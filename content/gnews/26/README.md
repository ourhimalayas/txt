## 台风雷伊过后 菲律宾死亡人数猛升

撰稿：东京樱花团/上善若水
![](https://assets.gnews.org/wp-content/uploads/2021/12/image-1083.png)[图片来源](https://www.dw.com/zh/%E5%8F%B0%E9%A3%8E%E9%9B%B7%E4%BC%8A%E8%BF%87%E5%90%8E-%E8%8F%B2%E5%BE%8B%E5%AE%BE%E6%AD%BB%E4%BA%A1%E4%BA%BA%E6%95%B0%E7%8C%9B%E5%8D%87/a-60179960)
根据最新消息，台风雷伊（Rai）已经造成超过300人丧生，目前仍有许多人失踪。灾害管理机构表示，迄今为止确认的大多数遇难者是被倒下的树木或者倒塌建筑物的碎片砸死的。还有一些人被淹死。在菲律宾许多地方，电力供应和通讯被切断。周日（12月19日），一些受影响地区的饮用水等物品分发点排起了长队。菲律宾当局表示，这次的受灾程度可以与2013年超强台风海燕相提并论。当时，菲律宾有超过7300人遇难、失踪。这场今年最强劲的热带风暴之一于12月16日登陆菲律宾，风速高达每小时195公里。它在登陆后已经有所减弱，从最高的5级风暴降为3级。台风雷伊在菲律宾南部和中部造成山体滑坡、洪水等灾害。救灾人员昼夜不停地进行抢险工作，成千上万的人被转移安置。

“我们失去了自己的家园。墙壁和屋顶被奥黛特（雷伊台风的本地命名）像纸片一样撕扯和吹落。”位于菲律宾东南部的迪纳加群岛省长巴奥(Kaka Bag-ao)在Facebook上说，“我们的食物和饮水供应捉襟见肘。”国家救灾部门发言人蒂姆巴尔（Mark Timbal）告诉法新社，超过18000名士兵和消防人员将加入受影响最严重地区的救援工作。雷伊台风对其所经城镇造成极大的破坏，树木被连根拔起，屋顶掀掉，电线被切断。

暴风雨过后，多个村庄被淹没在水中。棉兰老岛（菲律宾第二大岛）市长周四（12月17日）告诉本地媒体，台风雷伊该岛东北部的苏里高市扫荡数个小时，造成严重破坏。苏里高市有大约17万人口。菲律宾总统府发言人诺格拉斯(Karlo Nograles)此前说，超过33.2万人已经从高风险村庄撤离，其中大约1.5万人在疏散中心避险。为了避免台风造成的破坏，数十个航班被取消，数十个港口暂时关闭。雷伊是一场超级台风，相当于美国的5级飓风，周四对菲律宾锡亚高岛进行了猛烈的袭击。 到了周五，该台风风速已经放缓到每小时155公里。菲律宾风暴预警部门称，当地时间周五下午1点，风暴眼位于巴拉望岛省附近，正向西移向南中国海。在菲律宾，大多数风暴发生于7月至10月之间。而雷伊则是个例外，到来相对较迟。
