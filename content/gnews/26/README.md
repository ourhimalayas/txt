![](https://assets.gnews.org/wp-content/uploads/2022/01/A-1-2-scaled.jpg)
四川省自贡市地处墙国西南，自2015年起耗时六年，历经两个周期，终于在2020年获得了所谓的全国“文明城市”称号，只是这种涂脂抹粉的“荣誉”完全印证出中共的假与骗。在这座“文明城市”的街头巷尾，笔者亲眼目睹了中共当地政府如何造假，欺上瞒下。

在城市主干道的路边，只要是临街面的的破旧房子一概实行贴砖、刷涂料、做假阳台等翻新手段，使其能达到“文明城市”的标准。而在房子的背街一面，依旧破败，裸露着其真实破旧的面目。房子的内部设施更不会有任何实质性的改善，人们仍然生活在破败肮脏的环境里，经常走近楼道就能闻到一股臭味，有些地方下水道的污水甚至漫到了地面。
![](https://assets.gnews.org/wp-content/uploads/2022/01/Cok.jpg)![](https://assets.gnews.org/wp-content/uploads/2022/01/A-1.jpg)![](https://assets.gnews.org/wp-content/uploads/2022/01/A2ok-scaled.jpg)
在某些商业街，通常都有占道经营的情况，再宽敞的店铺都会把商品摆到店外一两米；小商小贩更是满街随走随买。当地政府在做网报资料时，把所有占道经营的商户全部都赶回店铺之内，把走街过巷的小商小贩也都赶到犄角旮旯里不准出来，商业步行街显出前所未有的“整洁”。但是一旦拍照完成之后，一切又恢复原样，待到审核单位现场检测时，清除占道经营和驱逐小商贩的流程又再重复一遍。

据申报“文明城市”的负责人说，自贡市在第一个申报周期里，花了三年时间申报都未成功，主要就是因为网报资料做得不好。其实无论网报资料还是后期的现场检测，都是一个作假和欺骗的过程。所以自贡市政府在第二个申报周期里做足了假，也终于拿到了“文明城市”的称号。

“文明城市”次年都要进行例行检查，自贡市需要接受“对标”检查。于是，2021年12月，笔者又再次目睹了迎检过程中检查与被检查的假和骗。迎检单位为此专门建立了微信群，派人全程跟随检查组，随时在群里报告检查组的行踪、车牌号、人员情况等，提前几天就对检查范围内的文化娱乐场所、公共文化服务站点等相关单位进行培训，要求熟记“社会主义核心价值观”，派出能背诵该价值观的人于检查当天站立于显眼位置，以便检查方提问时可做到对答如流；网吧和酒店等，则要求不能出现烟头，扫码和测温做到滴水不漏。等检查完毕，确认检查队已到达别的城市，立刻解散了迎检微信群。
![](https://assets.gnews.org/wp-content/uploads/2022/01/ok-scaled.jpg)
中共明目张胆对烂房子的外墙贴砖刷墙漆做假，而背街面的烂砖烂瓦裸露，烂房子内部依然腐烂，臭污水横流，所以政府根本就不是为了改善民生。令人难过的是，住着烂房子的人却是那样的顺从，很可能连一句质疑的话都没有问过，更没有想过这房子是谁的，既然中共并不打算从根本上改变房子的现状，又有什么权利对私人房产的外观进行改造？！

笔者家人前几日到迎检过的网吧玩网游，说网吧里已经允许吸烟，允许不戴口罩，这情形距离检查队伍离开自贡市还不到一个月。

中共的作假和欺骗，自它成立起延续到今天已到了无以复加的地步，新闻假、外交假、国防假、体育假、人口数据假、经济数据假、粮食假、文化假、就业率假、脱贫假、股市假、教育假、食品卫生假、商品假、药品假……就连演唱会现场都是放录音对嘴型，不知道墙国还有什么是真的？
