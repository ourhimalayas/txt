**1.****日本13县开始防止蔓延重点措施**

[据NHK 1月19日报道，为了应对中共病毒（新冠病毒covid-19）的扩散，日本政府已决定自1月21日至2月13日期间，将13县列入 “防止疫情蔓延重点措施 “地区。东京都要求餐馆缩短营业时间，选择营业时间带及是否供应酒水 。另一方面，优先措施并不要求商业机构和其他非餐饮机构缩短营业时间，而是要求遵守该行业的行为准则。对于大型活动，若不超过1万人，最多允许接纳半数，如果超过1万，则封顶5000。](https://www3.nhk.or.jp/news/html/20220119/k10013438191000.html)

**2.****文部省首议日本有关延长国际空间站的运营政策**

[据NHK 1月19日报道，受美国公开发表，将把包括日本在内15个国家参与的国际空间站的运作延长到2030年，文部省下一个委员会首次开会讨论有关日本的方针政策。国际空间站，由包括日本在内的15个国家参与，计划运营至2024年。但美国国家航空航天局（NASA）已宣布将其运行时间再延长六年，至2030年。该委员会还将审查空间站的运营成本和迄今取得的成果，并将在今年4月前最终确定其政策。](https://www3.nhk.or.jp/news/html/20220119/k10013439591000.html?utm_int=news-new_contents_list-items_024)

**3.****鸠山总政务官感染中共病毒（新冠病毒covid-19）**

[据NHK 1月19日报道，总务省已确认，总政务官鸠山先生感染了中共病毒（新冠病毒covid-19）。根据总务省的公布，总务省政务官鸠山昨晚（18日）发烧38.5摄氏度，19日接受了PCR测试，结果显示呈阳性。鸠山先生在17日出席了众议院全体会议后，回到了位于福冈县久留米市的故乡，目前正在久留米市的家中休养。鸠山总政务官去年已完成了两针疫苗，他是第18位被确认感染的国会议员。](https://www3.nhk.or.jp/news/html/20220119/k10013439821000.html?utm_int=news-politics_contents_list-items_002)

**4.****日本将向汤加提供超过100万美元的赠款援助**

[据NHK 1月19日报道，在南太平洋汤加附近发生大规模火山爆发后，磯崎副官房长官宣布，日本将提供超过100万美元的紧急拨款援助，以支持当地的恢复和重建。除财政合作外，日本政府正在协调提供饮用水及清除火山灰的工具等紧急救援物资，对于恢复该国交通运输，考虑派遣自卫队作为国际紧急救援队实施作业。](https://www3.nhk.or.jp/news/html/20220119/k10013439451000.html?utm_int=news-new_contents_list-items_048)

**5.****东京新增7377例确诊**

[据NHK 1月19日报道，当日东京的确诊感染人数为7377人，比去年8月13日的5908人的前纪录多出近1500人。按感染途径划分，在学校和老人护理设施等“设施内感染“的人数连续第三天达历史新高。至19日的每周平均数为4598.4，是前一周数字的四倍，也是自去年8月28日以来首次一周平均数超过4000。](https://www3.nhk.or.jp/news/html/20220119/k10013438651000.html)

**6.****去年赴日游客人数为24万人**

[据JIJI 1月19日报道，日本观光局发布，由于中共病毒（新冠病毒covid-19）扩散后的入境限制，2021年日本的外国游客数量为24.59万人，是1964年开始统计以来的最低数字。 同19年相比下降少99.2%，同20年相比亦下降94.0%。](https://www.jiji.com/jc/article?k=2022011901075&amp;g=eco)
