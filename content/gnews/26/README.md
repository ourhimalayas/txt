与此同时，一项新的临时停火协议，以疏散被围困的数千名。在南部城市马里乌波尔的平民。该市似乎在短短几个小时后就溃败了。这已经是连续第二天失败了。我们在乌克兰的记者夏洛特·加拉格尔，为我们带来了最新的消息。

俄罗斯对伊尔平的暴力袭击造成恐慌。这个靠近首都的城市正在经历激烈的战斗，越来越多的人试图逃离。许多人正在前往靠近波兰边境的城市利沃夫。这里的火车站被人群挤得水泄不通。这些是来自孤儿院的儿童，是最后离开的一些人。

他们中的一个看护人[说]，实在是太累了。她说，我的心都碎了。太难了。在离火车站几英里的地方。正在分发武器。不是发给士兵和警察，而是那些想保卫他们城市、家庭和乌克兰的市民。这一幕在全国范围内重复上演。

马里乌波尔的废弃街道。一个被俄罗斯军队包围的城市。大约20万人已经没有食物、水和电力，生活在黑暗的地窖里。

昨天，逃跑的希望被俄罗斯的炮击打破。达成的停火协议，允许人们离开。尽管莫斯科指责乌克兰。今天，将再次尝试疏散平民。市长说他们一直受到威胁。在战争的最初几天，我们数了数受害者有几十人。今天，数以百计，很快就会到几千人了。他们甚至不给我们机会计算伤者和死者人数。因为炮击已经六天没有停止过了。这就是人们害怕听到的从天而降的声音。
