**俄乌前沿今日战况**

- 乌克兰总统泽连斯基2月28日签署了要求该国加入欧盟的正式请求。欧盟外交和安全政策代表正在召集欧盟国防部长会晤，讨论议程将集中在”紧急需求”和协调欧盟对乌克兰人的援助上
- 俄罗斯和乌克兰官员之间的谈判正在白俄罗斯边界进行中。这是自俄罗斯入侵以来，俄乌双方第一次见面，但突破的希望并不高。在会谈之前，乌克兰表示希望停火和俄罗斯撤军，而克里姆林宫表示希望达成一项符合双方利益的协议。乌克兰总统泽伦斯基在周日的讲话中表示，即使机会很小，他们也应该尝试利用这个机会，以避免有人责怪乌克兰没去试图停止战争
- 基辅国防部声称，在乌克兰战斗的头四天中，已有 5,000 多名俄罗斯士兵丧生。乌克兰官员在 Facebook 上发布的一份声明中说，大约 5，300 名俄罗斯士兵被打死，并声称乌克兰军队摧毁了 191 辆坦克、29 架战斗机、29 架直升机和 816 辆装甲运兵车
- 基辅内政部长顾问安东·赫拉申科在脸书上写道，莫斯科军队用格拉德导弹轰炸了居民区，他说：”数十人丧生，数百人受伤！”全世界都必须看到这种恐怖！”
- 联合国人权事务负责人米歇尔·巴切莱特今天早些时候表示，至少有 102 名平民丧生，另有 304 人受伤。”这些平民中的大多数是被具有广泛影响的爆炸性武器杀死的，包括重型火炮和多管火箭系统的炮击以及空袭。我担心真实数字要高得多，”巴切莱特说
- 此前我们报道称，乌克兰为俄罗斯士兵家属开通了热线电话。基辅独立报报道热线至今已经接到”数百个电话”。这项名为”从乌克兰活着回来”的服务于上周六由乌克兰国防部启动。一位官员说：”通过这条电话线，您可以了解您的亲人是否还活着、被俘、受伤，您可以决定何时以及如何带走已故亲人的尸体。”


**美欧国家今日动态**
