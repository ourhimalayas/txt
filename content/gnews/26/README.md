根据同花顺财经10月11日报道，随着美国消费支出放缓、美联储发出缩表信号，高盛经济学家上周日再度下调对美国今明两年经济增长预期，这已是三个月以来的第3次。

以Jan Hatzius为首的高盛经济学家们在报告中写道，预计美国今年经济增速将放缓至5.6%，略低于此前的5.7%，同时预计明年经济增速为4%，较此前的4.4%有所削减。不过，高盛上调了2023、24年的经济增速预期，这让总体的经济增速与此前预计的大差不差。就在今年年初，高盛还预计美国今年增速为6.6%。然而，自8月开始，高盛以德尔塔变种、财政支持减弱、消费放缓为由不断下调增速预期。

高盛在报告中指出：“根据我们对消费增速中关键指标的最新评估，包括经济开放、财政刺激、积蓄和财富效应，同时考虑到更加持久的疫情，这影响了一些对疫情敏感的服务性消费支出，预计整体消费支出的复苏有更长的路要走。”
