乌克兰总统沃洛基米尔-泽连斯基于4月4日周一，进行了一次访问，以 “向世界展示 “布卡镇的万人坑和其他暴行，他发誓要追究俄罗斯对待乌克兰人 “比动物还糟糕 “的责任。
![](https://assets.gnews.org/wp-content/uploads/2022/04/图片-1.jpg)RONALDO SCHEMIDT/AFP via Getty Images
据CNN报道，泽连斯基在基辅北郊对记者说：”我们希望你们向世界展示这里发生的一切。俄罗斯军队所做的这些，俄罗斯联邦在和平的乌克兰所做的这些。你们看到的这些都是平民，这一点很重要。”

“你在周围看到的，他们对这个现代城镇所做的一切，是俄罗斯军队的特点，他们对待人民比动物还糟糕。据乌克兰媒体UNIAN报道，泽连斯基说：”这些是战争罪，这将被世界承认为种族灭绝。

布卡市副市长塔拉斯-沙普拉夫斯基说，克里姆林宫部队上周末撤出后，在那里发现了大约50名俄罗斯军队法外处决的受害者。

访问布卡的路透社记者还在当地教堂看到了一个“万人坑”，手和脚从堆在上面的红色粘土中探出。该通讯社报道说，发现了三名身着便衣的男子的尸体，每个人都是头部中弹。其中一人双手被绑在背后，脸上有火药烧伤，表明是被近距离枪击。

乌克兰总检察长Iryna Venediktova周一表示，最近从俄罗斯军队手中夺回的基辅地区城镇中，410名平民的尸体已经被移走。

媒体公布的这些照片非常触目惊心，有被坦克压扁的汽车、散落在高速公路边的断臂、躺在泥地里的尸体，还有死在自家屋外的一家人，说尸横遍野，一点不为过。从古自今，任何战争都是残酷的，生灵遭到涂炭，百姓流离失所，战争对每一个人带来的创伤都是难以抚平的。普京为了恢复前苏联的“大一统”而发动了这场无谓的侵略战争，让多少家庭破碎，让多少乌克兰人流离失所。不管是俄罗斯的军队，还是乌克兰的军队都是死伤惨重，而乌克兰的无辜平民被杀戮实在令人愤慨。

美国总统拜登周一说：”我们必须继续向乌克兰提供他们所需的武器以继续战斗。我们必须收集所有的细节，以便这可以成为一个实际的–进行战争罪审判。”

一旦俄乌战争结束，等待普京的将是战争罪的审判。同样这也是对以习领导的中共政府的警示，如果中共政府胆敢对台湾发动战争，等待习的也将是战争罪的审判。
