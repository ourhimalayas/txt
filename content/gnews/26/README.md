没有气候紧急情况。这对于任何关注的人来说都是显而易见的。甚至投票赞成宣布该市进入气候紧急状态的卡尔加里市议会成员也承认，这是一个象征性的声明。他们是对的，就是以任何有意义的方式帮助环境而言，这个宣言是空洞的美德信号。然而，虚假的美德并不是所有这些紧急声明的信号。很明显，进步的政治家希望我们永远生活在恐惧的状态中，这样他们就可以做他们想做的事并获得更多的权力。我们正在从一个紧急情况跳到下一个紧急情况。

新冠导致的当地紧急状态终于在卡尔加里失效，虽然不幸的是这并不意味着口罩和疫苗的强制要求结束，但它确实取消了市议会对新冠措施发布指令的部分权力。现在，正如无数所谓的“阴谋论者”所预测的那样，气候的危言耸听是新的恐惧原因，我们又一次陷入了理事会宣布的紧急情况。

我们可能需要数百年的时间才能完全过渡到可再生能源，但这些政客抨击为汽车提供动力并为家庭供暖的阿尔伯塔省石油。有多少加拿大政客在石油输出国组织的石油中大肆抨击管道？阿尔伯塔省生产的一些世界上最清洁、最合乎道德的石油。但在这里它没有意义。

大卫铃木、比尔·奈和格蕾塔·桑伯格等人的职业生涯都是通过传播对气候的恐惧来实现的。卡尔加里市市长乔蒂·冈德克（Jyoti Gondek）在几年前还是阿尔伯塔省能源的主要倡导者，现在他也表达了对生态歇斯底里的看法。什么时候停止？
