中（共）国国家主席习近平2021年7月1日（中共建党百年庆典）的讲话因其对世界和平的威胁和潜在的大男子主义而被全世界广泛报道。但基本上没有报道的是发表于中（共）国大陆杂志的一篇文章，其中展示了中（共）国将如何对台湾发动突然袭击。

《南华早报》在周五的一篇文章中报道了中（共）国将部署以武力攻占台湾的可行性战略。《南华早报》称：“文章和视频详细描绘了该岛如何为解放军登陆做准备，恰逢中共的百年庆典。该文章并未探讨可能的反击或其他关键角色在其方案中的反应”。

这篇文章发表在中国船舶工业集团公司发行的北京月刊《舰船知识》上，以纪念中国共产党成立一百周年。文中详细描述了中（共）国军队在登陆台湾海岸前将发起的三个阶段的攻击方案。

韩国报纸《韩国时报》称，中（共）国对台湾攻击的第一阶段是发起弹道导弹攻击，以摧毁台湾的机场、雷达、防空导弹基地和指挥中心等信息搜集与决策设施。这一阶段可能使用东风-16短程弹道导弹来完成，也可能使用中（共）国的H-6轰炸机和J-16战斗机对台湾军港进行袭击。

第二阶段攻击将采取密集的巡航导弹轰炸，例如从陆地、船舶和潜艇上发射YJ-91和CJ-10等导弹。据中（共）国杂志报道，攻击目标将包括军事基地、军火库、通信基础设施和公路交汇处。

第三轮攻击将是将是陆基火箭部队的炮火打击。

这三个阶段将为中共解放军海军陆战队和两栖登陆部队入侵台湾进行最后打击和实现“统一”清除最后的障碍。该文章还向台湾人传递了一个不祥的信息：“……我们必须郑重警告一些人，台湾独立的道路只会走向死胡同”。

这篇发表在《舰船知识》的文章强化了习在中共百年庆典上传达的信息，即中（共）国致力于台湾与大陆的 “统一”。

在过去的几年里，中（共）国在台湾领空进行了多次威胁性飞行。6月中旬，中（共）国向台湾出动28架次战斗机，创造了震慑这座小岛的新纪录。中共认为台湾是一个叛省，必须被强行回归大陆。
