# 重要事件

- **[0717 Zero Hedge：欧盟和美国领导人就中国制定人工智能标准发出警告](https://www.zerohedge.com/geopolitical/eu-us-leaders-warn-about-china-setting-standards-ai)**
美国和欧洲高层领导人最近谈到，民主国家应该制定人工智能（AI）的规则，而不是像中国这样的独裁政权。


“美国商务部长吉娜-雷蒙多6月13日在华盛顿举行的全球新兴技术峰会上说：”我们不能让中国编写围绕人工智能的道路规则。该活动由国家人工智能安全委员会（NSCAI）组织。雷蒙多说，相反，规则应该由美国和其志同道合的盟友来写。

她解释说：”重要的是，人工智能的发展方式和监管方式要符合我们的民主价值观，即自由和开放，保护我们的知识产权，尊重人权和尊重隐私。”

出席峰会的还有国务卿安东尼-布林肯和国防部长劳埃德-奥斯汀。

布林肯特别指出了美国和中国之间的差异–前者将 “普遍权利和民主价值 “置于其所有创新的中心，而后者在人工智能等新兴技术方面未能 “区分民用和军用 “应用。”因此，我们必须以不同的方式思考如何保护我们的创新和产业免受这种滥用，”他补充说。

奥斯汀说，在数据驱动的决策和人机协作等领域，人工智能对美国未来的军事优势将至关重要。但他补充说，与中国相比，美国将采取不同的方法来发展人工智能。

“在人工智能领域和其他许多领域一样，我们明白中国是我们的节奏挑战，”奥斯汀说。”我们将在竞争中获胜，但我们将以正确的方式来做。我们不会在安全、安保或道德方面偷工减料。”

五角大楼负责人补充说：”人工智能将改变军事行动的许多事情，但没有什么能改变美国对战争法和我们民主原则的承诺。”

近年来，中国共产党（CCP）多次将人工智能确定为其国家发展的首要任务之一。2015年，它是中国被称为 “中国制造2025 “的工业路线图中列出的关键产业之一。

“欧盟委员会执行副主席玛格丽特-维斯塔格说：”就像在中国一样，人工智能可能被滥用于大规模监控或社会控制。

- **[7.16 Zero Hedge：因世界意识到中共的弊端，中共被 “逼入绝境”](https://www.zerohedge.com/geopolitical/chinese-communist-party-cornered-world-awakens-its-abuses)**

“特克尔在年度国际宗教自由峰会上告诉大纪元记者：”在最近的记忆中，中国从未被如此孤立，而这种孤立正使他们变得好战。”他补充说：”这就是为什么他们要与难民营的幸存者挑衅。

特克尔指的是中国官员试图公开羞辱那些亲口讲述在新疆拘留营中遭受性虐待的维吾尔族妇女，在那里有多达100万维吾尔族和其他穆斯林少数民族被关押在政权声称的 “反恐 “运动中。

在2月份的新闻发布会上，中国外交部发言人举起前被拘留者的照片，称其中一位名叫Tursunay Ziyawudun的人为 “演员”，并指责她 “散布谎言”。特克尔说，这种行为出自一名政府官员之口，与 “人人害怕的国家 “的形象不符。

Ziyawudun在峰会第二天的开幕式上分享了她的故事。齐亚乌敦说，从2018年开始，她在营地的一年里，看到警察把维吾尔族妇女从牢房里带走，做 “他们想做的事”。一些妇女被带回来时已接近死亡边缘。还有一些人已经失去了理智。她目睹了一名20多岁的维吾尔族妇女被强奸，而三名汉族警察也对她做了同样的事情。

特克尔指出，周三几乎每个发言者都强调了新疆的镇压，继美国之后，越来越多的国家已经承认新疆的镇压是 “种族灭绝”。”这是北京方面的误判，”特克尔在谈到该政权的人权侵犯行为时说。”他们认为他们可以逍遥法外，就像他们多年来对法轮功学员和对西藏人所做的那样。”

全球对中国人权暴行的呼声越来越高，立法者们在各自的政府内部推动抵制2022年北京奥运会。

在过去的一周里，美国将十多个在帮助新疆地区的虐待行为和该政权的军事现代化中发挥作用的中国实体列入黑名单，同时它还对新疆的商业风险发出了警告。强制摘取器官是国家认可的做法，主要针对法轮功学员，但也包括其他良心犯，也受到了更多的关注。

北京方面也没有忽视这种不断增加的压力。特克尔认为，中国领导人习近平呼吁共产党官员创造一个 “可爱的 “中国形象–这本身就是一个不安全的信号。”他说：”他们被逼到了绝境。

- **[07.17 BBC NEWS： 中共当局称未接种疫苗的父母不能送孩子上学](https://www.bbc.com/news/world-asia-china-57859356)**
中国的一些地方政府表示，除非整个家庭完全接种疫苗，否则学生在9月份将不被允许回到学校。一些城市还表示，人们需要接种疫苗才能进入医院和超市等公共场所。


中国加入了包括法国和希腊在内的其他国家的行列，这些国家在某些领域强制要求接种疫苗。中国已经设定了一个国家目标，即在今年年底前为64%的人口接种疫苗。

本周早些时候，广西省发布的一份通知称：”那些尚未接种的人应该抓紧时间接种，以免影响你的孩子重返学校。”该通知补充说，这一规定适用于所有年龄的学生。

包括江西和河南在内的其他省份的地方政府也发布了类似的指令，称只有家庭接种了疫苗的学生才可以参加秋季新学期的学习。但是各地区的规定有很大不同。

在河北北部的萍乡市，12至17岁的学生除非完全接种疫苗，否则不允许返校。目前还不清楚这是否适用于其他年龄段的学生。

同时，陕西省韩城表示，未接种疫苗的人将不被允许进入酒店、餐馆和娱乐场所等。

各个城市的截止日期各不相同，但大多数城市都将7月底作为人们在严格措施生效之前接受疫苗接种的最后期限。

这些公告在网上遭到了反击，网民称其 “不公平”。

迄今为止，中国已经注射了超过14亿剂的Covid-19疫苗–尽管其卫生委员会没有说有多少人已经完全接种了疫苗。

- **[07.17 AP News：俄罗斯将在莫斯科航展上发布新的战机](https://apnews.com/article/lifestyle-business-europe-russia-moscow-aec52e68fc3b482dfc6a4132a2bafcbc)**
俄罗斯飞机制造商表示，他们将在下周开幕的莫斯科航展上展示一架潜在的新型战斗机。


这架隐藏在防水布下的新战机被拍到被拖到莫斯科郊外朱科夫斯基机场对面的一个停车点，那里的MAKS-2021国际航空和航天沙龙星期二开幕。俄罗斯总统弗拉基米尔-普京将参观该展览的开幕式。

俄罗斯媒体报道说，这架新飞机是由苏霍伊飞机制造商在轻型战术战斗机的开发计划中制造的。与俄罗斯最新的苏-57双引擎隐形战斗机不同，新飞机更小，只有一个引擎。

苏-57的制造是为了配合美国的F-22 “猛禽 “隐形战斗机，但与自2005年以来一直在服役的美国飞机不同，其系列化生产刚刚开始，旨在使其具有超音速巡航能力的新发动机仍在开发中。这款新战机的名称不详，也没有关于其能力和部署前景的信息。

未来的俄罗斯战斗机似乎打算与美国F-35 “闪电II “战斗机竞争，后者于2015年开始服役。俄罗斯希望最终向外国客户提供这种新飞机。

包括俄罗斯飞机制造商在内的国有企业Rostec表示，”根本性的新军用飞机 “将在周二的莫斯科航展上亮相。”它说：”俄罗斯是世界上少数几个拥有生产先进飞机系统的全周期能力的国家之一，也是公认的战斗飞机制造领域的潮流引领者。

克里姆林宫已将该国武装部队的现代化作为首要任务，因为与西方的关系出现了激烈的紧张，在2014年莫斯科吞并乌克兰克里米亚半岛、指责俄罗斯干预选举、黑客攻击和其他刺激因素之后，西方的关系已降至冷战后的低点。

- **[07.16 CNBC：参议院民主党人的3.5万亿美元预算如何应对气候变化](https://www.cnbc.com/2021/07/16/how-senate-democrats-3point5-trillion-budget-tackles-climate-change.html)**
乔-拜登总统和参议院民主党人发誓要推进一个3.5万亿美元的预算决议框架，为清洁能源转型和应对气候变化的政策提供资金。


该蓝图几乎包含了总统的美国家庭计划的所有内容–包括对儿童保育、带薪休假和教育的资助–是在拜登的气候建议在与参议院共和党人的谈判中被从两党的基础设施协议中削减之后提出的。

该计划涉及对清洁能源和电动汽车的税收优惠，以及对经济从化石燃料向风能和太阳能等可再生资源过渡的重大投资。该决议还提出了一个清洁能源标准，要求美国的部分电力来自可再生能源。

这种授权得到了环境活动家和科学家的广泛支持，他们说这对实现总统在未来十年内将碳排放减半的承诺至关重要，并使美国在2050年前实现碳中和。

民主党人希望在今年夏天晚些时候以党派投票的方式通过该法案。如果预算决议被签署成为法律，这将是美国历史上应对气候变化的最大立法推动。

国会正在与1.2万亿美元的两党基础设施计划同步进行这项决议，该计划仍在起草之中。

参议院多数党领袖查克-舒默（Chuck Schumer）表示，他希望在参议院8月休会前对预算决议和基础设施法案进行表决。
