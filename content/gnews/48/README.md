#### 编译：小红帽

根据《RT新闻》1月6日报道，**法国的在线隐私监管机构已经命令谷歌（Google）和脸书（Facebook）支付大约2.1亿欧元（2.37亿美元）的费用，以对这两家公司在其网站上使用数据跟踪“cookies”的做法进行罚款。**

法国国家信息学和自由委员会（CNIL）周四在一份声明中宣布了这一举措，称谷歌将被要求在三个月内支付1.5亿欧元，而脸书则需要支付6000万欧元，否则将面临每天10万欧元的额外罚款。

该委员会说，这两家公司使用 “cookies”的方式–或在用户浏览网站时产生的少量数据，可用于跟踪他们的活动–“影响了同意自由”，因为脸书和谷歌让网民更容易授权数据跟踪而不是拒绝它。

该委员会数据保护和制裁团队负责人的卡琳·基弗（Karin Kiefer）说：“当你接受cookies时，只需一次点击就可以完成。而拒绝cookies应该和接受它们一样简单。”

“这些做法违反了《法国数据保护法》，现命令这些公司‘向位于法国的互联网用户提供拒绝cookies的方式’。”

谷歌和脸书都发表了声明，发誓要与法国当局合作解决这个问题，尽管后者坚持认为其“cookie同意控制使人们可以更好地控制他们的数据”，并对法国国家信息学和自由委员会提出的同意问题提出异议。谷歌也辩称：“人们相信我们会尊重他们的隐私权并保证他们的安全”，但仍表示它将进行 “进一步改革” “以遵守该命令。
