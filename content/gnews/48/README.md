《市政大厅》（townhall）网站发表文章，“众议院民主党人否决了解密中共病毒（COVID）起源情报的法案”。

全文如下：

周二（7月20日）晚上，众议院以216票对207（的投票结果），民主党人否决了要求国家情报总监解密武汉中共病毒疫情源头相关信息的议案，特别是关于武汉病毒研究所在疫情爆发中可能扮演的角色的信息。

乔希·霍利（Josh Hawley）和迈克·布劳恩（Mike Braun）在美国参议院提出了中共病毒溯源法案，并于5月获得一致通过。

在周二晚上民主党投票否决《中共病毒起源法案》（the COVID-19 Origin Act）前不久，众议员迈克尔·伯吉斯（Michael Burgess）将此法案提交给了众议院审议，布拉德·温斯特鲁普（Brad Wenstrup）和达林·拉胡德（Darin LaHood）解释了它的重要性。

“最好的消毒剂是阳光，这是我们今天所能提供的。”温斯特鲁普解释了《中共病毒起源法案》，“我们必须确定中共病毒的确切来源，因为它对于预防未来类似的疫情至关重要。”

温斯特鲁普接着说：“我要强调的是，这项法案无论如何都没有争议。”事实上，它在5月份获得了参议院的一致同意——没有一位参议员反对。不是参议员特德·克鲁兹（Ted Cruz）或兰德·保罗（Rand Paul），也不是伯尼·桑德斯（Bernie Sanders）或伊丽莎白·沃伦（Elizabeth Warren），如果这四位议员能够参与这项法案，我们难道不应该也这样做吗？”

拉胡德议员进一步解释了立法的目的，他说，“这将要求拜登政府国家情报总监解密与武汉病毒研究所（又称武汉实验室）和中共病毒起源之间任何与潜在联系有关的情报信息，以便更好地准备和避免未来的疫情。”他补充说：“归根结底，美国人应该充分了解中共病毒疫情的起源。”

今年4月，霍利参议员在介绍参议院版本的《中共病毒起源法案》时指出，“一年多来，任何质询武汉研究所是否是病毒来源的人，均被冠以阴谋论者的称号。“他补充说，”全世界都需要知道这一疫情是否是武汉实验室疏忽大意的产物，但（中国共产党）已经尽一切努力阻止了可信的调查，这就是为什么拜登政府必须解密它所知道的，关于武汉实验室和北京试图掩盖疫情起源的信息。”
