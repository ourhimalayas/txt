对，北京开那个十一届六中全会那天下的那个雪呀，是假雪，fake snow。中南坑为了营造“瑞雪兆丰年”，动用了气象造雪。世界上的其他国家都具备这项技术，人家都不敢去用，因为会造成的太空污染，大气污染，是无法想象的。然而共产党就是敢这么干，管什么空气污染不污染，政治第一。

北京如期在六中全会期间下起了共产党的“瑞雪”，我在上篇文章[“北京下雪了”](https://gnews.org/zh-hans/1644200/)中提到，共产党操纵不来天气，我就是那傻乎乎的老百姓。共产党已经狂妄到敢欺骗上天，还啥天气不天气的，想要晴天就晴天，想要雪天就雪天。我管你气候啊，污染啊，只要天气不符合地面上的政治需求，那一定会给你管控天气。管天管地管空气，让天气也要符合党的利益，让天气也来爱党。怪不得每年的10月1号国庆节，永远不会下雨，尤其是碰上重要的年景要大阅兵的时候，天安门从来都是阳光灿烂。而在10月1号之前，尤其9月月底的时候，记得读书的时候那几天总是要开校运会，每年的校运会几乎都在瓢泼大雨里举办的。

最近几天六中全会已经开完了，北京的瑞雪也下完了，然而全国各地尤其内蒙古东北等地，遭遇了多严重的雪灾？从社交媒体上看到，很多人的家门整个被堵上了，有的住楼房的，下雪直接把一楼埋了，还有的地方城市公共运输全部停止，市民走路上班，不知道这些地区的大雪是否和北京刻意造的假雪有关，但能够看到的是给老百姓的生活带来极大的不便。你老百姓算什么东西，共产党都敢欺天。

![](https://assets.gnews.org/wp-content/uploads/2021/11/2-54.jpg)大雪封门、走路上班 / 图源：网路

六中全会的这场“瑞雪”，兆示什么样的丰年？整个世界都不知道共产党此次会议的内容，而受雇中共的外宣媒体，让报道啥就报道啥，一句真话没有。什么粮食大丰收，什么中共国和全世界100多个国家建立全天候战略合作伙伴关系。全世界的经济都可能因为共产党房地产的美元债到期无法偿还而崩塌，共产党就还敢对全世界撒谎。如果没有文贵先生，没有爆料革命新中国联邦，全世界都可能被欺骗，被屠杀，死的时候不知道怎么死的。北京下假雪这事儿，不听爆料革命，相信我永远不会知道。好比中共国篡改过的历史，经过那么多年，如果当事人全都死了，还找谁去查找真相去？爆料革命给我的成长，就是我在上篇文章的认知错误，这么短的时间就能得到验证，得到修正。这么快速及时的信息给到我，从而改变了我对天气，对共产党邪恶的不断深入认知。这仅仅是对个人来说的一个小方面，尚且能得到如此大的改变，如果一个国家，比如美国在疫情一开始的时候就打算查清楚这个病毒来源，而不是让共产党收买的媒体报道不实信息，那么哪有后来的疫苗，而且强制疫苗还死了这么多人？

这个六中全会，那张纸，11个第一。随便哪一条拿出来都能共产党死一万遍都不解恨的。共产党已经把中国人快全杀光了。全球75亿人，因CCP病毒导致的死亡是500万人，中共国14亿人，CCP病毒死亡人数是1370万。CCP病毒死亡人数1370万人，染病人数7000万人，单单就这两项加起来，算上背后的家庭人口，八九千万快一亿人口了。一个亿人口，1/14，共产党总人口到底有没有14亿？按照国外的统计，中共总人口只有10亿来计算的话，10个人里面就有一个人染了冠状病毒，这么多人染上了冠状病毒并且有的已经死了，共产党外宣媒体报道的数字，连个零头都没有。这些人口还不包含在全世界统一统计的感染人数和死亡人数中。还有那1100万癌症怎么来的，怎么一下子就突然多了那么多癌症，反正永远不会说那是疫苗造成的，疫苗比冠状病毒的危害要大得多得多，目前为止是1100万，到明年5月份会增加到多少？国内现在的情况，根据宣传，已经有一半以上的人打了两针疫苗了。虽然说有很多人打的是生理盐水假疫苗，但是这个比例仅仅是江浙沪地区，仅仅是总人口的十分之一多一点，况且又不全是打的生理盐水。

在过去的17个月中，中国人成为全球最不喜欢的人种，排华情绪正在抬头。过去的17个月，疫情到美国是去年的三四月份，差不多，就是在疫情和疫苗期间，中国人成为全球最讨厌的人。共产党捆绑着中国人到处叫嚣，动不动就是人民是铜墙铁壁，人民要给共产党挡子弹。中国人被共产党害死这么多人，还被共产党绑架着打算让全世界人来讨厌，中国人到底做错了什么？
