## 疫苗灾难已在美国顶级医院显现
` GTV新闻访谈` [轉載自GNews](https://gnews.org/zh-hans/1921606/)

作者：纽约香草山文艺部 – 宁文静
![This image has an empty alt attribute; its file name is Screen-Shot-2021-08-30-at-7.37.59-PM.png](https://assets.gnews.org/wp-content/uploads/2021/08/Screen-Shot-2021-08-30-at-7.37.59-PM.png)
郭文贵先生在1月27日的大直播中披露，中共病毒疫苗所造成的灾难已经在医院的医护人员中显现。同时，这些顶级医生的亲身经历将极大可能促成各国疫苗政策的根本扭转。

![](https://assets.gnews.org/wp-content/uploads/2022/01/Slide3-13.jpeg)

郭先生提到，近期，国内外的医生战友接连爆料称，由于中共病毒疫苗的严重副作用，他们周围经常有同事，甚至是一直热衷推广疫苗的专家去世，这让医生群体感到困惑与害怕。其中，据一位在美国顶尖医院工作的战友透露，他所在的医院早先强迫所有员工接种中共病毒疫苗，这导致该院目前出现了异常高的员工死亡率：从2021年8月至今，该院已经有12名医护人员去世，当中甚至包括一位身体健康的顶级医生，这一现象引起了全院员工的恐慌。此外，新加坡的脑科顶尖医生战友也陈述了类似的情形：他的医院中，中共病毒疫苗造成了4位同事的死亡，还剥夺了该院院长的行动能力。

郭先生感慨，发生在医生群体中的中共病毒疫苗灾难是不幸但又是必然的，尤其是那些主导强制疫苗接种而改变了很多人命运的顶级专家，他们的遭遇，甚至死亡，可以让医学界、科学界人士醒悟并尽快改变现在的疫苗政策——这未必不是一件功德无量的事情。
