**据《jamiemetzl》作者：Jamie Metzl 2021年4月13日稿：**

2021年4月7日，由我组织的一群科学家和其他专家组成的团体发布了一封公开信，呼吁对所有Covid-19的起源假设进行全面和不受限制的调查，充分接触中国境内以及适当的境外相关记录、样本和人员。然而，中国外交部发言人赵立坚并不欢迎我们的建议，他也不认为这是一个更好地了解这一可怕的大流行病是如何开始的机会，以造福于所有人民和所有国家，而是选择攻击我们的团体以及我个人，同时表示中共国当局已经表现出模范式的“开放和透明”。

但是，中共国政府非但没有表现出可以防止或组织大流行病的那种公开性和透明度，反而从疫情爆发的最初几天就一直在努力压制最基本的信息。如果赵发言人认真对待他的政府对公开性和透明度的承诺，我请他回答以下问题。

1，要证明大流行病不是源于实验室的意外事件，最简单的方法是提供所有相关的实验室记录和武汉病毒研究所和其他从事冠状病毒研究的武汉实验室的生物样本的全面访问。为什么这种类型的访问被拒绝了？你们能否承诺立即提供这些资源？

2，尽管武汉病毒研究所的病毒数据库对于任何公开和透明地调查大流行病的起源都是特别重要的，但这些数据库在2019年9月被下线，无法访问。你能解释一下为什么中国隐藏了这些关键信息，不允许世界卫生组织组织的国际委员会访问这些数据库吗？

3，美国情报机构和国务院在2021年1月15日宣布了他们的联合调查结果，即中国人民解放军参与了武汉病毒研究所的机密研究。这一说法被武汉病毒研究所新发传染病中心主任石正丽断然否认。鉴于中国政府认为SARS-Cov-2病毒没有从武汉病毒研究所意外逃逸的大部分理由是基于石博士的可信度，这种可信度的破坏将引起对该大流行病来源的新的严重关切。你是否坚持石博士的说法，即中共国军方在疫情爆发前没有以任何方式参与武汉病毒研究所的工作？

4，所有政府都有保护自己不受不利信息影响的倾向，但只有一些政府会对媒体进行积极的压制。中国公民记者张赞于5月被中共国当局逮捕，因为他对大流行病的起源提出了尖锐的问题，并被指控”寻衅滋事“，于2020年12月28日被判处四年监禁。据Quartz报道。其他三名公民记者——陈秋实、方斌和李泽华——都在2月失踪，因为他们在大流行期间对武汉的报道开始在网上受到关注（尽管有些人后来又重新出现）。鉴于你对公开性和透明度的承诺，你是否愿意呼吁立即释放张某，并鼓励中国记者在了解这场可怕的悲剧是如何开始的过程中发挥更重要的作用？

5，2020年1月3日，中国国家卫生委员会命令中国的科研机构不得发布与新型冠状病毒有关的任何信息，并命令实验室将他们拥有的任何样本转移到指定的检测机构或销毁它们，对中国科学家形成了有效的禁言令，并持续到今天。您能否确认贵国政府支持中国科学家和其他人自由分享他们关于大流行病起源的知识和专长的权利？

6，在许多情况下，对某一特定危机有最深入了解的人，会因害怕受到影响而不敢说出真相。在这些情况下，保密的举报人机制可以为提供重要情报的举报人提供一个安全的途径。您能否承诺贵国政府支持建立一个国际举报人制度，使中国科学家和其他专家能够毫无顾虑地分享重要信息？

7，如你所知，已知与SARS-CoV-2病毒最接近的遗传亲属是RaTG13病毒，该病毒的样本来自武汉以南一千多英里的云南省墨江矿洞。2012年，在该矿洞暴露于蝙蝠粪便的六名中国矿工在暴露后出现了类似COVID-19的肺炎症状，其中三人死亡。后来，WIV（武毒所）从同一矿井中提取了其他8种冠状病毒，它们都与SARS-CoV-2病毒密切相关。尽管这一事实模式提出了一种明显的可能性，即通过在墨江矿区的广泛取样，可以发现有关该大流行病起源的重要线索，但进入该矿区的通道已被完全切断，而且从该矿区的样本已被强行从科学家手中拿走。你能承诺国际科学界将有充分的机会对墨江矿区进行研究和采样吗？

8，我们对疫情如何开始的科学理解，将通过不受限制地获取湖北省从2019年下半年开始的匿名健康记录和血液样本而得到大大加强。因为中共国政府已经在其他情况下提供了这些记录，很难想象为什么不能向国际专家提供这些资源。您能否确认，从2019年下半年开始，[世卫组织]组织的国际委员会将被允许匿名访问湖北省的所有相关健康记录和血样？

9，在2021年3月30日新冠病毒调查国际委员会/中国政府联合报告发布后，世卫组织总干事谭德塞-阿达诺姆博士表示，他认为该研究没有充分考虑到武汉的实验室意外事件有可能引发该大流行病，并声称他将任命更多专家对该假设进行更全面的研究。您能否保证贵国政府全力支持这一进程，并确认您将为这些专家提供签证和充分获得所有必要资源的机会？

10，所有的人和所有的国家都将受益于我们对这一毁灭性的大流行病的起源获得更深入的了解。然而，即使[世卫组织]组织的国际委员会成员承认，目前的研究过程也丝毫不构成这种全面和不受限制的调查类型。鉴于您对公开性和透明度的承诺，您能否承诺贵国政府支持一项新的世界卫生大会决议，要求对所有Covid-19的起源假设进行全面和不受限制的调查，并充分接触所有相关记录、样品和人员？
