撰文：台湾宝岛农场 快慢机

一架隶属于美军特种作战司令部旗下的“C-146A型”猎狼犬专用军机，于7月15日上午，从驻日冲绳美军基地起飞，直达台北松山机场。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/07/16102938/%E8%9E%A2%E5%B9%95%E6%93%B7%E5%8F%96%E7%95%AB%E9%9D%A2-2021-07-16-222719.jpg)
当地时间2021年7月15日上午09：30。一架隶属于美军特种作战司令部，印太防区旗下，担负全球快速机动运输任务的，编号13013军用C-146A猎狼犬型专机，由驻日美军冲绳基地起飞，直抵台北松山机场。并于落地后迅速完成任务物资卸载及整补等相关作业，随即升空返航。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/07/16102946/%E8%9E%A2%E5%B9%95%E6%93%B7%E5%8F%96%E7%95%AB%E9%9D%A2-2021-07-16-222735.jpg)
据悉，机上除任务机组成员外，并无搭载其他任何乘客。该机本次所担负的主要任务为，向美国驻台协会（AIT）处长交接活动，运送相关文件及物资。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/07/16102955/%E8%9E%A2%E5%B9%95%E6%93%B7%E5%8F%96%E7%95%AB%E9%9D%A2-2021-07-16-222752.jpg)
美国驻台协会（AIT）的新任处长“孙晓雅”女士，已于当地时间7月12日顺利抵台，目前正在医学隔离程序中，待隔离期结束之后，其将随即接替上任处长“郦英杰”先生继续履新。

前后相距短短一月有余，即有两架美军专机相继起降台北松山机场。中共的所谓红线已被美军彻底打破，除了战狼口炮满负荷运转之外，竟别无它法。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/07/16103003/%E8%9E%A2%E5%B9%95%E6%93%B7%E5%8F%96%E7%95%AB%E9%9D%A2-2021-07-16-222805.jpg)
中共国防部随即对本次美军军机的抵台行动发表了官方回应，内容依旧是陈词滥调，除了对事件本身表示严重关切与极度不满之外，期间有一句表述更是甚为搞笑：“任何外军飞机未经中共政府的官方许可，都不得擅自进入中国领空及台湾空域，由此引发的一切严重后果，将由擅闯者自负”。

抛去美军近期频繁进出台湾西南空域的，多型反潜与电侦机不说。就这先后两次飞抵台北松山机场的美军专用军机来看，又有哪次是经过你中共的官方授权与批准才降落台湾本岛的？

你中共党卫军的扰台军机，每架都是经过北京当局官方授权许可的，但又有哪架是能够成功顺利降落台湾本岛机场的呢？当然，此前曾成功驾机叛逃至台湾的那批军机不在此列哈！

一个月前是美军C-17环球霸王大型战略运输机抵台，这次是担负美军特种作战司令部，全球快速机动运输任务的C-146A猎狼犬专用军机二次抵台。下一次又会是美军的哪个当家主力机型飞抵台湾呢？
