**博主: 冲天小卤蛋**

日本莫德纳第三针，酸爽…

去年6月和9月接种了辉瑞两针，第三针据说莫德纳混打效果比较好，鉴于本人准备于6月回国，想着打了比较靠谱。今天3.24上午十点半多打了第三针，这酸爽，可真是…具体表现如下

1.肌肉疼痛。第一针辉瑞也有过，但痛的范围不可比，注射完后4个小时左右开始感知到不可忽视的疼痛，现在晚上近一点，疼痛已经爬到了肩膀且愈演愈烈，翻身不顺畅，所以我睡不着。

2. 低烧，注射后8个小时左右体温开始爬升。平时体温大约36.2左右，从晚上6点到现在慢慢到了37.5.听说过会发烧，做好了明天还会烧的准备。

3. 发寒。手脚冰冷，脸和鼻息烫，手脚一阵一阵起鸡皮疙瘩，也是没想到久违的在睡觉的时候开起了空调。

4. 异常口渴。平常不爱喝水的人今天喝了1升以上，可能是跟体温上升有关。

5. 些许反胃，没到呕吐的地步，但胃里时不时会有些汹涌，本来这个点要是没睡我就该饥肠辘辘解嘴馋了…

各位打了莫德纳的都是勇士，打了莫德纳还没有副作用的都是天选之子，希望副作用快快消失，且第三针是否有必要打也是因人而异了 慎重选择呀。

3.25补充：3.24晚上12点之后体温上升速度异常之快，近2点已经到了38.1，才吃了退烧药，可能在37.5左右吃会更好吧。鉴于太疼了我实在睡不着，索性开灯起来不睡了，发烧也不适合开暖气，就关掉了。然后开始身体发热，3点左右强行躺在床上等睡着的时候烫到感觉要烧穿床垫了且由于我喝了很多水，一晚上跑了10来次厕所…最后看手机大约4点，然后终于睡去，早上9点左右量过体温，37.9，感觉头昏昏沉沉，鼻子像感冒了一样通气不畅，一上午都在补觉。12点量体温，好家伙又来到38.2了…

所以大家一定要备好充足的水和退烧药呀。

3.26 我又来补充了

今天睡到中午12点才起床，体温降回平常，手臂疼痛也减轻了。但是！我洗澡的时候才发现，注射疫苗的地方肿了一大片！好红！生生多出一块肱二头肌，刚打完之后貌似是没有的，第二天没注意，第三天了才发现，范围还挺大，比拔罐的一个罐还大两圈，但鉴于身体呈现好转状态虽然还是很疲劳…猜测过几天应该会消失
