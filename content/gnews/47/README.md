有人指出，该《[历史决议](https://www.12371.cn/2021/02/21/ARTI1613872222158969.shtml)》的重要性在于“稳固权力”，中共国所谓的《历史决议》是自1945年和1981年以来，分别由毛泽东和邓小平通过该“历史决议”。

根据邓小平1981年的《历史决议》看，邓在中共全党内史无前例地批判了毛泽东在第一个《[历史决议](https://zh.wikipedia.org/wiki/%E5%85%B3%E4%BA%8E%E8%8B%A5%E5%B9%B2%E5%8E%86%E5%8F%B2%E9%97%AE%E9%A2%98%E7%9A%84%E5%86%B3%E8%AE%AE)》中所做批判“左、右错误路线”问题的”“决议”是错误的，它导致了文化大革命的爆发。这的确堪称是“历史性”的“决议”，它触发了中共国的改革开放。

[邓小平的《历史决议》](https://jamestown.org/program/cultural-revolution-50/)认为：

***1966年至1976年，文化大革命使党、国家和人民遭受了建国以来最严重的挫折和损失。文化大革命是由[毛泽东]发起和领导的…… 文化大革命的历史表明[毛泽东]提出的文化大革命的主要信条与马克思列宁主义和中国的现实。[那些信条]对中国阶级状况以及当时党和国家政治状况的评价是完全错误的。***
