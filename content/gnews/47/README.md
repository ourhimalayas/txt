## 【独立观点】ZOOM的安全漏洞曾让数百万电脑面临被恶意软件感染的风险
` 纽约香草山MOS02` [轉載自GNews](https://gnews.org/zh-hans/1404931/)

作者：纽约香草山翻译部 GBW



![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/05/02105519/GNEW-GTV-MOS-LOGO-2-2-1.jpeg)



在2021年6月27日的GTV直播中，郭文贵先生发出了对视频软件Zoom的风险警告。据郭先生爆料，Zoom由中（共）国安全部和政法委背书，经习近平批准，占全球教育机构市场份额的90%，其他相关市场份额例如法院的86%，标普500强公司的60%，政府的70%，军队的25%-30%。

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/07/18171245/11-12.jpg)图源网络

媒体上有很多关于Zoom安全漏洞的报道，其中一个最大的漏洞估计涉及400多万台苹果电脑用户，是由一个安全研究员乔纳森-莱舒（Jonathan Leitschuch）在2019年发现的。虽然该漏洞在同年被修复，但这个故事提醒我们，个人隐私是非常容易被入侵的。

2019年3月，莱舒对Zoom视频电话相当流畅的用户体验感到很好奇，他随即点击了一个Zoom会议邀请。尽管莱舒之前已经在他的苹果电脑上卸载了Zoom客户端，但浏览器还是自动启动了Zoom客户端，这让莱舒有了警觉。事实上，出于安全考虑，网络浏览器在没有得到许可的情况下，一般是不能在用户的电脑上启动其他应用程序的。经过一番调查，莱舒发现，Zoom在他的电脑上秘密安装了一个Zoom服务器。为什么说“秘密”呢？因为莱舒翻阅了Zoom的公开文档，却找不到任何有关这个服务器的描述。更令人惊讶的是，即使卸载了Zoom应用程序也不能删除该Zoom服务器。当莱舒点击他朋友的Zoom会议邀请链接时，他电脑上的Zoom服务器被激活了，随后，Zoom服务器未经莱舒允许就下载并安装了Zoom客户端应用程序。

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/07/18171347/12-12.jpg)

莱舒在他的博客中解释说，苹果电脑上秘密安装的Zoom服务器可能会被黑客利用：

- 黑客可以在受害者不知情的情况下强迫任何人加入Zoom会议。由于Zoom的默认设置是打开摄像头，因此，黑客可以通过视频会议监视受害者。
- 黑客还可以将用户的电脑作为武器，发动拒绝服务攻击，即指示数百万装有Zoom服务器的电脑同时访问某个网站，以巨大的流量瘫痪这个网站。


此外，另一家安全研究公司Assetnote发现，黑客可以在网页链接中嵌入代码，指示秘密Zoom服务器下载第三方应用程序或恶意软件。如果苹果电脑的Zoom用户不小心点击了黑客的网页链接，黑客就能完全控制该用户的电脑。

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/07/18171428/13-9.jpg)

2019年3月26日，莱舒通过电子邮件告知了Zoom这个安全问题，并给其90天的公开披露期限。第二天，Zoom要求莱舒保持沉默，并愿意给他一大笔钱，但莱施拒绝了这个交易。2019年6月24日，即90天披露期限的截止日，Zoom推出了一个“快速修复”方案，但莱舒表示，这个方案并没有修复该漏洞。随后，莱舒将这个漏洞公之于众。最后，在Zoom CEO的介入下，这一秘密服务器功能终于被删除了。

我们不知道Zoom采用这一极不安全的秘密服务器方案是为了创造流畅的用户体验，还是像郭文贵先生所说的另有目的。自2019年以来，Zoom增加了更多的安全措施，但像谷歌和美国国家航空航天局这样的组织至今仍然禁止使用Zoom。作为用户，我们必须对软件的选择应保持谨慎态度。
