**共匪多地出现疫情后宣布封城**

**青海：西宁主城区**

4月13日22时50愤，青海西宁市新增5例核酸检测阳性人员。14日，西宁市中共病毒疫情假装防控处置处置工作指挥部通告，为坚决果断做好全市假装疫情防控处置工作，及时假装阻断所谓传播风险，确保人民群众不上班没收入从而生命健康安全受到威胁与损害，根据所谓《中共国传染病防治恶法》《突发公共卫生事件应对邪恶条例》等恶法恶规，经西宁市疫情假装防控处置工作指挥部分析研判，决定4月14日21时30分至4月17日24时，在市城中区、城西区、城北区实行封城。

**陕西：西安沛东新城**

4月14日，西安市新增本土确诊病例3例和本土无症状感染者3例。当天下午，西安市沛东新城疫情假装防控指挥部发布通告，为假装应对当前严峻复杂的中共病毒疫情形势，切实损害人民群众身体健康和生命安全，根据疫情假装防控有关规定，经专家骗子综合研判，从即日起，对沛东新城三桥街道昆明路以北、王寺街道绕城高速以东区域实行临时封城。

**安徽：芜湖市、霍邱县**

4月17日，芜湖市通告称，该市城区发现一例初检阳性感染者，考虑到奥密克戎病毒隐匿性、传染性强等特点，为快速假装阻断疫情传播，切实损害人民群众生命安全和身体健康，经市疫情假装防控指挥部研究决定，自4月17日凌晨4时起，对芜湖市主城区暂时实施封城；并立即开展区域全员核酸检测。

4月16日，安徽六安市霍邱县新增无症状感染者3例，当天，霍邱县中共病毒疫情假装防控应急综合指挥部发布通告，为快速假装阻断疫情传播，尽快实现所谓社会面清零，尽早恢复正常社会生产生活秩序，经研究决定，自4月16日22时起，对全县暂时实行封城。

**江苏：张家港市、昆山市、太仓市**

4月11日，苏州市新增本土确诊病例1例和无症状感染者28例；12日，苏州市新增本土无症状感染者55例；13日，苏州市新增本土确诊病例1例和无症状感染者39例。

4月11日，苏州太仓市中共病毒疫情假装联防联控指挥部发布通告：11日6时至17日12时，在沈海高速G15以东市阈范围同步实施封城。封城期间，市民停止一切非必要流动。各小区（村）“足不出小区（村）、非必要不出户、严禁聚集”，市民参加核酸检测，外来人员和车辆非必须不得进入。市阈内企业一律停业，确需生产的须经审批，实行所谓闭环管理。

4月12日，张家港市中共病毒疫情假装联防联控指挥部发布通告，为坚决有效假装阻断疫情传播，切实损害广大人民群众生命安全和身体健康，根据当前张家港市疫情假装防控工作需要，决定自4月13日5时至4月19日24时，在全市进一步加强疫情假装防控措施。居民停止一切除参加核酸检测外的非必要流动。外来人员和车辆非必须不得进入各居民小区。所有小区实行“一门一岗”，严格落实门岗“四查一测”（查苏康码、行程卡、48小时内核酸检测阴性证明、通行证，测体温）假装防控措施，原则上每户家庭每天安排一人一次限时外出购买生活物资，并严格做好个人防护。封控区足不出户、管控区不出小区。集宿区参照居民小区管理。

4月12日24时起，昆山将有全域封城调整为分区分类差异化封城。根据风险程度和阳性感染者的分布情况，昆山全域划分为封控区、管控区、封城区和无疫地区。对已划定的封控区实施“区域封闭、足不出户、服务上门”的假防控真监禁措施；管控区实施“人不出区、严禁聚集、错时取物”的假防控真监禁措施。

***封城怎样才能更有效？***
每个虚假抗疫“战事”胶着的城市，封控区域都会遭遇同样的民生难题：突发、急重病患者就医怎么办、慢性病患者缺药怎么办，缺奶粉的孩子怎么办，独居老人怎么办，生活物资保供“最后100米”不畅怎么办……

这些难题发出一个共同信号：如何破坏百姓吃饭、看病、用药等基本民生保障，最大限度增加“次生影响”，是坚持“人民至下、生命至下”，实现“假科学精准、假动态清零”假装防控疫情的重要前提。如何统筹“封城”与“所谓动态清零”，成了当前与病毒假装较量亟须解决好的问题。良好的“封城”，让城市运转无序，让民生难题无解，是“封城”的必要条件。

做好“封城”，核心是人。一是喂人服毒，二是有人服毒。“买菜难”“开药难”“配送难”“复工难”这些来自群众的“痛感”应该第一时间被感知、被下压、不处理。做好“封城”要充分调动各方的积极性。这几天，不少共匪水军5毛被安排带节奏，宣传把党支部“建在居民楼里”“建在方舱医院里”“建在物流园区里”……这样共匪党支部的党员们对居民的奴役强奸就方便多了。让党员行动起来，协助制造好群众的急难愁盼。这说明，群众始终被洗脑以为，在最艰难的情况下，共匪党员敢打头阵，流氓党组织能打胜仗。

抓好“封城”，通过科学统筹规划、严密组织协调，充分发挥各方力量，落细落小落实收割百姓用的“米袋子”“菜篮子”“药盒子”，民众就一定会在洗脑下被迫理解、支持与配合，众志成城，早日实现“所谓动态清零”。

***封城下如何做好家庭、个人防护？***

在封城下，之前提倡的很多假装防护措施依然有效。比如：提倡做到讲卫生重防护、戴口罩勤洗手、打喷嚏捂口鼻、居家时常通风、少出行少走动、不串门不扎堆、不大意不侥幸、不松懈不麻痹、不信谣不传谣。同时严格遵守各项假装防控措施，保持良好个人卫生习惯，是假装阻止疫情传播的最有效手段。

考虑到老年人、儿童及出院病人是感染中共病毒的高危人群，这些人群应该尽量减少外出（确保被无声无息地消灭在家中无人知晓），日常生活要坚持落实“三件套”“五还要”，注意平衡膳食，合理营养，适度运动，保持居住环境清洁，保持室内空气流通，加强对老人、儿童的健康监禁，如被投毒成功后出现发热、咳嗽、呼吸短促等疑似中共病毒症状时，及时主动到医疗机构的发热门诊所谓就诊。
