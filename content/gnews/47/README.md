## 白宫就国内航班的疫苗护照问题进行辩论
` 洛杉矶盘古农场 HIMALAYA LA PANGU` [轉載自GNews](https://gnews.org/zh-hans/1546786/)

**据《新闻大全》作者：Eric Mack，2021年9月19日报道：**
![](https://assets.gnews.org/wp-content/uploads/2021/09/a9d2h-yr6ll-3.jpg)图片来源：盖蒂
在安东尼·福奇博士站出来支持推出国内航空旅行的疫苗护照后，白宫正在对此措施进行内部辩论。

白宫新闻秘书珍·普萨基（Jen Psaki）在周四的每日新闻发布会上说：“我们没有取消任何选项，但目前我没有任何更新可以与大家分享，我们的重点是实施我们上周宣布的重大举措。”

“目前我们的重点是实施这些。其中的一部分是对飞机上不戴口罩的人进行加倍罚款 —— 我们觉得这个步骤将有助于保证人们在飞行中的安全，并减少病毒传播。”

普萨基被问及福奇出面支持美国航空旅行疫苗护照的情况。

根据《国会山报》报道，福奇对theSkimm播客说：“如果你想乘坐飞机和其他人一起旅行，你应该接种疫苗。”

白宫办公厅主任 罗恩·克莱恩（Ron Klain）承认正在考虑这种可能性。

克莱恩告诉Pod Save America播客：“这是我们继续关注的问题，我们想权衡这些要求，可以接种疫苗的人数和接种疫苗的负担，每次上飞机时都必须出示接种记录凭证，不得不在运输安全管理区排队，等候更长的时间。”

“但我认为，随着我们的不断进步，我们会关注这一点。”

报告声称，白宫关于航空旅行疫苗护照的问题并不侵犯美国人的隐私和个人权利，但它会给联邦雇员带来强制执行的负担。

克莱恩在播客中补充说：“我们认为最有效的疫苗要求是人们在某种固定的情况下，在工作中，或在军队中，他们验证一次，那么他们就被验证了。”

白宫冠状病毒应对协调员杰弗里·齐恩特（Jeff Zients）上周告诉记者，疫苗授权或测试都在考虑之中。

他说：“我们不会采取任何措施。”

美国旅游协会主席罗杰·道（Roger Dow）提到，国内航空旅行的疫苗接种的强制执行将“极其困难”，但他对重振国际航空旅行的做法“感到满意”。

“挑战是有超过65%的人口接种了疫苗，有35%到40%的人口由于某种原因可能无法接种疫苗，但他们仍然愿意进行COVID测试，并表明当他们走进门时，他们没有感染新冠病毒。”据《国会山报》的报道，他在周三告诉记者。

虽然国内航空旅行下降，但是达美航空公司首席执行官埃德·巴斯蒂安 (Ed Bastian) 担心实施疫苗接种规定“实际上会阻碍国内旅行系统”。

美国航空公司对国内疫苗护照的讨论感到担忧。

集团在给《国会山报》的一份声明中写道：“我们被告知，没有关于国内旅行的迫在眉睫的政策提案，并回应了政府对此类政策的实施和执行所表达的担忧。我们仍然与政府保持沟通，并继续依照科学的指导来优先考虑公众旅行的安全和健康的政策。”
