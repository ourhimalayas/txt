## 乌克兰声称夺回关键军用机场 俄军占领切尔诺贝利

翻译：Steven Hu
![](https://assets.gnews.org/wp-content/uploads/2022/02/3-82.jpg)
图片来源：合众国际社

合众国际社2月24日报道，乌克兰官员周四表示，他们夺回了一个关键的军用机场，同时俄罗斯特种部队占领了该国包括切尔诺贝利核电站在内的地点。

合众社引用美国广播公司新闻报道，乌克兰议会的官方电报账户称，位于霍斯托梅勒的安东诺夫机场“完全处于乌克兰军队的控制之下”，内政部副部长安东·格拉申科（Anton Geraschenko ）称这是“第一次重大胜利”。

当天早些时候在社交媒体上发布的视频似乎显示俄罗斯军用直升机正在前往基辅西北郊区的国际货运机场和测试设施。

“俄罗斯联邦的 20 架 KA-52 和 Mi-8 直升机降落在 Gostomel 机场，战斗正在进行中。敌人继续沿着整个共同边界线采取侵略行动，”前泽连斯基发言人 罗利亚.门德尔（Iuliia Mendel ）发推文说。

另据美国有线电视新闻网报道，在争夺机场的战斗中，有未经证实的俄罗斯直升机被击落。

早些时候，乌克兰官员表示，俄罗斯军队占领了切尔诺贝利现已废弃的核电站周围的整个区域，其中包括该国其他核电站核废料的储存库。

据《华盛顿邮报》报道，乌克兰总理丹尼斯·施米哈尔在周四晚间的电视讲话中说：“切尔诺贝利区——禁区——以及切尔诺贝利核电站的所有设施都已被俄罗斯武装团体控制。”

而白宫新闻秘书琼.普萨基（Jen Psaki）表示，美国对俄罗斯士兵将发电厂文职人员扣为人质的“可靠报道”表示“愤怒”。

“这种非法和极其危险的劫持人质行为可能会颠覆维护和保护核废料设施所需的日常公务员工作，这显然是令人震惊和难以置信的，我们谴责这种行为并要求释放他们。”

由于当时该工厂的一个反应堆熔化，大量核泄漏使切尔诺贝利是成为1986 年世界上最大的核灾难发生地。

这是在俄罗斯士兵和乌克兰国民警卫队之间地面交战之后发生的，作为安全措施，乌克兰国民警卫队经常驻扎在工厂周围。

周四早些时候，乌克兰总统泽连斯基在推特上表示，俄罗斯军队正试图夺取切尔诺贝利核电站。

”我们的战士正在献出生命，以免重演 1986 年的悲剧，”泽伦斯基在推特上写道。
