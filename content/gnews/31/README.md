2月7日，江苏徐州市政府官方微博徐州发布，发布了关于徐州丰县被锁女子的调查进展情况。

没想到激起墙内网友的愤慨，纷纷发布不同的观点和看法，尤其对“徐州发布”发布的内容提出很多质疑。

其中重庆雷登峰律师提出了几点质疑：在丰县被锁女精神病人事件中，有几个观点，看似有理，实则非常反人类。
![](https://assets.gnews.org/wp-content/uploads/2022/02/114.jpg)图片来自网络截图
一种观点认为，女精神病人不是被拐卖的，而是被家人遗弃的，所以别的男人捡回来，与她发生关系、生孩子，是做好事，比她流浪无人管好！这种观点，无理之极，混淆了不同的关系：第一，家人遗弃她，按现行刑法，该追究责任，公安机关依法追究就行。如果家人宁愿坐牢，也不想（或无能力）再管女子，那么，政府有义务对这类弱势群体进行照顾。第二，男子（或家人）捡到女子（这不是遗失的物品，是有人格尊严的同类），除了短暂的人道照顾（给水给食，提供睡眠）之外，无权占有，应该报警或交给民政部门（如果是收买的，本身就构成了犯罪，我个人认为，应该修法，收留无行为能力的外人后不在规定时间内上报的，就应该处以刑罚），第三，与无行为能力女性发生关系，生下孩子，是强奸重罪，都应按刑法时效规定进行追究（结婚证本就无效）。

另有一种观点认为，现在女子生了这么多小孩，再去追究男人的强奸罪，小孩怎么办，对小孩成长是不是更不利？这种观点也是典型的“有奶便是娘”论，做个简单对比，那有几个小孩的男人，杀死一个孤家寡人的流浪汉，为了孩子有人照顾，是不是不用负刑责（在一些父母虐待孩子的案件中，许多人也是这种观点，认为把父母抓了，孩子更可怜）？还是那句话，政府应该为这类人的生存问题兜底。

还有一种观点认为，精神病人几千万，政府没有人力、物力管。一些人看不到底层人民的艰辛，倒是为一年税收以十几万亿计、每年援外堪比天文数字的政府操碎了心。虽说精神病人（其实是指有一些精神疾病的）的确有数千万，但不都是完全丧失行为能力（很多还在上班挣钱）。家人无力照顾的，细算下来，真正完全需要交给政府（医院）照顾的，哪有那么多？说到底，就是一个愿不愿、值不值的问题，一个税收怎么安排、谁优先的问题！
