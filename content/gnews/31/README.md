## 一周中共病毒及疫苗伤害信息汇总。
` NFSC News` [轉載自GNews](https://gnews.org/zh-hans/2000055/)

作者：华盛顿DC农场 火来、香草山农场Zion文恩
![This image has an empty alt attribute; its file name is Screen-Shot-2021-08-30-at-7.37.59-PM.png](https://assets.gnews.org/wp-content/uploads/2021/08/Screen-Shot-2021-08-30-at-7.37.59-PM.png)
2022年2月2日至9日，除中共国外，全球新增中共病毒确诊患者1790万余例，同比上周下降了16.5%。此外，本周新增确诊死亡病例超过76195例，同比上周增长了7.39%，增长率较上周下降了11.51个百分点。连续三周，中共病毒全球新增死亡率持续上升。截止2月2日，除中共国外，全球累计接种了73.6亿剂中共病毒疫苗，同比2日的接种数据增长了1.52%，但疫苗接种增长率较前一周总体下降了0.3个百分点。
![](https://assets.gnews.org/wp-content/uploads/2022/02/截屏2022-02-13-上午10.06.46.png)
本周，全球疫苗护照、防疫等政策持续放缓。2月1日，丹麦首相弗雷德里克森宣布告别疫苗通行证，丹麦成为第一个解除疫苗护照等防疫政策的欧盟国家。2月8日，加拿大阿尔伯塔省省长杰森·肯尼也宣布，该省的疫苗护照政策将于当日午夜结束。以色列作为首批推行疫苗护照的国家之一，也开始逐步停用疫苗护照。以色列政府的疫情工作组决定，除了“公众密集聚会的高风险场所外”，餐馆、酒吧、健身房和酒店等大部分公共场所，都将放弃使用疫苗护照。法国负责欧洲事务的国务卿克莱门特·博纳向媒体指出，法国将放宽病毒防疫措施，完全接种的旅客将不再需要病毒检测阴性证明就可以入境法国。马来西亚冠状病毒恢复委员会于2月8日表示，作为加速经济复苏计划的一部分，建议最早在3月1日起全面开放边境，不再对游客实行强制隔离政策。另一方面，加拿大卡车司机针对强制疫苗政策抗议活动仍在继续。2月7日，“自由车队”抵达美加边境的底特律—温莎口岸，堵住了美加边境的大使大桥，抗议升级。

此外，美国参议院议员罗恩·约翰逊致信国防部长劳埃德·奥斯汀，要求国防部关注国防医学流行病学数据库(DMED)的数据， 并开始调查2021年该数据库登记的美国军人流产、癌症以及其他众多疾病的诊断病例，与往年相比显著增加的原因。
