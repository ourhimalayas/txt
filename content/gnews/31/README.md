4、[**格拉斯哥气候峰会前 约翰逊与习近平通电话 敦促提升减排目标**](https://www.aboluowang.com/2021/1030/1665991.html)

《联合国气候变化框架公约》第26次缔约方大会（COP26）即将在格拉斯哥开幕，大会主席国英国首相约翰逊星期五（20日）在和中共国家主席习近平通话时表示，所有国家都需要在联合国气候峰会上提升各自的减排目标，包括逐步放弃使用煤炭。习近平并不准备亲自出席格拉斯哥的联合国气候峰会。代表中国与会的很可能是中国生态环境部副部长赵英民和气候变化事务特使谢振华。习近平到目前已经有一年零九个月没有跨出中国国门。为了落实习近平向国际间作出的减排承诺，中国政府给全国各省市区都制定了减排的年度目标。

5、[**未获两大佬提名 谭德塞成世卫秘书长唯一候选人**](https://www.aboluowang.com/2021/1030/1665818.html)

世界卫生组织（WHO）今天表示，秘书长事谭德塞接受28国提名连任下一届秘书长，且是唯一候选人，将在明年5月世界卫生大会投票。但他未获母国衣索比亚支持，美国、中国也未提名他。在公布候选人之后，世卫将在2022年1月执行委员会进行初步审查，确保候选人符合资格，并进行面试，然后将提名送交5月的世卫大会（WHA）投票，若表决通过，谭德塞的第二任期将在8月开始，任期为5年。

6、[**疑涉网络攻击 中国百富环球在美遭FBI突击搜查**](https://www.voachinese.com/a/pax-technology-fbi-20211029/6291462.html)

美国联邦调查局（FBI）和海关与边境保护局的官员周二（26日）执行了一项法院命令，搜查了百富环球子公司在佛罗里达州的办事处和仓库。该公司星期五交给香港证券交易所的一份声明证实，一些物品被没收。媒体报道，搜查行动可能是怀疑该公司与发生在美欧的网络攻击有关。专家认为，中国政府可能利用该公司的支付终端进行信息盗窃。百富环球科技有限公司是中国一家电子支付销售点终端供应商，产品供给全球数百万企业和零售商使用，在全球120 个国家和地区拥有数千万台销售终端。
