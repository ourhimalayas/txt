## 六名香港抗共人士遭港共政府拘捕
` GTV新闻访谈` [轉載自GNews](https://gnews.org/zh-hans/1805213/)

作者：纽约香草山 医疗部 云豹
![This image has an empty alt attribute; its file name is Screen-Shot-2021-08-30-at-7.37.59-PM.png](https://assets.gnews.org/wp-content/uploads/2021/08/Screen-Shot-2021-08-30-at-7.37.59-PM.png)
当地时间12月29日，香港抗共民主人士何韵诗在自己的脸书帐号上发文表示，当天早晨6点，香港警方国安处前往何韵诗住所，并以所谓的“串谋发布及复制煽动刊物罪”将其拘捕。随后，何韵诗被带往香港西区警署。
![](https://assets.gnews.org/wp-content/uploads/2021/12/截屏2021-12-30-下午10.19.36.png)
据相关报道提及，而今已有6位与《立场新闻》相关的前任或现任主要参与人员被港共政府拘捕，包括《立场新闻》的前董事吴霭仪、何韵诗、周达智、方敏生，以及代总编辑林绍桐、副采访主任兼香港记者协会主席陈朗升。

资料显示，《立场新闻》是香港在地的非营利性独立新闻媒体，其成立的宗旨在于传播真相，争取香港的民主、人权、自由、法治与公义，并致力于守护香港的核心价值，是香港少数的抗共媒体之一。目前，立场新闻已在其官方网站上明确公告称，由于港共政府拘捕了该公司多名高层并收缴了其办公室内的多部电脑及相关文件，立场新闻已于12月29日停止运作并遣散公司内所有员工。
