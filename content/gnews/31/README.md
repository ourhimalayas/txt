## 默克公司向美国收取40倍于政府资助的COVID药片的成本费用

翻译：纽约香草山健身部 – Ivy001
![This image has an empty alt attribute; its file name is Screen-Shot-2021-08-30-at-7.37.59-PM.png](https://assets.gnews.org/wp-content/uploads/2021/08/Screen-Shot-2021-08-30-at-7.37.59-PM.png)
据The Intercept援引哈佛大学公共卫生学院和伦敦国王学院医院上周发布的一份报告称，默克公司的新“非伊维菌素”Covid-19治疗方法，莫努匹拉韦，生产成本为17.74美元，但该公司却向美国政府收取712美元的治疗费用，加价40倍。
![](https://assets.gnews.org/wp-content/uploads/2021/10/Untitled.001-1.jpeg)
这种药片最初是利用美国政府的资金开发的，作为委内瑞拉马脑炎的可能治疗方法，在一项针对775名患有新冠病毒的轻症/中症成年人的随机试验中，住院和死亡的风险减少了一半，这些人由于肥胖、糖尿病和心脏病等合并症而被认为是疾病的高危人群。该试验被提前停止，以便该公司能够申请紧急使用授权（EUA）。该药物对那些已经因严重疾病住院的患者没有帮助。

有关这种口服“神药”的消息使默克公司的股价在上周走高，因为该公司说它可以在今年年底前提供10MM的剂量。

显然，该药可以为默克公司及其在该药物上的合作伙伴Ridgeback Biotherapeutics带来巨额利润——该公司在2020年从埃默里大学获得了该药物的许可，然后以未披露的金额将该药物的全球权利出售给默克公司。

同时，据非营利组织知识生态国际发现，国防部下属的国防威胁减少局（DTRA）在2013年至2015年间为埃默里大学的药物开发提供了1000万美元的资金。

然而，正如Quartz指出的那样，只有默克公司和里奇克公司将从这种新的抗病毒药物中获利——他们说这种药物“可能是有史以来最有利可图的药物之一”，仅在今年年底就能带来高达70亿美元的收入。

尽管有了最初的投资，美国政府似乎面临着价格的急剧上升。6月，政府与默克公司签署了一份12亿美元的合同，以712美元的价格供应170万个疗程的药物。该交易将在莫努匹拉韦获得食品和药物管理局的紧急使用授权后立即进行。

好政府的倡导者指出，由于联邦机构在该药物的开发上至少花费了2900万美元，政府有义务确保该药物是可负担得起的。知识生态国际的高级研究员路易斯-吉尔-阿比纳德（Luis Gil Abinader）说：“公众资助了这种药物，因此公众有一些权利，包括你在合理条件下获得它的权利”。

Ridgeback公司的联合创始人温迪-霍尔曼上周在接受CNBC采访时说，他们要求但 “从未得到政府的资助”来制造莫努匹韦。该公司还在一份吹捧该研究的新闻稿中声称，“自从获得Ridgeback公司的许可后，用于开发莫努匹韦的所有资金都是由默克公司和Ridgeback公司的韦恩和温迪-霍尔曼提供的”。

同时，批评者说700多美元的价位是荒谬的，健康倡导者担心一些国家的人将无法负担这种新药。

报告的共同作者、国王学院医院的医生Dzintars Gotham说：“在一个人还没有感觉到生病的时候向他提供700美元的治疗，这将意味着很多人不会接受它”，该报告表明，如果默克公司将莫努匹拉韦定价为19.99美元，他们仍将获得10%的利润率。

“如果你买不起药，是因为它比你能承受的价格高1000倍，或者是因为它比你能承受的价格高100倍，这都不重要”，哈佛大学公共卫生学院博士生、莫努匹拉韦报告的共同作者梅丽莎-巴伯说，“这些都很糟糕”。

Barber和Gotham承认，生产一个五天疗程的抗病毒药丸的17.74美元的成本是一个估计值，但他们说，他们使用的算法，以及用于估计数百种药物的生产成本的算法，从长远来看，往往会导致高估。

同时，私营公司对药物的收费价格往往会上升而不是下降。她说：“对于所有这些发生在治疗剂或疫苗上的交易，价格只是随着不确定性的减少而增加。给出了一个价格，然后，在下一次的销售中，价格上升了。其他药物和疫苗的价格上升了，所以如果这个价格不上升，我将非常惊讶”。
