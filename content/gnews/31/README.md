**1．****原子能机构****将验证****福岛核电站排入海****中处理水****的安全性**

[据NHK 2月7日报道，国际原子能机构IAEA已决定于本月14日至18日向福岛第一核电站派出一个小组，检查将排入海中处理水的安全性。日本政府计划稀释福岛第一核电站不断增加的含氚处理水，待浓度降至标准以下后，于明年春天之前开始排入大海。原子能机构计划在今年年底前编撰一份临时报告。](https://www3.nhk.or.jp/news/html/20220207/k10013471901000.html?utm_int=news-social_contents_list-items_004)

**2****．****因回避****摩德纳疫苗****而导致****接种****预约缓慢**

[据产经新闻2月7日报道，围绕中共病毒（COVID-19）疫苗第三剂，在各地方政府提供的群体接种中，辉瑞疫苗的大受欢迎程度，而大家都在避免使用摩德纳疫苗。日本全国有80%的人在第一和第二针时接种的是辉瑞公司的疫苗，许多人担心摩德纳疫苗强烈的副作用和交叉接种。然而，在第三轮疫苗接种中，摩德纳疫苗的供应预计将大幅增加。但长崎大学“传染病和疫苗专家”森内博之教授说：“首要任务是完成第三轮疫苗接种，无论选择哪种疫苗，我们希望大家尽快接种。”](https://news.yahoo.co.jp/articles/757e76d55c028bf2b22d1b747eab6feef07d94ab)

**3****．为使用国产云服务****管理国家机密****，日本加速****技术****开发**

[据读卖新闻2月7日报道，日本政府已经决定，为共享行政数据而开发的“政府云”中，对被列为国家机密的数据仅限于使用日本企业的服务，在防止机密信息泄漏到海外的同时，推动日本企业的技术开发。计划将于今年列出选定服务的公司，目标是2023年启用运行。](https://news.yahoo.co.jp/articles/cc4f577f40d454c60c14f129c603df00e6a36575)

**4****．使用****熟练 “技能 “的机器人进行细胞培养**

[据读卖新闻2月7日报道，理化学研究所等机构将于2022年在神户建立一个研究基地，使用机器人进行细胞培养等实验。目的是将再生医学和生命科学领域中需要研究人员付出大量努力的简单任务自动化，从而腾出足够的时间进行研究和加速新疗法的开发。日本产业技术综合研究所和东京大学计划参与该项目，并与从事工业机器人技术的公司合作。该项目旨在同时开发能够思考实验过程本身的人工智能，从而带来人们可能不知道的新发现。](https://news.yahoo.co.jp/articles/3c5a8a1b0cd09f8e785d60b4916fe29835428d50)

**5****．****中小企业提高工资****，****价格****转嫁****是个挑战**

[据时事通信社报道，日本工会联合会会长芳野友子2月7日在富山市的新闻发布会上表示，她们已认识到无法将工资增长转为价格是中小企业的一个问题。她强调有必要改变现状，即中小型公司由于难以将原材料价格飙升造成的成本增加转嫁给客户而无法提高工资。为了保证增加提高工资的资源，她强调她将“坚定地呼吁中小型企业创造一个能够转嫁价格的环境”，并表示将敦促大公司停止“贱买”。](https://news.yahoo.co.jp/articles/2d596c3cd124f4fc0b486d798886ac290c34263e)

**6****．盐野义制药公司的新冠药物****将在月底前提交审批**

[据朝日新闻数据部报道，盐野义制药公司2月7日表示，它将在本月为其正在开发的一种治疗中共病毒（新冠病毒COVID-19）药物向日本政府申请批准。该公司总裁手代木功说，公司正在寻求申请有条件的早期批准，这将使其无需完成临床试验就能将该药物商业化。该公司于去年9月开始进行最后阶段的试验涉及约2000人，对其中69人的分析结果显示，该药物在减少病毒量方面比对照组假药更有效。发烧等症状也得到了改善。研究人员说，副作用很轻的，没有安全问题。](https://news.yahoo.co.jp/articles/026ea99575a5f8de628d73e3f99faaaa415a91d5)
