廷科夫是廷科夫银行的创始人，他在Instagram上写道，没有人从这场自2月24日开始的冲突中获益。

“在这场疯狂的战争中，我没有看到一个受益者！无辜的人和士兵正在死去。”根据路透社的翻译，这位寡头用俄语写道。

他用英语呼吁 “西方集体””请给普京先生一个明确的出口，以挽救他的面子并停止这场大屠杀。”

廷科夫的评论是在俄罗斯入侵乌克兰的第七周发表的。英国在3月底制裁了这位寡头。

英国外交、联邦和发展办公室表示，据报道，他的估计净资产为34亿欧元，或约36.7亿美元。

一些俄罗斯寡头和政府官员，包括俄罗斯总统弗拉基米尔-普京及其家庭成员，已经受到美国和其他国家的制裁。

廷科夫在他的帖子中声称，90%的俄罗斯人反对战争，10%的人是 “白痴”。他说 “白痴画Z”，这已经成为支持俄罗斯入侵的象征。

“克里姆林宫官员感到震惊，他们和他们的孩子都不会在夏天去地中海。据路透社报道，丁可夫补充说：”商人们正在努力挽救他们剩余的财产。

他说，俄罗斯的将军们 “在宿醉中醒来，意识到他们有一支低劣的军队，”他补充说，”而如果国家的其他一切都很糟糕，陷入裙带关系、奸商和奴性的泥潭，军队怎么可能好？”
