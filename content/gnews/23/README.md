俄军在靠近波兰边境的布列斯特训练区集结（图片来自网络）

就在俄罗斯全面进攻乌克兰遭到全世界的谴责和制裁之际，最新卫星影像又揭示，俄军突然在波兰边境集结！

据《希望之声》2月27日报道，长期追踪俄罗斯军队动态的美国太空科技公司“Maxar Technologies”的最新卫星影像显示， 俄罗斯军队正在波兰边境集结。

据该公司2月24日公布的卫星影像显示，俄罗斯士兵和火炮正集结在白俄罗斯城市布列斯特（Brest），而布列斯特距离波兰东边国界只有大约16公里。

美国《外交政策》（Foreign Policy）杂志负责采访五角大楼的记者德奇（Jack Detsch）也在推特上发文称，俄罗斯在靠近波兰边境的布列斯特训练区集结了士兵、装甲部队、火炮和50多辆重型装备运输车，并在白俄罗斯一处火车站附近增加装备。

虽然俄罗斯集结这些军队的确切原因目前尚不清楚，但美国国务卿布林肯已表示，担心俄罗斯将战事扩大到乌克兰以外的国家。

布林肯2月24日对ABC表示，俄罗斯总统普京的侵略行动有可能会越过乌克兰，但强大的北约将会制止这样的行动。布林肯指出，如果俄国入侵波兰、斯洛伐克、匈牙利或罗马尼亚，就将意味着美国、英国、法国、加拿大和其他北约成员国也会加入战场。

布林肯强调说：“攻击一个北约成员国就是攻击所有的北约成员国。 拜登总统非常清楚，我们将保卫北约的每一寸领土。”

布林肯同一日还在接受CBS采访时指出，普京明确想要重建苏联帝国，想在曾经属于苏联的邻国周围重新确立一个势力范围。
