**俄罗斯入侵乌克兰的军事行动持续到第****14****天，俄军继续在乌克兰首都基辅北部和西北部发起攻击。**

汇总过去24小时最新局势发展：

- 乌克兰表示，在受战斗影响最严重的六个地区，已经与俄罗斯达成了12小时的停火协议。希望平民能够通过人道主义通道逃离
- 俄罗斯证实，外交部长拉夫罗夫（Sergei Lavrov）和乌克兰外交部长库莱巴（Dmytro Kuleba）将于周四在土耳其会面。这将是自俄罗斯2月24日发动入侵行动以来两国外长的首次会面
- 在俄罗斯同意暂停轰炸后，约7000人已从东北部城市苏梅（Sumy）撤离
- 俄罗斯在轰炸苏梅期间，有22名平民丧生，其中包括三个儿童
- 乌克兰称切尔诺贝利的前核电站在被俄罗斯军队占领后已失去电力供应
- 联合国表示，自俄罗斯入侵以来，已经有超过200万乌克兰人逃离。波兰边境当局表示，已经有133万人从乌克兰入境，其中超过九成是乌克兰人
- 苏梅市长表示，周二开放的人道主义走廊将在周三继续
- 据俄罗斯国家媒体报道，莫斯科宣布再次停火，让受攻击城市的平民逃亡。当地时间周三10点（即格林尼治标准时间7点）再次为基辅、切尔尼耶夫、苏梅、哈尔科夫和马里乌波尔设立人道主义走廊
- 波兰提议将境内所有俄制米格29战斗机通过美国在德国的空军基地送往乌克兰，美国表示拒绝。美国国防部发言人说，该提议前景引起北约的“严重关切”，而一位美国军事专家称波兰的计划“令人费解”
- 英国公布了对俄罗斯的新制裁措施，将赋予政府部门有权利扣押在英国的俄罗斯飞机，俄罗斯飞机在英国飞行或降落可被定为刑事犯罪，英国还将阻止向俄罗斯出口所有航空或空间技术
- 美国总统拜登宣布美国单方面全面禁止进口俄罗斯石油、天然气和煤炭，目的是对俄罗斯总统弗拉普京进行“有力打击”
- 麦当劳、星巴克和可口可乐成为最新暂停在俄罗斯运作的西方公司
