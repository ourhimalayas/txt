![](https://assets.gnews.org/wp-content/uploads/2021/11/2-44.png)[图片来源](https://ec.ltn.com.tw/article/breakingnews/3734123)
11月22日，中共“环球时报”引用多个大外宣媒体的“报道”称，针对立陶宛与台湾的外交关系升级，中共正在采取外交、经济、商业和供应链等多重手段对立陶宛进行惩罚。同时，还利用借用大外宣之口放话，未来要拿掉几个台湾“邦交国”。

“环球时报”引用大外宣《南华早报》20日报道称，立陶宛外长兰茨贝尔吉斯19日表示，“由于受到来自中国的压力，立陶宛面临着某些经济和金融问题”。需要欧盟的支持“包括重新安排供应链和融资，以减少对中国的依赖”。

“环球时报”还引用大外宣俄罗斯《观点报》称，中共已几乎停止从立陶宛进口木材、鱼类和乳制品，并将立陶宛从中国到欧洲运输贸易线路上删除。

对于惩罚台湾，“环球时报”引用路透社称，在本月晚些时候举行的洪都拉斯总统选举之后，“台北可能会失去另一个盟友”。

立陶宛虽是“小国”，但其作为欧盟和北约组织成员国的身份，使得台湾在立陶宛设立办事处的意义变得极其特殊，甚至可以说可能形成中共国际外交空间和合法性全面崩塌的多米洛骨牌效应。
