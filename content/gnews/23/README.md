最近，辉瑞疫苗爆出了两个令人不安的消息，可以说一个比一个令人震惊。可奇怪的是，外网却鲜见报道。

第一个是瑞典隆德大學實驗研究，此报告发表在分子生物学的现代问题的杂志上。实验过程如下: 研究人员们把人体的干细胞放在一个培养器皿里，同时把辉瑞的疫苗也放入这个器皿里，然后发现辉瑞疫苗很快就会在人体的肝脏细胞里面产生很高的浓度，而且在六个小时之内，它就通过病毒本身携带的一个反转录的机制，从辉瑞疫苗（信使核糖核酸）反过来变成了DNA(脱氧核糖核酸DNA)。

这就意味着它有可能被整合到细胞的基因里。但如何整合? 整合进去的后果是什么？以及会对人体造成什么样的伤害，这一点还不太清楚。但值得深思的是，它既然能进入肝细胞，是否也能进入其他的比如生殖细胞,脑细胞或肾脏细胞等等。虽然这些只是猜测，但有专家认为这是疫苗未来会带来后续和长期的慢性疾病的一个可能条件。

美国科学家还提到，此报告也存在着不足之处，因为毕竟这只是一个体外实验，研究人员们并没有从打完疫苗的人体内直接提取干细胞再做一个类似的PCA检测，以证明确实有这种情况发生。因此还不能准确判断，但它至少说明在体外，这种可能性是明确存在的。

而且，此份报告还与之前的一个报告吻合了，之前的报告曾提到过，感染过病毒的人群中发现有部分人的病毒基因序列会被反转嵌入到染色体里面。

看到上面这则消息，普通老百姓担忧的是，辉瑞疫苗会不会跑到肝脏细胞里，或会不会改变DNA的基因序列，以及会不会影响下一代等等现实的问题。

总之，此报道已引起了广泛的注意，有专家质疑辉瑞有干扰基因序列之嫌。

大纪元网站新闻来源也提到，3月1日，美国食品药品监督管理局（FDA）公开了一份辉瑞（Pfizer）疫苗数据文件，其中提到高达1,291种副作用。 接种辉端后异常件高达四万多宗 另外其中一个章节为“授权后异常事件报告累积分析”（Cumulative Analysis of Post-authorization Adverse Event Reports），共38页，其中指出截至2021年2月28日，共接获42,086名在接种疫苗后出现异常事件的个案，涉及158,893宗异常事件，大部分个案来自英国、美国、德国、法国、葡萄牙和西班牙，其余个案来自56个不同国家。 在42,086宗异常个案其中，平均年龄为50.9岁，29,914人为女性（71.1%），9,182人为男性（21.8%），其余不详；其中有1,223人死亡（2.9%）、19,582人已康复和康复中（46.5%）、520人已康复不过有后遗症（1.2%）。 接获最多的异常事件包括头痛（24.1%）、发烧（18.2%）、疲倦（17.4%）、发冷（13.1%）、接种疫苗位置疼痛（12.3%）、恶心（12.3%）、肌肉痛（11.7%）等等。其中，有4.6%异常个案是在接种疫苗后确诊。
