## 上海民众和警察发生冲突因其住宅被强征为隔离设施

翻译&评论：金生水
![](https://assets.gnews.org/wp-content/uploads/2022/04/image-2608-15.png)![](https://assets.gnews.org/wp-content/uploads/2022/04/1-317.jpg)图片来自网络
《雅虎新闻》近日报道，周四，被封锁的上海出现了罕见的公众抗议场面，居民们因政府决定将他们的住宅改为新冠隔离设施而与警方发生冲突。

一段在中国微信平台上广泛分享的视频显示，浦东新区张江镇居民与30名身穿防护服的警察发生冲突，他们恳求警察让他们留在自己租住的公寓里。

据《金融时报》报道，周四，在警方将大院内的九栋住宅楼改为临时隔离设施后，爆发了冲突。

有1万多人观看的现场直播被微信突然切断，因为审查人员宣布该片段含有 “危险内容”。

“不是我不想与国家合作，但如果你住在一栋楼里，楼与楼之间只有10米的距离，每个人都检测呈阴性，而这些人不被允许进入，你会有什么感觉？”视频结束前，制作视频的妇女说。

据路透社报道，虽然该视频的真实性没有得到独立核实，但该住宅区的管理层证实了这一争议。

据张江集团称，当局将属于张江集团的五栋空置建筑改建为检疫设施，该集团也拥有该住宅区。它补充说，另外九栋楼也将用于同样的目的。

该集团在一份声明中说，冲突的发生是因为 “公司组织修建隔离栅栏时，一些租户阻挠了施工现场”，并补充说情况已经解决。

一位不愿透露姓名的目击者告诉路透社，工人们在星期四下午出现，警察很快就跟了上来。这名妇女说，”这个地方完全不适合成为检疫中心”，因为她表示担心从受感染的人身上感染到新冠。

根据中国的“动态清零”政策，每一个检测结果呈阳性的人都必须在一个指定的设施中进行隔离，邻居们也必须保持隔离，这激起了公众对感染病毒后果的恐惧。

上海拥有大约2600万人口，已经成为中国自该大流行病开始以来最大的新冠疫情爆发的中心，自3月以来记录了30多万例感染者。

当局正急于建造床位以安置新冠患者，该市被迫将学校、最近完工的公寓和展览厅改为隔离设施。该市宣布，它已经在100多家临时医院中设立了另外16万张床位，试图解决每天激增的感染问题。

上海的新冠病例在周五有所下降，但仍然是每天约23000例感染。虽然这比周四的约27000个新病例有所下降，但周五的数字包括创纪录的3200个无症状病例，而前一天是2573例。
