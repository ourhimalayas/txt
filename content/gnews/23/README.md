普京大帝在俄乌边境屯兵30万，而乌克兰则组织起了29万大军，并且把居民都组织起来，连老人小孩也不落下，进行军演，随时准备迎击俄国的入侵者。包括美国、英国、澳大利亚政府在内的各国政府纷纷撤离驻乌使馆人员，并且号召本国侨民离开乌克兰。战争似乎箭在弦上，一触即发。

普京大帝也真是LOW到家了，都已经是太空时代了，他还在玩侵略别国领土的游戏，乌克兰不是共匪的卖国贼，可以把二百多万平方公里的领土大笔一挥，苏俄兵不血刃就唾手得到，甚至把世界最北的不冻港海参威都纳入自己的版图。退一万步讲，就是乌克兰领导人跟中共一个德行，乌克兰人民也不是任人蹂躏的民族，加上美国等北约国家为了自己的利益，他们也不可能任其西扩。退一万步讲，就是乌克兰在你手中又能怎样呢？共匪心心念念要把香港抢到手，现在香港是到手了，同时把这个会下金蛋的香港也毁了。同样的，凭普京统治俄罗斯二十多年，俄罗斯不也是仅一万亿的GDP吗？凭普京大帝的治国能力，就是吞下了乌克兰又能如何，能咽得下去，并且能消化吗？美帝等北约国家只是不想干而已，如果真干，十个俄罗斯也不在话下吧。打仗打的是什么？不就是钱吗？俄罗斯有钱吗？不就是碰上了共匪这样的二货，为了自己的独裁统治，认贼作父，把俄罗斯当成了亲爹，把整个中国老百姓当冤大头，才让他每年捞个几千亿岁贡。否则，普京大帝说不定早就在哪个山清水秀的地方颐养天年去了。

当然，作为政治大玩家，世界上少数的几个流氓国家的领导人而已，他陈兵乌克兰，无非是跟习太阳在国际上玩个二人转的游戏，完成跟习太阳的协议，配合他攻打台湾，吸引全世界的眼光，来个“明修栈道暗度陈仓”，虚张声势而已。这样，既可以继续从习太阳身上骗取中国人民的血汗钱，二来，也借机把乌克兰当作筹码，讹诈国际社会，特别是美国等北约国家，在经济上得到好处，甚至把天然气管道直接铺到欧洲。即使北约真的强硬到底，一点好处也捞不到，反正有习太阳兜底，普京又不亏。再说了，普京只是陈兵乌克兰，战争未起，美、英、德、日、澳等国已经严阵以待，做好了一切准备，先是纷纷撤离使馆人员和侨民，然后美军已经进驻进了东欧诸国，甚至，把核武器都布置在了德国的领土上了。相信普京大帝一旦轻举妄动，最后吃不了兜着走的肯定是他。就算美帝他们不用武力，搞几个经济制裁，也能让本来就糟糕的俄罗斯经济雪上加霜，俄罗斯老百姓可不是中共国的顺民，一任普京大帝胡来。普京这么一个大玩家，会陪着习二把自己的身家性命搭进去吗？

从我个人的角度来看，我希望普京玩把大的，真把乌克兰给侵占了。上千年来，俄罗斯这个民族在侵略中国的领土抢夺中国的财物上，占尽了便宜。他侵占乌克兰之后，也让他偿偿北约飞机炸弹的轰炸，尝尝美国等西方国家更严厉的经济制裁，让普京陷入乌克兰的战争泥潭中，落得玩火自焚的下场，以解上千年来中国人民饱受北极熊欺负之恨。当然，这只是我一厢情愿的想法，我们新中国联邦人从不靠别人替我们报仇雪恨。冤有头债有主，特别是七十多年来，如果没有共匪卖国贼窃政，我们也不会失去海参威，也不会有山河之恨。现在共匪除了出卖领土之外，还在出卖中国人民的血汗养他的俄爹，只有消灭共匪，无论普京在于不在，无论俄罗斯是专制还是独裁，他们都休想从中国人民身上得到一寸的土地，一分的不义之财。
