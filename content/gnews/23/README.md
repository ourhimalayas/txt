新加坡——中共国股市周一领跌亚太市场，因为投资者对中共国3月份的通胀数据作出了反应，并监测了大陆的觊觎情况。

内地以及香港股市全天都在下跌，但到周一收盘时损失加深了。跟踪最大的大陆上市股票的沪深300指数下跌3.09%，至4,100.07点。上海综合指数下跌2.61%，至约3,167.13点，而深圳成分股暴跌3.671%至11,520.21点。

香港的恒生指数收盘下跌3.03%，至21,208.30点。中国电动汽车制造商尼奥在香港上市的股票暴跌11.44%，此前该公司宣布因Covid爆发导致其供应链伙伴中断而暂停生产。

中共国3月份的生产者通胀率高于预期。周一官方数据显示，生产者价格指数与一年前相比上升了8.3%，高于路透社调查的7.9%的预期。

中共国3月份消费者通胀率的增长也超过了预期，消费者价格指数同比增长1.5%。这高于路透社调查的1.2%的增长预期。

Vontobel资产管理公司的投资组合经理Ramiz Chelat周一对CNBC的“亚洲街景”节目说：“我认为更值得注意的事实是CPI和PPI之间的巨大差距，这表明中国大多数公司的定价能力很弱，他们的利润率受到了打击。”

数据发布时，中国大陆正在努力控制自2020年初开始的大流行以来最严重的Covid浪潮。上海报告了周日的综合病例数，有症状的914人，无症状的25173人，创下历史新高。

“鉴于Omicron的传染性，我们可以看到更多的局部封锁是一个反复出现的主题”，切拉特说。“我们认为你在中国需要有很强的选择性，寻找那些能够在增长受限的环境中实现的公司。”

在其他地方，日本的日经225指数当天下滑了0.61%，至26,821.52点，而Topix指数下跌了0.38%，至1,889.64点。韩国Kospi指数下跌0.27%，收于2,693.10点。

澳大利亚S&P/ASX 200指数与区域整体趋势相反，攀升0.1%，以7,485.20点结束交易日。

在东南亚，科技公司GoTo的股价从发行价开始飙升了约13%，因为它们在印度尼西亚首次亮相。另一方面，更广泛的雅加达综合指数下跌0.1%，至7,203.79点，摆脱了先前的涨幅。

MSCI最广泛的日本以外的亚太股票指数下滑了1.55%。

**油价跌幅接近2%**

油价在亚洲交易时段下午走低，国际基准布伦特原油期货下跌1.78%至每桶100.95美元。美国原油期货下跌1.92%，至每桶96.37美元。

追踪绿币对一篮子同业的美元指数，在最近越过100关口后，报99.739。

日元兑美元汇率为125.22，与上周兑美元汇率低于123.2的水平相比，仍然较弱。澳元在上周从0.763美元上方下跌后，报0.7433美元。
