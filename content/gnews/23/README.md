台湾总统蔡英文表示，她相信拜登政府会在台湾遭到中共袭击时为台北辩护，甚至证实美军已经在岛上，尽管规模很小。

蔡英文在周三晚间播出的独家专访中告诉美国有线电视新闻网(CNN)，“我确实相信”美国会帮助保卫台湾，并指出“与美国的广泛合作旨在提高我们的防御能力”。

蔡英文补充说，岛上有美军，但“没有人们想象的那么多”。据美国有线电视新闻网报道，五角大楼已确认台湾有32名军事人员，比2018年的10人有所增加。该网络不清楚这是否是由于守卫美国驻台北大使馆的海军陆战队人数增加，或者他们是作为价值约50亿美元武器合同的一部分而来的军事顾问。

蔡英文的言论似乎反映了台北当局对上周发生在美国的事情的看法，当时总统乔·拜登在一次公开活动中表示，华盛顿将向台湾提供援助，但第二天白宫却收回了这番言论。

台北政府相信中共国可能会在2025年之前入侵，并已开始实施军事现代化计划，发展“不对称能力”。
