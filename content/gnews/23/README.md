## 粮食进口第一大国竟称“农业强国”？
` 洛杉矶盘古农场 HIMALAYA LA PANGU` [轉載自GNews](https://gnews.org/zh-hans/2276856/)

**据网易2022年3月31日报道：党的十八大以来，我国农业综合生产能力上了大台阶。2021年，全国粮食总产量达到13657亿斤。把中国人的饭碗牢牢端在自己手中。**
![](https://assets.gnews.org/wp-content/uploads/2022/04/a1hxr-ncynl-4.jpg)图片来源：kknews.cc
中共宣传机器报道农业粮食产量 一直用斤为单位，而国际上衡量一个国家的粮食产量最小单位是吨。2021年，各个产粮大省经历水灾、旱灾和虫灾，中共国粮食总产量仍然创增产新高，达13657亿斤。
众所周知，中共的数字，没有一个是可信的。要想里了解真实情况，一般规律是，报忧的数字请自动乘以2，甚至加2个0，报喜的数字请自动除以2，甚至减2个0。

即便如此，13657亿斤粮食，按照14亿人口来计算，每人每年是大约是976斤粮食，每人每天约为2.7斤。这一数字还要包括各种粮食总和，如果去掉不能食用部分，那就更少。全国粮食还要用来做畜牧业饲料、酿酒还有其他工业用途，那么最后每人每天到底有多少粮食，每个人都能算明白。就这已经造假的数据都能让人明白粮食自给自足是个笑话，粮食都不能满足老百姓日常生活，何以称得上农业强国?

路透社引用海关总署进口粮食数据对比发现，粮食丰收就是个骗局。中共国是世界上最大的谷物消费市场，2021年全年进口2835万吨玉米，比2020年1130万吨的年度记录增长了152%；小麦进口达到977万吨，比2020年的838万吨增长16.6%；此外，进口大豆9652万吨，1248万吨大麦（比上年增长54.5%），以及942万吨高粱，增长95.6%；进口粮食总和达15654万吨。强国人一直吹嘘数学好，这个进口粮食和自产粮食差多少倍一算便知。如果2021年粮食大丰收，为何要进口如此多的粮食来补充国内缺口？
![](https://assets.gnews.org/wp-content/uploads/2022/04/vfff.jpg)图片来源: 网络
早有毛灾时代有“亩产万斤”的神话，现有进口粮食来填补国内粮食危机。进口粮食第一大国竟然在疫情封城期间饿死国民，人神共愤！在中共体制下，如果国人的饭碗仅仅靠自产粮食，必然还会出现第二次大饥荒。世界看清中共恶行，与中共国经济全面脱钩，墙内人若不警醒，老百姓吃草的日子很快就到来！
