## 印度医生谈儿童感染中共病毒
` 索尼克` [轉載自GNews](https://gnews.org/zh-hans/1355272/)

译评：文复
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/27131658/children-63175_1920.jpg)
无症状和轻度 Covid 的儿童可以在家中进行管理。

据报道，在第二波 Covid-19 期间，有更多儿童受到感染。为什么会这样？

在第二波期间，Covid-19 病例总数很高，因此检测呈阳性的儿童的绝对数量也有所增加。在上一波疫情中，数据表明，在检测呈阳性的总人数中，大约 11% 至 12% 是 20 岁以下。这一比例在第二波期间没有变化。因此，儿童并未受到比例失衡的影响，而且在未来的浪潮中，儿童不太可能受到比例失衡或是更严重的影响。

现有数据还表明，在所有检测呈阳性的儿童中，只有约 5% 至 6% 需要住院治疗。大多数住院的儿童都有一些潜在的疾病，如慢性肾病、慢性肺病、肝病、恶性肿瘤或血液学问题。

从本质上讲，当儿童接触病毒时，他们会被感染，但他们通常保持无症状或表现出轻微的症状。与成人和老年人相比，感染的严重程度较低。

唯一令人担忧的是，在 Covid 之后，在极少数情况下，儿童会患上称为多系统炎症综合症 (MIS-C) 的潜在严重疾病。因此，由于病例总数较多，因此MIS-C病例的数量也有所增加。这些儿童中有很大一部分是 RT PCR 阴性，但显示出针对 Covid-19 的抗体。我们需要准备好在未来治疗更多的 MIS 病例。

您能否详细说明多系统炎症综合症？

它是一种免疫介导的疾病，可影响心脏、肝脏、肾脏等多个系统。它可能表现为高烧、皮疹、结膜炎或眼睛发红、严重腹痛、灌注不良。有明确的诊断标准。它可能会危及一部分儿童的生命，尤其是那些心脏受影响的儿童。但如果及时诊断，是可以得到有效治疗的。

受影响儿童的平均年龄约为 9-10 岁；年龄较小和较大的儿童也可能发展为 MIS-C。在西方，大约每 1,000 名 Covid-19 儿童中就有 1 名患有这种综合症。

我们如何对待患有 Covid-19 的儿童？

无症状和轻度感染Covid的儿童可以在家中进行管理。对于无症状的儿科患者，我们不建议使用药物，但我们建议对他们进行隔离和监测。应该有一个成人留下照顾孩子，他们也应该保持隔离。

轻者若有发热或全身疼痛，应给予对症治疗。儿童应在定期医疗监督下接受治疗。他们可能需要远程医疗支持。一些患有基础疾病和轻度 Covid的儿童可能需要住院治疗。

患有中度至重度 Covid 的儿童需要住院接受支持性的护理，包括氧疗/呼吸支持。有些孩子可能需要类固醇。

国家是否有足够的儿科护理单元来应对重症患者？

该国拥有大量儿科重症监护病房，但这些都集中在城市地区。此外，大多数重症监护设施都在私营部门，因此可通达性和可承担的能力是一个问题。此外，训练有素的人力来管理儿科疾病也存在挑战。全国各地的当局都在努力增加医疗保健设施和儿科护理人力。

此外，我们需要以一种父母可以陪伴孩子的方式准备医院，尤其是患有新冠肺炎的年幼孩子。

为儿童接种 Covid-19 疫苗有多重要？

已经开始在儿童中进行试验以研究一些可用疫苗的抗体反应。这些很可能在未来几个月内完成，如果发现疫苗具有免疫原性，它们将被批准。

话虽如此，孩子们也不太可能患上有症状的疾病，而且他们很少患上严重的疾病。当前的疫苗降低了一个人患严重疾病和死亡率的机会，因此应优先为高危人群接种疫苗。渐渐地，当我们有足够的疫苗时，我们也可以为所有儿童接种疫苗。

疫情影响了儿童的身心健康。应该怎样做才能将其影响降到最低？

对儿童有许多直接和间接的影响。与成人相比，直接影响不会太多也不会太严重，但间接影响儿童的因素是多方面的。比如，很多家庭的收入在过去一年半的时间里都会受到影响，这会影响孩子的整体成长，因为它会影响他们的营养、教育等。缺课、与同龄人的互动有限或没有互动也会导致儿童出现许多行为问题。

父母或监护人为孩子提供一个愉快的环境很重要，让他们在家中参与某种形式的体育活动。此外，失去父母的孩子需要政府、大家庭和社会的支持。

（作者为新德里AIIMS小儿科重症监护室教授、负责人）
