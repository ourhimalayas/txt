## 有声｜匈牙利支持用卢布支付俄罗斯天然气

**By:秘翻新闻**

**2022-04-07**
![](https://assets.gnews.org/wp-content/uploads/2022/04/IMAGE-2022-04-07-163506.jpg)
**[路透社](https://www.reuters.com/world/europe/hungary-working-solution-pay-russian-gas-may-foreign-minister-2022-04-06/)** 4月6日报道，匈牙利周三表示，它准备以卢布支付购买俄罗斯天然气，这打破了欧盟联合反对以卢布支付的行动。

普京此前要求欧洲应以卢布支付天然气费用，否则将会削减供应，以此反制西方的制裁措施。

欧盟正努力争取一起抵制卢布支付天然气费用。周一，斯洛伐克表示将与欧盟一致行动；波兰期望今年年底前不改变与俄罗斯的天然气合约；奥地利正在与俄罗斯就以卢布支付天然气费用进行了沟通；拉脱维亚不支持以卢布支付，必须有一个共同的欧盟方法；立陶宛已表示将不再进口俄罗斯天然气，成为欧洲第一个抵制俄罗斯能源供应的国家。

而在周三，匈牙利总理欧尔班回答记者提问时表示，如果俄罗斯提出要求，匈牙利将以卢布支付费用。在欧盟成员国中，匈牙利是少数几个拒绝因俄罗斯入侵暴行而制裁其能源产业的国家之一。

目前，俄罗斯通过三个主要管路向欧洲输送天然气的总量基本保持稳定。
