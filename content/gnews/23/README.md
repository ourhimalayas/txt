## 现身直播举反战标语 俄女编辑被罚3万卢布

采集：玫瑰新闻组
![](https://assets.gnews.org/wp-content/uploads/2022/03/屏幕截图-2022-03-16-200232.jpg)
（早报讯）周一晚举反战标语打断俄罗斯国营电视台新闻直播节目的女编辑奥夫夏尼科娃，被法院罚款3万卢布（约378新元）。

奥夫夏尼科娃（Marina Ovsyannikova）周一（3月14日）晚间在第一频道的新闻直播节目中，突然出现在节目主持人身后，高举反战标语和高喊口号。标语牌用英语和俄语写着：“不要战争。停止战争。不要相信宣传。他们在说谎。”

电视台立即切断直播画面，奥夫夏尼科娃被拘留且一度失联，引发外界担忧她恐遭遇不测。

在抗议事件发生前，戴着乌克兰国旗蓝黄相间项链的奥夫夏尼科娃在推特上发布了一段视频，介绍自己的职业，并称以多年来为克里姆林宫宣传机器工作为耻，为被允许在电视屏幕上说谎而羞愧。

她在视频中说：“乌克兰现在发生的一切是在犯罪，俄罗斯是侵略者。对这次侵略负有责任的只有一个人，就是普京。”

她还透露，自己的父亲是乌克兰人，母亲是俄罗斯人，并说：“现在整个世界都离我们而去，我们的下一代也无法洗去这场手足相残的耻辱。”

她敦促俄罗斯同胞加入反战抗议活动，结束这场冲突，“只有我们有能力阻止这一切的疯狂行为”。

国际文传电讯社（Interfax）周二（3月15日）报道，奥夫夏尼科娃当天在律师陪同下出庭应讯，法院裁定她违反公众示威法，罪名成立，罚款3万卢布。

英国《卫报》指俄罗斯法院对奥夫夏尼科娃的惩罚相对轻，颇令人意外。

不过，这笔罚款并非针对她硬闯直播节目的行为，而是针对她事前发布的视频。

奥夫夏尼科娃的大胆行为在俄罗斯社会引发震惊，却赢得西方领导人的喝彩，法国总统马克龙甚至表示其领事可为她提供“保护”，并表示会向俄罗斯总统普京提及她的案子。

乌克兰总统泽连斯基对奥夫夏尼科娃的抗议行为表达感谢。他说：“我感谢那些不断试图传达真相的俄罗斯人，那些与虚假信息作斗争，告诉他们的朋友和亲人真相的人。”

此前媒体称，奥夫夏尼科娃可能在俄罗斯3月4日通过的一项诋毁武装部队名誉法律下被提控。该罪名的最高刑罚为监禁15年。
