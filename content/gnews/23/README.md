![](https://assets.gnews.org/wp-content/uploads/2021/12/image-1212.png)*香港在严格的新法律下举行了第一次选举，赋予政府更大的权力来审查新候选人。 一些民主活动人士呼吁抵制，导致选民投票率远低于 2019 年的选举。*
自《天空新闻》2021年12月21日报道，在中共对香港选举制度进行“大修”之后，香港的选举被五眼联盟谴责为没有任何“有意义的政治反对”。

美国、英国、澳大利亚、加拿大和新西兰在香港选举周日结束后发表联合声明，选举投票率创历史新低。这一结果为30.2%，是自1997年香港回归中国以来的最低水平，远低于2000年记录的43.57%的结果。

五眼联盟的合作伙伴批评中共国今年早些时候“侵蚀了特别行政区选举制度中的民主元素”。“破坏香港权利、自由和高度自治的行为正在威胁我们看到香港成功的共同愿望，”五眼联盟说。“自回归以来，具有不同政见的候选人都在香港参加了选举。昨天的选举扭转了这一趋势。”中共国带来的变化看到直接选举的席位数量减少，并建立了新的审查程序，限制了选票上的候选人数量。

五眼联盟将这些变化描述为“严格”限制性的，只是为了消除“有意义的政治反对派”。“该市的许多反对派政客，尤其是‘47名反对香港国安法(NSL47)’中的大多数仍在监狱等待审判，其他人则流亡海外”他们说。对《国家安全法》“更广泛的寒蝉效应”和“对言论自由和集会自由的限制越来越多”的担忧也受到谴责。

声明继续说：“不支持政府议程的非政府组织、工会和人权组织被迫解散或离开，而媒体自由正在迅速受到限制。”声明最后敦促中共国通过尊重香港“受保护的权利和基本自由”来履行其国际义务。

五眼联盟的发布得到了七国集团（G7）和欧盟的回应，他们对周一该市立法委员会选举的结果表达了类似的担忧。他们说：“我们再次强烈呼吁中国按照《中英联合声明》和其他法律义务行事，尊重香港基本法规定的基本权利和自由。”声明补充说：“我们还呼吁中共和香港当局恢复对香港政治机构的信心，结束对提倡民主价值观和捍卫权利和自由的人的无理压迫。

数小时前，美国对五名驻香港的中共国官员实施制裁，将今年早些时候制裁的中联办其他同名官员名单增加了。他们是何静、陈东、卢新宁、谭铁牛和尹宗华。制裁意味着美国公民和企业被禁止与目标人物打交道，这些人物在美国的资产通常也会被冻结的。

香港行政长官林郑月娥去年与其他10名市政府官员一起因与中共政权合作实施破坏自由和民主的政策而受到制裁。

周一，美国国务院提供了一份关于香港的最新报告，其中提到了一些事态发展，例如该市的《苹果日报》于6月被关闭。随着香港选举制度的改变，公务员也提出了新的宣誓要求。本月，苹果日报创始人黎智英和两名共同被告被判犯有与2020年天安门广场守夜活动有关的未经授权的集会指控。

简评：昔日亚洲金融中心被誉为东方之珠的香港如今已经面目全非，早已丧失了自由、民主和安全，正如文贵先生早在2017年就指出上午，大陆资金频频在香港以超高价买地，并非是搞发展，而是经济“掺沙子”，控制整个香港。房地产市场，也就是“控制香港的生死”，是一场政治阴谋。文贵先生在YouTube直播中又明确谈到了“北京绝对控制林郑月娥！”警示世界所提出的，香港将不再有民主自由！

七国集团（G7）与欧盟之后也加入行列，就香港立法会选举发布声明，批评选举削弱了香港在「一国两制」原则下的高度自治。

英国外交大臣特拉斯后在发表联合声明时附文称“昨天的选举暴露了北京正在努力消除香港的权利、自由和自治。英国与澳大利亚、加拿大、新西兰和美国站在一起，支持自由，并呼吁中国按照其国际义务行事。”
