美国不会被这些行动所阻止，而是会继续使用一切外交和经济工具来推动问责，并与那些为人权和基本自由发声的人站在一起。北京持续对那些为人权而发声的人进行恐吓和噤声，这只能让国际社会加大对新疆持续发生的种族灭绝和反人类罪的审视。布林肯再次呼吁中共国停止其跨国压制行为，包括监禁美籍维吾尔人的家属以及剥夺其行动的自由。

中共没有人权，天下人都知道。但是中共敢做不敢当，更受不了别人指责。于是各种恐吓、威胁、禁声、制裁。但正义的声音是压不住的，全世界人民的眼睛是雪亮的，早晚中共欠下的债务要自己来偿还。
