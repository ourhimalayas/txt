那么是否真是如刘先生所说，他真是接连两年中千万大奖吗？笔者认为他的说辞肯定是胡编乱造的，众所周知，原民政部长李立国贪污2000亿彩票款项的事这还没过5年，这中共国彩票就拼命制造中奖新闻想给自己洗白，诱惑更多韭菜自愿送来收割。

你要是不知道轰动中共国的彩票贪污案，那笔者就来翻翻历史，说说中共官员李立国的“光荣事迹”： 1953年出生的李立国，历任西藏自治区常委、秘书长、副书记、秘书长、副书记、政协副主席、党组书记，民政部党组副书记、常务副部长、党组书记、常务副部长、党组书记、部长。第十八届中央委员。李立国在小金库开设27个帐号，存款超过11.57亿元。李立国包养11名情妇，自2005年7月至2016年7月，李立国通过挪用公款、侵吞彩票资金，送给11名情妇逾11亿2千470余万元。而且把15套在北京、天津、珠海住宅送给其中9名情妇。中共巡视组、审计署、监察部自2014年查证、审计出25省（区）直辖市福利彩票涉嫌作假、挪用、侵吞资金1700余亿元。李立国自知大事不妙，部署外逃计划。搜查发现李立国在北京、天津、珠海住宅保险柜里有12本护照，其中9本是用假名以及假身份证。有讥讽称“那些戴着面具去领双色球大奖的可能都是李立国的情妇”。

从2017年1月9日中共对李立国进行立案调查到现在还没到五年呢，不过由于此事对中共所控制的彩票业是一个巨大的打击，绝大多数老百姓从那时起就不再相信彩票了，以前街上每过几百米就有一个彩票销售点，而现在可能几条街都难见到一个彩票销售点，笔者也在N多年前也为中共国的彩票骗局“送菜”百来块吧，现在想起来真让人笑话。

现在中共政府为了提高销量好收割更多的韭菜，中共在媒体上隔三岔五地就放出消息，某某中了千万大奖，当然这只能对新韭菜有点效果，对于90后及之前出生的人来说几乎都不管用。那么此刘先生所谓的第二次中千万大奖是否是真的呢？当然细看就会发现一些破绽，刘先生所称上次是母亲代领奖，笔者看来这点实在难以解释，众所周知，中共国的人都知道财不外露的道理，哪怕自己最亲的人都不会告诉，仅仅因为太过激动就让母亲代领这个理由确实太过牵强，一个大男人难道还不如自己年迈的母亲？而且刘先生也戴头套，无法查证这个人倒底是谁。另外还有就是，根据中共假骗偷本质，以及彩票的黑历史，我们完全可以得出结论，此次彩票开奖又是假的，是一次炒作，为了骗更多涉世未深的年轻人去买的。
