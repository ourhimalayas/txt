![](https://assets.gnews.org/wp-content/uploads/2021/12/12-7.jpg)[图片来源](https://www.gvm.com.tw/article/85386)
德国之声中文网12月18日消息，台湾周六（12月18日）举行全民公投，四大公投案分别为“重启核四”、“反莱猪进口”、“公投绑大选”和“珍爱藻礁”。其中莱猪进口议题被认为可能影响台美关系。

周六的公投从上午8时持续至下午4时。根据台湾中央选举委员会公告，四大公投有投票权的人数为1982万5468人，依照《公民投票法》的“四分之一有效同意票”规定计算，通过门槛为495万6367票，而且同意票必须高于不同意票。最终投票结果，四大公投皆未通过。

此次公投，国民党与民进党意见相左，以“四个都同意 台湾更美丽”为口号，呼吁支持者全部投下赞成票。国民党提出的同意理由包括反莱猪进口守护食品安全、公投绑大选更省钱、重启核四让台湾不缺电。

台湾总统蔡英文在当地时间晚间7时半召开记者会，针对公投发表谈话。她感谢所有参与公投的台湾人民，无论投下的是同意或不同意票，都是公民意志的展现。蔡英文表示，此次公投结果传递了四个讯息：台湾人民希望走向世界，愿意积极参与国际；人民支持能源转型，期待稳定供电，维持经济成长动能；人民重视生态环保，希望经济和环保达成双赢；人民期待公共政策资讯透明、被理性讨论。她也强调，会将不同的意见纳入政策评估与规划中。
