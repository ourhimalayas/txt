德国开姆尼茨市的一个诊所负责人因为 “关于疫苗接种的谎言 “而自杀。 这位名叫托马斯·扬格的医生近日从医院的屋顶上跳下自杀。  据称，在他的遗书中，医生这样为自己的举动解释：他不能再容忍与Covid有关的 “关于所谓无害疫苗的持续谎言和欺骗”。

扬格医生写了一封令人印象深刻的遗书，他要求将其公布。据说，他在信中尖锐地批评了政府关于Covid-19疫苗接种危险性的信息政策。这位医生再也无法忍受不断的谎言，即疫苗接种是无害的。他也不能再欺骗病人了。 在说明中，扬格医生强烈谴责用实验性的、致命的Covid-19疫苗为民众接种疫苗，这实际上是一种生物战剂。在他看来，这是种族灭绝和反人类罪。

由于市长威胁他说，如果他不遵守规定，拒绝为诊所的病人接种疫苗，就会被解雇，杨格医生大概没有其他选择。他不想宽恕联邦政府、州政府和他们的党羽所犯下的任何罪行。医生认为他的自杀是发出反对这一切信号的唯一途径。 这些是整个遗书的单独摘录，现在正在流传。尽管死者要求公布遗书全文，但据报道，开姆尼茨市市长斯文·舒尔茨继续阻止这一公布。
