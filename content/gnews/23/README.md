四川蓝光发展股份有限公司(下称蓝光发展)继续深陷债务风波，其逾期未偿还债务规模仍在扩大。11月11日，中共国人民法院公告网显示，新增一则以蓝光发展为被告的开庭公告，其中提到其再牵涉一起票据追索权纠纷。中新经纬统计发现，自2021年7月以来，因流动性紧张及债务风险发酵，蓝光发展已收到了来自数十家金融机构及合作方的起诉状，诉讼涉案金额合计达145.35亿元人民币。

在蓝光发展发布的相关公告中，共罗列了100个诉讼案件，具体诉讼事由包括融资租赁合同纠纷、买卖合同纠纷、建设工程施工合同纠纷、金融借款合同纠纷、借贷纠纷、履约金返还纠纷等。蓝光发展6日发布的公告显示，截至2021年11月5日，蓝光发展累计到期未能偿还的债务本息金额合计241.71亿元(包括银行贷款、信托贷款、债务融资工具等债务形式）。

在蓝光发展逾期未偿还债务规模扩大、诉讼缠身的背后，是其下滑的经营业绩。蓝光发展2021年三季报显示，前三季度，蓝光发展实现营业收入148.56亿元，同比下降52.65%，净亏损67.05亿元，同比下降351.06%。截至9月30日，蓝光发展总资产为2045.87亿元，较2020年末减少20.86%。此外，其房地产业务的销售金额、销售面积均明显缩水。

另据三季报，截至9月30日，蓝光发展的货币资金余额为100.29亿元，其中可自由动用资金仅3.30亿元，各类受限或者限定用途的资金为96.99亿元。这组数字与蓝光发展到期未能偿还的债务本息和重大诉讼涉案金额相比，还相差颇多。Wind数据显示，截至11月12日，蓝光发展境内债券存量规模为83.73亿元，其中46.23亿元将于一年内到期。

在蓝光发展于2021年7月出现债务违约后，此前已连续下调其信用评级的国际评级机构标普和穆迪先后撤销了蓝光发展的信用评级。大公、中诚信国际、东方金诚等评级机构也在7月再次下调其信用等级。此后，蓝光发展多次表示公司正在努力化解债务风险，但从公司的财务数据来看，其债务危机才刚刚开始。
