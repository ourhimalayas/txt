2022年4月3日，郭文贵先生在大直播中爆料，中共在五天前已将红旗导弹系列部署到塞尔维亚，直接对欧洲和北约造成了威胁，这表明中共已经正式参与了俄乌战争。

2022年4月9日，郭文贵先生在盖特上置顶了中共给塞尔维亚送去了红旗导弹系统的帖子，并附上了一张来自于推特的截图，推主为Evergreen Intel。该推主在4月9日在推特中跟帖另一位推主Amelia Smith的推文，分别公布了通过开源飞机追踪器查询到的飞往塞尔维亚的六架“运-20”运输机信息。

运-20又称“西安Y-20”，是由西安飞机工业公司开发的一种大型军用运输机。该项目于2006年正式启动，飞机的官方代号为“鲲鹏”，取自《庄子》中所描述的一种可飞行数千公里的神鸟。

最早推主Amelia Smith在[推文](https://twitter.com/ameliairheart/status/1512602756913246212)中写到：“解放军空军的运-20，编号20045，从土耳其伊斯坦布尔起飞，向西飞行。不知道它是什么时候飞到这的。”

该推主同时对比使用了Flightradar24和ADSBexchange的数据。Flightradar24是一家提供实时航班飞行状况的互联网服务商，它提供航班的飞行轨迹、出发地、目的地、航班号、注册编号、飞行器型号、当前位置、高度和空速等，同时也可以显示过往航班的信息。ADSBexchange则是世界上最大的未过滤飞行数据的公共来源。

根据以上查询到的开源数据显示，在同一条航线上，每架运-20相距约100公里，从土耳其的伊斯坦布尔一直延伸，几乎到了格鲁吉亚的边境。推主Evergreen Intel称，她已经追踪到总共43架运-20A/U的机身，其中至少有7架测试机，至少1架运-20U的加油机。

根据中共公开信息显示，2019年末西安飞机工业公司仅生产了20架运-20飞机的机身。已追踪到的43架已知的运-20，占中共空军现役机队的15%。并且这只是使用开源平台所观察到的情况，实际上此次飞行机队的规模可能要大得多。

中共在凌晨时间飞出如此众多的运-20，并且都使用MLAT多点定位，这很显然是在进行预定的武器交付。之前北约军队在使用运输工具向波兰或乌克兰运送货物时也采用类似的MLAT多点定位。

4月10日，三架运-20运输机出现在塞尔维亚首都贝尔格莱德的尼古拉·特斯拉机场，证实了推主Evergreen Intel公布的信息的真实性，而中共这些运-20飞机向塞尔维亚军方运送的正是红旗HQ-22地对空导弹系统。
![](https://assets.gnews.org/wp-content/uploads/2022/04/8-26.jpg)
运-20的集体飞行引起了人们的注意，从飞机飞行的轨迹可以看到，其经由的国家有吉尔吉斯坦、乌兹别克斯坦、阿塞拜疆、土耳其、保加利亚等国家。在中共向欧洲后院运送战略性进攻武器的时候，不仅只是塞尔维亚为中共开了绿灯，以上这些国家也都批准中共军机进入了他们的领空。
![](https://assets.gnews.org/wp-content/uploads/2022/04/9-26.jpg)
由此可见，中共势力之大，已绝对不可小觑。并且郭文贵先生还爆料称，中共不仅在塞尔维亚部署了红旗导弹和反导弹系统，已经或将要在中东多国以及南亚做出同样的部署，这意味着中共已经公然向美欧宣战，美欧等国家又将何去何从？！
