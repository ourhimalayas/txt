#### 撰稿：东京樱花团/喜马拉雅的微尘
![](https://assets.gnews.org/wp-content/uploads/2021/10/1234.png)[图片来源](https://sunnews.cc/baijiahao/198013.html)
中国基金报10月17日讯，对于日本制铁公司控告侵犯其专利并索赔约200亿日元（约合人民币11亿元）一事，宝钢股份最新回应称，对于日本制铁单方面主张，宝钢股份不予认同。针对其所提出的技术专利诉讼，宝钢股份将积极应诉，坚决捍卫公司权益。

10月14日，全球第五大钢铁企业、日本最大钢铁制造商日本制铁公司公开声明，称该公司持有的电磁钢板相关专利遭到侵权，已向东京地方法院提起诉讼，对丰田汽车和中国钢铁巨头宝山钢铁分别索赔约200亿日元（约合人民币11亿元）。

日本制铁指出，宝山钢铁制造违反钢板的成分和厚度等自身专利的电磁钢板，向丰田供应，而丰田使用这种钢板的马达用于电动汽车。

对于上述事项，10月16日，宝钢股份回应称，针对日本制铁所提出的技术专利诉讼，其将积极应诉，坚决捍卫公司权益。

长期以来，中共通过各种手段大肆剽窃外国企业的技术，并利用行政和司法权力进行袒护，加之技术专利和知识产权纠纷调查取证的复杂性专业性，造成中共国实施侵权和剽窃的企业往往违法后还有恃无恐。因此，对于此次日本制铁对宝钢股份的技术专利侵权诉讼事件的进展，值得我们持续关注。
