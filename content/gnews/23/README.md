据官方朝鲜中央通讯社报道：7月1日，朝鲜领导人金正恩向中共主席习近平发去贺电，庆祝中共国产党成立100周年。

金正恩在发去贺电的同时还表示，朝鲜劳动党与中国共产党坚定团结，他将把朝中两国友好关系提升到时代要求和两国人民期望的新战略高度。他还在发言中还说，敌对势力对中国共产党的恶毒诽谤和施压都只是暂时的，他们永远无法阻止中共国人民前进的步伐。

本周三，朝鲜官方媒体称金正恩在政治局会议上说，由于抗击疫情运动中的“关键”失误才酿成“巨大危机”。外界猜测金正恩可能在向国际社会求助，希望提供包括疫苗在内的援助。

自从中共病毒疫情爆发以来，朝鲜维持着世界上最严厉的一些疫情预防措施，包括封关一年半时间，这一系列严格的防控措施摧毁了本已陷入困境的经济，金正恩也曾说过朝鲜正面临“有史以来最糟糕”的情况，目前尚不清楚朝鲜何时会重新开放边界。
