周四，俄罗斯军队从陆、空和海上入侵乌克兰，这是自二战以来欧洲一国对另一国发动的最大规模袭击。

俄罗斯导弹如雨后春笋般袭击了乌克兰城市，乌克兰报告称，大批军队越过边境进入切尔尼戈夫东部、哈尔科夫和卢甘斯克地区。 俄罗斯军队还通过海上登陆南部的港口城市敖德萨和马里乌波尔。

在拥有 300 万人口的首都基辅，黎明前就听到了爆炸声。 枪声震耳欲聋，警笛声响起，随着居民逃离，出城的高速公路交通堵塞。 中午时分在基辅发生爆炸后，乌克兰军事情报总部上空冒出黑烟。 这次袭击是在西方领导人为避免战争而进行的几周徒劳的外交努力之后发生的，并意识到他们对俄罗斯总统弗拉基米尔·普京的野心最担心的事情。

乌克兰总统泽连斯基（Volodymyr Zelenskiy）在推特上写道：“俄罗斯在早上对我们国家进行了背信弃义的袭击，就像纳粹德国在二战期间所做的那样。”  “截至今天，我们国家处于世界历史的不同方面。俄罗斯走上了一条邪恶的道路，但乌克兰正在捍卫自己，无论莫斯科怎么想都不会放弃自由。”

他呼吁乌克兰人在城市街道上保卫国家，并表示任何准备战斗的人都将获得武器。 他还敦促俄罗斯人走上街头抗议政府的行为。 欧盟外交事务负责人何塞普·博雷尔说：“这是自二战以来欧洲最黑暗的时刻。”乌克兰第二大城市哈尔科夫的居民说，这是距离俄罗斯边境最近的大城市，公寓楼的窗户都在颤抖。 不断的爆炸。 在马里乌波尔郊外，靠近俄罗斯支持的分离主义分子控制的前线，俄罗斯轰炸袭击的森林中的大火冒出滚滚浓烟。 一支乌克兰装甲纵队沿着道路前进，士兵们坐在炮塔上，微笑着向过往的汽车挥动胜利的标志，这些汽车鸣喇叭以示支持。

在附近的曼古什和别尔江斯克城镇，人们排队领取现金和汽油。 有人看到来自马里乌波尔的平民正在收拾行李。  “我们要躲起来了，”一位女士说。 初步的伤亡报告未经证实。 乌克兰报告称，在赫尔松南部地区，俄罗斯炮击造成至少 8 人死亡，3 名边防警卫死亡。 乌克兰军方表示，它在哈尔科夫附近的一条公路上摧毁了四辆俄罗斯坦克，在卢甘斯克地区的一个城镇附近杀死了 50 名士兵，并在东部击落了六架俄罗斯战机。

俄罗斯否认有关其飞机或装甲车被摧毁的报道。 俄罗斯支持的分裂分子声称击落了两架乌克兰飞机。 普京在凌晨的电视宣战中表示，他已下令进行“特别军事行动”，以保护包括俄罗斯公民在内的在乌克兰遭受“种族灭绝”的人——西方称这是一种荒谬的宣传。  “为此，我们将努力实现乌克兰的非军事化和去纳粹化，”普京说。  “在来自现代乌克兰领土的持续威胁下，俄罗斯无法感到安全、发展和生存……所有流血事件的责任都将落在乌克兰执政政权的良心上。”
