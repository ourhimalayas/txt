## 脱钩中共国，新加坡正在成为半导体巨头
` 墨尔本雅典娜农场` [轉載自GNews](https://gnews.org/zh-hans/1803945/)

整理报道：文扬
编辑：文泓
![](https://assets.gnews.org/wp-content/uploads/2021/12/2-110.jpg)图片来源：Nikkei Asia（格罗方德芯片新加坡工厂）
12月28日，据TFI新闻报道，新加坡早已抛弃了中共国，现在又走得更远了。新加坡决定在最要害的地方打击“纸龙”——半导体行业。新加坡计划通过与美国半导体设备制造商应用材料公司（Applied Materials）的一项新的2.1亿美元的联合研究投资，来满足下一代设备的需求。

总部位于美国的应用材料公司和新加坡科技研究局的微电子研究所（IME）正在将其研究合作关系延长至2026年。最新项目的投资，将升级和扩大两个实体在2011年建立的联合实验室，并使其能够加快混合粘接和其他3D芯片集成技术的材料、设备和工艺技术解决方案。

此举对于中共国而言是一个巨大的挫折，其本身就急需半导体并且正在努力解决供应问题。

**新加坡试图再次成为一个半导体巨头**

新加坡一直拥有健康的半导体产业。该国拥有优惠的税收和良好的监管制度，这也使其成为世界上最先进的经济体之一。新加坡也拥有良好的人才库和一些高技能工人，以及便利的衔接，都促进了半导体行业的蓬勃发展。

在亚洲“四大”——台湾、中国、韩国和日本之外，新加坡拥有该大陆最大的半导体制造产业。

早在2000年，新加坡的半导体行业就表现出色，生产了价值615亿美元的计算机、电子和光学产品，包括半导体。然而，由于来自台湾的激烈竞争，这个东南亚国家的电子行业出现了下滑，台湾是与另一个与中共国敌视的岛国。

然而，新加坡仍然是电子和半导体生产的巨头。截至2020年，包括半导体在内的电子产品制造业增长至全国制造业总产值的46.3%。新加坡也占全球半导体设备份额的19%。

新加坡是外国芯片制造商和本地合同制造商的天堂。一些全球行业巨头，如第三大半导体生产商格罗方德、三星、英特尔、SK海力士、美光和高通，都在这个岛屿国家设有机构。

新加坡也是一些著名的本地半导体公司的所在地，如UMS控股公司和测试解决方案供应商AEM控股公司。与此同时，新加坡也在寻求与印度合作，一个来自新加坡的财团也表示有兴趣在这个南亚国家建立半导体制造设施。

**新加坡的半导体行业复兴如何伤害中国**

这一切都与供应链有关。与任何其他国家一样，对于中共国来说，半导体也是不可或缺的产品。从汽车电池到笔记本电脑，再到智能手机，从家用电器到游戏机，以及介于两者之间的一切，半导体在消费电子产品中不可或缺。

然而，自从美国川普政府禁止向中共国出口美国设计的机型，使中国电子行业陷入瘫痪以来，中国正在努力满足本地半导体的发展。在川普政府将华为和中芯列入黑名单并拒绝向其出售半导体后，华为和中芯被迫削减了生产规模。

中国正在努力建立本地半导体产业，但它根本没有办法起飞，因为在如此复杂的技术中自力更生绝非易事。现在，**如果像新加坡这样的国家全力发展自己的庞大半导体产业，那么北京的危机只会恶化，因为这些关键芯片的供应链将在一个试图与中共国脱钩的国家得到本土化**。

**新加坡试图摆脱中共国**

新加坡对半导体的大力推动，只是其在面对中共国威胁时觉醒的一段插曲。这个东南亚国家明白，其价值数十亿美元的证券投资和与共产党国家的广泛贸易关系并不是一个很好的局面。

中共国房地产的崩溃已经伤害了新加坡，现代置地（中国）未能支付在新加坡交易的2.5亿美元债券的利息和本金。

在这一事件之后，**新加坡意识到它必须将其经济与中共国彻底分开。现在，新加坡明确表示将单打独斗，重新发展成为半导体巨头，彻底与中共国脱钩。**
