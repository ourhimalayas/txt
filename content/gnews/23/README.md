上帝为人类创造了美好的地球，每个人都有资格享受地球上供应的一切，土地被赐福很容易就能获得出产，人类本可以在地球上简单而舒适地过活，但私欲污染了人心，土地被私有化，资产被少数人占有，权力被少数人掌握，尤其是中共独裁下的中国各地，虚假的繁荣下尽是衣衫褴褛食不果腹的凄惨人生。香港这个国际都市，同样也有一群无家可归的露宿者，他们常年住在通州街的桥底，几块托架铺上纸板已可蜷缩度日，有张破床垫已是舒适的家。他们每个人都有生活的无耐，但并没有寄希望于越来越专制的政府，他们中大部分人白天会外出打零工，晚上在公园的公厕内梳洗后回到桥底。通州街桥底的露宿者已经存在了很久，虽然有碍观瞻，但露宿者十分自律，与附近的居民相安无事多年，居民们对露宿者大多数持同情的态度，许多慈善机构和有心人更将这里的露宿者定为固定关顾的对象，经常免费派饭、送上物资，社工和区议员们也提供咨询和帮助，通州街的露宿者也有喜乐。

中共对香港的集权欲望越来越强，暴政也愈演愈烈，借口清洁城市，维护城市安全，对露宿者接二连三地进行驱逐， 2012 年 2 月15日，深水埗民政署、食环署在没通知下，在通州街天桥底「清场」；2015 年7月31日，油尖旺民政署、食环署把油麻地澄平街行人隧道廿多位露宿者物品丢弃；2019年12月21日, 警务处数十位防暴警, 以灭罪为理由驱赶深水埔通州街公园无家者, 同一时间康文署安排数十位清洁工人 把无家者家当变垃圾丢掉。时逢反逃犯条例运动被中共黑警血腥暴力镇压，警方这场夸张的暴力驱赶露宿者的行动被全港市民炮轰，媒体纷纷报道露宿者被黑警虐打被夺家当的凄惨，社会的声讨不绝于耳，其中有14名露宿者更勇敢站出来为自己寻求公道，入禀小额钱债审裁处向港府索偿，可惜其中一位未等到判决已于去年离世。香港人从来没有嫌弃过露宿者，每个人都有选择如何生活的权力，政府无能解决他们的困境，更无意支持他们，却要代表港人压迫最底层的同胞，这种丑恶的行径只会被更多人唾弃和不齿。

国安恶法治港，为民发声的组织和代表悉数被捕被禁声，中共有更多理由清除它眼中的“低端人口”，露宿者的境遇将更加凄凉。反观那些居住在政府公共屋邨的低收入港人，他们同样要面临被港府逐步清理的结局，中共的共同富裕口号从来不会包括低收入者，这些令穷人翘首以盼的忽悠最终会砸碎所有香港人生存的希望。
