笔者在日本雅虎网站，以及今晨的日本电视上看到这样的内容：

一个新的社交网站GETTR诞生了，好像是由川普的前公关人员负责监管。

美国时间7月1日，Politico报道，由前总统唐纳德-川普的前发言人杰森-米勒领导的团队推出了GETTR，这是一个类似Twitter的社交网络。

该平台自称是 “为全世界人民提供的无偏见的社交网络平台”，并在谷歌游戏商店和苹果的应用程序商店提供应用程序。 它在设计上与Twitter相似，并提供与Twitter相同的功能。包括发推热门话题，可关注喜欢的账户、为喜欢的帖子点赞，以及发布照片和视频。

虽然该应用在Google Play和App Store的描述中的 “新内容 “部分指出这是该应用的 “第一个版本”，但GETTR上有一些帖子可以追溯到2015年。 右翼媒体网络Newsmax的几个帖子的日期是2015年4月11日；还有一个2019年的帖子，作者是Tim Murtaugh，他是2020年美国总统选举中川普竞选团队的通讯主管。

米勒在3月份告诉福克斯新闻，川普将在 “几个月内 “带着自己的社交媒体平台回来。
