## 苟延残喘 中共国房地产龙头恒大卖飞机偿还债务
` 日本东京方舟农场` [轉載自GNews](https://gnews.org/zh-hans/1644066/)

**撰稿：日本东京方舟农场  子辰**

据《大纪元》11月6日报道，中共国房地产龙头企业恒大集团近日通过出售两架私有飞机，筹集了五千多万美元，用来偿还逾期的美元债利息。

《华尔街日报》援引知情人士的消息报道称，此次恒大出售的两架湾流喷气式飞机是由美国飞机投资者购买，交易于10月份结束。同月，恒大在30天宽限期到期前支付了两笔逾期的美元债利息。

![](https://assets.gnews.org/wp-content/uploads/2021/11/图片1-19.jpg)图片来源：https://www.epochtimes.com/gb/21/11/5/n13356427.htm

报道称，恒大最近出售的这两架湾流喷气式飞机是其拥有的较小型飞机。该公司正在为更大的空客ACJ330宽体喷气式飞机寻找买家。此架飞机是许家印在公司鼎盛时期挥霍购买的豪华飞机之一。

此前，恒大未能按时支付9月23日和9月29日到期的金额分别为8350万和4750万的美元债利息。若恒大在30天的宽限期结束时仍未能还债，投资者就可宣布恒大违约。 《路透社》引述一名恒大债券持有人的话指出，恒大公司拆东墙补西墙，偿还的也只是利息而已，该公司只是在买时间。
