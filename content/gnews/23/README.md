草根小哥：有好几种方式。现在（从）国内投资甚至我觉得比海外还要更容易些，当然是需要得到农场的协助和配合，这方面请跟农场保持联系，跟农场负责人沟通一下，请他/她协助你来完成这个投资。

16. 如果把报备卡项目改为（投资）重建项目的话，喜币的0.2配额还会有吗？应该怎样操作？

老班长：会有的，不受影响。

17. 战友想参加联盟数字银行（项目）投资，可不可以加入农场的投资群？可以联系谁？

长岛哥：

- 目前我们数字银行投资群有两个，一个是联盟专门的群，另外一个就是香草山的专门的群，因为人比较多，所以我们分开建群了。
- 如果你现在要参加联盟基金走数字银行的，就联系木兰；当然你也可以通过各农场主来联系，把信息交换一下。
- 加群的话，如果你是香草山的，你就联系我；如果你是其他农场的，也可以联系我或者各自的农场主，都可以加入这个数字银行群里。


18. 去年10月14日战友邮寄支票购买G|CLUBS卡，到现在还没兑现，担心支票过期，请问现在怎么办？

草根小哥：

- 大家再往G|CLUBS寄支票的时候，一定要选择可随时追踪状态的这种支票，不要是现金支票就寄在信封里直接邮寄，这种是不推荐的，不可以的。


19. 请问躺平币是按投资给配额吗？

老班长：躺平币正在做规划，大概率会在年底才会推出，相关的具体的措施还没出台。但是可以肯定的是，就像法治基金、法治社会的捐款者还有我们这些G系列的投资者，还有我们积极的辛勤工作的义工们，肯定都是最大的受益者，具体的细则要等到公布之后才能告诉大家。

20. 投资汇款比较难，能不能申请农场协助汇款？协助汇款的条件有哪些？

长岛哥：

- 尤其是墙内的战友，我们更要去帮助他们，很多时候他们的钱出来不像我们在海外那么容易，我们海外的有时候汇款都会被刁难，那墙内的（资金）出来就是难上加难，所以这块我们需要农场去协助，或者农场间合作帮助战友去完成。
- 战友有什么问题就跟农场主或者农场的投资义工联系。

草根小哥：加勒比海有很多岛国，不知道说的是哪个？我就说两个比较著名的：BVI这个国家是不支持的；开曼群岛它的政策是可以支持的。所以目前战友想走离岸公司持有的话，开曼群岛是个不错的选择。具体的大家还要去研究研究，因为我也不是专业的。

老班长：

- 当然你由你亲友代持了，那是另外一件事；或者你通过基金的方式作为你资产的一部分，是可以的。我们未来的联盟基金会为大家解决一系列的这样的相关问题。
- 哪个国家的政策不是一成不变的，据我们所知，某些国家可能正在出台数字（货币）相关法令，可能很快就会开通，具体要看时间的发展。
- 因为我们今天说的重大利好——推迟到11月30日作为汇款到账截止日期，对每位战友来说都增加了可选择性。乍一听是汇款时间推后了，实际上它影响的周边效应是很大的，懂的战友自然明白。

长岛哥：

- 至于你是升级卡，你刚刚升级，你完全没必要现在就支付年费，快到一年的时候G|CLUBS也会通知你，那时候再付也可以的。

草根小哥：我相信一定会要求进行KYC的，但具体是什么标准现在还不得而知，我相信只要不违法什么反洗钱条例或者跟中共相关的勾结，应该都是可以轻松通过的。

老班长：联盟基金正在完善它最后几步的法律手续，很快就会跟大家见面了。针对不同的需求我们设立了不同的（方法）来为大家服务，所以联盟基金它会一步一步去提供服务的，一个项目一个项目的提供服务的。只要有需求，只要合法合规，都会给战友们考虑到的。现在因为还没到这一步，我还不能具体地去说，但大家都知道设立联盟基金就是为大家解决这些痛点难点的。

- 怎么解决？如果你属于这种情况的，就跟各农场卡项目工作组联系，把大家的信息全部汇总，再汇总到联盟工作组，我们会集中力量把这部分问题先解决。
