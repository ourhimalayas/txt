## 世界各国如何监管加密货币
` 西班牙巴塞罗那喜悦农场` [轉載自GNews](https://gnews.org/zh-hans/2200206/)

编译：七叶之芒

一些国家宣布比特币为官方法定货币，而其他国家则宣布直接禁止加密货币。下面的世界地图由Statista的Katharina Buchholz制作–基于美国国会法律图书馆收集的数据–显示了哪些国家一直在试图阻止加密货币的炒作，哪些地方的加密货币或多或少得到了自由支配。

![](https://assets.gnews.org/wp-content/uploads/2022/03/image-2282-edited.png)zerohedge.com

一个拥抱加密货币的国家的例子是萨尔瓦多，民粹主义总统纳伊布克尔在2021年9月宣布比特币为官方货币。该国还对加密货币进行征税和其他方面的监管。萨尔瓦多处于一个特殊的位置，因为它没有自己的货币，而是像该地区的其他一些国家一样，依赖美元。

其他正在应用法律来监管数字货币的国家可能不会像萨尔瓦多那样走得那么远。相反，这些地方–最典型的是发达国家–一直在投资项目，推出自己的中央银行数字货币。这可以说是一种与原始加密货币非常不同的使用区块链技术的方法，原始加密货币明确地独立于任何国家控制，但也会因此而非常不稳定。在探索这一概念的国家中，有美国、欧洲国家、俄罗斯和澳大利亚。印度和泰国也在对加密货币进行广泛监管，它们已经有了更具体的计划来发行自己的数字货币。

乌克兰也是一直在监管加密货币的国家之一，但该国在周三为该国的加密货币行业立法制定框架时更进一步。该国在被俄罗斯入侵后，收到了加密货币的捐款，因此采取了这一举措。在欧盟、英国、加拿大、美国、墨西哥、智利、日本和韩国等国，也存在必须注册或获得加密货币交易所许可证等规则。

中共国是第一个在2021年初开始在区块链上发行其国家货币的主要经济体。该国采取了更极端的方式来监管加密货币，对其发布了绝对禁令。根据美国国会法律图书馆的数据，到目前为止，有九个国家采取了这一措施，而更多的国家则通过其他法律暗中禁止使用加密货币。这种做法在非洲、中东和亚洲最为常见。
