![](https://assets.gnews.org/wp-content/uploads/2022/03/入境限制造成日语学校经营困难.png)

NHK4日消息，据一家日语教育机构的调查表明，受新型冠状病毒带来的限制留学生入境措施的影响，全国至少有80所日语学校陷入经营困难，甚至会在3个月内停止经营，关闭学校。

由“日语教育机构六团体”主导，为调查限制留学生入境带来的影响，对所属469所会员学校进行了问卷调查，320所学校做出了答复。

其中，占比7%的24所学校，因周转资金不足决定本月底停止经营。占比18%的57所学校，在不能接收留学生的情况下，陷入3个月内不得已将停止营业的状态。此外调查还发现，有122所学校将在六个月到一年之内将停业营业，占比高达38%。

该团体的一位负责人说：“想到日本留学的人数在减少，取消入学的人数在增加。希望政府能够同时考虑防止感染和留学生的入国问题。”

关于放宽边境措施，日本政府3日宣布，从本月14日起将每日入境人数上限的5000人提高至7000人。但是，等待入境的留学生有15万人，要实现15万人入境日本还需要花很长的时间。一位经营日语学校的校长说, “完全不能预测自己学校的留学生什么时候可以入境…。”

专家表示，“如果现在的情况继续下去，日本留学的根基会遭到破坏。”一桥大学专门研究国际学生政策的大田博教授说：“我认为这很严重。许多在大学和研究生院学习的留学生都是在日本语言教育机构学习了日语后继续深造的。目前的情况如持续下去，日本留学的根基将受到破坏。希望政府尽快让等待入境的留学生进入日本学习。”
