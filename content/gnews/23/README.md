图根哈特认为，在习近平主导下，中共比过去更加集权、自负，比以往更积极使用经济捆绑、胁迫手段来要挟其它国家。英国必须尽快调整对中国的立场，并重新评估上述互动及所谓伙伴关系，是否符合英国的长远利益。
