## 天上人间，你最珍贵

![](https://assets.gnews.org/wp-content/uploads/2021/12/sky-g86d09472e_1280.png)图源：网路
三国里说东风不与周郎便，铜雀春深锁二乔，指的是历史上著名的赤壁之战。赤壁之战周瑜兵力本处在弱势，只是刮来的强尽东风一下子让火攻之势越发猛烈，东风相助于周朗，铜雀不必深锁二乔。赤壁之战在三国时代是具有划时代意义的一场战争，每当在某些关键的节点，天气总是能够起到一些作用，好似冥冥之中的一股力量，不能触摸却为后世口口相传。

天公作美，百事兴矣。人间的一切喜事总是会求个好兆头、好彩头，天气，或者是天上的祝福就变得如此重要。天有不测风云，变换的云风闪电等自然现象成为人们解读上天的原始凭借。人有旦夕祸福，人的起起落落总也伴随着某些前兆。若问前世因，今生受者是，人间弘扬正道，惩恶扬善的基本逻辑也是因为因果报应。

看古代的神话，人们对天的理解比现代人要亲切。古代神话故事里，总是有下凡的仙女，或者仙子，来到人间，和人谈恋爱，结婚生子，甚至为了世间的凡人放弃天上的荣耀，承受上天的严重惩罚。董永和七仙女的故事，董永出身贫寒，为了埋葬已故亡父，卖身为奴，此举孝行感动了七仙女，七仙女下凡和董永私定终身并有孕在身，玉帝活活拆散这对夫妻，把七仙女强行召回天上。一年以后七仙女带着孩子又来看董永。牛郎织女的故事，织女是天上的仙女，跑到人间来和牛郎恋爱生子，同样也是触怒了天上，惩罚他们每年只能相见一次，喜鹊感动于他们的爱情，都在七月七飞来架起一座鹊桥，牛郎织女相见就被成为鹊桥相会。这是民间传说里的两个关于天上的仙女和人间的凡人相爱的故事，除了传说，历史上却也有过记载。周穆王是周朝天子，他经常去找他的爱人西王母，西王母是住在瑶池的神仙，这么看来就是周穆王就是人间之人身份与神仙相恋。

看了这么多故事，就算是古代人对美好爱情的歌颂，凭空瞎编这么多的故事是不可能的。好比共产党拍抗日剧就是拍不好，看起来很假，没有逻辑，因为共产党就没有真正参加过抗日战争，没有实际操作怎么会有文学创作呢？这些神话故事全部充满了神秘色彩，就算流传了几千年仍然为人津津乐道。很惊奇的一点就是这些神仙与凡人相恋的原因，都不是平白无故的，七仙女感动于董永的孝，织女枯燥与天上的生活，跑到人间来看繁花似锦，而西王母也垂涎的是周穆王的英俊与神勇。人间如此美妙，人的感情如此真实，人们常说感天动地，确实太有道理了，人的真善让天地为之动容，何况天上的神仙呢？人间烟火，不似平常，原来我们习以为常的一个小小善良，小小真实，竟然就是让人类连接天地之间交流的密码。

**人是这个人间最珍贵的东西，也可能是天上最珍贵的东西。**

天上的神仙有神仙的生活，地上的凡人有凡人的生活，看起来各自并不相干，甚至人和神仙发生一些接触，神仙会受到来自上天的惩罚，而人却并不会受到惩罚。传说里人会和神仙相恋，其实现实中也可能如此。 人具有和上天和土地交流的某些通道，也许这些东西本来就是上古时期的人类，或者说每个人的前世。

而为什么现在这些共产党和邪恶组织的魔鬼们要拼命去给人洗脑，不让人去独立思考。也许独立思考本身就是在和上天沟通，真正的信仰具有坚不可摧的定力，因为见过上天见过人间见过那些人类未知的潜力，而且不是一两次见，是每次都见过，那还有什么可动摇的。共产党和邪恶组织这些魔鬼，他们不可能拥有这些，他们妄图把人类的这些感觉器官全部关掉，制造空气污染，给人打疫苗控制基因锁，给人不断地洗脑，这些邪恶的源头就是嫉妒。

佛教里有一个道，叫阿修罗道，就是俗话说的魔道。传说里阿修罗的女人貌美如花，男人丑陋无比，但是男女嫉妒心都非常之重。如果无法理解的话，那看看吴征杨澜就很好理解了。嫉妒的下一步就是生恨，只要生恨，杀人放火，什么事儿都可以干得出来。单伟健也好，中南坑那些军官也好，习近平，王岐山，包括盗国贼的这些常委，他们的特点就是看不起任何人，不管你是谁，你是老百姓，你是川普，你是普京，表面上对你的趋炎附势只是一种收买人的策略。贪嗔痴慢疑充满，让他们的内心灌满了嫉妒，因为他们都是恶魔，他们无法拥有人类这样生自天成的灵长，得不到的就毁灭你，这几乎是嫉妒心最好的报复。人类的身体就是这样，进入灵魂的是贪嗔痴慢疑，那上天就会远离。

每个人不要小看自己，每个人都不要小看人类团结的力量，上天在我们头顶，在我们心里，在我们的无处不在，还有什么可退缩的。
