在利益面前，这些药厂这些个大资本们，简直就是沾满血的吸血鬼。吴征把这些医药公司引进了中国，中间拿了多少好处，又绑架了多少人来帮他做坏事？钥匙澜找马云一次钥匙高达34亿美金，这些药厂难道会给的比马云少吗？一找钥匙就拍小视频，一拍小视频这些药厂都得趴下，让干什么干什么。辉瑞总部很早就搬到上海去了，这次疫苗事件中，很多辉瑞的员工出来爆料的，说FDA来检查的时候，辉瑞拿黑布遮住了窗户，这完全就是中共做派，背地里拿钱拿小视频一勾兑，害的都是无辜老百姓。

说到这里，就真得气愤。中南坑里那些王八蛋，还有吴征杨澜这种骗子，大资本这种贪婪嗜血的性质，以及辉瑞这种大药厂，他们手上拿了青蒿素会怎么样。所以那个要跟七哥合作卖药卖2500-3000美金的，他可以说是一个例子，我拿着解药，我可以拿你的命来要挟你，无异于共产党给喂的人尸丸。而每个普通人，他是如何被要挟到的，仅仅因为他打了疫苗，被无知和侥幸以及强迫，施打了疫苗。

青蒿素这么好，中南坑里都吃了好几年了。习神到底吃没吃？习神还给屠呦呦老人家挂了奖牌的，不可能不知道啊，如果习神吃了，那怎么都做了四次手术了，对Biden的殷勤要求见面也避而不见，习神到底得的什么病啊？再说王岐山和其他中南坑里的老杂毛们，掉头发掉得太狠了，白头发也肯定比七哥多了。他们到底吃没吃青蒿素？对于共产党这个绞肉机体制来说，寿命长，应该就是惩罚。可以通过各种保健方式延长寿命，但是架不住内斗腥风血雨，说不定哪天就像金仁庆一样，直接家里火化。记得疫情一开始的时候，八宝山的火化炉就没停过，天天烧人，也许这些死人中也有很多吃青蒿素保健的人。

灵丹妙药是给好人用的。好人吃过药，很快就好了，很多人可以通过自己的经历告诉更多的人，让更多的人得救。这也是战友染病以后，吃过药马上就好了，文贵先生说，你的人生以后一定会变得不一样的。因为有亲身经历，在讲述的时候，更有说服力，还有什么比真相更能让人信服的呢？另外我相信小酒窝战友，在通过青蒿素医治癌症后，一定会好起来，到时候又会鼓舞不少得癌症的人。而坏人就算吃再多，中南坑的寿命又臭又长，他们身体好了，只会干坏事，把全人类都灭族他们也做得出来。所以青蒿素，只能给到好人手里。
