![](https://assets.gnews.org/wp-content/uploads/2022/04/56628319-10720167-image-a-4_1649968635709.jpg)4月14日，美国两党议员抵达台湾台北松山机场时与台湾外长吴钊燮合影。（路透社图片）

英国《每日邮报》4月14日报道，在美国两党参议员团体突然访问台湾后，中共狂怒。中共国外交部发言人赵立坚警告说，美国不坚持一个中国的原则，是在“走一条错误和危险的道路”。赵立坚称这次访问“居高临下”且“不负责任”。

参议员林赛·格雷厄姆（南卡罗莱纳州共和党）、外交关系委员会主席鲍勃·梅嫩德斯（新泽西州民主党）、参议员本·萨斯（内布拉斯加州共和党）、参议员罗伯·波特曼（俄亥俄州共和党）和众议员罗尼·杰克逊（德克萨斯州共和党）都参加了这次访问。他们乘坐的是美国空军的飞机，这意味着拜登政府批准了本次访问。

议员们在台北机场会见了台湾外交部长吴钊燮。此前，他们前往澳大利亚讨论增加美国在那里的驻军以抵御中共国的挑衅。

预计代表团预计将在周五晚间离开前与台湾总统蔡英文和国防部长邱国正会面。台湾总统府发言人张惇涵将这次访问称为美国和台湾之间 “坚如磐石”的关系的证据。

众议院议长南希·佩洛西上周一直打算访问台湾，而中共国外交部长王毅称，议长计划中的访问影响是美中关系的“红线”。

中共国将台湾视为其领土的延伸；台湾认为自己是一个独立的国家。美国几十年来一直实行战略模糊政策，不正式承认台湾的自治权，但警告中共国不要入侵该岛国领土。

尽管有时提供武器援助台湾，并且美国官员偶尔也会访问台湾以示支持，但美国对于如果中共国真的试图入侵，它是否会来支持台湾这个问题依然有意模糊化。

近几个月来，中共国对台湾的挑衅已经升级——中国人民解放军几乎每天都会派出战斗机飞向台湾岛。就在本周，台湾军方出版了一本官方手册，建议平民如何为中共可能发动的入侵做准备，包括在哪里找到防空洞，以及如何储存应急物资。

台湾在对抗中共国的武器装备上仍然远远落后，但这个多山的岛屿对任何军队来说都是一个巨大的挑战，难以征服。

上个月，美国海军上将、印太司令部司令约翰·阿基里诺(John Aquilino)说，中共国在过去一年中表现得“很放肆”——从其对台湾日益强硬的立场到去年7月的高超音速武器试验。他说，俄罗斯对乌克兰的入侵，应该让世界保持警惕。
