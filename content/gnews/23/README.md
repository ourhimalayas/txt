![](https://assets.gnews.org/wp-content/uploads/2021/12/03-image-7.jpg)
据加拿大[CTV新闻]12月25日报道，随着“奥密克戎”变体病毒的全面迅速传播，安大略省卫生部当天报告了首次超过 10,000 例的新增病毒感染人数最高记录。

报道指出，由于“奥密克戎”疫情的持续恶化，安省政府早些时候已采取措施以限制假期聚会的规模，并因此削减了大多数室内场所的人员容量限制。同时，针对不断暴增的感染者，该省卫生官员警告称，由于病毒检测需求急剧增加，最新感染人数可能大大高于真实病例数。

尽管政府数据显示，大部分感染者都已接种了疫苗，但当局仍继续强调注射加强剂的必要性，并以此作为防止感染该病毒变体的一种保护方式。为此，大多伦多地区的许多诊所在圣诞节当天都仍在进行接种活动
