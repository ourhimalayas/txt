## 【加拿大广播公司】几乎所有员工均已接种“新冠病毒”疫苗

据加拿大自媒体《真北方》10月26日报道，目前，在【加拿大广播公司】的7429名雇员工中，仅有1%已向当局报告了其“新冠病毒疫苗”接种情况的人尚未完全接种该疫苗。

此前，在总理特鲁多宣布对联邦公务员及联邦管理行业的雇员执行该疫苗接种的任务后，【加拿大广播公司】所有工作人员都被要求必须在10月底前向该公司申报其疫苗接种情况。据悉，目前已有75%的员工报告了他们的接种状况，而其中仅1%的人没有完全接种该疫苗。

报道还称，【加拿大广播公司】的“强制性疫苗接种”规定将于2021年12月1日生效。此外，该公司目前仍在试图解决应如何完成此项强制任务的具体问题，包括如何处置那些拒绝接种该疫苗的员工。
