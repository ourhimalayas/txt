据观察者网报道：1月25日，俄罗斯驻华大使杰尼索夫（Andrey Denisov）表示，中俄已形成“能源联盟”。

据俄罗斯卫星社1月25日，俄罗斯天然气工业公司（Gazprom）24日发声明称，1月18日在与中国石油天然气集团公司的合同框架下，创下向中国供应天然气的单日新高。

上周，俄气表示，1月上半月通过“西伯利亚力量”输气管道向中国出口的天然气是去年同期的1.5倍。

2014年，Gazprom与中石油签订了为期30年的天然气合同，通过“西伯利亚力量”输气管道，每年向中国供应380亿立方米的天然气，合同总额4000亿美元。

卫星社介绍，2020年，俄罗斯天然气工业公司向中国供应了41亿立方米天然气。2021年前10个月供应量为80亿立方米。

杰尼索夫称，预计中国对俄罗斯天然气的高需求将持续，双方正在积极讨论天然气供应领域的新项目。
