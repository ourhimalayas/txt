## 香港第5轮抗疫基金仅40亿元
` 東京櫻花団-Tokyo Sakura Group` [轉載自GNews](https://gnews.org/zh-hans/1859231/)

作者：东京樱花团｜阿恩
![](https://assets.gnews.org/wp-content/uploads/2022/01/w1200.jpg)
港府宣布今日(14日)再推防疫抗疫基金5.0纾困，但只使用基金40多亿港元结余。据悉，港府同时公布再延长收紧措施14天，即受影响的行业农历新年继续停业。资料显示，第1轮防疫抗疫基金共涉300亿港元；第2轮资助金额为1,375亿港元；第3轮资助金额为240亿，第4轮资助金额为64亿港元。

香港受中共病毒影响已经两年，疫情依旧没有停止的迹象，港府不断出台防疫收紧政策自毁根基，一次次重创民生行业和失信于民。如今第五波疫情开始了，港府又重蹈覆辙，令经济雪上加霜，而且港府的防疫抗疫基金已经见底，仅剩40多亿元。据悉，港府并不打算增加防疫基金，只会把这剩下的40多亿分了完事。立法会的议员们在港府公布防疫基金报告前夕，分别代表自己的界别向港府索要行业资助，有的要求帮补受影响企业的开支，有的要求现金补助相关的员工，有的要求港府提供就业培训……这些没有任何力量的台面功夫，只不过是他们应付市民的手段罢了。

港府设立的这笔防疫基金加起来也不超过2000万元，相比配合中共大湾区发展的北部都会，港府预计投入资金达数千万亿元。中共眼中的人民都是可以吃三年草的奴隶，破了香港人高收入的神话，就能让港人向港府摇尾祈食，奴化港人就水到渠成了，之后再也不会有什么行业补贴、工资补贴，中共政权家族将会霸占香港所有的社会财富。
