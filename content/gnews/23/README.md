## 中日财经爆|雷诺汽车无限期暂停在俄罗斯的生产

撰稿：东京樱花团/peterwong

![](https://assets.gnews.org/wp-content/uploads/2022/03/1-266.jpg)[图片来源](https://gzrydn.com/carcollection/104473.html)

法国雷诺汽车3月23日宣布，将无限期暂停在俄罗斯的生产活动。乌克兰总统泽连斯基曾呼吁撤出俄罗斯业务，董事会当天决定关闭莫斯科工厂。

“我们将考虑当前环境并考虑可能的选择，”雷诺在一份关于俄罗斯汽车巨头 AvtoVAZ 的声明中说。

雷诺在俄罗斯汽车市场占有约30%的份额，是仅次于其总部所在地法国的第二大销售市场。在俄罗斯拥有约4万名员工。

雷诺在苏联时期就持有汽车制造商AvtoVAZ 68%的股权，且其约10%的营收来自俄罗斯，因此，雷诺在汽车同行中受到的影响最大。由于莫斯科工厂停工，2022年营业利润率前景已从之前的4%以上下调至3%左右。

自俄罗斯入侵乌克兰以来，由于西方国家对俄罗斯实施更严厉的制裁，以及公众舆论呼吁，外国公司在俄罗斯继续运营变得更加困难。
