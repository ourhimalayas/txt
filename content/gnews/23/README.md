![](https://assets.gnews.org/wp-content/uploads/2021/11/111301-1.jpg)
就像加密货币中的很多东西一样，喜币有很多东西可以解读，它在不到两周的时间里悄然升至270亿美元的估值。

首先，有音乐视频“Hcoin to the Moon”，突出流亡的中国亿万富翁郭文贵先生抽着雪茄，在游艇上闲逛，开着一辆红色跑车的镜头。

这首歌的歌词，大部分是中文唱的，描绘了一个暗淡的世界：“自由，多么熟悉的字眼/却不知不觉离我越来越远。”其他说唱部分列出了喜币和相关的喜美元的优点：是一种绑定美元的稳定币。

由最先进的加密技术保证其安全

你的财富没人能夺走

稳定币，浮动币，设计天下无双

还有20% 黄金挂钩

摧毁不停压榨你的机器

从此不再做金钱的奴隶

喜联储的大门 永远向你敞开

前途光明 就在喜币

一遍遍合唱地喊着：“欢迎来到世界/来到新世界/欢迎来到世界/喜马拉雅币登月。”

“到月球”的说法之所以引人注目，因为这已经成为一个集结号，是加密货币交易者的指导性希望，即持有的价格将巨大地飙升。

但是，来自郭文贵先生和前总统川普的顾问史蒂夫·班农的支持表明，喜币的工作不仅仅是加密货币的欢呼。

多年来，郭文贵先生和班农一直在公开反对中共政府，班农上周发表的评论表明，这是这场战斗的最新前沿。喜币本月的首次亮相也很引人注目，因为9月份，中共国加大了对加密货币使用的打击力度。

据喜马拉雅交易所的首席执行官称，郭文贵先生已经在社交媒体上赞扬了该数字货币及与该数字货币交易的交易所有业务关系。班农在上周的一次采访中称喜币的首次亮相是“丰碑式的”，并将其置于与中共国抗争的背景之下。“如果你发现这是从制度上对于中国共产党的反击，这相当的令人印象深刻”班农说。

喜币似乎只在喜马拉雅交易所交易。它没有被包含在加密资产价格跟踪网站 （CoinMarketCap）的全球加密货币长名单中，该名单已成为数字货币估值的事实上的来源。在喜交所，其价格已从月初开始交易时的10美分跃升至现在的约27美元。

喜马拉雅交易所首席执行官杰西·布朗(Jesse Brown)表示，已经发行了10亿枚喜币。市值大约270亿美元，类似于CoinMarketCap网站中的柴犬币的市值，这种数字币最近的受欢迎程度猛增。

他补充说，美国、加拿大和日本的人们不能使用该交易所，尽管有计划让美国进入。

郭文贵先生正在“利用他的平台让人们参与加密货币和获得财务自由”布朗说。“我认为这与他发起的爆料革命以及他努力让中国人获得更多的独立和自由有关。”
