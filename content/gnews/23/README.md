![](https://assets.gnews.org/wp-content/uploads/2022/02/1-97.png)[图片来源](https://newsin.us/%E4%B8%AD%E5%9B%BD%E5%9B%BD%E8%88%AA%E5%AE%A2%E6%9C%BA%E4%B8%8A%E6%9C%89%E4%B9%98%E5%AE%A2%E8%87%AA%E6%9D%80%E8%BA%AB%E4%BA%A1/)
美国交通部周三（2月9日）表示，中国国际航空公司同意就2018年在美国机场停机坪严重延误支付14.3万美元的罚款。

2018年国航有两架国际航班在纽约肯尼迪国际机场的停机坪上停留五个多小时，没有为乘客提供下飞机的机会。按照美国规定，如果国际航班在停机坪上延误超过4小时，航空公司应允许乘客下机。

该部称根据30万美元的民事处罚令，国航必须在30天内支付14.3万美元，如果它在一年内再次违反该规定，则必须再支付14.3万美元。国航可以免缴已向乘客赔偿的1.4万美元。

美国交通部并称，中国国航不同意其计算民事处罚的方式，但表示为了避免诉讼，已同意这项和解。
