【张陶打院士：你能打得，我们为何议论不得】 航天投资控股有限公司党委书记、董事长张陶涉嫌殴打国际宇航科学院院士王晋年和吴美蓉的消息在网上热传。王晋年和吴美蓉被打导致多处骨折，“伤得挺严重”，目前仍在住院。

打人原因目前网上说法较多，但是打人确实已是事实。张陶所在航天投资控股有限公司暂未公开回应此事，有记者采访该公司称张陶还在上班。

航天投资控股有限公司的上级是中国航天科技集团有限公司，该公司常群工作部发布通知称“正在全面了解事件真实情况”。但是，该公司常群工作部有几句话确实不舒服，“希望广大党员干部不要对此次事件进行散布和传播，严守政治纪律和政治规矩，着力营造风清气正的舆论环境”。

我们就不懂了，对于这样暴力殴打老年人特别是老年学者的行为，应是天下人皆可诛之。但是法治社会，我们也不能随便诛阿猫诛阿狗的，但是我们应该对此行为进行至少道德上的谴责和唾弃。而在这个时候，这个常群工作部根本就不顾及群众的想法，竟然发文不让议论和传播。中国航天科技在人们心目中应该是高高在天上的工作，对于其道德和素质的要求应该要比地面上的人们高的多。而这种动辄就用“政治规矩”和“风清气正”的帽子扣押同事，可见其确实不懂规矩 ，也是对“政治规矩”的歪读。
