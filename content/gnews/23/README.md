中共国官方公布，截至去年底人口出生1062万人，较2020年进一步下降，去年人口仅增48万，创43年来新低。

出生人口陡降的原因，首先是育龄妇女相比2020年减少了473万。其次是，90后00后是新的婚育主体，面对就业压力，婚育推迟十分突出。第三生育成本不断增高，也让许多年轻人在生育上犹豫不决。第四就是当前疫情影响，很多人不愿意孩子在这个时期出生。

同时，年轻人不愿结婚，不愿生育，其实就是对这个国家的前景不看好，没有信心，对自己未来的生活保障问题不确定，可见中共国正面临国内民众的极大不信任和对外国逐渐失去吸引力的双重危机！
