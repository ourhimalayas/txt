**班農先生：**首先我想代表班農家族和我父親對郭先生、新中國聯邦人、G-News、GTV說，我們非常感謝！

太暖心了。 記住，我爸爸只是一個美國老百姓，他是個小人物。 他是一個電話公司的藍領工頭，因為我們擁有自由，讓他能在美國支持我們一個有5個孩子的家庭，媽媽是全職家庭主婦。 他供養我們上天主教私立學校，我們5個孩子全部大學畢業。 所以他為我們犧牲了一切。

新中國聯邦的每個人傾注大量的愛和感情（舉辦這個活動），實在是難以置信的令人難忘。 我們有一個大家庭的聚會。 作為100歲的老人，我爸爸仍然很有精神。 但是那一整天他累壞了。

真的難以置信。 我只想感謝每一個人的付出，一切都是難以置信且鼓舞人心。 向人們展示了中國老百姓的善良，正如美國人的善良。 某種意義上是世界各地的人聚集在一起，所以我認為這非常特別。 我想感謝每一個人，尤其是文貴，他非常慷慨。
