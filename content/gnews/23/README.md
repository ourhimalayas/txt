## G时评：看破红尘、思维定势与三性

【看破红尘】

当我们说起“看破红尘”时，往往隐含失望、消极、厌世之意。这世间的名利、欲望令人厌烦，我们想逃离世俗的羁绊，寻一处安宁之地度过余生，遁入空门、隐居避世，放下对金钱、名利、美色等欲望的执着，不再纠缠现实俗物，保持内心的安宁。

“到了次日，并不约我，却一人独往，岂非看破红尘，顿开名缰利索么？”

——《镜花缘》第四十回

这里面的“看破红尘”，说的就是这么回事。

红尘，代表我们肉身所处的物质世界。

《红楼梦》第一回：“原来是无才补天，幻形入世，被那茫茫大士、渺渺真人携入红尘、引登彼岸的一块顽石。”

佛教中的“看破红尘”，原本是希望我们突破表象思维，透过世间万物的表象，明心见性，找到事物的本源与规律。

遁入空门固然可以减少名利诱惑，但并不意味着就能达到明心见性的境界。

摆脱了物质、名利的羁绊，找到了人生的意义，了解了世间的游戏规则，就是看破了红尘。

【思维定势】

思维定势，是统治奴役的前提，是顺从的镣铐。

很多时候，局限我们的，不是肉体，是思维。

决定我们行动的，是潜意识。

潜意识的存在，让我们的行为，在主观意识中被合理化，画地为牢而不自知……

【宗教】

宗教，某种意义上是一种自我约束，比如：我们相信，举头三尺有神明，在这种约束之下，我们慎独、有底线，相信轮回、报应。

然而，目前大多数宗教，事实上已经沦落成为少数人的洗脑工具……被曲解、被误导、被利用。

所以我们要回归人性，以人为本，走正道主义，找到真正的信仰，找回我们作为造物主的杰作、作为人类本身，逐渐失去的、人性最美好的特质：

爱、宽容、理解、真诚、善良、勇敢、正义、坚持、智慧、逻辑……

【三性】

在目前人类世界里，新材料石墨烯已经达到了一个原子的厚度；纳米技术突破了2纳米；社交媒体实现了个人推送……

一切都在往微观、粒子方向发展……

作为个体，你准备好了吗？

还在执迷于金钱、权势与物质？

局限于国家、民族、宗教、种族的宏观世界？

回归人性：

想想你为什么来到这里，想想造物主与你的契约，想想什么是人、什么是人生、什么是人生目的。

回归个性：

个性推动创新与发展，个性使你与众不同，尊重自己的兴趣与爱好，顺应自己的天性，不要为了从众，轻易扼杀造物主赋予我们每一个人独特的、宝贵的天赋。

明心见性：

灵魂不死不灭，身体只是灵魂的容器；地球只是灵魂的试炼场，我们的“敌人”是肉身在物质世界里产生的“贪、嗔、痴、慢、疑”，不要在“打怪通关”的过程中被“怪兽”同化，不要迷失自己……
