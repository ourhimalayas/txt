《零对冲》发表文章，谈到中共国镍业巨头青山控股集团是世界上最大的金属生产商，由项光达控制，在俄乌战争引发了镍价的空前上涨后，面临数十亿美元的交易损失，镍价在两天内飙升了250%的记录。

周一的纸面损失为80亿美元–这是LME持有的约10万吨镍的结果–在镍价猛烈冲高导致伦敦金属交易所周二暂停该金属的交易之前，镍价从1月份的每公吨20,175美元上升到了101,365美元的记录。此后，该交易所表示，它预计在周五之前不会恢复交易。

据《华尔街日报》报道，青山的创始人项光达说该公司正在与有关方面积极谈判，但没有具体说明他们是谁以及谈判的内容。

报道还引用了项光达的话说，”相关政府部门和领导都非常支持青山公司。根据财经新闻媒体第一财经的报道，”青山是一个坚实的中国企业，我们的立场和运作没有问题。

不用说，LME (伦敦金属交易所) 现在由香港交易所拥有，而香港交易所的最大股东是香港政府，在过去两年里一直是中共国的傀儡，如果LME强迫他支付保证金，他的金融帝国就会破产了。除了即将离任的LME伦敦金属交易所首席执行官马特·张伯伦（Matt Chamberlain）外，其他人都承认了这一点。
