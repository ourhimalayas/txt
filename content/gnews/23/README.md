## 普京说俄罗斯国防能力超越世界上多国包括美国
` 索尼克` [轉載自GNews](https://gnews.org/zh-hans/1361951/)

作者：yyshere
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/30090932/h_56974522-1.jpg)
根据俄罗斯政府网站公开的信息，俄罗斯总统普京当地时间6月30号在莫斯科一个电视访谈节目中表示，他相信俄罗斯的国防能力是 “超越世界上许多国家 “的，包括美国。

普京面对镜头表示，”你们应该从对世界迅速变化的理解出发。无论对俄罗斯实施了何种制裁，无论存在着何种威胁，俄罗斯都在发展。它的经济主权增强了。它的防御能力达到了一个非常高的水平。在许多方面，它超越了世界上的许多国家，包括美国。”

普京还指出，美国依然保持它世界老大的地位，但也意识到世界已根本改变了，”世界从根本上发生了变化。一方面，我们的美国伙伴明白这一点，因此，我们在日内瓦的（举行了）会晤。另一方面，他们仍试图保持其垄断地位，不惜一切代价，这里有威胁，有进一步的破坏性行为，包括演习、挑衅和制裁。”

熟悉俄罗斯政治的评论人士指出，俄罗斯一贯在外交上保持强硬的态度，尤其是在于美国打交道的时候。但俄罗斯这个民族和它的政权管理者，无论在怎样的政体下，一直以来都是遵循实际利益高于一切的原则，无论是在历史上的历次战争中，或是在前苏联解体的过程中，以及之后在与美国、与中共，和与欧洲的关系中，始终如一。俄罗斯政客，包括普京在内，屡屡在国际事务中发狠话，但这不过是一种“语言习惯“，这并不妨碍在接下来的谈判或者勾兑中与对手达成协议。唯一的要求就是俄罗斯必须中获得真真实实的利益。而普京在于拜登会晤前，曾公开表示，俄美之间是存在有共同利益的地方的。因此不需要过度解读普京这个电视讲话的表态。

同时这位分析人士指出，普京在28号应中共要求，在中共百年诞辰到来前与中共国家主席习近平视频会晤时，明确指出他是以个人名义向习近平表示祝贺，虽然中共官媒将其翻译为“亲自向您表示祝贺”。这意味着，普京依然是之前他自称的那只“聪明的猴子”，在中美之间，他只会站在最后的胜利者一边，就目前中共恍如惊弓之鸟的虚弱状况，美国及其盟友才会是那个笑道最后的胜利者。
