## 从文贵先生的待人接物说起

从文贵先生的直播中我们听到的关于待人接物方面的一些细节，比如，七嫂给保镖吃的饭，和给七哥郭美等家人吃的饭都是一样的；七哥当年在盘古，就连和员工喝的酒都是一样的，这个很难啊，因为好酒真得太贵了。很多次直播中，我们都看到七哥非常会照顾人，纳瓦罗先生，班农先生，Jason先生当时一起直播的时候，七哥给大家拿水。和Rachel战友一起直播的时候，拿水，拿咖啡，非常贴心。记得七哥自己还有说过，如果看到大家在一起吃饭，他就走开了，因为他一走过去，让人家吃得不自然了。他这些方面好像个女性啊，细致而温柔。

七哥这些待人接物方面的细节，让人顿生好感。但是七哥还说，以前在一起吃饭，看到个人不喜欢，马上会说，我不喜欢你，你快走，你不走我走。为此得罪了不少人。还说起一个非常漂亮的女士，说以后再也不想看见你，再艰难也放开了。与人交往在一些方面讲究的是舒服相待，普通人拥有这样性格的一半都会非常受欢迎，起码面相上不会看着那么拧巴。爆料革命一开始的时候，九指妖那个人，其实一开始七哥不太喜欢她的卫生习惯，邋邋遢遢，和班农先生一起吃饭也不能说一句英语进行交流的，按照以往文贵先生的交友准则，这个人应该一开始就被否定的。但是因为在爆料革命一开始，九指妖是做出了很多贡献的，文贵先生的准则就是你只要不砸郭不砸爆料革命，你就是好战友，我们永远不会先动手。路大脑骗，蛇妖更是，郭先生从来不和蛇妖单独通话，每次通话都是路大脑骗三个人一起，更别说单独见面了，不可能的。按理说，蛇妖一来美国，就用文贵先生给她的卡，去奢侈品店买了那么贵的好多衣服，这个行为已经很掉价了，就更别说随便去敲别人的门了。而路大脑骗，动不动就要咳一口痰出来，恶心死了。这两个人的这些行为，相信都是七哥在以前那种让他走，他就必须得走的人，哪还会让他们留那么久？但也是因为爆料革命灭共事业，他们一开始也有贡献，正如对待九指妖一样，让他们自爆。

因为文贵先生的爆料革命，后来又成立了新中国联邦，G系列，包括现在喜联储，灭共事业变成了这个世界目前的主流。一路走来，从九指妖，路大脑骗，蛇妖等事件，无数的战友被出卖，只能让坚定灭共的更加坚定，不坚定的早就被排除出去了。因为这几个如此肮脏的人，导致爆料革命内部总是在不断地抓特务，文贵先生很多次都告诉战友，不要抓特务，用爆料革命的力量，用真相的力量影响他们，把这些特务变成咱的坚定战友，那才是厉害呢。就是的，这个世界本来就是好人多，坏人就算有那个坏心眼，但不要做坏事，不要害人，就在心里想想，过个意淫的瘾也就行了，但凡做出害人举动一定会承担责任的。这在任何国家都是一定的，害了人一定要付出代价的。为什么不要刻意去抓特务，而且一定要坚持唯真不破，特务也有特务的难处，他能枪口抬高一寸，在重要的事情上选择保护战友，那么这个人就应该是被感谢的。七哥直播中说到七嫂和郭美当年被抓，那些警察都很客气，都把七嫂当神看，没有做出对七嫂郭美的过分举动，就凭这一点，他们得到了七哥的感谢，七哥给他们留了喜币。七哥也说过很多墙内体制内的帮过战友的人，未来都会得到新中国联邦的感谢。新中国联邦就是，向善，是有回报的，巨大的回报。
![](https://assets.gnews.org/wp-content/uploads/2021/11/截屏2021-11-25-上午8.55.23.png)喜币上市当天的金枪鱼 / 图源：网路
喜联储上市那天，切了一条金枪鱼给战友，但是很多战友都只吃了几块，剩下那些都被办公室的其他人偷走了，就这么小的事情，文贵先生都记在心里的。并不是可惜那个鱼，只是因为战友没吃着几块，让其他人给吃了，就很不爽。一条鱼没吃到战友肚子里，七哥都能如此惦记，那你说那些卖了喜币套现战友钱的人呢？喜币这两天遇上了大额套现者。一个水池，有进水管，有出水管，两相保持平衡，才是健康的市场。现在情况就是入口的那个水还堵在KYC的路上，出水的这个口把里面的水都拿走了。喜币的大额机构投资者还没有入场，现在账面上的钱，大部分都是战友的，且都是一开始很低价格给的福利币。所以这些大额套现喜币的人，他拿走的那个钱，都是战友的钱。骗钱不要紧，你骗走的是爆料革命拿命出来支持灭共的真战友的钱，拿文贵先生的话说，你就是跑到天涯海角也不会和你拉倒的，这事儿没完。

七哥是一个照顾别人体贴入微的人，他又怎么能不知道这些套现喜币的人是来干什么的？这些人你伤害的是战友的利益，你获得的东西会把你害死。就像当初那个拿战友的钱去投资房地产的九指妖陈其生伪类一样，等待他们的是无限的司法诉讼，爆料革命一定会把他送进监狱。这几个套现战友钱的人，喜联储拥有全世界最严格的KYC，在这里上了黑名单，你觉得你还能在其他地方活下去吗？况且这中间面临的法律诉讼无关乎钱的多少，真战友的利益，一分一厘也是钱。况且文贵先生带领的爆料革命新中国联邦，一旦认定的真战友，尤其那部分获得福利喜币的都是坚定跟随的真战友，文贵先生吃啥保镖也吃啥，喝酒都喝一样，更何况是真战友的切身利益受到了损害呢？咱走走看，帮助过真战友的人，我们会感谢，伤害过真战友的人，呵呵，等待你们会是漫长的牢狱。
