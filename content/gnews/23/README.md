看中国2021年12月9日讯，近日大陆流传多省公务员正在悄悄降薪，降薪幅度高达20%至30%，影响范围包括浙江、广东、福建、上海等地，消息引发外界关注。

浙江一名自称女公务员的网友12月初在微博披露，杭州财政部门此前在未公布具体原因的情况下，宣布裁减公务员绩效福利，她的薪水骤减25%，约5万元人民币。

这种情况不只局限在浙江，还有大陆网友表示，上海派出所所长已由年薪35万元人民币减至20万元，科级公务员年薪由24万，减至15万元。江苏、浙江、广东、福建以及上海等地都可能会收到这波减薪冲击。

据“香港01”报导，广东某些非珠江三角地区的公务员已经开始换发补贴，只保障基本工资的发放。公务员的补贴被延迟2个月，教师则被延迟1个月。报导还提及，“深圳是全国财政最充足的城市之一，可想而知其他城市的补贴基本都很难兑现”。

有网友分析中国大陆疫情下经济成长缓慢，地方政府为开源节流，才降低公务员福利，估计不少省市随后也会跟进。也有人认为，公务员降薪与推动“共同富裕”政策有关。

据中国财经专栏“格隆汇”今年8月发布官方数据，中国31省市上半年财政收支，仅有上海市是“财政盈余”，其余30个省市均存在收不抵支问题。压力最大的河南、四川、云南等省，收支缺口都超过2500亿元。

中国财政部11月23日公布的中国地方政府债券发行数据显示，截至2021年10月底，全国地方政府债务余额约296549亿元。2021年1至10月，全国发行地方政府债券6.5万亿元。去年5月，中国总理李克强在《政府工作报告》中提出，各级政府必须真正过紧日子，中央政府要带头，中央本级支出安排负增长，其中非急需非刚性支出压减50%以上。不过讽刺的是，如今各地公务员和教师带头“过紧日子”，但却未见中央政府官员带头“过紧日子”。
