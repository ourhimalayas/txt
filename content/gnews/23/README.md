欧美强制新冠疫苗政策的风向转变，各国政府有意甩锅制药厂和研究机构，所谓的严肃媒体依然一本正经地胡说八道，三方都是说谎者，对渐次变得严重的新冠疫苗灾难，有不可推卸的责任。要不是长期跟随爆料革命，有多少人能明辨是非曲直，让自己置身事外免受伤害呢？辗转流离躲过纳粹杀戮而建国的以色列，却成了重灾区。看来有钱并不一定有智慧，更不一定有真相和希望。只是西方文化里有很多妥协成分，勾兑是必然的，特别是灾难严重的时候，民众压抑的怒火无处释放，在谁都不想担责的情况下，各国政府必将矛头指向中共，中共也会拉海外华人下水，帮自己挡子弹，所以郭先生提前警告，必须审慎对待可能出现的大规模排华事件，并一直努力分开中共和中国人。其实行动早就开始，郭先生计划立足英国和日本，与沼泽地对薄公堂的法律战，主要是斩断中共的海外黑金，也有主动出击，扩大新中国联邦的影响力，至少让文明社会最大程度避免排华的用意。

但是有些国家或者地区，郭先生也表示前景令人担忧。不难预见，那些还在帮习太阳抬冬奥这顶破轿子的国家，可能是最先排华的。物以类聚、人以群分，喜欢和独裁者走在一起的，必定不是什么好鸟。沙特王子托病不来，一方面是利益得不到满足，另一方面已经被爆料革命所触动。伊朗、卡塔尔和土耳其首脑不出席冬奥，不得不说是对中共底细的了解造成的。不管他们信什么教义，都忌惮中共没有任何信仰，做事毫无底线。郭先生说，兔肉汉堡包实际上是老鼠肉，既然这都能骗，那还有啥不能骗的。那些还在吃老鼠肉汉堡的外国人这下心慌慌了，进退已经身不由己，就像《封神榜》里的周文王，吃了自己儿子的肉，还不能让别人看出来自己知道，只能没人的时候再吐出来。开个冬奥，什么都是假的，雪道是俄罗斯的造雪机造出来的，金哨子是铜的，一百多道菜变成了盒饭，观众是军人装扮的，想想确实挺吓人，特别是当你已经发现周围的一切都是骗局，却还得附和称赞的时候。

普京灰头土脸地打道回府，至少证明他还知道黄金和人民币哪个是真、哪个是假，没有兴趣在习太阳假大空的戏里跑龙套。回去之后又和马克龙黏糊上，一番眉来眼去叙旧情，看起来能抗二百斤的假太阳还不如性取向不甚明确的小马哥。失去了中共的经济支持，普京多少有点尴尬，打乌克兰显得心有也人力不足，不打乌克兰，面子和权力可能都要失去，只能硬着头皮，继续周旋在欧美之间，东欧西扩无非是想卖天然气，欧元总比人民币好些，双头鹰计划看起来要吹。不知道习太阳的脸和心，哪个更疼。当然正如郭先生所言，忽悠习太阳打台湾，无疑是很好的选择。俄罗斯不插手，把习太阳卖了换回和欧美的互信，可以一石二鸟，玩三角关系。这就是政治游戏，想在里面找到正义和真相，几乎没什么可能。只有爆料革命说了政客们不能也不敢说的事情，并逐一得到验证。

然而在中共国，政治不但是游戏，而且是演戏，还不让人说破，冬奥无论如何狼狈到不能收场，都会走向辉煌“胜利”。这就不难理解为什么裕达夜总会的妈咪，会让小姐在里面找“真爱”了。这样才能走出中原，到上海滩发展壮大。有什么样的土壤，就能长出什么样的草木，这是千真万确的。小姐并没什么错，你情我愿的交易，至少还是公平的。没有小姐们的辛勤劳动，哪有扫黄的丰硕成果。中共只有本事对自己老百姓放水养鱼，定期捕捞，美国却在更高维度，用撤馆为由，来来回回吸血，就像一个经验丰富的小姐，只是玩着穿脱衣服的游戏，就赚了大把嫖资。中共既不如小姐也不如嫖客，对内是吃白食的恶棍，对外已沦落到见谁都自卑、倒贴没人要的地步。中共不但能在明星里找小姐，而且可以把演艺圈培养成夜总会，会讲多种版本的“爱情”故事。更能在人贩子手里找出被铁链锁住的婚姻，编出每天不重样的剧情。这就是中共，不断制造惨绝人寰的事实，然后百般抵赖尽显无耻。

也许一千个人会遇到一千种不同的爱情，每个家庭又都有冷暖自知的婚姻，正如对自由的理解不尽相同一样，但一切正常生活的基础都已经被中共破坏了。既要躲避随时到来的疫情，又要提防新冠毒疫苗的陷阱，面对无可挽回的经济下行，如何走出失业和房贷的困境？答案在新中国联邦和爆料革命，在未来的躺平币和已经登场的HCOIN。
