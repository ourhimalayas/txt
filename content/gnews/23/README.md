79年、80年就是唐山地震后，你别说全国转一圈儿，你到当地的镇上、县城转一圈儿，就跟今天你出个国差不多，真的是。然后呢等到了82年、83年的时候，你能去一趟北京就跟今天往太空去一圈儿差不多！啊。我家是没吃没喝情况下我从小就那么折腾，我就能免费获得照片，那时候人家是骑着嘉陵摩托车到那块儿去照相，拿着120海鸥，照相的时候多少钱。你那时候到那个食堂，78年、79年到食堂去，要拿着粮票、拿着食堂票，发糕、米饭、馒头、还有花卷儿，还有分粗粮和细粮。有的地方大米是细粮，有的地方大米是粗粮。那打饭都是饭盒，到哪儿都带饭盒呀，那是开啥玩笑呢那是？那都是最高级人吃的饭，农民哪有饭盒呀？农民拿手抓的都没有，有饭吃就不错了。
哎，那时候七哥能带着我的兄弟姐妹冲到食堂去偷它的饭票、发糕票，咱就能解决兄弟姐妹们一顿大餐。那当时相当于今天咱们战友们拥有了喜币一样的感觉，一点儿不夸张。就是你进了食堂了，地上竟然拿水能洗地，把地洗干净，我那时很纳闷，怎么这地上我们冲进去以后也没人了，大家都跑了，那地上弄得湿乎乎，是洗地、水泥地，我们都在土地上哪有什么水洗地呀，高级！发糕拿着还盖着，我们那时候使劲儿吃，吃了怀里面装一堆，是吧，他们吃，我就拿发糕票，拿着那粮票，还有那个兑换的钱票，啊，我就是抓一大堆，结果我们吃了很长时间，他们傻乎乎光在那儿抢发糕了、抢花卷儿了，我拿这个。当时我记得很清楚，它那个是烤的那个猪油，这么一个罐子里边儿油脂喇儿，我们进去的小孩儿手伸进去以后吃脂喇儿往嘴里面捂着吃的。吃完以后那旁边的缸，它是压水的呀，那个还不是什么电动水，它还压水的呢，我们就爬在缸上就喝，就差点儿处缸里边儿去。Eric你别说你全国转一圈儿，那你是多牛啊！七哥我去了一趟深圳，去了一趟广州，啊，是吧？我回去以后带着牛仔裤、电子表，我给人家说，你知道我跟你们讲过，我就短短的几周，本来要到北京卖的，在火车上我就卖了。卖完以后最后这哥们儿我给了人家，没有地方，在厕所里待着啊，是吧？人家哥们儿就把我弄到餐厅那儿去了，我就一人送一电子表。哇噻，这哥们儿一下牛了，“哎，你就坐在这儿吧，你在餐厅坐着吧，” 能进餐厅多牛啊？“你别走了啊，小郭，你就在这儿啊。” 那时候能跟餐厅、餐桌上的人认识多牛啊？你知道我们保持到现在还有关系呢，他的孩子的孩子还跟我保持联系呢。结果我都卖完了，我就别往北走了，下来吧。当时到哪儿呀?当时只到了湖南我就下来了，下来他给我找个车还得给我再送上餐车我再回到广东去，再贩卖回去，就这么一折腾几千人民币就出来了，几千就像现在几千个亿的感觉，不是几万元，是几千个亿你赚了。就这么牛啊！就这么牛啊！十二岁、十三岁我当时不算是小，我十三岁就是1米75，早上量，晚上1米74，啊，从来个儿没改过，就处男你不是了，再没长过个儿，我就这么高的个儿当时。而跟我一起闯江湖比我小的还有呢，都十来岁的孩子，七八岁的孩子。你能坐过火车，闻过汽油味儿，你回到东北就没人相信，你甭说城了，Eric刚才描绘的说明你是城市人，你不是农村人，我们是真正的农村人。知道了吗？而且像Rachel这个以为十二岁，现在很多战友下边儿说，十二岁是孩子，你们都过得太好日子了。如果十二、三岁你不在家干活，你就是劳动力了你在那个时候，你在想啥呢，是吧？我说的很多都是发生在之前的事儿，十二三岁以前的、十岁左右的时候。唐山大地震我已经是回山东老家了，独立坐火车，将近一星期，你看从红旗岭镇坐客车到磐石县，磐石县再坐客车直接到梅河口，中间还倒一次，从梅河口才能坐火车是从梅河口坐上哈尔滨到北京的火车，你得等大概一天时间，在火车站就是流荡在那儿，你大厅里不让你待，所有人都带着被子，中国那时就跟北朝鲜一模一样，都带着被子，你有上哪儿跑去？你在车站旁边儿遛着吧你呀，厕所都没你的地方。要等一天还得饿着，到晚上凌晨蹭上过路车上火车，你能抢着位置吗？大盆儿小碗儿的，你背着这破行李烂被子的？上去以后你坐在厕所里你就不错了。这个中间要停大概将近110站，我要到山东禹城下来，从禹城下去再坐客车这就已经三天后了啊，这已经出家第五天了已经啊，不饿死已经不错了。从山东禹城下去你还得想办法到客车站去，你怎么去客车站呢？找到客车站，再坐客车到聊城，从聊城到莘县，莘县再到我们老家的曹城，曹城走路或搭车回到古城，从古城再回到我们家西曹营，一星期后的事儿。你在想啥呢？刚才Eric一讲起兜一圈儿，你想想那时候，那时候七哥哪儿十三呐？唐山地震哪一年？毛泽东死哪一年呐？那时候儿七哥都已经这样的江湖了，我还带着我弟弟回去第二次，我带着比我小的八弟回去，你去想想。我爹给了我18块钱，我说：“爹，你能不能给我买双鞋？” 一大巴掌乎过来了，我那爹的大巴掌乎得我眼睛一花，鼻子流血，差点给我流昏过去了，你去想想那是什么感觉。还十二、十三呢，就是大家你们十二、十三已经是老人家了，真的是，老人家了！
