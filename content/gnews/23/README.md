## 英版“下乡送温暖”：疫苗小组到农村上门接种
` 西班牙巴塞罗那喜悦农场` [轉載自GNews](https://gnews.org/zh-hans/1798362/)

**翻译：Lulumi**

![](https://assets.gnews.org/wp-content/uploads/2021/12/image-1591.png)图片来源 : thegatewaypundit.com

**声明：下文所提及的疫苗是指“COVID-19疫苗或新冠疫苗”，中共病毒是指“新冠病毒COVID-19”。**

英国官员认为，他们对抗冠状病毒的最新举措是被公众支持的，而在这里，我们认为水蛭放血法是野蛮的。

据《每日邮报》报道：在英国部长们正在考虑的计划中，带着COVID注射器的小组将被挨家挨户派往未接种疫苗的英国人家中，来帮助估计有500万人尚未接种疫苗的民众。

在过去的一周里，英国卫生部、英国国家医疗服务系统（NHS）和唐宁街10号首相府之间的讨论已经考虑在全国范围内向接种率低的地区派遣疫苗小组，作为避免封锁和其他限制的一个重要途径。

这也被认为是向农村地区或人们不容易到达接种中心的家庭提供疫苗注射的一种方式。

据《星期日邮报》报道，英国当局正在考虑将护士送到人们的家中，以此来避免新的限制性措施，并将疫苗送到农村地区，因为那里的疫苗接种中心可能不太方便。

该媒体引用了一位不愿透露姓名的内阁部长的话，称他支持任何“鼓励不愿接种疫苗的人”的做法，并认为“国家对拒绝接种疫苗的人的态度正在变得强硬”。

政府没有对这些报道作出回应，这些报道是在首相宣布进一步的中共病毒Covid措施之前发布的。鲍里斯·约翰逊预计将在周一发表声明，因为Omicron变种的快速传播已经促使重新采取一些限制措施。仅在周五，英国就报告了超过12.2万个新的中共病毒Covid病例——这是大流行病开始以来的又一个单日确诊记录新高。
