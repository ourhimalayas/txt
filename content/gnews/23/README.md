## 埃隆-马斯克收购Twitter的9.2%股份

编译：七叶之芒
![](https://assets.gnews.org/wp-content/uploads/2022/04/xin_png.001-1-1280x140-4.jpg)
特斯拉CEO埃隆-马斯克周一在一份美国证券交易委员会文件中披露，该社交媒体巨头已经收购了Twitter 9.2%的股份，此举是在这位亿万富翁指责Twitter未能遵守言论自由原则数周后发生的。

![](https://assets.gnews.org/wp-content/uploads/2022/04/image-363.png)

关键事实

- 推特在其监管文件中称，马斯克购买了该社交媒体平台的约7340万股普通股。
- 这一披露导致Twitter的股价在美东时间周一上午7:15的盘前交易中大涨25%。
- 金融时报指出，这次收购将使马斯克成为公司的最大股东，几乎是创始人杰克-多尔西2.25%股份的四倍。


巨额资金

28.9亿美元。根据周五的收盘价，这就是马斯克的Twitter股票的价值。

关键背景

马斯克尚未就此次收购发表公开声明，但这位亿万富翁此前曾表示，他希望看到Twitter的变化，甚至暗示要建立一个竞争对手的平台。3月25日，马斯克在Twitter上进行了一项投票，询问他的粉丝是否认为Twitter“严格遵守”言论自由的原则，并补充说投票的结果将是重要的。在200万Twitter用户中约有70%投了“反对票”后，马斯克指出，Twitter是“事实上的公共市政广场”，并指称它未能坚持言论自由原则，并在此过程中破坏了民主。12月，这位亿万富翁发布了一条神秘的推文，将社交媒体平台新宣布的首席执行官Parag Agrawal描绘成斯大林，而Twitter的创始人和前首席执行官Jack Dorsey则被显示为被扔进河里。

趋势

根据福布斯的实时亿万富翁榜，马斯克的净资产为2876亿美元，是目前世界上最富有的人。亚马逊创始人杰夫-贝佐斯（Jeff Bezos）以1893亿美元的净资产遥遥领先，位居第二。
