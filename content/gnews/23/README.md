## 中共又一次谎言！杭州疫情源头是“进口咖啡机芯”？

整理编辑：东京樱花团/peterwong

![](https://assets.gnews.org/wp-content/uploads/2022/02/6162971.jpg)

中共浙江杭州新一轮本土中共病毒（新冠）Omicron疫情已累计113人确诊。官媒报道，首发病例来自位于杭州滨江区的慧而特（中国）餐饮设备有限公司，传播源为「进口咖啡机的机芯」，由「物传人」再到产生「人传人」。

杭州市2月3日下午3时许举行新冠疫情防控工作记者会上公布，此轮疫情累计确诊病例113例，最近2天新增病例为个位数，从数量来看，疫情转捩点已经出现。

中共国杭州市政府副秘书长毛根洪表示，从病例被发现的脉络来看，近5天54个病例都是从集中隔离点筛查出来；从病例感染路径来看，传播链显得清晰；此轮疫情持续关注的几个风险较大场所和人群，已经没有新病例出现，「这次疫情已经社区清零，疫情防控工作从社区清零转为隔离点清零。」

中共国媒体《澎湃新闻》指出，Omicron变种毒株传播力极强，能迅速产生多代传播，杭州就出现「1传22」的情况。至于感染源方面，首发病例来自位于杭州滨江区的慧而特（中国）餐饮设备有限公司，传播自「进口咖啡机的机芯」，再产生人传人。

对此，不少网友提出质疑：「我不信，这机芯从哪里进口的，什么品牌，什么时候发货的，漂洋过海多少天到的杭州，病毒需要宿主才能活，这些信息能不能公布下！ 」、「进口货物至少一个月起步」、「问题是，作为一个国际物流从业人员了解的来说从德国进口商品到杭州，从德国工厂装箱—到德国海关清关—再到码头堆厂—上船—宁波或上海港—进口清关—陆路运输—货主手中，按最近这个物流速度。少说也得45天以上」、「别问，问就是咖啡芯片」。

关于中共从武汉爆发中共病毒疫情之初就故意误导世界，从“病毒从蝙蝠传人，病毒不会人传人”，到“病毒通过海鲜传人,水果传人”，如今无耻到说是“咖啡机芯传人”， 总而言之病毒就是境外传播到国内的，独裁政府的特色就是把谎言说上一千遍，直到让你相信这就是真理，果然病毒还是姓党，想让他怎么传就怎么传。
