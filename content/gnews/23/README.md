## 港警以违反国安法为由 拘捕6名立场新闻高层

整理编辑：东京樱花团/peterwong
![](https://assets.gnews.org/wp-content/uploads/2021/12/image-1624.png)[图片来源](https://www.secretchina.com/news/b5/2021/07/01/976737.html)
香港警务处国家安全处周三（12月29日)表示，今早拘捕六名网媒公司高层人员或前高层人员，涉嫌违反《刑事罪行条例》第9及10条串谋发布煽动刊物罪；同时，国安处上午执行由法院根据《香港国安法》第43条实施细则附表1发出的法庭手令，搜查观塘区一网媒公司办公室。

国安处声明称，行动中，警方共派出超过200名军装及便衣人员，搜查行动仍在进行中。搜查令包括授权警方搜查及检取新闻材料的权力。被拘捕六人包括三男三女，年龄介乎34岁至73岁；警方亦到被捕人士的处所进行搜查。

据网媒“立场新闻”Facebook上发文，国安处人员早上约6时前往立场新闻副采访主任陈朗昇住所，称有法庭手令，以“串谋发布煽动刊物罪”作调查。他亦为香港记者协会主席。

本地多家媒体报导，涉及网媒公司为立场新闻。而路透目击至少有四架警方车辆停泊于立场新闻办公室所在大楼外面。

今年6月警方国安处动员数百人搜查壹传媒大楼，并以涉违国安法拘捕多名苹果日报编采高层。《苹果日报》其后以基于员工安全及人手考虑，决定停止运作，并于6月24日出版最后一份实体报纸。而香港财政司长于9月向法院提交将壹传媒清盘呈请。
