## 美橄榄球联盟跪拜中共 国际分布图将台湾并入中共国
` 西班牙巴塞罗那喜悦农场` [轉載自GNews](https://gnews.org/zh-hans/1778512/)

编译： Jenny Ball

![](https://assets.gnews.org/wp-content/uploads/2021/12/image-1143.png)图片来源 : [westernjournal.com

美国国家橄榄球联盟（NFL）已完全向中国跪倒，将台湾作为中共国的一部分列入其在海外营销的国家地图。

据 NFL 发布的消息，NFL 周三宣布，18 支球队拥有 26 个所谓的国际家庭营销区的独家营销权。

“对NFL的狂热始于我们的俱乐部，”NFL 执行副总裁兼首席战略与增长官克里斯托弗·哈尔平 (Christopher Halpin) 说。 “这一重要举措，使 NFL 球队能够与海外 NFL 球迷建立有意义的直接关系，推动全球球迷的增长和热情。在最初的申请期间，我们对俱乐部提案的数量、创造力和承诺水平感到非常满意，并期待球队在明年初开始付诸行动。”

中共国是将成为洛杉矶公羊队两个国际球迷基地的国家之一。

但引起最大关注的不是哪些球队现在会增加外国球迷，而是 NFL 决定将台湾纳入中共国，以迎合中共。

1949 年，中国的国民党政府在革命败给毛泽东后，在离中国海岸不远的岛上避难——当时称为福莫沙岛。中共领导人认为台湾理所当然地是其地盘的一部分。最近几个月，他们加大了对这个极度独立的岛屿的外交和军事压力，该岛历来保持独立，主要是因为美国等国家的军事援助。

中共极其重视地图。今年早些时候，在 NBC 没有将台湾纳入其共产主义国家地图作为 NBC 奥运报道的一部分之后，中共外交部愤怒地跺脚。

据美国全国广播公司报道，当时中共驻纽约总领事馆表示，NBC 不包括它想要但不能拥有的东西，“伤害了中国人民的尊严和情感”。

据彭博社报道，其声明称：“该地图是国家领土的表达，象征着国家主权和领土完整。”

声明说：“我们敦促NBC承认这个问题的严重性，并采取措施纠正错误。”

中共官方控制的《环球时报》也批评美国全国广播公司在其地图上没有包括台湾，称这是针对中国的“肮脏政治把戏”的一部分。

据福克斯新闻报道，拜登白宫周五切断了一位台湾官员的视频源，因为它在地图上将中国和台湾显示为不同的颜色。
