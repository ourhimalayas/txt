在[《7·24系列之“单伟健”——沃顿商学院助理教授》](https://gnews.org/zh-hans/1734501/)一文中，我们可以从单伟健的履历看出其极不寻常的背景，从他留学美国一开始就在执行中共的“特别任务”。根据郭文贵先生的爆料，单伟健的特务级别非常高，直接向国家现任常委汇报工作。那么单伟健在美国都“结交”（蓝金黄）了哪些名人？

从来自公开网络的信息可以看到，单伟健与美国政坛的常青树、加州资深联邦参议员黛安·费恩斯坦（Dianne Feinstein）有着千丝万缕的联系。

据单伟健自述，“当初在选择美国学校的时候他也完全没有概念，就根据中国大学的传统——顶尖大学多是以城市的名字来命名的，因此他选择了入读旧金山大学。而当时初到美国，在欢迎会上遇到一位女士，就是日后成为联邦参议员的黛安·费恩斯坦，那时初到美国的兴奋仍然历历在目。”

- ![](https://assets.gnews.org/wp-content/uploads/2021/12/weijian-shan-and-mayor-of-san-francisco.jpg)https://weijian-shan.com/timeline/1990/


这位黛安·费恩斯坦女士，出生于1933年，是美国目前年纪最大的参议员，自1992年起一直担任加州参议员一职，蝉联至今，历时近三十年。此前费恩斯坦是旧金山的首任女市长，任期由1978年12月至1988年1月，而单伟健1980年至1986年则分别在旧金山大学和加州大学伯克利分校读书。

1990年，费恩斯坦竞选加利福尼亚州州长但没有成功，输给了当时的共和党候选人彼得·威尔逊（Pete Wilson）。在这次选举中，费恩斯坦因为没有如实申报选举中的献金和开支而被罚款190,000美元。

2018年媒体曝光，费恩斯坦有一名为她工作了20年的工作人员被美国联邦调查局查出是中共间谍，这位“司机”在工资单上被列为“办公室主任”，通过中（共）国驻旧金山领事馆向中国国家安全部报告。费恩斯坦对此事予以承认，她在声明中说：“五年前，联邦调查局告诉我，我在加州的工作人员中，一名行政人员可能被中共政府要求提供信息”，此时费恩斯坦正在参议院担任情报委员会主席，但是费恩斯坦坚称，“他从未接触过机密或敏感信息或立法事项”，并立即解雇了这位“司机”。

根据公开信息，单伟健曾担任TPG资本（TPG Capital，德州太平洋集团）的资深合伙人、带领“新桥资本”投资了深发展银行。而这家“新桥资本”和费恩斯坦参议员也有很大的关联性。

新桥资本（New Bridge Capital），据称是亚洲最大的私人股权投资机构之一，管理的资金达17亿美元。1994 年，由TPG资本、百隆资本（Blum Capital）和ACON Investments联合成立了新桥资本，主要投资于新兴市场，尤其是亚洲。在巅峰时期，新桥管理的资金超过32亿美元。新桥资本的总部与TPG一起，位于沃思堡和旧金山，在亚太地区的香港、墨尔本、孟买、首尔、上海、新加坡和东京均设有投资办事处。

而拥有新桥资本的这三家公司，其中之一的百隆资本的老板——理查德·布鲁姆（Richard Charles Blum），正是黛安·费恩斯坦的丈夫。据维基百科，布鲁姆对西藏和藏传佛教有着浓厚的兴趣，他是美国喜马拉雅基金会(AHF)的主席和创始人，但并未参与中（共）国对西藏的控制。

另据维基百科，加州参议员费恩斯坦因其丈夫的政府合同、百隆资本与中（共）国的广泛业务往来，以及费恩斯坦过去在与中（共）国贸易问题上的投票而多次受到审查。布鲁姆否认有任何不当行为。

2004年12月份，新桥资本以每股3.54元受让深发展17.89%股权，成为深发展的第一大股东。截至2009年6月，新桥资本总共拥有5.2亿股深发展股份，占比16.76%。而这一收购就是单伟健在新桥资本时创下的“杰作”。

无论如何，单伟健与费恩斯坦的“友谊”一直保持。2019年单伟健出版《走出戈壁》一书时，费恩斯坦为此书写到——“这（书）是由中国最聪明的人之一撰写的毛泽东和后毛泽东时期的中国个人历史。值得一读，更重要的是了解大跃进！”

“中国最聪明的人之一”，就是费恩斯坦参议员对单伟健的评价。而我们不仅要问，对中共特务这么高的赞美，背后的代价是什么？
