新中国联邦和爆料革命创始人郭文贵先生11月15日晚在盖特（GETTR）上发布盖文称，辉瑞公司首席执行官阿尔伯特·波拉的妻子11月10日早上被送入纽约长老会劳伦斯医院急诊室，之后死亡。

郭先生进一步指出，据保守派网站“比弗”报道， 根据从辉瑞公司总裁太太的医师那里获得的消息，阿尔伯特·波拉的妻子是死于辉瑞公司疫苗并发症。

在阿尔伯特·波拉的妻子去世后，其家人要求保护隐私。法官已经批准了临时媒体封锁令，因此所有美国大媒体都没有报道此事。

不过，在网上通过搜索可以搜到几家自媒体的报道，其中包括非右派的自媒体。而保守派网站“比弗”的总部设在加拿大，因此不受法官裁决的约束。

辉瑞公司首席执行官阿尔伯特·波拉在他妻子去世的前一天，即11月9日曾声称，传播疫苗错误信息的人（反对疫苗者）是罪犯，他们正在造成生命的损失。岂料仅仅时隔一天，他的妻子就死于他所在公司生产的疫苗所导致的并发症。
