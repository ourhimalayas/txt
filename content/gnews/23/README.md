2022年元旦大直播中，郭先生提到一命二运三风水，四积阴德五读书，天道酬勤，地道酬人，战友们之间都是有姻缘的，一定要相信轮回，五行中金木水火土，金生木，木生火，火生土，土生万物，土就是包容的。

对于ccp冬奥会之前把盘古大关龙头变包头，把佛手拿走，以及秦岭发生的事情，郭先生回忆起了盘古大观奠基石与黑蛇白蛇的故事，并提到了当年的风水先生，风水先生做奠基石的时候，拿五湖四海之土培养盘古奠基石，取土之人属相，出生八字，风水先生亲自跟着认定的人，而且告诉每个人，取了这个土可能会有生命之余，因为这是涉及到天机。

风水先生告诉郭先生，动奠基石之日 ，就是中国人重新开始之时。让郭先生盼望着那一天的到来，而不是是盼望没人动他。

据郭先生爆料，风水先生说他的日子到了，进不了水年，做了一个最懦弱风水先生不该做的事情。他一生研究奇门遁甲、八卦，给盘古看过风水，给郭先生家看过风水，给习总加速师也看过风水，并说出了习家的风水出现了问题。

郭先生接着提到龙是中国的神秘文化，是吉祥的象征，现在ccp把龙的脖子给套住了，脖子给拧住了，龙一定会挣扎，龙不会坐地等死。日本人说两龙之争，必有一伤。

郭先生详细讲述了盘古的风水，盘古对面奥运村跨过了四环，从南亚运区到北亚运区是个水龙，水为阴，盘古龙头突出为阳，一阴一阳形成了阴阳。鸟巢为圆，水立方为方，天圆地方。

中间中轴线，再一中轴线，直对中南坑对面正好十公里的水蒸蛋，盘古南北中轴线，东西在连接重要的地方，并且盘古龙头与陕西，山西重要地方相连。
