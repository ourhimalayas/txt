2月1日，日本国会下议院通过一项决议，表达对中共国人权议题的关切，包括中共当局对待维吾尔穆斯林和香港的方式。

据《中央社》报道，日本下议院当天通过的这项决议对中共国新疆地区、西藏、香港和内蒙古的“人权状况表达严重关切”，并呼吁政府就中共国人权议题积极表态。

去年12月，日本因中共国的人权问题表示不会派政府代表团参加北京冬奥，并呼吁中共国尊重人权与法治。
