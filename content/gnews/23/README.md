**#****一针新冠疫苗的后劲真大#**

之前听很多人说一针的疫苗反应比较大，吓得我一直对这个疫苗很恐惧。现在疫情有反复，必须接种，昨天上午和单位很多人一起接种的一针剂腺病毒载体疫苗。

刚开始30分钟现场观察，没见到有人有反应，大家觉得可能是之前的人夸大了，都该吃吃该喝喝。下午我还去同事家喝茶吃点心，晚饭后去看人家砍芭蕉摘杨桃。

结果快八点的时候，感觉不太对，好像开始犯困乏力。因为自己一个住，害怕发烧太高，还专门去买了个温度计和布洛芬，这可能是做的最理智的事情。

回来之后，赶紧收拾好，在床头准备了很多水，想着躺下来也不怕。然后不到九点的时候，37℃多低烧，九点多就38℃。那时候有反应的人还不多，有的同事还在怀疑自己打了假疫苗。

等到10点，陆续开始有人发烧，有的直接39度多。我一直在疯狂喝水，想把温度控制在38.5以下。那时候自己还比较清醒，朋友聊天，他们说这反应也不大嘛。再到后来温度到了38.7℃，实在降不下去，吃了一粒布洛芬，开始断断续续睡觉。中间不知道跑了多少趟卫生间，温度降不下去，直到凌晨两点左右降到了38.3℃，才开始真正睡觉。中间醒了几次，今早起来全身酸痛，并伴有间歇性头疼，和我妈说像被人打了一顿。

昨夜后期很多同事也开始发烧反应。一针剂的疫苗，药效有点猛，希望自己早点恢复正常状态。也希望打针的朋友们，一定备好水银温度计和退烧药，以防万一。
