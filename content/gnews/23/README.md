[中共2018年3月曾修宪，使国家主席和副主席得以连任超过两届。预计现任中共党魁习近平将在今年的二十大上打破先例开启第三个任期，继续担任中共最高领导人，并于次年继续担任国家主席。](https://www.epochtimes.com/gb/22/3/11/n13639587.htm)
