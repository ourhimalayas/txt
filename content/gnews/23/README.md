《南华早报》10月16日报导，美国总统拜登15日在康涅狄格大学“陶德人权中心”落成典礼上发表演说，把中共国对新疆维吾尔族的政策，连结到第二次世界大战期间的纳粹大屠杀，并警告世界领导人若对侵害人权的行为保持沉默，就是“共犯”。

“陶德人权中心”以美国前联邦参议员克里斯-陶德及父亲汤玛斯-陶德命名。汤玛斯是二战结束后，在纽伦堡审判中起诉纳粹战犯的检察官之一。

拜登强调，人权是美国对外政策的重中之重，包括美中关系。当人权受到侵害时，任何一位美国总统都不应该袖手旁观。美国国务院认定中共国打压新疆维吾尔族与其他少数民族的作为，构成“种族灭绝”，而其他专家则界定为违反人道罪。
