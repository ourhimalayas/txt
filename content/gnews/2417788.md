```
东京，4 月 25 日（路透社） - 瑞穗金融集团首席执行官表示，该集团计划加强其在美国的并购咨询和股票承销业务，
因为日本第三大银行的目标是在全球最大的投资银行费用池中获得更大的份额。
```

```
瑞穗在 2015 年收购苏格兰皇家银行的北美公司贷款组合后，近年来在美国加强了债务承销能力，又推出了最新的举措。
```
 
木原正弘(Masahiro Kihara)在接受路透社采访时表示，瑞穗在美国投资银行业务中尚未充分利用的是并购咨询和股权承销。“我们现在正在制定我们的战略，并分析我们可以在多大程度上实现有机增长” 他说。
 
他说，内部讨论包括关注哪些行业、如何扩大分析师的覆盖范围以及如何与其销售和交易团队建立合作，但拒绝提供更多细节。
 
受国内超低利率困扰，日本银行一直在探索日本以外的增长机会，瑞穗专注于美国和亚洲。

```
根据 Dealogic 的数据，去年美国并购咨询和所谓的股权资本市场 (ECM) 的费用总额为 428 亿美元。
ECM是银行帮助公司筹集股权资本的市场。
```
 
在主要银行部门的一系列电子系统故障引发高层管理人员重组后，木原正弘于 2 月接任瑞穗。
 
瑞穗是软银集团的最大贷方之一，软银集团是一家科技集团，由于股价和投资组合估值暴跌，其财务面临压力。截至去年 3 月，瑞穗有 8500 亿日元（66 亿美元）软银未偿还的贷款。
 
“我们一点也不担心，” 木原正弘在接受采访时被问及软银的财务状况时说。他补充说：“他们致力于他们的 LTV（贷款与价值）比率和其他财务纪律政策，我们在管理层面上相互信任。”
 
（1 美元 = 128.6600 日元）
