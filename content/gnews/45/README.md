**1.东京中共病毒疫情持续蔓延，奥运会举办期间日感染人数预计将超过1千人**

[朝日新闻7月11日报道，东京都内中共病毒疫情持续蔓延，10日新增确诊病例950例，离5月13日以来单日超千人确诊已经近在咫尺。政府将从12日起宣布东京进入第4次疫情紧急事态，但在东京奥运会23日开幕之前此宣言的效果有限，奥运会举办期间日感染人数超过1千人已经难以避免。东京都内第3次紧急事态宣言解除后不久，疫情反弹的趋势就开始显现。](https://news.yahoo.co.jp/articles/1e62d7aed35b9c82982aaaef660b619e1ba3b48c?tokyo2020)

**2.****前首相安倍晋三对下届众院选举表示感受到了危机**

[据共同社7月11日的消息，日本自民党前总裁、前首相安倍晋三11日在北海道苫小牧市发表演讲，对即将于秋季前举行的下届众院选举表示感受到了危机。在提及结果不理想的东京都议会选举时称，“自民党正遭遇严峻的选情风向，我们必须谦虚地对待。”围绕中共病毒疫情，安倍晋三重申支援企业等的重要性，他还认为尤其需要援助旅游业。](https://news.yahoo.co.jp/articles/fa1b08ab80febc022ab727c47c72d9aee8e5642e)

**3****.****日本官房长官加藤胜信称26日起受理申请疫苗接种证明**

[根据时事通信社7月11日的消息，日本官房长官加藤胜信在11日的NHK电视台的节目中，言及中共病毒疫苗接种史的官方证明（疫苗护照）时透露，将从26日起在市区町村的窗口受理该证明。加藤胜信称主要对象是“打算从日本前往海外，在入境时希望放宽防疫措施的人。”加藤胜信强调，疫苗接种绝不能导致歧视与强迫等现象。对于在国内是否运用该接种证明，他表示“必须另外进行探讨”。](https://news.yahoo.co.jp/articles/3f6249ec11aa8657965ccec50ef12bee878f0da4)

**4.****中共病毒疫情在大阪蔓延，知事吉村洋文称“处于第5波的入口”**

[据朝日新闻7月11日的报道，大阪府知事吉村洋文11日出席了在线举行的全国知事会中共病毒应对总部会议。针对大阪府内的中共病毒的感染状况，吉村洋文称，“我们正处于第5波的入口。抑制疫情出现高峰十分重要。”10日，大阪府新增确诊病例200例，这是自6月3日以来首次达到200人。与上周（6月27日至7月3日）的感染人数相比，本周感染人数增加了将近30％。会后，吉村洋文对记者说，“感染人数不多，因此未被关注。但从增加的百分比来看，大阪与东京首都圈的增速相当。人流外出的状况与第4波前的3月份相同，但本次换成了感染力更强的Delta变异毒株。”](https://news.yahoo.co.jp/articles/8627b9311e2b4d0cecfed3ac56681692f4fa0bab)

**5.****大阪邮政小包运输车起火，约400个邮包被烧毁**

[据产经新闻7月11日的消息，日本邮政宣布，在大阪府堺市公路上行驶的邮政小包（Yu-Pack）的运输卡车11日凌晨5时30分左右发生火灾，装载的约400个邮包被烧毁。据悉，该包裹原定于当日送至大阪府河内长野市与大阪狭山市。卡车司机没有受伤，起火的原因正在调查中。](https://news.yahoo.co.jp/articles/e42f28eb3f6a04732cd90e100ec56f9bb2433b36)

**6.****制药企业第一三共集团宣布年底前进行数千人规模的中共病毒疫苗临床试验**

[据NHK7月12日的报道，研制中共病毒国产疫苗的制药企业之一的第一三共集团（DAIICHI SANKYO）负责研发疫苗的薮田雅总经理接受NHK采访时披露，他们正准备在今年年底前对一种国产的“mRNA疫苗”进行数千人的临床试验，并根据试验结果向政府申请生产许可。目前，在日本使用的“mRNA疫苗”全部由海外研发，但第一三共已经开始利用独自的技术研发的国产“mRNA疫苗”进行临床试验。如果获得批准，埼玉县的工厂将进行批量生产。](https://www3.nhk.or.jp/news/html/20210712/k10013133761000.html)
