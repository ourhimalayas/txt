## 美国指责中共国策划全球黑客事件后，北京威胁要“报复”
` 洛杉矶盘古农场 HIMALAYA LA PANGU` [轉載自GNews](https://gnews.org/zh-hans/1410783/)

据《零对冲》作者：TYLER DURDEN，2021 年 7 月 19 日报道：

更新（13：10 ET）：尽管北京威胁要报复美国，如果它实施新的制裁或更糟的是——试图用自己的网络攻势进行报复。拜登政府刚刚发表声明称，它保留采取额外单边行动的权利。尽管它不排除单边行动，但它更愿意与其国际盟友一起采取行动。

更新（12：30 ET）：北京刚刚回应了拜登政府及其“盟友联盟”的最新指控。在英文《环球时报》发表的一篇社论中，亲政府的报纸抨击美国政府声称中共国政府与独立黑客合作的说法是“一个巨大的谎言”，并补充说这种做法在中共国的体制下是无法实行的。

社论称，中共国故意允许黑客犯罪以牺牲中共国“国家声誉”为代价获利的想法也是荒谬的。

**中共国有严格的问责制度。多年来，美国一直指责中共国进行网络攻击。这已成为中美之间高度敏感的问题。我们不禁要问：中共国像美方指责的那样对美国发动大规模网络攻击，严重打击中美关系的动机和利益是什么？美国的哪一条信息或经济利益比中国的国家声誉更重要？**

**美国甚至声称中共国允许黑客承包商谋取私利；也就是说，中共国愿意用自己的国家利益来为那些黑客及其所代表的特定组织的利益买单。我们不知道美国和西方资本控制的社会是否有这样的逻辑，但这在社会主义中共国是完全不可想象的。**

作者补充说，网络攻击极难追踪，华盛顿正在“利用”其网络困境来“陷害”中共国。 “几乎所有国家都发生了网络攻击，中共国遭受的损失比美国更大。”

最后，北京指责华盛顿对中共国的“诽谤”、“过分”，他们补充说，如果美国“采取激进措施，对中共国进行国家级网络攻击，或对中共国实施所谓的制裁，我们将进行报复”。

***“*****美国不能利用这些抹黑对中共国进行实质性攻击。如果美国采取咄咄逼人的措施，对中国进行国家级网络攻击，或者对中国实施所谓的制裁，我们将进行报复。*****”***

与此同时，《环球时报》的首席编辑、北京最引人注目的英语喉舌之一胡锡进正在推特上谈论美国警察的“压迫性”。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/07/21011551/E8ED50E8-F316-4B8D-B6AA-D70FCB69F18E.png)
随着拜登总统和他的政府继续提升他“对中共国强硬”的资历（这将成为 7 月份白宫的“主题”），美国刚刚宣布，中共国情报机构的成员雇佣网络犯罪分子进行全球黑客和网络入侵运动。

预计美国将于周一表示，中共国政府是一系列恶意勒索软件、数据盗窃和其他针对全球公共和私人实体的网络间谍攻击的幕后策划者，其中包括今年早些时候发现的 Microsoft Exchange 黑客攻击。

据彭博社报道，美国将提供证据，证明中共国国家安全部是如何利用合同黑客犯罪在全球范围内进行未经批准的网络活动，包括为自己谋取私利。此外，美国国家安全局、中央情报局和联邦调查局将揭露中共国国家资助网络黑客所使用的 50 多种策略。

据英国《金融时报》报道，一位美国高级官员表示：

***“*****中共国的*****MSS –*****国家安全部*****–*****使用合同黑客犯罪在全球范围内进行未经批准的网络操作，包括为自己谋取私利。*****”***

***“*****他们的活动包括犯罪活动，例如网络勒索、黑客加密和从世界各地的受害者那里窃取经济利益。*****”***

在此之前，美国并没有公开指责北京对微软的黑客攻击。

当然，这并不是我们第一次听说 Microsoft Exchange 黑客攻击或其他中共国网络入侵。继勒索软件攻击导致殖民地天然气管道瘫痪之后，今年网络安全已成为越来越重要的政治优先事项。这些措施也让人想起拜登的老上司奥巴马总统，他曾指控 5 名中共国军人涉嫌参与对美国公司的网络攻击。当然，结果证明这是奥巴马的“捶胸”操作而不是真反击。

从表面上看，美国并不孤单。据报道，北约成员国与欧盟、新西兰、日本和澳大利亚正在建立新联盟，该联盟涉及共享网络威胁情报并合作提高网络安全。据 CNBC 报道，微软将这次攻击归咎于一个名为 Hafnium 的中共国支持的组织。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/07/21011718/B716415C-FE61-4511-AABB-65FC0BF8206A.jpeg)
预计他们还将与美国一起指责中共国及其情报机构下令进行对包括 Microsoft Exchange 入侵在内的黑客攻击。

CNBC 指出，新联盟是在拜登政府游说北约和欧盟成员国支持对中共国的网络活动采取更具“对抗性”的方式之后建立的。
