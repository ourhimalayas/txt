**西班牙2021年7月21日电/西喜社**

7月21日塔斯社东京报道， 世界卫生组织负责人谭德塞警告会出现比德尔塔更危险的毒株。

对此，周三在东京国际奥林匹克委员会第138届会议上发表讲话时提到。“（病毒）传播的范围越广，其变体就会出现越多，可能会变得比造成巨大损害的德尔塔变体更为危险。变体越多，从中出现抗疫苗变种的几率越高，那样的话我们都将回到最开始的时候。”他强调。

世卫组织主任说，经常有人问他这样一个问题：”疫情何时结束？”事实上，新冠病毒和疫情给我们提出了许多问题。疫情是一个考验，而且世界还没有战胜它，”他说。”在我讲话的同时，将有一百人死于COVID-19，而到8月8日奥运圣火熄灭时，全世界范围内将有超过10万人死于冠状病毒。”

谭德塞表示他相信，尽管拥有所有必要的工具，但世界无法结束这一疫情的原因是缺乏真正的政治义务。 “各国政府，特别是G20，必须表现出集体意愿，并使用各种手段拯救人类生命。 如果世界主要经济体愿意，他们能够在几个月内控制这一流行病，如果他们分享疫苗并创造激励措施来扩大生产，”他补充说。根据他的观点，到明年年中，每个国家人口的70％接种疫苗不仅可以阻止冠状病毒大流行，还可以重新启动世界经济。

“我与国际货币基金组织，世界银行和世界贸易组织一起，呼吁在今年9月之前每个国家至少为10％的人口接种疫苗，在今年年底之前至少为40％，到明年年中，每个国家人口的70％接种疫苗。如果我们设法实现这些指标，我们不仅能够阻止这场疫情，而且还能够重新启动全球经济，”他补充道。 世界卫生组织负责人表示，为了完成设定的任务，世界将需要110亿剂的冠状病毒疫苗。

谭德塞向公司和企业呼吁。 “他们有社会责任， 大多数疫苗是使用国家资源研发的。许多公司已经做出了分享的承诺，但许多仍然没有实现，”他强调，生命受到威胁时，”利润和专利应该让路”。”如果国家能够进行动员和动员工业进行战争，那么为什么他们不能为打击共同的敌人做同样的事情？ 每个人都知道，对新冠的投资只是投资用于杀人的资金的一小部分，”世卫组织负责人表示遗憾。

世卫组织负责人说到底，还是在强调战胜疫情的关键是疫苗。是被人蒙蔽，还是自欺欺人？
