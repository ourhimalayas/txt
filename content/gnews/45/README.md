## 【战友寄语】中共终于实现“爱国者治港”的阴谋
` 喜馬拉雅-粵語組` [轉載自GNews](https://gnews.org/zh-hans/1410217/)

#### 搜集：文燕 \ 撰文：岁月如歌



大纪元时报19日报道：香港选举事务处18日公布，选委会界别分组的临时选民登记册，改制后，选委会仅余7891名选民，包括2547张个人票及5344张团体票，选民人数大跌近97%，有政治学者指，北京修改的选举制度着重对选举结果的可掌控性，而非反应不同界别声音。

战友评论：
经过这20多年的「挂羊头卖狗肉」，在林郑月娥集团的里应外合配合，终于有了今天的结果。九七回归之前，港英政府在立法会问题上花了不少心机，与中共谈判扯皮一两年，最终还是被中共收买的名人富豪等配合中共，使港英政府谈判节节败退，到处退让。末代港督临尾一脚欲通过政改，大量聘用港人执政等方式还政于民，除了要“两局分家”，取消所有委任议席，还新增九个功能组别议席（新九组），留下的是争议和质疑。现在应该很多港人怀念彭定康港督这一政策。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/07/20214341/7211.jpg)图片来自大纪元
港人质疑新九组只代表香港既得利益者，在当时没有真正的一人一票情况下，由原来的7个功能界别，增加九个，无疑是把民主扩大化，功能界别议席越多代表港人民意就越多。现在好了，政改改来改去不但毫无进展，还被中共一掌打回解放前。 2007年至今无数次上街游行、请愿、诉求，要求一人一票选出特首，多少人为此付出生命和鲜血的代价，均打不倒中共掌控香港的野心。 2021年3月4日，中共两会拍板通过《关于完善香港特别行政区选举制度的决定》，令香港选举制度倒退几十年，比彭定康的新九组还惨。

看全国港澳研究会副会长刘兆佳的说话：选举改制的目的标就是实现“爱国者治港”，如果保留过去的选举制度，反对派或者有能力推举自己的特首候选人，或许会出现建制派候选人争取反对派支持的情况，但改制后就不会再出现这些情况，他直言改制的结果「有利于一国两制的实现」，而非讲求“西方民主”。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/07/20214323/7212.jpg)图片来自自由亚洲粤语部
刘兆佳的讲话就是当年邓小平“一国两制，港人治港”，赤裸裸再现，当年邓小平在这个问题上还有所隐晦，到了刘兆佳就没必要遮遮掩掩了。我们看夏宝龙的“爱国者治港演”说，以及中共两会出台的《关于完善香港特别行政区选举制度的决定》，再到刘兆佳讲话，无一例外，都在紧扣邓小平当年的方针。可见中共的统一战线有多厉害。

相对夏宝龙的讲话和今年两会《关于完善香港特别行政区选举制度的决定》，刘兆佳的说话是直接告诉世界：中共就要这样耍流氓，容不得反对派，听不得反对声音；中共不可能让反对派有生存空间，必须捏死在萌芽状态；以前的「小范围选举」还是不够把握，通过政改把所有不同声音赶尽杀绝，全部剔除出共产党的圈子，保证万无一失。中共讲的民主是共产主义下的民主，不走西方“那一套”！

至于选举委员或会选民人数大跌97%～99%，无所谓，留岛不留人嘛，没人大陆有的是。香港中文大学政治与行政学院高级讲师蔡志强说，选民登记册反映中共希望百分百控制【行政长官】的选举。掌控香港、 统治香港才是目的，香港不再容许民主、法治的存在，邓小平如是说：「我们不全盘照搬西方民主那一套」。
