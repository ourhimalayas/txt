1. **美国和盟国发表联合声明谴责中共实施恶意网络攻击**


周一，美国与欧盟、北约、澳大利亚、加拿大、新西兰、日本发表[联合声明](https://www.whitehouse.gov/briefing-room/statements-releases/2021/07/19/the-united-states-joined-by-allies-and-partners-attributes-malicious-cyber-activity-and-irresponsible-state-behavior-to-the-peoples-republic-of-china/)，谴责中共国在全球范围内实施恶意网络攻击。

[布林肯国务卿](https://www.state.gov/responding-to-the-prcs-destabilizing-and-irresponsible-behavior-in-cyberspace/)表示，“这对我们的经济和国家安全构成了重大威胁”。

[一位高级政府官员](https://www.reuters.com/technology/us-allies-accuse-china-global-cyber-hacking-campaign-2021-07-19/?taid=60f58ec5c5720500017c366e&amp;utm_campaign=trueAnthem:+Trending+Content&amp;utm_medium=trueAnthem&amp;utm_source=twitter)称，中共政府支持的网络黑客使用了超过50种策略、技术和方式，对美国和盟国电脑网络发动袭击，包括今年三月对微软电邮系统的攻击。隶属中共国家安全部的黑客，利用微软Exchange服务器的漏洞地破坏了数以千计的计算机和网络，其中大多数受害者属于私营部门。

拜登政府表示，已经为联邦政府的五项网络安全现代化工程提供资金，以实现网络防御的现代化，保证国家的网络安全；并将继续与私营部门密切合作，解决关键基础设施的网络安全漏洞。

2. **美国司法部宣布对来自中共国家安全部的四名黑客提出刑事指控**

7月19日，[美国司法部](https://www.justice.gov/opa/pr/four-chinese-nationals-working-ministry-state-security-charged-global-computer-intrusion)宣布对来自中共国家安全部的四名黑客提出刑事指控。

四名被告中，丁晓阳（Ding Xiaoyang）、程庆民（Cheng Qingmin）和朱允敏（Zhu Yunmin）是负责协调的海南国家安全厅（HSSD）官员，负责协调、促进和管理海南仙盾和其他中共国家安全部（MSS）幌子公司的计算机黑客和语言学家，为中共国及其国有和赞助机构进行黑客攻击。另一名被告吴淑荣（Wu Shurong）是一名计算机黑客，就职于海南仙盾公司，创建恶意软件，入侵外国政府、公司和大学的计算机系统，并监督海南仙盾公司的其他黑客。

他们的黑客活动主要针对美国、加拿大、德国、挪威、沙特阿拉伯、南非、瑞士和英国等国家。目标行业包括航空、国防、教育、政府、医疗保健、生物制药和海事等。被盗的商业秘密和机密商业信息已经延伸到关键的公共卫生领域，包括埃博拉、MERS、艾滋病毒/艾滋病、马尔堡和土拉菌有关的传染病研究。

**3. 美国军机今年第三次公开进出台湾**

7月19日，一架[美军C-130](https://www.voachinese.com/a/US-military-charter-plane-lands-in-taiwan-to-make-deliveries-to-ait-/5971421.html?utm_source=twitter&amp;utm_medium=social&amp;utm_campaign=dlvr.it) 由菲律宾马尼拉机场起飞，在台湾桃园机场停留约一个半小时后离开台湾。

据台湾媒体的报道，这架美国军机是美国在台协会（AIT）的包机，除新任美国在台协会驻台北办事处处长孙晓雅（Sandra Springer Oudkirk）的私人物品外，运抵台湾的还有美国在台协会的外交邮袋。

这是一个半月来第三架在台湾落地的美国军机。

4. **智库报告称，中共国是对跨境数据流动限制最多的国家**

周一，智库信息技术与创新基金会(ITIF)公布报告称，中共国有29项数据本地化政策，是对跨境数据流动限制最多的国家。

各国政府常以保护本国公民隐私安全为由实施数据流动限制，而中共国以国家安全为由，出于民族主义或专制主义实施数据限制。

报告建议美国与盟友签订 “日内瓦数据公约”，以此作为数据共享的共同框架，并将中共国、俄罗斯等限制数据流动的国家排除在外。

**5. 欧洲中共病毒感染病例超过5千万**

7月20日，据[路透社](https://www.reuters.com/world/europe/europe-becomes-first-region-cross-50-million-covid-19-cases-reuters-tally-2021-07-19/?taid=60f6459ac5720500017c3a68&amp;utm_campaign=trueAnthem:+Trending+Content&amp;utm_medium=trueAnthem&amp;utm_source=twitter)报道，欧洲已成为全球首个超过5000万例中共病毒感染病例的地区。

欧洲是全球受影响最严重的地区之一，报告的病例占全球的27%，死亡人数占31%。约每八天，就有一百万新增感染病例。目前为止，欧洲已有130万人死于中共病毒。其中，俄罗斯是受影响最严重的欧洲国家，感染人数即将突破600万。

根据路透社的统计，欧洲前2500万个病例是在350天内发生，而后面2500万个病例报告仅用了194天。
