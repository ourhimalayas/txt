## 爆料革命继续起底SPAC的黑箱操作
` GTV新闻访谈` [轉載自GNews](https://gnews.org/zh-hans/1616296/)

作者：纽约香草山文艺部 宁文静

![This image has an empty alt attribute; its file name is Screen-Shot-2021-08-30-at-7.37.59-PM.png](https://assets.gnews.org/wp-content/uploads/2021/08/Screen-Shot-2021-08-30-at-7.37.59-PM.png)



10月24日郭文贵先生直播爆出，与前总统川普的新社交媒体合并上市的，SPAC结构的“数字世界收购公司”，在20日官宣之后的股市走向，进一步揭开其背后中共资本运作的身影。
![](https://assets.gnews.org/wp-content/uploads/2021/10/Screen-Shot-2021-10-25-at-7.50.19-PM.png)
这家空壳公司于9月2日刚刚在纳斯达克上市，上周一交易量1300多手，周二跃升至4万多手，周三高达6万多手。 21日，“数字世界”的股价一路高歌猛进，从开盘价9美元，涨到收盘时的45.6美元。22日，盘前价格直接高开118.8美元，最高达到175美元，最后收盘于94.2美元。 三天时间股价飙升10倍，洗走大约50至80亿美元。目前，“数字世界”的场外交易价格，已狂跌至 45 美元。

郭先生说，这些惊人的交易，其实是，由11个基金公司左右手互倒的资本操作。吴征操纵并购和上市，目的是阻止川普总统加入盖特，与爆料革命合作。同时，趁机让川普总统陷入内幕交易、虚假陈述以及操纵股价等刑事罪责，将彻底断送其政治生命。
