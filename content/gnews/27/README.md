**1. 美、日、印、澳四国领导人将于9月聚集华盛顿，商讨应对中共国调整**

2021年7月19日，据日本媒体报道，[美国白宫](https://www.voachinese.com/a/quad-summit-in-person-20210719/5970745.html?utm_source=twitter&amp;utm_medium=social&amp;utm_campaign=dlvr.it)提议在今年9月下旬举行美、日、印、澳四方峰会，商讨共同应对中共国的挑战。本次峰会将于联合国大会后在华盛顿举行。这也将是这四国领导人首次举行面对面峰会。

2. **英国BNO特许入境许可于7月19日到期，港人纷纷离港**

据报道，由于英国BNO”[特许入境许可](https://www.rfi.fr/cn/%E4%B8%AD%E5%9B%BD/20210718-%E8%8B%B1%E5%9B%BDbno%E7%89%B9%E8%AE%B8%E5%85%A5%E5%A2%83%E5%B0%86%E5%88%B0%E6%9C%9F-%E6%B8%AF%E4%BA%BA%E7%BA%B7%E7%BA%B7%E7%A6%BB%E5%A2%83%E6%9C%BA%E5%9C%BA%E7%88%86%E6%BB%A1?ref=tw_i)“LOTR将于7月19日到期，周日，香港机场挤满搭机及送机人潮。

因港版国安法的实施，英国于2020年中宣布，自今年1月31日起，持有BNO的香港居民可与家人根据“特许入境许可”(LOTR）机制前往英国居留，满6年后可取得永久居留资格。此后，英国政府数度延长“特许入境许可”LOTR期限，最新一次的延长期限至7月19日。

受访的港人纷纷表示，港版国安法是离港的主要原因。

3. **赞助商丰田撤回与东京奥运会相关的电视广告**

由于公众对[奥运会](https://www.reuters.com/lifestyle/sports/two-thirds-japanese-doubt-pandemic-olympics-can-be-safe-says-poll-2021-07-19/?taid=60f50801b881cc0001a152e0&amp;utm_campaign=trueAnthem:+Trending+Content&amp;utm_medium=trueAnthem&amp;utm_source=twitter)的支持率不高，三分之二的日本人怀疑本次奥运会能否在中共病毒大流行期间安全举行，赞助商丰田公司决定将不会投放与奥运会有关的电视广告。丰田公司周一表示，丰田汽车首席执行官和其他高管将不会出席奥运会开幕式。

在朝日新闻的一项民意调查中，68%的受访者对奥运会组织者能否控制病毒感染表示怀疑，55%的人表示反对奥运会继续举行。

7月18日，在奥运村的每日检测中，两名来自[海外的运动员](https://www.rfi.fr/cn/%E5%9B%BD%E9%99%85/20210718-%E4%B8%9C%E4%BA%AC%E5%A5%A5%E8%BF%90%E5%89%8D5%E5%A4%A9-%E4%B8%A4%E5%90%8D%E8%BF%90%E5%8A%A8%E5%91%98%E6%B5%8B%E5%87%BA-%E9%98%B3%E6%80%A7?ref=tw_i)被测出中共病毒呈阳性，这也是首次有运动员被测出中共病毒阳性病例。

东京第四度进入紧急状态，感染病例上升，公众越来越担心本次奥运会会加速东京的感染率，并引入更具传染性或更致命的变种病毒。

4. **日本国防部长警告，美日同盟将面临来自中共国和俄罗斯的联合攻击**

继中共国和俄罗斯于最近一起举行了几次联合军事演习之后，[日本国防部长](https://americanmilitarynews.com/2021/07/japan-warns-us-a-pearl-harbor-style-attack-from-china-and-russia-could-happen/)中山泰秀（Yasuhide Nakayama）表示，中共国和俄罗斯正显示出越来越多的军事合作迹象，他们在太平洋的活动显示出他们可能计划对美国发动珍珠港式的攻击。

中山泰秀建议，必须对中共国和俄罗斯显示出[威慑力](https://www.zerohedge.com/geopolitical/did-japan-just-warn-us-impending-joint-attack-china-and-russia)。如果中共国入侵台湾，这不仅仅关系到日本，也关系到美日联盟。
