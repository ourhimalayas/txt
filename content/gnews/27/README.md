## 担心中共军事影响力，美国拟派团游说印太三国

![](https://assets.gnews.org/wp-content/uploads/2022/04/image001-2.jpg)图片来源：澳大利亚人报
4月18日，《新闻极限》引述路透社的文章内容，报道说白宫在周一表示，由于担心所罗门群岛正在与中国达成安全协议，白宫印太协调员库尔特·坎贝尔（Kurt Campbell）和国务院负责亚洲事务的最高官员本周将前往所罗门群岛。

白宫国家安全委员会在一份声明中说，坎贝尔和负责东亚及太平洋事务的助理国务卿丹尼尔·克里滕布林克（Daniel Kritenbrink）将率领一个代表团访问三个国家，分别是所罗门群岛、斐济和巴布亚新几内亚，代表团包括国防部和美国国际开发署官员。

该代表团还将在夏威夷停留，以“与美国印太司令部的高级军事官员和区域合作伙伴进行磋商”，声明中没有给出旅行的具体日期。

对此，国务院发言人内德·普莱斯（Ned Price）说，对中国问题的关切将是与所罗门群岛官员讨论的主题之一。在例行新闻发布会上，他说，美国对该地区的政策是确保各国了解与华盛顿接触的好处，“而不是和中国或任何其他国。 我们将让他们把我们提供的东西与其他国家，包括该地区相当大的国家（指中国），可能提供的东西进行对比”。

报道说，2月份，作为拜登政府努力向印太地区投入更多外交和安全资源，以对抗中国争取更大影响力的努力的一部分，美国宣布它将在所罗门群岛开设大使馆。

然而，3月份，所罗门群岛方面说，它正在与中国建立伙伴关系，以应对安全威胁并确保安全的投资环境，这将是北京在美国盟友澳大利亚和新西兰几十年来一直视为其“后院”地区的一个重大突破性进展。

文章指出，虽然在一个地区性的反弹之后，所罗门群岛说它不会允许在那里建立中国的军事基地，然而，普莱斯表示担心，因为与中国拟议协议的广泛性质为在所罗门群岛部署中国军队敞开了大门。

普莱斯说，“我们认为，签署这样一项协议可能会增加所罗门群岛内部的不稳定因素，并将在更广泛的太平洋岛屿地区树立一个令人担忧的先例。”

针对中国的意图，澳大利亚国防部长彼得·达顿（Peter Dutton）说，中国希望在太平洋岛屿上获得一个军事立足点，包括在巴布亚新几内亚建立一个“军港”。 报道说，2018年，中国提出在巴布亚新几内亚重新开发一个海军基地，但与澳大利亚最接近的这个北方邻国最终决定由澳大利亚来开发这个基地。
