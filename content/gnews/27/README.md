## 河南发生学生集体食物中毒 校长曝教育局垄断配餐
` GTV新闻访谈` [轉載自GNews](https://gnews.org/zh-hans/1705640/)

原作者：澳洲天好蓝
![This image has an empty alt attribute; its file name is Screen-Shot-2021-08-30-at-7.37.59-PM.png](https://assets.gnews.org/wp-content/uploads/2021/08/Screen-Shot-2021-08-30-at-7.37.59-PM.png)
北京时间11月27日，中共国媒体曝出，在河南省封丘县越岗镇戚城中学，学生食用学校提供的食品后出现了呕吐和腹泻现象。该事故发生在23日，但是中共负责监督的官员试图掩盖，直至27日才被媒体曝出。
![](https://assets.gnews.org/wp-content/uploads/2021/11/屏幕快照-2021-11-29-下午11.02.35.png)
11月23日该学校校长已经接受了媒体采访，他面对镜头痛哭流涕，曝光出学校的送餐公司由教育局统一招标，自己无权干涉。

据媒体报道，该县教育体育局早前推出“学生营养改善计划午餐采购项目”，预算金额3年8300多万元人民币。中标结果公布后，曾有另一家公司对招标进行投诉，被县财政局以“缺乏事实依据”等理由驳回。
