上海4月9日新增本土新冠病例高达24,943例，自4月7日首度单日突破2万例以来，连三日创新高纪录。上海副市长宗明于4月9日的疫情防控工作新闻发布会上表示，“动态清零”的总方针不动摇，并将开展二次全员核酸检测，且实施分区分级的差异化防控，依风险程度采阶梯式分级管理，将上海划分为“封控区”、“管控区”及“防范区”等三区。

![](https://assets.gnews.org/wp-content/uploads/2022/04/图片1_ccexpress-2.jpeg)中国官媒新华社发布的照片显示，第一批上海感染新冠病毒的人康复后从上海新国际博览中心改建的方舱医院出院。（2022年4月9日）

根据官方统计，上海本波疫情自3月1日以来，已累计超过18万例感染者，当局正加快脚步，将上海国家会展中心改建为上海最大的方舱医院，预计可提供5万张床位。

在全球多数国家都已改采与病毒共存的防疫思维后，中国仍坚持“清零”政策和严厉的封控措施。上海此波疫情是继武汉两年前首度爆发新冠病毒以来，封控规模最大的一次。目前上海2600万市民足不出户，面对民生物资供应不稳和防疫乱象百出，市民怨声四起。另外，全面封城期间，极需透析或化疗等病人的医疗照护需求受到排挤，使其生命安全陷入险境。种种的次生灾害已经远超过奥密克戎(Omicron)病毒本身所带来的伤害，让上海市民的容忍度消耗殆尽。

针对上海的防疫成效，位于纽约的独立学者虞平表示，上海不乏一流的医学专家，许多专家早已就肆虐上海的奥密克戎病毒，做出“不封城”、针对污染源“切块式”、“网格化”管理等符合科学的“精准防控”建议，但中共政治体制僵化，并以少数领导班子个人的意志为行事方针，导致为了清零，付出极大的封城代价。

对于此，新中国联邦创始人郭文贵先生在文贵大直播中透露，针对一线城市的所谓病毒“清零”政策，既是中共内部斗争的需要：让中国老百姓忆苦思甜，感激中共的一种措施，也是中共预备发动对台战争时，进行的社会压力测试。不仅上海，之前的深圳以及后面的北京、广州等城市也会和上海一样的遭遇。中国共产党是全体中国人民灾难的根源，只有消灭中国共产党才能拯救全体中国人民。中共不灭，灾难不止！
