## 西安智飞疫苗让西安“忍饥挨饿”
` wenwu` [轉載自GNews](https://gnews.org/zh-hans/1808002/)

编译：西班牙巴塞罗那喜悦农场-wenwu

![](https://assets.gnews.org/wp-content/uploads/2021/12/image-1937.png)

中国最大城市之一的西安居民在严格疫情规则下表示，他们在（当地政府）被禁止外出获取食物后，面临饥饿；管理西安市的官员周一告诉该市 1300 万居民，他们只有在被邀请参加新一轮大规模 Covid-19 测试时，或在出现医疗紧急情况时，才可以从家里出来（为国家掏钱）。

在（西安）采取严厉措施之前，每户人家每两天可以有一名成员外出购买食物（英国的第一份官方报告显示 Omicron 住院的风险比 Delta 低50%到70%）；英国卫生官员一再表示，中共病毒COVID加强型疫苗可以防止Omicron，并提供了度过病毒大流行的最佳机会。

由于中共国继续实行“零感染”政策，西安在周二记录了175个新病例（西安居民是否会被饿死，根据的是伪政府官员的一句话）。

靠近西安的城市也记录了与疫情爆发有关的病例，距离西安约185英里的延安在周二命令私人企业关闭，并告诉一个地区的数十万人留在室内（可见会有更多的“西安模式的挨饿”出现）。

“挨饿”

![](https://assets.gnews.org/wp-content/uploads/2021/12/image-1935.png)

西安居民在社交媒体上呼救，说他们”快饿死了”，要求邻居为他们提供物资。

“我快要被饿死了，”一个人在微博上写道，微博相当于中共国的 Facebook。”没有食物，我的住房大院不让我出去，我的方便面也快吃完了……请帮助！”

“我不想再听到任何关于一切都很好的说法，”另一个人说。”即使物资如此丰富又如何–如果你不真正把它们给人们，它们就没有用。”

据称，西安目前中共病毒（COVID-19）病例的增加是由 Detal 变体导致的，并被认为与一周前的巴基斯坦旅行有关。

上周四，当大规模疫情检测发现一个病例逃脱了隔离区，并传播了中共病毒时（从来没有停止传播过），西安市采取了封锁措施。

本月迄今为止，西安共报告了810例中共病毒（COVID-19）病例，这是该病毒在武汉出现后中共国（官方报道）最大的一次疫情爆发。

这是自2020年初武汉市被封锁以来，中共国最大的一次人口封锁，影响到1100万人。

城市工人在周日被派出去对公共场所进行消毒，当地人被警告在化学品有时间消散之前，不要接触任何东西。

大规模疫情测试

周一晚上，随着第五轮大规模疫情检测的进行，限制措施进一步收紧：到目前为止，检测结果显示病例分布在整个城市和各个年龄段，这引发了人们对实际病例可能远远高于检测结果的担忧；任何被发现违反车辆通行规定的人都会被警方拘留10天，并被罚款500元（78美元）；在进行大规模疫情检测时，任何不遵守规定的人也将面临被拘留和罚款。

香港大学病毒学家金冬雁说，鉴于人们从室外表面或空气中感染 COVID-19 的风险很低，因此对公共场所进行大规模消毒似乎没有必要。

他说：“这是用大炮打蚊子；他认为室内表面的消毒是必要的，特别是在被感染者访问的地方。”

除了西安所在的陕西省之外，广西地区以及浙江、广东和四川等省也发现了新增感染（其实一直都在感染，没有停过）。

虽然中共国已经记录了Omicron变体的病例，它被认为比Delta的传染性强得多，但它们都与旅行有关（全是境外输入），而且尚未在国内爆发最新变体。

人们认为中共国将继续其”零感染”政策，将至少到2022年2月在北京举行的冬奥会结束。
