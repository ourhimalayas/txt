#### 【雅典娜快讯】，紧贴时事脉搏

- ![](https://assets.gnews.org/wp-content/uploads/2022/01/F-16V_1.jpeg)图片来源：theaviationist.com


编译：小红帽

根据《DW新闻》1月12日报道，台湾立法院周二（1月11日）通过海空战力提升特别预算，经费约为新台币2369.5亿元 (546亿元人民币)。据悉，朝野对该预算有高度共识，目的是要因应中国政府对台湾的威胁。

然而，就在该预算案通过后几个小时，一架在台湾西部外海训练的F-16V战机坠海。战机飞行员目前仍然失联。这是台湾F-16V机队改装升级后的首次重大事故。

台湾空军周二紧急召开记者会表示，嘉义基地一架F-16V单座战机于下午2时55分，由飞行员陈奕上尉驾机执行例行性训练任务。该战机在雷达上的光点于3时23分消失，即起飞约半小时后失联。

F-16V战机失事的前几天，台湾军方在嘉义向媒体示范多架次紧急升空，并展示12架F-16V战机在跑道列队，其中失联的编号6650的F-16V战机当时也参与演示。

台湾空军督察长柳惠千表示，战机当时正执行对地炸射科目、进行大角度导入时快速坠入海中，并称当没有人看到飞行员跳伞，不排除是机务、人为或天气等因素导致，当局已组成专案小组调查中。
