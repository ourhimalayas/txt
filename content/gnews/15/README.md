## 恒大、佳兆业债务违约导致信用评级下调
` GTV新闻访谈` [轉載自GNews](https://gnews.org/zh-hans/1741575/)

作者：日本方舟农场 Andy5
![This image has an empty alt attribute; its file name is Screen-Shot-2021-08-30-at-7.37.59-PM.png](https://assets.gnews.org/wp-content/uploads/2021/08/Screen-Shot-2021-08-30-at-7.37.59-PM.png)
当地时间12月9日，惠誉评级宣布，中共恒大集团和佳兆业集团目前处于“限制性违约”状态。
![](https://assets.gnews.org/wp-content/uploads/2021/12/屏幕快照-2021-12-11-上午11.16.31.png)
据了解，恒大和佳兆业此前均未能如期支付已经到期的债务，为此，评级机构下调了二者的信用评级，并首次将恒大集团的评级降至“RD”状态，而这也标志着全球负债最多的房地产公司——恒大正式进入违约状态。

相关报道提及，恒大和佳兆业本应于12月7日分别偿还到期的八千两百四十九万美元和四亿美元的利息和债券，但两家公司在超过期限后非但没有还款，甚至没有做出任何公告，也没有回应惠誉的请款要求。

对此，有日本媒体指出，恒大和佳兆业的行为可能会加速惠誉提出立即还款要求的时间，或将导致本已周转极为困难的两家企业在资金方面进一步雪上加霜，进入恶性循环。

另外，穆迪和标普全球评级虽然尚未正式宣布恒大和佳兆业的违约，但穆迪投资者服务公司也在当天表示，最近几个月里，中共房地产开发商的在岸以及离岸债券市场都变得非常不稳定，同时，其他领域的中共企业也或多或少地面临着同样的债务违约困境。

另有评论指出，中共房地产爆雷所带来的中共经济崩溃或已迫在眉睫。
