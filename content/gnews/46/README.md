## 反华危机｜“大翻译运动”被揭刻意撷取极端言论 妖魔化中国人 煽动反华情绪

**发布时间｜17/04/2022**
![](https://assets.gnews.org/wp-content/uploads/2022/04/「大翻譯運動」被揭刻意擷取極端言論-妖魔化中國人-煽動反華情緒.jpg)

[点击关注官方盖特](https://www.gettr.com/user/hktwfrontline)


外媒报导，自俄乌战争爆发以来，社交媒体Twitter上忽然开始兴起“大翻译运动”（The Great Translation Movement），有一批匿名用户在中国大陆地区的微博或微信上刻意挑选出一些极端言论或亲俄报导，将它们翻译成英文、日文并发布到Twitter上进行广泛传播，旨在向西方世界营造出一种中国人普遍“亲俄贬乌”的极端立场。

报导指出，虽然该Twitter帐号的其中一名管理员对外媒表示，“大翻译运动”旨在揭露中国人的真正立场，提高国际对中国舆论状况的认识，但透过那些被截图翻译的内容可以发现，他们只会挑选某一部分中国人的偏激言论，并将之放大至辐射到全体中国人身上。

比如，撷取某中共军事博客“杜撰”的报导（乌克兰自导自演攻击东部城市的克拉莫托斯克（Kramatorsk）火车站并嫁祸俄罗斯），并翻译留言区一些过激的评论；撷取某个播客拍摄的一条对乌克兰女性使用侮辱性词汇的视频，将部分网友发表的低级留言翻译出来，但却没有将大量批评该播客言论失当的留言同步呈现出来。

“大翻译运动”官方Twitter于今年3月正式运营至今，已吸引超过11.9万人追踪，目前人数还在持续增加中。有外国媒体专家批评认为“大翻译运动”的立场过于偏颇，有刻意抹黑中国人形象，煽动反华情绪之嫌。

评论指出，鉴于中国社交平台均受中共严格审查，平台上极端民族主义声音向来高涨，而自由意志则被大幅压制，因此能被看到的内容并不能全面展现出中国公众舆论的整体观点。而这一点，相信同在中国出生并成长的“大翻译运动”的管理员们更加清楚，因此猜测行为背后或别有用心。
