自1983年以来，每年7月，一小群媒体大亨、科技巨头、投资者、政治家和情报机构的内部人士，都会聚集在爱达荷州的太阳谷小镇，举行为期一周的会议，就主流媒体、社会媒体和新兴通信技术的政策达成共识。

这基本上就像媒体的比尔德伯格会议，由于Facebook、Twitter、苹果和YouTube等科技公司已经成为媒体行业的主要参与者，他们每年都会聚集在太阳谷，试图确保没有新兴平台能够威胁到他们的权力…
