2. 中共勒令下架滴滴出行应用软件

新闻：7月4日，国家互联网信息办公室（以下简称“国家网信办”）发布通知称，“滴滴出行”App存在严重违法违规收集使用个人信息问题。国家网信办依据《中华人民共和国网络安全法》相关规定，通知应用商店下架“滴滴出行”App。

战友评论：滴滴刚在纽交所上市5天，这短短的5天之内，经历了过山车般的风暴。起初，中共网络安全审查办公室7月2日做出对滴滴出行的网络安全审查决定，并停止新用户注册后，滴滴股价立即下跌超过10%，之后在周末，又勒令苹果商店和谷歌的应用商店下架滴滴出行的应用软件，可见中共对滴滴这次的封杀是彻底的。

但回忆滴滴上市到这次被制裁，整个过程也是疑点重重。首先滴滴在经历前两次上市被拒绝后，此次的上市过程异常匆忙，无敲钟，无庆功，距离理想的招股价格差了将近三分之一，可见有部分资本想急于通过上市来套现的意愿非常强烈。

但中共对滴滴的制裁足以见得，北京当局对滴滴的此次上市十分不满，或者说中共内部的利益团体角逐又上了新的台阶，滴滴背后的资本主要有软银、红杉、阿里、腾讯等，这些白手套背后的势力是否与一尊正在全面冲突中，我们不得而知，但在中共治下的民营企业却是被牺牲的棋子，这点毋庸置疑。
