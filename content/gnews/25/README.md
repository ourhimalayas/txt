## 简析中共国猪肉供应情况及与外汇储备的关系

东京樱花团/满天樱花
![](https://assets.gnews.org/wp-content/uploads/2021/11/2-129.jpg)[图片来源](https://finance.sina.com.cn/roll/2020-01-17/doc-iihnzhha3069790.shtml?source=cj&amp;dv=1)
官方数据：2016-2020年中国猪肉消费量稳定在5500万吨左右。

假设小孩和老人不吃猪肉，目前官方人口统计16-59岁为64.9% （90199万人），按照真实数据10/14=0.71计算，16岁—59岁的人为6.4亿人，其中宗教信仰原因4000万元加上本身不吃猪肉的按3%计算，共5800万人不消费猪肉，总共5.8亿人消费猪肉，每年550亿斤猪肉，平均每人年消费猪肉94斤，每天消费猪肉0.26斤。

感觉和生活中习惯基本相符合，因此**笔者认为年消费量5500万吨具有一定的参考性**。

2019年猪肉产量4255 万吨，猪肉消费量3793万吨，进口猪肉210万吨，累计进口金额323亿元。出口2万吨，累计出口金额9.8亿元。

出口价格明显高于进口价格，**笔者认为可能是出口深加工产品如罐头，香肠等同时有部分是高品质猪肉出口**。

2020年猪肉产量4113万吨，消费猪肉5451.8万吨，猪肉累计进口量达439.22万吨，同比增加108.34%，累计进口额达120.41亿美元，同比增加157.59%，平均到岸价格2742美元/吨，同比上涨23.64%。

通过核算，2019年的数据和2020年的数据衔接没有问题。2020年的消费量5451.8万元高于2020年的猪肉产量4113万吨1338.8万元，缺口部分由2020年进口439.22万吨和2019年的剩余672万吨合计1111.22万吨基本补足，不足部分227.58万吨笔者全部算为由猪肉储备补齐。

通过2年数据可以得出结论：

**即使在有猪瘟和猪肉价格高位运行这两种极端影响猪肉消费的情况下，猪肉的消费需求依然可以达到近4000万吨/年**。

经过查询网上数据，猪肉储备数据为国家机密，网上有业内人士估计，中央冻猪肉最大储备量不会超过60万吨，地方和企业的储备量最大可能是100万吨。笔者从谨慎性原则出发，估算猪肉储备具有250万吨的能力，本文假定永远有250吨的猪肉在储备状态，相对于整体需求也只是个安慰作用。

2021年前三季度猪肉产量3917万吨，季产量为1305万吨，预计全年可以突破5000万吨。1—10月份，猪肉进口累计334万吨，进口猪肉呈现连月下跌的态势，自7月份进口猪肉35万吨后，进口猪肉持续减量，10月猪肉进口仅有20万吨，全年保守估计进口猪肉350万吨。

2019 年生猪出栏量 5.44 亿头，生猪存栏量 2.25 亿头。

2020年全国生猪出栏5.27亿头，生猪存栏4.06亿头，能繁殖母猪存栏4161万头。

2021年前三季度生猪出栏4.9亿头，生猪存栏量4.3亿头，三季度末能繁母猪存栏4459万头。

由数据分析得出：

**1****、在当前猪肉价格低位徘徊，消费者随心消费猪肉的情况下，可以算出猪的出栏数量在6-6.5亿头可以满足全国需求**。

**2****、目前猪肉供应稳定，明年乃至后年的猪肉供应有保证**。

**3****、以前共识是7亿头猪可以满足全国供应，现在只需要6-6.5亿头就可以保证，说明每头猪养的更大了**。

综上所述，在目前资本大量进入养猪行业后，养猪行业的规模得到较大的提升，猪的出产周期得到进一步缩短，猪的出肉率也有一定的增加，**中国猪肉供应目前看来较为充足，近2年出现紧缺的可能性较小**。

但是因为种猪和饲料基本要依靠外汇，因此**人民币的稳定对于猪肉供应至关重要，所以即使现在猪肉供应如此强劲，一旦外汇产生剧烈波动，养猪成本将急剧增加，养猪企业受金融风暴的影响资金将不能正常运转，猪肉供应就会瞬间埂塞**，请战友紧跟时事变化，在灾难快来时提前做好准备。
