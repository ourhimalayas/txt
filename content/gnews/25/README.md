中东近日再次陷入乱局，接连发生袭击事件。新中国联邦奠基者郭文贵先生一针见血地指出：在中东点燃战火制造混乱，是中共和普京合演的“围魏救赵”把戏。

郭文贵先生几日前刚在盖特上发出“中共要在中东制造事端”的警告。话音刚落，中东便点燃了战火——伊朗对伊拉克库尔德斯坦地区进行了导弹袭击；胡塞武装则对沙特阿美在利雅得的炼油厂发动了袭击。

3月13日，郭文贵先生在“文贵大直播”节目中指出，共产党在中东点燃战火，是中共总书记习近平与俄罗斯总统普京一早做出的计划。

该计划是：在普京发动对乌克兰的侵略战争后，共产党配合普京截击美国和西方的金融制裁。而几个邪恶轴心国——中共国、俄罗斯、伊朗、北朝鲜，以及中东几个邪恶小国，在中东点燃战火，把中东的整个世界能源供给的所有基础给砸烂，以达到“围魏救赵”的目的。

何为“围魏救赵”？郭先生解释说，就是一旦西方制裁俄罗斯，将其踢出SWIFT系统，或日后对共产党做出SWIFT制裁，以及美元与中共国的人民币和香港的港币脱钩，影响中共攻打台湾时，就实施搞乱中东计划，让西方整个石油系统崩溃。

郭先生进一步指出，美国虽然是石油的净出口国，但石油的价格影响着美国的经济和就业。而整个欧洲是完全依赖于石油和能源的进口供应。欧洲一旦没有了石油和天然气，就会在俄罗斯的事情上做出妥协，就得继续买俄罗斯的石油，并且也会在金融上妥协，停止对俄罗斯进行SWIFT制裁。而中共，则可以趁机攻打台湾了。这就叫在中东开展“围魏救赵”。

郭先生最后还发出警示称，接下来马上就会看到，在叙利亚、沙特、伊拉克等地，在美军基地，美军会遭受到超乎想象的打击，甚至还有可能会再次被投毒。
