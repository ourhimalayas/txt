美国《蒙大拿标准报》4月13日转发美联社乌克兰基辅报道，位于俄罗斯门口的四个国家的总统周三访问了乌克兰，并强调他们对乌克兰陷入战争困境的支持，在那里他们看到了严重受损的建筑物，并要求俄罗斯为俄罗斯军队犯下的战争罪行负责。

波兰、立陶宛、拉脱维亚和爱沙尼亚总统的访问强烈表明了北约东翼国家的团结，其中三个国家与乌克兰一样曾经是苏联的一部分。这些领导人乘坐火车前往乌克兰首都基辅，与他们的同行沃洛基米尔·泽伦斯基(Volodymyr Zelenskyy)会面，并访问了附近的一个城镇博罗江卡，在俄罗斯军队撤集中到该国东部后，那里发现了俄军暴行的证据。

立陶宛总统吉塔纳斯·瑙塞达(Gitanas Nauseda)说：“这里正在为欧洲的未来而战。”他呼吁对俄罗斯进行更严厉的制裁，包括对俄罗斯的石油和天然气运输以及该国所有的银行进行制裁。

据西方国家的官员称，俄罗斯于2月24日入侵乌克兰，其目的是占领基辅，推翻现政府并建立一个亲俄的政府。但地面推进慢慢停滞不前，俄罗斯可能损失了数千名战士。这场冲突造成了无数乌克兰平民死亡，并迫使数百万人逃离。它还扰乱了世界经济，威胁到全球粮食供应，打破了欧洲在冷战后的平衡。

周三，瑙塞达、爱沙尼亚总统阿拉尔·卡里斯(Alar Karis)、波兰总统安杰伊·杜达(Andrzej Duda)和拉脱维亚总统埃吉尔斯·莱维茨(Egils Levits) 与泽伦斯基一起出现在基辅历史悠久的马林斯基宫(Mariinskyi Palace)的一个华丽的房间里，他们重申了在政治上和军事援助上支持乌克兰的承诺。

杜达说：“我们知道这段历史。我们知道俄罗斯的占领意味着什么。我们知道俄罗斯的恐怖主义意味着什么。”他补充说，那些犯下战争罪的人和下达命令的人都应该被追究责任。

他说：“如果有人派飞机，如果有人派军队去轰炸居民区，杀害平民，谋杀他们，这就不是战争。这是残暴，这是强盗行为，这是恐怖主义。”

泽伦斯基周三晚些时候说：“俄罗斯军队一定会被追究责任。我们将把所有人都拖到法庭上，不仅仅是因为他们在布查所犯下的罪行。”他还说，清除俄罗斯人离开后在乌克兰北部留下的数万枚未爆炸的炮弹、地雷和绊线的工作正在继续。他敦促返回家园的人们对任何陌生的物体保持警惕，并向警方报告。

同样在周三，欧洲安全与合作组织(Organization for Security and Cooperation in Europe)委托撰写的一份报告发现，“俄罗斯军队在进行战争行动时有明显违反（国际人道主义法）的行为”。该报告由乌克兰挑选的专家撰写，并由总部设在维也纳的促进安全和人权的组织出版。

该报告说，乌克兰也有违法行为，但结论是俄罗斯实施的违法行为“在规模上和性质上要大得多”。乌克兰此前承认，可能存在“个别的违法行为事件”，并表示将进行调查。

俄罗斯军队现在正准备在东部的顿巴斯地区发动一场大规模进攻，自2014年以来，与莫斯科结盟的分离主义分子一直和乌克兰军队在那里作战，而俄罗斯已经承认分离主义分子的独立主张。

俄罗斯战役的一个关键部分是马里乌波尔，它位于顿巴斯地区，自战争开始以来，俄罗斯人几乎一直在对它进行攻击。乌克兰官员一直在调查一项指控，即一架俄罗斯无人机在马里乌波尔投放了有毒物质。
