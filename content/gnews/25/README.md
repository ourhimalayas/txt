全文如下：

据最新的联邦外交机构披露，过去两年，中国共产党向华盛顿特区的一家广播电台支付了440万美元，用于广播宣传。

总部位于弗吉尼亚州的“波托马克传媒集团”在上周四（12月16日）向司法部提交的文件中，详细说明了其与中共的“国际传播规划局”（International Communication Planning Bureau）签订的利润丰厚的合同。作为交易的一部分，波托马克传媒的WCRW，一家AM广播电台，播放来自“中共环球电视网”（CGTN）的内容，以及一系列宣传中共国的脱口秀节目。

“波托马克媒体”向司法部提交的文件提供了其与中共宣传部下属的“国际传播规划局”安排的节目的大量细节，中共的规划局可以根据合同审查广播和核实播出的节目。“波托马克媒体”被要求向规划局提供有关受众范围、反馈和“国际组织评估”的报告。

北京与一家AM广播电台的合作突出了其在美国的宣传活动范围，近年来，随着中共试图转移对其侵犯人权行为的注意力，在美国的宣传活动已变得过于活跃。中共的国营媒体机构通过社交媒体以及与美国报纸和杂志的出版协议，积极向美国观众宣传其内容。国营报纸《中国日报》向《时代周刊》、《外交政策》和《华尔街日报》支付了数百万美元为了在其网上发表文章，中共驻纽约领事馆最近聘请了一家公关公司，招聘社交媒体上有影响力人士以宣传北京奥运会。

440万美元的支付，涵盖了从2019年7月到2021年8月，为广播CGTN内容，社交媒体推广，以及脱口秀节目《桥》（The Bridge）的制作。

“波托马克媒体”作为外国代理人注册之际，正值司法部近年来加大了外国代理法的执法力度，要求中共国、俄罗斯和土耳其的国有媒体公司向政府注册其活动。司法部于2019年2月强制CGTN注册为外国代理人。

WCRW自1992年起播放了国营的“中共国际广播电台”（China Radio International）的内容，但之前没有在司法部注册为中共国的外国代理人。俄罗斯卫星通讯社向总部位于佛罗里达州的RM广播公司支付了140多万美元，在华盛顿特区和其他城市的AM广播电台进行广播。

“波托马克传媒集团”没有回应置评请求。
