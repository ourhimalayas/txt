## 美中共病毒感染两周内增二倍 政府归咎于疫苗接种率低

#### 编译：脐橙君
校对：萌萌的朋克
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/07/21212431/%E6%8E%A5%E7%A7%8D.jpg)图片来源：网络截图
在疫苗错误信息的猛烈冲击下，美国本土的中共病毒病例在两周内几乎增加了两倍，这让医院不堪重负，让医生筋疲力尽，并使神职人员陷入困境。

根据约翰霍普金斯大学的数据，在美国各地，过去两周每日新增病例的7天平均值从7 月6日的不到13,700上升至周二的37,000以上。卫生官员将责任推给delta变体病毒和疫苗接种率下降。根据疾病控制和预防中心的数据，只有 56.2% 的美国人至少接种了一剂疫苗。

在路易斯安那州，卫生官员周三报告了5,388例新的中共病毒病例，这是自2020年初大流行开始以来的每日病例数的第三高数值。全州因该病住院的人数上升至844人，自6月中旬以来增加了600多人。新奥尔良的政府官员敦促人们在室内恢复戴口罩。

犹他州报告有295人因该病毒住院，这是自2月以来的最高数字。过去一周，该州平均每天约有622例确诊病例，大约是6月初感染率最低点数字的三倍。同时，政府也将这种激增与未接种疫苗扯上关系。

根据皮尤研究中心 2019 年的一份报告，白人福音派新教徒对疫苗接种的反对尤其强烈，现在，如何让信仰社区发挥疫苗的主观能动性，是如何提高疫苗接种率的方法之一。

纽约市市长比尔·德·布拉西奥 (Bill de Blasio) 周三表示，该市每天接种的疫苗剂量已从4月初的10万多支高峰下降到1.8万支以下。但与此同时，该市的病例数数周以来一直在上升，卫生官员表示，在他们测序的10个病例中，Delta变异约占7个。

可见，面对美国激增的病例，政府将责任推给了未能积极接种疫苗，这与拜登政府推出的所谓的“疫苗上门”的概念是一致的。媒体对疫苗给健康人士带来的危害闭口不提，反而将感染上升与疫苗挂钩，并用“社会责任”来进行道德绑架，这样的宣传对于降低感染数没有丝毫帮助，相反会激起“已接种群体”对“未接种群体”的攻击。

根据昨天塔克卡森节目的观点，英国有60%的住院病例是完全接种疫苗的，那么既然疫苗如此有效的话，为何还会有如此高比例的接种人群不得不住院治疗？每个人在选择接种之前，应该遵循其个人意愿，但政府和主流媒体的责任至少是将疫苗可能产生的危害真实无误地告知民众，这才是正常人性的做法。
