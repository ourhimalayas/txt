## 与中共国的动荡关系成为澳大利亚选前首场辩论的焦点

![](https://assets.gnews.org/wp-content/uploads/2021/12/michael-4.jpg)

在布里斯班刚刚结束的5月21日选举前的首次领导人辩论中，澳大利亚总理斯科特·莫里森（Scott Morrison）和他的反对派竞争对手中左翼工党领袖安东尼·艾博年（Anthony Albanese）就该国与中共国的动荡关系发生了激烈辩论。

![](https://assets.gnews.org/wp-content/uploads/2022/04/r0_450_5472_3539_w1200_h678_fmax.jpg)

最近几个月，工党在大多数民意调查中一直领先于莫里森的保守派政府。在所罗门群岛与中共国签署安全协议后，与中共的外交政策成为今天双方辩论的焦点。

今天，所罗门群岛总理马纳西·索加瓦雷（Manasseh Sogavare）证实，他的政府已与中共国签署了一项新的安全协议，该安全协议允许中共国向所罗门群岛派遣警察和军事人员“协助维持社会秩序”，同时也为中共国军舰在该国港口停靠“后勤补给”打开了大门，这引发了人们对中共在澳大利亚和新西兰门口可能建立海军基地的担忧。

澳大利亚曾敦促所罗门群岛不要签署该协议，并于上周派出国际发展和太平洋部长塞德·塞塞利亚（Sed Seselja）与索加瓦雷进行会谈。美国国家安全委员会印太事务协调员库尔特·坎贝尔和负责东亚和太平洋事务的助理国务卿丹尼尔·克里滕布林克两名美国高级官员也将于本周访问该国进行会谈。

在中共国宣布签署协议后，塞塞利杰和澳大利亚外交部长玛丽斯﹒佩恩在一份联合声明中表示，他们“深感失望”，担心该协议的制定缺乏透明度，并指出它有可能破坏地区的稳定，并希望索加瓦雷声明“所罗门群岛永远不会被用作外国势力的军事基地或其他军事机构”。密克罗尼西亚、其他太平洋岛国以及所罗门群岛的主要反对党也对该协议表示担忧。

马纳西·索加瓦雷为安全协议辩护称，该协议完全针对“我们的内部安全局势”，这是对2017年与澳大利亚达成的安全协议的补充，这不会像反对派和包括美国和澳大利亚在内的国家所担心的那样“破坏我们地区的和平与和谐”。

在今天进行的选前首场辩论中，与中共国的关系成为辩论最激烈的焦点。

安东尼·艾博年指责澳大利亚总理斯科特·莫里森的保守派政府在太平洋地区犯下了二战以来最大的外交政策失误，该协议引发了对中共国在距离澳大利亚东北海岸仅2,000公里（1,240 英里）的所罗门群岛建立海军基地的担忧。意指莫里森政府未能阻止所罗门群岛与中共的交易，标志着政府的外交政策失败。

![](https://assets.gnews.org/wp-content/uploads/2022/04/1000-2.jpeg)

莫里森说，新的威胁应该归咎于中共国，而不是澳大利亚，澳大利亚是所罗门群岛的主要安全伙伴。工党说这是澳大利亚的错，而不是承认中共国干涉该地区，这很奇怪。“这是一个非常严重的问题，但我今天发现有趣的是……工党说这是政策失误。不，发生这种情况是因为中共国正在寻求干涉太平洋”。

莫里森反问艾博年，“我不明白的是，当这种重大的事情发生时，你为什么要站在中共国一边？”

安东尼·艾博年将斯科特·莫里森这一反问描述为“令人发指的诽谤”，指责莫里森将国家安全问题政治化。称“这并不是太平洋地区的进步，这是太平洋的东西，这是澳大利亚在外交政策上的一次重大失败，事实上，我们都知道中共国已经改变了，中共国已经改变了姿态。它更具侵略性，在该地区更加活跃，我们需要了解这一点并做出回应。”

这位工党领袖指的是澳大利亚加强与岛屿邻国接触的政策，保守派政府将其描述为太平洋地区的进步。

斯科特·莫里森辩称，工党在中共国问题上软弱无力，并认为工党政府会屈从于中共国的经济胁迫。

今年早些时候，莫里森曾声称，近年来，随着中共国对澳大利亚实施官方和非官方贸易制裁，北京希望工党在选举中获胜。一个保守的游说团体在卡车上投放广告，描绘中共国国家主席习近平投票支持工党。

2014年，工党因挫败莫里森政府与中共国签署引渡条约的计划而受到赞誉。此后澳中双边关系恶化，莫里森政府警告，如果澳大利亚人到中共国旅行，他们将面临被任意拘留的风险。

莫里森在辩论中表示，工党指责他对中共国“过于强势”，并向其他世界领导人指出中共国正试图胁迫澳大利亚，他还因要求对冠状病毒的起源进行调查以及在大流行初期禁止从中共国起飞的航班入境而被指责为种族主义。

无论如何，中共威胁已经成为双方争论的焦点！
