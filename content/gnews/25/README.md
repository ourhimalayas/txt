- ![](https://assets.gnews.org/wp-content/uploads/2022/01/加拿大一父亲痛诉辉瑞疫苗害死其17岁儿子.png)图片来源：FT中文网


加拿大多伦多一位17岁孩子的父亲丹·哈特曼（Dan Hartman），上周在多伦多市议会作证说，辉瑞公司的新冠疫苗杀死了他的儿子。虽然这位父亲确信他儿子死于新冠疫苗，但该案仍被报道为“不确定”。

听证会面向公众开放，多位发言人分享了他们对当地政府应对冠状病毒大流行的看法。哈特曼在会议上公开谈及因辉瑞疫苗强烈副作用而死的儿子肖恩。

这位父亲在他的证词中说：“我儿子从8岁起就开始打曲棍球……因为疫情的原因他休息了一年，他坐在卧室里感到非常无聊，所以决定今年再去打他喜爱的曲棍球，为此他必须接种疫苗才能继续这项体育爱好。”

为了应对疫情，安大略省的许多曲棍球俱乐部要求所有希望参加此项运动的人士接种新冠疫苗，所有12岁及以上的运动员都被要求提供疫苗接种证明才能参加。

2021年8月25日，肖恩接种了第一针辉瑞疫苗。在接种后，这名少年立刻开始出现了疫苗引起的副作用。他的父亲说：“他的脖子和脸上布满了皮疹，眼睛周围泛起了褐色的黑眼圈。”四天后，肖恩被送进医院治疗，后又被送回家，医院只开了布洛芬的处方。哈特曼说：“9月27日早上，他的母亲发现他死在床边的地板上。他只是想打曲棍球而已，却因此丢了性命。”

丹和他的妻子要求进行尸检，试图查清肖恩的死因。然而，尸检并没有提供任何答案，反而出现了更多的问题。

验尸官拒绝对肖恩可能的死因作出裁决，只是将他的死亡列为“未确定”。令人震惊的是，“未确定死因”只在2%的尸检中有这样的结论。有趣的是，尸检确实显示肖恩在他去世时有一个“心脏略微肿大”，但是肖恩没有被诊断出患有任何心脏疾病，而疫苗一直与心肌炎和心包炎等心脏问题有关。

丹·哈特曼想从另一个渠道寻找答案，因此将尸检结果发送给加拿大一位著名的病理学家，这位病理学家因害怕失去工作而希望保持匿名。医生的结论是：新冠疫苗是导致肖恩死亡的最终原因。

他在作证时说：“我内心确信这位病理学家的结论——新冠疫苗杀死了我的儿子。这并不像大家之前认为是罕见的疫苗副作用，它正在普遍地发生。推特上每天都有类似信息，人们因疫苗而死亡。我根本不是一个反疫苗主义者，我真的不是。但我认为这个疫苗确有问题。”

丹后来质疑新冠疫苗的安全性，他也相信疫苗已经杀死了很多人。“从来没有人谈论过死亡，但它正在发生。没有人知道具体数字，因为真实的死因被掩盖，死者的亲人被噤声了。”
