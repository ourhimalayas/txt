资料来源：法新社

自 2019 年以来，朱利安·阿桑奇一直被关押在伦敦的贝尔马什监狱。现在他的一项申请成功了：维基解密创始人被允许与他两个孩子的母亲斯特拉·莫里斯结婚， 甚至可以邀请客人参加他们的婚礼。

在一个拖延多年的案件中，这个决定出人意料地迅速：10 ​​月，朱利安·阿桑奇 (Julian Assange) 申请与未婚妻斯特拉·莫里斯 (Stella Morris) 结婚。现在，不到一个月后，正如《卫报》等英国媒体报道的那样，批准来了。

来自美国的引渡程序正在等待这位 50 岁的维基解密创始人。美国指控他获取和披露国防信息。他创立的平台发布了数十万份关于阿富汗和伊拉克战争的文件，因此据美国称，这危及士兵和特工人员的安全。

作为律师也为阿桑奇辩护的未婚妻斯特拉莫里斯说：“我感到欣慰的是，理性已经获胜，我希望我们的婚姻不会受到进一步的干扰。”这对夫妇此前曾宣布，如果他们的申请未获批准，他们将起诉英国司法部长多米尼克·拉布。
