## 天价罚款后，美团发布3年来最差季报
` GTV新闻访谈` [轉載自GNews](https://gnews.org/zh-hans/1703497/)

作者：MOS – 柳是多水土透
![This image has an empty alt attribute; its file name is Screen-Shot-2021-08-30-at-7.37.59-PM.png](https://assets.gnews.org/wp-content/uploads/2021/08/Screen-Shot-2021-08-30-at-7.37.59-PM.png)
据外媒11月26日消息，从今年4月开始，腾讯投资的美团受到中国市场监管机构反垄断调查，并在上个月以2020年3%营收，即34亿元罚款结束。 本周五提交的季报显示，罚单带来了近三年最大单季亏损，同时美团下调2022年展望。

![](https://assets.gnews.org/wp-content/uploads/2021/11/Slide3-18.jpeg)

美团业务包含餐饮评价，单车共享，现在正面对大陆经济减速带来的挑战。公司CEO王兴告诉分析师，由于中原洪灾和新冠封城对餐饮业的冲击，7-9月核心业务食品快递增速减至29.5% “我们预期今年Q4，而且可能明年前几季度都将受到负面影响”

美团第三季度报亏100亿元人民币，作为对比，去年第三季度则有63亿利润。 美团之前因快递员待遇过低受到批评，其大多数骑手没有基本医疗和社会保障。
