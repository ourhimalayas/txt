台媒1月29日报道，在中共持续打压香港民主自由之际，港共政府授意香港大学，遮盖校园内一幅巨型六四标语牌。

记者当天看到，建筑工人正以金属围板，在港大校园内太古桥上，将写有“冷血屠城烈士英魂不朽，誓歼豺狼民主星火不灭”的，巨型六四标语的行人道封住。这行标语是港大学生，在中共镇压六四民主运动后不久，铺着黑布用白色油漆写上的。30多年来，它一直伫立在校园，提醒民众不要忘记六四。
