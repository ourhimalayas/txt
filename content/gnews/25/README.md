路透社2月9日报道，瑞士银行UBS(瑞银)在阻止中国商人郭文贵先生对其起诉的努力宣告失败，后者于周三将案子成功递交伦敦高级法院并立案。

伦敦法官总结说伦敦瑞银在文贵先生和其他申诉者在法律诉讼中明显参与了有害性的活动。法官说：“根据伦敦瑞银的解除该案的申请，我宣誓这个法庭对索赔人的申诉有管辖。”

该起诉围绕在文贵先生和其他申诉者说瑞银无视他们在有关中国金融机构海通的股票买卖中的权益。
