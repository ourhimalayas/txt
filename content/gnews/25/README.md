![](https://assets.gnews.org/wp-content/uploads/2021/11/53B20E69-A1C5-42A3-8717-0A64FDFA5143.jpeg)（图片来源：网络）

11月18日，美国弗吉尼亚州诺福克法院开庭审理保罗·马里克（Paul Marik）医生起诉圣塔拉医疗集团（Sentara Healthcare）禁止其使用有效安全的药物伊维菌素治疗中共病毒患者的案件。

据《生活新闻》报道，近期，圣塔拉医疗集团告知保罗·马里克医生，不能再对感染上病毒的危重病人进行包括使用伊维菌素在内的一系列治疗，尽管这些治疗方案是高效的。

马里克医生医学经验丰富，是圣塔拉·诺福克总医院重症监护室主任。他曾成功使用包含伊维菌素在内的治疗方案，将重症监护室病毒感染者的死亡人数减少了50%之多。然而，圣塔拉医疗集团发布的禁令却使病人的死亡率急剧上升。

马里克医生表示，不能再眼睁睁地看着病人在没有适当治疗的情况下无谓死去，因此向法院提起诉讼，希望允许他和他的同事使用美国FDA批准的药物及其他治疗方法。他特别强调说，在过去的18个月中，他们用伊维菌素治疗方案挽救了成千上万的中共病毒重症患者。

马里克医生的律师弗雷德·泰勒（Fred Taylor）表示，弗吉尼亚州的法律很明确，主治医生有权使用他们认为合适的治疗方法，而不是由医院来决定。

《生活新闻》的报道称，法院在审理这一案件前不仅收到了“前线重症护理联盟”（FLCCC）提交的支持声明，还收到了来自世界各地的医生对马里克医生的支持申明。

FLCCC联盟成立于2020年3月，该联盟是由一群出版过大量学术论文的世界知名重症监护医生和学者联合成立。该联盟在来自世界各地专职医生的学术支持下，研究开发在疾病各阶段预防和治疗中共病毒的救命方案。该联盟发布的治疗方案I-Mask+预防+伊维菌素是早期在家问诊治疗中共病毒的有效方案，也是解决全球大流行病的一个潜在方案。
