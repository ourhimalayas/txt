## 立陶宛前总理：中共国需要吸取教训

去年，立陶宛退出了所谓的 “17+1 “集团，这是一个由17个东欧和中欧国家与中国合作的论坛。鉴于中共在该地区的众多商业利益，尤其是以基础设施项目为重点的所谓 “一带一路 “倡议，欧洲的任何形式的反击在北京都是不受欢迎的。

然后在11月，立陶宛成为欧洲第一个允许台湾以 “台湾 “名义开设事实上的大使馆的国家。台湾外交部表示，台湾驻维尔纽斯代表处的开设将 “为台湾和立陶宛的双边关系开启一个新的、有希望的进程”。

此举激怒了北京，它认为这是对其 “一个中国 “原则的侮辱，并立即降低了与维尔纽斯的外交关系。立陶宛声称，中共阻止立陶宛商品进入中国，有效地制造了一个贸易壁垒。中共政府一再拒绝这些说法，指责立陶宛损害了中国的 “核心利益”，使双边关系深陷冰点。

台湾的反应是购买立陶宛运往中共国的产品—包括20,400瓶朗姆酒–并承诺向立陶宛工业投资数亿美元，以支持该国面对中共的压力。

这场争吵拉拢了欧盟，欧盟正在支持成员国立陶宛。布鲁塞尔认为北京对待立陶宛的做法是对其他欧盟国家的威胁。

周四，欧盟在世界贸易组织对中共国提起诉讼，指责北京 “对立陶宛的歧视性贸易行为，这也打击了欧盟单一市场的其他出口产品”。

1990年，立陶宛成为苏联第一个宣布脱离莫斯科执政的共产党的成员。随后，它在2004年加入了欧盟的欧洲联盟和北约—这个组织的目的正是要对社会主义的扩张进行制约。

在这种情况下，像中共国这样的国家在自己的地区表现出侵略性，特别是对台湾的侵略，以及利用贸易作为武器对付较小的欧洲国家，自然会使那些记得苏联统治下生活的人感到震惊。

立陶宛前总理安德里乌斯-库比略Andrius Kubilius告诉CNN：“中共国需要吸取教训，因为到目前为止，他们的行为不符合我们的价值观和规则，只是因为他们非常富有而已”。

长期从事人权运动的香港观察组织首席执行官本尼迪克特-罗杰斯说，虽然 “中共国已经显示出它非常善于分而治之，能够把各国玩弄于股掌之间……当各国站在一起，共同反对中共国时，北京的欺凌策略就不那么有效，对中共的压力就能产生更大的影响。”
