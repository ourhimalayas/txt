今届立法会选举首度为内地港人设口岸票站投票，政制及内地事务局局长曾国卫称，有2万2千多名身处内地的香港选民登记投票，其中7Awesome成居于广东省，其余约7000人身处其他省市及自治区，包括福建、上海、北京甚至内蒙古，他指这批选民须长途跋涉，仍坚持回港投票令他非常感动。

虽然面对中共政棍们带风向，称“中共不在乎选票”，港府高官依旧不遗余力地宣传投票。虽然中共选举造假的功力享誉国际，但碍于中共公开抢夺了香港控制权，又用国安恶法暴力“平息“街头运动，暗杀潮、抓捕潮、移民潮不断刷新国际友人对中共的认识。因此中共阉割的选举制度能有更多的人参与投票，是唯一能证明中共霸占香港的”正义“性。但面对香港人持续冷淡的投票意欲，港府深知不妙，史无前例地为爱港爱国的“内地港人”提供赴深圳口岸投票，并大加赞许，试图令本港几百万选民蒙羞。港府打温情牌的伎俩不过就是诱骗香港人接受垃圾的选举制度，恶心港人罢了。
