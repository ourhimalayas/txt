据中央通讯社7月20日报道，中共国务院开放三胎政策后，于20日再次发文“取消社会抚养费等制约措施、清理和废止相关处罚规定，配套实施积极生育支持措施”。

报道指出，中共的国家卫生健康委员会强调实施三孩生育政策及配套支持措施“仍然是计划生育”。该委员会还强调说之前因为生三胎而被罚款的人不会得到任何的退款。

据报道，有位中共国的经济学家表示这些关于计划生育的新政策的推出，意味着从现在开始中国百姓是允许生第4胎或是第五胎的，也意味着计划生育实质性的终结。有位社会科学院法学研究所研究员也表示“社会抚养费已经完成了历史使命，退出历史舞台顺理成章”。

报道称2000年开始的“社会抚养费”是由20世纪80年代初的“超生罚款”演进而来的。

很明显，中共想要通过这些计划生育的新政策来得到更多愿意被他们控制的奴隶去继续为他们的子孙卖命。

中共国的经济学家和研究员们从未解析过以前中共向老百姓刮取的所谓“社会抚养费“都被用到哪里去了。中共制定的计划生育政策，无论是让中国老百姓少生，还是中国老百姓多生，都是为了服从中共的指命而存在，只要中共存在一天，计划生育就不会退出中共国的历史舞台。
