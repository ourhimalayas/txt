## 乌克兰战争：世卫组织称对卫生设施的袭击每天都在增加

翻译: 里奇

![](https://assets.gnews.org/wp-content/uploads/2022/03/xin_png.001-2-23.jpg)

![](https://assets.gnews.org/wp-content/uploads/2022/03/123875527_hospitalphoto_2.jpg)位于哈尔科夫附近的伊久姆中央医院，在乌克兰当局称是俄罗斯炮击之后。图片来源：bbc.com

世界卫生组织（WHO）表示，乌克兰已经发生了 70 多起针对医院、救护车和医生的单独袭击事件，而且数量“每天”都在增加。

它说，袭击目标针对医疗保健设施已经成为现代战争战略和战术的一部分。

3 月 8 日最近发生的伤亡事故是哈尔科夫南部伊久姆新翻修的中心医院。

乌克兰当局称医院被俄罗斯炮弹击中。

该市副市长在网上发布的视频和照片显示，医院主楼遭到严重破坏。去年新建的接待区被彻底摧毁。

这段视频已经被 BBC 和其他媒体证实，但目前无法确定袭击的确切情况。

“在第一次轰炸后，医院的窗户被炸飞了”副市长沃洛迪米尔·马佐金告诉BBC。

他补充说，第二次袭击摧毁了医院的手术室。

据乌克兰当局称，当天医院工作人员正在治疗儿童、孕妇和三名新生儿以及在该地区激烈战斗中受伤的士兵和平民。

袭击发生时，他们正在地下室避难，没有人丧生。

“政府已投资数百万美元来提供配备现代设备的良好设施”马佐金先生说。

“病人必须自己爬出瓦砾才能逃脱。”

BBC已就此次袭击事件联系了俄罗斯驻伦敦大使馆，但未收到任何回应，尽管莫斯科过去曾否认故意针对平民。

第二次世界大战后扩大的公约规定了平民和军人的基本权利，并规定了对伤病员的保护。1954 年，当时的苏联批准了这些规定。

根据公约第 18 条，民用医院“在任何情况下都不得成为攻击的对象，但在任何时候都应受到尊重和保护”。

海牙国际刑事法院可以对违反该规则的行为进行调查，如果认定为战争罪，可以起诉和惩罚个体肇事者。

也存在公约豁免。

如果医疗设施被放置在合法的军事目标附近或被认为正在实施“对敌人有害”的行为，则将失去免受攻击的保护。

根据红十字国际委员会 (ICRC) 的说法，这可能包括使用医院作为健康战士的盾牌，或将医疗单位部署在阻止敌人进攻的位置。


```
伦敦玛丽女王大学国际法和人权教授内夫·戈登（Neve Gordon）说：“实际上，我们今天所处的情况是，医院和医疗单位已经处于公平竞争中。”
```


“如果医院外面有士兵，或者只是在火车站旁边，就有可能遭到袭击。或者可能是一名受伤的士兵拿着手机，正在给其他部队打电话，告诉他们附近有人。

“所有这些漏洞都使得声称攻击是合法的成为可能。”

红十字国际委员会表示，理论上，在针对可能违反这些规则的医院之前，攻击方应始终发出警告，并有时间限制，而另一方肯定忽略了该警告。

没有证据表明这种情况发生在乌克兰冲突中。

戈登教授希望看到根据国际法对任何袭击医疗设施的行为实施更强有力的全面禁令，类似于联合国于 1987 年通过的禁止酷刑的禁令。

**从越南到叙利亚**

《日内瓦公约》的豁免已被用来证明在二战后朝鲜和越南之后的冲突中袭击医院和医疗单位是正当的。

不过，这一趋势似乎正在迅速加快，部分原因是弹道导弹、无人机和其他远程弹药的使用。

美国倡导组织“人权医师”声称，自 2011 年以来，俄罗斯或当地军队与至少 244 起针对叙利亚医疗设施的单独袭击有关。

有一次，慈善机构无国界医生甚至决定停止与叙利亚政府或其俄罗斯盟友分享其运营的一些诊所的 GPS 坐标，因为担心这些诊所更有可能成为直接目标。

俄罗斯官员否认蓄意袭击叙利亚的医院，并暗示该国的“圣战分子”经常躲在受保护的民用建筑中。

世卫组织担心这一切意味着对医疗设施的袭击正迅速成为现代战争更广泛的“战略和战术”的一部分，而不管日内瓦公约的规则如何。

它警告说，摧毁医疗设施“是对希望的毁灭”和对基本人权的剥夺。

“我们从未在全球范围内看到过这种对医疗保健的攻击速度”。其紧急情况主任迈克尔·瑞安本周在新闻发布会上说。

“这场危机已经到了乌克兰卫生系统濒临崩溃的地步。”

“它需要得到支持……但如果给予支持的人要进入的这些基础设施受到直接攻击，你怎么能做到这一点？”
