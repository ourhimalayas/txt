## 内华达州将对疫苗未接种者征收附加费

![](https://assets.gnews.org/wp-content/uploads/2021/12/图片1-2-2.jpg)图片来源：大纪元
12月3日据零对冲 (Zero Hedge)报道，内华达州本周四成为美国首个对未接种中共病毒疫苗的工人征收附加费的州，尽管该处罚到明年中旬才会生效。

该州公共雇员福利计划委员会（Public Employees’ Benefit Program Board,PEBP）除两名成员外，其他成员在一次会议上投票，批准对未接种疫苗的工人每月征收55美元的附加费。（会议）批准的提案还规定，工人的配偶、伴侣和18岁以上的家属每月要缴纳175美元的附加费。这可能会在今后进行调整。此项附加费将在2022年7月1日生效。

该委员会执行官劳拉·里奇（Laura Rich）说，这将帮助抵消中共病毒的测试费用。到9月份的测试费用估计为330万美元。另外，里奇称委员会没有分析中共病毒住院费用的提案，因为这将使配偶和家属的附加费“大大提高”。州政府规定，禁止向工人收取更多的附加费。但内华达州劳工部上个月发布指导意见称这些附加费都是合法的并且里奇将其与过去计划对吸烟者征收的附加费相比较。

会议期间公众评论员和提交书面声明的人在投票前反对该提案。公众评论员艾伦·克雷西留斯（Ellen Crecelius）在一份声明中说：“我认为拟议的附加费是不适当的，并且是过度的。”她指出许多人享有天然免疫，或者从中共病毒康复后自身会（产生抗体）保护自己。莎娜·科布·亚当斯（Shanna Cobb-Adams）说，她已经支付了每月255.06美元。新的附加费将使其（每月）增加90%。她表达了对她18岁儿子接种疫苗的担忧，因为研究表明年轻男性在接种疫苗后患上心脏炎症的风险升高，而中共病毒对没有严重潜在健康状况的健康青年来说风险很小。

该州民主党人史蒂夫·西索拉克(Steve Sisolak)是强迫工人在不接种疫苗的情况下每周接受测试的人。西索拉克的政策主管杜安‧杨（DuAne Young）称这种大流行病“已经增加了所有人的负担。而现在这个特殊的负担——测试费用，应该由拒绝接种疫苗的人承担”。
