![](https://assets.gnews.org/wp-content/uploads/2022/02/Peterson-Pierre.20220220.jpg)
Peterson Pierre是美国治疗冠状病毒感染患者的前线医生。在最近发表的视频上，他认为冠状病毒感染患者住院治疗是很危险的事情, 因为美国医院完全遵循CDC、NIH的那套害人的指导方案。患者住院治疗死亡率奇高。病人和家属被医院欺骗，完全被蒙在鼓里。其原因有如下：

1. 根据目前的健康法案，医院急诊室提供冠状病毒检测即可获得政府的资助。每诊断出一个冠状病毒感染患者，医院还会从政府再获得一笔额外的奖金支持。
2. 在接受国家老年人医疗保险（Medicare）和医疗救助计划 (Medicaid) 的医院，病人的权力被取消。
3. 每一个冠状病毒感染患者住院治疗，医院还会从政府再获得一笔额外的奖金支持。
4. 每一个冠状病毒感染患者用瑞德西伟治疗，医院又会从政府再获得一笔额外的奖金支持。
5. 每一个冠状病毒感染患者用使用呼吸机治疗，医院又会从政府再获得一笔额外的奖金支持。
6. 无论是否死于冠状病毒感染，只要死亡诊断证明书上写明“冠状病毒感染患者死亡”，每有一个冠状病毒感染患者，医院又会从政府再获得20%的一笔额外奖金支持。
7. 每个环节，医院都会从政府再有额外的奖金支持。
