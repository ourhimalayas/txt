郭文贵先生1月6日在其盖特提前曝光中共20大常委名单，习近平准备连任中共国家主席.
在这个名单中除习近平之外均是新任常委。团派背景的胡春华将出任中共下届总理，而习的亲信李希、陈敏尔、丁薛祥将分别担任中共政协主席、常务副总理、人大委员长。原中共新疆党委书记陈全国将出任中纪委书记。郭先生没有曝光主管中共意识形态的常委人选，留下了一丝悬念….在这个常委人选安排中，习近平明显排除了上海帮势力，而让胡春华出任国务院总理，应该是要联合以胡锦涛为首的共青团派，共同打击江泽民的上海帮。

 习近平擢升陈全国为中纪委书记让人颇感意外。去年陈全国因为种族灭绝、反人类罪，已经被美国政府列入制裁名单。而习近平反其道而行之，提拔陈全国任中纪委书记，应该是想利用酷吏陈全国铁腕整肃异己、震慑全党。 

郭先生基于准确情报，提前揭露习近平的政治野心，曝光中共20大常委名单应该是有其周密部署和安排，这是一着敲山震虎、分化瓦解中共的妙棋。因为江、曾、王、孟等上海帮势力绝对不会坐以待毙，肯定会绝地反击。

 相信随着郭先生提前踢爆中共20大常委名单，中共党内肯定会掀起新一轮的血雨腥风。但这将更加有利于以共灭共，加速中共分崩离析。
