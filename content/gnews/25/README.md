## 拜登面临访问乌克兰的压力越来越大

![](https://assets.gnews.org/wp-content/uploads/2022/04/GNEWSLOGO1-1-1-2.jpg)
《国会山》4月17日报道，**周日，乌克兰总统泽伦斯基和外交部长德米特里·库莱巴都表示，有兴趣邀请拜登访问乌克兰。此前，英国首相鲍里约翰逊已经访问了这个饱受战争蹂躏的国家。**

![](https://assets.gnews.org/wp-content/uploads/2022/04/220103073851-biden-zelensky-split-dec-2021-super-tease-e1650241287513.jpg)美国总统拜登与乌克兰总统泽连斯基（图片来源：国会山报）

报道称，要求拜登访问乌克兰的呼声是在俄罗斯入侵乌克兰的第七个星期发出的。俄罗斯已经控制了乌克兰的一些地区，但是由于乌克兰军队的坚决抵抗，俄罗斯在一些地区的攻势已经停滞。

此前，**拜登政府表示正在考虑派遣一名高级官员前往这个陷入困境的国家后，拜登总统访问乌克兰的压力越来越大。**

最近，**美国加大了对乌克兰的致命武器军事援助，俄罗斯希望将重点放在乌克兰东部，并最终占领马里乌波尔。**经过数周的袭击，虽然马里乌波尔没有被占领，但目前处于俄军的包围之中。

泽伦斯基在美国有线电视新闻网的《国情咨文》节目上说，**他预计拜登总统会这样做。他表示，我认为他是美国的领导人，这就是为什么他应该来这里看看的原因，当然这是他的决定。至于具体行程，则视安全情况而定。**

![](https://assets.gnews.org/wp-content/uploads/2022/04/泽伦斯基.jpeg)乌克兰总统泽连斯基（图片来源：网络）

外交部长库莱巴表示，**拜登的访问将是一个重要的信息，他指出，两国领导人之间的会晤可以为新的物资、武器，以及讨论结束冲突的政治解决方案 铺平道路。**

**拜登政府表示，由拜登去乌克兰访问并不在考虑之列**。本周，白宫新闻秘书普萨基在「播客救美国」（Pod Save America）节目说，**拜登政府已经准备好前往乌克兰，但我们不会向乌克兰派遣总统，这不在美国的总统计划中**。

总统泽伦斯基表示，**为了保护我们的国家，为了保护欧洲的民主，我们需要来自西方伙伴的更多制裁。**我们需要更多的弹药来保护我们的国家和欧洲边境。**他还表示，乌克兰绝对不会以割让领土来换取侵略战争的结束。乌克兰绝对不会放弃自己的土地。**

据报道，自冲突开始以来，乌克兰已经失去了2500至3000名军人。国际刑事法庭的一名检察官已经在调查任何个人在乌克兰任何地区犯下的战争罪、反人类罪和种族灭绝罪行。
