7月19日中午，美军一架C-130军机降落台湾桃园机场。而就在4天前的15日上午，一架美军C-146A行政专机从冲绳飞抵台北松山机场。这也是短短几天内美军飞机第二次抵达台湾。此前，中共新闻发言人曾针对7月15日美军一架运输机降落台湾时表示：任何外军飞机降落我领土都必须经许可，任何外国舰机擅闯我领空的行为，都将引发严重后果。

中共所谓的“严重后果”来的很快，7月16日，中共党卫军宣布，将在台海进行军事演习。还记得绿帽将军王宏光的叫嚣吗：美军舰停靠台湾之际，就是解放台湾之时。现在军舰不但早已停靠了，连军机都落地了，怎么还不“解放”台湾？而中共仅仅是奓着胆子举行军演。就像一个无赖对一位侠客说：敢打我一下试试？马上挨了个耳光；无赖擦干了嘴角的血，秀了秀二头肌说：敢踢我一脚试试？然后又挨了一记飞腿。中共就是那个无赖，欺软怕硬的丑恶嘴脸暴露无遗。

台湾何时成你中共的领土和领空了？台湾一直是中华民国的领土，包扣沦陷的大陆，都是中华民国的领土。中华民国在联合国是有席位的，是联合国正式认可的。是你中共窃取了政权，奴役中华同胞70余年。

更可笑的是此次军事演习的地点，位于浙江省海域，地处台湾海峡北部，这个“北部”真的有点北，再往北一点就是上海了。怎么不选择福建海域，那里可是台湾海峡，离台湾很近。怎么不选择南海，那里有美军的航母和北约的舰艇。在这两个军演地方更能达到对美国和台湾的“震慑”。中共现在震慑的不是美台，而是中共国内的人民。所谓的军事演习完全是演给老百姓看的。告诉老百姓，中共在维护“国家利益”，在维护“领土完整”，在完成“统一大业”。中共哪里有什么“国家利益”。盗国贼们有的只是红色江山、政权的稳定，有的只是党的利益、家族利益、个人利益。对外没赢过、对内没输过是中共的真实写照。对外撒币、对内放狗是中共的一贯做法。这次也不例外，借着美国军机抵台的名义，无耻地表演其所谓“爱国”。就像曾经和印度在边境剑拔弩张，而中共竟然远离边境几千里之外的内蒙古举行大规模军演，以此表达“强硬”，让人笑掉大牙。其外强中干的丑态展露无疑。此次中共的军演，表演和震慑对象只有一个：老百姓。

虽然中共的腿在发抖，但是嘴巴却很硬。什么“粉碎台独”，什么“玩火自焚”。请问，为何中共军机一再绕台飞行，中共军舰一再逼近台湾沿海？其目的何在？对台湾一再军事威胁、政治胁迫、经济打压。中共一口一个“不放弃武力收复台湾”、“宁可台湾不长草，也要收回台湾岛”，公然威胁恐吓台湾人民，使2300万台湾同胞生活在恐惧之中。美国的态度很明确，保护台湾人民免受流氓无赖的欺辱。

美国的军机、军舰抵达台湾将成为常态，中共的叫嚣和军演也将成为常态。如果有一天真的擦枪走火，那么中国人民有福了。美国必然将中共击得粉碎。美国在台湾和南海正在“擦枪”，或许在寻求“走火”，寻求结束中共的契机。
