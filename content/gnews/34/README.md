**1. ****香港民研：林郑最新民望****38.3****分****三成人评零分**

[据【独立媒体】10月12日讯，香港民意研究所公布最新调查结果，刚宣读施政报告的特首林郑月娥的民望评分为38.3分，当中28%受访者给予零分，支持率净值达-35%。在核心社会指标方面，法治及民主指标依然不合格，分别为4.86分及4.28分。学者表示，民间对新一份施政报告的态度普遍冷淡，政府硬销的「北部都会区」未能刺激民望。](https://www.inmediahk.net/node/政經/香港民研：林鄭最新民望383分-三成人評零分)

[五项核心社会指标以10分为满分，安定、繁荣、自由指标依次为5.48、5.26、5.12分。法治指标及民主指标则属不合格分数，分别录得4.86及4.28分。相比一个月前的调查结果，民主指标未有变化，其余四项指标均有升幅，其中安定指标上升0.41分。](https://www.inmediahk.net/node/政經/香港民研：林鄭最新民望383分-三成人評零分)
