根据2021年11月26日新浪报道， 央行中国反洗钱监测分析中心党委书记、主任苟文均表示“虚拟资产极易成为不法分子洗钱工具，因此必须明确虚拟资产的非金融属性，完善新兴虚拟资产的监管政策。”

苟文均表示，虚拟货币对国家货币主权、反欺诈、反洗钱和反恐怖融资构成日益严重的挑战，目前，已有50多个司法辖区已经或计划将虚拟资产交易纳入监管框架，有一些司法辖区将虚拟资产交易作为非法金融活动而予以禁止。

针对虚拟资产洗钱犯罪，苟文均提议“银行、支付机构作为法币与虚拟资产的中介，应实名认证虚拟资产交易方，监测用户并及时报告可疑交易。”
