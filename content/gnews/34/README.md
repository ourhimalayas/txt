来自台灣英文新聞 2021年12月17日的报道，中共国分析人士称在台湾问题上做的有点过可能促使胡锡进提前退休。

一段时间以来，有传言说胡锡进作为民族主义对英文世界宣传的负责人可能会提前退休。据中国媒体项目的报道，中共高层对胡锡进的“高调的国际言论”感到不满。

据中共国长期分析家比尔·毕晓普（Bill Bishop）说，这些言论可能是指胡锡进针对台湾的好战言论。毕晓普在其最新一期通讯《中共主义》中引用了他“自己的可靠信息来源”，说胡锡进“今年早些时候被允许延迟三年退休，但后来发生了一些事情导致他被迫提前退休。”

毕晓普继续推测，胡锡进“在台湾问题上的极端好战行为”并没有得到中共高层的认同。中共国领导人习近平很可能不希望如此重要的问题上由“不严谨的舆论机构”来引导大众。

胡锡进在担任《环球时报》总编辑的15年中放出了许多攻击性言论，他的好战派言论在2021年进一步升级。台湾新闻网报道了他今年以来最令人难忘的一些威胁，包括：

“《环球时报》编辑说，美国应该向台湾派遣240名士兵，这样他们就可以被中共解放军空军‘干掉’了”

“中共国官媒告诉澳大利亚在保卫台湾方面‘准备好牺牲’”

“中共国《环球时报》总编用‘鸡爪’的比喻抨击西方政府”

“中共国官媒体对美国官员不会‘给北京奥运会带来病毒’‘如释重负’”

“中共国说‘解放军的战斗机必须飞越台湾岛’”

具有讽刺意味的是，胡锡进卸任正值他在国际上被越来越多认可的时候。《卫报》最近对这位中共喉舌做了深入的介绍，称他为 “中共国的大魔王”。

《卫报》对胡锡进的未来做了认真思考：“他帮助中共国在世界舞台上建立了一个战狼式的新基调——但他还能跟上他创立的节奏吗？”他提前退休似乎已经回答了这个问题。

他在微博上被称作“老胡”，这不可能是他事业的终点。毕晓普说，我们可以期待他继续跳出来，因为他可以通过演讲和在媒体上露脸赚大把的钱。

简评：胡锡进作为《环球时报》经常对外发表一些好战的言论，对中国人在海外的形象带来了极大的负面影响。西方世界的民众，由于对中共国的无知，很容易被胡锡进所影响，进而认为所有的中国人都是共产党，所有的中国人都是极端的好战主义者。胡锡进的角色就是中共版的戈培尔之一。胡锡进卸任《环球时报》总编辑一事并不能说明他在中共高层失宠，在给中国普通民众洗脑方面，他一定会得到新的任务。
