## 纽约市向公立学校工作人员批准530名学校工作人员对周一生效的新冠病毒疫苗的医疗和宗教豁免

![](https://assets.gnews.org/wp-content/uploads/2021/09/kendall_0-11.jpg)网络图片
**摘要翻译：**市教育局官员说，纽约市教育部已授予530名学校工作人员医疗和宗教豁免，免除周一生效的新冠病毒疫苗任务。

豁免申请将于本周早些时候到期，并由教育部中央办公室的工作人员进行审查。官员们表示，他们仍在审查一些申请，可能会给予更多豁免。

该机构没有具体说明授予的豁免中有多少是出于医疗原因，有多少是出于宗教原因，也没有说明有多少人申请。

教师联合会主席迈克尔·穆尔格鲁（Michael Mulgrew）周五表示，约有3,000名成员要求退出，而代表行政人员的工会负责人马克·坎尼扎罗 (Mark Cannizzaro) 表示，约有150名成员已申请。

接种疫苗但无法产生足够抗体来保护自己免受病毒感染的教育工作者也可以获得照顾。教育部官员没有立即说明这些工作人员是否包括在 530 人中。

简评：随着各国对疫苗护照的通过或者强制疫苗注射。各国民众为了不被强制注射疫苗，寻求的理由和借口也是五花八门，如同本文所涉及到的：**医疗及宗教豁免。**而此种种的所谓豁免的前提都是民众被强制和控制黑幕下向政府申请的一个小口子，所以我们不能屈服申请所谓的豁免，而是需要站出来抵制疫苗强制及疫苗护照。
