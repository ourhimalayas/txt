据【福克斯新闻】11月6日报道，美国保守派媒体人士奥基夫的住所在当天凌晨遭联邦调查局突袭搜查。该事件牵涉到美国总统拜登女儿阿什利“日记被盗”案件的调查。

奥基夫是非盈利新闻媒体【真相工程】的创始人，该媒体以暗访、卧底等方式来搜集和曝光各种内幕消息而著称。此前，奥基夫曾在其主要网络社交媒体上，揭露了有关“2020美国大选”以及“新冠病毒疫苗公司”等背后的大量黑幕而成为焦点人物。

据悉，该被盗日记的部分内容已于去年10月被另一家保守派媒体发表。由于无法确认其内容的真实性以及是否属于阿什利，奥基夫在收到该日记后并未对外发布，而是试图将它交还给阿什利的律师。但因该律师拒绝对此进行鉴定，奥基夫最终就将其交给了政府执法部门。

奥基夫最后表示，[联邦调查局]此次针对其住所的突袭搜查颇具“政治味道”，但他本人及其媒体[真相工程]都不会因此而屈服。
