1991年12月8日，俄罗斯总统叶利钦、乌克兰总统克拉夫丘克、白俄罗斯最高苏维埃主席舒什克维奇，举行秘密会议并签订了《关于设立独立国家联合体的相关协议》。此次会议决定三国脱离苏联独立，并成立独立国家联合体取代苏联。12月25日，戈尔巴乔夫宣布辞职，苏联正式解体。

![](https://assets.gnews.org/wp-content/uploads/2022/02/蓝天飘扬的乌克兰国旗-高质量照片-196413259.jpg)

欧洲最穷国1991年苏联解体后，乌克兰继承了苏联的军事基础，并维持着仅次于俄罗斯的欧洲第二大军事力量，是世界上拥有核武器数量排行第三的大国。

- 销毁核武器


1994年12月5日，美国、英国和俄罗斯及乌克兰在匈牙利布达佩斯签署了《布达佩斯备忘录》，美、英、俄都向乌克兰提供安全保证，前提是乌克兰必须销毁核武器，成为一个无核国家。这份备忘录写着“尊重乌克兰现有领土的独立和主权”、“避免使用武力等方式威胁乌克兰”。

1986年发生在乌克兰境内的切尔诺贝利核电站爆炸事件，受辐射死亡的人数高达10万人，百万人被迫搬离该地区。乌克兰民众始终笼罩在切尔诺贝利的阴影中。加之乌克兰政府无力支付高额维护核武器的费用，因此乌克兰销毁了全部的核武器，由核武器大国瞬间成为了无核国家。

2014年，克里米亚自治共和国脱离乌克兰投向俄罗斯后，乌克兰才意识到《布达佩斯备忘录》就是一张废张。因为俄罗斯没有遵守约定，而且英美两国也没有出手为乌克兰提供安全保障。

- 明斯克协议


《明斯克协议》是以协议签署地白俄罗斯首都明斯克命名的。协议共有两个版本，第一个版本签订于2014年9月，协议签订后不久，乌克兰东部武装就与乌克兰政府再度全面开火。第二个版本签订于2015年2月，也称为《[明斯克协议II](https://www.chathamhouse.org/2020/05/minsk-conundrum-western-policy-and-russias-war-eastern-ukraine-0/minsk-2-agreement)》。协议提出了13点要求，包括立刻全面停火、双方撤走所有重型武器、交换囚犯、豁免参战人员、提供人道主义援助、撤走所有外国士兵雇佣军和武器、欧安组织负责监督停火执行。虽然《明斯克协议II》签订后，乌东部停止了大规模战斗，但各方对于其他条款特别是涉及关键政治问题条款的执行顺序存在分歧，地方小规模武装冲突仍时有发生。

乌克兰是欧洲面积第二大的国家，仅次于俄罗斯，拥有辽阔的平原、肥沃的黑土地和成熟的生产流程，被誉为“欧洲粮仓”，曾是第三大粮食出口国。乌克兰自然资源丰富，经济结构多样化程度较高，既有规模巨大和种类齐全的工业，又有集约化水平较高的农业。

![](https://assets.gnews.org/wp-content/uploads/2022/02/图片2-26.png)

自乌克兰宣布独立以来，乌克兰政府摒弃了计划经济体制，选择了市场经济道路。在经济转轨过程中，乌克兰没有制定自己的经济政策，而是选择维持旧的苏联体制，从而导致经济危机、恶性通货膨胀，经济形势持续恶化，人民生活水平急剧下降，直到2000年乌克兰才实现了经济正增长。可惜好景不长，2008年爆发的全球经济危机，导致依靠出口的乌克兰经济严重衰退。之后乌克兰因为政局动荡，东部战争等诸多原因，经济形势一直没有好转，最终导致乌克兰成为欧洲最穷的国家。
