美国媒体《保守简报》11月11日报道，加州州长纽森自10月27日以来已经有两周未公开露面，并缺席了本月初在苏格兰格拉斯哥举行的【联合国气候峰会】。尽管纽森此前曾为该峰会做了大量准备，但最终是由副州代替他出席。 

据《每日邮报》当天新闻表示，与州长家人接近的某位消息人士透露，纽森于10月27日在奥克兰最后一次公开露面并接种了“新冠病毒”疫苗加强针后出现了肌肉无力和疲劳等症状。

对于以上状况和疫苗接种副作用的传闻，加州政府办公室发言人予以了否认并声称，纽森州长一直都在国会大厦工作，他对接种加强剂疫苗没有任何不良反应，并且将于本周参与有关经济和疫苗方面的公共活动。而此次缺席峰会完全是由于其“家庭事务”所致。
