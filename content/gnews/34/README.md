据加拿大媒体《真北方》2月1日报道, 萨省已成为第一个宣布即将取消所有疫情限制和疫苗护照的省份。

报道称，于2月28日结束所有限制措施的决定是省长斯科特在民意“希望一切恢复正常”之后作出的。新的民意调查显示，54%的加拿大人支持尽快结束各种与疫情和疫苗相关的限制措施。 此外，阿省和安省也计划进一步放宽相关疫情政策。而魁北克省长勒戈在遭到公众强烈抵制后，已被迫终止向该省未接种疫苗者征税的计划，并同时公开承认此举将导致魁北克人之间产生严重分裂。
