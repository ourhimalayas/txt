## 有声|欧盟将对中共国铝箔征收额外关税

配音：香草山美食部 – 玫瑰园
![This image has an empty alt attribute; its file name is image-40-1-11.png](https://assets.gnews.org/wp-content/uploads/2021/10/image-40-1-11.png)
据[OAN 网站](https://www.oann.com/eu-to-impose-extra-tariffs-on-china-aluminium-foil-over-subsidies/)援引路透社12月22日报道，欧盟官方期刊周三表示，在欧盟委员会发现中共国生产商受益于过度和不公平的补贴后，欧盟将对来自中共国的铝箔征收额外关税。

将于周四生效的、对特定薄型铝转换膜征收的关税从8.6%到18.2%不等，是对现有6.0%到28.5%的反倾销税的补充。

欧盟委员会在收到来自6家欧盟生产商的投诉后进行了调查。该委员会表示，中共的国家补贴形式包括优惠条件的融资、出口信贷保险、补贴、免税以及政府以低于市场价的价格提供土地使用权和电力。

调查显示，中共国已将铝箔确定为一个关键行业，其在欧盟铝箔市场的份额从2017年的18% 增长到2019年的24%。
