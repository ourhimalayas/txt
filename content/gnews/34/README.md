7月12日，日本时事通讯社报道：今天，“荷兰海牙仲裁法院驳回中共国对南海领土权利主张的裁决”满五年。日本外相茂木敏充发表声明：就中共国无视这一裁决，指责中共违背了“根据国际法和平解决争端”的原则，破坏了法治。

五年前的今天，荷兰海牙仲裁法院根据《联合国海洋法公约》裁定：中共国欲在南海的海洋扩张，声称拥有主权的独特边界线 “九段线 “之说，没有国际法依据。

笔者在日生活近三十年。尊规守法在日本被视作理所当然的事，而中共国则恰恰相反。所以，这样的指责，对中共来说是不疼不痒，不会感到一丝廉耻。
