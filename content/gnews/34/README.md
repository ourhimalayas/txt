## 中共国肉制品含禁药，反禁药组织提醒冬奥运动员别吃

德国反禁药组织（NADA）日前强烈呼吁准备参加北京冬奥的运动员别吃中共国的肉制品，以免摄取到“瘦肉精”违反禁药规定。而中共官媒去年3月也曾揭露河北养殖户违禁使用“瘦肉精”达十年之久。

法新社报导，德国反禁药组织10日在业务通讯中呼吁将参加北京冬季奥运的运动员，不要在中共国食用肉制品，以免摄取到肉品中俗称瘦肉精的类固醇“克伦特罗”（clenbuterol）。这种类固醇是用来让牲畜在屠宰前增加瘦肉、减少脂肪。

据报导，反禁药组织长久以来都有提到中共国产制肉品受克伦特罗污染的风险，不过近期没有传出相关案例。不过，中共国中央电视台在2021年3月15日 “3.15”国际消费者权益日当天曾揭露，每年大约出栏70万只羊的重要肉羊产地河北省沧州市青县，有养殖户违禁使用莱克多巴胺、沙丁胺醇和克伦特罗等瘦肉精，持续销往多个省市约达10年之久。

据报导，中共国在2002年就明令严禁瘦肉精作为兽药和饲料添加剂，虽然青县相关部门一直严查违规添加瘦肉精，许多养殖场的大门上都有监管部门醒目张贴的「禁止使用瘦肉精」告示，但实际上养殖户官商勾结，事先得到消息，提前应对检查。

报导引述一名当地业者表示，曾接获电话通知会有检查，“像我们这有的时候来查了，村支书都通知。如果是市的来，他先给县通知，一级一级都有通知”。

消息来源：[**中国瘦肉精引德国组织示警官媒一年前也曾揭露**](https://tw.news.yahoo.com/%E4%B8%AD%E5%9C%8B%E7%98%A6%E8%82%89%E7%B2%BE%E5%BC%95%E5%BE%B7%E5%9C%8B%E7%B5%84%E7%B9%94%E7%A4%BA%E8%AD%A6-%E5%AE%98%E5%AA%92-%E5%B9%B4%E5%89%8D%E4%B9%9F%E6%9B%BE%E6%8F%AD%E9%9C%B2-061405026.html)
