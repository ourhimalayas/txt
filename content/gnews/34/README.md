“打架啦！”

李拐子和长生打起来了，好家伙！撒腿跑去看看，长生正骑在拐子身上开打，拐子像猪一般嚎叫，人们都看着，谁也不去拉，我也是这样想的——“打死这个狗日的”。不一会儿支书领着民兵连长海旺来了，一嗓子喊开了长生。

李拐子流着鼻血爬起来，一看人来了立马来劲：“长生，你给爷记住了，不是爷这赶马车的，你全家早饿死㞗了，看看你那㞗样能养活你的媳妇和娃？你给爷等着！”说完后恶狠狠的吐了一口嘴里的血，一瘸一拐走了。

这李拐子年轻时赶马，从马上掉下来让车压坏了一条腿，后来好了以后落下了拐腿的毛病。这家伙是个马屁精，整天把支书和会计满村子夸，每次上公社供销合作社拉回全村的物资都要先去支书和会计家，村里人也都知道怎么回事——那年头，村里记公分制，谁家多点，谁家少点，都是支书会计说了算；每年春节前马车官都要去拉供应物资，村里向上多报点，向下多扣点，多出来的就到了村里这些头头的家里，马车官中间也得抹些油，而这点可怜的物资就足以让拐腿的李拐子睡了村里好几个女人。写到这，相信战友们也明白了，七哥直播里提到的厨师用馍睡女孩的故事也就不足以为奇了！

今天中国满大街的大白，有一大部分是都是来自农村的民兵，很多都是光棍汉。前年疫情爆发前我给老家打电话，有个远方亲戚一直吵着让我在城里给他找个活干，后来我给他找了个保安的活，结果他又说不来了，说是正在老家做民兵训练，还跟我说将来国家会有大用处，听他声音兴奋的不行。这段时间再问他时，说已在老家的市里管人，穿著白色防护服，说他想管谁就管谁，好吃的食物很多，关健是一个月能领4500元工资。这让我一惊，看来先生说的真没错——中共早就已精心设计好了，管住老百姓的吃饭喝水，也就管住老百姓的自由。今天发生在全国各个城市里的打人和诱奸事件就很好理解了。

现在村子里稍微勤快一点的年轻人都在过着自己的小日子，有些头脑灵活的还在城里兼一份事情做，村子里剩下的大部分都是些老人，而这些民兵平日在村里大多都是些游手好闲偷鸡摸狗的主，多数都找不到媳妇。这两年老听周边的人说现在村里的中壮年都当了民兵，我还一直理解不了。前几天看《大纪元》的节目说，上海去了好多山东的民兵，这让我一下明白了，于是大白把狗打死也就能够理解了，因为现在的民兵大白和以前的红小将都一样，低素质和邪恶。

显而易见，中共是要用这些不睁眼的僵尸来奴役人民，把辛辛苦苦工作、老老实实过日子的普通百姓再次拉回到那个荒唐的年代，让我们这些人为了能吃口饭而整天哀求他们，失去做人最基本的尊严，为了吃饱饭再也不关心究竟是谁在奴役着我们，而他们自己却可以将皇位世代传承下去。

醒来吧！我的同胞们，一起来消灭这邪恶的中共魔鬼。
