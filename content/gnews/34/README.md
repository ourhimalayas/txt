[日本新闻周刊报道](https://news.yahoo.co.jp/articles/1e033647bcda440fc877454f0403b4cf0b4fb93c)：
一个国家对另一个国家的好印象，在很大程度上受对其领导人印象的影响。根据皮尤研究中心今年2月至5月在17个国家进行的调查，与美国和中国双双好感度很弱的去年相比，拜登总统当选后的今年，人们对美国的印象明显改善。

在全球范围内，有平均61%的人表示他们对美国有良好的印象。好感度最高的是韩国，达到77%，其次是意大利的74%和日本的71%。

与此同时，人们对习近平主席继续执政的中共国的印象普遍保持负面印象。只有平均27%的受访者说他们对中共国有好印象，但新加坡（64%）和希腊（52%）是仅有的”爱中共国 “的国家。

对美国有好印象的人的世界平均比例为61%。
对中国有好印象的人的比例为27%。
在韩国对美国有好印象的人的比例为77%。
在新加坡，对中国有好印象的人的比例为64%。
