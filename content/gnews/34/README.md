恒大海花岛39栋楼/43万平方建筑被要求10天内拆除

据中共国官媒上海报业集团主管的财联社报道，2021年12月31日，一份海南省儋州市综合行政执法局发布的行政处罚决定书在网络流传，该行政处罚决定书称，儋州信恒旅游开发有限公司位于儋州市白马井镇海花岛2号岛三(六)期项目共建设39栋楼，总建筑面积434941.46平方米，因违法取得的规划许可证已被撤销、项目存在违反《中华人民共和国城乡规划法》第四十条之规定情形，依据规定，现责令该公司在10日内自行拆除上述违法建筑物。如逾期不拆除，执法局将依法组织拆除。
