近日，一艘载有60名官兵的[澳洲潜艇柯林斯号停泊于凯恩斯港口圣三一水道](https://www.rfi.fr/cn/%E5%9B%BD%E9%99%85/20210626-%E6%BE%B3%E5%A4%A7%E5%88%A9%E4%BA%9A%E6%9F%AF%E6%9E%97%E6%96%AF%E5%8F%B7%E6%BD%9C%E8%89%87%E5%81%9C%E6%B3%8A%E5%87%AF%E6%81%A9%E6%96%AF%E5%9B%A0%E5%BA%94%E5%8D%97%E6%B5%B7%E5%B1%80%E5%8A%BF)。柯林斯号潜艇完成补给后，将加入澳洲东部沿海的军事演习，这是为了预防中共在南中国海局势升温期间模仿俄罗斯与英国海军开火的做法。一旦南海发生冲突，澳洲军舰将伴随美国舰队介入南海的军事争端。由于担心军舰会先遭到中共的袭击，澳大利亚决定将位于澳洲东北部邻近南海的凯恩斯作为[澳洲重要的国防前哨据点](https://www.cna.com.tw/news/aopl/202106260012.aspx)。

此前在2021年6月24日，媒体报道了澳洲陆军将和日本离岛防卫专门部队“水陆机动团”、美国海军陆战队和英国海军陆战队一起，实施名为[“护身军刀”](https://china.kyodonews.net/news/2021/06/8ad3353a57b0.html) 的水陆两栖作战联合训练。之前参加该训练的只有美日澳三国，而作为首次参加的英国，将派遣以最尖端航母“伊丽莎白女王”号为中心的打击群加入该训练。本次“护身军刀”训练旨在通过在太平洋周边的联合行动来彰显与英国的合作，对加强海洋活动的中共进行制约。

而早在3月份，中共其实已经知晓美、日、澳等各国将集结南海，中共察觉不妙后，通过[国防部发言人](https://finance.sina.com.cn/tech/2021-06-16/doc-ikqcfnca1337891.shtml)做出表态，试图消除世界各国对于中共南海野心的疑虑。中共[国防部发言人](https://finance.sina.com.cn/tech/2021-06-16/doc-ikqcfnca1337891.shtml)称：“当前南海局势总体稳定，地区国家致力于将南海打造成和平之海、友谊之海、合作之海。中方维护南海和平稳定的决心意志坚定不移，坚决反对任何国家以‘航行自由’为名，加大在南海的军事存在，制造紧张气氛，插手地区事务，损害地区国家共同利益。”

由于全球有48%左右的海上贸易均通过南海，因此南海的自由航行集聚了全球的眼光。自总加速师上台以来，中共与南海地区周边国家的关系剑拔弩张，包括在南海建设人工岛礁，导致南海局势不断升温。对于中共使出“贼喊捉贼”的伎俩，世界各国并不买账。出于战略安全考虑，澳大利亚一直与美国等盟友进行合作，澳军舰曾多次穿越南海，以维护南海的航行自由。

由于澳大利亚政府坚定地维护国家安全和在国际上要求对中共病毒进行独立的调查，中共恼羞成怒，企图通过经济手段威胁和打压澳大利亚，澳中关系一路走低。6月23日，即将卸任的澳洲外交贸易部常务副部长弗朗西斯•亚当森发表演讲，称中共“充满不安全感”。亚当森在演讲中称中共“担忧外部威胁又试图对外扩张……在充满不安全感的同时受到野心的驱使……在对威胁严防死守的同时又把自己的利益凌驾于其他国家之上”。

据中共媒体《[新浪新闻](https://news.sina.com.cn/c/2021-06-24/doc-ikqcfnca2958648.shtml)》，中共外交部发言人赵立坚回应称：“澳方官员的有关言论完全不符合事实，希望澳各界民众擦亮眼睛、明辨是非，客观看待中共国的发展，自觉抵制澳国内一些人对中共国的污蔑抹黑，理解和支持中澳友好和合作事业。”

赵立坚向来以“战狼”闻名，澳洲人民早已有所耳闻。如赵战狼所望，澳各界民众一定会擦亮眼睛和明辨是非，将邪恶的中国共产党和善良的中国人民分开。孙力军藏于澳大利亚的病毒秘密已被澳洲政府所缴获，澳大利亚已经进入灭共模式，中共向全世界发动生化超限战的真相即将公布于天下，中共灭亡已经指日可待！
