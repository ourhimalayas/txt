近期，中共央行等监管部门将恒大、融创、绿地等“三道红线”试点房企的商票数据纳入监控范围，要求房企将商票数据随“三道红线”监测数据每月上报。

当前，中共国每家大型房企的债务规模动辄都上千亿级，甚至高达万亿级，除涉及银行等金融机构的债务之外，还以商票形式来对上下游企业进行“赊账”。由于每家大型房企都涉及数千家的上下游企业，加之上下游企业的数量众多、情况复杂多样，任何一家上下游企业因商票兑付出现的不可控情况，都可能成为压垮一家大型房企这个“骆驼”的“稻草”，从而引发行业和系统性暴雷。

可见，商票已经成为引发房地产和金融机构暴雷最不受控的因素和最大的隐患。中共央行等监管部门将房企商票纳入监管，主要目的就是通过监控商票来对实现对大型房企的全链条监管，最终目的是为了提前预警和应对大型房企以及由其引发的暴雷。但当前因中共盗国而产生的天量债务无法解决，加之中共国政治、经济和社会形势急转恶化，各种不确定性和变数加大，又将加剧商票承兑违约。可见，中共央行监管房企商票最多延缓，但却无法避免中共房地产和金融暴雷。
