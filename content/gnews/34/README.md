在人类文明几千年发展的历史长河中，宗教一直扮演着重要角色，我们从何而来，为何而生，上天之神究竟在哪。这些问题永远无法解答却存在于我们的内心。今天将会给大家讲述一部日本神级动漫《新世纪福音战士》。毫不夸张的说，这部动漫我已经看了三遍，但还是不能理解其全部内容。表面上看这部动漫展现出科技感十足的人机混战，超大型机器人与外星生物的精彩决斗，然而不难发现，这部动漫始终能看到《圣经》和宗教文化的身影，其中更是将人类的精神力量得以表现，那就是人的精神意志力会超越机器本身的极限，爆发出更多的力量。

《新世纪福音战士》里的战斗机器叫EVA，是一种超大型机器人，驾驶者进入机器后与它进行神经连接，如果EVA认为驾驶员不是合格的人选，将会拒绝连接，没错，机器会选择人。如果人机匹配度高，机器将会更好操控，如果人机匹配度低，可能让EVA行走都困难。

在浩瀚的宇宙中，不定期会有外来生物入侵地球，它们被称作“使徒”，人类为了生存就必须要消灭这些使徒，说到这里，有没有联想到《圣经》的十二使徒，他们是和耶稣一起救世的帮手。而动漫中也在不断暗示，使徒入侵地球实际上是帮助人类进化，也被称为“人类补完计划”。在眼花缭乱的战斗场景中，有一个让人疑惑又不能忘记的画面，每当一个使徒被消灭时，在爆炸的一瞬间，它会以一个巨大的“十字架”样子出现在空中，而不是以往的一团烟云。仿佛作者在告知我们，这是上帝的旨意。更奇怪的是，每一个使徒死后，都会涌出大量的红色血液，大片的土地和海水都被染红了，那些生物也无法在那里生存，最后人类的栖息地也越来越小。当画面中看到人类为了拯救物种时，不得不建立人造保护区，将少量的生物保护起来，以便未来进行人工繁殖避免灭绝，这个场景让我想起了诺亚方舟的故事。上帝对人类的杀戮和亵渎无法容忍，于是想彻底毁灭人类，然而诺亚一支族人却保留了善良的品质，上帝便命诺亚打造方舟，带上族人到船上去避难，所有的生物也带上一公一母，并告诉诺亚，等大洪水退却后，你会在鸽子的引导下找到大陆。

在动漫中，当地球的海水全都变成红色时，绝望的人类看着大海，闻着发臭腐烂的味道，这一画面和“启示录”里所讲太相似了，天使吹响了

第二个号角，海水的三分之一变成红色，海洋生物死去三分之一，大地在燃烧，人类无处可去。

动漫中的使徒拥有强大的力场AT Field，这是一种生物自我保护力场，不接受外来力量的侵入，在这种力场的保护下很难伤害使徒，所以EVA必须要突破力场才能消灭它们。然而整个动漫一共出现了十七个使徒，在大结局中暗指，人类自己就是那第十八使徒，这便是上帝对人类实施的“人类补完计划”。因为人类个体太过于渺小，力场十分薄弱，只有大家联合起来才能对抗外来入侵者。整个动漫用宗教的故事启发我们，不要杀戮，同求共存，相信上天的神，而不是狂妄无知的自己。
