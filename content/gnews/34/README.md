**2.股市因德尔塔变异病毒株而紧张**

[全球股市周二从纪录高位小幅回落，原因是担心中共病毒在亚洲爆发会破坏经济复苏，而投资者仍对美国退出宽松政策感到不安。以泛欧STOXX 600指数衡量的欧股上涨0.4%，工业、金融和矿业类股大幅上涨，这些板块将从经济好转中受益。市场对经济稳步复苏的乐观情绪，使这一欧洲基准股指有望连续第五个月上涨。相比之下，摩根士丹利资本国际(MSCI)除日本外亚太地区股指下跌0.5%，因一些国家重新实施封锁以遏制德尔塔变异病毒株的传播，近期的积极势头停滞。日本日经指数下跌0.8%，澳大利亚ASX/200指数收盘下跌0.1%，原因是该国对中共病毒的控制力度加大，影响了市场人气。韩国股市收盘时下跌0.5%。中共国股市下跌0.92%。](https://finance.yahoo.com/news/asian-shares-pressured-fears-over-024212830.html)

**8****.****三菱UFJ金融集团董事会否决气候决议**

[三菱UFJ金融集团股东周二否决了一项要求该行将业务与全球气候变化目标保持一致的提议，令董事会在投资者推动更多环保行动方面取得了胜利。在欧洲和美国，试图阻止气候变化的激进投资者已经成功地迫使企业撤资，但到目前为止，这一策略在日本没有奏效。这是自2020年以来，日本上市公司第四次否决所谓的股东气候决议。日本董事会受益于倾向于服从管理层的国内机构投资者的支持。三菱UFJ是日本最大的银行，并支持了许多煤炭和化石燃料项目。三菱UFJ银行的一位发言人通过电子邮件证实，该决议在其年度股东大会上未能通过，该大会获得了约23%的股东投票。三菱UFJ金融集团董事会反对该决议，称该提案的“基本内容”“已被纳入公司的管理战略”，最近还做出了碳中和承诺，并做出了其他政策变化。](https://money.usnews.com/investing/news/articles/2021-06-29/mufj-board-beats-back-climate-resolution-as-activists-falter-in-japan)
