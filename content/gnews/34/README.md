2021年7月20日，河南郑州市区及农村发生了历史以来特大洪水，根据网上流传的视频以及郭文贵先生7.21日上午的直播爆料，当地房屋冲毁、大坝决堤、地铁被淹、洪水肆虐，把路上的行人汽车冲走，大地时不时塌陷的把行人和路面吞噬，现场群众死伤无数，伤者惊魂未定，死者死不瞑目，初步估计死亡人数达一万以上，情况惨不忍睹。

根据一则网上流传的2018年郑州市政府发布的一则城市规划报告，到2020年郑州市将投入534.8亿元建设海绵城市项目，彻底告别看海城市。结果是：啪…啪…啪，自打脸！这场洪水不但没有防住，还彻底撕掉了政府的虚假宣传，而郑州市政府不仅没有道歉问责，反而第一时间把责任推给不会说话的老天爷，歌功颂德领导救灾，掩盖洪水整个真相，威胁死者亲属不准发声。

那么为什么投入534亿的项目，还是让郑州变成了欧洲的威尼斯呢？当然是因为工程款被层层转包，层层贪污，层层腐败了。郑州乃至河南的医疗、教育、基建、民生等生活设施在中共国内都是倒数的，因为河南当官的从来只想着玩政治，当大官，只知道贪财行贿上级领导，而不是努力为老百姓工作、干实事，真心为人民服务。
