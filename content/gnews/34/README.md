## 我们的神山住满了妖孽
` 骄子` [轉載自GNews](https://gnews.org/zh-hans/1843116/)

文：银河勇气星｜骄子
![](https://assets.gnews.org/wp-content/uploads/2022/01/5f49cb858d194058b5f312655ec25019.jpeg)昆仑山 / 图源：网路
以前听人讲，说地球上这些各种煤啊石油啊，都是上古那些大树和各种动物的尸体变的，经过了几十上亿年的演变，生生把这些东西熬成了石油变成了煤。阐释还很有道理，比如说煤和树都是可以燃烧的，本质没有变化，只是两种不同的形态而已，毕竟在自然界，连水的存在都是可以互相转化的，冰、水，遇到某些化学反应可以相互转化。自然界既有液态的水也有固态的水，只是他们不会在相同的条件下存在啊，冰和水起码不能在高温下都保持着原有的形态哦，但是大树和煤是完全可以在相同的条件下存在的。这种很抽象的问题，并且往往是要经过上亿年的演化的，往往只能听从于权威，他们穷尽几十年的研究去搞这个，好像大众不相信他们，产生了质疑，就不对了。

以上只是我们的教育中，经常见到的一个例子，还有一个例子就是人是猴儿变的，无论有多少问题，比如猴子怎么还长毛啊，人怎么不长毛了要穿衣服，或者猴子怎么就不会说话呢，而从人生下来开始只要没有先天缺陷，说话是最常见的生理活动功能。书本给人的知识充满了各种怀疑，但是这种东西依然写在了书本上，美其名曰“科学”。直到后来有了互联网，世界各种奇闻怪录，并不仅仅就是看而已，因为事情太真实了，尤其是亲眼所见的那种，没有见过的，因为出现的频率太多而被称为“世界未解之谜”。地球上出现过无数次的神秘的麦田圈就是一个，还有众所周知的外星人。因为人类科技的发展，比如谷歌地图实时可以观测地表的变化，包括那些要引起冲突争端的国家部署的各种导弹武器，只要在地表，或者就算挖坑，地表一定也是有痕迹的，更别说地表的大规模灾难，比如大洪水，河流断流等，这些都是有历年来的数据的。现在的人类已经可以通过科技手段看到一些与书本知识违背的内容，但是很多大科技公司都不打算公布，要么三缄其口，比如美国五角大楼始终不提外星人的事情，就算公布也会遮遮掩掩；要么就撒谎，比如共产党。
![](https://assets.gnews.org/wp-content/uploads/2022/01/66a04eef7fba49f1bd50bf56ae69ebf0.jpeg)麦田圈 / 图源：网路
地表的一切，河流，高山，瀑布，沙漠，这些大规模的暂且就不说，就算是面对一花一草，有时候看到那种悬崖峭壁上开出的花，真得会很惊讶，自然太神奇了。是什么力量让这种严酷条件下都能长出这么好看的花儿来？七哥直播里讲昆仑山的故事，那更加感受到自然的神力。中共国境内有很多山啊，旅游风景区，目前情况看来开发好的，无论收不收门票，都是些非常适合人类游玩的地方。去看一趟迎客松，很多小孩都被要求回来要写一篇记忆深刻的日记，他个小孩子，旅游的时候到处都是人，啥也看不上，迎客松前面一波一波的人流都在合影照像，你让他看那棵树，他看到的都是讨厌的人类，并且这趟旅程一路上都是人类的嘈杂，还在大巴车上唱我和我的祖国，歌颂完共产党，还非逼着他写出迎客松的自然神奇时刻，那不写共产党写啥呀，网上一搜都是歌颂共产党的，随便嫁接一下就成为了老师要在班里给全班同学朗读的优秀作文。我们的教育环境就是这样的，一切鬼斧神工都是共产党造化的。旅游名胜地点一般都有个tips可读的，很多人都请了导游专门给讲解。讲解内容大部分都是共产党教材里随意篡改的，什么大力怪神乱世毛贼东，只要上面表扬，啥都敢乱编，编造了亩产万斤，当然可以编造某些超自然神力。只是这种编造要私底下偷偷来，毕竟共产党不允许老百姓有神论。很多小地方的人，把毛贼东当神一样供着，车子里面挂个毛像，家里有毛像，办公室各种地方都有毛像，对这些人来说，毛可以保平安保发财。

![](https://assets.gnews.org/wp-content/uploads/2022/01/1598933346877988.jpg)迎客松 / 图源：网路

人类能够到达的旅游景点，都是可以随意给编排个什么神奇故事的，这要看投资方的意愿，一切遵照这人的喜好，对这些商人来说，真正的大自然并不重要。更别提那些人类尚未开发的旅游景点了，很多陡峭的山峰，因为气候和环境的影响，经常会形成叹为观止的周边景象，光是看到就够震撼了。并不是说去人类开发好的旅游景点就不会感受到这么淳朴的自然，去哪儿都可以感受自然的力量，但是所有的前提就是推翻一切书本上的东西。人迹罕至的地方之所以神奇，就是因为他们还未被列入人类的课本，一个人去一个地方带回去一个游记，一个movie，都是这个人的自己的看法，不同的人在不同的阶段去，感受也会大不相同。共产党对昆仑山的了解已经非常非常丰富了，他们才可以在那里设祭坛，摆风水大阵来诅咒中国人，力保邪恶统治。这些事情，他们绝对不会告诉老百姓，课本里也从来不会有，有也是篡改的。

我们的神山之上，如今住满了妖孽。不仅仅是对这些真实壮丽的山河之神的屏蔽，也是对人类真正起源的误导。妖孽横行，各种各样的真假参半的消息肆虐，通过互联网带来更多误导。我们了解山河大地，了解太空宇宙，不过是想知道人到底是怎么来的，想找到最初的一切。世界各个地区和宗教里关于人类起源的传说都提到了大爆炸，我比任何时候都相信，人类一定是有主人的，别再拿什么人是猴儿变的这种弱智低能的骗术来引诱和误导。如果能够被引诱，这是我的耻辱，我就不适合做人，我对不住给我智慧的上天。
