**港府宣布入境条例****8****月生效****「预先通报旅客资料系统」稍后落实**

港府今日（19日）宣布，《2021年入境（修订）条例》将于8月1日生效，条例赋权保安局局长可制订规例，落实「预先通报旅客资料」系统，同时赋权其可禁止航空公司运载个别乘客。

保安局表示，政府会根据国际民用航空组织建议的标准，和参考外国做法，研究「预先通报旅客资料」系统整体运作安排，再制订附属法例。强调系统会咨询立法会，不会在现阶段生效。

新例下，免遣返声请人须按入境处要求出席会面，声请人如缺席会面，入境处可就声请作出决定。

条例修订后，雇主若雇用非法入境、逾期逗留或被拒入境者，最高刑罚将由现时罚款35万元和监禁3年大幅提高至罚款50万元和监禁10年，相关公司董事、经理、秘书、合伙人亦可能负上刑责。飞机拥有人及代理人，若运载未带备有效旅行证件抵达香港的乘客，最高罚款由1万元提高至10万元。

羁留方面，当局将继续利用现有的羁留设施集中羁留对社会治安有较大风险的声请人。

入境处辖下的青山湾入境事务中心，连同重新启用的大潭峡惩教所，借用于羁留对社会治安有较大风险的声请人。

**教局更新指引****每**** 3 ****年查教师刑事纪录****新入职须签守则注意网上言论**

教育局更新学校聘任教学及非教学人员须注意事项，上周五（ 16 日）发信学校。当中新增指引列明学校在聘用教师时指，应透过不同途径，如要求教师签署文件作承诺，清楚列出学校对教师专业操守的要求和期望，并提醒应征者，如获受聘，他们须承诺秉持教师专业操守，包括注意在社交媒体的言论。另外教育局更新守则指，2021/22 学年起会为所有注册教师，每 3 年一次全面查核其刑事纪录。

教育局发出的通告表明，就教师注册方面，对曾干犯严重罪行或作出违法行为，并新增「涉及失德行为」的申请人，会拒绝其注册申请；有关守则适用注册教师，包括「检定教员」和「准用教员」，当局会考虑取消其注册。

教育局又说，学校聘任教师时，应查阅教师的注册资格、是否有刑事定罪纪录、是否涉及任何在进行中的刑事诉讼或调查，包括但不限于被警方逮捕或拘捕，或是否正被学校或教育局调查有关其专业失德的指控。

至于在职教师，通告列明学校需要求教师当知悉自己涉及任何在进行中的刑事调查或诉讼，包括但不限于被警方逮捕或拘捕，必须即时向学校报告；教育局亦在通告新增备注列明，2021/22 学年起教育局会为所有注册教师每3 年一次全面刑事纪录查核，杜绝漏报、隐瞒或未报而出现遗漏，进一步保障学生福祉。如学校发现教师有漏报或隐瞒，应按事件性质和严重性采取适当跟进，包括考虑暂停有关教师职务，甚至将其解雇。

美国之音7月16日报道，亚马逊掀起针对中共国电商的封杀大潮，涉及电子产品、家居健康、服装美妆等多项类别，深圳各大跨境电商一片哀嚎。

报道说，号称「深圳跨境五虎」的电商，现在除了安克（ANKER），其他四家傲基（AUKEY）、通拓（TOMTOP）、泽宝（SUNVALLEY）、帕拓逊（PATOZON）已全部被亚马逊封杀。

美国之音指，亚马逊对中共国电商下狠手，主要原因是中共国跨境卖家把他们在中共国司空见惯的欺诈行为带到了国外：为追求流量虚假刷单，送礼品卡给买家换取好评等等。

香港某广告公司顾问Tina Wang告诉美国之音，亚马逊封杀对中共国大企业肯定是致命的打击，因为在大陆拼多多平台上只能卖出去一些低利润的伪劣产品，高端产品只能依赖国外市场，中共政府说什么「内循环，双循环」，把出口不了的内部消化，但根本就不可能消化掉。
