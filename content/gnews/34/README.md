## 日本两栖部队和美国海军陆战队首次共同演练空降攻击

![](https://assets.gnews.org/wp-content/uploads/2022/03/16473535341.png)图片来源：路透社
根据《新闻极限》3月15日引述路透社的报道，作为日本和美国之间军事合作深化的一个标志，周二，日本两栖部队和美国海军陆战队首次共同演练了空降登陆攻击。

面对中共日益增长的军事强硬态度，日本今年正在修订一项十年前的国家安全战略。根据对国防政策指导方针的升级要求，预计将要求该国在区域安全方面与华盛顿一起发挥更积极的作用。

作为为期三周的联合演习的一部分，周二，演习在富士山脚下举行，包括来自日本两栖快速部署旅（ARDB）的400名士兵和600名美国海军陆战队员，目的是磨合盟友之间的互操作性。演习中使用了鱼鹰直升机。

美国海军陆战队退役上校格兰特·纽森（Grant Newsham）曾在日本建立两栖部队时作为联络官为其提供建议，他说：“这次训练的真正意义在于，海军陆战队和日本两栖快速部署旅正在进行严肃的战斗训练，而这种训练在十年前是不可想象的，这表明美国和日本的关系更加牢固。”

报道说，日本两栖快速部署旅于2018年启动，是日本自第二次世界大战以来的第一支海军陆战队，组建该部队是为了加强其在东中国海边缘的西南岛屿的防御。

这支由约1500名士兵组成的部队接受过使用直升机、鱼鹰和两栖登陆艇夺回被占领岛屿的训练，很可能将是首批行动的部队之一，以对抗中共对日本岛屿的任何攻击。

文章分析，联合演习也是在俄罗斯入侵乌克兰引起东亚地区新的安全关切之际举行的，中共在镇压香港之后对台湾施加压力。日本还面临着俄罗斯从远东基地行动的部队，而俄罗斯的部队与中共军队的合作越来越多。
