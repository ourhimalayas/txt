**2.美国房屋交易在5月意外反弹**

[据美国房地产经纪人协会（NAR）公布的待售房屋销售指数显示，美国房屋交易在5月意外反弹。根据出售房屋的合同数量，美国5月房产交易量比4月上升了8%。美国所有地区的合同签定量均较4月均有所上升，该指数达到2005年以来的最高水平。据悉，4月房屋交易量较3月下降4.4%。根据彭博社分析师的一致估计，这一反弹令人震惊，远远超过预期的1%的降幅。另据标普周二发布的美国最新全国房价指数，4月房价飙升至新高，创下30多年来最快速度。NAR发布的数据显示，5月末美国房屋库存总量为123万套，比4月份增加7.0%，较去年同期下降20.6%。](https://finance.yahoo.com/news/pending-home-sales-may-2021-135934665.html)
