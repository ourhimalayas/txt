据中共天津市疫情指挥部12月13日消息，天津市疾病预防控制中心对12月9日境外输入的新冠病毒无症状感染者呼吸道病毒测试分析，经过中国疾病预防控制中心二次复检，确认从这名无症状感染者中检测出奥密克戎（Omicron）变异毒株(B.1.1.529进化分支)。
![](https://assets.gnews.org/wp-content/uploads/2021/12/WhatsApp-I46-AM.jpeg)图片来源：微博天津
**据悉，这是中国内地发现的首个奥密克戎（Omicron）感染病例。**

2021年11月9日，奥密克戎（Omicron）变异毒株在南非首次被发现。**相关研究数据显示，奥密克戎毒株具有很强的疫苗逃逸能力，且传染力比德尔塔病毒增加了37.5%，群体间传播速度更强。**

截至目前，**奥密克戎（Omicron）变异毒株已在全球包括中国在内的64个国家和地区出现。**辉瑞CEO Albert Bourla近期表示，预计在接下来的几周内，奥密克戎确诊病例数量将从几十例激增至数百万例。
