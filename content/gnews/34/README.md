《国会山报》12月14日报道，周二，来自两党的立法者都表示支持西藏自治，呼吁拜登政府推进一项有助于解放西藏的政策，该领土仍受到北京中共的严格控制。

众议院和参议院立法者在给政府的一封信中写道：“几十年来，西藏在美国国会获得了两党的大力支持。倡导西藏人民的权利和尊严对任何有原则的外交政策都至关重要。”这封信是在负责公民安全、民主和人权事务的副国务卿乌兹拉·泽雅（Uzra Zeya），在她被任命为拜登总统的西藏问题特别协调员之前发给她的。这封信由参议员马可·鲁比奥（R-Fla.）和帕特里克·莱希（D-Vt.）在参议院领导，在众议院由众议员吉姆·麦戈文（D-Mass.）和克里斯·史密斯（R-N.J.）领导。

议员们主张采取近十几项他们随时准备支持的行动和政策，包括拜登邀请达赖喇嘛访问白宫，扩大美国对西藏的接触，重振美国对西藏的外交关注，并号召盟友加强美国对西藏的立场。议员们继续提到：“西藏对我们很重要，不仅是一个对我们的选民和美籍藏族社区很重要的问题，而且也是一个优先考虑人权和追求人类尊严的有原则的外交政策的具体体现。我们随时准备在这个紧迫的道德问题上与你合作。”

这封信受到国际西藏运动（ICT）的赞扬，该组织总部设在美国和欧洲，专注于西藏人民的人权和民主自由。ICT的政府关系主管弗朗兹・马茨纳（Franz Matzner）表示：“这些信件为下一任特别协调员提供了一个重要框架，以解决中国在西藏的压迫问题，同时明确表示，国会期望拜登政府迅速采取有意义的行动，支持西藏人民。”

北京于1951年吞并西藏，并声称它是中共国不可分割的一部分。但独立活动人士及其支持者表示，中国的统治压制了藏人的宗教生活和文化。人权组织表示，中共国当局实施了许多侵犯人权的行为，包括使用致命武力镇压抗议活动，任意拘留和酷刑——最明显的是自2008年以来，北京对西藏支持独立的抗议活动进行了残酷镇压。

美国对西藏的直言不讳和强有力的支持可能会引起北京的强烈谴责反应，北京认为这些言论是外国对中共国内政策的干涉。拜登政府试图召集盟友和伙伴，以更有力地对抗它提出的北京侵犯人权的行为，特别是针对中共国新疆省的维吾尔族少数民族，美国已将这些行为称为种族灭绝。

美国政府本月早些时候宣布，它将对定于2月在北京举行的2022年冬季奥运会进行外交抵制，以反对中共国的人权侵犯行为。拜登于11月与中共主席习近平举行了一次虚拟会议，根据白宫的一份声明，他“对中共在新疆、西藏和香港的做法，以及更广泛的人权问题提出了担忧。”
