基辅和莫斯科之间关于乌克兰-白俄罗斯边界可能的和平解决方案的谈判正在进行中。尽管如此，俄军仍在继续进攻。

俄罗斯总统弗拉基米尔·泽连斯基的顾问米哈伊洛·波多利亚克在短信中说，俄罗斯和乌克兰之间的谈判已经开始。泽连斯基的办公室此前曾表示，会谈的目的是立即停火并让俄罗斯军队从乌克兰撤出。据乌克兰报道，这还包括俄罗斯自 2014 年以来吞并的克里米亚和顿巴斯的分离主义地区。

俄罗斯总统弗拉基米尔普京周一下令武装部队继续袭击邻国，尽管会谈。战斗继续没有改变，双方都声称成功。这无法独立验证。
