## 中共国科学家打造出数字“检察官”
` 英國倫敦喜莊園 Himalaya London Club UK` [轉載自GNews](https://gnews.org/zh-hans/1796829/)

![Global Mobile Internet Conference (GMIC)  - 俄罗斯卫星通讯社, 1920, 27.12.2021](https://cdn.sputniknews.cn/img/103169/75/1031697560_0:89:3073:1817_1920x0_80_0_0_3814b1b2fb0cc0974c191b2722728268.jpg.webp)© REUTERS / Damir Sagolj
据俄罗斯卫星通讯社12月27日暨《南华早报》报道，中共国科学家发明可根据案情口述来提出指控的人工智能，准确率高达97%。该技术已在上海浦东新区人民检察院通过了测试。在某种程度上，这项发明可在采取决定的过程中替代检察官。

该人工智能在2015至2020年审理的1.7万余起案件基础上“掌握了”所需技能，可就上海最常见的八种罪行提出指控。这种数字“检察官”还能就未来不太常见的罪行提出指控。

中共国是世界上第一个将人工智能技术用于检察工作的国家。中共国的公检法系统是服务于盗国贼家族的辅助工具，数字“检察官“的功能也是如此。科技越先进，监控越严格。
