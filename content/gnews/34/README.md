## 三星撤了，中国梦也该断了
` 银河勇气之星` [轉載自GNews](https://gnews.org/zh-hans/1542373/)

![](https://assets.gnews.org/wp-content/uploads/2021/09/未标题-1-1.png)图片来源于网络![](https://assets.gnews.org/wp-content/uploads/2021/09/12-7.jpg)图片来源于网络
有着三四千员工的三星重工宁波公司撤离中国，尽管工人拉着令人潸然泪下的“感恩三星十几载，不放弃，不抛弃”的横幅强烈要求不要走，但她还是毫不犹豫地走了，留给国人的是一个决绝的背影。岂止是三星，阿迪达斯也因为新疆棉的影响，近四分之一的在华企业将从中国转移。还有日本东芝大连公司，一汽马自达、彪马、夏普、起亚、微软、索尼……等等。
![](https://assets.gnews.org/wp-content/uploads/2021/09/11-11.jpg)图片来源于网络![](https://assets.gnews.org/wp-content/uploads/2021/09/13-5.jpg)图片来源于网络![](https://assets.gnews.org/wp-content/uploads/2021/09/14-7.jpg)图片来源于网络
“风萧萧兮易水寒，壮士一去兮不复还。”在几十年坚持不懈的抵制X货运动，终于让这些企业夹着尾巴逃走了，小粉红们赢了，义和团们也胜利了。可是这些工厂的员工呢？当他们卷起铺盖离开容不下他们的城市准备踏上回家的路途时，发现曾经生养他们的农村已经满目疮痍，百业凋敝，再也回不去了。

世界上最遥远的距离，不是生与死的距离，而是，我无路可走时，耳边只有一首歌陪伴：心若在梦就在，天地之间还有真爱；看成败人生豪迈，只不过是从头再来。

真想问一问刘欢，梦在哪里？到哪里从头再来，十多年的青春，从青年到中年，从中年到老年，以前哪怕苦一点，还有一个容身的场所，可以凭一身蛮力养家糊口，安身立命。现在他们撤了，撤向了自己的国家和东南亚的国家，留给这些员工的是空空的行囊。就是摆个地摊也要被抢走称砣，推倒三轮，甚至把人狠狠往地上死命地砸，天地之间除了蛮横的城管，只有生活一地鸡毛的零乱。哪怕是做个三陪，也因新冠而门庭冷落，即使想凭一身蛮力买把斧头做个“生意”，却因实名而一刀难求。

即使如此，还有媒体不停地自说自话地讲着“励志”的故事：80岁小伙子，推着一辆破旧的三轮车，自力更生，年入百万，把日子过成诗……这难道就是所说的“人类命运共同体”还是“同一个世界，同一个梦想”？

王朔说：世界上最阴险、最歹毒的正能量，就是用穷人的艰辛和苦难，当做励志故事愚弄底层人。

林肯说：你可以在所有的时间欺骗一部分人，也可以在同一时间欺骗所有的人，但你不可能在的所有的时间欺骗所有的人。

二十年前，中共以双赢的口号，以各种优惠作为诱饵，使得外企纷纷来中国开办工厂。二十年来，外企带来了先进的技术，带来了先进的管理理念，在给中国创造了许多财富的同时，他们自己也获得了相对丰厚的利益。本以为双赢的事情，可以天长地久，却不料，中共把从外企在这里办厂所得财富变成了一颗颗射向他们国家和自身企业的子弹，不仅要夺了他们的财富，而且还要灭了他们的种族，以此来实现共产主义一统世界的阴谋。

当爆料革命的影响力越来越深远，当世界各国的正义力量逐渐集结，中共的画皮本质也逐渐被揭得一丝不挂。撤离，似乎是所有企业保护自己同邪恶划清界限的规定动作。

不要问我恨你有多深，撤离代表我的心。

没钱抢了的中共，还靠什么来给没饭吃的韭菜洗脑和维稳？毕竟你自己作死，猛踩倒车油门，再蠢的人都不愿跟着你掉进万丈深渊而万劫不复。毕竟，时代变了，再也回不去文革全民疯狂的年代了。
