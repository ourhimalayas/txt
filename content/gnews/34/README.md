## 美国敦促奥运选手使用一次性手机，以防中共的间谍活动

据《布雷特巴特新闻网》（Breitbart News）1月14日报道，为了防止中共在种族灭绝奥运会期间的间谍活动，美国奥运代表队建议美国运动员把他们的个人电子产品留在家里，并使用一次性手机。
![](https://assets.gnews.org/wp-content/uploads/2022/01/WhatsApp-Image-2022-01-15-at-13.54.20.jpeg)图片来源：原新闻
据《今日美国》（USA Today）报道，美国奥运队在最近的一份咨询备忘录中告诉美国运动员说：“与电脑一样，手机上的数据和应用程序也会遭到恶意入侵、感染和数据泄露。而且在冬奥会期间有被中共监控的风险。”

美国并不是唯一一个开始警告其运动员在去北京参加即将到来的奥运会时要留下个人电脑和电子设备的国家。与美国一样，荷兰奥委会、英国奥林匹克协会、加拿大和澳大利亚奥委会也同样警告他们的运动员注意中共的电子间谍活动。

加拿大警告参赛选手要“格外警惕”他们的个人信息。加拿大奥委会说：“我们已经提醒所有加拿大队成员，奥运会为网络犯罪提供了一个独特的机会，建议他们在奥运会上要格外注意，包括考虑将个人设备留在家里，限制存储在设备上的个人信息，并随时保持良好的网络卫生（cyber-hygiene）。”

事实上，运动员到中共国后，中共当局已完全公开地收集了他们的个人信息。中共奥运当局宣称：“除非有必要，否则北京冬奥运会收集的个人信息不会被披露。”但问题是他们将根据谁的定义来决定何时“有必要披露”？

来北京参加冬奥会的运动员所面临的威胁远远不止间谍活动和被监控，郭文贵先生在大直播中屡次强调，中共很可能在冬奥会期间对外国运动员投毒，这绝非空穴来风。在2019年10月于武汉举行的世界军运会期间，加拿大、法国、意大利、德国等许多国家的运动员就染上了不明原因的疾病，有些人甚至直至今日仍受其困扰。世界不可低估中共的邪恶，全面抵制北京冬奥会才符合最佳利益。
