## 法国千万人注射三针疫苗 单日新增感染仍破五万
` GTV新闻访谈` [轉載自GNews](https://gnews.org/zh-hans/1726469/)

作者：东京樱花团 – 在远方
![This image has an empty alt attribute; its file name is Screen-Shot-2021-08-30-at-7.37.59-PM.png](https://assets.gnews.org/wp-content/uploads/2021/08/Screen-Shot-2021-08-30-at-7.37.59-PM.png)
法国近日宣布，超过1000万民众已经接种了第3针疫苗加强针。12月4日，政府表示，在过去24小时内，仍新增5万感染病例。其中694人住院治疗，119人病情危急，113人死亡。
![](https://assets.gnews.org/wp-content/uploads/2021/12/6-11.jpg)
据悉，两周前，法国平均单日新增病例28000起，一周前猛增至平均每天41000起。大面积接种却造成单日感染暴增，法国卫生部长维宏仍极力煽动民众，继续接种所谓加强针剂。

维宏公布，明年1月15日后，法国18至64岁民众，必须出示疫苗加强剂证明，而且接种时间不能超过第2针的7个月期限，以维持疫苗护照的效力。民众出入公共场所，必须出示疫苗护照。

资料显示，截至目前，法国已有792万7361人感染，死亡12万零440人。
