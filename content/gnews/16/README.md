我最近在为GClubs 晚会练习唱歌，突然间就在前天就有了感觉，所以爆料革命真的是得天助啊！我接着说那个反华排华的事情，我的歌也是很好的例子。就好几个海外的华人带着孩子参加了那个“好声音”，说我们也看了，想带着孩子参加《我是音雄》，说但是七哥我想要喜币，说实在话如果他想参加爆料革命或者想唱歌，他不要求我也会给他喜币，但是他说完，我说门都没有，拒绝了。我说你保护好你自己的命就不错了，我问，你在旧金山打疫苗了吗？他说打了。我问，你孩子打了吗？他说打了。 我说，那你的孩子没有机会参加爆料革命了，再见吧。他作为华人在海外生活二三十年，他的家庭也是被共产党残害的，可是当孩子长大，他带着孩子回到中国去参加《好声音》！现在看到爆料革命逐渐壮大就来要喜币，甚至还打了疫苗，兄弟姐妹们，你去想想，他怎么能不被排华呢？

你看看，华人在加拿大起诉大卡车司机的事情。好多加拿大的白人给我发信息，一个哥们儿问我，“你们中国人疯了吧？我们在跟共产党的疫苗作战，你们中国人来起诉我们的卡车司机还要赔偿？”这哥们儿很愤怒。我说，你看到的不是真正的中国人，那是共产党的中国人，新中国联邦的中国人不会有一个人像这样的！这个是加拿大人，他说我一定要跟像这样子的中国人算账！这位加拿大人曾经是中国和加拿大两国的军事合作局局长，他曾经亲共，后来他受到我和爆料革命的影响开始反共。在跟我的聊天中，他震惊于加拿大华人的堕落，他说我现在在温哥华，看到所有的华人我都觉得很奇怪。兄弟姐妹们，这种现象很吓人。你再看看西雅图，那么里有好多华人，我的美国朋友说，你知道我们周围的华人有多奇怪吗？他说这些奇怪的华人讲着完全听不懂的英文，并经常说自己的国家是多么的伟大，他说，伟大那你回中国去呀！在圣地亚哥，我的美国朋友说我看到任何亚裔人，我都觉得他们是来窃听我们的航空母舰和潜水艇的。圣地亚哥是美国军事基地的所在地，我听到后感到毛骨悚然，我说你不能这么看，亚洲人是无辜的，都是共产党造成的。类似的事情也发生在德国、英国和日本。比如在日本，我的一个日本朋友说，我现在很害怕，在我们丰田内部，所有日本人看中国人的眼神都变了。这些日本人都是曾经对中国很友好的人，他们一旦看到中国人眼神就变了。我的一个台湾高雄的朋友住在日本，有家庭有事业。最近他说我要搬回台湾去。他说，我身边跟我很好的日本人经常问我，你是台湾人你支持共产党吗？他说我没法在日本呆下去了。

排华无处不在，排华现象最惨的最危险的会发生在三个地区，俄罗斯东欧国家、印尼东南亚和非洲。非洲是太危险了！在排华这件事情上，七哥我一点信心都没有，我们努力吧！
