#### **英伟达股票正在反弹**

英伟达在2022年有一个艰难的开始。即使该股在周五走高，下周也为更大的转机提供了潜在的催化剂。芯片制造商英伟达（股票代码：NVDA）将在下周一至周四举办名为GTC的旗舰技术会议，与首席执行官黄仁勋的主题演同时进行。对于一家今年股价下跌13%的公司来说–尽管该股在过去12个月里仍然上涨超过100%，并在2021年11月飙升至历史最高点–这些活动是一件大事。

花旗银行分析师Atif Malik和Amanda Scarnati在周五的一份说明中说，焦点将是一波新产品，因为该公司往往不提供目标财务模型。

花旗的团队强调了去年英伟达是如何通过宣布强大的新Grace计算机处理单元（CPU），以及提出人工智能软件货币化的前景，给投资者带来惊喜。对关键的游戏行业的销售和来自汽车行业的80亿美元的合同管道的更新是另一个额外的好处。

英伟达一直是云计算和人工智能基础设施消费热潮的受益者。该集团还因其在metaverse（描述虚拟世界的新兴平台和技术）中占据主导地位的举措而受到推动。但是，英伟达也不得不放弃从软银手中收购芯片设计公司Arm，其中包括一笔巨额的分手费。

Atif Malik和Amanda Scarnati预计，下周会有四个关键的发展。其中任何一个都可能被投资者扑捉到。

*第一个是宣布用于数据中心的新的5纳米处理单元，用于游戏的5纳米单元可能在9月公布，以及关于强大的Grace CPU的更新。*

*第二条是元空间方面的消息，预计英伟达的Omniverse会有新的机会，其三维虚拟世界模拟和协作平台有广泛的工业应用。*

*第三条是与电动汽车行业发展相关的销售管道的更新。去年，汽车制造商NIO(NIO)与英伟达合作，将其芯片用于开发自动驾驶解决方案。*

*第四项是以投资者为中心：股票回购。”在Arm交易失败后，投资者预计会有一些回购，”花旗银行的团队说。*

*Malik和Scarnati将英伟达评为买入，目标股价为350美元。周四该股收于247.66美元，这意味着有超过40%的上升空间。*

*分析师们大多看好英伟达；接受FactSet调查的经纪人绝大多数将该股评为增持，平均目标价为340美元。*

#### **美联储对GDP和通货膨胀的良性预测忽视了经济衰退的风险**

联邦公开市场委员会在上周将其联邦基金利率目标范围提高了四分之一个百分点，至0.25%-0.50%。同时，该政策制定小组公布了一份经济预测摘要，与12月发布的上一份经济预测摘要相比，通胀和联邦基金预期大幅上升，2022年的预期经济增长大幅下降。

FOMC预计*通胀率将温和走低–不会对国内生产总值或失业率造成任何损害。它预计2023年的通胀率为2.7%，12月的展望为2.3%，接近美联储2%的长期目标。这也将低于2022年的预期4.3%，*与个人消费支出平减指数–美联储首选的通胀指标–在截至1月的12个月内上涨5.4%相比，仍是一个巨大的进步。(PCE的数字明显低于更广泛关注的消费者价格指数，后者的最新读数为7.9%，处于四十年来的高位）。

根据美联储的预测，*今年和明年的失业率保持不变，为3.5%，而2024年的失业率预测被提高了0.1个百分点，达到3.6%，这一点并不明显。*

2022年实际国内生产总值增长的预测中值被调低，*从4.0%降至2.8%，这基本上包含了今年已经发生的事情，而2023年和2024年的国内生产总值预测与12月的预测持平，分别为2.2%和2.0%*–尽管联邦基金利率可能会比之前的预期更大幅度地收紧。

AllianceBernstein的前首席经济学家Joseph Carson补充说，*要在2022年底之前将年度通胀率从目前的速度降低4个百分点，就需要比美联储预期的额外6个季度加息（到12月达到1.75%-2%）更严厉的紧缩政策。*根据圣路易斯联储的数据，国债通胀保值证券市场认为未来五年的通胀率远远高于美联储的预期，约为3.5%，这是2003年1月以来的最高值。

罗森伯格研究公司的同名作者大卫-罗森伯格尖锐地警告说，*美联储的紧缩周期在历史上有75%的机会引发经济衰退。*

毫不奇怪，美联储的老板杰罗姆-鲍威尔对前景的看法要良性得多。他将美国经济描述为令人难以置信的强劲，其特点是劳动力市场强劲，每一个失业者有1.7个职位空缺。政策紧缩的另一个组成部分，即美联储资产负债表的流失（耗费流动性），将相当于再次提高联邦基金利率。这一部分政策调整可能会在5月3-4日的FOMC会议上宣布。

#### **最大的比特币基金有一个问题——只有SEC能解决这个问题**

Grayscale Bitcoin Trust（股票代码：GBTC）持有262亿美元的资产，它拥有的加密货币比任何其他基金都多。作为一个像股票一样交易的私人置换信托，GBT已经成为以股票形式获取比特币的流行工具。摩根士丹利和ARK Invest等机构投资者在基金中为客户持有它。而且它看起来很便宜，交易价格比其资产净值折价27%。如果GBT的股价与它的资产净值一致，即使没有比特币的推动，投资者也会有大约37%的收益。

然而，自2021年2月以来，GBT的折扣一直存在，导致该基金的表现大大低于比特币。从那时起，GBT的总回报率下降了约40%，而比特币的价格却下降了15%。GBT 2%的费用率也受到了影响，而直接拥有比特币并不产生费用。

通常情况下，对冲基金会在像GBT这样的封闭式基金中进行套利游戏。他们建立股权，并向基金发起人施压，要求其以资产净值回购股票。然而，Grayscale的基金章程阻止了积极的投资者。Grayscale的母公司Digital Currency Group已经以市场价格收购了6.98亿美元的GBT，但这并没有缩小与资产净值的差距。

这就留下了另一个解决方案：将GBT转换为交易所交易基金。在大多数ETF中，相关资产的交易与股票价格一致。但赢得批准的比特币ETF在证券交易委员会遇到了障碍。

在民主党主席Gary Gensler的领导下，SEC已经拒绝了几个比特币ETF申请，包括Fidelity、Valkyrie和VanEck的提议。Gensler对加密货币采取了强硬的立场，呼吁对代币和交易所进行更多的监管，并敦促国会通过法律来控制他认为的行业的 “狂野西部 “做法。SEC拒绝就Gensler对ETF的做法发表评论。

该机构否认的主旨是，*比特币现货市场容易受到欺诈和价格操纵的影响，可能会蔓延到基于现货的ETF。比特币在全球范围内的交易主要是在不受监督的交易所和分散的平台上进行。*相比之下，比特币期货在芝加哥商业交易所交易，并受到商品期货交易委员会的监督。此外，在美国证券交易委员会看来，旨在上市比特币ETF的美国证券交易所还没有满足美国证券交易委员会关于与基础比特币市场签订 “监督共享协议 “的要求，也没有提出令人信服的豁免理由。

美国证券交易委员会通常为ETF提案设定240天的最后期限，因此可能在6月中旬做出决定。共和党任命的美国证券交易委员会专员Hester Peirce长期以来一直敦促她的民主党同事批准比特币ETF，*其部分理由是比特币期货市场可能会引领现货市场的价格，使比特币交易所交易基金很难操纵价格。*
