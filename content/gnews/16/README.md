— 一年多以前，有大量经过同行评审的随机对照试验为伊维菌素在每个疾病阶段治疗中共病毒（ COVID-19） 的功效提供了强有力的证据。

— 利物浦大学的主要作者希尔（ Andrew Hill） 博士为世界卫生组织的中共病毒指南制定小组撰写了一篇，被认为经过多项研究的论文。希尔是伊维菌素的早期积极支持者。他的论文表明，如果在世界范围内使用伊维菌素，可以将死亡人数减少 75%。

——莫名其妙，就在发表前几天，这篇论文出现在预印本服务器上，结论却发生了变化。与其认为伊维菌素——世界上最安全、最便宜的药物之一——应该在全球范围内推广，它现在得出的结论是，需要对伊维菌素进行更多的研究，然后才能在全球范围内推荐伊维菌素。鉴于伊维菌素的全部科学证据，希尔博士对这一结果进行了惊人的——实际上是令人震惊的——逆转。

![](https://assets.gnews.org/wp-content/uploads/2022/03/image-1253.png)

在循证医学咨询主任 Tess Lawrie 博士发起的紧急[视频会议中，希尔 博士向她承认，他的研究赞助商之一](https://www.oraclefilms.com/alettertoandrewhill)**Unitaid**对他的论文结论有发言权. 但他不会透露改变论文结论的人的姓名。

**但现在，****Substack ****上的****“****挖掘者****”****（又名制片人****/****导演菲尔哈珀）透露了可能编辑论文结论的人的姓名****——****这导致世界卫生组织不推荐使用伊维菌素。这一决定可能导致全世界数百万人不必要的死亡。**

**叹息…**

哈珀先生研究了该论文的 PDF 文件，想了解其“幽灵”作者的身份。“希望 PDF 上的一些人工痕迹会揭示一些东西，也许字体不同，也许有隐藏的评论，也许一些跟踪的更改已保存到文档中，” 哈珀说， “这些调查没有任何结果。”

然后他灵机一动。会在 PDF 的元数据中吗？“有时这是最明显的事情，” 哈珀写道。“论文的‘v1\_stamped’版本确实有元数据。它甚至在元数据中包含作者信息。我期待看到安德鲁希尔在作者一栏，相反，我看到了一个我认识的名字：安德鲁欧文。

“除非有人使用他的电脑，否则安德鲁·欧文在安德鲁·希尔的论文上留下了他的数字指纹。”

![](https://assets.gnews.org/wp-content/uploads/2022/03/image-1255.png)

**据称，安德鲁****·****欧文教授编辑了安德鲁****·****希尔关于伊维菌素的批判性论文。他还从具有竞争产品的制药公司收取咨询费。**

事实证明，安德鲁.欧文（Andrew Owen ）是利物浦大学的药理学和治疗学教授，也是长效治疗卓越中心 (CELT) 的联合主任。他还是世界卫生组织中共病毒 COVID-19 指南制定小组的科学顾问。就在希尔博士的论文即将发表前几天，该论文的赞助商**Unitaid**向 CELT 提供了 4000 万美元的赠款，其中欧文是项目负责人。*“*这份价值 4000 万美元的合同实际上是**Unitaid**、利物浦大学和Tandem Nano Ltd（一家将 ‘固体脂质纳米颗粒’递送机制商业化的初创公司）之间的商业协议——安德鲁·欧文是该公司的最大股东，” 哈珀说.

![](https://assets.gnews.org/wp-content/uploads/2022/03/image-1257.png)

此外，哈珀写道，“安德鲁·欧文在从制药公司收取资金方面花样繁多。他获得了 ViiV Healthcare、默克、杨森、勃林格殷格翰、葛兰素史克、雅培、辉瑞、阿斯利康、蒂博泰克、罗氏制药和百时美施贵宝的研究资助。”
