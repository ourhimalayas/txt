另一方面，我们也要小心谨守, 自以为站得稳的,須要謹慎,免得跌倒。我们彼此提醒互相代祷。若不与主连接就不能结果子，属灵生命会枯干，就不能作什么了。类似的，面对病毒，每个人都都需要谨慎，作好防护避免感染，小心备药以备不时之需。另外小心周围有匪类、潜伏在我们中间的假弟兄，会透过不同的方式收集我们的个人信息，我们应该灵巧谨守的对付，务必小心保護自己個人的信息，当我們在各個农场与小組的聚集時，彼此提醒不要收集或是透露戰友的個人信息。

不过我们也不可让恐惧完全占居我们的心，我们蒙神的呼召，是有使命来传福音建造门徒。船在港口里面虽然是最安全的，但是却不是船被造的目的。圣经给我们最大的提醒是“不要惧怕”，我们在恐惧中不能思考，我们可以靠主刚强，因为神赐给我们，不是胆怯的心，乃是刚强、仁爱、谨守的心。
