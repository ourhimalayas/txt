![](https://assets.gnews.org/wp-content/uploads/2022/01/image-2136.png)[图片来源](https://www.aa.com.tr/en/economy/imf-revises-down-global-growth-projection-to-59-for-2021/2390043)
国际货币基金组织（IMF）预测，今年世界经济整体将增长4.4%，比去年少1.5个百分点。 这是因为美国长期的通货膨胀和中国严格的“清零”政策，将拖累全球经济增长速度。

国际货币基金组织25日发布了最新的经济展望，预测今年全球经济增长将为4.4%，比去年下降1.5个百分点，去年由于疫苗的普及，经济曾出现了强劲反弹。

美国预计增长4.0%，低于去年的5.6%，原因是供应链中断导致的长期通货膨胀，以及拜登政府的一项标志性政策《重建美好未来法案》通过的不确定性。

中国将从去年的8.1%降至4.8%，政府的“清零政策”限制了个人消费增长，房地产行业的萎缩也带来来消费减缓。

欧元区预计也将从去年的5.2%下降到3.9%，英国从7.1%下降到4.7%。

另一方面，日本由于长时间限制经济活动使去年的增长止步于1.6%，预计今年将提高到3.3%。

IMF表示，未来美国货币政策收紧可能对新兴经济体的发展带来一定风险，预计明年全球经济增长将比今年进一步放缓至3.8%。
