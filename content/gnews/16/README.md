【[#环卫工蹚水背学生过马路获奖五千#](https://s.weibo.com/weibo?q=%23%E7%8E%AF%E5%8D%AB%E5%B7%A5%E8%B9%9A%E6%B0%B4%E8%83%8C%E5%AD%A6%E7%94%9F%E8%BF%87%E9%A9%AC%E8%B7%AF%E8%8E%B7%E5%A5%96%E4%BA%94%E5%8D%83%23&amp;from=default)：只是举手之劳】9月15日，甘肃兰州。一中学校门口在大雨后积水很深，淹至道牙，当时马上到上课时间，一群学生在马路边不便过去，环卫工人王兴德、王兰英看到后，轮流背着学生们过马路。王兰英称，和搭档王兴德各背了七八个学生，同学们都表达了感谢，觉得这就是举手之劳。事后，[@天天正能量](https://weibo.com/n/%E5%A4%A9%E5%A4%A9%E6%AD%A3%E8%83%BD%E9%87%8F?from=feed&amp;loc=at) 为两人颁发5000元特别奖励，点赞这份善良。[@白鹿视频](https://weibo.com/n/%E7%99%BD%E9%B9%BF%E8%A7%86%E9%A2%91?from=feed&amp;loc=at) [L白鹿视频的微博视频](http://t.cn/A6MA6C5m)

得了5000元特别奖励的环卫工人王兰英说的是：当时那天早上是兰州雨下的特别大，那个污水井里面的脏水全都冒到马路上了，当时同学们过马路过不去，我在那个地方是搞环卫的，看见同学们着急的样子，那八点就要上课了，急急忙忙就背过去，他们就说谢谢阿姨，这是举手之劳呗，当时背了七八个，还有我的搭档也背了好像七八个。
![](https://assets.gnews.org/wp-content/uploads/2021/09/捕获_副本-14.png)
在背人事件完成后，天天正能量为两人颁发5000元特别奖励，我们特别注意到了，这份荣誉证书是由兰州晚报作为合作媒体的阿里巴巴天天正能量评选活动的产物。其实对于王兴德和王兰英两位环卫工人来说，能得到这样的奖励自然是好事一桩，对于两位热心肠环卫工人的义举也的确是令人暖心。

但是对于这样的正能量我们还是要说两句的。为什么雨稍微大一点路面就积水了呢？不但雨水积水，而且污水井里面的脏水全冒到马路上了，这说明兰州这个城市的污水和雨水是合流的，而且污水和雨水的合流系统完全无法满足正常情况下的降雨排放要求，所以微稍大一点雨就让排水系统不堪负荷，就把污水和雨水混在一起在路面上横流了。

那么流在马路上的污水是什么呢？它无非是厨房污水和卫生间的污水，厨房的污水经过出户的隔油池后汇集到污水系统，而卫生间的污水出户经过化粪池后也最终汇集到污水系统，所以在兰州路面上的污水是包含着人类粪尿和厨余污染物成份的污水，在兰州人民的日常生活中，在每一场因降雨引起的污水横流的马路上都流淌着含有粪尿成份的污水。

在这一场兰州晚报与阿里巴巴合作营造的正能量中，如果我们只看到了两位热心肠的环卫工人，而忽视那条含有粪尿成份的过水马路，或者说如果我们只看到了它们刻意营造出来的正能量，而忽视了城市管理者们对于最起码的城市市政建设的漠不经心，那么每个淌水过路的兰州市将永远与马路上的粪尿污水相依相伴。
