据[《目标新闻》](https://www.newstarget.com/2022-04-06-us-national-security-threatened-china-nuclear-arsenal.html)4月6日报道，美国战略司令部负责人就中共国战略核武库的惊人扩张向美国立法者发出警告。

在众议院的书面证词中，美国海军上将查尔斯-理查德表示，中共国在2021年7月首次测试一种新型洲际弹道导弹，严重影响地区战略稳定。中共使用了配备DF-ZF高超音速滑翔器的DF-17中程弹道导弹发射器。飞行了25,000英里，超过100分钟，在全球范围内盘旋，然后在其测试目标的20多英里范围内进行打击。

据理查德说，这是陆地攻击武器中最远的飞行距离和飞行时间。随着中共国最近对其核能力的大规模投资，它现在有能力与俄罗斯一起单方面将冲突升级到任何暴力程度。

中共国重点研发的是全球投送高超音速核武器，以及能够击败敌国的导弹防御系统及其反卫星、反导弹和反无人机能力。
