#### 撰稿：东京樱花团/喜马拉雅的微尘
![](https://assets.gnews.org/wp-content/uploads/2021/10/err.jpg)[图片来源](https://ee.ofweek.com/2021-09/ART-8130-2805-30525862.html)
日经新闻网10月15日消息，丰田已将11月全球生产计划削减约15％。受半导体不足和中国电力短缺双重影响，将减产约15万辆。此次与丰田9月下旬发布的最新的11月生产计划相比，日本国内减产约5万辆，海外减产5～10万辆。

据报道，东南亚的疫情趋于平稳，汽车零部件的生产正在恢复，但对11月的全球产量仍有影响。近期，中国发生的电力短缺导致的零部件生产低迷也产生了影响。预计产量与2020年11月（82万辆）处于相同水平。

9～10月，由于受到东南亚疫情再次扩大、零部件采购停滞影响，丰田已将计划产量削减4成。11月再次出现减产，但丰田仍力争2021年度的全球生产计划（900万辆）不发生调整。丰田力争12月恢复正常并挽回之前的减产。

针对12月的产量，丰田9月下旬向一部分供应商提出了比2020年12月高出3成的100万辆规模的全球生产计划。不过，丰田旗下供应商高管表示“半导体短缺的情况至今仍很严重”。在自9月起连续3个月减产的背景下，丰田年度的生产和财务目标是否能够实现仍值得关注。
