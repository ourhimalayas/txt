## 中共外交受辱，向星巴克泄愤

撰稿： 天鵝
![](https://assets.gnews.org/wp-content/uploads/2021/12/Untitled1-3.png)图片来源：unsplash
12月13日江苏无锡市场监管部门针对媒体曝光的两家星巴克门店有“更换配料标签使用过期食材”的问题，已经组织工作专班全面开展核查处置工作[1]。初步核实相关企业有更改食品原料内控期限标识、使用超过内控期限原料的行为，已责成2家涉事门店停业整改 。市场监管部门同步对全市其他82家星巴克门店开展排查，发现从业人员未戴工作帽、加工区物品摆放零乱、消毒记录不全等15处问题，均已责令整改。市场监管局对星巴克（中国）华东北区进行了行政约谈，要求公司对存在的食品安全问题进行全面自查、落实整改。

我们来看其他餐饮企业有没有类似的食品安全问题的报道呢，他们的问题与星巴克是否类似呢？

1、今年9月，北京市朝阳区市场监督管理局对“餐饮业食品安全大检查”处置情况进行了通报，其中涉及品牌包括瑞幸咖啡、喜茶、鲜芋仙、大鸭梨、攀成钢小郡肝串串香等。北京瑞吉咖啡有限公司双桥第一咖啡店（瑞幸咖啡双桥万达店），存在未建立食品进货查验制度、未登记食品进货验收记录。

2、今年10月，北京市、区市场监管局在北京全市范围内对连锁餐厅、“网红”餐厅、美食城等餐饮企业开展了食品安全大检查，发现部分连锁餐饮企业存在食品安全问题。华莱士检查时发现，未取得健康证明的人员却在从事接触直接入口食品的工作，涉嫌使用污秽不洁的食品原材料；杨国福麻辣烫店存在汤料污秽不洁等食品安全问题等。

可见墙内餐饮企业的食品安全问题普遍比星巴克糟糕的多，但却没有任何官媒引起重视，更没有全网声讨，墙内老百姓的身体早已被训练的五毒不侵了。

我们再看看无锡市市场监控部门对82家星巴克门店的排查结果，只发现了“未戴工作帽”“ 物品摆放零乱 ”“消毒记录不全”这三条拿的上台面的问题，看来星巴克的食品安全问题也并非宣传中的如此严重、吓人。但舆论方面却是咄咄逼人，此次事件从新京报记者卧底开始点炮，迅速以文章、短视频、长视频为载体疯狂发酵，短短24小时就传播至了墙内全网。媒体圈一致重点炮轰美国咖啡品牌星巴克的食品安全问题，人民网、环球网、新华社、五毛、自干五等纷纷加入到讨伐的行列，甚至还报道星巴克因为该事件股价应声下跌1%。我们看一下星巴克半年的走势就可以知道这完全是中共的宣传，可笑至极，低劣至极。
![](https://assets.gnews.org/wp-content/uploads/2021/12/Untitled-27.png)图片来源：google search
战友们和明眼人都知道这其实就是中共下三滥的招数，北京冬奥会被五眼联盟抵制了，老大哥俄罗斯也把他甩了，这下只能找个私人企业来出气，于是找上了星巴克。我们翻一翻中共之前的报道就看可以看出其可笑的操作。2020年8月6日深圳市市场监督管理局在其网址发布了“民生微实事丨社区食品安全社会共治建设走进应人石社区 ——“食品安全工作坊”活动走进星巴克咖啡”，该活动由深圳市食品药品安全委员会办公室、深圳市市场监督管理局、深圳市民政局指导，人石社区党委主办，其中写道“联合星巴克咖啡实施本次食品安全工作坊活动，以独特的形式向参观的社区居民展示了食品安全知识”[2]。

一年前还是食品安全的示范企业，被食品、药品等监控机构作为宣传窗口，一年后就遭到卧底、批判，中共这波操作实在是LOW爆了。中共一波波的卑鄙操作最终将会点燃美国私营企业家、美国人民的民意，随着美国中期选举的到来，中共必将自食其果。
