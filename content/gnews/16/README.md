在英国政府推出的一个项目后，英国公民正在敞开大门，开始接待乌克兰难民。首相官邸发言人和伦敦市长表示不会用私人住宅来接待难民，但考虑启用俄罗斯的豪宅。

据美国周刊3月14日报道，为了帮助流离失所的乌克兰人，英国政府周一推出了“乌克兰家园”网站。该网站允许英国公民在他们的家里接待乌克兰难民。

根据协议，政府每月支付每个接待难民的家庭350英镑（455美元）。在最初的五个小时内，有近44000人在该网站注册。住房和社区部长，迈克尔·戈夫（Michael Gove）在一份声明中说，英国一直在“支持在战争时期中那些最脆弱的人”，接待的数量“没有限制”。

首相发言人说，为了安全，建于17世纪的首相官邸唐宁街10号不会接待难民。 总理的乡间别墅，契克斯（Chequers）为私人所有，所以首相约翰逊无权决定将难民安置在乡村庄园。他没有直接问约翰逊是否愿意接待一个家庭。

英国其它知名政客已承诺通过该项目接收难民。交通部长格兰特·沙普斯（Grant Shapps）周一在推特上表示，他将向一名乌克兰人开放自己的家。他说：“过去几周，我们一家人在讨论乌克兰的灾难性局势。因此，我们打算今天申请与其他英国家庭一起提供我们的家园，为乌克兰人提供避难所，直到他们安全返回他们的国家”。

其它的一些政客表示，他们暂时不会参与这件事。其中包括伦敦市长萨迪克·汗 (Saidiq Khan)，他曾表示，目前他的家中没有空间收容难民。然而他提出了另一个想法；把乌克兰难民安置在受到制裁的俄罗斯寡头在伦敦的豪宅里。

这位市长周一告诉时报电台，“让人倍感心碎的是，俄罗斯寡头们购买的豪宅已空置多年，这些房子不是他们的家，而是用来洗钱的金砖。” 他提议，政府应该在出售这些豪宅之前扣押它们。因为在它们被用来安置逃离乌克兰的难民之前，还需要一段时间的整理才能启用。 “我们将在伦敦为乌克兰难民提供一个避风港，这将成为“诗意般的正义”。
