【路透社**4**月**6**日报导】  上海周二新增本土中共病毒阳性感染者逾**1.7**万人，连续第二日破万例，并连续五日创出新高。
![](https://assets.gnews.org/wp-content/uploads/2022/04/image021.png)
上海市政府副秘书长顾洪辉周三表示，从**4**月**4**日的全市核酸筛查结果来看，形势仍**”**严峻复杂**”**，**4**月**6**日在全市范围内再开展一次核酸或抗原检测。上海市委书记李强指出，要超常规加速方舱医院建设和集中隔离房源筹措。**

**顾洪辉在上午的疫情发布会上称，新一轮筛查是将前一次筛查发现的阳性感染者转运、收治起来，然后再做一次筛查，**”**在本次检测全部完成前，全市继续实行分区管控。**”****
**上海市卫健委公布，**4**月**5**日上海新增本土新冠确诊病例**311**例和本土无症状感染者**16,766**例，合计超过**1.7**万例，连续第五日创新高。
