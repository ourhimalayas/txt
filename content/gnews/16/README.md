## 中共决定制裁向台湾军售的美军工企业
` 首尔天池农场` [轉載自GNews](https://gnews.org/zh-hans/2041291/)

撰稿—八木
校对/上传—无明逆流

2月21日，中共外交部发言人汪文斌在例行记者会上宣称，中国政府决定对长期参与向台湾出售武器的美国军工企业“雷神技术”公司和“洛克希德·马丁”公司实施制裁。

![](https://assets.gnews.org/wp-content/uploads/2022/02/44.jpeg)图片取自网络

汪文斌表示，“美国向中国台湾地区出售武器，严重违反一个中国的原则和中美联合公报817公报的规定，严重损害中国主权、中美关系和台海和平稳定；为了维护中国主权和安全利益，根据中国《反外国制裁法》的有关规定，中国政府决定对长期参与美国向中国台湾地区出售武器的美国军工企业雷神技术公司和洛克希德·马丁公司的行为实施反制。”

![](https://assets.gnews.org/wp-content/uploads/2022/02/Wh45.jpeg)图片取自网络

此前，美国务院在2月7日宣布了一项对台军售服务计划。该计划决定对台湾出售价值1亿美元、为期5年的“爱国者”系统工程勤务计划，其中包含工程服务支持，用作改善、维持台湾的“爱国者”防空导弹系统。

美国务院军售服务计划公布后，中共外交部随即表示，“中方必将采取有力措施，坚定捍卫自身主权和安全利益，请大家拭目以待。”

据悉，这是拜登政府执政以来，第二次批准对台军售计划。
