据Summit媒体报道，加拿大已将资产冻结作为其紧急情况法案的一部分，该法案用于针对自由车队抗议活动的支持者，这是一项永久性措施。 为了阻止特鲁多政权所谓的“非法封锁”，政府威胁要冻结示威者和任何向他们捐款人的银行账户。

根据《紧急情况法》，银行可以在没有法院命令的情况下冻结账户，而所有众筹平台和支付提供商都必须向 FINTRAC（加拿大金融交易和报告分析中心）提供信息。

副总理克里斯蒂亚弗里兰宣布，许多针对抗议者“临时”采取的措施（在他们被适当地妖魔化为暴力极端分子之后）现在将成为永久性措施。

“我们使用了在援引《紧急情况法》之前拥有的所有工具，我们确定我们需要一些额外的工具，”弗里兰宣布， “其中一些工具我们将提出措施以永久使用这些工具。我认为，FINTRAC 的权威确实需要扩大到涵盖众筹平台和支付平台。”

罗纳德·里根再次被证明是正确的。 “没有什么比临时政府计划更持久的了。” 同时，正如我们之前强调的那样，此类措施可能会排除受保护的阶层，而移民和难民等群体则享有豁免权。
