撰稿：东京樱花团/asket
![](https://assets.gnews.org/wp-content/uploads/2022/02/1-80.jpg)
北京冬奥会开幕的2月4日上午，维吾尔人、香港人、西藏人和蒙古人团体在东京的中国大使馆前举行了抗议活动，抗议中共打压人权，呼吁人们抵制北京奥运，并发表共同声明。

共同声明指出，奥林匹克宪章中规定“保障人权的尊严”，但中共政权至今仍在持续进行人道主义犯罪。大规模强制收容维吾尔人、破坏藏传佛教的佛像和寺院、镇压法轮功修炼者的[群体灭绝](https://www.epochtimes.com/gb/tag/%E7%BE%A4%E4%BD%93%E7%81%AD%E7%BB%9D.html)行为丝毫没有改善。人权团体要求中共政权立即停止侵犯人权，保障信仰和言论自由。要求中共允许国际调查团体的到中国调查，保障参加冬奥会的选手们有表达人权问题的权利。

共同声明呼吁，今年正值日中邦交正常化50周年，日本政府不要被中共政权的“友好和亲善”之类的言论所迷惑，要密切关注实际的人权状况，以便做出正确的应对。
