![](https://assets.gnews.org/wp-content/uploads/2022/02/Unknown-14.jpg)亚马逊被要求赔偿因披露强迫劳动而入狱的中共维权人士 网络图片
据美国之音2月17日报道，美国两党及两院的国会及行政当局中国委员会（CECC）16日）致函亚马逊，要求该公司支持中共国劳工维权人士唐明芳向法院提出免除罪责和赔偿的请求。唐明芳因举报亚马逊的供应商衡阳富士康工厂侵犯劳工而被监禁两年。

唐明芳曾是衡阳富士康的一名物控工程师。他向总部设在纽约的中国劳工观察揭露富士康工厂违法使用学生工和派遣工，并不能给予其基本的尊严和福利。随后，他在2019年9月被逮捕，在审讯期间遭到刑讯逼供和殴打，被迫在认罪书上签字。2020年7月1日，衡阳法院以侵犯商业秘密罪判处唐明芳两年有期徒刑，并处罚金一万元人民币。

在中共国，有正义感并且付诸行为是危险的，轻则入狱，重则丢命。富士康公然对举报人采取报复行为，以及亚马逊的不作为，在中共国绝不是特例。中共国有劳工法，但实施不力。中共国的工会是针对工人的维稳机构，从来不会保护工人。像唐明芳这样敢于揭露工厂中的非法劳工和侵犯劳工行为，遭到的就是打击报复。在中共国，他人若有难处，很少有人敢发声，因为承受不了后果。这是中共体制的邪恶，只有推翻中共，中共国工人的权益才能得到保障。
