2021年4-8月份，中共国先后组织了年龄在18-60岁以及60岁以上国民接种毒疫苗，8-9月份又组织了12-18周岁青少年接种毒疫苗，10月至今，正在组织3-11岁儿童接种毒疫苗。据我所知，村里面绝大多数人都已经接种了毒疫苗。我曾与村民们讨论是否接种疫苗，有些村民并不认为疫苗有毒，但认为疫苗没有什么作用，他们说病毒具有强变异性，这次打的疫苗下次就不管用了，所以并不是所有国人都相信疫苗。那为什么我身边的村民接种率接近100%呢？我想这跟共产党打造的基层网格员制度让共产党牢牢控制住了基层密切相关。

什么是基层网格员？什么叫网格化管理？在这近半年的时间里，村里面经常用大喇叭一遍又一遍的广播网格员的含义：“一个自然村或一个社区是一个网格，在村里网格长就是村支书，网格员就是生产队长。”其实网格员制度很容易理解，这就好比抗日神剧里“皇军”到村里面扫荡，把乡亲们集中起来的就是网格员。亦可以追溯到《商君书》中将五户编成一伍，十户编成一什的连坐制度。共产党搞的网格员制度则是什伍制度、生产队模式的数字版、升级版。网格员制度的存在即维护了共产党对于基层的有效控制，又使得基层尤其是乡村的一部分人获得了在网格范围内强大的权力。在乡村，正是传统“差序格局”的亲属关系与共产党所授予网格员的权力结合，使得疫苗接种率非常高。

文贵先生在12月3日的大直播中对大陆同胞将要面临疫苗灾难以及经济崩溃所造成的混乱表示极为担忧，并号召所有战友加大力度传播疫苗灾难与青蒿素解药。如果我们对于基层网格员的运作流程进行研究，就可以寻找到一条传播病毒真相、疫苗解药的路径。

我亲身经历本镇执行疫苗接种的过程，在前段时间，每个村的大喇叭都由网格长也就是村支书广播接种疫苗。每天镇卫生院都有各村网格员代表在门口，询问你是哪个村的，指引你接种。对于没有接种的，则由网格员也就是队长，挨家挨户的找。大家可能以为村里大喇叭你喊你的，我反正就不打也可以呀。那么你听听大喇叭里是这么喊：“我们有的同志，不接种疫苗，以后可能要限制你的出入。人都打就你不打，你比人能！你不接中疫苗，我们以后可能由派出所，强制你打。到时候强制接种，你不光丢脸，可能还要花钱。”这不仅仅是宣传，更是威胁。如果你不知道疫苗有害，听了大喇叭绝对会打；你知道疫苗有害不想打，听了大喇叭你会十分煎熬；你对疫苗持保留态度，也大概率会打。村里面的恐吓和中共国官方的虚假宣传知情自愿打疫苗完全不一样，镇政府给了村里面接种率98%的任务，村里面绝对按照100%接种，村民也认为村里的意思就是政府的意思。当时我问一个村民，你不打不行吗？她说：“不打能行？村里到我家已经找了很多次了。”村民考虑的不光是疫苗安不安全，而是你不能得罪村干部，起码不能撕破脸，跟村干部的私人关系是否融洽在乡村太重要了。

当下，村里面如果因为纠纷报警，派出所先找到村支书联系再找当事人。未来的疫苗灾难中，这些网格员在基层依旧扮演重要角色。如果你遇到一个氏族关系浓厚，比较讲究人情的网格员网格长那还好说。如果面对一个奸邪的网格员村干部，那么我们必须有震慑他的力量。不久的将来，共产党失去对基层的控制之时，这些人依然是控制乡村的主流力量。

因此我对即将到来的疫苗灾难有几点建议。首先，利用村里面的舆论是一个古老有效的办法，村里坏事传的很快，让村干部有所收敛。其次，要与本片的邻居搞好关系，这也是为什么文贵先生常说疫苗灾难是对我们平日里做人做事的一场大检验。最后就是建好自给自足的小生态，有水有电有吃的。我举一个例子，我们那每年最冷的几天都会把水泵冻坏，自来水，井水都没有。我们家做生意的缘故，水泵放在室内，结果十几户邻居都来我家接水。如果周围邻里能互通有无，能屯粮的屯粮，能打水的打水，互相帮助，友好往来，谁都不希望出现文贵先生说的拿着玉米睡遍全村妇女的情况。我们不仅需要玉米，也需要能团结的人。

我所在地区因为大多数村民已经接种完疫苗，尤其是上学的儿童直接由学校组织接种，速度太快了，就好比纳粹毒气室，一下子全都进去了。我们要与亲友保持联系，不抛弃不放弃，从关心他们身体亚健康的角度出发，把青蒿素保健品推荐给他们。现阶段最紧要的就是根据自己的经济能力购买囤积青蒿素，有突发情况可以救急。
