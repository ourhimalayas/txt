2月28日，美国及其盟国对俄罗斯中央银行采取行动，阻止它动用6300亿美元的国际储备，并将一些俄罗斯银行从环球银行金融电信协会（Society for Worldwide Interbank Financial Telecommunication，简称SWIFT）支付系统中剔除，大大限制了俄罗斯从事国际银行业务的能力。俄罗斯陷入前所未有的国际孤立。
