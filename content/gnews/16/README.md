2、[**制定汽车安全系统标准应对司机突发疾病**](https://china.kyodonews.net/news/2021/11/29dc4624c64e.html)

共同社23日消息，日本国土交通省将对结合车内摄像头和自动驾驶技术的安全系统设置技术标准，以防司机驾车时突发疾病导致事故。2023年9月以后发售搭载安全系统的新车须符合标准。对象为所有汽车，包括巴士与卡车。国交省明年1月将修改《道路运输车辆法》的安全标准，敦促厂商研发和采用。

3、[**构建儿童数据库 汇总贫困与虐待信息**](https://china.kyodonews.net/news/2021/11/e4270648c17f--.html)

共同社23日获悉，为预防儿童贫困与虐待，日本政府已基本决定构建汇总家庭经济状况、儿童学习能力等各种信息的数据库（DB）。此举旨在尽快发现处于困境的儿童，以提供支援。政府力争最快2023年度在全国推行。

4、[**日政府将携手ILO支援企业人权应对**](https://china.kyodonews.net/news/2021/11/c6ac461509b7-ilo.html)

22日获悉，为了让在海外有供应链的企业强化人权问题应对，日本政府将提供支援，与致力于维护劳动者权利的国际劳工组织（ILO）合作。日本政府拟向联合国专门机构ILO出资，ILO通过派遣熟悉劳动法规的专家，为在亚洲地区开展业务的日本企业提供建议。政府还将制作包括交易伙伴在内的人权问题应对范例，用于强化日本企业整体对策。

5、[**东芝新技术可凭数码照片精准测算实际尺寸**](https://china.kyodonews.net/news/2021/11/fc1688484bf8.html)

东芝公司22日发布消息称，研发出凭数码相机照片数据测算被拍物体长度和面积的技术。用普通数码相机就能进行高精度测量。该技术需要准备多份从不同角度拍摄的物体数码照片。以照片的散焦程度为线索，通过人工智能（AI）进行分析，计算拍摄对象的大小与纵深。据称该技术为全球首创。
