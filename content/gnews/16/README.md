此举引发中国不满。中国把实行民主自治的台湾视为有待统一的领土。中国国台办发言人朱凤莲北京时间星期三说，中国“坚决反对美国和中国台湾地区进行任何形式的官方往来，敦促美方切实恪守一个中国原则和中美三个联合公报规定”，“不要向台独势力发出错误信号。”
