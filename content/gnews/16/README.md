26. Ambcrypto报道，尽管俄罗斯银行对加密货币的立场坚定不移，但当局已同意通过制定“路线图”来规范该行业。根据当地报纸的报道，该路线图可能要求俄罗斯投资者在进行加密货币交易之前进行测试。该法律草案的作者之一、政府最高官员安德烈•卢戈沃伊（Andrey Lugovoi）表示，数字资产监管法案将在春季会议上提交给国家杜马。而且，虽然该草案旨在规范交易和采矿，但它还将纳入强制性测试要求。报告指出，如果每年交易金额超过60万卢布，当局将通过测试检查“不合格投资者”。值得注意的是，测试仅对使用数字货币进行交易是强制性的，而不是对包括代币在内的其他加密资产进行交易。
