12月28日共同社消息：中共国籍原留学生，因试图根据军事命令非法购买软件，日本警方宣布通过国际刑警组织，对其实施通缉。

东京警视厅对36岁的原中共国籍留学生王建斌发出了逮捕令，王建斌的现住址和职业不详，现已离开日本。他曾试图根据一名军事官员的指示，非法购买涉及中国人民解放军的网络攻击相关的日本安全软件。因此，涉嫌欺诈未遂，据称警视厅已通过国际刑事警察组织（ICPO）发出国际通缉。

逮捕令称，2016年11月，在一名中共国军人的妻子的指示下，王建斌试图利用虚构的公司名称从东京的一家销售公司购买软件。而该公司的销售对象仅限于日本公司。当时，因为其公司注册无法得到确认，该软件销售公司拒绝了王建斌的订货请求。
