根据《华盛顿观察家报》4月19日的报道，一位美国高级国防官员称，五角大楼没有看到“任何迹象”表明俄罗斯正考虑在乌克兰使用核武器。

这位官员星期二告诉记者，虽然在俄罗斯入侵乌克兰以来的近八周时间里，人们一直担心俄罗斯可能会诉诸于使用小型核武器，但五角大楼并不认为这种部署“迫在眉睫”。“我们没有看到任何迹象表明俄罗斯正要使用核武器，我们每天都在尽可能地密切关注此事。正如你们以前听到我说的，我们相信我们拥有恰当的战略威慑。”

同样，当天晚些时候，五角大楼发言人约翰·柯比（John Kirby）表示，这是他们“几乎每天都在监测的事情，特别是考虑到普京先生在这场战争初期发表的升级言论。”

文章报道了相关背景事件，在2月底入侵乌克兰的几天后，普京将俄罗斯的核力量置于高度警戒状态。自那时起，克里姆林宫官员表示，该国面临的“生存威胁”可能迫使俄罗斯部署其核武库。

上周，中央情报局局长威廉·伯恩（William Burns）斯也警告说，如果俄罗斯的军事进攻继续不成功，它可能会诉诸于使用核武器。

而乌克兰总统泽连斯基（Volodymyr Zelensky）在接受电视采访被问及伯恩斯的评论时，他警告说，“所有国家都必须担心”可能发生的核事件。

此外，俄罗斯还威胁说，如果芬兰和瑞典申请加入北约，俄方将在波罗的海附近动用其核武库。
