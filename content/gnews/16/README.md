## 恒大的命运威胁着中共国影子银行系统

#### 编译：小红帽
校对：文泓
![](https://assets.gnews.org/wp-content/uploads/2021/10/打倒-14.png)
图片来源：雅典娜农场设计师：姐夫说说

中共国的恒大似乎准备好赚取一大笔现金。然而，这些现金最终会流向何处，谁也说不准。

**周一，这家摇摇欲坠的房地产巨头价值70美元（约96澳元）的上市子公司——恒大物业集团的股票在香港停牌。**中共国媒体立即将这一公告与另一家香港上市开发商合生创展的股票同时停牌联系起来，“等待发布有关该公司一项重大交易的公告”。

**媒体报道推测，合生创展计划以约51亿美元的价格收购该物业服务部门51%的股份，这意味着恒大将面临大幅减持。**

**上周，恒大以15亿美元将盛京银行20%的股份出售给沈阳当地政府，随后将出售其在恒大服务的股份。通过预期出售其服务业务61%的大部分股权，恒大将因此筹集超过40亿美元的资金。**

**恒大因流动性问题和（可能的）偿还能力危机而通过被迫出售（股权）来筹集大量现金，其债券持有人已不太可能看到。**出售盛京银行股份的所获收益，被该银行要求用作贷款偿还。

中共国当局还明确表示，他们的首要任务是确保恒大的困境不会引发银行危机，确保其预售公寓能够完工，并偿还散户投资者。

**恒大集团的债券和财富管理产品的投资者的命运一直悬而未决，这些产品为其提供3,050亿美元负债。同时，**恒大债券发行未付利息的清单几乎每天都在加长。

**虽然上个月恒大与投资者在一批境内债券中进行了某种形式的非现金结算，但恒大已经错过了8,353万美元和4,520万美元的利息支付，据彭博社报道，周一恒大未能兑现由相关实体发行的2.6亿美元票据的担保。**

虽然债券发行有30天的宽限期，可以将违约推迟到本月底，但对Jumbo Fortune Enterprises的贷款没有宽限期。正式违约可能引发恒大其他债券的交叉违约。

**在一周内，恒大集团还有三个系列的美元债券的利息支付总额约1.4亿美元，并在明年1月初还有超过10亿美元的利息到期需支付。**

恒大还错过了一些高收益理财产品的兑付（注：恒大严重依赖该产品作为短期资金的来源），这引发了众多投资者的抗议。恒大试图通过象征性付款和向投资者提供大幅折扣的公寓来代替现金以满足其中一部分人。

**中共当局似乎试图阻止恒大的内爆发展为房地产市场崩盘和社会动荡，并指示当地政府部门尽其所能接管和完成集团未完成的开发项目，并为“群体性事件”或大规模示威做好准备。**

预防骨牌效应并不容易。其他房地产开发商的股价一直在跳水，他们通过银行和财富管理渠道获得资金的渠道正在被限制。

除了指示地方政府进行干预外，北京当局还在不到两周的时间内向金融体系注入了超过1,200亿美元的流动性，同时敦促国有银行向购房者提供更多贷款，并支持作为中共国经济核心的房地产行业。

**然而，中共尚未直接做任何拯救恒大的尝试，而是试图影响其崩溃的速度并在一定程度上遏制其后果。**

**当局希望降低该行业的杠杆程度——正是他们自己的去杠杆政策切断了恒大获得资金的渠道并引发了危机——并摆脱了与之前救助国有企业或被视为大到不能倒的公司相关的道德风险。他们不会同情外国银行或债券持有人。**

恒大的困境对中共国影子银行系统的影响可能对中共当局构成挑战。过去几十年，尤其是自2008年全球金融危机以来，庞大的非银行金融体系是中共国房地产繁荣和泡沫的主要资金来源。

如果恒大要在该系统内引发挤兑，中共当局可能别无选择，只能进行更直接的干预。

恒大的资产出售将产生一些现金和流动性，使其能够满足中共当局的一些优先事项，但其负债的规模和性质以及危机引起的资产贬值，意味着它无法履行其所有义务或控制更广泛的影响。

**当前，**中共当局**面临的挑战将是将损失限制在外国债券持有人和富有的本地投资者身上，同时避免系统性问题、房地产价值（体系）崩溃以及关键行业参与者之间的连锁反应。**
