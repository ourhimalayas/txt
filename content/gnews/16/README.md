听当地的居民说，他们小时候这条河里有鱼、有乌龟、有甲鱼、有螃蟹，他们可以在这里饮水，在这里游泳，现在都不可以了。

这条小河的附近有很多工厂，工厂的污水处理是名存实亡、形同虚设。污水直接排入这条小河。当地的居民知道下游的人民会深受其害，还庆幸自己没有受害。
