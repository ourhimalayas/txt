## 朝鲜可能很快准备进行核试验

消息人士告诉韩国新闻社，朝鲜正准备让丰溪里核试验中心再次投入使用，”捷径 “隧道可能在一个月内投入使用。

Kcna Via Kns/AFP/Getty Images

据韩国媒体报道，朝鲜可能正在迅速准备进行核武器试验，这是四年多来的第一次。

韩联社援引政府消息人士的话说，朝鲜似乎正在丰溪里先前关闭的核试验场中挖掘一条通往3号隧道的 “捷径”。

不扩散研究中心（CNS）的分析师在3月初曾表示，卫星图像显示该试验场有重建的迹象，但让它恢复运行可能需要几个月或几年的时间。现在，金正恩的政权可能已经改变了策略。

“韩联社援引一位消息人士的话说：”[朝鲜]突然停止了恢复3号隧道入口的初步建设工作，它正在挖掘[隧道]的侧面。”这样一来，似乎有可能在一个月内恢复[设施]。”

丰溪里是朝鲜唯一已知的核试验场。从2006年到2017年，它在该地点的隧道里进行了六次核武器试验。朝鲜最后一次也是最大的一次核试验似乎引发了地质不稳定，造成了多次小地震，但分析家和美国情报官员表示，该地点很可能可以再次使用。

这一事态发展是在朝鲜打破联合国禁令并发射一枚洲际弹道导弹几天后发生的。日本和韩国说，这枚导弹飞行了1080公里（670英里），最大高度超过6200公里，在日本最北端的主岛北海道的大岛半岛以西90英里处落下。

分析人士说，这枚被称为 “华颂-17 “的导弹是有史以来从公路移动发射器发射的最大的液体燃料导弹。这是自2017年以来朝鲜首次测试洲际弹道导弹，代表着平壤在发展可能能够打击美国境内目标的武器方面迈出了重要一步。

朝中社当时说，金正恩下令进行这次试验是因为 “朝鲜半岛及其周边地区的军事紧张局势每天都在升级”，以及 “与美帝国主义的长期对抗不可避免地伴随着核战争的危险”。

朝鲜在1月份测试了创纪录的导弹数量，而且似乎还准备发射一颗间谍卫星。

总部设在加利福尼亚的CNS的专家在一份报告中说，3月初卫星拍摄的图像显示丰溪里的包括建设和维修的活动迹象非常早。

“报告说：”一种可能性是，朝鲜计划使试验场恢复到准备就绪的状态，以恢复核爆炸试验。”朝鲜需要多长时间才能在该地恢复爆炸试验，这取决于隧道本身的损坏程度，而这一点我们并不清楚。”

自从朝鲜在2018年自我规定暂停核武器试验以来，丰溪里一直被关闭。然而，随着无核化谈判的停滞，金正恩表示，他不再感到受到暂停试验的约束。
