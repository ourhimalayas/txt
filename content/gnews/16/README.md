## 中共国禁直播未授权游戏 腾讯禁连外网

![](https://assets.gnews.org/wp-content/uploads/2022/04/Unknown-13.jpg)中共国禁直播未授权游戏 腾讯禁连外网 网络图片
据美国之音4月24日报道，自5月31日起，中共国互联网游戏巨头腾讯将“腾讯网游加速器”升级为“腾讯游戏管家”，仅供中共国国产游戏改善网路速度，不再提供连结外网的服务。

腾讯网游加速器是一种虚拟私人网路(VPN)的技术，中共国玩家可借此“翻墙”，连接到海外伺服器，或提高网路速度畅玩腾讯旗下游戏的海外版本，升级游戏。6月份之后，中共国玩家需另寻其他VPN服务来玩海外游戏。腾讯最新调整引发中共国网友热议，部分网友批评这是“降级”而非“升级”服务。

腾讯2021年的总营收为5601.2亿人民币(868亿美元)，来自海内外游戏的年营收约占全公司的三成收入。其中海外游戏市场收入占全年游戏营收的26%。因此，取消加速器服务对腾讯影响有限。但是，对于仰赖中共国市场的海外游戏公司而言，中共国玩家连不上外国伺服器，会影响其营收。因为中共国也是2021年全球最大的游戏市场，以逾460亿美元的产值及游戏玩家6.85亿人高居全球第一。

除了不能透过VPN翻墙，中共国国家广播电视总局4月15日出台包含“严禁网络视听平台传播违规游戏”、“督促网络直播平台建立并实行未成年人保护机制”等六大要点。中共国游戏直播市场规模于2020年达到343亿人民币，估计2022年的游戏直播产值将达530亿人民币。

针对北京严禁直播推广海外游戏的新规，台北政治大学传播学院特聘教授林日璇表示，许多非主流游戏未获中共官方授权，仍广受中共国玩家欢迎。禁令对于直播平台是非常大的冲击。台湾学者曾俊霖也说，直播主或玩家的言论自由将进一步受到限制。

过去一年，中共国为防止青少年沉迷网路，重拳打击游戏产业。去年八月，中共官媒《经济参考报》重批网路游戏为“精神鸦片”，引发游戏类股价爆跌。中共国国家新闻出版署紧接着于8月底正式出台规定，限制所有网游企业只能在周末和国定假日晚上8点到9点，对未成年人提供一小时的网路服务，。

中共国持续收紧游戏产业的管控，再加上腾讯在游戏领域一家独大，中共国电子竞技及中小型游戏营运商不得不转往海外发展。早在2003年，中共国体育总局便将电竞列为体育项目，成为全球最早将电竞归为体育竞赛项目的国家之一。中共国电竞队于2021年击败强敌韩国，荣获竞技网路对战游戏《英雄联盟》的世界冠军。上海公司米哈游于2020年开发的跨平台手游游戏《原神》上市第一年的收益达20亿美元，创下全球手游纪录，而且大部分收入来自海外。
