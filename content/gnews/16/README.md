当地时间2月4日上午，河北省廊坊市固安县突因两例确诊病例封城。据网传消息，确诊病例疑为由京返乡人员。

目前，该县城全域内所有小区、公共场所、高速路口已全部封闭，除全员核酸外，所有街道空无一人，进京公交线路全部停运。有墙内网友戏称，仅仅一个上午的时间，整个县城就如遭空袭。

据了解，固安县直接接壤北京市大兴区，距离北京市中心仅有50公里，距离大兴国际机场仅有8公里，每日都有大量务工人员需往返于北京与固安之间。
