[正文]

年来，因中共积极游说国际电信联盟（ITU）、国际电工委员会（IEC）等机构，让中共的企业获得竞争优势。面对此冲击，美在北京冬奥开幕当天通过《2022年美国竞争法》，旨在促进美半导体制造业，抵消中共扭曲市场的贸易行为，包括加强反倾销规则等。

美国也和日本连手建立6G无人化技术的国际标准，防止中共企业垄断；并强化尖端技术保护，防范中俄经济间谍战。

欧盟主管竞争事务的执委维斯塔哲（Margrethe Vestager）表示，中共近期也重新定义网络的基本标准，也因而让中共对使用国有网络的使用者有更大的控制权，因此，欧盟委员会也提出对行业标准系统实施改革，藉以避免欧洲的数码产业等沦为被中共主导、垄断的惨境。

然而，北京大学在其报告中却罕见承认，美中科技脱钩后，中共无论在技术面或产业面等明显落后，尤其是芯片制造、人工智能及信息科技产业等，更会面临发展瓶颈，可说中共的损失比美国更大，然而却因滥中共立场反调而被火速删除，足见中共对言论思想的极端宰制，同时某种程度也印证了中共的野心。
