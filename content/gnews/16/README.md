## 北约首长寻求在1月举行北约-俄罗斯理事会会议

北约发言人周日表示，北约主席延斯-斯托尔滕贝格已寻求在下个月召开北约-俄罗斯理事会会议，并与莫斯科联系以确保其出席。

斯托尔滕贝格近几个月来多次提出通过这一机构恢复与莫斯科的对话，该机构成立于2002年，但由于乌克兰的冲突目前没有活动。

但俄罗斯当局并没有作出积极回应。

北约发言人说，”我们正在与俄罗斯就1月12日的会议进行接触”，他要求不透露姓名。

北约一直谴责俄罗斯2014年从乌克兰吞并克里米亚的行为，并呼吁莫斯科尊重其邻国的领土主权。

西方国家长期以来一直指责克里姆林宫向乌克兰东部的亲俄分离主义分子提供直接军事支持，这些分离主义分子在2014年莫斯科吞并克里米亚后不久夺取了两个地区。

俄罗斯否认这些说法，而普京则表示，这场已经造成超过13000人死亡的冲突是种族灭绝性的。

克里姆林宫越来越坚持认为，西方和北约正在危险地蚕食俄罗斯的边界。

本月早些时候，莫斯科向西方提出了全面的安全要求，称北约不得接纳新成员，并寻求禁止美国在前苏联各共和国建立新基地。

1月12日的会议是莫斯科提出要求后斯托尔滕贝格提议的第一次会议。

为期两天的北约30个成员国的军事首长会议定于同一天在布鲁塞尔开始。

周四，斯托尔滕贝格重申了对乌克兰的支持，反对俄罗斯边境对面的军事集结和克里姆林宫日益咄咄逼人的言辞。
