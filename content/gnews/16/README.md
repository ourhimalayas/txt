昨天，中国时间10月9日，我长期关注的一个做外贸和跨境电商的微博，博主“风中的厂长”发了一条微博，并配上了他微信截图。
![](https://assets.gnews.org/wp-content/uploads/2021/10/135.jpg)图片来源：微博
截图上写“好消息好消息，树脂粉涨到新高，大家准备好可以放假了”，有回应称“亏死”、“完了”、“这样下去还不如关门，树脂粉已经涨到1万4000多了，下个星期我估计涨到1万5000，这么搞，这生意没法做了，昨天价格报出去，跟今天差一双鞋，相差3块钱……”。另有人说“一天一个价，当天有效”。微博博主“风中的厂长”说，“今天办公室里鬼哭狼嚎，原材料大涨，很多订单没法做了，各个供应商群，也是一片哀嚎。”

这是在一线的制造业企业主的真实声音和目前制造业的真实情况。有的人会问，原材料涨价了，就不能给终端客户涨价吗？这个问题其实是老生常谈，可以涨，但是已经签订的合同怎么办？撕毁吗？签订的合同必须按原价格去做，但是按原价格来操作它就是亏损的。并且所谓的“终端客户”也就是普通老百姓，要是手里都没钱了，你涨价的话，客户可以选择不买，最后惨的还是中间的制造业。而且最主要的原因是没有人能够预测产能，突然一夜之间减少40%的产能，就是因为很多工厂都受到拉闸限电停产的影响，这是最主要的原因之一。

所有的工业、农业都跟煤电油有关，一旦煤电油们涨价，就会一路传导下来，未来所有东西都要涨，短期内也许会有一小段过渡时间，因为某些小企业撑不下去的时候只有大量低价出货以回笼资金，但是等到这部分小企业死的差不多了，剩下的寡头垄断定价权后，价格肯定会涨的更疯。现在很多化工原材料都是翻倍的涨，已经慢慢传导到日用消费品了。供应链上大家都是一根绳上的蚂蚱，要么涨价，要么毁约，要么停产，要么扛着。现在国内物价上涨的几个种类，价格上涨得非常厉害，基本上都是从国外进口的，因为中国的石油、煤炭、铁矿石、玉米基本上都是靠进口的。

另新华社北京报道，10月8日召开的国务院常务会议，进一步部署做好今冬明春电力和煤炭等供应，保障群众基本生活和经济平稳运行。会议明确，在保障居民、农业、公益性事业用电价格稳定的前提下，将市场交易电价上下浮动范围分别不超过十百分之十五，调整为原则上均不超过20%，对高耗能行业可由市场交易，形成价格不受上浮20%的限制。在电力供应紧张背景下，近期全国多地出现限电。会议指出，今年以来国际市场能源价格大幅上涨，国内电力、煤炭供需持续偏紧，多种因素导致近期一些地方出现拉闸限电，给正常经济运行和居民生活带来影响。看完这个新闻大家应该明白，接下去电价马上上涨，全面铺开，最低要涨20%。

而全球物价如此疯涨，最主要的原因就是美联储大放水，还有拜登上台以后搞的刺激政策，印发了海量的钱，现在债务上限又提高了。有的网友问，美国为什么要一直提高债务上限呢？要知道，不提高债务上限的办法就是省吃俭用、少花钱、多赚钱、多收税，像川普的政策那样让制造业回流，让人们有工作做，少拿福利，另外也扩大税基，再减少非法移民，少在非法移民身上花钱。但是美国的利益集团却把川普弄下台了，因为川普的政策让很多靠这些赚钱的议员或官僚没有了来钱的地方。

拜登上台后，他要继续给支持他的议员和官僚各种明里暗里的好处，也就是“补贴”，就像佩罗西发给美国普通民众的钱加一起不到补助总金额的十分之一，其余的都拨给了这个组织那个管所，其实背后都是议员家族的利益。所以即使佩罗西那么烂，还是有很多议员支持她，拜登也一样。另外，很多支持民主党的人都是不愿意出去工作的、拿福利的人，或者是在政府机关里工作的所谓的公务员的那些人。这些人都是吃财政饭的，如果把这些人的福利给取消了，或者减少，这些人立马就不会投民主党的票，不会投拜登的票。虽然拜登有“多猫腻”投票机，但是多多少少还是要有一些来投他的人，一个都没有的话，只靠“多猫腻”也是没有用的。所以为了这些选票和支持他的这些议员，他绝不可能节衣缩食，只能大放水。因为美元毕竟是世界通用货币，放出来的这么多的水就会涌到全世界，再加上全球央行跟着大放水，全球物价暴涨也就顺理成章了。

在中共国，由于习近平要对抗澳洲，不让进口澳洲的煤炭，虽然现在偷偷的卸了一些煤，但毕竟没有大规模的进口，中国非常缺煤少炭，导致了发电企业的亏损，所以发电企业不愿意发电。拉闸限电之后，习近平看到这一招可以拉高全球的通胀，让美国也承受这种通胀的后果，所以就更加拉闸限电了。本来不需要限电限产如此严重，现在很多珠三角、长三角的制造业工业区都是停五天干两天，停六天干一天，产能一下子减少百分之四、五十左右。产能减少了这么多，原材料价格能不上涨吗？所以，就发生了文章刚开头说的制造业主“风中的厂长”和他的供应商对于原材料涨价的一片哀嚎。

全球一体化，也就是美国大放水，搞刺激政策，导致各种原材料上涨，中国承接了通货膨胀的压力，然后又把这种压力传回给全世界。这中间，习近平又再加了把劲，人为拉闸限电限产，也就是火上浇油，让这股火儿烧得更旺一些，价格涨得就更多了，意思就是要死大家一起死。当然，最后拜登和习近平倒不会怎么样，毕竟人家有大把的钱，是Big boss，所有的恶果都会由全球的老百姓来承担。这也就是说，拜登和习近平搬起了一块巨大的石头，砸了全球老百姓的脚。

老百姓会发现，原来100块钱可以买很多东西，现在100块钱只能买一点点东西；原来圣诞节可以出去游玩，现在还是在家玩电子游戏或看电视剧吧，省钱啊！原来节日还能吃个大餐，澳洲龙虾之类的，现在改成小螃蟹或者白虾吧，龙虾就不要吃了。

还有一点，2020年疫情危机爆发以来，几乎所有国家的政府全部都债台高筑，被困在“政府债务”这个牢房里，为了把自己“解放”出来，包括美国在内的各国政府，唯一的办法就是印钞，让经济体通胀，然后通过同样的税收比例收上来更多的钱，而债务却还是原来的那个债务，自然就不再是个问题了。

总之，鹅毛是肯定要拔的，但高水平的操作方式就是既能把鹅毛拔下来，又不让鹅叫唤。各国通过印钞这种途径，使劲儿拔储蓄者的鹅毛，他们中的大多数又毫不知情，这就是拔鹅毛最高级的手段，更是1971年脱离金本位以来世界各国政府一直都在干的事情。

但通胀以后，大部分人收入绝对不会增加，特别是工薪阶层，相反原材料物价飞涨，企业和个人成本增加，苦的只会是老百姓。通胀还能转移财富，把财富从穷人手里转到富人手里，这也正是现在全球的奢侈品卖得非常火的原因。

《福布斯》在10月6日发布的榜单同时指出，新冠大流行虽然对美国大部分地区来说是一场经济灾难，但是富豪们却过得不错。去年（2020年）上了富豪榜的400人的总财富增加了40%。
