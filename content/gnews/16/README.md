日本每日新闻3月16日消息：15日，莫斯科法院对在没有许可证的情况下，手持呼吁反战抗议标牌出镜的，俄罗斯政府所属电视台的新闻节目女导演玛丽娜-奥夫沙尼科娃处以30,000卢布（33,000日元）的罚款，并将其释放，

14日，奥夫沙尼科娃在第一频道的直播新闻节目中举起标语牌，抗议俄罗斯入侵乌克兰，之后被当局拘留。据法新社等消息来源消息，当局的审讯持续了14个多小时。获释后，奥夫沙尼科娃告诉记者：”两天时间，几乎没有睡觉，我很累”。

据该通讯社报道，奥夫沙尼科娃被指控举着反战标语牌出现在新闻节目中，而且之前在自己拍摄的视频声明中呼吁俄罗斯公民参加反战示威活动。

奥夫沙尼科娃女士是两个孩子的母亲，父亲是乌克兰人，母亲是俄罗斯人。她今后可能面临更严重的刑事指控，包括传播有关俄罗斯军队的虚假信息的指控。15日，法国总统府表示，如果奥夫沙尼科娃女士认为自己处于危险之中，法国大使馆等机构已做好对她实施保护的准备。
