中国的互联网监督机构已经命令该国一些最大的网络平台删除与儿童有关的不当内容。

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/07/22020341/%E5%9B%BE%E7%89%8736-1.png)图片来源： Getty Images

瓜子网、腾讯的短信工具QQ、阿里巴巴的淘宝网和微博已经被中国网络空间管理局（CAC）传唤。中国互联网协会表示，这些平台必须 “纠正”和“清理”所有非法内容，并对它们进行了罚款。

这一宣布是在中国政府对科技公司进行打击的时候做出的。在一份声明中，CAC说 “这次行动的重点是解决七类危害未成年人身心健康的突出网络问题”。它们使儿童在直播中，成为社交媒体的影响者，宣扬“拜金主义”和奢靡之风。

不当内容包括以下：

色情和暴力内容

含有色情和暴力内容的不适当的动画片

鼓励自杀或从事儿童色情等行为的论坛

儿童参与筹款活动的粉丝俱乐部

网络欺凌等 “不良社会行为”

由于打击青少年网络成瘾的措施不力，这些平台被要求在最后期限内删除违反准则的内容，并被处以罚款，但声明中没有详细说明公司必须遵守的时间和罚款的数额。

今年以来，随着中国政府对科技行业的严格控制，中国的主要互联网公司受到了越来越严格的审查。本月早些时候，中国叫车巨头滴滴出行的股票暴跌，因为中国证监会下令网店不得提供滴滴出行的应用程序，称其非法收集用户的个人数据。

3月，中国国家市场监督管理总局（SAMR）表示，它已经对12家公司违反反垄断规则的10项交易进行了罚款。国家市场监管总局在一份声明中说，这些公司包括腾讯、百度、滴滴出行、软银和一家由字节跳动支持的公司。

据国家广播公司CCTV报道，习近平主席已下令监管机构加强对互联网公司的监督，打击垄断并促进公平竞争。
