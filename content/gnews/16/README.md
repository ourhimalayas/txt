1、[**中国人口出生率首次跌破10‰ 引热议**](https://www.aboluowang.com/2021/1120/1674150.html)

11月19日，第一财经日报报道称，《中国统计年鉴2021》显示，2020年全国人口出生率为8.52‰，首次跌破10‰；人口自然增长率（出生率-死亡率）仅为1.45‰，均创下了1978来的新低。此前，中共官方公布2020年出生人口为1200万人，相比2016年的1786万人，少了1/3左右。2016年全面放开二孩之后，人口出生率出现小高峰，但随后逐年下滑。

2、[**怕啥？北京天安门广场将需预约参观**](https://www.aboluowang.com/2021/1120/1674111.html)

北京天安门广场将不再是民众可以随意进出的“人民广场”。天安门管委会以疫情防控为由，宣布下个月15日起，所有中外游客，无论散客或团客，要进入天安门广场都必须事先透过手机或上网预约。在实施预约参观后，天安门广场仍将维持免费参观，入内无须购票，现阶段也不限定预约总量。

3、[**被拐20年后与母相认 深圳男自己取的名字竟是本名**](https://www.aboluowang.com/2021/1120/1674157.html)

11月19日，居住在深圳松岗的谢女士与失散了20年的儿子相认，母子二人抱在一起痛哭。20年前的一个傍晚，儿子意外走丢，疑似被拐卖了。从此，一家人开始了漫长的寻子过程。20年来，谢女士从没有离开过松岗，就是希望有一天儿子能找到回家的路。谢女士的儿子曾某说，他从小在潮汕揭阳长大，养父母对他很好。三年前，曾某来到深圳工作，嫌自己的名字拗口就改了花名。当他得知自己的本名时惊讶地说，“没想到我本来就叫这个名字”，“似乎冥冥之中一切都是安排好的”。

4、[**大陆161人犯罪团伙遭逮捕 涉案金额高达4亿多**](https://www.aboluowang.com/2021/1120/1673951.html)

今年8月，丹东市一个以“国氏兄弟”为首的犯罪集团被举报，该集团在丹东市东港地区，常年通过非法手段肆意攫取海洋渔业资源，同时还涉及走私、故意伤害致人死亡、非法拘禁、寻衅滋事、聚众斗殴、保险诈骗、敲诈勒索及非法经营等犯罪事实。涉案人数多达161人，涉案金额高达4亿多元人民币。犯罪嫌疑人国某兄弟等161人被采取强制措施，其中，21人被逮捕。中国黑恶势力背后往往有中共官员充当保护伞。

5、[**丹麦驻重庆总领事馆12月1日起正式撤馆**](https://www.epochtimes.com/gb/21/11/19/n13386978.htm)

丹麦驻华大使馆周五（11月19日）称，丹麦驻重庆总领事馆将于12月1日起正式撤馆。该总领事馆官方微信新闻稿称，“由于行政原因，从2021年11月18日起，该总领事馆所有业务移交丹麦驻华大使馆。”根据公开资料显示，丹麦驻重庆总领事馆位于重庆市渝中区邹容路68号大都会商厦31楼，其前身为丹麦驻重庆领事馆，于2005年1月设立，2012年11月正式升格为总领事馆，有商务、领事保护职能，领区涵盖重庆、四川、云南、贵州。
