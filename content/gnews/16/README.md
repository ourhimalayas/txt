深陷债务困境的中国恒大集团表示，将延迟刊发2021年经审核业绩；其中恒大物业更披露在审核去年财务报告过程中，发现有约134亿元人民币的存款，已被相关银行强制执行。

恒大在公告称，考虑到集团目前面临的经营上和财务上的挑战，尤其是债务压力，公司风险化解委员会正在积极研究解决方案并与债权人进行沟通，建议增加聘请金杜律师事务所作为法律顾问，协助开展债务风险化解工作，遭强制执行的134亿元存款是第三方提供的质押保证金。

据公司中期业绩报告，截至去年6月底其银行存款和现金总额约140.28亿元，其中约2,429万元为受限银行存款，主要为根据当地政府部门规定存作物业管理服务执行保证金及恒大保险经纪公司的受限制股本。

恒大为全球负债最高的房地产开发商，总负债超过3,000亿美元。公司的境外债合计227亿美元，从去年12月开始出现部分违约。同时由于未能支付供应商和施工队的欠款，在售物业陆续出现交房困难，资金链断裂。
