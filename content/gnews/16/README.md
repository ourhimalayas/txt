从抗日战争（1931－1945年），朝鲜战争（1950－1953年），三年大饥荒（1959－1961年），文化大革命（1966－1976年），六四事件（1989年4月15日－6月4日）等重大历史事件中所发生的事实做了全面伪造，利用独裁统治掌控媒体、电视台大肆宣传，写入历史书籍，编入学生教科书中，对中国大陆老百姓进行洗脑教育，在思想上实行愚民政策，导致普通老百姓以为在大陆看到中共宣传的历史就是中国的近代史。

CCP制造的三年大饥荒（1959－1961年），也有西方学者称为大跃进饥荒、中国大饥荒，被认为至今为止人类历史上最大的人类饥荒，三年中国时间在大饥荒中非正常死亡人数高达5000万人左右。几十年过后CCP体制内一些海内外高级学者也承认历史上确实发生了过这些惨绝人寰的悲剧。在CCP统治下人为制造三年大饥荒，却被CCP顽固的称为三年自然灾害或三年困难时期，妄想把这场由CCP制造的大灾难、反人类罪推给老天爷。

以毛贼为首的CCP统治集团，违反自然规律搞起工业化大跃进，造成千万百姓饿死，甚至出现“人相食”的人间惨剧。CCP至今为止未对毛贼在大跃进中犯下的错误进行清算，官方也没有公布大饥荒数据资料档案。CCP虽然后期总结是毛贼一意孤行极左路线造成的大饥荒，其根本的原因是CCP独裁统治制度造成的人间悲剧。CCP独裁治集团历来原则最高领袖凌驾党和政府之上，媒体又是一党专政控制的喉舌，任何不同声音都被视为异端予以消灭，这种独裁体制往往只会为领袖的错误助纣为虐。如果中华大地实现民主、言论自由、媒体独立，这样大规模饥荒造成千万百姓死亡人间惨剧是不可能发生的。

在大跃进期间，毛贼以个人意志为中心在农村搞人民公社，强行推广公共食堂，这种违背人性公共食堂制度剥夺了农民自己拥有食物权力，农民甚至连自己做饭的权利都没有。当饥荒开始蔓延的时候，人民公社各级干部以维护公共食堂制度为名不允许农民家里私下存粮，对农民家里搞突然袭击，悉数搜刮他们历年积存的余粮充公，不服从者甚至体罚。有历史资料记载，当公共食堂没有粮食的时候，人们开始吃树皮、草根。尽管发生如此艰难困境，上级还是强压下级交公粮，农民不仅口粮没有了，连种子粮、饲料也全部上交了。CCP干部利用掌管食堂仓库的特权，获得大吃大喝、贪污、腐败的最大便利，农民则沦落为“人民公社”的农奴，农民生存权力完全掌握在这些干部手中。面对这种强制性、压迫性的“人民公社”及“公共食堂”制度，农民唯一能做的便是极度的消极怠工，这又使粮食产量进下降。大饥荒发生最严重期间，出现了了失去人性的“人相食”的世间惨剧。

CCP建立的高干特供制度，CCP为保障其高级干部的食物、肉类及烟不匮乏而建立的“特需供应”制度，依官员级别满足所需。1960年8月CCP中央决定对国家级高干实行食品特供应，又在大饥荒严重时期，1961年2月起对高干以下的干部实行特供。在大饥荒期间，CCP执行全民计划配给制，食品分配级别是高干比机关干部好，机关干部比城市老百姓好，城市老百姓比农村农民好，而且高干之间、机关干部之间又分级别划分供给。这种不公的分配制度，大饥荒期间造成大量食物链底层的农民死亡，一位CCP高干子弟在接受《国家人文历史》杂志专访时回忆：“北戴河即便是困难时期，供应一直不错”。一位历史学家回忆三年困难时期：“在寒冬腊月，北风成天呼啸的时候，村里三天两头死人嚎丧。死的都是老人，其实是在饥寒交迫之下，得了感冒，便成不治之症”。
