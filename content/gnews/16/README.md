就连非洲那种古老到不会用手机的老部落都知道有选票这回事，村子里的长老那一定是大家伙一人一票选出来的，长老既然被大家推出来，当然事事都要考虑族人就是选民的意见，可不能啥啥都往自己兜里偷，一次两次大家也还能接受，俗话说有一次两次，没有三次四次，到了第三次的时候，这个长老也就不用做了，选民肯定要把这个人投票投下去。这就是人类社会的基本运行规则。是的，中共国自诩上天入海，又是嫦娥奔月，又是地下钻井，感觉科技发达到无所不能了，就这样的厉害的国家，居然没有选举这回事儿。共产党自打习近平上台以后就开始以家为天下，以家为单位治理天下。

就这么个模式，人类社会存在了几千年，从未发生过本质的改变。今天各个国家拥有的一切选举的形式不过都是随着一些科技的发展而产生了一些小小的变化，但人们依然深信不疑选举是神圣的，那个选票是用来表达自己的政治态度的，也是为了保护自我的人权不受伤害的利器。中共历史几千年，从来都在打打杀杀中过来，汉中地区土壤肥沃，民风淳朴，老百姓安居乐业，勤勤恳恳做着自己的老百姓，只要政通人和，几乎都会感谢皇恩浩荡，庇佑众生。到了战争年代，汉中也往往成为兵家必争之地，老百姓也最惨，一个人上辈子可能在太平盛世中过来，到了下半辈子遇上王朝末年战乱无休，那这个大家族很有可能毁于一旦。有聪明的劫掠者，会选择和这些大家族达成合作，也是为自己赢得江山拿下一些筹码。历史上很多次的农民起义，大多数都是因为农民实在吃不起饭了，只好起来造反，造反的下场就是把所有的仇恨都发泄给无辜的当地老百姓，进一座城屠一座城。农民起义的结果就是想把这个烂摊子打碎，重新建立一个不受剥削自己吃得起饭的制度，但是往往，还没成功呢，就先把和自己一样的老百姓给杀完了。

七哥说人类历史就是一个墓碑历史，总是要有无数人牺牲掉然后才能迎来更高层次的发展。但是在中国历史里，墓碑就是墓碑，并没有为之后的发展带来养分，没有化作骨灰，没有成为肥料，在历史里书写好了暴虐和残暴，然后没几天就被遗忘，后人偶尔看到的此人，也是经过了无数次篡改。人们常常说，不要和任何人比什么成功和失败，只要一个人活得足够长，他就能够拥有足够的阅历，能够跑赢时间，作为地球上的人来说，足够了。可是又为什么有个说法叫做老不死的，尤其中南坑的盗国贼，他们的生命为什么可以又臭又长，一百多岁活过了，还想再活五百年。因为有了时间，每个人在时间面前都是平等的，因为有了四季，有了太阳、空气和水，每个人也都是平等的，但是因为阶级，财富，每个人能够得到的生命体验度是不同的。在中共国，一个家人的一场大病足够把一个家庭拖垮。小粉红们到处嗤之以鼻的美利坚，人家的医疗是免费的，美国看病要排队，但是国内看病就不需要排队吗？人家只要预约好了，你去看病肯定能看上，你试试在北京协和医院的专家号，你排队排半年，到你了，愣是能把你给挤走，这就是现实。我们总是说民主，自由，这就是民主自由最基本的表现形式，有秩序，有规则，不会歧视你，预约的时间就是规则，永远不会因为任何，就算川普来了医生也是按照预约的顺序给你看病。有钱你就找私人医生，能够给到更好的医疗建议，中南坑的老杂毛个个都把国内顶级的医疗专家当自己的私人医生用，你老百姓有这个特权吗？甚至连那些个大专家的面儿都见不上，人的病等不起，就死了。
