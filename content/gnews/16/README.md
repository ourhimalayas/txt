今天主要想探讨下鬼神，灵魂以及生命的意义这个话题。首先人死为鬼，鬼死为魙，聻死为希，希死为夷，而希夷这个词曾多次出现在道教典籍中，《道德经》中就曾说：“视之不见名曰夷，听之不闻名曰希，无声曰夷，无色曰希。”意思就是说最终会化为虚无成为宇宙能量的一部分，我想这就是本源吧，既然人来自本源，如果不是神注入灵气，本源又怎会成为灵体轮回转生呢？
