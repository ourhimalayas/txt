一份新的报告称，美国官员正在考虑禁止向中国芯片制造商中芯国际 (SMIC) 出口产品，这让全球半导体混乱和与中国的地缘政治紧张局势再次恶化。

据路透社周五上午报道，中芯国际一直利用监管漏洞购买美国技术，但国防部已提出一项阻止出口的提议。

与此同时，一些商务部官员正试图阻止国防部的提议。

报告称，中芯国际去年被列入美国黑名单。 因此，它应该被拒绝“从美国供应商那里获得先进的制造设备”。 列入黑名单是由于中芯国际与中国军方之间存在潜在联系。

回想一下，上个月中国指责美国阻止英特尔在中国的扩张计划，从而“夸大了国家安全的概念”。

中国外交部发言人赵立坚表示，美国正在利用国家安全作为“筑起贸易壁垒”的手段。

李健还指责拜登政府强迫三星和台积电交出与其供应链相关的“敏感信息”。 因为众所周知，中共总是置身于大型中国科技集团的计划之外。

李健随后指责拜登的行为损害了美国和其他国家的供应链。 他建议美国“摒弃零和博弈思维，维护公平非歧视的营商环境，以实际行动为建设开放型世界经济贡献力量”。
