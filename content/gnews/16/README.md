作者：一颗星星

【接上篇：世纪骗局之：1981~1985（改革开放）(四)】

**笔者观点：**

提起“改革开放”，似乎国内很多人总是会觉得是一次具有划时代意义的壮举，有些人会觉得是“改革开放”让国人富起来了。殊不知，“改革开放”也是中共政权的一次迫不得已的救党之路。在“毛腊肉”的时代，中共不但把中国人折腾得死去活来，中共的政权也是命悬一线。那一时期，在中共的折腾下，中国实在是太穷了。朝鲜战争让老毛丢了儿子，“文革”后“毛夫人”（江青）又进了大狱，“毛腊肉”家天下的美梦彻底破灭了。在“文革”中苟活下来的那些中共老杂毛们清楚的知道，这样穷下去是不行的。即便他们不会在乎饿死多少中国百姓，但是继续走老路是要亡党的。

想要赚钱就要改革。虽然“改革”了，依旧没有改掉中共独裁的本性。

经济自由化一旦开始启动，政治民主化的要求就会提上日程。犹如日后，中共加入WTO而急需要西方自由世界承认的“市场经济”，便是与中共彻头彻尾的政治独裁相矛盾的。没有政治的民主化，经济的自由也会难以维系。经济越发展，经济的改革越深化，那么其与中共政治独裁的矛盾也会越尖锐。这对于贪婪的中共独裁政权来说，是一个解不开的难题。它们一边想要赚得盆满钵满，一边还要想让它们世世代代的子孙们独裁统治着中华大地，贪婪成性的它们，只好用千万个谎言和亿万个无辜的生命去维系着它们那邪恶的独裁政权。

反观这一时期的中共领导人——被称为“总设计师”的邓小平，虽然改革的“号角”吹的震天响，但是他根本是不会去搞彻底的政治改革的。他所讲过的政治改革，都是在“四项基本原则”的框架之内的。而彻底的政治改革，是必须要解决掉中共一党专政这根本性的体制问题。中国共产党是中国最大的利益集团，坚持中国共产党的领导，是保证其利益最根本的一条。亲爱的读者们，再去复习一下“四项基本原则”的内容，我们不难看出其中的任何一条都是为中共要保持一党专政而设计的。有了“四项基本原则”这个“紧箍咒”，中共再如何的宣扬、叫嚷“市场经济”、“政治改革”、“法治社会”、“民主选举”……这一切的一切，都是骗、都是扯淡。

高举“改革开放”大旗的中共，在内部依旧斗得热火朝天。邓、陈两大阵线的人，在中央党校（意识形态阵地）和人民日报（宣传阵地）之中斗得死去活来。时至今日，控制意识形态和掌握宣传的话语权，依旧是中共欺骗中国百姓，欺骗全世界的不二法宝。篇幅所限，就不一一赘述了……

而“改革”之初发起的“严打”运动，更是让中共政权狠狠的抽着自己的嘴巴。口口声声号称“依法治国”的中共政权，在后毛时代依旧是中共太祖“毛腊肉”那副“和尚打伞”——无法无天的德性。所谓的法律，在中共体制之下只不过是用来“摆摆样子”的一纸空文。
