产经新闻1月20日消息：岸田文雄首相在20日国会上院的一次全体会议上，针对立宪民主党的水冈俊一先生提出的问题答复说：日本政府将要求地方政府考虑为那些因离婚或其它原因而没有得到18岁以下未成年人福利的单亲父母提供该福利。

岸田文雄首相解释说：我将要求地方政府考虑使用应对中共病毒（新冠病毒Covid-19）感染的地方发展临时补助金，以便考虑为那些在规定日期之后离婚的，未成年人抚养者们提供福利。

像这样想方设法为国民提供福利支援的事，笔者在日本多年，亲历过很多。这在中共国是不可能有的事儿！中共国国民被派去其它国家工作，通常来讲是福利、是高兴的事儿。而在日本的企业，被派到中共国去工作，被认为是去受罪。所以，企业对愿意去中共国工作的员工会给予高福利待遇。
