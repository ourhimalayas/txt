## 澳大利亚本周末再次举行反强制疫苗自由集会

当地时间11月27日周六，澳大利亚民众再次于多地首府和主要城市举行反对疫苗强制令的游行集会，包括墨尔本、悉尼以及黄金海岸。

虽然政府自称不会进行疫苗强制，但事实上在许多地方都需要进行疫苗接种情况的检查才可以进出，例如商店、工作单位以及一些活动。例如在维州，目前只有完全接种两次疫苗的工人才有资格返回岗位， 拒绝疫苗则意味着失去工作和经济来源。

相比上周的自由集会活动，本周的集会在规模上依旧十分庞大，可见人们对于疫苗强制政策（不论硬性还是软性）以及疫苗护照的担忧和恐惧。

根据社交媒体上人们的说法，当地家长尤其对于儿童强制疫苗以及儿童疫苗护照的话题十分敏感，这事关下一代人的健康成长和未来。

同时政客们以及各路主流媒体将异议人士统统贴上反疫苗标签的CCP行为也令部分民众感到不满和愤怒，想必这也是本周再次集会的另一成因。

在墨尔本地区，就有民众打出“敬畏上帝而不是州长”的口号讽刺维州州长的独裁行径。而在新南威尔士州，人们则打出“少一点政府，多一点自由”的标语，反应他们已经受够了政府的压迫和管制。
