![](https://assets.gnews.org/wp-content/uploads/2022/02/七国集团领导人电话会议将于24日举行.png)图片来自本文网站

2月21日日本电视新闻在线消息，内阁官房长官松野在新闻发布会上宣布，岸田首相将出席24日在德国举行的七国集团领导人电话会议。会议将讨论包括乌克兰局势在内的外交政策。

松野强调，对日本来说，团结七国集团，引领国际社会极为重要，对此日本将积极参加讨论。

随着乌克兰局势日趋紧张，截至19日，还有约120名日本人仍在该地区。对此，松野表示，除了继续呼吁疏散外，政府正在做各种准备，包括已经在邻国安排了包机，以便能够对任何情况做出适当的反应。
