乌克兰政府透露，俄罗斯军队的大规模军事入侵只在24日就造成57人死亡，169人受伤。

昨天开始的俄罗斯军队的军事入侵。袭击在乌克兰各地进行，俄罗斯国防部宣布在日本时间黎明前摧毁了乌克兰的83处军事设施。据乌克兰卫生部长称，仅24日，至少有 57 人丧生，169 人受伤。

与此同时，乌克兰总统府透露，俄罗斯军队已经控制了位于白俄罗斯边境附近的切尔诺贝利核电站。路透社报道称，乌克兰官员表示，来自白俄罗斯的弹道导弹袭击，和俄罗斯军队攻入首都基辅的行政区，随时会发生。这可能会让战斗更加激烈。
