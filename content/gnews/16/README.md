（早报讯）路透社报道，美国官员称，袭击伊拉克北部库尔德自治区首府埃尔比勒的火箭弹是从伊朗发射的。

埃尔比勒星期天（3月13日）遭12枚[火箭弹](http://www.zaobao.com/realtime/world/story20220313-1251804)袭击，袭击事件没有造成人员伤亡。

目前还没有组织认责，美国国务院发言人指责这是一次“令人发指的袭击”。
