**中共病毒肆虐香港多时，港府闭关锁港肃清政敌已近尾声，香港社会如今已是红旗插遍港九，政商界同撑中共暴政。**

闭关锁港的香港经济不再繁荣，主题公园不再有游客收入成为亏损无底洞，零售业大幅收缩至本地供应规模勉强度日，运输业因燃油供应被财团垄断而成本激增，各国降低清关能力导致供港物资减少……近期香港的媒体不断吹风粮食供应短缺，大米、肉类和水产的来货量直线下降，价格显著上升。此外能源供应的垄断导致电费和燃气价格飙升，收入减少的港人要承担越来越贵的生活开支。

此时的港府不但不解决民生疾苦，反而全力配合中共锁紧大门。前几日新上任的海关关长何佩珊以国安为由，扬言进一步加强海关对人和物的检控，龙虾也成为有损中共国威的禁品。今日港府以确保与内地通关为由取消大部分入境豁免强制检疫群组，包括海外金融机构人士及艺人，保留豁免的主要是中港司机和少数特别类别的人员。港府此举引发海外金融巨头的不满，称有损香港金融中心地位。但港府高官和政要均称要全力配合中共要求，坚持闭关锁港至明年11月。

爆料革命发起人郭文贵先生揭露中共内斗已白日化，各地人祸灾害频发，政权岌岌可危。中共内部政治角力不断在国内引起轩然大波，房产泡沫接连爆破，银行次贷火烧联营，地方政府财政濒临破产，引发各地人祸频发，中共试图以更多的灾难抹去债务，用“多难兴邦”的谎言转移民众关注的目标，“共患难”将人民对政权的不满和追责扼杀在摇篮中，中共的诡计屡试不爽，愚弄了善良的中国同胞七十年。

BNO护照和各国救生艇计划带走了许多勇于与中共割席的香港人，他们中的许多人放弃了原本安逸舒适的工作和生活，选择内心的指引远赴他乡重新开启新的人生。但大部分的港人选择留在越来越黑暗的香港等候，他们都有各自的理由去安慰自己的心灵。但事实上局势越发动荡，他们手中能抓住的就越少，当他们手中的一切都被夺去而无力反抗的时候，他们生命就如草芥般只是一个数字。

爆料革命郭文贵先生无数次提醒同胞们不要留恋大城市虚幻的繁华，尽快逃离或退居到小城市或农村，以免灾难来临的时候与中共一同被卷入毁灭之中。
