根据韩国疾病控制和预防机构的报告，该国正在努力阻止该病毒的传播，但在周三(2021年12月1日)，每日感染人数首次超过5000人。达到了一个新的记录， 韩国还报告了新出现的Omikron变体的前五例病例。

根据丹麦血清研究所的数据，丹麦周三报告了每日新冠状病毒感染数量也创了新纪录，并且已经确认了一些新的Omikron变体病例的北欧国家在周三报告了5120个新的COVID-19病例。

第一张图片所示丹麦和韩国接种疫苗的情况，两个国家已经接近80%。 然而第二章图片显示，随着接种疫苗的达到峰值，感染中共病毒也达到了历史以来的高峰。另一方面也说明了疫苗并没有起到预防冠状病毒的作用。

12月5日，在柏林举行了一个关于疫苗副作用病理会议。下面第三张图片为会议中展示的超正常死亡率资料，统计年龄组为19岁到59岁。从图表可以明显的看出接种疫苗的超正常死亡率在统计学意义上明显高于非接种疫苗种类， 并且这个年龄组都是青壮年期间，可想而知，超过60岁的情况会有多么糟糕。疫苗数据统计到9月末，从英国的官方国家统计中心。
