## 为防土地不当使用，日本内阁专设部门全面摸底重要区域领土

![](https://assets.gnews.org/wp-content/uploads/2021/12/10-9.png)
[*图片来源*](https://cloudfront-ap-northeast-1.images.arcpublishing.com/sankei/G3DA44OBHFL3LFJZ7MFE75QJZA.jpg)

据产经新闻23日消息，根据6月颁布的《土地利用规制法》，作为防止征用对国家安全有重要意义的土地的措施，政府在内阁办公室设立了一个约70人（包括防卫人员）组成的特别部门，负责目标区域的指定和调查。为了集中管理信息，内阁办公室将增加总干事级政策主管的职位，确保明年9月全面实施的准备工作将正式开始。

内阁办公室将设立一个专门负责土地利用调查的部门，工作人员约为30人，包括从各部委借调的人员。由于法律涵盖了防卫设施周围约一公里的区域，大约40人将被分配到日本各地的国防部防卫局。且该法涉及到偏远的岛屿，所以也会有数人被分配到海洋政策总部。

此外，目前内阁办公室在不同政策领域的七个总干事职位将增加一个，以负责土地测量工作，理事和参事也将各增加一位。

政府的目标是在明年6月之前成立的新部门能正式运行，届时《土地使用管理法》将部分生效，并在9月全面生效之前能确定需要调查的区域。

法律规定，国防设施和关键基础设施周围约一公里的“观察区”，以及偏远边境岛屿的土地，都是需要调查的区域。法律允许政府建议或命令停止不适当的土地使用。特别重要的设施和偏远的边境岛屿将被指定为“特别观察区”，参与土地交易的各方将被要求提前通知政府。
