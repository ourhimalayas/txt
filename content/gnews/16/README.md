#### 编译 ：小红帽

著名疫苗专家、病毒学家、比尔和梅琳达·盖茨基金会前高级官员海尔特·范登·博舍(Geert Vanden Bossche)博士敦促**那些尚未接种新冠病毒（COVID-19）疫苗的人，如果他们想要保护自己的先天免疫系统，就不要接种“新冠疫苗”。**

海尔特以视频的形式播放了这些信息，该视频于11月20日发布。“永远不要允许任何东西或任何人干扰或抑制你的先天免疫系统。一个人的自然免疫系统是抵御包括新冠病毒在内的任何疾病的第一道防线。”

“**新冠病毒疫苗产生的所谓‘抗体’会抑制一个人的自然免疫力，并且该‘抗体’不能替代自然免疫力。不要相信–只要有足够多的人接种新冠疫苗，整个社会都将受到保护。新冠疫苗“无助于群体免疫力”**。

建议那些更容易感染新冠病毒的人，例如老年人，免疫系统受损或患有合并症的人，与其他人密切接触时要小心。

“这不是歧视疫苗，但未接种新冠疫苗的人应该特别避免在这个时候与接种过新冠疫苗的人接触，”海尔特说，“不要向儿童提供实验性和致命的新冠疫苗。”这一行为是“绝对不允许的”。

研究表明，儿童的免疫系统很容易被新冠疫苗提供的所谓“抗体”抑制。由于新冠疫苗抑制儿童的免疫系统，可能使他们暴露于更多的疾病，甚至可能导致自身免疫性疾病。
