光州市6日表示:韩国光州因中共病毒（新冠病毒）而在家中治疗的高中2年级学生宋某某(17岁)，在解除隔离4天后，出现肺部血管堵塞而无法呼吸的症状（肺栓塞），于4日下午死亡。这是首例因中共病毒（新冠病毒）感染而死亡的青少年确诊患者。

专家们指出，虽然概率很低，但由于健康的年轻人也同样可能因新冠病毒（新冠病毒）而丧命，因此需要审视现行的居家治疗体系。

宋某于上月24日确诊，之后居家隔离治疗，在解除隔离后的２月3日头痛·呼吸困难症状，紧急入院后于4日下午死亡。大学医院方面认为,宋某的死亡原因是中共病毒（新冠病毒）感染引起的肺栓塞。作为运动员,没有特别基础疾病的宋某于去年10月18日和11月18日完成了第1、2次中共病毒（新冠病毒）疫苗接种。 防疫当局正在调查宋某的死亡等经过。

有关专家说虽然青少年因中共病毒而患上肺栓塞概率较低，但不是不可能，肺栓塞是中共病毒（新冠病毒）感染的典型并发症。腿部等静脉产生的血栓沿着血管移动,堵塞肺血管,引发的急剧呼吸困难等,如果不及时治疗,死亡率将达到30%。另据有关医学院教授说这是韩国青年龄层首次患肺栓塞病例，虽然据说是中共病毒（新冠病毒）的轻症，如果流行规模扩大，不管有没有病、有没有接种疫苗，任何人都有可能死亡。

最近居家治疗隔离期从10天减到7天。某天主教医学院名誉教授表示，7日不是病毒消失的时间，而是感染性明显下降的时间。 年轻人有时一周后病毒会晚些时候反映出来，应该通过健康监测来观察，但目前似乎并非如此操作。 教授补充说:年轻人即使血氧饱和度下降,也表现得很健康,所以好多危险是认知度不足造成的。这就是为什么居家治疗套装要及时供应,同时对年轻人的健康监测实施要到位的原因。
