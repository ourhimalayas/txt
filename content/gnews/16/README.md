据英国媒体《每日内幕》12月4日报道，【英国卫生安全局】最新公布的报告显示，尽管大部分易感染群体已注射了“新冠病毒”疫苗加强针，但目前该国绝大多数病毒感染及其死亡者都已接种了该疫苗。

报道称，60%病毒感染和住院病例以及80%因此死亡病例为已接种者。该数据表明，疫苗既不能预防病毒感染，也无法防止其传播，更未能阻止死亡。
