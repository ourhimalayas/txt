11月26日，南非新冠状病毒新变种的发现引发了全球恐慌，各国争先恐后暂停来自南部非洲的旅行，金融市场遭受重创，大西洋两岸的股市遭遇了一年多来的最大跌幅。

![](https://assets.gnews.org/wp-content/uploads/2021/11/670b-52668f211fc794cc82150057e2fb3503.jpg)图源网络

当天，道琼斯工业平均指数收盘下跌2.5%，这是自2020年10月底以来最糟糕的一天，欧洲股市也经历了17个月来最糟糕的一天。邮轮运营商嘉年华公司、皇家加勒比邮轮公司和挪威邮轮公司各暴跌10%以上，而联合航空公司、达美航空和美国航空公司的股价也遭遇了几乎相同幅度的暴跌。

在恐慌情绪下，石油每桶暴跌了约10美元。

有分析人士称，市场恐惧的并非病毒本身，而是各国可能的应对病毒的政策。如果各国再次实施严格的封锁令，对经济的复苏无疑是当头一棒。
