这就是VAERS疫苗不良反应报告系统。我们知道在该系统中，这些（数据）是由CDC认证的。因此，报告数据每周更新一次，由CDC认证。这意味着所有这些（不良反应）事件是真实发生的，因为这些报告开始只被当作临时数据，然后被认证核实。

Meissner博士及其同事在COVID之前的一篇论文中告诉我们，这些（不良反应）报告中约有85%-86% 是由医生、护士或其他医护人员以及制药商完成的。只有约14%-15%是由病人自己提交的。

他们是如何得出那个漏报比例的？哈佛大学的一项研究表明漏报率很高，报告率仅为1%，也就是说他们只报告了1%，对吗？哈佛大学的研究是针对HPV也就是人乳头瘤病毒疫苗的。

这就是全部吗？是的。问题是，大人和孩子们都打了疫苗，所以（漏报）可能是严重的，他们可能严重的报告不足。COVID的人都很紧张，对吗？CMS（医疗保险和医疗补助服务中心）的数据基本上体现了接种疫苗的人数，我们知道CMS服务对象所占美国总人口的比例，所以通过推断可以计算出真正的数字是什么。

因此，相较于VAERS显示的数据，（疫苗不良反应的）真实数字大约是45000，这就是我们如何能得到实际数据是（VAERS数据）4-5倍的原理。我们认为这是一个合理的数字。即（真实疫苗不良反应数据）是（VAERS数据）的4-5倍。
