据美国之音11月25日报道，正在日本和韩国进行访问的美国国会众议院的五名议员预计26日抵达台湾进行访问，计划停留二至三天，会晤台湾国防部官员和台湾总统蔡英文。这是美国国会议员本月第二次访问台湾。11月9日，美国参众两院六名议员抵达台湾访问并会晤了蔡英文总统和国防部官员。

美国和世界上大多数国家一样，与台湾没有正式外交关系，但美国是台湾最重要的支持者和武器提供者。台湾与西方的民主世界有着同样的理念，与中共国的独裁暴政格格不入。中共一直对台湾虎视眈眈，视台湾为中共国领土的一部分，叫嚣武统台湾。自从民进党执政以来，两岸关系日趋紧张，解放军加大了对台湾的军事压力，导致美国对台湾军售增加，与台湾的关系也日益密切。但是中共不灭亡，对台湾和整个世界的威胁就不会解除。
