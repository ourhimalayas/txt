#### 撰稿：日本东京方舟农场 青衣

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/07/06044221/2C8EC904-9559-456E-8929-D8B33AEFC120.jpeg)

据《希望之声》2021年7月5日报道，日本副首相兼财务大臣麻生太郎今天（7月5日）在东京的一场演讲中表示，如果中共入侵台湾，日本将可能根据安全保障相关法规，行使有限的集体自卫权，“日本和美国将不得不合作防卫台湾”。

据日本《时事通信社》报道，麻生在演讲中表示，根据安全相关法律，中共对台湾的入侵可以被认定为 “生存危机事态”，日本可以行使有限的集体自卫权。

“生存危机事态”是指与日本关系密切的他国遭到武力攻击，进而威胁到日本国家存亡的明显危险事态。这也是日本自卫队行使集体自卫权的条件之一。

麻生还表示：“台湾如果发生重大问题，完全有可能将关系到日本的生存危机。那样的话，日美将不得不一同防卫台湾。”

日本防卫副大臣中山泰秀也在推特上转贴了上述相关新闻。

7月1日，中共国国家主席、中国共产党总书记习近平在中共百年党庆活动上发表讲话，表明了统一台湾的决心。虽然习近平声称“和平统一”，但也警告称“大陆不会放弃使用武力”。

此外，中共将在明年秋天召开第二十届全国代表大会。有观察人士表示，习近平已为当选前所未有的第三个任期展开权力布局。北京的一位学者对此进一步推测称，“习近平将在他的第三个任期后入侵台湾。”

美国前印度-太平洋司令部司令菲尔·戴维森在今年3月美国国会参议院的一个听证会上也警告称，入侵台湾的威胁正在加剧，中共可能准备在未来6年内占领台湾。

美军参谋长联席会议主席马克·米利将军（Mark Milley）6月23日在国会众议院军事委员会作证时也表示，在对华战争中支持军事行动会给美军带来严重问题。但目前美国军队比北京（中共）军队更强大。他还重申，尽管中共目前缺乏对台湾进行军事接管的能力，但这种袭击可能导致美国军队介入保卫台湾。

对此，中共的国际事务专家也指出，中共军队自1979年中越战争以来，没有经历过正式的对外战争，现在仍无法战胜经验丰富的美国军队。

不过，中共虽然在正面战场上无法战胜美军，但却可以使用所谓“超限战”的手段，包括生物武器、化学武器，甚至核武威胁等，与美国和世界对抗。而如今肆虐全球的中共病毒就是最好的例证。

中共现在正处在末路狂奔中，不惜一切代价殊死一搏很可能会发生。而攻打台湾又可以转移调查病毒真相带给中共的巨大压力，再加上中共还绑架了10几亿国民做“肉票”，因此，中共武力攻台的时间表，或许比美国军事专家所预测的要早。

新中国联邦和爆料革命的奠基者、领导者郭文贵先生6月20日已经根据他所获得的情报，就此对全世界发出了警告，世界必须尽早做好准备。
