**西班牙2021年7月13日电/西喜社**——

美国商务部周五已宣布将一些公司列入实体制裁名单，其中包括一组美国公司不能出口或转让货物的公司和组织。

中共国商务部今天在官方网站上反对美国政府对23家中国公司的制裁，因为这些公司被指控有虐待新疆穆斯林少数民族的行为，该部称这些制裁 “严重违反了国际贸易规则”。据中共当局称，美国滥用进出口管制，不注重事实，“以‘人权’为借口”。

美国国务院称，被制裁的公司中有14家是“北京镇压、大规模拘留和监视新疆维吾尔人、哈萨克人和其他穆斯林少数民族运动的同谋”。五家因与中国军方有联系而被列入实体制裁名单，四家因与已被美国政府制裁的公司做生意而被列入实体制裁名单。

中国商务部称美国的制裁是“对中国公司的不合理镇压”，并敦促美国 “纠正其违规行为”。此外，声明还保证，中共将采取“必要措施保护其合法权益”。
