**游戏行业审查和监管**

从“实名制”，到“防沉迷”、“人脸识别”，中共对于游戏行业的审查与监管逐年提升，各大游戏厂商在压力下，不得不建立和强化各自的防沉迷、信息监控体系。中共的诸多政策多管旗下，其目的就是要牢牢控制住年轻人的思想，孩子们的大脑，决不允许有自由思考、独立思想的出现，必须做到“爹亲，娘亲，不如党亲”。

作为审查方的中（共）国机构国家新闻出版总署虽然有一些所谓的明文的规定。——例如，不得侵犯版权，不得泄露国家机密——但其大部分的指向性都不那么精确。禁止“危害社会公德或者民族文化传统”的作品；以及“宣扬邪教和封建迷信”的媒体。但在决定允许和不允许的内容时，这种模糊性赋予审查人员几乎无限的权力和灵活性。这些类似于“口袋罪”、“文字狱”的操作手法使得国外开发商深受其害，他们不知不觉中就触到了所谓的“红线”，导致游戏无法发行，项目无法推进。

**打击宗教、人权、自由**

自习近平 2013 年上台以来，中（共）国政府在墙内的镇压愈演愈烈，国际社会对其处理香港民主抗议活动和新疆种族灭绝、反人类罪的行为的批评越来越愤慨。中（共）国类似的言论审查在娱乐圈、游戏圈都是屡见不鲜。全世界大部分的企业都跪倒在中共的淫威之下。

例如，在 2018 年，派拉蒙影业与腾讯影业合作的《壮志凌云》的续集。在预告片中，汤姆克鲁斯的标志性飞行员夹克与 1986 年原版电影中的那件有一个关键的不同——缝制的台湾国旗被移除。而在2019年，《魔兽世界》、《炉石传说》等海量热门游戏的创造者美国开发商暴雪驱逐了一名顶级职业游戏玩家并收回了他的奖金。仅仅由于他表示支持香港的民主运动，这名来自香港的名叫Blitzchung的选手Chung Ng Wai接受了现场采访，他用普通话说：“光复香港，时代革命”。

**对个人言论进行攻击**

大纪元的一则报道[2] 韩电竞手说“赚中国钱就要当狗” 粉红被刺激，2021年4月12日，韩国电竞选手朴钟烈（游戏ID：saebyeolbe）结束在中（共）国直播平台（斗鱼）的游戏实况后，回到Twitch频道直播时说出了对中（共）国市场的看法。根据网友翻译，朴钟烈说：“在中（共）国，台湾不能叫台湾、香港不能叫香港，搞得我都不知道怎么说话。所以也就不说了。不知道为什么不能这么说，就是不能说。”，“如果你想赚中（共）国人的钱，就要当中（共）国的狗。所以我现在就当中（共）国人的狗，连中文都认真学了。我连‘谢谢你的礼物’的中文都会说，我中文说得可好了。””

此举极大的刺激了墙内“自干五”、“小粉红”的神经，他们开始攻击朴钟烈。4月13日迫于压力朴钟烈在微博留言表示歉意，但大批网民仍“不买账”，甚至开始攻击朴钟烈所属战队“首尔王朝”，以及所属GenG电子竞技俱乐部。中共宣传部也跳出来，声称这是公然的“辱华”，挑战“一个中国”政策，要求韩国战队对朴钟烈做相应处罚和处理。随后，国内电竞战队也纷纷响应中共号召，表态提出抗议，包括抵制游戏赛事，抵制韩国战队。4月14日朴钟烈被迫再发致歉信。但在Twitter 和 Reddit 上，粉丝发布了“I stand with Saebyeolbe”，以表示对这位韩国球员的支持。
