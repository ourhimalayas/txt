**1****.****爱沙尼亚和日本外长会晤，同意共同维护和加强国际秩序**

[据NHK报道，在波罗的海三国之一的爱沙尼亚，茂木外长会见了里梅茨外长，在分享了对中共国霸权主义行为的严重关切后，他们同意共同努力，维护和加强自由和开放的国际秩序。 在参加完G20（20国集团）外长会议后，茂木外长访问了波罗的海三国，7月1日晚，他在他访问的第一个国家爱沙尼亚与里梅茨外长进行了45分钟的会谈。会议期间，茂木外长欢迎欧盟计划在日本提出的 “自由开放的印度洋-太平洋 “概念的基础上制定新的印太战略。](https://www3.nhk.or.jp/news/html/20210701/k10013114471000.html?utm_int=news-politics_contents_list-items_018)

**2.****从10月开始三菱UFJ银行降低向其他银行转账的费用**

[据读卖新闻报道，7月2日，三菱UFJ银行宣布，从10月1日起降低对其他银行的转账费用，转账金额在3万日元以上的，降低110日元，转账金额在3万日元以下的，降低66日元。 这反映了银行间结算费用的减少，使其成为第一个宣布新费用的巨型银行。 从10月起，30,000日元或以上的存款费用将从柜台880日元降至770日元，通过互联网从330日元降至220日元。 三菱UFJ银行内部的转账费用不会有变化。](https://news.yahoo.co.jp/articles/21ab33dff1a53be8eb595aba28c9ff98f8836564)

**3.****首相菅义伟表示，他将 “尽快 “就疫苗供应作出决定**

[据产经新闻报道，7月2日，大阪府知事吉村洋文在首相官邸与首相菅义伟会面。 鉴于目前希望获得CCP病毒疫苗的地方政府的供应无法跟上需求的情况，吉村要求将可能出现感染扩散的城市地区指定为 “优先接种区”，并战略性地供应疫苗。 出席会议的大阪市长松井一郎说，首相打算尽早决定从7月中旬开始供应的疫苗数量，并通知地方政府。虽然全国各地正在加速接种疫苗，但由于缺乏全国性的供应计划，一些城市正在暂停接种疫苗。 在会议上，首相说：”我们将倾听地方政府的真实声音，并尽我们所能做出回应。”](https://news.yahoo.co.jp/articles/879ac3dc2918be55ffeb4c15a6d2ccc4e793a1d2)

**4.****NTT ****Plala****或泄露800万条个人信息**

[据NEWS24报道，互联网供应商NTT Plala宣布，多达800万客户的个人信息，包括姓名和地址，可能由于未经授权的访问而被泄露。据NTT Plala称，2009年8月至2021年6月期间收到租赁设备（如路由器）的800万客户的个人信息可能被泄露。这些个人信息包括地址、姓名和电话号码，但不包括信用卡或金融机构账户信息或其他与支付有关的信息。NTT Plala表示，”如果今后个人信息明显被泄露，NTT Plala将以书面形式单独联系受影响的客户。”](https://news.yahoo.co.jp/articles/58d90088a038ae3f0aae35bccef0becd31a79534)

**5****.****日本邮局银行从明年1月起在夜间和节假日对部分ATM机收费**

[据JIJI报道，7月2日，日本邮政银行宣布，它将从2022年1月17日起对位于电车站和FamilyMart便利店的自动取款机（ATM）收取费用。 夜间，周末和节假日，每次存款或取款的费用为110日元。 在全国32,000台自动提款机中，约有7,600台设置于邮局和日本邮政银行之外的地方。 除此以外，在ATM机上处理硬币将收取110-330日元的新费用。 在柜台和自动取款机上支付水电费和其他费用时，仅现金支付的费用将增加110日元。同时，该公司还将提高使用附属罗森银行自动提款机和E-Net自动提款机的费用。](https://news.yahoo.co.jp/articles/d6b9dee0492f070a2617a69ae8af422131edd85e)

**6****.****公共养老基金GPIF的投资创下37.8万亿日元的盈余记录**

[据NHK报道，7月2日，管理公共养老基金的政府养老金投资基金（GPIF）宣布，其上一财年即2020年的投资结果是有史以来最大的，盈余约为37.8万亿日元。根据该报告，GPIF在上一财年的盈余为37.7986万亿日元，回报率为25.15%，均为历史最高。 GPIF分析说，因为主要国家实施财政刺激措施以刺激经济，使日本和国外的股票价格上涨。](https://www3.nhk.or.jp/news/html/20210702/k10013115811000.html?utm_int=news-politics_contents_list-items_006)
