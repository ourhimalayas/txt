据《美国之音》7月14日报道，英国反间谍高官警告称，英国国民要以警惕恐怖主义的态度看待中（共）国和俄罗斯的间谍威胁。

英国军情五处处长麦卡勒姆（Ken McCallum）预计将于星期三（7月14日）在总部发表讲话。据悉，该讲话将发出警告指出，“外国势力在英国从事间谍活动对民众带来的影响包括不安和不便；失去生计；以至于丧失生命。”

军情五处事先提供的讲稿节选中写道：“我们必须要像多年来对待恐怖主义那样，建立公众对外国威胁的了解，并加以应对。”

英国方面称，中（共）国和俄罗斯各自寻求偷窃商业敏感信息和知识产权；干预英国国内政治，以及播种虚假信息。

据悉，麦卡勒姆的讲稿还将提到，英国杰出的大学和研究机构的研究结果被盗窃或被复制；英国的公司也因失去自己曾精心建立的高超技术而被挖空。

目前，除了英国之外，美国、加拿大和澳大利亚等国也对中（共）国做出了类似的指责。

麦卡勒姆去年10月在伦敦首次发表公开讲话时曾表示，俄罗斯是英国目前面临的最大国家威胁，但中（共）国将在未来变得更加危险。他比喻称，俄罗斯过去制造“突发恶劣天气”，北京（中共）则是“改变气候”。

麦卡勒姆还指出，中（共）国和俄罗斯等敌对国家不再是集中进行窃取政府机密等传统的间谍活动，而是针对英国的经济和基础设施采取行动，并试图破坏英国的民主。
