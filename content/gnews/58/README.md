有些简单的知识需要提前说明，大概说一点点就是了，因为几乎这个东西是学无止境的（其实因为我不会）。中国人中特别流行的风水，周易，八字，等等，所有的这些都是以金木水火土五行为基础的，常见的生克为，生：金生水生木生火生土，接着又是土生金；克：金克木，木克土，土克水，水克火，火克金，然后又接着是金克木了，生克最后都可以绕成一圈循环起来。这个生克原理构成了最朴素的基础，无论哪一种术，都依照这个理论开始铺排。

伏羲发现了四季的变化，时间的轮回，轩辕黄帝把天干地支应用成为记录时间的符号，并加入农耕的重要节气，叫做历法，就是现在所说的老黄历。打开老黄历可以清晰地看到每个时辰都有一串代表历法的字，一共八个，比如2021年12月18日早上7点，换算成老黄历就是：辛丑/庚子/庚子/庚辰，就是这八个字，从左到右依次为年月日时，一个时辰是两个小时。如果这个时辰有个孩子出生了，那么这个八字就赋予了这个孩子。在婴儿潮时期，相同八字的人非常非常多，那个时期一个医院一个时辰同时出生的孩子听哭声就知道有很多。相同八字的人又因为出生地点、家庭环境的不同，导致人生有些许不同，大概率的走向不一致的也有，这个就说明八字只是一种参考，无论从哪些方面都不能作为唯一的方面。况且，所有的术数取到的都是时间，太相信时间对人的影响，很容易困在里面，人能不能超越时间，可以的，只要速度够快，人的意识完全可以做到这些。所以不要把这个东西过度妖魔化了，迷信的人，很把这个当回事儿，本质上是对自身的戕害，很容易什么都看不到，走进误区。
![](https://assets.gnews.org/wp-content/uploads/2021/12/48935286c0834f5e8ff9aa1095800c19.jpeg)中国历史上的婴儿潮 / 图源：网路
算命里说的八字就是出生时间的天干地支，融合了五行的符号，年月日时称为四个柱，年柱月柱日柱时柱，上面的叫天干，下面的叫地支，比如年柱的两个字又可以叫做是年干和年支，其他的依次类推。大范围的分析方法一般认为，年月代表外部，日时代表内部，就是自己，日就是哪天出生的，时就是哪个时辰出生的。时辰这个其实只有家人知道，但生日，基本上该知道的都知道，因为人总要过生日的。有个现象就很迷惑，在维基百科上查询到的人物资料基本都是精确到生日的，而在国内公开的共产党员的资料，大部分只有年月，没有生日（江泽民有公布生日）。共产党员无论官职大小，有一种普遍的公开资料的格式，不公布生日。众所周知共产党迷信是比任何人都迷信，到了要做妖术的程度了，盘古龙头给包上就是最好的例子。这个公布党员资料的方式就是代表了一种强制，加入了共产党呢，就只有年月这个外部环境，就是这个邪恶魔党，而没有日，人就好像被刻意抹掉了自己。孩生日，母难日，共产党不但让你忘掉自己，还让你忘掉母亲，认贼党做爹妈。
![](https://assets.gnews.org/wp-content/uploads/2021/12/4-27.jpg)共产党不公布生日，美国政要有生日 / 图源：网路
共产党把自己成立的具体时间是忘记了，究竟是不是7月23日，存疑很多。而后来毛贼东钦定的每年7月1日又没有什么说服力，这就造成了时间上的模糊，就像共产党这个组织一样，学俄罗斯，学欧美，啥啥没学会，学了个四不像，总之就是看不到自己，只能看到这人身上有无数个旁人的影子，有的甚至是互相排斥的，只要不妨碍作恶就可以。共匪给自己定的成为合法政府的时间是1949年10月1日下午二点，这个会议上决定了这个政府的领导班子，排一下黄历，这个时间是：己丑/癸酉/甲子/辛未。开国大典是下午三点，对于独裁国家来说，这个时间就没什么作用了，因为三点以后的所有的仪式都是为了证明二点的那个决定。从左到右年月日时依次为：己丑/癸酉/甲子/辛未，到这里了，看八字的老先生就要开始念念有词了，甲木生于酉月，官格，化官为印，财来合我，并可助官，官多为杀，印贴身化之，当有权柄，此为救应。哈哈，这些专用术语是结合了八字理论的诸多因素，此篇不做过多赘述，只从五行生克方面来谈一个问题：为什么共产党害怕寅虎年？

己丑/癸酉/甲子/辛未，从阴阳上来看，只有日柱一柱是全阳的，甲是阳木，子是阳水，而其他剩下的字全都都是阴的，如果看作一个人的话，这个人就是非常虚假的那种人，表里不一，因为日柱代表的自己和所有的外部都不同阴阳，这怎么能是一路子人呢？再者看日柱的下面，子水，子水是生甲木的，甲木在这个全盘是没有根的，木要长在土上，丑和未都是土，但是一个冻土，一个燥土，温湿失调怎么能长得好呢。甲木只能依靠坐下的子水来做一个水培的植物，随波逐流。子水是全国广大的老百姓，甲代表的这个政府不生产一分钱价值，全都是坐下的子水勤劳打工养着的，甲同时高高在上，把这子水狠狠踩在脚下。而为什么共产党害怕寅虎年，因为寅是甲木的最强根。本来这个盘里甲木可以做一个水培植物随波逐流，然而寅来了，就有一股冥冥之中的力量，要生根发芽，要长在土上，直向沙泥万丈埋，可以把全盘的平衡力量都打破。寅是甲的比肩，比肩就是要和你齐肩，一争高下，独裁国家只有1，一个领导一种声音，怎么能容忍这种行为呢，一定要斗个你死我亡，所以在这些年景里，内斗都导致了一些巨大的人道灾难。1950虎年，抗美援朝；1962年虎年，大饥荒；1974年，因为内斗周恩来而开展批林批孔；1986年，八六学潮；1998年，特大洪水。这些收录里，可能只是一些细枝末节，共产党的党史本来就是以篡改为主。2022年，也是虎年，很快就到了。
