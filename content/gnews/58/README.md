## 中共掌权者才是盗国卖国贼
` GTV新闻访谈` [轉載自GNews](https://gnews.org/zh-hans/1857999/)

作者：纽约香草山文艺部 – 宁文静
![This image has an empty alt attribute; its file name is Screen-Shot-2021-08-30-at-7.37.59-PM.png](https://assets.gnews.org/wp-content/uploads/2021/08/Screen-Shot-2021-08-30-at-7.37.59-PM.png)
1月12日，郭文贵先生在直播中指出，倘若政掌权者以国家财产为筹码私下换取政治与经济利益，他们就是名符其实的盗国、卖国贼。

![](https://assets.gnews.org/wp-content/uploads/2022/01/Slide1-11.jpeg)

郭先生解释道，公民在国家宪法下宣誓后却背叛了该宪法契约，这是叛国；如果把国家资产非法盗卖给另一个国家，这叫卖国。然而，中共国的普通公民既没有在宪法下宣过誓，也没有权力动用国家资产，因此，“判国卖国”于他们而言就是莫须有的罪名。相反的，只有那些真正掌握权力的人才是判国、卖国的罪魁祸首。

郭先生举例道，2002年，江泽民、朱镕基以“上海合作组织”的名义与周边国家签订了所谓的战略性合作协议，该协议却把中国28万平方公里土地的99%都卖给了哈萨克斯坦；而早在1960年前后，毛政府也把帕米尔高原96%的山脉都划给了阿富汗。

郭先生指出，这些人才是真正的卖国贼。同时，盗国贼家族不但掏空了中共国的家底，他们还制造出无数愚昧、无知且顺从的民众。而今，中共国内几千万个杨改兰在苟且偷生，直至被逼上绝路，其后，他们的惨死还要被中共的舆论机器拿来消费。

郭先生强调，新中国联邦人一定要厘清这些概念，传递真相，不能被无中生有的恐吓吓倒。
