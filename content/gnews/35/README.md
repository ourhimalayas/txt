## 非法政府共产党

![](https://assets.gnews.org/wp-content/uploads/2021/10/1-158.jpg)传统山水文化 / 图源：网路
自从美利坚独立宣言以来，开辟了自由民主法治的人类生活模式，吸引了世界上多少精英，法治+信仰成为天赋人权的有效保障。在世界其他追求先进的国家，也在效仿这种方法。有些邪恶组织，比如共产党，得知自己不合法的情况下，就强行立法让自己合法，然后挨个蓝金黄那些资本主义见钱见色眼开的高官，投其所好来进行勒索。共产党占领了中国这片大地以后，他也知道自己是违法政权，所以用了很多投机取巧的办法，比如啥改风水啊，挑日子啊，做法事啊，加入邪恶组织得到庇佑啊，总之就是什么能续命就干什么，无论付出什么代价。

中华人民共和国，非法政府；中华民国，合法政法。从名称上来看，区别虽然是多了几个字，实则不然，中华民国，民是一个大概念，通俗点说老百姓都是民，民国，民在国之上，民、国中间没有任何间隔，是可以认真沟通的，这就奠定了民、国的关系，颇有孟子“民为贵，社稷次之，君为轻”理念之遗风。而中华人民共和国，人民是个政治概念，按共产党的说法，今天说你是人民你就是人民，明天说你背叛了人民那就马上把你拉入大牢关起来施以暴行，叫做“人民的惩罚”，实际上谁打的，是共产党指派的走狗。很多共产党的电影，都是以人民为幌子，什么“你背叛了人民背叛了党”，那个时候国内老百姓以为自己是“人民”，古汉语中确实是人民，简简单单的老百姓，如《孟子·尽心下》诸侯之宝三：土地、人民、政事。共产党使用的人民这个词是日本翻译过来的政治词汇。如果用另外一种断句方法，中华人民共，和国，人民共就是独裁共产党，共在国之上。和国，就是依附，啥意思呢打个简单比方，就是“我和你”，不是“我们”。清晰点说，就是把人分得特别清楚，人人都是有阶级的，他吃一口肉，一口骨头都不给你留的那种。这和天赋人权人人平等本身就是背道而驰的。

![](https://assets.gnews.org/wp-content/uploads/2021/10/social_media.jpeg)中共国国庆日浪费民脂民膏的面子工程巨大花篮 / 图源：网路

共产党给自己选的开国大典的日子是北京时间：1949年10月1日下午3点。毛贼东宣布“中华人民共和国，中央人民政府成立了”。这个时间只是对老百姓，或者新闻媒体宣布的时间，真正的“人民”早都知道了。中央人民政府成立的时间是往前推一个小时，下午2点，在大典前已经召开了会议，给各色人等任命了职位，从独裁的角度来说，这个会比所谓的大典要重要多了，这才是中央人民政府成立的时间。中华人民共和国成立的时间是1949年9月21日。开国大典的意义好像是在用大型集会来判定共产党是合法组织。非法就是非法，清晰的红线，哪有啥好商量的？西方民主国家从未正式承认共产党的合法性，就连最著名的历史文件《中英联合声明》里面都很甄词酌句，但怕用错了。用的acknowledge，很多人解读过这个词在这里的用法，但是作为英语为母语的国家，我不觉得任何人比大英帝国更加懂得这个词用在这里的意义。

对比美国来看，美国相同的日子即国庆日为7月4日，就是签署独立宣言的那一天。尽管这个日子在后面被各种质疑过，到底是不是那一天。我想大多数美国人并不是认可什么日子，主要是认可那件事，就是签署独立宣言。而中共国的做法是宣布，宣告，广而告之，具体背后怎么操作的，老百姓没有权利知道，老百姓只要被告知结果就好了。从来不把人当人看，老百姓对自己合法的政治权利连知情权都没有。

网上有很多神棍分析1949年10月1日下午三点是个好日子。呵呵，共产党选的那个日子怎么说呢，共产党就是刻意选的好日子，想承接天地五行的美好气场，但你得是个合法政府，你得真正给人民一人一票选举，天地才会让你这种气场up&up啊，我们都知道此消彼长，阴阳相合的道理，逆天地而行，再好的日子都能给用废掉。

网上还有神棍说2020年6月4日17:37新中国联邦成立日是个大凶的日子。我始终认为一个会择日的神棍和此人的见识有太大关系了，比方说此神棍长期给小卖铺，小餐馆择日，接过的最大的业务是给大超市择日。给一个大银行择日估计都择不好，甭说再大的机构，再大到国家层面的了。新中国联邦这个日子，那是有8964的历史来由的。郝海东先生勇敢站出来宣读战友书写的庄严的新中国联邦宣言，还有本来还乌云密布，怎么又突然放晴，七哥咬破手指，天上来了一道闪电那种震撼，这种又怎么解释呢？

不过也说得有道理，即然是大凶日，则不破不立。最近七哥又兴奋地在车里直播告诉战友喜联储上市择日的来由，奇门遁甲一句也听不懂，怎么7个1就和一年中三个最大的日子其中之一给重合了，哇好神奇。
