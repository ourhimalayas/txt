## 西安考研乱象频生，部分考生投考无门
` 纽约香草山MOS02` [轉載自GNews](https://gnews.org/zh-hans/1801073/)

作者：香草山教育部 – Noemi
![This image has an empty alt attribute; its file name is Screen-Shot-2021-08-30-at-7.37.59-PM.png](https://assets.gnews.org/wp-content/uploads/2021/08/Screen-Shot-2021-08-30-at-7.37.59-PM.png)
据中共国内媒体12月27日报道，西安封城后，当地近期组织的研究生考试现场一片混乱，出现了部分被隔离考生因各种防控原因投考无门的乱象。 据了解，近期陕西省西安市因疫情失控而大规模封城，而每年一度的全国研究生考试却迫在眉睫。在西安城中，如何能顺利完成线下考试就成为了当地10万考研学子最关心的问题。在考试之前，部分被隔离的考生就曾多次向当地的防疫指挥部询问相关考试计划是否有变，都被工作人员告知考试安排无误并被要求要耐心等待。在得到官方的肯定答复后，这些考生都表示十分放心，但是最终却没能等来自己的考卷，被迫放弃了苦等一年的考研机会。

![](https://assets.gnews.org/wp-content/uploads/2021/12/6080997.jpeg)

据当地考生反映，在过去几天的考试经历中，波折和焦虑成为了考生们共同使用的高频词汇。在开考前不到20个小时，曾有个别考点因为出现确诊病例而被取消，考生们不得不深夜拖着行李箱绕过半个西安城赶往新的考点。开考之前，西安市政府也曾组织当地的网约车司机志愿者，组成5000辆车规模的送考车队，但在开考前这些司机却只被分配到十六七名考生。无奈之下，司机们只得自行发布义务助考信息后，才有大批考生与车队联系。而在咸阳一家的隔离酒店里，一名从年初就开始紧张备考的女生，最终还是没能等来自己的考卷。

在开考前一天，该名女生还因就考问题连续多次向当地的研究生考试院、学校研招办和所在区的疫情防控指挥部致电，仅拨出话费就高达100多元。在当晚七点钟，该生终于得到相关防疫部门的肯定回复说，会有专人配送考卷上门。然而直至第二天中午第一门考试结束之后，也未有任何人员联系到这位考生，更无任何人配送考卷上门。情急之下，该女生自行联系到了考点的负责老师，却被告知未曾收到任何通知需要送考卷上门。随后，她就此事再次询问了政府的工作人员，对方却称即便是该生能继续参加考试意义也已不大，不如趁早放弃这次考试。 据悉，这名女考生已经是第二次参加研究生考试，家中经济条件十分困难。在这一年中，家中母亲确诊了乳腺癌，自己却无时间回家探望。高昂的医药费、考研辅导费都担在了父亲一人的身上。备考期间，她还经历了骨折、失眠等各种病痛的折磨。该女生在采访中说，黄金时期的人生又有几个一年，这一年太苦了，却没有在该结束的时候结束。

有知情人士表示，西安的考研乱象仅是中共疫情、疫苗灾难的冰山一角，事实上，中共的生化超限战争所带来的将会是几代人都难以抹去的伤痛。
