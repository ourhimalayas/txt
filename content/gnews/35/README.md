**摘要翻译：**

由赖纳·费勒米奇（Reiner Fuellmich）博士领导的由1,000多名律师和10,000多名医学专家组成的团队，已开始针对美国疾病控制及预防中心（CDC）、世界卫生组织（WHO）和达沃斯集团的危害人类罪提起法律诉讼。
![](https://assets.gnews.org/wp-content/uploads/2021/12/120601.png) 
费勒米奇和他的团队提出，有问题的PCR检测以及命令医生将任何合并症的死亡标记为新冠病毒死亡是一种欺诈。PCR检测从来就不是为检测病原体设计的，并且在35个周期内是100%有问题的。美国疾病控制及预防中心监督的所有PCR检测都设定为37到45个周期。疾病预防控制中心承认：任何超过28个周期的检测都不可能获得可靠的阳性结果。仅仅这一点，就使通过使用这种有问题的检测跟踪到的90%以上的所谓新冠病毒病例/“感染”无效。

除了有缺陷的测试和虚假的死亡证明外，“实验性”疫苗本身也违反了《日内瓦公约》第32条。根据1949年《日内瓦第四公约》第32条，禁止“对受保护人进行非医疗所必需的致残和医学或科学实验”。根据第147条，对受保护人进行生物实验是严重违反《公约》的行为。

***“实验性”疫苗违反了《纽伦堡守则》的全部10条规定，对企图违反这些国际法的人将处以死刑。***

**该“疫苗”不符合下列被视为疫苗的五项要求，根据定义，该“疫苗”属于医学“实验”和试验：**

***提供对病毒的免疫力***

这是一种“有漏洞”的基因疗法，它不提供对新冠病毒的免疫力，以及其声称的可以减轻症状，但现在有60%需要在急诊室或重症加护病房就诊的新冠病毒感染患者，是接种了两针疫苗的。

***保护接受者免受病毒感染***

这种基因疗法不能给接种者提供免疫力，且两针疫苗接种者仍然可以感染和传播病毒。

***减少因病毒感染而导致的死亡***

这种基因疗法并没减少感染者的死亡。感染了新冠病毒的两针疫苗接种者也有死亡。

***减少病毒的反复扩散***

这种基因疗法仍然使病毒散播，因为它对病毒的免疫力为零。

***减少病毒的传播***

这种基因疗法仍然允许病毒传播，因为它对病毒的免疫力为零。

***违反《纽伦堡守则》的情况如下：***

**《纽伦堡守则》 1：受试者自愿同意是绝对必要的**

在没有知情同意的情况下，不得强迫任何人进行医学实验。许多媒体、政界人士和非医学人士都在告诉人们要接种疫苗。他们没有提供关于这种基因疗法的不利影响或危险的信息。你从他们那里听到的只是–“安全有效”和“利大于弊”。各国正在利用封锁、胁迫和威胁来强迫人们接种这种疫苗，否则就会在疫苗护照或绿色通行证的授权下被禁止参与社交。在纽伦堡审判期间，甚至媒体也被起诉，其成员因向公众撒谎而被处以死刑，许多医生和纳粹分子被判犯有危害人类罪。

**《纽伦堡守则》2：试验目的必须能为社会带来福祉，且无法以实验以外的其它方式获得**

如上所述，基因疗法不符合疫苗的标准，也不提供对病毒的免疫力。针对新冠病毒，还有其它卓有成效的治疗方法，例如伊维菌素、维生素D、维生素C、锌，以及提高免疫系统对流感和感冒的免疫力。

**《纽伦堡守则》3：以动物实验结果和疾病的自然发展为基础进行实验**

这种基因疗法跳过了动物试验，直接进行了人体试验。在辉瑞使用的mRNA疫苗研究中—一项对恒河猴使用新冠病毒mRNA疫苗进行的候选研究时，在该研究中，所有猴子都患了肺部炎症，但研究人员认为风险较低，因为这些是2-4岁的年轻健康猴子。以色列使用了辉瑞公司的疫苗产品，国际法庭已经接受了一项索赔，即有关80%的接种者因注射这种基因疗法疫苗而患有肺部炎症。尽管有这种令人震惊的发展，辉瑞公司还是在没有动物试验的情况下，继续开发他们的新冠病毒mRNA疫苗。
