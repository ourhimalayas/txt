## 沙特阿拉伯一天内处决81人

据官方媒体报道，沙特阿拉伯在一天内执行了 81 起死刑判决。这些罪犯被指控与恐怖组织有联系。该国长期以来一直因其高死刑率而受到批评。

据官方媒体报道，沙特阿拉伯一天内有 81 人因恐怖主义罪名被处决。 SPA 官方通讯社周六报道称，这些罪犯与“伊斯兰国、基地组织、胡塞武装和其他恐怖组织”有联系。因此，他们计划袭击重要设施或安全部队成员，并将武器走私到该国。

这是沙特阿拉伯单日报告的最高处决人数。去年全年，海湾国家共执行了 69 起死刑判决。

根据 SPA，被处决者中有 73 人是沙特阿拉伯国民，7 人是也门人，1 人是叙利亚人。所有 81 人都被沙特阿拉伯法院判定犯有“多项令人发指的罪行”。

沙特阿拉伯正在领导一个军事联盟，在也门与伊朗支持的胡塞叛军作战。最近几个月，胡塞叛军在沙特阿拉伯发生了数起袭击事件。

长期以来，沙特阿拉伯一直被批评为世界上处决率最高的国家之一，其司法制度不透明。
