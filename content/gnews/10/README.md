那么到了下一个时期，就是宪法共和国，就是类似于像法兰西，就是法国大革命之后的共和国这一套体系。它是一个职业政客和技术官僚治国的兴起，我们现在的大部分国家，都是在这个体系里面的。这个法制的程度也是中等到中等偏上，这个是看每个国家，就像委内瑞拉虽然也有法制也有民选，但最后它也变成了一个共产主义的独裁国家。每个国家具体情况是不一样的，它管理复杂度是比较中或者是高的，个人自由度也是比较高的，自由协作也是比较高的，这个就是一个规则的兴起。但是大家可以看得出来，即便是像加拿大这种法制，就是一个标杆的法制国家的时候，也可以随时因为统治者的自身原因，会把它一下就打到解放前，回到一个类似于独裁国家的这种趋势，大家可以看到这种规则的脆弱性是非常非常大的。

第三个阶段就是我们未来的一个方向，就是Rules without rulers ，我们有规则有法治，但是没有统治者。第一个阶段就是我觉得未来几年内可以实现的,有一种理念叫 Start up  nation，叫“创业国家”。这个现在已经有很多地方都有了，特别是在美国、加拿大或者澳洲，这种联邦属性的地方，只要自己有一块土地，就可以宣传我叫一个什么袖珍国，比梵蒂冈还小的就可能有几十号人，类似国中国的这种东西。但是它为什么不行呢？就是因为它没有规模效应，没有相关的像金钱体系。大家知道金钱也是一种信息和能量，就是它没有这种议价能力跟各个国家。我们未来的新中国联邦可能就是这种的一个比较成熟的模式。第一，我们有独立的金融体系和独立的运作的组织架构。我们有金钱就是资本主义，因为现在我们还是处于一个资本主义的阶段，我们必须要跟其他国家靠经济实力才能有议价。包括法治和科技治国这一部分，我们现在就已经开始使用了，类似于像喜交所、喜联储，还有G-clubs。我们没有一个人去过 G-clubs，这是一个实体的Club,我们所有东西都是在线上申请完的,你发现没有，这是个很强大的东西。全世界没有任何一个国家可以做到这么多的事情，能够做到在线无国界。你看我们现在直播，在线看这个直播，还有播出直播，包括我们后面的义工做的这些，每个农场都有很多不在本地的战友，他就是一个无国界、虚拟化的东西，而且我们的管理复杂度也是很高的，个人自由度也是很高的，自由协作也是非常高的。
