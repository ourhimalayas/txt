## 这样“护犊子” 国际奥委会表演大跌眼镜

![](https://assets.gnews.org/wp-content/uploads/2021/12/9-6.png)
路透柏林12月15日，国际奥委会主席巴赫(Thomas Bach)说，参加2022年北京冬奥会的90个国家的政府多数没有选择加入对此次冬奥会的外交抵制行动。

美国、英国和澳洲等国最近因中共国的人权记录宣布对2月4日至20日的北京冬奥会进行外交抵制，中共国将这一立场视为“政治姿态”。

欧盟方面，法国等国家已寻求欧盟的共同回应，但欧盟对是否加入外交抵制感到纠结，担心中共国的报复行为会损害贸易。

“有像美国、英国和澳洲这样的国家做出了那样的决定，”巴赫周二晚间对德国媒体ZDF表示。

“其他几个国家，不仅是几个，如果我们说将有90个国家奥委会出席这次冬奥会，很可能有70多个甚至80多个国家的政府没有做出这样的声明。”

“但他们都有一个共同点，那就是他们支持他们的奥运队伍……这样他们的运动员才能实现他们的奥运梦想，”巴赫说。

国际奥委会作为一个非政府、非营利国际体育组织，理应不牵扯政治，对国际体育事务更应作出中立表述，不应不宜有任何立场。显然国际奥委会的表述带有风向，作为国际大型的体育赛事机构，本应不会为外界事务所影响。但是面对中共这个“蓝金黄”的老手，高手，显然已经深深被共产党影响和渗透。近年来，共产党加大了对联合国，国际刑警组织，等各种国际组织以及各国政府要员的渗透拉拢，深深地影响着国际事务发展方向和进程，对全世界的损害到了骇人听闻的地步。国际社会意欲消除病毒给全世界带来的危害，恢复正常国际秩序，不消除“中共”这个“毒母”是不可能完成的。
