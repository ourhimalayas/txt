**摘要翻译：**

波士顿凯尔特人队的篮球运动员埃内斯·坎特（Enes Kanter）周二直言不讳地批评中共政府压迫维吾尔穆斯林，称中共领导人习近平是“无情的独裁者”，称中国共产党（CCP）是邪教，并宣称尽管中共极力压制香港的民主，但香港总有一天会获得“自由”。

坎特之前曾抨击习近平是“独裁者”，但他对香港独立的全力支持，是他最近对共产主义中国的批评的补充。2019年，时任休斯顿火箭队总经理的达里尔·莫雷(Daryl Morey)，在推特上分享了一条支持香港民主运动的信息，该协会对其做出了回应，这对于一名美国国家篮球协会 (NBA) 球员来说尤其重要。

由于担心共产党针对其商业利益的报复，NBA与莫雷保持距离，并向中共政府做出了谄媚的道歉。批评者抨击NBA的懦弱，尤其是当他们发现该协会的声明被翻译成中文后更加卑躬屈膝。

当NBA球员勒布朗·詹姆斯站出来支持联盟并批评莫雷时，坎特表示他对詹姆斯对独裁邪恶的明显无视感到惊讶。“自由不是免费的，”坎特在2019年10月提醒詹姆斯。

中共国事件发生一年后，莫雷从休斯敦火箭队辞职，这促使中共国官媒兴高采烈地猜测他为支持香港民主“付出了代价”。当莫雷签署了一份非常有利可图的五年合同，担任费城76人队的篮球运营总裁时，共党的快乐很快就消退了。

坎特上周使用了职业运动员武器库中最强大的武器之一：鞋子，加强了与中共暴政的斗争。

在10月25日的社交媒体视频中，坎特呼吁耐克“立即停止现代奴隶制！”——这指的是中共国工厂在生产耐克运动鞋时，使用强迫劳动的指控。坎特质疑耐克公司的保证，即它已采取措施消除其供应链中上的强迫劳动。

“你知道整个服装和鞋类行业，都受到中共对维吾尔族人强迫劳动这种罪恶的污染吗？” 坎特在视频中问道。“许多知名的全球品牌都受到牵连，是的，其中包括NBA最大的赞助商之一，耐克。”

坎特在与数百名维吾尔族、西藏和香港自由活动人士一起参加国会山集会后，穿着自己定制的那款鞋子参加了上周六与华盛顿奇才队的比赛。

坎特还有一双运动鞋，为1989年天安门广场大屠杀向中共叫板，其中与传奇“坦克人”对峙的坦克炮塔已被小熊维尼头像取代。党在多年前就禁止了小熊维尼，因为肥胖的习近平被比作卡通熊而受到嘲笑。

为了向他周五参与的粉丝致敬，坎特不仅加入了#自由西藏和#自由维吾尔人的标签，还加入了#自由香港和#与台湾站在一起的标签。

中共国对坎特的最新言论做出了回应，封锁了波士顿凯尔特人队比赛的流媒体视频，并阻止了互联网对坎特名字的搜索。到目前为止，NBA一直在默默地支持坎特说出自己的想法。

至于香港，中共政府违反其尊重该岛自治的承诺，实施了严厉的《国家安全法》，使民主运动受到了沉重打击。

工会和民间社会团体最近在国家安全法的压力下解散了，其中包括数十年来赞助香港天安门广场守夜活动的团体。支持民主的报纸被赶出了该行业，记者受到恐吓，电影被审查。支持民主的立法者已被清除出立法机构。甚至连国际人权组织都撤出了香港，担心受到中共傀儡政府的迫害。
