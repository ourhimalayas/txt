郭文贵先生11月12日大直播中说到（摘要）：为了把六中全会开好，恒大会在六中全会之后爆雷，谁也挡不住，谁也救不了。恒大爆完了就是富力、融创、碧桂园、合生创展、花样年……

今天就来谈谈富力。广州富力地产股份有限公司（以下简称富力地产），成立于1994年，从广州旧厂改造起步，2002年承建北京富力城，2005年在香港联交所主板上市，2007年开始在中国十一个核心城市进行地产开发，并在华南、华北、中部地区的一二线城市开发大量商业地产，包括商业写字楼和星级酒店。至今富力地产业务已经扩展到国内外145个城市和地区，450个项目，包括海外马来西亚的柔佛新山，澳大利亚的墨尔本、布里斯本、韩国仁川、英国伦敦。富力已成为房地产华南五虎之首（其余四家是：碧桂园、恒大、雅居乐、合景）。

富力地产最巅峰的时刻恐怕就是2017年7月，在中共“降杠杆”大背景下，斥资199亿一举鲸吞万达77家酒店，当时王健林出售万达酒店、万达文旅产业、万达广场，不断套现还债，富力却逆势而动，可谓风光无限。此时富力总共拥有超过100家酒店，成长为全球五星级酒店霸主。富力紧接着又并购了伦敦九榆树广场地块，成功跻身千亿房企之列。

正所谓：“曾见金陵玉树莺声晓，秦淮水榭花开早，谁知道容易冰消！眼看他起朱楼，宴宾客，眼看他楼塌了。”

到2021年，富力地产上半年年报数据显示公司借款总额为1433亿元，一年内到期债务占总借款的36%。截至2021年6月30日，富力地产的现金持有量是288亿元（包括无限制现金余额为128亿元），短期借款总数为519亿元，短期债务覆盖率为25%。今年下半年，富力地产有2亿美元的美元债到期，另外还有17亿元的可回售公司债券。2022年，富力地产大约有190亿元的还款。

2021年9月3日，穆迪投资者服务公司将广州富力CFR评级从B1下调至B2，并将富力香港的CFR评级从B2下调至B3。与此同时，穆迪还将富力的评级置于审查之中，以便进一步下调其评级。

穆迪高级副总裁曾文健表示：“广州富力的CFR下调至B2，反映了该公司的再融资风险增加，因为其获得离岸资金的渠道减弱，且到期债务数量庞大。富力香港的CFR下调至B3反映了其母公司在需要时提供财务和运营支持的能力减弱，以及子公司的个体信用质量较弱，规模小，且酒店业务的经营环境波动较大。”

根据这些公开的信息与数据，可以想见富力地产将不得不依靠新的融资或资产出售为其即将到期的债务提供资金。只是如今境内外债券资本市场融资渠道急剧减弱，借新债还旧债肯定行不通，使用内部资源偿还到期债券唯有卖卖卖。

风水轮流转，仅仅时隔三年，富力就走上了万达“卖家当”的老路。

2020年11月，富力向黑石集团转让了广州国际机场富力综合物流园70%权益，作价44.1亿。2020年末，富力地产将其在广州珠江新城的写字楼、旧改等多个项目，都质押了股权，富力地产质押了十家子公司的股权。据媒体报道，富力拟向一家国企出售海南富力海洋欢乐世界的股份。富力董事长李思廉透露，富力去年拟出售的投资性物业和地产项目规模在120亿元左右。

2021年9月，富力掌门人李思廉和张力将个人名下的境外控股公司富良环球股权，以不超过约100亿元人民币的价格转让给了碧桂园服务，富良环球间接持有富力物业100%股权，而富力物业也属于李思廉和张力的个人资产。换句话说，张力和李思廉把自己的552个物业项目都卖给了碧桂园。

2021年9月20日富力地产发布公告称，作为公司执行董事和大股东李思廉和张力将在未来一两个月内提供约80亿港元的股东资金支持公司发展。

2021年10月初，港交所披露的信息显示，李思廉将永升（亚洲）16%的股权转让给了新世界发展董事会主席郑家纯全资持有的Celestial Pioneer Limited。

2021年10月，富力地产董事长李思廉将北京幻想纵横网络技术有限公司个人持股2.0281%转让给了上海七猫文化传媒有限公司。

显而易见，李思廉和张力自掏腰包80亿港币补充富力流动资金，从侧面说明了富力的资金链非常紧张，如果再不提供支持，恐怕就要发生债务违约。

天眼查的资料显示，目前由李思廉担任的法定代表人、高管及股东的企业总共达到138家。其中，担任法定代表人企业106家、担任高管的企业达121家，仅担任股东的企业有28家。

目前来说李思廉转让个人名下的公司股权还仅仅是一小部分，是否会继续退出其个人投资的项目，拭目以待吧。

富力地产所面临的严重的债务危机，仅靠大股东输血和卖掉一些非核心资产显然不够。目前富力依然是脚踩三道红线，现金短债比仅为0.55，净负债率为123.5%，剔除应收账款后的资产负债率在74.9%。

在富力资金链紧张、业绩下滑的背景下，更难过的或许还有数万名已经被裁员和随时存在“下岗风险”的员工。

在富力地产的2020年半年报中显示：截至2020年6月末，富力共有员工约39244人，而2019年末为62305人，半年内员工人数降幅达37.01%；总员工成本则同比减少18.89%至14.6亿元。

去年11月又有传闻称，富力集团已暂停公司所有岗位的招聘，2021年要在2020年基础上裁员30%。

尽管富力集团此后发布公告澄清，并表示并无裁员计划。但有员工称：“我们的薪资都降了30%，好多人因此而离职，也有的是被劝退。许多部门上半年减少了约15%的人。”

富力地产已经快到山穷水尽的地步，那么何时爆雷呢？

郭文贵先生在10月29日的直播中说道：明年是中共国房地产爆雷的开始，最严重应该是5月份，然后最夸张的就是7月份，这跟明年的二十大和两会是息息相关的。李嘉诚的和黄60%股权是江家的，背后会扯出很多金融问题。张松桥、刘銮雄和恒大，还有融创、碧桂园、合生创展，花样年、富力的李思廉、张力，所有的这些人……导致的结果就是给做融资担保的，会形成一个崩溃效应。
