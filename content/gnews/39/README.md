**By:[秘翻新闻](https://gtv.org/broadcast/watch/61baff8d7882000d5101f1b9)**

**2021-12-16**

大纪元12月15日消息。中共国家主席习近平周三在北京同俄罗斯总统普京举行视频会晤。在俄方发布的影片里，意外出现习连咳三声的画面。

据俄方发放给国际媒体的两分多钟影片显示，习近平和普京两人在会晤开始时互相挥手，在影片播出约40秒后，习近平连咳三声。

根据俄罗斯电视台直播的图像，普京说，他计划于明年2月在北京与习近平亲自会面，他说，“按照约定，我们将举行会谈，然后参加冬奥会的开幕式。”

习近平近年的健康倍受关注，此前已有几次异常表现引发外界猜测。

习近平2020年10月上旬南下广东考察，参加深圳经济特区建立40周年庆祝大会并讲话，也曾因为频频咳嗽，意外成为关注点。

中共央视在进行活动直播时，却在这些时刻将画面切换到台下的来宾，但收看直播的观众仍可清晰听到习近平咳嗽和拿水杯的声音。

习近平2019年3月25日访问法国时，也曾因“走路不稳”的姿态引起社交网络关注。社交媒体一直广泛流传着习近平与法国总统马克龙在凯旋门检阅仪仗队时“身体两边摆动，脚步不平衡”“步伐零碎”“似乎无法迈开双腿”的说法。

对此，有人认为或与习健康异常有关，一些国际媒体还因此引申到习近平的接班人问题。

中共领导人的健康状况向来是“国家机密”，除非本人去世，官方不会披露任何相关的资料。1976年夏天，中共最高领导人毛泽东已病入膏肓，但北京当局对外仍然保持一致口径，指毛“身体健康”。

今年11月上旬的中共十九届六中全会，据官方称是确立了习的核心地位和习“新时代”思想的指导地位，但并没有任何有关习近平安排未来接班人的迹象。

有台湾学者表示，习当政十年，从官方报导的画面看，他到外面调研两三天后，体力就会有一些下降。但是对于威权的领导人来说，通常不会直接在他准备要交出权力之前，就开始安排接班人。
