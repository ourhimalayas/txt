乌克兰总统弗拉基米尔-泽连斯基（Volodymyr Zelensky）周五告诉美国有线电视新闻网，“世界所有国家”都应该为俄罗斯总统弗拉基米尔普京可能在对乌克兰的战争中使用战术核武器的可能性做好准备。

泽连斯基周五在基辅总统办公室接受美国有线电视新闻网的杰克·塔珀（Jake Tapper）采访时表示，普京可能会转向核武器或化学武器，因为他不重视乌克兰人​​民的生命。

“不仅我——全世界，所有国家都必须担心，因为它可能不是真实的信息，但它可能是事实，”泽伦斯基说，转而用英语强调他的观点。

“化学武器，他们应该这样做，他们可以这样做，对他们来说，人民的生命，什么都​​没有。这就是为什么，”泽伦斯基说。“我们应该认为不要害怕，不要害怕，但要做好准备。但这不是乌克兰的问题，不仅是乌克兰的问题，也是全世界的问题，我认为。”

美国官员警告说，如果普京陷入困境，可能会转向在乌克兰使用战术核武器。中央情报局局长比尔伯恩斯周四表示，中央情报局“非常密切地”关注这种可能性，同时强调美国尚未看到任何迹象表明俄罗斯准备采取这样的步骤。

“鉴于普京总统和俄罗斯领导层可能陷入绝望，鉴于他们迄今为止在军事上面临的挫折，我们任何人都不能轻视可能诉诸战术核武器或低当量核武器所带来的威胁，”他在佐治亚理工学院的公开讲话中说。

本周，俄罗斯最重要的海军军舰之一在黑海沉没，乌克兰称这是导弹袭击的结果，而俄罗斯则声称这是由于弹药爆炸引起的火灾。

一名美国高级国防官员周五表示，两枚乌克兰海王星导弹击中了莫斯科，一名美国官员表示，袭击和随后沉没的船只是乌克兰导弹的结果。然而，泽连斯基对乌克兰是否发挥了作用持谨慎态度。

“我们知道它已经不存在了。对我们来说，它是对付我们国家的有力武器，所以它的沉没对我们来说不是悲剧。我希望你们，其他人，意识到这一点。武器越少， “攻击我们国家的俄罗斯联邦对我们来说更好。他们的能力越差。这很重要。”

泽连斯基说，乌克兰官员认为大约有 2,500 至 3,000 名乌克兰士兵在战争中丧生，将这一数字与他所说的俄罗斯 19,000 至 20,000 人的伤亡人数（俄罗斯承认有 1,351 名军人伤亡）进行比较。泽伦斯基补充说，大约有 10,000 名乌克兰士兵受伤，“很难说有多少人能幸存下来”。

他说，平民伤亡更难量化。

“谈论平民是非常困难的，因为在我国南部，城镇被封锁——赫尔松、别尔江斯克、马里乌波尔更东边，以及东边沃尔诺瓦卡所在的地区——我们只是不知道有多少人在被封锁的地区死亡，”泽伦斯基说。

当被问及本周发布的令人痛心的视频显示他的国家的死亡和破坏，包括一名乌克兰妇女在井中发现她儿子的尸体时，泽伦斯基说：“这对我来说是一个巨大的痛苦。”

“作为父亲，我不能看，只是因为你想要的只是报复和杀戮，”他说。“我必须看着这个州的总统，那里有很多人死去，失去了亲人，还有数百万人想要活下去。”

“我们所有人都想战斗，”他继续说。“但我们都必须竭尽全力，让这场战争不至于无休止。时间越长，我们的损失就越多，所有这些损失都会像那一场一样。”
