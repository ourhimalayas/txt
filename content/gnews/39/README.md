**二）教育，为圣教之意**

终于我们明了，原来“教育”的真正含义是对人施行“与人为善”的理念和行为规范教化。写到此，我想起来了一个经典的佛教故事：有一天，唐代著名大诗人白居易问鸟窠禅师：“禅师，依你之见，究竟何为佛法？”鸟窠禅师想了想回答说：“哦，你问这个呀，很简单，佛法教就八个字，诸恶莫作，众善奉行。”白居易听完哈哈大笑道：“禅师啊禅师，你开玩笑吧，就这两句话，三岁孩童都懂，还用你说啊！”鸟窠禅师不紧不慢，缓缓说到：“三岁孩童，虽懂得；但八十老翁，却行不得啊。”白居易听完，脸一阵绿一阵白，半响说不出话来。

鸟窠禅师不愧是大师，他用最浅显的道理，把佛教的精髓告诉了白居易。佛教它究竟到底要说什么？要告诉世人什么？佛教创始人释迦牟尼佛，说法讲经四十九年，那么他究竟要告诉世人什么呢？其实，就是鸟窠禅师告诉唐代大诗人白居易的这“八个字”：诸恶莫作，众善奉行。

这八个字，其实就是四个字：诸恶莫作。因为“诸恶莫作”反过来不就是“众善奉行”吗？而“众善奉行”反过来不就是“诸恶莫作”吗？所以，说“诸恶莫作”可以，说“众善奉行”也行，它们的意思完全是一样的，这就是前文我们说到的“教育”的本质含义和真谛。释迦牟尼佛，四十九年说法讲经，就是这四个字：“诸恶莫作”，大家不要以为佛教真谛是那么的高深、那么的高大上，不是的，绝对不是！伟大的真理都是简单的，越是伟大的真理就越是简单。因此，我们完全可以得出一个惊世的结论：佛教的起源非宗教，而是教育。释迦牟尼佛，就是一个自愿的，地地道道的多元化义务教育工作者，他哪是什么神呢？

那么，什么是“诸恶”呢？在佛教里，它讲“恶”共有十种：身体有三种，分别是杀生、偷盗和邪淫；嘴巴有四种，分别是妄语（说假话）、绮语（花言巧语）、两舌（挑拨离间的话）和恶口（骂人的话）；思想（说思维、观念、意识都是一个意思）有三种，分别是贪、嗔、痴。这叫“诸恶”，或叫“十恶”；那么反过来当然就是“诸善”，就叫“十善”。“ 诸恶莫作”，就是让你“十恶”千万不要去做！如何让你从出生到死亡，都不要做“十恶”，这就叫教育！

其实，只要我们用心，去仔细认真看一看世界上“四大宗教”的创始人，无论是释迦牟尼佛，还是默罕默德，还是基督，或玛利亚，以及我们中国的道教、儒教，哪一个宗教的创始人，不是教育“与人为善”的呢？我们再仔细去琢磨，我们就会发现，无论是《圣经》，无论是《古兰经》，无论是《道德经》，佛经就更不用说，统统讲的都是这“十善”，只是它们各自所表达“与人为善”的用词不同，但意思完全一样！

佛法讲，世间法和出世间法，很多人不了解佛教，以为”世间法和出世间法”是两回事。很多虔诚的佛教徒误解佛教之真意，只修“出世间法”而不问“世间法”；有的甚至把“世间法”和“出世间法”对立起来，把“大乘法”和“小乘法”对立起来。而他们不晓得，“十善”是一切“出世间”之本、之根！犹如大树，如果没有根，树何以长成？就是这么一个简单得不能再简单的道理，可有的人就是死活都悟不到。

人，为何要对其进行“与人为善”的教化？善，即为“十善”。为何，人一生下来就要告诉他一生都要行“十善”？就在这篇文章的前几日，普京大帝下令向乌克兰发动军事打击。战友们，普京大帝，包括习太阳未来的一天攻击台湾，这难道说他们是在行“善”吗？NO！这是十足的恶！大恶之中的大恶！杀害生命为一切恶之首！我们来看看普大帝发动战争带来的十恶：一恶，侵略士兵必杀害生命；二恶，侵略士兵必偷盗；三恶，侵略士兵必邪淫；四恶，妄语假话欺骗世人；五恶，花言巧语玩弄西方；六恶，恶口骂人六亲不认；七恶，挑拨离间发动战争；八恶，贪婪意图统治欧洲；九恶，嗔心仇恨人民民主；十恶，与整个世界为敌之愚痴。

习皇帝与邪恶之徒的普大帝苟谋侵略台湾，不也是十足的“十恶之徒”吗？我们假设，如果这两人从生下来就接受“诸恶莫作，众善奉行”的“十善”教化，一直学习到现在，他们俩人会“疯狂”到如此畜生不如的地步吗？不会的，绝对不会！反过来说，如果咱七哥一生下来就接受“人不为己，天诛地灭”的教化，接受“诸恶必作，众善绕行”的教化；七哥一出生就接受杀生、偷盗、邪淫为正道之事，试问：七哥今天还会成为全球第一反共义士吗？NO！恐怕，七哥早就是中共麾下一个杀人不眨眼，像王岐山、孟建柱之流恶贯满盈的匪徒了！

可见，教育是圣教，即圣贤（圣人和贤人）对人，使其从善的教化。人，为何需要“善”的教化？佛经里面讲得很明白：人，在无始劫的轮回当中，总是喜欢做恶的事，行恶的业，已经习惯成自然了，它控制不住。因此恶的“念头”（俗称的动机）总是要远远大于善的念头。因此，人，必须要从小就要施于“善”的教化；让他懂得因果、懂得感恩、懂得果报之恐怖，最终让他的恶念一生不要发生或少发生。这，就叫教育；这，才是我们所说的教育之真谛。而能把这个果报和道理，说的如此之透彻、说得如此之圆满的，唯有开悟之圣人。所以，我们说“教育”亦为圣教就是这个道理。

佛法曰：“十善”，为一切诸佛成佛之正因。什么意思？就是说，一切法门，八万四千法门，无论你修什么法门，想成佛，都必须要修“十善”。不仅如此，我们还知道，基督教、伊斯兰教、道教、儒教，一切正教，其本源都是教育。至于当今宗教变成“让人们崇拜的对象和精神的依托”，那是邪恶势力故意歪曲宗教是教育本源而达到它统治人民的目的。这个道理，前面我们讲过，中国就是这么干的。战友们，如果这个道理，我们不懂，或者误解了教育的真正含义，那么你所做的一切，一定不是善了。我记得一位修行者写过一本修行佛法的书。在书的一开篇，他就说到：“开悟之前，无论你修何法门，如何用功，都跳不出盲修瞎炼的范围”。他说的“悟”就是指彻底明白了佛教真谛者。在我们这里，如果套用这位修行者的话来说就是：在没有彻明教育之真谛以前，无论你如何教授什么课程，无论你如何用功备课，都跳不出盲教瞎育的范围。

这位修行者说的这句话，看起来简单，但是它却是对释迦牟尼佛49年讲经说法的最高归纳。这句话的核心是一个“悟”。“悟”就是七哥经常说的“开智”，表述不同，用词不同，意思完全一样。如果，谈教育，说教育，今天不是我来写此文章而是你来写此文章，那不就是等于说，你对于教育的真谛，是你“悟”到了，是你开智了嘛。

如果有人——其实就是你、我、他，从小就接受了这样的圣教，一生都懂得“诸恶莫作，众善奉行”的真理。他有了这个基础，再往上提升一步，那么不就是佛教讲的“出世间法”了吗？不就是《圣经》里讲的“信我者升天堂”了吗？教育，是指对人根基的教化。有了根基的教育，就好比一座大厦，基础打得深、打得牢、打得坚固，那么这座大厦就可以盖的很高、很高。战友们，别人不要看，就看看天天在我们眼前的七哥，他取得今天如此巨大无人难比的辉煌成就，不正是他从小就接受了“十善”的教育吗？他最初的启蒙老师，不就是他的父母，言传身教吗？不就是七哥从小这个“十善”的根基打得坚固吗？这个就叫教育，而不是学习数理化。知识和智慧是两极世界的问题，关于这一点，我们下面会谈到。（待续）
