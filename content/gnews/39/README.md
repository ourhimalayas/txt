## 新中国联邦的教育项目将由郭先生直接负责
` NFSC News` [轉載自GNews](https://gnews.org/zh-hans/2053787/)

作者：波士顿五月花农场-葛朗台太太
![This image has an empty alt attribute; its file name is Screen-Shot-2021-08-30-at-7.37.59-PM.png](https://assets.gnews.org/wp-content/uploads/2021/08/Screen-Shot-2021-08-30-at-7.37.59-PM.png)
2月20日，郭文贵先生在大直播中涉及了新中国联邦教育项目G-EDU的相关事宜。

![](https://assets.gnews.org/wp-content/uploads/2022/02/Slide3-9.jpeg)

郭先生表示，教育项目是新中国联邦除了灭共以外的第二件大事，而这一项目当下已经正式转由他本人直接负责。

据郭先生透露，所有参与新中国联邦教育项目的老师都可以获得一节课100美金的授课报酬。此外，老师们还可以自由选择实时支付、周付或者月付的薪资形式，且薪水的支付货币也将包括美元、人民币、HCoin或HDollar等各类币种。同时，所有参与课程的老师和学生都将得到相应数量躺平币的激励。

郭先生还补充道，目前，除了已经存在的线上课程外，线下课程也正在准备当中，而所有课程的开设基准将根据学生的需求决定。另外，在评估标准方面，老师的奖励等级将取决于学生对其的评分，而学生的学业水平将交由第三方评估机构评估。

至于资金方面，郭先生提到，教育项目1000万美元的启动资金眼下已经到位。在未来，新中国联邦还将成立人类历史上最大的教育基金，其中至少包含的一百亿美元的资金将为所有加入新中国联邦的学生提供免费的课程。

郭先生最后还分享了他对G-EDU教育项目的期待，即他希望新中国联邦的教育项目能够引领大家探索未来、破除无知与愚昧、靠近并且创造文明、重新建立信仰。
