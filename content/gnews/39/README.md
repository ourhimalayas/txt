据3月22日英国每日之星报道，一名向乌克兰军队交出坦克的俄罗斯士兵取得了公民身份的机会，和1万美元（7500英镑）。

官员证实，一名被确认为米沙（Misha 是Mikhail的缩写）俄罗斯士兵，在一次战役后，给乌克兰内政部打电话並進行了讨价还价后，自愿交出了一辆主战坦克。乌克兰内政部长丹尼斯·莫纳斯蒂尔斯基（Dennis Monastirski）的顾问证实了這條消息。

俄罗斯军队成员维克托·安德鲁西夫（Victor Andrussif）说，米沙的提议被转达到了乌克兰国防部，后来他被安排去了一個会面地点。据《沙巴日报》报道，他在抵达指定的地點时，被特种部队拘留。因为他向乌克兰军队提供了帮助，而获得1万美元。

安德鲁西夫在脸书 （Facebook）上的一篇帖子中写道：“原來，他（米沙）被其他的坦克兵給丟下了，那些人都逃回家了，剩下他孤身一人，米沙认为沒有必要再接着打下去了”。“他也不能回家，因为他的指挥官说他会被开枪打死，并将他纪录为战斗牺牲”。

他补充说：“米沙说，当时也没有什么可做的了，军队的指挥混乱，几乎没有人在场，士气非常低落。现在，这名士兵已经获得了舒适的拘留条件。战争结束后，他还将获得1万美元，并有机会申请公民身份”。

乌克兰中央国家军事公司首席执行官尤里·古塞夫（Yuri Gusev）表示，给与投降的俄罗斯战斗机驾驶员的奖励是，100万美元（75.4万英镑）
