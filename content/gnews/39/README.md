![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/29031907/WhatsApp_Image_2020-10-24_at_04.33.05-4.jpeg)



据多家香港传媒报道，笔名“卢峰”的前《苹果日报》英文版执行总编辑冯伟光，于当地时间6月27日晚，在香港国际机场准备前往英国时，被港警国安处拘捕带走。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/30000435/Untitled-design-2021-06-29T215923.230.png)图片来源：DW
香港警方随后证实，警务处国家安全处继6月17日、23日执行与壹传媒集团有关的拘捕行动后，6月27日晚上再在香港国际机场，以“串谋勾结外国或者境外势力危害国家安全罪”为名，拘捕一名57岁男子，该名男子涉嫌与上述案件有关，并正被扣留调查。行动仍然继续，香港警方不排除有更多人被捕。

毫无疑问，这是北京当局和港共政府对香港民主人士的迫害和清算。现今的香港，已经彻底失去新闻自由和言论自由，香港再也不是以前民主自由的香港了。赤色恐怖笼罩香港。

冯伟光自1997年在《苹果日报》撰写社论至今，这是继6月17日杨清奇被捕后，第二位被捕的撰写评论文章主笔，也是第七位被当局以违反国安法名义逮捕的《苹果日报》前员工。以言入罪，是专制政体的标配之一，显然，香港政府已经成为专制政体。

由此可见，北京当局和港共政府扩大对民主人士的迫害，已经到了无所顾忌的地步，希望西方民主国家能加大对北京当局和港共政府的制裁力度，以及能给予香港的民主人士更多的支持和帮助。只有消灭中共这个邪魔恶党，才能让正义得到伸张，无辜的人得到真正的民主和自由。
