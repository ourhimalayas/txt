- 你们不要论断人，就不被论断；你们不要定人的罪，就不被定罪；你们要饶恕人，就必蒙饶恕（“饶恕”原文作“释放”）。（路 6:37）
- 为什么看见你弟兄眼中有刺，却不想自己眼中有梁木呢？（太 7:3）


前言：战友们，上述经文与本文的主题是否有关系？那就在于我们如何看待“论断人”的教义在实际运用中的合理性——我说的这个“合理性”是合乎圣经的教导。如律法中的一条这样说道：“使瞎子走差路的，必受咒诅！百姓都要说：‘阿们！’”（申命记 27:18）那么，到底谁是瞎子？谁使瞎子走错路？今天我要谈论的就是这个领域的问题。这也和揭露CCP的假恶丑、偷盗、欺骗有着直接关系。当然，我们解决这个问题的前提就是要扫除“不要论断人”的教义带给信徒的错误判断以及其所造成的混乱。如此一来，我们就不会心存疑虑。

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/07/18213855/19-6.jpg)

爆料革命战友们是如何看待王腚缸、闫蛇妖二人的？大家众说纷纭，总体上当然是同仇敌忾。当然，也有反对的声音——他们说，你们不要论断人！

基督徒作为爆料革命中的一员，应该如何看待这个问题？其中一个令人困惑的问题是——“你们不要论断人。”这句经文会成为一个敬虔基督徒所遵循的道德伦理准则吗？对的，那一定是说阿们！这段《圣经》的教导在马太、路加福音里都有记载。路加福音第六章37节说：“你们不要论断人，就不被论断；你们不要定人的罪，就不被定罪；你们要饶恕人，就必蒙饶恕。”（“饶恕”原文作“释放”）

这里，我们首先要明白“论断”、“定罪”、“饶恕”分别是什么意思？事实上，这些都是审判与定罪的意思。约翰福音第五章29节说：“行善的复活得生，作恶的复活定罪。” 又有经文讲到，“如今那些在基督耶稣里的，就不定罪了。”（罗马书 8:1）显然，使徒约翰和保罗对这些问题的解释很明了，这些可以被理解为在末日审判的一种预表，或者也可解释为，当恶人在他作恶而不肯悔改时，他的罪已经定了。耶稣基督给了十二个门徒们赶逐污鬼并医治各样病症的权柄。（马太福音 10:1）不仅如此，“他叫有权柄的失位，叫卑贱的升高。”（路加福音 1:52）因此，那些作恶的不仅在末日审判时被复活后要继续被定罪，而且我们现在就有权柄驱赶他们、谴责他们。因为神曾允许路缺德社（王腚缸）升高，如今神也要使他“失位”。

上述问题，我们可以进一步分析：一句俗话说的好，上帝要灭掉魔鬼，必先使魔鬼自高、疯狂。箴言二八:26节说： “心中自是的，便是愚昧人；凭智慧行事的，必蒙拯救。”而保罗说，“在基督耶稣里的，就不定罪了。”这句话恰好可以作为使徒约翰说的“作恶的复活定罪”这句话的一种补充解释。因此，我们就可以得出一个结论：恶人一定是被定罪了，而且罪已经定了，行善的就不被定罪——真战友就不被其腐蚀拉拢，这也是毋庸置疑。另外，一旦对恶人“断定罪名，不立刻施刑，所以世人满心作恶。”（传道书 8:11）这也是因为“恶必害死恶人；恨恶义人的，必被定罪。”（诗篇 34:21）《圣经》中明确指出，亡腚肛、闫蛇妖等伪类们恨恶灭共的义人，“必被定罪”。如此一来，我们基本上解释了“论断人”的问题。

我们一旦从《圣经》中明白神已经给那些恶人定罪了以后，这时若不立刻施刑，才会导致人性普遍堕落的结果。实际上，有关罪人遭现世报的例子有很多，这就是为什么我们解释人性里恶的一面时，打击往往只看新约，不联系旧约，这其实是将整本《圣经》（三位一体的神）割裂开看问题。神是一个整体，而这正是中共撒但政权混淆视听的所在，也是中共利用宗教打击人性的惯用伎俩。如果有基督徒搞不明白“论断人”的道理，他就不是全面看待《圣经》的基督徒，至少是缺欠的。而这些问题被中共魔鬼政权利用后，又会催生出许多假基督徒。比如，当你向牧师指出某某弟兄或姊妹是异端时，牧师会责怪你说，“为什么看见你弟兄眼中有刺，却不想自己眼中有梁木呢？”（马太福音 7:3）尽管你掌握了大量充足证据证明你说的是事实，但那个牧师“眼中有梁木”，或着他故意视而不见，而你却因此被斥责了。

可以说，哪里有反抗暴政，哪里就聚集魔鬼的攻击，因牠惧怕你们高举基督的名。如果我们在真理上昏昏欲睡，中共派出的魔鬼首先就会攻击这样的人，而这就成为中共寻找攻击的突破口。为此，我们需要在基督的真道上站稳，至少保全自己免遭攻击。

《圣经》中充满着许多宝贵的教导，但这些教导却一次又一次地被撒但利用，屡试不爽。中共就是这样的魔鬼帮，因为魔鬼利用经上的话说：“你们怎样论断人，也必怎样被论断；你们用什么量器量给人，也必用什么量器量给你们。”（马太福音 7:2）牠似乎在说，我虽然很坏，但因为你论断我了，你就有罪。但这其中最大的一个事实是，牠掩盖了自己有罪，却指出你犯罪了。这就像是典型的法利赛人，他们只把淫妇揪出来问耶稣该如何处置，却没有把那个偷情的男人抓过来。如果你对《圣经》不熟悉，魔鬼会立刻激活你里面的自由意志，使你做出错误的判断。牠会扰乱你说，耶稣饶恕了那个淫妇啊，还说“你们中间谁是没有罪的，谁就可以先拿石头打她。”事实却是，那些混淆主道的伪类们一直都在断章取义地拿出有利于他们的这句话，却决口不提那些法利赛人从老到少一个个地都出去了。这是因为他们所做的见证虽然也是凭着三两个人的见证，但他们的见证是假的，其动机是要拿住耶稣基督的把柄。那些认为自己的选择绝对正确的伪类连伪君子的名号都不配，难道我们基督徒还看不懂主的教训吗？

当人们在处理复杂情感问题时，倘若人的自由意志，即心中的老我被激活了，这时候你也许会说：是啊：我论断人了，我有罪了；我这样衡量一个犯错误的人，人家同样也会这样衡量我呀。然而，如此恶性循环，你岂不越来越偏离神吗？于是，你的自由意志会选择原谅亡腚肛、闫蛇妖之流，说他们就算是淫妇，主都饶恕他们了，我们难道不能饶恕吗？问题是，《圣经》中那个淫妇是怎么做的？主问她，“没有人定你的罪吗？”她说：“主啊，没有。” 其实，当罪人开口称“主”的时候，她的心已经悔改了。而如今的情况却是，亡腚肛、阎蛇妖不但没认识到自己有罪，反而变本加厉地陷害战友、执意砸郭，他们哪里有机会让我们饶恕他们？因此，不要把教义中主耶稣有关爱与饶恕的教导所曲解。事实上，在末日审判中，死了的人都要“照他们所行的受审判”，希望所有爱主的人都是“无罪”宣判，得以进入天国。

总而言之，基督徒要从整本《圣经》的基础上全面地看待《圣经》的教导，不要让自由神学用“毫无原则的爱心”虏获了基督徒的心。我们要知道，主耶稣是怎样回击撒但的试探的？当主耶稣成为人的样式，在传道之前主动来到旷野接受撒但的试探时，魔鬼对他说：“你若是神的儿子，可以从这里跳下去。”耶稣回击撒但的话说：“经上又记着说：‘不可试探主你的神。’”（马太福音 4:7）其实，主耶稣就是创造宇宙万物的神，跳下去又算得了什么？但耶稣此时此刻是完全以人性的存在面对试探，魔鬼很懂这一点：如果耶稣为了证明自己跳下去，就等于拜在撒但权势下。因此，这就是耶稣基督给我们基督徒所做的表率：我们在面对亡腚肛之流混乱主道时，最好的回击就是运用《圣经》中的话。因此，我们不要落入他们的圈套。如果你感觉自己绝对不会被忽悠，其实你这是对自己人性的软弱面不了解。这时，如果伪类用激将法问你，把“你若是神的儿子”的话，变成你若是基督徒，你就不应该论断人；你若是基督徒，你就应该用爱心遮盖他，而不是相反。如果你听信了他们的话，神的灵就不在你里面，你就拜在亡腚肛、蛇妖闫的门下。

因此，当我们处在看不懂的骗局面前，在揭露他们之前，多看看《圣经》，看看神怎么说？我们可以用《圣经》这个量器来衡量他们。我们如此做，就能抵挡魔鬼的欺骗。
