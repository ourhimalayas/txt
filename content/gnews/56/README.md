据《阿拉伯新闻》2021年11月19日报道：
![](https://assets.gnews.org/wp-content/uploads/2021/11/image-565-e1637604439173.png)图片来源：Hector RETAMAL/AFP
利雅得：彭博社援引联合国贸易和发展会议（UNCTAD）的报道称，**2023年远洋运输昂贵的价格可能会使全球通胀率至少上升1.5%，其中依赖贸易的国家可能遭受的损失最大。**

这家总部位于日内瓦的机构表示，在过去十年中，运费上涨了四倍多。目前从中国运往美国的40英尺集装箱单位的价格现在比大流行前的平均水平高出348%。

贸易发展会议周四在年度海事报告中表示：“如果目前这种状况持续，集装箱运费的飙升将显著提高进口和零售价格。”

报告称，海运运费上涨的影响可能对依赖进口的较小经济体的消费者造成更大的打击，这些经济体的零售价格可能累计上涨 7.5%。

贸易发展会议预计，由于持续的需求、供应方面的不确定性以及对港口效率的持续担忧，集装箱货物的费率将保持在高位。

报告还提出了一些改善海上航运环境的政策建议，例如改善港口基础设施、聚焦规模经济、解决贸易失衡、改善贸易便利化和加强航运互联互通。联合国估计，海运部门的重大结构改进可以将运输成本降低约 4%，并减轻未来运输中断的影响。

**联合国表示，航运量的上升趋势将在未来四年逐渐放缓，并稳定在2.4%，略低于20年历史平均水平 2.9%。**
