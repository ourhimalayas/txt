据《路透社》记者：[Guy Faulconbridge](https://www.reuters.com/authors/guy-faulconbridge/)，2022年4月13日报道：
![](https://assets.gnews.org/wp-content/uploads/2022/04/image-1650.png)2022年4月5日，俄罗斯总统普京在莫斯科郊外的一处住所通过视频连线主持一场关于该国农业和渔业的会议。图片来源：Sputnik/Mikhail Klimentyev/Kremlin 通过路透社
伦敦，4月12日(路透社)——总统弗拉基米尔•普京(Vladimir Putin)周二表示，与乌克兰的和平谈判陷入了死胡同， 他在一个多星期以来首次就冲突发表公开评论，发誓他的军队将获胜，并刺激西方，说制裁“闪电战”已失败。

普京承诺，俄罗斯将在乌克兰实现所有的“崇高”目标。这是俄罗斯军队在基辅门口被阻止，而后从乌克兰北部撤退以来，普京首次公开发表讲话。

迄今为止，这是战争将持续更长时间的最强烈信号。普京表示，基辅通过对俄罗斯战争罪行的虚假指控以及要求覆盖整个乌克兰的安全保障，破坏了和平谈判。

自1999年以来一直担任俄罗斯最高领导人的普京，在访问莫斯科以东 3450 英里（5550 公里）的东方港航天发射场（Vostochny Cosmodrome）期间在新闻发布会上表示：“我们谈判再次回到了死胡同。”

当被俄罗斯航天局工作人员问及在乌克兰的行动是否会实现其目标时，普京说：“当然。我一点也不怀疑。”

普京表示，俄罗斯将“有节奏地、冷静地”继续行动，但最重要的战略结论是，美国在冷战后建立的单极国际秩序正在瓦解。

普京表示，俄罗斯别无选择，只能战斗，因为它必须保护乌克兰东部讲俄语的人，并防止这个前苏联邻国成为莫斯科敌人的反俄跳板。

西方谴责这场战争，认为它是针对一个主权国家的野蛮帝国式土地掠夺。乌克兰表示，在普京于2014年吞并克里米亚并于2月21日承认其两个叛乱地区为主权国家后，乌克兰正在为生存而战。

虽然制裁使俄罗斯陷入自1991年苏联解体以来最严重的衰退，但是普京认为西方的制裁是失败的。

普京说：“我们的敌人指望的闪电战没有奏效。美国已经准备好与俄罗斯战斗，哪怕只剩下最后一个乌克兰人——这就是事实。”

在战争初期，普京在俄罗斯电视上无处不在，但自从两周前俄罗斯从乌克兰北部撤军以来，他基本上已经从公众视野中消失了。

过去一周，他唯一一次公开露面是在一名民族主义议员的葬礼上，在葬礼上他没有直接谈到此次战争。周一，他在莫斯科郊外的一处乡间住宅会见了来访的奥地利总理，但会议的照片没有公布。
