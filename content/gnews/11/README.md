2021年10月7日，据《黑龙江日报》报道：近日，记者从省委编办获悉，黑龙江省已如期高质量完成了改革任务。一年来，全省共精简事业单位2735个，收回事业编制8.3万余名。其中，中省直精简事业单位902个、占比62.7%，收回编制1.3万余名、占比10%，精简内设机构1694个、占比19.3%，精简领导职数1159名、占比4.9%；市县精简事业单位1833个、占比8.8%，收回编制7万余名、占比8.5%，精简内设机构3193个、占比9.2%，精简领导职数3118名、占比6.9%。

2020年4月，中共中央印发《深化事业单位改革试点工作指导意见》，9个省份被选为试点。公开资料显示，除了黑龙江省外，内蒙古、江西、山东等省份被列入全域试点省份。据新华网报道，截至2021年3月31日，内蒙古自治区事业单位改革试点工作按照时限要求已基本完成。改革后，内蒙古自治区涉改事业单位整体精简比例约为35%。自治区本级精简占比44%，压缩挂牌机构80个，减少内设机构375个。盟市、旗县事业单位机构实行限额管理，盟市级事业单位减少35%左右，旗县级事业单位减少30%左右。

另外，作为中共中央确定的9个试点省份之一，江苏省启动深化事业单位改革试点，由省委编办组织、指导无锡市本级及江阴市、常州市本级及溧阳市、扬州市本级及江都区、徐州市本级及贾汪区这4市4县(市、区)开展试点工作。

看到这个消息的第一感觉就是中共真的缺钱了，开始对自己人动手，要知道事业编制是吃财政饭的，中共这么多年一直屹立不倒的原因之一就是，让他们自己人，也就是公务员、事业编制人员等享受超国民待遇，他们除了工资外，还有各种津贴、补贴，如住房补贴、防暑降温费、交通补贴、带薪休假，并且还有福利房、内部集资房等福利，这些房子的房价只有周边楼盘价格的60-70%，甚至更低。在房产已成为中国老百姓最主要资产的情况下，他们以如此低的价格购买房子，中间的差价就只能由老百姓的税收来弥补了。另外，公务员、事业编制人员的退休工资是企业人员的两三倍以上，共产党就是要让全国的普通百姓供养这一小部分人，使得这些所谓的“自己人”成为支持共产党的中坚力量。那到底这部分人有多少呢？

中（共）国财政供养人员主要由三部分组成：一是党政群机关工作人员；二是各类事业单位人员；第三是党政群机关和事业单位的离退休人员。穆正新在《北京之春》发表的文章中，依据由《中国统计年鉴》提供的数字，中共目前有44067个乡镇街道党委、2861个县级党委、333个地市级党委、31个省级党委和一个庞大无比的党中央。此外还有成千上万的院、校、系、所、军、师、旅、团党委等等，养党经费之巨可想而知。中共政协委员、陕西省政协副主席李冬玉在两会上提案，为避免人口流失严重，她建议可以对人口规模低于10万人的内地小县先行合并试点。她还举例说，某县2019年常住人口3.02万，地方财政收入3,661万元，一般公共预算支出8.65亿元，行政事业和社会组织120余个，财政供养人员6,000余人，财政供养人员比为1:5，通过合并这些内地小的行政区可以减少行政资源的浪费。

前几年网上有一篇文章说，中国财政供养人员与中国人口比为1：26，也就是中（共）国官方给出的中国人口为14亿，那财政供养人员就是5384万。随后网上就出来一些辟谣的文章，说中（共）国公务员只有700多万。这其实就是混淆概念，公务员只是中共财政供养人员的一部分，中共还有大量的事业编人员是财政供养的。医生护士是可以靠医院的收入来养活自己的，根据国家卫健委2016年的数字，中国卫生系统人员为1117万人，现在医生已经取消事业编制，全部改为聘任制，但由于医院属于盈利单位，这部分取消事业编并不受影响。高校教师也要取消事业编，在大学生收费就读的情况下，高校的老师也不太受影响。但事业编有很大一部分是中小学教师、卫生院、血站、防疫站、科研院所、各级质量技术监督局、县农机局、乡镇农机站、农经站、农技站、兽医站、林业站、水利站、各水库管理局/所、各级档案局、广播电台、电视台、公立图书馆、博物馆、纪念馆、美术馆、群众艺术馆等等，这些都属于事业编。事业编是属于中央给政策，工资由地方发放的一种制度。

2000年以前，我有一个同学，在辽宁一个非常边远的叫建昌的县镇做中学老师，被就开了40个月的白条，就是因为县里没钱。所谓的财政拨款，一般中央市直机关是由中央拨款，其余的全部由所在在县市拨款。在中国两千多个县、300多个地级市中，有钱有税收的有多少？撑死了也就五分之一。这些年这些公务员事业编小日子过得不错的原因是各地房地产大发展，卖地的收入较多。现在地卖不动，地方政府就失去了经济收入，即使不缩减也没钱发饷银。

中共做了很多恶，但有一点是中共很擅长的，它从古今中外各国的历史中学习和研究这些集权国家都是如何灭亡的，然后极力加以避免。所以历史上李自成、刘邦这种小吏被裁员之后揭竿而起的历史，中共是极力避免的，这也是为什么中共这么多年一直养着这些闲人的原因。但现在黑龙江开始的事业编的缩减，其他9个省市随后跟上，说明中共是真没钱了，养不了这些人了。中小学教师工资不到万不得已一定会保证，但那些县农机局、乡镇农机站、农经站、农技站、兽医站、林业站、水利站、各水库管理局/所、各级档案局、广播电台、电视台、公立图书馆、博物馆、纪念馆、美术馆、群众艺术馆肯定是能精简就精简，能不发工资就不发工资，这些人有一两千万之众，省下这部分钱可以发给军队、武警和警察这些武力维稳的人。

有了精简事业编的试点，接下来推进就会加快。现在先精简事业编，然后减少离退休人员的工资，最后是公务员，随着中共兜里的钱越来越少，中共所有的“自己人”都会被扔下船去，因为再不扔船就沉了。就像当年的国企几千万人大下岗，和公务员下岗分流一样，这个节点很快就将再次来临。
