## 上海官方深夜宣布“全域静态管理”进入实质上封城状态
` NFSC News` [轉載自GNews](https://gnews.org/zh-hans/2274881/)

作者：纽约香草山医疗部 – 云豹
![This image has an empty alt attribute; its file name is Screen-Shot-2021-08-30-at-7.37.59-PM.png](https://assets.gnews.org/wp-content/uploads/2021/08/Screen-Shot-2021-08-30-at-7.37.59-PM.png)
当地时间3月30日深夜，上海官方宣布，即刻起采取全域静态管理、全员核酸筛查、全面疫调排查、全民清洁消杀等综合防控措施，尽早实现社会面清零。

![](https://assets.gnews.org/wp-content/uploads/2022/04/19.jpg)

中共上海市委书记李强在30日晚上临时举行的全市领导干部大会上强调，在新的一轮全面核酸筛查期间，要严格落实“人不流动、足不出户”，确保“不漏一户、不落一人”。 此表示，上海市已进入实质性的完全封城状态。

随着疫情爆发失控，坚称不封城的上海当局开始逐步提升防控等级，从封锁部分小区到部分交通管制，最后到现在的全域静态管理。

分析人士认为，上海的疫情失控与中共的内部斗争交织在一起，造就了当前的局面。承受代价和痛苦的，永远是被控制的普通百姓。
