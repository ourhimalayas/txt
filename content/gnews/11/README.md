《自然新闻》以色列当局证实，辉瑞声称其生产的中共冠状病毒（Covid-19）“疫苗”是“95%有效”的说法是个赤裸裸的谎言。

以色列新闻机构*Ynet*报道说，辉瑞中共病毒疫苗的最新疗效数字已降至约64%，因为有报道称，各地接种疫苗的人正在检测新型“变种”的“阳性”率，据称这种“变种”目前正在传播中。

由于数字继续下滑，表明辉瑞疫苗是一种欺诈行为，当局建议人们接受检测，即使他们已经注射过疫苗，以确保他们对抗中共病毒时是“安全”的。

据报道，辉瑞公司发言人拒绝提及以色列的新数据，而是选择引用公司数据，声称注射疫苗仍然”安全有效的”，而完全不管最新科学显示了什么。

同一家企业喉舌说，辉瑞疫苗接种对所有已知的变种都是有效的，尽管”强度下降”（不管这话意味着什么……）。

大约930万以色列人，约占以色列人口的60%，现在至少接受了一次”剂量”的辉瑞注射。因此，政府当局开始放宽一些限制，包括强制戴面罩面具和社会交往准则。

尽管注射率很高，但中共病毒在以色列的”阳性”病例继续上升。7月4日，每日病例数达到344例，而重病病例达到35例，高于6月19日的21例。这表明注射导致更多的疾病。

**至少一半已经接种了疫苗的以色列人测试”Delta变种”呈阳性**

早在6月份，以色列政府专家顾问团的一位官员就承认，这些疫苗接种并没有声称的那样有效。在那些对所谓的”Delta变种”检测呈阳性的人中，至少有一半已经注射了中共病毒疫苗。

当然，这意味着注射疫苗不能像声称的那样起作用，实际上正在引起更多的疾病。英国公共卫生部发现，注射疫苗的人们”感染”Delta变种的可能性至少是没有注射疫苗者的八倍。

以色列官员罗恩·巴利瑟（Ron Balicer）说，大约90%的中共冠状病毒（Covid-19）新发感染病例与Delta变种有关，该变种于去年年底在印度首次发现。

主流媒体说，与早期的中共病毒毒株相比，Delta变种具有高度传染性，而且正在迅速传播。然而，现在说它是否比对中共病毒之前的其他变种更致命还为时过早。

一旦公开宣布，媒体准备并有意无穷无尽地报道变种病毒的踪迹来散播恐怖。已经有谈论一个新的”Lambda”变种，随之可能会跟着一个Epsilon变种，一个Zeta变种等直到永远。

《大纪元时报》的一位评论员写道：”我比相信我们自己还信任以色列政府（这还不算多）。当然，疫苗的有效性将被报道为下降，以推动’疫苗加强剂’的注射。

另一些人指出，如果人们仍然需要接受”测试”，才能知道他们是否有感染了一些可怕的新变种，然后说变种可能不是一个风险，甚至可能不存在。

另一位评论者总结大多数人们的感受写道：”在这一点上，我认为卫生专家和政府所说的关于这种病毒的说法，疫苗的有效性还不到零！
