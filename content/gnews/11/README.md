**撰稿：蓝莲花农场 文斌**

1、俄罗斯无人机在基辅的波迪尔上空被击落；据说基辅西北方向的俄军70公里的长龙有分散开的迹象，并且有些已经排成射击队形。

2、切尔尼戈夫：俄军轰炸了这座城市的标志性建筑之一的乌克兰酒店，“这座古城的历史可以追溯到 1,300 多年前”。

3、苏梅地区的国土防卫部缴获了几件俄罗斯设备，包括自行火炮。俄军昨天从苏梅州的Trostyanets（Тростянець）猛烈开火，发射阵地的位置坐标是50.481108, 34.965646。

4、在哈尔科夫乌军抓住了俄军小股破坏分队，他们的任务是炸毁重要的基础设施，缴获了其武器和炸药。该市Sharikova街遭到炮击，附近一辆救护车的医生因此受伤。

5、哈尔科夫与顿涅茨之间：从早上开始，俄罗斯军队用Tochka-U弹道导弹和多管火箭炮轰炸Avdiyivka和塞维罗涅茨克市，此外，奥谢尔耶内也遭到炮击；在Avdiivka郊区有战斗发生；俄罗斯军队不断用多管火箭炮轰击塞维罗涅茨克市；在Lysichansk市，一玻璃厂遭到炮击。

6、乌克兰中部：俄罗斯导弹袭击克罗皮耶夫尼茨基附近的机场；而第聂伯市在第一次遭受空袭以后继续不停地遭受俄军袭击，乌军防空部队在该市上空拦截了两枚导弹。

7、梅利托波尔：该市居民在被占领的政府楼前举行集会，要求释放市长伊万-费多罗夫。俄军抓捕了伊万市长以后又抓捕了乌克兰活动家Olha Haysumova。

8、俄罗斯开始在顿巴斯和克里米亚调集部队。据乌克兰军事情报负责人Kyrylo Budanov称，俄罗斯也在从俄罗斯中部和东部地区调来新的部队，以补充兵源。乌克兰总参谋部的消息说，俄罗斯正利用图-95MS战略轰炸机向乌克兰发射巡航导弹。一架 Ka-52 直升机在俄罗斯境内的罗斯托夫地区坠毁，机组人员全部遇难。

9、白俄罗斯说将向乌克兰边境派遣五个营战术小组，以取代目前驻扎在那里的部队，但是否认这是为白俄罗斯的参战做的准备。

10、乌克兰总统办公室主任顾问 Volodymyr Zelenskyy：以色列总理Bennett与其他参与调停的中间国家一样，不会让乌克兰同意俄罗斯的任何要求。由于军事和政治原因，这是不可能的。相反，以色列敦促俄罗斯更充分地评估这些事件。
