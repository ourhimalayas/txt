皮质醇是压力荷尔蒙。高皮质醇会耗尽维生素D，维生素D不足会提高皮质醇。如果皮质醇过高，会导致许多不同的健康问题。高皮质醇可能会导致问题有：

• 抗压性低 (不耐压力) / 焦虑 / 睡眠障碍!

• 发炎 / 高血压

• 过敏 / 气喘

• 血糖问题 / 钾减少 / 小腹肥胖

• 性欲减退

高皮质醇的第一征兆是：

每晚凌晨 2:00 – 2:30 左右会醒来，脑中很多想法一直转，醒着直到早上 8:00 左右。这是因为皮质醇依照生理时钟，通常会在凌晨 2:00 左右最低，而在上午 8:00 左右最高。但是当你有高皮质醇时，一切都颠倒过来了。

高皮质醇症状累计之后会：

1. 疲劳

2. 无法入睡

3. 情绪低落

4. 生活缺乏活力、意志消沉

高皮质醇的自然疗法：

1. 补充维生素 D3

2. 补充钾和镁(喝电解质)

3. 补充维生素 B1（营养酵母）

4. 进行体力劳动直到感觉很累

5. 缓解压力/在大自然中散步/聆听音乐

维生素D在睡眠中的主要功能之一，是在深度睡眠时，让肌肉停顿，或在睡眠时关闭肌肉，从而提高睡眠质量，降低压力荷尔蒙高皮质醇。
