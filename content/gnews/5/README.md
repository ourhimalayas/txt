最近半岛新闻报道，从俄罗斯对乌克兰侵略战争以来，美国的俄罗斯餐厅受到冲击，有些电话 “非常令人不安”；紧接着是“糟糕的在线评论、取消预订和威胁性的说明”。尽管加利福尼亚州圣地亚哥普希金俄罗斯餐厅的老板称他的大部分员工都是乌克兰人；同时，在俄罗斯发动毁灭性的军事攻势时，这家餐厅“悬挂了乌克兰国旗并捐款支持乌克兰”。可是，这“并没有阻止一些公众在俄罗斯总统弗拉基米尔普京全面入侵该国后的几周内瞄准他的餐厅”。

该报道引用相关人士的观点：由于俄罗斯的侵略战争，“任何带有‘俄罗斯’这个词的东西都会成为每个人的危险信号”。文章指出，西方国家一再将乌克兰冲突“归咎于普京”，甚至在冲突开始之前就称其为“俄罗斯总统的‘选择之战’”。可是俄罗斯军队“对乌克兰城镇的持续袭击，摧毁了房屋，损坏了医院，迫使数百万人逃离”；在世界上“引发了公众对整个俄罗斯的愤怒”。

文章指出，虽然一些俄罗斯公民“确实支持战争和普京本人”；但“全国各地仍有数千人走上街头谴责入侵，还有数千人担心被迫征兵而逃离俄罗斯”。尽管俄罗斯和国外的许多人“反对这场战争”，可是通过俄罗斯社区的反映，他们经历了“明显的寒意”，遭受“自入侵开始以来引起了公众的强烈反对”。

综上所述，笔者认为，这些在海外的俄罗斯文化背景的企业所受到的冲击，是非常令人遗憾的。这些因为自己与俄罗斯相关的文化背景和血脉关系而无辜躺枪，并深受其害的人们，真的让我们深感同情，也深为关注。

这个动态，一方面体现着，世界正义一致性的表现：力挺乌克兰，谴责发起侵略战争的俄罗斯邪恶者普京。另一方面，也表现出，民众在这个战争的本质归向时，有时会很难明确定位到战争狂人普京。相反，民众看不到俄罗斯民众被洗脑，被绑架，也会很简单、轻易将俄罗斯侵略战争与俄罗斯，俄罗斯侵略战争和俄罗斯人民联系起来。因此，一切与俄罗斯有关联的个人、企业和机构，都是会被带着强烈正义感的世界人民置于一种抛弃、打压和反对的状态。

笔者认为，这些正义人民的行动和感受，是可以理解的；可是也极容易让那些有良知的俄罗斯本国民众和俄罗斯文化背景关联的人们受到伤害。同时，我们也可以充分理解到，为什么那些独裁者和专权魔头要保持国家权力，垄断宣传工具，就是要让本国人民为自己的邪恶目的送死；甚至，就是让一切个人的疯狂和邪恶，导致整个国家，全体人民和一切与之有关联的人士，都要为他们的邪恶疯狂举动买单。

我们从俄罗斯战争狂人普京发动侵略战争之后，连普京自己家乡的人民都出来上街游行，示威反战，可以清楚看到，普京的这些荒唐邪恶举动是不得俄罗斯民心的。可是普京仍然还在用自己保持的喉舌为之唱赞歌，给俄罗斯民众洗脑。这个报道中，我们可以清楚看到，独裁者洗脑，绑架民众和国家的邪恶所在。

同时，我们也从这里看到，只有消灭邪恶的少数人，就可以解救更多的民众。最终地，只有传播真相，分享事实，才能让世界认识到，普京不能等同于有良知的俄罗斯民众，并形成将攫取国家权力和绑架人民的独裁者与国家、人民分开的清醒意识。俄罗斯民众，也需要在民间更多地分享普京及其邪恶集团的各种真相，才能真正地解决被普京发起的邪恶侵略战争所绑架的现实。拯救来自真相，拯救来自人民自己。
