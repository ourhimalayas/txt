**摘要翻译：**

反数字仇恨中心 (CCDH) 是英国极左派的非营利组织，目前正在向脸书施压，迫使社交媒体巨头脸书进一步打击保守派媒体。该组织的背后是一个神秘的瑞士投资集团，而该投资集团正是共产主义中国争夺全球主导地位的资助者。

正如《布莱特巴特新闻网》昨天报道的那样，反数字仇恨中心组织最近发表了一篇报道，将《莱特巴特新闻网》和其他保守派媒体的出版物污蔑为“气候（变化的）否认者”，试图将它们列入脸书的黑名单。

这个极左组织将布莱巴特新闻电台的一位批评绿色新政的受访者的表述，等同于“否认气候（变化）”，甚至脸书也对该报道的表达方式表示怀疑。

反数字仇恨中心组织是由橡树基金会资助的，橡树基金会是一家位于瑞士日内瓦的极其富有的隐秘投资实体。

该基金会一直资助各种各样的左翼事业，包括反对执行美国移民法所做的努力；为倾向民主党和广泛传播左翼环保的群体，进行选民登记和动员工作。

并且，它还一直在资助共产主义中国争夺全球贸易网络的主导地位。

2018 年，橡树基金会曾经向一个名为“气候工程基金会”的组织捐赠了100万美元，以支持绿色“一带一路”倡议。

“一带一路”倡议是中国共产党最雄心勃勃的项目之一，这是一项庞大的全球基础设施项目，旨在通过一个启动的贸易网络​​连接欧洲、俄罗斯、非洲、中东、亚洲、印度次大陆和南太平洋，并形成一个以中共国为起点和终点的贸易网络。

该项目包括港口、铁路、机场、隧道、水坝、发电站和全新的城市，所有这些都旨在通过铁路、海运、公路和航空线路的组合，将世界贸易路线集中在中共国。

美国盟友澳大利亚最近取消了其维多利亚州与“一带一路”倡议达成的交易，引发了中共国政府的强烈反对。

美国国务院批评“一带一路”倡议，指责中共国对发展中国家使用 “债务陷阱外交”，因为斯里兰卡在拖欠中共国的债务后失去了对一个主要港口的控制。
