#### **总统已下令开枪杀人的街头抗议活动的起源**

**编译：七龙珠**
![](https://assets.gnews.org/wp-content/uploads/2022/01/tempsnip10.png)在阿拉木图市抗议期间部署安全部队.EP
哈萨克斯坦作为一个独立国家的历史与努尔苏丹-纳扎尔巴耶夫有着密不可分的联系。哈萨克斯坦是纳扎尔巴耶夫的。自脱离苏联独立以来，直到2019年，总统职位一直由他担任，他代表了主导这些前苏联国家很大一部分的寡头，这个国家是一个火药桶。哈萨克斯坦起义只是压垮骆驼的一根稻草，尽管抗议的原由是基于燃料价格的上涨，但其背景要深得多。为什么会变成这样？

燃油上涨的抗议……但不只是燃油问题。

从历史来看：哈萨克斯坦是最后一个宣布独立的苏维埃共和国。自1991年以来，权力掌握在纳扎尔巴耶夫手中，纳扎尔巴耶夫在三年前辞职，情况与现在类似，虽然没有那么严酷。他把指挥棒交给了他最亲密的盟友之一卡西姆-约马尔特-托卡耶夫，在目前的情况下，他已经下令对示威者“格杀勿论”。这表明该国的政治结构是由寡头统治的，使哈萨克斯坦成为一个专制国家。它有举行选举，是的，但执政党总是有接近百分之百的支持率。

2022年初，该国金融中心阿拉木图市的燃料价格上涨，点燃了这一火花。哈萨克斯坦拥有巨大的石油和天然气储备，占整个中亚地区国内生产总值的60%左右，这种在资源方面的“好处”没有转化为民众的收益，这就是民众对经济的抱怨根源。

然而，这些抗议活动显然具有社会性质，因为示威者要求进行深刻的政治变革，提高公民参与度，不迫害持不同政见者，并限制总统的权力（恢复1993年宪法）。政府却将这场起义定义成一场对“麻烦制造者”的斗争。

**纳扎尔巴耶夫的遗产**

努尔苏丹-纳扎尔巴耶夫在国外的形象显然是他的政府自以为是的形象。目前的抗议活动在哈萨克社会并不常见，但有两个重要的先例：2011年抗议和2019年抗议，但是这两者都比现在要胆小得多，2019年那次则以总统辞职而告终，但该政府体系仍在实施中。继任总统托卡耶夫的当选在当时和现在都被理解为前任总统纳扎尔巴耶夫2.0版，因此，前任总统仍在暗处继续积累着大量的权力。

示威活动已经达到了几乎前所未有的程度，数字也证实了这一点。到目前为止，已有26人被杀，多达3000人被捕，因为政府决定用武力镇压他们，而且不需要任何预警。此外，它也不会听取公民的要求，托卡耶夫已经给示威者贴上了“叛乱分子”的标签，甚至把正在发生的事情归咎于外国势力。

阿拉木图、阿克托比和阿拉木图地区的局势已经稳定下来。实行紧急状态的做法正在产生效果。托卡耶夫解释说：“宪法的合法性正在全国范围内得到恢复。”他希望继续保持纳扎尔巴耶夫的铁腕政策。哈萨克斯坦是一个相对稳定的国家，但正是因为这种一党一派的领导体制，从对抗议活动的反应中可以看出，其前任和仍在执政的总统的权力仍然存在。
