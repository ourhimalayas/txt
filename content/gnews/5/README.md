## 为什么俄罗斯的入侵会导致全球粮食短缺 -1

弗拉基米尔-普京对乌克兰的入侵战争所产生的连锁反应正向全球扩散。

在美国和英国等发达国家，由于石油和天然气价格上涨，消费者不得不支付更高的费用来取暖和驾车。

然而，许多其他发展中国家则面临着一个更根本的危机——粮食短缺。

众所周知，在战争之前，俄罗斯和乌克兰占了全球小麦出口的30%。

在俄罗斯攻击乌克兰后，小麦的价格立即飙升。

一蒲式耳小麦的价格从入侵时的8.84美元飙升至3月8日的13.64美元左右，其生产的面粉足以烘烤70个1磅的白面包或90个1磅的全麦面包。

这对大型小麦进口商来说已经够糟了。

不过，让事情变得更糟的是，乌克兰正努力将其小麦运出国境。

乌克兰在2021-22还有大约2000万吨小麦和玉米需要出口，但由于俄罗斯封锁了黑海和亚速海的港口，导致100艘外国船只搁浅，因此无法出口。

乌克兰谷物协会主席米科拉-戈尔巴乔夫（Mykola Gorbachev）说，我们正面临60亿美元的潜在损失上。

乌克兰去年的粮食出口总额为270亿美元。

对那些购买乌克兰小麦的国家的影响是可怕的。这些国家包括印度尼西亚、埃及（世界上最大的小麦进口国）、巴基斯坦、突尼斯和摩洛哥等国。这些国家不得不寻求更贵的替代品。
