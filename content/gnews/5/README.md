**‘脸书’诬陷及封锁提供正确医疗讯息者**

现已被解僱的Ventavia新药研究集团区域主管布鲁克．杰克逊Brook Jackson向《英国医学杂志》提供了数十份公司内部文件、照片、录音和电子邮件，并透露 去年在德克萨斯州多处进行的中共病毒疫苗试验出现重大问题，包括伪造数据、违反基本规则以及拖延对不良反应的报告。
