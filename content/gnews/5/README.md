## ASML 要求客户回避涉嫌侵犯专利的中共国竞争对手

![](https://assets.gnews.org/wp-content/uploads/2022/02/图片1-73.png)（图片来源：ASML）
据《汤姆的硬件网》马克·泰森（Mark Tyson） 2022月9日报道

中共国的东方晶源公司与XTAL 公司有关联，ASML 此前曾赢得XTAL 8.45 亿美元的知识产权侵权赔偿。

在今天早些时候发布的年度报告中，荷兰光刻系统制造商ASML 告知投资者，一家中共国竞争对手可能正在使用侵犯其知识产权的技术。中共国显然对 ASML 先进的半导体制造机器的出口限制感到沮丧，而且很明显，受到怀疑的中共国竞争对手拥有中共政府批准的印章，是一家被称为“小巨人”初创公司。彭博社表示，ASML公司已向中共国官员表达了担忧，但尚未采取法律行动。

ASML 密切关注东方晶源电子有限公司，该公司与 XTAL Inc. 有联系。AMSL 在 2019 年成功赢得了与 XTAL 8.45 亿美元的商业机密诉讼。东方晶源是一家被中共政府打上 “小巨人” 标签的中共国科技创业公司。据彭博社报道，这个标签是“政府认可的重要衡量标准，它向投资者和员工发出信号，表明这些公司不受监管处罚”。

在提起法律诉讼之前，ASML 正在调查和收集东方晶源的任何知识产权侵权行为的证据。然而，情况看起来并不乐观，因为向 XTAL 和东方晶源发送的有关事项的询问和信件仍未得到答复。因此，ASML 已将其担忧转达给了中共国官员。如果公司间的沟通不起作用，并且与政府监管机构的联系被置若罔闻，那么法律诉讼可能是 ASML 的下一步。

很容易理解，中共对目前ASML 阻止向中共国客户提供其关键半导体制造机器的贸易不满。中共国热衷于追赶外国半导体巨头的先进技术，所以对东方晶源这样的公司相当扶植。

在上述 ASML、美国和中国之间的不安推拉发生的同时，ASML 已告知主要客户不要以任何方式帮助东方晶源。ASML由于其在半导体产业生态系统中的重要性，而具有相当大的权重，因此这种软实力可能比打官司更有效。

很多读者听到ASML业务创新高，一点不会感到惊讶。英特尔和台积电等半导体巨头依靠该公司最新的 EUV 机器生产最先进的工艺芯片。希望今年不会再发生火灾。
