由于当地政府部门极端的封闭式管理，常驻人口50万人的瑞丽市，在短短不到一年的时间里，人口极速下降到仅剩10万人。民不聊生的瑞丽人民纷纷逃离了瑞丽。网民戏称：“要不是老弱妇孺腿脚不便，可能这10万人也不会留在这个地方。”

瑞丽市地理位置特殊，市内边境贸易区接壤缅甸。瑞丽病毒疫情始于3月29日，截至10月底，累计发现境内外感染者114例，瑞丽地方政府以边境疫情严重为由，已对瑞丽全市实施完全封闭式管理长达七个月之久。

在封城以后，街边商户几乎每天都是被关停状态，各处关卡林立，民众不能轻易进出，每天也只能领到一斤米，最基本的生活物资保障均不能得到满足。民众表示他们也不能外出打工，离开瑞丽需要向当地疫情指挥部门申请自费集中隔离14天，而百分之七八十的民众在封城近一年当中没有任何收入，对于高昂的酒店隔离费用也都负担不起，高压的管理方式让大家几乎都处在心理崩溃边缘。

从7月份开始，被封城隔离到绝望的瑞丽人民就不断的在地方媒体热线上求助和在社交媒体平台上发声，呼吁当地政府倾听民众的意愿，改善封城后人们受到的种种不人道的待遇。不仅毫无结果，瑞丽民众上述的自救行为，不是被当地政府粗暴拒绝，就是在网络上被删帖禁言。当地政府部门对民众、商户的补助措施却一直停留在口头上，从没有任何实际的行动。
当地政府不仅对外封锁消息，对当地市民的管制方式更是残忍到令人发指。民众爆料：“瑞丽市民每三天做一次病毒核酸检测，几个月下来，人均核酸检测至少五十次。一个小区如果发现一个感染病例，附近几个小区的居民都将会被强制拉去隔离，稍有不配合，一群人马上就砸门上手铐强制带离。”

就在瑞丽真实情况越来越捂不住的时候，中共瑞丽市长发表声明说瑞丽不需要援助，多少还在饥寒交迫中的瑞丽人民听到这句话不知会作何感想。肉食者鄙，未能远谋。中共官僚集团失尽了人心，通过各种途径逃出瑞丽的几十万民众可能都在庆幸摆脱了恶魔的控制，而留下来的那些民众的生存环境也将会在瑞丽实情曝光于众后得到些许改善。至于中共官僚集团对这座城市人心残酷的践踏，得到的必将是四十万颗对立的民心，它们离被反噬的日子也将不远了…
