日本政府打算加快建立一个新的机制，以保护国家间冲突中的逃难者有资格获得“必要保护”，并使他们能够得到与难民相同的待遇。

![](https://assets.gnews.org/wp-content/uploads/2022/04/1-106.jpg)[图片来源](https://www.bbc.com/news/60735121#:~:text=How%20can%20I%20offer%20a%20UK%20home%20to%20Ukrainian%20refugees%3F)

鉴于俄罗斯对乌克兰的入侵，日本政府打算加快建立一个新的机制，以保护冲突中的逃难者和其他不属于《难民条约》中 “难民”这一狭义定义的人，作为 “准难民”。

对此，在去年被废除的移民难民修正案预计在夏季参议院选举后的临时国会上会被重新提交。

难民条约中将难民定义为因“种族、宗教、国籍、特定社会团体，政治观点”而受到“迫害”的外国人。鉴于此，日本政府对逃离国家间冲突的人不被视为 “难民”。对此，修正案的重点之一是建立一个机制，使冲突中的逃难者有资格获得“必要保护”，并使他们能够得到与难民相同的待遇。欧洲国家也有类似的制度，如果得到认可，可以得到落户资格等良好的保障。

另一方面，为了避免非法居留的外国人被长期拘留，修正案对难民认定程序中暂停驱逐出境的规定也作了例外规定，允许在第三次申请后驱逐出境。但是，日本立宪民主党及部分党派则坚持要求删除这一内容。他们还呼吁对一名斯里兰卡妇女在移民拘留所的死亡进行调查，这导致了该修正案的废除。

截止今年1月，政府和执政党决定将该修正案推迟到夏季的参议院选举之后，因为这将使执政党和反对党相互对立。然而，鉴于乌克兰的局势，政府内部要求必须以某种方式通过该修正案的呼声迅速上升。在4月5日的新闻发布会上，内阁官房长官松野博一表示，“他正在考虑重新提交，并且强调，将努力建立必要的制度，适当保护那些真正需要庇护的人”。
