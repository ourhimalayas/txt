## 奢侈品巨头路易·威登集团和爱马仕将暂时关闭在俄罗斯的商店

![](https://assets.gnews.org/wp-content/uploads/2022/03/图片1-26.jpg)莫斯科红场上的爱马仕。该公司表示，它将从周五晚上开始暂时关闭其在俄罗斯的商店（谢尔盖-卡尔普欣/路透社Sergei Karpukhin/Reuters）
据《纽约时报》记者萨普纳-马赫什瓦里（Sapna Maheshwari）2022年3月4日报道：

法国奢侈品巨头酩悦·轩尼诗·路易·威登（LVMH）集团和爱马仕(Hermès）周五分别表示，由于乌克兰危机继续升级，他们将暂时关闭在俄罗斯的商店。

在俄罗斯有120多家门店的酩悦·轩尼诗·路易·威登集团一位发言人表示，“鉴于该地区目前的情况，集团将于周日关闭这些门店”。该发言人补充说，“该公司将继续向其在该国的3500名员工支付工资和福利”。

爱马仕公司在LinkedIn的帖子中说，从周五晚上开始，它将暂时关闭在俄罗斯的商店并暂停商业活动。

爱马仕在帖子中还说：”对此时欧洲的局势深表关切，我们很遗憾地决定暂时关闭我们在俄罗斯的商店，并暂停我们所有的商业活动。

近日，宜家、H&M集团和耐克等西方零售商已经宣布，停止在俄罗斯的销售并暂时关闭商店，并表示支持乌克兰人民。大多数人表示，他们将继续支付其俄罗斯员工的工资。除了乌克兰的暴力升级之外，零售商还对俄罗斯更广泛的贸易中断和针对该国金融交易的制裁做出了回应。

总部设在斯德哥尔摩的宜家公司本周表示，它将暂停与俄罗斯和白俄罗斯的进出口业务，停止零售业务，并暂停其生产木制家具的部门的业务。该公司周五在一封电子邮件中告诉《纽约时报》：”由于目前的国际制裁，以及制裁的意图，我们没有进行交易，包括进出俄罗斯或白俄罗斯的付款。”

美国服装和鞋类协会总裁兼首席执行官史蒂夫-拉马尔在一份声明中说：“随着各国协调和部署一系列有针对性的金融和贸易制裁，影响俄罗斯的制裁机制正在迅速演变。影响零售业及其工人的是对金融机构和资金流动的限制，如工资支付。”
