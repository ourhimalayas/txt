### ｜简述

长岛伟哥：

1. 美东时间每周日上午9-11点是周例会，农场负责人和法治基金、法治社会董事列席，总结上周工作和下周的工作计划，讨论各农场遇到的问题，有必要的，会以联盟公告形式通知下去。
2. 由于联盟组织的活动和下发的信息，在传递过程中出现了问题，不能及时或准确传达给战友们，所以我们启动了每周直播，每次1到1个半小时左右，把重点内容和大家分享，战友们也可以把关心的问题提出来，我们进行回答。


老班长：

1. 农场联盟委员会是由全球农场负责人组成的委员会，长岛伟哥是秘书长，我是主席。委员会的根本任务和方向是，在爆料革命大局下，团结战友凝聚战友服务战友，展开一致性的行动，开展游行、文宣等各种活动，同时服务G系列，在全球范围展开G系列的布局和运作。
2. 每周日的这个直播主要是解决和战友间的信息不对称的问题，增加联盟委员会的透明度，增加和战友们的互动，回答大家的问题。
3. 过去的几次会议中，我们分别完成了农场间的合作协议，农场的议事规章，定稿了农场联盟的总章程。即将要完成的是对农场的审定，也就是哪些农场符合委员会章程标准成为了合格的农场，哪些尚未达标，是准农场。
4. 联盟委员会鼓励成立新农场，并帮助完善准农场。依照章程，管理和审核我们现有的农场。
5. 通过评级之后，被认定是农场的，我们也依然会在每1年或半年进行一次重新的审核，将来还会做评级。
6. 所有的过程我们争取做到透明化，并把信息及时地反馈给战友们。同时对正在进行的，比如说G-Club、借款项目等大家关心的问题，在这个平台上和大家进行交流。


长岛伟哥：

1. 我来给大家介绍下**G-Club**的操作流程。G-Club会员卡现在购买时，要先在官网登记信息，付款方式可以做选择。比如选汇款，就会获得一个订单号，汇款账号会通过邮件发给你，在银行汇款时，你需要把订单号备注清楚，后续方便G-Club对账。
2. 完成付款后，你需要填写两个链接，一个是G-Club公司登记付款信息和个人信息的，第二个是每个农场的信息登记，包括名字、订单号和6字头临时卡号。
3. 1字头卡号是G-Club公司给的正式卡号。农场的表格是要交给G-Club公司做更新1字头卡号用的，农场定期分批交上去做更新。
4. 之后会做实体卡。不方便接收实体卡的战友，可以写农场的地址。
5. 1字头卡号，比如香草山农场的卡号是1000001003等等，这个1代表美国，是区号，3是各农场的区分码。
6. 目前每个农场拿到的卡号是足够的，排名的话肯定都在1万以内的。
7. 汇款难的是一直存在的一个问题，农场能帮忙的帮忙，但最终还是要靠战友。可考虑换家银行电汇，或者寄支票买卡。
8. 现在发现汇款后，对账确实有问题，尤其在官网正式开通前付款的，对账时间会更长一些。


老班长：

1. 还有一个问题是，很多战友的款项汇不出来，但我们不会抛弃战友的，会让所有战友都享受到相应待遇。
2. 农场第一是保护战友，凝聚战友，服务战友。换了农场，应享的利益也不会少。
3. 今天有战友说被农场踢出来了。农场都有自己的所谓家规，但是战友可以加入其他农场，也可以反映问题解决后再回原农场。
4. 如果是带节奏的违规的，不仅会被踢出农场，还会报到联盟委员会，查明属实的，所有农场会全部拒绝他加入。
5. 联盟委员会有投诉邮箱和日常交流邮箱。
6. 我和长岛伟哥接到新任务，之前郭啸天负责的，负责大家游行等活动的报销。之前法治基金/法治社会对有些活动诸如游行、灭贼行动等规定了报销标准。那么在这次我们交接过程中，发现之前的活动报销并没有达到规定的标准。还有在灭贼行动中产生了新的经费和其他事情。
7. 在德州行动中，由于当时战线拉得比较长，后勤不是很足，所以出现给大家提供衣服等这些情况，延伸到后续，战友提出了一些新要求。但我们不是决策机构，这些需要提交法治基金/法治社会做处理的，我们得到回馈后，会再告诉大家申请是否符合标准、是否被批准。
8. 之后的这些活动申请，我们争取有一个公开透明的表格给大家填写。


长岛伟哥：

1. 在过去几次会议中，我们现在在规范农场的命名。基本上以国家开头，比如：美国纽约香草山农场、新西兰奥克兰伊甸农场，就是为了把农场命名规范化标准化，还有之后的一些规定都要规范化。
2. 其中包括，农场不能向战友集资，各个农场成立公司，怎么经营，这些我们还都没有定下来，在此基础上，向战友集资的话，可能会产生很多纠纷，所以联盟委员会叫停了这些项目，等到委员会出台规则，在委员会指导下，农场才可以开始操作。
3. 以意大利农场为代表的，内部发生了风波，联盟委员会进驻后，帮助组建农场，产生新农场主之后，进驻小组撤出。
4. 还有现在其他的一些农场，可能也会遇到这种问题，造成内部出现分裂，联盟委员会就会进驻进行协调解决，让农场更好服务战友。


老班长：

1. 农场和战友之间的关系，只能是合作关系。你做的不够好，最多是和其他农场不合作，但绝不能对抗。不合作是过渡期，最后必须调整成可合作。凡是对抗的，都是违规的，这是红线。
2. 不允许农场的战友在公共平台，推特、脸书等等，包括在农场内部，出现攻击、侮辱其他农场和战友的行为。任何人可举报这种行为，我们一定严肃处理。产生了矛盾，可通过联盟委员会协调解决。
3. 各农场只有紧紧联系在一起，才能形成战斗力。郭先生昨天对农场提出的7点要求，要让农场更加标准化、规范化、国际化。
4. 今天我们做直播也是要让农场更加透明化，让各农场负责人和管理团队知道我们的发展方向，也让每位战友知道这些事情和发展的方向，避免战友产生种种质疑，就像正在某些农场发生的。我觉得这个直播是必要的，我们不会回避大家的问题。


长岛伟哥：

1. 如果农场之间或战友之间出现了矛盾，农场负责人之间沟通不成的，再提交到联盟委员会处理；再不行，就要上升到法律层面了，这也不是联盟可以管的了。
2. 目前我们不允许官方地攻击兄弟姐妹农场和其他战友。战友是背靠背共同对敌的。我们的敌人是共产党，不是农场不是战友。


老班长：

1. 联盟公告是发到每个农场的，要求农场必须公开出来。联盟的要求会越来越规范严格，改变农场不透明的现状。
2. 我们是爆料革命，战友们要轻松的舒服的实现自己的灭共理想。
3. 我们不是为了任何人，不是为了组建公司分利益，争做大股东或二股东，以这个为出发点去组建农场的，绝对是灾难。
4. 联盟委员会推出投诉机制后，很多人不理解，但是这是我们接受大家监督的，权力过度集中和不透明，就是新中国联邦的灾难了。
5. 没收到这个公告的，请大家告诉我们。
6. 现在农场发展还不够快，我们对农场的筹建人是有要求的。可以是郭先生推荐的，或者是两个以上农场联盟委员会的委员推荐的，之后你需要提供信息给我们审核，最后郭先生批准后公开后，你可以开始筹建农场。
7. 当地已经存在的农场，你们之间形成协作关系，也可以单独活动。有些人提交了信息，我们正在审核、观察中。
8. 比如台湾，一定需要一个合格的农场。过去有一个台湾农场，但拒绝提供相关的信息给联盟，所以我们先请他们离开。
9. 我们期待着台湾有一个有组织结构的农场。农场负责人需要具备这样的能力，依附于其他农场才能活的所谓农场，是不行的。


长岛伟哥：

1. 联盟接下来的方向是规范现有农场，筹建成立更多新的农场，增加我们的战斗力。首先审核负责人达标与否，还有当地战友人数足不足够。
2. 我们更倾向于在完全没有农场的地方，组建农场，聚集更多的战友。
3. 可能以准农场或工作组的名义开展工作，何时成为正式农场是需要时间的，但这不影响我们为爆料革命做事。
4. 每个人的力量小，但聚集起来的力量很大。参加进来的战友一定要全力以赴，加入进行动来。美国大选结束后，我们还有很多事要做，战友们尽可能投进来更多精力，才可能完成灭共的最终目标。


老班长：

1. 农场规范化命名是符合我们发展新农场的目标。任何人不要幻想一统哪个国家或地区，管好自己那个地方的农场就很不容易了。
2. 如果有人提出在奥克兰筹建第二个农场，当然可以，但鼓励他去其他城市建农场，现在首要任务是在其他没有农场的地方建农场。


### ｜答疑

长岛伟哥：

1，因为银行拒付，G-Club付款方式变了。

答：付款方式变了会产生新的订单号，按照步骤流程操作即可。

2，已经拿到卡号，以后想要升级是不是按网上操作拿到订单号直接去银行电汇就可以？

答：升级需要咨询客服，或者让农场负责人去咨询也可以。

3，G-Club是不是得等VOG拿到编号后才能注册呢？

答：VOG有编号吗？它不属于农场，Sara是凤凰农场，农场联盟不管VOG，只管农场。

4，邮寄过去的支票需要多长时间处理？

答：时间不能把握，如果支票上写了订单号，会很快对账的。支票比较安全，不会被退票。我们几百上千号人还是在等待中，只要钱没被退回去，就是到了G-Club公司。长时间没对上账的，请联系农场专门负责G-Club这块的人，通过他去和总部进行协调解决。

5，关于农场借贷项目

答：大部分农场已经完成了借贷项目，现在对账需要时间。没有备注编号的款项需要较长时间对账，由于安全考虑，参与对账的战友很少，工作量比较大，所以对账速度不会很快。钱只要到账了，借款协议就算数了，这点不需要担心，只差农场回签这一步了。联盟在关注各农场的对账工作，现在是正常的状况。

老班长：

1，5月份寄出的支票购买D-Dollar，要过期了怎么办？

答：只要没被退款，就是购买成功了。

2，G币预购的时候被退回了怎么办？

答：现在还没有这方面信息。被退回的，起诉的过程很漫长，现在能处理的是把已经汇款成功的部分先处理好。

长岛伟哥：

1，G-Club升级会和一次性买5万的福利一样吗？

答：第一批买卡还没结束。G-Club的福利和椅子等等福利，会通过农场来发放。我们希望先买和后买的要有一些区别，比如说今年买的和明年买的。文贵先生也提到，11月3日到1月间买卡的会有小福利啊；有些战友在官网开通前就汇款了，这都是对G系列的信任。接下来会推动这些，体现出战友的优先福利差别。

2，G-Club16号汇款成功，还没拿到号，正常吗？

答：正常的。

3，喜庄园15万当董事咋回事？

这个没听说过，可以把信息发到我们的两个邮箱，我们需要了解一下。

老班长：现有的农场和遇到问题的农场，以及改组的农场，战友的福利都不会受到影响的，有任何问题你们都可以跟我们沟通反应。大家有问题的可以通过邮箱反映给我们，也可以在下次直播中提给我们。

长岛伟哥：我们每周基本上都会在这个时间把这周的工作内容汇报给大家。没回答的问题，请通过邮箱发给我们，我们会进行回复。谢谢大家。
