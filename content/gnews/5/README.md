郭先生从2017年开始爆料，其中有讲关于明星艺人处在共产党的控制之下，表面越光鲜的背地里越不堪。无论是来自哪里的明星，港澳台那都是小盘菜了，共产党都打算进军好莱坞了，妄图把给好莱坞女明星也全都穿上不合身的蓝绿军装。对好莱坞的渗透有多严重，从每年的各种大片都看得出来，尤其是那种世界型大危机，总会有共产党的影子。而好莱坞的创作者也总会为了迎合中共市场，“共产党国家广电总局”让删减就删减。为了钱，没有什么艺术，没有什么思想。HBO就是很好的例子，背后有共产党的资本，想要进入中共国市场，一定要符合社会主义核心价值观，众所周知，共产党的影视剧就是以篡改历史撒谎瞎编为主，还能有啥优秀作品出来吗？如果没有这些作品，那大媒体大影视公司靠啥呢？基本上就成了空壳公司。你看杨贝贝用一年的时间来采访郭先生，素材时间那么长，最后只用了几分钟，郭先生面对那么刁钻的问题依然保持着风度，保持着心理状态，能挑出来造谣的只有这几分钟。经恶意剪辑然后企图伤害郭先生，更严重的是爆料革命的战友，甚至是排华事件，一整年的时间做这一件脏事儿，说成功也成功了，太成功了，这件事一定会成为媒体界的耻辱柱，HBO这样的大媒体会倒在这件事上，背后那些亲共资本包括共产党红色资本更会死在这件事上，这件事带来的深刻影响可能会历经千年，仍被后世津津乐道。
![](https://assets.gnews.org/wp-content/uploads/2021/12/6d33bd4a8ac28b905f7b543046a5afe6.jpg)
这些媒体大资本无法抗拒共产党的邪魔引诱，他们贪婪成性，跟在共产党屁股后面走。世界上几乎没有人深刻了解共产党，除了郭先生。资本主义社会想要的就只有钱，他们不要命，毕竟美国有法律，他们的家庭和成长背景让他们不可能这样做，人最怕的就是先入为主，共产党的大量资本都给这些西方大媒体带来了大蛋糕，当然他们会认为共产党是优秀partner，他们想要的一切不能够在美国社会操作的犯罪行为比如娈童，也可以去中共国操作，还有人主动给他们献上。这让他们愉悦，但是这一切都让共产党录好了小视频，成为要挟的借口，方便操纵，“让你干啥你就得干啥”，“听党话跟党走”，要你的命就是分分钟的事儿，没有什么电影特效的慢动作，什么激情昂扬的背景音乐，呵呵，想多了，一个派出所所长就可以要你的命了。

大资本大媒体集团尚且是这样被共产党牵着鼻子走的下场，何况明星艺人呢？明星艺人的背后有演艺经纪公司，制作公司，发行方，最后才是大资本。中共国的女明星几乎都是共产党的玩物，很多给共产党做白手套的，基本上靠诈骗挣钱了做的第一件事就是抛弃发妻，娶女明星，或者玩女明星。共产党的官员就更不用说了，谁没睡过女明星？每个共产党的官员都以睡过女明星为荣，早期都是睡大陆的，后来就是港台的，最后日本的，再后来要睡遍全球女明星。明星艺人的上位，在共产党这里，都是睡一觉，可以给你某些资源，很多为了事业也要和这些官员睡觉。七哥爆料，港台明星，大陆明星，先开始是黑帮控制的，最后都变成共产党控制了。这些在社会上有巨大影响力的艺人们，在接受电视采访的时候，总是会说自己当年是被星探发现的，那怎么普通人家的孩子永远不会被星探发现？进而包装出道，明星获得了数额庞大的粉丝，粉丝简直为明星神魂颠倒，这些光鲜的表面背后是明星永远都被人身控制，所以明星上妆，服饰搭配极其重要，只有厚重的粉底才能掩盖他们内心的空虚。当然艺人中普遍存在的吸毒乱性等事情也是常见的，按他们自己的话说要去找创作灵感。我始终相信，灵感缪斯不会光顾一个不爱惜自己身体的人，可能常来光顾的是魔鬼。共产党控制了港台资本，这些艺人背后的老板全部都被控制，那艺人就更是瓮中之鳖了。
![](https://assets.gnews.org/wp-content/uploads/2021/12/30-194537_208.jpg)部分吸毒明星 / 图源：网路
在遥远的年代，演艺电视媒体事业是一项需要多人合作并且耗费巨大资本的活动，毕竟那时候无论是写剧本，演戏，拍电影，简单的，写歌词，唱歌，拍歌曲MV，这都是需要专业度极高的。但是现在随着手机、芯片的发展，这些技术限制已经逐渐在消失，再加上直播这种形式具有不可更改，有记忆，可查看，并且同时在线的互动，逐渐连接每个人，成为未来世界的趋势。之前的电影艺人明星在这几年都被网红打败，网红又被直播打败，尤其是郭先生这样不准备，说的都是有用的真相，连续直播十几个小时，不断创造新记录。这样还去看什么电影呢，还去听什么你爱我我爱你，爆料革命展示的是每个人最真实的一面，并且成本最低，比起一个明星上一场直播来说，他讲一句话可能就会要很多钱，而且是没什么营养的口水话，郭先生这样纯免费，并且讲得都是有用的话，让老百姓去选，都会来选看郭先生的直播。又能开智，还能保命，还有这么多可爱的战友，人生在世还图啥呢？

一个真正的艺人，无论是演戏还是唱歌，根本上都是灵魂散发的一种体会，对自然的思考，对社会的启发，好的媒体的采访也是这样，不加修饰，越真实越自然，旁人看到的会越多，受到的启发也会很多。然而共产党底下的明星艺人/大媒体集团，他们的作用是无限扭曲自然，配合进行洗脑。有没有那种本真一点的，有，但那个时代已经过去了。
