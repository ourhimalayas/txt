澳大利亚联邦财政部长弗莱登伯格于澳东周二晚7:30公布了政府2022-23财年预算案。预算案涵盖的移民配额、福利补助、育儿养老、中小生意以及国家经济复苏等最新信息。

**预算案移民政策****主要看点：**

本年度永久移民水平维持在2020年的水平，即16万个名额，包括109,900个技术类（Skillstream）名额。

州际移民将有利于昆士兰，今年将有3万人迁往该州，未来三年每年将超过1.9万人。

澳大利亚的人口将在明年超过2600万，并在2025-26年超过2700万。

从以上预算案中看出未能将年度吸纳移民水平提高到16万以上，这将使那些寻求通过增加移民人数来解决劳动力短缺问题的行业领导者感到失望。

虽然这些名额的重新分配有利于技术工人，但尚不足以填补工程、制造、运输和建筑行业劳动力空缺。

经财政部的研究发现，新移民一生平均为国家财政贡献19.8万，而与之相比普通人对国家财政的贡献为8.5万。

2022-23年度打工度假者签证的上限增加了30%，这将允许增加11000个名额。国际学生也将能够每两周工作40小时以上，以解决零售、酒店和旅游部门的用工情况。
