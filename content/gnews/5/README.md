日本政府从3月1日起，允许外国留学生入境，并对入境日本后经济窘迫的外国留学生开展援助。

另外，日本政府11日还启动一项新措施，利用航班上的空座优先安排留学生入境。

日本政府将对3月底之前入境日本，因新冠疫情而无法打工，没有收入经济窘迫的大学等在读外国留学生，每人发放10万日元补助。

此外，因疫情长期化等而处境艰难的大学、大学院以及专门学校等的学生也可申请领取每人10万日元的紧急补助。

日本政府计划，由各大学等负责受理补助申请，经审查合格后，于3月之内发放补助。
