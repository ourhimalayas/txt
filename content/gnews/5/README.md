日本“每日新闻”11月2日报道，11月1日联合国秘书长安东尼奥·古特雷斯，在《联合国气候变化框架公约》第26次缔约方会议(COP26)上称,当前各国提出的温室气体减排目标，将导致本世纪末的平均气温比工业革命前高2.7摄氏度。

他警告说，如果本次会议失败，各国不是5年一次，是每年一次甚至是经常性地修改气候变化对策。

古特雷斯强调说:“我们面临着艰难的选择，是我们停止对化石燃料的依赖，还是化石燃料的依赖让我们停止。”他说:“最近发表的气候变化对策可能会让人感觉情况好转，但这只是幻想。”

他还指出:“对于小岛国和发展中国家来说，COP26的失败不是选项，而是宣布死刑。”并且呼吁“保护未来、拯救人类”，要求增加小岛国和发展中国家以应对气候变化的支援资金等。

邪恶资本以气候环境为代价扩充财富，对比当下发动中共病毒，以疫苗灾难来解决人口问题，从而保证他们的财富和权力，同样都是为逐利而不择手段，丧尽天良。
