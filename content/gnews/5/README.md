## 文贵先生教路：如何用常识劝人别打疫苗
` 纽约香草山MOS03` [轉載自GNews](https://gnews.org/zh-hans/1582552/)

作者：纽约香草山健身部 – Jack Li文阳
![This image has an empty alt attribute; its file name is Screen-Shot-2021-08-30-at-7.37.59-PM.png](https://assets.gnews.org/wp-content/uploads/2021/08/Screen-Shot-2021-08-30-at-7.37.59-PM.png)
郭文贵先生在10月7日直播里，向战友传授了如何用常识，劝人不要去打疫苗。

![](https://assets.gnews.org/wp-content/uploads/2021/10/0.模版images-2.jpg)

文贵先生说：“药厂在人类遇到最大危难的时候，不应该再花钱游说要救别人。哪有说我要救你，拉你上来，还为你准备一百万的。这到底是你把我推下去？还是拉上来以后还有什么事？我就搞不懂了。”

“全人类都不知道病毒哪儿来的时候，为啥疫苗公司就能生产出疫苗了？让别人打，而且还拿钱去游说，给别人钱打你的疫苗。政府强制打，打了还要你自己承担责任，这叫啥疫苗，这符合逻辑吗？所有西方宗教教人有人命天赋，每个人都有追求幸福安全、美好自由、基本人权、自我表达、追求幸福的权利。疫苗不打行吗？不打，你就要剥夺我的工作、不让我坐车坐飞机，救我还一手拿着钱，一手拿着枪，这样来救我，到底是救我还是害我？”

“关键的问题是：疫苗干了那么多好事，人们要在媒体夸你啊，为啥全球媒体同时不允许被救的人说说浑身的感受和“感谢”你的话？你要救我，让警察拿着枪强迫救我，否则不让我坐飞机火车，不但救我，还要救我全家，用同样方式救，这什么逻辑？”

“如果在这个问题上还有人问到底要不要打疫苗的，那他的智商已经有大问题了。人类过和平时期的安乐生活70年了，人类已经完全不会思考，那就该出大事了。”

“他们说我没有工作咋办？你没有工作？你是没有命重要？还是没有工作重要？你是不能上学重要？还是没命重要？所以说兄弟姐妹们，这是常识。”
