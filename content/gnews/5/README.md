## 有声｜应对朝鲜威胁 美韩强硬表态 日本演练导弹拦截

By:**秘翻新闻**

**2022-04-19**
![](https://assets.gnews.org/wp-content/uploads/2022/04/419-2.png)
[新唐人](https://www.ntdtv.com/gb/2022/04/18/a103403777.html)4月19日报道，朝鲜近期多次试射导弹。本周一，美国国务院朝鲜特使金星容与韩国外交部朝鲜半岛和平交涉本部长鲁圭德在首尔会面后，共同公开表态将强硬应对来自朝鲜的威胁。同一天，日本某航空自卫队基地进行了拦截导弹的演练。

据韩联社报导，金星容与鲁圭德18日的会面，是最近两周内两人的第二次会晤。这次会面后，鲁圭德告诉媒体，美韩之所以在短时间内两次会谈，是因为朝鲜近期频频发射导弹显示其核实力，美韩双方都担忧朝鲜的这种挑衅行动将持续。

同时，日本也针对朝鲜的这种挑衅作出了强硬的反应。日本防卫大臣岸信夫曾对媒体表示，朝鲜频频发射导弹明显是为了提高其导弹技术，这对包括日本在内的整个国际社会来说都是严重的问题。他说：我们将考虑所有选择，包括拥有攻击敌方基地的能力，并将继续致力于大力加强防御能力。
