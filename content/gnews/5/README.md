我对应酬的认识是小时候的红白喜事，大人们你来我往地大声吆喝、大口喝酒，我们小孩子呲牙咧嘴地看着热闹大口吃肉。尤其喜欢看大人们推杯换盏划拳，随着划拳呼词地抑扬顿挫，旁边看客也跟着一起一伏，嬉笑怒骂，挤眉弄眼，喜笑人间，各展高低；气氛起来了，划拳者更是如打了鸡血一样，坦胸露乳，架腿站凳，一顿张牙舞爪，或称兄道弟或祖宗八辈地损骂。反正结果是必须喝好，喝好的标准是喝高，什么是喝高呢？就是必须有人当场“壮烈”喝倒。更有甚者，被喝高的壮烈者，被众人架着放到村中草垛上展览公示，美其名曰“醒酒”，自然引来好事者来此观阅评论，以此达到主家待客的热情厚道。反正不论红事白事，必须喝好，必须热闹！

参加工作后，由于年轻，自然是领导带着应酬挡酒的生力军了。记得第一次和我的部门领导见面，哼哼哈哈问了基本情况，领导突然冷不丁问喝酒还可以吧？为了给领导好印象，打肿脸充胖子，不行也得说行。有一次快下班的时候，突然接到领导电话，晚上一起搞接待，问有什么要求和注意事项，说只要把领导招待好、喝好即可。我用我仅有的老家红白喜事应酬经验理解，此事既简单又艰巨。

晚上主宾落座，寒暄开场，您好我好大家好，一顿好之后，自然是轮流敬酒表达敬意，领导在前，我紧随其后，依葫芦画瓢装傻充愣，说一些不男不女不南不北的奉承话。谈的来的，称兄道弟，恨不得上去亲他一口；谈不来的，就想拿酒灌他个酒肚满肠。

话说那次接待的是一个省领导文部长，很有文采，爱写作，笔耕不辍，常年在各类报刊杂志发表文章，也有真情实感，最为人称道的是每天都坚持写作。大家见缝插针，见部长有如此才华，自然众人轮番捧场赞叹，农村人实在，又不知道如何拍，我就趁敬酒之际唯唯诺诺傻乎乎地说：真想拜读拜读您的大作，学习学习。结果现场一下子鸦雀无声，我的领导也以为我捅篓子了，赶紧跑过来说喝酒喝酒，打圆场。虽然农村人不按套路出牌，文部长已经被灌得晕头转向了，但还是立马从自己随身携带的手提包里面，拿出一个日记本，小心翼翼地打开，醉意朦胧说：小兄弟，您是唯一真心要读我作品的，小老弟，我认你了！老哥哥我今天只是随身携带了日记本，从小养成就这样坚持了几十年，来看看。里面字迹工工整整、一丝不苟，记录着每天的经历、所思所感，文部长语重心长地补充：一定要做您喜欢的，用心享受，希望您有所得！

那次号称几斤不倒的文部长，第一次和我这个只有几两酒量的“新兵蛋子”喝酒喝高了，一直拉着我的手不放，说了很多我也听不懂的话，就差海誓山盟了。事后听其他同事说，由于酒桌上大家乱嚷嚷，听不清，我和文部长喝的互相揪着对方耳朵说话，其他人面面相觑第一次见揪文部长耳朵的。那次我俩吐得一塌糊涂，反正我是连胆汁都吐出来了……

第二天，领导单独找我，瞧了我半天，笑着问我：行啊，没看出来，能让文部长喝好！我就想问问你，你昨晚喝高了一直要找什么草垛，拉文部长去什么草垛干什么……领导的话还没说完就把我臊得落荒而逃。到现在文部长和我还是无话不谈的忘年交，他一有作品也都是第一时间和我分享。

听七哥讲很多的在毛屎坑你来我往的应酬，感叹七哥的为真不破，出污泥而不染和坚持到底！尤其是听到七哥讲和Jason Miller开会每次都不超过20分钟，而跟长岛伟哥、跟Rachel等战友们吃饭，一坐就是几个小时，每次都是意犹未尽；北京医院和七哥素未谋面的医生一起喝酒聚餐，喝到一半的时候就一句话，为那个教练哥们儿喝一杯，大家心领神会，然后说一切都是刚刚开始，大家哈哈大笑，酣畅淋漓……

昨晚做梦梦到战友们一起盘古相聚，旁边是龙虾山，还有数不清的美酒佳肴，反正都是我没见过的（就是没有包子和馒头），我也说不出名字，自然也写不出来，嘿嘿；反正大家都在喝酒庆祝，我也喝高了，到处找七哥，结果很奇怪在盘古竟然有个大草垛，在高高的草垛上，七哥也喝高了，惬意地躺在温暖舒适的草垛上笑眯眯地望着大家……
