## 第三次世界大战人类将面临的四个战场
` NFSC News` [轉載自GNews](https://gnews.org/zh-hans/2181193/)

作者：纽约香草山农场 马克文，朱丽叶
![This image has an empty alt attribute; its file name is Screen-Shot-2021-08-30-at-7.37.59-PM.png](https://assets.gnews.org/wp-content/uploads/2021/08/Screen-Shot-2021-08-30-at-7.37.59-PM.png)
3月16日，爆料革命战友据盖特消息综合分析指，人类近年来面临的挑战堪称二战后最大的危机，甚者可以被称为第三次世界大战。

![](https://assets.gnews.org/wp-content/uploads/2022/03/Slide6-6.jpeg)

而据郭先生多次爆料的消息显示，这次世界大战的始作俑者与支持者就是中共，以及其他邪恶轴心国俄罗斯、伊朗和北韩等。这些的独裁政权一直伺机而动，启动环环相扣的计划，让全球陷入四个主要战场中。

首先，自2019年起，中共利用病毒向世界发起攻击，妄图击垮西方国家的经济并扰乱其政治，使得世界各国动荡不堪。同时，中共对内制定了严格的扫码打卡监管制度，并以此实现了对百姓的以家为牢政策。

其次，今年2月底，俄罗斯大举入侵乌克兰，至此双方爆发海陆空全面战争。

再者，胡塞武装开启中东石油战场，并随即轰炸了沙特阿拉伯首都的石油设施以及其他目标。目前，胡塞武装正在试图扩大战势，发动更大规模的地面战争。

最后，仍有一个尚未启动的战场落在台湾海峡。据郭先生透露，中共正在等待时机，发起对台湾的武装入侵。但是，台海战争也将是世界独裁国家走向结束的开始。

眼下，第三次世界大战已经从中共病毒和疫苗的隐形战场转成为地面战争，这当中，民主国家与独裁政权及世界黑暗势力的最终对决已然开启。但郭先生早先也表示，此次对决的结局必定是正道主义获得胜利，而以民为主的新世界即将要到来。
