![](https://assets.gnews.org/wp-content/uploads/2022/03/截屏2022-03-22-上午10.53.46-1536x113-5.png)
据法新社3月25日报导，美国总统拜登访问波兰。拜登总统在周五搭乘空军一号转赴波兰靠近乌克兰边境的城镇热舒夫（Rzeszow），这里距离乌克兰仅80公里。
![](https://assets.gnews.org/wp-content/uploads/2022/03/image005-17.jpg)
拜登首先前往了美军基地，与驻地美军第82空降师官兵进行了会晤。随后与波兰总统杜达听取人道救援情况简报。

拜登在热舒夫发表讲话时说：“当你看到一个三十多岁的女子站在一辆坦克前，手持步枪，（这情景）令人想到发生在天安门广场的事情，那就是天安门广场。”拜登以此称赞乌克兰人民的“勇气”和“韧性”。此外，拜登再次将普京称为“战犯”。

拜登发表讲话时说：“我们要做的最重要的事情，就是以我们的决心和努力保持民主国家团结一致，以减少那个我认为是‘战犯’的人所造成的破坏”。

据美国国务卿布林肯说，美国政府已于3月23日正式确定“俄罗斯军事人员在乌克兰犯下了战争罪行”。

拜登对现场协助人道救援的团队表示，目前已有1000万人流离失所，380万人逃往他国，还有成千上万的人遭到俄军围困。应持续扩大协助，与乌克兰政府密切协调。

拜登此前曾宣布，再提供10亿美元的人道援助，并且，将接收1万名乌克兰难民。
