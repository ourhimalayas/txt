2021年12月17日，郭文贵先生在大直播中提到孙正义的远景（另译“愿景”）基金：

##### 是我推荐了孙正义给马云，四年前我（给孙正义）的建议是，能拿的钱赶快拿走，永远记住，不能长干。远景基金投了将近1000亿美元，到今天连本带利都没了。孙正义就是和中共太近了。这次马云卖日本的资产，当时他说把资产做了担保抵押，马云什么都拿不走。

孙正义，出生于1957年8月11日，生于日本佐贺县鸟栖市，是软银集团的创办人兼社长，软银股份有限公司会长。据《福布斯》杂志报导，他在2011年拥有81亿美元净资产，名列日本富豪榜第二位。孙正义在《福布斯》2019年亿万富翁排行榜中名列第43位，资产达到216亿美元。孙正义曾在20世纪90年代雅虎估价最高时，超过微软创办人比尔·盖茨，成为一天时间的世界首富。他依靠软银数次对阿里巴巴集团的控股与投资回报，多年高踞日本首富之位，仅有数度被柳井正超越。

奠定孙正义江湖地位的三笔投资，分别是对阿里巴巴的投资、对美国著名互联网门户站的雅虎的投资，以及2016年对英国ARM近250亿英镑的巨额现金投资收购。

软银远景基金（SoftBank Vision Fund）成立于2017年，是隶属于软银集团的风险投资基金，拥有超过1000亿美元的资本，是全球最大的以技术为中心的证券投资基金。2019年，软银远景基金2号（SoftBank Vision Fund 2）成立。截至2021年3月31日，这两只基金的总公平价值为1540亿美元。

2017年初，孙正义与新当选美国总统的川普一起高调亮相，宣布500亿美元的投资计划。同年5月份，远景基金完成第一期930亿美元的募集。根据《金融时报》调查获得的一份数据资料显示，930亿美元来自于沙特阿拉伯公共投资基金、阿布扎比投资基金、苹果、高通等，软银自身投入的资金为280亿美元。

从基金管理结构上来看，远景基金具有区别于传统风险投资基金管理结构的特点，具体表现在，远景基金所募集的很大比例的资金是以债权的形式，而非股权的形式存在。例如，沙特阿拉伯公共投资基金所投资的450亿美元资金中，约280亿美元以优先债形式存在，剩余170亿美元以股权形式存在；阿布扎比投资基金总共投入的150亿美元中，有93亿美元以优先债形式存在。在所有融资来源中，只有软银自身投入的280亿美元全部为股权形式。

2021年，软银公布了截至第一季度的财报，其净利润为1.93万亿日元（约合177亿美元），创下1990年以来日本上市公司中季度净利润最高记录。其中，大部分利润来自“远景基金”，净利润为2.3万亿日元，主要受到了韩国电商公司Coupang成功IPO上市的推动。

远景基金成立之初就在各个前沿领域广泛展开布局，包括无人驾驶、虚拟现实、癌症检测和基因诊断、人工智能机器人、室内种植等。远景基金的投资包括Brain Corp——一家从事室内机器人控制软件开发的初创公司；Plenty——进行室内种植，宣称可以是传统土地产出530倍的初创公司等。

远景基金在中国的投资占比23%，主要投资包括：字节跳动、商汤科技、滴滴出行、满帮集团、众安等。由于中共政府打击滴滴出行和阿里巴巴的监管行动，冲击了软银的估值，给软银的前景蒙上阴影，软银股价较2021年3月创下的最高点下跌三分之一，尽管该公司已完成创纪录的2.5万亿日圆股票回购。

2021年7月8日，孙正义在2021世界人工智能大会开幕式上表示：“软银远景基金所投资的公司超过90%，甚至95%，依然还没有实现盈利，甚至仍在亏损。”

2021年11月8日，软银集团发布业绩报告，三季度实现营收1.5万亿日元，同比下降3%；净亏损为3979亿日元，其中远景基金亏损8251亿日元（约合465亿人民币），为远景基金成立以来最大季度亏损。

孙正义在财报发布会表示，公司目前正处于“暴风雪”中，公司的净资产价值在一年的时间里下降超过770亿美元，其中远景基金第一大重仓股阿里巴巴的百分比已经下降至不足三成。

当被问到中国科技行业以及初创企业未来发展前景时，孙正义表示软银集团将继续投资中国企业，特别是人工智能行业。
