有很多著名的算命先生，比如韦千里，他怎么出名的呢？据说当时蒋介石被张学良土匪扣在西安城里，宋美龄就去找韦千里给蒋介石算命，问蒋的安危，韦千里给看了蒋的八字，还用六壬给占了一卦，从八字和占卦看来，蒋都是性命无忧的。后果然应验。从算命这个学科来说，韦千里是民国算命大师，还有一位大师叫做袁树珊，民国当时其实也算不上是非常抵触命学的，社会风气也相对很开放，但总是一些政客和商人被人们记住得多，比如蒋介石杜月笙，知道他们的人比知道韦千里的人要多多了。只有在接触算命的时候，有了一些基础了，才开始知道韦千里，他有写过很多关于命理的书，比如著名的《千里命稿》，这个书写得非常笼统，看半天也看了，也就只知道个皮毛，还是什么都不懂。

![](https://assets.gnews.org/wp-content/uploads/2022/01/截屏2022-01-12-18.43.59.png)宋美龄 / 图源：网路

很多算命的书都是算命先生写的，从现代汉语的角度去理解，一是有些晦涩，二是理论型的章节过后，为了方便理解，这些命书上都记载了一些例子，况且都是很有名的人的命，后人可以通过对这个人的生平来套在那个理论上得出结论，但是如果仅仅凭着一个命盘去推断这个人的命局，虽然也有很多人做得到，但能做80percent就不错了，就可以称大师了。但无论如何，算命这件事很神奇。那么一个人的命是不是都能被算中，七哥那天直播里有讲，这些影响只占到20percent，说白了，毕竟这些书都是人写的。算命和信佛呢，有点类似，大部分都是拿着人写出来的东西当成是给神，给上天的颂。话多讲一句，*大众文学二连*，每个人有每个人的性格，每个人写出来的东西其实都充满了个人色彩，他对算命的理解那是他的理解，他对佛陀的理解也属于是他自己的理解，后人看他们写出来的书，整理出来文字，只是为了给想要多点理解的人一些参考，并不能代表这件事的全貌。后人在读书的时候会过度解读，会产生误读，都是再正常不过的事了。所以书里记载的东西很局限，读万卷书，行万里路，还不行，还得阅人无数，难道这些就能足够了解了吗？不一定的。每个人都是一个造化，自己的造化自己成就。

无一例外的，所有的命书都会介绍一点关于地球大爆炸之始的故事，比如太极啊，混沌啊，两仪四象八卦，理气象数之类的，毕竟这些东西都是形成算命的基础，但是，很难说，基本上没有一本书正儿八经把这个讲得清楚全面的，很多书都是一隅。或者说，大众文学三连，我根本看不懂。西方的科学和逻辑的发展，对探索外太空带来了很多便利，在教科书上，很容易很清晰地能看到，现代物理学、地质学关于地壳变动的原理，地球上经历的那几个世纪，侏罗纪、白垩纪、寒武纪、等等，每个时期有不同的地表情况，不同的气候环境。甚至有的课件还会做一些生动的视频来展示这些变迁。但是最初那一刻，地球怎么形成的，月球为什么和地球有如此恰到好处的距离，多一点地球就会海啸洪水，甚至毁灭？一些权威机构大多数赞成大爆炸学说，但是大爆炸之前是什么样的，很多很多的研究都是尚有疑问的。

![](https://assets.gnews.org/wp-content/uploads/2022/01/p2694752a514946395-ss.jpg)大爆炸。/ 图源：网路

七哥直播中讲到的关于大爆炸时期如何形成地球，月球又是怎么形成的，以至于地表很多山啊，河啊，到底除了山河之外，那种所谓的脉气是什么样的存在，又是怎么影响到人的，这些问题，从目前听到的情况来看，七哥的解释最清楚也最好理解。并且这个解释里有人和地球万物的联结关系，毕竟之前我们看到的各种研究，要么都是形，要么都是气，没有联系，这也解释了为什么我们对之前的这些研究从来不敢苟同，因为这些知识总是缺乏一个链条，一根绳子，碎片一样一块一块的掉在那里，七哥这么一说，就是给了一根绳子串起来了。七哥老中医，哈哈，还能讲天文地理，乐哉。

再回来说命书，一个写这种书的人，可能穷极一生也没能走遍万里河山，不走完这些山，没见过昆仑山附近的巨大磁场，和死亡之谷，怎么可能在脑海里有关于这些沉默山河潜藏巨大力量的感知？放在今天，七哥都在不断强调自己去过了地球上多少地方，亲自去过，摆放高人，才知道了这些。说白了，不要太把所谓的书当成圭臬，不要把什么人使劲儿捧上去，越这样，这个东西就完全成负担了。还怎么找真正的信仰啊，信仰不是宇宙里的一粒沙，让人走路硌脚了都不扔，该放都得放，毕竟那始终是别人的，不是我。正如一场梦，梦里再好的山河，再香的五花肉，或者再脏再艰难的处境，一定要有醒来的时刻。沉溺了梦魇了醒不过来，假作真来真亦假，恍恍惚惚一梦一生。
