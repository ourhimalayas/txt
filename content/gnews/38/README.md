乔·拜登27日声称，几乎百分之百的美国人都需要接种疫苗，这样国家才能安全地恢复正常。

![](https://assets.gnews.org/wp-content/uploads/2021/09/图片24.png)图片来源：Dailycaller

拜登在周一接受新冠疫苗强化注射时说，“97%，98%”是美国的接种率目标。他没有解释他是从哪里得到这些数字的，白宫在记者采访时也没有提供这一说法的来源。

拜登的声明与医学专家今年早些时候的预测相冲突。包括Anthony 福奇博士在内的医学专家预测，群体免疫只需要70%至85%的美国人接种疫苗。

截至2021年9月底，大约55%的美国人口已经完全接种疫苗。拜登政府正在推动这些美国人在最后一次接种疫苗的几个月内接受强化注射。CDC在9月中旬批准了辉瑞公司的一种疫苗加强剂，用于65岁以上的美国人，或那些面临新冠病毒特别危险的人，如免疫力低下的人。

CDC主任罗谢尔·瓦伦斯基（Rochelle Walensky）在周四晚间推翻了她之前的建议，也建议一线人员接受强化注射。

CDC在周五早些时候的一份声明中说：CDC的这一更新的临时指南允许数百万处于新冠病毒最高风险的美国人接受辉瑞-生物技术公司的新冠病毒加强针，以帮助增加对他们的保护。

目前仍不清楚需要加强注射多长时间，因为福奇曾多次表示，美国可能几乎无限期地面临新冠病毒的爆发，就像季节性流感一样。

一次又一次的调整，毫无道理的疫苗逻辑：从施打疫苗可以结束新冠，到疫苗并不能阻止传播，但可以加强免疫力，从两针疫苗可视作完全接种，到需要接种第三针，从70%多可以达到群体免疫到需要100%接种，一次又一次地表明了疫苗的无效性。政客却对之视而不见，更不顾疫苗对群体造成的伤害，仍在一味地强制接种。正如维加诺大主教所说，这是全球范围对人类有计划的大屠杀。
