这个世界上为什么有独裁？为什么有的独裁还能统治那么久？并且独裁已经危害到家门口了，我们依然能够看到永远的懦弱和妥协。二战时期丘吉尔的至暗时刻，把老头子整个都拉垮了，直到日本偷袭珍珠岛以后，美国直接进入盟军的名单，丘吉尔才从至暗时刻转向了光明，他可算看到了希望。在希特勒的独裁下，老百姓也许有无数次机会不选择合作，但是能起作用的结果终究太少。希特勒把犹太人全部都杀掉，虽然有不少犹太人事先已经逃走，然而大部分犹太人还是死在了独裁的屠刀之下。杀人恶魔到最后身边没有一个人和他说真话，希特勒“以梦游者的姿态走向他的人生”，多讽刺啊，他杀了那么多人，把全世界都快毁了，他说只是一场梦而已。死掉的那些人命都是这场梦的铺垫者，尸体累累，再厚也是一层底色。所以，这些死去的人命生前有没有后悔？

共产党从一开始统治中国，就开始了荼毒与残害。在文革中受到创伤的被迫改写那些屈辱，一切残忍都得放下，以党为重，给党拿好这块遮羞布，那是国家政策的不得已调整，人人都要为了党，从来没人会提这根本就是毛贼东的个人仇恨。红卫兵漫山遍野要赞颂毛主席，红小兵小小年纪就开始举报自己的父母亲生兄弟姐妹，说这都是忠于毛主席，可是毛贼东在中南坑里吃着红烧肉，而老百姓吃糠咽菜，甚至没有吃的，这种时刻如何谈忠诚？每个人拥有的不过都是一副肉体，为什么肉体和肉体能有如此大的不同？凭什么中国人就得逆来顺受，遭受苦难，国内还热衷于一直洗脑勤劳致富。谎言欺骗，没有真诚，在中国大地上是常见的，从生活的时时处处都可以看到。二战的希特勒对民众的统治也是如此，不可以有独立的思考，不可以有个人空间，集体大于一切。希特勒和共产党所处的不同年代让人唏嘘，中间有将近百年的跨度，独裁好像随时随地都能存在，并且存在的范围可以在短时间内急速扩大。

习近平自从上台以后，再也不是常委大家一同议事参政了，共产党内几个家族一起盗国一起剥削老百姓的局面不复存在了。习近平的家人操持了整个国家的运营，李木子彭丽媛彭磊，所有的亲戚全都上，不管会不会，不管有没有能力，那是一整个国家的治理啊，就这么草率和鲁莽地全部圈在习近平的家族之下。那些盗国贼拼命把钱往海外运，王岐山甚至不顾一切全盘掌控了华尔街，江泽民家族更是在海外拥有深耕的基业，盗国贼的家人资产全部都在海外，表面上看来习近平虽然在国内霸占着点什么，但是海外势力还是以盗国贼家族为主。习近平仿佛也知道这个道理，除了彭丽媛和他本人，他们全家都是海外国籍，这叫什么呢，一圈海外的人统治着国内14亿老百姓，所以他们怎么会有怜悯呢，怎么会有恻隐之心呢。加入任何别的国家的国籍，原则上是不允许再回去参政的，尤其去做机关要员。因为在加入国籍的时候是有宣誓的，比如加入美国国籍的时候，宣誓词里有很多关于这样的词句，我誓死效忠美利坚合众国，只要法律要求我会为了我的国家去战斗。加拿大的入籍誓词也是类似的，我誓死效忠加拿大女王，我维护我的国家，我旅行加拿大公民的责任。中南坑的盗国贼家人几乎全部都在海外，这个入籍誓词宣读过后，他们又回来统治中国老百姓，当然了，中国老百姓就是案板上的肉啊，想怎么剁就怎么剁。毕竟他们都是外国人了，不受中国法律的制约，也可以不受户籍地的法律制约，因为他不在国籍的国土上，况且不是每个国家都会有引渡条例。

邪恶势力，独裁势力，都倾向于把家人和偷盗的老百姓资产放在国外，因为他们自己也知道自己这个国家没有希望没有未来，一切不过都是洗脑，是政治需要，不然这些奴隶老百姓怎么会给这些盗国贼创造源源不断的财富呢？所以那些帮助他们藏钱的各种海外机构，包括为此事增加便利的一些官员，他们是不知道这是邪恶的财富吗？他们当然知道，但是蓝金黄的利益让他们眼馋啊，毕竟保护私有资产的民主国家的公务员待遇太低了都。

![](https://assets.gnews.org/wp-content/uploads/2021/12/未标题-1.jpg2_-1280x720.jpg)

当独裁给全世界带来巨大的人道危机的时候，那些个平时设立了用来干预此事的国际性机构，全部可以替独裁掩盖罪行，比如WHO，联合国，断然漠视共产党屠杀香港人，西藏人，新疆人，中共国内各省市的老百姓。因为这些机构是没有钱的，他们都靠着每个会员国给的供养费过活，谁拿钱谁是大爷，大爷让他跪他就得跪着。这些国际性机构指望不上来遏制独裁，那一些大的国家，比如美国，世界的灯塔，他就可以了吗？答案是不可能的，因为毕竟这是一个主权国家，他自己国家的日常事务都需要打点，尤其是民意这里，你无法让一个不了解全世界的普通老百姓来投票关于共产党大屠杀的罪孽，只能通过媒体来唤醒，朝鲜战争的结束就是一个记者带回了关于战争中到处死人的报道引起美国国内老百姓的激愤，才结束了这场惨无人道的战争。所以媒体起到的作用是巨大的，那么独裁就去收买媒体，给人播送的都是共产党的好，甚至有人还向往共产主义的统治。BLM就是典型的共产主义洗脑，真得有人在公园里搭帐篷种菜了。
