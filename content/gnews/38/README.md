台湾，台北。去年6月13日，是我国与中共国建交四周年纪念日。两国便民联姻的蜜月期比北京任何官员想象的都要短。中共国对拉美和加勒比地区的最大误解之一就是认为我们国家处理外交关系就好像它们是国家政策一样，也就是说，无论哪个政党或人物掌权，都使双边关系保持连续性。这远没有准确理解本土政治家在外交政策管理中的个人主义和以自我为中心的本质，本世纪迄今为止，在台湾与中共国的外交问题上，同样的模式至少重复了四次。

巴拿马与中共国建交四年后，双方誓言的爱意已荡然无存，初恋的风华也已荡然无存。中共国再次表明它不了解巴拿马精英政治和本土经济的特质；他还低估了该国选举民主的动力；更糟糕的是，它似乎挑战了巴拿马和美利坚合众国之间的战略关系。中共国认为，促进与中国公民或海外后裔的关系的政策，以及将巴拿马列入拉美和加勒比国家名单，都得益于中共国政府主办的旅行计划，以参加“以了解中共国及其治理”（不维持外交关系）——数千名巴拿马人参与——将改变巴拿马在建立外交和政治关系之前时期的精英和社会行动者的看法。

在商业层面，根据主计长的数据，去年中共国成为我国商品出口的主要目的地，总额达3.7亿美元。这一增长主要是由于巴拿马向该国出口铜——该国消耗了世界上生产的铜的一半——用于其制造和建筑部门。当然，对中共国发展而言，巴拿马关键原材料的出口从来不取决于建交与否。近几个月来，地峡媒体强调了巴拿马牛肉对华出口的情况，但没有分析为什么该项目之前没有进入中共国市场，尽管两国都设有商业代表处二十多年。而且，虽然中共国几十年来一直将巴拿马作为该地区的商品分销平台，但北京以缺乏正式关系为由，不允许巴拿马的各种商品进入该亚洲市场。

在投资层面，与巴拿马的看法相反，中共国在我国的存在的特点不是作为大型投资来源国发挥突出作用，而是作为支持者根据国家预算参与公共基础设施工程的资助。也就是说，应该指出的是，这种类型的参与不符合外国直接投资 (FDI) 的条件。中共国公司，主要是国有（国有企业，SOE），他们认为巴拿马——与他们在该地区其他国家的经历相反——一个财政稳定的国家，我们国家项目的利润有助于为结构上无利可图的中共国国有企业体系注入新的资金。具有讽刺意味的是，虽然我们的产品直到 2017 年才进入中国，但中共国公司在我们国家确实有一个开放的领域来定位自己以获得百万富翁的合同。

巴拿马方面认为，建立外交关系将促进中共国在该国的投资。据墨西哥自治大学（UNAM）拉丁美洲和加勒比地区中共国学术网络项目《中共国对我地区外国直接投资监测》显示，2019年该项目的投资额将达到2000多亿美元，主要用于两个项目，一个以天然气为基础的电厂，一个集装箱港口和一个物流园区。不幸的是，第一个项目被搁置，第二个项目正在取消特许权。

四年后的今天，巴拿马与中共国的外交关系停滞不前。与中共国大使最近所指出的相反，关系没有发展，不稳定，更不健康。巴拿马外长在欧洲之行时对中共国的评论，指出巴拿马新政府采取了“中立”的立场，更何况“过去的错误是卷入了巨人之间的争端。”在第二部分，我将分析这些评论的深度。
