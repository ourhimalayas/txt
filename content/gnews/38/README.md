## 新疆再教育营与纳粹集中营一样种族灭绝
` NFSC News` [轉載自GNews](https://gnews.org/zh-hans/2275965/)

作者：静言
![This image has an empty alt attribute; its file name is Screen-Shot-2021-08-30-at-7.37.59-PM.png](https://assets.gnews.org/wp-content/uploads/2021/08/Screen-Shot-2021-08-30-at-7.37.59-PM.png)
纳粹的奥斯维辛集中营是纳粹德国时期建立的最主要的集中营，也叫灭绝营，位于波兰南部全国第二大城市克拉科夫西南小镇奥斯维辛。
![](https://assets.gnews.org/wp-content/uploads/2022/04/11-3.jpg)
奧斯维辛集中营于1940年4月27日，由纳粹德国亲卫队领导人希姆莱下令建造。共有3个主要营区，和39个小型营地或工厂。最主要的目的是对收容者进行强迫工作，集体处决，或进行非人道的人体试验，对犹太人进行有系统的杀害。据估计，约有110万人在奧斯维辛被杀。1947年，波兰国会立法，在此建立纪念纳粹大屠杀博物馆，作为第二次世界大战中纳粹德国犯下极恶罪行的历史见证。

新疆再教育营是中共对在新疆境内设置的多个特殊机构的一个称呼，新疆民间称其为“转化所”“学习班”，外界称其为集中营。2013年，习指挥成立“国家反恐怖工作领导小组”，2014年首先在新疆发起了针对维吾尔人的所谓“反恐人民战争”。2016年陈全国调任新疆，具体地把再教育营纳入了新疆政策。2017年起, 再教育营的建设规模化、扩大化。当局引进最新科技，打造并强化了全面覆盖的网络监控系统。先后增加招募了十几万警察和辅警，在新疆全境设立了超过7,300个检查站，新疆被拘捕人数猛增，中共的新疆再教育营建设逐渐达到高潮。

中共官方对外谎称，建立新疆再教育营是为收留受极端思想影响，且尚不构成犯罪的人员，包括参与活动但并未造成实际危害后果，自愿接受培训的，刑满释放前经评估仍具有社会危险性的穆斯林偏差人员，学习汉语、法律、职业技能，使他们去极端化，以便重归正常社会生活。

但实际上并非如此。据统计，在遍布全疆的再教育营里，因信仰、民族身份或其他非违法原因被关压的人数达150万。

据侥幸逃脱的人员透露，维吾尔再教育营内部条件恶劣，拥挤不堪，被关人员时常经历被虐待，强迫劳动，甚至死亡；有报道称，再教育营破坏维吾尔族等民族的信仰，强迫他们唱红歌、吃猪肉、喝酒，拒绝会被罚禁食、坐老虎凳，以及不让睡觉等；对维吾尔族人进行语言及文化清洗；强制孩童与父母分离；强制绝育；对被关压人员强行进行器官摘除等。

新疆再教育营与纳粹奥斯维辛集中营一样，是地地道道的维吾尔种族灭绝营。
