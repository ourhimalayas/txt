近一个月不见的杨总今天来了，整个人我都快不认识了，又憔悴又苍老，他说，不知何故睡眠不好了，我疏通他头部经络时，发现有个硬硬的疙瘩，他自己也担心出现脑梗啥的，问他可有买青蒿素药吃？他拒绝回答我。
