## 奥密克戎疫情恶化 危及中共经济核心上海和广东
` 日本東京方舟農場` [轉載自GNews](https://gnews.org/zh-hans/2172185/)

**撰稿：日本东京方舟农场 北极**

据法新社3月16日报道，近期中共国奥密克戎病毒疫情急速恶化，导致上海和广东多处封闭停产，危及中共经济核心。

![](https://assets.gnews.org/wp-content/uploads/2022/03/20020316-5.jpg)2022年3月15日身穿防护服的工作人员在上海一处因发现新冠疫情而被封闭的小区的屏障外。（图片来自美国之音）

中共卫健委3月15日在官网发布通报，周一共有3,507例确诊的本土感染病例，比一天前的1,337例翻倍暴增，连创新高。

随着近日奥密克戎病毒感染的快速扩散，中共国制造及科技重镇深圳被迫封城七日，上海也因疫情突变而不得不封区。深圳附近的制造业重镇东莞周二也下令封城七天，有感染的工厂则必须停产。

广东省是出口大省，2020年其出口占全国出口总额的24%，仅东莞2021年就为中国GDP贡献了1.09万亿元的产出，在全国类似大小的城市中名列第五。

分析人士表示，上海和深圳是两个中共国最富有的城市，目前都在封城，将会重创中共的经济。

中共外宣机构仍然宣城，目前的“动态清零”总方针和一系列的防控措施，在应对奥密克戎变异株疫情上是“行之有效”的。

截至3月14日，中共国大陆累计报告疫苗接种近32亿剂，疫苗接种总人数达到12.7亿多人。
