## 【新闻速递】海康威视被揭——聘请前美国官员为说客

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/07/08025536/WhatsApp_Image_2020-10-24_at_04.33.05-6.jpeg)



有记录显示，与中共国政府关系密切的视频监控公司——海康威视正在与一名前美国政府官员合作。海康威视的游说公司 Mercury Public Affairs 于 6 月宣布聘请前财政部外国资产控制办公室制裁政策顾问 Peter Kucik。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/07/09214116/Untitled-design-1.png)图片来源：网络
据IPVM 称，中共国政府创立了海康威视，是其“控股股东” 。据Axios报道，Kucik 将帮助该公司对美国政府最近对中共国采取的制裁展开公关行动。

2019 年，美国商务部限制美国公司与海康威视开展业务。去年夏天，美国国防部将海康威视列入中共军方“拥有或控制”的公司名单。海康威视对此提出异议，并表示它从未“为中共军事应用进行过研发工作”。

今年6 月，拜登总统禁止美国实体投资海康威视和其他数十家中共国公司，以应对中共“军民融合”构成的威胁”。据称该公司还参与了建造新疆维吾尔清真寺和拘留营的监视系统。海康威视也否认了这一指控，称其没有“在新疆采取任何不当行为”。

《华尔街日报》 5 月报道称，“海康威视在 2019 年向中共空军出售了无人机和其他辅助设备，并在 2014 年被认为是中国军队的顶级供应商。”据报道，国家电视台的画面显示，一名中共士兵手持带有海康威视标志的无人机干扰武器。
