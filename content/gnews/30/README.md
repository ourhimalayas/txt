## 蓬佩奥爱荷华演讲：“我们不是种族主义国家”

#### 编译：脐橙君
校对：萌萌的朋克
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/07/21212357/mp.png)图片来源：网络截图
前国务卿迈克·蓬佩奥（Mike Pompeo）周五在爱荷华州向一大群有影响力的福音派人士发表了讲话，他强调了他的社会保守派资历，该州也是半个世纪以来，一直在总统提名日程上领先于其他地方。

蓬佩奥在今年第二次访问鹰眼州期间，从早到晚的活动日程填满了他的行程，引发了更多关于他在2024年共和党总统候选人提名竞选中是否会有更进一步的抱负的猜测。

当被问及潜在的白宫竞选时，蓬佩奥在接受福克斯新闻采访时说：“我只是没有像其他关注目光那样，把思绪腾挪来考虑这些事情。”

他在第10届年度家庭领袖峰会上发表演讲，他强调说：“事实上，我们是美国历史上最支持生命的政府。该峰会由家庭领袖主持，家庭领袖是位于爱荷华州的白人福音派社会保守派顶级组织。

来自堪萨斯州的陆军退伍军人和前国会议员蓬佩奥在成为美国最高外交官之前曾担任时任川普总统政府的中央情报局局长，他认为“当我们盯着今天看到的所有问题聚焦点时，就会发现美国家庭的崩溃是其中大多数问题的核心。”

蓬佩奥一再抨击拜登总统政府、国会中的民主党人和“左派”，他指责他们“破坏了家庭单位”。

他指责当今学校不合格的师者，他们用各种伪装和方法来批判种族理论。并且将整个国家打入本质上和系统上的种族主义者。

“我们不是一个失败的国家，”他宣称。 “我们不是衰落的国家。我们不是种族主义国家。我们是上帝保佑的基督教国家。不要受社会主义的影响，我们应该继续工作，重新对社会做出承诺。”

他还指责拜登政府“不愿保护我们南部边境的安全”，并抨击中共国是“一个无神的国家，每时每刻都挑战着敬畏上帝的人民”。

他还批评拜登总统政府对古巴反政府抗议活动的反应，他告诉福克斯新闻“我很遗憾拜登政府没有全力支持，来帮助示威者。

这位前国务卿在家庭领袖峰会上的讲话是周五在爱荷华州繁忙行程的一部分。忙碌的行程不仅仅是他在爱荷华州的现状。

今年早些时候，蓬佩奥在新罕布什尔州为州党和候选人举办了几场虚拟活动，新罕布什尔州在爱荷华州之后举行总统初选，并在提名日历中获得第二名。两周后，他成为南卡罗来纳州主要州党筹款活动的头条新闻，该州举行了第一次南部初选，并在共和党提名日历中获得第三名。蓬佩奥在全国各地忙碌地出行，以帮助共和党同僚在2022年中期选举中获胜。

被问及川普总统是否会参加2024总统竞选时，蓬佩奥说“川普总统会做他的事，他有他自己的选择。我们会尽自己的一份力量。”

他发誓“我将继续参加这场斗争，我相信川普总统也将继续参加这场斗争。”
