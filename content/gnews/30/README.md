最近有读到一篇文章，链接在这里：[Beijing Is Intentionally Underreporting China’s Covid Death Rate (Part 1)](https://www.forbes.com/sites/georgecalhoun/2022/01/02/beijing-is-intentionally-underreporting-chinas-covid-death-rate-part-1/?sh=57fe559d4352)

文章的作者各种头衔都非常实在，在某个行业工作超过25年，搁共产党这儿来说，叫半辈子了，这位作者称得上是沉甸甸的前半生。这个文章主要内容是分析在新冠病毒大流行期间，西方民主体制下和共产党一党专制独裁体制下的金融市场和经济活动的某些联系。文章里罗列了很多种数据，比如近两年的金融市场震荡跌幅完全是根据新冠病毒走向来的，病毒稍微控制住了，人们的生活也就恢复了，金融市场就会回暖，反过来，病毒猖獗，人们不得不停止一切经济生活，lockdown，供应链出现问题，金融市场就会走低。然而中共国的病毒痕迹和西方民主国家是一致的，不可能全世界都在大流行，唯独中共国这里没有病毒，只是文中说中共国的金融市场走向没有和病毒的情况保持一些相关性，所以作者怀疑共产党少报了中共国内的新冠病毒死亡人数。这不是作者的唯一判断理由，他的理由还包括共产党在整个新冠病毒大流行期间对病毒溯源问题的不配合，真实数据不共享，导致了卫生状况的来不及应对和正常科研进展的阻碍，人类死了这么多人，付出了惨重代价，但是共产党北京方面还是对病毒溯源的问题或者三缄其口，或者各种歪曲事实带其他的风向。

![](https://assets.gnews.org/wp-content/uploads/2022/01/unnamed-3.jpg)共产党propaganda / 图源：网路

文章还没有写完，还有两部分，这第一部分已经处处是逻辑思维大坑了，但是作者居然得出了正确的理论，共产党确实在掩盖新冠病毒死亡的真实人数，从郭先生直播中得知，在11月8日共产党的六中全会之前，中共国内死于新冠病毒的人数为1370万人。还是回看文章，首先，新冠病毒的传播和大众公共卫生对病毒的处理，防疫措施，并不是和金融市场呈正相关的。西方民主国家的市场是自由经济，这方面来说有一些相关性，然而一个阶段的数字并不能代表什么，从整个新冠病毒大流行的两年时间来看，金融市场整个市值疯了一样上涨，但问题是，人类在这个期间几乎都没怎么创造价值，要么就是在家里隔离，要么就是好好的突然lockdown了，不工作但是股票市场一个劲儿地涨，那钱从哪儿来的，毫无疑问美联储超发的，相当于美股市场那个是虚高的价格，因为生活用品的价格在不断上涨，通货膨胀时时刻刻都在发生着。这并不是一个能够拿来参考的数据，往些年经济活动的真实数据不会全部反映到金融市场上去，大部分的金融市场都是些大科技公司，这类公司背后大多数都有共产党的红色资本，从另个意义来说是最容易被操控股价的，尤其是美股市场上还有共产党的房地产公司，互联网公司，恒大不还美债，或者之前阿里巴巴公司，突然一下子涨上去，然后一下子跌下来，这就明摆着是操控价格，钱全都被共产党盗国贼们套走了。
![](https://assets.gnews.org/wp-content/uploads/2022/01/Slide7-1.jpg)图源：news talk
至于共产党内部的金融市场，那就完全是擀面杖子经济，都是电脑操作的，想怎么改就怎么改。作者是西方人，他的数据模型和逻辑思维是典型的西方式的民主框架里形成的，有时候看得人很着急。他的结论是北京谎报了新冠病毒的真正死亡人数，他认为死亡人数比那个官方的4636要多得多，这个结论都得出来了，居然还在根据共产党的金融市场数据来推理。既然共产党连人的真实死亡数据都可以更改，那为什么就不能操控一下金融市场的数据呢，这个比那个要容易得太多了。作者的意思就是要去中共大陆投资或者还在大陆的外资企业要注意了，共产党不说实话，小心你们的资金血本无归。美股市场的背后大鳄都是共产党，起码华尔街这里就是王岐山做老板的，美股市场的震荡绝大多数原因是共产党那里的金融出了问题，或者就是七哥爆料中所说的，让那些被抓起来的人都把资产一个一个拿回来。金融市场看似是非常具有参考意义的一个方面，但促使作者得出北京在掩盖真正的新冠病毒死亡人数的原因，是共产党对新冠病毒溯源的阻挠，这是一个正确的前提，沿着这个思路一直往下走，不管是什么思维，一定能够发现共产党撒谎，并且撒了不止一个谎。让人挺惊讶的，原来共产党这两年终其全部精力在做的事情就是掩盖关于新冠病毒溯源的事情，在西方的媒体表现上，起码从作者的字里行间看，这个策略完完全全失败了。看来但凡是个人就明白，一些拼命不配合，拼命要掩盖的东西，肯定不对劲儿。那么这个道理沿用下去，就说到新冠疫苗上来了，几乎全世界所有的媒体都在掩盖关于新冠疫苗造成的人的健康问题和死亡状况，推特上支持新冠疫苗成为了政治正确，多少用户被推特永久封禁了。一样的道理，越是掩盖越有问题，并且是大问题。
