## Eglise医生建议病毒高发区战友强化预防
` GTV新闻访谈` [轉載自GNews](https://gnews.org/zh-hans/1770817/)

作者： 纽约香草山信息部 诺雅

![This image has an empty alt attribute; its file name is Screen-Shot-2021-08-30-at-7.37.59-PM.png](https://assets.gnews.org/wp-content/uploads/2021/08/Screen-Shot-2021-08-30-at-7.37.59-PM.png)

12月18日，Eglise医生在盖特发布了《疫情严重地区加强用药指南》，建议中共病毒感染高发区的战友，通过联合服用两种以上抗病毒复制药来降低感染风险。
![](https://assets.gnews.org/wp-content/uploads/2021/12/Screen-Shot-2021-12-19-at-9.16.41-PM-1.png)
其中联合2-3种抗病毒辅助药指的是：

1. **必须选择服用伊维菌素或硫酸羟氯喹之一，**
    - 如果是选择服用硫酸羟氯喹：一份的剂量是200mg，第一周连续用药五天，之后每周一次；
    - 如果是选择服用伊维菌素时：第一周口服2次，按体重 0.2毫克/公斤，两次之间要间隔48小时，之后则每周服用一次相同的剂量。
2. **槲皮素 1000毫克（mg）和 NAC1000毫克（mg）则是选择至少服用其中一种。**


此外在免疫强化方面，她建议每天服用：锌 25-50毫克（mg），维生素C 1000毫克（mg），维生素D 5000国际单位（IU）或125微克（mcg）。

至于青蒿素保健品则因人而异，可以根据厂商包装上标注的建议量，按需服用。Eglise医生特别提示，由于青蒿素与维生素C、槲皮素、NAC的作用方式相反，所以要和它们以早晚分开的方式服用。
