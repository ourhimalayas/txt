## 解直锟之猝死是充当中共白手套的必然结局
` GTV新闻访谈` [轉載自GNews](https://gnews.org/zh-hans/1772996/)

作者：纽约香草山农场 -文真
![This image has an empty alt attribute; its file name is Screen-Shot-2021-08-30-at-7.37.59-PM.png](https://assets.gnews.org/wp-content/uploads/2021/08/Screen-Shot-2021-08-30-at-7.37.59-PM.png)
在12月19日大直播中，郭文贵先生直言，解直锟的猝死，与海航王健之死一样，是充当中共白手套的必然结局。

![](https://assets.gnews.org/wp-content/uploads/2021/12/Slide2-17.jpeg)

解直锟早年靠倒买倒卖起家，建立“哈尔滨信托”后，通过放高利贷等诈骗手段发迹。他建立的中植集团没有为社会创造过任何财富，仅利用假合同和金融票据，通过洗钱和操纵空壳公司上市套利。中植集团和中共控制的其他各种金融控股集团一样，无一例外是中共盗国贼家族的腐败集团。这些集团的掌门人为中共家族代持资产，俗称白手套。早在2017年爆料中共海航腐败事件时，郭先生曾提及解直锟与王岐山家族的关系。

郭先生说，解直锟兄弟都是王岐山的白手套，负责执行中共蓝金黄的核心计划。早在2018年，王岐山曾警告解直锟，不要再提与高燕燕和贯军的关系，并警告他要么自杀，否则被灭口，二选其一。与海航王健一样，解直锟在死前已得知，自己终将逃不过被中共杀害的结局。
