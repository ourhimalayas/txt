## 俄罗斯背弃中共 转战东亚
` GTV新闻访谈` [轉載自GNews](https://gnews.org/zh-hans/1738132/)

作者：纽约香草山信息部- 诺雅
![This image has an empty alt attribute; its file name is Screen-Shot-2021-08-30-at-7.37.59-PM.png](https://assets.gnews.org/wp-content/uploads/2021/08/Screen-Shot-2021-08-30-at-7.37.59-PM.png)
在12月8日的直播中，郭文贵先生爆料称，俄罗斯已经完成和美国的勾兑，被默许重返东亚。

![](https://assets.gnews.org/wp-content/uploads/2021/12/Slide2-9.jpeg)

郭先生提到，多年来，习近平耗费约1万亿美元意图收买俄罗斯，而今，却在这最后一刻遭到背弃，失去了其绝对的自信和疯狂的靠山。

郭先生表示，俄罗斯近期已经与缅甸、柬埔寨签订了军事基地合约。另外，俄罗斯明了中共即将倒台，因此重新在南海地区开采石油，其采油量是供应中共国的一倍还多。且据郭先生透露，印度、越南、缅甸、柬埔寨、菲律宾等国家宁可转向俄罗斯，也不愿意与中共合作。

与此同时，美国也做出了战略调整，将基本默许俄罗斯称霸亚洲，甚至可能会将越南的金兰湾军事基地让给俄罗斯。 郭先生解释，于美国而言，只要俄罗斯愿意合作，不再攻打乌克兰，那俄罗斯通往德国的天然气管道可以继续供气，并且也将允许俄罗斯继续在南海地区采气、采油，俄罗斯的军事基地也可以得到恢复。

郭先生强调，现今，美俄联合抗共局势已经基本成型，中共此番必将腹背受敌。
