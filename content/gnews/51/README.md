自然新闻：被监禁的维基解密创始人朱利安·阿桑奇的未婚妻斯特拉·莫里斯 (Stella Moris) 发出警报，她的伴侣处于困境，他在监狱中正遭受黑暗势力试图对他的谋杀。

这位 50 岁的维基解密出版商在争取引渡到美国的过程中被关押在最高安全级别的监狱中，最近遭受了现在被认为是一次小中风。

据称，由于美国法院对他的持续诉讼，阿桑奇承受了如此大的压力，以至于他的健康状况下降到神经损伤的程度（要么就是他们正在给这个人注射神秘的“疫苗”）。

据报道，阿桑奇现在右眼睑下垂，记忆力有问题。  《每日邮报》（英国）称他遭受了“短暂性脑缺血发作”，也称为大脑供血中断。

据说阿桑奇现在正在服用抗中风药物，因为他担心如果导致中风的病情和压力持续下去，他可能很快就会患上全面中风，这似乎是基于他的可怕情况。

“朱利安正在挣扎，我担心这种小中风可能是更严重攻击的前兆”，同时也是一名律师的莫里斯说，“这加剧了我们对他能否在这场漫长的法律斗争中生存的时间的担忧”。

阿桑奇的父亲说，他在狱中被迫“接种”新冠肺炎疫苗

正如我们去年报道的那样，中央情报局已经抓捕试并图毒害阿桑奇，并从他的家人那里窃取他的 DNA。

一家名为 UC Global 的西班牙安全公司泄露了有关如何在阿桑奇居住的伦敦厄瓜多尔大使馆内安装“侵入性和复杂的”秘密监视设备的信息。

现在，阿桑奇似乎受到了一种新的攻击，他的父亲在最近的一次直播活动中表示，这可能是由于他儿子在狱中接受了“疫苗接种”。 据报道，由于害怕拒绝注射会遭到报复，据报道，阿桑奇同意接受接种疫苗，这据说是为了帮助他“安全”对抗武汉冠状病毒 (COVID-19)。

“当然他已经接种了疫苗”，阿桑奇的父亲约翰希普顿说，“所以他在这方面别无选择，没有任何选择，他无法掌控自己的身体”。

阿桑奇服用或不服用的每一种药物都会被记录在案，而这些信息很可能会在法庭上宣告终节。 正因为如此，阿桑奇应该遵守注射福奇流感疫苗，以增加他看到积极结果的机会。
