【香港】政务司司长李家超上月提出检讨本港法援制度的“建议”，包括三大改革方向，不准刑事案法援申请人自选律师代表等。表示，新制度会在年底前全面落实，但未有透露确实日期，近日，有民众表示，新的法援制度其实已开始实施。

有媒体调查得悉，多名月前申请法援的《国安法》被告证实，法援通知，将有署方委派律师，目前，有至少三人打算放弃法援，请愿自费数百万打官司。

首宗国安法定罪案件，因“煽动他人分裂国家罪”及“恐怖活动罪”两罪罪成，被判入狱九年的唐英杰，早前就量刑及定罪提出上诉，将于本周五 ( 26 日) 进行指示聆讯。由于该项新制度，法援已经指派“杜伟强律师事务所”于上诉案中代表唐英杰，而原审的代表律师中，刘伟聪大律师被踢出局，不被允许继续代表唐英杰。

据杜伟强律师事务所的网页资料显示，创始人杜伟强身兼深圳市政协常委，亦是北京仲裁委员会、北京国际仲裁中心仲裁员等等，并以“深得各地、尤其来自中国大陆客户的信赖”作为评价。

另外，有法援署职员透露，现时所有刑事案被告均不可自选律师，皆由署方指派。
