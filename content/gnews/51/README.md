**答**：我不敢说今年就能够有那么大比例的人口变化，但到2023、2024年会有几何倍数的非常大的变化，我觉得75亿人口最多只能剩下一半。

只有解药，有效的介入治疗，才能挽救一部分人的生命。

威廉王：那我十分希望这个药物能够从我们新中国联邦这边推出，那么我们就是真正地在拯救人类了。

**答**：我们今天说的内容就是在拯救人类，如果有谁真的能听进去的话，我们每天都在拯救人类啊。

14. 请问印度的青蒿琥酯和双氢青蒿素可靠吗？

**答：**印度产的青蒿琥酯和双氢青蒿素其实是大多数战友已经购买和很多人已经在使用的，特别在海外的。我还没看到所谓的假药。除非这个药本身有一些副反应，好像服药的人都比较正常，至少这个药物的治疗目的是达到了的。印度的药物管理市场还是比较严格的。

15.（1）我没感染过病毒，打了疫苗两个月后去做了一个lgG的冠状病毒抗体测试还能测出来吗？如果测出来了是不是就说明疫苗里还有一些刺突蛋白，而不全是氧化石墨烯？（2） NAC能过血脑屏障吗？能过是不是就可以降解脑子里神经细胞里血液里的氧化石墨烯？

**答：**(1) 做免疫球蛋白检查，如果比较高的话说明你的肌体曾经有过病原体的侵犯，就是刺激可以是来自于疫苗，也可以来自于天然病毒本身。氧化石墨烯不刺激抗体的产生，是刺突蛋白诱发抗体的产生。(2) 不能。我们预防需要的药物清单里面任何一个都不能过，可以过的是阿司匹林，还有一个是治疗抑郁症的氟伏沙明或百忧解。

16. BESTVITE青蒿素保健品的说明是饭前吃，与饭后吃有矛盾？

**答：**我本人是不主张在无特殊原因时,饭前空腹服用任何药物，特别是长期服用的。哪怕是NAC，曾经也有说明NAC在不完全空腹时，或在两顿饭之间。要保护你的胃，但如果没有引起胃肠任何不适，实在想饭前吃, 可以试。

17. 好奇问一下，mRNA之外的疫苗（灭活之类的）造成堵塞是什么原理？血栓是怎么造成的？也是抗体和免疫系统造成堵塞吗？到底是刺突蛋白造成的还是抗体造成的？

**答**：抗体没错，是刺突蛋白引起的血栓。我们在第13期讲了有许多机理可以形成血栓，最主要的还是红细胞本身的凝集，血小板受损失后引起凝血障碍，就会出现奇葩的现象：血栓性溶血，就是一边溶血一边因为细胞损伤而导致各种免疫细胞过来检查，免疫细胞本身就可以成为血栓组成部分，所以有很多可以引起血栓的机理，不是抗体的错。免疫系统不会造成堵塞。

mRNA进入细胞产生刺突蛋白，附着在上皮细胞表面，同时血管上皮细胞壁损伤，二者都会吸引淋巴细胞、体液细胞等来检查，这些细胞聚集在一起也可形成血栓。

威廉王：那可不可以结合断食来让身体产生自噬，然后吃这些破损的老化的或是癌细胞来减轻血管的堵塞？

**答：**不能说没有用，但我觉得自噬最多可能是吞噬脂肪细胞。大量的有病变的有损伤的细胞单纯靠自噬的作用肯定是不够的。自噬其实就是一种人体的天然的抗氧化作用的一种表现。需要一批志愿者以后可以自觉地去进行这种叫生酮饮食–间歇性断食来看看对疫苗副作用的控制有无帮助。

18. 刺突蛋白需要宿主才能复制，如果物品上有刺突蛋白，会一直都有传染性还是经过一段时间就没有了？

**答：**刺突蛋白的宿主必须是人体，只有人体有ACE2 受体。物品上没有那些受体，自然就死亡了。气溶胶传染不会传染物品，它传染人体的细胞，这就是这个病毒恶的地方，因为它是在实验室里被改造过的只认人体ACE2的受体, 而不是动物的ACE2受体。

19. 气雾疗法中提到氯化镁溶液，可以用氯化镁盐自己制作吗？它是不是和生理盐水一样浓度是0.9%？

**答：**你必须买商品的氯化镁，因为它里面矿物质的浓度调到最好。我不建议你自制氯化镁，它并不昂贵，而且每次加的量极少，所以最好买现成的。

20. 槲皮素、NAC、黑孜然籽油、青蒿素四者中，是不是只有青蒿素不能在生理期时吃，别的都可以？

**答：**是的，因为青蒿素亲铁，又是间歇性的服药，所以生理期那几天休药是比较稳妥的。

21. 我天天猫在家里，没打疫苗，感染了还能生娃吗？

**答：**天然病毒感染，不是接种疫苗，那你会产生中和抗体，对你健康没有什么大的负面影响，但病毒的一些残体会在某些细胞内隐藏数月以后才出现症状。所以你需要一定时间的排毒,一定要有持续性地用预防剂量的伊维菌素或者其他的营养素, 和间歇性服用青蒿素衍生物或保健品。

22. 氧化石墨烯会导致血栓吗？还是会割破血管壁呢？如果溶栓后氧化石墨烯又开始活动了，那是不是还会持续造成血栓呢？脑子里的这些东西怎么清除呢？沸石可以吗？

**答**：沸石也是一种保健品，有一些报道说它跟NAC相结合可以综合掉化解掉一部分氧化石墨烯。氧化石墨烯这个主题真的需要一些实验室数据来支持，我看到很多文件或报道也是相互抄袭的。所以对于这个主题我一直不愿意深度探讨，因为我没有临床数据，只有确实数据才能说服我，进而才能说服观众。

23. 请问用双氢青蒿素哌喹片（40mg的双氢青蒿素，0.32g的磷酸哌喹）排毒，有怎样的建议？

**答：**双氢青蒿素哌喹片比青蒿素哌喹又进了一步。因为青蒿素进入体内也要转化成双氢青蒿素才起药效的，所以用它是很不错的。同时磷酸哌喹的半衰期（6-9天）比哌喹要短，那么你就要按照抗疟药说明的剂量和疗程来服用。如果是国内产的双氢青蒿素，它是8粒一盒一个疗程。它建议第一天服用双剂（四粒），然后第二、三天每天两粒。其实对付病毒的刺突蛋白，不是针对疟原虫, 每天均匀剂量就可以了，不用第一天给自己那么大的刺激。每天两粒，四天一个疗程，如果当中没有任何不适反应，可以间隔两周以后再用第二次。如果条件允许就去做一个肝脏的检查，为了保证你后续的几次疗程能按照这个剂量按部就班稳妥地排毒。因为你至少需要三个疗程。还只是起始排毒, 以后每年都有可能需要维持排毒, 次数,疗程因人而异。

24. 一个朋友打了三针，最近脖子淋巴肿大，医生说是下颌下腺炎，用抗生素一个星期都没有好转。最近加了地塞米松，没有发热，口腔牙龈疼痛有所缓解，淋巴结节还存在，请问有什么好建议？

**答：**这个就是典型的疫苗接种以后的副反应，叫淋巴结炎。淋巴结肿大是个症状不是一个诊断，它一旦肿大，身体的什么地方肯定有问题。因为它是重要的免疫器官，说明你免疫系统出了问题，也就是刺突蛋白攻击了你的整个免疫系统，包括所有的淋巴细胞，因为淋巴结就是淋巴细胞的聚居地。患者要考虑的是按照我们建议的排毒方案的治疗预防剂量服用我们推荐的所有病毒复制抑制剂和多种营养素。 同时还是青蒿素，青蒿素，青蒿素。

25. NAC是谷胱甘肽的前体，那么可以用谷胱甘肽替代NAC作为预防服用方案吗？

**答：**可以直接用，但是相对比较昂贵。而且使用方法有很多种，甚至静脉谷胱甘肽都可以。因为它是抗衰老的很有效的一个氨基酸。

26. 维生素B可以吃吗？

**答：**当然。维生素B可能是维生素最大的家族，从B1到B12对人体都是非常有用的，特别是B1, B2, B6, B12。尤其之前感染过Delta病毒的吃B1会对味觉恢复有帮助。

27. 买到锌和维C的复合剂，但听说二者要分开用，那这个复合剂是不是不好？

**答：**如果不服用青蒿素或其保健品，那么二者是可以一起用的。

28. 戴隐形眼镜和眼镜在PAND答MIC时期有什么不同？
