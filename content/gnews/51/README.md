神经病，又称傻子，这句话似乎触动了某些东西。就像我一个长辈，年纪轻轻家里的顶梁柱，得了癌症晚期。家人亲友都主张放弃治疗，这时候医院的老专家出来了。说话当然是冠冕堂皇，能延长寿命。放疗化疗保守治疗，人受罪不说，最后钱花光了人也死了。落了个妻离子散家破人亡。

可能不少传播真相的战友会被人嘲笑为神经病、傻子。就像疫苗，普遍给来一句，人家都打了都没事，怕什么？这几天家里一个长辈和我聊疫苗问题，问我：“既然疫苗有问题，那为什么这么多人都打了？图什么？”这个问题我觉得可以和另外一个对爆料革命有点了解家人的另外一个问题一起回答。问题是：“中共开放三胎四胎，为什么又放病毒打疫苗杀人？”我用了养猪场的比喻做了回答：“这批猪该出栏了，就得杀了吃肉，但是人又不是真猪，是有思想的，只能放病毒，再打毒疫苗的方式来处理。”
