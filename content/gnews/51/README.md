**9月24日汇编**

1、[**战狼秦刚狂言“中国民主”体现林肯理想**](https://www.aboluowang.com/2021/0924/1650835.html)

9月22日，中共驻美大使秦刚出席卡特中心和乔治·布什美中关系基金会联合举办的视频对话会，并发表演讲。演讲中，秦刚列举了中国人民某些所谓“参与选举和协商重大政策的权利”，作为美国民主制度与中共政治制度相似的“证据”，并借此宣称中共是建设“社会主义民主政治”，东西方不存在“民主和威权”之争。秦刚并称，中共的“全过程民主”是“为人民服务”，“和林肯总统的‘民有、民治、民享’都是为了人民”，中共政治制度“体现了美国前总统林肯和发明代议制政府的古希腊人的理想”。彭博社报道指，秦刚的言论是中共试图从新定义一些关键西方概念的最新作为，旨在争取习近平所指示的更大“话语权”。中共外交官经常辩称，美国、日本等西方国家的价值观“不代表国际社会”。

2、[**人行紧急放水4400亿人币 应对恒大危机**](https://www.aboluowang.com/2021/0924/1650829.html)

中国人民银行紧急向市场输血，为中国恒大(03333)一旦债务遗约做好准备。报道统计指，人行连续4个交易日进行千亿人民币等级的逆回购，总计放水4,400亿人民币。中国人民银行为维护季末金融市场的流动性平稳，周四(23日)开展了7天期和14天期逆回购操作，规模为1,200亿元。在中秋假期前，9月17及18日分别进行1,000亿人民币逆回购操作，假期后第一天22日再度进行1,200亿人逆回购操作，加上今日的1,200亿。报道指，中国央行积极对市场注资主要是缓和恒大财务暴雷引发的市场不安情绪，大举放水的结果使得银行同业拆款利率下降，隔拆利率下跌14个基点至2.01%，为9月3日以来的最低水平，10年期政府债券孳息率也跟着下跌1个基点，至2.85%。

3、[**中国经济亮红灯 李克强开会要求“稳增长”**](https://www.aboluowang.com/2021/0924/1650801.html)

近期，恒大危机是否会成为中国版“雷曼时刻”一直吸引着市场的目光，但与此同时，中国经济已经亮起红灯。经济学家任泽平指出，消费、投资、出口三架马车全面放缓，中国经济正在处于衰退周期。周三，中共总理李克强在国常会上，要求“保持经济平稳运行”。中共统计局数据显示，8月宏观经济数据全面明显回落，供需两端放缓，8月规模以上工业增加值两年平均增速放缓至5.4%，社会消费品零售总额两年平均仅增长1.5%。有经济学家判断，中国三季度经济增长可能趋近于零。尽管中共官方近来一直试图稳定大宗商品价格，加大了原材料储备投放，中国8月工业生产者出厂价格指数（PPI）依然继续攀升，同比升幅9.5%，创13年来最大涨幅；与此同时，由于消费疲软，8月居民消费价格指数（CPI）同比上涨0.8%，环比升0.1%。两个指标之间的剪刀差再度扩大，显现出经济滞涨的特点。然而，这些还不是中国经济最大问题，更大的问题来自政治环境改变带来的潜在危机。

4、[**大陆15家银行安抚投资者 宣称涉恒大业务风险可控**](https://www.aboluowang.com/2021/0924/1650758.html)

恒大债务问题一石激起千层浪，不少贷款给恒大的银行担心受到波及，使投资者退出，故连日来大陆不少银行急忙安抚投资者，强调与恒大的业务风险整体可控，或澄清自己与恒大没有多大的业务往来。安抚投资者的多数为地方银行，如浙商银行、江苏银行、郑州银行等。据澎湃新闻等统计，中国大陆至少有15家上市银行借由不同的方式，回应投资者“关于银行与恒大业务往来”的问题。投资者最关心的问题包含“对恒大的贷款规模多大”、“目前风险是否可控”等。报道指出，多家银行对于投资者的疑虑均表示“抵质押充足，整体风险可控。”

5、[**恒大将连续支付三笔巨额款项**](https://www.aboluowang.com/2021/0923/1650587.html)

中国房地产龙头恒大集团近日深陷债务危机，恒大于09月22日早间宣布，透过“场外方式协商”，23日会如期支付人民币2.32亿的债劵利息。与此同时，根据彭博社报道，中共央行向银行系统注入了1200亿元人民币。这让担心中国第二大房地产商违约、影响可能波及全球金融体系而紧张不安的市场，暂时得到缓解。另外，恒大在23日当天，还另需支付于2022年3月到期债券相关的8,350万美元利息。9月29日还需支付于2024年3月到期债券相关的4,750万美元款项。总计金额高达1亿3千万。恒大若未能在预定还款日期后30天内付清利息，这两笔债券都将违约。
