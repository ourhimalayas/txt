## 共和党赢得弗吉尼亚州长选举

作者：英国喜庄园 – I770770
![This image has an empty alt attribute; its file name is Screen-Shot-2021-08-30-at-7.37.59-PM.png](https://assets.gnews.org/wp-content/uploads/2021/08/Screen-Shot-2021-08-30-at-7.37.59-PM.png)
11月3日，美国弗吉尼亚州州长选举结果出炉——共和党候选人格伦·扬金以50.7%对48.6%的得票率击败民主党候选人特里·麦考利夫，成为自2009年以来首位赢得该州州长选举的共和党人。

![](https://assets.gnews.org/wp-content/uploads/2021/11/Slide5-2.jpeg)

据了解，弗吉尼亚州是美国少数在总统大选次年就进行州长选举的州，该州拥有大量受过大学教育的郊区选民。在过去的十年间，该州州长一直由民主党人担任。消息称，这场在2022年美国中期国会选举前进行的州长竞选，被视为明年两党竞争国会控制权的关键。 前国务卿蓬佩奥表示，对于整个美国来说，这是个好兆头。

弗吉尼亚州长选举结果甫一出炉，郭文贵先生也在盖特上发文表示了祝贺。郭先生表示，世界有很多大事儿都正在发生，弗吉尼亚的选举将再一次震撼世界。此时此刻发生的每一件大事儿，都是对新中国联邦的重大优势和对中国共产党的致命打击。
