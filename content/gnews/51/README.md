## 非洲猪瘟肆虐！华东人只吃华东猪？

撰稿：东京樱花团/喜马拉雅的微尘
![](https://assets.gnews.org/wp-content/uploads/2021/11/2-62.png)[图片来源](https://finance.sina.com.cn/roll/2020-01-17/doc-iihnzhha3069790.shtml?source=cj&amp;dv=1)
近日，上海、江苏、浙江、安徽、山东、河南六省区发布《关于协助东部区实施生猪调运有关措施的函》。

文件表示，根据农业农村部《非洲猪瘟等重大动物疫病分区防控工作方案（试行）》（农牧发[2021]12号）要求，结合东部区实际，东部区六省（直辖市）将于2021年12月1日起实施生猪调运有关措施：自2021年12月1日起，除种猪、仔猪及非洲猪瘟等重大动物疫病无疫区、无疫小区生猪外，不再接收东部区外30公斤以上生猪进入东部区育肥，不再接收东部区外未经“点对点”调运备案的生猪调入东部区屠宰。

此外，文件还要求，除种猪、仔猪（重量在30公斤及以下且用于育肥）、非洲猪瘟等重大动物疫病无疫区、无疫小区生猪、经东部区“点对点”调运备案管理的生猪外，禁止辖区内出具调运至东部区的生猪检疫合格证明。

中国是世界上头号猪肉消费大国，生猪存栏数约占全球一半。自2018 年，在东北沿海省份辽宁首次确认病例以来，非洲猪瘟（ASF）已在中共国扩散流行。虽然，非洲猪瘟不直接威胁人类健康，但对猪具有很强的毒性，感染后死亡率近 100%。

随着中共实施非洲猪瘟分区防控，猪肉供应将以各省自给为主，即华东人只吃华东猪的时代开始了，由此猪肉价格恐出现较大地区差。
