《福克斯新闻》发表文章，题目是，“美国将微软的Exchange 黑客攻击归咎于中共国政府的附属机构”。副标题称，微软 Exchange 软件遭到黑客入侵，全球数万台电脑遭殃。

全文如下：

拜登政府周一（7月19日）指责中共国要对今年早些时候微软 Exchange 电子邮件服务器软件遭到黑客入侵，全球数万台计算机遭到入侵的事件负责。

美国政府是与欧盟、英国、加拿大、澳大利亚、新西兰、日本和北约等一批盟友和伙伴共同宣布这一消息的。

拜登政府一名官员说，这是北约首次谴责中共国的网络活动。

“任何单一的行动都不能改变中共国在网络空间的行为，单一的国家也不可能。”一位政府高级官员说，“我们紧密的盟友和伙伴关系，美国独特的优势，以及我们对网络威胁信息共享和防御的共同应对，都是巨大的力量源泉。”

微软交互平台黑客在今年1月首次被发现，并被某私营组织确定为中共国网络间谍所为。一位政府官员表示，政府之所以将此事归咎为隶属于中共国家安全部的黑客，部分原因是发现了勒索软件和营利性黑客活动，政府希望在宣布这一消息的同时能为企业提供有关中共国人一直使用的策略的指导。

中共外交部发言人在被问及微软交互平台黑客事件时，此前对美联社表示，中共国“坚决反对和打击各种形式的网络攻击和网络盗窃”，并提醒说，网络攻击的认定应以证据为依据，而不是“无端指责”。

但拜登政府和盟国还披露了来自北京的一系列其他网络威胁，包括来自政府下属黑客的勒索软件攻击，这些黑客攻击公司的目的是索要数百万美元。

一名官员说，中共国家安全部利用刑事合同黑客在全球进行未经批准的网络行动，包括为自己谋取私利。这位官员说，活动范围从犯罪活动，如网络敲诈勒索，以加密劫持，并从世界各地的受害者那里盗窃经济利益。

与此同时，美国政府一名高级官员说，国家安全局、网络安全和基础设施安全局以及联邦调查局将揭露中共国家资助的网络行为者在针对美国和盟国网络时使用的50多种战术、技术和程序，以及应对这一威胁的技术缓解建议。

拜登政府官员说，他们已经向中共国政府高级官员表达了对微软事件和中共国“更广泛的恶意网络活动”的担忧，称他们“明确表示”他们的行为威胁到“网络空间的安全、信心和稳定”。

美国政府一位高级官员说：“美国及其盟友和伙伴不排除采取进一步行动追究中共国的责任。”与此同时，美国司法部周一（7月19日）宣布对四名中共国公民的指控，检察官说，这四名中共国公民与国家安全部合作进行黑客攻击，攻击目标包括公司、大学和政府实体等数十个计算机系统。

副总检察长摩纳克（Lisa Monaco）在一份声明中说：“这些刑事指控再次突显，中共国继续利用网络攻击窃取其他国家的产品，公然无视其双边和多边承诺。中共国黑客攻击活动的广度和持续时间，包括针对医疗保健、生物医学研究、航空和国防等方面及对十几个国家的黑客攻击，提醒我们没有一个国家或行业是安全的。”

摩纳克补充说，“今天的国际谴责表明，世界需要公平的规则，各国投资于创新，而不是盗窃。”联邦调查局副局长保罗·阿巴特（Paul Abbate）说，联邦调查局以及联邦和国际合作伙伴仍然致力于令施加风险的恶意网络行为者承担后果。
