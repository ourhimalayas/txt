欧盟委员会（EU）29日宣布，将启动“欧洲新闻室”项目，16家欧洲新闻机构将参与其中，共同报道与欧盟有关的新闻。

此项目将于2022 年1月开始。欧盟将支持176万欧元（约合2.25亿日元）。

参与的新闻机构包括德国的DPA、法国的AFP、意大利的ANSA、西班牙的EFE等欧盟周边国家的主要新闻机构。总部设在布鲁塞尔（欧盟总部所在地）。此项目是欧盟支持媒体的独立性，多样性以及应对数字化措施的一部分。

据法新社报道，所有参与媒体将共同分享信息，安排联合采访，并在一个专门的网站上发布新闻。参与的记者还将接受数字技术和事实调查方面的专业培训。

衷心希望欧盟此项目能真正为真相报道提供一个透明的平台，而不是统一喉舌，开始言论管控，成为像中共一样的大外宣。
