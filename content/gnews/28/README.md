据美国之音3月1日报道，俄罗斯入侵乌克兰之后，中共当局明显偏袒普京，民间许多人甚至为普京点赞叫好。中共国南京大学孙江教授、北京大学王立新教授、香港大学徐国琦教授、清华大学仲伟民教授和复旦大学陈雁教授，共5位学者发表公开信，谴责俄罗斯入侵乌克兰，以及中共国部分民众幸灾乐祸，反对非正义战争，并支持乌克兰人民保家卫国。不过，他们的公开信在上网不到3小时便被封杀。

该文受到非官方智库欧洲对外关系委员会亚洲项目主任、历史学家顾德明(François Godement)的推介。他为这几位中共国历史学家感到骄傲。北大、清华、复旦、人大、山东大学等中共国大学的130位海内外校友2月28日发布反对俄罗斯入侵乌克兰的联署，强烈呼吁中共国政府履行1994年乌克兰废核后，与乌克兰签署的为乌克兰提供安全保证的声明及其承诺。
