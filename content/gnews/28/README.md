## 共军对台心战用方言喊话遭耻笑
` GTV新闻访谈` [轉載自GNews](https://gnews.org/zh-hans/1838559/)

作者：纽约磐石农场 – 在远方
![](https://assets.gnews.org/wp-content/uploads/2021/12/条幅.png)
台媒1月7日报道，中共军方近日发布一个宣传视频，出动30多架陆航飞机对台进行心战喊话，用方言与普通话轮番播报，要求敌方赶快投降，缴枪不杀，遭到墙内和台湾民众耻笑。
![](https://assets.gnews.org/wp-content/uploads/2022/01/10-2.jpg)
台湾前空军副司令张延廷认为，这只是中共做样子给人看的宣传片而已。如果真要对台湾心战喊话，就应该使用普通话或者台湾话。而共军用方言，只能是为了宣传。

台湾学者认为，如果中共对台动武，绝无可能使用闽南语。该视频是共军完全不顾对敌劝降喊话标准作业规范，而胡乱对号入座所为。
