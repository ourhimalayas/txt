![](https://assets.gnews.org/wp-content/uploads/2022/03/图片1-100.jpg)图片来源：福布斯
据《国会山报》（The Hill）3月18日报道，正在兴起的数字资产是数字和金融领域的一场革命，它让消费者以互联网的速度在全球范围内直接进行货币支付。数字资产的广泛使用将使消费者能够近乎即时地转移资金，而不是依靠一系列中间商来进行交易。

数字资产的非凡前景和日益上升的受欢迎程度，促使立法者、监管者和经济学家研究中央银行数字货币（CBDC）等工具。CBDC是中央银行货币的一种数字形式。目前，全世界有90多个国家正在研究、开发或实施某种形式的CBDC。美国的中央银行，即美联储，在今年早些时候发布了一份文件，将研究CBDC的潜力。但是CBDC是一个非常危险的工具，它威胁到消费者的隐私和安全，以及数字资产市场的持续创新。

正在开发的CBDC中，最值得注意和最令人不安的是中共的数字人民币。作为数字独裁主义的巅峰，这种数字货币将与社会信用评分挂钩，使得中共政府可以即时了解和控制其人民的财务状况，并提高其镇压异议人士和控制言论的能力。

传统的现金支付为消费者提供了从隐私到个人自主权的广泛好处，这是CBDC无法复制或取代的。但这并不是说，美国应该忽视或回避数字资产市场。相反，我们应该接纳正在蓬勃发展的私营部门替代方案，提供金融包容性的机会。

稳定币（Stablecoin）是一种价值与法定货币挂钩的数字资产产品，它可以满足消费者的需求，并且降低了支付成本、提高了支付速度，并促进对包容性支付系统的使用。像稳定币这样的创新让许多人能以可承受的价格进入金融市场，否则就会被传统银行系统抛弃。一个设计不良的CBDC有可能损害消费者，并排挤像稳定币这样的私营部门创新。随着21世纪经济的成型，我们必须警惕不会走上数字独裁主义的道路。

郭文贵先生屡次提到，由于法币货币的超发，数字货币的时代无人能挡，以后将是数字货币为主，法币为副的时代。而喜币、喜联储和喜支付则走在了数字金融大变革时代的前列。喜币每年10亿的限量，良好的流通性，中心化的管理，20%的黄金锚定，最先进的三代区块链技术打破了限制货币流通的物理和地域限制，零误差交易，以及易交易的支付系统，其特性超过了各国政府的CBDC，更不用说数字人民币了，所以必将受到世人追捧。
