## 【中日财经爆】三星和美光发出警告 西安封城影响内存芯片生产

整理编辑：东京樱花团/peterwong
![](https://assets.gnews.org/wp-content/uploads/2021/12/image-1853.png)[图片来源](https://unwire.hk/2021/12/29/samsung-8/fun-tech/)
世界上最大的内存芯片制造商韩国三星电子和美国美光科技在12月30日同一天警告称，中共国西安为控制中共病毒疫情而封城，可能会影响他们在该地区的芯片制造基地。

美光周三表示，封锁可能导致其广泛用于数据中心的DRAM内存芯片的供应延迟。西安本月早些时候实施的严格防疫限制措施可能越来越难以缓解，已经导致其生产基地的人力变得较为稀少。

三星电子周三也表示，将暂时调整其在西安的NAND闪存芯片制造厂的运营，这些芯片用于数据中心、智能手机和其他科技装置的数据存储。

中共国官员从12月23日开始对西安市内以及离市旅行实施严格限制。

美光在博客中说：“我们正在利用我们的全球供应链，包括我们的分包商伙伴，帮助满足我们的客户对这些DRAM产品的需求，我们预计这些努力将能够满足大部分客户的需求，但在我们启动我们的网络时，近期可能会有一些延迟。我们正在努力将病毒传播的风险降到最低，并采取了包括物理距离和现场测试在内的措施。”

据分析机构TrendForce称，三星在西安有两条生产线制造先进的NAND闪存产品，占其NAND闪存总产能的42.5%，占全球总体产能的15.3%。

首尔的分析师表示，三星西安NAND工厂生产的芯片主要发货到中共国内市场，向海外市场的出货量有限，对该工厂生产的芯片的一些最大需求是来自中共国的服务器公司。

三星在10月底的财报电话会议上表示，进入7-9月的季度时，其NAND芯片库存较低，并打算在该季度将库存水平恢复正常。预计该公司将在1月份公布10-12月的业绩。
