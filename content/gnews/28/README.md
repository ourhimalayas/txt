我们被告知实验性的中共（新冠）病毒疫苗可以“阻止病毒传播”。

然而，最近公布的州政府数据显示，俄勒冈州有620多名居民尽管已经完全接种了中共病毒疫苗，但还是死于感染该病毒。

俄勒冈州卫生局的统计数据显示，自疫情爆发以来，共有622名完全接种了中共病毒疫苗的居民死于该病毒。

在死于中共病毒的患者中，314人接受了辉瑞疫苗，167人接受了摩德纳疫苗，84人接受了强生疫苗。

周二（12月21日），乔·拜登花了22分钟谴责未接种疫苗的美国人伤害了已接种疫苗的人，并声称未接种疫苗的人最容易死于中共病毒。

拜登说：”如果你没有完全接种疫苗，你有充分的理由感到担忧。在过去的许多个月里，几乎所有死于中共病毒的人都没有接种过疫苗，都未接种疫苗。”

他对未接种疫苗的美国人说：“你将其他人置于危险之中——你的亲人、朋友、邻居、陌生人，你可能认为你只是在让自己处于危险之中。但是，这是你的选择，你的选择不仅关乎你自己，还影响到其他人，你正在将其他人置于危险之中，你的亲人、朋友、邻居、你遇到的陌生人，而你的选择可能决定生死，病毒传播的时间越长，风险就越大，可能的病毒变异形式可能比以前出现的更致命。”

拜登的白宫甚至在周一（12月20日）发表声明，威胁未接种疫苗的人，如果他们在今年冬天继续未接种疫苗，他们将患重病并导致家人死亡。
