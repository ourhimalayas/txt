## 朝鲜有能力继续增加铀产量
` 纽约香草山MOS02` [轉載自GNews](https://gnews.org/zh-hans/1645571/)

作者：纽约香草山农场 – 柳是多水土透

![](https://assets.gnews.org/wp-content/uploads/2021/11/Screen-Shot-2021-08-30-at-7.37.59-PM.png)



斯坦福大学国际安全与合作中心(Center for International Security and Cooperation)和亚利桑那州矿业咨询公司的顾问通过对卫星图片的分析，在Science & Global Security期刊上发表研究报告，认为朝鲜目前的铀产量仅是其潜在产能的一小部分。

![](https://assets.gnews.org/wp-content/uploads/2021/11/捕获4-1.png)

川普总统破冰之旅后，朝鲜从2017年主动停止了核试验。但最新研究显示，朝鲜能够制造的核弹的裂变材料多于之前的评估，金正恩政权拥有加速核武最初阶段生产的潜力。

此前北韩唯一被证实正在运作的铀矿综合体位于朝鲜非军事区以北约30英里的平山县，用于生产粗提纯铀矿，又名“黄蛋糕”。粗提纯铀是核燃料提纯的关键原材料，当被加载入5兆瓦反应堆后，初级裂变副产品即可再处理为武器级钚。

朝鲜曾公开声称将继续扩张其核武器库。

另有卫星图片显示，朝鲜最近升级了其延边核电站反应堆，学者认为者可以用于制造武器级高纯度铀.

关于北韩的核计划情报有限，国际科学与安全机构主席David Albright认为其产能为每年4-6枚核弹头。国际原子能机构称“朝核武计划正全速前进，包含钚分离，铀浓缩和其它活动”。
