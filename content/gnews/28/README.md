**摘要：**

7月4日法国《世界报》刊发保加利亚籍欧盟议员伊尔汗·库楚克评论文章。自从库楚克为新疆维吾尔人发声之后，他成了中共当局的封杀对象。文章中他讲述了中共国的历史以及西方国家对中共政权的错误认识。

从1921年7月1日到2021年7月1日，100年过去了，在中国大地上从来没有过真正的改朝换代。

18世纪初期，一个叫毛泽东的图书管理员，从最初的政党领导人变成了毫无人性的军师，最后演变成了“受人爱戴”的独裁者。

21世纪，深受毛思想熏陶的习近平，把毛泽东那套老办法重新用了起来。先是个人崇拜，又推出“文革2.0”打压政治对手、异见人士和企业业主。中国共产党的历史就是一部整人历史。从被毛打压的陈独秀、彭德怀、林彪到现在被习用反腐名义打掉的江派、胡派的人马，更不用说数不胜数的民主人士。

中国共产党对维吾尔人的迫害和种族灭绝，逐渐被西方自由国家所了解，纷纷采取行动对中共进行制裁。2021年3月，欧盟对中共国参加迫害维吾尔人的集体和个人进行了制裁。中共国立即反制，制裁了包含作者再内的15名欧盟人士。

从2012年习近平执政以来，中共国的野心极度膨胀，不再掩饰称霸世界的美梦。通过“一带一路”中共国在亚洲、拉丁美洲、欧洲一路撒网，出现了一个个永远无法偿还债务陷阱。中共国成了别国想躲都躲不开的恶梦。

现在西方社会是时候明白，一个依靠数以千万计的监控摄像头控制社会的独裁政权，从20世纪20年代的暴力开始，到1989年镇压天安门民主运动，现在又通过扼杀香港的任何一丝抗议活动和武力威胁台湾，是对世界和平的威胁。

我们必须用勇气和清醒来武装自己，避免重复过去的错误。
