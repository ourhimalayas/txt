据加拿大法语媒体《魁北克新闻》11月8日报道，美国政府于当天凌晨，向那些完全接种了“新冠病毒疫苗”的加拿大旅行者重新开放了其陆地边界，并由此结束了长达20个月的疫情限制封关措施。

自8月9日以来，加拿大边境已重新对来自美国已完全接种该疫苗的非必要旅客开放。该国总理特鲁多曾为此辩解称：联邦政府充分认识到，已完全接种疫苗者在第四波疫情中并不构成大的传播风险，这是一个几乎完全只影响未接种者的病毒浪潮。

此前，随着“新冠病毒三角洲”变体在美国迅速蔓延而导致该国疫情的恶化，总统拜登当时决定继续执行关闭美加边界的管制措施。尽管目前重新开放了大部分关口，但加拿大人通过陆路进入美国并非完全自由，因为当局将监测所有旅行者的“新冠病毒疫苗”接种状况，只有完全接种者才被允许入境而无须进行针对该病毒的阴性结果测试。
