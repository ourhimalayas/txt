尚卡拉·切蒂（Shankara Chetty）博士是南非的一名家庭医生，他在改善新冠病毒（Covid-19）的早期治疗效果而受到赞誉。切蒂博士认为，**大规模 “新冠疫苗接种 ”计划的目标是 “控制并杀死我们人口中的一大部分，而没有人怀疑我们被毒死”。**

在最近的一份声明中，切蒂博士解释，由新冠疫苗接种引起的死亡过程的设计方式无法追踪。人们会因这样或那样的问题而生病，而且症状范围很广，很难明确地把病症归咎于疫苗的接种。

切蒂博士表示：“接种新冠疫苗后的死亡事件将永远无法归咎于毒疫苗。疫苗副作用太多样化，数量太多，而且发病的时间范围太广，我们无法理解我们已经中毒了”。

切蒂博士称其已经成功地治疗了7000多名新冠病毒患者，“没有一人住院或死亡”。他也明显意识到政府对他提供给病人的早期治疗方案的审查运动。

“我认为围绕正在发生的事情的观点是极其重要的，”切蒂博士在一次Zoom电话会议上解释说。“我们需要了解（新冠疫苗）目的是什么。每个人都知道有不一致的地方，有胁迫，但我们需要了解为什么。为什么会有这种情况？”

在切蒂博士看来，似乎有一样东西导致了新冠病毒的死亡原因，而那东西就是刺突蛋白。

正如切蒂博士所描述的那样，这种 “病原体”要么存在于（强生公司、阿斯利康公司），要么由（辉瑞-生物技术公司、莫德纳公司）的所谓“疫苗”生产。而正是这种病原体最终将杀死数百万，甚至数十亿人。

只有少数人对 “最精心设计的毒素”立即出现 “过敏反应”，他说，通常是在症状出现后的头八天。然而，疫苗接种会将这种过敏反应 “延长”一段时间。

“我们开始看到这种新冠疫苗以其刺突蛋白造成的内皮[血管内壁]损伤，以其对ACE2受体的影响，”切蒂博士警告说。“”这些是注定要发生的死亡。而且它们永远不会被归咎于刺突蛋白上，这是一种精心设计的毒素。”

“刺突蛋白也是一种膜蛋白。因此，mRNA将把这个蛋白分布在我们的全身。它将在我们身体周围的各种组织中复制。刺突蛋白将被整合到我们身体周围的那些膜中，以及那些特定的组织。”

由于这些刺突蛋白被身体识别为外来入侵者，免疫系统以自身免疫的方式作出过度反应–意味着免疫系统会攻击自己。这就是缓慢死亡过程开始的地方。

他警告说：“从长远来看，这种毒素会让原本就有疾病的人的病情加重”。

更糟糕的是，这些毒素包括 “少量HIV蛋白”，这清楚地表明它们是经过基因改造的，切蒂博士说。患有癌症的人“会让他们的癌症发作，他们会说他们死于癌症”。

“有血管损伤或易患疾病的人，如我们的糖尿病患者和[那些]高血压患者，将在不同的时间出现中风和心脏病发作，而其他人将把这些归因于他们先前存在的状况，”切蒂博士补充说。

“随着时间的推移，人们将会患上自身免疫性疾病，任何药物干预都无法解决其多样性问题，因为它们的针对性太强了。”
