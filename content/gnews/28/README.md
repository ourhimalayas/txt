中国欧盟商会近日发布一份调查称，部分外资企业担心在中国的“十四五”规划中参与度受到限制。对此，中共官媒“经济日报”刊发专题文章回应称，外资企业在中国市场的参与度不仅不会降低，反而会迎来更多发展机遇。

文章称，当前国际形势下，部分外资企业对在华发展环境有担忧，是可以理解的。不过，外资企业中对中国市场持乐观态度和坚定信心的是多数、是常态，尤其是在新冠肺炎疫情之下。相信只要外资企业紧紧把握中国经济变化的趋势和特点，必定会获得更广阔的发展空间。

多年来，外资企业对中国经济发展作出了重要贡献。如今，外资企业以占全国各类企业总数约2%的比重，创造了全国约2/5的对外贸易、1/6的税收收入，直接或间接拉动了1/10的城镇就业。可以说，没有外资企业就没有中共国近30年的发展，也就没有中共政权执政的物质条件。

当前，中共在政治、经济上已经全面转向，特别是在经济上全面推进国有化， 做大做强国企的同时，不断打压民营企业和升级对外企的限制，导致外企发展环境持续恶化。鉴于中共作死的德性，外资企业在中共国的境遇只会愈发恶劣，不断撤离将是常态。
