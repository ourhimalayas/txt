## 欧盟为谷歌、Meta 制定新的网上规则以遏制非法内容

布鲁塞尔：根据欧盟国家和欧盟立法者周六（4月23日）达成的新互联网规则，Alphabet旗下的谷歌、Meta和其它大型网络平台将不得不采取更多措施来处理非法内容，否则将面临巨额罚款。

经过16个多小时的谈判后，该协议达成。数字服务法 (DSA) 是欧盟反垄断负责人玛格丽特·维斯塔格 (Margrethe Vestager) 约束 Alphabet旗下谷歌、Meta和其它美国科技巨头的战略的第二个方面。

上个月，她赢得了27个国家和立法者的支持，制定了名为“数字市场法案”(DMA) 的具有里程碑意义的规则，这可能会迫使谷歌、亚马逊、苹果、美达和微软改变其在欧洲的核心商业惯例。

维斯塔格（Vestager）在推文中说：“我们就数字服务法DSA达成协议：《数字服务法》将确保线下的非法行为也被视为在线非法行为——不是口号，而是现实。”

欧盟立法者迪塔·查兰佐娃（Dita Charanzova）八年前曾呼吁制定这样的规则，她对该协议表示欢迎。

“谷歌、Meta和其它大型在线平台将不得不采取行动更好地保护它们的用户。她在一份声明中说：“欧洲已经明确表示它们不能充当独立的数字岛屿。”

谷歌在一份声明中表示：“随着法律的最终确定和实施，细节将变得很重要。我们期待与政策制定者合作，确保剩余的技术细节正确无误，以确保法律适用于所有人。”

根据数字服务法（DSA），这些公司因违反规定将面临高达其全球营业额 6%的罚款，而屡次违规可能会导致它们被禁止在欧盟开展业务。

新规则禁止针对儿童或基于宗教、性别、种族和政治观点等敏感数据的定向广告。黑暗模式，即误导人们在网上向公司提供个人数据也将被禁止。

广大的在线平台和在线搜索引擎将被要求在危机期间采取具体措施。此举是由俄罗斯入侵乌克兰和相关的虚假信息引发的。

这些公司可能被要求向监管机构和研究人员交出与它们的算法有关的数据。

这些公司还面临每年高达0.05%的全球年收入的费用，以支付监控其规范性的费用。

欧盟立法者马丁·施德万（Martin Schirdewan）批评给予中等规模公司的豁免的规定。

他说：“在保守派的压力下，中型公司的例外规则被整合，这是一个错误。由于在数字领域有大量公司属于这一范畴，例外就像一个漏洞。”

数字服务法（DSA）将于2024年执行。
