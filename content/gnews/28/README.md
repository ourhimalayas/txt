## 【中日财经爆】腾讯参股英国数字银行 中共数字人民币正式布局欧洲

撰稿：东京樱花团/喜马拉雅的微尘
![](https://assets.gnews.org/wp-content/uploads/2022/01/image-152.png)[图片来源](https://www.toptiao.com/8413.html)
路透社2021年12月31日消息，英国数字银行Monzo12月31日表示，中国游戏和社交媒体公司腾讯控股有限公司(Tencent Holdings Ltd.)[0700.HK](https://www.reuters.com/companies/0700.HK)已持有Monzo的少数股份。这是该银行融资计划的一部分，几周前其估值达到了45亿美元。

Monzo没有披露腾讯持股的细节，但有媒体援引一位知情消息人士的话报导称，腾讯正在进行1亿美元的少数股权投资。

Monzo成立于2015年，是一家备受瞩目的新银行，拥有500多万客户，但一直难以盈利，去年由于披露面临潜在的民事和刑事洗钱调查，年度亏损扩大。

本月早些时候，Monzo获得了阿布扎比增长基金(Abu Dhabi Growth Fund)和Coatue等投资者的注资。

海外数字银行是中共数字人民币国际化的重要依托。作为中共权贵互联网巨头的腾讯公司，此次入股英国数字银行Monzo，表明中共数字人民币已经将正式布局欧洲。
