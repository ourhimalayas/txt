## [ 聚焦港澳 ] 从“洪门宴”看香港政治乱象
` 澳喜农场` [轉載自GNews](https://gnews.org/zh-hans/1970510/)

撰稿：云彩

2022年1月3日，Omicron冠病变种毒株在香港社区传播之际，逾30名政府高官和立法会议员被爆出席洪为民百人生日派对，引发社会哗然。

香港时事评论员刘细良先生认为，从这事件看出，香港沦为大陆村官东莞式治理，爱国者治港是一场闹剧。

洪为民是前海管理局香港事务首席联络官。前海项目是被中共列入十四五规划的重头戏，其业务涉及土地、金融、创新科技等，与参与“洪门宴”14个政府官员负责的范畴相似。

为何这些问责官员要出席洪为民的生日宴会？因为他背后的老板，是中共在香港的权力代表。现在政府官员的升迁靠的是拉关系。

刘细良先生以民政事务局为例。在港英政府时期，民政事务局扮演重要角色，是政府与下面社会的沟通桥梁。经过基层磨练的民政专员凭工作表现提拔到政府重要部门。如前特首曾荫权曾是沙田区的民政专员。现在中共管制下的香港政府，选拔人不再看工作能力而是政治挂帅，利益输送。

林郑月娥对此事的处理看得出她在借机将政治对手打下去。所谓爱国者内部的矛盾将会是香港的主要矛盾，内斗。

笔者很痛心看到香港迅速大陆化。如果共产党继续存在，西方法治社会势必迅速香港化。幸好，这个世界有新中国联邦和许多正义的人们一起来灭共！
