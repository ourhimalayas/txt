## 韩国庆北安东市一家酒吧发生了青少年集体感染，防疫进入了紧急状态
` 首尔天池农场` [轉載自GNews](https://gnews.org/zh-hans/1924609/)

翻译：鱼子酱
校对：势在必行
编辑：追着曙光跑

据韩国《N News》1月27日的报道，庆北安东市27日透露，安东市玉洞的一家酒吧共发现了74名新型冠状病毒确诊患者。 其中23日4名、24日19名、25日35名。另外，58名是10多岁的学生，16名是从业人员和其他客人。
![](https://assets.gnews.org/wp-content/uploads/2022/01/WhatsApp-Image-2022-01-29-at-01.40.23.jpeg)图片来源：原新闻
经确认，这些人大部分是全程疫苗接种者，是属于“突破感染”病例。 进入酒吧时，也认证了全程疫苗通行证。 特别是10多岁的确诊患者，去年为准备高考而接种了疫苗。

74名确诊患者中,接种到第二批的人数为48人,接种到第三批的人数为25人,共73人,其余1名未接种者是与去过酒吧的家人接触后确诊的第二批感染者。

74名确诊患者中，第二批确诊人数里有48名患者是接种了疫苗，第三批则有25名，共73人。其余1名未接种者是与去过酒吧的家人接触后确诊，是属于第二批感染者。

另外，在记录当时状况的视频中，捕捉到了人们挤在一起喝酒，部分女性爬到桌子上跳舞的场面。 数十名客人大部分没有戴口罩，也没有遵守保持社交距离规则。

一位在相关酒吧的学生说：“高三学生都打了疫苗后入场的”，“没想到所有打过疫苗的学生都会被感染”。

防疫当局叮嘱道：“为了防止欧米克仑扩散,在制定对策的同时,在5天的春节长假期间,请克制移动,彻底遵守防疫守则。”
