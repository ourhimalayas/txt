我们必须接受现实，相信科学并承认没有气候紧急情况，不是吗？

本周卡尔加里市议会将就宣布我们城市进入气候紧急状态进行辩论，该动议是由议员拉吉·达利瓦尔提出的，但被广泛认为是市长新人乔蒂·贡德克的项目。

上周，该议案获得一致通过辩论，表明气候危言耸听是一种意识形态叙事，现在已经清醒过来的市议会显然已经接受了。温哥华、哈利法克斯和埃德蒙顿等其他城市都处于自我描述的气候紧急状态，无论这意味着什么。

卡尔加里市议员皮特·德蒙承认，这一声明是一个象征性的声明，也是另一项毫无意义的进步政策，它的作用无非是为政客们提供了一个自我安慰的机会。

对不起，德蒙先生把它打破了，没有象征性的紧急情况。

卡尔加里建立在石油和天然气之上。乔蒂曾经是我们清洁和符合环境道德的能源部门的大力支持者，但不要相信我的话：我们已经收录了她对加拿大能源部门预告和赞美的片段。

乔蒂现在是一个城市的市长，该城市有三分之一的市中心空置，随着钻探的增加和潜在的石油繁荣即将到来，卡尔加里可能会随着经济的激增而重新焕发活力，她现在正在拒绝加拿大资源。无论她宣布什么紧急状态，人们仍然需要石油和天然气，因此通过谴责我们的道德资源，她实际上是在支持欧佩克的。

除了经济和能源方面的考虑，这里还有一个更令人担忧的问题，那就是我们孩子的福祉。环保主义者声称要为子孙后代保护世界，但在此过程中，他们创造了一个充满恐惧和歇斯底里的环境，以至于孩子们急于生活在今天的世界上。由于管道取消和反石油游说者而失去工作的父母的综合压力，以及世界即将结束的不断传递的信息，无疑助长了我们年轻人的心理健康。
