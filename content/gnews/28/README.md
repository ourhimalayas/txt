## 上海震旦职业学院教师宋庚一因发表“错误言论”被开除
` 澳喜特战旅Gnews` [轉載自GNews](https://gnews.org/zh-hans/1770646/)

撰稿：菜园子112～尕虎
![](https://assets.gnews.org/wp-content/uploads/2021/12/191.png)图片来自网络
本周，上海震旦职业学院一位老师的言论突然在互联网被广泛传播，并迅速引起墙内网民的热议。

根据该学院自己的说法，东方电影学院一位名为宋庚一的教师于2021年12月14日下午《新闻采访》课程中发表错误言论，造成重大教学事故和严重不良社会影响。根据《上海震旦职业学院关于教学事故认定及处理的办法》和《上海震旦职业学院教职工处分暂行规定》，学院决定给予这位老师开除处分。

所谓错误言论，是指这位东方电影学院的宋老师，在讲解南京大屠杀时说：“这段历史，如果没有史料支撑的话，那也可能只是民间说说而已，说得和真的一样……我觉得不应当永远去恨，而应当反思战争是怎么来的。”

上海震旦职业学院同时表态将高度重视师德师风建设，举一反三抓好抓实教育教学管理，严肃课堂政治纪律和行为准则，对违规违纪的行为始终坚持“零容忍”态度，一经查实，绝不姑息。

宋庚一老师被学校开除后，在墙内网络上引起了轩然大波，网友们纷纷发表个人看法和观点，大多数人都支持宋庚一老师，谴责举报人董讯。然而官媒环球时报却认为董讯并非“告密者”，相反将其称作“吹哨人”，可谓极为厚颜无耻。

举报者学生董讯直接毁掉了一个老师，学校领导间接毁了一个老师，我们再来一齐毁掉一个学生？我们为什么不去一齐毁掉一个毁人不倦的规则？我们为什么热衷于毁掉一个人而不是毁掉一个将人导向恶的规则？这是我们应该认真思考的。
