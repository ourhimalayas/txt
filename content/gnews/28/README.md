据德国之声7月22日报道，在东京奥运会开幕式前一天，奥运会总指挥桥本诚子解雇了开幕式总导演小林贤太郎，原因是这位前喜剧演员在1998年的喜剧节目中开了一个关于犹太人大屠杀的玩笑。

在作曲家小山田圭吾因欺负残疾同学而被迫辞职后，仪式已经计划在没有开幕音乐的情况下进行。
