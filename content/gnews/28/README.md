西方社会就三件事儿, 就是它这个社会出大问题了。
第一∶你看到超市供应，因为西方这个信息极度发达的社会，就你现在的感受就是开始要出问题了，当超市要出问题的时候，基本上这个社会就会出现政治经济社会安全的问题，一定是结局。

当你看到股票市场的这种起伏，暴跌暴涨的时候，它就是有人要操纵这个东西了。
这三件事情基本上就是让你面临这个3年到8年的一个困难期。

不是说你家准备点柴火，不是你们家准备点面包就能过的。它一定是3到8年。
西方的经济在正常情况下每8年一次重新的一种经济危机进行调整，泡沫吹灭再恢复一次。那时候是调整期在3年到8年, 但这一次已经不是了，这一次是经济的调整周期，加上人为的一场人类大灾难和整个人类的共产主义，和社会主义和资本主义的大比拼，和人类大灭亡和大拯救的一个全面开始，可能30-50年都有可能，3年5年10年调整期。
