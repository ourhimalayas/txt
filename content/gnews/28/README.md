刚宣布未接种疫苗者将失去所有基本人权的一天后，11月21日，奥地利更进一步，裁定整个国家将失去基本人权。

这次将实施全国范围内的封锁，以控制疫情爆发。这还有意义吗？另外，专家们“感到困惑”的是，在没有接种疫苗的非洲，那里没有封锁，只有极少的柯瓦伊病毒病例，死亡人数只是欧洲和美国人均死亡人数的一小部分。

上周五美联社的一份报告中指出，去年冠状病毒首次出现时，当地卫生官员担心这种大流行病会席卷整个非洲，导致数百万人死亡。尽管目前还不清楚新冠疫情的最终死亡人数，但这种灾难性的情景在（最穷国家）津巴布韦或非洲大陆的大部分地区还没有出现。

美联社承认，**非洲的情况太重要了，不容忽视，注意到非洲基本上摆脱了大流行病的深刻影响，令科学家和卫生专家感到困惑。**

但是，哥伦比亚大学全球健康主席瓦法·萨德尔说，在非洲（总）有一些“神秘”的事情发生，让科学家感到困惑。她说：**“非洲没有欧洲和美国那样的疫苗和资源来对抗新冠病毒，但不知为何，他们似乎做得更好。”**

在非洲，**只有不到****6%****的人接种了疫苗。**几个月来，世卫组织在其每周的大流行病报告中将非洲描述为“世界上受影响最小的地区之一”。

对西方官方说法的许多“怀疑论者”早就指出，年轻人被证明风险要小得多，另外，阳光和维生素D也有帮助。

现在，全球疫情大流行已过去一年半，这次主流媒体报道承认了以下内容，“一些研究人员说，非洲大陆的年轻人口——平均年龄为20岁，而西欧约为43岁；除了非洲较低的城市化率和花时间在户外的倾向外，到目前为止，可能使他们免受病毒的致命影响。一些研究正在探究是否可能有其他解释，包括遗传原因或过去感染寄生虫病。”

对照一下，特别是在大流行病的前半年，西方国家出现了警察在“封锁”中把人们从公共海滩上赶走的场景，以及父母因为让他们的孩子在公共露天场所进行户外活动而在公园被警察逮捕。
