![](https://assets.gnews.org/wp-content/uploads/2022/01/Screenshot-2022-01-04-201034.jpg)
根据2021年12月30日新浪报道，  北京冬奥会将推动数字人民币试点应用，用户可以通过多种渠道便捷地获取、开立数字人民币钱包，并在闭环区域内各个消费场景使用数字人民币。 据最新报道，数字人民币（试点版）App已上架至安卓苹果应用商店。

截至2021年10月22日，数字人民币个人钱包开立1.4亿个，对公钱包1000万个，累计交易笔数1.5亿余笔，累计交易金额约620亿元。2021年，中共国在北京、深圳、苏州、上海、等11地派发了累计金额超3.4亿元的数字人民币红包。

中国人民银行副行长范一飞表示，“北京冬奥会场景是下一步试点的重点领域。” 研究机构认为数字人民币将是冬奥会最确定的科技“名片”，此外数字货币是绿色低碳的货币工具和支付工具，非常符合中央经济工作会议指出的坚持节约优先，实施全面节约的战略。

央行数字货币研究所所长穆长春表示**，数字人民币不是一些人理解的加密资产，而是人民币的数字化，**它的定位就是现金的替代，跟现金一样是法定货币，相比较微信和支付宝，数字钱包里装的内容可能会增加央行数字货币，并且进行支付在使用时无需手续费，数字人民币的每一笔收付，相关信息都要同时发送央行进行被查账户的核查处理，数字人民币是以风险可控为前提的匿名，既保护用户隐私，又要符合反洗钱、反恐怖融资、反逃税等金融稳定要求，是“维护金融安全的客观需要”。

中国人民银行国际司司长朱隽指出，全球央行数字货币整体快速崛起，数字货币的国际合作也蓄势待发， 但是数字人民币重在提高国内零售支付的效率，尚无用于跨境支付方面的安排。

中共国央行数字货币研究所所长都说数字货币其实是数字化的法币，而且目前只能在中共国内流通，无法境外支付。中共国没有国家信用，没有人想要马上变废纸的人民币，数字化集中管理无非是更好的管理和收割中共国老百姓。
