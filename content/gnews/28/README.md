2022年伊始，恒大又遭暴击。1月1日，中共国网络媒体曝光一则新闻，恒大在海南海花岛的39栋建筑被判为“违章建筑”，要求在十日内自行拆

一份海南省儋（dan）州市综合行政执法局的处罚决定书显示，儋州信恒旅游开发有限公司开发的，海花岛2号岛三 期项目共39栋楼，总建筑面积逾43万平方米。因该项目违法取得的规划许可证已被撤销，现责令相关单位在十日内自行拆除上述违建。

公开信息表明，信恒旅游是恒大童世界集团有限公司旗下子公司，注册资金8亿元人民币。海花岛项目号称世界上最大的花型人工旅游岛。此次处罚决定书涉及的建筑，均位于居住功能区的2号岛。
