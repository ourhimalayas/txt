当地时间12月14日，土耳其好党在安卡拉公布了一份关于新疆维吾尔自治区人权的报告书，当中指控了中共在新疆全面性侵害人权的行为。
![](https://assets.gnews.org/wp-content/uploads/2021/12/6-29.jpg)
好党主席阿克谢纳尔在报告中指出，中共在新疆地区设立的“集中营”分为5类，包括临时拘留中心、集中营、奴役劳动中心、儿童集中营和监狱。其中，最基层的临时拘留中心遍布于维吾尔自治区的各个大小城巿。

报告中还提到，新疆集中营内的生活状况之恶劣，实为自第二次世界大战后的仅见。当中，维吾尔妇女被制度化的性侵，受害人也被施以残酷刑罚，甚至儿童也被强制与父母及原属文化进行隔离，成为中共系统性“汉化”计画的牺牲品。

阿克谢纳尔公开宣示，倘若任何个体或机构赞同中共残暴的行为，便与自身施行残暴无异。 因此，如果任何人对于新疆人遭受中共残暴迫害的现状默不作声，其就等同是施暴者中共的同路人。
