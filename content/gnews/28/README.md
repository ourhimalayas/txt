## 海地总统约韦内尔莫伊塞在家中被暗杀
` 索尼克` [轉載自GNews](https://gnews.org/zh-hans/1379590/)

翻译：神奇四侠

美联社7约7日报道：海地总统约韦内尔-莫伊塞在家中被刺杀，总统的妻子也被波及，正在医院接受治疗，临时总理克劳德-约瑟夫（Claude Joseph）证实了这起刺杀，并表示现在海地政府已经控制住局面。

莫伊塞继任的前总统米歇尔-马尔泰利说，他正在为第一夫人玛蒂娜-莫伊塞祈祷，并称这次暗杀 “对我们的国家和海地的民主是一个沉重的打击，海地正在努力寻找自己的方向。”

临时总理克劳德-约瑟夫说：“一些袭击者说的是西班牙语，但没有提供进一步解释。国家的安全局势在海地国家警察和海地武装部队的控制之下，我们将赢得胜利。”

白宫发言人詹-普萨基（Jen Psaki）在接受MSNBC采访时说：“这次袭击是可怕的和悲惨的，我们仍在收集有关情报。总统拜登（Joe Biden）将在周三晚些时候听取国家安全团队的汇报，我们随时准备并站在他们身边，提供任何需要的援助。”
