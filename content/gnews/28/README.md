## 比利时健将哭诉被隔离引关注 中共迫于压力妥协

![](https://assets.gnews.org/wp-content/uploads/2022/02/GettyImages-1238056142-700x420-1.jpg)2022年1月29日，布鲁塞尔机场，比利时俯式冰橇运动员金·梅莱曼斯在参加北京2022年冬奥会的比利时队运动员出发时的照片。(盖蒂图片)

比利时俯式冰橇明星金·梅莱曼斯2月2日在社交媒体视频中含泪哭诉，尽管多次中共病毒COVID-19检测呈阴性，但她还是被隔离了三天，并担心自己无法参加比赛。

这位25岁的选手在2月3日发布了最新进展，说在国际奥委会介入后，她获得北京奥运村一个房间。

梅莱曼斯在2月3日的Instagram视频中说：“我现在在一个与世隔绝的房间里，但至少我回到了奥运村。我感到很安全，我在这里能够进行更好的训练，谢谢大家。”

目前，梅勒曼斯作为中共病毒COVID-19患者的密切接触者，住在奥运村的一个单间里，她可以在这里训练，但必须独自乘坐交通工具，并独自吃饭。

**一段哭诉视频**

2月2日，梅莱曼斯在Instagram上发布了一段在隔离酒店录制的视频，她在视频中哭着求救，因为北京政权将她关进隔离室，并告诉她必须在那里隔离至少七天，也不能做适当的训练。

“我应该在这里再呆七天，每天做两次PCR测试，而且不能与其他人接触，”梅莱曼斯在视频中眼含泪水地说。“我求大家给我一点时间考虑下一步行动。因为我不确定我是否能够再承受14天时间，现在的奥运比赛是这种隔离状态。”

梅莱曼斯于1月29日与她的比利时奥运同伴一起飞往北京。因为到达时的中共病毒PCR测试为阳性，她在进入中国后被隔离，并被告知，在测试结果呈阴性的三天后，才允许她进入奥运村，并将被视为测试阳性者的密切接触者。

这名比利时运动员在隔离中心的随后几天检测结果都为阴性，但当局拒绝释放她，而是将她转移到另一个隔离酒店进行进一步的所谓隔离。

梅莱曼斯在她的社交媒体账户上提到，她在1月早些时候检测出阳性。但后来在国内进行了12次检测，结果都是阴性，因此她被允许飞往中国参加比赛。

![](https://assets.gnews.org/wp-content/uploads/2022/02/GettyImages-1231115057-600x371-1.jpg)2021年2月12日，在德国东部的阿尔滕贝格，比利时选手金·梅莱曼斯在完成国际雪联雪橇世界锦标赛女子俯式冰橇比赛的第四轮比赛。(盖蒂图片)

**国际奥委会介入**

国际奥委会于2月2日对梅莱曼斯的视频作出回应。

国际奥委会表示：“当国际奥委会在她到达酒店后了解到她的个人情况时，立即与比利时国家奥委会取得联系，以确保迅速向她提供适当的支持。中共病毒患者亲密接触者可以在奥运村进行训练和比赛，并生活在奥运村。”

国际奥委会说，梅莱曼斯被隔离在第二家酒店是一个“临时”解决方案，因为在奥运村“没有这样可以直接使用的房间”。女子俯式冰橇项目将于2月11日在延庆国家滑雪中心举行，该中心的正式训练赛将于2月7日开始。
