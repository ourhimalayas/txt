6月29日，江苏省基督教两会主办的“[讴歌百年路 永远跟党走——江苏基督教庆祝中国共产党成立100周年歌舞音乐会](https://www.ccctspm.org/newsinfo/14606)”在南京基督教圣训堂隆重举行。

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/07/01100935/%E5%9B%BE%E7%89%8728.png)图片来源：网络

据悉，该庆祝活动有江苏省委统战部、民宗委领导以及中国基督教协会驻会副会长沈学彬牧师等出席。江苏各市教会主要负责同工以及基督徒演职人员280多人共襄庆典。

庆典晚会隆重，堪比平安夜盛况。演出节目有《没有共产党就没有新中国》《江姐》《太湖儿女》《十送红军》《黄桥烧饼歌》；以及“带我们回到战火纷飞的峥嵘岁月”，《映山红》《红梅赞》《永远跟你走》，“一首首耳熟能详的经典红歌，勾起革命成长的共同记忆”。

唯一能勾起中共国基督徒回忆的似乎只有十年文革中，中共对基督徒的逼迫。20世纪50年代的基督徒在那场逼迫中至死不渝、背起基督的十字架走上割他头颅地的信心先贤数不胜数，颇有今日香港年轻基督徒那样的勇气，他们是将“赞美诗变成抗议颂歌”，而不是相反。

如今的中共国基督徒见风就倒，完全丧失了灵性的力量。这就是中共政权利用和平时期向基督的教会渗透、控制的结果，使整个中共国教会处于[温水效应](http://sj.jidujiao.com/lockey.php)中失去灵魂。上述撒旦的颂歌说明了一切。这些和教会完全水火不容的歌曲被请进教会，显然，中共治下的教会完全失去了灵魂，使基督徒信仰的核心——敬拜上帝——被倾塌，而永远跟随基督却变成了“永远跟党走”。这种公然违背基督教导、敬拜撒旦的做法实属罕见。

作为基督教协会，一定要去讴歌共产党吗？还是被逼迫？根据中国基督教协会官网讯息看，中共国全国各地都在做此类庆典活动，一个显著特点是，基督教届敬拜中国共产党的活动都必须在7月1日前完成。中共似乎在担心在7月1日党庆日有基督徒自发的抗议活动。

据中共国某位牧师透漏，教会领袖变得越来越贪恋权势、而非教会事务，他们见到官员总是卑躬屈膝，见弟兄姊妹总是“我很忙”。听起来，宗教领袖似乎无须逼迫就跪拜在血旗下效忠。却完全忘记基督的教导说：“唯有胆怯的、不信的、可憎的、杀人的、淫乱的、行邪术的、拜偶像的和一切说谎话的，他们的份就在烧着硫磺的火湖里，这是第二次的死。”（[启示录 21:8](http://sj.jidujiao.com/lockey.php)）上帝曾指出：拜偶像的罪和奸淫、胆怯、不信的罪同等。
