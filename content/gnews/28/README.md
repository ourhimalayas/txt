## 多伦多消防局多名接种疫苗员工感染新冠病毒

多伦多第一消防局接种疫苗的多名工作人员感染了新冠病毒。

Canada.com 于12月11日报道，几个月前因拒绝强制接种疫苗，205名多伦多警察局工作人员被迫停职。但近期警察局的交通服务处以及11分局相继出现了新冠疫情爆发的现象。

多伦多警方发言人康妮·奥斯本说：“我们知道 11 分局和交通服务处有阳性病例。”

奥斯本说，“深度清洁、穿戴个人防护装备，持续及时地更新有关公共卫生命令和政府指令的信息以及实施强制性疫苗接种政策至关重要”。

一位消息人士称，在11分局的工作人员，至少有一名家庭成员的检测结果也呈阳性。

在交通服务部门，一整个重建小组的工作人员都被送回了家。

这里需要声明的是这些被感染的员工都是接种了双剂新冠疫苗，也就是完全接种的人。

一些未接种疫苗并且被迫停职无薪休假的工作人员也向多伦多警察局提出了一个问题：“如果接种疫苗的人都被感染了，为什么不让我们去工作呢？”
