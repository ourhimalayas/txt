最近国内网络疯传某小区业主群关于断供问题的讨论，根据讨论内容中关于冬奥会及高铁相关描述可以判断该小区位于河北张家口市某小区。业主群讨论内容显示上月5号楼十几户选择断供，不少业主对房价上涨完全失去信心，房价已经腰斩。
