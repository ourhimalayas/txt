据布赖特巴特新闻网报道，去年春天，欧盟委员会提交了一份气候法草案，并设定了到2050年实现气候中立的约束性目标。周四，欧洲议会成员以442票赞成、203票反对、51票弃权的表决结果通过了《气候法案》。 该法案计划将欧盟到2030年的碳排放从1990年水平的44%提高到55%。至此，欧洲议会已经正式确定了一个目标，即在2050年前使整个欧盟实现碳中和。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/28183420/2-52.png)
据瑞典电视台（SVT）报道，瑞典社会民主党议员纪特-古特兰认为，法案只会为欧洲和世界发挥越来越大的作用。瑞典中央党议员艾玛-威斯纳认为，法案还不够激进，但这些目标是一个很好的起点。

而保守派和民粹主义的欧洲议会议员们则大多反对该提案。波兰法律和司法议员安娜-扎列夫斯卡表示，通过的法案令人失望。目前没有钱来做这件事。没有适当的资金，就不可能实现公正的转型。气候法中包含强制和禁令，不仅会导致新的税收，而且还会增加日常生活的成本。

瑞典是欧盟中推动减排力度最大的国家之一。 但瑞典人在2019 年仍将气候变化支出视为纳税人资金的最大浪费。
