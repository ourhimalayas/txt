华盛顿 — 国际人权服务社（ISHR）7月15日发布的一份有关中共国在联合国经济和社会事务部（ECOSOC）不当影响力的报告指出，中共国利用其在联合国经社部独一无二的存在和影响力，确保设立标准和拨放资金都与北京的外交政策高度一致。

项目主任和中国问题专家、该报告的主要起草人萨拉∙布鲁克斯（Sarah M. Brooks）表示，这项研究展示出中共国政府谋求联合国正式和非正式领导位置，并将其国家优先目标与此结合起来。这包括寻求联合国对“一带一路”计划以及习近平的名言“人类命运共同体”的支持。

在一场新闻吹风会中，发言者谈到中共国与其他政府一道充当非政府组织进入联合国的看门人，与中共看法保持一致的组织才能进入联合国系统。中共国通过联合经社部推销“一带一路”计划，甚至通过联合国经社部分配资金，以此确保联合国对“一带一路”计划的政治支持。

报告建议各国政府和联合国对中共国政府的投资和影响力加深了解并积极回应；建议各国政府挑战中共国在联合国经社部的领导位置，并追踪联合国文件中可能出现的中共语言和概念。
