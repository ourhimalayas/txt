**摘要翻译：**

台湾海峡的两名官员申请加入跨太平洋贸易协定。北京方面试图阻止台湾的申请，而台北方面则捍卫其加入该区域协议的权利。

9月22日，中共政府向台湾防空识别区派出24架战机，这被理解为对台湾的参与施加压力的一种威胁。台湾政府重申它愿意不顾北京方面的威胁而介入，并表示它不会放弃。

中共国和台湾能否最终加入《跨太平洋伙伴关系全面进步协定》（CPTPP）？

决定将由11个CPTPP成员国做出，其中包括澳大利亚、文莱、加拿大、智利、日本、马来西亚、墨西哥、新西兰、秘鲁、新加坡和越南。

在被决定之前，他们每个人都会与申请人进行非正式接触。申请人只有在获得所有成员国的批准后才能加入商业伙伴关系。

尽管台湾是一个事实上的独立国家，有自己的军队、民选政府和宪法，但中共政府声称台湾自治岛是属于它的。
