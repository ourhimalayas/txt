![](https://assets.gnews.org/wp-content/uploads/2021/12/Unknown-38.jpg)中共监管再立禁区 国足球员不得刺纹身 网络图片
据美国之音12月31日报道，中共国体育当局要求国家足球队成员不要纹身，已经有纹身的要自己清除或者在比赛时自己遮盖。

习近平掌权后，扩大了对各行各业的管控，不仅收服了阿里巴巴、腾讯、百度等行业巨头、取缔了各种补习班，甚至学生每周上网打电子游戏的时间都有严格规定。如今，习太阳的光辉终于开始照耀中共国足球队。

众所周知，美元有国际长臂管辖权。东施效颦，中共国也开启了国内的“长臂管辖”，并且进入了个人生活领域。从头脑、思维、行动坐卧到球员纹身，也成了“公共问题”。中共国国家足球队自2002年在亚洲亮相世界杯后，一直未能再次入选。同种同根同在亚洲的韩国、日本则多次晋级。被太阳照耀后，中共国足球是否能突破，拭目以待。
