## 台湾新冠疫苗接种逾1340万剂 覆盖率即将破50% 副作用报告也如韩国一样攀升
` GTV新闻访谈` [轉載自GNews](https://gnews.org/zh-hans/1549317/)

栏目记者/责任编辑：AsG諾雅



![](https://assets.gnews.org/wp-content/uploads/2021/08/Gnews-Logo-29.png)





台湾疫情指挥中心医疗应变组副组长罗一钧在9月22日下午的疫情记者会中公布了中央疫情指挥中心最新统计，截至9月21日，累计接种新冠疫苗达1340万剂，涵盖率49.58%、剂次人口比57.16%。

同时，新增90件疫苗接种后非严重不良反应、68件其他疑似严重不良事件以及8例死亡通报个案。死亡个案中有6例和AZ疫苗相关，个案年龄为76岁至96岁间，于接种疫苗后1天到93天发生；另有2例和莫德纳疫苗相关，均为50多岁年轻族群，死因均为心脏疾病，1例为心肌梗塞，另1例为心因性休克、疑似心血管病变。

韩国也于19日星期日传出消息，自2月26日实施新冠疫苗大规模接种计划以来，全国接种首剂新冠疫苗的人数累计已经超过3600万人，接种率突破70%。然而，已经有将近24万例接种后产生不良反应的案例，为总施打人口的0.42%。各家药厂疫苗的不良反应报告率，强生疫苗为0.58%，莫德纳疫苗为0.52%，阿斯利康疫苗为0.51%，辉瑞疫苗则是0.35%。截至18日0时为止，已有922起通报死亡个案。随着施打率提高，接种后发生副作用案例也急速增加，光是15日至17日就有高达1万2703例。
