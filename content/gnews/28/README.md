![](https://lh3.googleusercontent.com/mv47IGB74jFPpGT5L7PES7MiGPqP5Aw5xrxmGYJykrxwAKlrY0dxP2E2lMzoeJfPJeD69c_yyhdbs2JS5eZzCmdkBVDwfkdNocevoQxAn8tyrn0mhDCXBNAD1kZQjPRkYG4fcA8h)[*图片来源*](https://www.gqjapan.jp/culture/article/20220215-beijin-olympic-2022)
北京冬奥会于20日落下帷幕。除了兴奋剂、问题裁判等竞技方面的质疑外，还有中共式的“管控”，将任何“不合宜”的东西屏蔽掉。当地的气氛与尊重自由人权的“和平盛会”相去甚远。

举办期间，在主媒体中心（MMC），当电视上出现针对新疆问题维吾尔族人的国外抗议活动相关报道时，节目被屏蔽的情况屡见不鲜，这意味着中共国的新闻管控也适用于奥运场馆。

在奥运会开幕前，北京冬奥组委会表示，将保障外国记者的权利，但任何违反中共国法律法规的行为或言论都将受到惩罚。事实上，不符合当局意愿的行为随时随地会被管控。

火炬传递的最后一名选手，中共国维吾尔族运动员，在比赛结束后并没按惯例出现在记者采访区。但第二天，他告诉新华社记者，很高兴能参加接力赛。据了解，中方在想方设法阻止外国记者询问有关人权的问题。

当问及志愿者对比赛的看法以及对人权问题的坦率意见时，往往被拒。甚至被反问“你提前申请了吗？”或者“这不是一个好问题”。志愿者有可能经历了某种“培训”，因为他们必须经过面试和培训才能成为工作人员。

举办期间，运动员在社交网站（SNS）上也甚少交流，看来，面对中共国无形的控制管理，运动员也无言以对了。事实上，运动员回国后纷纷指责让人权问题劣迹斑斑的国家举办奥运会是极不负责任的。

此外，一名芬兰运动员发布了一段奥运村大量漏水的视频，该视频很快被删除。美国媒体指出，是当局要求运动员删除视频。

一位德国记者质问，比赛为谁而办？批评声音完全被压制的冬奥会，毫无和平气氛可言。
