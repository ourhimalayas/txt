10月9日，《华盛顿自由灯塔》（Washington Free Beacon）发布文章，报道说早在美国从阿富汗撤军时，中共黑客就入侵了阿富汗一家大型电信公司，表明北京试图在该国获取立足点。

全球著名网络安全研究公司（Insikt Group）一份报告显示，今年早些时候，至少两名中共支持的黑客入侵了阿富汗电信发展公司（Roshan）。在8月和9月，随着美军开始撤离阿富汗，中共黑客组织加大了入侵力度。据研究人员说，这一入侵可能使得中共有权限访问有关网络本身和高价值个人的“极其宝贵”的信息资料。

文章分析认为，阿富汗是中共国试图扩大其影响力的最新国家。在美国，中共国已经成功向耐克、苹果和NBA施压，要求他们对中共国侵犯人权的行为闭嘴。在亚洲，中共国采用了“债务陷阱外交”，以灾难性的利率向小国贷款，从而在他们无法偿还时进行资产扣押。

尽管中共国每年在“一带一路”倡议外交上花费500到1000亿美元，利用基础设施项目扩大在所在国的影响力。但是，文章作者认为，鉴于阿富汗地区的偏远、贫穷和不稳定，它不太可能成为“一带一路”项目的目标。根据美国战略与国际研究中心高级研究员乔纳森·希尔曼（Jonathan Hillman）的说法，中共国可能将数字（黑客）攻击视为在阿富汗获得立足点的更有效途径。他说：“中国政府可能将阿富汗的主要电信供应商视为收集情报与监测政治和安全风险的重要途径”。

作者认为，这种方式的行动可以帮助中共国检测阿富汗与新疆地区边界附近的活动，因为中共国在新疆地区维持着一个集中营网络。目前，塔利班已经将维吾尔人的士兵从该地区撤离，这被视为塔利班对中共国的一个恩惠。但是，文章指出，阿富汗维吾尔人担心，随着塔利班政权的巩固，他们可能会被驱逐回中共国。

报告研究人员指出，中共国的情报机构非常庞大，黑客组织之间很可能没有相互协调，这意味着黑客行动中可能有多个情报目标。其中的一个黑客组织就与3月份微软邮件服务器（Exhange）的黑客事件有关，该事件导致几十万美国人的电子邮件信息被盗取。
