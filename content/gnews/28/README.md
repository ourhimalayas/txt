## 华盛顿特区将终止中共病毒口罩令和室内疫苗规定

![](https://assets.gnews.org/wp-content/uploads/2022/02/1-216.jpg)
图片来源：网络截图

据合众国际社2月14日报道， 华盛顿市市长穆里尔·鲍泽（Muriel Bowser）周一宣布，华盛顿特区将很快取消其室内中共病毒（COVID-19）强制佩戴口罩和企业对疫苗接种证明的要求。

在一次中共病毒的情况通报会上，鲍泽以改善大流行病指标为由取消了这些要求，这些要求是在12月为应对奥米炕（Omicron）变体引起的新病例激增而制定的。

鲍泽说：“自从华盛顿特区的奥米炕浪潮达到高潮以来，病例已经下降了90%以上，住院人数减少了95%。”我们现在处于一个更好的位置，可以宣布对该冬季应对计划的调整。”

市长说，室内强制佩戴口罩的要求将在2月28日到期，而且不会再延期。这意味着从3月1日起，餐馆、酒吧、体育赛事、健身房、教堂、零售企业和其他地方将不再需要口罩。

如果企业想保持面罩要求，他们可以。在学校、图书馆、公共交通、医疗机构和选定的其他场所，也将继续强制要求佩戴面罩。

鲍泽说，另一项要求室内场所提供疫苗接种证明的紧急措施将在星期二取消。

市长的行动反映了其他几个州和地方管辖区最近在中共病毒数据改善后采取的类似行动。纽约州、伊利诺伊州和马萨诸塞州都在周三宣布了放宽口罩规定的计划，而在加利福尼亚州，旧金山湾区的9个县中有8个宣布他们将遵循州政府的决定，从本周开始放弃普及室内口罩的规定。
