产经新闻11月1日报道，东京都调布市行驶的京王线特快列车上，17名乘客受伤，一名因涉嫌“谋杀未遂”而被捕的男子，在接受调查中时供述，“工作和朋友关系上都进展不顺，因此在人多的万圣节那天计划大量杀人”。警视厅调布署调查本部正在仔细调查，认为有可能是有计划地“无差别杀人”。

被捕男子表示，今年6月工作和朋友关系进展不顺，虽然想死但不想一个人死，想通过杀人被判处死刑。选择特急列车作案，是因为两站之间行驶时间长，同时参考了今年8月发生的小田急线的刺伤事件。

10月31日晚上8点左右，从京王八王子开往新宿的特急列车上，在离开调布站前往明大前站的途中，一名72岁的乘客被刀刺伤右胸（30厘米长），昏迷不醒。随后，嫌疑人泼洒油状液体，并点燃，造成16名乘客受伤。列车在国领站停车后，嫌疑人被抓捕。
