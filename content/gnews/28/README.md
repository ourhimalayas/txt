## 西方必须为与中共国的技术脱钩做好准备

![](https://assets.gnews.org/wp-content/uploads/2022/04/rep-tiffany-700x4202-1.jpg)2011年5月31日，美国科技巨头英特尔公司的参展商在亚洲最大的信息技术和通信展–台北电脑展的一个展位上。(Patrick Lin/AFP/Getty Images)
《大纪元时报》2022年4月1日报道，根据一份新的报告，整个西方和中共国信息系统的技术基础正在日益脱钩，变得互相独立和相互无法理解。专家表示，西方国家将不得不做更多的准备来应对这一后果。

前国家情报局局长丹尼斯·布莱尔(Dennis Blair)说：“西方志同道合的政府和私营领域公司之间的网络安全合作，必须不仅仅只是在发生危机时简单地将安全证书连接起来。”“不幸的是，这就是我们目前在许多领域开展国际合作的现实。”

布莱尔说，中共国和西方正在分化成两个截然不同的“技术圈”，各自拥有自己的技术、政策和做法。

他说，在一个领域中，中共国和俄罗斯正在开发与世隔绝的、具有国家特色的信息系统，而在西方民主国家，则正在推动开发更加普遍有效或可互操作的系统。他说，随着脱钩的加速，这种分裂将对网络安全产生巨大影响。

该报告发现，中共国已成为全世界技术主导地位的有力竞争者，如果其模式在世界范围内得到更广泛的采用，其隐私和安全做法可能会深刻改变信息格局。

“在与西方国家的竞争中，中共国在推动全球数字技术进步方面处于领先地位。”该报告称，“中共国技术实力的崛起正值地缘政治紧张局势加剧之际，中共国的实力和野心日益增强，欧洲日益分裂，而美国则处于政治动荡。”

这种情况导致了一场在发展中国家争夺国际影响力的斗争，西方正努力在国外推广与其市场上正在创建的信息技术系统相兼容的系统。

“中共国正有意识地寻求拉拢全球的不结盟国家，以便与其技术圈保持一致。”报告称，“西方国家在寻求与不结盟国家在这一领域接触时，在合作方面面临着挑战和选择。”

这种状况正日益导致布莱尔所说的一个“分割的空间”，即世界各国都在以一种彼此脱节的方式运行技术，阻碍了国家之间的数据流动。

为此，报告发现，西方国家在推动技术发展所需的价值观方面基本达成一致，但他们缺乏在中国共产党专制模式控制下的自上而下的协调。

该报告强调，俄罗斯、委内瑞拉和整个非洲许多国家都在采用中共国模式。报告认为，西方可以采取一些行动来对抗中共国模式的发展。

报告称：“政府和私营领域公司领导人应该联合起来，了解挑战，并致力于采取必要的长期行动，使网络空间成为一个充满活力和富有成效的领域。”
