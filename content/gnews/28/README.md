中共成立百年之际，法国《世界报》发表《西方民主制度面对中共考验》的社论。文中肯定了中共作为前苏维埃的“学生”已经成功地超越“老师”。苏维埃共产政权没有熬过75年，中共国政权已经到达了74年，并且达到前所未有的“盛世”。 作为世界上唯一一个执政如此长久的政权，其“成功”是不可否认的。中国的发展使其对周边国家不屑一顾，尤其是中共病毒散播世界以后，中共国是全世界第一个走出病毒阴影的国家，再次“巩固”其执政合法性。

然而成功是有缺憾的。一个严重践踏人权和自由的政权，中共国的发展道路是不可持续的，因为中共国挑战现有的世界秩序，摒弃多边主义，而世界的发展不是由中共国说了算。

尽管中共领导人号称没有输出“中国特色社会主义”，但是欧洲并不能存在侥幸心理。中共国的成功是西方的挑战。目前来看中共国不能成为民主国家，大部分中国人民也没有要求民主的欲望。虽然灌输民族主义不是共产党的专利，但是中共长期利用民族情绪维持其统治。

面对中共国这个巨人，除了美国以外，其他国家都不是对手。正因为中共国试图分裂世界，那么最简单的方法就是从经济、战略上联合起来。这还不足以应对中共的挑战，欧盟不能只是谴责中共国对人权的践踏，只有加强自身的发展，证明民主制度的成功。

中共国的下一代并不愚蠢，他们比欧洲和美国的同龄人更加“灵活变通”。让西方和北京之间谁占据主动的好方法就是让中国的下一代看到谁更藐视中共 。
