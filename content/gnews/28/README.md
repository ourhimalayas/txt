***短评：***

面对上海“防疫”封城期间的混乱、暴力、颠倒是非、草菅人命等残酷事实，面对上海沸腾的民怨和抵抗，面对国内外的一片质疑和声讨。新华社4月17日炮制的《上海战“疫”24小时》一文，堪称邪恶中共“谎言治国”的典范之作。

下面我们用文中造假的部分片断对照一下现实情况，以印证一下中共的邪恶嘴脸。

**片断1**

文中言：上海绝大部分人处于居家状态，尤其在“封控区”，居民日常生活所需物资的保障很重要。“团购”就成为邻里之间互帮互助、大家共渡难关的一种有效方法。

看来中共上海政府“保障”的非常有序。而事实上，一方面多处小区物质匮乏，居民忍饥挨饿，物价几倍几十倍的被人为上涨，外地务工人员没有任何食物配给，另一方面，多处绿色蔬菜堆积如山、腐烂丢弃却不予配送。甚至于为填饱肚子，有个竟zui c t拦截警车，要求被拘留的极端行为。更有甚者，发生了——老人吃自己大便以抵御饥饿，直至被活活饿死的人间惨剧。

**片断2**

文中言：崇明区花博园复兴馆方舱医院，280多位患者即将解除医学观察，正在准备出舱……“我在这里生活了6天，可以喝到牛奶，每顿都有蔬菜和肉。”一位即将出舱的男士告诉记者。

一片温情、幸福的“人们至上”场面。而事实上，有些方舱医院设备简陋、无药缺食、没有防疫、没有医疗，基本自生自灭。个别方舱医院上面漏雨下面大小便外溢。甚至于让孩子们与父母分开隔离，制造人道灾难。

**片断3**

文中言：上海市卫生健康委员会一级巡视员吴乾渝回应，一方面推进零售药店应开尽开，并建立药品配送“绿色通道”；另一方面针对封控区域居民的常见病、慢性病用药，由志愿者到就近社区卫生服务中心或医疗机构配药，按照慢病长处方政策，可配3个月的药量。

现实是，有关有基础疾病的老人因为没有药物的求救文字、音频在微信、抖音等互联网上屡见不鲜。很多慢性病和癌症患者因无法取药或者完成疗程，出现有人自杀，死亡案例。其中一些死亡案例的信息已得到了官方或者媒体的确认。

**片断4**

记者来到位于徐汇区的这家医院急诊大厅门口，实地目击急诊一线患者就医情况。……“您带她到对面核酸检测窗口再做一下，不需要等待结果，凭检测凭条就能进去看诊了。”

此段文字描写颇具戏剧化感染力，可谓“妙笔生花”。而事实是：哮喘发作未获救治的上海东方医院护士周盛妮、浦东航昌路哮喘发作未获救治的老人、天平路永昌街道高烧死的老人、等待核酸结果中死去的癌症患者、等待核酸结果中死去的5岁女孩、浦东街道寿二小区某透析病人、脑溢血无法获得救治的韦桂国、被封城无法去医院的陈伟、等待核酸结果中去世的郎咸平的母亲无法及时救治的郎咸平母亲……名单该有多长，我们无法估量，但是相比无一例死于奥秘克隆新冠病毒就已足够说明问题。

**片断5**

城市已一片静寂。但洋山、外高桥、张华浜、罗泾、龙吴等上海港区，依然灯火通明。桥吊、轨道吊、AGV（无人引导车）仍在有序作业。

是的，按此描述上海港区的作业非常有序，“蒸蒸日上”的上海经济应该不会受到海运物资的影响。但是，我们看到网络上的上海港夜景呈现的是——水泄不通的船灯。只不过，中共及其喉舌们把卫星地图给主动屏蔽了。
