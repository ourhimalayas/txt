年鉴数据显示，2020年相比2019年，全国人口净增加了204万人，而前一年人口增量还高达467万，2018年为530万。

另据中共民政部数据显示，中国结婚人数也在骤降，2020年官方统计的结婚人数为814万对，相较于2019年的947万对减少了113万对。
