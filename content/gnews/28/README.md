疫苗灾难重磅消息：新中国联邦创始人郭文贵先生在2021年12月3日文贵大直播中，受国内医疗界重量级的专家官员委托，再次呼吁全世界的家长们千万要保护自己的孩子，拒绝给孩子打疫苗，因为目前对已打疫苗的孩子检测结果显示，疫苗会摧毁孩子的正常生长周期，导致孩子身体的细胞变异和迅速成人化，马上早熟，即使孩子身体目前没有异常变化，以后也必然会导致肿瘤在体内生长。

该中国国内医疗界的专家官员是一位有良知的支持爆料革命新中国联邦的战友，她在直播前一晚与郭先生通话，郑重拜托郭先生在大直播中将此消息公布于世。她说，作为领域主管和专家，根据他们目前掌握的对国内各类打疫苗人群的研究显示，症状反应最多的是老年人，中共国老年人打疫苗后的爆发症状和全世界都不一样，最重要的就是突发癌症，甚至身体内同时爆发不同类型的癌症，而且几乎没人怀疑突发癌症是打疫苗导致的后果。而突发猝死的人群多是中、青年人；打了疫苗的女性群体子宫癌、卵巢癌、乳腺癌比以往几百倍的增长，而且很多都是没有结婚怀孕的年轻人。普遍打了两三针疫苗的孩子，突发症状反应最少，死亡率最低，但他们最近对打疫苗的孩子做了检测，结果显示孩子的身体细胞发生的变异、迅速成人化，马上早熟，是过去的很多倍，孩子即便不死，将来身体也一定会产生肿瘤，几乎是肯定的。因为任何医学界人士都知道，一个孩子突然被外界强加的力量促使其身体早熟，与国内吃的害人的催熟食品导致身体变化是同样道理，只会导致身体细胞处于变异之中，无休无止的对身体造成伤害。

当郭先生问该官员，国内大概死了多少人时，她说，台湾公布因疫苗造成死伤的人数是一千多还是一万多，以台湾的疫苗致死致残率乘以中国十四亿人是多少人？这是其一，第二，按照医疗条件，台湾用的疫苗、医疗措施比国内的先进程度，国内疫苗的致残致死率应该是台湾的1500左右倍，那你乘一下子，就会得出大概数字了。她具体举例佐证说，仅仅在她的范围内有例可循的，因为疫苗死亡的已经达到七万多。

该官员专家表示，新中国联邦文贵大直播在国内影响力非常大，特别像Eglis 医生对疫苗病毒专业通俗易懂的讲解、还有对青蒿素的介绍，使中共异常恐惧，墙内防爆料革命的声音超过一切，中共已出台内部文件称爆料革命新中国联邦是第三方中国人的政治势力。而且她最近看到一份文件让她非常惊讶，中共的各地疫苗病毒金融信息中，称郭文贵的爆料革命和他的新中国联邦，掌握病毒疫苗的内部信息。正因为如此，引发了她去调查到底疫苗发生了什么具体事情。她发现原来中共国内各地各省各地区，当地警察24小时在医院警戒，所有疫苗导致的灾祸、死亡、突发病情，已经全部进入国安状态了。
