当地时间12月14日，据消息人士透露，目前，中共政府正在以全球疫情严重、避免疫情在中共国内扩散为由，持续劝返试图出境的相关人员。
![](https://assets.gnews.org/wp-content/uploads/2021/12/7-27.jpg)
据有关报道显示，中共政府对墙内民众正常出境的限制越来越多，比如，没有正当理由或资料不齐全者不能正常出境。为此，不少被限制出境的民众不得不使用港澳通行证，通过澳门再辗转进入其他国家。然而，中共却将此举定义为“偷渡”出境，从而导致这些人员回国时面临被强制剪护照并拘留15天的处罚。

知情人士还披露，鉴于严惩之下仍不能阻止墙内民众出境，中共政府除了严管护照发放以外，还对港澳通行证的签发也进行了更加严格的管控。个别地区的民众在持港澳通行证出行时，必须先将护照抵押在当地公安局。

评论人士对此表示，中共政府多番作为的实质就是巧立名目，禁止墙内民众出境。而中共这般无所不用其极的行为或许表明，中共国的闭关锁国已经真正开始了。
