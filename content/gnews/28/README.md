1. **多国中共病毒感染病例激增，首尔、悉尼延长封锁时间**


[墨西哥](https://www.reuters.com/world/americas/mexico-adds-nearly-8000-new-confirmed-covid-19-cases-2021-07-06/?taid=60e4f207b9a1150001df57a1&amp;utm_campaign=trueAnthem:+Trending+Content&amp;utm_medium=trueAnthem&amp;utm_source=twitter)卫生局周二报告了7,989例新的中共病毒确诊病例和269个新的死亡病例，这是自2月底以来日新增病毒感染病例数最多的一天。到目前为止，墨西哥总的感染人数达到2,549,862人，死亡人数达到233,958人。

[巴西](https://www.reuters.com/world/americas/brazil-registers-1780-new-covid-19-deaths-2021-07-06/?taid=60e5084afdc7d30001f57610&amp;utm_campaign=trueAnthem:+Trending+Content&amp;utm_medium=trueAnthem&amp;utm_source=twitter)卫生部周二报告称，在过去24小时内，巴西登记了62,504例新的中共病毒感染病例和1,780个新的死亡病例。巴西的病例总数达到近1,900万，死亡人数达到526,892人。

[印度](https://www.reuters.com/world/india/india-reports-43733-new-covid-19-cases-last-24-hours-2021-07-07/?utm_medium=Social&amp;utm_source=Twitter)卫生部周三数据显示，在过去24小时内，印度共有43,733例新感染病例，使目前活跃病例上升至459,920例。

[韩国](https://www.reuters.com/world/asia-pacific/south-korea-reports-highest-daily-covid-19-caseload-months-2021-07-06/?taid=60e5385bbcdfda0001434329&amp;utm_campaign=trueAnthem:+Trending+Content&amp;utm_medium=trueAnthem&amp;utm_source=twitter)周三报告了其有史以来第二高的每日新增中共病毒病例数，达1,212例，其中大多数来自首尔。考虑到Delta变种病毒正在迅速蔓延，政府考虑将首都和周边地区的行动限制延长至少一周。

因担心Delta变种病毒，[悉尼](https://www.reuters.com/world/americas/sydney-covid-19-lockdown-be-extended-another-week-media-reports-2021-07-06/?taid=60e512d7bcdfda0001434264&amp;utm_campaign=trueAnthem:+Trending+Content&amp;utm_medium=trueAnthem&amp;utm_source=twitter)宣布将封锁时间再延长一周，至7月16日。

2. **美国务卿视频会晤维吾尔人，国务院发言人警告将实施新制裁**

[美国国务卿布林肯](https://www.voachinese.com/a/blinken-met-with-uyghurs-and-wished-dalai-lama-happy-birthday-20210705/5956099.html?utm_source=twitter&amp;utm_medium=social&amp;utm_campaign=dlvr.it)周二以网络视频方式，与曾在中共国新疆地区被关押的维吾尔穆斯林会面。布林肯希望直接从这七名曾被拘押的人、其他被拘押者的亲属和倡导人士那里听取有关他们的情况以及更广泛的维吾尔人社区所面临的处境，并就如何最有效地向中共国施压以制止其在新疆的打压咨询意见。

美国国务院发言人普赖斯说，“美国就这些践踏行为、暴行、正在新疆发生的种族灭绝已经非常明确和一贯地发出了声音”，并警告称，“将利用额外的工具对负有责任的官员”实施新的制裁。

3. **德国政治学者因向中共提供情报被捕**

[德国联邦检察官](https://www.voachinese.com/a/German-think-tank-man-arrested-for-spying-for-China-20210706/5955346.html?utm_source=twitter&amp;utm_medium=social&amp;utm_campaign=dlvr.it)办公室周二宣布，德国某智库的政治学者兼负责人克劳斯∙L（Klaus L）因涉嫌为中共国情报部门从事间谍活动，于7月5日被逮捕。

根据德国联邦检察官办公室发布的一份声明，克劳斯于2010年6月前往上海讲学时被中共国情报机构接触并收买。从那时起至2019年11月，他定期在国事访问或国际会议前后向中共国情报机构传递情报，并接受中共支付的报酬。检察官办公室指出，克劳斯向中共国提供的情报，主要来自他多年来以智库学者身份联络、接触和结交的“高层政治人物”所提供的信息。

4. **美国就朝鲜问题与中共国进行电话会谈**

[美国国务院发言人](https://www.reuters.com/world/asia-pacific/us-envoy-north-korea-speaks-with-chinese-counterpart-2021-07-06/?taid=60e4f45eb9a1150001df57a9&amp;utm_campaign=trueAnthem:+Trending+Content&amp;utm_medium=trueAnthem&amp;utm_source=twitter)普莱斯称，在平壤一再拒绝拜登政府发起的对话之后，美方朝鲜问题最高特使Sung Kim周二与中共国朝鲜半岛问题特别代表刘晓明进行了电话会谈。

普莱斯表示，华盛顿将与盟友韩国和日本在朝鲜问题上进行合作，而中共国 “显然对朝鲜政权有影响力”，“也可以发挥作用”，并说，拜登政府正在“等待朝鲜的建设性答复”。

5. **新西兰著名中国问题学者因嘲讽习近平被推特限制使用账号**

[新西兰](https://www.breitbart.com/asia/2021/07/06/twitter-restricts-account-of-china-scholar-who-dared-to-mock-xi-jinping/)坎特伯雷大学教授安妮·玛丽·布拉迪（Anne-Marie Brady）是一位研究中国政治影响的专家。上周，布拉迪在推特上嘲笑中共的100周年庆祝活动，并质疑为什么几乎没有人参加这个活动。在推文中，布拉迪修改了一篇关于此次庆祝活动的新闻报道的标题，写道：“习近平：这是我的党，如果我想哭，我就会哭”。布拉迪的两条推文被暂时标记为 “不可用”，账户在周末被暂时限制，周一才恢复使用。
