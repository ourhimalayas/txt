## 疫苗谎言“云南瑞丽疫苗接种率近97%却封城”

云南省瑞丽市再度爆发中共病毒疫情。目前瑞丽市主城区实行封闭管理，所有市民居家隔离。按照党媒说法，在这次疫情出现前，瑞丽市所在的云南德宏州就在力推疫苗接种，接种率接近97%。

但当地居民透露，这次疫情仍然十分严重。 在这次疫情出现前，瑞丽市所在的云南德宏州就在力推疫苗接种。央视的报导说，截至7月4号24点，德宏州应当接种中共病毒疫苗的人群，全程接种率达96.92% 7月7号0点起，瑞丽市主城区实行封闭管理，所有市民居家隔离。当天10点起，瑞丽市姐告国门社区调整为高风险地区，其他区域为低风险地区。 当局声称，前一天云南省新增本土确诊病例15例，其中两人是7月5号的无症状感染者。另外13例和新增的2例无症状感染者，都是瑞丽市全员核酸检测中发现的。由于中共一贯掩盖疫情真实情况，官方数据可能严重缩水。 目前当地的学校和各类培训机构已全部停课。除保供超市、农贸市场、医院、药店外，其他经营场所也全部停业。餐饮行业只允许外卖。网友发布的视频显示，7月6号晚，身穿防护衣的人员在街头，公安警察也一齐出动。光是今年，瑞丽就三次爆发疫情，最后一次从3月末封城到4月26号解封，持续近一个月。

之前《纽约时报》6月下旬曾披露，主要使用中国国药疫苗和科兴疫苗的四个国家——塞舌尔、智利、巴林和蒙古，目前都在疫情最严重的国家行列。“印度尼西亚医学协会”也表示，6月死于中共肺炎的26名医生中，至少有10人接种过中共科兴疫苗，并且已经接种了两剂。 BBC中文网7号也引述印尼一名不愿公开姓名的胸肺科医生说，她虽然已经完成接种两剂科兴疫苗，但一个月之后她在测试血液内中共病毒抗体水平时，发现效果远逊预期。有印尼专家呼吁，当地面对新一波变种病毒疫情，当局应该让医护人员接种第三剂疫苗，但印尼亚逸朗加大学（Airlangga University）讲师普尔诺莫 （Windhu Purnomo） 质疑“如果科兴疫苗不可以保障人们免受变种病毒感染，那么注射第三剂疫苗也没有用。”。

笔者认为，中共散布病毒制造民众恐慌，然后在国内急推疫苗接种，本质上利用疫苗进一步绑架人民群众，一是利用免费疫苗，这些小恩小惠，让人民群众对中共政府感恩戴德。二是利用疫苗毒害人民群众的健康，悄无声息的削减人口，进行无差别杀戮。对于青壮年人口来说可以消除社会不稳定因素，对于老龄化人口来说可以减少政府养老金支出。三是中共疫苗本就是中共权贵家族敛财工具，也是“蓝金黄”世界的一种手段，进一步拉拢医疗科技不发达的国家，为其邪恶政权摇旗呐喊，从而达到其控制全球以及其他不可告人目的。最后，笔者也劝告广大战友和能看到这篇文章的朋友，珍惜生命，拒打疫苗。
