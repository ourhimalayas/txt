## 新中国联邦让战友们安全地获得财富

1月14日，郭文贵先生于直播中指出，王毅在与中东六国的外交部长见面时，他向这六个国家提出了一系列针对新中国联邦的限制措施，其中包括绝不能支持新中国联邦、不能支持喜联储和喜币，以及屏蔽GTV与GNEWS等条件。而王毅的此番荒谬言辞也使得这些国家真切地感受到了中共对新中国联邦的恐惧。
![](https://assets.gnews.org/wp-content/uploads/2022/01/Screen-Shot-2022-01-15-at-8.50.02-PM.png)
此外，郭先生提到，目前，躺平币随时可能面世，但他不希望战友们因为躺平币承受任何风险。于躺平币而言，其最重要的特点，就是让战友们在不冒险的情况下得到币。因此，躺平币的推出时机非常重要。

郭先生解释道，在给战友们准备躺平币的时候，他考虑最多的就是战友们的安全——要让战友们能够安全地从墙内出来，还能前往更多的国家并真正拥有这个币。为此，郭先生也再三劝告称，包括墙内的战友在内，任何人都不要冒着生命危险去买币。

郭先生还强调，爆料革命和新中国联邦不是邪教，更不搞精神控制、金钱控制或安全控制，相反的，爆料革命和新中国联邦恰恰要让战友们在绝对自由和安全的情况下获得财富，是正义的、真正为人民服务的存在。
