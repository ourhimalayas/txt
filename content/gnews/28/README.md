文贵先生在2022年2月23日大直播里再次谈到“铁链女”，一句“这个世界不要我了”抒发出难以言表的悲愤之情。虽然文贵先生强忍着泪水，但相信每一个看大直播的人心里都在滴血。是啊，在21世纪的中华大地上竟然还存在着这样残酷和非人道的现实。谁敢相信？谁又愿意去相信呢？这位“铁链女”24年来受到何等非人的折磨？还有多少同样遭遇的女性没有被人知道？那些死去漂在河中的冤魂又有多少故事要述说？

文贵先生深刻地指出，更令人心痛的是“铁链女”周围的人们都冷漠地接受和容忍一个女性被当作性奴24年之久。事情被曝光后，各种官方托词和假证明接踵而至，足以证明如今的中国是如何被共产党把持和奴役的。为什么乡里乡亲的男人和女人们都可以熟视无睹？为什么政府的男官员、女官员们都可以如此麻木不仁？锈蚀的锁链在“铁链女”的身上，而真正生锈了的又何尝不是那条锁住人们心灵的无形的铁锁链？

这条心灵的铁锁来自哪里？

我记得我的母亲多少次说过她到甘肃农村搞“四清”时看到的情形。她开口的第一句话总是：“唉，你是没去过，你不知道甘肃农村的生活有多苦”。因为气候的原因，那里干旱少水，生活在那里的大多数人一辈子也洗不上几次澡，可数的有出生之后、出嫁之时，和入土之前。最穷的人家共用一床棉被，一条棉裤。兄弟共妻的现象十分普遍。这是发生在中共统治下的20世纪60年代初的情景。

我曾经天真地相信共产党的宣传，以为“改革开放”可以使人们脱贫致富。然而，2016年，还是在甘肃，又发生了杨改兰杀四子后又自杀的惨剧。2021年共产党宣布“全民脱贫”。可如今2022年，“铁链女”事件赫然爆出，令人不寒而栗。如果真的像共产党宣传的那样“全民脱贫”了，为什么还有人共妻？甚至有更恶劣的贩卖妇女做性奴的事件存在？沉浸在谎言中人们是不是可以醒醒了？

在共产党的宣传里，“妇女是半边天”，但中国妇女从来没有完成心灵的自我救赎。虽然表面上女性好像在家庭里说一不二，管着钱，管着丈夫，但大多数女性的内心没有真正的平等意识。女人不被平等对待，也不能平等地对待男人；男人的“大男子主义”更不曾有丝毫减弱。无论男人还是女人深入骨子里的仍然是千年来的男权思想。一个没有平等自由的社会就演变成如今这样一个“互害”的社会。
