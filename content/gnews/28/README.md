在这个框架内，事实并不能证明东利的理论。调查协议的语言模棱两可，因为其中一个条款使用了“犯罪”一词，而另一个条款提到了“商业调查”。但协议的其余部分显示，研究并不是用来帮助特定官司中的特定受害者。它预想对多达4000人的长期监测。它涉及每周或每月的报告，但它没有包含任何关于使用或保存证据的内容。它没有提到诉讼、受害者或犯罪。事实上，该协议的三个主题事项似乎更侧重于可能羞辱个别人和中共党员，而不是针对任何具体的犯罪或受害者。

此外，你们将听到的双方谈判的证词也证明了这一点。郭文贵、王雁平或韩春光从未表示过任何受害者或诉讼会在这里举行，也没有表示（调查报告）会在任何具体的诉讼中使用。相反，这是为了帮助郭文贵和他的爆料运动，以提升他在美国的形象，并据称可以推翻中共。这是政治目的，而不是准执法(quasi-law enforcement)目的。

最后，郭文贵作为东利的代理人，欺诈了愿景公司。这一声明的核心是检验郭文贵是否是真的异见人士。它建立在三大支柱上。郭文贵的资金从香港转移并与大陆有联系（郭文贵自己的说法）以及对其他异见人士的攻击。我们稍后将讨论这些问题。不过，这里的其他内容集中在愿景公司上。沃勒博士和华勒普女士都将作证说，他们长期以来比较依赖的同事——韩连潮和比尔-格茨——都是证人，你会在视频中看到，而不是在这四天里。他们将证明，他们不同意提供调查报告给任何会将它用来帮助中共的人。在这一点上，他们并不孤单，因为另一位著名的异见人士和记者龚晓夏(Sasha Gong)也在这一时期相信了郭文贵的说法。她甚至一度在一个机构的董事会任职，而这个机构现在已经调转枪口攻击她和其他异见人士。而韩连潮的母语是普通话，他将作证说，他在本案接受质询的时候才第一次看到郭文贵的关于自己的有害言论，尽管韩连潮是被雇来专门指导郭文贵申请政治庇护，而该申请是在郭文贵发表这些言论的两周后提交的。韩春光（可能是笔录错误，真实应该是韩连潮）并不知道这些言论。

现在我转到第一支柱，从香港和大陆的资金渠道。郭文贵的律师和发言人，金泉公司的律师，但不是今天在座的律师，回应愿景公司在本案中的声明时说，从郭文贵发声的那一天起，他没有从香港或大陆拿一分钱出来。但是ACA从香港给愿景和比尔·格茨，甚至给郭文贵的家族基金汇过款。龚晓夏，正如我之前所说，将证明郭文贵把ACA的唯一股东余建明介绍给她，称他财神爷。郭文贵在另一个联邦案件中声称，在同一时间段，即2017年至2018年，ACA通过他在香港的家族基金——中国金泉，向他支付了数百万美元，为他所谓的国内人脉给报酬，而他据称是中国头号异见人士，以及无法从这两个地方拿走一分钱，他怎么可能以一个香港公司的名义来起诉追偿？这不符合逻辑。

关于ACA的老板余建民，郭文贵说他是异见人士，他的老朋友，甚至不堪政府的骚扰而住院了。但是，当许多公开的来自中共国的消息显示余建民与中共国的的机构有密切联系时，郭文贵将无法解释他为何能容许和余建民如此亲密的关系。我们将听到关于这些的消息。

此外，郭文贵强烈要求，第一小组——即愿景雇用的第一个调查小组——的普通话翻译不能是中国公民或在中共国居住，据称是担心他们可能是中共特务。龚晓夏将证实，异见人士与大陆保持开放的通信是多么危险，然而谷歌的商业记录将显示，郭文贵的几个社交媒体账户的管理员使用了大陆的电话号码和电子邮件地址。

最后，郭文贵的家族基金、ACA，甚至一度东利的总部都在香港中银大厦的49层套房。郭文贵后来承认，中银大楼被中共国用于在国外进行间谍活动。该大楼是一个很特别的选择，用于容纳与郭文贵有关的所谓异见人士的机构(dissident entities)，这些机构在本案中都有问题。

在2017年5月郭文贵录了自己和政治局以及其他中共官员在他公寓开会的录音。在里面，郭文贵为他效力国安邀功，并宣称 “我坚决支持习主席”。最后，在2017年8月，他签署了一封信，然后参加了一个中文电视节目，在节目中他同意有限度的爆料。他指出，他之前的采访从未越过所谓的红线，他要求领新的任务来支持习近平主席。

第三支柱：继2017年3月保证抨击在美的中国异见人士之后，郭文贵试图收买或胁迫美国当地保守派的异见人士。法院将听到郭文贵威胁过他的异见人士的录音，如龚晓夏和傅希秋(Bob Fu)。郭文贵鼓励在美的中国公民跟踪和骚扰异见人士，要求他们通过他在纽约设立的非营利组织——法治基金和法治社会——获得批准。

因此，综上所述，法官大人，事实将表明，在这份合同作废后，郭文贵和东利公司从未雇用过其他人来调查和实施他的计划。没有其他作为。他们从来没有再找其他人来解冻这些资产。这些香港资产仍然被冻结，据说是因为中共的报复，但郭文贵却仍然可以自由地资助几十起诉讼，继续攻击在美的异见人士。出于这个原因，战略愿景认为，记录将显示，郭文贵并不是他本人所说的那样，也从未打算利用一个合同来打击中共。

证据不仅支持愿景的欺诈反诉，也为东利的不当得利(unjust enrichment)的衡平法索赔提供了不洁之手(Unclean Hands)的辩护。东利（通过郭文贵）认可这种欺诈行为的情况下，到法院来寻求救济是不公平的。

法官大人，总之，在接下来的四天里，本庭将听到东利公司证人讲述不一致的故事。东利花了很少的时间在他们上面，而且传唤他们在愿景的证人之后出庭。你很可能会看到原因。也就是说，你会听到关于调查目的完全不同的说法——是为了郭文贵的爆料运动，正如东利的证人在当时一直宣称的，还是像他们现在说的那样，是为了东利自身的利益解冻资产？谁在为该协议提供资金？是香港的投资者，如王雁平当时声称的那样，还是ACA的贷款，如东利在本案中很晚才最终声称的那样？如有疑问，本院应以当事人当时的陈述和本案早期的证词为依据。

不管怎么说，我们还是要回到我们的起点。东利是一个空壳公司，它在本案的参与度悬而未决。它必须以某种方式证明与ACA的所谓贷款不是一个骗局（该贷款对其主要证人来说，在本案发生近一年后仍然是个谜）。由于ACA向愿景支付的100万美元不是贷款，东利公司不应根据归还或依赖求偿，双方应该各回自家，或者愿景应该挽回自己的损失。事实将表明，这一百万美元的定金，就像这份2700万美元的合同下的其他付款一样，绝不可能来自一个在香港银行账户被冻结的8万美元的公司，而且它甚至需要世界历史性的政权更迭才能获得这微薄的资金。任何为该合同支付了一百万美元定金的人都不可能指望东利公司偿还本金或利息

法官大人，是时候结束这场闹剧了，其结果应该是倾向愿景的判决。谢谢。
