## 迟到60年的太空圆梦之旅 贝佐斯陪她一起飞
` 日本东京方舟农场` [轉載自GNews](https://gnews.org/zh-hans/1368564/)

#### 作者：日本东京方舟农场 铁蚁骑士

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/07/03001510/E9C60087-DF16-47CA-A4B1-81A48840AD0A.jpeg)

英国亿万富翁布兰森的“维珍银河公司”一日宣布，旗下太空船“团结号”订十一日试飞，布兰森将与另外五人一起升空；若天气和技术检测许可，布兰森按此计划将比对手亚马逊创办人贝佐斯提早九天上太空。

贝佐斯先前表示，订廿日搭乘自家“蓝源公司”的太空船上太空。“蓝源公司”一日宣布，飞行员出身、现年八十二岁的美国妇人冯克，将参与蓝源公司的首次载人太空飞行，成为贝佐斯的旅伴。 对冯克而言，这趟太空之旅可说是迟到了六十年。冯克曾是“水星十三”计划的一员，也就是在一九六○至六一年接受太空人培训的第一批女性，后来却因为性别无法上太空。 布兰森推文写道，“我向来是个梦想家，我的母亲教我绝不放弃前往星际；七月十一日，是时候让梦想成为现实，下一趟维珍银河的真实旅程。” 布兰森表示，“我真的相信，太空属于我们所有人；经过十七年的研究、工程和创新，新的商业太空工业已准备好为人类打开宇宙并永久改变世界”。

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/07/03001821/84DA6016-1BBC-440C-8DC0-47EB21BEC3DD.jpeg)

布兰森与贝佐斯的太空之旅高度大约距地表九十至一百公里，高度已经属于太空边缘，两趟飞行都将持续约十分钟。相较之下，SpaceX总裁马斯克的太空舱，包含载人与载货的舱体，将绕地球轨道运行。
