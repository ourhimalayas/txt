## 新州州长多米尼克-佩罗特建议避免非必要PCR测试
` 洛杉矶盘古农场 HIMALAYA LA PANGU` [轉載自GNews](https://gnews.org/zh-hans/1790843/)

**评论：经过20个月的谎言，美国CDC再也无法掩饰其使用COVID-19 PCR测试是假的“科学”，但该测试却制造了恐惧，可以预见越来越多的政府会放弃PCR测试。**

**据《天空新闻网》作者：Zoe Zaczek，2021年12月23报道：**

每天数以万计的新南威尔士州居民在圣诞节前排队等候PCR测试，州长现在要求人们避免去做不必要的测试，并宣称这“给系统带来巨大压力”。

州长多米尼克-佩罗特建议非密切接触者或患有COVID类似症状的居民，停止排队进行PCR检测。

新南威尔士州周四记录了5,715个COVID-19病例——在高达160,471次测试中发现——以及一例死亡病例，由于高度传染性的Omicron变体病毒在该州肆虐。

佩罗特先生说：“整个州许多人没有症状，州卫生局不要求他们进行测试，但他们依然在排队等候PCR测试，这给圣诞节前的医疗系统带来了巨大压力。”

他在周四的COVID-19新闻发布会呼吁：“我们需要确保那些需要接受检测的人及时得到检测。更重要的是我们要尽快将结果反馈给人们。”
![](https://assets.gnews.org/wp-content/uploads/2021/12/amq6p-x6e25-10.jpg)**周一，在马鲁布拉Heffron Park驾驶式COVID-19测试中心出现了巨大的汽车排队现象。图片：Julian Andrews.**
佩罗特先生说：“一些居民通过州服务局的应用程序被通知他们是密切接触者并一直在排队接受测试。”

他说：“如果你确实感到不适，请去做PCR测试。如果你没有感觉到不适，就没有必要进行这种测试。”

“这将有助于减轻我们全州检测中心的一些压力。”
总理说，大约20%的PCR检测是为了跨州旅行的需要，这也给州卫生系统带来了“巨大压力”。
![](https://assets.gnews.org/wp-content/uploads/2021/12/amq6p-x6e25-9.jpg)**总理多米尼克-佩罗特建议非密切接触者或患有COVID类似症状的居民，面对延误，应停止排队等候PCR拭子测试。图片：NCA Newswire / Gaye Gerard**
他说：“我确实知道外面有相当多的挫折感。许多人排队等待数小时，特别是在圣诞节期间。”

“我们正在尽一切努力来减轻这种压力。”
“如果你没有感觉不舒服，请不要做PCR测试。如果你在这个时候占用了那些感到不适人的位置，将延长排队的时间。”

卫生部长Brad Hazzard重申了佩罗特先生的要求，即除非受州卫生部要求或感到不适，否则不要进行PCR检测。

他建议假日期间探亲的居民选择快速抗原测试，而不是采取PCR检测。
![](https://assets.gnews.org/wp-content/uploads/2021/12/a9ky9-mtr5w-1.jpg)**在圣诞节前，悉尼人在中央商务区的COVID-19检测诊所排队等候。图片：NCA Newswire / Gaye Gerard**
他说：“如果你要去见保姆、家人或朋友，认为需要进行检测，请使用快速抗原测试方式。”

“快速抗原测试方法并不像PCR检测那样准确，但单就满足你上述访问目的需求来说，它们肯定非常好。”

州病例自本周初以来增加了一倍多，周一报告了2501个病例，周二报告了3057个，周三报告了3763个。

目前有347名阳性患者住院，高于周三的302人，有45个病例在ICU，高于前一天的40个。

住院和进入ICU的人数逐日增加——自本周初以来增加了86名病人和12个ICU病例。
