## 【NewsExpress】华为聘请了三家新的游说公司

#### 翻译整理：蓝精灵

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/07/09044618/f34g34gnipaste_2021-0sdafsaf7-09_16-45-48.png)

根据提交给国会的披露报告，陷入困境的中共电信华为最近聘请了三家新的游说公司。

拜登总统保持原先川普总统制定的政策扼杀了华为在国际上开展业务的能力。

根据最近的游说文件，华为聘请了前众议员李特里(R-Neb); Glenn LeMunyon，前众议院多数党党鞭汤姆·德莱（R-Texas）的前助手史蒂芬·宾哈克（Stephen Binhak）他们是白水调查前总统克林顿的前检察官。三人都经营着自己的公司。“华为已与这些公司合作，以加深华为与美国政府之间的了解，”一位知情人士告诉希尔。

根据游说文件，特里和勒蒙将就与电信和基础设施相关的问题进行游说。Binhak将就贸易、经济制裁和年度国防开支法案等相关问题进行游说。

在川普政府实施一系列限制并由拜登政府进一步收紧之后，华为在中国以外的销售额直线下降。美国指责华为代表北京利用其技术监视美国实体，该公司否认了这一指控。2019 年，川普政府禁止华为与美国公司开展业务，包括谷歌、高通和英特尔等主要合作伙伴。美国敦促其欧洲盟友阻止华为建设他们的 5G 网络。

上个月，拜登签署了一项行政命令，禁止美国投资华为和其他据称帮助中共政府镇压维吾尔穆斯林和香港民主活动人士的中共国公司。参议院上个月通过的中国竞争力法案将阻止商务部在未证明该公司不再对国家安全构成威胁的情况下取消美国对华为的限制。众议院不会采取参议院通过的措施，而是将重点放在几项较小的法案上。

华为的目标是在拜登官员审查川普的中国相关政策时影响他们。该电信公司还推动释放其高管孟晚舟，后者因涉嫌规避美国制裁而被监视居住在加拿大自己的豪宅里。
