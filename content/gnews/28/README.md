据《悉尼先驱晨报》11月24日报道，由于担心中共国网球运动员彭帅的安危状况，澳大利亚正在考虑通过外交手段抵制明年2月举行的北京冬奥会。

报道称，堪培拉正在考虑不派遣任何政府官员参加冬奥会，因为议员中要求外交抵制的呼声越来越高。然而，这项措施不会阻止澳大利亚运动员参加比赛。

如果澳大利亚决定不派出政府官员，它将加入美国、英国和加拿大的队伍，考虑对这场全球体育赛事进行外交抵制。外界认为，这是对中共国侵犯维吾尔族穆斯林人权的回应。

星期四晚些时候，中共国严厉批评澳大利亚的报导，说外交上抵制北京冬奥会只会损害全世界运动员的利益。

中共国外交部发言人赵立坚在北京对记者表示：“中共国一贯坚决反对任何将体育政治化、违背《奥林匹克宪章》精神的言行。”他还表示，任何抵制奥运的企图“都不会成功”。

《悉尼先驱晨报》报道称，澳大利亚正在考虑不派遣任何政府官员，包括其驻中共国大使出席冬奥会。但不会公开称其为抵制。报导指出，澳大利亚政府正在等待美国总统拜登(Joe Biden)政府作出决定。同时援引英国执政党自由党议员埃里克·阿贝兹的话称，美国和英国正在积极考虑外交抵制北京冬奥会。
