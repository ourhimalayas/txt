## 生命的意义是什么？（第四集）

![](https://assets.gnews.org/wp-content/uploads/2021/09/GNEW-GTV-MOS-LOGO-2-2-1-1-400x54-1.jpeg)



你是否需要灵感来疗愈自己并从抑郁中解脱？你是否需要忘掉过去的挣扎苦恼并让自己的生活向前迈进？你是否期待发掘自己的潜力，并找到一个分享自己想法的平台？在这一集里，我将继续和大家谈论生命的意义，也将继续分享朋友们的故事和他们对生命的看法。

**生命就是要脱离不需要的记忆或事件。**

大多数人都有美丽的梦想和不需要的记忆，他们对某件事情爱恨交加了很久。有些记忆是带着无法逃避的情绪，仇恨、愤怒、怨恨和痛苦。走近它，触摸它，不要试图躲避那些不开心的感觉。一个有天赋的射手多年来一直在为一场比赛做准备。在开始前的几个月，他失去了他的右手。手术后他几乎放弃了希望，但他的内在力量拒绝接受残酷的命运。他开始用自己的左手不停地进行射击练习，而不是放弃自己的梦想。最后，他用左手赢得了比赛。所以一定要坦然面对生活提供给你的任何东西。生活中有些变化是好的，有时也会发生意想不到的事情。生活就是要保持中道，不偏不倚，就像你是观众中的一员，观看一部由你自导自演的名为 “我的生活 “的精彩大片。

**生命就是要勇敢面对可控和不可控的事情。**

人生并不总是像彩虹一样，有着美丽的色彩。我们必须学会在美好但有时艰难的情况下保持快乐。你会抱怨雨夜天空中的星星吗？当然不会。我打赌你可能偶尔会享受在宁静的夜晚倾听窗外雨滴的声音。要充满感恩地活着，感受大自然和我们周围的一切。乐观的人知道如何享受和度过他们有时狂野但又难忘的生活。活着就是要一直做真实的自己。有时感到恐惧是可以的；偶尔觉得悲伤或沮丧也是可以的，但记住，你有力量和勇气面对你的恐惧，克服你的焦虑情绪。当你开始按自己的意愿生活时，你就能过上无悔的生活，即使你经历了起伏，包括成功的日子或失败的夜晚。这就是生活。好日子是有期限的，同时电闪雷鸣的夜晚也迟早会结束。请放松心情，尽力了就好。

**保罗**。对我来说，生活就像一块比萨饼，可以选择在上面添加我想要的调料。生活就像雨，我们永远不知道它何时开始，也不知道它何时停止。没有人能够永远一帆风顺，所以我会尽力享受与所爱的人相伴，珍惜快乐的时光，做我最喜欢的事。是的，没有人知道下一个时刻会呈现给我们什么。我已经意识到，我应该对自己有信心，以享受我的人生旅程，就像我小时候父母给予我的无微不至的关怀和无条件的爱。我曾经抱怨我的父亲不富裕，责怪他在高中时没有提供我需要的一切。我曾经为了从母亲那里得到更多的零花钱而与妹妹争吵。现在我每周至少和父母电话聊天三次。总之，我开始为自己的生活负起全部责任，不再寻找借口或指责他人来为自己的懒惰辩解。突然间，我觉得生活就像美丽的花朵，不再是一种折磨人的毒药。对我来说，生命意义的关键是心态的转变。

**丽莎：**两周前，我叔叔因肺癌去世了。他就像我的父亲一样，因为我父亲在我13岁的时候就去世了。叔叔一生都专注于他的汽车维修生意。“我宁愿死在我的修车厂里，因为我把我的生命和灵魂放在这个地方”，上个月他在医院的病床上这样跟我说。我叔叔几乎没有任何朋友，只有家人、几个雇员和一些供应商。“我至少会去十个国家旅行，看三百部电影，和我的家人在全国各地最有名的餐馆用餐”，他在去年的家庭圣诞聚会上对他的妻子和三个孩子说。他的去世对家庭的所有成员来说是一个巨大的损失。悲伤氛围持续了三天，叔叔的妻子和孩子们就开始争夺财产，因为律师宣布的遗嘱内容在家人们的意料之外。他们甚至大打出手，要求修改遗嘱条款，全然忘记了叔叔为这个家里的每个人做出的牺牲、给予和忍耐。我不确定我叔叔现在是否真的安息了，家人们的行为是否会让他寒心。他在退休前后对未来十年有很多计划，可惜都没有办法实现了。能说他的生命是充实的吗？他忙碌到连去电影院看电影的时间都没有。叔叔只考虑到未来，为了明天的梦想拼命工作，但忽略了自己的健康和家庭关系。我从叔叔的生命经历中学到了一件事，那就是生命就是要活在当下，才能死而无憾。
