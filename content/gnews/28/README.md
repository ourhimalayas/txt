## 【NewsExpress】沙特阿拉伯资助美国新的数字新闻平台 启动白宫游说工作
` 闪电侠` [轉載自GNews](https://gnews.org/zh-hans/1382836/)

译者：新兴媒体接踵而至，新闻会更加成为影响政治的一部分。传统的拜访游说将会有新的变化。这篇文章报道了这个事实，也让我们了解最新的数字媒体的时代的到来，以及对于打造新媒体的信心。以下是文章翻译件，供读者参阅：

布赖恩施瓦茨2021 年 7 月 8 日在CNBC报道：沙特阿拉伯正在资助一个数字新闻平台，平台将开始对白宫和国会游说工作，平台将邀请一些美国媒体专业人士参加，将在华盛顿特区设有工作室。这是在华盛顿邮报记者和沙特王室评论家贾马尔·卡舒吉被杀近三年后。

沙特阿拉伯正在资助一个尚未公布的数字新闻平台，平台将在华盛顿特区设有工作室，为开始对白宫和国会进行新的游说活动。根据向司法部提交的新的外国游说披露，这项尚未公布的新项目得到了沙特技术开发和投资公司的子公司 Taqnia 的支持。参与该项目的记者和主持人过去曾在 Fox News、Al Jazeera、NBC 和 Sirius XM 卫星广播电台工作过。一份文件称“新闻平台”创建之际，沙特已开始招聘一支新的游说团队，因为沙特寻求接触乔·拜登总统的政府和新国会。这是在华盛顿邮报记者和沙特王室评论家贾马尔·卡舒吉去世近三年后。美国最近的一份情报报告称，沙特王储穆罕默德·本·萨勒曼批准了一项逮捕或杀死卡舒吉的行动。

然而，拜登选择不惩罚王储，虽然他涉嫌在卡舒吉之死中扮演角色。“我们对那个组织中的所有人负责但不是王储，因为我们从来没有意识到，当我们与一个国家结盟时，会去找代理国家元首并惩罚那个人并排斥他，”拜登在最近接受美国广播公司新闻采访时说。

支持该计划的子公司是 Taqnia ETS，这是一家专注于为沙特阿拉伯提供工程和技术服务的公司。PitchBook表示， Taqnia 总部位于沙特阿拉伯，它“更喜欢投资于生命科学与健康、安全与国防、信息与通信技术、材料科学、能源与环境以及水技术领域。”Taqnia 的网站称，它由本国超过4000 亿美元的公共投资基金 (PIF) 所有，这个基金是本国庞大的主权财富基金，投资项目并帮助提振沙特阿拉伯的经济。游说披露报告称，Taqnia ETS“由沙特信息部监督和资助”。PIF 的执行委员会包括王室成员，如王储。

游说文件让我们得以一窥即将到来的沙特领导的新闻事业。其中一项游说披露显示，一家名为 Prime Time Media 的公司正在帮助领导这项工作。Prime Time Media 的首席执行官 Elie Nakouzi 正在协助创建新的数字新闻机构。文件显示，他的公司至少获得了 160 万美元的报酬帮助指导此项目。游说披露中包含的采购订单将 Prime Time Media 所做的工作描述为“新闻平台的英文内容”。文件称，已经在设备、自由职业者和华盛顿特区的工作室上花费了超过 75,000 美元。其中一份列出 Nakouzi 的文件称，Prime Time Media 将“协助客户制作英文新闻内容的视频。这样做时，注册人将不会参与所述新闻内容的传播。”

据披露，纳库兹拥有黎巴嫩国籍，但居住在美国，曾在中东媒体圈工作过。在为一家阿拉伯电视网络工作期间，纳库兹在伊拉克和阿富汗战争期间采访了当时的总统乔治·W·布什。Nakouzi 在一封电子邮件中向 CNBC 证实了 Taqnia 项目的几个方面，包括它将成为一个新的在线平台，并且已经创建了一个工作室。他拒绝对即将发布的网站的新闻内容发表进一步评论，并将所有问题转给 Taqnia，后者没有回应 CNBC 的评论请求。“Taqnia 计划在今年年底前推出一个国际在线平台，”Nakouzi 周二在一封电子邮件中说。“至于工作室，我们已经建造了一个设备齐全的虚拟绿色房间供将来使用，”他补充说，同时指出它将设在华盛顿。“黄金时间媒体无权代表他们发言。我们是一家视频制作公司，不提供游说或公关服务，”Nakouzi 解释说。

埃里克·汉姆和克雷格·博斯韦尔，两位美国记者，将为沙特资助的合资企业主持在线节目。到目前为止，Ham 和 Boswell 都因披露报告中描述的作家和主持人的身份而获得了 6,000 美元的报酬。“Eric Ham 和 Craig Boswell 先生将主持仍处于制作阶段的在线节目，”Nakouzi 在电子邮件中说。博斯韦尔和哈姆在发表前没有回复评论请求。
