## 郭先生：去中心化保险不久之后将面世
` 日本东京方舟农场` [轉載自GNews](https://gnews.org/zh-hans/1889034/)

#### 撰稿：日本东京方舟农场  幽蘭自得

![](https://assets.gnews.org/wp-content/uploads/2022/01/73E0E275-A32A-47D7-9BFB-120CD7CA61B1.jpeg)（图片来自GNEWS）

1月19日，新中国联邦创始人郭文贵先生在GTV大直播中透露，其带领的秘密团队设计开发的去中心化保险不久后面世。该保险锚定数字货币并受第三方监管，客户能100%地得到赔付。

郭先生在直播中指出，现在市场上所有的保险都是没有任何锚定，也没有第三方监督的中心化保险，都是基于客户的保险费做的金融安排，比如存储、经营等。若保险公司替客户做的经营失败，客户的保险费也会一并亏损，完全没有保障。

另外，在中共国，客户能否得到赔付几乎就由保险公司说了算。比如去年郑州因人祸造成的大洪水泡坏了数以千计的车辆，但这些车主并不能得到相应的赔偿，因为保险公司将其归为不可避免的自然灾害以逃避赔付。郭先生说：“现在的保险纯粹是金融安慰剂，是某些人用来骗钱的金融工具。”

为此，郭先生融入区块链技术理念，设计出去中心化的保险，一举打破传统保险行业客户被动的局面。他透露，该保险不仅有数字货币锚定，还有第三方监管，在任何情况下，只要满足条件，客户就能100%地得到相应赔付。郭先生强调，这样的设计既不存在欺骗，也没有霸王条约，该不该赔或者赔付多少都不是由保险公司决定，一切以约定好的白皮书为基准。

郭先生还透露，他的秘密开发团队马上就能完成这个项目，随时都能掀开这个“新娘”的盖红。

此外，郭先生在直播结束宣布送给爆料革命战友一份大礼，即在去中心化保险的诞生之日，所有GTV的投资者和G-club会员将100%拥有一份与投资同等价值、完全免费的去中心化保险。
