辽宁丹东东港市3月6日凌晨确诊一例阳性感染者，之后，对感染者所在的前阳镇实行“只进不出、人不出区”的管控措施，镇内所有商铺暂停营业，所有活动一律暂停举办。

续吉林省连续爆出疫情感染事件后，中共又一轮残暴的疫情管控已经开始。这种灭绝人性的管控所引发的次生灾难远远大于疫情本身。曾经的武汉、西安等地的管控措施已经证明了中共防疫措施的失败。据内部人士透露，吉林省、辽宁省等多地正全面部署全员核酸检测。

据笔者分析，吉林、辽宁两省采取如此极端的防疫政策有以下2个原因。

1、中共的两会召开，由于中共的政策倾斜东北经济发展相对于南方省市发展落后，由此产生的官僚、腐败严重，上访事件频发。利用疫情管控管理人员流动就成为有效的选择。

2、利用疫情管控发财。由于经济、政策的落后，东北对于南方开放的省市思维方式也相对落后。如流行歌曲、服饰、文化等，由这种思维模式所产生的时间差大约半个月至两个月不等。像全员核酸这种发财路子东北人还需要学习一段时间。

东北由于历史原因文化、教育等相对封闭，受俄罗斯远东文化影响较多。汉族、朝鲜族、蒙古族、满族等民族文化交融，东北的汉族基本由两次闯关东的百姓组成，乡村百姓纯朴，既得利益者凶狠奸诈两级分化严重，极端事件多发。文化大革命对本来文化底蕴不多的东北形成的冲击至今还未消退，对暴力、权利的崇拜已经影响了2代人，也是东北黑社会猖獗的原因。黑社会与官僚勾结已成常态。
