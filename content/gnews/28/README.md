## 百分百接种疫苗的邮轮又爆发疫情
` 喜马拉雅意大利罗马达芬奇农场` [轉載自GNews](https://gnews.org/zh-hans/2265214/)

据《大纪元》3月29日报道，公主邮轮公司（Princess Cruises）证实，它的一艘邮轮“红宝石公主号”（Ruby Princess）在停靠旧金山之前报告了COVID-19疫情。
![](https://assets.gnews.org/wp-content/uploads/2022/03/C.png)freeread.causeaction.com
该邮轮运营商要求乘客和船员全面接种COVID-19疫苗，该病毒由CCP（中国共产党）病毒引起。根据公主邮轮的网站，它还强制要求乘客在登船时出示COVID-19的阴性测试和疫苗接种证明。

那些检测结果呈阳性的人被 “隔离和检疫，同时由我们的船上医疗团队进行监测和护理”，公主邮轮在周一给新闻媒体的一份声明中说。它没有说有多少人检测出阳性，也没有说他们在游轮上何时检测出阳性。

该公司说“他们都没有症状或只有轻微的症状”。声明补充说，一些感染CCP病毒的乘客没有完成检疫，他们或者被送回家，或者 “被提供到岸上事先协调好的酒店进行隔离和检疫的住宿”。

该公司告诉《旧金山纪事报》（San Francisco Chronicle）说：“与公主号的所有行程一样，这艘邮轮是按照美国疾病控制和预防中心的定义，以接种疫苗的方式运营的。客人和船员的疫苗接种率为100%。”

该公司说，红宝石公主号于周日停靠在旧金山，此前该船在巴拿马运河进行了为期15天的巡航。公主号邮轮的发言人内金·卡马利（Negin Kamali）在给《今日美国》（ USA Today）的一份声明中说，该船当天晚些时候离开，前往夏威夷进行为期15天的巡航。

根据美国疾病控制和预防中心（CDC）的邮轮监测网站信息，红宝石公主号被联邦卫生机构描述为 “正在观察”。

这一发展是在CDC将其对邮轮旅行的疫情警告降低到 “2级”，即“中等 ”风险之后的两周。此前，该机构对邮轮旅行发出了 “4级 ”警告，这是最高级别的警告，因为几个月前奥米克戎（Omicron）变体在美国蔓延。

在疫情大流行期间，邮轮业一直受到封锁和联邦对邮轮的限制的打击，在早期的猜测中，邮轮成为该病毒的 “超级传播者”。行业数据表明，邮轮公司在2020年和2021年总共损失了630亿美元。

3月18日，CDC为邮轮业发布了新的COVID-19指南，发言人告诉《今日美国》，该指南包含了该机构关于社交距离、检疫要求和港口协议的建议。

《大纪元》已与公主邮轮公司联系，要求其发表评论。

**快评**：这样的消息不是一个两个，就是主流媒体不肯承认，接种了疫苗的人不仅自己会感染病毒，还会传染病毒，并增加CCP病毒的变异。

《大纪元》将COVID病毒直接称为CCP病毒，要为他们点个赞。

疫苗灾难还没有真正显现，但邮轮行业在疫情大流行期间已经受到重创。
