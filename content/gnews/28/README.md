**1. ****钟南山称中国国产疫苗对德尔塔变异毒株有效**

中国工程院院士、国家呼吸系统疾病临床医学研究中心主任钟南山在接受中国官方媒体采访时称，中国国产疫苗对德尔塔变异毒株有效。钟南山周五在接受中国央视采访时指出，对于入境人员不需要再增加隔离时间，因大多数病毒潜伏期并不长；加强对这些人员在隔离时间内的核酸检测的频度才是有效的办法。

据钟南山说，“目前国产常用疫苗对德尔塔变异株还是有效的，大家应该打，”钟南山称，“德尔塔变异毒株的传播中，主要的重症的病人多为老年人，同时也有孩子。因此，这两组人群应加强接种。”
