## 美国情报机构更新病毒起源调查报告
` 纽约香草山MOS02` [轉載自GNews](https://gnews.org/zh-hans/1645040/)

作者：日本方舟农场 – Andy5

![](https://assets.gnews.org/wp-content/uploads/2021/11/Screen-Shot-2021-08-30-at-7.37.59-PM.png)

10月29日，美国情报机构于更新了今年8月的90天病毒起源调查报告。

![](https://assets.gnews.org/wp-content/uploads/2021/11/捕获.png)

本报告第一版于8月被提交给总统拜登，之后总统拜登要求继续调查。29日的更新版本即是这个继续调查的结果报告。

报告内容与结论跟第一版的大致内容并无二致，依旧是“没有足够的证据来说明起源”，而其原因也还是“中共国政府妨碍调查、拒绝提供情报”。

报告称，美国4家情报机构认为病毒起源于受感染动物或者相关病毒的可能性较低，另有1家情报机构称，它“有适度的信心”认为第一次人类新冠病毒感染很可能是实验室事故造成的，可能涉及武汉病毒学研究所的实验或动物处理。
