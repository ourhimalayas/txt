检察官在一份声明中说，这名男子是一名政治学家，自2001年以来，他一直在一家智库工作。

办公室称，他于2010年在上海的一次演讲之旅中被中共国情报部门招募，并定期为其提供信息以换取金钱和前往中共国的旅行，这份工作持续到2019年11月。

嫌疑人将于周二在慕尼黑法庭出庭。多年来，他利用为该智库工作的便利，收集各种高级政治人物的信息。
