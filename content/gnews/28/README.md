1、[**美台经济对话，讨论芯片供应和中国经济胁迫等**](https://www.voachinese.com/a/taiwan-talks-chips-chinese-coercion-in-us-meeting-20211123/6324642.html)

美台第2届经济繁荣伙伴对话周二（23日）以视频方式举行。双方就4大议题进行了深入的讨论，其中包括半导体芯片等供应链韧性、如何应对北京的经济胁迫、数字经济与5G网络安全以及科技合作。美方主谈人是主管经济成长、能源及环境的国务次卿费南德兹，台湾方面则由经济部长王美花和科技部长吴政忠牵头。双方的视频对话共持续了5个小时。

2、[**荷兰众院力挺台湾参与国际刑警组织 **](https://www.aboluowang.com/2021/1124/1675378.html)

第89届国际刑警组织大会11月23日至25日在土耳其伊斯坦布尔举行年度大会，开会第一天，荷兰众议院以149票对1票的压倒性票数通过动议，吁请荷兰政府与理念相近国家共同支持台湾参与，这是继2019年荷兰国会通过支持台湾参与国际组织的历史性动议案后，再次发声力挺台湾有意义国际参与。

3、[**意总理第三次否决中国公司并购计划**](https://www.voachinese.com/a/italy-veto-3rd-chinese-takeover-20211123/6325213.html)

路透社报道说，中国浙江晶盛机电公司周二（23日）在香港正式提交的文件说，意大利政府阻止了该公司与美国应用材料公司香港分公司成立合资企业，收购应用材料公司在意大利的丝网印刷设备业务的计划。这项决定是11月18日在意大利政府内阁会议上做出的，是意大利总理德拉吉今年2月就职以来第三次否决中国公司的并购计划，理由是保护本国的战略产业。

4、[**格陵兰撤销中国公司的开采许可**](https://www.voachinese.com/a/greenland-chinese-firm-delicensed-20211123/6325150.html)

路透社报道，中国煤炭和铁矿石进口商俊安集团2015年取代破产的伦敦矿业公司，获得格陵兰首府努克附近Isua 矿业项目的控制权，也因此成为第一家在格陵兰获得矿业开采许可的中国公司。格陵兰政府周一（22日）宣布，撤销俊安集团在当地开采铁矿石的许可，让中国公司进入这个资源丰富的北极岛屿的企图遭受打击。格陵兰政府在一份声明中说，撤销俊安集团的开采许可是因为当地没有开采活动，而且俊安集团未能按照规定支付保证金。

5、[**拟建立致力于言论自由的美国大学**](https://www.voachinese.com/a/university-of-austin-free-expression-20211122/6323493.html)

一群学者和活动家正计划在美国建立一所致力于言论自由的新大学，因为他们说，他们对美国著名高校“普遍存在的不自由主义和审查制度”感到不满。这一举动引起了外界尖锐的批评，反对者认为这所学校是美国保守派右翼的平台，不会改善美国高校言论自由的现状。这所学校将以德克萨斯州奥斯汀地区的实体校园为基础，被称为奥斯汀大学。于明年夏天进行软启动。其创始人说，学校将提供被称为“禁止课程”的非学分项目，来提供针对其它许多高校可能回避的、有挑战性的话题的热烈讨论。这些话题经常在许多大学导致审查或自我审查。
