## 硝烟中仍决定冒险返乡的乌克兰人
` 日本东京方舟农场` [轉載自GNews](https://gnews.org/zh-hans/2361494/)

**撰稿：日本东京方舟农场  Miyali7768(文玉)**

据《BBC》4月15日报道，近期，俄乌战争局势趋于稳定，一些乌克兰人开始返乡，甚至愿意回到那些直接受到俄军打击威胁的地区。

报导称，波兰边境的一些乌克兰人向记者表示，已经厌倦了在国外当难民的生活，迫切想回到自己的家乡。其中来自乌克兰东部顿涅斯克的斯拉维扬斯克难民奥莱娜和玛丽亚在国外逗留了一个多月后徒步返回了祖国。

![](https://assets.gnews.org/wp-content/uploads/2022/04/图片-1-5-4.jpg)（图片来源：路透社）

由于奥莱娜居住的城市在3月时受到俄军的威胁，她的丈夫让她带女儿逃往到波兰，但他没有跟随母女二人逃亡，而是作为救护车司机选择留在故乡工作。

于是，奥莱娜母女二人便前往与德国接壤的波兰城市什切青投奔亲友。然而，在那里语言不通，找工作很困难，并且没有熟悉的家人和朋友。因此，尽管俄军新一轮军事打击在即，但她们仍然决定一路向东回家。

当奥莱娜母女二人到达乌克兰后接受采访时，面露笑容说，“回到乌克兰我们非常开心！”

此前，西方分析人士预言乌克兰首都基辅可能在48小时内被俄罗斯攻陷。然而，如今俄乌战争已经超过48天，打破了预言，乌克兰人顶住了俄罗斯的进攻，也稀释了国家可能迅速瓦解的恐惧。而且，对乌克兰人来说，家庭、家人和朋友的吸引力太大，不能不回家去。
