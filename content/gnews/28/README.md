## 众议院通过的国防法案削减了一些旨在对抗中共国的条款

来自《大纪元》2021年12月8日报道

**摘要翻译：**

美国众议院于12月7日通过了《2022年国防授权法案》(NDAA)的两党妥协版本，即年度国防开支法案。该法案包括许多旨在反抗中共国政权的条款，但排除或淡化了其他几个中共国早期草案中的相关项目。

总的来说，该法案将授权五角大楼在2022财年获得7680亿美元的资金，比拜登总统最初要求的多出约250亿美元。它以654票对70票通过，51名民主党人和19名共和党人投反对票，现在它将送往参议院进行投票。

该法案的很大一部分致力于更好地定位军队以对抗日益咄咄逼人的中国共产党。这场竞争的范围是全球性的，但国防授权法案尤其关注印太地区。具体而言，该法案为太平洋威慑倡议授权71亿美元，这是一项广泛的战略努力，旨在发展更多力量，并将印太地区的战略努力集中在对抗中共国政权上。

妥协后的法案还指出，核心目标是“维持或恢复美国相对于中共国的军事优势”。为此，该法案支持继续努力为台湾提供不对称的军事能力和训练，以避免中共国入侵的“既成事实”。这符合《台湾关系法》，该法案要求美国为台湾提供自卫能力。

周二晚上众议院通过的最终版本中，还有几项与反制中共国有关的条款被淡化或彻底删除。“中共国仍然对我们的国家和经济安全构成威胁，美国可以而且应该做更多的事情来追踪其恶意活动。”

同样，该法案的先前众议院版本将禁止使用五角大楼的任何和所有资金用于在中共国进行的开发或研究。最终版本将该订单限制在一个组织内：生态健康联盟。

实验室泄漏新冠病毒起源理论，新的《国防授权法案》条款禁止将国防部资金转移给生态健康联盟，用于在中共国进行的任何研究或国防部长确定为中共国所有或控制的任何实体。

**简评：**

新的《国防授权法案》在众议院修改后去除了要求国务卿报告中共国遵守《不扩散核武器条约》第6条（为避免军备竞赛而进行的善意谈判）的规定； 一项旨在通过鼓励美国与国际货币基金组织和金融稳定委员会合作来减轻来自中共国的全球金融威胁的条款；将建立一个工作组来协调美国和台湾之间的半导体制造的条款； 一项要求报告美国为打击中国和俄罗斯在非洲的恶意影响所做的战略努力； 一项拒绝在九段线内适用中国《海上交通安全法》的条款；一项旨在建立外交影响力行动以对抗中国全球影响力的条款；禁止将五角大楼的资金用于中国大陆的任何研究或产品的条款。

法案还禁止美国国防部采购中共国新疆地区的强迫劳动生产产品。美国将中共国（中共）对待新疆维吾尔族穆斯林少数民族的行为认定为种族灭绝，国会议员一直在推动禁止进口强迫维吾尔族人的劳动生产产品。
