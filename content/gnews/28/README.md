## 中共国创下冠状病毒感染人数的记录
` 西班牙巴塞罗那喜悦农场` [轉載自GNews](https://gnews.org/zh-hans/2128531/)

**编译：七龙珠**

![](https://assets.gnews.org/wp-content/uploads/2022/03/tempsnip中共国创下冠状病毒感染人数的记录.png)香港PCR檢測中心。

尽管中共国对Covid病毒采取了严厉的零容忍政策，但本周记录了几个月来的最高日感染人数。

中国大部分地区正在遭受疫情，本周录得数月来最高的每日感染人数，尽管中国对Covid采取了严厉的零容忍政策。

该国今天连续第三天报告了大陆至少300个新的Covid病例，与其他国家相比，考虑到中国14.11亿的人口，这个数字很小，但自2020年上半年以来在中共国中闻所未闻。

在中国大陆，已经接种了31.6亿剂covid-19疫苗，根据官方数据，这足以让所有符合接种条件的人口接种两剂疫苗。

受影响最严重的地方之一是与香港半自治区接壤的南方省份广州，自2月25日以来，已经有1000多人死亡。

深圳市是规模最大的疫情爆发地之一。据当地媒体援引深圳防疫专家领导的话说，这是受其靠近香港的影响，因为“一些人非法越境”和“许多卡车司机往返香港和广州”。深圳的反应是收紧了对香港旅客的规定，并要求有PCR才能进入地铁，以及其他措施。

面对病例的增加，许多街区被禁闭。“本周一，他们禁闭了我的住宅区，在此之前，我们不被允许外出，我们必须每天接受测试”，深圳的乌拉圭居民贾纳-罗德里格斯-赫兹说，他已经连续8天每天都必须接受核酸检测。

罗德里格斯-赫兹居住的蛇口区继续受到限制。他说：“直到上周，所有餐馆和超市都已关闭，从上周日开始，餐馆现在只允许提供外卖食品”。

根据官方数字，在拥有1250万人口的深圳市，有320个活跃的病例，据这位居民说，深圳受到了“完全不相称的措施”，他将这种“过度反应”归因于当局热衷于避免“任何可能导致连锁传染的人为错误”。

在经历了两年多的大流行之后，这位乌拉圭居民强调了民众的疲惫感，“尤其是在外国社区的人”，以及一些中国公民“在当局做得太过火时感到恼火”，尽管他也看到有些家庭“感到很高心，这样一些父母有时间与他们的孩子在一起”。

无症状病例的数量相对较多，自上周日以来，官方报告中单独统计的无症状病例每天超过300例，这使得日常追踪工作变得复杂，而这是中共国将感染者及其密切接触者隔离和观察的战略中不可或缺的一部分。

在东部城市青岛，有8000多人被关在检疫中心，65000多人被关在“禁闭或看守”的街区，大多数感染者“没有症状，因为他们感染的是Omicron变体，其症状较轻”，当地政府说，他们没有说明疫情复发的来源。

在上海，有595个活动病例，占该市整个大流行病中报告的所有病例的12%，他们已经对15万多人进行了检测，对街区和办公室进行封锁，以及大规模PCR检测，周一发现了51个无症状病例。

在其他对冠状病毒采取零容忍政策的国家，如澳大利亚或新加坡，放松了他们的策略之后，中共国是唯一一个仍然坚持所谓的“动态零病毒战略”的国家，尽管这个中共国越来越多的专家强调“动态” 这一形容词。

中共国的边境仍然对非居民的外国游客关闭，所有登陆中共国的人必须在酒店里接受至少14天的严格检疫。

流行病学家、前疾病控制和预防中心主任曾光最近在断言中共国“不急于开放边界”的同时，宣布“限制不会永远持续下去”，中共国“将在适当时候提出与病毒共存的路线图”。

根据官方说法，自该大流行病开始以来，该国已有111,520人被感染，其中102,832人已被治愈，4,636人死亡。
