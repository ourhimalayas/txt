美国气候特使约翰-克里呼吁中国提高其减碳努力的速度和深度。克里先生说，如果中国不进行充分的减排，将温度控制在1.5℃以下的全球目标 “基本上是不可能的”。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/07/20225541/Untitled-design-13-1.png)图片来源 EPA
克里先生说，他相信中国可以做得更多，美国愿意密切合作，以确保一个合理的气候未来。每个主要经济体现在都必须承诺在2030年前进行有意义的减排。

在本周晚些时候在意大利举行的20国集团环境部长会议之前，克里先生在伦敦邱园发表讲话。在提到11月在格拉斯哥举行的关键的COP26会议时，这位前国务卿说，”在100多天内，我们可以拯救未来100年”。

他说，要做到这一点并不容易。在2015年巴黎气候协议期间和之后作出的承诺仍将看到世界气温上升2.5-3摄氏度。他说：”我们已经看到升温1.2摄氏度带来的灾难性后果，”他提到最近美国和加拿大的热浪，以及欧洲的洪水。”试试想想，将这一数值翻倍是在招致怎样的灾难。”

他指责了一些仍在建设新的燃煤发电站的国家的努力。他对那些非法砍伐雨林的国家表示强烈谴责。”他们正在清除世界之肺，破坏不可替代的生物多样性，同时破坏气候的稳定。”观察家们说，这位特使可能指的是巴西和印度尼西亚。

克里先生特别关注中国的努力，说该国现在是 “气候变化的最大推动者”。中国已承诺到2030年达到排放峰值–但这位美国外交官说这还不够好。他警告说：”如果中国坚持其目前的工厂，直到2030年才达到排放峰值，那么世界上所有其他国家必须在2040年甚至2035年之前达到零排放。

“根本没有其他选择，因为如果中国不进行充分的减排，1.5C的目标基本上是不可能的。在这个具有非凡国际影响的问题上，中国的伙伴关系和领导力至关重要。”克里先生说，他相信中国可以超越它自己设定的目标，尽管他们有外交分歧，但美国热衷于进行合作。

他还说，所有其他主要经济体都必须加紧努力，为未来十年制定更宏伟的目标和计划。

克里先生的评论是在新的研究表明G20国家继续支持化石燃料的方式与巴黎协议的目标不一致的情况下提出的。

根据彭博社的《气候政策实录》，G20国家从2015年到2019年总共削减了10%的化石燃料资金。然而，包括美国、加拿大和澳大利亚在内的八个国家在此期间增加了对石油、煤炭和天然气的财政支持。

上周，克里先生前往莫斯科，双方都认为气候问题是一个共同关心的问题。普京总统说，莫斯科 “非常重视 “实现巴黎气候协议的目标。
