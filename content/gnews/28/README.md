【接上篇：为什么欧洲需要乌克兰（一）】

看看我们看似无聊的“排放交易系统”，这是世界上最大的系统；它现在覆盖了欧洲经济的一半以上，并开始运作得越来越好。或者看看我们的火车网络，它是世界上整合得最精细的网络之一。如果说有什么体现了欧洲的梦想，那就是乘坐火车旅行。法国在1981年开始运营其第一批高速TGV列车；40年后，美国仍然没有一条高速线路。还要考虑到我们城市的设计。哥本哈根和阿姆斯特丹是世界上最好的两个骑自行车的城市，像巴黎、巴塞罗那和米兰这样的大都市现在有激进的计划，把他们的街道也还给行人和骑自行车的人。

![](https://assets.gnews.org/wp-content/uploads/2022/04/image-858.png)图片来源：《大西洋》杂志

硅谷开发的算法使我们尽可能多地点击广告，华尔街开发的金融产品使我们的经济处于崩溃的边缘。欧洲的创新更具有可持续性。

近年来，太阳能和风能的价格急剧下降，这是政治决定–欧洲政治决定的结果。由于德国的“能源改革”，太阳能已经变得更便宜了。或者以丹麦的风能产业为例，它建造了世界上最大的涡轮机。那个有三个叶片的现代白色风力涡轮机？它被称为“丹麦概念”。

在国际合作方面，欧洲也处于领先地位。欧盟在制定1997年《京都议定书》和2015年《巴黎协定》中发挥了重要作用。作为世界上最大的经济集团，欧盟有能力强制执行全球标准。政治学家将此称为“布鲁塞尔效应”。如果我们写了一部法律，规定例如冰箱应该消耗更少的能源，那么世界各地的制造商就会被迫遵守该法律，因为为每个产品制造不同版本的成本太高。他们宁愿制造一个符合地球上最大单一市场标准的产品。

因此，也许我们必须从美国人那里借用绿色新政这样的吸引人的术语，当我们有一些很酷的想法时，就引用肯尼迪的演讲。欧洲仍有很多问题。我们已经沉迷于廉价的俄罗斯天然气，忽视了我们的国防，并经常将我们国家政治家的失败归咎于布鲁塞尔的“欧洲官僚”。

然而，大多数年轻的英国人从未想过退出欧盟，而现在又想回到欧盟，这是一个原因。像我遇到的那些乌克兰学生，多年来一直梦想着加入欧盟，这是有原因的。

新一代人意识到，欧洲可以而且应该成为世界的灯塔；没有远见的紧缩时代已经结束，我们必须开始大规模投资；我们需要富人和穷人、年轻人和老年人、北方和南方、东方和西方之间的团结；国家主权和欧洲一体化之间没有不可避免的权衡，相反，如果我们一起努力，我们都会变得更强大。我们在应对气候变化方面取得的显著进展不仅说明了我们在制定标准和微调监管方面的技术能力，也说明了我们的价值观–投资、团结、合作。

我们不要忘记，2014年俄罗斯对乌克兰的入侵和对克里米亚的吞并是在基辅推进与欧盟的结盟协议之后开始的。乌克兰人在弗拉基米尔-普京的独裁模式和欧盟的民主模式之间得到了一个简单的选择。这一选择是很容易做出的。在2014年之前，乌克兰对欧盟和俄罗斯的出口是平均分配的；在随后的几年里，对欧盟的出口翻了一番，而对俄罗斯的出口却萎缩了。越来越多的乌克兰人到欧盟成员国旅行，这主要是因为不再需要签证了。

简而言之，乌克兰选择了欧洲。而普京认为这是不可容忍的。现在该由我们来选择乌克兰了。是的，通常情况下，加入欧盟的道路是漫长而复杂的，而且有充分的理由。但现在不是正常时期。数百万勇敢的乌克兰人重振了欧洲的理想–自由、民主和合作，而且许多人付出了生命的代价。

为了我们的价值观和原则，我们应该尽快停止进口俄罗斯天然气，以各种方式支持乌克兰争取独立的斗争，慷慨地接收难民，并启动一个欧洲马歇尔计划–我们可以借用另一个美国术语–在战争结束后重建乌克兰。最后，我们必须为乌克兰打开大门，让它加入我们的欧盟。

在这场战争的黑暗中，有一件事是肯定的。乌克兰的未来在欧洲。我们的共同未来在欧洲。或者像一个美国人说的那样“我是一个欧洲人”。
