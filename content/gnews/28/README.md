## 越南CCP病毒感染激增 Delta病毒为主
` 索尼克` [轉載自GNews](https://gnews.org/zh-hans/1377053/)

编译：yyshere

据越南卫生部7月6日号公布的数据，当天越南新增1029例Covid-19（CCP病毒）确诊病例，创下疫情爆发以来的单日新增最高记录，病例中有1019例为本地感染，仅有10例为境外输入病例，目前越南的累计确诊病例已增至22064例。

在本土感染病例中，南部疫情最严重的胡志明市占到710例，同塔省南部新增了9例，平阳省南部则是增了92例，感染者大多曾与先前确诊病例有接触，或着属于已知的群聚感染群体。

根据此前的统计，越南自6月开始的新增病例中，超过6成都是感染的Delta变种病毒，更值得注意的是，6月15号胡志明热带病医院的55名工作人员被检测出Covid-19（CCP病毒）阳性，而这批感染者之前已经全部完成了英国阿斯利康疫苗的两剂接种。

据印度和英国科学家最新的联合研究，Delta变种病毒对现有疫苗的不敏感度是源自中共国武汉的原始毒株的8倍，也就意味着，疫苗对Delta变种毒株的保护性极低。同时Delta变种毒株可以轻易躲过先前感染Covid-19（CCP病毒）后出现的中和抗体，并可导致已康复的感染者再次被感染。
