据AP News7月18日报道，韩国首尔一艘海军驱逐舰上近70名海军队员的冠状病毒检测结果呈阳性后，韩国将于周日派出军用两艘多功能空中加油机，把4400吨级驱逐舰 “文武大帝 “号上正在东非海域执行反海盗任务的300名海军队员带回家。

到目前为止，有68名士兵的检测结果呈阳性，300名士兵中的200人的检测结果仍在等待中。15 名士兵在非洲某国家住院治疗，其余的则留在驱逐舰上。据卫生和军事官员表示，没有人接种过CCP病毒疫苗，因为他们在2月初离开韩国，军事当局怀疑病毒可能是在6月底驱逐舰停靠在该地区的一个港口装载货物时传播的。据参谋长联席会议称，由150名海军人员组成的替换人员将抵达空中油轮，并转移到停泊在海上的驱逐舰上，并将其驶回韩国，这段旅程大约需要一个月。

自2009年以来，韩国参加了亚丁湾的反海盗行动。”文武大帝 “号在进行为期六个月的轮换部署后，将在下个月被另一艘驱逐舰替代。第二艘驱逐舰目前正在前往该地区的途中。

韩国卫生当局周日报告了1,454个新病例，使该国的病例总数达到177,951个，死亡2,057人。韩国最近对其人口众多的首都地区已经实施了最严格的隔离规则，当局正在考虑在其他地区也实行更严格的限制。
