## 美国会议员因北京冬奥，正寻求剥夺国际奥委会的免税地位

![](https://assets.gnews.org/wp-content/uploads/2022/02/GettyImages-1233797766-700x4201-1.jpg)图片来源：www.theepochtimes.com
佛罗里达州共和党众议员迈克·华尔兹(Mike Waltz)和弗吉尼亚州民主党众议员詹妮弗·韦克斯顿(Jennifer Wexton)正在寻求剥夺国际奥委会(IOC)在美国的免税地位，因为它对中共的种族灭绝和其他暴行视而不见。

“北京是我们最不应该举办奥运会的地方。”华尔兹告诉新唐人电视台的《国会山报导》，指出中共国是如何将大约100万维吾尔人和其他穆斯林少数民族关在集中营中，在香港践踏自由，以及对中共病毒起源的不透明。“我认为美国的纳税人最不应该做的事情就是补贴这种行为或赋予他们免税的地位。”

华尔兹和韦克斯顿于1月19日提出了《追究国际奥委会与中国同谋法》(H.R.6433)，称国际奥委会违反了其作为联邦税法501(c)4规定的社会福利组织的注册。根据美国国税局的规定，社会福利组织必须“专注于慈善、教育或娱乐目的”，才有资格获得501(c)4的免税地位。

根据提案的内容，如果法案通过，这一立法将修改美国税法第501条，取消国际奥委会和类似组织在任一纳税年度的免税资格。这些组织被定义为那些主要目的是进行国际多项目体育比赛，并且在前三个纳税年度中任何一个年度的总收入超过1亿美元的组织。

国际奥委会的大部分收入来自于广播和市场营销协议。2014年，国际奥委会与NBC环球集团达成了一项77.5亿美元的合约，NBC环球集团获得直到2032年奥运会在美国的转播权。

美国是认定中共政权在新疆地区的政策等同于种族灭绝的西方国家之一。“那么我们什么时候才会觉得‘够了’呢？如果正在进行的种族灭绝还不足以让国际奥委会另寻他处，我不知道什么才是‘够了’。”华尔兹说，他指的是国际奥委会拒绝听从一再提出的转移2022年冬季奥运会地点或推迟比赛的要求。

华尔兹还强调了中共对法轮功学员的持续迫害，同时表示，中共政权批准的从有良心犯身上摘取器官的做法是“逆天背理和令人作呕的”。在英国的中国特别法庭在其2019年的报告中得出结论，中共背书的强制活摘器官行为在中共国已经发生了多年，“规模很大”。该报告指出，“可以肯定”，器官的来源是被羁押的法轮功学员，他们“可能是主要来源”。

华尔兹还批评了作为2022年冬季奥运会赞助商的美国公司，说他们是中共侵犯人权的“共犯”。

1月27日，包括自由之家和人权观察组织在内的243个全球组织发表了一份联合声明，呼吁更多国家加入对2月4日在北京开幕的2022年冬季奥运会的外交抵制。到目前为止，澳大利亚、加拿大、立陶宛、美国和英国都是宣布外交抵制的国家之一。声明还呼吁冬奥会的13家企业赞助商，包括爱彼迎、英特尔、宝洁和Visa，说他们应该“立即披露其人权尽职调查战略”。

声明中写道：“这些公司对他们的赞助造成或导致侵犯人权行为的担忧，或他们是否已经采取行动来减少这些侵权行为，没有做出有意义的公开回应。”华尔兹认为，归根结底，当涉及到中共国的侵犯人权行为时，这就不是一个共和党或民主党的问题。他说：“这是关于我们的基本自由、道德和价值观，坚决支持那些应该享有这些基本人权的人。”

国际奥委会能无视中共在新疆犯下的种族大屠杀，活摘人体器官等等恶行，继续授予中共2022年冬奥会的举办权，说明国际奥委会根本不在乎人权、公平和正义，它也只是独立于国家行政机构之外的特殊的黑暗势力的代言机构。能在这样侵犯人权的恶劣情况下，仍然为中共政权站台，国际奥委会的委员们完全有可能已经被中共蓝金黄手段拿下。美国开始寻求解除国际奥委会在美国免税地位的做法标志着美国人开始醒悟，是正视和清算邪恶中共政权爪牙的重要一步。随着中共政权经济崩溃的趋势继续，类似于国际奥委会这样的组织势必很难再从中共政权处获得利益，在蓝金黄手段失效的情形下，中共将进一步走向被国际社会孤立的境地，这种国际形势，将为它最终走向灭亡，铺平道路。
