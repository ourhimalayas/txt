## 【热点播报】Delta Plus变体的症状及其更强的传染性

Delta Plus变体是Delta（德尔塔）变体的变异形式，早先在印度发现。据卫生专家称，到目前为止，Delta变体已在85个国家被发现，并且是南非感染率激增的主要原因。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/30102034/1-103.png)图片来自网络
冠状病毒的Delta Plus变体一直在全球范围内肆虐，其传播性和对某些药物的抗药性都在增加。印度已经宣布它是目前为止最令人担忧的变种，已在12个州的49个样本中发现了这种变种。马哈拉施特拉邦报告的病毒感染病例中Delta Plus变种的病例数量最多。

事实上，南非的传染病专家认为，由于Delta变体，该国已经经历了第三波的感染。

本月早些时候，世卫组织将Delta列入其关注的冠状病毒变体名单。

为什么Delta Plus变种会引起人们的关注？这种新的冠状病毒变体，也被称为AY.1，比其前身Delta变体的传播速度快近60%。它还更容易与肺部细胞结合，并对一些用于对付Covid-19的药物显示出更高的抗药性。ICMR的流行病学和传染病前首席科学家Raman R Gangakhedkar博士说，有两个分级突变发生在Delta变体中，而Delta变体是该国迄今为止最主要的冠状病毒菌株。这些分级突变L452R和P871R。

“这些特殊的突变增加了更高的传播效率，因此与其他存在的毒株相比，该变体可以迅速从一个人传播到另一个人，或者可以更有效地进入细胞，”Gangakhedkar博士说，他称Delta Plus是冠状病毒的最关键突变之一。

印度的顶级病毒学家说，Delta Plus变体带有Delta以及其伙伴Beta变体的症状。其中一些症状包括咳嗽、腹泻、发烧、头痛、皮疹、手指和脚趾变色、胸痛和气短。

专家们列出的其他症状并归因于Delta Plus变体的是：胃痛、恶心和食欲不振。
