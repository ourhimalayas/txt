## 【NewsExpress】欧盟与英国指控中共对Microsoft Exchange服务器发动网络攻击
` 索尼克` [轉載自GNews](https://gnews.org/zh-hans/1406671/)

#### 文章撰写:【默凡】

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/07/19093328/119487407_h5sadf165r4g56trh56i068438660.jpg)

BBC新闻网7月19日报道称：英国和欧盟指责中共国在今年早些时候发动了一次针对Microsoft Exchange服务器否认重大网络攻击，该攻击影响了全球超过25万台服务器。

欧盟首先发表声明称袭击来自“中共国领土”，而英国则表示中共国政府支持此次网络攻击，并应对此负责，预计美国也会跟进。这些国家还表示，中共国国家安全部应对其他间谍活动负责。

此次针对Microsoft Exchange服务器的网络攻击中，黑客利用了一个漏洞，该漏洞允许将web shell放置在可以充当后门的系统上，从而被进一步利用，该漏洞随后也被其他黑客组织利用。

“中共国政府支持的团体对微软 Exchange 服务器的网络攻击是一种鲁莽但熟悉的行为模式，”外交大臣多米尼克·拉布说。“中共国政府必须结束这种系统性的网络破坏，如果不这样做，可能会被追究责任。”微软早在3月份就公布了这次黑客攻击的细节，并表示与中共国有关的一个名为Hafnium的组织对此负责，随后中共国否认了这些指控。

欧盟声明称，此次入侵Microsoft Exchange服务器破坏了全球数千台计算机和网络的安全性和完整性，包括欧盟成员国和欧盟机构。这种不负责任和有害的行为给欧盟政府机构和私营公司带来安全风险和重大的经济损失，并对其安全、经济和整个社会产生重大的影响。

欧盟与英国一样，将这些网络攻击活动与同样位于中共国的两个组织（称为APT 40和APT 31）联系起来，这两个团体被指控从事间谍活动和知识产权盗窃。
