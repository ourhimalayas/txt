图片来源：The guardian

**【雅典娜快讯】，紧贴时事脉搏**

在中共病毒大流行期间，英国医院的数千名脆弱的卫生服务部门患者将受益于一种名为Ronapreve的新抗体疗法。该疗法曾提供给美国前总统川普用于治疗中共病毒。

【注：Ronapreve是第一种专门用于治疗中共病毒的中和抗体药物，已获药品和保健产品监管机构（MHRA）授权在英国使用。】

Ronapreve是两种单克隆抗体的组合，最初将针对那些尚未对中共病毒产生抗体反应的住院患者。

去年，当川普对中共病毒检测呈阳性时，该疗法被用作给他的实验药物疗程的一部分。

英国卫生部长萨吉德·贾维德(Sajid Javid)说：“我们已经为英国各地医院中最脆弱的患者提供了一种全新的治疗方法，我很高兴它最早在下周开始用于拯救生命。”

该药物将针对免疫功能低下的人群，例如患有某些癌症或自身免疫性疾病的人群。无论他们通过接触中共病毒还是接种疫苗，都难以建立对中共病毒的抗体反应。

英国卫生部门表示，很快将向临床医生提供指导意见，以便他们尽快开始处方治疗。
