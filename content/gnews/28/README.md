今日文贵先生所说中共国农药化肥价格猛涨，从210元涨价到270元，这就是中共国经济滞胀已经开始的信号。

中共僵化的体制和高度的集权，造就了习包子以小学博士的智力和见识统治中共国的奇葩现象。为迎冬奥会塑造假的蓝天白云，从去年四季度开始对煤炭限停限产，由此造成一系列恶性影响：

1. 到2020年大陆煤火发电占比例68%，虽然是掺了水的假数据，但也绝对表明煤炭短缺必定导致电力短缺，相信大陆的朋友都已经亲身体会了吧，去年四季度全国各地都在拉闸限电，大众日常生活和企业经营生产都受到严重影响；
2. 电力短缺必定加剧工业原料减产和短缺，比如钢材、建材、化工等产品，这就进一步抬高很多行业的成本，那么下游的企业的生产投资积极性就降低了；
3. 很自然经济发展就减慢了，这就是滞胀中的“滞”字表现，经济停滞，很多企业不再扩大规模甚至都关停了，大众感觉就是工作难找了，生活会很艰难。
4. 工业基础原料和电力短缺，又会影响到很多民生行业，比如蔬菜价格猛涨，还有如今天文贵先生所说的农药化肥等价格增高，另外汽柴油价格也会涨价，运输成本太高又会抬高全社会的物流成本。所以大众会感受到滞胀中的“胀”，通货膨胀，也就是什么都涨就是工资不涨。


因此，滞胀的经济停滞是关键，经济停滞了生产少了，但社会需求旺盛，更多的钱对更少的商品自然通胀。解决的办法只有加速经济活跃度，营造更良好的企业发展空间，更多的社会投资建设，并实行例如减税、减少监管、减少吃拿卡要等问题，解决了“滞”的问题，“胀”的问题才好解决。

但习包子凭“小学博士”的见识肯定认知不到这最基本的经济思路，一定会一条道跑到黑，越缺钱越大撒币，然后掠夺全社会的资源和财富，不断上演杀鸡取卵的恶行，最后中共一定崩溃，不用怀疑！
