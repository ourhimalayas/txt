## 灾难频发的中共国

一个普通的中国人在中共统治下随便逛个街，就有可能面临被广告牌砸死、被钢化玻璃砸死、被跳楼自杀的人砸死，这姑且还能称之为意外。可如果连集各种高科技与一身的飞机、轮船、动车都先后在一个国家接二连三的发生诡异的事故，还能被称之为意外吗？

眼下坠毁在广西梧州市藤县的东航（MU5735）航班飞机，在毫无预警的情况下，在两分钟之内从8869.68公尺骤降至1333.5公尺。3月22日晚间，中共民航局在新闻发布会上通报了中国东方航空股份有限公司客机坠毁事件的相关调查进展情况。

民航局通报内容如下：东航MU5735航班于3月21日13时16分从昆明起飞，14时17分保持巡航高度8，900米进入广州管制区。14：20管制员发现飞机高度急剧下降，随即多次呼叫机组，但未收到任何回复，14：23飞机雷达信号消失，后经核实，飞机在广西梧州市藤县境内坠毁。通报还提到，机上共有旅客123名（无外籍旅客），机组人员9名。截至目前，未发现机上幸存人员。事发飞机为2015年6月22日引进，截至事故发生时共飞行8，986架次，总计18，239小时。

当地一家矿业公司的监控摄像头拍下了失事飞机，在最后一刻从高空极速呈现”断崖式“坠落。
飞机坠落视频：来源网络
从监控画面来看，基本上可以断定几乎不可能有生还者出现。事发当日，中国东方航空股份有限公司对机上所有人员的遇难表示了沉痛的哀悼。这是中共国近30年来，遇难人数最多的坠机事件。事发24小时之后，周二（3月22日）中共官方在新闻发布会上称，空难搜救工作仍在进行，目前尚未发现幸存者。此前中共央视官媒报导援引广西梧州市消防救援支队欧灵表示：“我们协助搜救飞机残骸，没有找到生还者，但看到比较大的碎片机翼、（遇难者）遗体。”但相关报导随即被删除。当日下午，中共国务院调查组举行新闻发布会，会上民航事故调查中心主任毛延峰表示，在现场已找到失事飞机的一部黑匣子，外部破损比较严重，目前正在确认是飞行数据记录器（FDR）还是驾驶舱话音记录器（CVR）。23日晚间，中共民航局航安办主任朱涛确认，该黑匣子为驾驶舱语音记录器。
![](https://assets.gnews.org/wp-content/uploads/2022/03/image-2800.png)图片来源：网路
在3月22日晚的新闻发布会上，当路透社记者围绕东航失事飞机维修记录、飞行时长、事故发生时天气等问题进行提问时，东航云南公司董事长孙世英“照稿读、答非所问”，将共产党的傲慢、对生命的淡漠、对公众的愚弄，一如既往展现在全世界面前。当时路透社记者问：“这架波音737-800飞机的维修记录是怎样的？近期维修方面有发现任何问题吗？飞行员的飞行小时记录是怎样的？事故发生时的天气是怎样的？在空难发生前，机组有无和空管联系？如果有联系的话，都说了什么？飞机坠落之前，飞机的基本能否完整？最后是，USNTSB（美国国家运输安全委员会）的调查者有被提名，中方会不会邀请他来中国协助调查？”

官媒直播画面显示，孙世英回答时低头看着稿件说：“飞机当时是在平飞巡航阶段。第二个，东航正全面开展安全隐患大排查，目前航空运力比较充足，我们的安排检查也包括一些临时性、预防性措施。我们将严格按照民航安全管理规定，有序开展相关工作。”

由于客机坠毁地点树林相当茂密又位于三面环绕的山区，因此无法动用大型机械，只能靠着人力搜救，在拉网式的过程中，发现了一些私人物品和钱包，特别是一张边缘已经烧毁的纸张，上面写满平安祈福的话语，看完无不让人动容。曾经写满祝福话语之人，此刻再也见不到平安扣的主人。在没有被导弹击中，没有发生空中爆炸、解体、烟雾的情况下，不可能正副两名驾驶员同时不省人事吧！至少在两分钟应该能发出”Mayday Mayday“国际通用的求救信号。却以头朝下的方式坠落，这在世界航空灾难史上，也属于是极其罕见的。郭先生在3月23日，“文贵大直播”中为战友解密，东航神秘坠机事件始末，又是一场中共国“互害”模式的悲剧事件，中共不灭，永无宁日。

2015年6月1日，官方宣称：“东方之星”号突遇罕见强对流天气从南京驶往重庆途中，在长江中游湖北监利水域沉没。由于事发突然，共产党赖在当日，湖北省华容县发生了百年难遇的特大暴雨，降水量超过250毫米。而且当客轮经过湖北省境内的长江中航道301公里处时，又遇上了风力高达12级以上的龙卷风。直接拍打在“东方之星”上，将轮船长76.5米，总吨2200，型宽11米，型身深3.1米的客轮掀翻侵入江水中。造成454人全部落水，其中442人遇难，最小的年仅3岁，被发现时，手里还紧紧攥着一袋曲奇饼干。仅有12人幸存，其中就有船长张顺文和船的大副。该船核定乘客定额为534人。就像河南郑州发大水，千年不遇、五千年不遇的降雨让河南人赶上一样。我就弱弱的问一句，既然都知道降水量，风力高达12级，你船长为什么还要继续往里面开呢？而且你和大副都没死，这一条就够玩忽职守罪。

2011年7月23日，浙江温州市境内，由北京开往福州的D301次列车与杭州开往福州的D3115次列车发生追尾事故。在“7·23”动车特大事故新闻发布会上，有记者提问道，为什么要在现场掩埋车体？是不是想毁灭证据？时任铁道部发言人，一句“不管你信不信，我反正是信的”语惊四座，让原本是给公众答疑解惑发言人该做的事情，就因一句大实话使得政府的公信力荡然无存。王勇平完整的回答是：“事实是无法掩埋的……在现场抢险中，环境非常复杂，下面是个泥潭，施展很不方便，还要处理其他车体。把车头埋在下面，便于抢险。不管你信不信，我反正是信的。”仔细推敲这句话的逻辑，目的是“便于抢险”，手段是“把车头埋在下面”。但有一个大前提泥潭的存在，导致他话语中的手段与目的之间不具有一致性了。正确的做法应该是处理掉泥潭，将淤泥挖出运走，再填进干土压实，为大型机械的进场抢救工作开创出工作面来，然后才能实现便于抢救的目的。把车头埋进去，车厢里面都是空的，埋进去以后表面看是平的，大型机械往上一压不又陷进去了。明明是抢险的机械导致自己反而深陷泥沼需要救援，你怎么能说是便于抢险呢？

有的战友在直播中经常开玩笑地说，“让领导先走”，殊不知这句在1994年12月8日，竟要付出323条生命的代价，其中包括288名中小学生。尽管当时的副市长否认有人说过这句话，但是对于在场的幸存者来说，这句话却是难以忘怀的。近30年后，请记住那些288个生命在瞬间凋零的事实，记住那132个被大火永远改变了命运的幸存者，任何的遗忘和否认事实的真相是对逝者和幸存者的再一次伤害。

二十多年来，遇难者家属都在追问3个重复的问题：
1、是谁喊的那句“让领导先走”，到底是况丽还是唐建？
2、事故的责任人为何被轻判？
3、你们当初承诺为那些孩子造的纪念馆，现在在哪里？

当年，周永康用“稳定压倒一切”，当年事件发生时，资讯并不发达，了解事情真相的同胞并不是很多。随着时间的推移，互联网技术的快速发展，已经有越来越多的人知道当年发生过什么。2009年制作完成时长6小时的电影纪录片《克拉玛依》于2010年3月在香港科学馆首映，纪录片记录了邪恶的共产党竟连悼念活动都不允许进行，现场观众无不动容流泪。共产党反对的”四风“，即：反对形式主义、反对官僚主义、反对享乐主义、反对奢靡主义，在处理这个事情，将它所反对的，发挥到淋漓尽致。

1994年12月8日下午6时10分，新疆克拉玛依友谊馆的舞台上，几块被烤燃的纱幕条忽然落在796名师生和官员面前时，坐在前面的某官员声嘶力竭地喊了句：“同学们坐下，请领导先走”。学生们很听话，都坐在自己的位子上不动，随后，离火源最近的上级政府与教育局等所有在场的26名官员从第一排撤退到最后一排的出口处“先走”之后，教师才开始组织学生撤离，但此时电灯已全灭，大火已蔓延到剧场四周，唯一的逃生之路已被熊熊火焰堵住。就这样300多条鲜活的生命被围困在烈火燃烧的整个场馆之中，288名如花的小学生以及为抢救他（她)们没有离开火场地36名教师，被大火夺去了生命……

大火被扑灭以后上述事实得到证明，作证的便是他们自己的尸骸。还有许多具是他们老师的。同时又证明着那些上级领导和官员没有遭受到任何灾难。共产党肯定会像动车追尾事故中被发现的那个小女孩一样，面对之前对搜救工作不力的质疑时，把小女孩的得救称之为“这是一个生命的奇迹”。在这场大火中，稍有常识的人都知道，因为那26名官员离火源最近，离逃生门最远，竟真的“奇迹般”的无一人伤亡，而且走出剧场门口时还个个衣冠楚楚。

宇宙间最为邪恶的中国共产党，唯有它的党员才能说出那句，最卑鄙最无耻的一句话，“让领导先走”。中国共产党是中国人民、全世界人民遭受灾难的根源所在，我们爆料革命的战友是消灭邪恶中国共产党的新中国人，在这场史无前例的推翻中共暴政的战斗中，新中国联邦、爆料革命的战友们一定会做到让老杂毛们先走，以告慰那些冤死的灵魂。
