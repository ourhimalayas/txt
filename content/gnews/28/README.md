## 地方财政再告急，多地公务员及教师被追讨已发奖金

中国河南、江西及广东等地公务员及事业单位教师，近期接到当局追讨已发放的奖金、清退绩效奖励。最多的每人被追讨第一季度两万元人民币，并宣布以后停发奖金。基层公务员和学校教师叫苦连天，感叹莫非“紧日子”真的来了。

7月7日，江西省南昌市水利局发出文件，指根据有关规定，各单位在2021年6月7日以后发放的政府性奖励，要在十天内完成无条件退款程序。德兴市政府则要求全体教师必须在7日当天把原发放的高质量奖金统一退还到学校银行卡。

各种迹象表明国家财政出现了问题：

“不管当局怎么样宣传粮食大丰收，经济增长有多少，但是我们可以看到到处有店铺倒闭，企业倒闭，一些人失业。再加上疫情持续到现在。国家的税收必然减少。她现在财政困难是必然的，奖金等福利被削减是实实在在的。”

**追讨已发奖金不是单一事件，各地政府强制执行。**

微博网民聊天内容显示，上海、江西、河南、山东、重庆、湖北及广东均出现公务员奖金被停发，并被责令全部退回。而广东省潮州市早在7月3日，就有单位接到停发住房补贴和绩效的通知。三天后，汕尾市也跟随推行。深圳市有单位正在统计员工近几年的收入。上海亦已停发第一季度奖金。

去年上半年，中国财政赤字激增三成，地方债务激增三万四千亿元人民币。官方报道，31省市的一般公共预算收支，仅上海市出现财政盈余，其余30个省市都存在收不抵支的问题，压力最大的河南、四川、云南等省，收支缺口都在2500亿元以上。
