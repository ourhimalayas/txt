4月23日，媒体引用乌克兰情报部门的消息，俄军由于兵源不足，开始强制动员赫尔松和扎波罗热等占领区的年轻人参军，并要求哈尔科夫州沃尔臣斯克市的初级医务人员上前线为俄军伤员治疗，他们如果拒绝可能会被当场处决。

在赫尔松，俄军将从赫尔松去往乌克兰控制地区的道路进行了全面封锁，以确保强制动员能够完成。在扎波罗热地区，俄联邦安全局和俄军官员正在紧急应征入伍者，并将他们送往前线作战。

另外，俄罗斯还试图模仿2014年吞并克里米亚的模式，在赫尔松与扎波罗热举办了伪造的独立公投。但是，所谓的“公投”尚未举行，他们就开始强制征兵，暴露了俄军兵源严重缺乏的事实。

对此，英国国防情报局指出，俄罗斯此举已经违反了《日内瓦第四公约》，其中第51条规定，占领国不得强迫被占领地区的人在其武装辅助部队中服役，也不得为确保他们自愿入伍而施加压力或进行宣传。
