产经新闻12月4日消息：警视厅有组织犯罪调查组，以涉嫌违反移民管制和难民法，逮捕了东京都杉并区高井东的行政书士，61岁的石川和仁，以及他的两名中共国籍助手，住在埼玉县川口市川口的，31岁的温建欧和餐饮店员工周春明。警方称，这三名嫌疑人已经承认了这些指控。

据警方称，一名中共国籍男子在中共国的一个社交网站上宣传他可以办理在留资格变更。周某看到后，与石川和文某工作的行政书士事务所联系，另一名中共国人，也是主要嫌疑人，向他们介绍了房地产情况。

当时，石川等犯罪嫌疑人，为留学身份的周醒嫌疑人办理了虚假的，从事零售业务的在留资格。作为主犯的中共国籍嫌疑人，在2016年8月至2021年10月期间帮助大约80名中共国籍留学生办理了旨在延长在留资格的签证变更手续。非法获利达4500～6000万日元。

该嫌疑犯于2020年4月24日，向东京入国管理局提交了虚假的居留资格变更许可申请，并于6月25日获得许可，将其居留身份从 “留学”变更为 “经营管理”。
