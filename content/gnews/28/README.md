北领地首席部长迈克尔·冈纳在喧嚣的新闻发布会上说，很简单，反对强制新冠疫苗的人是“反疫苗者”，并指出他所在州“预期”会出现更多阳性病例。

在一次新闻发布会上，他问应在新冠疫苗强制接种问题上政府持强硬立场是否会使一些人放弃接种疫苗时，冈纳很清楚表示，任何达不到100%支持政府日益严厉的疫苗接种规定的人，都是真正的“反疫苗者”，即便他们自己已完全接种疫苗。澳洲人要么伴随着病毒，要么伴随着（政府）的强制规定。

“如果你为任何反对疫苗的人开绿灯，给予安慰，支持他们，你就是一个反新冠疫苗主义者。你的个人疫苗接种状况就会完全地失效，”冈纳宣称。

“反疫苗主义者”，这个词以前是指反对为孩子接种疫苗的父母，在新冠大流行期间，它的含义已经扩大到任何反对为自己接种疫苗的人——现在，在狂心的疫苗支持者眼中，包括任何反对强制接种的人。

冈纳继续说：“如果你以任何方式、形式站出来开展反对政府强制疫苗的活动，那么你绝对是反疫苗的。”他坚持认为该州的疫苗接种授权对拯救生命是“绝对至关重要”。“我永远不会放弃支持疫苗，任何为这项强制令而来的人，你们都是反疫苗的，”冈纳咆哮道。

北领地凯瑟琳镇的封锁将延长至当地时间周三晚上6点，因为有1300个测试结果仍未公布，当局“仍在期待更多的阳性病例”。周一发现了两个病例——一名67岁的男子和一名33岁的母亲，据说她是“确诊病例的密切接触者”。然而，她的孩子多次检测结果均为阴性。

尽管病例很少，冈纳坚称该地区“极有可能”还有其他阳性病例，理由是该社区的废水集水区“持续出现阳性迹象”，但允许附近的罗宾逊河地区解除封锁，只要居民不离开社区，就可以自由走动。

部长坚持认为，“强制，而不是鼓励”是使疫苗接种水平达到预期目标的唯一途径。
