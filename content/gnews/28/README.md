## 伊州成为美首个要求学校开设亚裔美国人历史州

#### 编译：文复   
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/07/09172047/Screenshot-2021-07-09-2.19.20-PM-edited.png)网络裁图
（路透社） 伊利诺伊州州长普利兹克 J.B. Pritzker 周五签署了一项法案，要求公立学校教授亚裔美国人的历史，这是美国各州在对针对亚裔人的暴力行为日益关注的情况下迈出的第一步。

该法案要求从 2022-2023 学年开始在公立小学和高中教授亚裔美国人的历史单元。

“我们正在为真正回顾我们的历史的意义设定一个新标准，”普利兹克说，他在芝加哥郊区斯科基的一所高中签署了这项立法。

这位民主党州长说，让儿童了解亚裔美国人在美国历史上的贡献将有助于遏制错误刻板印象和歧视的蔓延。

在芝加哥亚裔美国人推进正义倡导组织的领导下，支持者自 2020 年初以来一直在推动该法案。但在 3 月亚特兰大地区对亚裔女性的致命袭击之后，该法案在州立法机关获得了支持。

在为应对在中共国中部城市武汉出现的冠状病毒大流行而采取封锁措施后，许多亚裔社区在针对他们的一波暴力事件中处于紧张状态。

《公平教学亚裔美国人历史法案》规定，课程应包括教授亚裔美国人为促进民权所做的努力，以及他们对艺术和科学以及国家经济和文化发展的贡献。

芝加哥亚裔美国人推进正义组织执行董事 Grace Pai 表示，该法案将使伊利诺伊州从幼儿园到高中的 100,000 多名亚裔美国学生受益。

Pai在一份声明中说：“很多学生没有机会了解他们社区的贡献或他们

家庭的移民故事，从而致了一种受排挤的感觉。
