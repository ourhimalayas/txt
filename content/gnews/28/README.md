**◇“台独”定义明确化的主张**

对使国家统一进程法制化的方案也有审慎的论调出现。

据3月9日中国台湾网(国台办系) 新闻网站报道，全国人民代表大会代表、北京大学台湾研究院院长李义虎对记者团表示，“制定统一法的条件还不具备”。他说，有必要首先由全国人大常委会给出反国家分裂法第八条的具体解释，弄清楚哪些行为定义为“台独”，那些是跨越本土方面的红线。

第8条规定：(1) “台独”分裂势力以任何名义、任何方式造成台湾从中国分裂出去的事实；(2) 发生将会导致台湾从中国分裂出去的重大事变；(3) 和平统一的可能性完全丧失，国家需要采取非和平方式及其他必要措施，捍卫国家主权和领土完整。一般认为是设想台湾宣布独立或外国军事干预等，但并未提及具体情况。

无论是《祖国统一法》，还是《反国家分裂法》第8条的解释，看起来都代表了并不看好吞并台湾前景的习近平政权的焦躁情绪。中共方最近点名批评与民进党对立的台湾最大在野党、国民党的一部分接近美国的举动，对整个台湾的态度越来越盛气凌人。

在独裁体制企图吞噬民主体制的意义上，中台关系与俄乌关系相似。而且，拥有先进工业的台湾经济发展水平远高于身为高于发展中国家的中国。对于享受政治自由和经济富裕的台湾人来说，没有任何理由乐意进入既无自由，经济发展水平又低下的中共国统治之下。

如果中共方高举金科玉律般的“一个中国”旗帜，坚持以武力和单方面法律威吓为依据的政策，台湾只会更加远离。
