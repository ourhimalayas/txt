` developer001` [轉載自GNews](https://gnews.org/zh-hans/1363165/)

致中共灭爆小组的公安、国保及混杂于各农场和编辑部的各特勤人员：

近期G-News编辑部收到多位国内作者反馈，由于个人信息被中共国灭爆小组的公安、国保及混杂于各农场和编辑部的各特勤人员起获，而发生了对部分G-News作者喝茶、恐吓事件。

针对这数起G-News作者被喝茶、被恐吓事件，G-News编辑部现严正声明如下——

1、请所有被喝茶、被恐吓的G-News作者第一时间给[email protected]

邮箱联系，将您的被喝茶、被恐吓经过详细说明，将被喝茶、恐吓过程中的直接参与者和指挥者的详细信息一一列明，被喝茶的G-News作者们若持有中共的官方文件或者在此过程中保留有电话录音请一并提供官方文件的影印件及电话录音附本；

2、编辑部将对收到的被喝茶、被恐吓的G-News作者提供的信息进行认真甄别，对于所有侵犯G-News作者个人权利和自由的责任者，一经核实将通过法律渠道向相关国际人权机构报备；

3、在得到当事G-News作者授权的情况下，G-New编辑部保留随时在G-News上公布严重侵害G-News作者个人权利和自由的责任人及相关事件细节的权利；

4、所有参与抓捕及恐吓G-News作者的军警特及国保人员，请务必注意，你们每个人都有枪口抬高一寸的权力，这不是对你们的威胁也不是对你们的恐吓，你们的每一次抬高枪口都是对你们自己的救赎；

5、在任何情况下，保护G-News作者的安全都是G-News编辑部的第一要务，G-News编辑部不会放弃任何一位身临险境的战友，G-News编辑部也永远不会忘记任何一位把枪口抬高一寸的体制内战友。
