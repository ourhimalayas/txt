据《零对冲》报导，mRNA疫苗领域的先驅者罗伯特•马龙（Robert Malone）博士在7月16日推特评论说：“世界上接种中共病毒疫苗人数较多的国家，中共病毒感染人数正在激增，而接种人数较少的国家的现状则相反。这个情况让我有点担心。”

马龙博士推文中简介：欧洲接种率较高的国家比如84.95%的马耳他、67.82%的英国、67.38%的荷兰，现在的中共病毒感染人数正在激增，而其它15个接种率较低的欧洲国家则没有这问题。全世界有15个国家分别是阿联酋、马耳他、塞舌尔、冰岛、巴林、以色列、智利、乌拉圭、蒙古英国、卡塔尔、加拿大、新加坡、丹麦和荷兰，人均接种了超过1剂中共病毒疫苗，感染人数也在激增。而据称每人都接种了两剂疫苗的小国直布罗陀，感染人数突然暴增。伊丽莎白女王号航母打击群上的1600名英军士兵均接种了两剂中共病毒疫苗，在遵从了严格的卫生条例和保持间距的情况下，仍有100多位士兵感染。

中共病毒疫苗是实验性疫苗，没有经过第三期临床试验，也没有获得美国食品药品监管局的批准，只获得了紧急使用授权，采用的mRNA技术也不成熟。这样的疫苗仓促地在人群中大规模推广使用只会酿成巨大的人道灾难。要想解决大流行危机并阻止下一场大流行，必须找罪魁祸首中国共产党，要求他们公布病毒的全部真相。只有这样才可能有安全的疫苗，在此之前可以用已经被广泛证明有效的硫酸羟氯喹和伊维菌素等药物来预防中共病毒。
