## 日本无印良品商标遭中企侵权案山寨再次告赢正版

（图片来自网络）

在中共国，山寨品牌抢注横行。许多国际品牌与名人在与中共国企业的商标专利战中，屡屡败诉，赔偿案例多不胜数。日本原厂“无印良品”就是其中之一。

据《希望之声》2021年11月7日报道，中共国的盗版“无印良品”状告日本正品“无印良品”的商标侵权案，日前已由中共国法院做出判决。日本原厂无印良品再度被判败诉，须向同名的中共国原告赔偿人民币40万元，并需做出公开说明以“消除影响”。

据中共国媒体报道，第2轮“无印良品”商标案由北京市朝阳区法院一审审理，判决书于11月4日对外公布。最终判决日本原厂赔偿40万元，其中含30万元的经济损失费及10万元的“合理开支”。

“MUJI无印良品”于1980年由日本“株式会社良品计划”成立，并注册商标。2005年4月，“无印良品”进军中共国，并在上海成立“无印良品（上海）商业有限公司”，负责中共国市场的营运。

中共国版的“无印良品”商标是由一家中共国公司于2001年注册，之后商标被转让给北京棉田公司。2011年7月，北京棉田公司又再次注册“无印良品”商标，且授权给“北京无印良品公司”使用。

外界普遍认为，这是中共国企业抢注日本“无印良品”原厂商标，因此质疑其是“山寨版”。2015年，北京棉田及北京无印良品将日本原厂告上法院，称日本“无印良品”“构成商标侵权”。这是北京棉田发起的第1轮诉讼。

2019年11月4日，北京市高级法院判决日本无印良品“侵权成立”，不但要求其立即停止“侵权行为”，并判其赔偿62.6万元，且要发布30天声明以“消除影响”。

日本原厂对此发表声明，指其在中共国销售的布、毛巾、床罩等商品的一部分，被其他公司抢注商标。这一声明令北京棉田不满，认为“抢注”一词“足以导致公众认为棉田公司是不法之徒”，进而认为棉田公司的商品是“山寨货”，因此又再告上法庭，双方陷入第二轮诉讼。

最终，不出所料，“山寨”又一次“战胜”了原厂。
