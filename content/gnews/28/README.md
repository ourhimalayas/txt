## 伊朗、中共国和俄罗斯举行第三次联合海军演习

伊朗、中共国和俄罗斯周五(1月21日)在北印度洋举行了第三次联合海军演习。伊朗武装部队发言人塔约尔迪尼（Mostafa Tajoldini）少将告诉伊朗国家电视台，伊朗武装部队和革命卫队的海军正在1.7万平方公里的海域范围内参与“2022年海上安全带”演习。

此次演习将包括各种战术演练，例如营救着火的船只、解救被劫持的船只以及夜间对空中目标的射击。塔约尔迪尼说，演习于周五凌晨开始。他说，这三个国家从2019年开始进行年度联合海军演习。塔约尔迪尼还说，“此次演习的目的是加强区域安全极其基础，扩大三国多边合作，共同支持世界和平、海上安全，打造所谓“海洋命运共同体。”

伊朗强硬派总统总统莱希（Ebrahim Raisi）自去年6月上任以来奉行“向东看”政策，以深化与中共国和俄罗斯的关系。去年9月，伊朗加入了中、俄领导的中亚安全组织“上海合作组织”。伊朗外长阿米尔-阿卜杜拉希安（Hossein Amir Abdullahian）上周访问了中共国，总统莱希则在周四访问了俄罗斯。
