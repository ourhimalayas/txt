![](https://assets.gnews.org/wp-content/uploads/2022/04/C7C06EBB-BF58-42F2-9175-937D6FFDFCFC.jpeg)
美国国家安全委员会（NSC）印太地区协调官库尔特-坎贝尔。图片来源：路透社/托马斯-彼得

据路透社4月25日消息，由美国国家安全委员会（NSC）印度-太平洋地区协调官库尔特-坎贝尔率领的美国政府代表团于4月22日访问了南太平洋岛国所罗门群岛，并拜会了总理索加瓦雷。他对索罗门群岛和中共国签署的安全协议表示关切，并警告说如果中共国军队驻扎在索罗门群岛，美国将采取适当措施。

根据白宫的一份声明，索加瓦雷先生承诺说该国与中共国的《安全协议》不会导致建立军事基地、长期驻扎或建立武力投射能力。

白宫没有提到在这种情况下将采取怎样的具体对应措施，但解释说，“美国强调，它将与区域伙伴协商，并密切关注事态的发展”。

据报道，美国政府代表团还转达了其加快美驻所罗门使馆的开放和扩大提供疫苗的意向。

所罗门和中共国宣布两国在20日之前签署了一项《安全协议》。美国及其盟国对此已表示关切。
