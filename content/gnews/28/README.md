## 中共国缺的不仅是电，还有智商

by：银河勇气星｜天雷滚滚

9月30日，中共国媒体公布了第一批纳入中共精神谱系的伟大精神，我用仰慕的眼光数了一下，有46种之多。从这标题来看，下面应该还有第二批、第三批……。我的乖乖，不用说，中共国多的是“精神病”，缺的是智商。我们来扒拉扒拉一下

最近拉闸限电这波操作涉及面实在太广，不仅把半导体的企业都拉了，甚至连东北民用电都是拉闸范围，现在东北也差不多入冬了，有没冻死人，这个不得而知，要死也死在屋里，谁知道是冻死的、猝死的还是得了新冠病死的，咱中国别的不多就是人多，死几个人不算什么。你看光头李毅不就说了吗，全世界得新冠死了几百万人，咱中国就死了4000人，哪简直就是没有死人吗？说了后，还不够踏实，又追问了一句，“是吧”。这4000多人死了都不算死，冻死几个算啥，更何况，你能证明这是冻死的吗？废话不说，这东北居民区无预警拉闸，人被关在电梯，居民做不了饭，手机没信号，23人因停电导致煤气中毒送进医院，甚至红绿灯信号也没有

黑龙江的这一波骚操作可能影响了我党伟大光荣正确的一点点形象，于是，连中南坑的首辅大人李中堂都出来说话了，说要保障民生用电。这李中堂说话就是管用，话音未落，各地都纷纷行动，不仅要保民生，制造业也要保，以体现天朝大国世界工厂的担当。你郭文贵不是说中共国没外汇了吗，我就给你打脸“啪啪啪”响，我从越南进，从印尼进，从美帝进，至于越南这些国家产不产煤，煤质好差，你管不着，煤的价格比澳洲贵多少，你也管不着。我只买贵的，不买对的。只要皇上开心，只要不直接从土澳买煤，我甚至不远万里（15700公里）从哈萨克斯坦进口煤炭，绕半个地球历时一个月到达浙江的舟山岛，以显示我朝“为国为民的仁心”
![](https://assets.gnews.org/wp-content/uploads/2021/10/未标题-1-1.jpg)图片来源：网路
终于有点点明白了，原来就是这“一盘大棋”！这下大棋目的就是让中国人世代吃草，也要搞断世界的产业链，从而让美帝恶性通货膨胀，让全世界通货膨，导致全球性经济危机，然后你们这些昔日对我党说三道四的帝国主义跪下来求饶臣服，甘愿永世为奴，再趁机把老是扒我底裤的郭文贵和爆料革命的气焰给打下去，体现我党英明神武，一举多得！厉害，真是厉害，我谁也不扶，就服中南坑老杂毛：为了饿死人家，自己先表演吃屎；为了让人家跪下，自己先干趴下

原来这棋不是国际象棋，也不是中国象棋，而是中共象棋，或者是反智象棋。规则由我党制定，棋盘暂定在我天朝媒体，在我横店影视城，想怎么下就怎么下，怎么下都是我党赢，帝国主义都是搬起石头，不，是搬起棋子砸自己的脚。这不，刚刚手撕完鬼子，现在听说吴京又开始在《长津湖》脚踹美帝坦克，然后又驾着美帝坦克把美国大兵杀得满地找牙了。本来，电影就是电影，演给老百姓嗨一下，让他们自觉长成镰刀的样子，方便收割，自己明白也就行了，不料入戏太深，感觉不够刺激，要把棋盘搬到世界去，改变世界的下棋规则

1840年，中英第一次鸦片战争，为克制强敌，有傻逼向林则徐进言：洋人形似犬羊，喜食肉类等不易消化之物，如无中国茶叶和大黄，则会排泄不通。林则徐深以为然，慨然写下《示谕外商速缴鸦片烟土四条稿》：况茶叶大黄，外夷若不得此，即无以为命……这战法太强大，不卖茶叶给洋人，他们就拉不出粑粑，最终必腹胀而死。您看，好大的一盘棋。道光阅罢，龙颜大悦，遂下旨：那还跟洋人逼逼个啥啊，速开战

据说，现在今上也越来越自信了：这些西方国家，只是因为这几百亿的养老金入投到恒大，就急得求爷爷告奶奶地到处找他，澳洲总理也为了多卖一点矿产资源开始四处求人联系他，只为通一次电话。看来帝国主义真是纸老虎，一碰就破，嗯，毛主席说得不错。加上咱天朝煤不多，镰刀形的韭菜多；武器不精，血肉砖块多；粮食不多，二氧化碳能造淀粉的技术高。还有更多振奋人心的消息：无人机世界领先太多，无人能模仿；太空发电站即将建立，可供人类用电十万年。技术又是世界领先。千年不遇的好时机，干，这是成就万年一帝的好机会

我们都处在这以假乱真的世界：韭菜被割着割着，就当自己是镰刀了；太监看皇帝跟妃子做着做着，也真高潮了；演员演着演着就当自己真是皇上了；谎言说着说着，就当真是宇宙真理了；我们被迫在这个世界生存着，逃无可逃。只有一种情况才可能清醒：当社会主义铁拳砸烂韭菜时，也就反目成仇了。据说，有一个小企业主崇拜卢克文，深深为其睿智的思想和战狼精神折服，订了会员，时时打赏，见不同意见者便冲上去撕咬……直到限电潮来临，刚复工就被迫停产的他看到偶象卢逼文居然写了一篇“大棋”文，力陈拉闸限电的好处，瞬即粉转黑，捶胸顿足骂其伤天害理，翻脸比翻书还快。

韭菜如此，中南坑的影帝们呢？投毒全世界，拿疫苗再来第二次生物战，又拿经济超限战来威胁胁迫他们。你当西方的民众都是中共国的韭菜呢？当灭共浪潮汹涌而来，正道主义的铁拳砸向你们时，不知道这些老杂毛有没有道光皇帝的好运。
