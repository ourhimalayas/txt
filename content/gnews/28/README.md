**新闻评论：**

强制疫苗接种已经造成西方民主社会结构的改变，并人为地制造族群撕裂的二元社会和医疗歧视。现在，美国拜登总统强制疫苗接种令遭到军队前所未有的抵制，美国空军首当其冲，已有12000名军事人员拒绝疫苗强制接种，其他各个军种都有不同程度的抵制。虽然，军方领导人威胁解雇或者军事法庭指控，但是，军事专家指出，此举将影响美军战力，因为军事人员的大规模解雇可能导致美国无法成功应对国外军事挑战和各种危机。

**翻译正文：**

多达12000名空军人员已经选择违背五角大楼的疫苗强制令而选择不接种新冠疫苗。因为这些人已经不可能在规定时间内完成接种，对于军方领导人而言，这将是第一次重大考验。

根据美国空军的数据，绝大多数现役飞行员（超过96%）至少完成了一剂疫苗的接种。但官员们警告称，除非获得医疗或宗教豁免，否则违抗军令拒不接种的人将受到处罚，可能面临开除或者受到军事法庭的指控。

空军领导人们所面临的问题是，如何能够解决最高医疗优先级的疫苗接种问题，但同时这一高度政治化的问题也会引起潜在的大规模异议。

从今年11月底到明年夏天，伴随着规定最后期限的到来，疫苗接种已成为针对于各个军种部门的头号难题。甚至在某些情况下，拜登总统的强制接种令遭遇了前所未有的抵制。

华盛顿智库新美国安全中心的军事政策专家凯瑟琳·库兹明斯基说，一波解雇浪潮可能会动摇空军人事制度，并在被要求必须随时准备好应对危机的作战单位内部造成重大挑战，尤其是如果过多的解雇飞行员或飞机维修人员等一些重要岗位的人士。

她说：“令人吃惊的事实是强制接种令可能会导致我们应对突发战争的准备不足。”

空军是仅次于海军的第三大兵种，拥有32.4万名现役飞行员，所以即使有3%的人员离职也是相当可怕的。

空军拒绝透露飞行员中拒绝接种疫苗与寻求豁免的人数。目前他们没有告知多少飞行员正按计划退出军队。空军发言人安·斯特凡内克说，空军将在下周的最后期限过后公布其中的一些细节。

五角大楼发言人约翰·柯比本周说，一般来说，对于任何疫苗的宗教豁免批准的数量是“非常有限的”。官员们说，陆军是最大的人员服兵役机构，但它只能批准一项永久医疗豁免，而不能对冠状病毒疫苗给予宗教豁免。在过去的七年里，海军没有对任何疫苗接种批准任何宗教豁免——其中包括冠状病毒疫苗。

据官方人士透露，目前，空军的疫苗接种率已经在这几周放缓，而且，现在接种也已赶不上规定的最后期限。这也表明，空军基本对所有希望接种军人完成了疫苗接种。

总体而言，自8月份国防部领导人通知全国210万军队必须接种疫苗以来，军队的疫苗接种率一直在攀升。柯比说，近87%的现役军人已经完全接种了疫苗，尽管预备役军人和国民警卫队成员的犹豫拉低了军队整体的接种率到68%左右。

拒绝接种疫苗的服役人员面临一系列潜在的纪律（处罚）。空军表示，拒绝接种的飞行员在与指挥官交谈后，或者如果他们的请求被拒绝，可以改变主意，不过，进一步不遵守规定将面临一系列不断升级的惩罚，包括非自愿离职或军事法庭指控。

类似的违规行为在其他服务部门可以以不同的方式处理，前空军律师、国家军事司法研究所所长雷切尔·范·兰丁汉姆认为这种方式不公平。
