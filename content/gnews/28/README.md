据路透社2021年7月18日报道 在凯恩斯的赛前训练营中，澳大利亚参加东京奥运会的整个田径队在发生CCP疫情恐慌后被隔离在房间里。

澳大利亚奥委会（AOC）周日证实，一名从墨尔本赶来的不愿透露姓名的官员进行了一次测试，之后的测试结果是阴性。

运动员们从周六中午到当天晚上一直被隔离在房间里，之后才被允许参加热身赛。

澳大利亚田径队采取了谨慎的态度，因为澳大利亚田径队实施了超出要求的额外测试协议。要求运动员们在解决问题之前，营地的运动员和官员都被隔离在他们的房间里。

这名官员的测试结果不确定，随后又进行了两次阴性测试，他必须在重新加入团队之前完成额外的测试。
