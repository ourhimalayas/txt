**中共对教育的投入与义务教育现状**

由于数据缺失，中共对教育的投入，在此仅以2018年为例。
2018年国家财政性教育经费支出，约占GDP的4%，居世界第110位以后。2018年中共中央财政支出，公共安全（维稳）支出近1991.10亿，教育支出1711.22亿（中共数据，包括对教育官僚的支出）。2018年联合国“人类发展署”的数据指出，中国经济总量的70%被政府用于行政开支。

以上是面上的数据，不包括大量被中共盗国贼窃走的财富。因而，如此投入显然无法使所有适龄儿童入学。为此中共青少年发展基金会成立了“希望工程”忽悠人们捐款助学。此外，还有众多以海外华人为主提的基金会，如海外中国教育基金会（OCEF）、全美华人文化教育基金会（ACCEF）、一对一彩虹助学计划、爱心基金会、德国逸远基金会等积极救助中国义务教育。尽管如此，中国儿童少年基金会表示，“2018年中国300万适龄儿童因贫困造成失学”。“以1个儿童1年学杂费450元计算，300万失学儿童1年教育经费需10多亿元”。

**凛冬开始，义务教育教师的工资发不出来了**

早在2021年9月份，美国跨国投资银行与金融服务公司高盛集团的经济学家在一份报告中称，中共地方政府隐性债务总额已增至2020年底的53万亿元，这个金额相当于中共国GDP的52%。

尽管如此，中国依然无耻地自称2021年GDP增长8.1%，并预计2022年增长8%以上。然而，江苏苏州，贵州、安徽、河南以及辽宁等省份日前都传出有教师被拖欠薪水；同时，最近网上还盛传江苏、浙江、福建及上海多个省市公务员、医生、教师等大幅降薪，降薪幅度二成左右。可见，国务院教育督导办发出提醒涵，“将义务教育教师工资列为政府必保支出”。又是在欺骗愚弄奴隶们！
