## 拜登刚满79岁民主党便开始担忧哈里斯

![](https://assets.gnews.org/wp-content/uploads/2021/11/image0-8-7.jpg)11月6日，总统乔·拜登在白宫国宴厅向副总统卡马拉·哈里斯打手势。（图片属于联合通讯社）

拜登总统周六的79岁生日，是理解本周民主党对副总统卡马拉·哈里斯发出警报的关键。她的支持率是否如调查所说的那么糟糕？她是否被白宫不公平地排挤了？这种忧虑事关乔·拜登和2024年总统选举。

民主党人慢慢发现，拜登不太可能寻求第二个任期，特别是当选民观察发现拜登在总统位置上的工作举步维艰。尽管与2012年他和保罗·瑞安的辩论相比，他的状态有了明显的变化，但在2020年的竞选活动中，因为年龄而质疑他的能力被视为禁忌和歧视。

但根据哈佛-哈里斯公司最近的一项民意调查，58%的选民现在认为拜登先生“太老了，不能当总统”，其中包括27%的民主党人。同时，53%的选民对他是否“精神上适合任职”表示怀疑，包括16%的民主党人。上周五，拜登先生接受了年度体检。我们希望他能顺利通过体检，并能像现年97岁的吉米·卡特一样长寿。

不过说到卡特先生，他在2019年是这样说的：“如果我再年轻15岁，即使我只有80岁，我不相信我可以承担我当总统时经历的职责。”拜登在2024年将82岁。当民主党人的目标是驱逐川普时，他们倾向于忽视这一点。但川普已经被他们拉下马了。如果拜登的4万亿美元支出法案获得通过，特别是如果共和党在2022年重新夺取国会，那么他对民主党的作用将进一步下降。

那么，还有谁能胜任？哈里斯女士并没有证明自己是一个令人信服的答案，因此（民主党）出现了恐慌。她的支持率正在下降，最近的一项民意调查显示其支持率为28%。CNN本周的一份报告显示，哈里斯女士的圈子里有人抱怨她没有得到成功的安排，她的通讯主管即将离职。

拜登确实要求哈里斯女士在棘手的问题上承担起领导责任：移民和边境问题，还有国会的投票法案，这些问题都不会通过。其他副总统也在努力寻找自己的角色定位，特别是当椭圆形办公室里的人知道拜登的行事方式的时候。

白宫新闻秘书珍·普萨基说，对哈里斯女士的批评带有性别歧视和种族主义色彩，特别是来自“右翼的一些人，他们对她进行攻击，因为她是第一位女性，第一位有色人种女性”担任副总统。对不起，一个巨大的右翼阴谋并不能产生28%的支持率。只有哈里斯女士能做到这一点，而她在前述方面的挣扎几乎不新鲜。

在第一批选票投出之前，她就退出了2020年的总统竞选。她在南卡罗来纳州的民调中排在第七位，那里的民主党初选选民大多是黑人。该党提名拜登先生为看守总统，唯一的工作是击败川普先生。作为对身份政治的一种让步，拜登先生承诺挑选一位女性副总统，然后他选择了哈里斯女士。
