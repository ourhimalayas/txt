**1.习近平同普京举行视频会晤并发表联合声明，正式决定《中俄睦邻友好合作条约》延期**

北京时间6月28日，中共国国家主席习近平同俄罗斯总统普京举行视频会晤。两国元首宣布发表联合声明，正式决定《中俄睦邻友好合作条约》延期，并发表联合声明。其中，中俄重申恪守《不扩散核武器条约》义务；重申坚持《关于禁止发展、生产、储存和使用化学武器及销毁此种武器的公约》的目标和任务；双方强调，《禁止发展、生产及储存细菌（生物）及毒素武器和销毁此种武器公约》作为国际安全体系的支柱之一具有重要意义。据悉，杨洁篪、王毅参加会议。

**2.王毅将与29日出席二十国集团外长会**

北京时间6月28日，中共外交部发言人汪文斌宣布，中共国务委员兼外长王毅将于6月29日以视频方式出席二十国集团外长会，并将就多边主义、全球治理等议题进行讨论。

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/29093538/003-3.png)

**3.中共国汽油、柴油价格上调**

北京时间6月28日，中共国家发展改革委决定，自2021年6月28日24时起，中共国内汽油、柴油价格每吨分别提高225元和215元。

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/29093703/004-2.png)

**4.中共国猪肉价格再次大涨**

北京时间6月29日，中共称国家出手回购猪肉，猪价再次大涨。其中东北地区和华北地区上涨明显，华东、华中、华南等地上涨缓慢。

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/29093839/005-3.png)

**5.四川养老金参保15年以内，每满一年仅上涨0.7元**

北京时间6月29日，四川省公布养老金上涨细则，与缴费年限挂钩一项引人关注。据悉，与缴费年限挂钩一项规定，参保年限15年以内部分，每满一年只上涨0.7元，还不到1元。这是目前全国单项调整最低水平。

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/29093953/006-1.png)

**6.中共国下半年预计发放3万亿地方债**

北京时间6月28日，按照财政部数据，今年前5个月，中共国地方政府发行新增债券9465亿元，按照今年全年地方政府债券4.47万亿元限额估算，下半年预计发行3万亿元左右的新增地方政府债券。
