太好玩了，原来好的穿一条腿裤子的地方政府和开发商，现在大难临头却互相掐了。

先看两个通知：

一个是“郑州航空港经济综合实验区国有建设用地使用权收回公告”。因郑州超营生活服务有限公司未按国有建设用地使用权出让合同，合同编号儿为（豫正港出让第039号），约定缴清土地出让价款按出让合同约定，现解除郑州航空港经济综合实验区国土资源局与郑州超营生活服务有限公司签订的国有建设用地使用权出让合同，并收回土地。

另一个是“关于暂停办理绿地地产济南有限公司及其子公司商品房预售许可”的函。发送单位：市行政审批服务局、济阳市住房城乡建设局、济南管委会建设管理部。近期，绿地地产济南有限公司及下设子公司为开发主体，在我市开发建设的多个房地产开发项目存在延期交房，涉嫌违规销售、延期交付等问题，引发多次群访、群诉，严重扰乱了我市房地产市场秩序。为保障购房群众利益，我局责令绿地地产济南有限公司及其子公司进行整改。整改完成前，请贵局暂停绿地地产济南有限公司及其下设子公司商品房预售许可证办理工作。

中国有句俗语，叫做夫妻本是同林鸟，大难临头各自飞。其实不只是夫妻，很多的利益相关方在大难来临的时候，也是各自飞的。以前房地产行情好的时候，地方政府和房企恨不得穿一条裤子。有的时候为了烘托一下土地拍卖市场的气氛，开发商明知道买了这块儿地是要亏钱的，但依旧会为了给地方政府站台，用高价买了这个亏钱的地。但现在房地产行情不好了，开发商没钱买，明知道是亏钱的地，地方上看到开发商都舍不得掏钱了，也开始处处防着了。就此，曾经哥俩非常好的亲密关系就这么破裂了，而且互相掐了起来。这就发生了济南市把绿地的新房预售证儿给暂停了，郑州把恒大以前拍的一块儿没交钱的地给收回了，后续估计还会有很多城市都会和开发商撕破脸的。



中国这么多年屡屡飙升的房价，原因就是地方政府与房地产开发商合谋，当然还要加上银行，一起搜刮老百姓未来几十年劳动的成果。说过多次，自从1994年地税、国税改革之后，中央政府拿走了地方上的大部分税收，但是像教师工资、公务员工资、当地的道路桥梁维修，这些责任，还是要交由地方政府来管理。所以地方政府是没钱做这些事的，这是钱与责任的不匹配。之后，中央就开了一个口子，是允许地方政府用土地出让金恶来弥补地方财政的不足的。于是，地方政府一个个就发了疯似的开发房地产，他们为了这个土地出让金，而房地产开发公司当然也是这条线儿上的一个利益相关方，开放房产他们有钱赚啊。于是就合谋一起，把房地产这个市场烘托了起来。

这才是中国房地产这20年来迅猛、快速发展的最主要原因。但是，也是由于房地产价格的飙升，使得现在的人们越来越不敢结婚，越来越不敢生孩子了。有人统计，这四年来，从2016年到2020年，新生儿的出生率就下降了45%，从2016年的1700多万下降到了2021年应该不到一千万。所有的经济最后都是靠人的，很多人说，我们有AI，有高科技呀，有机器人，这些都是没有用的，因为人除了生产之外，最主要的一个功能是消费。机器人和AI是不可能消费的。

而现在，因为出生人口越来越低，已经没有了接盘侠。很多现在年轻的孩子，九零后、零零后绝大部分都是独生子女，父母早已经给买好了房子，所以这个房地产市场已经到了拐点，就是不可能再像过去那么多年大力发展了。在这种情况下，那房地产开发商和地方政府当然就是要为了自己的利益了，就开始互相掐了起来。你没给我钱，我就把你的预售证儿给收回，或者是把你没交钱的土地给收回。之前地产行情好的时候，这都不是事。这也说明地方政府已经明白房地产市场走到了尽头。

接着再看一下各个大的地产公司最近的情况。自9月20日股价暴跌87%后，新力控股已经短暂停牌超过20天，新力的债务问题正走入更加严峻的形势中。10月11日晚间，新力控股集团发布公告称，此前的一系列问题已经导致境外融资安排出现技术性违约，且目前没有资源可以用于偿还本月18日到期应付的2.5亿美元债本金及利息，在补救措施上尚未制定任何具体方案，接下来将继续短暂停牌。这意味着新力将很快从公开市场中的技术性违约走向债务实质性违约。

10月11日，当代置业宣布：10月25日到期的2亿美元的债券余额延期兑付，目前正在寻求投资人的谅解和同意。一位香港投资人士认为，这一宣布等同于官宣违约。

上一个交易日，当代置业发行的5笔美元债全线下跌，其中三笔跌幅超过30%，两笔跌幅超过10%。彼时资本市场普遍认为，10月到期的美元债违约将不可避免。

当代置业美元债暴跌是新一轮地产美元债下行行情的缩影。过去一个月，有140只地产美元债的跌幅超过20%；受花样年违约事件影响，10月4日-10月9日一周，有44只地产美元债跌幅超过20%。

一位机构投资人士告诉经济观察网：近期美元债暴跌与9月以来恒大、新力、花样年等房企美元债付息和兑付不及时有关，房地产行业频繁的流动性风险影响了投资者的信心。恒大10月11日再次未能履行1.48亿美元的债务利息。危机正在蔓延，当代置业与新力控股将偿债日期再往后延，中国房地产债务总额超过5万亿美元，已超过了第三大经济体日本2020年的GDP。所以说，除了恒大，新力控股，当代置业，花样年都已实质违约、爆雷，国外投资中国的基金里的养老钱、退休钱都打了水漂。后面还有一大堆房企在排队违约呢。这些年中国房地产企业赚的钱都到了银行高管、地产公司高管，以及地产公司背后的家族，还有华尔街高管手里，而普通百姓的钱就这么没了。

这也是中共和暗黑势力控制媒体的原因，就是不让老百姓知道真相。如果西方的老百姓知道他们的退休金和养老金全部都打了水漂儿，他们非反了不可。但是绝大多数的西方老百姓和中国老百姓是一样的，一是被媒体控制；二是很少有人对财经这方面儿的消息感兴趣，不了解这里边的门道，所以即使财经新闻上说爆雷不断，但是他们也不懂这与将来的退休金和养老金有什么关系。他们还在相信着政府，相信着将来可以拿着退休金，过着和以前一样的舒适的退休生活。
