据日本[朝日新闻](https://www.asahi.com/articles/ASPDH6SQRPDFUHBI033.html)16日消息称，北京冬奥会开幕距离明年2月冬奥会还有50天。中共国一直向外界鼓吹“严格的防疫措施抑制了新型冠状病毒而感到自豪”。然而，随着高度传染性的奥密克戎（Omicron）变异毒株在世界各地蔓延而失控，最近感染已经蔓延到上海和邻近的浙江省。

该消息表示，过去两周，浙江、上海等地新感染者的数量每天大约有 40 到 90 人，它正在扩散。这是中共国今年以来中共病毒（Covid-19）感染人数较高的水平。

显然，奥密克戎新变种从南非到[美国](https://www.cbsnews.com/news/omicron-variant-covid-19-first-united-states-cases/)，在欧美非等地扩散后又回到中共国，这显然表明了中共病毒始作俑者中共国也失控了，更加表明了毒疫苗的欺骗性，有[数据表明](https://apnews.com/article/coronavirus-pandemic-business-health-africa-south-africa-c8d09bed79be4a25f752237c0470961b)，Omicron虽然更温和，但更擅长逃避疫苗。

- ***延展阅读：***[***广州发现全省首例奥密克戎阳性病例，疾控：不影响现有检测试剂敏感性***](https://3g.163.com/dy/article/GR6M2LVB0519APGA.html)


另据中共政府13日消息称，所谓首次在一名从波兰入境并被隔离的妇女身上检测到Omicron毒株，这对奥运会的影响很大。但中共国政府承认并表示，“有抗击新冠疫情的经验，可以顺利成功举办（冬奥）。”那是因为，中共知道自己释放的生物武器，并知道怎么控制它。

对于新变种奥米克戎病毒此时传回到中共国大陆——病毒大本营，对于习近平政府来说，冬奥恐怕是一个残酷的冬季。替中共掩盖罪行的世卫组织总干事谭德塞又开始警告世界说：“Omicron是以前任何变体都没见过的速度在传播。”这也就意味着在这个冬季中共国要承受巨大的境外输入压力。奥密克戎变异毒株不仅传播速度快，而且对比德尔塔毒株二天就会有更多的突变，还有较强的疫苗抗药性，可能导致人们二次感染中共病毒的风险增加。

对此，人们也因此加大了对疫苗无效而有害的怀疑力度。更让世界感到惊讶的是，中共国疾控中心（CDC）病毒预防控制所所长[许文波](https://www.thepaper.cn/newsDetail_forward_15635990)在11月30日新闻发布会上介绍说，“针对奥密克戎变异株，中国已经做好了技术储备，灭活疫苗、蛋白疫苗、载体疫苗都已经做了前期的技术研究，包括一些序列的设计已经开始了。”这表明，中共国对Omicron变异病毒防备应早于11月6日在所谓南非发现世界第一例时间。

不但如此，中共国仍有最后为疫情溯源追责在不遗余力地做自我洗涤的铺垫，11月29日，一篇[中新社](http://www.chinanews.com.cn/gn/2021/11-29/9618612.shtml)针对中共病毒奥密克戎变异株的消息援引中共疾控专家表示，“奥密克戎变异株在南非首先发现和报道，但不代表这个病毒是在南非演变形成的，变异株的发现地不一定是溯源地。”
