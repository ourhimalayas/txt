## 美国参议院举行打击威权主义听证会 矛头直指中俄

![](https://assets.gnews.org/wp-content/uploads/2022/03/1-176.jpg)
美国参议院外交事务委员会主席梅嫩德斯（图片来自美联社）

美国参议院外交事务委员会星期二（3月15日）上午召开题为“打击威权主义：美国的工具和应对措施”听证会，讨论如何更好的打击俄罗斯和中共国等威权主义国家所带来的日益增长的威胁。

据《美国之音》3月16日报道，美国参议院外委会主席梅嫩德斯（Bob Menendez）在听证会上表示，普京对乌克兰的入侵向世人显示了威权主义死灰复燃所带来的危险。美国及盟友必须采取行动打击21世纪新型威权支持系统。

梅嫩德斯指出，威权领导人不是在专制孤立的情况下运作，而是通过新的盗贼金融机制、虚假信息专业人员和一系列安全服务网络来相互保护，免受民主压力，并确保他们的压制性统治。从委内瑞拉到古巴，从白俄罗斯到缅甸，独裁者都得到了中共国和俄罗斯的支持。

梅嫩德斯说：“我们必须切断他们的生命线，削弱他们相互缓冲制裁带来的冲击的能力；我们必须打击用于颠覆民主原则和促进独裁者利益的数字威权主义——包括虚假信息、宣传和审查制度；我们必须对抗威权主义者操纵和散布的用以分散注意力的危险叙述，因为这些会使人们质疑民主是否能为现代世界提供任何东西。”

此外，梅嫩德斯还表示，乌克兰的例子也表明，在威权主义威胁日益增长的情况下，民主正在重新觉醒。因此美国必须充分利用民主热潮的复苏，带头支持民主，并加倍努力对抗威权主义。

俄罗斯问题专家、《大西洋》杂志撰稿人阿普鲍（(Anne Applebaum)与美国“国际共和研究所”所长特温宁（Daniel Twining），也在听证会上对如何利用美国的政策来遏制威权主义提出了他们的建议。

阿普鲍姆建议终结跨国盗贼统治，将民主建设作为美国外交政策的中心。她也强调了打击虚假信息的重要性，并特别点名俄罗斯、中共国和委内瑞拉。

参议院外委会资深成员里施（Jim Risch）参议员也在发言中表示，威权主义正在兴起，绝不能让独裁者获胜，因为没有人愿意生活在一个由普京或习近平控制的世界。
