## 慈善团体视频显示缅甸仰光CCP病毒死者激增
` 曹操` [轉載自GNews](https://gnews.org/zh-hans/1403896/)

编辑：yyshere
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/07/18043548/%E7%BC%85%E7%94%B8%E4%BB%B0%E5%85%89.jpeg)（图片来自网络）
缅甸最大城市仰光的一个殡葬服务慈善团体在社交媒体公布的一段视频显示，他们在上周五即7月16日在一幢民宅内协助处置两具Covid-19（CCP病毒）病毒感染者遗体。

视频中，志愿者进入一栋房子，在卧室里发现了一具70岁老人的遗体，老人42岁的儿子被发现死在门边的塑料椅子上，椅子旁边有一个小型血氧仪。

志愿者透露，根据家庭成员的说法，这对父子生前已有一周时间处于呼吸困难状态。

而在该团体的另一个视频中，同样是在周五，志愿者从一辆汽车里抬走了一名男子的遗体。随后遗体被送往仰光德貢新區港口的基祖公墓。志愿者在公共墓内拍摄的画面显示，尸体沿着火葬场外的走廊一字排开。

虽然不能确定每一位死者的死因，但这个慈善团体透露，在仰光这样的大城市，民众正面临Covid-19（CCP病毒）病毒的快速扩散，同时医院超负荷且氧气短缺，死亡人数正在迅速攀升，以至于他们难以应付激增的葬礼安排。仅星期六一天，仰光的四个公墓里已经埋葬了600多具遗体。这一数字明显高于此前的数据。

同时这个慈善团体表示，死于Covid-19（CCP病毒）病毒的人本不属于他们处理的范围，应由官方处置，但目前他们都是接到求助前往处理那些死在医院外的遗体，虽然不确定他们都是死于病毒感染，但其中很多人都是因为氧气耗尽而死亡，由此他们推断这部分人应该是死于病毒感染。

而目前仰光光放并没有公布每天举行火化和埋葬的确切数字。

有分析认为，缅甸目前因Covid-19（CCP病毒）病毒感染和死亡的的实际人数应该高于官方公布。

根据缅甸卫生部17日的公告，当天新增5,497例COVID-19（CCP病毒）确诊病例，总病例数已达到224,236例，同时新增233例死亡，死亡总数总数达到4769例。
