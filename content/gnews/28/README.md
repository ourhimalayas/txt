## 有声｜上海疫苗灾难爆发 死亡人数攀升 防疫乱象继续

By:**秘翻新闻**

**2022-04-25**
![](https://assets.gnews.org/wp-content/uploads/2022/04/425-6.png)
上海疫苗灾难已经显现，因疫苗反应导致的病例持续飙升，死亡人数也在不断增加。

据[新唐人](https://www.ntdtv.com/gb/2022/04/24/a103409056.html)综合消息，4月24号，中共国家卫健委通报说，上海23号单日染疫人数再增2万多例，死亡人数也增加了39例。来自上海市卫健委的消息称，这次疫情爆发以来，上海市染疫死亡病例共有87例，平均年龄81.10岁，最大101岁。

但据郭文贵先生在直播中所称，上海在隔离期间的所有死亡人数已在百万左右。西方媒体也普遍认为，中共公布的死亡人数，只是上海染疫死亡人数的冰山一角。

据《华尔街日报》报导，上海市东海老年护理医院上月底新进一批外来务工的家政人员和保姆做护工，而该院中不少医生和护士都已经因染疫而被隔离。一名获聘的工人在进入该院不到一周的时间里，便开始从病房搬运尸体到停尸房。

此外，上海疫苗灾难的大规模爆发，致使防疫管理一片混乱。

[据报道](https://www.ntdtv.com/gb/2022/04/24/a103408931.html)，一名被送到方舱医院隔离的女子在网上说，她被安排在男女混住的房间里，她的床位与旁边男子的床位相隔只有40厘米，她感到自己没有被当作人对待。
