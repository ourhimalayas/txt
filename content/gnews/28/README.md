## 伊核谈判再添变数 伊朗和美国及其盟友交换人质未果
` 曹操` [轉載自GNews](https://gnews.org/zh-hans/1404362/)

编辑：yyshere
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/07/18092644/%E7%BE%8E%E4%BC%8A2.jpeg)（图片来自网络）
伊朗核谈判首席代表阿巴斯·阿拉奇7月17日通过推特发文表示，“我们（伊朗）正处于一个过渡时期，因为我们的首都正在进行民主的权力移交。因此维也纳谈判显然必须等待我们的新政府。这是每一个民主国家所要求的。美国和英国需要了解这一点，并停止将已经准备好的人道主义交换与JCPOA（核问题全面协议）联系起来。将这种人质交换冠以政治目的，不可能达到任何目的。如果美国和英国履行他们的协议部分，各方的10名囚犯可能在明天被释放。

美国国务院发言人随后就阿拉奇的言论发表声明，”这些言论是一种令人愤慨的尝试，旨在转移对目前可能的，相互恢复遵守JCPOA（核问题全面协议）的僵局的责任。我们随时准备返回维也纳，一旦伊朗做出必要的决定，就会相互返回核协议的工作。关于对伊朗违背其意愿不公正关押的美国人的评论，我们看到的只是为增强被拘押者家人的希望而做出的另一种残酷尝试。阿拉奇说我们把协议作为‘人质’，而正是他的政府多年来一直不公正地扣押着四名无辜的美国人，在被拘留者的问题上还没有达成协议。如果伊朗真的有兴趣做出人道主义姿态，它就会立即释放这些被拘留者。释放我们被不公正地拘押的公民是我们的首要任务，我们仍然希望尽快看到这一点实现。我们一直在维也纳进程的背景下参与关于被拘留者的间接会谈，而对于重启这一进程的拖延毫无意义。虽然如果我们在维也纳举行会议，取得进展会更有效，但我们也准备在这一时期继续进行关于被拘留者的会谈“。

有分析认为，美伊两国外交官员的骂战显示，除开核协议的谈判，伊朗和美国及其盟国还存在着其他的谈判，并且陷入僵局，双方都希望通过施压，尽快获得符合各自利益的结果。

而另一则由伊朗外交部发言人卡提布扎得发布的推特也间接证实这一点，“’离谱’=美国否认‘在被拘留者问题上有一个商定的交易’这一简单事实，甚至连如何宣布（都不知道）。与美国和英国在维也纳达成的人道主义交换协议，这与JCPOA（核问题全面协议）不同，是关于释放各方的10名被拘押者，伊朗已经准备好在今天进行（交换）“。

但截至目前，美伊双方还未被拘押人质的交换协议或者进程做出进一步说明。
