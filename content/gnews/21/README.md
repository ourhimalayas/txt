墨西哥政府刚刚取消前往墨西哥旅行的所有 Covid新冠旅行的限制要求。 在此之前，墨西哥海关也没有严格限制过国际旅行者入境，仅仅要求填写包含 3 个问题的单子，主要询问你是否感到身体不适。 现在墨西哥政府在海关连3个问题的单子都不需要国际旅行者填写了。

墨西哥和萨尔瓦多很有可能是目前地球上绝无仅有的两个完全没有 Covid 入境要求的国家。**评：**新冠大流行以来，冷眼旁观各国的国际入境旅行政策，发达国家的限制越来越多，让很多旅行者望而却步。反观，墨西哥对新冠病毒从来没有采取任何封锁和出行限制，赢得很多国际人士的青睐
