## 疯狂之举&#8212;-英超球队考虑在新合同中加入疫苗接种条款

据英国媒体《每日电讯报》（Daily Telegraph）12月4日报道，英超俱乐部正在考虑在新合同中加入关于新冠疫苗的条款，以确保球员全部接种疫苗。
![](https://assets.gnews.org/wp-content/uploads/2021/12/WhatsApp-Image-2021-12-05-at-23.07.23.jpeg)
据悉，几家顶级俱乐部已经探讨了新合同的可行性，其中规定球员必须接种两针疫苗。英超联赛在10月份时宣布，68%的顶级联赛球员已经打了两针疫苗，81%的球员至少打了一针。

现在有很多球员担心他们谈论疫苗接种时的公众反应。爱尔兰前锋卡勒姆·罗宾逊（Callum Robinson）在本赛季早些时候谈到了他不接种疫苗的决定时说道：“人们对于疫苗接种的评论非常恶毒，不得不让你在谈及此事前保持谨慎。”拜仁慕尼黑的德国球星基米希因为担心疫苗的长期作用，一直对接种疫苗保持犹豫，因此招来了德国各界人士的轮番批评。拜仁慕尼黑董事会前主席鲁梅尼格批评基米希说，作为一个榜样，他必须接种疫苗。根据巴伐利亚州的规定，未接种疫苗的人必须被隔离，雇主也不用支付隔离期间的工资。尚未接种疫苗的基米希现在就处于停薪隔离的状态。

在足球界，尽管支持新冠疫苗的声音占据主流，但随着最近大量的足球运动员出现心脏问题或在球场上突然晕倒，其中包括巴塞罗那球星阿奎罗和丹麦国脚埃里克森，这些反常现象让很多人开始质疑这是否与疫苗有关。前英格兰国脚特雷弗·辛克莱（Trevor Sinclair）和马修·勒蒂希尔 (Matt Le Tissier)最近就发推说，越来越多足球运动员出现心脏问题，这是否与疫苗有关，并呼吁立即展开调查。但无奈审查疫苗言论的网络太大太广，他们的声音立即就被打压和埋没。
