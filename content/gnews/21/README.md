随着汽油价格飙升，优步平台争相减轻工人的痛苦——但可能为时已晚。

面对欧洲各地不断上涨的燃料成本，优步的涨价可能不足以平息司机的怒火——这可能会给他们另一个争取员工地位的理由。

欧洲对俄罗斯实施制裁的后果，使欧洲大陆争先恐后地适应飙升的燃油价格，影响从物流到商业航空业的各个领域。优步、Bolt 和 FreeNow 等打车应用的司机们处于特别不稳定的境地：虽然这些公司可以向消费者收取附加费，但当乘客预订乘车时，他们的司机依赖平台来定价。

这种定价平衡行为需要叫车平台来补偿更高的燃料成本以便他们的司机在街上空跑来保持足够的竞争力，不至于把乘客赶走。一个明显的提醒是，最终控制定价的是平台而不是司机，这也可能迫使司机更加努力地争取作为雇员而不是自雇承包商。

在美国，优步很早就出来支持面临更高油价的司机。3 月 11 日，优步宣布将对每次优步行程征收 0.45 美元或 0.55 美元的燃油附加费——取决于地点——并声称“这些钱100% 全归工人” 优步在声明中强调，涨价是“暂时的”，将在 60 天内重新评估。

**微妙的机动**

优步向 POLITICO 证实，与其在美国的行动相比，优步在欧洲的定价政策方面没有那么直言不讳——但在多个国家/地区的票价已经提高。优步提价的欧洲国家是法国、德国、西班牙、荷兰、葡萄牙、波兰、捷克共和国、波兰和罗马尼亚。在英国，票价最近也有所上涨，但原因是承包模式发生了变化。工会还报告了比利时的价格上涨。

虽然它在美国引入了固定燃油附加费价格，但优步在欧洲的做法涉及不同程度地提高所谓的基本票价。这些上涨考虑了司机的运营成本和他们在每个国家/地区赚取的不同收入，基于优步的竞争对手和每个国家/地区不同的运营要求。基本票价是司机获得的最低金额，无论在任何给定时间需求有多高或多低。 （优步在需求旺盛时引入“动态定价”）。

“在欧洲的许多城市，我们提高了优步的价格，以帮助司机应对最近运营成本的飙升”优步发言人说，并补充说，上涨的价格将提前呈现给乘客：“与往常一样，乘客在预订他们的旅程之前就知悉行程价格。”

然而，定价政策是一个微妙的问题。如果司机和乘客不喜欢建议的价格，他们都可以轻松转而选择竞争对手的应用程序，比如爱沙尼亚的Bolt。但在平衡价格方面，Bolt面临着与优步相同的问题。出租车行业、公共交通和电动滑板车是优步在定价决策中必须考虑的其他竞争对手。


```
Bolt在与 POLITICO 分享的一份声明中表示，它“目前正在审查”其定价，以确保它能够呈现来自司机“公平地反映正在发生的成本”。
```


除了提高票价之外，还有其他一些争议较小的方法可以缓解司机的痛苦，例如与加油站达成燃料折扣协议，或者让司机更能负担得起的电动汽车。例如，Bolt在 Circle K 加油站设立了忠诚度计划，以帮助控制燃料和汽车维修价格。与此同时，优步拥有 8 亿美元的资金用于到 2030 年让司机使用电动汽车，为驾驶电动汽车的人提供购车援助和奖励。

**完美风暴**

尽管平台正试图减轻燃油价格飙升造成的损害，但它仍可能影响已经迫在眉睫的司机就业状况摊牌。

不断上涨的燃料成本可能会给司机游说团体提供新的弹药，以争取平台工人的更多权利，甚至是充分就业状态。两周前，App Drivers and Couriers Union (ADCU) 在贝尔法斯特为送餐快递员举行抗议，称高油价是司机面临的“完美风暴”的一部分。抗议的目标之一是确保司机的最低收入。

在欧洲其他地方，司机代表也声称平台的努力还不够，并要求政府加紧支持工人。

比利时豪华轿车司机协会 (ABCL) 主席费尔南多·雷东多 (Fernando Redondo) 说：“油价上涨如此残酷，以至于我们无法成功实现盈利。”由于该平台在布鲁塞尔的法律问题，雷东多过去曾站在优步一边。 “我们仍请求得到[比利时]国家政府和该地区的支持，以帮助我们应对燃料价格上涨，即使各个平台做出了非常、非常、非常小的努力。”

政府支持的请求是在平台工人的命运已经受到审查的时候提出的，无论是整个集团在各国的首都还是在布鲁塞尔。欧盟立法者正在加紧启动 12 月初提出的欧盟平台工作指令的工作，该指令可能将多达 410 万工人重新归类为雇员，而不是自雇承包商。


```
对工人薪酬的控制是确定平台工人是否实际上是雇员的五个标准之一。司机们被提醒，是优步而不是他们，来决定乘车价格——因此，他们获得了多少经济救济——他们可能会提高平台工作计划的热情，以确保作为受雇工人的同等权利和条件。
```


“我们如何监管平台？”法国 INV 联盟秘书长 Brahim Ben Ali 表示，这是政府面临的问题。“我们不能就这样把它丢下，那是不可能的。他们创造了一种反社会工作的形式。我们是一个民主国家，一个社会国家。而今天，我们恰恰相反。这是不可能的。”
