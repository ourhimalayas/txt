**二．中共北京政权的由来以及罪恶**

**1      中共北京政权的前身是中华苏维埃共和国，中共政权是来自非中华民族的侵略者**

1.1   1931年11月7日，适逢十月革命纪念日，中华苏维埃共和国临时政府在江西瑞金宣布成立，**开始分裂中华民国**。

1.2   中华苏维埃共和国是中国共产党在苏联和共产国际的支持下，**创建的第一个分裂中国的外来政权**，是中华人民共和国的前身。于中华民国大陆时期建立的苏维埃政权，先后辖有18省、4直辖县。该政权主张推翻国民政府和中国国民党的合法统治，在中华大地实行消灭地主富农、实现邪恶共产主义。是一个彻头彻尾的暴力流氓机构。
![](https://lh4.googleusercontent.com/qYJdlvltQOu8LSVeRJ-kelQc0J3kAUyKD98ewGh11kemU0bx7IXT1kA0ej4HJZfbwkRStPuq9uRM-OBnaZhfSMtydnyzSqJIXhkB4MDLgeGmQJANhcTEOb8c02x3DmFWtt7O475Q)
1.3  “苏维埃”这个词与在中文字典里没有，是一个外来音译词汇，单单从这一点看苏维埃共和国就和中国人民没有一毛钱的关系，**“中华人民共和国”作为这个政权的徒子徒孙机构显然是一个外来政权，其本质是奴役中国老百姓，为实现共产国际奴役全世界的其中一环。**

1.4  2007年6月1日新华出版社出版的《中苏关系史纲》写到：“莫斯科不顾外交受损坚决支持中共按照俄国革命的模式发动苏维埃革命，从政治方针一直到具体政策文件的制定，从决定中共领导人到选派代表亲临上海，甚至直至苏区，就近帮助工作和指导作战，可以说是事无巨细，几乎一包到底。”**可见中国共产党根本没有外交主权，只不过是共产国际的一个棋子，从领导人到政府机构都是一个效忠于苏联（外国机构）的一个卖国贼政权。**

2      如下图所示，中华人民共和国只不过是中国共产党的一个子公司。中国共产党是苏联第三共产国际的直接下属机构，是中华名族的败类，是真正的卖国贼。
![](https://lh3.googleusercontent.com/gh7fXF33FGACUiqbzfcISo0Jw160scIxDokRekv96LKl42DI9DAMfo5c2iRfyCpgJ7kHoJ4oxezKktgqBPb5LBNZ_G7W6q_56anTOuZ84Y1ehib4lz16nHRwEh2ddYqABU4I83Ot)
2.1   共产国际（俄语：Коммунистический èíòåðíàöèîíàë，缩写为Коминтерн），通称第三国际（Третий èíòåðíàöèîíàë），也称赤色国际或红色国际，是一个共产党和共产主义组织的国际联合组织，第三国际成立于1919年3月2日第一次代表大会，在列宁领导下成立，总部设于苏联莫斯科。第三国际本名“共产国际”，区别于第二国际的本名“社会主义国际”。

2.2   共产国际进入中国：**1920年4月，俄文报纸《生活报》的记者、共产国际代表维经斯基（吴廷康）受命组建中国共产党，在翻译杨明斋的陪同下来到中国。**1920年5月维经斯基在上海成立了“临时的集体中心机构”———共产国际东亚书记处，下设中国、朝鲜和日本三个科，书记处全会选出三人临时执行局，维经斯基任东亚书记处临时执行局主席。

2.3   中共成立：**1921年7月23日，在共产国际的资助下，来自中国各大城市和日本留学生代表的中国共产党的13名党员（共有57名党员）和共产国际代表马林、尼克尔斯基在上海举行第一次全国代表大会**，最后一天会议移至嘉兴南湖举行，会议选举陈独秀、张国焘、李达三人组成的中央局为领导机构。这也是中共通常认可的成立日期。

2.4  1922年7月，中共二大正式决定参加共产国际，成为它的一个支部，在很长一段时间里，第三国际成了中共党的实际领导者，**中共也从第三国际获取卢布、鸦片（方便在中国南方兑现）等援助及形势指导。**

2.5   西安事变改变了中共的命运，蒋介石曾说过中共是嗜脐之患，也就好比是人身体上肚脐周围被腐蚀，不除掉的话就会危及人的生命，果不其然。

2.6  郭文贵先生在2021年11月5日直播中指出：**共产国际是以消灭家庭、消灭婚姻为目标，《共产党宣言》里边就认为一夫一妻制是私有制的产物。**

2.7   郭文贵先生在2022年2月1日直播中指出：**中国共产党是共产国际资助的一个恐怖组织，它在过去颠覆了中国的合法政府**，中共在中国的极权统治造成了骇人听闻且践踏人权的反人类暴行。

**3      中共奴役百姓的手段：枪杆子，笔杆子，钱袋子，刀把子**

3.1  共匪自称的三大法宝：①党的建设，②人民军队，③统一战线。**说白了就是①秘密结社，②枪杆子挟持老百姓，③对外蓝金黄。**

3.2   除了以上三条，**第四条共匪还非常重视宣传**，在共匪的眼里，不仅所有的报纸，收音机，电视等媒体可以利用，甚至小说，音乐，艺术都可以用来做政治宣传。

3.3  **第五条共匪还非常重视运用特务间谍**，打入对手内部，在对手的内部分化瓦解，周恩来就是共匪间谍的第一代特务头子。国共对战时期他们曾经把国民党高级将领傅作义的女儿发展为共产党员，直到现在共匪把间谍派到了全世界。根据郭文贵先生爆料才知道，共匪的特务间谍通过和白人组成家庭，结婚生子，渗透西方几十年用于关键时刻策反，获取情报。**在日本以及有很多国会议员被腐蚀，甚至有共匪的间谍通过和自卫队军官结婚的形式获取日本军方的情报。**

3.4  **本质上共产党就是一个模仿黑社会的组织形式，用武力夺取政权后，对内用秘密警察统治人民，对外用金钱美女和宣传腐蚀全世界的非法流氓统治集团。**
