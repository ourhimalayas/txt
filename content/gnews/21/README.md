## 论文解读：灭活疫苗对人生理病理的持续影响
` 喜马拉雅农场新西兰站` [轉載自GNews](https://gnews.org/zh-hans/1662897/)

作者：Stephen(文文)
![](https://assets.gnews.org/wp-content/uploads/2021/11/屏幕截图-2021-11-14-154810.jpg)
Nature旗下子刊《Cell Discovery》于10月26日在线发表了上海同济大学的一篇研究论文，论文比较全面的揭示了接种灭活疫苗（北京生物制品所）免疫后，被接种者的一些持续生理变化，论文题目“Comprehensive investigations revealed consistent pathophysiological alterations after vaccination with COVID-19 vaccines”。

该论文得出的比较明确的结论有：

1. 接受灭活疫苗免疫后糖化血红蛋白A1c会有持续升高，到第 1 次接种后的第 28 天，11 个人中有 3 人达到前驱糖尿病范围。

2. 疫苗对电解质平衡有影响，接种后血清钾、钠水平都有显着降低。

3. 疫苗注射后还会产生凝血功能障碍，凝血曲线发生了显着变化，但到第 90 天，曲线恢复到疫苗接种前。

4. 导致肾功能障碍，在第一次接种后 28、42 和 90 天，血清肌酐水平显着高于接种前的水平，导致 eGFR（估算肾小球滤过率，肾小球滤过率是评估肾脏排泄废物功能的一个指标）降低。

5. 接种疫苗导致了几乎所有免疫细胞的基因表达都发生了巨大变化。其中“炎症反应”基因在接种疫苗后进一步高度表达；“糖酶解”、“胆汁酸代谢”和“I型干扰素（IFN-α/β）反应”相关的基因被下调。

重要的是：I 型干扰素反应降低，表明在第一次接种后至少 28 天，一般抗病毒能力减弱，反而更容易感染病毒。如果被接种者在这段时间内接触SARS-COV-2病毒，则更容易被感染，这也许疫苗是病毒“激活剂”的其中一个原因。

不能确定的是，免疫细胞的基因表达变化是否会长期存在，如果接种灭活疫苗所导致的基因表达变化长期存在，那么对人体的免疫系统影响也将是长期的。“炎症反应”基因的上调意味着人体会持续处于“炎症状态”，这种慢性炎症也是诱发癌症的因素之一。

论文中值得一提的还有在选择11名疫苗受试者时，竟然发现了一例COVID-19抗体阳性，说明他之前已被病毒感染过，但自己却不知道。在仅仅选择11名入组人员过程中，就能发现了一例曾经的”无症状”感染者，那么有可能实际的被感染人数是相当庞大的，而且是在人口众多的上海，这些曾经的“无症状”感染者是极易将病毒传播给他人的，中共国内的真实感染人数可见一斑！

论文美中不足的是仅入组了11名受试人员，没有考察疫苗对各类群体的全面影响。而且仅观察了受试人员在接种后90天内的某些病理生理变化，究竟疫苗对人的长期影响，特别是免疫方面的影响研究还不够充分。
