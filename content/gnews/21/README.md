## 【NewsExpress】中国共产党宣扬的100年带给中国的真正的是什么
` 闪电侠` [轉載自GNews](https://gnews.org/zh-hans/1363404/)

作者：米莎·凯切尔

本周，中国共产党（CCP）正全力为建党100周年做准备，并大力宣传其成就。

然而，中共为中国所做的事情并没有什么值得庆祝的。它的主要成就在于它如何能够生存并长期掌权。那么，中共到底宣扬的是什么，真相又在哪里呢？

1、中国主权

名单上最重要的主张是中共通过 **1949 年中华人民共和国的成立统一了国家**并获得了独立。中共指责由中国国民党（KMT）领导的前任政府是美国帝国主义的傀儡。

2、经济繁荣

中共还吹嘘自己的经济成就，声称它释放了中国的潜力，使其成为经济超级大国。

**但在中共上台之前，工业化和城市化已经在顺利进行**。尽管发生了连续的战争，但现代城市、交通、工业、商业和金融的基础设施正在全国大部分地区建立起来。例如，上海在 1930 年代就已经是一座精致的大都市，被称为“东方巴黎”。

3. 消除贫困

中共及其支持者对中共“使”数亿中国人摆脱贫困的说法感到特别高兴。事实上，今年早些时候，习近平大肆宣扬中共消除农村贫困的目标“完全胜利”，一些分析人士称，这一举措不符合成本效益或可持续性。虽然近几十年来赤贫确实急剧下降，但必须记住，**中共失败的政策首先使数百万人陷入贫困。中共在大跃进运动中杀害了多达****4500万人的罪行，导致了中国历史上最严重的人为饥荒，这是永远不应被原谅的。根据世界银行现在使用的不同贫困衡量标准，仍有数亿中国人可能生活在贫困线以下。**

4. 建立“人民民主”

中共声称要在中国建立“人民民主”，也就是毛泽东所说的“民主为民”和“敌人专政”。但实际上，**该党建立了一个极权主义国家，中断了中国走向宪政民主的进程**。中华民国成立于1912年，常被称为亚洲第一个民主共和国。它拥有现代化的法律体系、充满活力的公民社会、基本自由的新闻以及自治的学校和大学。1928年，国民党以武力统一中国，以“监护民主”取代不稳定的多党民主，国民党承诺将引导民众建立完全民主，从而垄断政治权力。1946年后国民党的专制党国开始民主化进程。次年制定宪法，1947年和1948年进行了国民议会和总统的多党选举。

中共结束了这些政治发展。在习近平的领导下，当前的强硬领导层**通过对普世价值发动全面战争、扼杀中国公民社会并消除任何向宪政民主和平转型的机会来保持对权力的掌控。**香港最近对民主机构的镇压是这个国家走向的一个明显例子。

5. 社会主义和更大的平等

中共也说要给中国带来社会主义和更大的平等，**但它在文革期间创造了最残酷的反对“黑五类”的种姓制度，造成了最骇人听闻的不平等。**从1950年代到1970年代的30年间，数以百万计的人被共产党政权归类为“地主”、“富农”、“反革命”、“坏分子”和“右派”。这些人经常被隔离参加斗争会（一种公开羞辱的形式）、劳动教养、殴打甚至处决。他们的许多孩子被剥夺了教育和国家就业机会。此外，**毛泽东时代通过特殊的户籍制度，农民及其子女被登记为农村居民。他们被永久排除在城市地区的国家福利、就业和学校之外**。这种户籍制度和城乡差距，今天仍然对农村居民和向大城市的流动人口产生严重影响。共产党政权今天还在新疆地区使用“再教育营”作为根除其认为对其权力构成威胁的方法。

6. 保护中华文化

最后，中共现在**假装代表中国文化，但它早就根除了中国传统文化**。毛泽东时代，**中国传统文化的主要捍卫者，农村绅士和城市知识分子根据共产党国家意识形态被淘汰或改造。**广告活动在文革期间发动系统地消灭“四旧”被视为不受欢迎-即旧思想，旧文化，旧习惯和旧风俗。当前共产党国家对中国文化的推广只不过是一种利用任何机会推动中国民族主义作为为政权提供合法性的工具的愤世嫉俗的举动。
