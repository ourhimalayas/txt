这些特殊的弹出式网站专门面向 5-11 岁的儿童，他们现在有资格根据 Pedo Joe Biden 的Build Back Better（重建得更好）计划接受辉瑞公司引入的由上届政府为预防武汉冠状病毒“曲速行动”疫苗计划。
