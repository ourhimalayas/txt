本周二（1月11日），大众汽车中共国区总裁冯思翰（Stephan Wöllenstein）在北京向媒体表示，去年德国大众汽车集团（中共国）与其合资企业在全球最大汽车市场——中共国大陆和香港市场共交付汽车330万辆，同比下降 14.1%，造成这一情况的原因是芯片短缺及供应链瓶颈。

冯思翰表示，2021年主要是大众、斯柯达销量大降；高端品牌奥迪只降了3.6%，而保时捷和宾利的销量增加了8%和43%。尽管整个汽车市场增长了4%，大众集团所占份额从15%降至11%。冯思翰指出，因半导体短缺、中共病毒疫情、停摆措施，以及一家日本供应商的火灾，最终导致减少生产60万辆车。今年大众集团希望能迎头赶上，他指出“需求仍在”。整体市场预计将增长4%，但大众希望增长15% -16%，希望弥补回去年的损失。他表示，希望大众汽车开发的ID系列电动车销量也在增长。 2021年，大众实现销售7万ID系列电动车的目标。冯思翰希望今年的数字“至少”翻倍，他确信大众汽车能够售出每一辆可以制造的 ID 系列车。他称16万至20万辆ID车的芯片供应是有保障的。去年，中共国汽车销量为 2100 万辆。

冯思翰承认，不确定因素依旧存在， “芯片风险难以预估”。对中共国首个奥密克戎染疫病例发生在北京邻城天津，他也担心当局可能会采取更严厉停摆措施，大众天津工厂已不得不暂时停产。由于中共国实施严格入境限制和长达３星期的强制隔离，导致外国专家入境困难。冯思翰承认“制造业受到影响”。
