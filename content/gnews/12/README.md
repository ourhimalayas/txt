## 乌克兰96名儿童国际大救援-“辛德勒的名单”再上演
` 纽约香草山MOS02` [轉載自GNews](https://gnews.org/zh-hans/2123251/)

作者：纽约香草山农场教育部-马克文、朱丽叶
![This image has an empty alt attribute; its file name is Screen-Shot-2021-08-30-at-7.37.59-PM.png](https://assets.gnews.org/wp-content/uploads/2021/08/Screen-Shot-2021-08-30-at-7.37.59-PM.png)
2022年2月24日，俄国独裁者普京下令入侵乌克兰。25日乌克兰总统泽连斯基宣布成立最高统帅部，并自任主席兼武装部队最高统帅，率领乌克兰人民坚决抵抗俄罗斯入侵者。大多数乌克兰男性公民，安排妻子孩子离境逃亡邻国后，自己留下来保卫家园。俄国的入侵战争造成几百万难民背井离乡，其中大多数是儿童和老人。

![](https://assets.gnews.org/wp-content/uploads/2022/03/image0-19.jpg)

新中国联邦创建人郭文贵先生看到乌克兰涌现的大量难民，立即让法制基金、法制社会与联合国救援机构合作建立国际救援队，奔赴乌克兰进行人道主义紧急救助。法制基金、法制社会立刻响应，开展组织国际大救援。西班牙农场接到任务后，立刻组织人员赶赴一线。

救援工作人员在乌克兰境内联系酒店时，一位Teresa女士，前来求助，她是巴塞罗那本地一家儿童慈善机构工作的工作人员。希望能把战区一家慈善机构里的几十名儿童救出来。文戈七雄站长立即向联盟委员会汇报此事，联盟会员会全力支持，第一时间协调车辆和人员配合西班牙站救援。最终成功营救出96名儿童，加上陪同老师共计113人，最终全部安全到达波兰境内，新中国联邦免费为孩子和老师提供食宿。之后，新中国联邦将用飞机把他们送往西班牙，让Teresa女士所在的慈善机构来安排孩子们未来的生活。

《辛德勒的名单》是一部1993年上映的美国电影,获得七个奥斯卡金像奖。它讲述了一个德国商人奥斯卡·辛德勒在二战中拯救超过1100名他工厂里的犹太人的故事。

新中国联邦国际大救援活动只是刚刚开始，大量救援队将赶赴战区，计划租用上百辆大巴车，数十架飞机做百次飞行，将成为史上一次大规模的国际救援大行动。
