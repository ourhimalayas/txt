根据这封信，随着拜登政府的应对中共国战略的形成，“进步人士”认为气候变化方面的合作应优先于所有其它问题。

“为了应对气候危机，并建立一个为美国和中共国的人民服务的全球经济，我们必须从竞争转向合作。” “进步人士”在信中继续说道。人类如果跟随这些所谓的“进步人士”，一定会“进步”到万劫不复。

报道称，挑战中共国侵犯人权和地区侵略扩张是拜登外交政策的核心，”坚持美国式的民主和中共独裁主义斗争是他担任总统期间的理念。”

随着拜登总统要求情报界递交病毒真相的调查报告90天期限的临近，中共指使其海外纵队在多领域展开反扑。在科学界，由诺贝尔奖得主领衔，纠集了一群包括马利克、彼得打杂客加盟的科学家来替中共站台。另一边，又有所谓的“进步人士”向白宫进谗言，居然说什么解决环保问题比查清病毒真相，遏制中共的侵略扩张更迫切，企图阻碍西方反共灭共的进程。说明西方对抗中共的方向走对了，中共还会出尽所有的牌来为自己续命。欧美各国应该再接再厉，排除杂音，清除障碍，加大灭共力度。同时我们可以通过这些来自西方内部的阻力，进一步考验拜登的反共决心和意志。
