## 美国务院就《香港国安法》实施一周年发表声明

<p>声明如下：</p>
<p>《香港国安法》实施一周年<br>新闻声明<br>安东尼J.布林肯，国务卿<br>2021年7月16日</p>
<p>在过去的一年里，中共国（PRC）和香港官员系统地破坏了香港的民主制度，推迟了选举，取消了当选议员的资格，迫使官员们采取效忠宣誓来保住他们的工作。自2019年抗议活动开始以来，地方当局逮捕了数千名公开反对政府政策的人，其中包括那些在社交媒体上发布帖子和参加守夜活动的人士。记者仅仅因为在报道政府活动和镇压抗议者方面的工作而被捕，香港当局已经发起了一场持续的、政治性的打击自由新闻的运动，监禁了一些报纸和相关人，并强迫关闭了这些刊物——这些独立报道的捍卫者。北京削弱了香港负责任、透明的治理和尊重个人自由的声誉，违背了香港保持高度自治50年不变的承诺。</p>
<p>我们将继续支持《中英联合声明》和《基本法》保障香港人民的权利和自由，面对北京去年的决定，这些决定扼杀了香港人民的民主愿望，我们正在采取行动。今天，我们发出一个明确的信息：美国坚决与香港人站在一起。</p>
<p>促进问责制：国务院指定（对以下人实施制裁），陈东（Chen Dong）、杨建平（Yang Jianping）、仇鸿（Qiu Hong）、卢新宁（Lu Xinning）、谭铁牛（Tan Tieniu）、何靖（He Jing）、尹宗华（Yin Zonghua），这些人是香港特别行政区中央人民政府联络处的副主任，中联办是中共国在香港展示影响力的主要平台，多次破坏了香港在中英联合声明中承诺的高度自治，这七名官员是根据《13936号行政命令》（Executive Order 13936）任命的。</p>
<p>透明度：香港的商业环境在过去的一年里恶化了，在中共大陆长期存在的许多法律、金融、运营和声誉风险，现在在香港已经越来越普遍了。一个健康的商业和投资环境，需要一个透明的监管框架和法治的坚守。今天，国务院与商务部、国土安全部和财政部一起发布了一份商业警告，警告香港企业面临的风险增加，这些风险包括《国家安全法》和其他新立法引入的风险、潜在的电子监控和缺乏数据隐私、信息访问量的减少，以及可能因公司遵守美国制裁（规定）而遭到报复。商业咨询概述了这些新出现的风险，以告知美国个人和企业，并建议提高认识以及尽早防范。</p>
<p>（声明完）</p>
<p>此声明对中共在香港行径的揭露算是比较到位的，相对美国对中共认知更清醒的保守派人士，拜登政府对中共的措施虽然让人感到低于预期，但在国际范围内，还是领先的，欧洲包括欧盟、G7对中共邪恶的认知远不如拜登政府，还需要拜登政府的启蒙。</p>
<p>希望这次德国默克尔访美，在美国进一步的呼唤下，对魔共的认知有所提升，非但不去为中共承担“基辛格使命”，还应该站在坚决反共的一边，奉行正道主义，为过度亲共的恶名降损。</p>
