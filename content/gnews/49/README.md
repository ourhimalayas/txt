这是美国第四次制裁中港官员了。 2020年8月7日，美国第一次宣布制裁11名香港官员，林郑月娥首当其冲，其次是陈国基、郑若骅、李家超、曾国卫、邓炳强、卢伟聪、骆惠宁、夏宝龙、张晓明、郑雁雄。美国财政部谴责林郑月娥2019年在试图通过修订《逃犯条例》，推动《香港国安法》中扮演关键角色，实施北京打压自由和民主程序的政策，这意味着林郑与独裁者、恐怖分子以及纳粹分子同等级别危险。

时任国务卿彭配奥的声明中表示，制裁的法理基础是根据川普总统颁发的「撤销对香港特殊待遇的行政命令」以及《香港人权与民主法》、《香港自治法》。他表示，美方对港共发出一个清晰的信息：香港当局的举动是“不可接受的”，违反了中共自己在「一国两制」和《中英联合声明》的承诺，林郑月娥对于美国的制裁表示并不害怕，她在美没有资产也不打算到美国去。一副死猪不怕开水烫的样子。美国制裁这11人的效果从林郑月娥的表态上可以看出，这制裁对他们毫无作用，香港的独裁政治还在继续，对香港人的秋后算帐反而变本加厉。中共在背后撑腰并嘉奖这些官员，让林郑团伙更疯狂，纷纷表态表忠。

2020年8月19日，国务卿彭培奥表示，中共压制港人自由与自治，美国将暂停或终止与香港3项双边协议：罪犯引渡移交协议、转移判刑者以及国际船运所得收入双重课税宽免。制裁11人，停止双边协议效果甚微，港共官员继续作恶，2020年6月，香港立法会选举中，香港选举管理委员会取消了多名参选人资格，列出5宗罪。这是香港历史以来最大规模的DQ，并扬言不排除有更多人被DＱ。 2020年12月7日，美国财政部宣布制裁包括副国级的中共中央政治局常委王晨在内的第十三届全国人大代表大会常务委员会副委员长等14人：王晨、曹建明、张春贤、沉跃跃、吉炳轩、艾力更·依明巴海、万鄂湘、陈竺、王东明、白马赤林、丁仲礼、郝明金、蔡达峰、武维华等。
