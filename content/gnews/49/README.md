美国国务院向媒体发布了“挑战与机遇”的C5+1国际会议联合声明，5个国家（C5）是：哈萨克斯坦共和国、吉尔吉斯共和国、塔吉克斯坦共和国、土库曼斯坦、乌兹别克斯坦共和国，2021年7月15-16日C5以及美国在乌兹别克斯坦塔什干举行了会议。

声明如下：
发言人办公室
2021年7月16日

哈萨克斯坦共和国、吉尔吉斯共和国、塔吉克斯坦共和国、土库曼斯坦、乌兹别克斯坦共和国，以及美利坚合众国政府，2021年7月15-16日在乌兹别克斯坦塔什干举行的“中亚和南亚：区域互联互通国际会议上通过了以下声明，挑战与机遇”。

文本开始：

C5+1代表团——哈萨克斯坦共和国、吉尔吉斯共和国、塔吉克斯坦共和国、土库曼斯坦、乌兹别克斯坦共和国和美利坚合众国——参加了2021年7月15日至16日在塔什干举行的“中亚和南亚：区域互联互通”国际会议，“挑战与机遇”。

C5+1国家申明致力于通过我们的区域外交平台加强接触，并寻求机会通过贸易、运输和能源联系，加强中亚和南亚区域之间的联系。C5+1认识到，加强互联互通有助于实现中亚繁荣安全的共同目标，中亚经济增长的远见卓识和与南亚经济体更密切的联系，也加强了C5+1对加强该区域安全与稳定的承诺，包括通过阿富汗和平谈判。

C5+1致力于在互联互通方面取得的合作进程，包括：

使中亚的基础设施和中转能力实现现代化，包括通过实施投资项目；
促进加强区域合作，支持跨界企业间联系和人员交流；
改善能源部门的连通性和绩效，包括扩大和有效整合可再生能源；
实现并加强气候适应、空气质量和水资源合作方面的目标，保护最易受气候变化影响的那些部分；
维护人权和自由，进一步加强法治和善政，以普遍的人类价值观以及南亚和中亚人民的历史、文化、文明和精神特性为基础，恢复文化和人道主义联系，在所有努力中促进包容所有人，支持本地区经济可持续发展；
简化南亚和中亚之间的贸易、运输和通讯，包括通过阿富汗；
创造有利于阿富汗和平进程的稳定和繁荣条件，包括向各方重申：就政治解决进行实质性谈判的紧迫性，实现公正和持久和平的唯一途径是通过谈判达成政治解决，从而形成一个包容各方的政治制度，尊重所有阿富汗人的基本权利，没有人支持以武力在阿富汗建立新政府，决不允许恐怖分子和第三方势力利用阿富汗领土威胁或袭击C5+1国家或任何其他国家；
推进与阿富汗在安全、能源、经济、贸易、文化等方面的合作；而且，
合作应对地区安全、繁荣与稳定面临的挑战和威胁。
（声明完）
