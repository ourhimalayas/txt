[美国贸易代表戴琪(Katherine Tai)周三表示，美国必须把贸易政策重点转向重建国内制造业，并减少与不友好经济体的联系。戴琪向众议院筹款委员会表示，俄罗斯入侵乌克兰等全球性事件以及新冠疫情引发的供应链冲击表明需要有新的优先事项。“我们当前面临的问题是，在经历了两年的新冠疫情和俄罗斯对乌克兰的入侵之后，我们所处的这个全球化版本并没有把我们带到一个让我们感到更安全的地方。” “在我们的供应链方面，我们的不安全感正在加深，我们所依赖的合作者并不能让我们放心。”](https://cn.wsj.com/articles/%E7%BE%8E%E5%9B%BD%E8%B4%B8%E6%98%93%E4%BB%A3%E8%A1%A8%E6%88%B4%E7%90%AA-%E7%BE%8E%E5%9B%BD%E5%BA%94%E6%8A%8A%E8%B4%B8%E6%98%93%E6%94%BF%E7%AD%96%E9%87%8D%E7%82%B9%E8%BD%AC%E5%90%91%E9%87%8D%E5%BB%BA%E5%9B%BD%E5%86%85%E5%88%B6%E9%80%A0%E4%B8%9A-11648689954)

**2.欧洲向卡塔尔示好，寄望该国来摆脱对俄罗斯天然气的依赖**

[卡塔尔这个波斯湾小国正在成为欧洲摆脱对俄罗斯天然气依赖的最大希望之一，卡塔尔和欧洲官员表示，德国、法国、比利时和意大利正与卡塔尔就长期购买液化天然气进行谈判。德国经济部长哈贝克(Robert Habeck)本月飞往卡塔尔，宣布与卡塔尔建立能源合作伙伴关系，并承诺建设该国首个液化天然气接收站，接收来自卡塔尔和其他天然气生产国的液化天然气。卡塔尔已经成为世界第二天然气出口国，长期以来一直希望将业务扩展到欧洲，但欧洲买家不愿意购买卡塔尔的天然气，因为来自俄罗斯的天然气更便宜，可以通过现有管道运输，而且合同期限更短、更灵活。](https://cn.wsj.com/articles/%E6%AC%A7%E6%B4%B2%E5%90%91%E5%8D%A1%E5%A1%94%E5%B0%94%E7%A4%BA%E5%A5%BD-%E5%AF%84%E6%9C%9B%E8%AF%A5%E5%9B%BD%E6%91%86%E8%84%B1%E5%AF%B9%E4%BF%84%E7%BD%97%E6%96%AF%E5%A4%A9%E7%84%B6%E6%B0%94%E7%9A%84%E4%BE%9D%E8%B5%96-11648695364)

**3.ONS表示，英国经济回到接近中共病毒大流行前的水平**

[英国经济在2021年最后三个月以1.3%的速度增长，比最初想象的要快，并使GDP略低于大流行前的水平。根据最新的ONS估计，英国经济去年增长了7.4%，从毁灭性的2020年反弹，创下了历史新高，但刚刚错过了7.5%的初步估计。”GDP增长比我们第四季度最初想象的要强一些，这意味着它现在仅比大流行前的水平低0.1%，”ONS经济统计总监Darren Morgan表示。国家统计局（ONS）此前估计，去年10月至12月期间，GDP增长了1.0%。国家统计局还修改了其对大流行开始时GDP崩溃程度的估计 – 将这一数字从9.4%降至9.3%。](https://news.sky.com/story/uk-economy-back-close-to-pre-pandemic-levels-says-ons-12578609)

**4.拜登考虑援引《国防生产法》以促进电动汽车电池的矿产开发**

[一位知情人士说，美国总统拜登正在考虑最快于本周援引《国防生产法》(Defense Production Act)，以促进电动汽车所需电池中使用的矿物的国内生产。该知情人士说，政府将把锂、镍和石墨、钴和锰等矿物纳入朝鲜战争时期的国家安全动员法，以帮助采矿公司获得资金。该人士说，这些公司仍将接受常规的环境审查程序，并且不会加快批准。彭博(Bloomberg)早前报道了拜登的计划。拜登政府努力让美国车主改用电动汽车，这是他应对气候变化计划的关键部分。](https://cn.wsj.com/articles/%E6%8B%9C%E7%99%BB%E8%80%83%E8%99%91%E6%8F%B4%E5%BC%95-%E5%9B%BD%E9%98%B2%E7%94%9F%E4%BA%A7%E6%B3%95-%E4%BB%A5%E4%BF%83%E8%BF%9B%E7%94%B5%E5%8A%A8%E6%B1%BD%E8%BD%A6%E7%94%B5%E6%B1%A0%E7%9A%84%E7%9F%BF%E4%BA%A7%E5%BC%80%E5%8F%91-11648683954)

**5.特斯拉签署了一项”秘密协议”，以避开镍价飙升**

[随着镍市场前所未有的波动余震继续上演，一直受到关注的附带损害领域是电动汽车市场。全球最知名的电动汽车制造商特斯拉（Tesla）已经能够通过与矿业公司淡水河谷（Vale）达成的”秘密”协议来规避市场的波动。彭博社今天上午报道说，一项多年的镍供应协议已经到位，涵盖来自加拿大的镍。”与大多数同行汽车制造商不同，特斯拉多年来一直专注于如何确保自己的镍供应，”报告称。Loup Ventures的Gene Munster表示：”特斯拉对镍所做的是一种隐藏的竞争优势。特斯拉继续领先其他公司几步。](https://www.zerohedge.com/markets/tesla-signed-secret-deal-sidestep-spike-nickel-prices)

**6.房价以2004年以来最快的速度增长**

[英国的房价正以2004年以来的最快速度增长，连续八个月上涨。根据Nationwide的数据，上个月，房价年增长率从2月份的12.6%上升至14.3%。英国房屋的平均价格现在为265，312英镑，比2020年中共病毒大流行爆发前高出约21%。威尔士仍然是今年前三个月表现最强劲的地区，而伦敦仍然是最弱的地区。与此同时，自大流行爆发以来，独立式房产增加了近68，000英镑，平均公寓价格上涨了24，000英镑。”鉴于家庭预算压力越来越大以及借贷成本稳步上升，房地产市场保持了令人惊讶的势头，”Nationwide首席经济学家罗伯特加德纳表示。](https://news.sky.com/story/house-prices-growing-at-fastest-rate-since-2004-12578656)

**7.小麦供应面临“夏季断崖”**

[小麦价格在夏季到来之前进一步上涨的隐忧正在加强。这是因为小麦出口大国乌克兰和俄罗斯的收获量有可能下滑，导致对全球的供应减少。小麦的国际价格3月上旬创出约14年来的高点。虽然目前正在下跌，但如果叠加其他出口国的气候因素等，存在重新上涨的风险。除了能源之外，如果粮食通胀加剧，也将成为拖累世界经济的因素。美国高盛集团3月8日的报告显示，俄乌两国合计的小麦出口的全球份额在8～10月前后会提高至超过4成的水平。如果除了夏秋的收获和出口减少之外，春季的播种也减少，到收获时期或将形成“断崖”。](https://cn.nikkei.com/politicsaeconomy/commodity/48112-2022-03-31-10-03-18.html)

**8.富士通开发出世界最快的模拟量子计算机**

[富士通3月30日宣布，借助应用能进行超高速计算的量子计算机原理的“模拟量子计算机”，成功实现世界最快的计算速度。富士通表示，与德国研究机构等开发的同类设备相比，能以最高约2倍的速度进行计算。模拟量子计算机的特点是配备与日本理化学研究所和富士通联合开发的超级计算机“富岳”相同的CPU（中央处理器）。为了容易完成计算而将数据排列起来，借此实现了高速处理。由于实现了较高的计算速度，还将启动应用方面的研究。将于2022年度应用于携手富士胶片计算化学反应的方法，讨论能否进行改善。](https://cn.nikkei.com/industry/scienceatechnology/48110-2022-03-31-08-42-33.html)
