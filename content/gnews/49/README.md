# 重要事件

- ** **[**10. 07：**Childrenshealthdefense:研究显示完全接种疫苗的国家新增COVID病例数最多Fully Vaccinated Countries Had Highest Number of New COVID Cases, Study Shows](https://childrenshealthdefense.org/defender/fully-vaccinated-countries-highest-number-new-covid-cases/?utm_source=salsa&amp;eType=EmailBlastContent&amp;eId=33296b2c-02fa-450e-8a2d-4fa3482cfe39)


9月30日，发表在同行评议的《欧洲流行病学疫苗杂志》上的一项研究发现，完全接种疫苗的人口比例与COVID新病例之间 “没有明显的关系”。事实上，该研究发现，根据研究人员对9月份七天期间新出现的数据的分析，接种疫苗最充分的国家的COVID新病例数量最高。作者说，仅仅依靠接种疫苗作为缓解COVID-19及其不良后果的主要策略 “需要重新审查”，特别是考虑到Delta（B.1.617.2）变体和未来变体的可能性。

作为研究的一部分，研究人员调查了68个国家和2947个美国县的完全接种疫苗的人口比例和新的COVID病例之间的关系，这些国家和县都有第二剂量的疫苗，并有COVID病例数据。在9月3日之前的七天里，研究人员计算了每个国家每百万人中的COVID病例，以及完全接种疫苗的人口比例。研究人员发现，在完全接种疫苗的人口百分比类别中，COVID新病例有很大的县域差异。在完全接种疫苗的人口比例较高的县，似乎也没有COVID病例减少的明显迹象。在完全接种疫苗的人口比例最高的前五个县中（99.9%-84.3%），疾病控制和预防中心（CDC）将四个县确定为 “高 “传播县。在被列为 “高 “传播的四个县中，有三个县的完全接种率达到90%或更高。相反，在57个被CDC列为 “低 “传播的县中，15个县的完全接种率为20%或更低。

研究结果还显示，如果考虑到疫苗生效需要14天的时间，那么在COVID病例和完全接种率之间没有明显的联系。作者建议对政策叙述进行修正，因为提高疫苗接种率是不够的。他们写道：”随着关于疫苗现实世界有效性的新的科学证据的出现，这种课程的纠正，特别是关于政策叙述的纠正，变得至关重要。“作者引用了以色列卫生部的数据，显示两剂辉瑞公司的COVID疫苗对预防SARS-CoV-2感染的有效性被报告为39%–大大低于96%的试验有效性报告。

新的研究还显示，从辉瑞公司COVID疫苗中获得的免疫力可能不如通过感染获得的自然免疫力强。研究人员说，也有报告称，免疫接种后六个月，mRNA疫苗的免疫力大幅下降，同时完全接种的人中突破性病例的数量也在增加。作者说，在疫苗问题上对人群进行污名化可能弊大于利，需要重新进行非药物预防工作，以便学会与COVID共处，”就像我们在100年后继续与1918年流感病毒的各种季节性变化共处一样”。

中情局周四证实，在中情局局长比尔-伯恩斯今年早些时候下令进行重大审查后，它已经建立了一个新的以中国为重点的 “任务中心”。伯恩斯局长发表的一份新声明说，新的中国任务中心 “将进一步加强我们在21世纪面临的最重要的地缘政治威胁方面的集体工作，即一个日益敌对的中国政府。”

这是在批评中国最终 “超越 “美国的情况下发生的，也是在多年来针锋相对的网络入侵指控中发生的。本周《纽约时报》的一份单独报告也表明，美国情报官员越来越担心，中国是越来越多地逮捕、处决或经常危害美国海外特工的少数几个国家之一。中国中心将在与中国有关的情报事务方面拥有更狭隘的任务，这将包括更广泛的资源。 而不是在先前更大的 “东亚和太平洋任务中心 “的主持下收集有关中国的情报。彭博社先前的一份报告指出：任务中心是独立的实体，根据机构的优先事项利用整个中情局的资源。显然，整个中国任务中心已经讨论了多年，但没有任何局长或前任政府愿意下令成立，可能是担心这会引发冷战式的升级和针锋相对的秘密行动。

周四中情局的公告还透露了内部部门的进一步重组。同时，另外两个传统上高度优先的领域将不再是独立的，而是其区域的一部分。伊朗和韩国任务中心将分别并入近东任务中心和东亚及太平洋中心。其他调整包括一个新的跨国和技术任务中心和一个首席技术官职位。中情局一位高级官员说，这第二个任务中心将专注于 “对美国全球竞争力至关重要 “的问题，包括全球健康、经济安全、气候变化和技术。

在这次重大调整之前建立的最后一个主要任务中心，即韩国任务中心，是在2017年特朗普执政时期，当时白宫正试图评估北方的核能力以及如何对抗平壤。中国领导人肯定会认为这一最新的内部举措是更广泛的紧张局势升级的一部分，并可能会反映美国的情报工作，特别是考虑到最近几个月中国专门指控中情局黑客攻击北京长达十多年。

- [** 10.07. 法广：**忧引发心脏疾病，丹麦和瑞典暂停年轻人接种莫德纳疫苗](https://www.rfi.fr/cn/%E5%9B%BD%E9%99%85/20211007-%E5%BF%A7%E5%BC%95%E5%8F%91%E5%BF%83%E8%84%8F%E7%96%BE%E7%97%85%EF%BC%8C%E4%B8%B9%E9%BA%A6%E5%92%8C%E7%91%9E%E5%85%B8%E6%9A%82%E5%81%9C%E5%B9%B4%E8%BD%BB%E4%BA%BA%E6%8E%A5%E7%A7%8D%E8%8E%AB%E5%BE%B7%E7%BA%B3%E7%96%AB%E8%8B%97)


由于担心莫德纳新冠疫苗会引发接种者出现心脏疾病，瑞典和丹麦官员10月6日宣布暂停为年轻人接种莫德纳疫苗。

据《每日邮报》网站报导，瑞典卫生局6日宣布，从即日起至12月1日，暂停为1990年以后出生或30岁以下的年轻人接种莫德纳疫苗，因为数据显示，年轻人，特别是男性年轻人在接种疫苗后出现心肌炎和心包炎的病患在增多。瑞典卫生局在声明中表示，清晰的证据显示了接种莫德纳疫苗和病患出现心肌炎（Myocarditis）和心包炎（pericarditis）之间的关系，特别是在接种第二剂疫苗后，接种者出现心肌炎和心包炎的可能性更高，因此，他们暂停30岁以下的年轻人接种莫德纳疫苗。已经宣布主要为12岁至17岁的青少年接种辉瑞疫苗的丹麦当局则于6日表示，“出于审慎原因”，暂停为18岁以下的年轻人接种莫德纳疫苗。

心肌炎是心肌本身发生炎症；心包炎是包裹心脏且充满液体的囊发生炎症，这都是年轻人，特别是男性年轻人接种新冠疫苗后可能会出现的“罕见”的副作用；但是心肌炎和心包炎也是人们染疫后容易出现的一种疾病。一些研究的结果显示，染疫后出现心肌炎和心包炎的机率高于接种疫苗后出现的机率。美国南加州凯撒医疗（Kaiser Permanente）最近的一份研究显示，每100万人中会有7个人在接种两剂新冠疫苗后出现心肌炎；同一个研究还发现，每100万感染新冠病毒的病患中，有47.5人出现心肌炎；而另外一份尚未经过同行审查的美国研究发现，20岁以下的男性年轻人感染病毒后出现心肌炎的病患数量是接种疫苗后出现心肌炎的人的6倍。

美国疾控中心（CDC）今年6月份发布的一份声明表示，年轻男子接种新冠疫苗后出现心肌炎的风险在增加。美国疾控中心虽然并未暂停为年轻人接种这两款疫苗，美国对于辉瑞和莫德纳疫苗的说明中已经就此问题进行了更新。莫德纳公司的一位女发言人表示，该公司知道丹麦和瑞典两国卫生当局做的决定，并声称这些疫苗接种者出现的心肌炎和心包炎的症状都比较轻微。声明写道：“出现炎症的病患往往都能够在接受治疗和休息后很快康复。然而由于染疫后出现心肌炎的风险却在实质性地增加，因此接种疫苗仍然是防止染疫后出现严重心肌炎的最佳方式。”
