## 所罗门群岛禁止中共在该国建立军事基地
` NFSC News` [轉載自GNews](https://gnews.org/zh-hans/2275967/)

作者：纽约磐石农场 – 在远方
![This image has an empty alt attribute; its file name is Screen-Shot-2021-08-30-at-7.37.59-PM.png](https://assets.gnews.org/wp-content/uploads/2021/08/Screen-Shot-2021-08-30-at-7.37.59-PM.png)
台媒报道，4月1日，太平洋岛国所罗门群岛总理苏嘉瓦瑞表示，不允许中共在该国建立军事基地。此前，所罗门群岛与中共国签署了安全合作框架协议草案，一份外泄的草案显示，允许中共国海军船舰在索罗门岛进行补给等内容，引发了南太平洋区域国家强烈反弹。
![](https://assets.gnews.org/wp-content/uploads/2022/04/10-3.jpg)
密克罗尼西亚联邦领导人于3月31日，敦促所罗门群岛不要签署该协议，对此表达了严重的安全关切，并担心太平洋地区会卷入中共与美国之间的战争。新西兰政府警告说，该协议可能破坏长期的区域安全合作。澳大利亚国防部长杜登对苏嘉瓦瑞的观点表示尊重，但敦促他谨慎行事。杜登说，中共已经在南海设立了20个军事据点，尽管中共告知美国不会将该地区军事化。但澳洲政府担心北京当局，会在太平洋岛国采取一贯的欺骗手段。

目前为止，该安全协议尚未正式签署。
