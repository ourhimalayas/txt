**郭文贵先生2018年1月4日直播：**

王恩哥有一次跟我讲，“文贵赶快用我，我现在别的不敢说，关系、朋友多，绵恒那儿、江志成那儿，你都说说，有啥事跟我说”，建议我多见见毛毛，毛毛就是江志成，江志成是1986的1月22号出生的，他现在有个女朋友叫夏玉，是1993年4月15号出生的。我今天不是爆料夏玉女士，我们今年春节的时候好好说说夏玉和江志成，86年江志成和93年的夏玉，我们到春节的时候、三十的晚上再说。

然后王恩哥强烈推荐我要跟江志成、或者上海银行其它方面的合作，他来牵头。后来因为王恩哥要用我私人飞机飞美国，他和李友，还有李友的女儿，后来还上来了几个美女，这咱都有飞行记录的，飞了美国一大圈，中国飞美国，美国飞中国，花了一千多万吧。在我和李友闹僵了以后，北大支付了一千多万的飞机费用。那次来美国是什么事呢？因为王恩哥的儿子和儿媳妇在另外一个大学上学，他想上哈佛，还有一个是在斯坦福继续读博士，一个斯坦福的儿媳妇。然后我一个朋友帮他们忙，进入哈佛继续读博。结果是在几十个人的竞争中，三百多个，让他儿子和儿媳妇，让他入了哈佛。

后来我到美国来，他儿子儿媳妇到迈阿密跟我一起喝了两天酒，都喝大了，还有变形金刚的导演麦可·贝(Michael Benjamin Bay)，我们还去了他家。结果他们很高兴，因为他爹都搞不定，（我）最后搞定入了哈佛大学读博，他很高兴。

那次王恩哥和李友携众多美女来美国，除了度假，除了感谢哈佛，还有一个目的：安排江绵恒在美国一些事情。江志成和夏玉他们主要在洛杉矶。其中他就跟我谈到了田国立的儿子，叫田丁，是1988年10月14号出生的，然后他太太小李，我就不说了……

**郭文贵先生2018年1月15日直播：**

我今天春节的时候，要向田惠宁先生——65年出生的田惠宁（说这么多），我不想（说）关于你老婆周圆，我也不想（说）关于你儿子田雨楠那么多，你知道我为什么，我希望你最近的活动要停下来。我也不想和1986年1月份出生的江志成（说那么多），我不希望你太嚣张，你去你的达沃斯，你享受马云带去的七个厨师，去行贿全世界的领导、精英，和马云带七个厨师去做饭、去吃饭、去喝酒，你别惹我郭文贵，我也不想对你老婆夏玉做任何事情。你老婆93年出生，我就不说，你知道为什么，包括你其他人我都知道。

江志成先生，我在说的话，你过去的事我都没说，你都明白，咱们之间、中间的人我都没说，但是你和孙力军、你和孟建柱、你和王岐山，你继续想通过这些人收拾郭文贵、想灭郭文贵，你继续相信吴征这个烂人，你继续在海外这样扩张，你认为世界上没办法，咱们走着瞧。

**郭文贵先生2018年8月5日直播：**

记者招待会我们一定会开，会开的大家无法想象。海航已经是死驴一个了，陈峰、陈国军、王健，你们的使命完了，叫你们享受了一辈子，拥有了全人类唯一的787，你们跟了王岐山、跟了共产党，你们用这种挣扎式的堕落和被杀来挽回你们下入更深层次的地狱唯一的机会。盗国贼你们记住，下面还会爆三家公司。

江志成，你和我共同的朋友，这位英国领导人，你让他给我打电话，打电话之后24小时，我哥哥第二次被抓。我为啥没打给孙力军，你去问，从我申请政庇以后，孙力军、孟建柱你们干了啥？

孟建柱你的绿地,、江西铜业，孙力军你搞肖建华的现金，包括郭广昌和你之间这种利益，孙力军当年你带着我最好的朋友进了郭广昌的办公室，你说了啥，后来你两啥关系你很清楚，我等着呢，潘多拉盒子没有打开，南普陀会议不是假的，习近平当时就被定为一个托，就是个临时的托。

**郭文贵先生2018年8月13日直播：**

我有很多话不能说，但我可以告诉所有的战友们，跟你们站在一起的、跟我爆料革命站在一起的后边的人，超出所有人想象中的想象。还是那句话，阳谷莘县县搭县。

我不说别的，我没有任何组织。那些组织我看他靠我还差不多，我靠他们什么啊？哪个组织我不认识？你们看我跟谁在一起，我跟哪个组织，你们没有注意到，那个老爷子——罗斯柴尔德（家族）老爷子Jacky跟我的照片。在那之后，我还推出过两三个比他还厉害（人物）的照片，我不说大家没人注意，很多人不知道。这些人都是绝对支持我的，都是我的哥们儿，好得很好得很，包括台湾香港日本新加坡很多大家族。

战友们，我要是需要他们的帮助，那真是太多太多人了。但是这些人从根本上帮我们实现不了喜马拉雅。实现喜马拉雅只能靠你们现在能想象中之外的那几种力量。

接下来说这些盗国贼的家人。我就举很简单的例子，你说江泽民的孙子江志成拿着国外护照，拿着外国人的钱，你说外国人能拉倒吗？孟建柱的钱能拉倒吗？中南坑里现在哪个领导人的家人不是外国护照、澳大利亚的加拿大的美国的欧洲的（护照）？哪家人没有海外联姻的婚姻？跟我说一家！有一家没有的？从习近平到李克强、韩正、赵乐际，找找，包括我们这郭声琨，没有亲戚？没有联姻？没有海外资产的给我找出一个，一个没有！

西方只要不形成一个集团性的对我们的帮助没有用，没有用的，哪个人也没有用的。
