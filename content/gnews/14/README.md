## 为什么我写了《真正的安东尼·福奇》这本书

**编译： Jenny Ball**
![](https://assets.gnews.org/wp-content/uploads/2021/12/tempsnip01-1.png)图片来源：childrenshealthdefense
编者注：以下是《真正的安东尼·福奇：比尔·盖茨、大型制药公司以及全球民主和公共卫生战争》的摘录，该畅销书由儿童健康防御董事长兼首席法律顾问小罗伯特·肯尼迪 (Robert F. Kennedy, Jr.) 撰写。

虽然一些共和党人对福奇博士不断积累的权力和看似武断的言论持谨慎态度，但政治部落主义的炼金术和 COVID-19 的无情煽动的恐怖说服，着迷的民主党人对他的 COVID-19 政策是一场灾难性失败和危险的确凿证据，闭眼无视。

作为公共卫生、强大科学和独立监管机构的倡导者——不应该腐败和与制药公司的财务纠葛——我与福奇博士斗争已经很多年了。

我个人认识他，我对他的印象与我的民主党同胞大不相同，他们第一次见到他是在电视转播的白宫 COVID 新闻发布会上，他是一位优雅、谦逊、认真、可爱和长期受苦的明星。

福奇博士作为**“**机构俘获**”**的主要建筑师，发挥了历史性的作用**——**制药业对美国公共卫生机构的公司侵占。

可悲的是，福奇博士在 COVID 大流行期间，未能实现公共卫生目标，并不是特别的错误，而是他与制药利润和自身利益相一致，在他们的祭坛上，反复牺牲公共卫生和安全。他始终将制药行业的利润置于公共卫生之上。

读者将在[这些页里](https://childrenshealthdefense.org/fauci_info/)了解福奇博士在其漫长的职业生涯中如何提升专利药物，经常伪造科学，欺骗公众和医生，并在安全性和有效性方面撒谎。

在本书中详述的福奇博士的罪行包括，他对数百名黑人和西班牙裔孤儿和寄养儿童的罪行，他对这些儿童进行了残酷和致命的医学实验，以及他与比尔·盖茨一起，将数十万非洲人变成危险实验药物的低成本临床试验的实验室老鼠的角色，一旦药物获得批准，大多数非洲人却在经济上仍然对其遥不可及。

您将了解福奇博士和盖茨先生如何将非洲大陆变成过期、危险和无效药物的倾倒场，其中许多药物在美国和欧洲因安全原因而停产。

您还将了解他这两年宣传虚假流行病的战略，以此作为促进新型疫苗、药物和制药公司利润的计划。

您将了解到他隐瞒血液和疫苗中广泛污染的行为，他对挑战制药范式的科学家的破坏性仇杀，他故意破坏从** HIV **到** COVID-19 **等传染病的专利过期疗法，为那些不太有效但更有利可图的药物的补救措施。

您将了解到在他冷血地关注行业利润,而不是公共卫生之后积累的奇怪的身体数据。

他在 COVID 期间的所有策略——伪造科学，将危险和无效的药物推向市场，压制和破坏利润率较低的竞争产品，即使代价是延长流行病和失去数千人的生命——所有这些都有一个共同的目的：对制药企业的奉献。

这本书会告诉你，福奇不是在做公共卫生；他是一名商人，他利用自己的办公室来丰富他的制药合作伙伴，并扩大影响范围，这使他成为人类历史上最强大和专制的医生。

对于一些读者来说，要得出这个结论需要跨越一些新的桥梁。然而，许多读者凭直觉就知道真正的安东尼福奇，只需要看到书中阐明和组织的事实。

我写这本书是为了让美国人——包括民主党人和共和党人——能够理解福奇博士在允许制药公司主宰我们的政府和颠覆我们的民主方面所扮演的有害角色，并记录福奇博士在当前反对民主的政变中发挥的关键作用 。
