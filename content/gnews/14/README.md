## 7·24系列——“毒刺”导弹，俄军的梦魇

在俄乌战争爆发后，乌克兰的防空，尤其是反导弹能力，一直被西方专家担心可能是首先倒下的最薄弱环节。

但接下来的情况表明，俄罗斯未能完全消除乌克兰的防空力量并确保绝对的空中优势。据五角大楼称，乌克兰军队每天有5到10架次的飞行，而俄罗斯进行的飞行大约有200架次。但乌克兰还是成功地保住了其空中力量。

据美国国防部官员称，在许多方面，这要归功于乌克兰人在地面上有效地使用了灵活、便携式的防御导弹系统。**乌克兰不仅有效使用了9K38 Igla（北约报告名称为SA-18 Grouse）等老式苏联便携式系统，由西方提供的毒刺导弹（FIM-92 Stinger）则更加发挥了重用的作用，有效的打击了俄罗斯的空中优势**。

2月25日，作为白宫批准的一揽子计划的一部分，美国首次批准直接向乌克兰运送“毒刺”防空导弹。

FIM-92毒刺便携式防空导弹（FIM-92 Stinger）中文一般称为便携式防空导弹或肩射导弹，为小型便携式防空导弹，采用红外线制导热追踪以及紫外线物体追踪，由美国研发并使用于所有美军部队，1981年服役至今。在美国和其他29个国家中。主要由雷神导弹系统公司制造，授权德国的EADS和土耳其的ROKETSAN生产导弹，已经生产超过70,000枚导弹。经证实有270架飞机和直升机被FIM-92毒刺便携式防空导弹击落。

FIM-92毒刺便携式防空导弹的设计是便于搬运和操作的防御型导弹，虽然官方要求两人一组操作，不过单人亦可操作。FIM-92B可以攻击15,700英尺（4800米）范围内和12,500英尺（3800米）高度下的飞机。其可安装在悍马 (军用汽车)改装的复仇者防空系统载具上或M2布莱德雷步兵坦克上，并可以由伞兵携带快速部署于敌军后方，至于直升机发射版被称为ATA或AIM-92 Stinger。

毒刺导弹弹体长1.52米、直径70毫米，尾翼100毫米，导弹重10.1千克，包括发射架在内重15.2千克（33.5磅）。毒刺由小型两节固体火箭推进，喷焰在发射者后方安全距离外，导弹最高速度为2.2马赫（750米／秒），战斗部为3千克穿透战斗部和延迟引信火药。

发射时要将电池冷冻模组（BCU）插入手把，这将注入一股氩气进入系统，并使瞄准器和导弹通电，但电池只有有限的气体量而不能随便使用，在缺乏保养的情况下，经过长时间电池会变得无法使用，而发射器瞄准天线则使用充电电池。

**毒刺有三种衍生型；基本型、被动光学型（POST）、软件电脑型（RMP）。**

软件电脑型（RMP）主要针对近年来战机常装设的反制系统提供反反制的可能性，因为其可以载入随时升级的软件和参数表以克制各种推陈出新的反制手法，其还可以在插槽上的只读内存中载入预设设定程式，如果软件发生差错还能用只读内存中的程式维持基本导弹功能，其有4个CPU每1CPU有4KBRAM，程式运作时载入到RAM，因此并没有许多备用内存空间，尤其是跑目标搜索和分析时。软件电脑型RMP有双导向：红外线和紫外线双波段追踪，再加上软件控制，使它更有机会躲过敌机的反制手段击中敌机，其与只有红外线的红眼（Redeye）导弹相比，效能高出极多。

毒刺导弹初次实战是在1982年5月21日。英国空军特种部队（SAS）在福克兰群岛（马尔维纳斯群岛）战争期间使用了少量的FIM-92A“毒刺”，摧毁了一架阿根廷空军的FMA IA 58A Pucara双螺旋桨近距支援飞机。此后，毒刺导弹参与了印巴战争、安哥拉内战、乍得与利比亚的冲突、阿富汗战争、马岛战争、斯里兰卡内战、海湾战争、塔吉克斯坦内战、车臣战争、叙利亚内战等。尤其在苏联与阿富汗的战争中，美国向圣战者提供了毒刺导弹，对苏联的直升机和固定翼飞机造成了沉重打击。

各型“毒刺”导弹系统销售到世界范围内的许多国家军队中，包括英国、日本、韩国、以色列等。除了正式渠道，一些第三世界国家也能采用“非正式”渠道获得。“毒刺”导弹系统现已整合和配置在超过40个军种的20种车辆和直升飞机平台上。在世界范围全部被击落的飞机中，约300架归于被“毒刺”击落。

至于价格，1980 年一枚毒刺导弹的售价约为 38000 至 40000 美元。目前，这种导弹的成本约为 119000 至 120000 美元。这是制造商设定的官方价格。

毒刺导弹具有两大优势，就是“射后不理”和“敌我识别”。

“射后不理”是毒刺导弹的最大优点。射手一旦按动发射按钮，导弹飞离开发射管后，即可迅速装配另外一枚导弹用于下一步的交战（小于10秒内），隐蔽或移动到另外的一个作战位置。毒刺导弹具有双导向：红外线和紫外线双波段追踪，再加上软件控制，能够提供全方位探测和自导引能力。当导弹接近它的目标时，导引头在碰撞目标前的一秒内启动它的“目标适应引导”电路，去调整弹道从尾喷口排气热流转而朝向红外目标自身。引信系统允许碰撞激活或在导弹达到20秒的飞行时间后激活自毁。毒刺可以等飞机一接近就进行迎头攻击，这一反应往往能在短程弹药或开始地面扫射之前实现。而导弹依靠三级火箭发动机实施动力推进，攻击射程有充足保障。

同时毒刺导弹安装一套综合AN/PPX-1IFF（敌我识别）系统，能够对敌我目标进行区分，避免误射。当射手在光学瞄准器的视域中捕获目标时，只需按下IFF呼叫开关，该系统能够在在0.7秒内给射手一个音频讯号提示，告知目标是友军或无法识别。IFF系统的方位角在10公里范围内是相同的。当战场情况复杂时，识别敌我的能力至关重要，这样才能保证防空导弹，对敌机实施有效的打击，保护己方的战机，免遭己方防空导弹的打击。
