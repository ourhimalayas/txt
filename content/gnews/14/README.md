## 疫苗接种超8成，新加坡单日新增感染人数再创新高
` GTV新闻访谈` [轉載自GNews](https://gnews.org/zh-hans/1568856/)

栏目记者/责任编辑：云豹

![This image has an empty alt attribute; its file name is Screen-Shot-2021-08-30-at-7.37.59-PM.png](https://assets.gnews.org/wp-content/uploads/2021/08/Screen-Shot-2021-08-30-at-7.37.59-PM.png)



新加坡自9月15日开始为60岁及以上的年长者、养老机构居民施打加强针，截至目前，全境疫苗注射率已超过8成。但是从8月中开始感染病例不断攀升，已经连续4天单日新增病例超过2000人，死亡病例也随之增加。10月1日，新加坡单日新增2900多人确诊，再创单日新增感染人数新高。
![](https://assets.gnews.org/wp-content/uploads/2021/10/2021.10.2AM-PICTURES-1234689.007.jpeg)**疫苗接种率（粉色：一剂；红色：二剂 的注射）**![](https://assets.gnews.org/wp-content/uploads/2021/10/2021.10.2AM-PICTURES-1234689.008.jpeg)**每日新增病例的7天平均值**
对于新冠疫情，新加坡总体采取「与病毒共存」的应对策略，但在大量注射疫苗后，新增病例仍不断攀高，证明疫苗无法防止感染，且病例增加速度如此之快，迫使新加坡政府宣布本周起，全境将再度实施加强防疫措施。
