## 有声|《灭共的雪》是前无古人的史诗般的音乐
` MOSREC` [轉載自GNews](https://gnews.org/zh-hans/1957501/)

配音：香草山巡管部– 文凯琳
![This image has an empty alt attribute; its file name is Screen-Shot-2021-08-30-at-7.37.59-PM.png](https://assets.gnews.org/wp-content/uploads/2021/08/Screen-Shot-2021-08-30-at-7.37.59-PM.png)
在1月31日的壬寅年新春大直播中，郭文贵先生现场首次演唱了歌曲《灭共的雪》，而该曲的摇滚版MV当天也在盖特平台上同步推出。此外，郭先生还在直播中解密了这首改编歌曲的幕后故事。

郭先生特别感谢了墙内无数默默奉献的战友，他们以全家的安全和生命为代价，一直支持着新中国联邦。当中，一位参与了中共冬奥会的战友向郭先生透露，《塞北的雪》已经被中共作为冬奥会开幕式的主打曲之一，而彭丽媛当年也是借此曲成名得以进入北京。

这位战友透露，按照中共一贯的伎俩，只要是郭先生唱过的歌曲，其在中共国内必然遭到封禁。为此，郭先生决定在冬奥会开幕前公开演唱这首改编版的《灭共的雪》。
