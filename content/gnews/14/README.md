**北欧国家丹麦、挪威、瑞典相继解除中共病毒（****COVID-19****）疫情限制措施**

继2021年9月10日，丹麦成为第一个解除中共新冠病毒（COVID-19）限制的北欧国家之后，从2021年9月25日开始，挪威政府取消了大部分剩余的COVID-19疫情限制措施，不再要求企业实施保持社交距离的措施，体育文化场所和餐馆可以充分利用其服务空间正常营业，夜总会可以重新开业。至此和平时期推出的最严厉疫情限制措施，在拥有530万人口，且约67%已经完全接种了中共新冠病毒疫苗的挪威执行了561天后嘎然而止。当天挪威各地成千上万的市民走上街头狂欢，街道、酒吧、餐馆和夜总会挤满了人，庆祝中共病毒限制措施的结束，庆祝社会生活秩序恢复正常。庆祝活动一直持续到2021年9月26日凌晨。

挪威公共卫生研究所（NIPH）助理所长布克霍尔姆（Geir Bukholm）表示，我们现处于一个新阶段，COVID-19病毒现在可以被视为具有季节性变化的几种呼吸道疾病之一；COVID-19在挪威已被列为一种普通危险的疾病，但官方对其的分类可能很快会改变。

![](https://assets.gnews.org/wp-content/uploads/2021/10/16-3.jpg)

瑞典本月早些时候也宣布，自2021年9月29日开始，取消大部分对COVID-19疫情的限制措施，让民众回归正常社会生活。

**未接受每周中共新冠（****COVID-19****）病毒检测的宾州大学学生被临时停课**

据2021年9月27日零对冲的报道，宾州大学在秋季学期开始之前已经宣布，没有接种COVID-19疫苗和没有疫苗接种信息的学生和员工，必须全学期进行每周COVID-19病毒测试，并警告不遵守病毒测试要求这类的学校健康与安全政策的人，将面临严重后果。错过了两周病毒测试的学生将被记录在案，并被禁止注册新课程；错过了三周以上病毒测试的学生，学生操守办公室将通知他们进行临时停课。117名学生因此已被临时停课，且不得在校或线上上课；不得进入大学校园设施；不得参与宾州大学赞助的任何项目和活动；并被迫暂时从宿舍楼搬走。

上周，为了避免更多的学生被临时停课，宾州大学工作人员给错过三次病毒测试的每位学生都打了电话，帮助他们尽可能解决遇到的问题，这让数百名学生及时避免了被临时停课。同时，没有完全接种疫苗的学生每周都会收到电子邮件，提示他们及时完成学校要求的病毒检测。学校人力资源部也正在检查教职员工的病毒测试数据，以确定还有多少人未符合此要求。
![](https://assets.gnews.org/wp-content/uploads/2021/10/17-3.jpg)
负责学生事务的副校长西姆斯（Damon Sims）表示，在疫情期间，学生和员工都要遵守我们的病毒测试要求，并开设校园诊所供其做病毒测试；所有无特殊情况的人都应该接种疫苗，并尽快将接种数据上传学校，且为学生和员工提供疫苗接种奖励。学校已经事先做了一切能做的事情，以确保学生意识到自己的义务并做他们该做的事情。学校最不希望的就是让学生停课，但很遗憾有些学生仍没有听取学校的多次劝诫和警告，希望他们能做出必要的改正，其他人不要重复他们的错误。

被临时停课的学生，在完成疫苗接种或在校完成中共病毒COVID-19病毒检测这两项措施中的一项后，可在网上提交一份表格申请取消临时停课，但每位学生只有一次解除临时停课的机会。在接到学校关于取消其临时停课的明确通知之前，临时停课仍然有效。计划在校内诊所完成COVID-19病毒检测的临时停课学生，必须选择最短的路线前往检测地点，并在检测完成后立即离开。

**辉瑞提交针对****5-11****岁儿童的疫苗试验数据，并宣布一项类伊维菌素药物进入最终实验；赛诺菲放弃了****mRNA ****疫苗的开发**

2021年9月28日，辉瑞（Pfizer）向美国食品药品监督管理局（FDA）提交了其针对5至11岁未成年人的mRNA 新冠疫苗“Comirnaty”之2/3期临床试验数据，随即福奇就表示希望FDA能在10月底前为儿童对该疫苗的接种“开绿灯”。人们普遍猜测：FDA将很快批准该疫苗用于低于5岁的患者。因为今年5月，辉瑞完成了针对12-15岁儿童的疫苗试验，并称该疫苗在预防该年龄段儿童的COVID-19重症方面“100%有效”后，辉瑞就在9月20日宣布该疫苗对年龄低至5岁的儿童都是“安全有效的”。辉瑞完全故意混淆乃至颠倒了预防性疫苗和治疗性药物的概念！
![](https://assets.gnews.org/wp-content/uploads/2021/10/18-5.jpg)
辉瑞昨晚还宣布已经开始了一项预防性的COVID-19治疗药物的最终试验，并称就功能而言，该治疗药物与伊维菌素（Ivermectin）类似。价格低廉且对中共COVID-19病毒感染者疗效显著的传统药物伊维菌素，一直被大型制药公司及其合作伙伴极尽诋毁，因为伊维菌素妨碍了他们利用自己昂贵的疫苗和新药赚大钱。迄今，吉利德（Gilead）非常昂贵但并不是特别有效的瑞德西韦（Remdesivir）是唯一被FDA批准用于治疗COVID-19的药物。

同时，赛诺菲（Sanofi）今天却宣布放弃自己基于mRNA的COVID-19疫苗开发计划，因为担心无法与BioNTech-Pfizer和Moderna生产的疫苗竞争。赛诺菲表示，尽管I/II期试验结果相对强劲，但还是决定放弃自己的mRNA疫苗的临床开发，并称此举是赛诺菲收购Translate Bio的一部分。赛诺菲还表示将与英国制药巨头葛兰素史克（GSK）合作，基于一种更“传统”的方法研发一种替代疫苗。

**强生公司核心职员表示，孩子们不应该接种****Covid-19****疫苗**

据零对冲2021年9月29日报道，“真理计划（Project Veritas）”推出了疫苗业内人士系列节目的第三期，本次节目秘密记录了强生（J&J）两名官员的讲话，他们都说儿童不应该接种Covid-19疫苗。

强生公司地区商业主管沙特（Brandon Schatt）表示，就Covid-19疫苗未来的影响而言，这并不是一件未知的事情，孩子们不应该接种该死的COVID-19疫苗。大力推动儿童接种疫苗是为了钱，而不是为了公共安全。强生现在就像踩在一堆臭气熏天的大便里，众目睽睽之下无地自容。

强生公司科学家杜兰特（Justin Durrant）明确表示，就当我没告诉你，千万不要接种强生COVID-19疫苗，孩子同样不要接种该疫苗。杜兰特还评论了限制未接种疫苗者社会活动权利的施压运动：如果你没有接种COVID-19疫苗，你几乎就像一个二等公民，不能做任何普通公民能做的事情，如不能离开本州、更不能出国；甚至失业导致口袋没有收入这样最严厉的惩罚。当局的疫苗强制接种令在社会上形成如此严峻而苛刻的压力氛围， 让未接种疫苗的人在社会活动中的不方便程度达到令人窒息的状态，使其萌生还不如直接第一个去排队接种中共COVID-19病毒疫苗的想法和冲动，以达到公民最后不得不都去接种疫苗的结果，这就是疫苗公司与当局正在做的事情。
