## 泽连斯基称乌克兰绝不会被征服
` NFSC News` [轉載自GNews](https://gnews.org/zh-hans/2169346/)

作者：磐石农场 在远方
![This image has an empty alt attribute; its file name is Screen-Shot-2021-08-30-at-7.37.59-PM.png](https://assets.gnews.org/wp-content/uploads/2021/08/Screen-Shot-2021-08-30-at-7.37.59-PM.png)
盖特3月14日消息，近日乌克兰总统泽连斯基发表视频演说表示，俄罗斯侵略者无法征服乌克兰，呼吁欧洲各国共同抗击俄国侵略。
![](https://assets.gnews.org/wp-content/uploads/2022/03/6-28.jpg)
泽连斯基说，俄罗斯侵略者缺乏勇气和意志，他们只是延持續使用暴力和恐怖袭击、只是武器多于乌克兰而已。但是他们却没有人类正常生活的自然思维，以便让人民感觉幸福和拥有梦想。

泽连斯基嘲讽俄罗斯侵略者天生没有正常生活的能力，无论俄军的铁蹄践踏在世界任何地方，这样的梦想都无法会实现，只能非常艰难的为生存而斗争。

泽连斯基表示，他不断的向友好国家表明，希望他们为乌克兰人民做出更多的事情來，因为这不仅仅是为了乌克兰，而是为了全体欧洲人民。俄罗斯侵略军蓄意轰炸和平城市，袭击救护车和医院的行为，不会只是针对乌克兰。一旦实力足够，普京政府就会变本加厉的继续侵略下去。
