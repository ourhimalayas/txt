据《The Verge》记者 Emma Roth，2021年10月30日报道：
![](https://assets.gnews.org/wp-content/uploads/2021/11/image-27-e1635902507831.png)图片来源: Alex Castro / The Verge
**Facebook提醒用户该疫苗现已可供儿童使用**

就在美国食品药品监督管理局(FDA)正式批准辉瑞(Pfizer)公司的针对5岁至11岁儿童的COVID-19疫苗之际，**Facebook的全新身份Meta公司宣布，将对儿童疫苗的错误信息推出更严格的政策。**该平台此前在2020年底对COVID-19疫苗的错误信息进行了限制，但没有专门针对儿童的政策。

Meta在一篇新的博客文章中表示，**它正在与美国疾病控制与预防中心(CDC)和世界卫生组织(WHO)合作，删除平台上传播的与儿童COVID-19疫苗有关的“错误”内容。这包括任何暗示COVID-19疫苗不安全、未经测试或对儿童无效的帖子**。此外，Meta公司还将提供英语和西班牙语的即时提醒，告知该疫苗已被批准用于儿童，并提供在何处可接种该疫苗的信息。

![](https://assets.gnews.org/wp-content/uploads/2021/11/image-28-e1635902683951.png)图片来源：Facebook

























Meta 指出，自中共病毒COVID-19疫情开始以来，它总共从 Facebook 和 Instagram 删除了2000万条涉及COVID-19 和疫苗的“错误信息”。

这些数字与我们从 Facebook 泄露的内部文件中看到的情况不一致——Facebook 文件清楚地表明，该平台对与 COVID-19 疫苗相关的错误信息准备不足。 如果 Facebook 准备得更充分，它可能会在疫情早期发起打击（针对儿童和成人的） “错误信息”的活动，从而可能会删除更多“虚假内容”。
