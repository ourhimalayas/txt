## 安徽某民警将一女子强奸致死后嫁祸其夫

![This image has an empty alt attribute; its file name is Screen-Shot-2021-08-30-at-7.37.59-PM.png](https://assets.gnews.org/wp-content/uploads/2021/08/Screen-Shot-2021-08-30-at-7.37.59-PM.png)
当地时间12月28日，据中共国内媒体报道，安徽省一名被指控杀妻的男子在服刑17年后被改判为无罪。另据中共国检察网最新消息显示，该男子的妻子实为被当地一名公安局民警强奸致死。
![](https://assets.gnews.org/wp-content/uploads/2021/12/截屏2021-12-30-下午10.19.50.png)
据了解，该男子曾与1996年被指控因家中2800元来源不明的现金而将妻子残忍杀害。对于该项指控，这名男子在最初遭拘捕时被逼认供，随后被当地法院判处死刑，缓期两年执行。

在法院宣判后的十几年间，该名男子不断进行上诉和申诉。直至八年之后，该案件被提交给当地最高检察院。该男子入狱十七年后，最高检终于认定该案件系误判，真正的杀人凶手是当地公安机关的一名民警，而这名男子也最终被无罪释放。跟据相关公告显示，该案件翻案的关键证据在于，当年公安机关在现场所提取到的指纹和被害人阴道中的精子均与该名男子的生物信息不相符。

对此，郭文贵先生也在盖特中评论道，在过去的中共国，类似这样的冤案实际上不计其数。而郭先生也强调，只要中共还存在，这样的悲剧就将继续发生。为了所有无辜的同胞，我们必须将灭共事业进行到底。
