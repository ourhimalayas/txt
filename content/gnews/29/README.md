## 中共讳莫如深的林彪“九一三”事件
` NFSC News` [轉載自GNews](https://gnews.org/zh-hans/2067717/)

作者：纽约香草山 – 侠肝义胆
![This image has an empty alt attribute; its file name is Screen-Shot-2021-08-30-at-7.37.59-PM.png](https://assets.gnews.org/wp-content/uploads/2021/08/Screen-Shot-2021-08-30-at-7.37.59-PM.png)
“九一三”事件亦称林彪事件，指的是1971年9月13日凌晨，时任中共副主席的林彪及其妻子叶群、儿子林立果等9人，从山海关机场乘机起飞，最终坠毁于蒙古人民共和国温都尔汗附近，机上人员全部死亡。

![](https://assets.gnews.org/wp-content/uploads/2022/02/11-35.png)

回溯到1966年，毛泽东为了从政治上彻底打倒刘少奇，发动了文化大革命。当时，为了给文革造势助威，毛拉林彪出山，并于同年8月，将林彪安排为中共唯一的副主席。一时间，林彪成为党内二号人物和“毛泽东的亲密战友”。

但打倒刘少奇后，毛泽东和林彪的关系逐渐产生隔阂。1970年8月的中共九届二中全会上，二人的政治关系迅速恶化。会后，毛泽东开展“批陈整风运动”，表面上针对陈伯达等人，实际上对林彪含沙射影。随后，毛又用一系列敲山震虎的办法逼迫林彪做检讨，承认错误，但被林彪拒绝。林彪认为，毛的指责是莫须有的罪名，他即使认罪也没有好下场，也不想写检讨让毛抓把柄。

1971年8月，毛泽东为了彻底打倒林彪，抛开中共中央，私自南巡到各地方“打招呼”，试图给林彪正式定罪。在各地讲话中，毛公开指责“有人”要搞“反革命政变”。

9月13日凌晨，就发生了中共党史所谓的“林彪外逃事件”。中共对这桩党史上最大的政治丑闻讳莫如深，一直没有公开历史真相，各种历史研究者则对事件有不少推测。如今半个世纪已经过去，“九一三事件”仍然扑朔迷离。
