![](https://assets.gnews.org/wp-content/uploads/2021/12/t_35269201a8524c6eb768e34ac62c1067_name_vax_pass_gif2.jpg)本月初在韩国举行的集会上，有示威者声称他们的家人被Covid-19疫苗杀害。© YouTube / Covid-19疫苗受害者和家庭委员会
《RT》12月26日报道，在韩国，这个全球接种疫苗最多的国家之一，越来越多的公众开始对Covid-19疫苗进行抵制，因为韩国的示威者要求政府对他们归咎于疫苗的死亡案例负责。

在圣诞节那天在首尔举行了类似的集会后，抗议者于周日在釜山再次举行集会。愤怒的示威者举起已故家庭成员的大型肖像——就像通常在韩国葬礼上展示的那样——并阐明他们的亲人是如何在接种Covid-19疫苗后不久死亡的。

周六，在首尔政府大楼举行的示威活动中展出了数十幅葬礼画像。抗议者呼吁政府查明不良反应的原因，并承认是疫苗是罪魁祸首。

超过1000名韩国人在接受Covid-19疫苗注射后不久死亡，但政府仅承认其中少数病例与疫苗有因果关系。在一个罕见的被承认的Covid-19疫苗严重不良反应案例中，一名护理人员在接种英国阿斯利康的疫苗后出现瘫痪，并于8月被认定为疫苗事故的受害者，获得政府津贴。

在2月底和3月初推出Covid-19疫苗仅一周后，韩国疾病控制和预防机构表示，有7人在接种阿斯利康疫苗后死亡，24人报告有严重的不良反应。据报道，在一名没有潜在健康问题的青少年接种辉瑞的Covid-19疫苗死亡后，政府于8月开始进行调查。

一个名为Covid-19疫苗受害者和家庭委员会的协会在韩国几个城市举行了集会。据韩联社报道，示威者周日从釜山市政厅游行到釜山国立教育大学。

疫苗安全可能成为韩国总统选举中一个有争议的问题。反对党国民力量党上周举行了一次关于疫苗副作用的公开听证会，邀请据称的受害者及其家人就总统候选人尹锡悦（Yoon Seok-yeol）可能采取的支持措施提出建议。

据报道，该党的竞选主席金钟仁指责文在寅总统政府对疫苗伤害漠不关心。政府已承诺对遭受疫苗副作用的受害者进行赔偿，但政府同时也是决定伤害和死亡是否归因于疫苗的仲裁者。“我认为人民已经到了无法信任政府的地步”金说。

韩国已是世界上各种疫苗接种率最高的国家之一，Covid-19疫苗也不例外。大约83%的韩国人已经接种了Covid-19疫苗，是G20国家中接种率最高的国家。
