本文描述了当台湾面临共产党军队入侵的时候，作为全球各地的台湾人，可供的避免选择。本文参考波黑战争中，真实的经历、经验及大陆的生存经验，以下以最优到最劣的选择排列：

1. 现居住台湾的同胞，立即离开台湾，去美国或者加拿大。不用多考虑，美国加拿大近200年未发生过战争，是非常好的躲避战乱的地方，也是法律非常健全的地方，合法居留基本不会有任何问题。另外持台湾护照去美国办理好电子认证 ESTA 可以居留 90 天，不需要签证。还需要换美元现金，将银行账户资金转移到美国。

2. 或者去澳大利亚，新西兰，英国，德国，法国等欧洲国家，也是非常好的选择，同样需要携带本地货币，将资金转到这些国家。

3. 在大陆的台湾人，很多是做生意的，建议离开大陆，同样需要到北美或者欧洲避难，如果共产党入侵台湾，肯定会将台湾人扣住作为人质，很可能投入到监狱，失去人身自由。

4. 不想离开或者不能离开台湾的，可以去小城市或者到山区的城市避难，由于台湾山普遍不高，山林不够密集，很难找远离人烟的地方，所以只能是了胜于无。

5. 居住在台北等大城市，是最糟糕的选择，共产党军队入侵台湾，首先就要控制大城市，居民在战火下，很难保证人身安全。 如果实在无法离开，也要避开军事基地，空军海军基地，发电站，通信大楼，民用机场等地，这些都是高价值目标，会优先被攻击。
