![](https://assets.gnews.org/wp-content/uploads/2022/03/324_7.jpg)白宫国安顾问沙利文( 图片来源 Getty Images )
[大纪元](https://www.epochtimes.com/gb/22/3/23/n13668212.htm)3月24日报道，拜登政府一直在努力阻止中共帮助俄罗斯躲避西方制裁。周三（3月23日），白宫对此设置了红线，警告中共不要利用制裁创造的商业机会，帮助俄罗斯逃避出口管制或处理其被禁的金融交易。

据路透社报导，白宫国家安全顾问杰克‧沙利文（Jake Sullivan）告诉记者，七国集团国家将很快宣布统一回应，以确保俄罗斯即使得到了中共的帮助，也无法逃避西方实施的制裁。

在与拜登飞往布鲁塞尔途中，沙利文在空军一号上表示，拜登总统将出席北约紧急峰会。并表示，这不是专门针对中共国的，而是适用于每个重要经济体，以及这些经济体中的任何一个以故意和主动的方式试图破坏或削弱我们所实施的制裁决定。
