## Marine Le Pen玛丽勒庞在欧盟反欺诈办公室的一份新报告中涉嫌贪污公共资金

![](https://assets.gnews.org/wp-content/uploads/2021/08/logo-5.jpg)





在“Mediapart”披露的一份OLAF报告中，全国集会总统候选人和其他三名前欧洲议会议员被指控从斯特拉斯堡议会挪用了近62万欧元的公共资金。

![](https://assets.gnews.org/wp-content/uploads/2022/04/图片1-103.png)图片来源于网络

根据Mediapart4月16日星期六披露的信息，欧洲反欺诈办公室（OLAF）3月11日转交给法国司法系统的一份报告对包括Marine Le Pen在内的四名前极右翼欧洲议会议员滥用近62万欧元的公共资金提出了质疑。

根据发布该文件摘录的在线新闻网站，欧洲反诈骗局指控国民议会候选人在担任欧洲议会议员（2004-2017年）时亲自从斯特拉斯堡议会挪用了近137,000欧元的公共资金。RN的高管Bruno Gollnisch和Louis Aliot以及欧洲反诈骗局要求略高于303,000欧元的Jean-Marie Le Pen也受到质疑。

Mediapart引用的报告称，所有人都会通过“可能导致刑事起诉的欺诈行为”来“损害联邦机构的声誉”。就巴黎检察官办公室而言，它表示，该文件正在“分析中”。

详细而言，欧洲反诈骗局指出了几项可疑的费用，例如2014年交付给党总部的23,100欧元的促销物品“似乎是为里昂的FN[国民阵线]大会购买的”，或4,107欧元的Beaujolais瓶子，据Mediapart称，这些瓶子由Bruno Gollnisch在同一届大会上分发。

**已经被起诉**

就她而言，Marine Le Pen向欧洲反诈骗局证明了议会资金用于的事件的欧洲性质，或者回答说，她不知道某些指控的事实。她的律师还告知Mediapart，她“忽视了”报告中的结论，尽管她是“一个特别关心的人”。

自2018年以来，国民议会候选人已经因议会助理的“滥用信任”和“挪用公共资金”而被起诉。该调查于2016年开始，部分基于欧洲反诈骗局的一份报告，涉及该党及其主席（他们对此提出异议）组织的可能“系统”，通过转移为雇用议会助理保留的欧洲议会议员信封，从欧盟基金中为其长期工作人员的工资提供资金。欧洲议会估计损失为680万欧元。
