从上海爆发疫情以来，上海人民苦不堪言，民怨四起，死亡人数不计其数，却没有一个外国人身处水深火热的，中共官员的所作所为很好地证实了这一点，简单搜集了几个例子：

1. 疫情初期，上海永康路外国人正常酒吧喝酒聚餐，而中国人只能封闭隔离，没有自由。
2. 在上海方舱医院环境设施很差的情况下，中国人只能挨饿受冻，交叉感染，里面有老外抱怨了几句，就立马转送至瑞金医院高级病房，享受顶级待遇。
3. 上海一护士因哮喘病发，医院不予收治导致死亡，但第二天，上海医院就为一外国人成功做了阴茎手术，还自豪地发文章表扬。
4. 抖音一上海男子发视频歧视外地人，直言外地人来上海只能扫大街、端盘子，叫嚣让外地人滚出上海。
5. 上海市民一菜难求，花钱都买不到食物的情况下，外国人和高官家属们配送的是免费战斧牛排和帝王蟹等高级食品。
6. 某市民事项受理服务站只给当地人发放物资，外地户口不给，可笑的是物资都是外地人送过来的。
7. 上海几百个婴儿被强制与父母分开隔离，医护人员人手不够，导致小孩子们的大便不能被按时被清理，很多婴儿不会翻身，屁股都烂了。根据视频显示，现场婴幼儿哀嚎一片，有一个2岁儿童因持续发热无人看护、治疗导致死亡，母亲也因此悲伤自杀。但是上海没有一个外国人和小孩分开隔离的新闻。
