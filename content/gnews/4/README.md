14. Cointelegraph报道，加拿大下议院议员Michelle Rempel Garner提出了一项法案，建议政府建立一个框架，鼓励加密货币行业在该国的发展。在周三下议院对C-249号法案的一读中，Garner提议让加拿大财政部长克里斯蒂娅-弗里兰与行业专家协商制定一个监管框架，旨在促进围绕加密货币的创新。该法案名为”鼓励加密资产行业发展法案”，还将要求财政部长就该框架提出报告，并在通过后的三年内提出立法。该法案称，”加密资产对加拿大具有重大的经济和创新潜力。
15. Finextra报道，HoFT基金会、普华永道卢森堡公司在卢森堡基金业协会（ALFI）的积极支持下，出版一份关于卢森堡加密资产行业的全面市场研究。报告显示，2021年对加密资产来说是杰出的一年。比特币达到了历史性的市值门槛，NFT市场飞速发展，比特币首次被一个主权国家采纳为法定货币。在卢森堡的金融服务领域，围绕着加密资产的热情和务实的态度并存：我们61%的受访者已经在评估、开发或提供加密资产产品/服务。
16. Finextra报道，英国的一项新倡议正在寻求推进零售央行数字货币（CBDC）的案例，发布了一份绿皮书并准备了一套试点。英格兰银行和英国财政部一直在积极寻求建立中央银行数字货币的案例，被称为Britcoin，因为现金的使用不断减少，私人数字货币对货币主权构成威胁。然而，这个想法受到了很大的阻力，最近的一次是来自上议院的一个贵族委员会，该委员会上个月得出结论，数字英镑没有令人信服的理由。
17. 2月11日消息，俄罗斯政府似乎正在为比特币和加密采矿业的合法化和监管奠定基础——并且已经在计划让矿工向海外买家出售代币的一种方式，即使在该国开始监管加密货币之后也是如此交易。据报道，在总统普京的直接干预下，财政部与坚决反对加密货币的中央银行之间的长期僵局似乎已经结束。据报道，各方现在赞成一项法案草案，该草案将寻求消除加密领域的匿名性，并迫使加密交易所向新的超级监管机构注册。
18. Blockworks报道，美国商品期货交易委员会（CFTC）主席Rostin Behnam周三在参议院委员会听证会上表示，CFTC要求追加1亿美元联邦预算以支持其对数字资产现金商品市场的监管。Bankman表示，目前监管错综复杂，没有明确的监管机构，意味着消费者面临风险、系统性风险和行业缺乏明确性。
19. TheBlock报道，美国新罕布什尔州州长克里斯•苏努努（Chris Sununu）宣布成立一个新的州委员会，重点研究加密货币及其影响，以及该州可能希望执行的立法类型。根据2月9日的一项行政命令，该委员会将对管辖加密货币和其他数字资产的现行州法律法规的作用和有效性、对此类法律法规进行修改和改进的原因等领域作出调查和决定。该委员会将包括来自州银行部、总检察长办公室、州议会两院的利益相关者，以及“三位在加密货币、数字资产以及向机构或消费者提供数字资产服务方面具有公认经验的公众成员”。在接下来的几个月里，该委员会将编写一份关于其调查结果的报告，然后将公布于众。
