附近的战友自发组织起来去王定刚和家人租住的豪宅进行和平抗议，但是这只缩头乌龟不敢露面。昨日，警察部门通知将对其寓所进行入室搜捕后，王定刚带着其家人连夜潜逃。四年来追随爆料革命的真战友知道，文贵先生的家族基金无偿赠予王定刚及其家人居住，但是作为共产党长期隐藏在爆料革命的间谍，他极度心虚，一直没有签署相关的赠予合同，完美错过了这栋价值近200万美元的别墅。

没有文贵先生对路德社的支持，没有爆料革命战友们的关注和宣传，路德社才有在爆料革命中的影响力。王定刚反而称爆料革命蹭他的流量，此为恬不知耻。
