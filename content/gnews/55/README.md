## 中共国家安全部骇客任务，对全球发动情报超限战

7月19 日美国、英国、加拿大、澳洲、日本、欧盟、纽西兰和北约一同谴责中共对全球的骇客攻击。美国国务卿布林肯表示「中共任意破坏全球网路安全，对经济和国安造成重大威胁，中共国家安全部培养骇客生态体系，执行国家资助的网路犯罪攻击。根据路透社报导，微软今年3 月Microsoft Exchange Server 中的零日漏洞遭骇，和先前对美国政府软体供应商Solarwind 发动的网路攻击，都是隶属于中共国安部的骇客DEV-0322 组织。美国官员强调，白宫将公开中共国家安全部如何利用犯罪合约骇客（criminal contract hackers）在全球范围内攻击，向受害企业勒索数百万美元。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/07/20093021/DEV-0322-338x225.jpg)FBI
目前美国联邦调查局对4位中共国安部的骇客下达通缉令，其中3人为海南省国家安全厅官员，另一人属海南国安的门面公司海南仙盾，在美国司法部5月起诉书，此4 人涉嫌在2011至2018年间入侵美国及海外数十间大学、政府部门、企业的电脑系统，窃取商业机密、关键技术和情报资讯。 4名被告其中3 位海南国安厅官员分别为丁晓阳、程庆民和朱允敏和海南仙盾的骇客吴淑荣，被控密谋犯下电脑诈欺罪及经济间谍罪，最重分别能判处5年与15年有期徒刑。美国司法部声明，骇客入侵的对象包括美国、加拿大、德国、英国、奥地利、柬埔寨、印尼、马来西亚、挪威、沙乌地阿拉伯、南非和瑞士，入侵产业涉及航空、国防、教育、政府、医疗保健、交通、生物制药、病毒研究和海事领域。

**笔者认为中共拥有的超级电脑技术是窃取于美国尖端科技实验室，无论是透过骇客窃取，还是专利欺诈，甚至透过技术合作BGY得来的成果，中共将数以千计的处理器组成来计算复杂的程式，并非只用于攻击全球的骇客任务，除了气象预报、石油勘探、药物开发、DNA测序、模拟分析各种疾病等，更是将骇客任务用于研究尖端生化武器SARS-CoV- 2中共病毒，掌握病毒变种状态，甚至控制病毒和疫苗解药，当然也会利用骇客来发展核导弹、太空军备、飞弹防​​御系统和电子货币加密技术。**
