## 【百年罪恶看中共】消失的中国（三）——土地改革与乡绅阶层
` 温哥华扬帆农场` [轉載自GNews](https://gnews.org/zh-hans/1372722/)

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/07/04205639/2b41acf3066245c8.jpg)瑞士摄影师瓦尔特·博萨特30年代摄于中国
说到“绅士”，我们首先会想到“英国绅士”。“绅士”是gentleman的中文翻译，也就是说，在中国是有一个与gentry相类似的绅士阶层的，所以才会有这两个词语之间的互译。但为什么我们现在说起绅士，只能想到英国，不会想到中国呢？你看题图中，一个乡绅带着孩子回到村里，村民们向乡绅鞠躬、乡绅向村民回礼。无论是村民还是乡绅，都彬彬有礼，和谐自然。这不就是一个中国的绅士吗？可是这样的场景，不要说在当代的中国农村，就算在中国最繁华的大城市里也不容易见到了。绅士，或者说乡绅，在现代中国，是一个完全消失的阶层。

乡绅曾经在中国社会生活中有着举足轻重的地位。中国作为农耕社会，农村人口一直是中国历史上最重要的人口组成部分，农村人口的生活水平和文化素质影响着整个社会的文明程度。农村中相对富有的家庭形成了乡绅这个阶层，在大多数人不能接受教育的时代，乡绅们受过教育，有文化有思想，在乡村中是中流砥柱式的人物，因而受到广大民众的尊敬。可以说，乡绅阶层是中国传统农耕社会的精英阶层。中国文化的传承，在很大程度上，是依赖于乡绅阶层的。

中国传统社会，并不是一个真正意义上的法理社会，它是靠中国传统的价值观来约束人们的行为的，每个村庄都有德高望重的人维护礼法，而不是凭枪杆子去管理。乡绅们都是儒教的坚定守护者，虽然其中不乏迂腐之人，但不得不承认，他们对于维护社会风气和秩序起到过极大的作用。乡绅近似于官而异于官，近似于民又在民之上。中央政府派来的地方官往往不了解当地情况，而民众因为等级差异等原因难于和地方官员沟通，这时亦官亦民的乡绅，便成为官民之间沟通的桥梁。他们一方面协助官员治理地方，另一方面替乡民向官方争取权益。官员和乡民都是乡绅的衣食父母，可以说，乡绅是一个能够平衡各方利益的阶层。他们担负了政府缺失的部分职责，怜贫恤老、救济鳏寡孤独、救灾赈灾、修桥补路、调解纠纷、兴办义学、兴修水利等等，在农村中起着稳定社会的重要作用。

在中国近代社会，开明的乡绅阶层基本都受过中国传统儒家文化的熏陶，他们诗书传家，世代耕读，他们遵守的仁义礼智信廉耻等伦理道德成为当地人的行为准绳。他们了解经济规律，集中了当时的农业资料，代表了当时最先进的生产力。他们有知识有思想，能够接受外来文化，他们上新式大学，或者到外国留学，推进社会改良。最著名的例子就是抗战期间高校内迁，李庄乡绅罗南陔以“同大迁川，李庄欢迎，一切需要，地方供给”的十六字电文支持同济到李庄办学。这一举措大大带动了当地科技、医学、社会及教育的发展。

有学者指出，乡绅阶层的财富来源大体有三种：1、祖上传下来的家业；2、在外地做官、经商后衣锦还乡，兴家置业，购买大量土地；3、勤劳俭朴，聪明好学，善于经营，一辈子省吃俭用，有点积蓄便买田。当然，难免也有靠欺行霸市而成为乡绅的人，但这应该不是主流。

毋庸置疑，乡绅阶层曾经在中国的社会稳定和文化发展中起到了至关重要的作用。49年以后，这个阶层被定义为“地主”，因为他们有田产，相对富有。“地主”这个词，现在听来完全是个贬义词，它是剥削、残酷和压迫的代名词，但它在49年之前是指土地的主人，或者指住在一个地方的人，例如“略尽地主之谊”，并无褒贬。它在近代的含义出现在毛泽东的《怎样分析农村阶级》中：“占有土地，自己不劳动，或只有附带的劳动，而靠剥削农民为生的，叫做地主。地主剥削的方式，主要地是收取地租，此外或兼放债，或兼雇工，或兼营工商业。但对农民剥削地租是地主剥削的主要的方式，管公堂和收学租也是地租剥削的一类。”所以说，毛泽东完全歪曲了“地主”原本的涵义，然后，中共在这个定义的指导下，又编造出了黄世仁、周扒皮、南霸天和刘文彩四大地主欺压百姓的故事，在宣传上树立起了反面典型，让“地主”形象深入人心，完全颠覆了乡绅的概念。

从1927年秋收暴动，搞武装割据开始，毛泽东领导的中国共产党，一直是沿袭“打土豪”的办法，解决军粮军饷问题。他们每“解放”一个地方，便把那里的地主杀掉，夺取他们的财富。1949年后，财政危机相当严重。西边要进藏，南边要“解放”海南岛，东南准备攻打台湾，北边要“抗美援朝”，军粮军饷从何而来？于是，历史上最大一次“打土豪”，掠夺农村地主财物的“土地改革”运动，便应运而生了。1950年6月，中共七届三中全会提出了“八大任务”，其中第一项就是土地改革。毛泽东毫不讳言，“在全国范围完成土地制度的改革，是实现新民主主义建国纲领的一项主要任务，也是争取国家财政经济基本好转的首要条件”。这就是土地改革的第一目的：掠夺地主的财富，解决中共的财政危机。

关于土改，毛泽东有过一个说法，中国农村有3600万地主，其中有400万地主是坏的，要杀100万，关100万，管制200万。有学者统计，土改实际杀了200万，美国学者则统计为450万人。除被杀之外，到1979年中共给所有地富分子“摘帽”时，竟然还有400万人。粗略计算，土改斗、杀人数在600万—900万之间。所以，这并不是一场所谓的“土地改革”运动，而是一场杀人运动，杀人越货，巩固统治。

随着中国乡绅阶层的消失，中国农村的精英阶层和乡村文化完全被催毁。本来农村的生产生活，秩序井然。自从“土改”将谋财害命颠倒为备受赞扬的“正义事业”之后，传统道德观念崩溃了，人心涣散了，代之而起的是自私自利，互害互斗，道德沦丧。今日之中国，尔虞我诈、物欲横流，追本溯源，“土改”难辞其咎。“土改”人为地制造仇恨，摧毁了中华民族的道德传统。

乡绅代表着当时农村的先进生产力。他们农具齐全，资金充足，生产具有一定规模；他们经验丰富，接受新生事物较快；他们的农、副业产量总是达到当地最高水平；他们先进的耕作方法给农民起了示范、带头作用……但不幸，“土改”通过斗争和杀戮，把善于经营的地主、富农打压下去，使农村缺乏经营管理人才，农业生产力遭到严重破坏。

“土改”利用“流氓无产者”（扒手、小偷、地痞、恶棍、无业流民……）打冲锋，斗垮地主、富农后，这批“苦大仇深”的“土改根子”纷纷入党做官，趾高气扬地成为“书记”、“委员”、“主任”、“乡长”、“村长”……使农村基层领导彻底恶质化。

土改对中国的历史进程和社会各方面的影响极其巨大，它不仅改变了中国几千年的土地制度、改变了中国乡村的社会结构和生态平衡，而且改变了人们的价值观甚至基本人性。现在我们面对的不少社会问题，都可以从那场运动中找到原因。比如：它摧毁了中国农村的精英阶层和乡村的文化艺术，让不少痞子型的人登上舞台；它破坏了中国乡村的和谐，使残暴和血腥在中国乡村大行其道；它改变了农村贫富价值观——以穷为荣，以富为恶；它把不劳而获、抢劫瓜分别人财物的土匪行为当作反剥削、反压迫的革命行动；它使农业生产停滞不前，长期处于落后状况，直至今天，中国农村依然贫穷落后，农民依旧苦不堪言。

土地改革，看似消灭了“地主”乡绅这个所谓的“剥削”阶层，实质上是抽掉了农村社会和经济的主心骨，让众多的农村人口一直处于愚昧无知、一盘散沙和无法凝聚的局面，这对于中共来说，很大程度上减少了对政权的威胁。

土地改革，败坏了人心和道德，又为社会的乱象埋下了隐患。这些恶果影响至今。
