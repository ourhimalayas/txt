**1.日本发布2021年防卫白皮书，关注美中对立和台湾局势**

[据NHK7月13日的报道，13日在内阁会议上发布的《2021年防卫白皮书》中首次加入分析美中对立的内容，美中两国在政治、经济、军事间的竞争日益明显，并表示将密切关注中共国对台湾展开日益活跃的军事活动。白皮书分析指出，中共国对台湾展开军事活动的同时，美国拜登政府与特朗普政府一样，明确其向台湾提供军事支持的立场。台湾局势的稳定不仅是日本的安全保障，还关系到整个国际社会的稳定，因此必须高度警惕台湾局势。](https://www3.nhk.or.jp/news/html/20210713/k10013136281000.html?utm_int=news-politics_contents_list-items_012)

**2.****东京晴海奥运村13日低调开园，受疫情影响没有举行任何开园仪式**

[据共同社的报道，7月13日，也就是东京奥运会开幕前10天，供奥运会运动员及相关人员居住的东京晴海奥运村正式开园，并做好了迎接各个国家和地区代表队的准备工作。由于中共病毒疫情再次扩大，东京宣布进入紧急事态。考虑到首都圈1都3县以及北海道、福岛县的奥运会比赛采取零观众入场措施，东京晴海奥运村的开园仪式也没有依据惯例举行，奥运村的开园变得非常低调。奥运村共有21栋宿舍楼，约18000名运动员及相关人员将入住。](https://news.yahoo.co.jp/articles/568e52374d8ffffe77ae958c96d006b055a68ab5?tokyo2020)

**3****.****政府撤回要求酒类销售企业“停止交易”的命令**

[据NNN电视台7月13日的报道，作为应对中共病毒疫情蔓延措施的一部分，政府命令酒类销售企业停止与未按防疫要求停供酒类的餐厅交易。但由于此要求没有相应的补偿机制而遭到一系列批评，以及难以实际掌握餐厅与酒类经销商间的交易行为的原因，政府于13日晚间突然宣布撤回该命令。](https://news.yahoo.co.jp/articles/26a31f455fa57755646c1cd7d9ad244af3ae2e1b)

**4.****国际奥委会主席巴赫将于16日访问核爆地广岛**

[据共同社7月13日的报道，日本广岛县与广岛市13日发布消息称，国际奥委会（IOC）主席巴赫已决定将于16日访问广岛市内的和平纪念公园。巴赫将向核爆慰灵碑献花并参观核爆资料馆。长崎县知事中村法道13日也透露，IOC协调委员会主席科茨将于16日访问长崎市。巴赫原计划5月访问广岛市并参加广岛市圣火传递仪式，但因东京等地中共病毒疫情紧急事态宣言延长而放弃。](https://nordot.app/787619209709289472)

**5.****日本呼吸道合胞病毒感染患者急增，超过去年400倍**

[据产经新闻7月13日的消息，可能导致婴儿肺炎的呼吸道合胞病毒（RS病毒）感染患者数量正以不同寻常的速度增加。由于戴口罩和洗手等应对中共病毒的措施，去年RS病毒流行似乎得到了抑制。但专家警示，今年没有免疫力的孩子可能会被立即感染。RS病毒感染呼吸道，主要引起流鼻涕、发烧和咳嗽。根据国家传染病研究所的数据，6月28日至7月4日该病毒的感染人数为去年同期的413倍。](https://news.yahoo.co.jp/articles/a67285cf910f9737f7434e122cbbd02e51fc0ef9)

**6.****四名访日的外籍奥运会工作人员因吸食可卡因被捕**

[据TBS新闻7月13日的报道，几位作为东京奥运会的工作人员访问日本的外籍电子工程师因在本月月初吸食可卡因被警视厅以涉嫌违反《禁毒法》逮捕，包括美国公民Bouquet Ray（22岁）和英国公民 Rockwood John（46岁）等四人。他们负责维护奥运场馆使用的发电机，于今年2月至5月来到日本。四人于本月3日在港区六本木饮酒后，Bouquet嫌疑人侵入附近的公寓被警察盘问后东窗事发。但四人均否认指控，称“完全不记得了”。警视厅正调查可卡因的来源。](https://news.yahoo.co.jp/articles/519ce696b2b7b8cf09e9bb4a0bb35f0290a8c5df)
