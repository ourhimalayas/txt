撰文：台湾宝岛农场 快慢机

当地时间2021年7月19日中午12时许，一架美军C-130型大力神战术运输机，携”外交邮袋”类免检物资，飞抵台湾桃园机场。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/07/19094304/%E8%9E%A2%E5%B9%95%E6%93%B7%E5%8F%96%E7%95%AB%E9%9D%A2-2021-07-19-214152.jpg)
据悉，该架C-130型美军战术运输机，是从菲律宾马尼拉起飞，直抵台湾桃园机场执行运输投送任务的。与4天前，即7月15日，飞抵台北松山机场卸载物资的那架，隶属于美军特种作战司令部，全球空中快速反应运输任务组的，C-146A型军机，所执行的是类似性质运输投送任务。

这架美军的C-130型战术运输机，在桃园机场顺利降落后，驶进615号接驳停机坪处，随即展开物资卸载及燃油加注等地勤作业程式。这次担负该机地勤作业支援任务的，为驻桃园机场长荣航勤小组全程跟进。

“外交邮袋”免检物资！
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/07/19094315/%E8%9E%A2%E5%B9%95%E6%93%B7%E5%8F%96%E7%95%AB%E9%9D%A2-2021-07-19-214205.jpg)
该机在桃园机场卸载了数个体积不等的木箱封装货物，及多件红色帆布袋子包裹物品。由于上述卸载物资均属于「外交邮袋」范畴，拥有外交豁免权及海关进出港免检权的，特殊机密性质。因此，外界对该机本次所运抵台湾的神秘物资具体情况几乎一无所知。

前后短短4天之内，相继飞抵台湾，并顺利起降卸载所运物资的，两架美军军机，已经用美式风格的实际行动，向中共直接表明，北京当局此前所谓，单方面对台设限等一切恐吓性言论。都不足以撼动美台之间的战略伙伴关系，朝着更加紧密的方向加速前进。

中共战狼外交部口中曾叫嚣的，有一有二，绝不可能再三再四。如今距上次美军军机降落台湾，才不过短短的4天，美军的战术运输机，便第三次踏上了台湾本岛机场的跑道，且涵盖地勤保障程序在内的，几乎完整执行了一套常规起降作业流程。为日后美军军机常态化起降台湾，做好试航铺垫。

中共此前已经仓促组织，东海浙江近海水域的”实际使用武器训练”军演，做为对7月15号，二次降落台北松山机场的，美军C-146A型军机的激烈回应。这回美军再次三度派遣C-130型战术运输机，飞抵桃园机场卸载”外交邮袋”等机密物资。不知中共还能拿出什么招式，来表演其激烈与尴尬的反应呢？
