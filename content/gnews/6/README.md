## 有声|欧盟：所有加密货币交易即将被监控？
` MOSREC` [轉載自GNews](https://gnews.org/zh-hans/2268251/)

配音：香草山商业部——文锤
![This image has an empty alt attribute; its file name is image-40-1-11.png](https://assets.gnews.org/wp-content/uploads/2021/10/image-40-1-11.png)
在本月早些时候考虑限制加密资产开采后，欧洲议会经济和货币事务委员会（ECON）正在考虑一项新提案。如果经过验证，它将可以监控受监管实体进行的所有加密货币交易，无论金额大小。

![](https://assets.gnews.org/wp-content/uploads/2022/03/1-blob-1.jpeg)图片来源于网络

**欧盟的一系列新措施**

欧洲议会经济和货币事务委员会 （ECON）认为，加密货币必须符合规定，并作为反洗钱和反恐融资制度改革的一部分进行监管，该制度已适用于其他形式的货币：

“ 直到今天，加密资产仍然超出了该法规的范围，该法规仅适用于传统资金，定义为’纸币和硬币、代表性货币和电子货币’，而不适用于加密资产的转移。 ”

毫不奇怪，欧盟通过在加密货币和犯罪活动之间建立联系来证明这种新的监管准入是合理的：

“ 全球影响力、交易处理速度以及加密资产交易提供的匿名可能性使它们特别适合希望进行非法转移[…]并在国界之外进行活动的犯罪分子。 ”

根据分析公司 Chainalysis 的数据，我们借此机会回顾一下，2021 年非法交易仅占加密货币总交易量的 0.15%，到 2020 年则为 0.62%。因此，加密资产生态系统不是欧洲议会经济和货币事务委员会 （ECON）提出的“非法资金的老巢”，但这并不妨碍它根据这种解释提出特别限制性的措施。

以下是该提案的要点：

**1. 转账信息的收集，无论金额大小**

在欧盟内部，对超过 1,000 欧元的转账进行监控，并收集有关发送者和接收者的信息。但是，ECON 委员会希望在加密货币领域摆脱这一上限，并收集所有金额的信息：

“ 由于加密资产的特定特征和高风险状况，无论转移的价值如何，通知义务都应适用于[它们] 。 ”

具体而言，这意味着受监管服务进行的任何加密货币交易都必须附有双方的信息，然后将其提供给政府。该提案还设想收集有关以这种方式交换的加密货币历史的信息：

“ 加密资产服务提供商还应获取有关这些加密资产来源和目的地的信息。 ”

**2.监控自托管钱包**

最近几天，加密社区一直担心可能会禁止自托管钱包。实际上，这不是禁令，而是非常广泛的监控。该法案解释说，新的监控规则将适用于分类账和其他类型的钱包。受监管实体应直接向投资组合持有人索取信息：

“ 信息应由加密资产服务提供商直接从其客户处获取，并应保存并提供给主管当局。 ”

因此，它是对进出此类投资组合的转账的监控。具体来说，这意味着分类账和中心化交易平台之间的任何发送都将被记录，并收集识别信息。

这种措施当然会极大地阻碍欧洲加密生态系统的发展，更不用说如此广泛的监控可能带来的道德问题了。对交易所提出的要求确实可以鼓励他们简单地停止使用自托管钱包进行交易：

评论：欧洲政府阻止不了数字加密货币的发展，因此他们必将把监管手段用到极致以保护法币的存在，从而保护政府对货币发行的垄断地位。然而，数字加密货币全球流通的巨大优势必将使法币退出历史舞台。最终，全球流通的数字加密货币一定会成为各国政府不得不认可的唯一选择。
