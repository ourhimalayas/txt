## 中共盗国家族将难逃被清剿的命运
` NFSC News` [轉載自GNews](https://gnews.org/zh-hans/2169785/)

作者：磐石农场 在远方
![This image has an empty alt attribute; its file name is Screen-Shot-2021-08-30-at-7.37.59-PM.png](https://assets.gnews.org/wp-content/uploads/2021/08/Screen-Shot-2021-08-30-at-7.37.59-PM.png)
在3月13日的《文贵大直播》中，郭文贵先生表示，中共国的盗国贼家族，将难逃独裁者必由的悲惨结局。

![](https://assets.gnews.org/wp-content/uploads/2022/03/Slide2-10.jpeg)

1989年，罗马尼亚共产党独裁者齐奥塞斯库夫妇被人民枪决之后，直到九十年代，他们在英国的盗国资产还未被罗马尼亚政府追回。然而，在俄罗斯侵略乌克兰的17小时内，美国和欧洲诸国就祭出了金融核武级对俄制裁的决定。短短几日，俄罗斯的国际贸易已陷入瘫痪，诸多俄财阀纷纷表示对普京政权的不满。郭先生揭露，这一变化深深地撼动了中共独裁者们。由于当今国际金融交易高度发达, 独裁者们所标榜的 “闭关锁国” 等手段抵抗国际制裁的策略已毫无可行性。中共不可能再随意对14亿共产奴隶肆意妄为。

郭先生进一步评述，齐奥塞斯库夫妇从所谓“国家英雄”到被枪毙，只有短短的4天时间，他们的家人被连带的却不多。然而，21世纪初的独裁者萨达姆和卡扎菲的家人，几乎全部被群众歼灭；而科索沃战争战犯米洛舍维奇，也受到了全球追剿和围捕。

郭先生表示，中共国的“齐奥塞斯库事件”一定会突然发生，在极短的时间内就会有无数中共的国贼巨蠹被清洗，他们将会难逃人财两空的必然命运。中共盗国贼在全世界所藏匿的全部盗国资产会被瞬间查封，他们的亲属子女也会得到各国法律的制裁。
