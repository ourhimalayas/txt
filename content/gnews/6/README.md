## 新中国联邦以信任为基础继续救人

作者：纽约磐石农场—在远方
![This image has an empty alt attribute; its file name is Screen-Shot-2021-08-30-at-7.37.59-PM.png](https://assets.gnews.org/wp-content/uploads/2021/08/Screen-Shot-2021-08-30-at-7.37.59-PM.png)
3月2日，郭文贵先生在一则盖特视频中指出，全世界目前已经不再相信习近平和普京，但新中国联邦将以信任为基础，在乌克兰战火中拯救更多的民众。

![](https://assets.gnews.org/wp-content/uploads/2022/03/Slide1-2.jpeg)

郭先生解释道，习近平和普京已经成为了孤家寡人：不仅世界不相信他们，他们自身也不相信任何人。相反的，全球越来越信任新中国联邦，因为我们只会拯救和唤醒中国人，这将给人类世界带来巨大的变化。

郭先生还提到，中共通过洗脑将无数国人变成了无知的可怜人，这让很多身陷乌克兰的华人仍旧不敢接受新中国联邦的救援，这更导致在此次俄乌战争中，中国人成了各个民族的仇恨目标。

郭先生最后强调，无论如何，救援战火之中的同胞们将依然是当前新中国联邦的首要任务。
