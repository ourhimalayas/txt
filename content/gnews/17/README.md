要说中共国政府是世界上最富的政府，应该没人反对吧？美国虽然是世界GDP最大的国家，按理说世界上最富的政府应该是美国政府，但是美国的政府是受管控的，而且美国老是因为开销、政策等原因被迫倒闭近四百多次，政府实际上很穷是的，富的是美国民众。所以我大天朝的政府才是世界上最有钱的，由于国企和黄金储备量的原因，我国政府的国库拥有大量的财富。

按照每个政府的黄金储备量来看，中共政府的黄金储备高达3900039358441美元，远超同期日本、美国，居世界第一。（注：日本为1260680415997美元、沙特为744440558276美元。）

而且中国的土地全部是国有的，所以中共政府是全世界最大的地主，能不有钱吗?还有，中共把人民的居住权、生命健康权和受教育权都拿来带动国家经济，能没有钱吗？这些都是我们的感觉，中共政府到底多有钱呢？我们可以用数字来说话。

以下数字均来自中国政府网，我们很多人的感觉是这几年中国经济在下行，很多企业和个人的收入不仅没增长，反而还下降了，那中共政府的收入相应也应该减少吧，实际上到底是不是这样呢？

2021年1-4月累计，全国一般公共预算收入78008亿元，同比**增长**25.5%。其中，中央一般公共预算收入36657亿元，同比**增长**28.5%；地方一般公共预算本级收入41351亿元，同比**增长**23%。全国税收收入67450亿元，同比**增长**27.1%；非税收入10558亿元，同比**增长**16.6%。

主要税收收入项目情况如下：
1.国内增值税24864亿元，同比**增长**24.7%。
2.国内消费税6214亿元，同比**增长**13.9%。
3.企业所得税16250亿元，同比**增长**25.7%。
4.个人所得税4948亿元，同比**增长**22.8%。
5.进口货物增值税、消费税5862亿元，同比**增长**28%。关税986亿元，同比**增长**25%。
6.出口退税5397亿元，同比**增长**6.8%。
7.城市维护建设税1977亿元，同比**增长**27.8%。
8.车辆购置税1365亿元，同比**增长**41.5%。
9.印花税1600亿元，同比**增长**49.6%。其中，证券交易印花税1025亿元，同比**增长**57.9%。
10.资源税732亿元，同比**增长**22.1%。
11.土地和房地产相关税收中，契税2674亿元，同比**增长**51.4%；土地增值税2700亿元，同比**增长**40.6%；房产税1124亿元，同比**增长**13.4%；耕地占用税349亿元，同比下降6.6%；城镇土地使用税713亿元，同比下降3.8%。
12.环境保护税103亿元，同比**增长**7.1%。
13.车船税、船舶吨税、烟叶税等其他各项税收收入合计388亿元，同比**增长**9.8%。

看看绝大多数的税收不仅没有下降，相反还增长了很多，基本上都是二十、三十，甚至百分之四十左右，少的也是百分之七八左右，所以虽然老百姓的收入下降，很多企业也在苦苦挣扎之中，但是中共政府的税收收入依然还在快速往上涨，中共政府怎么可能没钱？

《日经中文网》特约撰稿人柯隆说：如果谁能说清楚中国人的收入，应该离诺贝尔经济学奖就不远了。在北京一个普通的处级干部，孩子都会在美国留学。据我所知，北京的一个处级干部的月工资一般不到1万人民币，满打满算年薪2万美元（12万人民币），夫妻俩的工资加起来4万美元（24万人民币），根本不足以供一个孩子在美国纽约的大学读书和生活，那他们的钱哪来的呢？与政府连年增长的财政收入有没有关系呢？所以中共政府官员的灰色收入就很有猫腻。

在中国越接近权力的人越有钱，这是大家所熟知的。中国老百姓辛辛苦苦挣的钱，都被各种名目繁多的税收给抢走了，然后又通过某种渠道转到了某些家族和某些接近权力的人的手中。所以才有外国人看到的一幕幕——中国人在海外各种买豪宅，犹如买白菜一般，奢侈品店都是中国人，法拉利、兰博基尼等也都被中国人一枪而光……

要知道中共政府不止有中央政府，还有地方政府，尤其是2000多个县级政府，这些县市政府也那么富有吗？在94年分税制改革后，中央政府拿走了大部分税收，除了极少数沿海省份的政府之外，中国绝大多数的省份、市、县都是很穷的，这几年很多地方政府看着有点钱，但那也是卖地收入。现在由于房地产的发展，很多地方政府也是欠了一屁股债。

根据财政部、地方财政厅（省、自治区）和财政局（直辖市）网站数据：全国共有17个省份地方政府的债务负担高于100%，主要分布在西北和东北地区，其中天津、贵州、内蒙古3个省份政府债务负担超过150%。

从政府债务负担高于150%的区域来看，2020年全国仅有天津、贵州、内蒙古三个区域政府债务负担超过150%，其中天津达到184.7%，位居全国第一位，主要因天津经济产业结构不佳，税收收入始终处于低位，加之房企对天津房地产市场的战略性收缩，拖累当年政府性基金收入下滑约36%，同时区域上年大规模的非税收入不具有可持续性，综合导致区域财力下降约23%，对历史大规模无序投资积攒的政府债务难以形成覆盖。

贵州同样存在相似的问题，政府债务规模超过万亿元，位居全国第8位，但其一般公共预算收入仅排名全国第22位，在中央大力支持下政府债务负担仍较高（2020年转移性收入占区域综合财力约45%）。

内蒙古因其地理区位特性，目前仍处在资源产业转型升级的过程，前期较大规模的投入目前仍未看到有效产出，同时区域土地市场景气度低也是区域财力较弱的主要原因。

从政府债务负担低于100%的区域来看，全国共有14个区域地方政府债务负担低于100%，主要体现在以下两个维度：（1）广东、江苏、浙江、上海等区域经济结构较为优质，经济发展动能较强，税收规模较大，土地市场景气度较高，无需依赖较大规模的中央转移性支持，即可拥有雄厚财力保障政府债务；（2）而北京作为全国首都、政治中心，拥有全方位的资源优势和独特的发展模式，对举债式发展依赖程度低，造就较低的政府债务负担。

由此可见，即使东南沿海省份和我们认为最富裕的广东、江苏、浙江、上海、北京，都是有一定债务负担的，只是债务负担小于100%，而其余的东北、西北和中西部大部分地区债务负担都非常重，而天津更是超过了百分之一百八十多。所以说，中共国政府既是世界上最富有的政府，也是世界上最穷的政府，就像恒大一样。恒大的许家印是那么那么的有钱，好几架私人飞机，玩最顶级的明星，曾经一度是全中国的首富，但是他的钱全都是借来的，后边欠了一屁股的债。现在连海外的美元债，欧洲、日本、德国、美国各国各种债都已经违约还不上了。

中共政府也是一样的，中共政府包含中央政府和地方政府，中央政府拿走了全中国最多的税收，百分之七八十的税收都归中央，而且所有的央企的税收也都归中央政府来管控，所以中央政府很有钱，所以他们就穷兵黩武，大力发展军事，军舰像下饺子一样，从全世界购买各种武器。中共还拿巨量的金钱去海外各种蓝金黄，收买各国的政府政客，让各国政府都跪倒在中共面前。因为中共政府是不受人民监督的专制政府，人民对国家收入没有任何支配权。
