(Photo by Jonathan NACKSTRAND / AFP) (Photo by JONATHAN NACKSTRAND/AFP via Getty Images)](https://media.breitbart.com/media/2022/02/GettyImages-1207393821-e1645042072655-640x480.jpg)瑞典最大的机场阿兰达(Arlanda)机场
来自布赖特巴特新闻网2022年02月17日的报道，瑞典最大的机场阿兰达(Arlanda)机场与一家中共国公司签订了提供安检的合同，据称该公司与中共军方和中国共产党有密切联系。

中共国公司同方威视(Nuctech)技术股份有限公司赢得了瑞典国有公司斯威达维亚(Swedavia)的采购，该公司拥有并经营阿兰达机场，为机场5号航站楼提供安检服务，该机场将于明年5月进行现代化改造。

据《晚报》(Aftonbladet)报道，斯威达维亚公司的新闻官员彼得·瓦林(Peter Wärring)表示，与同方威视的合同已于上周五签署，同方威视预计将采用最先进的X射线扫描技术来检查机场的行李。

斯威达维亚战略采购主管安妮卡·巴拉兹(Annika Balazs)声称，同方威视赢得了合同，因为他们“只是提出了最好的报价”，但有人批评过同方威视，并提到了安全问题，美国也将该公司列入其工业和安全局实体清单。

荷兰国防部前网络安全主任、现为欧洲议会成员的巴特·格罗特休斯(Bart Groothuis)就潜在的安全风险发表评论说：“这些设备所处理的数据是非常敏感的。它是个人数据、军事数据、货物数据。这可能会危及商业机密。”

格罗特休斯补充说：“你依赖于一个外国机构，而它是一个地缘政治敌手和战略对手。”

其他政府也对同方威视持批评态度，包括加拿大，加拿大在2020年确定该公司的扫描设备可用于收集和获取电子设备的数据，并允许可能被禁止的物品通过。

I还有人称，同方威视利用从中国共产党那里获得的补贴削弱竞争对手，参与了不公平的商业行为，补贴使得它的出价比其他公司低30%到50%。

斯威达维亚在回答广播公司瑞典电视台(SVT)关于同方威视合同的问题时说：“同方威视的设备是由欧洲民航会议(ECAC)批准的，ECAC是欧洲负责批准审计设备的机构。关于该协议，我们遵守了适用的采购法规，该采购也符合《瑞典安全保护法》(Swedish Security Protection Act)的规定。斯威达维亚将与供应商签署安全保护协议”。
