## 恒大汽车直线暴跌19% 11月两次配售依然入不敷出
` GTV新闻访谈` [轉載自GNews](https://gnews.org/zh-hans/1722094/)

作者：纽约香草山文艺部 – 宁文静
![This image has an empty alt attribute; its file name is 1-28.png](https://assets.gnews.org/wp-content/uploads/2021/12/1-28.png)
12月2日港股早盘，恒大汽车再一次闪崩，最低触及2.90港元以下，最大跌幅一度达到19% 。当日收盘时，股价回到3.04港元，全天暴跌接近14% 。

![](https://assets.gnews.org/wp-content/uploads/2021/12/9.png)

据12月1日港交所股权披露，恒大继续减持恒大汽车9亿股，融资27亿港元。交易完成后，其持有的恒大汽车股份从目前的62.55%降至57.36% 。同时，恒大汽车在刚刚过去的11月份进行了两次配售，分别募得资金5亿港元，及3.47亿美元，以增强该公司的财务状况并为新能源汽车的研发和生产提供资金。11月26日晚，恒大汽车发布公告，已退还未动工的生活配套用地和工业用地266万平方米，涉及金额达12.84亿元。

自9月份债务危机爆发以来，恒大一直在与潜在买家洽谈，出售集团名下的各种土地、项目和资产，以维持公司基本的财务运营支出，尽可能地拖延爆雷的时间点。但恒大近期出售资产所得，相对于超过3,000亿美元的负债，无异于杯水车薪，恒大破产只是早晚问题。
