到目前为止，还没有一个简单、严格的分析能准确和明确地传达COVID-19封锁的真实成本。因此，《左轮手枪新闻》开始委托进行一项研究，正是为了做到这一点：最终以一种被称为 “生命年 “的指标来量化封锁的净损失。简而言之，我们借鉴了现有的关于失业对健康影响的经济研究，计算出美国因封锁而损失的生命年数，并将其与封锁所挽救的生命年数进行了权衡。结果是非常惊人的，表明封锁最终将使美国人失去的生命年数是他们从病毒中拯救的年数的10倍以上。

美国人在2020年的大部分时间里不得不忍受的COVID-19封锁措施是这个国家历史上采取的最具戏剧性、影响最大、破坏性最大的政策措施之一。在其历史上，美国第一次经历了如此严重和危险的局势，以至于长期的金融和社会稳定都受到了合法的威胁。

与2020年美国的一切一样，这些封锁措施已变得高度政治化。时任总统川普反对进一步的破坏性封锁。他在共和党全国代表大会演讲中指出，封锁对许多美国人的生活产生了非常真实和非常毁灭性的影响。相反，当时的民主党总统候选人乔-拜登表示，如果 “科学家 “告诉他，他愿意实施进一步的封锁。但不清楚乔-拜登会听取哪些科学家的意见，因为科学家和专家们之间对于封锁是否值得给普通人带来惊人的成本并没有达成共识。

2020年3月11日，世界卫生组织正式将COVID-19列为全球大流行病。在接下来的几周里，世界各国开始实施以前无法想象的措施来防止病毒的传播。在美国，一些州迅速封锁了几乎所有的实体企业、场所和公共区域。随着短暂的隔离迅速发展为无限期的封锁，一些立法者和经济学家开始质疑封锁是否会造成比病毒本身更大的损害。通过实证研究，可以对史无前例的封锁在生命年损失方面的影响进行首次估计。然后可以将其与美国因COVID-19损失的估计生命年数进行比较。

为什么是生命年？众所周知，社会经济地位（SES）似乎与预期寿命有关，其中一些关联是因果关系，较高的SES通过一些渠道导致预期寿命延长。大多数政府政策分析家使用统计生命价值（VSLs）进行决策—该价值约是1000万美元。例如，如果一项法规可以拯救一条生命并花费900万美元，那么挽救生命是值得的。如果花费了1100万美元，那么就不值得为了拯救一条生命而实施。这种方法的一个问题是它的粗糙程度。因为SES与预期寿命有关，政府的行动如果不直接导致生命损失，就很可能在这种方法中被忽略。例如，为什么不把VSL设定为1亿美元？或者1美元？如果你把VSL定得太低，你将无法实施许多非常便宜的救生条例。也许更微妙的是，如果你把VSL定得太高，那么你实施的法规将通过延缓经济活动而减少收入，以至于你将通过SES-预期寿命的渠道而最终减少预期寿命。实际上，为所有公民使用单一的VSL有更多的基本问题（见Sunstein的《珍视生命》，有一个很好的概述），但许多人发现它的一般均衡问题非常直观的。

令人惊讶的是，对VSL问题敏感的公共卫生分析家、生物统计学家、经济学家和政策决策者之间的COVID-19对话一直被上述标准的 “粗略 “VSL计算所主导。尽管各种研究都试图估计每例COVID-19死亡的平均生命年损失，但我们没有看到使用生命年方法的完整政策分析。对统计生命的正确价值估计的争议，在上述分析中实际应用的问题，以及也许与以美元计算的人类生命相关的某种可憎性，促使一些健康经济学家和公共卫生分析家使用生命年的方法。这种方法很简单，原则上不涉及将人的生命明确转换为货币形式–尽管这种权衡在任何政策分析中都是隐含的。从生命年的角度来看，政策制定者可以计算他们采取具体行动所损失和获得的生命年数。例如，一项新的航空安全法规可能会使航空公司的用户变得非常安全，从而为每个旅行者平均挽救0.1个生命年，但较高的航空旅行成本可能会诱使潜在的乘客转向安全性较低的汽车旅行，使转向者平均损失0.3个生命年。如果有足够多的人转向汽车旅行，那么从整个交通系统的角度来看，航空安全法规实际上会减少总的生命年数。

寿命年最大化者会说，不应该采用这种规定。生命年方法的一个优点是，它在原则上平等地重视所有人的生命年。一个80岁老人的生命年与一个20岁的人的生命年具有同等价值。穷人的生命年与富人的生命年价值相同。VSL方法错误地估计了异质消费者和劳工愿意为安全支付的金额，可能会使富人（他们对安全有更高的要求）和老人（他们不必面对更多安全法规的 “一般均衡 “成本）比穷人和年轻人更有特权，因为他们可能更喜欢不太安全但更便宜的商品和服务或高薪但不安全的工作，而不是低薪但安全的工作。随着生命年被确定为我们的操作指标，我们将继续分析Covid封锁政策对生命年影响。

COVID-19的封锁导致了大规模的全球经济衰退，几乎没有一个国家、公司或经济部门能够幸免。美国失业的规模几乎可以与20世纪30年代的大萧条相媲美，2020年4月的失业率高达14.7%。使用粗略式计算，我们可以为中期失业率的增长设定下限，可以合理地估计为8.5%左右。值得注意的是，即使是中期失业也会导致永久性的工作离职，因为失业超过一年的雇员不太可能回到他们以前的岗位。

先前关于失业和死亡率的研究发现，失业者面临着死亡率的显著上升，从中可以估算出损失的生命年。失业和永久性离开工作岗位已被证明与心脏病、药物过量、肺癌和肝病的增加相关，以及其他增加死亡风险的因素。Sullivan和Von Watcher关于失业和死亡率的论文估计，失业导致每个人损失约1.5个生命年。

经济合作与发展组织（OECD）估计，2021年美国的失业率将在单波情景下的8.5%和双波情景下的11.5%之间。假设单波情景发生，美国的失业率将比COVID-19之前的2020年水平增加约5个百分点。根据COVID-19之前的薪资，2021年的总就业人数可能会达到820万左右。2020年2月至7月期间各年龄组失业率的增加可以用来估计各年龄组额外失业的分布。把820万个工作岗位转移，乘以Sullivan和Von Watcher对每个年龄组的生命年损失系数，可以估计出由于工作岗位转移而损失的生命年总数超过800万个。

除了直接的工作岗位流失之外，其他研究表明，劳动力市场的衰退对进入劳动力市场的人的未来死亡率有显著但滞后的影响。根据Hannes Schwandt和Till von Wachter关于在经济衰退期间进入就业市场导致的死亡率增加的线性模型，这一分类中，每个个体损失 了0.629个生命年。这个数字可以乘以2020年16-24岁年龄组的1300万工人，然后再乘以2020年的390万大学毕业生。Till von Wachter在其关于Covid-19危机对劳工的长期影响的工作论文中也使用了这种方法。根据这些估计，美国因COVID-19衰退而损失的长期生命年总计约1870万人。

为了更好地了解封锁的影响，重要的是在未实施封锁的情况下估计美国的COVID-19死亡人数。虽然对该病毒仍有许多未知数，但来自其他国家的数据说明了不同程度的封锁的效果。

与其他大多数国家不同，瑞典因没有颁布封锁措施而面临争议。约翰-霍普金斯大学的数据显示，瑞典的COVID-19死亡率为每10万人中有56.62人。在采取了全面封锁措施的美国，COVID-19的死亡率为每10万人中有50人。英国提供了另一种观点，因为它最初采取的方法更接近于瑞典，然而在大流行期间改变了方向，导致每10万人有70人死于COVID-19。通过使用瑞典的人均死亡率或英国的死亡率作为 “经济成本最低的政策 “的反事实估计，可以估算出如果没有颁布封锁措施，美国的COVID-19死亡人数。为简单起见，我们保持死亡时的年龄分布不变。图2和表3按年龄组显示了这些估计值。表4显示了反事实和实际COVID-19死亡人数之间的差异。为简单起见，我们假设美国COVID-19的 “死亡年龄结构 “将继续适用，并应用一个简单的乘数（基于美国和英国/瑞典的人均死亡比例）来构建我们的反事实估计。

第二个更值得商榷的关键假设是，如果没有封锁措施，失业率就不会急剧上升。Krugman和其他经济学家认为，如果没有基于瑞典和丹麦等相邻国家经验的居家令，导致失业的消费削减就会发生，瑞典和丹麦等国都遭受了类似的GDP损失。这种逻辑忽视了这样一个事实，即北欧国家集体GDP的很大一部分是通过贸易联系在一起的，因此，直接的差异化忽视了芬兰、丹麦和挪威对其邻国实施的负面外部性封锁。即使不是封锁导致消费削减，主流媒体也几乎没有采取任何措施来反击灾难叙事，并通过选择性报导，积极误导公民对COVID-19的实际（小）风险。

事实是，一旦COVID-19发生，与未发生COVID-19的世界相比，将会出现经济和生命年损失。然而，这些收入和生命年损失的最终规模，以及这些生命年损失的地理和年龄分配，相当于一种政策选择。一些学者可能会质疑，我们利用对美国数据的时间序列分析构建了失业反事实，而我们利用外国经验构建了“无封锁”COVID-19生命年损失反事实，有效地混合了两种不同的设计。

我们鼓励公平的读者查阅这篇文章的标题。正确的反事实是不可能的。来自瑞典或英国等国家的真实结果优于流行病学模型的结果，该模型的样本外有效性极其有限，并且根本上无法识别参数。这个定量思维实验的重点主要是定性的，旨在向公民和政策制定者提出一个观点：由封锁引起的失业率的短期或队列级小幅增加很容易抵消封锁带来的小规模记录的好处**。**美国失业率的实际增长是巨大的，超过了大萧条的规模。失业率的长期增长不能轻易地从同期的跨国数据中得出，原因很简单，这些国家的长期就业演变尚未发生，但可以合理地假设COVID-19已经在纽约或瑞典结束 – 现在每天约有1-2例COVID-19死亡。如果有人为《柳叶刀》窃取了这些见解，*Revolver.news*将感到很荣幸，柳叶刀周转速度很快（回想一下他们的羟氯喹政策），CDC的内部期刊或NBER关于流行病/COVID-19的工作论文系列。

封锁造成的经济破坏将在病毒得到控制后持续数十年，并可能导致更严重的连锁反应。美国在其历史上第一次经历了几乎可以比作一个新兴国家突然停止的情况——这种情况如此严重和危险，以至于长期的金融和社会稳定受到了合法的威胁。

这一切是怎么发生的？值得反思的是，学术界的制度性激励措施导致了大流行病的泛滥和美国未来几乎确定的财政崩溃。我们的计算表明，从失去生命年的角度来看，美国的COVID-19封锁客观上对每个年龄段造成的伤害远远大于好处。生命年损失如此之大，以至于很难看到任何改进来证明当前美国政策组合的合理性。事实上，人们需要争辩的是，如果没有精英恐慌，一种疾病的毒性只有流感的两到四倍，就会引发萧条，而萧条本身就可以对宏观金融决策界提出指控。

为什么COVID-19的政策权衡从未在生命年数的形式向政治家，媒体或公众展示过？为什么经济学家和整个公共卫生领域在普及社会经济地位和预期寿命之间的联系时，在涉及COVID-19时突然没有考虑这些联系？为什么贸易经济学家在考虑COVID-19政策时忽略了封锁的贸易溢出效应？

颁布这些封锁的美国政客受到恐惧和政治动机的双重驱使。是什么驱使资深专业人士夸大病毒的潜在危害，并尽量减少封锁的成本？公民向学者提供大量的金钱和时间，并承诺知识可以帮助制定更理性的政策。事实上，一些经济学家直言不讳地警告潜在的封锁措施会造成损害，但大多数人要么保持沉默，要么煽动恐慌。美联储和麻省理工学院斯隆分校的一些经济学家匆忙发表研究报告，这些研究积极误导决策者做出生死攸关的决定。

荒谬的是，公共卫生专家鼓励“黑人的命也是命”的抗议和骚乱，并声称它们实际上减少了COVID-19感染。事实上，在学术界的高度政治化的氛围中，很难想象任何“学者”会保留他们的立场，以常识性的公共卫生为由谴责抗议和骚乱。基于这种不对称性（以及在处理种族和死亡率、污染、性少数群体以及经济分析在公共卫生中的作用方面的类似不对称），公共卫生必须被谴责为“对政策分析尚无用处”。
