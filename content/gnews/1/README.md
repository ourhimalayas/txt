普京和他的两个成年女儿（资料图片来自网络）

美国和西方盟国星期三（4月6日）宣布对俄罗斯立刻实施最新的严厉经济制裁。俄罗斯总统普京的两个成年女儿的资产也被列为制裁对象。

综合多家外媒4月7日的报道，美国及7大工业国集团（G7）周三（6日）宣布新一轮制裁俄罗斯的措施，制裁对象包括俄罗斯总统普京的两个成年女儿。美国官员向当地传媒透露，普京的两个女儿之所以成为目标，是因为美方相信普京透过其家人隐瞒资产。

据美媒引述政府官员的话指出，美方相信普京37岁的长女沃龙佐娃（Maria Vorontsova）和35岁的女儿吉洪诺娃（Katerina Tikhonova）有资产在美国，而这些在美资产将会被冻结。

据了解，吉洪诺娃是莫斯科国立大学复杂系统数学研究所的副主任，其工作是支持俄罗斯的国防工业；沃龙佐娃据报是一名儿科内分泌学家。有报道称，沃龙佐娃所领导的由国家资助的遗传学项目是由普京亲自监督。

除了普京的两个成年女儿之外，俄罗斯外长拉夫罗夫的妻子及女儿、俄罗斯联邦安全委员会（Russian Security Council）成员也被列为制裁对象。

美国周三祭出的新制裁措施还包括禁止美国人投资俄罗斯。白宫还表示，将制裁“关键的重要俄罗斯国有企业”。

最新制裁措施将对俄罗斯最大的金融机构、持有国内1/3银行资产的俄罗斯联邦储备银行（Sherbank），以及俄罗斯国内第4大金融机构、俄罗斯最大的私营银行——阿尔法银行（Alfa Bank），实施全面封锁制裁。

在宣布新的制裁措施之前，白宫新闻秘书珍·莎琪表示，美国正在与七国集团和欧盟协调采取措施，将使莫斯科“进一步走在经济、金融和技术孤立的道路上”。

有关对俄新一轮制裁，欧洲理事会主席夏尔·米歇尔表示，新的欧盟制裁方案将包括禁止进口煤炭。他同时还表示，有关石油甚至天然气的制裁措施，他认为迟早也是需要的。
