金龙鱼已连续十余年蝉联中国小包装油销量冠军，在传统销售渠道有将近2000家经销商、100万个以上的销售点。但是在2月22日，金龙鱼（300999.SZ）在深交所公告，2021年全年净利润为41.32亿元，同比下降31.1%。这则公告令人大跌眼镜，同时造成第二日股价重挫将近8%。本文以金龙鱼企业为切入点，探讨下2022年中共国粮油行业的趋势。

金龙鱼企业在2021年度财务报表中指出，“因原材料成本上涨幅度较大，公司生产经营成本高于去年同期。虽然公司上调了部分产品的售价，但并未完全覆盖原材料成本的上涨，利润同比减少。此外，市场竞争加剧，叠加消费疲软的因素，公司的中高端零售产品销量受到较大影响”。

这则消息透露出以下几点信息:1、国际粮食以及大豆价格上涨，造成企业成本上升；2、中共国粮油行业企业内卷加重，造成企业不敢加价，甚至低价亏本保持市场份额；3、国内消费市场疲软，高端油品购买力下降。

目前随着俄乌局势的紧张，推动国际小麦等粮食的上涨，会更一步加剧粮油企业的成本压力。而且金龙鱼母公司丰益国际在俄罗斯和乌克兰有葵花籽油的工厂，这更是不可控因素。其次随着国内经济的萎靡，会更一步压缩企业生存空间，造成盈利能力下降。
