## 艾伯塔省对特鲁多政府实施《紧急状态法》提出法庭挑战
` 英國倫敦喜莊園 Himalaya London Club UK` [轉載自GNews](https://gnews.org/zh-hans/2042975/)

![](https://assets.gnews.org/wp-content/uploads/2022/02/CP150599587-700x4201-1.jpg)《大纪元时报》
《大纪元时报》2022年2月19日报道，艾伯塔省省长杰森·肯尼(Jason Kenney)表示，他将对自由党政府实施《紧急状态法》提出法庭挑战，称这是不必要的、不恰当的，并且侵犯了省级管辖权。

肯尼在推特上发布的一段视频中说，联邦政府援引该法案的决定，赋予当局更多权力驱逐渥太华反对中共病毒疫苗强制和限制措施的抗议者，这可能侵犯公民自由，并为未来树立“危险的先例”。他说“现在我可以宣布，艾伯塔省将对加拿大联邦政府启动紧急法案的行为提出法庭挑战。”

该法案于2月14日开始实施，赋予当局处理抗议活动的额外权力，包括强迫拖车公司移除在渥太华市中心扎营的卡车。

还增加了几项额外的金融措施，目的是减少抗议活动的资金，包括将反洗钱机制扩大到众筹平台和数字货币。金融机构无需法院指令即可冻结参与抗议的个人或公司的账户。

2月19日，公共安全部长马尔科·门迪奇诺(Marco Mendicino)证实，与渥太华正在进行的抗议活动有关的“至少76个”金融账户已被冻结，金额达320万美元。

然而，肯尼指出在没有该法案的情况下，不久前阿尔伯塔省和渥太华成功解决了最近封锁加拿大与美国之间边界的示威活动。
这些封锁示威活动是为了声援渥太华的抗议活动，抗议活动最初开始反对联邦政府的疫苗强制令，要求从美国返回加拿大的卡车司机如果想避免在重新入境后接受14天的隔离，就必须全面接种疫苗。随着大型卡车车队于1月29日抵达渥太华进行抗议，抗议很快演变成一场全国性运动，许多人加入此次活动来反对所有的中共病毒强制措施。

“让我们明确一点，必须实行法治。无论人们支持哪种主张，法律和秩序都必须优先。这个国家的人们有权利来和进行和平的、合法的抗议示威活动，我鼓励那些对疫苗强制令、公共卫生限制措施有强烈感受的人合法、和平地这样做。”肯尼说。

“那么问题是为什么联邦政府使用不必要的权力来没收银行账户和资产，例如，只是基于某个人的立场和他给谁捐了钱，就在没有法院命令的情况下，任意地、未按法律程序地从人们手中夺取银行账户和资产。对他们使用原本是阻止资助恐怖主义之类的事情的权力，这一点都不合乎常理。”

阿尔伯塔省和其他几个省政府已公开反对援引《紧急状态法》，肯尼表示，他们还将考虑申请作为干预者来支持加拿大公民自由协会和加拿大宪法基金会，它们也对渥太华的紧急情况提出法律挑战。

“我们需要采取行动捍卫，是的，法律和秩序，还有公民自由、我们的宪法和加拿大。艾伯塔省将这样做。”肯尼说。

2月18日，肯尼给总理贾斯汀·特鲁多写了一封信，他在信中说，实施该法案可能会对加拿大人民和国家的民主造成“无法弥补的伤害”。

本周早些时候，省长与萨斯喀彻温省省长斯科特·莫伊(Scott Moe)和16名美国州长共同签署了一封信。信中要求特鲁多和美国乔·拜登总统立即恢复以前对跨境卡车司机的疫苗和隔离豁免。

简评：
每个国家的公民都有和平示威的权利，加拿大卡车司机在渥太华的和平示威是在行使他们的公民权利，表达对特鲁多政府强制实施疫苗的政策表示不满。特鲁多政府不但拒绝与卡车司机对话，用和平的方式解决卡车司机的合理要求，还强行实施《紧急状态法》派遣警力暴力逮捕和驱散示威的民众。更为恶劣的是还指挥金融机构，在没有法院指令的情况下冻结示威活动者和捐款者的银行账户。

特鲁多政府的行为自然会让我们回想到2020年的香港和平抗议受到中共和香港政府残酷镇压的惨案。在那次的事件中，郭文贵先生就警告说，如果西方国家的民众和政府对香港发生的惨剧视而不见，那么香港发生的黑警暴力镇压民众和平抗议的行为就会传播到整个西方社会。很不幸的是郭先生的这个警告再次成为了现实，从2021年9月澳大利亚维州警察出动反恐特警和装甲车镇压反抗强制疫苗的和平游行群众，到今年1月比利时首都布鲁塞尔警方动用喷水车驱散示威群众；再到这次发生在加拿大的特鲁多政府实施《紧急状态法》来暴力镇压和平示威的卡车司机。我们看到西方民主国家的各级政府正在向独裁的方向全面发展，并且越来愈清晰地看到了中共以黑治国、以警治国、非法断你的钱、抓你的人这些违法的手段正在被这些独裁者们使用。

艾伯塔省省长杰森·肯尼对特鲁多政府的行径提出了法庭质疑，称其是不必要的、不恰当的，并且侵犯了省级管辖权。肯尼的行为无疑受到了加拿大和其他民主国家的人民的广泛支持。另一方面也说明了不光是普通民众，很多优秀的政治家也对那些在中共病毒和疫苗强制中，独裁本色一览无余的各级政府官僚表现出了强烈的置疑和抵制。

中共发动的这场病毒和疫苗战是全球邪恶势力和正义势力的一场对决。一定会把中共这个邪恶的体制彻底消灭的同时，还会把西方民主国家中的邪恶势力和其在各个国家培养的独裁政客带上审判台。
