## 【中日财经爆】日本麒麟将清空与华润合资公司股份 退出中共国饮料制造业务。

《日经亚洲评论》1月14日援引消息人士透露，日本麒麟控股（Kirin Holdings）计划以约1000亿日圆（8.74亿美元）的价格出售其与中共央企华润集团在中共国成立的软饮料合资企业中的所有股份。

麒麟最早将于下个月将其持有的华润麒麟饮料（China Resources Kirin Beverages）公司40%的股份出售给一家中国基金。此举将成为麒麟海外业务的转折点。麒麟正在对与这家中共国基金的合同进行最后调整，预计这家酿酒商将在其2022年的合并财务报表中记录销售收益。

麒麟公司打算将其重点转移到利润更高的行业，如精酿啤酒，并远离中共国低迷的软饮料业务。

华润麒麟饮料公司成立于2011年，麒麟持股40%，华润持股60%。麒麟在这家生产和销售茶和咖啡饮料的合资企业中投资了约330亿日圆。在中国，麒麟还通过一家全资的子公司生产其旗舰产品麒麟一番榨啤酒（Ichiban Shibori）。

《日经》认为，与华润合资的终结意味着麒麟将退出软饮料的生产，不过这家日本公司将寻求把其品牌授权给麒麟的前风险合作伙伴。

根据华润集团网站的介绍，华润的前身是于1938年在香港成立的“联和行”；1948年联和进出口公司改组更名为华润公司；1952年隶属关系由中共中央办公厅转为中央贸易部（现为商务部）；1999年12月，与外经贸部脱钩，列为中央管理；2003年归属国务院国资委直接监管，被列为国有重点骨干企业。

麒麟控股是一家位于日本的饮品、酒精性饮料以及制药工业公司，拥有麒麟啤酒品牌。其子公司分布于美国、澳大利亚、台湾、韩国、中国大陆、香港以及东南亚地区。
