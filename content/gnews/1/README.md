自乔·罗根加入后，社交平台盖特48小时内猛增50万新用户

自播客名人乔·罗根加入盖特两天后，以自由言论，灭共闻名社交媒体平台盖特（GETTR）在短短两天内新增用户达54.5万，盖特于2021年7月正式向用户开放，在不到6月的时间内总用户已接近400万。
![](https://assets.gnews.org/wp-content/uploads/2022/01/截屏2022-01-04-21.42.56.png)
在周日罗根向其780万推特粉丝宣布加入盖特后，当天超过 171,000 个新用户在盖特创建账号。第二天更是翻了一番，超过 341,000 人加入该平台。根据盖特最近的官方新闻通稿显示，超过 545,000 人争先恐后地加入这个该社交平台，其中不乏一些有影响力的社交媒体大V，包括UFC选手，前NBA球星，MMA选手，嘻哈说唱歌手等等。

“这是社交媒体领域的巨大颠覆，”GETTR 首席执行官杰森米勒在一份新闻稿中表示。 “人们一直在寻找自由表达的新渠道，而 GETTR 就是目的地。

作为爆料革命战友主要使用的社交媒体平台，战友的活跃引起了很多西方盖特用户的关注，盖特CEO杰森米勒被问到盖特上活跃的中国用户是否都是反对共产党的爆料革命者时表示: “我们都反对共产党，实际上，在盖特运行之前，我曾问到郭文贵究竟哪个更重要？是灭共还是上市，文贵回答我说我们唯一的使命就是灭掉共产党，让民主去到那个从未有过民主的地方，这就是盖特真正吸引我的原因，就是那个消灭共产党的承诺。”
![](https://assets.gnews.org/wp-content/uploads/2022/01/截屏2022-01-04-22.13.09.png)
随着盖特支付，直播，私密直播，打赏等功能的逐步开放，盖特即将成为拥有改变世界能力的社交媒体新星。郭先生曾在去年10月直播中表示盖特用户很快将达到千万。
