根据路透社3月16日报道，受俄乌战争影响，两周多来，石油市场一直处于震荡状态。

过去30天内，全球布兰特原油及美国原油期货两大指标合约的日内交投区间都达到了自2020年年中以来的最阔水平。

3月16日，全球指标布兰特原油期货在97.55-103.70美元的六美元区间交投，结算时报98.02美元/桶，下跌1.89美元，跌幅1.9%；美国原油期货结算价下跌1.40美元，报95.04美元/桶，跌幅1.5%。

布兰特原油在六个交易日内上涨28%，然后在六个交易日内下跌24%。3月7日，油价曾触及14年高点，随后回落。

国际能源机构(IEA)周三表示，如果战争继续，更多的石油供应将会中断。

俄罗斯和乌克兰和平谈判取得进展的迹象增加了看跌气氛。乌克兰总统泽连斯基表示，乌克兰和俄罗斯的立场听起来更加现实，但还需要时间。
