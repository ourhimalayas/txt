国务院报告还特别指出尼加拉瓜不公正地监禁和拷打反对者，记录了古巴对和平抗议者的镇压，并提到了西班牙抗议活动中警察的虐待行为。

美国国务卿安东尼-布林肯本周二提交了与2021年相对应的世界人权状况年度报告，该报告按国家和大陆详细审查了《世界人权宣言》和其他国际公约所承认的个人、公民、政治和劳工权利的状况。与2020年一样，俄罗斯和中共政府是去年侵犯基本权利最多的国家，而西班牙则因为在镇压几次抗议活动中滥用警力的案件也被提及。

在周二发布的报告中，华盛顿汇编了一些国家的政府不公正地监禁、拷打甚至杀害政治反对派、活动家、人权维护者或记者的案例。除了俄罗斯和中共国之外，朝鲜、尼加拉瓜和叙利亚也在名单上占据了重要位置。该报告记录了缅甸、白俄罗斯、古巴、香港和苏丹的和平民主示威者受到的虐待。报告强调了令人担忧的跨境镇压案例，如白俄罗斯强迫一架国际商业航班改道以逮捕一名批评明斯克政府的独立记者。

关于中共国，美国国务院记录了中共政权的任意杀戮，以及强迫失踪、酷刑和大规模拘留少数民族的案件。特别的例子是北京对新疆自治省的维吾尔人以及该地区的其他穆斯林少数民族实施的“种族灭绝”。2020年的报告中已经谴责了对维吾尔人的系统性迫害。另一个具体提到的问题是，监狱的条件很恶劣，有时会危及囚犯的生命。与俄罗斯的情况一样，中共国的情况也许是文件中最详细的，虐待行为包括对信仰和信念自由的限制，以及在恐吓下实施的强制绝育或堕胎。

尽管报告没有反映2月24日开始的入侵乌克兰的战争罪行指控，但关于俄罗斯的章节中充斥着各种违法行为，例如2020年8月反对派领导人阿列克谢-纳瓦尔尼被毒死，这种神经毒剂也用于两年前对前间谍谢尔盖-斯克里帕尔的袭击。国务院文件还指出，联邦安全局特工（俄罗斯情报机构，前克格勃的继承人）对数十名反对者进行监视，并参与了其中一些人的谋杀，有时伪装成自杀。该文本将这些事件归纳为题为“由当局指导的剥夺普通人的生命和其他法外或政治动机的杀戮”。

在华盛顿举行的报告发布会上，布林肯说，美国无法证实任何有关在乌克兰使用化学武器的指控，但他说这是一个“真正的担忧”。他补充说：“我们正在与我们的合作伙伴进行直接对话，试图确定真正发生了什么。”

西班牙镇压抗议活动中的虐待行为

与其他发达国家一样，西班牙也受到了一定的批评。报告提到了警察在镇压抗议活动中滥用职权的情况，并接受了各党派和非政府组织的投诉，列举了说唱歌手巴勃罗-哈赛尔因美化恐怖主义而被逮捕和定罪，一次示威活动中，一名妇女因泡沫子弹的冲击而失去了一只眼睛。还回顾了2021年2月在利纳雷斯（哈恩）发生的事件，其中两名警察因殴打一名男子和他的未成年女儿而被捕，这引发了一些民众的抗议。关于所谓的procès政治犯，即2019年被西班牙最高法院以颠覆国家罪定罪的加泰罗尼亚支持独立的领导人。

尽管全球形势暗淡，该文件也显示了进展的迹象和一些希望的曙光。华盛顿称，在伊拉克，去年举行的立法选举，比2018年更加可信和透明。在博茨瓦纳，一家法院通过维护同性关系的非刑事化，推动了LGBTQI+的权利。在土库曼斯坦，所有被囚禁的耶和华见证会依良心拒服兵役者被赦免，这是宗教自由的胜利。
