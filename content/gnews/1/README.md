- 基辅市长克利奇科宣布，基辅将实行35小时宵禁，他警告说，“今天是困难、危险的时刻”
- 俄罗斯对基辅的炮击周二早上继续进行，基辅多个地区住宅建筑被炸。据乌克兰国家紧急服务部门称，基辅的一座住宅楼遭空袭后导致两人死亡
- 基辅市长说，昨夜基辅遭攻击导致四名人丧生
- 美国一位高级国防官员表示，几乎所有的俄罗斯军事攻势仍然停滞不前，俄军在周末几乎没有取得进展
- 俄罗斯和乌克兰代表之间举行的第四轮会谈在没有取得突破的情况下结束，并于周二在土耳其继续进行
- 乌克兰总统泽连斯基在Facebook上发布了一段视频讲话，称他被告知谈判进展“相当不错”；他补充说，乌克兰军队在周一帮助3806人逃离卢甘斯克和基辅地区；一支运送100吨物资的车队仍滞留在别尔江斯克
- 波兰、捷克、斯洛文尼亚总理启程前往基辅，会晤泽连斯基，波兰总理办公室负责人说，他们将向泽连斯基提出具体支援乌克兰计划的细节
- 乌克兰马里乌波尔市当局表示，由约2000辆汽车组成的疏散车队离开被围困的马里乌波尔，大约还有2000辆汽车在等候离开。昨天有大约160辆汽车成功离开。
- 联合国说，自从战争开始以来，共有300万乌克兰人逃离祖国，大约180万前往波兰
- 俄罗斯导弹摧毁东部城市第聂伯罗机场跑道，当地市政府说，夜间受到两次导弹袭击，机场受到“严重破坏”
- 欧盟出台第四套针对俄罗斯的制裁措施，旨在打击俄罗斯支持入侵战争的金融能力；
- 英国扩大对俄罗斯的制裁，宣布新的370个个人或实体为制裁目标，其中包括俄罗斯前总统梅德维耶夫。另外还出台制裁措施，针对俄罗斯产伏特加进口和奢华产品出口
- 日本财务省宣布冻结更多俄罗斯人的资产。自俄罗斯入侵乌克兰后，日本财务省宣布制裁的俄罗斯人已达61人。此前日本已宣布对俄罗斯11名国家杜马成员、俄罗斯议会下院、银行家尤里·科瓦尔丘克（Yuri Kovalchuk）的5名亲属和亿万富翁维克托·维克塞尔伯格（Viktor Vekselberg）进行制裁
- 新西兰移民部长克里斯·法福伊（Kris Faafoi）宣布，新西兰计划接收4000名乌克兰出生的新西兰公民和居民的亲属。这个特殊签证类别将开放一年。成功的申请者将获得两年期的工作签证，他们的孩子将被允许在新西兰上学。
