全文如下：

为被忽视的自然免疫科学摇旗呐喊，并谴责福奇“前后矛盾的”政治学“腔调”，前副总统候选人莎拉·佩林说，她已经战胜了中共病毒，现在不需要接种疫苗。

佩林周日（12月19日）在亚利桑那州凤凰城举行的“美国转折点”（Turning Point USA）会议上对查理·柯克说（Charlie Kirk）：“我绝不会去打疫苗，也不允许他们碰我的孩子。”

总统乔·拜登在击败前总统唐纳德·川普后，正在推动疫苗强制令，以此来实现他应对疫情的竞选承诺。佩林嘲笑福奇在自然免疫方面的变调，称他是 “我所遇到的言行最不一致的人”。

“你记得他说的，如果你感染了中共病毒，你会获得自然免疫力。这是几个月前他说的。” 佩林说，去年3月，她自己战胜了中共病毒对她的侵害，那时疫苗接种刚开始。

“所以当时我们被引导相信我们不需要接种疫苗。然后他们改变了论调。我们这些感染过中共病毒的人，他们告诉我们，即使我们已经感染过——我们有了自然免疫力——现在我们仍然必须接种疫苗。”

上述言论发表之际，共和党人正在谴责拜登的疫苗强制令。不仅是在选择上，在科学上，自然免疫力如果不是更好也是足够好的保护。

“够了。” 佩林呼吁年轻的保守派“挺起腰杆”。

“告诉政府官员：‘不，别管我们了。’”

佩林对不希望美国人谈论自然免疫的“思想警察” 表示担忧。

“你不能害怕谈论它，尤其是对不一致的地方。” 她说，“不一致，这真是个笑话。”

“你也需要问问自己：‘我们是怎么走到这一步的？’”

佩林指出，“根除病毒”的努力止于政治。

“如果他们是认真的，边境或者会被关闭，那些越过边界的人需要经过病毒测试。” 她总结道，“这就证明了这是一个多么大的笑话。”
