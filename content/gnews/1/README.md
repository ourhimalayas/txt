精糧：

我也補充一下，除了酒店以外在現場我們接觸好多媒體，包括有一些大媒體或者一些自由記者，乃至於在這個現場的一些其他團體的誌願者，他們看到我們胸口這個要”消滅共產黨”這個標語，走過來就指著這裏，不是來恐嚇，就覺得，真的「I like it」，「I like it」 太好了、太好了。就是覺得共產黨就是一個不好的一個機構。說到這裏，允許我再補充一點，到底共產黨怎麼把我們華人的形象，怎麼給搞壞。

有一次我們遇到一些，從南韓來的一些教會的成員，我們就問他，他們過了境，到了烏克蘭境，然後又回來，那過程是怎麼樣的？他們初步給我們一些講法，然後再深入一些呢？人家問我們，你們是不是中國共產黨派來的？不相信！就要我們的護照看，你們是持什麼護照？講到這種話來！你說這個，如果我們這一次救援活動，我們缺席，沒有華人在這裏，你想想看， 全世界是怎麼看華人的。全球戰友，(容許)我就抓住這一次機會。這是我親眼所見， 我自己被人家這麼問啊！所以這是一個見證，所以也補充了 。

為什麼人家會喜歡我們這個主張？還要求，不止是明道，我和其他戰友，文耀等等，都是被要求，轉身過來，我要拍你後面的那個字眼等等。都是每天在發生，而且的確，越來越多人主動會走過來問。你們這個NFSC是幹啥的啊？什麼組織啊，等等。就是說這些都是發生在我們每一天 ，好多戰友的中間，有好多人主動來問，我們都給他們解釋。我們新中國聯邦，法治基金的目標是什麼，等等…… 這都是我們全體的戰友這麼多禮拜(星期)以來辛辛苦苦建立起來的一個形象，大家都看在眼裏。剛才有波蘭的軍人也在巡視，也在問我們。到底你們是幹啥的？等等……每天都在發生，我們繼續會傳播我們新中國人的形象，謝謝。
