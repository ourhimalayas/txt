![](https://assets.gnews.org/wp-content/uploads/2021/12/2-49.jpg)图片来源：The Register-Herald
在美国时间周六，前橄榄球明星和电视名人迈克尔·斯特拉汉搭乘杰夫·贝索斯的火箭飞往太空，他与美国第一位宇航员的女儿分享了这次旅行。

“现在（橄榄球比赛中）‘落地’被赋予新的含义！”他在飞船成功落地后在推特上写道。

蓝色起源公司的“新谢泼德”火箭从德克萨斯州西部起飞，承载着两位贵宾和四位付费顾客的太空舱一起进行了10分钟的飞行。自动太空舱飙升到66英里（106公里）的高度，在降落伞打开到沙漠之前提供了几分钟的失重状态（体验），火箭助推器也成功返回着陆。

这比美国第一名宇航员艾伦·谢泼德1961年5月5日从佛罗里达州卡纳维拉尔角起飞的水星（飞船）飞行，短了5分钟和50英里（187公里）。他的长女劳拉·谢泼德·丘奇利带上了他的自由7号太空舱的一小部分，以及他的阿波罗14号登月的纪念品，还带了一些高尔夫球——她父亲在月球表面打了几个球。

作为美国广播公司“早安美国”节目的联合主持人，斯特拉汉在整个星期的节目更新中都兴奋不已，他带着他的超级碗戒指和退役的纽约巨人队92号球衣。贝索斯含在飞船上藏了一个橄榄球，这只球将被送往职业橄榄球名人堂。

从太空舱里一出来，斯特拉汉就说他还想再去。但贝索斯开玩笑说，下次他必须得自己买票。

在他后来发布的一段视频中，斯特拉汉称这种经历是超现实的，令人难以置信的。“哇！这就是我所能说的。哇！”

在范霍恩附近的发射场，贝索斯在发射塔的桥上印着“点燃这支蜡烛”，借用了艾伦·谢泼德在自由7号内因延误而发出的著名抱怨：“你为什么不解决你的小问题，点燃这支蜡烛？”

谢泼德·丘奇利——她自愿参加了蓝色起源的第三次客运航班——借用了她已故父亲的话，大喊“让我们点燃这支蜡烛！”在等待起飞的时候，狂风使她的（太空）航班延误了两天。

她是宇航员奖学金基金会的董事会主席。

“我想到爸爸上次的太空之行，天哪，他一点没有享受到我所感受到的任何东西”，谢泼德·丘利在落地后说：“他那时在工作，他必须自己做，我上去兜风了！”

随着太空（商业）旅游终于起飞，周六的发射是美国私营公司今年最后一次发射。维珍银河在7月启动了这项计划，派出亿万富翁创始人理查德·布兰森，其次是蓝色起源和SpaceX。如此多的人在飞行，以至于美国联邦航空管理局周五宣布，将不再指定谁是商业宇航员或提供机翼徽章（纪念品）。
