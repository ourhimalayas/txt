4月16日，乌克兰总统泽连斯基在接受数家乌克兰媒体采访时，谈及被俄军包围的东南部城市马里乌波尔的局势时表示：“如果我们的士兵全部牺牲，将给任何谈判打上终止符。” 据认为，如果俄军在马里乌波尔采取最终的强硬路线，（他们）将打算与停战谈判决裂。

据乌克兰网络媒体（LigaNet）报道， 泽连斯基强调：“我们不会拿领土和人民做交易”。

俄罗斯从马里乌利市区”清剿士兵”，要求其投降

4月16日，俄国防部宣布，已在乌东南部的马里乌利将乌军从市区清除。 （他们）唯一的生存机会是缴械投降。 据俄媒体报道，乌克兰总统泽连斯基向当地媒体表示：“如果马里乌利的守军全部牺牲，（我们）将给（停战）谈判打上终止符，不会把领土和人民作为交易对象。”

据报道，乌方（士兵）固守于该市的一个钢铁厂大院里，继续抵抗。 据国际文传电讯社报道，俄国防部4月17日要求被围困的乌军于当天上午6点（日本时间中午）至下午1点之间缴械投降。
