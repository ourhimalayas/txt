据N12新闻12月22日独家发布：第四剂疫苗接种活动预计将于本周日在以色列开始，但须经卫生部总干事Nachman Ash教授批准，如果卫生部总干事批准开始行动，那些人将有可能在周日预约接驳种疫苗，该指示是在昨晚流行病治疗小组达成同意为60岁以上的老人接种第四剂疫苗，86%的团队成员投票赞成这一举措，这还需要卫生部总干事阿什教授的批准。

在针对69岁以上人群的废气疫苗获批后，以博阿斯博士为首的疫情救治小组代表就此召开新闻发布会，并解释了决定，列夫博士说：“这个Omicron变异病毒在所有国家及我们国家变得严重，整个事件的特点是非常高的不确定性，我们不知道很多事情，例如变异导致疾病的严重程度，很清楚的是，疾病的严重程度较轻。”

总理贝内特说：“这是一个好消息，将帮助我们克服席卷世界的Omicron浪潮、我呼吁所有符合标准的人一一接种疫苗，为我们的健康和生计负责，疫苗可以挽救生命。”

编者按： 以色列是第一个开打第四针的国家，四个月前以色列是全球首个开打第三针疫苗的国家，此后全球多国陆续开打。在对Omicron变异病毒并不了解的情况下，强行为60岁以上老人接种第四剂疫苗，并且总理大力推动人人去接种疫苗，最智慧的犹太民族却沦为强力推进疫苗，民众受害最严重国家，不禁令人感到悲哀，但庆幸的是现在因为真相的传播，越来越多的犹太人甦醒过来，有民众在社交媒体留言说，“总理贝内特是收了钱要灭绝犹太人的凶手。”愿真相的传播令每个国家甦醒过来的人越来越多！
