![](https://assets.gnews.org/wp-content/uploads/2022/01/34649C67-6B88-4AF0-8BC5-EC7DD4ECA336.jpeg)（图片来自《大纪元》）

1月20日，美国总统拜登澄清立场，称如果俄罗斯只要越过乌克兰边境就算入侵，就会受到严厉的惩罚。

拜登在前一天的新闻发布会上说，俄罗斯“小规模入侵”乌克兰会比全面入侵引起较小的反应。此话立即引起基辅方面的抗议。

乌克兰总统弗拉基米尔‧泽连斯基（Volodymyr Zelensky）在一篇微妙但措辞强硬的推文中向美国发出了“提醒”。提醒表示，入侵战争中，没有大小之分，就如战争中的伤亡和失去亲人的悲痛没有大小区别一样。

随后，拜登在周四两党的一次活动上澄清说：“我已经和普京总统说得非常清楚了。他没有任何误解。如果任何——任何——集结的俄罗斯部队越过乌克兰边境，那就是入侵。”

拜登说：“如果他们真的利用在边境集结的部队做了他们有能力做的事，即进一步入侵乌克兰，那么这对俄罗斯来说将是一场灾难。”
