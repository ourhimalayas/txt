![](https://assets.gnews.org/wp-content/uploads/2022/03/9-37.png)图片来源 STR/AFP via Getty Images
[大纪元3月17日报道，在上海市变相封城后不久，相邻的南京市疫情防控措施近日也迅速升级，整座城市的正常运转似乎已按下了暂停键。](https://www.epochtimes.com/gb/22/3/16/n13651469.htm)

[据《南京发布》微信公号3月16日通告称，南京市疫情防控措施升级，在当前疫情多点散发、多链传播和社区传播的严峻形势下，校园集聚性疫情传播风险加大。](https://www.epochtimes.com/gb/22/3/16/n13651469.htm)

[自3月16日起，南京全市中小学暂停线下教学，各校组织开展线上教学，幼儿园暂停入园。官方要求实行弹性工作制、居家在线办公、远程办公等方式，最大限度减少人员聚集。](https://www.epochtimes.com/gb/22/3/16/n13651469.htm)

[自2022年3月15日起，南京市部分区域提高了中高风险等级。而南京东面的上海市，疫情管控早在3月13日就已升级，处于变相“封城”的状态。](https://www.epochtimes.com/gb/22/3/16/n13651469.htm)
