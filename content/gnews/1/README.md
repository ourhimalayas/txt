## 美国将释放5000万桶石油以缓解能源成本

**【简评：】**郭*文贵先生直播中提到：带来美国政治灾难的三个核心物价：第一个就是油价，永远不接受超过4美金；伴随美国10月份通货膨胀率飙升6.2%，汽油（无铅普通汽油）上涨了 57%，平均每加仑 3.48 美元，眼看就要超过4美元的红线。拜登政府急忙决定将一定数量的石油储备投放市场，并与中国、日本等主要消费国合作推动油价下跌。此举可能会让拜登政府在 2022年中期国会选举之前避开批评，避免被批评在解决油价上涨问题上几乎没有采取任何行动*

据《美联社》记者JOSH BOAK and COLLEEN LONG，2021年11月23日报道：
![](https://assets.gnews.org/wp-content/uploads/2021/11/image-760-e1637880851662.png)2020年4月21日，星期二，底特律的一家炼油厂的储油罐。白宫周二表示，它已下令从战略储备中释放5000万桶石油，以降低能源成本。 
图片来源：AP Photo/Paul Sancya, File
华盛顿（美联社）—**总统乔·拜登周二下令从战略储备中释放5000万桶石油，以帮助降低能源成本，并与包括中国、印度和英国在内的其他主要能源消费国合作。**

此举针对的是全球能源市场，但也为了安抚那些在感恩节和冬季假期旅行之前面对通货膨胀和物价上涨的选民情绪。根据美国汽车协会(American Automobile Association)的数据，目前汽油价格约为每加仑3.40美元，比一年前高出50%以上。

拜登急于围绕通胀问题重塑其经济议程，称他最近通过的1万亿美元基础设施一揽子计划将通过提高货物运输效率和降低运输成本来减轻价格压力。

共和党议员抨击拜登政府‘通胀在10月份触及31年高点’。消费者价格指数较去年同期飙升6.2%，为1990年以来最大的12个月增幅。

参议院共和党领袖米奇·麦康奈尔（Mitch McConnell）上周在白宫发表讲话时猛烈抨击称物价上涨的受害者是美国中产阶级。

这位肯塔基州参议员说:“上个月惊人的6.2%通胀率的三大驱动因素是住房、交通和食品。这些不是奢侈品，而是必需品，而且它们在中低产阶级家庭的预算中占的比例要大得多”。

**战略石油储备是一种紧急储备，用于在发生自然灾害、国家安全问题和其他事件时保持获取石油的途径。这些储备由能源部负责维护，储存在德克萨斯州和路易斯安那州海湾沿岸盐丘形成的洞穴中。储备中大约有6.05亿桶石油。**

能源部长詹妮弗·格兰霍姆在一份声明中说：“随着我们摆脱前所未有的全球经济停摆，石油供应跟不上需求，迫使工薪家庭和企业付出代价。这一行动突显了总统致力于使用现有工具降低工薪家庭成本，并继续我们的经济复苏的承诺”。

拜登政府认为，储备是帮助缓解供应问题的正确工具。美国能源情报署(Energy Information Administration)的数据显示，9月份美国人平均每天消耗2,070万桶石油。这意味着释放的供应量几乎相当于两天半的额外供应量。

与其他一切事物一样，疫情使能源市场在多个方面陷入混乱。随着2020年4月开始的因疫情而实施的封锁，原油需求暴跌，石油期货价格转为负值。能源交易商不想被无法储存的原油困住。但随着经济复苏，价格在10月份飙升至7年来的最高点。

美国的生产尚未恢复。美国能源情报署(Energy Information Administration)的数据显示，美国国内石油平均日产量约为1100万桶，低于疫情爆发前的1280万桶。

**共和党人还把拜登减少钻探和支持可再生能源的努力作为石油产量下降的原因，随着全球化石燃料价格上涨，市场仍存在多种不定因素。**

怀俄明州共和党参议员约翰·巴拉索(John Barrasso)说:“拜登总统的政策是为美国人民提高通货膨胀和能源价格。动用战略石油储备并不能解决问题，我们正在经历价格上涨，因为政府和国会中的民主党人正在对美国能源发动战争。”

白宫在经过数周的外交谈判后做出了这一决定，此次释放将与其他国家同时进行。日本和韩国也参与其中。

美国能源部将通过两种方式从战略石油储备中获取石油。白宫表示，未来几个月将释放3200万桶石油，并将在未来几年恢复储备。另外1800万桶将成为国会先前授权的石油销售的一部分。

白宫新闻秘书普莎琪(Jen Psaki)周一晚间说，白宫也将密切关注石油公司的动向。

普莎琪说:“我们将继续向那些利润创纪录的石油公司施压，在有石油供应或石油价格下降而天然气价格没有下降的情况下，我们会持续监督我们认为的这些公司的价格欺诈行为。不需要经济专家也知道这是个问题”。
