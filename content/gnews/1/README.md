我们家的家庭教育归功于七嫂。我是一个宠孩子的人，如果按照我宠孩子的方式教育，这俩孩子肯定不是今天这样子的。你嫂子在家教方面是个认真严肃的人。她家的家教就很严，她爸妈属于那种非常强悍型的父母，而且有非常非常严格的家规。你七嫂很强势，我这样人都被她整得服服帖帖的。所以她对这俩孩子就管的很严。家庭教育主要归功于你七嫂。

一.我们家的家规第一条是孩子不要讨论大人的事。小屁孩儿怎么能讨论父母的事？你不要介入到父母的私人感情或者介入父母的恩恩怨怨。

我最小的和最好的发小是跟我爹娘非常不合的。但是半点不影响我跟他的好。他就是那张照片里前面的那个小伙子。他爸爸就是我们当地的队长，现在还在世。我的发小的父亲就跟我父亲非常不合。孩子不要介入到父母的纠纷当中去或者感情当中去。

二.我们家里边的孩子绝对不能撒谎。所以他们从来不撒谎。我的孩子在社会上从不撒谎，绝对不欺负软的，绝对善良。

三.们作为山东人在家里非常孝敬。我儿子和女儿的孝敬，是你们不可相信的程度。他们从小按照孔子家教的原则教育

四.他们财务上特别独立。我们家里的孩子从小跟我的事业和生意完全没关系。他们从小在金融的生意上都做得很成功。

所以说，我觉的我的家庭是一个中国文化和现代文化相结合的家庭，一个成功的家庭。

不要管孩子的私人生活，孩子是自由的

我和你七嫂对孩子的私人生活没有权利管，也不能去管。孩子爱谁，不爱谁，那是他们的事儿。我觉得，父母去管他们属于纯粹多余的事情。我从小就这么早的成熟并且进了玉米地。我更没有资格去管孩子。我也不能和跟孩子们讨论这个事情。孩子是自由的。而且我认真学过生理学，社会关系学和婚姻学。这就是为什么我和七嫂私奔结婚。这就是为什么我从来不服世俗的这套东西。是因为我尊重自然规律。战友们不要宠孩子，把你们的想法强加给孩子。

长江后浪推前浪，一浪和一浪都不一样，这才叫浪！

如果你孩子像你的时候，你就彻底失败了。哪怕他们的优点像你，你也失败。孩子不像你，不管孩子比你强还是比你弱，只要不像你，这就是成功的。中国人常说长江后浪推前浪。 我说，长江后浪推前浪，一浪和一浪绝不一样。 这才叫浪。哪一浪都不一样，没有重复的浪。长江后浪推前浪。 一浪和一浪都不一样，这叫浪。 中国人一定要一浪把前浪拍在沙滩上，这是中国人的对立心理嘛。 对立心理永远是对立的，不是你死就是我活。我儿子和女儿身上有的优点，我身上都没有。我每天都能从他们身上学到很多很多的东西，“都是为了你好”不要挂在嘴边

战友们对孩子不要望子成龙。 特别中国父母在孩子面前，永远在孩子面前表现自己都牛。这是因为中国人爹妈都不自信，极度自卑。“都是为了你好”挂在嘴边。我觉得这个世界上没有比这句话更糟糕的了。爸爸妈妈总是正确的，他们总告诉孩子应该怎么做。

给父母的建议： 尊重孩子并爱其所爱

人生有如果的话，我给父母天下建议是， 如果你让你孩子孝敬你，你要先学会去尊重你的孩子。还有一个就是爱你的孩子所爱。不要强调你孩子的爱。我和我女儿的男朋友就非常非常好。我女儿爱他，我就要爱他。你不要带着观点进去，如果这样，那是你的爱。 不要带着观点并乱评价。 所以说，一旦懂了这个道理，你当父母就开心了。否则，当父母就一生痛苦。 小时候替他打工，大了替他操心。 然后，嫁人或娶人后，还跟他操心。这不是混死了嘛！
