习近平上来以后，第一，抓人清党，一两百万家庭进去，反腐革命；第二，整理太子党，死了很多个太子党；第三，整这些军阀，整这些财阀。解放军将军、中将几乎全杀完了，跟斯大林一样。然后马云，七哥说马云被控制，马云除了在西班牙船上转一圈，你们还见过马云吗？你唯一听到昨天单伟建接管了马云的财产。你知道单伟建的他妈他爹不是一般的共产党员，你想啥关系？习能相信单伟建你想想他是什么感觉？所以说所有财阀，许家印、马化腾、董文标、史玉柱、柳传志，七哥2017就爆料，今天所有说的人，你见过几个今天还能活着旅行啊？

王健林那时候全世界买，当时我在洛杉矶都特别有意思，突然过来4辆警车，说是中国的万达集团王健林来了，王健林从车上一下来，“咔咔咔”的就是总统的那种规格啊，花多少钱呢？17000美金。就是当地的安保公司，在美国谁都可以。我当时就说王健林将是最后一次出国，因为他不知道，就在他进来之前，有人告诉我说，内部已经签发，王健林全部资产冻结，没收，然后实施边控。当天晚上我就看他儿子还发信息啊，吹娘叉，玩这明星，玩那明星的。从那天回去，还见王健林出国吗？七哥告诉大家了，王健林已经被边控，他不可能出国了。后来被抓，6个月瘦了几十斤，再出面，几十斤没了。

他以为给桥桥投资了，他就牛了呢，他不了解习近平，习近平这个人是个混不吝的主。他那酒一喝下去，大腿往上一撸，挠着是吧，端着杯。我告诉你，习近平从来不劝酒，他最讨厌就是劝酒，喝一杯，人家一杯喝下去。就很少有这样的人，真心的喝酒，而且绝不偷。放在这儿，我说啥，你说啥，一句啰嗦话都没有。习在生活中的个性就像黑社会老大，老子下了手了，下决心弄死你了，我就要把你挫骨扬灰，把你彻底弄死，这就是习。我觉得够意思了，今天你就是魔，我也挺你。

他到美国跟奥巴马那一次失败的会面，回去的时候就咬著牙，在飞机上就骂了很多人，然后直接到莫斯科，跟普京说，咱俩个性是一样的，我们今天在一起干，咱们就决定了世界的命运。当时习的原话，我们要不被他们给搞死，要么我们就搞死他们。听说普京也吓一哆嗦，咱俩都是流氓出身，但是现在都还穿着西装呢，这话也太直接了吧，是不是有点儿粗鲁？
