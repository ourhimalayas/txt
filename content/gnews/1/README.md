今天，7月1日，是中国共产党（CCP）成立100周年的日子，也是纪念中国共产党长期以来的暴行和其高压统治的持续阴暗面的日子。

中国共产党通过不分青红皂白的暴力和残忍的行为而上台。二战结束后，中国共产党背叛了国民党政府，在全国范围内发动了一场毁灭性运动。其合法性是值得怀疑的。

中国共产党会说或做任何事情来扩大其权力。 通过 “大跃进”、”文化大革命”、”一孩政策 “等更多的政策，导致数以亿计的人，从胎儿到老人，都失去了性命。他们每个人都在用自己的鲜血来书写历史。

几十年来，中国共产党一直在密谋改变和破坏西藏的种族身份，现在轮到了新疆。这些文化抹杀的运动慢慢地剥夺了西藏人、维吾尔人和其他少数民族的文化、语言、信仰以及其生活方式。正在新疆进行的大规模种族灭绝是这些运动中最新和最恶劣的表现。

生活在中国大陆的中国人持续生活在政治暴力的威胁之下，遭受着在体制内的人权侵犯。即使是已散居在海外的中国人，他们之中的大多数人也持续地生活在恐惧之中，遭受着威胁，并始终处于中共的支配之下。

只要中共能保持其控制力，并实现其成为取代美国的世界霸权的最终目标，中共就会毫无悔意地犯下滔天罪行和倾尽滔天劫难。
