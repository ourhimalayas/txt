这篇用西班牙语写的关于中国的原创文章的作者预测了两种情况：悲观的说法是，西方民主国家将无法与亚洲模式进行公平竞争。

克劳迪奥-F-冈萨雷斯教授（ Claudio F. González 马德里，1963年）拥有完美的职业经历来解释过去习近平在位十年在中共国的情况。他是一名工程师和经济学家，从2014年到2020年他一直“生活在未来”，沉浸在中国共产党打算用技术变革来书写世界历史的下一章，即是它的霸权。作为马德里理工大学的亚洲区主任，他的日常工作是与亚洲国家最富盛名的大学机构，以及主要的研究、创新和创业中心合作。回到西班牙后，他在理工大学领导技术创新支持中心（CAIT）工作，并刚刚出版了一本书《中国的伟大梦想》，这是他经验三部曲的第一卷。他相信，“如果西方民主国家要有一个长期的未来，西方就有义务更多地关注世界另一端发生的事情”。

问：对于在中共国正在发生的事情，有两个不同的、有时是相互冲突的版本。一个是住在北京的西方人告诉我的，他们一般与政府机构合作，这个版本是批判性的和悲观的版本，消极居多。另一个版本通常是从上海这个经济力量的中心讲述的，你要有在上海生活的经验。

答：这是真的，但在中国的27个省中，我只有四个省没去过。由于大学散布在全国各地，我不得不经常出差。但上海确实是一个更适合居住的地方。还有就是从地理方面来讲，当你离北京越远，事情就会越加变得更容易。

问：你是在2014年年初开始任职的，当时习近平已经执政一年了。这是一项有许多“光”和“影”的变革。

答：对我来说，是一个大转折，特别是在习近平的第二个任期。比如说，当我刚来到这里时，Gmail还没有完全关闭，你还可以使用WhatsApp，而现在这些东西是无法想象的。审查管控一直在增加。

问：在你的这本书的第一部分，你提出了一个中心论点，即中共思想，即通过数码和技术让社会主义社会看上去更好。萨尔瓦多-阿连德（Salvador Allende）的东方版本的机器。

答：这是个很好的问题：是否可能有一个有效的计划经济？这是当时苏联的一个大问题……但它失败了。 什么是更好呢？一个无计划的系统，最好的决定按自己的方式做，还是一个有计划的系统，由掌握所有的人指明方向，哪个更好？传统的答案是，自上而下做出决定的系统起不了作用，因为它缺乏真实的数据，也因为它无法处理所有的信息。一个系统的运作就像一群鸟，和谐地飞翔，因为它们都互相看着对方。这就是自由市场理论，特别是奥地利经济学家的理论。这是一个似乎是一个有明确答案的问题：得出的结论是应该允许一些自由和神秘，总是比所有这些集中决定的系统更好。
