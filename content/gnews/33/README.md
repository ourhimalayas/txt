原定于4月下旬在日本东京举行的日美澳印四国首脑会议，有可能推迟至5月下旬举行。

据日美外交相关人士称，政府已在协调4月下旬在日本举行的面对面四国首脑会议，由于参与国澳大利亚正在准备大选，协调工作遇到了困难，因此，四国会议推迟举行的可能性很大。

岸田首相在3月时曾呼吁拜登总统在4月下旬访问日本，举行四方会谈，拜登总统对日本的首次访问也将被重新调整。
