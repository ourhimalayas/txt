## “滴滴出行”将从纽交所退市

中共国网约车巨头 “滴滴出行”12月3日宣布该公司将从纽交所退市，而这距离其赴美上市尚不足半年时间。

据《德国之声》中文网12月3日报道，中共国网约车服务巨头“滴滴出行”周五（12月3日）在微博上发布消息称：“经认真研究，公司即日起启动在纽交所退市的工作，并启动在香港上市的准备工作。 ”

另据《路透社》报道，滴滴出行随后在一份英文声明中表示，其董事会已经批准了这个决定。

《纽约时报》的报道指出，“滴滴出行”下市的决定可能会激怒投资者，这些投资者曾在该公司上市时帮助推高股价。目前中美两国的官员都对中（共）国公司与华尔街之间的关系愈发感到怀疑，而中国（中共）则正在收紧对私营科技行业的控制。

“滴滴出行”今年6月30日才在纽交所上市，每股定价14美元。它是继2014年阿里巴巴赴美上市之后，在美IPO（首次公开募股）规模最大的中共国公司。

然而，该公司刚在美国上市便遭到中共国家网信办的调查，并且在上市后仅4天，其旗下的25款app在中共国遭全面下架。

滴滴出行12月2日的股价收在7.8美元，较6月底的上市发行价已大跌超过4成。

《路透社》上周援引消息人士的话报道称，滴滴退市是因中共监管机构向其施压，要求其制定从纽约证券交易所下市的计划。该消息人士称，之前中共监管机构曾敦促滴滴暂停上市，同时对其数据做法进行网络安全审查。但该公司还是继续推进其在纽约上市，这引发了中共政府的愤怒。

“滴滴出行”过去曾因在中共国击败美国的“优步”而广受称赞，如今却成为了中共政府整顿网络企业的重点打击对象。
