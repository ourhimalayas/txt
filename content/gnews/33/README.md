因币安赵长鹏将客户信息悉数出卖给中共，11月24日，郭文贵先生在大直播中呼吁，所有持有币安账户的爆料革命战友，尽快联合起来，向币安发起集体诉讼。利用美欧的法律，将赵长鹏绳之以法。
![](https://assets.gnews.org/wp-content/uploads/2021/11/11264-1.png)
文贵先生分析道，中共一旦获取到战友们的KYC信息，将采取两种途径伤害战友。第一，以偷税漏税和非法漏报税的名义，将战友遣返回国，战友们的人身，财产都将毫无安全可言。第二，将战友的KYC信息发布在暗网上贩卖，战友们因个人及账户信息被泄漏，将面临无穷无尽的骚扰和未知风险。

据文贵先生透露，因币安泄露客户信息，一位在美国生活了多年的战友已经被抓，甚至还将面临被遣返回国的命运。

目前，币安已因无执照、无总部、涉嫌非法操作等问题，被多国暂停交易。
