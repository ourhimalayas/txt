◎我谈谈药物的开发流程。药物现在分为自然提取药、半合成药物、全合成药物。动植物提取药就是发现这个自然界存在的物质对某种疾病或某种微生物存在治疗或抑制活性，那么我们对其有效成分进行分离，就得到药物，这方面最有名的是奎宁、青蒿素等。二、半合成药物。因为纯化学合成产量很低或合成过于复杂等因素，在原有药物的基础上进行再合成，达到增强活性、降低毒性的效果。如青蒿素的衍生物双氢、琥酯、羟氯喹这些。三、纯化学合成药物。自然界不存在，我们根据既往药物的开放经验和基团活性，再实验室里完全合成，通过临床实验验证的药物，如现在大量使用抗生素、阿司匹林等。无论那种药物，一般都要做体外细胞实验、体内细胞实验。来研究其EC50 IC50 CC50等指标，计算SI，SI越大说明药物的毒性和有效比值越高，越有选用价值。然后进入一期动物实验。如小鼠、大鼠、家兔等。在动物实验对其有效性和毒性进行验证，推测在人体的可用剂量。二期临床实验。这个最开始是验证药物的安全性。一般选择药物安全剂量。如果药物之前治疗别的疾病使用过，从既往资料选择安全剂量，如果没有就根据动物实验结果对人体安全剂量进行推测。这一步安全性验证通过，再分高中低三个剂量组和对照药物研究非劣效实验。进一步验证安全性的同时和验证可能的治疗最优剂量。三、三期临床实验。人群进一步扩大。同时验证大人群药物的安全性有效性，并得出有害时间、严重有害事件的统计学数据。三期安全性和有效性全部通过就可以提交FDA批准。四、FDA批准药物上市后进一步收集全人群用药的疗效及副作用异常反应信息，如果发现任何疗效不佳和异常反应严重超出三期数据的情况，药物要进行召回。总之，药物开发和临床上市是一个复杂过程。我个人做的最多是临床实验，所以我对这些比较熟悉。我的推荐剂量就是根据既往疟疾治疗的安全剂量及目前体外实验的结果推出。
