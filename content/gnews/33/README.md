俄罗斯总统弗拉基米尔·普京2016年与亚历山大·德沃尔尼科夫合影。

俄罗斯入侵乌克兰七周以来，莫斯科在一次重大改组中任命了新的最高指挥官亚历山大·德沃尔尼科夫将军。据西方官员和分析人士称，在平民死亡人数不断增加、破坏范围广泛和进展缓慢，以及俄罗斯军队陷入了后勤问题和军事失误的情况下，他负责监督这场战役。

在他被任命之前，所有俄罗斯部队都没有一个单一的军事领导人。这种缺乏凝聚力的情况在德沃尔尼科夫的领导下可能会改变，一位美国高级官员周六向《华盛顿邮报》证实了对他的任命。

这位将军一直在指挥俄罗斯的南部军区，这是他在担任俄罗斯在叙利亚的空战行动的第一任领导人后获得的一个关键职位。俄罗斯被指控在这两场冲突中犯有战争罪。

德沃尔尼科夫在乌克兰东部的顿巴斯地区有丰富的经验，莫斯科表示，在未能夺取基辅和首都周边郊区后，将把战争的重点放在那里。关于这位俄罗斯军事领导人，还有很多事情不为人知——但他在乌克兰东部的经验和在叙利亚的时间是他背景的关键因素。

以下是对俄罗斯在乌克兰的新任战争负责人德沃尔尼科夫的了解：

- 亚历山大·德沃尔尼科夫是谁？
- 他为什么被任命？
- 他在叙利亚做了什么？


谁是亚历山大**·**德沃尔尼科夫？

德沃尔尼科夫一直在监督乌克兰南部和东部的部队。伦敦大学学院研究俄罗斯武装力量的名誉教授马克·加莱奥蒂说，他有可能取代俄罗斯参谋长瓦列里·格拉西莫夫。加列奥蒂说，他被称为“他那一代人中真正的重磅人物之一”。

德沃尔尼科夫是俄罗斯监督其在叙利亚的残酷战役的第一任指挥官，在那里，俄罗斯军队配合巴沙尔·阿萨德总统自己的空战和围攻，对叙利亚平民、街区和医院进行了大规模的狂轰滥炸。德沃尔尼科夫因其在那里的暴行于2016年被授予“俄罗斯联邦英雄”称号。

五角大楼新闻秘书约翰·柯比（John Kirby）周一表示，他有“完全无视”避免伤害平民和战争法历史。他在叙利亚的战术非常残酷，足以为他赢得“叙利亚屠夫”的称号。

为什么他被任命？

在俄罗斯从基辅郊区撤军后，西方情报和国防机构表示，莫斯科正在重新集中精力夺取乌克兰的南部和东部——德沃尔尼科夫已经在那里指挥。

![](https://assets.gnews.org/wp-content/uploads/2022/04/image-1577-edited.png)图片来源：https://www.washingtonpost.com/

乌克兰第聂伯罗的一家鞋厂，3月份被一枚导弹击中。

据《邮报》报道，此举因为俄罗斯夺取基辅和乌克兰其他地区的最初努力因乌克兰军队比预期更强的抵抗而失败，加上其他因素，包括后勤失败和俄罗斯士兵士气低落。

根据战争研究所4月9日的评估，关键障碍之一是“缺乏单一的整体指挥官”，这“明显阻碍了俄罗斯部队间的合作”。

“鉴于已宣布的俄罗斯主要努力目标几乎都在他的责任区，现在指定德沃尔尼科夫为总指挥官是合理的，”战争研究所总结道。根据该研究所的评估，在这一变化之前，德沃尔尼科夫是其他两到三名负责乌克兰各条战线的指挥官之一。

加莱奥蒂说，德沃尔尼科夫“是一个有趣但并不意外的选择”，因为他在东部的经验以及他作为一个创新的指挥官和“能够应对新的和不熟悉的情况的人”的声誉。

“任何将军的任命都无法抹去俄罗斯在乌克兰已经面临战略失败的事实，”白宫国家安全顾问杰克·沙利文周日在CNN的“国情咨文”节目中说。“这位将军的履历包括在叙利亚的其他战场上对平民的残暴行为，我们可以预见在这个战场上有更多同样的事情发生。”

他在叙利亚做了什么？

德沃尔尼科夫在叙利亚战争的关键时期监督俄罗斯军队——2015年，长期盟友阿萨德看起来可能无法战胜反政府叛乱组织，这些叛乱组织是在叙利亚2011年的起义陷入内战和区域代理战役后形成的。

战前俄罗斯在叙利亚有一个海军基地，从一开始就在外交上支持阿萨德。从2015年开始，莫斯科帮助领导阿萨德对叙利亚城市和平民的空战，这使局势转向对阿萨德有利。此后，这位叙利亚领导人重新获得了对该国几乎所有地区的控制权，其中大部分地区仍然被阿萨德联盟的焦土战术所深深伤害。

自2011年以来，叙利亚有多少人死亡仍然不得而知；估计有30多万到60万。但总部设在伦敦的Airwars监测叙利亚、伊拉克和利比亚的空中轰炸活动，发现根据当地消息来源，在2015年9月至2022年3月期间，据说有近2.5万名叙利亚平民被俄罗斯空袭杀害。

根据该组织最近的一份报告，“迄今为止，俄罗斯还没有公开承认对战役中任何一个平民的死亡负责。”

在乌克兰看到的许多俄罗斯战术——使用集束炸弹，对平民区进行无情的轰炸，以医院为目标，炮击一个地区，然后在紧急服务响应后再次返回进行打击——是莫斯科在叙利亚的战术手册的一部分。

![](https://assets.gnews.org/wp-content/uploads/2022/04/image-1578-edited.png)图片来源：https://www.washingtonpost.com/

62岁的纳塔利娅·蒂托娃在乌克兰北部城市切尔尼戈夫生活了30多年，她的家在俄罗斯军队数周的炮击后被摧毁

德沃尔尼科夫在多大程度上是这些方法的设计者尚不清楚。但这位俄罗斯将军之前也曾与瓦格纳集团合作，这是一个由俄罗斯私人安全承包商组成的阴暗网络，自2015年以来一直活跃在叙利亚，也在乌克兰开展活动。叙利亚人权网络(Syrian Network for Human Rights)执行主任法德尔·阿卜杜勒·加尼说，德沃尔尼科夫以这种身份可能在俄罗斯的空袭和瓦格纳在叙利亚的地面战斗人员之间进行协调。

加莱奥蒂说，在夺回叙利亚城市巴尔米拉的战斗中，德沃尔尼科夫展示了他对“任务的首要性”的信念，如果你需要用强硬手段来完成任务，他会去做。
