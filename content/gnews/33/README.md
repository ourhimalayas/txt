## 中共公布了“接管”拉丁美洲的计划
` 洛杉矶盘古农场 HIMALAYA LA PANGU` [轉載自GNews](https://gnews.org/zh-hans/1837702/)

据《washingtonexaminer》 作者：JOEL GEHRKE，2022年1月02日报道：
![](https://assets.gnews.org/wp-content/uploads/2022/01/image-543-e1641672790811.png)图片源自BBCnews
中共官员在与美国最近的邻国举行新的峰会后，公布了与拉美国家的“合作行动计划”，这是一个培养影响力和威胁美国利益的“全面”计划。

“中共没有说‘我们想接管拉丁美洲’，但他们明确提出了一个多维的合作战略，如果成功，这将大大扩大他们的影响力，并令美国产生巨大的情报关注。”前国务院政策规划人员、美国陆军战争学院研究教授埃文-埃利斯告诉《华盛顿观察家》。

**中共官员在与拉美和加勒比国家共同体的一次峰会后概述了他们的野心。这个各政府间的论坛（“中拉论坛”）于2011年在已故委内瑞拉总统乌戈-查韦斯的主持下成立，查韦斯希望有一个与美洲国家组织相抗衡的场所，并挑战美国在拉丁美洲的影响力，现在它为中国总书记习近平提供了一个平台，以聚集一个符合北京利益的左派和威权领导人联盟。**

**“中国共产党和政府正在积极寻求加强其在整个西半球的关系，特别是与反美分子的关系。”佛罗里达州参议员、参议院情报委员会共和党人Marco Rubio在给《华盛顿观察家》的一份声明中说。“北京正在寻求在各个领域超越美国，我们必须认真对待这一威胁。”**

数十名国会议员为与中国共产党有联系的智囊团有联系的非营利组织提供建议。

本月早些时候，中国对拉丁美洲意识形态断层的利用更加突出，尼加拉瓜总统丹尼尔-奥尔特加在美洲国家组织大会斥责他监督的选举“不自由、不公正、不透明，没有民主合法性”仅数周后，选择关闭台湾大使馆，转而与北京建立新关系。

“奥尔特加-穆里略政权已经宣布与台湾断绝外交关系并结束官方接触，但11月7日的虚假选举并没有为其提供任何将尼加拉瓜从美国民主国家大家庭中剔除的授权，”国务院发言人内德-普莱斯在12月9日的回应中说。“如果没有自由和公平选举带来的授权，奥尔特加的行动不能反映尼加拉瓜人民的意愿，他们继续为民主和行使人权和基本自由的能力而奋斗。”

卢比奥也是参议院外交关系小组委员会中负责西半球事务的共和党一方的领导人，他对这一回应和乔-拜登总统对拉丁美洲的更广泛的做法感到不满意。

“奥尔特加-穆里略政权认为中共是比美国更好的意识形态盟友，因为奥尔特加的一长串罪行和侵犯人权行为对北京的种族灭绝政权来说并不关心。”他在周二发表的一篇专栏文章中写道。“奥尔特加还希望中共继续提供资金，在尼加拉瓜修建一条从大西洋到太平洋的运河，然后人民解放军的军舰可以利用这条运河进入加勒比海。”

据中共国媒体报道，尼加拉瓜外交部长丹尼斯-蒙卡达表示有兴趣加入中共的“一带一路”倡议，即美国官员认为是购买帝国的掠夺性贷款计划的海外基础设施投资。该地区的一些经济项目已经引起了美国国家安全官员的注意，特别是当时的国务卿迈克-蓬佩奥的全球游说运动，警告盟国，中共的电信基础设施代表着间谍威胁。

**就在尼加拉瓜与台北断绝关系的前几天，中共官员对他们在该地区的野心做了更详细的预告。12月3日举行的中国-拉加共同体峰会最终通过了一项计划，不仅要加强经济联系，而且要加强“政治和安全合作”，同时深化中共国在高科技领域的参与–从网络空间和人工智能到“空间科学、卫星数据共享、卫星应用、地面基础设施建设”，甚至是核能。**
