《福克斯新闻》2022年4月4日报道：

随着乌克兰危机继续升级，分析人士将其与台湾危机相提并论，并警告中共入侵这个主权岛屿的可能性令人恐惧。

“与此同时，另一位独裁者，中共领导人习近平，正在密切关注这一切，而不是悄悄地策划着他对台湾可能的接管时机和方法。”

“像普京一样，习近平认为台湾已经是中共国的一部分——如果习近平成功占领台湾，对全球的影响将是毁灭性的，”福克斯新闻主持人警告说。

台湾岛的面积与马里兰州差不多，位于中国大陆东海岸附近。据中共国问题专家兼作家章家敦（Gordon Chang）称，台湾以大规模生产微芯片而闻名，已成为“强国”。

“台湾半导体制造公司生产了全球约90%的定制芯片，”章家敦在特别节目中说。“台湾是全球供应链的核心，因为我们购买的所有不可食用的东西都含有芯片。”这些微芯片是手机、电脑、电视、汽车等等的大脑。

“没有芯片，你的苹果手机就无法工作。我们国家不生产这种产品，”前中央情报局官员丹.霍夫曼（Dan Hoffman）强调说。“因此，保持台湾独立对我们来说是有至关重要的战略利益。”他补充说：“我们必须向中共明确表示，他们在经济和军事上入侵台湾将付出高昂的代价。”

“这是习近平遗产的一部分，对中共国的外交政策至关重要，”霍夫曼说。“这其中有情感的一面，那就是他们想要统一中国。还有一个战略目标：他们想要控制南中国海，而控制台湾将使他们能够做到这一点……并在此过程中在太平洋地区威胁我们。台湾也是一个充满活力的民主国家；这对中共构成威胁。他们不希望再台湾这样一个经济充满活力的地区，出现一个正常运转的民主国家。”

中共国家主席在对待台湾问题上已经表明了他的意图：他不承诺放弃使用武力，将台湾独立视为一种威胁。对此，章家敦将习近平与普京相提并论，提醒《福克斯国家网》（Fox Nation）观众，就在一个多月前，每个人都认为俄罗斯总统是在虚张声势。四十天后，乌克兰陷入动荡之中，而且看不到结束的迹象。“这不是我们可以认为是理所当然的事情，”章家敦警告说。
