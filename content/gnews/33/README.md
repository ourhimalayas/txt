## 【中日财经爆】日本房地产价格要下跌？

日本人气经济评论员渡边哲也在预测2022—2023年日本经济形势中分析指出，基于以下两个因素，房地产价格将会下跌。

**来日本的中国人数量骤减**

2020年东京奥运会原定举办时，日本的目标是每年吸引4000万游客到访日本，但由于中共病毒的影响，该计划全部落空。

2021年10月1日，宣布紧急状态解除，旅游业的复苏似乎即将开始，但不幸的是，可以说像疫情之前的入境旅游盛况一去不复返。因为占据入境流量核心的中国，其入境人数大不如前。

中共国不想从海外输入文化。时至今日，中共国已接近文化孤立的境地。习近平害怕的是“自由的味道”。不用说，一旦尝到了自由的滋味，就像香港的情况一样，自然会有越来越多的人反抗中共政府。因此，中共政府不再希望国民自由出境。

从2021年秋季开始，中共国严格控制护照签发，中国人已不能自由出国。只要这种情况持续下去，中国人在海外拥有房地产就毫无意义。自然地，中国人会出售他们迄今为止购买的房地产。

**市中心的办公室空置率增高**

根据三鬼商事的调查，2021年9月东京中央5区（千代田区、中央区、港区、新宿区、涩谷区）的写字楼空置率平均超过6%。这是自2015年6月，五年零八个月以来首次超过5%。其中，大型写字楼集中的港区以及IT行业密集的涩谷区空置率较高。

主要因为疫情影响，远程工作变得越来越普遍，导致对办公楼的需求急剧下降。并且此形势将可能一直延续。

笔者认为这两个因素只是显而易见的表面影响，疫苗灾难对供应链的摧毁性打击已经影响到了各行各业，后续的失业潮，断供潮，将会成为房价下跌的有力推手。
