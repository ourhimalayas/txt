**前情简介**

Angela七月底家中就有症状，首先是Angela的女儿Jennifer，接着八月初、八月中旬全家都先后感染上病毒，Angela 在五月份就开始服用羟氯喹加锌，一星期一次从未中断过，全家就她一人没有感染。Justin与 Jennifer都很健壮，相当于职业运动员，免疫能力高于常人，Jennifer七月底外出参加音乐会，回家第二天就出现症状，头痛、发热，核酸检测为阳性。

Angela 刚开始服药时家里其它人都拒绝服用，于是就当着家人的面服用，在家人感染而 Angela 无恙时，家人就很好奇。本来想让家人早晚各服一次，共400mg 的量，但家人拒绝，只同意服用一次200mg的量，连续服用了5天。Jennifer 在服药的第三症状就完全缓解，仅用12天就恢复阴性。此次服用量只是预防的用量，不是治疗用量，青年少身体本就非常健 康，这一点点药对他们就非常敏感而有效，还添加了鱼肝油，而且鱼肝油全家都已服十多年。

Justin 因为妹妹感染就也去做了核酸检测，结果也是阳性，但没有症状，仅嗅觉和味觉有所减退。Justin 是全家唯一接种过疫苗的人，因为要代表大学参加比赛，必须接种，但依然被感染。由于同住一个屋檐下，他并不感到吃惊，但当嗅觉和味觉出现问题才意识到严重性，这完全是新冠的症状，他接种之后应该有免疫能力，不应该出现这种症状，所以他才开 始接受用药。嗅觉和味觉在用药5~6周之后恢复，即使像他这样强壮的年轻人也需要这么久的时间，其它的战友要多点耐心，不要着急。

Angela的先生早出晚归，与家人接触较少，感染也最晚，第四天开始肌肉有萎缩现象，无法喝水、吃东西，但肚子很硬、头痛、喘气很费力像拉风箱一样，体温在38.5度左右。羟氯喹200mg加锌25mg、阿奇霉素500mg(只吃了一次)， 当时伊维菌素已收到，但由于Angela本人没有服用过，Angela的先生拒绝服用，只吃Angela吃过的药，过了一周左右先生的病情好转之后才服用伊维菌素，服药第二天背部出现皮疹。当时 Angela的先生躺在床上血氧含量才90，起床活动之后一 直掉到 80-70(指夹式的不太精确，应该是低于90高于80，低于80基本就失去自主呼吸能力了)，去洗手间要扶着走，服药5天后 可以开车出门了。

**Eglise医生点评**

硫酸羟氯喹容易产生副作用，美国用槲皮素替代了硫酸羟氯喹，硫酸羟氯喹并不适用于所有人，特别是心脏病人，会有心悸、胸闷症状。Angela 当时给先生用药时是空腹服用，对胃是有损伤的，会有烧灼感，应饭后服用，或服用抗胃酸药后10分钟再服用。

伊维菌素是抗寄生虫的，用药目的不一样，服用方式就有区别：如果主要针对胃肠道，需要空腹服用，可以让药物的集中于胃肠道；如果需要进入血液作用于人体细胞上，就需要大量吸收，而脂溶性物质就能促进这一过程，这就需要脂肪类食物，如高蛋白食物就可帮助吸收。英国大卫在服用伊维菌素前吃了羊肉，有助于吸收，因此大卫在服用数小时之后就有明显好转。

Angela的先生始终没有服用地塞米松，在服用伊维菌素之前有持续7天左右的咳嗽， 如果抗病毒的药量够大，控制了病毒的载量，即使有咳嗽(这往往是细胞因子风暴出现的症状，引起肺泡改变)，此时服用少量地塞米松有可能会缩短病程，但 Angela的先生没有服用地塞米松，说明病毒的载量得到了完全的控制，所以没有继续发展下去，因此地塞米松并不是必需的药物，这种情况下晚用比早用要好，因为地塞米松是类固醇制品，属激素类药物， 作用就是抑制免疫力，如风湿性关节炎、红斑狼疮患者都采用免疫抑制药物，因为他们不需 要强烈的免役反应。

而在病毒感染的早期，人体的天然免疫开始调动，在这个病毒复制期需要天然免疫力去杀灭病毒，减少复制。此时如果过早地服用地塞米松就相当于过早地卸下武装，而Angela的先生经过了一个自然的控制病毒的过程，症状缓解，说明她先生的免疫力很好。

病毒有一个生命周期，到末期时病毒会自行解体，解体所形成的碎片会激怒免疫系统， 免疫细胞就会大量调动起来，释放出大量细胞因子，身体会难以承受，因为免疫细胞会攻击 所有碎片所在的位置(如体细胞、血管细胞，这些在人体中是无处不在的)，包括周边的健 康细胞，因此造成全身性的症状，包括心脏、肾脏、肺泡、胃肠道，所有症状都会出现，器 官衰竭往往就是这样引起的。

**问答环节**

**文耀：**

43岁女性，5月份打了两针科星疫苗，6月份停经，干咳乏力、味觉嗅觉减弱，鼻塞流涕，咽痛，有结膜炎出现，肌肉疼痛，腹泻等现象，但没有跌倒过，10月底去医院做了超声波检查，发现子宫多发肌瘤，内膜回声不均，右卵巢内囊性块，医生只开了处方醋酸甲羟孕 酮片，每天1次，每次5片。

**Eglise：**

非常典型的新冠疫苗接种后的并发症，不管是感染还是接种，它们的并发症都有重叠之处，因为疫苗本身就是S蛋白，是病毒入侵人体最主要的蛋白质结构，以此为模板制作的疫 苗，不论是灭活疫苗还是mRNA 疫苗，都与S蛋白有关，虽然人造的S蛋白与病毒的有许多不同之处，但它们引起的症状有很多重叠之处，都会作用于血管上皮细胞，不管是中枢大 脑神经还是机体的末梢循环，都有可能发生大大小小的血栓。

在用青蒿素哌喹片排毒时有一些严重的副反应，如发热甚至肺水肿，但这些副反应并非所有人都有。有些人在第一个疗程后期有症状，在休息三周恢复后进行第二次排毒时这些症 状没有出现，这是一种全新运用，完全没有经验可以借鉴，有可能这种症状是排毒过程中必 要的强烈反应，可能克服了第一次，第二次第三次排毒时会越来越轻松(仅仅是猜测，无确切答案)，可能涉及个体差异问题，因人而异，很遗憾因没有条件无法给出确切答案。如果青蒿素哌喹片对你不适用的话，可以采用伊维菌素，至少对控制症状是非常有效的，可以不 用硫酸羟氯喹，但一定要用伊维菌素，加上免疫调节剂，如槲皮素、鱼肝油。

**文耀：**

70多岁，打了两针疫苗一直不舒服，且查出有白血病症状，做了几天化疗，是否可以吃 青蒿素哌喹片排毒?服用剂量可否给个建议?

**Eglise：**

前面已经讲过，血小板上有许多受体，容易被侵犯成为受害者，所以血小板减少是很常 见的接种副反应，甚至成为白血病，还会引起其它的肿瘤的恶化。比如良性肿瘤、原位癌(早 期癌症)在接种之后会很快恶化。用青蒿素解毒是完全可以的，如遇个体差异出现严重副反 应的情况还是要考虑停药，换成伊维菌素。

双氢青蒿素与青蒿素哌喹片相比多了双氢键，使得血液中药物有效浓度的持续时间延长， 对于解毒应该也是一个很好的选择。但因为没条件进行比对，无法给出确切的信息。

**文耀：**

亲戚打了两针疫苗，服用伊维菌素一周后出现嗜睡现像，接下来应该如何服药?

**Eglise：**

依然按方案进行。

**文耀：**

“快乐鸟”战友和他儿子一直在服用硫酸羟氯喹片，每周一次，但体重增长了不少，听了 郭先生直播才知道与激素有关，有没有办法避免这个问题?

**Eglise：**

从医学生理学来讲，硫酸羟氯喹来源于秘鲁的金鸡纳树皮，属天然成份，与体重增加没 有必然的关联性，个人不认为与激素有任何关系。除非药物促进了食量的增加，这可能是一 个诱因。(文耀:也许与大家在疫情期间不太出门、不做运动有关系)硫酸羟氯喹对肠道有刺激，有些人服用之后会不自觉地吃更多的食物，让自己的胃肠道舒服一点，无形中就增加 了食欲。

**文耀**：

“喜国丽影”问谷胺酸可以与青蒿素保健品、槲皮素同时服用吗?

**Eglise：**

青蒿素含氧化基，而维生素 C、槲皮素、NAC 它们是抗氧化剂，也就是说它们有还原键，放在一起就会产生氧化还原反应，但不是很强烈的反应，是一种相互抵消作用的反应。 最好分开服用，以保持各自药效的最大化，建议早晚分开服用。

**文耀：**

“众人平安”问伊维菌素、羟氯喹、NAC、槲皮素都有，打了两针疫苗应该如何服用?小 孩 12-15 周岁如何服药?

**Eglise：**

12-15 周岁建议按成人用量服用，很多药物通过肝脏转换酶代谢，12 周岁是一个标志性年龄，肝功能已发育成熟。伊维菌素、硫酸羟氯喹对胃肠道有较强刺激，这两个加在一起很 多人的胃吃不消，所以为了实用考虑而非从药理上讲，伊维菌素配合槲皮素相对而言温和一 些，NAC 可一直服用，属锦上添花，维生素 C、维生素 D3、褪黑素都可以用。维生素 C 尽 量和其它药分开服用，隔数小时即可。

**文耀：**

“追寻自由 TW”问服用伊维菌素会拉肚子是正常反应吗?

**Eglise：**

这种反应这是有可能的，因为伊维菌素刺激胃肠道，如果达到令人很不舒服的程度可以换药，换成硫酸羟氯喹或是槲皮素。或是槲皮素加 NAC，保证两种添加剂对于抑制病毒复制是有保障的。

**文耀：**

“every 莫扎特”问半年前打了两针，上星期体检贫血，病毒阳性要复诊，不知是否是疫 苗的关系?

**Eglise：**

可见接种疫苗不能防止病毒感染，还有就是病毒的检测手段本就很有问题，特别是所谓的核酸检测 PCR，这个技术本就不是用于 RNA 病毒感染的诊断，CDC 莫名其妙地引入了这 个检测方法，这个方法不是标准的用于 RNA 病毒的诊断手段，因为它会把检测时的背景噪音扩增，而且它不能鉴别新冠和其它任何 RNA 病毒的不同之处，也就是不能进行质的区别， 很可能会检测出阳性，至于是流感病毒阳性、还是疱疹病毒阳性、还是新冠病毒阳性它是无 法区分的，这不应该成为新冠的诊断手段。

务必按时按量服药。
