## 安德鲁·赫夫称：彼得·达扎克效命于中央情报局，生态健康联盟是“中情局前台组织”
` GTV News Talk` [轉載自GNews](https://gnews.org/zh-hans/1904188/)

作者：首尔天池农场 – brandnew
![This image has an empty alt attribute; its file name is image-40-1-9-6.png](https://assets.gnews.org/wp-content/uploads/2021/11/image-40-1-9-6.png)
据《InfoWars》1月20日报道，生态健康联盟的副主席助理安德鲁·赫夫博士1月12日公开表示，在中共国的武汉病毒研究所进行功能增强研究的生态健康联盟主席彼得·达扎克曾告诉他说，他在为美国中情局工作。另外，赫夫博士认为生态健康联盟是“中情局的前台组织”。
![](https://assets.gnews.org/wp-content/uploads/2022/01/FireShot-Pro-Webpage-Screenshot-001-REPORT_-Peter-Daszak-Worked-For-CIA-EcoHealth-Alliance-Is-A-‘CIA-Front-Organization-www.infowars.com_.png)
由达扎克领导的生态健康联盟，获得了多个美国政府机构的资助。早在新冠疫情最初爆发前，该组织就与北卡罗来纳大学的拉尔夫·巴里克博士和武汉病毒研究所的石正丽博士合作，在中共国对蝙蝠携带的冠状病毒进行功能增强研究。

据报道，达扎克督导了对“数千个蝙蝠样本”的筛选，“以获取新型冠状病毒”。这项有争议的研究还包括“筛选与活体动物打交道的人”。

赫夫于1月12日发布了多条推文指出，在2015年里，彼得·达扎克在三个不同的场合说了他与中情局合作的最新情况。后来达扎克说中情局的人对他们工作的地方、相关的人员及收集的数据感兴趣，并表示与中情局的合作会继续进行。

赫夫详述了他认为达扎克的生态健康联盟是“中央情报局的前台组织”，其目的是收集病毒样本和外国实验室能力的相关情报“。在过去两个月里，美国政府耗费数百万美元对他和MTRX公司（注：一家科技公司，赫夫是创立人之一）员工进行监控，包括军用飞机，试图摧毁他的住所，窃听里面的一切，盗窃他的财产，以及非法入侵他所有的设备。

在另一系列帖子中，赫夫表示，“如果说作为福奇、弗朗西斯·柯林斯、达扎克、巴里克和许多其他人之间的中间人，中情局和情报界精心策划了对中共病毒（新冠）疫情的掩盖，他对此毫不惊讶。说的好听点，这是美国历史上官僚或政客代理人最大的犯罪阴谋。”
