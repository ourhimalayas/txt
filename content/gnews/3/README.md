## 从当前的粮食危机想到中共刻意打造的袁隆平神话
` 澳喜农场` [轉載自GNews](https://gnews.org/zh-hans/2185560/)

撰稿：三只松鼠
![](https://assets.gnews.org/wp-content/uploads/2022/03/Picture3-9.jpg)来自网络截图
近日，俄乌战争尚牵绊着全球人民的目光，由于俄乌都是产粮大国，世界上关于粮食危机的可能性也越来越大了，尤其是现在越来越依靠粮食进口的中共国尤甚。说到中共国粮食问题，不得不提到一个当年被中共国神话的人物，中共国的粮食专家袁隆平。

[袁隆平](https://www.epochtimes.com/gb/tag/%E8%A2%81%E9%9A%86%E5%B9%B3.html)已于2021年因病逝世，享年91岁。[袁隆平](https://www.epochtimes.com/gb/tag/%E8%A2%81%E9%9A%86%E5%B9%B3.html)生前是中共的红人，被誉为“[杂交水稻之父](https://www.epochtimes.com/gb/tag/%E6%9D%82%E4%BA%A4%E6%B0%B4%E7%A8%BB%E4%B9%8B%E7%88%B6.html)”，曾获得过“[共和国勋章](https://www.epochtimes.com/gb/tag/%E5%85%B1%E5%92%8C%E5%9B%BD%E5%8B%8B%E7%AB%A0.html)”、入选中国工程院院士。有意思的是，中共党媒CGTN当天提前误报了袁隆平的死讯，并为此公开道歉，称“报导不慎”。几个小时后，党媒新华社才再次确认袁隆平去世。

中共党媒对袁隆平逝世消息的宣传调子则从“让中国人不再挨饿”、“养活了中国人”到“养活了全世界”。

其实很多中国人对中共硬把袁隆平树立成英雄也是颇有微词的，袁隆平是研究杂交水稻，水稻产出大米，但是中国很多地方是不吃大米的，他们的主食是面食，像是馒头、面条等，这些都是小麦生产来的。

旅美独立学者、时评专栏作家 戈壁东：“水稻专家袁隆平是一个比较有争议的农业专家他在改良水稻的品种，提高产量上是有贡献。但是中国网友的一个评价是最有趣的。他们说，袁隆平在水稻产量的上的最大的贡献，是从中共的《人民日报》吹嘘的亩产三万六千斤恢复到了正常的两千斤。”

下面我们选取一些网友们留言：

“袁隆平在农业科技方面是有成就和贡献的，但是他的功绩无限夸大了，成了中共的政治宣传机器。他研究的杂交水稻在市场上不受欢迎，反正我在中国的超市和商场里基本没见过他的杂交水稻，中国最受欢迎的水稻是东北大米。而且杂交水稻的很多科研成果不是袁隆平本人的。”

“共和国中央的造神运动自[文革](https://www.epochtimes.com/gb/tag/%E6%96%87%E9%9D%A9.html)结束以后就没有停止过，不仅把领导人造神，甚至把科学家造神，使得这些本来与民为伍的知识分子群体与群众越来越远。有没有吃到过他本人研究的杂交水稻我不知道，但是造神这种行为我很反感，这种行为会打压以后的科学研究人员的研究兴趣，制造大量不必要的、只会得到负反馈的官僚系统，可能会给共和国未来的科研之路带来严重灾难。”

还有网友评论说：“袁让中国人吃饱，那是谁让中国人挨饿的呢？又不让说了。”

“他可能在杂交水稻方面有一点贡献，但远没有共匪宣传的那么高。现代农业的发展，主要还是归功于西方科学技术的进步，归功于化肥的发明和转基因技术的应用。”

“袁隆平是中共捧上去的。具体来说是华国锋捧上去的。他上去后把其他工作人员的贡献据为己有。还引起了其他研究人员的愤怒，但是都被压下来了。”

“他把很多其他研究水稻的人都搞得很惨。黑历史后来都被中共抹去了。有研究人员试图告他，但都被打压下去了。因为他是中共的典型人物”。

“对中国粮食的贡献，美国人远在袁隆平之上。”

不过袁隆平还是说了一些大实话。袁隆平2009年接受《广州日报》采访时也谈到：“三年困难时期，饿死了几千万人啊。大跃进把树都砍了去炼钢铁， 把生态破坏了，1959 年大干旱，一年基本上没有收成，饿死了四五千万人啊。”

袁隆平研究的[杂交水稻](https://www.ntdtv.com/gb/%E6%9D%82%E4%BA%A4%E6%B0%B4%E7%A8%BB.htm)产量比一般水稻提高了一些，但是缺点是口感并不好。中国从1976年开始，大范围推广了袁隆平的杂交水稻，但随着生活的改善，人们更加追求米的口感和味道。因此目前中国人大部分吃的是常规稻和进口稻，杂交水稻主要用于战略储备和工业生产等用途。

旅美时事评论员邢天行：“（中共）它一直在宣传是袁隆平发明了这个杂交水稻之后，解决了中国人的多少多少亿人的吃饭问题。这个谎言一直都是在这样讲。中国其实每年都是从国外买入大量的粮食。”

邢天行：“（中共）它已经是把袁隆平这个个人，打造成它的一个党产。就是中共它在给老百姓保证了温暖等等这些方面，中共的一个贡献。因此它不允许老百姓对于袁隆平有半点的质疑。”

从以上这些评论员及网友们的发言可以看出，中共是典型的以假治国，把袁隆平吹成“养活了全世界”。“养活了全世界”其实是华人到西人媒体上发表的文章，然后又出口转内销到国内做宣传。另外杂交水稻其实也不是袁隆平一个人的功劳，它是几代科学家们的努力。单说中国除了袁隆平之外还有石明松、陈日胜、谢华安等等众多科学家，尤其是谢华安培育的水稻（汕优63）在国内种植面积最大，号称80%的人吃的杂交稻都是谢华安的产品。

不管怎样，袁隆平个人是因为大饥荒饿死人的刺激才发奋搞了农业研究，他再有本事也不可能把水稻亩产提高到万斤。共产党千方百计掩盖、否认大饥荒饿死几千万人的事实，他们吹嘘“亩产万斤”导致饿死人的罪责是抹不掉的，并且想用科学技术发展掩盖他们吃人的罪恶制度。共产党种下的种种罪恶，到了算总账的时候了。共党必灭！新中国联邦必胜！
