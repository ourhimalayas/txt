## 美国的通胀已经升至40年来最高
` 加拿大温哥华英里农场` [轉載自GNews](https://gnews.org/zh-hans/2318580/)

编辑：文宝



最近几个月，美国的通胀已经升至40年来最高。据知名投行瑞银集团（UBS）的数据显示，美国2月份的年通胀率跃升至7.9%，最新预测显示3月份的通胀率可能达到8.5%。这样的比率将是自1981年以来的最高水平。

本周早些时候，摩根大通首席执行官杰米·戴蒙（Jamie Dimon）在致股东的年度信函中表示，高通(136.69, -2.53, -1.82%)胀、俄乌冲突和对俄罗斯的制裁可能会“极大地增加美国未来的风险”。他警告说，该国经济将面临不可预测的后果。

《财富》杂志称，美联储为应对国内通胀问题，几周前已经加过一次息。更高的利率会导致短期收益率突然变得比长期收益率更高，形成收益率曲线“倒挂”。哈奈特认为这种“倒挂”是衰退临近的迹象。
