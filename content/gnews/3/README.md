摘要翻译：

《国会山报》2022年2月22日报道，

在俄罗斯总统弗拉基米尔·普京(Vladimir Putin)承认乌克兰的两个地区为独立国家并将俄罗斯军队调入乌克兰东部后，澳大利亚、加拿大和日本周二对俄罗斯实施制裁，加入了越来越多采取经济行动的国家名单。

美国、英国和欧盟都对俄罗斯实施了制裁，以回应普京承认卢甘斯克和顿涅茨克这两个由俄罗斯支持的分离主义分子控制的地区为独立国家。普京周一还下令俄罗斯进入这些地区以履行“维和职能”。

据澳大利亚广播公司(Australian Broadcasting Corporation, AB)报道，澳大利亚总理斯科特·莫里森(Scott Morrison)在宣布澳大利亚的制裁时表示，这些制裁将包括针对俄罗斯国家安全委员会成员的有针对性的旅行禁令和金融制裁。

莫里森表示，更广泛的制裁将扩大到分离主义分子控制的地区，并补充说，他预计将对寻求从俄罗斯入侵乌克兰中获利的俄罗斯公民实施更多制裁。澳大利亚总理还表示，乌克兰的签证申请将被移至“最顶端”。

根据ABC报道，莫里森说：“对乌克兰的入侵实际上已经开始。这种入侵是不合理的、没有根据的、无端的、不可接受的。我们不能有一些暗示，俄罗斯在这里有一些他们正在起诉的正义案件——他们的行为就像暴徒和恶棍一样。”

在一次新闻发布会上，加拿大总理贾斯汀·特鲁多(Justin Trudeau)称俄罗斯周一的行动“违反了俄罗斯在国际法和联合国宪章下的义务”。

“加拿大也谴责俄罗斯的军事行动，包括进入乌克兰的命令，这显然是侵犯乌克兰主权和领土完整。毫无疑问，这是对一个主权国家的进一步入侵，这是完全不可接受的。”特鲁多说。

作为加拿大对俄罗斯制裁的一部分，特鲁多宣布，所有加拿大人将被禁止参与“与卢甘斯克和顿涅茨克所谓独立国家的金融交易”。加拿大人也将被禁止购买俄罗斯主权债务。

加拿大还将针对投票承认卢甘斯克和顿涅茨克的俄罗斯议会成员，以及两家国家支持的俄罗斯银行进行制裁。

据路透社(Reuters)报道，日本首相岸田文雄(Fumio Kishida)说：“俄罗斯的行动非常明显地损害了乌克兰的主权，并违反了国际法。我们再次批评这些举动，并强烈敦促俄罗斯重新进行外交讨论”。

岸田说，日本的制裁将包括禁止俄罗斯债券在日本发行，冻结某些俄罗斯人的银行账户以及限制赴日旅行。莫斯科的精英人士和俄罗斯银行也将成为制裁目标。

这位日本领导人表示，制裁的细节将在未来几天内制定并公布。他补充说，由于石油和液化天然气储备充足，日本的能源供应在短期内不会受到影响。岸田说，如果能源价格开始上涨，他将考虑所有可能的措施，以防止对日本企业和家庭产生影响。
