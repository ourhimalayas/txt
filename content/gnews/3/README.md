**建立境外公司如何才能委托到专业的律师或机构？**

**当我们讲到建立一个境外公司时，难免担心如何才能委托到适当和专业的律师或机构。这个是个很广泛的问题，所谓专业人士简单来指的就是这个人是全职在做这个事情，怎么看他具体有多专业呢？比方说律师，他有多少经验？业务范围多大？做房地产、信托还是遗嘱类的？做的海外信托还是加拿大信托？做的海外信托的地区是哪里？对该地区了解多少？对建立信托的国家之间了解多少？所以专业能力、专业范围和我们的预期目标都是重要的参考指标。**

**如何能驾驭财富，有正确识别律师和会计的能力？**

**有钱不代表你有驾驭财富的能力，要学会做财富的主人，提高自身能力，包括财富管理能力非常重要，同时也要有正确识别律师和会计的能力。建立境外公司这种事情，首先你会看到一个合同，你对合同里的条款的理解是否跟你的业务提供商的理解是一致的？他给你的出的报表精细到什么程度？这个精细程度有没有在合同里写下来？这时候需要找专业的律师来读他们的文件，使之能够在法律上能彼此读懂。如果是律师的话，看他在法庭上有没有过纠纷？纠纷如何解决的？他的客户对他的满意度评分是怎样？非常重要的是他是否已经在那边接到类似的业务？当然，千万不要告诉他们你有多少币，要会把控风险。**

**我们应该成立公司、基金还是信托？**

**控股公司也就是资产公司，信托里面可以持有一个基金，基金可以是单独的一个实体，也可以是控股公司下面持有的一个投资账户，账户里面可以持有不同的证券。信托可以设置，控股公司的信托可以是信托，基金可以被信托或控股公司持有。这三者怎样选择，怎样建立，整件事情要考虑的是你所在国家的管辖区设怎样来处理另外一个国家建立的公司、基金和信托里面的所有税务问题，法律允许，且你衡量下来哪种结构适合你，你就可以选择哪种。**
