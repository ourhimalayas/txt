当地时间2月9日，瑞典取消了几乎所有的疫情限制措施，并停止大多数的中共病毒检测。

从9日起，瑞典的酒吧和餐馆获准在夜间23时以后可以继续营业，而且没有人数限制。对于大型室内场合的人员要求，以及疫苗护照的使用也同时被解除。

在整个大流行期间，瑞典政府选择采取自主防疫，而非强制措施。据统计，该国死于中共病毒总数为1万6千人左右，远低于大多数的欧洲国家。
