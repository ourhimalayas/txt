## 世界卫生组织点名批评日本新规禁止外国人入境

![](https://assets.gnews.org/wp-content/uploads/2021/12/image-55.png)[图片来源](https://news.yahoo.co.jp/articles/62fd4de1191487af563a74a29fbcb6e062d98351)
日本雅虎网站12月2日消息：据韩国媒体报道，因日本为防控中共病毒新变种Omicron毒株的传播，宣布新规禁止外国人入境，世界卫生组织（WHO）对日本提出“点名批评”。

韩国媒体2日报道，世卫组织应急小组组长迈克尔-瑞安在1日的新闻发布会上指出，日本新规禁止外国人入境的做法从流行病学角度难以理解。“病毒不看国籍或居留证”，用是否是公民来区分是否可以入境这种做法是与之矛盾的。

世卫组织对全面限制入境持否定态度，敦促使用病毒检测和其它措施。瑞安还说，日本的反应“从公共卫生的角度来看是不符合逻辑的”。

之前，日本政府从11月30日午夜开始禁止所有外国人进入日本，以防止Omicron毒株的传入，并宣布包括南非在内的10个国家的，具有日本居留身份的外国人，也不可以再次进入日本。
