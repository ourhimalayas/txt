当地时间12月30日，据《国家评论》报道，美国前线医生组织(AFLDS)在最新一期简报中警告说，由实验性新冠mRNA疫苗产生的刺突蛋白，能够通过“血脑屏障”导致神经损伤；让接种疫苗的人“脱落”刺突蛋白，导致未接种疫苗的儿童和成人生病；导致女性不规律阴道流血。

该文件于上周发布，标题为“确定接种疫苗后并发症及其原因：对新冠患者数据的分析”。该非营利组织强调了与这些“疫苗”相关的数千个不良事件，并由CDC的疫苗不良事件报告系统（VAERS）所记录。“然而，这些并发症只在强生疫苗血栓争议中只得到了一小部分关注”。

在仔细研究这些数据时，AFLDS提出“FDA或CDC尚未公开解决的一些主要问题类别”，断言这些监管机构未能“考虑这些和其他‘已知的未知数’是一种基础医学研究的失职”。具体如下：

首先，人们对这些新疫苗的广泛接种存在严重担忧，**这些新疫苗采用新技术并且在没有得到FDA的完全批准的情况下仍只是实验性的**。**这些实验性试剂不会像传统疫苗那样使用减弱的抗原反应，而是将一种称为“刺突蛋白”的物质引入人体系统。**

AFLDS文件证实：“需要多年的时间才能确保新生事物是安全的，**没有人确切地知道这种刺突蛋白对身体和大脑的长期健康影响，特别是在年轻人中**。此外，**如果这种蛋白质确实出现了记录在案的问题，将永远无法逆转那些已经接种疫苗的人的不良反应**。”

其次，与传统疫苗不同的是，这些刺突蛋白和“脂质纳米颗粒”能够穿过“血脑屏障”，为身体的这些敏感区域提供特殊保护。但文件警告说：“根本没有足够的时间来了解会产生哪些脑部问题，以及由此而发展出的脑部问题有多频繁。”

“这种渗透的风险包括“神经系统中的慢性炎症和血栓形成（凝血），导致震颤、慢性嗜睡、中风、贝尔麻痹和萎缩性脊椎侧所硬化症（ALS）症状；脂质纳米颗粒可能与脑细胞融合，导致迟发性神经退行性疾病；mRNA诱导的刺突蛋白与脑组织的结合能力是原始病毒的刺突蛋白的10到20倍。”

第三，**由于这些实验性疫苗会在接受者体内产生数以万亿计的刺突蛋白，因此这些接种疫苗的个体“可以将其中一些（刺突蛋白）颗粒传播给密切接触者”，从而导致他们患病**。

AFLDS的创始人西蒙·戈尔德博士将这位记者介绍了4月29日的推文，她在推特上发布了辉瑞实验性疫苗试验的一份文件，她写道，这家制药巨头“承认刺突蛋白潜在脱离的这种机制”。

正如该文件所述，一个人可能“因环境暴露而接触到这项研究干预”，包括“通过（空气）吸入或皮肤接触”给参与研究的人等。

根据AFLDS的说法，这可能是危险的。随着问题深入，“刺突蛋白就像完整的病毒一样具有致病性（引起疾病）”，此外，这些“刺突蛋白比完整的病毒结合得更紧密”，因此**世界各地都有“****心包炎、带状疱疹、肺炎、四肢和脑部血栓、贝尔麻痹、阴道出血和流产的病例报告，这些病例都发生在靠近接种过疫苗的人身上**”。**这种脱落也“也会导致某些人患上各种各样的自身免疫性疾病（身体攻击自身组织）”**。

此外，**由于“与传统疫苗不同的刺突蛋白可以穿过血脑屏障”。甚至对未接种疫苗的人也可能面临其他更严重的危险**。

第四，**如果孩子靠近接受这些实验性疫苗的父母和老师**，这些脱落的疫苗会使孩子受到伤害。“我们的另一个担忧是，因为儿童比成人他们早了几十年，让身体内有数万亿的上述刺突蛋白，**儿童可能会患上长期慢性自身免疫性疾病，包括神经系统问题”。**

第五，“AFLDS了解到，数以千计的关于新冠疫苗接种后阴道流血、绝经后阴道流血和流产的报告，以及与接种疫苗的密切接触者发生类似不良事件的传闻”，**许多报告（提到）的疫苗接种后阴道出血事件，确定了“疫苗和不规则出血之间的明确关联**”。

“**尽管有这个明确的证据，但月经周期变化并未被列为FDA在其三期临床参与者中的常见副作用**。妇女的生殖健康需要得到认真对待，而不是被议程驱动的公共卫生官员所抛弃，”简报中写道。

最后，AFLDS认可“制药公司在不可抑制的经济利益驱动下”销售不必要和危险的儿童新冠疫苗、加强针等，并坚持认为，“公共卫生专家应该在（疫苗）灾难到来之前，停止接种并评估关于可能的疫苗副作用和相关接种后问题的数据”。

因此，**辉瑞自己的内部报告警告医护人员不要接触接种过疫苗的人**——考虑到大多数医护人员都需要自己接种疫苗，这颇具讽刺意味。
