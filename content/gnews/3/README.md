近日，新西兰推出一项新的安乐死法案，允许政府为医疗从业人员每次执行生命终止程序支付1087美元。

一个名为DefendNZ的反安乐死组织上个月要求政府澄清这项名为《生命终点选择法》（EOLCA）的新安乐死法案，特别是它与 “重症住院”的中共病毒患者有关。

该组织质问：”如果医生认为一个患有中共病毒的重症住院病人的预后少于6个月，那么他是否有资格根据该法案获得协助自杀或安乐死？

新西兰卫生部（MOH）于12月7日已经回应证实：”在某些情况下，患有中共病毒的人可能有资格获得协助性死亡。

卫生部表示：

协助死亡有明确的资格标准。这些标准包括，一个人必须患有可能在六个月内结束其生命的绝症。末期疾病通常是指治疗无效的长期疾病。《EOLCA》规定，资格由主治医生（AMP）和独立医疗从业人员决定。在某些情况下，患有中共病毒的人可能有资格接受协助性死亡。

《天主教先驱报》报道说：”医生每执行一次安乐死，就会收到1000美元的政府费用，外加开支。”不过，全国16000名医生中，只有96名医生表示愿意参与，而全国32家临终关怀机构中，除一家外，都表示他们不会允许安乐死。”

英国缓和医学教授芬利夫人告诉《先驱报》，EOLCA背叛了生物伦理学的基本原则。

芬利说：”奇怪的是，一个一直试图通过完全封锁来保护公民避免一种可以完全康复的病毒的国家……现在却建议这些病人应该被他们的医生杀死，这很奇怪。”它颠覆了医学的道德观。”
