有储户拍到在该银行网点的门上贴的告示：目前社会上存在对禹州新民生村镇银行的诽谤，谣言。请大家保持清醒的头脑，大家不要信谣，对传谣者我行将追纠其法律责任。” 有人推断按照中共的宣传套路银行没钱了是真的。
