## 中共最终将把特斯拉挤出中国，玩弄市场换技术的把戏将会重演

![](https://assets.gnews.org/wp-content/uploads/2022/01/image-1238.png)2021年3月29日，中国上海的特斯拉上海超级工厂。（Xiaolu Chu/Getty Images）
自《大纪元》2022年1月14日报道，总部位于华盛顿的非营利性信息技术与创新基金会（ITIF）主席罗伯特·阿特金森（Robert Atkinson）表示，中共强制技术转让的国家战略意味着特斯拉最终将被挤出中国市场，就像过去其他西方公司所遭受的命运一样。

“归根结底，中国的战略非常明确。就是他们会利用你，然后把你推到一边，”阿特金森在最近接受大纪元姊妹媒体NTD采访时说。

“我相信中国对特斯拉的战略是，嘿，我们将从特斯拉那里获得一些好处。但最终，我们希望像吉利和其他中国电动汽车制造商这样的公司能够主导，不仅是中国市场，还有很多世界市场，”他补充道。

众所周知，中共政权要求外国公司与包括许多得到共产党政权支持的中共国公司建立合资企业，以获得他们的专有技术、有价值的技术和知识产权 (IP)。

特斯拉似乎已经避开了这种风险，2018年，它成为第一家在没有中共国公司做为合作伙伴的情况下在中国设立全资工厂的外国汽车制造商。该工厂位于上海，于2019年底开始生产特斯拉Model 3s。

不过，阿特金森表示，中共确实从特斯拉那里获得了“直接或间接的技术转让”。他补充说，更重要的是，中共国电动汽车制造商也将受益，因为目前特斯拉的一些工人最终将转行，转而为中共国电动汽车制造商工作。

例如，他提到了两家欧洲电信公司诺基亚和爱立信，称这两家公司过去在中共国市场上占有很大份额，后来割让给中共国竞争对手中兴通讯和华为。

“我认为这就是我们最终会在某个时候看到的（特斯拉）的情况。真正的问题是多久？” 阿特金森说。

阿特金森说，但是也许对所有外国公司来说更重要的是，知道中共并没有放弃目前获取西方技术的方法。

“强制技术转让、知识产权盗窃、知识获取，无论你怎么称呼它，至少在未来十年或二十年内，仍将成为中共经济的核心部分，”他警告说。

根据哈佛大学增长实验室的全球经济复杂性排名，中共国经济正变得越来越复杂，在过去20年中迅速赶超美国。其经济复杂性指数(ECI)排名越高，表明一个国家出口的商品更加多样化和复杂。

2000年，中共国在ECI指数中排名第39位，而美国则是世界第六复杂的国家。 2019年，中共国跃升至第16位，而美国则跌至第11位。日本在2000年和2019年均排名第一。

为了扭转这一趋势，阿特金森表示，去年6月在参议院通过的一项法案《美国创新与竞争法》(USICA)是建立美国在先进技术领域能力的一项重要立法。

如果获得通过，该法案将授权约1900亿美元用于科学研究和开发，以及520亿美元的资金用于促进美国半导体—为从智能手机、电动汽车到导弹等各种产品提供动力的微型芯片—的生产和研究。

阿特金森补充说，美国政府可以做的还有很多，比如拒绝那些已知从知识产权盗窃或政府巨额补贴中受益的中共国公司进入美国市场。

美国贸易代表在2018年的“301条款”调查中发现，中共国的非法贸易行为包括政府过度补贴和知识产权盗窃。这推动了中美贸易战和2020年1月签署第一阶段贸易协议。根据协议，中共国同意在2020年和2021年期间额外购买至少2000亿美元的美国商品和服务，并结束强制技术转让。

“我认为归根结底，我们在美国必须做的是我们必须开始更加努力，”阿特金森说。“如果我们不这样做，那么我认为它基本上已经结束了。”
