3月17日时事通信社伦敦消息：俄罗斯中央银行和中国人民银行（CBM）正在考虑合并各自的银行支付网络，作为全球银行支付网络–环球银行间金融电信协会（SWIFT）的替代方案。

这是在一些俄罗斯银行因入侵乌克兰，而遭制裁被排除在西方主导的SWIFT之外的背景下做出的。如果这个声明成为现实，中共国和俄罗斯之间的关系将更加紧密，同时中俄与西方国家的经济将进一步瓦解。
