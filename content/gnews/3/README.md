## 台湾33案获得疫苗接种受害救济
` GTV新聞訪談` [轉載自GNews](https://gnews.org/zh-hans/1977605/)

[撰稿日期]：2022.02.07

[署名]：台湾宝岛农场\_阿尔道夫

[新闻连接]：《https://udn.com/news/story/122190/6081787?from=udn-catelistnews\_ch2》

[正文]：
据台媒报导，台湾疾病管制署公布卫生福利部预防接种受害救济审议小组(VICP)于一月十三日审查46例疑似因接种疫苗后产生不良反应或死亡的个案，依最新会议纪录显示，其中有3例经判定与疫苗有关，分别如下：

个案（编号3139）经医师诊断为心肌炎，判断为与接种疫苗相关。

个案（编号2586) 接种疫苗后出现胸痛、全身酸痛、呼吸困难而后失去意识，可能致死原因为急性心肌梗塞、主动脉剥离等心血管疾病。但因个案有糖尿病、 高血压，因未有充分之检验及检查数据，无法确定与疫苗之关联，但仍核予救济金。

个案(编号3184) 接踵疫苗后出现皮肤痒症状，但因个案本身有异位性皮肤炎，所以无法确定与接种疫苗的关联性，但仍核予救济金。

从上述案例，虽只有个案（编号3139）审查结果判断为与接种疫苗相关外，其余两例（编号2586、编号3184）无法确定与接种疫苗的关联性，但显示疫苗接种的安全性已受到正视，且对新中国联邦一直呼吁应正视疫苗安全性的行动具有指标性意义。
