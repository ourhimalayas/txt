#### 编译：小红帽

从2017年到2019年，再从2020年到今年秋天，担任奥地利总理的塞巴斯蒂安·库尔茨(Sebastian Kurz)宣布离开政坛。

库尔茨周四在维也纳举行的新闻发布会上表示，他将于周五正式卸任奥地利人民党主席一职，在未来几周内解除所有其他政治职责。

这位前总理称，他刚出生的儿子是使他重新考虑其优先事项的关键因素之一，并补充说，直到最近，他几乎把所有的时间都投入到他所担任的政治职位上。

10月9日，库尔茨宣布，他决定在正在进行的腐败调查中辞去总理职务，这标志着他在保守的人民党和绿党达成联盟协议后，于2020年1月开始的第二个任期结束。

在库尔茨离职之前，奥地利经济犯罪和腐败检察官办公室宣布对当时的总理和他的一些亲密伙伴进行调查。他们涉嫌在2016年至2018年期间操纵民意调查，并使用公共资金确保媒体对库尔茨的政策进行有利的报道，目的是提高他当选为政党领导人的机会，以及随后的总理机会。库尔茨一直否认有任何不当行为，只是在绿党联盟伙伴发出最后通牒后才被迫下台。

库尔茨的第一个任期也因腐败丑闻而结束，当时德国《南德意志报》和《明镜周刊》于2019年5月将 “伊维萨事件”曝光。丑闻发生后，奥地利自由党领导层下台，联合政府垮台；然而，视频中的人坚持说他们是被陷害的，指责德国报纸选择性地编辑视频。
