中共教育部此举不仅被视为是中共教育部企图拍习近平的马屁，同时也是一场企图集体奴化学生心智、摧毁人权的教育灾难。诚如北京法学博士王先生表示，这是中共当局企图通过奴化教育来箝制学生的思想，清除他们的人权、法治等观念。

因此，我们可说这是中共继对新疆维吾尔人权的最糟糕迫害后，转以采取更隐晦、更隐形、更全面性、同时也更具摧毁性的对中小学学生进行人权思维的教育迫害行动，不仅凸显了中共只有党治，没有法治的政治扭曲立场外，更是将中国岌岌可危的人权思潮更加矮化、危及化。因此不仅堪称是教育灾难，更可谓是国之灾难呀!
