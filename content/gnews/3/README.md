## 官员：向美驻伊拉克领事馆发射的导弹来自伊朗

据《福克斯新闻》作者：Kyle Morris，2022年3月12日报道：
![](https://assets.gnews.org/wp-content/uploads/2022/03/image-2040-e1647620983378.png)2017年2月23日，伊拉克摩苏尔南部，美国陆军士兵站在装甲车外。位于与伊拉克军队的联合基地。
图片来源：Associated Press
前中央情报局站长丹-霍夫曼称伊朗正在向美国传递一个“信息”。

一位美国高级官员告诉福克斯新闻，周日从伊朗向美国驻伊拉克埃尔比勒领事馆方向发射了一系列导弹。

据美联社报道，这些导弹的数量多达12枚。

这位美国官员告诉福克斯新闻，没有一枚导弹炸到领事馆。

领馆建筑群中的所有美国人员都有下落，位于伊拉克北部库尔德自治区首府埃尔比勒的领馆附近也没有伤亡报告。

国务院发言人告诉福克斯新闻：“美国政府的任何设施都没有损坏或伤亡，伊拉克政府和库尔德地区政府正在对这一事件进行调查，我们请你们向他们征求意见。我们谴责这种无耻的攻击和暴力表现。”

一位国家安全官员告诉福克斯新闻，没有迹象表明这些导弹是针对美国设施的。

美联社报道说，这次袭击代表了伊朗和美国之间敌对行动的升级，这些敌对行动主要是在伊拉克境内进行的。

美联社报道，袭击发生后不久，总部位于领事馆附近的卫星广播频道Kurdistan24播出了其演播室地板上的玻璃碎片和散落的残骸。

据美联社报道，这次导弹袭击发生在伊朗表示将对以色列在叙利亚大马士革附近的袭击作出回应的几天后，这次袭击导致伊朗革命卫队的两名成员死亡。

美联社报道，伊朗国营的伊斯兰共和国通讯社报道了周日在伊拉克的袭击，但没有说导弹的来源。

但一名伊拉克官员告诉美联社，这些导弹是从伊朗发射的，是伊朗制造的Fateh-110型导弹——这次打击可能是对大马士革袭击的回应。

核谈判处于“暂停”状态

据美联社报道，周日的袭击也发生在维也纳关于可能恢复奥巴马时代的伊朗核协议的谈判“暂停”之际，因为俄罗斯要求对其因入侵乌克兰而面临的经济制裁进行处理。

据美联社报道，美国海军陆战队将军弗兰克-麦肯锡在12月表示，美国军队已经转向在伊拉克的非战斗角色，但伊朗及其盟友仍然希望美国结束在那里的存在。

报道说，留在伊拉克的美军在伊拉克军队与伊斯兰国部队作战时提供空中支援和其他援助。

一位官员告诉福克斯新闻，上一次从伊朗境内向美国在伊拉克的基地发射导弹是在2020年1月，当时美国无人机在巴格达机场附近的袭击中杀死了伊朗的卡塞姆-苏莱曼尼中将。
