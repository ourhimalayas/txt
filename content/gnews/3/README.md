2021年9月20日，求是网首页刊文《做到“两个维护”是首要政治任务》，中工网，理论网等多家媒体跟进转载。在中共国无论做什么，一旦定调为政治任务，就是不讲条件必须完成的，不能质疑这个任务是否合理。

“两个维护”是2018年习近平为了巩固自己的政治地位提出的口号，即维护党中央的核心地位，维护党中央的权威和集中统一领导。

这篇文章赞扬习近平“高瞻远瞩、统揽全局、运筹帷幄、指挥若定”，“作出一系列重大科学判断，提出一系列重大战略策略，推动一系列重大工作”，具有“马克思主义政治家高超的政治领导艺术”，结论是，“习近平不愧为党中央的核心、全党的核心，赢得了全党全国全军衷心拥护，赢得了国际社会高度赞誉。” “两个维护”在本质上是一体的，

正如丁薛祥所言，两个维护其实是一个维护，就是维护习近平的核心地位。这篇文章再次印证了郭文贵先生直播中所说，中共政治局没有什么七常委，就是一个常委习近平，中南坑已经变成“习一坑”。 文章疯狂赞许习近平，显然是中共二十大前为习近平造势。
