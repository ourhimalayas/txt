2月14日，《新闻极限》发布文章，根据《每日邮报》的报道，周一，俄罗斯国家电视台播放了新的宣传片，被认为是即将入侵乌克兰的理由。

周日，俄罗斯受欢迎的新闻主持人和国家宣传员德米特里·基谢廖夫（Dmitry Kiselyov）在周日的节目中说，乌克兰民族主义者准备屠杀居住在该国东部的俄罗斯人。

报道认为，之前白宫已经警告，说军事入侵最早可能在周三到来，于是，俄罗斯国家电视台的这个动作可能是朝着这个方向迈出的最后一步。基谢廖夫的这个节目为军事入侵做了掩护，尽管是未经证实的说法，但他声称乌克兰在顿巴斯地区“折磨和野蛮杀害”了数千名平民。

众所周知，基谢廖夫被称为“普京的喉舌”，因此，他这篇题为“他们正在准备杀人”的报道很可能得到了克里姆林宫的批准。

同时，该节目还包括对一名（乌克兰）分裂主义战士的采访，声称乌克兰民族主义者说，他们将“杀死和屠杀你们所有人，并将你们的孩子挂在电线上”。

文章指出，俄罗斯在2014年入侵乌克兰之前，也采用了同样的宣传手法。当时，克里姆林宫声称要“保护”居住在乌克兰的俄罗斯族人。

据信，多达14.8万名俄罗斯军队与坦克和大炮一起从三面包围了乌克兰。

报道说，虽然围绕乌克兰危机已经进行了几周的谈判，但是，美国情报官员担心，如果普京决定进攻乌克兰，几周的危机谈判给了俄罗斯准备大规模进攻的时间。
