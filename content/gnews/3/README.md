## 马克龙击败勒庞在法国大选中获胜

翻译&评论：金生水
![](https://assets.gnews.org/wp-content/uploads/2022/04/image-2608-21.png)![](https://assets.gnews.org/wp-content/uploads/2022/04/1-547.jpg)图片来自网络
《BBC》近日报道，马克龙在以令人信服的方式战胜竞争对手玛丽娜-勒庞后，赢得了法国总统的五年任期，但勒庞获得了极右派迄今最高的选票份额。

他以58.55%对41.45%的优势获胜，比预期的要大。

这位中间派领导人在埃菲尔铁塔脚下对欢呼雀跃的支持者说，现在选举结束了，他将成为一个 “所有人的总统”。

他是20年来第一位获得连任的现任总统。

53岁的勒庞女士说，尽管她输了，但她的巨大得票率仍然标志着一场胜利。

她告诉她的支持者，她的国民大会所代表的思想已经达到了新的高度。但极右翼的竞争对手埃里克-泽穆尔指出，她最终还是失败了，就像她之前的父亲一样。”这是勒庞的名字第八次失败”。

玛丽娜-勒庞接管了她父亲让-马里-勒庞在2011年创建的政党，以使其成为可选举的政党。她在周日赢得了1300多万张选票，其纲领是减税以解决高昂的生活费用，禁止在公共场合佩戴穆斯林头巾，并就移民控制进行公投。

马克龙先生在他的胜利演说中说：”必须为导致我们许多同胞投票给极右派的愤怒和分歧找到一个答案。”这将是我和我身边的人的责任。”

超过三分之一的选民没有投票给任何一位候选人。投票率略低于72%，是1969年以来第二轮总统选举中最低的，有300多万人投了废票或空白票。

投票当天，法国大部分地区都在休假，但低投票率也反映出选民的冷漠，他们抱怨两位候选人都不能代表他们。那些说自己投了空白票的选民告诉BBC，他们想惩罚现任总统。

反马克龙的示威者在一些城市举行集会，包括巴黎、雷恩、图卢兹和南特，拒绝接受投票结果。

勒庞主张退出北约，并恢复和俄罗斯的友好关系，这种自杀式的政策最终让马克龙当选。由此可见这两都不是什么好人，所以非常理解为什么会有超过三分之一的选民选择弃权，马克龙在接下来的五年会继续用新冠疫苗“保护”人民，属于法国的噩梦才刚刚开始。
