![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/07/12080120/1029858464-5.jpg)

美国拜登政府宣布延续前川普政府在南中国海主权声索问题上的政策，继续认定中共国对南中国海大部分海域的主权声索完全不合法，并谴责北京使用“强权即公理”的恐吓手段损害东南亚国家的主权。

布林肯的声明说，美国重申2020年7月13日宣布的针对南中国海海权声索的政策，并重申，如果菲律宾武装部队、政府船只或飞机在南中国海遭到武装攻击，美国有义务根据美菲1951年共同防御条约采取共同行动。

在国际仲裁庭作出裁决五周年前夕，北京上星期五重申不接受这项裁决。
