## 安老界配合港府施压院舍为长者注射新冠疫苗
` 東京櫻花団-Tokyo Sakura Group` [轉載自GNews](https://gnews.org/zh-hans/1918004/)

作者：东京樱花团丨阿恩
![](https://lh4.googleusercontent.com/4rJA5LSucBBBfrgaqfrAbHbw4J6nXTPPecSWmrIVPvbUXevcA5ZirfZeRu7s6l_H0LKpEpQFtM1x9I_h4_neje6hEUqYDpapbriLPW7C57UHmR_WyO0oAql10fLpdzDQjX2kgVJ2)图片源自本文链接
香港安老服务协会执委李辉表示，与政府开会商讨推高院友接种率，包括探讨疫苗气泡，如长者没有接种，其家人不可探访、只可留在自己的宿位、亦不可参与活动。会上有人提议为赋权医生替同意接种的长者接种疫苗，家属不能提出反对。李辉虽然认为这些安排不合适，但仍需要在长者安全及公共卫生之间取得平衡。

从前的香港是个忙碌有序安逸的城市，每个人都可以照着自己心中的希望在这个城市生活，即使入住安老院的长者也有生存的盼望。港府一直漠视养老体系，畸形的楼价迫使许多港人蜗居，公立的安老院轮候时间超过三年，许多有需要长者只能自费先入住私人安老院，虽然院舍的生活枯燥，但定期家人的探访和社会组织前来问候，表演节目为长者解闷，送上小礼物哄长者开心，为长者们将逝去的人生增添不少喜乐。

中共病毒的阴霾在香港挥之不去，港府对安老院实施一波又一波没有人性的限制，最开始是长达数月禁止家人探望，期间许多长者至死也没见到自己的亲人；之后港府在长者中推新冠疫苗，没有人知道究竟有多少长者因此死亡，只有港府高官在媒体上欣然宣布“公立安老院轮候的时间缩短了三分之一”。如今港府依旧对长者的注射率表示不满，尤其是安老院的长者，声称只有两成的注射率。港府不断逼迫安老业界提高注射率，有人提出授权医生剥夺长者家属意见，直接替长者接种疫苗，虽然业界对此举有质疑，但仍然抗衡港府的防疫强权，将注射疫苗的责任全部推给安老院的医生，港府卸磨杀驴的表演炉火纯青。

这些被中共视为耗费港府资源的长者都是建设新香港的障碍，新冠疫苗就是杀人于无形的生物武器，未卜先知的港府提前五年就为港人预备了上千公顷的安葬所，目前已可提供60多万个风凉水冷环境优雅的廉租骨灰安置位，还有免费的绿色殡葬撒骨灰服务，时刻盼着港人速速为中共的新香港赴死。
