我是一个财经小白。七年前，我的旧同事介绍了迈克尔·马洛尼 (Michael Maloney)的油管视频。看了视频后，我才知道纸币和货币的区分。之前，迈克尔一直认为黄金与白银是保值的工具，所以大多数的视频是以金属为避险的途径。后来，他发表了关于虚拟货币技术的视频，来探讨虚拟货币跟金属的相似度和诚信度。
