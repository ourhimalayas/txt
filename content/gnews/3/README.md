首先，不讲普京先说一下俄罗斯的国家性格。

俄罗斯，全称俄罗斯联邦，简称俄罗斯、俄国。位于欧亚大陆北部的联邦共和国，国土横跨欧亚两大洲，拥有1,707万平方公里的面积，占地球陆地面积八分之一，为全世界领土面积最大的国家；它也是世界上人口第九多的国家，拥有1.44亿人口，77%居住于其较为发达的欧洲部分。**俄****罗****斯****拥****有世界上最大****储****量的****矿产****和能源****资****源，是世界上最大的石油和天然气****输****出国。**

由此可见俄罗斯的第一个特点是地广人稀，资源丰富。简单做一个类比，俄罗斯的人口和日本差不多，但是国土面积是日本的45倍。打个比方也就是说在日本一个人住一间房的话，一个俄罗斯人住45间房。每天换房子住需要一个半月。再加上俄罗斯的石油天然气甚多。**就造成它的第一个国家性格就是老是担心****邻****居****们****来侵占它的地****盘****，****惦记****他的能源。**俄罗斯从内心深处渴望在对它有威胁的大国之间一定要有缓冲地带，也就是普京说的中立非武装地区。因此历史上俄罗斯曾经一手制造了外蒙古的独立，还曾经和纳粹德国平分了波兰。1945年在日本投降之后迅速南下侵占北方四岛至今作为和日本的缓冲地带。

俄罗斯的第二个特点：俄罗斯位于欧洲东部和亚洲北部，大部分地区处于北温带和北寒带，温差普遍较大，1月平均温度为5度摄氏度到负50摄氏度（全世界有记录以来最低的温度之一）。**俄****罗****斯大部分国土****纬****度高冬季****严****寒漫****长****。因此在****军****事上俄****罗****斯需要有可以冬季出海的不****冻****港。**虽然俄罗斯的国土面积世界第一，但是俄罗斯可以走向大洋的出海口少之又少。
