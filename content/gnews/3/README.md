1月9日早间电视新闻消息：据悉，世界著名的电影导演张艺谋将担任下月4日开始的北京冬奥会开幕式的总导演。本次活动规模将比2008年小，而且将 “不使用人海战术”。

张艺谋以《红高粱》、《初恋之路》和高仓健主演的《千里走单骑》等电影而闻名，曾担任2008年北京奥运会的总导演。

在接受中共媒体采访时，他说：是因为我们的国家变得更强更大了，我们才能够在这么短的时间内举办两届奥运会。
