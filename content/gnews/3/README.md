中共国外交部发言人汪文斌周二（3月1日）在例行记者会上表示，尽管有国际制裁，中共国仍将与俄罗斯保持正常贸易关系。北京将继续进口俄罗斯天然气，同时反对国际制裁。

他表示：“中俄将继续本着相互尊重、平等互利的精神开展正常的贸易合作。”

发言人表示，中方的立场是反对任何“非法实施”的单边制裁，认为“制裁不是解决问题的一个有效办法”。

外交部周二的声明重复了周一新闻发布会上的声明。据中共国外交部网站的官方记录显示，汪文斌在回答路透社记者表示，中俄两国将“继续进行正常的贸易合作”。

周二的新闻发布会上，这位发言人还表示，中方对乌克兰冲突的立场没有改变。在回答有关人道主义援助的问题时，他说中国“准备好在缓解乌克兰局势方面发挥建设性作用”。

中共国常驻联合国代表张军周一在联合国紧急会议上说，通过直接对话和谈判来解决俄罗斯和乌克兰之间的冲突是可能的。

2022年初，俄罗斯总统弗拉基米尔·普京和中共国国家主席习近平发表联合声明，呼吁停止北约扩张。

俄罗斯上周四对乌克兰发动袭击，声称必须动用武力去停止乌克兰对顿涅茨克和卢甘斯克分离地区的袭击，并使该国非军事化和“去纳粹化”。基辅和西方国家谴责俄罗斯无端侵略的行为，对俄罗斯及相关寡头进行经济制裁。
