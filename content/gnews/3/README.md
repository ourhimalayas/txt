## 美国佛州反对儿童接种新冠病毒疫苗
` 扬帆枫叶快讯` [轉載自GNews](https://gnews.org/zh-hans/2139550/)

加拿大温哥华扬帆农场    平民正道

据【每日电讯】3月8日报道，佛罗里达州卫生局日前宣布，将正式反对为该州健康儿童接种新冠病毒疫苗。

报道称，在相关医疗专业机构的指导下，佛州政府认为，由于疫苗潜在的严重副作用，以及儿童感染新冠病毒并因此患重症的风险极低，因此，将停止儿童疫苗接种的推广。

拜登政府当天则对此表示，佛州的以上决定是在 “兜售阴谋论”，并仍然坚称接种疫苗是保护各年龄段人群免于感染甚至死亡的有效手段。

评论指出：随着疫苗无效及其健康灾难的逐渐显露，各地政府与医疗界都已纷纷开始改弦易辙以逃避即将到来的追责。而那些仍在欺骗公众的政客及专家们必将受到其国民最严酷的惩罚。
