自普金和CCP邪魔勾兑，在2月24日发动了入侵乌克兰犯罪战争。随着俄罗斯军队对基辅和其他乌克兰城市发动攻击和侵略，各个国家纷纷谴责俄罗斯独裁和霸凌。好多国家都表态支持乌克兰.

3月5日美国白宫发表一份正式声明对俄罗斯这种侵略进行制裁，其中之一就是把俄罗斯从国际结算SWIFT系统踢出去。这种金融经济制裁意义非常深远，它不仅及时对俄罗斯战争行为进行严厉回音，同时用SWIFT有效冻结了俄罗斯国家和普金个人资产来遏制战争会给无辜人们带来的极大伤害。

环球银行金融电信协会 (SWIFT) 是连接全球银行的重要管道。它是一个消息系统，允许银行快速安全地转移资金，支持数万亿美元的贸易和投资流动

该公司总部位于比利时，是一家由来自世界各地的金融机构拥有的合作社，该公司成立于 1973 年，旨在取代电传网络（一种发送信息的电传打印机网络）。 SWIFT连接了来自 200 多个国家的 11,000 多家银行和其他公司。

从之前伊朗进行禁止核武器和核试验，美国用SWIFT系统非常有效制约伊朗。2012.3月美国财政部对23 家伊朗银行列入黑名单，其中大部分也是欧盟的目标。美国立法者表示，因德黑兰正在利用其他20多家银行为其核计划提供资金并支持地区激进组，所以美国和一些欧盟国家利用金融手段对伊朗寻求发展核武器制裁。伊朗被提出Swift系统，伊朗石油出口被限制，从而导致伊朗经济极度下滑，没有外汇收入。伊朗自2012年财政一直严重亏损。

俄罗斯和邪恶CCP勾结为了独裁者个人权力和控制发动战争，最终受到国际金融制裁。 俄罗斯和邪恶CCP会像伊朗一样，出口受到限制，外汇受到管制。这样制裁会使独裁国家经济有效打击。
