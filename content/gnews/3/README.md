## 中共正派出最先进的战机在争议海域巡逻

据《CNN》记者：Jessie Yeung 和 Brad Lendon，2022年4月15日报道：
![](https://assets.gnews.org/wp-content/uploads/2022/04/image-2150.png)去年9月在中共国珠海举行的2021年中国航展期间，两架J-20隐形战斗机在空中表演。图片来源：CNN
香港《美国有线电视新闻网》——据中共国官方媒体报道， 中共国已开始派遣其最先进的战斗机J-20在东海和南海巡逻。

J-20 隐形喷气式飞机的早期版本使用俄罗斯发动机，但后来被国产双发发动机取代。去年，这些喷气式飞机在中共国航展上首次向公众展示了配备新型中共国发动机的飞机。

官方小报《环球时报》周三援引军事专家的话说， 此次部署旨在“更好地维护中共国的领空安全和海上利益” 。

据《环球时报》报道，J-20 国营制造商的发言人任玉坤补充说，现在 J-20 配备了中共国发动机，开始进行巡逻是一种“训练程序”。

几周前，美国太平洋空军司令肯尼斯·威尔斯巴赫将军表示，美国的 F-35 和中共国的 J-20 在东海上空相互靠近。

东海和南海长期以来都是有争议的地区，许多国家的领土主张重叠。

中共国声称几乎所有广阔的南海都是其主权领土。它一直在那里建造设施并将其军事化，将岛屿变成军事基地和简易机场，并据称建立了一支可以拥有数百艘船只的海上民兵。

与此同时，在东海，中共国声称对日本控制的尖阁诸岛（又称钓鱼岛）拥有主权。近年来，美国重申了在外来侵略时保卫日本岛屿的承诺。

专家表示，J-20 的部署表明了两件事：中共国对其军事能力的信心增强，以及对其他与领土争端有利害关系的国家发出警告。

“据报道，有大约200架 J-20 在服役，中国人民解放军空军（PLAAF）现在拥有一支与美国一样优秀的先进隐形战斗机机队定期服役，他们仍然是标杆。”访问学者彼得·莱顿（Peter Layton）在澳大利亚格里菲斯亚洲研究所表示。他补充说，中国向世界传达的信息是：“任何侵入中共国声称拥有的东海和南海领空的外国军用飞机现在都可能被歼20拦截。”

莱顿说，虽然这样的举动在政治上会令人担忧，但 J-20 的广泛行动范围意味着它可以在更远的海域巡逻，或者在东海等地区停留更长时间。

少量喷气式飞机等小型编队也可以偶尔深入南海巡逻，在中共国的一个岛屿空军基地着陆加油，然后返回大陆。中国人民解放军空军甚至可以飞越任何进入南海的美国航母战斗群。

莱顿说，从俄罗斯发动机向中国发动机的过渡也表明中共国在军事制造方面的独立性日益增强。“这不仅仅是中共国不再需要俄罗斯的帮助，而是中共国制造的飞机现在优于俄罗斯。”

莱顿说，J-20 现在拥有比俄罗斯更可靠的双引擎，这使得这些巡逻“成为一个更合理的选择”，这可能是他们以前没有被派去巡逻的原因。

J-20 长期以来一直被吹捧为中共国对美国 F-22（被认为是世界上首屈一指的隐形战斗机）和 F-35 的回应。

战略与国际研究中心中国电力项目 2017 年的一份报告称，新的中共国发动机将赋予 J-20 低超巡航能力，这意味着它们可以长时间以超音速飞行。

美国官员长期以来一直表示 J-20 无法与美国隐形战斗机相提并论——但美国太平洋空军司令威尔斯巴赫上个月表示，J-20 在与美国 F-35 交战时给人留下了深刻的印象。

“我们对与 J-20 相关的指挥和控制印象比较深刻。”威尔斯巴赫在米切尔航空航天研究所的一次会议上说。

他说，美国观察到中共国飞行员“相对专业的飞行”，但指出美国不确定中国将如何使用 J-20。

莱顿表示，美国、日本和其他国家将“积极收集有关任何 J-20 进行海岸巡逻的电子情报数据”，希望收集有关其隐身特性以及任何无线电和数据链传输的更多信息。
