代号“冷静反应”（ Cold Response ）的北约大规模防御性军演在挪威启动主体部分。 在俄乌战争已持续两周的背景下，北约此次军演受到格外关注。

挪威军方发言人表示，此次有德国和众多其它北约成员国参与的军演已从3月14日拉开帷幕，预定4月1日结束。本周主要演习活动在水上和空中进行。

“冷静反应”每两年在挪威寒冷季节举行一次。北约秘书长斯托尔滕贝格是挪威人。迄今，由挪威组织的该演习与取名“精彩跳跃”（Brilliant Jump ）的另一北约军演衔接。

据悉，参加本年度《冷静反应》演习的有来自27个北约国家约3万名官兵，包括陆、海、空部队。
