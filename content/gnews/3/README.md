## 俄对乌克兰医疗基础设施的攻击旨在粉碎乌克兰人的居住地及意志

![](https://assets.gnews.org/wp-content/uploads/2022/03/Picture2-3.png)

图片来自原文

俄军在哈尔科夫（Kharkiv）击中了一家血库，里面的捐赠者正在排队。集束弹药在大院里留下了类似迫击炮的小弹坑，而且–目击者说–杀死了一个出来抽烟的人。

在这之前，俄军在沙斯提克（Schastye）轰炸了电力和抽水站，导致整个城镇没有自来水，并迫使乌克兰百姓在炮击期间到水井边排队打水。

昨天，俄军在马里乌波尔（Mariupol）炸平了一个妇产医院–幸好已经疏散–据当地政府称，俄军还试图切断供水。

连同对居民区的无情轰炸，俄军向乌克兰其他城市的领导人发出了警告：你真的想让你的公民遭受这种待遇吗？

这些打击的目的是为了让城市变得无法居住，这种威胁似乎在赫尔松起了作用，俄罗斯在周三（3月2日）控制了乌克兰南部的战略性河流过境点。

该市市长伊戈尔-科利哈耶夫（Igor Kolykhayev）在脸书上说，”武装访客 “参加了一次城市行政会议–他同意了某些条件，包括宵禁和行人将以一到两人一组的方式行走，以保持城市运行。

市长伊戈尔-科利哈耶夫说，我没有对俄军做出任何承诺。没有什么可承诺的，我只是要求不要向人开枪。我们的城市里没有乌克兰武装部队，只有平民和想在这里生活的人！

克里姆林宫希望乌克兰其他城镇也能效仿。

乌克兰总统沃洛季米尔-泽伦斯基（Volodymyr Zelensky）昨天上午敦促乌克兰人保持勇气。他和他的人民希望俄罗斯人的士气在抵抗面前最终会崩溃–只要他们能再坚持一段时间。
