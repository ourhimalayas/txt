#### **撰稿：**日本东京方舟农场 子辰

![](https://assets.gnews.org/wp-content/uploads/2022/01/FF8003F7-A5C3-4842-84EA-ED98E0F838DD.jpeg)（图片来自《美国之音》）

据《美国之音》1月5日报道，朝鲜进行了一次导弹试射。而在前不久的新年讲话中，金正恩还谈到要把重点放在国内经济困难方面而不是军事发展上。

韩国联合参谋本部周三在一项声明中说，朝鲜当天清晨向其东部海岸附近的海域发射了至少一枚抛射体。日本海上保安厅也说，这次发射可能涉及一枚弹道导弹。

随即，美军印太司令部发表声明：“我们了解这次的弹道导弹发射，并在与盟友和伙伴们密切磋商。虽然我们认为这次发射对美国的人员或领土，以及对我们的盟友不会立即构成威胁，但它突显了朝鲜非法武器项目对地区稳定的损害。美国对保卫韩国和日本安全的承诺依然坚持有效。”

此前，金正恩在新年到来之际的讲话中，重点谈到了他要提振困难重重的朝鲜经济的计划，以及国际制裁、自然灾害和新冠疫情严重冲击了朝鲜经济的问题。并没有释放多少外交政策方面的信息。

截止发稿前，尚未得到进一步关于朝鲜此次发射导弹的细节内容。包括发射了多少枚导弹以及射程多远和溅落位置。
