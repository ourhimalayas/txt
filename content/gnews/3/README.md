![](https://assets.gnews.org/wp-content/uploads/2021/11/image-444.png)图片来源：美国新闻与世界报道
11月16日，《美联社》对美国乔·拜登总统与中共总书记习近平之间的视频会议作了雾里看花式的报道。

美联社基本引用中共外交部和中共国内喉舌的话语来评价此次视频会晤，如引用中国外交部发言人赵立坚的话说：“此次会晤是坦率和建设性的。中美关系如果回不到过去，就应当面向未来。”

美联社继续之前的立场，抨击中美关系的恶化源自于川普白宫时期，认为此次会晤是“旨在结束两国关系的恶化”。也正如其所说，在会议开始时，拜登毫无骨气的说：“正如我以前说过的，在我看来，作为中国和美国的领导人，无论是有意还是无意，我们的责任是确保我们国家之间的竞争不会转为冲突”。作为回应，习近平在开场白中呼应了拜登的亲切语气，说：“中国和美国需要加强沟通和合作”。

对于台湾问题，美联社只是引述中共方面发表的声明，指习近平将台海紧张局势归咎于台湾谋求独立而寻求美国的支持，以及美国方面一部分人利用台湾来遏制中国。

对此，白宫国家安全顾问杰克·沙利文则为拜登打圆场，说拜登提醒习近平，他作为参议员时投票支持1979年生效的《台湾关系法》。

虽然报道说白宫方面表示，拜登重申美国将遵守其长期奉行的“一个中国”政策，但美联社却解读说该政策是承认北京是中国政府，但允许与台北建立非正式关系和国防联系，却没有解读“一个中国”的真正含义是模糊定义的，可以指北京政权，也可以指中华民国等。

对于当前美国紧迫急需要解答的中共病毒（COVID-19）真实起源问题，美联社也没有特意对此进行评论，但却字里行间透露了一个细节，说在今年6月份，当记者问拜登会否迫使他的老朋友习近平配合世界卫生组织（WHO）对冠状病毒起源的调查时，拜登却勃然大怒。

综观美联社对此次视频会议的报道，对于与中共国的关系，正如参议员乔尼·恩斯特（Joni Ernst）所说，她认为国家领导人之间确实需要对话，如简单的握手和问候，但这不适合于与共产主义中国的关系。“我们知道他们是邪恶操纵者和行为者，我们需要对他们进行反击，我们许多盟国和世界上其他国家都期待美国的领导，但乔·拜登在所有方面都无所作为”。

而究其原因，也正如11月18日美国前国务卿彭佩奥在盖特发文所讲，拜登之所以表现的软弱，是因为中共一直将美国视为敌人，但拜登却没有这样做。

但最根本的原因，正如郭文贵先生所言，任何不涉及中共病毒的首脑会议都是在作秀，都是勾兑的结果。
