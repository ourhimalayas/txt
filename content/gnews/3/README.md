## 日银问卷调查结果显示，77.4%民众深感价格上涨

![](https://assets.gnews.org/wp-content/uploads/2022/01/pasted-image-0-12.png)图片源自本文链接
据日本银行2021年12月的 “生活现状问卷调查”结果显示，77.4%的人表示，与前一年相比，物价上涨了。这是自2015年12月调查（78.8%）以来的最高水平。调查表明，消费者已经开始感受到因日元疲软而导致的原材料和进口商品价格上涨的影响。

该调查每三个月进行一次，以确定个人对商业条件和价格的看法。

在去年12月的调查中，有2216人给出了答复，其中16.6%的人认为价格大幅上涨，60.8%的人认为略有上涨。当被问及价格变动幅度时，回答为平均6.3%，这也是自2009年3月（6.4%）以来的最高数字。
