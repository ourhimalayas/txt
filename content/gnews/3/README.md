由于印尼历史上多次的暴力排华，目前很多先知先觉的华人已经准备撤离。当年印尼血腥排华的时候，很多华人在家里无辜地被杀、被强奸，有些浑身是血的小孩子跑到中共驻印尼大使馆寻求庇护。结果，这些孩子被中共使馆人员以不干涉印尼内政为由全部轰出。这些孩子被轰出去的结局就是死，在当时中共驻印尼大使馆西侧墙的台子上被强奸、杀害，年龄最小的女孩子只有7岁。

那时的大使馆已经有监控录像设备，所有工作人员都只是看着从大使馆出来的华人有的当场被砍头，有十多岁的女孩在被强奸、轮奸后，被用棍子插进肛门从胸口穿出挑放在大使馆门前。印尼暴徒们用惨绝人寰的手法告诫那些妄想获得中共大使馆保护的华人，找大使馆没用的，中共政府不会关心华人的死活。

至于目前有些在乌克兰留学生被中共所救，某种程度上是因为新中国联邦切实有效的救援行动促使了中共撤回部分学生，但是这些学生分别要付出的代价是1.8万人民币的高昂机票，加21天、7000人民币的隔离费。即便是这，党媒还要求留学生们要知足并感恩祖国，这才是能平安回家的同胞们应有的态度。
