我是每天都面临着很多人在国内战友的人问我，体制内的人在问我，世界上各国人都在问我，好像我真的很懂似的，好像我有权力做这些。首先我只代表我本人，我没有权力告诉大家正确的答案，因为每个人都身处环境不同，而且也不等同于我是爆料革命的创始人我就有资格提供什么你们要遵守的方案，完全不是啊。你们一定要独立的思考，独立的判断。就我本人而言，我是做好了长期的适应一个新的人类的生活方式的准备。在我这灭共之前，我一切都是，反正我想的就是一切都灭共，怎么安全地、怎么有效地灭共。2025年以前我就想这个，2025年以后我要想的事情真的是隐居山林。

最近我们看到国内很多战友说：“七哥，我听到你的话了，从大城市——京上广回到了县城，回到了老家。” 回到了县城老家，随之而来的就是发现了“锁链女”事件就层出不穷。这不是今天发生，也不是几个月，从去年很多战友跟我说。就是各个县城和城镇人那个无知和那个利益保护形成所谓几千年的村、族利益保护，几个家族承包了当地的公检法、税务、工商，所有的连什么粮局、卫生局、工商局，就是个县镇的十字路口，你别人都插不进来，都是已经是霸王，已经是全都是圈地形式给它搞起来了。像我们安徽这位战友回家以后，他说：“七哥，我从清华毕业的，回来给我安排了一工作，一来叫我到法庭去，说给我弄了一个叫什么法庭助理。我也花了钱了，我家有点后台，但是跟另外一个人形成冲突了在考核当中。我现在就进入了一个所谓我老家县城两大派系的斗争”，他说：“要比在北京那个要残酷得多，而且这里没有底线，你死我活。” 他说：“你能想象吗？我亲眼看见法官把一个律师，就在法庭把人撵出去以后，直接在法庭就给强奸了，我就听到这个律师大叫。” 他说：“你说我，我就犹豫，我是冲进去？我还是装听不见？我就给我家亲戚打电话，我亲戚说你找死呢，这事儿成天发生，你敢喊喊试试？你肯定被消灭，你肯定被车祸。”他说：“我就只有闭嘴，七哥，但这种事后来我发现我已经习以为常了。”

他不是法官强奸律师，他法警也敢强奸律师，他为什么？律师全拿人家钱，当法官和法警一发现他俩之间有勾当、有把柄的时候，直接就把你强奸，就办了，然后告诉你：“你想不想要律师执照？你想不想继续执照？你想要你就给我闭嘴！” 他说：“这个县城就这么黑暗，我当一次英雄，我没了，下一个事继续下去。”所以说现在生活方式，他说：“我不能再待在县城了，我到大城市去我可能还能把我安全起来，我待在这儿我成了枪靶了，所有人都想干掉我，但是现在我发现每个人都想杀我的感觉。”我说：“那你赶快回去离开吧。”

所以说弟兄姐妹们，你知道在中国是什么灾难？中国就是越底层越不适合人居住，除非你就不把自己当人，你进入到那个原始时代，是吧？是姐姐也可以睡，嫂子也可以睡；然后就坚信，所有的共产党高官相信的：只要不和自己上床的女人就不能相信的原则，都得搞上床。
