## 俄罗斯向伊朗寻求航空制裁方面的援助

![](https://assets.gnews.org/wp-content/uploads/2022/03/image-2731-edited.png)2021年9月16日，俄罗斯航空公司的空客A320飞机在莫斯科谢列梅捷沃国际机场 … [+] Sopa images/lightrocket via getty images

莫斯科正在期待伊朗帮助它找出如何应对对其航空部门的削弱性制裁。

3月22日，交通部长维塔利-萨维利耶夫在俄罗斯议会上院经济政策委员会的一次会议上说：“俄罗斯正在接受伊朗在类似情况下如何为飞机提供服务的经验的指导”。

德黑兰当然有很多经验，几十年来，它一直是美国和其他国际制裁的目标。然而，它的记录表明，它的建议对莫斯科的作用可能有限。

多年来，伊朗一直被阻止从国际制造商那里获得备件或新飞机，但该国的航空公司设法保持飞行，主要是通过使用一些飞机作为备件。2018年，在其约250架飞机中，约有100架飞机停飞，因为它们要么出现故障，要么被拆解为零部件。

当伊朗有机会购买西方喷气机时，它就抓住了这个机会。在2015年签署核协议后国际制裁的短暂停滞期间，伊朗的航空公司向空中客车、波音和ATR订购了300多架新飞机。然而，在当时的美国总统唐纳德-川普于2018年退出核协议并重新实施制裁之前，只订购了少数飞机。

此后，它四处寻找西方飞机的替代品，但发现任何交易都难以完成。2018年宣布了两家伊朗航空公司–阿斯曼航空公司和伊朗航空旅游公司–从俄罗斯购买40架苏霍伊超级喷气式飞机的协议，清单价格约为20亿美元，但由于制裁，该交易失败了。伊朗甚至开始研究开发自己的客机，但该项目似乎受到了资金短缺的影响。

在没有任何新飞机可用的情况下，伊朗不得不转向二手市场，购买更老的飞机。3月16日，伊朗民航组织（CAOI）负责人穆罕默德-穆罕默德-巴赫什（Mohammad Mohammadi-Bakhsh）说，在过去三个月里，伊朗的航空公司购买了10架平均机龄为15-20年的客机。目前还不清楚这些飞机的来源。

制裁切断了俄罗斯的空中通道

俄罗斯在2月入侵乌克兰后对其实施的制裁比对伊朗的制裁影响更深远。它们包括禁止俄罗斯飞机使用美国、加拿大、欧盟和英国的空域和机场。此外，飞机制造商已停止交付新飞机或提供备件或服务。

俄罗斯航空公司使用的500多架飞机是从爱尔兰或百慕大的公司租赁的，但这些国家的监管机构已经暂停或取消了它们的适航证书。作为回应，弗拉基米尔-普京总统已经签署了一项法律，允许俄罗斯航空公司将这些飞机放在当地的登记册上，然后由其颁发适航证书。

国际租赁公司已经发现自己无法收回资产。萨维利耶夫在3月22日说：“我们正在寻找与出租人谈判并解决这一问题的法律途径，但到目前为止，我们还没有成功做到这一点。但我们并没有失去希望，我们也不会交出任何东西。这样做将意味着让我们没有[一个]机队”。

萨维利耶夫说，迄今为止，近800架飞机–在大约1367架飞机的俄罗斯机队中–已经被转移到俄罗斯的注册地。至少有一些在俄罗斯注册的飞机没有在其他地方取消注册，这违反了国际航空规则。爱尔兰航空管理局已经发现至少有11架波音飞机现在是双重注册，包括俄罗斯航空公司和阿尔罗萨公司的飞机。

国际网络

由于制裁，俄罗斯航空公司的国际航班受到了严重的限制。3月8日，国家航空公司俄航因制裁而暂停了所有国际航班，但到白俄罗斯首都明斯克的航班除外。

本月早些时候，该国第二大航空公司S7也暂停了所有国际航班。其他航空公司包括Smartavia和俄航的低成本子公司Pobeda也采取了类似行动。

然而，最近几天，俄罗斯国际航空公司已经开始重新拼凑其国际网络。3月14日，它恢复了飞往吉尔吉斯斯坦比什凯克和奥什的航班，一周后开始再次飞往阿塞拜疆首都巴库。从4月2日起，它将重新启动飞往德黑兰的航班。
