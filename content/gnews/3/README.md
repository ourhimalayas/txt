中国上海被誉为“超一线国际化“大都市，曾经的它高楼大厦，灯火辉煌，然而这一切都随着中共政府的封城防疫政策而不复存在。看到互联网上被封锁下的上海的种种惨状，您是否觉得很不可思议？上海正在上演一段史无前例的荒唐乱象。上海人的骄傲和尊严被邪恶的中共给拿走了，留给上海人的是饥饿、恐慌甚至是死亡。这就是中共口中的“文明，礼仪，安全”的国家。

历史曾记载，1807年第一位传教士马礼逊进入中国，而当时处于清朝的闭关锁国时期，传教工作困难重重。当时的马礼逊传教士的主要工作是学习汉语，并将圣经翻译成中文。但是在1826年，当时在位的道光皇帝将之前禁止巫术、妖术的法律修订后再次颁布，新的法律禁止传播基督教，违者将被判以死刑。已经入教的中国人若不放弃他们的信仰，会被发配到新疆做奴隶。从此中国又与西方文明远离。

19世纪早期，西方文明国家已经开始把英文作为国际交流的语言，在此期间，这些国家一直将彼此视为世界上最重要的盟友。由此可见，西方的文明和团结程度远远领先那时的中国。

而早在距今6000多年以前，上海最早出现的文明是马家浜文化，接着是崧泽文化（据今5500年），然后是良渚文化（距今4200年），它的晚期文明程度已经相当发达，尤以玉器最为著名。作为江南地区文明有机整体的一部分，上海早期成为大陆文明的先锋，具有相当的领先水平。

1860年以后，上海开始进入初步的繁荣时期。《北京条约》签订以后，西方国家都纷纷选择上海作为他们在中国的贸易基地，并设立了领事馆。

从1858年《天津条约》签订到1906年，随着鸦片买卖合法化，上海成为当时中国最主要的鸦片输入港。

曾经上海十分有名的圣约翰大学，在1887年开始完全使用英语授课，这是中国首座全英语授课的学校。到了1952年由于中共的政策是模仿苏联的教育体系，大量削减综合性大学，改办专门学院，原有教会学校多数被拆分。圣约翰大学硬件设施也被拆散并入上海多所高校，按照并入院系数量排序，依次是华东师范大学、复旦大学、同济大学、上海财政经济学院、华东政法学院、上海第二医学院。其中文、理学院主要并入华东师范大学和复旦大学。圣约翰医学院与震旦大学医学院、同德医学院合并成立上海第二医学院（后改名为上海第二医科大学，2005年改为上海交通大学医学院）。

原本与西方教育和文化已经逐渐结合的良好形态，却因为50年代的中共军管——“三反”、“五反”运动而终结。该运动导致仅上海一处自杀、中风与精神不正常的人数不下万人。这场运动实则是中共制造的一场惨无人道的大屠杀。

当“三反”、“五反”运动在上海掀起浪潮时，上海已经注定一夜回到蛮荒时代。先前所有融入的文明、教育和信仰统统被砸烂，共匪正式走上奴役上海老百姓的社会主义道路。

之后在以恶魔毛泽东为首的中共领导下，通过发动文革、土改，然后对百姓进行洗脑，一切要听从党，鼓励百姓互相举报，让百姓内斗，甚至让百姓人吃人。利用人的自私和想要存活的本能，把人性的丑恶面发挥到极致，从而达到让百姓看见党就害怕的目的。因为只有这样它才能更好的奴役百姓。

中共通过控制媒体和网络屏蔽真相，让谎话对老百姓重复70年，利用商鞅驭民五术——贫民、壹民、疲民、辱民、弱民统治老百姓，让百姓失去自信与反抗的意识和能力。其中壹民就是让百姓只有一种思想，那就是统治阶级需要的对其有利的思想。

上海的现状就是最好的例子，人与人之间毫无信任感，有的只是自私和冷漠。当乌克兰发生战争时，乌克兰人民拿起武器，团结一致，以死抵抗俄罗斯的入侵。而中国百姓遇到封城，却老实待在家里等待物资，无脑地相信政府讲的每一句话，甚至宁可饿死也不去反抗中共政府的纳粹屠杀式的防疫措施。这样鲜明的对比，让人悲哀。

但随着科技的发展，社交媒体已经成为最大的核武器。真相因为文贵先生的爆料被逐渐揭开，中国人是有大智慧的。中共对我们洗脑长达70余年，邪恶笼罩在中国大地70余年。文贵先生唱的好：“醒来吧，别再如此冷漠；醒来吧，抛掉所有懦弱。”这次的疫苗灾难带给人类的伤害难道还不够多吗？中共给我们的枷锁难道还不够深吗？
