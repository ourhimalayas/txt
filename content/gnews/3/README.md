## 维州《大流行病法案》通过，7名执政党议员退党
` 澳喜农场` [轉載自GNews](https://gnews.org/zh-hans/1725836/)

撰稿：Runaway
![](https://assets.gnews.org/wp-content/uploads/2021/12/维州议员.jpg)网络图片
备受争议的《公共卫生与福利修正案（流行病管理）》于上周四下午以20票对18票在维州议会上议院通过了表决。这意味着，从12月16日维多利亚州的紧急状态到期后，州长和卫生部长将有权宣布大流行并实施限制措施。

该法案在议会辩论期间引起了墨尔本民众数周的大规模抗议活动，甚至在执政的工党内部也出现了分歧。在上议院最后的表决中，工党不得不在最后一刻做出大量改变并依靠4名跨党派议员的支持，才通过了该修正案。

工党议员Jane Garret在对州长Daniel Andrews处理大流行病的政策表达了强烈反对之后，宣布了将退出工党的决定，她是最近一周内第七位宣布退出工党的议员。

执政的工党连续失去了七位议员，表明安德鲁斯政府大流行以来的隔离、强制疫苗接种以及种种破坏人权的政策不但不得人心，而且渐渐失去了党内的支持。一意孤行的“独裁者丹”还能走多远，我们拭目以待。
