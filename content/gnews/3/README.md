![](https://assets.gnews.org/wp-content/uploads/2022/03/image010.jpg)美国得克萨斯州米德兰附近的石油相关设施
随着俄军入侵乌克兰进入第六日，尽管西方国家努力避免打乱能源流动，但制裁俄国的行动仍被市场视为扰乱供应。原油期货价格周二(1 日) 大涨，

纽约原油期货市场在3月1日大涨，4月交割的美国标准油（WTI）原油期货价格大增8%，登上2014 年7 月以来最高收盘价。指数当晚在加时交易中暂处于每桶109美元区间。在俄罗斯入侵乌克兰引发的激战背景下，小麦和黄金期货也有所上涨。

WTI 每日收盘价为每桶103.41 美元，较前一交易日上涨7.69 美元。人们越来越担心俄罗斯将削减其原油供应以应对欧洲和美国的经济制裁。国际能源署成员已决定协调释放石油储备，但对市场的影响有限。由于担心供需紧张，俄罗斯的主要出口产品小麦也被购买。

即使国际能源署(IEA) 宣布，其成员国已同意自紧急储备中释出6000 万桶石油，以缓解俄军入侵乌克兰后造成的供应短缺，但是，原油期货价格仍持续走涨。
