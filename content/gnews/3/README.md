## 从西安集中隔离看中共体制之恶

作者：波士顿五月花 淘喵先生
![](https://assets.gnews.org/wp-content/uploads/2022/01/20220101-1.jpg)
12月31日，中共国陕西省卫健委发布消息称，自12月9日以来，陕西省中共病毒（Covid-19）感染累计确诊病例1475例，其中西安市1451例。

因为世人皆知，中共国官方数据一向是好的使劲儿往大了说，坏的使劲儿往小了说。所以从陕西伪政府公布的数据可以想见西安疫情的严重程度。

面对本轮疫情，西安伪政府的应对办法还是老一套：强制隔离。可是习惯于拍脑袋决策的中共伪政府官员却不知道这次疫情不同以往之处。

以前出现疫情，主要是中共病毒（Covid-19）爆发引起的，共匪采取的灭绝人性的强制隔离措施，对于隔绝病毒传播确实有着显著作用。

但这次不同。因为本轮疫情是在西安近1300万人中95%以上都已打了毒疫苗、而且是高抗原含量毒疫苗的情况下爆发的。这就意味着西安此次疫情是高浓度毒疫苗本身引起的，随着时间的推移，打了毒疫苗的人中会有越来越多的人将会爆发出症状。

因此将所有人进行强制隔离，对于减少病例几乎毫无作用，只会徒增西安市民的生活成本，甚至重现饿死人的惨剧。但是对于中共邪教体制下的官员来说，绝对是只敢照方抓药，“别人咋干咱咋干”。强制隔离是他们唯一的选择。

因此，西安当地伪政府的抗疫措施完全是乱打仗，干着“捏着鼻子哄眼睛”的鬼事。对于打了高浓度毒疫苗的一千多万西安市民来说，如果不能及时获得解药，简单粗暴的强制隔离只会导致更为恐怖的局面，其后果简直不堪设想。

与此同时，由于中共国有着极高的毒疫苗接种率，而且接种同种高浓度疫苗的地方应该不止西安一处。随着西安疫情的首先爆发，其他注射同样疫苗的地方一定会接二连三的爆发。到最后，整个中共国很有可能成为一个骇人的大疫区！无法想象那时将会是什么样的人间惨剧。

作为没有打毒疫苗的爆料革命一员，唯有祈祷共匪灭亡前的愚蠢和疯狂不要带走太多的同胞！
