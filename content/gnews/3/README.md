## 人口普查：COVID推动了一个世纪来美国死亡人数最大飙升
` 西班牙巴塞罗那喜悦农场` [轉載自GNews](https://gnews.org/zh-hans/2224405/)

编译：七叶之芒

![](https://assets.gnews.org/wp-content/uploads/2022/03/xin_png.001-2.jpg)

根据美国人口普查局最新发布的数据，COVID-19推动了一个世纪以来最大的死亡高峰，2020年比2019年多出53.5万人。

![](https://assets.gnews.org/wp-content/uploads/2022/03/image-2837.png)图片来源：Sarah Grillo/Axios

**它为什么重要：**新的数据显示了这一流行病对美国人口产生了多么深刻的影响，因为美国人死亡或逃离城市，到更便宜或人口少的地区避难。

**我们关注的是：**奥扎克、卡茨基尔和波科诺斯是拥有季节性住房的目的地之一，在2020年中期和2021年中期，迁入–或决定留下–的人数显著增长。包括哥伦布和印第安纳波利斯在内的大都市郊区的各县也出现了增长。

- 与此同时，纽约-纽瓦克、圣何塞-旧金山-奥克兰和洛杉矶-长滩等价格昂贵、人口众多的地区经历了明显的“国内净移民减少”。
- 这意味着更少的人迁入或更多的人迁出或两者兼而有之。这三个大城市地区的国内净移民下降幅度是所有其他类似地区中最大的。
- 主要城市地区–那些拥有100万或更多居民的地区–在这一年里总共失去了人口。布鲁金斯学会的比尔-弗雷说，这是十年来第一次发生这种情况。


**从数字上看：**在2019年和2020年之间，美国的死亡人数有19%的跳跃。在此之前，十年来的最大增幅仅是2015年的3.3%。

- 根据该年最新的临时数据，2021年美国的死亡人数仍然很高，这场大流行病打乱了曾经可预测的季节性死亡趋势。


**在****2020****年中期和****2021****年中期，**死亡率的整体上升促使美国73%以上的县的死亡人数超过了出生人数，这是一个历史性的高点，高于前一年的56%和2019年的46%。

- 半数州的死亡人数多于出生人数，这种现象被称为“自然减少”。根据人口普查局的数据，这一趋势在东北部和南部尤为明显。
- 特拉华州、缅因州、新罕布什尔州和罗得岛州的每个县都经历了自然减少。


**是的，但是：**与此相反的是，在2020年7月开始到2021年6月底期间，美国近五分之三（58%）的县经历了人口增长。

- 这在很大程度上是由于人们避免了更拥挤的城市地区，而是留在或前往该国人口较少的地区。
- 在许多县，这些决定有助于抵消指向人口下降的更广泛的人口趋势，如死亡人数激增、出生率放缓和美国国际移民减少。


**字里行间：**美国三分之二的县在2020年中期和2021年中期经历了正的国内净移民–这意味着他们看到更多的人留在或搬进去，而不是离开。

- 这一比例比前一年的46%有所提高。
