俄罗斯与乌克兰结束第三轮谈判，乌克兰代表随后称谈判取得小幅进展，俄罗斯则说谈判未满足其所提出的条件。然而，俄罗斯再提停火要疏散平民，对此乌克兰总统泽连斯基则表示质疑。

此前，俄罗斯军方在7日已宣布在乌克兰开放多个 “人道走廊”，但由于俄罗斯公布的路线中，有几条路线是将乌人民疏散至俄罗斯或白俄罗斯，此提案遭乌官方否决，并批评俄罗斯的举动完全不道德，强调乌克兰人民应该有权利被疏散到乌克兰境内。乌总统泽连斯基同日指责俄罗斯军队阻碍了乌克兰平民的撤离行动。他在双方结束会谈后，于Telegram上发布一段影片，表示：“双方有达成一个关于人道走廊的协议，但这个协议起作用了吗？俄军的坦克、多管火箭发射器与地雷都仍在持续运作”。泽连斯基重申，他仍待在基辅，且他对目前的情势并不感到害怕。

此外，泽连斯基周一也与美国犹太人组织主席会议进行电话会议，并把俄军的行为与纳粹做比较。他指责俄军队不让乌克兰平民疏散，甚至扣留他们的食物与水。他说：“这些情况在纳粹时期都发生过。乌民族数以百万计的人都将被灭绝” 。

联合国负责人道救援事务的副秘书长格里菲斯周一在联合国安理会紧急会议上说：“马里乌波尔、哈尔科夫、梅利托波尔和其他地方的平民迫切需要援助，特别是拯救生命的医疗用品。”美驻联合国大使格林菲尔德在会议上说，世界应该为“非常漫长和非常困难的道路做好准备”。她补充说，普京显然愿意牺牲成千上万俄罗斯士兵的生命来实现他的个人野心。
