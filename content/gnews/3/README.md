![](https://assets.gnews.org/wp-content/uploads/2022/03/image-1470.png)[图片来源](https://gnews.org/wp-admin/%E6%97%A5%E5%AA%92%EF%BC%9A%E9%B4%BB%E6%B5%B7%E9%9B%86%E5%9C%98%E8%81%AF%E5%90%88%E7%99%BE%E5%AE%B6%E6%97%A5%E6%9C%AC%E5%BB%A0%E3%80%80%E5%BC%B7%E5%8A%9B%E6%8E%A8%E9%9B%BB%E5%8B%95%E8%BB%8A%E4%BA%8B%E6%A5%AD)
根据《日经亚洲评论》报导，鸿海电动车联盟中目前预计约有100家日本公司将参与联合开发，其中不乏与传统日本车厂关系密切的供应商，此举将对汽车行业产生重大影响。

鸿海是在2020年10月宣布了其EV电动车计划。集团内的相关核心公司都将共同开发EV车身和通信基础设施。其他参与的公司将销售鸿海根据合约生产的电动汽车零组件和系统，并免费使用设计资料。透过这些基础软件和相关底盘设计，将可以为鸿海计划中的目标市场客制化相关产品。

截至3月初，约有2200家公司打算参与这个由全球最大的电子代工制造商领军的项目。与2021年4月该计划启动时相比，参与企业的总数增加了70%，而日本公司的数量则是一口气增加了5倍，这其中，一些公司已经开始与鸿海和其他参与企业一起开发零组件。

日本企业中日本电产、NSK 和其他汽车零部件制造商以及包括旭化成、昭和电工和东芝在内的主要材料和电子供应商都将参加；以旭化成来说，目前正在准备为其联合开发的动电车内部装饰材料提供材料。

而与日本主要汽车制造商关系密切的公司也参与其中，如丰田供应商电装、捷太格特和丰田通商、以及与本田汽车关系密切的FCC等。

鸿海从手机代工到汽车代工，再到自我进化，想要抓住新能源汽车发展的关键节点一跃而上，实现转型，以摆脱苹果代工对鸿海的束缚。在鸿海强势插入未来产业的过程中，台湾企业也都在实现着质的飞越，文贵先生大直播中提及台湾将成为亚洲未来的重点，这也将为台企插上飞翔的翅膀，实现企业和台湾的逆势崛起。
