## 乌克兰马里乌波尔的死亡人数可能超过2万人

![](https://assets.gnews.org/wp-content/uploads/2022/04/image-1827.png)

（图片来自原文）

2022年4月11日周一，乌克兰港口城市马里乌波尔 (Mariupol)的市长瓦迪姆-博伊琴科（Vadym Boychenko）说，俄罗斯对该城市的围困，导致超过10,000名平民死亡，死亡人数可能超过20,000人，街上到处都是尸体。

俄罗斯军队将移动火化设备运到马里乌波尔处理尸体。该设备以卡车的形式抵达。打开它，里面有一个管道，这些尸体被烧掉。

他指责俄罗斯军队拒绝允许人道主义车队进入该市，试图掩盖大屠杀。

乌克兰总统弗拉基米尔·泽连斯基(Volodymyr Zelensky)在其最新的国际支持请求中告诉韩国议会，马里乌波尔有数万人死亡，这一数字没有得到独立证实。但即使如此，俄罗斯人也没有停止他们的攻势。

俄罗斯声称摧毁了几个乌克兰防空系统，包括击中了中部城市第聂伯罗附近的四台S-300发射器。

周一，乌克兰表示，预计俄罗斯将很快发动一场新的大规模攻势。因为在其入侵部队被赶出基辅城门后，莫斯科将重点转向夺取东部地区的领土。

这似乎是俄罗斯要重新获得空中优势，在乌克兰东部广泛的新攻势之前，摧毁基辅称为关键的武器。

另外，顿巴斯地区以外的哈尔科夫遭到了猛烈的炮击。

市长伊霍尔-特列霍夫（Ihor Terekhov）说，造成多人伤亡，包括一名儿童被杀。
