根据大纪元3月19日讯，因中共病毒疫情在各地爆发，大陆多地采取封锁措施，民众不允许出门，影院被迫歇业。

截至３月16日，大陆影院营业率已跌破60%，约有5000家影院暂停营业。

目前上海、吉林影院基本停业，辽宁、陕西、江苏等省份影院营业率低于20%。

有媒体在《从电影票房看中国经济的可能性》的文章中分析说，电影市场规模庞大，关联着诸多其它行业，是衡量一个国家社会经济发展水平和文化软实力水平的重要考量因素。所反映出的任何衰退和下滑都有可能成百上千倍地扩大反映在国家经济层面上。

中共当局面对疫情所采用的“清零政策”严重打击了大陆经济，服务行业被严重冲击更是不能幸免。

评论人士文小刚认为，房地产和疫情是影响大陆GDP增长的两个重要变数。如果当局不像其它国家那样改变“清零政策”，大陆经济就不会好转。
