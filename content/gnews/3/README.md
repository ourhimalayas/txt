以下原文內容：

美国总统拜登说，俄罗斯总统普京或因入侵乌克兰自我孤立后，将部分幕僚软禁起来。

据法新社报道，拜登周四（3月31日）首次就俄乌战事导致克里姆林宫内部关系紧张发表上述看法。

针对美英情报显示普京在俄军表现和俄受制裁影响等问题被军方和幕僚误导的提问，拜登说：“他看似自我孤立，并有迹象显示他已开除或将部分幕僚软禁起来。”

拜登尝试淡化其言论，称“（目前）有很多揣测”，因此不愿“过度解读（当下情况）”。

不过，美国国务院发言人普赖斯过后说：“我们看到报道指有好几名俄罗斯官员被雪藏，疏离权力中心，就像总统所说的被软禁。”

普赖斯说，俄罗斯显然没料到会在俄乌战事中遇到顽抗，也对西方的制裁感到吃惊，这再次证明普京因被误导作出错误决定。“从他过去五个星期的决策，不难看出有好些失算情况。”

拜登也质疑莫斯科声称已减弱在乌克兰的军事行动。他表明，无法确定俄罗斯此说法的用意。

他说，下令从任何城市撤退，将允许部队在乌克兰另一地区加强攻势，如乌东顿巴斯地区。“目前没有明确证据显示他（普京）从基辅撤出部队，但有证据表明他在顿巴斯增强兵力。”
