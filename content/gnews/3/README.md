《路德时评》2021年6月28日晚间节目主要内容：

一、《中俄睦邻友好合作条约》背后的荒唐与陷阱

网易新闻6月28日报道[\[1\]](https://www.163.com/news/article/GDJJF0CG0001899O.html)：国家主席习近平28日下午在北京同俄罗斯总统普京举行视频会晤。两国元首发表联合声明，正式宣布《中俄睦邻友好合作条约》延期。

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/29003732/%E4%B8%AD%E4%BF%84.jpg)[图片地址](https://lh3.googleusercontent.com/proxy/MyPA_-Ilz2-bwc3SLG6BV0PhzcK6bgEdeOV15xlZd5dLf77ZX7xmfqBdXJ1YwBPGhb7GNfgwldp-oZDd9ugHCA4Mx7nQ-jGwtyZdibnnl6Tig3fspEyb9oh7kz4qqtG6xFvw)

路德社评论：

1、《中俄睦邻友好合作条约》于2002年2月28日正式生效，该条约第二十五条写道：本条约有效期为二十年。如果在本条约期满一年前缔约任何一方均未以书面形式通知缔约另一方要求终止本条约，则本条约将自动延长五年，并依此法顺延。[\[2\]](https://baike.baidu.com/item/%E4%B8%AD%E5%8D%8E%E4%BA%BA%E6%B0%91%E5%85%B1%E5%92%8C%E5%9B%BD%E5%92%8C%E4%BF%84%E7%BD%97%E6%96%AF%E8%81%94%E9%82%A6%E7%9D%A6%E9%82%BB%E5%8F%8B%E5%A5%BD%E5%90%88%E4%BD%9C%E6%9D%A1%E7%BA%A6/2787097?fromtitle=%E4%B8%AD%E4%BF%84%)中俄两国在2021年2月28日均未表示要终止该合作，所以该条约继续生效，习神居然特意与普京举行视频会议还正式对外宣布两国延续合约，何等滑稽可笑。

2、中共特意在中共百年大庆前拿此事进行大肆宣传，可见中共现在已无稻草可抓，外交也陷入绝境之中。中共借此消息在墙内大肆宣传，忽悠老百姓说中俄关系世代友好，但是中共却避而不谈中共向俄罗斯出卖了大量中国领土，而且该合作也没有让中俄两国建立同盟关系。

3、按照国际惯例，同盟协议双方要明确两国国防等方面的责任与义务，所以才有有效期，只有不需要承当义务和责任的合约才能无限期延长，《中俄睦邻友好合作条约》是否延长对俄罗斯意义不大，中共国也只是急需借此对内进行宣传。

4、北约成员国签署北约同盟协议后，各成员国每年都需要向北约提供固定比例的财政拨款，各国也必须承担相应责任。《中俄睦邻友好合作条约》则不需要中俄两国为此提供任何财政支持，其中的大部分条款都要求两国共同遵守《日内瓦国际公约》等国际公约，而该公约就是以美国为首建立起来的国际秩序，只有中俄两国都遵守国际法，《中俄睦邻友好合作条约》才有效。

5、《中俄睦邻友好合作条约》中大部分条款都引用国际法，中共如果违法其中的任何一项条款，该合约就失效，这就是普京的政治智慧。两国元首此次特意强调：《禁止发展、生产及储存细菌（生物）及毒素武器和销毁此种武器公约》作为国际安全体系的支柱之一具有重要意义，支持遵守和巩固此《公约》，包括通过达成具有法律效力和有效核查机制的《公约》议定书，呼吁各缔约国不要阻挡上述议定书谈判，增加其国内外生物军事化活动透明度，呼吁各方通过积极磋商和相互合作解决《公约》实施过程中遇到的任何问题。[\[2\]](http://www.xinhuanet.com/2021-06/28/c_1127606620.htm)俄罗斯在此给中共国布下了一个非常大的陷阱，这和中共病毒密切相关，俄罗斯也为自己脱身埋下伏笔。

6、两国还明确：中俄基于条约第七条所述原则，秉持全面战略协作伙伴精神，开展不针对第三国的军事及军技合作。[\[2\]](http://www.xinhuanet.com/2021-06/28/c_1127606620.htm)这已经明确俄罗斯不可能与中共国形成联盟，同时中共也无权干涉俄罗斯与其他国家形成军事同盟，所以该条约也只是表明双方拥有松散的合作关系而已。

7、中共近几年不仅大量生产核武器，还向全世界发动病毒超限战，中共早就违反了《核不扩散条约》和《禁止生物武器公约》，所以该合作条约毫无意义，俄罗斯早晚得切断和中共国的合约关系，俄罗斯未来还有可能会依据本条约向中共国索赔。

二、美乌主导联合军演

德国之声中文网最新报道[\[3\]](https://www.dw.com/zh/%E5%8C%97%E7%BA%A6%E9%BB%91%E6%B5%B7%E5%86%9B%E6%BC%94-%E4%BF%84%E7%BD%97%E6%96%AF%E8%A7%86%E4%B8%BA%E6%8C%91%E8%A1%85/a-58077231)：此次由美国和乌克兰主导的“海风”军事演习，共动员了数千名官兵及数十艘军舰飞机，参与国家为32个。美国军方表示，整个军演将持续两周。来自美国海军的消息称，这是1997年开始在黑海举行定期军演以来，规模最大的一次军事演习。莫斯科方面认为，北约军队在自家门前举行军演无疑是一种挑衅行为，并敦促美方放弃军演计划。

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/29003724/%E9%BB%91%E6%B5%B7%E5%86%9B%E6%BC%94.jpg)[黑海军演](https://static.dw.com/image/17916963_303.jpg)

路德社评论：

1、前天，乌克兰刚刚宣布撤回其向联合国递交的要求彻查中共在新疆犯下反人类罪和种族歧视罪的声明，后来有媒体揭露中共利用疫苗威胁乌克兰。今天，美国和乌克兰就主导了此次黑海军演，这种联合演习意味着美乌两国已形成同盟关系，况且乌克兰正在努力加入北约组织。

2、美国与乌克兰联合三十多个国家举行联合军演就是在给乌克兰站台，俄罗斯默认这些国家举行联合军演也是拜登与普京在6月16日两国首脑会谈上达成的默契，这也预示这乌克兰即将加入北约。

3、乌克兰军事工业曾是前苏联时期的国防工业基础，乌克兰有很多船厂和兵工厂，乌克兰一旦加入北约，其武器系统就可以与盟国的军队指挥系统进行对接，俄罗斯也就不可能再侵犯乌克兰，这也彻底破灭了中共企图利用俄罗斯搅乱欧洲秩序的阴谋。

4、申请加入北约的要求非常高，首先就要求成员国必须行使三权分立的制度，集权国家绝不可能加入，其次就是要对北约有所贡献，本国人均GDP还要满足一定要求，同时每年还要向北约缴纳一定比例费用，最后就是本国的军事系统要具备与北约对接的实力。

5、美俄过去十年一直在争夺乌克兰，当乌克兰将加入北约写入本国宪法之后，这意味着乌克兰与美国战队。中共国也仅从乌克兰偷走了二流的武器技术，中共军舰动力所使用的燃气轮机和多项技术都来自乌克兰，乌克兰一旦加入北约，那中共海军将毫无秘密科研。
