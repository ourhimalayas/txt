By:**秘翻新闻**

**2022-03-03**
![](https://assets.gnews.org/wp-content/uploads/2022/03/8-6.png)图片来源 CNN
[据CNN 3月2日报道，美国司法部3月2日表示，将成立一个名为【盗贼抓捕（KleptoCapture）】的特别部门，帮助执行对俄罗斯政府官员和寡头的制裁，目标是他们的游艇、飞机、房地产和其他资产。该部门被获准使用最尖端技术，没收受制裁个人和公司的资产，包括游艇、喷气式飞机、房地产等等。](https://www.cnn.com/2022/03/02/politics/yachts-russian-oligarchs/index.html)

[美国司法部长梅里克·加兰德表示，新的工作组将包括检察官、联邦探员和来自联邦调查局、国税局、美国法警局和美国邮政检查局的洗钱、税务执法和国家安全调查方面的专家。加兰在一份声明中表示：将不遗余力地调查、逮捕和起诉那些犯下罪行，使俄罗斯政府得以继续这场非正义战争的人。](https://www.cnn.com/2022/03/02/politics/yachts-russian-oligarchs/index.html)

[据美国司法部新闻稿介绍，该制裁工作组将致力于执行美国与盟国和合作伙伴一起实施全面的制裁，包括出口限制和经济对策，以应对俄罗斯对乌克兰的无端军事入侵。该部门将确保制裁行动能将俄罗斯与全球市场隔离，并通过制裁俄罗斯官员、与政府结盟的精英以及帮助隐瞒罪行的援助者，让他们为这种不合理战争行为付出严重代价。](https://www.cnn.com/2022/03/02/politics/yachts-russian-oligarchs/index.html)
