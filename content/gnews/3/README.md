立法会会议厅过去悬挂的特区区徽已抢闸换上中国国徽，立法会秘书处回复：「根据《国旗及国徽条例》，举行立法会议员宣誓的场地必须悬挂国徽。林郑月娥将会为新一批90名立法会议员监誓。立法会正门玻璃蒙上黑布，暂时不可以内进。目前未知立法会会议厅日后会否长期悬挂中国国徽，还是在宣誓仪式之后恢复悬挂区徽。但根据规定，立法会亦须展示区徽。

中共公然撕毁了《中英联合声明》，代表香港的一国两制已提前结束，所有公务人员、立法会议员和区议员已经向中共宣誓效忠，曾经代表香港人的立法会也变成了代表中共的举手机器，立法会换上中共国徽不足为奇。至于立法会必须展示区徽的规定，在朝令夕改的中共政权管制下也不过如《中英联合声明》般撕毁而已。
