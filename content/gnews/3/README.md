## 樱花时政┃欧洲航天局停止与俄罗斯的火星探测

撰稿：东京樱花团/上善若水

![](https://assets.gnews.org/wp-content/uploads/2022/03/欧洲航天局停止与俄罗斯的火星探测.png)[图片来源](https://gnews.org/wp-admin/post-new.php#imgrc=NBrG6xyJxS4qVM)

在俄罗斯和西方因乌克兰战争而关系紧张之际，欧洲航天局也暂停了与俄罗斯的一项关键太空合作——火星探测任务（ExoMars）。该航天局于本周四（3月17日）宣布了这一消息。

该机构在一份声明中表示，欧空局目前不可能与俄航天集团就2022年发射ExoMars探测器继续合作。

欧洲航天局还写道：”作为一个被授权在充分尊重欧洲价值观的情况下发展和实施太空计划的政府间组织，我们对乌克兰遭受侵略而造成的人员伤亡和悲惨后果深表同情，并严厉谴责俄罗斯的侵略行为。
