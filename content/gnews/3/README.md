一、美9.35亿打造月球轨道房屋

路透社7月10日报道[\[1\]](https://www.reuters.com/article/northrop-grumman-nasa/northrop-to-build-homes-on-moon-orbit-under-935-mln-nasa-contract-idUSL3N2OL3I6?edition-redirect=uk)：美国宇航局和诺斯洛普·格鲁曼公司敲定了为月球轨道空间站GateWay开发HALO居住后勤舱的合同，价值9.35亿美元。阿尔特弥斯计划正式进入实施阶段，这一计划的实施将改变人类探索外星的模式。



> 美國宇航局和諾斯洛普·格魯曼公司敲定了為月球軌道空間站GateWay開發HALO居住後勤艙的合同，價值9.35億美元。阿爾特彌斯計劃正式進入實施階段。這一計劃的實施將改變人類探索外星的模式。

路德社评论：

1、这意味着美国NASA月球开发计划正式向民营企业开放，同时将人类登月分成两步走，地球到中转站，中转站再到月球表面，NASA正在为将来从月球进入火星做准备。

2、马斯克参与了几个重要项目，第一是特斯拉电动车，第二是太阳能产业，第三是速成房屋建设，第四是数字货币，第五就是地面轨道，第六是星链计划，美国从第一次登上月球至今六十年一直在进行太空领域研究，美国已经开始发展星际文明，中共居然还在国内开历史倒车。

3、特斯拉一直在进行火箭回收实验，亚马逊也在进行，美国宇航局未来还会向民企开放越来越多的项目，像贝索斯这样的全球富豪都愿意亲自参与发展星际文明，这未来将带来巨大的经济效应。

4、深受皇权统治和儒家思想桎梏的清朝因闭关锁国已经错过了大航海时代的黄金发展时期，美国现在已经开始引领全球进入星际文明时代，中共居然还在国内建防火墙，靠假骗偷提升本国技术，更可悲的是，中共还对新疆少数民族进行种族灭绝迫害。

5、诺斯洛普是一家为美军生产B2轰炸机的军火商，这家公司要建设的房屋绝非普通房间，未来还有更多企业将参与建设太空房屋。美国率先开启人类星际文明，那美国就将制定太空秩序。诺斯洛普还会将大量零配件分包给盟国的企业共同参与研发制造，但是美国已经明确禁止中共参与。

二、拜登政府制裁34家中共军方关联企业

美国商务部7月9日正式发文称：美国商务部工业和安全局 (BIS) 将34家实体添加到实体名单中，因为它们参与或有可能参与违反美国外交政策和国家安全利益的活动。在这34个实体中有14个来自中共国，这些实体帮助中共对维吾尔人、哈萨克人和新疆维吾尔自治区其他穆斯林少数民族成员进行镇压、大规模拘留和进行高科技监视活动。商务部还将另外五个直接支持中共国激光和C4ISR计划相关的军事现代化计划的实体添加到实体清单中。[\[2\]](https://www.commerce.gov/news/press-releases/2021/07/commerce-department-adds-34-entities-entity-list-target-enablers-chinas)

路德社评论：

1、美国商务部发布的这份制裁令已经表明美国的强硬态度，中共国如果真有本事就不要依赖美国，自己研发和生产高科技产品，但是中共国专制体系绝对不可能生产出高科技产品。美国完全有权禁止本国企业向中共国出口高科技产品，靠的就是实力，这是建立在强大的体系之上，美国价值观就是该体系的根基，所以才有科技创新。

2、中国人的能力没有问题，有问题的是中共体制，中共假骗偷体制不但不保护创新，还彻底阉割了中国人的创新能力。而美国已经建立起一套非常完善的专利保护体系，任何只要涉及抄袭的行为都会被处以巨额罚款，所有专利使用者都需要向专利发明人缴纳专利使用费，而且美国看中谁先发明创新，而中共国则是谁先注册专利就属于谁。

3、社会每到一个阶段都要推出新行业，只有不断地推陈出新才能淘汰落后的生产力，推动社会发展。而中共靠着瓷器、茶叶、丝绸和农业这四个行业吃了几千年，几乎没有什么创新，整个社会因为没有创新就会形成垄断，这是行业的必然属性。任何一个行业在经过充分竞争后只留下三种公司，分别是成本最低型公司、技术领先型公司和个性差异化公司，如果该行业没有诞生新技术及很容易形成垄断，垄断就会降低效率，这时就需要靠其他刺激来盘活整个行业。

4、传统能源汽车发展到今天突然出现特斯拉电动车，这大大刺激了传统能源汽车行业，特斯拉因此开创出一个新行业，其他汽车企业都不得不跟进投入。特斯拉又创造出星链计划，其他网络运营商又不得不参与投入研发，如果星链计划的成本在未来出现大幅度下降，其他运营商必死无疑，所以竞争能让垄断自然消亡。每个行业本身都容易形成垄断，但是中共国居然还要动用国家力量促使某些企业垄断整个行业，中共政府已经成为剥削老百姓的工具，这就是计划经济和市场经济的巨大区别。

三、伊朗铁路系统遭黑客攻击出现瘫痪

雅虎新闻7月10日报道[\[3\]](https://news.yahoo.com/humorous-hackers-hit-irans-railroad-184436270.html)：据一家半官方通讯社报道，伊朗的铁路系统周五遭到网络攻击，黑客在全国各地车站的显示板上发布了关于涉嫌火车延误或取消的虚假信息，他们敦促乘客致电索取信息，并列出了该国最高领导人哈梅内伊办公室的电话号码。这次黑客攻击导致火车站出现“前所未有的混乱”。

路德社评论：

1、伊朗近日发生多次离奇事件，例如斩首行动，未知原因的爆炸，核设施瘫痪，这次又遭黑客攻击，西方国家就是借攻击伊朗警告中共国。如果黑客未来将习神的手机号码公布在大屏幕上，那对中共国绝对是巨大打击。

2、火车的调动是依赖独立的铁路调度指挥系统，绝不允许接入其他网络系统，完全是自成体系，黑客居然能入侵这种独立系统，可见该组织黑客技术非常高明，中共看到后肯定非常恐慌。由此可以看到，黑客很轻松就能扰乱中共国的秩序，社会一旦陷入混乱，那中共统治地位就将不保。如果要玩邪招，科技发达的美国和以色列绝对强于中共。
