## 泽连斯基谴责炮击医院是“战争罪”

发布 / 2022年3月10日 9:07 AM
采集：玫瑰新闻组

![](https://static.zaobao.com/s3fs-public/styles/article_large_full/public/2022-03/2022-03-09t194320z_772811410_rc25zs91pihh_rtrmadp_3_ukrainre-crisis-mariupol.jpg?VersionId=2b8LWczqrd05f8eFtMnp.3sypIzHQN43&amp;itok=YaZbY1Ir)马里乌波尔一处医院被炸毁

乌克兰总统泽连斯基周三（3月9日）形容俄罗斯空袭东南城市马里乌波尔的儿童医院是“战争罪”，此行动遭到国际社会谴责。

据法新社报道，乌克兰官员说，医院的妇产科和儿科病房被空袭。他们上传网络的录像显示，所有窗户被震碎，隔间墙壁被炸开，医院外的车辆被纵火。

泽连斯基说：“我们没在顿涅茨克或卢甘斯克地区，或是其他地区犯下这样的战争罪行，因为我们是人。你们呢？俄罗斯是怎样的国家，是害怕医院和妇产科病房，所以要炸毁它们？”

俄罗斯外交部发言人扎哈罗娃在莫斯科记者会上没有否认上述袭击，但说乌克兰民族主义军队在撤离医院职员和病患后，在当地设立攻击点。

泽连斯基证实有17人受伤，并说在空袭警报响起时，在医院的人员立即躲藏。

另据路透社报道，美国白宫新闻秘书普萨基说：“看到军事力量在一个主权国残暴追杀无辜平民，实在令人震惊。”

联合国人权组织说，所属监督单位正核实马里乌波尔市的伤亡情况。该组织发言人特罗塞尔说：“我们对此深表关注，因为在人口众多的地区出现滥用武器的情况。”
