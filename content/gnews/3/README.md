## 中共国开始对美国外交官进行中共病毒肛门测试

![](https://assets.gnews.org/wp-content/uploads/2022/01/1-453.jpg)
图片来源：网络截图

据国家脉动1月27日报道，司法观察通过《信息自由法案》获得的电子邮件显示，中共让美国外交官进行肛门中共病毒（COVID-19）测试。

2021年1月底，就在乔·拜登宣誓就职的几天后，一连串交换的电子邮件显示，美国国务院官员在中共国时被进行了肛门中共病毒测试。

在2021年1月22日一封题为“新的测试方法 ”的电子邮件中，一名国务院官员解释了一名同事是如何接受中共病毒的肛门测试的。他说他的一位同事在他的公寓里被做了一次中共病毒肛门拭子测试。“这只是一个开始，如果你还没有处理它的话，我相信它很快就会扩大。最好在开始传播之前，先下手为强。”他的这位被隐去名字的同事在邮件写到。

对这封电子邮件的单独回复显示，一名国务院官员询问了有关测试前的细节。

“这是在哪个城市发生的？又是什么号码的测试？他是否说过中共是否事先通知过将以这种方式进行测试？他是否被告知有其他选择？”这位名字被隐去的官员问道。

随后的电子邮件显示，中共病毒肛门测试是在北京该官员自己的公寓里进行的，该官员没有得到“任何通知或选择”，他不得不同时做鼻腔和肛门拭子。

司法观察获得的其他电子邮件似乎显示，国务院各官员对中共病毒测试的潜在要求感到恐慌。其他电子邮件似乎将美国外交官被迫接受中共病毒肛门检查归咎于一个“错误”，因为据说该政策“不适用于外交官”。

这些邮件证实了之前的报道，即中共正在推出有争议的中共病毒测试形式。
