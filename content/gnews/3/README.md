## 國際｜黑海糧倉爆戰爭 引爆糧食短缺危機

烏克蘭為主要穀物出口國，眾多國家仰賴進口烏克蘭小麥、玉米和葵花油等，俄烏戰爭影響烏克蘭播種，將嚴重影響全球糧食供應，恐發生糧食短缺危機。

美國農業部資料顯示，烏克蘭是世界第4大玉米出口國、第5大小麥出口國，佔全球10％小麥出口、14％玉米出口及50％葵花油出口。而烏克蘭有85％作物是靠海運輸出，戰爭導致港口關閉，後改以鐵路方式輸出，成本增加10％至15％。

俄羅斯與烏克蘭出口小麥數量佔全球25%，摩根大通估計3月第2周俄國穀物出口量相較去年同期銳減6成。全球糧食政策倡議組織「農產品市場資訊系統」（AMIS）表示，有至少十多個國家糧食依靠俄烏供給，戰爭導致糧食危機發生。小麥和玉米期貨價，今年來漲幅分別達42％、27％。
