2、中共近年非常重视发展外太空技术，但中共国要么想方设法剽窃西方国家的太空技术，要么就想办法摧毁西方国家发射的卫星，却从来不思考如何一步一个脚印地发展本国太空技术，总想着利用规则的漏洞耍流氓。中共居然利用自杀式卫星来破坏西方国家卫星，防卫大臣也展示了多张图片证据，由此可以看出中共国已多次使用这种办法破坏他国卫星，中共也在太空领域向西方国家发动超限战，这就是北约与美国要开展太空防卫战的真正原因。国际秩序最核心的就是规则，绝不允许任何国家利用规则来破坏国际秩序，中共现在举一国之力破坏国际规则，这对全世界是一场巨大的灾难。

3、日防卫副大臣所说的内容与爆料革命所说的相吻合，美国肯定也掌握这些情报，但是对中共非常了解的日本人比较容易透彻地解读这些情报，所以日本加入亚约大大提升了美国解读情报的能力。同时，日本副防卫国务大臣还提及中共在香港问题上欺骗了全世界，并表示决不允许台湾步香港后尘。

4、这也是日本开始掌握亚洲话语权的表现，由于中共在香港问题、新疆问题和中共病毒问题等一系列问题上的邪恶行径已经让中共彻底失去了其在亚洲的话语权。美国对日本进行军事松绑之后，日本逐渐从中共手中夺回亚洲话语权。

5、日本副防卫国务大臣就是代表日本政府站出来公开表态，他拿着日本情报公开表态事先经过日本政府批准，可见日本内部已经达成共识认可台湾是一个民主国家，彻底否认中共政权的国际合法性。

6、日防卫副大臣此举也是在做舆论铺垫，日本政府绝不可能干预日本媒体，所以中共对日本媒体渗透严重，所以副大臣才在美国进行公开表态。此前美国军用运输机停降台湾，中共国没有采取任何实质性行动，现在由日本开始试探中共国底线。如果中共仍然没有任何行动，那台湾作为一个民主国家就逐渐成为国际共识，现在对中共国外交是一个巨大的考验。日本政府接下来如果公开支持日本防卫副大臣言论，中共很可能推动中日脱钩来要挟日本，那就正中下怀，如果中共对台湾发动战争，那中共政权必死无疑，无论中共如何选择都进退两难。

7、中共外交部汪文斌对此表示，中方对日本政府高官发表错误言论表示强烈不满，已向日方提出严正交涉。日本这位政客公然将台湾称为所谓的“国家”，严重违反中日联合声明等四个政治文件的原则，中方要求日本政府就此作出明确的澄清，确保不再发生此类的情况。[\[2\]](https://news.sina.com.cn/c/2021-06-29/doc-ikqciyzk2562659.shtml)
