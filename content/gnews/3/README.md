新中国联邦乌克兰国际人道救援团队前线总指挥大卫先生在波兰城市热舒夫（Rzeszow）迎接第一辆从乌克兰战区安全撤离的救援大巴，并现场采访了热舒夫市市长Konard Fijolek。

大卫先生询问市长先生如何看待俄罗斯和乌克兰的这场战争以及难民的遭遇。

市长先生表示，21世纪发生这样的战争令人不可思议，每天看到无数个家庭逃亡，承受妻离子散的痛苦，很难过，希望能够马上停止战争，通过和谈解决问题。

大卫先生询问市长如何看待新中国联邦的这次救援行动，以及法治基金和法治社会这两个慈善机构提供了这次无偿的救援行动，来解救陷入困境的中国同胞、乌克兰受困民众以及所有需要帮助的人们。

市长先生表示，全世界很多国家的媒体都来到了俄乌前线，加拿大、欧洲、澳大利亚、南美。市长先生很高兴看到中国人也来了，很感激，也很感动。

大卫先生表示，今后会有更多的巴士加入到救援行动，争取每一天每隔一个小时就有一辆救援的巴士Coach从利沃夫（Lviv）到达克拉科夫（Krakow），希望能解救更多的中国同胞和乌克兰受困民众。

大卫先生代表法治基金和法治社会特别感谢市长先生给予这次救援行动提供的支持和帮助。亿万中国人会因为新中国联邦的直播节目看到这个车站，这里会被整个中国所关注。

大卫先生以私人身份问起市长先生如何看待中国共产党对待这场战争的做法。市长先生对中共国政府没有反对这场战争、对救援行动只是采取观望的态度感到很失望。市长先生表示很喜欢中华文化，很高兴中国人与他们站在一起。

大卫先生摘下自己印有新中国联邦标志的帽子，向市长展示新中国联邦的信仰、愿景和目标——Take Down The CCP，解救中国同胞，建立一个属于中国人的正道主义的新中国联邦。

新中国联邦乌克兰国际救援行动，是中国近代历史上第一次由非官方认可的民间组织自发展开的大规模救援行动。新中国联邦成立仅仅两年，还很年轻，但是在战争面前，在同胞无依无靠、生命受到威胁的关键时刻，新中国联邦第一时间与国际救援组织合作，派出团队去往前线展开无分国界、无分种族的国际大营救行动，这是载入史册的一次壮举。

在中共国官员争先逃走，弃同胞于不顾的时候，新中国联邦人从四面八方来到前线为素未谋面的华人同胞和受困民众伸出援手。作为俄罗斯发动这场不义之战的支持者，中共的行为和国内媒体的欢呼雀跃，导致海外华人被世界各国所排斥。新中国联邦以救人为根本，没有政治需求、没有利益考量，不计成本地全力营救中国同胞、乌克兰和各国友人的行为，正在努力扭转华人的形象，塑造正义、团结、有爱心、有信仰、有行动力的新中国联邦人的形象，用行动告诉世界中国人不等于中国共产党。
