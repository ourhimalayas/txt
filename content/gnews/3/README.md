3月16日，美国总统拜登首称俄罗斯总统普京为“战犯”，这是自莫斯科近三周前入侵乌克兰以来，总统首次使用战犯形容普京。

据俄罗斯国有通讯社塔斯社报道，普京的新闻秘书回应，拜登的战犯说法不可饶恕。去年拜登曾将普京比喻为刽子手，俄方曾短暂召回驻美大使作为回击。此前一天，美参议院一致通过了一项决议，谴责乌克兰的暴力事件，并呼吁对普京及其政权成员的战争罪进行调查 。

对此，联合国表示，俄罗斯入侵乌克兰造成了欧洲自二战以来增长最快的一场难民危机。位于荷兰海牙的联合国最高法院下令俄罗斯立即停止其毁灭性入侵。作为对战争罪指控调查的一部分，海牙的另一家法院—国际刑事法院ICC也在调查这场冲突。
