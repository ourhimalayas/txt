## 一位乌克兰妇女自述：上海隔离期间犹如罪犯
` 日本東京方舟農場` [轉載自GNews](https://gnews.org/zh-hans/2370508/)

**撰稿：日本东京方舟农场  Miyali7768(文玉)**

近期，一名居住在上海的乌克兰妇女向西方主流媒体叙述了她在隔离中心的经历，遭受到了罪犯般的对待。

![](https://assets.gnews.org/wp-content/uploads/2022/04/0-29.jpg)（图片来源：零对冲）

据《零对冲》报导，居住在上海的一名乌克兰妇女因疫情管控而被带到上海世博展览馆综合大楼内隔离，在那里她度过了近三个星期，即使她有两次病毒检测呈阴性，仍被强迫留下。

她对记者表示，这个隔离中心挤满了成千上万的陌生人，并且没有淋浴或卫生设施。在整个隔离期间都不能洗澡，而且房间里的灯24小时亮着。她回忆道，最让人难以忍受的就是整天开着的明亮灯光，感觉自己住在监狱，像个“新冠病毒罪犯”。

此外，她还表示，她的男友虽然未进隔离中心，但一直在担心自己不断减少的食物储备。

据悉，上海借由疫情扩散已经建造了100多家隔离中心，总容量超过16万张床位。
