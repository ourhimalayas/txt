“毫无疑问，北京的作为是在破坏民主治理以及开放社会，试图获取自身战略利益，这是美中更广泛战略竞争的一环。”前白宫国安会亚洲事务资深主任、现任华盛顿智库战略暨国际研究中心（CSIS）资深副总裁葛林（Michael Green）6号在CSIS举办的民主高峰会会前媒体电话简报活动上，回答了自由亚洲电台记者提问时做了上述表示。

但葛林说，中共国越是抹黑反击，越是凸显中共国的内心不安，“北京将民主规则视为我们（美国）的优势”，他说，“所以北京才会对内、对外都在攻击美国。” 另一方面，中共国官方还“老王卖瓜、自卖自夸”，中共国务院新闻办公室4日公布的《中国的民主》白皮书，称赞中共国的“全过程人民民主”。其实中共国的制度好不好，老百姓最有发言权。

美国的德国马歇尔基金会负责民主倡议项目的主任卡兹（Jonathan Katz）在布鲁金斯学会的另一场有关民主峰会的研讨会上说，中共国与俄罗斯通过散播假信息、拼命破坏人们对民主制度的信任，但这无法根本解决两国内部的问题，像是面对人民质疑各种问题的压力与声浪愈来越大，不论哪种制度，如果你想要有善政，就必须有独立与自由的媒体。

葛林：民主峰会后不会启动新冷战

中共国外交部发言人赵立坚6日批评，美国打着“民主”的幌子拉帮结派、党同伐异，挑动分裂对抗。美国应该把不搞“新冷战”的表态落到实处。

在葛林来看，赵立坚恐怕多虑了。“我不认为民主峰会后会开启美中新冷战，但我想强调的是，一个民主的体制是要有健康的公民社会存在的，然而，是中共国官方自己早就主动订立相关非政府组织法，拒绝中共国公民自己办的、以及境外的非政府组织。”
