## 墨西哥、中国、美国不断变化的动态关系

![](https://assets.gnews.org/wp-content/uploads/2022/04/1-382.jpg)墨西哥塔巴斯科州的多斯·博卡斯炼油厂项目现场

AMLO（墨西哥总统名字的缩写）和北京之间的关系变暖，可能会给美国带来安全难题。

在中国古老的围棋游戏中，棋手们在木板上逐渐摆放黑棋和白棋，争夺战略地位。如果说中共国和美国最近为争夺在拉丁美洲的影响力而进行了一场围棋比赛，那么中共国正在坚定地将一颗新棋子放在墨西哥。总统安德烈斯·曼努埃尔·洛佩斯·奥布拉多（AMLO）似乎接受了与中共国加强联系的想法，因此总统乔·拜登和他的政府考虑他们的下一步行动将是明智的。

中共国对墨西哥影响力的大幅提升，将对美国安全产生强烈影响。到目前为止，华盛顿与南部邻国在安全合作方面一直保持着一种“特殊关系”。尽管墨西哥政府在管理公共秩序方面享有自主权，但在国内安全事务方面，它一直将美国作为其主要——实际上也是唯一的合作伙伴。强有力的墨中关系的发展可能会改变这种平衡，北京将成为协助墨西哥不断升级的毒品战争的设备和建议的替代来源。

到目前为止，在中共国对拉丁美洲的影响方面，墨西哥仍然是个例外。中华人民共和国（PRC）和该地区之间的贸易在过去20年里猛增——2002年至2019年期间从170亿美元增至3,150亿美元，而中共国的投资在过去十年里达到了1,432亿美元。但其中只有不到2%的资金流向墨西哥，而中共国在2019年仅吸收了墨西哥出口的1.6%。有限的经济联系意味着两国的政治关系薄弱，虽然中共国和墨西哥领导人只是定期互访，但外交对话已经落后于其他拉美国家的对话。

但是，接触迟缓的时代可能很快就要结束了。当墨西哥外交部长马塞洛·埃布拉德感谢北京在应对COVID-19大流行病方面的支持，包括在未来几个月供应3500万剂量的中共国康希诺疫苗的时候，他补充说，墨西哥“将扩大两国的战略伙伴关系”。在埃布拉德2019年7月访问北京之前，该声明是AMLO政府做出的一系列亲中姿态中最新的一个。

直到最近，有两个关键因素阻碍了墨西哥与中共国更密切的接触。首先是美国自1994年签署北美自由贸易协定以来前所未有的政治和经济影响力。第二是民族主义。作为一场血腥的独立战争以及随后一些法国和美国干预的遗产，墨西哥的政治精英们在历史上一直反对外国干涉。虽然美国和墨西哥由于两国间密集的人员、文化和经济交流而建立了不稳定的伙伴关系，但与遥远的中共国的关系仍然是陌生的。因此，直到现在，墨西哥当局仍倾向于选择他们知道的魔鬼。

这些接触的障碍已经开始被削弱。在墨西哥和唐纳德·川普政府之间多年的商业和政治摩擦之后，乔·拜登的政府可能不会导致双边关系自动改善，而是带来新的紧张局势。

在安全领域，冲突几乎是肯定的。AMLO很可能不愿意接受对他处理打击有组织犯罪和腐败问题的批评，特别是在川普政府同意撤销对墨西哥前国防部长萨尔瓦多·西恩富戈斯将军的一系列洗钱指控之后。这位将军被允许离开美国，以换取墨西哥当局继续调查的承诺，58天后，墨西哥决定不起诉，这一承诺被打破。在AMLO民族主义信条的另一个表现中，墨西哥国会最近还通过了一项法律，限制美国缉毒署（DEA）特工在墨西哥境内的工作，在贩毒集团扩大其影响力的同时，对安全合作造成了严重打击。

意识到这个新出现的机会，中共国已经迅速介入，展示了明显的支持。除了在中共病毒大流行期间发送医疗设备和疫苗外，中共国还参与了AMLO在其“第四次转型”（4T）反贫困和反腐败框架下的一些最具代表性的项目。由中国交通建设股份有限公司（CCCC）和葡萄牙公司Mota-Engil领导的财团赢得了建造玛雅列车（Tren Maya）第一阶段的合同，确保了AMLO促进墨西哥南部经济发展战略的一个关键立足点。与总统的公众形象密切相关的是，这个由军队管理的项目的利润也将被用于资助军队的养老金制度，使其在政治上更加重要。

中共国的投资在石油领域也变得尤为重要。2014年，中共国达成了一项协议，在墨西哥国家石油公司PEMEX、中国石油天然气集团公司、国家开发银行和中国工商银行的参与下，设立了能源基金SINOMEX。这个基金提供了6亿美元的贷款，为多斯·博卡斯炼油厂的建设提供资金，大多数专家认为这个项目在财务上是不可行的，但对于AMLO试图减少精炼石油产品的进口和维护墨西哥的能源独立来说却是关键。中共国当局再一次将政治置于盈利之上，支持一个经济上可疑的倡议，以赢得墨西哥同行的同情。

尽管是前总统恩里克·培尼亚·涅托重新启动了墨西哥和中共国的双边关系，在2013年习近平访问期间给予其“战略伙伴”地位，但AMLO可能持有更多意识形态方面的观点。中共国可能成为他的国家按照墨西哥革命的民族主义和国家主义思想进行转型的关键盟友。这些思想最初体现在革命制度党（PRI）中，该党是该国直到21世纪初的主导政治力量。AMLO将自己塑造成恢复这一遗产的民众支持者，在他看来，前几任墨西哥总统背叛了这一遗产，他们实施了以市场为导向的改革，并与华盛顿建立了密切合作关系。他的政治项目维护对民众的社会施舍政策，让国家在经济发展中发挥核心作用，并拒绝美国的影响，将能够得到北京的同情。

尽管美国和墨西哥之间的关系在短期内可能不会有大的改变，但前者应该预见到中共国扩大其在墨西哥经济关键部门的存在，并加强向前的政治合作。如果AMLO的近似战略在国内取得成果，它可能会使墨西哥政府中的反美部门更加大胆，从而与华盛顿产生更多的摩擦。当拜登的官员开始确定在拉丁美洲的外交政策优先事项时，他们应该注意不要把墨西哥想当然地视为中共国影响力的永久例外之地。

\*编者注：墨西哥和中共国的官员后来说，中共国为多斯·博卡斯提供的资金是一种误解，向PEMEX提供的贷款没有被指定用途。
