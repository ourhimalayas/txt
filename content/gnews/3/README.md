## 有声｜中共监管部门要求国有企业检查对蚂蚁集团的投资

**By:秘翻新闻**

**2022-02-23**
![](https://assets.gnews.org/wp-content/uploads/2022/02/图片88.jpg)图片源自:网络
[路透社](https://cn.reuters.com/article/china-ant-group-soe-sources-0222-tues-idCNKBS2KR0S5)2月22日报道，据三位知情人士称，中共监管部门上周要求国有企业启动新一轮检查，以查明它们对亿万富翁马云的蚂蚁集团的投资以及与该公司的其他关联项目。

消息人士说，监管当局要求进行检查的实体包括国有银行和非银行企业，具体是要求这些国有企业提交与蚂蚁集团有关的股票投资、资产支持证券风险敞口以及贷款的细节。目前消息人士拒绝透露身份，因为他们未获授权向媒体发言。

此举加剧了投资者对中共政府今年继续整顿科技平台企业的担忧，并引发周二股票市场科技板块持续遭到抛售。阿里巴巴9988.HK股价最多下跌5.3%，是1月28日以来最低。

目前蚂蚁集团拒绝发表评论。
