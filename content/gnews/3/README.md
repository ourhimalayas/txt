## 俄罗斯高技术人才因入侵乌克兰而逃离该国

![](https://assets.gnews.org/wp-content/uploads/2021/12/michael-4.jpg)

普京发起的这场对乌克兰的侵略战争，越来越受到俄罗斯人民的反感和唾弃。

尽管俄罗斯FSB（Federal Security Service）警察在机场的检查每个人的手机，并截留任何他们怀疑试图永久离开的人，但自战争开始以来，仍有100,000高技术人才设法逃脱了试图阻止他们离开俄罗斯的FSB警察的检查，逃离了俄罗斯。

![](https://assets.gnews.org/wp-content/uploads/2022/03/Snipaste_2022-03-28_19-43-37.png)

自入侵乌克兰以来，已有很多俄罗斯人逃离该国，其中许多人逃往土耳其以及芬兰、吉尔吉斯斯坦和格鲁吉亚。高技能的程序员和能够在任何地方立足的IT专家是新的俄罗斯“技术游牧民族”的主力军。每天的航班让更多的年轻专业人士摆脱了政治压迫、经济制裁和对被征召参战的恐惧。

他们中的大多数人选择降落在亚美尼亚，也有些人搬到伊斯坦布尔或邻近的格鲁吉亚，也有许多人留在中立国。

在战争爆发前，只有3,500名俄罗斯人在亚美尼亚工作和居住，但现在每天都有这个数字的年轻人到达。目前，约有80,000名俄罗斯人来到亚美尼亚，亚美尼亚首都的主要街道被称为“小莫斯科”，约有20,000人选择在那里永久停留。因为该国在冲突中选择保持中立，其公民也表现出更多的同情心。

42岁的安德烈（Andrey）和36岁的妻子达莎·阿帕里诺夫（Dasha Aparinov）曾因在反对派集会上献花而被监禁，但在莫斯科向乌克兰派兵时决定离开。安德烈在俄罗斯经营着自己的IT公司。阿帕里诺夫说，在普京这个怪物下台之前，她们不会回去。

因为欧洲领空对俄罗斯航班的关闭使土耳其成为剩下的少数几个可以作为逃生路线的枢纽之一，对俄罗斯人来说，伊斯坦布尔也是一个受欢迎的选择，无论是过境还是在这里建立新家。

化名阿图尔（Artur）和弗谢沃洛德（Vsevolod）夫妇于3月初逃离莫斯科前往伊斯坦布尔，他们的生活在一夜之间因俄罗斯入侵乌克兰而发生翻天覆地的变化。

弗谢沃洛德说，俄罗斯的恐惧气氛是前所未有的。她的一位朋友在网上签署了反对战争的请愿书，然后警察出现在她家中。而另一位已经逃往伊斯坦布尔的朋友在WhatsApp上接到警方的电话，要求见他。

曾在艺术界工作的阿图尔说，“这是我在俄罗斯人生中的第一次，我对未来失去了任何愿景，现在的事情太不可预测了，我们看不到未来。现在一切皆有可能——在精神上，我们无法留下来。”

在俄罗斯机场，当局正在到处检查，询问每个人的回程机票。阿图尔删除了Meduza和其他一些反对派的新闻应用程序，因为有传言说它们（FSB警察）会检查人们的手机。像许多人一样，阿图尔听说过边防人员搜索旅行者的电子设备、寻找可疑内容，例如支持反对派政治家阿列克谢·纳瓦尔尼（Alexei Navalny）或阅读在新的新闻审查法后在俄罗斯被禁止的网站的内容。他说，“机场很诡异；空荡荡的。但是这架飞机异常庞大，而且装满了人。你可以看到所有穿着考究的年轻人都带着悲伤的眼神，你立刻明白这是一个自我放逐的创意阶层”。

俄罗斯说唱歌手奥克美龙（Oxxxymiron）在伊斯坦布尔组织了一场名为“俄罗斯人反对战争”的慈善音乐会，门票一开售就售罄。

![](https://assets.gnews.org/wp-content/uploads/2022/03/Snipaste_2022-03-28_19-42-40.png)

为了保护仍在俄罗斯的亲属，一名前高级时装摄影师，在莫斯科的一次示威活动中避免被捕后不久，和他的妻子逃离了俄罗斯来到伊斯坦布尔的阿尔森（化名，Arsen）说，至少在这里或世界其他任何地方，我们可以做的比在俄罗斯内部抗议更多，至少在这个政权倒台之前，一切都不会改变。
