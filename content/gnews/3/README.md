当地时间2月16日，据相关媒体消息显示，香港当天通报了超过四千两百起中共病毒新增确诊病例，以及9起新增病毒死亡病例，死亡患者中还包括一名年仅三岁的女童。当地目前等候入院的患者已经高达一万两千人。

有报道提及，香港地区当下的疫情传播情况十分复杂，当地医疗系统中已有120多名医护人员确诊感染中共病毒，另有20名医院职员被列为病毒的密切接触者。一位香港医院管理局的经理人在接受采访时指出，这些医护人员的感染途径均为社区或家居感染，加之各大医院患者人数骤增，香港公立医院眼下已经不胜负荷。
