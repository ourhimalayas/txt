![](https://assets.gnews.org/wp-content/uploads/2021/12/图片187366-edited.png)图片来源：网络
22日，京阪巴士在京都市内运行的路线上开始运营四辆电动巴士。该公司采用了比亚迪日本公司的一辆小型巴士，是中国主要电动汽车（EV）公司比亚迪的日本公司。比亚迪已在日本销售了50多辆电动公交车，并将以其压倒性的价格优势发起进攻。中国一个豪华汽车品牌在大阪市开业，电动汽车销售将于明年开始。专家们对越来越多的中国公司在电动汽车领域的存在越来越谨慎。

京阪巴士是连接JR京都站和京阪七条站的“车站循环巴士”，运行的4辆巴士全部更换为电动巴士。据该公司称，这是日本第一次在一条生产线上的多辆汽车上实现所有车辆的电气化。

Kansai Electric Power Co., Inc. 将负责充电/放电系统，我们将通过促进有效的能源成本管理来考虑引入电动巴士。目前，他拥有大约600 辆巴士，铃木一成社长说：“我想将来都用电动巴士取代它们。”

日野汽车等国内厂商也涉足电动客车，但京阪客车将世界品牌的信誉和成本列为采用比亚迪的原因。

比亚迪日本总裁比亚迪汽车拥有全球业绩记录，强调“比亚迪公交车在50多个国家和地区、300多个城市运行7万辆汽车”。

此外，我们还以约 1,950 万日元的价格推出了为日本市场开发的名为“J6”的车辆。据说国内厂商生产的产品价格在7000万日元左右，相差很大。条件如此不同，京阪巴士的高管表示，“唯一的选择是比亚迪。”

比亚迪的目标是到2022年在日本销售4000辆，刘总裁热切希望不仅在东京和大阪市区提供电动巴士，还要在各地提供电动巴士。

比亚迪正在中国和欧洲开发乘用车电动汽车，比亚迪日本方面表示“在日本的计划尚未确定”，但也有可能考虑在日本市场进行全面销售。中国一汽集团还在大阪市开设了豪华品牌“红旗”专卖店。中国企业在电动汽车领域的存在正在增加。

补贴机制

评论家江崎道雄表示，“中共国在电动汽车领域的攻势是一个非常麻烦的问题，日本私营企业必须以成本来判断产品的推出”。对于能够以低价供应产品的背景，他指出，事实上“倾销出口”是在获得中共政府大量补贴后实施的。

美国正在要求中共政府披露对中国极其廉价的产品出口的补贴机制”，他说， “日本政府应该通过要求中国披露类似信息来保护自己的制造商”。

评：中国公司以极低的价格，用政府的补贴来倾销比亚迪电动客车，而卖完了钱都流到盗国贼的钱包里。日本政府应该要求中共政府披露其相关政策的信息来保护本国的制造商。
