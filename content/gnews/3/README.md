人生百态：英雄者未必成功，成功者未必英雄。君不见，历史上多少名人典故，现实中多少亲朋好友，聪明努力者未必事业有成，貌不惊人者兴许功成名就。令人击桌合卷，掩面唏嘘！道理何在？“总在遇缘不同”——此乃唐朝净土宗实际创始人善导大师的名言，使无数修行人茅塞顿开。

![](https://assets.gnews.org/wp-content/uploads/2022/03/1-354.png)

佛法深谙因果关系的宇宙真相，凡事都讲“因”和“缘”，但更重视“缘”。“缘”，是宇宙间一切有生灭的事或物所必备的各种条件，也就是说任何事物都是由众多的“缘”和合而生的。明白了这个道理，须时时刻刻提醒自己凡事都应持“缘聚缘散”、“缘生缘灭”、“缘来则应”的态度，不可过分执着，免得自生烦恼。人的一生亦如是，无论你是春风得意轰轰烈烈、或是逆水行舟默默无闻、或功成名就也罢、或碌碌无为也罢，都是由个人的善根和缘分决定的。相比较而言，缘分比善根更重要。那么，善根和缘分是哪里来的？是自己多生累劫修来的。往生世造的“业因”，就是这一生的境遇，即“果报”。包括人生能否遇到贵人相助提携，修行人能否遇到活菩萨善知识教导，都看你多生累劫是不是修得了福报——天下事无偶然，劝君莫存侥幸心。

“诸法因缘生”，天下发生的大事小情，都有其原因，人生亦如此。那么，人生于世该怎么处世才对？佛法特别讲“随缘”、“缘来则应”。所谓“缘来则应”其实很简单，就是“尽人事，听天命”。也就是说，做任何事，力所能及，尽力而为即可。努力了，尽心尽力了，还是无可奈何回天无术的话，就不必再执着。为什么呢？因为凡事，“总在遇缘不同”，缘分不足这事你无论如何也做不成，兴许别人可以成就，偏偏你就不能成就，“总在遇缘不同”，不是你的缘分不够就是你的福报不够，无须烦恼。

小资料： 善导大师(西元613-681年)，阿弥陀佛再来人。中国佛教净土宗第二代祖师，净土宗的实际创始人。临淄人(今山东淄博市)，生于隋大业九年，少年出家。唐贞观中到并州去拜谒道绰大师，接受念佛往生的法门。
