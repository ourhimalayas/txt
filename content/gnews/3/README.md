2022年注定是不平凡的一年，岁月的脚步跨过元旦，刚进入1月5日，哈萨克斯坦就突然发生暴乱。

哈萨克斯坦是中亚五国之一，也是最大的内陆国家，与中国接壤，同时也是与中共关系较好的国家之一。

哈萨克斯坦爆发的抗议示威从该国最大城市阿拉木图开始，并迅速蔓延至全国。“示威者”一度控制了阿拉木图的机场。阿拉木图处于一片混乱之中。上千名“示威者”冲入总统宅邸并纵火，总统宅邸“陷入火海”之中。此外，市长办公室所在的市政府大楼“从上到下燃烧”。随后曼格斯套州、阿拉木图州及首都阿斯塔纳均发生了大规模抗议。

哈萨克斯坦全境进入紧急状态，全国通讯中断。总统托卡耶夫称，在哈萨克斯坦国内组织骚乱的团伙是在国外受训的，他们正在试图占领阿拉木图的重要设施，包括军火库。有中共媒体报道说，美国是哈萨克斯坦暴力抗议活动的幕后黑手，这似乎印证了托卡耶夫总统的说法，但事实究竟如何？

文贵先生在1月5日的大直播中向世界揭示了真相：哈萨克斯坦的暴乱完全是中共一手策划和支持的，是中共内部利益之争的结果。

事情的原由是：盗国贼王岐山、江泽民在哈萨克斯坦藏有100吨黄金，习近平得知后想据为己有，便指派王毅督办。王毅身负“圣命”，亲赴哈萨克斯坦，面见总统托卡耶夫，表明来意索要黄金。没想到被托卡耶夫总统一口回绝，并大骂王毅，扬言要黄金就和台湾建交。王毅更是难得的强硬，撂下一句狠话：限72小时归还。然后扬长而去。王毅前脚刚走，随即就发生了文章开头的一幕。

王毅有说狠话的资本。多年以来，中共一直肆无忌惮地实施蓝金黄，周边国家更是重灾区。正像文贵先生爆料所说，孟建柱控制了整个东南亚的毒品、贩卖人口市场，俨然成了东南亚的地下之王。王岐山更是了得，世界金融玩的风声水起，华尔街的大佬亦仰仗其鼻息。总加速师也不甘人后，掌控中共后通过“一带一路”大肆贿赂、收买多国政客，培植自己的势力。

2022年1月3日，是中共和哈萨克斯坦建交30周年纪念日。1992年的中共国还是江家的天下，彼时的江家、王岐山和哈萨克斯坦首任总统纳扎尔巴扎耶夫建立起了良好的关系。现任总统托卡耶夫更是和王岐山关系非同一般。这也是江王把巨量的黄金藏在哈萨克斯坦的原因。
