4月3日朝日电视新闻消息：被俄罗斯军队暂时控制的乌克兰首都基辅及其周边地区的凄惨状况已经清晰可见。

城市里没有人，包括住宅和商业建筑在内的大多数建筑，都已被俄罗斯的军事攻击摧毁。

俄军撤出后的4月1日，红十字国际委员会首次进入首都基辅附近的伊尔平。红十字国际委员会称，据信有3500名居民仍在该地区，已对伤员进行了急救，并运送了救济物资。

另据路透社报道，在同样靠近基辅的布查，一些看似平民的尸体被丢在街上，该市市长说有300多名居民被俄罗斯军队杀害。
