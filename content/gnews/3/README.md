5、[**浙江丽水现“鹦鹉热”传染病 7人感染1人死亡**](https://www.aboluowang.com/2021/1005/1655644.html)

中国浙江省丽水市出现“鹦鹉热”传染病，其中一名患者已抢救无效死亡。10月5日，浙江省丽水市疾控通报称，9月上旬以来，丽水市中心医院和丽水市医院先后收治7例鹦鹉热病例。其中1例救治无效死亡。上述患者中，6例经实验室检测宏基因检测确诊为鹦鹉热病例，另一例是临床诊断。通报称，这是丽水市首次报告鹦鹉热病例。这些病例出现畏寒、发热、全身乏力症状；患者先是剧烈头痛、全身肌肉疼痛，而后出现咳嗽、咳痰、呼吸困难、发绀、烦躁、重症肺炎等症状，病程长，病情重。通报未披露这些病患者身份。鹦鹉热是一种由鹦鹉热衣原体引起的疾病。公开资料显示，鹦鹉热又称鸟热，是鹦鹉热衣原体所引起，这些衣原体主要在多种鸟类之间传播和感染，偶然也会由带菌动物传染给人，最初发现本病多见于玩赏鹦鹉者。
