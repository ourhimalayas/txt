**巨额债务都是奴隶们的血汗钱**

巨额的利息的背后一定是巨額的債務。1）2011年11月23日，中共财政部公布了最新的地方政府债券发行数据：今年头十个月地方债已直逼30万亿元人民币，创下历史新高。更令人关注的是接近一半新增债务并没有投入经济，而是用作偿还旧债。2）路透北京1月30日报道，2022年首月地方债发行6,989亿元人民币，环比去年12月增加逾一倍，3）搜狐财经报道：2021年中国债务规模达336万亿。

以上是中共提供的数据，这还不包括企业的债务，中共国究竟有多少债务是个迷，但是仅按400万亿估算，14亿人每人28万元。邪恶中共不创造任何财富，这些钱都是奴隶们的血汗。那么，这些奴隶的血汗钱花在哪里？

**拿奴隶的血汗大撒币**

中共国是世界第一的撒币国，已是路人皆知。1）据《中国数字时代》网站该数据库资料，从2000年到2014年的15年间，中国向140个国家提供了经济援助和贷款，累计金额高达3620亿美元，约2.45万亿元人民币（按照当前汇率计算）。同时期，美国的海外援助总计3990亿美元，相差不到400亿美元。2）据《环球时报》2017年8月7日统计，中国从2013年—2017年8月止，中国共对外援助8774亿美元，约6.1万亿人民币。

2017年以后的对外“大撒币”数据无法追踪，另一个重要援助对象北朝鲜，70多年来由于援助数据不公开，我们也无法得知。但是仅凭此数据就已超过美国和日本的总和。

**宁与外邦不与家奴**

虽然我们无法明确中共大撒币的明确数据。但是仅就2013年—2017年8月，约6.1万亿人民币的撒币金额计算，如果当时平均分配给国内3000家上市公司，每家可获得20.33亿元人民币；贷给国内小微企业，平均每户61万元人民币，可以彻底解决全部1000万户小微企业的融资难问题。如果用于“三农”，平均每户6.1万元人民币，可以一次性实现全部一亿农民的小康目标。如果平均分配给每个奴隶，按17年人口计算，每人约4388元人民币。

也许，上述想法可能太过理想，那么下面的二组数据一定会让你感动悲哀和愤怒。1）据中共国儿童少年基金会报告，近300万因贫困造成的完全失学儿童1年教育经费需10多亿元。2）中共卫生部副部长朱庆生表示，至今农村有一半的农民，因经济原因看不起病；中西部农民因看不起病，死于家中的比例高达六到八成。
