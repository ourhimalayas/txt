## 新冠大流行次生灾难：一千万英格兰人因封锁面临心理健康问题

作者：台湾宝岛农场 寻文客

【新闻链接】https://www.breitbart.com/europe/2022/02/21/second-pandemic-10-million-britons-in-need-mental-health-treatment/

据外媒21日报导，英格兰的卫生官员警告，由于在中共病毒流行时实行封锁，预估英格兰有1000万人，包括150万儿童和青少年，因此遭受了精神健康问题的困扰。

由于应对中共病毒实行的封锁限制，大量英格兰人民的精神健康受到了不利影响。相关专家透露，自中共病毒开始以来，因心理健康问题而需紧急治疗的儿童增加了52%，同期因饮食失调而需治疗的18岁以下儿童增加了72%。目前英格兰有160万人在等待接受心理健康方面的专业治疗，还有800万需要专业治疗的人没有进入等待名单。

大量患者的积压已经让英格兰部分地区的精神健康专家不堪重负，许多缺乏足够护理的患者未来令人担忧。英格兰的卫生官员目前精神健康问题的现状称为“第二次大流行”。

根据郭文贵先生的情报，中共病毒是中共释放的生化武器。要解决中共病毒的流行及其次生灾难，必须消灭释放病毒的邪恶中共。
