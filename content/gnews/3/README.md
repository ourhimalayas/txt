马里乌波尔的谢尔比・奥尔洛夫副市长警告说，钢铁厂的基本物资几乎用尽。

“这里什么都缺。水、食物、药品和救援物资都不足，俄罗斯完全切断了所有人道主义援助和疏散（走廊）”，奥尔洛夫告诉BBC的“新闻时刻”节目。

俄罗斯继续炮击和突袭钢铁厂。乌克兰总统助理指出，俄军正在使用可穿透厚重装甲的地堡破坏者，屠杀地下的目标人物。

人道主义走廊，几乎无人能撤离

乌政府宣布，已与俄方达成了一项临时协议，旨在建立一条人道主义走廊以救援平民。 马里乌波尔的瓦迪姆・博伊琴科市长说，已在4月20日部署了90辆巴士，以疏散约6000人。据说，全市约有10万人无法撤离。

然而，顿涅茨克州的保罗・基里连科州长后来说，实际抵达的巴士数量低于计划，导致许多人无法撤离。

他告诉路透社：“市民聚集在商定的集合点，但极少人能够登上巴士。”
