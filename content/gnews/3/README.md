## 美国白宫对俄罗斯和白俄罗斯最新制裁
` 波士顿五月花农场金融部` [轉載自GNews](https://gnews.org/zh-hans/2099428/)

作者：波士顿五月花背靠背（Frankie）
![](https://assets.gnews.org/wp-content/uploads/2022/03/Screen-Shot-2022-03-03-at-6.46.55-PM.png)图片来源：互联网
美国白宫周三（3月2日）对俄罗斯和白俄罗斯做出新的制裁，回应俄罗斯入侵乌克兰。

白宫表示：”正与盟国和伙伴协调，对俄罗斯和白俄罗斯施加额外的经济成本。美国将采取行动追究白俄罗斯促成普京入侵乌克兰的责任，削弱俄罗斯国防部门及其未来几年的军事力量，瞄准俄罗斯最重要的财富来源，并禁止俄罗斯航空公司进入美国领空。“【1】

白宫还表示，俄罗斯已经成为全球经济和金融的弃儿。

最新制裁包括:

1、对白俄罗斯全面限制其技术产品的进口。美国商务部将把对俄罗斯出口管制政策扩大到白俄罗斯，防止物品、技术和软件通过白俄罗斯转移到俄罗斯。

2、美国国务院对22个被指认的俄罗斯国防实体全面封锁制裁。包括为俄罗斯军队制造作战飞机、步兵战车、电子战系统、导弹和无人驾驶飞行器的公司。

3、针对俄罗斯军事主要收入来源制裁。商务部限制长期支持俄罗斯炼油能力的技术出口，美国和盟友在逐步减少降低对俄罗斯能源依赖有着共同利益。

欧盟国家对煤炭需求的减少，转向更加清洁的能源-天然气，加剧了对俄罗斯能源的依赖，比如欧盟天然气进口大约50%来自俄罗斯，德国对俄罗斯天然气依赖超过50%。因此德国在俄罗斯入侵乌克兰早期，态度不明确，左右为难。

郭先生大直播指出，就在俄罗斯入侵乌克兰时候，俄罗斯通过北溪2号线给德国供应天然气一刻也没有停过。

4、商务部在与机构间合作伙伴的协调下，通过管理支持俄罗斯和白俄罗斯军队的实体清单，确保军队以及航空航天、海事和高科技部门不会获得可用于支持俄罗斯技术维护和创新的美国技术产品和技术。

5、禁止俄罗斯飞机进入和使用美国国内空域，孤立俄罗斯，与世界上30多个最重要的航空市场站在一起，剥夺了俄罗斯航空公司开展业务的能力。【1】郭先生2月25日大直播中爆料，一个俄罗斯拥有5000架超级私人飞机，仅南法俄罗斯游艇价值就是俄罗斯GDP（1万多亿美元）一倍达2万亿美元， 每年固定成本20%即2000亿。
