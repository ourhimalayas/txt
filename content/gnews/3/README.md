财联社电报2021年12月6日报道，社科院今日发布的经济蓝皮书。蓝皮书关于2022年中国经济形势分析与预测显示，在执行房地产三条红线两集中政策过程中，积极推动房地产业软着陆，既要防止土地拍卖频繁流标，热点城市供地不足导致后期房价暴涨，又要防止四五线城市房价过快下跌引发连带风险。

“积极推动房地产业软着陆”，也即房价下跌已是事实。一个慢慢的下跌叫软着陆，一个很快的下跌叫硬着陆。而中共国政府是想要房价缓慢的下跌，从而实现软着陆，但是他们做到吗？

现在中共采取的办法就是降准，就是把商业银行存放在央行以备急用的存款准备金降下来，拿给商业银行。所以在12月6日，央行宣布于12月15日全面降准。本次降准后，金融机构加权平均存款准备金率降至8.4%，释放长期资金约1.2万亿。虽然这些释放的资金补充了资金流动性短缺，但是对房地产公司来说，是杯水车薪。中国现在最大的问题是，老百姓兜里没钱，他还会买房吗？没人买房，房价必然下跌。其次，再释放资金进市场也没用，因为钱不是直接进入老百姓钱包的。有关系离权力核心近的人拿到钱，而这些人本就有很多房产了。这些人比猴子都精，现在这个经济形势，他们不趁着降准机会，用这些政策忽悠其他人接盘他们抛售就不错了，怎么可能在这个时候往里冲呢？而对于普通人来说，首要的是保住工作机会，保住收入。至于房子，在目前的经济形势下，已经是排在他们考虑的后面了，所以也不会买房。只要没有买家进入，房地产就会暴跌。就是地动山摇的硬着陆。

房地产硬着陆还有一个原因，就是所有的房地产公司都借了很多美元外债，现在已经到了还债高峰。目前没有还上债爆雷的有大家所熟知的恒大，佳兆业，花样年，后面还有融创，和生创展，绿地，蓝光发展，阳光100，金茂，银城国际，嘉禾，当代置业等几百家知名房企。他们欠的债不比恒大少，只是恒大是第一个，又由于许家印比较高调从而使得恒大比较知名而已。其实欠债比恒大多的有23家呢。这些房企到时间还不上债，国外的投资者肯定逼着这些房企贱卖资产还钱，他们可不是国内交了房款的买房人和供应商那么好欺负。所以为了回笼资金还债，各个房企弄出了买房送面积，送车位，这种变相降价方式，采取这种方式的房企一多，拿房地产的价格就会快速下跌，形成硬着陆。

很多中国人有一种信念，就是说政府不让房价下跌，房价就不会下跌，政府能拖住，然后那些个有多套房的人都是既得利益者，他们也不会让自己受损失，让房价下跌的。所以房价最后一定能挺住。在他们眼里政府是万能的，如果是政府是万能的话，我们现在还生活在大秦朝，当时的秦朝对老百姓和对各个方面儿的控制不比现在的中共还要厉害？短时间政府确实能控制一些事情，但长时间，就是经济规律起作用了。

这些年不仅是中共，全世界都是财富向顶部集中，1%的人几乎就拥有了全世界百分之七八十人财富的总和，有钱的人越来越有钱，没钱的人越来越多。那些个有钱的人，就是买100套的房子，1000套的豪车，买亿万套衣服又能怎么样？他们的数量还是很少的。他们的消费是支撑不起来全世界的消费市场的，所以现在全世界最主要的问题就是因为普通民众没有钱，所以消费起不来。消费起不来的意思就是老百姓没钱去换车，没钱去换手机，没钱去换家电，没钱买衣服。所有这些，只要不是与吃饭相关的，能省就省。吃饭也是从以前的排骨，换成了青菜。

中国是制造业大国，中国的制造业分为三个部分，一是国内的消费市场，而现在中国老百姓的六个钱包儿都已经贡献给了房地产。中国的老百姓都没有什么钱了，所以为中国市场生产商品的这些制造业企业生产出来的产品，衣服、手机、鞋子、袜子，他卖给谁去？

还有一部分是出口创汇的工厂，专为欧美全球生产中低端产品的。现在这两年的疫情，国外的经济也是非常差，商铺，饭店，旅游公司，服装店也倒闭了不少。虽有政府发钱救助，但普通民众也不敢花钱了。再有西方国家为了刺激经济，搞了极低的利率，所有西方民众这两年拼命抢买房子，把钱花在了房子上，别的支出就少很多，所以消费也是除了基本吃的，别的也少花了。他们不买东西，国外消费市场萎靡，当然针对出口市场的中国出口制造业也完蛋了。

中国的制造业企业还有一部分是为房地产行业做配套的，像什么钢筋，水泥，建材，家电，而现在房地产不行了，所以这些个制造业也完了。

面向这三个市场的制造业生产出来的东西卖不出去，那工厂不倒闭它干嘛呢？所以企业大规模的倒闭潮已经来临。

接下来我们再数一数，一般来说就业行业都有哪些。中国的就业行业首先是制造业。刚才已经说了，制造业的倒闭潮已经来临。12月5日网易新闻报道，工厂众多、外来人口最多的深圳，今年大多数的企业都在苦苦挣扎。很多大型企业将生产基地都已经搬离深圳，还有很多企业已经开始停工停产，变卖公司设备来解散员工。广东可是电子企业的集中生产基地，新闻都这样报道了，实际情况只能是更差。那这些工厂的员工有多少，他们还能找到就业岗位吗？

还有就是房地产，建房子、铁路、公路机场的建设提供了很多就业。但是“铁公鸡”建设是需要钱的，这些钱财政出的资可以忽略不计。基本都是城投公司或其他公司在市场上融资，而现在融资非常困难，“铁公鸡”建设无奈很多也停止了，所以这部分的就业也少了很多。

制造业和基建房地产是农民工就业的主要行业，这两个大的行业导致了很多农民工失业返乡，没有了收入来源。

还有一个为农村劳动力提供就业的行业就是服务业，大家知道服务业是为制造业、建筑业做配套的，就像小商店，小发廊，小饭店，等等。当年深圳制造业红火的时候，每个工厂区都有很多商业服务业，现在制造业和建筑业不行的话，那依托于制造业，建筑业的很多商铺，饭店也都倒闭了。这又有一部分人失业。

这些都是所谓的低端人口的失业，中国的很多中产津津乐道的IT行业、互联网行业在今年年底也遭遇了大规模的裁员，很多公司都裁员百分之二三十左右。这可是高薪的职业。2021年12月9日经济日报发文称，300万医药代表可能将被时代淘汰。中国经营报12月6日报道，近2000家银行支行网店或营业场所获批终止营业，四大行上半年减少员工2万多人。

互联网、金融行业这些国人最热衷的就业行业都在裁员。再加上之前的教培、旅游，最后就剩下教师医生公务员了。虽然在降薪，可还有工作啊。

把这所有的就业行业都数了一遍，几乎每个行业都在裁员，就是多少的问题。杠精不要和我在下边抬杠，说我周围都活得好好的，都有工作啊，都没失业啊。在他们的眼中，他的亲戚朋友都是那种过的好的或者有钱有势的，像他的远房亲戚在工厂里打工的，或者是在小商铺里边儿做营业员的，那些就不是他的亲戚，也不是他的朋友了，他们是看不见的。现在多少农民工返乡在家没有工作。

以防杠精在这又说三道四，我先说一下，即使在大跃进，文革时期，也是有工厂，有人在上班有工作的。失业潮不是每个人都没有工作，只要有百分二三十的没有工作，失业，就是失业潮了，若是50%的人没有工作，那就会导致社会动乱了。

现在开始，房地产硬着陆，企业倒闭潮，员工失业潮已经开始了，这场危机只会越来越重，各位做好过苦日子的准备吧。
