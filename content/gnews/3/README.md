**1.****自卫队将开始运营中共病毒（新冠病毒****COVID****-19）疫苗的接种**

[据NHK 1月17日报道，报道防卫省决定本月31日在东京与之前的疫苗接种的同一地点开始让自卫队运营大规模中共病毒（新冠病毒covid-19）的接种。为了加速第三次疫苗接种，政府计划重新建立由自卫队管理的大规模疫苗接种场所，这些场所将为18岁以上的人使用莫德纳疫苗。预计到下月初时，可接种疫苗的人数将增加到每天约2000人。](https://www3.nhk.or.jp/news/html/20220117/k10013435181000.html?utm_int=news-new_contents_list-items_028)

**2.****日本将在岛根和山口近海开发30年来第一个新的海上气田**

[据产经新闻 1月17日报道，日本最大的石油和天然气勘探公司INPEX宣布，它将于3月开始在日本岛根县和日本山口县的近海地区进行勘探项目。 该项目如果成功将成为日本30年来的第一个新的海上油气田项目。 随着原油和液化天然气（LNG）等能源的价格在全球范围内飙升，从经济安全的角度来看，即使是在本国生产少量能源的能力也很重要，此外还有助于提高国内天然气的自给率。](https://news.yahoo.co.jp/articles/7246bd14253effcc134f2ea00ea7aec19733b678)

**3.****东京、埼玉、千叶、神奈川要求实施“防止扩散”措施**

[据NHK 1月17日报道，因中共病毒（新冠肺炎COVID-19）的快速传播，东京都及三个县共同要求政府采取优先措施，防止病毒的传播。为此东京都、埼玉、千叶和神奈川三个县的知事举行了一次视频会议。在会议上，一都三个县决定共同要求国家政府采取优先措施，防止该疾病的传播，并提出了要求。 这些措施的具体细节将在未来制定出来。东京的医院病床使用率截至16日为19.3%，预计17日将超过20%。在埼玉县过去一周的平均新阳性病例数为903.7，是上周的4.2倍。](https://www3.nhk.or.jp/news/html/20220117/k10013435361000.html)

**4.****新财年预算提案总额107.5964万亿日元，创历史之最**

[据NHK 1月17日报道，日本政府已向国会提交了2022年新财年的预算草案，其中包括应对中共病毒（新冠病毒covid-19）的措施费用，总金额为1075964亿日元，是有史以来最大的。在支出中，”社会保障支出” 是有史以来最大的，为36.2735万亿日元， 53,687亿日元用于 “国防开支”。243,393亿日元用于赎回政府债券和支付利息，以及5万亿日元作为应对疫情的 “储备基金”。](https://www3.nhk.or.jp/news/html/20220117/k10013434421000.html?utm_int=news-politics_contents_list-items_016)

**5.****日本将允许一些外国学生入境**

[据共同社 1月17日报道，日本计划允许87名政府资助的外国学生进入该国，这是对该国目前在疫情中实施的非居民外国人入境禁令的一个例外。这些学生可能从本月晚些时候开始被允许入境，他们到达后将被要求在其学校准备的酒店中隔离10天。内阁官房长官说，这87名学生距离毕业或完成学业只剩下不到一年的时间，政府决定允许他们入境，以便他们能够完成学校教育。](https://english.kyodonews.net/news/2022/01/87f3374bafe0-japan-to-allow-in-some-foreign-students-making-exception-to-ban.html)

**6.****白宫：美日领导人将于1月21日举行在线会晤**

[据共同社 1月17日报道，白宫表示，美国总统拜登将于周五与日本首相岸田文雄进行在线会晤，在中共国在印太地区日益强硬的情况下，两国寻求进一步深化关系。拜登期待着与岸田合作，推进他们 “对自由和开放的印度-太平洋的共同愿景”，并包括通过四国集团在新兴技术方面建立伙伴关系。据白宫称，两位领导人预计还将讨论在抗击中共病毒（新冠病毒covid-19）大流行和气候变化问题上的进一步合作。](https://english.kyodonews.net/news/2022/01/3c75a074202c-urgent-us-japanese-leaders-to-meet-virtually-on-friday-white-house.html)
