## 中共网信办对涉东航客机坠毁网络言论溯源及处置
` 儒为` [轉載自GNews](https://gnews.org/zh-hans/2239581/)

作者：意大利罗马达芬奇农场-儒为
![](https://assets.gnews.org/wp-content/uploads/2022/03/东航.jpeg)封面来源：网络图片
中新网2022年3月26日报道，3月21日，东方航空公司MU5735航班在执行昆明－广州任务时在广西梧州市上空失联并坠毁。事故发生后，广大网民密切关注救援进展和事故原因调查。

此次东航飞机坠毁事件不是机器故障而是人祸，中共未免事实曝光会引发不可预测的后果，针对在网上各种不利于党的言论的账号进行溯源和网上贴大字报处理，以达到震慑的作用。

此外，腾讯、新浪微博、字节跳动、快手、百度、哔哩哔哩、小红书、知乎等网站平台，针对在东航客机坠毁事故中发布真实的和不利于中共的信息和账号从快从严处置。

截至目前，共计清理违法违规信息27.9万余条，其中谣言类信息16.7万余条，处置账号2713个，解散话题1295个。老百姓报复社会所引发的各种恶性事件暴露的是中共统治下的黑暗，追根溯源这一切灾难的根源就是中共。

下一步，中共国家网信办将继续督导网站平台，切实履行信息内容管理主体责任，及时处置相关违法违规信息和账号，引导网民客观理性发声，共同营造清朗网络空间。“防民之口甚于防川”，互联网时代的来临，中共这些做法无疑是在饮鸩止渴。
