香港强制防疫两年之久，闭关锁港自断活路，适逢特首换届，中共独裁下的“一人选举”已是嘘声一片，而新特首接手的是一个破败的香港，百业萧条、高失业率、高通胀、国际资金离场等现状下，根本无法彰显新特首树立中共选举制度的优越性。

特首候选人李家超在拜票中不断透露要恢复本港经济，正如爆料革命郭文贵先生在早前的大直播中揭露，香港的新特首上任后，香港会有一段时间的平静日子。中共很乐意将香港的管制玩弄于股掌之间，利用林郑政府暴力镇压香港的民主运动，借反复的强制防疫摧毁港人的斗志，再把一个“一穷二白”的香港交给钦定的李家超手中“重生”。中共钦定的特首上任后驱走了病毒，强检、强制隔离和社交距离限制不再，生产恢复、生活回复平静安逸，港人可能对中共的愚民政策大为感恩。不过这一切不过是中共梦，不然前特首梁振英为何要急于大搞海外宣传美化香港的国际形象，还要利用驻港外国人去外国唱好香港，目的就是要扭转中国败坏的国际形象，吸引外资回港，重建国际金融都市。

中共一边掩耳盗铃支援邪恶轴心国俄罗斯攻打乌克兰，同时挑起中东战火，在国际社会纷纷远离中共的趋势下，还想借香港打温情牌。还有，中共在内地各大城市封城制造的大量人道灾难，已掀起海内外同胞极度的愤概，更多的受害者意识到要与邪恶中共政权进行殊死搏斗，中共内部瓦解的日子已近，重光香港的伟大工作，上帝只会交予和他同行的勇士们。
