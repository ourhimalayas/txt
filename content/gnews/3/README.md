2022年3月26日，美国总统拜登在波兰发表演讲。拜登总统以轻松诙谐的方式说：天啊，上帝啊，普京啊，你就赶快别干了！看似轻轻松松的一句话，实际上宣告了普京政治生涯的结束。

文贵先生告诉我们，拜登总统的这句话，超过了李根总统的“推到柏林墙”，跨过了美国和国际规则的底线。以文贵先生对拜登总统48年的了解，拜登先生作为老牌专业政客，会倾其一生之力，做出你想不到的动作。拜登总统不会放过这次千载难逢的机会，他就像一匹伺机而动的狼，会将猎物吃的非常精彩。

苏俄帝国一直以来是世界的威胁，是文明和正义的对立面。如今的俄罗斯悍然对乌克兰发动武装侵略，公然破坏和平、挑战世界秩序。在乌克兰人民的英勇抵抗和猛烈打击下，犹如掉进陷阱里的狗熊。拜登总统岂能放过这个天赐良机，解除对美国和世界的威胁。而且，今年11月份美国举行的中期选举，也不允许拜登放过普京。

拜登总统的演讲明里说的是普京，暗里针对的却是习近平。习近平与普京两个独裁者抱团取暖、狼狈为奸。习近平和中共才是乌克兰战争的真正元凶。正是习共源源不断地输血，才使得普京如此嚣张。如果说普京是美国和世界的威胁，那么习近平就是最大的威胁。

拜登总统的波兰演讲发出了一个强烈的信号，美国永远与独裁势不两立，追求民主自由的人民将得到美国的支持。但是支持终归是支持，最终刺向独裁者的致命一剑还要靠本国人民出手一击。

如果习近平胆敢武力攻打台湾，美国或许将中共的所有军事基地摧毁。但终结独裁者，奔向自由要靠中国人民自己。这就是文贵先生一再强调的不能指望美国灭共，最终灭共的一定是中国人。

俄罗斯人民可以推翻普京，中国人民也一定能够推翻习近平和共产党。美国可以解除中共的武装，但是不能解除中国人民思想上的禁锢。

幸运的是有爆料革命和新中国联邦。文贵先生领导的爆料革命已经走过了5个年头，唤醒了千千万万善良的中国人民，尤其是党内的正义之士。从众多墙内战友被“喝茶”时受到的优待可知，爆料革命已经深入人心。

真相的力量是无穷的。爆料革命让人们看清了中共的假、骗、偷、恶的本质，看清了习近平等几大家族的丑陋、贪婪，看清了只有铲除共产党中国人民才有希望和未来。

拜登总统的波兰演讲预示着中共终将覆灭的下场，残暴的独裁者习近平会被内部人干掉，这些人很可能是爆料革命的战友。拜登总统的演讲既宣告了普京的结束，也敲响了习近平的丧钟。
