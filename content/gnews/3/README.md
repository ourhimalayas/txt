据《纽约时报》2月18日报导，中共使用机器人、虚假帐号等工具选择性编辑宣传故事，以影响有关冬奥会讨论。

《纽约时报》与非营利调查新闻编辑室ProPublica发现了Twitter上一个中共虚假帐号网络。网络的超过3000个帐号通过分享带有相同评论的中共外宣推文，以协调行动影响冬奥会讨论。这些帐号普遍创建不久，没有关注者，推文仅以转发中共外宣推文为主。

目前Twitter已经通过电子邮件声明，《纽约时报》与ProPublica认定的数百个账户已被平台封禁，同时Twitter表示将进一步调查这些帐户与国家支持的信息运营的联系。

中共冬奥会的虚假不仅在于虚假帐号操纵舆论。根据郭文贵先生在大直播中的爆料，中共冬奥会场雪为人造假雪，裁判被中共买通非公正判罚，中共参赛运动员国籍造假。中共冬奥会的呈现只有虚假和丑陋。
