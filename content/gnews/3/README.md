美联储为何有这么大的威力，可以摧枯拉朽要了独裁者的千金之躯？会左右各个行业，各个国家人们的生活？那为什么不早结束这苦逼的人类命运？

美联储没有菩萨心肠，美联储不是慈善机构，美联储是美国人的美联储，美联储更不是法制基金和新中国联邦（帮助中国人只能是中国人自己——新中共联邦）。可是，当美国国家利益受到威胁时，美国人的命运受到挑战时，美联储定会挺身而出，让这个世界尘归尘，土归土。当然，美联储背后的拥有者是个人。但是长期来看，他们是为美国服务的。

之所以美联储会用美元数量的变化，以及使用货币的廉价和昂贵这种起起伏伏，忽高忽低左右世界的格局，制造和生产着独夫民贼。今天，美联储在维护美国利益的必须时，会收走曾经给予独裁者的权利和金钱。以往不远的历史给了我们很多的证据。不要以为前苏联解体是政治妥协和民主的力量。千万不要忽略，苏联垮台美联储呈现的巨大能力！

看看上世纪八十年代，里根总统时期美联储货币使用成本突然变得多么昂贵，今天的人们无论如何也是不敢相信，也不能接受。年基础利率11.5%/年，没错这才是射向苏维埃的致命子弹。美联储这一轮加息时间周期为1982年12月到1984年9月，当时基准利率从8.5%上调至11.5%。到苏联已是解体进行时的1989年，美联储的基准利率还维持在9.81%。在那个石油价格相对低廉，而前苏联境内缺少国际先进科技，科技产品和一些生活必须品，购买所支付的国际硬通货币（美元，英镑，马克）绝对稀缺，再加上使用成本突然飞入云端，俄罗斯的前身苏维埃不死才怪。
