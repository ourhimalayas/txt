近日，有网传消息称，自2022年3月1日起，微信、支付宝的个人收款码不得用于经营性收款。近4年的数据将被追查，一旦数额较大需按4.5%补税，还要缴纳滞纳金和罚款。该消息爆出后，引发全社会的关注。

2月15日，微信和支付宝针对传言紧急辟谣，但此举仍未打消社会普遍疑虑。去年，中共对网商、微商、网络名人大面积查税，被查者轻者均被补税和重罚，重者获牢狱之灾，引发大面积恐慌。

去年12月，中共官方两起查税案件至今让相关从业人员胆战心惊。去年12月初，“微商教父”龚文祥因用个人微信收款，被工商税务公安法院等专案组的联合查处，不仅补交**45%的**[**个税**](https://finance.sina.cn/app/taxcalc.shtml)，高额处罚个人已经到了负债累累，卖房卖车，倾家荡产，身无分文的灭顶之灾地步。龚文祥的一封公开信流传甚广，“无论你过去4年还是现在只要是微信收款的，被大数据检测到未来一定是按照你微信收款总额45%补交个人所得税。”

去年12月下旬，浙江省杭州市税务部门通过税收大数据分析，查处网络主播薇娅偷逃税款，并作出追缴税款、加收滞纳金并处罚款共计13.41亿元。

当前，中共经济开始全面崩溃，企业倒闭潮失业潮加剧，税源不足矛盾突出，中共开始进入全面收割韭菜的疯狂阶段。
