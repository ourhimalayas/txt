根据大纪元3月3日讯，美国司法部周三表示，将成立一个特遣队，帮助展开针对俄罗斯政府官员和寡头实施制裁。

俄罗斯自2月24日开始入侵乌克兰，目前战事激烈，引发全球谴责和全方位制裁。

据美国司法部新闻稿介绍，KleptoCapture特遣队将致力于执行美国与盟国和合作伙伴一起实施的全面制裁、出口限制和经济对策，以应对俄罗斯无端对乌克兰的军事入侵。

司法部声明表示，特遣队将利用美国司法部所有工具和权力，来打击逃避或破坏美国政府为应对俄罗斯军事侵略而采取的经济制裁的行动。特遣队的任务将包括：

调查和起诉违反针对乌克兰入侵而实施新制裁和未来制裁的行为，以及针对俄罗斯先前侵略和腐败事件实施的制裁；打击那些破坏对俄罗斯金融机构制裁的非法行为，包括起诉那些试图逃避“了解你的客户”（原则）和反洗钱措施的人；针对使用加密货币逃避美国制裁，清洗外国腐败所得或逃避美国对俄罗斯军事侵略反应的努力；以及利用民事和刑事资产没收授权，扣押属于受制裁个人的资产或被确定为非法行为所得的资产。

声明还透露特遣队可以使用最尖端的调查技术，以识别和逃避制裁相关的犯罪及不当行为。

司法部表示，在有事实和法律支持的情况下，将寻求逮捕和起诉。
