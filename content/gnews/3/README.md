柏林国防部发言人周三表示，一艘德国军舰近20年来首次驶入南中国海。这艘德国海军舰将在前往新加坡的途中穿越南中国海。

德国政府曾明确表示，这次任务是为了强调德国不接受中共国有关南海的领土要求，以保证流经该水域的、占欧洲40%对外贸易的通道安全。预计这艘护卫舰将不会驶过台湾海峡。

除德国外，英国、法国、日本、澳大利亚和新西兰等国家也一直在扩大其在太平洋地区的活动，以对抗中共国在南中国海的影响力。
