据ASKANEWS新闻社报道，俄罗斯政府已批准对俄罗斯、俄罗斯公司和俄罗斯公民采取敌对行动的外国国家和地区名单。俄罗斯总理米哈伊尔•米舒斯京已就此事签署命令。

敌对国家

根据国际文传电讯社引用的文件， 该名单包括欧盟国家——包括意大利、澳大利亚、阿尔巴尼亚、安道尔、英国（包括泽西岛和英国海外领土——安圭拉、英属维尔京群岛、直布罗陀）、冰岛、加拿大、列支敦士登、密克罗尼西亚、摩纳哥、新西兰、挪威、韩国、圣马力诺、北马其顿、新加坡、美国、台湾、乌克兰、黑山、瑞士和日本。
