**日本政府披露了美前总统老布什对中国“天安门事件”后如何确定共产党执政的合法性以及人权问题的担忧。回顾历史，它突出显示了当今国际社会的烦恼，这和美国对中共确立的“绥靖政策”紧密关联。给这个共产中国收买并试图统治全世界带来了无限的机遇和权力欲望。当国际社会仍在思考对策时，习近平对进攻台湾的定义是“两岸同胞的心灵契合”，这一表述是一个狠毒的攻台计划。**

据日本《[朝日新闻](https://www.asahi.com/articles/ASPDH5D3QPD6UTFK026.html)》22日消息称，日本政府按照惯例，披露了1990年美国总统老布什与日本时任首相海府俊树寻求恢复关系交流中的外交文件。文件曝光了中国1989年“以武力镇压民主化运动的天安门事件后，” 突出显示了时任美国总统老布什对“如何确定共产党执政合法性、促进人权问题的改善”所做的努力是当今世界的主要忧虑的根源。

消息透漏，1990年3月在美国西部度假小镇棕榈泉举行的日美首脑会谈的晚宴上，布什先生和海府俊树谈到与中共国的关系时表示，“我的基本方针是与中国保持联系，推动变革。”这即是基辛格主导的美国对中共的“绥靖政策”的由来，从日本公布的文件看，显然老布什在处理天安门事件时也心有余悸，他说，“但我对未来感到担忧。中（共）国跟上日本和美国（民主）的步伐很重要。”老布什在对中共国问题的处理表示担忧是可以理解的，这就是美国今天的困局以及中共国疯狂所在。

据日本外交部2020年12月23日发布的一份外交文件中显示，当时中共国的最高领导人[邓小平向日本商界访华团讲述了1989年6月4日天安门事件中国共产党政府打压寻求民主的学生](https://www.asahi.com/articles/ASNDQ7R6KNDCUTFK01H.html?iref=pc_rensai_short_1142_article_2)的情况，邓为镇压的正当化辩护称，“国家权利压倒人权”。

据称，1989年11月13日，日本经团联的会长斋藤英四郎访问中国时在人民大会堂谈到天安门事件时，中方代表谎称：“我们对最近动乱中的学生非常宽容，但这次动乱是由思想混乱造成的。”他还谈到了（同年7月）当时欧洲、美国和日本指责天安门事件“压制人权”恶劣事件，称，“如果我们谈论人权重还是国家权利重，国家权利是关于独立、主权和尊严的，它压倒了一切（人权）。”并强调了这是中共与民主世界“价值观的差异”。
