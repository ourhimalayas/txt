英国国家医疗服务系统(NHS)的血液和移植服务部门发出警告称，英国在冬季正面临着“危急”状况，因为如感冒和流感等季节性病毒停止献血，如感冒和流感，使血液库存处于一个危险的低水平。
尽管该部门寻求试图在整个英国各地维保持至少五天的（血液）储备，但其库存目前正面临有可能下降到两天以下的危险。

这是在过去12个月来中，国家医疗服务系统第二次表示它正面临重大事件危机状况。今年夏天，该机构系统警告称，表示其正在努力应对因中共病毒大流行新冠疫情使造成的急救电话的数量增至加到有史以来的第二高的情况。
英国国家医疗服务系统血液和移植服务部门在一份声明中表示：“冬季对血液库存来说一直是一个具有挑战性的时期，但季节性的压力正在提前发生，我们担心天气变冷和更高的感冒和流感发病率，再加上中共病毒新冠疫情，包括疫苗加强针计划，可能使今年的情况更加严峻。”

英国国家医疗服务系统呼吁人们在这一困难时期支持国家医疗服务系统，并敦促“现有的和曾经的献血者”“现在就行动起来，帮助建立库存……以确保病人继续获得他们所需的救命血液。”英国国家医疗服务系统之前曾警告说，如果不改变目前的状况，从2022年1月到3月，英国国家医疗服务系统将面临“库存不足”的问题，迫使国家医疗服务卫生系统审查其冬季过后的措施。
虽然不是供应量低的主要原因，但该服务提出引起了人们对中共病毒大流行新冠疫情的长期影响的担忧，过去两年的情况“引发了人们对捐赠水平的担忧”已经“促使人们关注”捐赠水平。
