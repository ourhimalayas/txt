**【墙国谎言】栏目**
![](https://assets.gnews.org/wp-content/uploads/2021/12/ay7sp-mtrwj.jpg)**图片来源： Getty Images**
作为西藏高原的主人，藏族人的基因与祖先及生活在平地的其他民族有重大区别。据《自然》杂志于2017年报道，考古发现藏族人的遗址最早可以追溯到127000年至7400年前。 2010年科学发现，西藏人和汉族人的基因组里有少数基因具有显着区别，分别是EPAS-1和EGLN-1这两个基因，主要用来调节血液中血红蛋白的能力，这一特征也是西藏人更适应高原气候的原因。西藏人的古代祖先可能是丹尼索瓦人（Denisovans，生活于40000前）。这些发现，远远早于中共鼓吹的“五千年中华文明”，更可以证实“西藏自古以来是中国的一部分”的荒谬。

中共对于西藏人的血腥镇压和屠杀由来已久，对外却大肆宣称说西藏人口从1949年以后，一直在增加，而且所占比例超过百分之九十五。利用中共自己发布的数据，依照科学规律进行分析，可以明确揭示中共对于西藏的人口谎言。

**人口数量不一致**

西藏人口数量从2017年的330万增加到2021年700万，人口接近翻了一倍（表1）。然而，2020年的总人口是364万，到2021年，人口增加336万，在西藏这种生存条件有限制的条件下，这种人口增长速度，人类历史上都没有出现过。郭文贵先生指出，2014年，西藏实际人口接近420万，较为接近中共国1997年报道的459万。

中共宣称，西藏自治区一直没有实施计划生育，而人口却没有增加。按照2.238%生育水平，根据种群增长理论，人口应该缓慢增加，为何到2014年时人口反而减少39万呢？据中共记载，西藏没有大自然灾害发生，也没有政治事件或者大量藏人流亡，人口骤减几十万，什么原因造成的？中共真的在西藏没有发生大规模镇压屠杀么？曾有国外研究报道，1961年，西藏人口约为600万，假如这一真实数据可信，那么减少约一半藏人该如何解释。中共国的造假数字游戏在西藏人口数量上体现的淋漓尽致。

**低于更替生育率却保持人口增长**

人口增长理论里面有个重要概念——更替生育率（也称世代更替水平），生育率接近更替生育水平是人口维持长期稳定的必要条件，一般更替生育率为2.1%左右。若要人口增长，则更替生育率要高于2.1%。西藏人口生育率仅为1.05%左右，远低于更替生育率，甚至低于中共国2021年所报道的总人口生育率1.8%（表1）。那么西藏仅为1%左右，人口怎么能增加呢？从2017年到2020年，增加的34万人口怎么凭空冒出来的？更不要提从2017年到2021年人口翻倍增加。这些不符合科学人口增长规律的人口增长神话只剩下两种可能——要么数据造假，要么大量外来人口涌入西藏。而中共国让西藏藏人保持自治，藏族比例占到95-92%（表 1），否认大量外来移民进入西藏。最后的可能只剩下结论——数据造假。

中共一贯对外坚称藏人治臧，没有大量移民进入西藏。藏人作为西藏人口主体，达96%。按照西藏人口生育率，不可能实现人口增长，那么增加的人口是什么人？不言自明，只能是大量外来移民。而汉族人口的逐年增加也有数据证实（表2）。从1964年，到2000年，汉人增加了一倍，造成这一结果只有二种可能，要么汉族人口生育率高，要么就是引进汉族人口。考虑汉族人必须计划生育的情况，不可能36年内，人口翻倍，加上汉人因为身体特性的原因，在高原的生育率会受到影响，那么只剩下引进汉族人口这一可能。

据《北京林业大学2021年非西藏定向西藏就业招生简章》说明，将从四川、云南和重庆进行高校招生，学生毕业签就业合同，定向到西藏去工作，合同期限约为5年。这正是从内地引入汉族人口进藏的一个佐证。这种招生政策从2005年已经开始，每年人数约为上千人。
