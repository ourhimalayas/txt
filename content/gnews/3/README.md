以假治国是中共一直笃守的信条之一，当然假话或隐或现，不过有关数字的造假，稍加推算就能戳穿。

今日环球网报道了一则中共央视消息：“就业新观察 | 数字经济催生更多新职业，到2025年带动就业人数将达3.79亿”。不读全文，仅看标题我们就可知有多假。

首先，我们了解一下什么叫数字经济。数字经济又称互联网经济、网络经济、新经济，是泛指通过数字计算技术为基础，由数字产业带动的经济活动，以及非数字产业借由数字科技的创新经济活动。也就是说，1）数字经济是一个“经济活动”；2）需要和互联网、物联网相关。

其次，我们来看看中共发布的二组数据：1）中共国家统计局公布，2021年末，全国就业人员74,652万人（7.4652亿），其中城镇就业人员46,773万人（4.6773亿）；其余的被称作农民工，总量为29,251万人（2.9251亿）。2）中共经济体制改革研究会副会长、北京改革和发展研究会会长陈剑称：到2014年年底，中国财政实际供养人数远超过6400万（0.64亿）。

最后，我们了简单计算分析一下；“到2025年，数字经济带动就业人数将达到3.79亿”。如果不计2022年到2025年适龄就业人口变化，到2025年就业人口结构同2021年末相仿，那么4.6773亿城镇就业人口减去0.64亿不参与“经济活动”的财政供养人员，从事“经济活动”的就业人员为4.0373亿；而农民工的工作基本不参与互联网、物联网相关的“数字经济”。那么，到2025年（3.79除以4.0373乘以100%）近98.9%的城镇就业人口将从事数字经济工作。即使农民工有所减少，即使不计未来3年劳动人口的减少趋势，甚者不计2015年到2025年10年间行政供养人数的增加，那么起码95%的城镇就业人口从事“数字经济”工作是只多不少的。

2025年中共国将有95%以上的城镇就业人口从事“数字经济”，类似这样漏洞百出的荒唐谎言，却出现在中共最高喉舌平台，并公然忽悠大众百年如一日，是中共的无耻傲慢，也是奴隶的愚昧懦弱。
