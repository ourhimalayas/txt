华盛顿时事报道，美军参谋长联席会议主席米利4月5日在众议院军事委员会听证会上表示，俄罗斯入侵乌克兰战争将“至少几年”。他表现预计将成为长期化。

他还坚持应在东欧建立一个新基地，以扩大美军的存在，以保护欧洲盟友免受俄罗斯的威胁。

米莉先生指出，“（乌克兰战争）将作为一场非常长期的冲突持续下去，不是几十年，而是几年。” “北大西洋公约组织（NATO）和所有支持美国和乌克兰的盟国将在相当长的一段时间内卷入这场冲突，”他说。

随着北约附属东欧国家对俄罗斯变得更加警惕，米莉“提议（在东欧）建立一个永久基地，美军在此不是长期驻扎，而是巡回驻留。”他表示希望美军驻留扩大的波兰、罗马尼亚和波罗的海国家承担建设基地的费用。
