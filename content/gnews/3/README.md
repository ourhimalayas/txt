日本正式宣布金融制裁侵略乌克兰的俄罗斯，在最后一刻跟著欧美各国一同谴责，未来日俄的经济关系也将降到谷底。回首过去，日本曾经在普京就任总统时，一度想修补两国外交关系，其中不但有历史背景，也有两国领土争议，而如今制裁下，两国未来对话短期将难以预见。日本首相岸田文雄在2月27日晚上紧急会议结束后对记者表示，将对俄罗斯进行金融制裁。主要的内容包括，支持将俄罗斯排除在环球银行金融电信协会（SWIFT）机制内、并冻结了包括普京在内的政府相关人士的资产。最大的支持，则是给予乌克兰一亿美元的紧急人道援助，坚定站在乌克兰一边。岸田文雄严肃指出“对于俄罗斯的关系已经无法像往日那样正常的看待了”，显示日俄关系也面临到关键时刻。此外，对于帮助俄国侵略的白俄罗斯，日本官方也在考虑对白俄罗斯的官员们进行一系列制裁。

随著日本正式制裁，这也代表过去日俄外交上的一段蜜月期将正式结束。特别是普京从2000年就任总统以来，始终与日本保持一定程度互动。爱好柔道的他，甚至还担任国际柔道总会的名誉主席，在这次入侵活动后该职位也遭到取消。无可讳言，在这次俄罗斯入侵乌克兰后，不仅给了欧洲各国一记猛棍，让各国开始计划加强军备与给予乌国援助外，也影响到日本的军武战备。身为G7中唯一与俄国邻土接壤的国家，前首相安倍晋三也在27日上电视台节目时，重申日本要有“适时快速地武器打击能力”，以应付未来俄罗斯、中共国与朝鲜等武力威胁。

日本除了是G7唯一亚洲国外，也是同时跟中俄相近，且都有领土等历史遗留问题的国家，身在其中如何选择，势必相当困难。与俄关系，从过去一度想修补，但最后仍是“一刀两断”。如果将来台海发生任何擦枪走火，同样的困境也会再度发生在日本身上，未来几年，日本也会走出更清晰地外交路线。
