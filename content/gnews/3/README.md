中美这两个经济超级大国之间的紧张局势现在重新浮出水面，这次摩擦的对象是中国电信公司。

美国广播和通信监管机构联邦通信委员会 (FCC) 实际上已经撤销了允许中国电信在美国运营的许可证，并提到这家中共国公司可能会渗透关键网络，从而带来安全风险。

作为独立机构的美国监管机构以强烈的态度一致通过：以四票对零的结果特别确定中国电信是受中共国政府的影响和控制，北京的影响构成了重大的国家安全和秩序保护风险，这家中共国政府公司为访问、存档、破坏和转移美国通信提供了机会。

美国联邦机构的决定要求在 60 天内终止中国电信在美国的服务。中国电信获准了在美国20个州开展业务：联邦牌照被撤销的消息导致在美上市的中共国公司股票立即下跌。
