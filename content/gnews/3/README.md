中国房地产危机已经导致部分地方政府财政破产，中国经济正在受到波及。世界银行对此发出警告，中国房地产低迷引发的房价下跌，会给中国经济造成重大冲击。

路透1月12日报导，世界银行（The World Bank）1月11日在最新发布的《全球经济展望》中警告，金融压力可能导致中国房地产行业的无序去杠杆化。

报告指出，动荡的去杠杆化可能会导致中国房地产行业的长期低迷。

英国《金融时报》1月11日报导也说，尽管恒大导致的直接问题似乎得到控制，但至少就目前而言，房地产行业下滑的影响正在波及整体经济。
