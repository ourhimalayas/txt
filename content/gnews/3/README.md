## 美国高级官员将因中国问题将访问所罗门群岛

翻译&评论：金生水
![](https://assets.gnews.org/wp-content/uploads/2022/04/image-2608-16.png)![](https://assets.gnews.org/wp-content/uploads/2022/04/1-386.jpg)
图片来自网络

《路透社》近日报道，白宫周一表示，白宫印太协调员库尔特-坎贝尔和国务院负责亚洲事务的最高官员本周将前往所罗门群岛，因为人们担心这个太平洋岛国正在与中国达成安全协议。

白宫国家安全委员会在一份声明中说，坎贝尔和负责东亚和太平洋事务的助理国务卿丹尼尔-克里滕布林克将带领一个包括国防部和美国国际开发署官员在内的代表团前往三个国家：所罗门群岛、斐济和巴布亚新几内亚。

该小组还将在夏威夷停留，以 “与美国印太司令部的高级军事官员和区域合作伙伴进行磋商”，声明中没有给出旅行的日期。

国务院发言人内德-普莱斯说，对中国的关切将是与所罗门群岛官员讨论的主题之一。

美国对该地区的政策是确保各国了解与华盛顿接触的好处，”而不是关于中国或任何其他国家，”普莱斯在定期新闻发布会上说。

“他说：”我们将让他们把我们提供的东西与其他国家，包括该地区相当大的国家，可能提供的东西进行对比。

今年2月，美国宣布将在所罗门群岛设立大使馆，这是拜登政府为对抗中国的更大影响力而向印度-太平洋地区投入更多外交和安全资源的努力的一部分。

所罗门群岛在3月说，它正在与中共国建立伙伴关系，以解决安全威胁并确保安全的投资环境，这将是北京在美国盟友澳大利亚和新西兰几十年来一直视为其 “后院 “的地区的一个重大进展。

在区域性反弹之后，所罗门群岛说它不会允许在那里建立中国的军事基地。

但美国官员仍然担心，与中共国拟议协议的广泛性质为在所罗门群岛部署中国军队打开了大门，普莱斯说。

“他补充说：”我们认为，签署这样一项协议可能会增加所罗门群岛内部的不稳定因素，并将为更广泛的太平洋岛屿地区树立一个令人担忧的先例。

澳大利亚国防部长彼得-达顿说，中国希望在太平洋岛屿上获得一个军事立足点，包括在巴布亚新几内亚建立一个 “军港”。

中共国在2018年提出在巴布亚新几内亚重新开发一个海军基地，但澳大利亚最接近的北方邻国决定由澳大利亚来开发该基地。
