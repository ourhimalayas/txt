中共不断清除香港殖民地色彩，各大纪律部队已转用中式步操，但司法系统的殖民色彩需要改革，如现时法庭审讯有部分使用英文，区域法院及高等法院的法官，审案时仍须穿戴假发及法袍。有学者及法律界人士认为，港府应加快除殖民步伐，安排让法官“脱发”，并尽量多安排以中文审讯，以增国民身份认同；另应考虑逐步撤换外籍法官、被指纵容及放生修例事件暴徒的“黄官”，或工作不理想的法官。

中共利用镇压反逃犯条例运动夺取了香港的控制权，却非常惧怕英国殖民痕迹的影响力，一直伙同港府偷偷清理殖民时期的建筑物，收购山顶欧式教堂进行重建，以结构问题拆除有文物价值的地下储水库，重置香港历史博物馆，重建香港警察博物馆……中共对香港的每一个祛殖民化动作都引起港人极大的愤慨，至少以人民的力量挽救了深水埗主教山上，罗马式地下储水库被拆毁的命运。中共在去除殖民文化方面遇到了阻力，但在政治体制上的去殖民化却顺风顺水，中共首当其冲毁掉了香港的民主制度，破坏了香港的三权独立，篡改香港的选举制度，将立法会彻底清理，由清一色的中共走狗建制成员进驻，用于辖制港府的运作，贯彻中共的意图。公务员被要求宣誓效忠中共，纪律部队被要求中式步操。今日的报章将香港的司法系统也列入去殖民化中，历数司法界不符合中共心意的殖民色彩，比如讲英文，带假发等，要求尽快处理。中共卑劣易碎的自尊心享誉全球，一句话、一张图、一首歌就能被定义为辱华。

香港被英国管制了百多年，无处不在的殖民色彩正是香港成为东方之珠的魅力源泉，但这一切的美好都被中共简单粗暴地捣毁。中共对外假骗偷、蓝金黄，对内奴役人民，对人类没有半点益处，因此中共在民富国强的民主制度面前极度自卑，但它却缺乏身残志坚的向上精神，一味愚蠢地推倒重建，要将内地和香港拉回到万恶的奴隶社会。
