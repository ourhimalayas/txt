![](https://assets.gnews.org/wp-content/uploads/2022/03/3071.jpg)乌克兰马里乌波尔政府说，俄罗斯军队周日（3月6日）再一次违反停火，当地民众未能安全撤离。 （彭博社）

早报讯：乌克兰南部港市马里乌波尔（Mariupol）地方政府说，俄罗斯军队周日（3月6日）再一次违反停火，当地民众未能安全撤离。

乌克兰马里乌波尔市政府当天较早前宣布，俄军同意于新加坡时间周日（3月6日）下午4时至周一凌晨3时暂时停火，让当地民众再一次撤离。这是马里乌波尔市政府[第二次与俄军达成暂时停火协议](https://www.zaobao.com.sg/realtime/world/story20220306-1249523)，试图让民众安全撤离。

路透社报道，负责撤离人员的车队周日未能离开乌克兰，目前仍被围困在马里乌波尔。当局说，地方政府已与俄军达成临时停火协议，但俄军仍在继续炮击。

市议会发声明说：“在这种情况下带人出去极其危险。”

据报道，俄罗斯周六宣布在乌克兰黑海城市马里乌波尔和附近的沃尔诺瓦哈（Volnovakha）暂时停火并开通人道走廊，让受困的居民离开。

不过，当地政府指俄军没有遵守停火安排，继续在市内外进行轰炸，撤离行动被迫延后。
