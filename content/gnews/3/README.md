## 美国CDC预测下个月新增COVID-19死亡病例62000例

![](https://assets.gnews.org/wp-content/uploads/2022/01/图片1-75.png)
合众国际社（UPI）12日报道。美国疾病控制和预防中心（CDC）周三预测，未来一个月，美国可能新增6.2万人死于COVID-19（中共病毒）。

CDC发布了根据23个建模小组数据编制的月度全国总体预测，预测从1月10日到2月5日，全美死于病毒大流行的人数将在36000到62000之间。

CDC说，在接下来的一个月里，每周死于COVID-19的人数将增加。

截至1月14日的一周将有8000至14100人死亡；截至1月22日的一周有8800至18000人死亡；截至1月29日的一周有8700至24700人死亡；截至2月5日的一周有10400至31000人死亡。

美国周一报告了近1900例COVID-19的新死亡病例，这是现有最新的数据。自中共病毒大流行开始以来，美国的七天移动平均数为每天1600多人，累计死亡人数为837274人。

根据CDC的统计，随着Omicron病毒的变体导致病例增加，自11月下旬以来，每日死亡人数与日俱增。

美国周一报告了破纪录的140万个新病例，自2020年初大流行开始以来，累计有6173万个病例。

虽然科学家们怀疑Omicron比COVID-19的其他变种更温和，但病例数量之多，已使全国医院的COVID-19入院人数创下新高。

美国卫生与公众服务部显示，截至周二晚上，有超过14.5万人因该病毒住院。

疾病预防控制中心预测，在接下来的四周里，每周的新住院人数都会增加，该预测的最后一天——2月4日，新住院人数将达18000至48000之间。
