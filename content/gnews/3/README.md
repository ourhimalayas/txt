## 泽连斯基呼吁以色列提供&#8221;铁穹&#8221;导弹防御系统

**撰稿：日本东京方舟农场 北极**

据新闻周刊3月20日报道，乌克兰总统泽连斯基当天向以色列议会发表视频演讲，呼吁提供”铁穹”导弹防御系统，同时敦促以色列制裁俄罗斯。

![](https://assets.gnews.org/wp-content/uploads/2022/03/0-11.png)乌克兰总统泽伦斯基（Volodymyr Zelensky）呼吁以色列分享其着名的”铁穹”导弹防御系统（图片来自新闻周刊）

泽连斯基表示，以色列的防空系统是世界上最好的，特别是”铁穹”反导防御系统。泽伦斯基随后呼吁以色列为乌克兰提供更多的军事援助。

据以色列媒体报道，该国并不打算转变外交路线从而向乌克兰提供武器。以色列总理纳夫塔利·贝内特（Naftali Bennett）一直都在乌克兰和俄罗斯领导人之间进行调解，期待达成和平解决方案。

自俄罗斯入侵开始以来，以色列向乌克兰提供了大量人道主义援助，但迄今为止拒绝了泽伦斯基提供军事支持或对莫斯科实施经济制裁的请求。
