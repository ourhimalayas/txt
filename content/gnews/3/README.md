## 美国的债务水平将自1945年以来的峰值

![美国债务将爆增，我们投资的美债还安全吗]()![美国债务将爆增，我们投资的美债还安全吗](https://pic1.zhimg.com/v2-4ef2b3a43ca72fb7f830cde4867fe1e1_1440w.jpg?source=172ae18b)

本周公布的预估表明，国会的赤字将相当于国内生产总值的13.4%，并将成为自1945年以来历史上第二大赤字，这主要是由于新冠毒大流行的相关支出所造成的。

乔·拜登总统和前总统川普批准了近6万亿美元的联邦救济，用以对新冠病毒的危机作出大规模应对，因为数百万美国人面临失业和金融不稳定的情况。

美国国会预算办公室（CBO）主席Mayra MacGuiness在一份声明中说：“美国国会预算办公室对经济强劲增长的预测表明，现在是时候从进一步的举债融资转向偿还债务了，并最终实现国家债务的减少。”
