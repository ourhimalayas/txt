## 美国通过136亿美元紧急预算支持乌克兰
` 東京櫻花団-Tokyo Sakura Group` [轉載自GNews](https://gnews.org/zh-hans/2173516/)

撰稿：东京樱花团/文小白

据NHK 3月16日报道，美国总统拜登签署了一项法案，包括向乌克兰提供武器和人道主义援助等总额为136亿美元的紧急预算，并表示，“我们将迅速采取行动为乌克兰提供更多支持。”该法案获得通过。

![](https://assets.gnews.org/wp-content/uploads/2022/03/美国通过136亿美元紧急预算支持乌克兰.png)[照片链接](https://gnews.org/wp-admin/アメリカ%20ウクライナ支援に日本円で1兆6000億円の緊急予算)

总计136亿美元的援助，其中35亿美元用于向乌克兰提供武器，30亿美元用于向乌克兰周边国家派遣美军，约40亿美元用于包括食品和食品等人道主义支援。

拜登总统表示，“我们将迅速采取行动，进一步支持乌克兰人民。”称将尽快落实方案内容。

自去年1月拜登政府就职以来，美国对乌克兰的军事援助总额达到了12亿美元，此次紧急援助远远超过了一前的规模。
