3月17日，在刚刚结束35小时外出禁令的乌克兰首都基辅，记者向留在基辅市中心的日本人询问了现在的街道情况。

高垣典哉（56岁）连日来一直在宣传在基辅的生活。不分昼夜的炮击声一直传到高垣的家中。

（3月9日）“这个时期几乎没有人走路。”

（3月15日）“攻击在10分钟前又开始了。这个声音太恐怖了。”

（3月16日）“从昨天开始下达了35个小时的外出禁止令。哇，这种响音从昨天开始渐渐传来了，好像一直通宵达旦。”

市内的电子屏幕上出现了批判俄罗斯士兵的话语。“俄罗斯士兵○○○…脏话连篇。”

市民的生活是怎么样的呢？图为，3月8日拍摄的地下停车场。没有车的空间里放着床垫和桌子等。据说，为了躲避炮击，有些人晚上都呆在这种地下停车场。 超市里好像有食材进货，但是听说价格正在上涨。——3月9日，“美国利百艾牛排”的肉价是多少呢？“这个一直300~400UAH的价位，现在是1535UAH（约4倍），价格正在大幅上涨。”
