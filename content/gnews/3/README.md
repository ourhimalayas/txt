威廉王：最开始的时候说要改女人花这首歌，小唐姐说：“这首歌一个是慢，一个是旋律线条都比较复杂，七哥能唱吗？ ”我说怎么不能？！我说七哥绝对能唱，我就是有这种信心，绝对能唱。我是一个特别感性的人，加入爆料革命之前像这些特别悲惨的事情，特别虐心的事情我是真的一点儿都没法干。但是加入爆料革命之后，我发现我真是变成了一个战士，就是不仅是从行动上，从心理上也是，我可以面对任何事情。现在任何的人间的悲惨不幸我都能面对，因为我知道发生这些的原因。

我觉得音乐是什么？音乐就是一种感觉。如果你做的音乐能把你内心的感觉完完整整地传达给倾听的人。一句话就听懂了，你的这个感觉、你要表达的东西就成功了。所以我一直觉得无病呻吟或者是你写的东西根本就不是你内心所想的，那出来的东西就是垃圾，绝对的垃圾。在写歌的时候就把自己放在那个角度去写，然后我就发现其实我就是这个锁链女，不是说我去站在她的角度上去想，而是我们每一个人就是锁链女。

我在写的时候，小唐姐说：“这个我本花一朵，不对吧？应该是她本花一朵。”我就跟她急了，我说这个不许改，因为这个你自己可以试一下，你改成“她”第三人称，感觉就完全没有了，而且这种引申的含义也没有了。我感觉一首歌非常（重要）的是第一有含义，第二非常有逻辑。我就想主歌要表达的是那种，我自己在改的时候觉得是挺难的，因为在写词的时候我是没有任何东西出来，就是那你写吧，就这么一句话，那我就要自己先在脑子里建立那种感觉，这个歌我想她是什么感觉，就有画面感出来，之后到副歌一定要唤醒人们对吧？你光说她惨是没有用的，你得做点什么？然后就有了副歌。其实一直到写完之后，我还是觉得差了点儿感觉，但是就是不知道差在哪儿，我觉得这句话“永世地狱做猪狗”一点毛病都没有，就是差了这么一点感觉，我觉得最后应该有一个轰轰烈烈的结尾，但是它的这个旋律到那儿就降下去了。但是当时曲还没有开始弄，制作还没有开始弄，只是我单纯去写词，所以我就没有办法突破当时的那种感觉，找不着那种最后轰轰烈烈的感觉。

为什么说七哥给这个首歌（增添灵魂之笔），就是你看我前面写了99%，就最后这一句我仔细听、反复听这首歌，我觉得最精华的应该最后一句，前面这种压抑的，小小的看法又压抑，最后也是压抑的，到结尾，尤其是那个升8度之后，喊出来：“砸碎了铁锁”，一下这首歌就升华了。

我不同意有人说是翻唱，我觉得这绝对是一个颠覆，从音乐上来讲绝对不是一个等级，绝对是一个颠覆！刚写完词之后我要给七哥录demo，然后录的时候，就是随便先编一个小小的编曲，然后我拿编曲做这个demo，真的是一张嘴，我写的时候还是觉得就是按照感觉这么写出来，哇，觉得真的有感觉。但是真到录的时候就真的是张不开嘴，我唱第一句话的时候就没办法唱，小唐姐还在那边工作，我录的时候（哭了），我还怕她笑话我。就跟七哥后来录歌的时候一样，张嘴就哭一首歌的时间。
