**马里乌波尔：废墟中的“英雄城市”**

到3月7日，马里乌波尔收到了最不受欢迎的花环，被乌克兰总统沃洛基米尔·泽伦斯基命名为“英雄城市”。没有一条街道是完整的。地面上到处都是尸体，未被埋葬，在灰烬、玻璃、塑料和金属碎片中腐烂。空白的眼睛回瞪着惊恐的孩子。

“一个女人的胳膊、腿和头都被撕下来了，”娜迪亚写道。“我确信我很快就会死。这只是几天的事。在这个城市，每个人都在不断等待死亡。我只是希望它没有那么可怕。”

![](https://assets.gnews.org/wp-content/uploads/2022/03/image-3277.png)人们在马里乌波尔接受人道主义援助。美联社照片

躲在市中心地下室的家庭只能喝水坑里的水或收集排水管里的水滴，并分享从被烧毁的房屋中抢救出来的食物残渣。唯一的救星是志愿者，他们用桶从一个运河里装满脏水，在地下室里分发。

3月8日，马里乌波尔的一名6岁儿童因脱水而死亡。她被埋在杀死她母亲的瓦砾下，被发现时已经太晚。第二天，儿童医院被炸毁。

![](https://assets.gnews.org/wp-content/uploads/2022/03/image-3278-edited.png)紧急救援人员和志愿者将一名受伤的孕妇从马里乌波尔被炮火炸毁的妇产医院抬出。婴儿是死胎。半小时后，母亲也死了。美联社照片

官方公布的平民死亡人数为1207人，但似乎是为了嘲弄这种接近可怕事实的假象，一名俄罗斯飞行员向剧院投下了一枚炸弹，该剧院有1300人用作避难所，上面的人清楚地看到两边用白色油漆写着“儿童”一词。

其中大约1000人在剧院的舞台下。尽管做出了各种努力，但此后没有任何关于他们的消息。只有这座庞大建筑中的其他地方的人能够踉踉跄跄地走出来，有些人怀里抱着他们瘫痪的孩子。该委员会周五说，至少有300人死亡，因为空袭后立即出现了内部屠杀的视频。

![](https://assets.gnews.org/wp-content/uploads/2022/03/image-3279.png)卫星图片显示，3月14日空袭前，该市一家剧院外的地面上用大号俄文写着“儿童”一词。

![](https://assets.gnews.org/wp-content/uploads/2022/03/image-3280.png)炮击剧院的后果图片，当时有数百名平民在那里避难。

顿涅茨克地区的巡逻警察部门负责人米哈伊尔·韦尔希宁告诉《卫报》：“我们离它大约200米，发生了空袭和巨大的爆炸，还有另一次巨大的爆炸……我们正沿着相邻的街道走，我们不知道剧院被击中。”

![](https://assets.gnews.org/wp-content/uploads/2022/03/image-3281.png)卫星图像显示3月19日马里乌波尔戏剧院被空袭后的情景。盖蒂图片

“我的一名士兵目睹了爆炸后发生的一切。他看到人们抱着自己的孩子，浑身是血。这对他的打击非常大，他看到后无法履行自己的职责。他崩溃了。”
