正如郭先生直播中提到的，北京是世界上污染最严重的城市之一，但是如果有一天乌云几乎消失、太阳出现，那或许意味着北京将举行重要的政治会议或具有国际意义的活动。而为了达到这个目的，中共就经常使用人工方式影响天气。

人工影响天气是指在适当的天气条件下，通过人工干预，使天气过程向符合人类愿望的方向发展。1946年就有美国的科学家提出, 在冷云中通过播撒干冰或碘化银（AgI）的方法，适当增加云中的冰晶数量，促使降水的形成，该方法开创了人工增雨作业的历史。中国的人工影响天气工作则始于1958年，目前作业规模为世界之首。2018年，全国有30多个省（区、市）开展了人工影响天气作业，使用飞机70架次，高炮和火箭作业点18057个，人工增雨作业目标的总面积约五万平方公里。

目前中国常用的人工降雪方式，是使用大炮，把化学药品装在炮弹里，然后用大炮发射到云层里。这种方法喷撒不均匀，药品浪费较大，增加了人工降雪的成本。还有人把它们装在土火箭里，让火箭飞到云里去喷撒。而这就是中共最常用的“造雪箭”。

这项技术使用的主要化学物质是碘化银。碘化银是碘和银的化合物，分子式为AgI，是一种亮黄色无嗅微晶形粉末，具有感光性，在光的作用下，会分解成极小颗粒的“银核”，逐渐变为带绿色的灰黑色。碘化银不溶于水和氨水。

碘化银对健康的危害包括：口服、局部接触或职业性长期接触，可引起局部或全身性银质沉着，发生局部或全身皮肤沉着症、眼部损害、慢性支气管炎等。

碘化银对环境的危害包括：对水体造成污染。

联合国环境保护署制定的清洁水法案中，碘化银被列为有毒物质。由于碘化银见光分解为碘离子和银离子，而银离子属于重金属离子，毒性很大，对微生物和鱼类的危害特别大。人体长期接触碘和银的化学物质也可出现中毒现象。

碘化银只要受热后就会在空气中形成极多极细（只有头发直径的百分之一到千分之一）的碘化银粒子。每一克碘化银可以形成几十万亿个微粒，这些微粒会随气流运动进入云中，在冷云中产生几万亿到上百亿个冰晶。而这些化学物质最后则会随着人工降雪，进入每一条河流，进入每一寸土地，进入每个人的饮用水、食物和空气中，这就是中共为了所谓的“瑞雪兆丰年”的六中全会，对中国人民犯下的又一滔天大罪。
