## 【每日聚焦】俄乌战事3月25日简要

![](https://assets.gnews.org/wp-content/uploads/2022/03/GNEWGETTR.jpg)
**撰稿：蓝莲花农场 文斌**

1、基辅西线：在过去24小时内，乌克兰军队没有进行任何新的反击，也没有收复基辅西北部更多的土地，但是迫使俄军处于守势。基辅州民政部门在当地时间3月24日上午6点报告说，在日托米尔公路沿线、伊尔平和霍斯托梅尔一代战斗还在进行，俄军主要意图是从这里突破，然后从西面包围基辅。基辅当局还报告说，过去24小时内，俄罗斯军队在Bucha和Irpin的西北部建造了新的战壕线，可能是为了抵御乌克兰未来的反击。

2、切尔尼戈夫和苏梅方向：3月24日，乌克兰军队击退了俄罗斯对Brovary的进攻。3月24日，俄罗斯军队继续进行零星的、不成功的攻击，包括对切尔尼戈夫以西30公里的Slavutych的进攻也是不成功的，其目的仍然是试图包围切尔尼戈夫，据一位美国国防部高级官员透露，俄军距离切尔尼戈夫市中心还有8-10公里。乌克兰总参谋部在当地时间3月23日午夜报告说，俄军正在切尔尼戈夫上空加强空中侦察，以进行战损评估并确定乌军的方位。乌克兰总参谋部报告说，俄罗斯部队，主要是指第一近卫坦克军的部队，在过去24小时内没有对苏梅进行任何进攻行动，但继续炮击该市。

3、哈尔科夫方向：乌克兰部队击退了俄罗斯对哈尔科夫东南的Izyum的攻击，并可能在3月24日进行了局部反击。乌克兰总参谋部在当地时间3月24日中午报告说，俄罗斯对Izyum的攻击失败了，俄军曾试图从南部突破乌军的防御；而在下午6点报告说，乌克兰炮兵在哈尔科夫州的一个未指明的地点重创了俄军第144摩步师的第59坦克团的一个营级战术小组。哈尔科夫市当局报告说，俄军于3月24日炮击了位于Nova Poshta邮局的人道主义援助运送点，造成6人死亡，15人受伤。

4、卢甘斯克州和顿涅茨克州方向：在过去24小时内，俄军在这两个州取得了几个小的进展。卢甘斯克州当地的乌克兰政府官员3月24日表示，”事实就是事实”，俄罗斯军队在Rubizhne和Popasna取得成功推进。此前乌克兰总参谋部称，截至当地时间3月23日午夜，俄罗斯对这两个城镇的进攻都失败了。但是在顿涅茨克州，乌军击退了俄军对顿涅茨克西部Marinka镇的进攻。

5、马里乌波尔：俄军于3月24日进入市中心的部分地区。乌军也证实，俄军于3月24日占领了马里乌波尔市中心的圣母代祷教堂。马里乌波尔市政府3月24日报告说，俄军使用扩音器向马里乌波尔居民谎称扎波罗热不再接受难民、乌克兰政府已经抛弃了他们，还谎称俄军已经占领了敖德萨。市政府还报告说，截至3月24日，俄军已经将大约6000名马里乌波尔居民强行带到俄罗斯。另外，通过击沉奥尔斯克号大型登陆舰，乌克兰展示了自己对停靠在已经被俄军占领的港口—-别尔江斯克的攻击能力，这可能会扰乱俄军再次试图从海上增援马里乌波尔和赫尔松附近的行动。

6、赫尔松：俄军在过去24小时内没有向米科拉耶夫进行任何进攻行动，但米科拉耶夫州官员报告说，在米科拉耶夫州和赫尔松州的边界，局部战斗仍在继续。Kryvyi的乌克兰当地官员报告说，俄军至今还无法让其炮兵进入射程范围。
