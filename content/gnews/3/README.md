## 泽伦斯基：欧洲国家还在为俄石油进行血腥交易
` 西班牙巴塞罗那喜悦农场` [轉載自GNews](https://gnews.org/zh-hans/2355888/)

翻译：噜噜咪

![](https://assets.gnews.org/wp-content/uploads/2022/04/xin_png.001-1-1280x140-32.jpg)



![](https://assets.gnews.org/wp-content/uploads/2022/04/1-324.jpg)泽林斯基说，仍在购买俄罗斯石油的欧洲国家正在进行血腥的交易

在接受英国广播公司采访时，泽伦斯基总统特别提到了德国和匈牙利，指责它们阻挠禁运能源销售的努力，而俄罗斯今年将从中赚取高达2500亿英镑（3260亿美元）。

乌克兰领导人对柏林越来越失望，柏林支持对俄罗斯的一些制裁，但到目前为止抵制对石油销售采取更严厉行动的呼吁。

“我们的一些朋友和伙伴明白，现在是一个不同的时代，它不再是一个商业和金钱的问题，”泽伦斯基先生周四在基辅的情况室告诉BBC。“这是一个生存的问题”。

总统还再次呼吁向乌克兰提供更多的武器，说他们没有得到足够快的供应来抵御俄罗斯的攻击。

他说：“美国、英国、一些欧洲国家——他们正在努力帮助，而且正在帮助。但我们仍然需要更快，更快，更快。关键字是现在。”

![](https://assets.gnews.org/wp-content/uploads/2022/04/2-128.jpg)乌克兰总统沃洛季米尔·泽伦斯基与英国广播公司的克莱夫·迈利交谈。这位乌克兰领导人说，他的国家仍然迫切需要来自西方的武器。

最近几周，俄罗斯军队从乌克兰首都基辅周围以及该国其他中部和北部地区撤出——显然放弃了用武力夺取整个乌克兰的企图。

但现在人们担心该国东部和南部会发生血腥和持久的冲突，因为俄罗斯总统弗拉基米尔·普京重新将军事行动的重点放在那里，试图夺取更多领土。

南部港口城市马里乌波尔——普京总统的一个战略目标——已经被俄罗斯数周的炮火轰炸所摧毁。

泽伦斯基总统告诉BBC，他认为该市可能已有数万人被杀。

他说：“我们还有消息说，除了那几万名死者之外，还有许多人失踪了。我们知道他们的文件被替换了，他们得到了俄罗斯护照，并被带到了俄罗斯的深处——一些人被带到了难民营，一些人被带到了其他城市。没有人知道这些人正在发生什么。没有人知道有多少人被杀害了。”

![](https://assets.gnews.org/wp-content/uploads/2022/04/3-115.jpg)泽伦斯基总统说，在乌克兰最大的城市之一马里乌波尔，95%的建筑都被摧毁。

泽伦斯基先生说，俄罗斯军队在马里乌波尔以及基辅郊区的布查和博罗迪卡明显犯下的暴行，进一步缩小了与俄罗斯人进行和平谈判的可能性。

自一个多星期前被乌克兰军队夺回以来，在布查发现了数百名死者，包括被发现双手被反绑头部中弹的平民，以及广泛的性暴力报告。

“布查正在关闭[和平谈判的可能性]，”泽伦斯基总统说。“这不是关于我——是关于俄罗斯。他们不会有更多的机会与我们对话了。”

他说，他上周访问布查时“经历了所有的情绪”，但在结束时“除了对俄罗斯军队的仇恨，什么都没有”。他指责普京总统和俄罗斯军队的其他成员“从上到下”都是 “战争罪犯”。

![](https://assets.gnews.org/wp-content/uploads/2022/04/4-107.jpg)本月早些时候，泽伦斯基总统（在中间的没有戴头盔）访问了布查镇。

泽伦斯基先生为他在2月开始的俄罗斯入侵前的领导力辩护，当时他的政府敦促乌克兰人民保持冷静。

他说，政府一直在后台工作，以商定武器和供应品的交易，并专注于避免可能引发银行挤兑和破坏乌克兰经济稳定的恐慌情绪。

“那是俄罗斯——不仅仅是俄罗斯——想要的，但我们没有让它发生，”他说。“但是，当全面入侵发生时，我们没有想到。”

俄罗斯在2014年克里米亚被吞并之后，试图瓜分更多的领土，乌克兰现在正面临着东部和南部的再次攻击。

泽伦斯基总统说，东部现在是乌克兰武装部队“最困难的情况”，”但这是我们最强大的部队集中的地方”。

他说：“他们可以摧毁我们，但我们会回应；他们可以杀人，但他们也会死。我不明白为了什么——我不明白他们为什么来。”
