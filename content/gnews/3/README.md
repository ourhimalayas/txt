根据RT新闻3月2日报道，瑞典电信巨头爱立信周三透露，根据美国司法部（DOJ）的通知，该公司违反了关于其在伊拉克的所谓活动的调查的延期起诉协议（DPA）。

该声明是在国际调查记者联盟(ICIJ)发表一篇文章后发表的，一份泄露的报告据称揭露了爱立信与伊斯兰国（IS，前身为ISIS）的“秘密交易”。

ICIJ称，泄露的报告显示“爱立信向被称为伊斯兰国的恐怖组织寻求许可，在伊斯兰国控制的城市工作，并付费将设备通过一条被称为‘高速路’的路线偷运到伊斯兰国地区”。报告称，该公司“在近十年内支付了数千万美元的可疑款项，以维持其在伊拉克的业务”。

爱立信已经承认，它在2019年对其可追溯到2011年的活动进行了一次内部调查，并审查了其员工、供应商和供货商在该国的行为。爱立信表示，其道德和合规计划已经“识别并调查了在伊拉克的不当行为”。然而，爱立信坚持认为，其无法确定任何爱立信员工直接参与资助恐怖组织。

尽管有这些披露，爱立信表示不认为它们对2019年的调查结论有任何影响。该公司已承诺“调查并采取行动处理任何新信息”。

“爱立信正在接受合规监督，与美国司法部签订的延期起诉协议仍然有效。”爱立信的声明中说。
