## 一家制药公司说它已有变异毒株奥米克戎的“疫苗”在测试中

![](https://assets.gnews.org/wp-content/uploads/2021/11/image-897.png)
来自《贝克尔网》2021年11月26日报道

最新的冠状病毒变体奥米克戎（Omicron）抢占了国际头条新闻，它已经在全世界造成了毁灭性的连锁反应。英国和欧盟的全球性主管机构正在收紧边界，以应对另一波冠状病毒疫情的爆发。

在美国，纽约州州长凯西·霍赫尔(Kathy Hochul)已经以奥米克戎变体为由，取消了该州的择期手术，尽管这种最新的冠状病毒变体的传播性和致命性尚不清楚。该变种被认为起源于博茨瓦纳，并可能在南非的冠状病毒病例激增中发挥了作用。该变种已在博茨瓦纳、南非、香港和荷兰的阿姆斯特丹被发现。

刚刚从政府的“疫苗”强制接种令中获得高额利润的辉瑞公司，已经在考虑“未来重新推出”一种冠状病毒疫苗，以应对奥米克戎变种。然而，看起来像一家鲜为人知的制药公司诺瓦瓦克斯（Novavax）可能会抢先一步。

据路透社报道，诺瓦瓦克斯医药公司周五表示，它“已开始研究其冠状病毒疫苗的一个版本，以针对在南非检测到的变种”，并且“将在未来几周内准备好测试和生产该疫苗”。

路透社指出：“该公司的冠状病毒疫苗含有一种真正的病毒刺突蛋白，它不会导致疾病，但可以触发免疫系统。” 疫苗开发商表示，他们已经开始专门根据已知的变种B.1.1.529的基因序列开发一种刺突蛋白。

该公司发言人说，“初步工作需要几周时间。”周五，该公司股价收盘时上涨了近9%。

该报告指出，诺瓦瓦克斯公司的疫苗本月早些时候在印度尼西亚获得了首次紧急使用批准，随后是菲律宾。

报告继续写到，“该公司已表示，有望在今年年底前向美国申请批准。”“它还向欧洲药品管理局和加拿大申请了批准。”

诺瓦瓦克斯公司的声明表明，冠状病毒疫苗获得专利并开发“过快”的记录可能会延续下去。在2019年12月新冠肺炎（SARS-CoV-2）疫情在全球范围内爆发之前，预期疫苗需要多年的测试和开发。

突然之间，几乎不需要任何时间就可以生产出一种所谓的“疫苗”，而实际上它只是一种预防性治疗，随后会在数月内失效并需要“加强针”。这不禁让人联想到国际权威机构得知新冠肺炎疫情后，中共国迅速获得疫苗专利的方式，而中国共产党显然自2019年5月开始就知道并一直对世界隐瞒新冠肺炎疫情。

在中共国申请冠状病毒疫苗专利之前，武汉突然出现了新冠肺炎疫情。事实上，“神速”推出疫苗几乎是不可能的，除非中共国在全世界知道之前就已经在对这种致命的冠状病毒进行了研究。

《澳大利亚人报》报道称，“美国支付了中国人民解放军研制冠状病毒的费用。” 五角大楼资助在中共国进行此类危险的研究，其影响是广泛而深远的。

《澳大利亚人报》联系了一位病毒专家，他证实了疫苗研制出的时间线几乎是不可能的。

情节更加复杂。在开始对冠状病毒的起源进行调查后，参与该研究的科学家之一周育森，于2020年5月突然“不明原因死亡”。作为拜登政府发起的调查的一部分，“五眼联盟”情报部门正在调查这起神秘的死亡事件。

《华尔街日报》早前报道称，科研人员认为，新冠肺炎（SARS-CoV-2）病毒中的RNA序列几乎可以肯定是非自然起源的。

一位一直在研究奥米克戎变体的科学家将其描述为“可怕的”。它还具有在其它冠状病毒变种中未曾见过的、不寻常的特性。

诺瓦瓦克斯公司可能很快就会成为下一家利用冠状病毒牟取暴利的制药公司。如果它能够在几周内生产出一种疫苗，它可能会发现其产品已被强制用于全球数百万人，而没有人知道副作用，甚至人们在接下来的六个月内还需要更多的加强针。
