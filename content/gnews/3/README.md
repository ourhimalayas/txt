参议员克里斯.库恩斯（Sen. Chris Coons ）在周日接受采访时表示，他主张美国派军队进入乌克兰与俄罗斯作战。 当被问及这个问题时，库恩斯说俄罗斯总统弗拉基米尔普京“只有在我们出手时才会停止。”

库恩斯在哥伦比亚广播公司新闻的“面对国家”节目中被问及他上周在密歇根大学发表演讲中所表示的“拜登政府和国会应该就我们何时愿意采取下一步行动达成共同立场，不仅仅是派遣武器，而且派遣军队来援助乌克兰……如果答案是永远不会， 那么我们无疑是在鼓励普京进一步升级残暴力行为。”

在接受哥伦比亚广播公司采访时，当被问及拜登总统说他永远不会向乌克兰派遣军队是否错误时，库恩斯说：“美国人民不可能无视发生在乌克兰的这场悲剧，我认为 21 世纪的历史将关注我们如何在乌克兰捍卫自由，而普京只有在我们出手时才会停止。”

拜登总统一再重申，他不会派兵与俄罗斯发生直接冲突，并拒绝了乌克兰关于设立禁飞区的呼吁，因为这意味着必须击落俄罗斯战机并轰炸其境内的导弹系统。 尽管美国和俄罗斯之间的直接战争有可能什级成核冲突，但向战区派遣军队的想法正在变得越来越一致。

超级鹰派众议员亚当·金辛格 (Adam Kinzinger) 是禁飞区的最大支持者之一，他在周日的推文中发出了这一讯息：“是时候明白乌克兰必须赢了……目前谈判已不再可能，”金辛格写道。  “我们这一代人正在吸取我们祖父母的教训：战胜邪恶的唯一方法就是摧毁它。”

上周末的《泰晤士报》报道称，英国特种空勤局 (SAS) 士兵已部署到乌克兰，训练乌克兰军队使用伦敦提供的反坦克武器，这是迄今所知的自俄罗斯2月24日入侵以来，北约成员国首次在该国驻军。

虽然仅仅是作为训练任务，但英国军队在乌克兰的存在可能会引起莫斯科的反应。——上述新闻引自《零对冲》4月19日的报道。
