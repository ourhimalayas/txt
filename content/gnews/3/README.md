## 强制接种政策加重安大略省教育业压力

- ![](https://assets.gnews.org/wp-content/uploads/2021/11/a-25-edited.jpg)图片来源：TURENORTH


据加拿大自媒体【真北方】11月7日报道，【多伦多地区教育局】近期已对该地区近百名长期从事教育工作的人员执行了无薪休假的处罚，原因是他们没有上报其“新冠病毒疫苗”接种状况证明。

报道进一步指出，目前，多伦多地区约有1016名教育工作者已证明他们没有接种该疫苗，并因此有可能在11月21日的截止期限后，被当局置于无薪休假的状态。虽然教育局声称可以批准宗教或医疗原因的接种豁免，但在所有提交的413例该类申请个案中，目前仅有五个医疗豁免被批准。

据悉，【多伦多地区教育局】主席在去年曾请求安大略省教育部，希望在学校重新开学时，增加雇员以解决疫情所导致的人力资源严重短缺问题。但由于目前仍然存在各种基于学校教育的大量资金缺口，因此，一些地方教育局采取了相对宽松的非强制性疫苗接种政策。
