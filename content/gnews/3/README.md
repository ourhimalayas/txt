马斯克在14日向推特发出了约430亿美元并购行动，并在提交给SEC的文件中指出，若收购案遭拒，他会考虑出脱持股。

马斯克给推特董事会下的这个最后通牒，实际上是在巨大民意的加持下，向带着独裁和共匪因子的资本发起的强力挑战。

对于推特董事会的这些成员来说，马斯克的意思已经非常明确：要么你们退出推特、拿着利润滚蛋，由我来全面掌控，要么我就不做这个股东，把手里低价购入的巨量股份全都抛掉。

如果推特被马斯克顺利全盘收购，那么对于言论自由和推特及其他社交媒体的发展无疑都是极好的。

但是如果出现第二种情况，以马斯克的号召力，推特用户数将会出现直线下降，其小股东也会追随马斯克一同抛售，如此一来肯定会把推特股价砸到地板上，从而让董事会的那些成员们血本无归。更有甚者，以马斯克的财力，这样的游戏可以时不时就玩上一把。如此循环往复，推特董事会的成员们受得了吗？

众所周知，推特现在的各大股东都是非富即贵，个个乃呼风唤雨非等闲之辈。因此，马斯克如此的豪气和霸气，敢于正面硬刚这些大人物，其中的原因可能有二：一是邪恶总源头的共匪已经日落西山，其财力已到捉襟见肘的地步；二是这次收购不是马斯克一个人在战斗。
