## “四大危机”、“一个必须”更加锁定习近平成为俄乌战争最大输家
` 波士顿五月花农场` [轉載自GNews](https://gnews.org/zh-hans/2303968/)

作者：波士顿五月花 跟随战神
![](https://assets.gnews.org/wp-content/uploads/2022/04/309B8163-DFE4-4000-8EB3-22DB083425B3.jpg)
2022年4月2日，一位世界前10号的牛人和文贵先生探教了俄乌战争局势。文贵先生明确地说，习近平是俄乌战争的最大输家。

文贵先生基于对中共国内情况的了如指掌，点出了“四大危机”和“一个必须”，使习近平成为俄乌战争的最大输家。

“四大危机”是民生危机、粮食危机、香港危机、政治危机。

中共赖以生存的三驾马车：外贸、基础建设、内需已经全部瘫痪。由于美国的制裁和中共病毒的威胁外贸额锐减，和高峰时期相比外贸额降至原来的零头。基础建设更是捉襟见肘，房地产开发企业大量倒闭。房产巨头债台高筑、大幅裁员，面临被清算的境况。中共国各行各业萎靡不振、百废不兴。失业人员的激增，使原本就脆弱的消费彻底坍塌，民生变成了民不聊生，变成了民生危机。

粮食危机更是严重。中共国是世界上最大的粮食进口国，乌克兰占中共粮食进口的四分之一。俄乌战争催发了世界粮食危机，包括欧洲在内的发达国家纷纷屯粮，使有风吹就草动的国际粮食市场更加风声鹤唳，粮食价格快速攀升。包括郑州大水在内的人祸天灾导致去年粮食大幅减产。中共面临的是外面没钱买、买不来，里面生产供应严重下降的双重困境。日前，中共拼命地在全国实行“退林还耕”。将林地、鱼塘、公园纷纷推到、填平，证明了粮食危机的严重程度。

习近平对香港的大屠杀导致香港成为了臭港。香港的国际金融地位已经不复存在。中共利用香港从欧美偷钱的渠道被习近平废掉。香港危机加剧了中共的金融危机和经济危机。

民生危机、粮食危机、香港危机等诸多危机形成了政治危机。习近平的疯狂让各政治派别感觉到了中共即将进入死刑执行期。包括习近平的部下对他恨之入骨的人大有人在。政治危机的结果是政治厮杀、拔剑相对。

以上的“四大危机”足以结束习近平的政治生命，还有个更要命的“一个必须”。

习近平是必须要打台湾、一定要打台湾的。习近平最大的政治抱负是收复台湾。这是他所谓的政治决定和个人爱好，也是他实行家天下的政治资本。据文贵先生爆料，早在2007年习近平和中共国防部长开会时，第一次暴露他的目标。习清楚地告诉大家，“台湾问题在我这一任必须解决，是新时代必须解决的问题。如果美国或西方任何国家参与，那就毁了它！”展现了习近平残酷的本性。

俄乌战争发生后，习近平喜出望外说：“给我们提供了很好的样板，更证明我们要快打！狠打！不惜一切代价地打！”但是中共高层包括军队的头领们十分清楚，美国有秒杀中共的实力和愿望，武力攻台的后果是脆断、是被团灭。没有人愿意和习近平一起去见马克思。文贵先生领导的新中国联邦一直在保卫台湾同胞上不遗余力，利用一切力量在消除习近平的“一个必须”。

如果说“四大危机”可以结束习近平的法身，“一个必须”则能立刻结束他的肉身。俄乌战争虽未结束，但已经决定了习近平和中共的命运。
