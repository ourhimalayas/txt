是的，这个人间处处都是修罗场，有人有了一点小权利他们都敢去撼动这个世界。正如共产党利用中共病毒和疫苗妄图统治世界一样，连人类发展的基本运作规律都没有分清楚的人，甚至深信人类是猴子变的，但是共产党就用这个思路开始改造人类基因了，对啊，人家就觉得人是猴子变的，所以人可以变回去做猴子。人的基因生理构造如此神秘，不是随便什么人就能够改造的，下场只有一个，这是一次失败的实验，产生排异反应的人类都会死去。但谁说这么大规模的人类尸体死后不会给地球带来什么坏影响呢？是的，地球是个母体，人类生活在地球上，从人体构造来说，每天都有数以万计的细胞死去，但那是自然现象，你要是用认为手段，见人就给人打两针疫苗，这些都是非自然死亡，地球能够承受这样的消亡吗？再一个，打上疫苗以后就算人没死，也会生病，几乎失去了生活的能力，好比大多数位于人体的细胞，按照目前人类打上疫苗的比例来说，半数以上的细胞都半死不活了，请问这个地球会活得好吗？我们看到最近两年天气反常，除了共产党恶意用气象武器制造的天气，自然的天气不也是挺紊乱的吗？冬天打雷，夏天落雪，地球需要自我调节，就只能从人类调节起了。

神爱世人，不惜派出他的独子来到人间。但是让神的儿子失望了，这个人间到处都是肮脏与邪恶，也许正是如此，神的儿子才来。若处在太平盛世之下，处处都是神与上帝，不用派任何人来，人人都是活菩萨人人都赛比神仙。
