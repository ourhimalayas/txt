单伟建此人并未多现于媒体，然而却关联了中共所有重大的海外布局。中共家族的盗国蓝金黄计划，单伟建是那个亲自操作的人。单伟建和吴征两个人，一个是面子，一个是里子，都是共产党的海外间谍部署。可笑的就是吴征这种形象，居然被选做面子，这不是自己打自己的脸吗？虽然自己的老婆杨澜看起来是形象完美的，只是碍于两人的婚姻关系的捆绑，就算是杨澜找钥匙的贡献，也都会分给吴征一半。单伟建的太平联盟，一直对文贵先生进行着各种超限战，灭爆小组，法律超限战，就连博讯的官司，都是这个太平联盟在背后支付各种款项。如果说吴征是在前面吸引注意力的那颗棋子，那么单伟健就是那个在背后真正煽风点火的人。

文贵先生说看共产党的行为不能用普通正常思维来看，如果能够按照正常思维来看，那共产党也就不会成为邪魔恶党了。共产党和邪恶组织所做的这些事，其最后的结果不过就是独裁全世界。在正常思维看来，无论是颠覆西方法治的基石，比如蓝金黄各国政要，律师，司法等，还是给全世界都打上疫苗，做完这些以后，人类就能完全被统治了吗？疫苗打进身体以后如果现在不会死去，看病也需要高昂的成本，这些成本从哪儿来，如果不给这些人看病，那么这个人就死了，再也无法给这些独裁政府做奴隶了。还有就是既然要侵吞合法的私人资产，共产党已经做到可以明抢了，为何还要假惺惺地写一张毫无逻辑徒增笑柄的告示？收买西方的律师法官等，不过就是为了让文贵先生背着这些法律的错判，让所谓的“法律”来给爆料革命定罪。人类法律的基准是事实和逻辑，法官律师向来都是要看证据，给出那些假证据，假逻辑，让法官给出假判断，法律已经不是法律，智慧不会服从这些虚假，善良不会让这些污蔑之名存续。如果法律的证据都不能保证是否是真实的，那谁又能保证法律的真实？
