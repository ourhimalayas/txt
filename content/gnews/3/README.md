## 欧洲议会外委会通过中国新战略，挺欧台投资协定

2021年7月15日

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/07/15205138/0-61.png)

编辑摘要：

据中央社今天报道称，欧洲议会外委会今天通过应对中共国新战略的报告草案，呼吁欧中持续讨论人权、气候变化、健康危机等全球性挑战，要求中共国允许对COVID-19（2019冠状病毒疾病）的起源和如何传播进行独立调查。

报告批评中共国系统性侵犯人权，呼吁欧中定期举行人权问题对话，以解决中共国对新疆、内蒙古、西藏和香港的侵犯人权等问题。报告明确表示，在中共国取消对欧洲议会议员和欧盟机构的制裁之前，将继续冻结“欧中全面投资协定”（CAI）的批准程序。

报告还再次呼吁启动针对欧盟与台湾的双边投资协定。
