## 炮灰进化史
` 银河勇气之星` [轉載自GNews](https://gnews.org/zh-hans/2344136/)

BY：银河系农场|楚天秋
![此图像的alt属性为空；文件名为%E6%88%AA%E5%B1%8F2022-03-22-%E4%B8%8A%E5%8D%8810.53.46-16.png](https://assets.gnews.org/wp-content/uploads/2022/03/%E6%88%AA%E5%B1%8F2022-03-22-%E4%B8%8A%E5%8D%8810.53.46-16.png)![](https://assets.gnews.org/wp-content/uploads/2022/04/图片1-67.png)
慈禧不揣冒昧，以泱泱天朝大国的名义，向十一国开战。令人啼笑皆非的义和拳运动，是其中原因之一。有什么土壤，就会长出什么样的植被。满清末世的矛盾重重，官民冲突不断，社会上流传洋人办教堂，以传教搞慈善为掩盖，吃婴儿肉，甚至摄人魂魄，种种令人震惊的消息。义和拳乘机挑起爱国大旗，扶清灭洋。这和江湖上多以反清复明为口号的各种势力相违背，却能大行其道，当然和清政府睁一只眼闭一只眼有关。政府想利用拳民，把内部矛盾引到洋人身上，慈禧入戏太深，居然信以为真了，大臣们其实心里很清楚，开战的结果。正规军队都没有战斗力，何况义和拳，刀枪不入，只是些骗人的把戏，谁信谁傻叉，但是老佛爷信了，也只能由着她作。慈禧并不傻，算盘打得噼啪响，战争失败，义和拳就是替罪羊，沾着人血签合约、盖玉玺，拳匪人头滚滚，天朝诚意满满。

江山到了中共手里，毛魔的行径还不如慈禧。清理国军降部的抗美援朝惨败之后，除了宣传美帝亡我之心不死，再也没敢有任何实际行动，只是一味关起门来搞内斗。阶级敌人遍布朝野，红卫兵应运而生。保卫毛魔的文革，也就是翻版的清君侧运动，把自己恨的人整死，把有可能挑战自己地位的人整死，各阶层互斗，这样人人自危，早把饿肚子的事情抛到九霄云外。斗死的人也就白死了，活下来的依旧没饭吃，咋办？再来个上山下乡，让红卫兵自生自灭，缓解就业压力。所谓的阳光灿烂的日子，就是毛魔把自己的粉丝赶到农村，被强暴虐待，各种非正常死亡的悲催过程。

中共国末世模式开启，习太阳集两者之大成，一面野心勃勃，要取代美帝成为世界核心，一面利用疫情，搞底层互斗。义和拳、红卫兵俱往矣，防疫志愿者一身孝服登上舞台。虽然过了一段时间太平日子，奴性和残暴程度，不如前朝，但是在蒙面的情况下，也颇有一些浑浊孟浪之徒，加入所谓的抗疫战争中来。中共利用完全捉摸不定的病毒，营造恐怖氛围，不遗余力地打乱正常社会秩序，制造经济灾难和人道灾难。志愿者们却还蒙在鼓里，一朝权在手，已经忘乎所以了，以为自己是赵家人。信息时代，一切都比以往来的要快，防疫时间长了，志愿者难免怀疑和觉醒，所以角色转变速度得更快。前一分钟，是作威作福的抗疫工作者，后一分钟成阳性人，只需要一个短信即可。剩下的路，和别的老百姓一样，隔离饿肚子，困顿抑郁，甚至自杀一了百了。就算啥事也没发生，顺利完成党的任务，毒疫苗发作起来，自然而然就被淘汰掉。

还有更可悲的一群人，只是为了赚钱，被骗去建方舱或者在方舱里打工，最后进退不得，有钱也不一定有命花。从来不去考虑一下，为什么那么多能力强的人，都被关家里出不来，而自己却出来了，还能打工赚钱，啥好事能轮到自己头上，既不是赵家人，又不是体制内，凭啥？被洗脑的后果真的非常恐怖，多数人心甘情愿往火坑里跳，拦都拦不住，至死方休。

中共官员也许正在窃喜，如此多的愚民，再用五百年管够，红朝江山稳固无忧。只是兔死狐悲、鸟尽弓藏，独裁之下，没有谁能笑到最后。炮灰固然无法摆脱前世今生的共同宿命，极权体制在无皇有主的时代，也只是等待收割而已。
