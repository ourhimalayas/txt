反观人类文明发展史，为什么总是在“战争-进步-再战争-再进步”的恶性轮回中得以发展？核武器的研发成功结束了二战，却让世界处在更危险的境地。

我的导师开示说“这个世界多一件武器，人类就多一个不安定因素，要想从根本上解决战争，唯一方法就是解决人心问题”。
![](https://assets.gnews.org/wp-content/uploads/2022/03/EsIjy_sU0AE8gT0.jpeg)
究其战争原因，无不是贪心所致，嗔恨心作祟。全部是心在作用。

无我利他也是一种心，这是一种没有一丝一毫过患的向善之心。全世界的人，人人都拥有这样一颗心，世界就会永久和平。
因为无我，我们没有贪心，没有贪心就不会因为得不到而引发嗔恨心，没有嗔恨心何来战争？

因为利他，我们释放的全部是爱心，是慈悲心。当我们给予别人爱的时候，就像7哥爱战友的时候，战友们也会爱7哥。这才是人类正常交往的最好方式。

我的导师开示说：“嗔恨心解决不了嗔恨心的问题，慈悲心能解决嗔恨心”。

这个生活经验相信每个人都有。嗔恨对嗔恨，只能引发更大的嗔恨。只有慈悲心对嗔恨，嗔恨心才能化解。

当然我不是说，乌克兰人对普京讲慈悲，普京就会停战。普京已经被他心中那个无限大的贪心，和嗔恨心占满，控制，对这一类恶魔释放再多的善意，都会被他恶魔的心吞噬，唯一的办法就是消灭之，让他停止犯罪。
