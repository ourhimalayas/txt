圣乔治-阿克雷马诺–一位27岁的意大利专业游泳运动员、游泳教练、硕士生和即将成为妻子的人去世了，因为全球各地的年轻猝死者正在加速。

玛丽亚索菲亚-帕帕罗女士最著名的身份是一名成绩优异的游泳运动员。她的Instagram页面，可以追溯到2013年，有许多她在游泳比赛中赢得奖牌和奖杯的照片。第一张重要的是2014年底的照片。标题写道：“战利品增加！从未感到如此健壮。

帕帕罗女士本应在4月18日满28岁。她还定于4月28日毕业并获得那不勒斯大学Parthenope导航科学与技术专业的硕士学位。

**突发心脏病和死亡**

据多个消息来源称，她于4月11日星期一因心脏病发作而“突然死亡”。没有确切的证据显示帕帕罗女士接受了mRNA和/或病毒载体DNA注射。但常识和主要证据的结论是，她至少接受了两次疫苗注射。

至少从2021年8月起，那不勒斯大学帕特诺普分校就有了疫苗授权。意大利在2021年7月对室内餐饮、体育场馆、游泳池和几乎所有其他场合都强制要求接种疫苗。12月实施了“超级绿色通道”，基本上禁止未接种疫苗的意大利人进入所有公共场所。意大利还要求所有运动员必须接种疫苗才能参加体育运动。

**意大利将遭受世界上最多的疫苗死亡？**

迄今为止，意大利是2021年初在The COVID Blog™上报道最多的国家之一。至少有13个国家，包括意大利，在2021年3月暂时或永久地禁止阿斯利康病毒载体DNA注射。截至6月，我们在意大利报道了至少10起阿斯利康之后的死亡事件，包括18岁的卡米拉-卡内帕。她在注射阿斯利康后仅两周就死于血凝块和脑出血。辉瑞公司和摩德纳公司在夏天之后成为平等机会的杀手。

所有的致命性注射都是危险的。但大多数西方国家去年夏天停止使用强生公司和阿斯利康公司的针剂。美国也暂时停止使用强生公司的针剂，因为在获得紧急使用授权后，有太多的猝死和血凝块。
