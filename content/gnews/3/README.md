## 中共重回计划经济“供销社”模式

新华社北京4月10日报道，《中共中央 国务院关于加快建设全国统一大市场的意见》10日发布。中共撕下了最后的一块遮羞布，用山寨版的“全国统一大市场”代替计划经济时期的“供销社”模式。

意见中明确，要加快建立全国“统一”的市场制度规则，打破地方保护和市场分割。中共利用“中共病毒”、“强制疫苗”、“核糖核酸”、“强制隔离”、“暴力封城”、“动态清零”……对全国老百姓进行“压力测试”，对物资进行集中管理和重新再分配，为供销社模式进行预演。

根据意见，加快建设全国统一大市场的工作原则是：立足内需，畅通循环；立破并举，完善制度；有效市场，有为政府；系统协同，稳妥推进。总之一句话，中共早就为“闭关锁国”做好准备，用“米袋子”和“菜篮子”控制老百姓的生死，让老百姓成为巩固自己权利的工具和奴隶。

中共重回计划经济“供销社”模式已经全面启动。七哥在直播中说，习近平认为吃饭喝水的权利和听啥说啥的权利，是最大的两个权利。食品和水变成武器，让所有人只能听到看到他想干的事，习近平做到了。
