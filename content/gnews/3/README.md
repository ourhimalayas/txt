【日本NHK】乌克兰总统泽连斯基将于3月23日傍晚向日本国会发表在线讲话，被视为呼吁日本加强对持续进行军事侵略的俄罗斯压力。

美国总统拜登还计划在本周访问欧洲，宣布欧美新的制裁措施，俄罗斯的压力。这将进一步增强国际社会对

3月22日，美国国防部的一名高级官员告诉记者，俄军正在蓄意攻击医院和避难中心，并指出，对于东部关键城市马里乌波尔，除了远距离猛烈炮击外，又开始从部署在亚速海的军舰开炮。

据报导，马里乌波尔市仍滞留着约10万名失去了避难的机会平民。

联合国秘书长古特雷斯22日谴责俄罗斯说，马里乌波尔迄今已被围困了两个多星期，遭受着无差别的轰炸和炮击

在此之上，他再次呼吁立即停战：“在乌克兰继续战争的行为从道德来看不被容忍、从政治来看没有开脱的余地、从军事上来看毫无意义。”
