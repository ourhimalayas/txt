澳大利亚新南威尔士州政府的最新官方数据显示，接种三针和两针疫苗的澳洲人感染新冠奥密克戎Omicron变体病毒的可能性比未接种者高十倍以上，这表明获得性免疫缺失综合症、也就是ADE效应可能已经大量出现。

我们根据新南威尔士州政府网站的数据，整理出的2022年第一周1月2-8日的疫情数据如下：


| 疫苗接种状况 | 染疫总人数 |
| --- | --- |
| 两针及以上接种 | 159,325 |
| 一针接种 | 1,468 |
| 未接种 | 787 |
| 接种情况未明 | 46,697 |
| 未到年龄(0-11岁) | 18,339 |
| 总计 | 379,056 |


在这段时间内，新南威尔士州有93.62%的人完全接种了疫苗，95.04%的人第一次打了第一针，有4.96%的人没有打疫苗。 因此，该州1月份第一周的两针以上接种者/未接种者的染病比率为（ 160,793×4.96% ）/（787 x 95.04%） = 10.66

。或者换个说法，今年第一周新南威尔士州的奥米克戎病例中，有787例是未接种疫苗的，有160793例是单次或两次或三次接种疫苗的。这意味着不到0.5%的病例是在未接种疫苗的人群中，超过99.5%的病例是在接种疫苗的人群中。看看吧！疫情简直成了疫苗接种者的专利了。疫苗有效吧？对，真有效！
