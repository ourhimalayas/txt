［路透社华盛顿10日电］美国10个州的共和党总检察长,10日向联邦地区法院提起诉讼，要求禁止拜登政府上周宣布的，强制联邦资助的医疗机构为其雇员接种疫苗的政策，因为这将加剧医护人员短缺。

诉讼由密苏里州、内布拉斯加州、阿肯色州、堪萨斯州、爱荷华州、怀俄明州、阿拉斯加州、南达科他州、北达科他州和新罕布什尔州的总检察长提起。

堪萨斯州总检察长施密特说，“对医疗机构和员工施加这些额外义务会加剧问题恶化，而一些服务不足的农村地区，因为他们无法雇用到所需数量的员工，可能导致一些设施关闭。”

这10个州声称，拜登政府侵犯了各州的监管权。他们说，在没有设定公示期的情况下宣布这项任务，违反《行政程序法》的。
