据加拿大媒体【CTV新闻】4月19日报道，加拿大一项在线调查显示，近四分之一的加拿大受访者已感染过新冠病毒。

报道称，加拿大正经历肆虐的新冠疫情第六波，但官方与公众分享的数据越来越少，所以加拿大研究协会制定了在线调查，值得注意的是，调查结果的感染率高于官方水平。此外，另一项民意调查研究了新冠患者后遗症的问题，结果显示四分之一是加拿大人，超过一半人的表示后遗症中最明显的是疲劳，肌肉和关节疼痛。

据悉，虽然互联网民意调查存在分配误差，但互联网调查结果存在的意义是让人们意识到新冠疫情长期的风险与症状，以更好地应对可能出现的状况。
