## 樱花时政┃五角大楼邀供应商 讨论长期战争武器供应
` 東京櫻花団-Tokyo Sakura Group` [轉載自GNews](https://gnews.org/zh-hans/2339295/)

撰稿：东京樱花团/asket
![](https://assets.gnews.org/wp-content/uploads/2022/03/LOGO-1-14.png)
五角大楼将于周三接待来自美国前8大武器制造商的领导人，讨论如果与俄罗斯的战争持续数年，该行业满足乌克兰武器需求的能力。

![](https://assets.gnews.org/wp-content/uploads/2022/04/8-33.jpg)[**图片来源**](https://www.reuters.com/world/pentagon-asks-top-8-us-arms-makers-meet-ukraine-sources-2022-04-12/)

华盛顿，4月12日路透社，两名知情人士周二表示，五角大楼将于周三接待来自美国前 8大武器制造商的领导人，讨论如果与俄罗斯的战争持续数年，该行业满足乌克兰武器需求的能力。

自俄罗斯于2月24日入侵乌克兰后，因乌克兰对武器的需求猛增，促使美国和盟国向乌克兰转让武器。不愿透露姓名的消息人士告诉路透社，预计会议将讨论补给以及计划更长时间的战争。

其中一位知情人士说，美国国防部的武器采购商，五角大楼的采购和维持办公室将主持这次90分钟的会议，预计国防部副部长凯瑟琳·希克斯将出席。

五角大楼曾表示，最有用的武器是小型系统，例如标枪反坦克导弹和毒刺防空导弹，华盛顿及其盟友几乎每天都向乌克兰运送这些导弹。

乌克兰军队的密集使用以及战场上表现出的效力，激发了人们对补充这些武器的兴趣。

Raytheon Technologies (RTX.N) 和 Lockheed Martin Corp (LMT.N) 联合生产标枪反坦克导弹Javelins，而Raytheon生产毒刺防空导弹Stingers。其他顶级武器制造商包括波音公司(BA.N)、诺斯罗普格鲁曼公司(NOC.N)、通用动力公司(GD.N)和L3Harris Technologies (LHX.N)。

白宫上周表示，自乌克兰入侵以来，它已向乌克兰提供了超过17亿美元的安全援助，其中包括5000多支标枪反坦克导弹和1400多支毒刺防空导弹。

五角大楼的国防安全合作管理局每周召开一次欧洲危机管理小组会议，以审查与乌克兰有关的具体要求。

为了加快美国政府批准美国国防承包商生产的武器的销售和转让，五角大楼重新组建了一个团队来应对不断增长的需求。
