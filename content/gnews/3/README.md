现在的大上海，很多人是都在找吃的东西。当宣布3月28日浦西封城的时候，很多家超市都是24小时营业，抢购的那个浪潮是前所未有的。其实封到现在，很多人家里面都已经开始断粮了，上海老头老太太都在说，自己要想办法出去买菜什么的，然后就是说，我要突围，我要越狱，我要去吃饭，我要买东西，我要自由！一封锁家里什么存货也没有了，那很多老人也不会上网买菜，没办法网购，所以他就要想办法出去，戏称自己是“越狱”。真的是让我听了心里真的是特别酸。像上海闵行的一个小区，一连被封了26天，中共所谓的“为人民服务”的政府是一直都没有送过物资，小区居民他们也开始反抗了。像刚才咱们PPT说的，有个小区开始在绿化带里找东西吃，无助的民众非常可怜。
