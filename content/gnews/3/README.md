## 有声｜中共国疫情封控 民众挨饿官员贪腐食材大发国难财
` 秘密翻译组-即时新闻播报组` [轉載自GNews](https://gnews.org/zh-hans/2370652/)

By:**秘翻新闻**

**2022-04-18**
![](https://assets.gnews.org/wp-content/uploads/2022/04/IMAGE-2022-04-18-235727.jpg)
综合消息。近日，有墙内网友发布视频称，当地城管没收民众为封闭小区居民送的蔬菜后，据为己有。无独有偶，近期上海封城期间，民众在网络指斥中共政府各级官员垄断物资采购与分配，造成物价飙升，甚至一些居民忍饥挨饿，处境窘迫。

日前，网络流传的一条视频称，辽宁省海城市响堂管理区的城管将老百姓给封闭小区居民送的蔬菜没收后，不上交单位统一分配处理，而是偷偷将其搬运到自己的私家车里，而小区居民却无菜可吃。

在中共国，这一现象并不偶然。封城已有半个多月的上海，在极端封控之下，也上演着官员垄断物资分配与采购，从而大发国难财的戏码。

一位自称上海餐饮业者的网友表示，现在上海的物价已飙升到了平时的5到10倍。

有居民声称，买30个鸡蛋花了478元，面包则涨到10元一个，牛奶20元一小盒，一个土豆要10元。

据一位发布视频的餐饮业者披露，实际上现在蔬菜的一手拿货价跟往常是几乎一样的，而且还有全国各地的救援物资源源不断地输送。他反问：“物价到底为什么会涨成这样？”

上海市民赵女士透露，她家楼下的志愿者表示，外省援助上海的物资够整个上海所有人吃两个月，但是都被上边的人给贪污了。

还有居民声称自己团购了“盒马”套餐，拿到的却是辽宁捐赠的物资。
