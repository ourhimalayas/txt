这可是公众媒体的时代。知道吗，现代人不比小孩聪明。为什么，因为他们没有思考。白天挥汗如雨，到了晚上烂醉如泥，打麻将和弹子球，或者是看电视和听广播。他们什么时候有时间思考，头脑里面只有一个空壳。我们的目标是明确的，要往他们空壳的脑袋里重复塞入我们的信息，广播、电视、电影等一切广告设备都可以用上。一切都是那样活生生的，媒体是现代社会的独裁者。

——《巨人与玩具》（1958）

很多人应该都知道在今年的奥斯卡上威尔史密斯上台打了脱口秀主持人克里斯洛克一巴掌，虽然事情传播很广但都没什么实质意义。不过台湾中坜信义国小的洪老师在课上拿这件事发起了一个小小的社会实验，实验步骤和学生表态如下。

1.先不管学生是否得知这个事件，告诉学生脱口秀主持人克里斯洛克在奥斯卡颁奖典礼上拿威尔史密斯老婆的光头来谈论，威尔史密斯不爽上台打了克里斯洛克一巴掌。请全体学生举手表态是否支持威尔史密斯的举动。统计3个班的人数，大约是4成的学生支持，尚未过半。

2.播放此事件的整个影像，学生得知威尔史密斯的老婆是因病掉发，却被克里斯洛克拿来嘲讽。这次表态结果接近9成支持威尔史密斯打耳光。

3.向学生提问，嘲讽当然是言语暴力但打人却是更严重的暴力，即便被取笑了我们不是应该在暴力以外寻求其它的解决方式吗?克里斯洛克的言论是否构成公然侮辱还有待商榷，更何况克里斯洛克事后说了他并不知道威尔史密斯的老婆是因脱发症才理光头的。没有给对方解释道歉的机会直接诉诸暴力是否恰当。这时学生表态的结果，支持者又降回到约4成。

4.告诉学生威尔史密斯是家暴目睹者，小时候常看到母亲被父亲打到浑身是血，从此他发誓要守护自己的家人。而这次他入围的角色恰好是一名捍卫家人的勇者。他的老婆因病掉发曾经封闭自己，好不容易在女儿的鼓励下愿意走出来；而如今因为克里斯洛克不知是无心还是蓄意的嘲讽让他老婆受伤，他当然选择保护自己最爱的人。听完这段说明后支持威尔史密斯打人的又飙高到8成左右。

5.告诉学生这是奥斯卡举办94届以来头一次发生暴力事件，这画面放送出去会有上亿人目睹。奥斯卡主办单位正在考虑取消威尔史密斯影帝资格，甚至追回小金人；克里斯洛克若提告，威尔史密斯将面临6个月的牢狱之灾以及至少10万美金的赔偿。这次表态大约是5成的人支持为守护家人开揍。

6最后也是最重要的问题，每位学生举手5次，从头到尾你的态度完全没有动摇过，无论支持或反对威尔史密斯打人的立场，5次都没有改变的，请举手。3个班加起来大约不到1/4。

这个实验的相关步骤其实洪老师本人在facebook上也有展示，为保证严谨以上步骤与原文几乎一字不差。社会学实验当然不只会给学生启发，相信世界上各行各业的人现在都感觉到了不对劲，首先就是各种矛盾日趋尖锐且不可调和。很多乐迷圈的人都有发觉像是音乐作品下面的评论区以非常激烈的方式走向两极分化，威尔史密斯当时用打人的方式表达自己的观点一定也有受到这样的氛围影响。如此剑拔弩张也说明现在世界必须要选择一个结果，而温和的手段以及模糊的立场都已经无法胜任。

另外这个实验结果仿佛让我们回到了董仲舒的时代，君要臣死，臣不死是为不忠；父叫子亡，子不亡则为不孝。并且现代人还误以为人的认知和科技水平成绝对正比，然而科技却可以更方便地操弄舆论，只要给出对自己有利的讯息，剩下的一概不提。慢慢的它就成为你的皇帝、你的父亲，然后你就无法自己做主了。文贵先生说的未来无皇有主的时代不只针对国家，个体的状态也是一样。

台湾值得感恩的一点是还有这样的国小老师愿意做如此的社会实验，并且实验范围仅限于班级，让学生把错误变成自己的生命经验但又不至于遭受灭顶之灾。但在中共国就没有这样的机会了，首先就像60多年前的电影所说成年人不比孩子聪明，中共病毒大流行又意味着实验对象变成了全世界，并且你不可能听得到让你左右摇摆的两种声音。它只会给你展示菜场看上去充足的蔬菜但快餐店的汉堡却开始变成无蔬菜版本。事实上中共的声音已经给空壳的成年人带来了很多灾难，军事训练般的全民核酸、激化地域矛盾、让父母与孩子分离、小区封锁至死不休，以及拉出高龄党员冲锋陷阵。2022年死亡5000万至1亿人也就在这一次次实验中成为现实。
