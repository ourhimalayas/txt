据《路透社》作者：john Chalmers, Gabriela Baczynska，2022年3月23日报道：
![](https://assets.gnews.org/wp-content/uploads/2022/03/image-3050.png)2022年3月23日，比利时布鲁塞尔，北约秘书长延斯·斯托尔滕贝格在北约峰会前的新闻发布会上。图片来源：Gonzalo Fuentes/路透社
布鲁塞尔，3月23日（路透社）——北约周三警告，俄罗斯在乌克兰的战争可能导致莫斯科和西方之间的核对抗。

北约秘书长延斯·斯托尔滕贝格(Jens Stoltenberg)在新闻发布会上说：“俄罗斯应该停止这种危险而不负责任的核言论。但毫无疑问，我们愿意随时准备保护和捍卫盟友，抵御任何威胁。”

他在布鲁塞尔举行的西方军事联盟国家领导人峰会前夕说：“俄罗斯必须明白，它永远不可能赢得核战争，北约不是冲突的一部分……它向乌克兰提供支持，但不是冲突的一部分。”

斯托尔滕贝格重申：“北约不会派兵进入乌克兰……向乌克兰提供支持是极其重要的，我们正在加紧努力。但与此同时，防止这场冲突演变为北约和俄罗斯之间的全面战争也极为重要。”
