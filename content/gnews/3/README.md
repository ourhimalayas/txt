**4.辽宁葫芦岛累计报告本土确诊120例**

2月18日，葫芦岛市召开疫情防控新闻发布会通报，2月17日0时—24时，葫芦岛市新增确诊病例9例，为绥中县报告，均已转入定点医疗机构治疗，病情稳定。2月8日—17日24时，葫芦岛市累计报告本土确诊病例120例。

2月8日以来，葫芦岛全市247个社区对所属居民进行了全覆盖排查，做到了底数清、情况明、动态数据更新及时。对1月25日至2月8日期间从绥中等疫情中高风险地区来(返)葫人员实行居家隔离措施，第一时间确定包保联系人，建立健康管理台账，详细记录外出地址及工作单位等信息。包保联系人、社区医务人员每天多次与居家隔离观察人员发起微信视频、位置共享，避免居家隔离观察人员在隔离期内外出。

同时，对于独居或生活不便的居家隔离观察人员，社区工作人员还提供了购买居家生活物品、处理生活垃圾等力所能及的生活帮办服务，确保隔离人员基本生活得到保障。截至2月17日，葫芦岛全市社区已落实居家隔离和居家健康监测措施5.6万余人。
