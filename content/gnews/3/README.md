## 麦当劳将关闭在俄罗斯的所有餐厅

图片来源：路透社

麦当劳公司 (MCD.N) 周二表示，将暂时关闭其在俄罗斯的所有 847 家餐厅，包括其标志性的普希金广场位置，这增加了其他全球品牌在莫斯科入侵乌克兰后暂停在该国运营的压力。

麦当劳餐厅的关闭在俄罗斯也具有象征意义，1990 年在莫斯科市中心开设的第一家餐厅成为苏联解体后美国资本主义繁荣的象征。

麦当劳表示将继续向其在俄罗斯的 62,000 名员工支付工资。由于其庞大的规模和全球影响力，如果该连锁店在某个问题上采取立场或进行重大运营变更，则通常会被其他品牌复制。

包括麦当劳和百事可乐公司 (PEP.O) 在内的主要全球品牌受到消费者和包括纽约州养老基金在内的投资者的压力，要求其退出俄罗斯。

32 年前，苏联解体之际，麦当劳在普希金广场开业。那家新餐厅代表了冷战时期紧张局势正在解冻，当时一些年轻的俄罗斯人迫切希望得到蓝色牛仔裤和其他美国人。

麦当劳首席执行官克里斯·肯普钦斯基（Chris Kempczinski）周二在全公司范围内通过电子邮件发送并发布在公司网站上的一份说明中说：“无法预测我们何时能够在俄罗斯重新开放我们的餐厅。” “我们正在经历供应链中断以及其他运营影响。我们还将密切关注人道主义局势。”

马萨诸塞大学政治学教授保罗·马斯格雷夫表示，麦当劳的关闭表明西方对俄罗斯实施快速而广泛的制裁将产生持久的经济影响。
