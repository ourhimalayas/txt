中共国去年实行双减政策后，多省市教育部门要求校外培训机构，在去年年底前完成非盈利性机构，若无法转型，则由相关部门予以注销。

在实际操作过程里，限制会更多，关于校外培训机构转型事宜，教育部门是完全停办的，所以转型基本是不可能的。勉强可以存续的一些培训机构，规定不允许在寒暑假和双休日培训，外语培训机构严禁使用海外紧扣教材，平时培训时间必须18点开课，而教育部门把学校放学时间都延长到了18点。

这个所谓“双减”，所谓整顿校外培训机构，其实就是要把所有校外培训机构彻底挤死。校外培训机构也是各省市的一大税收企业板块，为什么要断臂自残呢？据网友传闻，校外培训机构收益可观，教育部门想把这块肥肉自己直接吃下，把培训这块变成硬性指标压倒各个学校而且收费很高。
