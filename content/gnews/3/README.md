4月7日TBS新闻消息：俄罗斯外交部发言人谴责日本就乌克兰局势对俄罗斯实施制裁，并表示将采取进一步的反制措施。

在6日的例行记者会上，俄罗斯外交部发言人扎哈罗娃说：日本正在破坏两国之间花了多年时间建立的双边合作关系。

扎哈罗娃对岸田政府对俄罗斯实施制裁表示不满。批评日本紧跟西方步伐，说日本是 “顺从地听从他国的指示”。她还断言，将 “采取进一步的反制措施”。但具体细节没有披露。

俄罗斯外交部上个月宣布，将暂停与日本的和平条约谈判。

自民党外事委员会主席佐藤正久表示：我认为应该以一种协调的方式，与欧洲和美国一样，采取驱逐俄罗斯外交官的措施。同时，在今天上午的自民党会议上，外交事务委员会主席佐藤提到了从日本驱逐俄罗斯外交官。

针对在基辅附近发现大量平民尸体，佐藤先生说：有必要尽快实施严厉的追加制裁，以阻止进一步的屠杀。在会议上，一名外交部官员回应说，他们将考虑这些意见并做出决定。
