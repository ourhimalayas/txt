踪、分析及应对，打击中共对外国在军事、经济和政治领域的恶意影响，保护美国国家安全。

特尼对其法案进一步介绍说：“中共是当前美国国家安全和经济最重大的威胁之一。《中国观察法》的目的在于为美国政府提供更多资源，以便更好地掌握中共的各种势力扩张和影响力渗透，增加政府各部门之间的合作以应对任何可能威胁。中共猖獗太久了，现在是美国采取果断行动的时候了。”

众议院外交事务委员会（Foreign Affairs Committee）的首席共和党人麦克·麦克考 (Michael McCaul)说：“中共从事的恶意活动遍布全球各地，这项挑战与其它挑战都不同，需要具有全球战略眼光。美国国务院有必要成立一个专门小组，追踪中共的恶意活动。”

事实上，今年9月份已有消息指出国务院也准备建立一个“中国小组”应对中共的挑战，并且计划扩充熟悉中国的事务人员，进一步紧密追踪中共在世界主要国家的活动。

这项法案目前已获得众议院两党七位议员的联署支持。该法案将强化美国与国际伙伴的协调，以应对中共的恶性影响活动。美国政府长期以来一直警告说，中共在美洲和非洲的影响力在不断扩大。特别是中共病毒大流行期间，中共利用“一带一路”倡议来推进其在拉丁美洲的贸易、技术和基础设施建设。美国认为中共已经持有1650亿美元被认为是掠夺性的贷款。这些贷款通常以债务国的战略资产作为抵押，如果它们无法还款的话，中共政府可以没收那些资产。
