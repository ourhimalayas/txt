## 谈论外太空
` 骄子` [轉載自GNews](https://gnews.org/zh-hans/1800702/)

文：银河勇气星｜骄子
![](https://assets.gnews.org/wp-content/uploads/2021/12/20211012_1_01.jpg)太空梯 / 图源：网路
美苏争霸的时候，两方为了显示自己的强大实力，都拼命往太空发展，离地球最近的月球是一个迷一样的存在。人类自己制作的航天飞行器，把人能带到月亮上去，并且全程电视直播，这在那个年代来说，其实美苏争霸已经不重要了，对老百姓好奇心的启迪才是至关重要的。虽然苏联当时也有加加林完成了载人太空飞行，一那个没去月亮，二那个没有直播。美国阿波罗登月是进行了全球直播的，2个小时，那个年代中共国内的老百姓都在颤栗中经历着文革，小命都保不住，哪有什么好奇心求知欲以及探索，你多想一件事都可能被批斗。虽然这两个小时的直播后来引发了人们的巨大猜想，很多阴谋论开始分析那些直播画面，因为当时有断掉的部分，当然也就成为了各种各样的猜测基础。后来随着直播走向大众，随便谁都能直播的年代，在地球上的直播都会因为网路覆盖或者速度的原因造成各种各样的画面卡顿，声音卡顿，不禁会问，50多年前登月直播是怎么做到的，并且就那么无私无畏就敢直播，那可是人类第一次到达地球以外的地方。毕竟这事儿要是放在共产党这，不可能的，直播是一种最大的不可控因素，尤其是对未知地域的探索，怎么可能呢，况且这种直播是永远不会从人们的记忆之中抹去的，你再洗脑也改变不了曾经发生过的事实。

![](https://assets.gnews.org/wp-content/uploads/2021/12/7.jpg.jpeg)阿波罗登月 / 图源：网路

就算是到了现在，共产党依然无法做到全程直播一次登陆外太空的计划。去年登陆火星的宣传片都来自美国，拼拼凑凑，一个视频成片也就很短的时间，毕竟素材太少了，后期视频剪辑人员为了还原当时的景象，只能找类似的影片，本来这类素材就少之又少，精度高的更是凤毛麟角，所以很容易就被网友发现素材是美国的。搞笑的是还弄什么自拍手机，你都带了手机了，你拍照干什么，手机可以直播啊，哪怕不直播你拍一段真实视频也行啊，然而只能给大家欣赏经过高度PS后期合成和精修的照片。共产党的那些在太空的宇航员，拍的照片不符合太空的外力环境，是个人一眼就能看出来不对劲。与此同时，space X上太空全程直播，完完整整地直播，并且在今年第一次带着四个平民上了太空，只是经过短时间的太空常识培训，这在以前是想都不能想的。毕竟上去了太空，你得回来，这才是一次完整的旅行。这给所有热爱太空的人提供了更多的可能，即每个人都有可能去太空旅行，离开地球表面，不再是梦。

![](https://assets.gnews.org/wp-content/uploads/2021/12/164076167257930_P12291261.jpg)共产党抄袭的火星车 / 图源：网路

人类对太空的探索，从始至终都不是偶然。中国古代有夜观天象，观测行星四季的运动和对人间事务变化的影响。西方较实际，发明了望远镜直接去看外太空。一直到现在人类都在使用望远镜观测地球以外的世界。哈勃望远镜经过一代又一代的不断升级，在天文观测上起着重要重要，现在的哈勃望远镜可以从外太空直接来观测地球，一些地球上的限制因素可以全部抛开，拍摄检测到的各种在地球上看不到的地球自身现象。毕竟人类生活在地球上，不可能看到地球的全貌，也不会有进一步的关于地球的认知，跳出地球来看，会得到更多的发现。“不识庐山真面目，只缘生在此山中”，站在太空看地球，是完全不一样的体验，相信上过太空的那四个平民有更加深刻的体验。

外星人来到地球的传言一直就没有间断过，在大家的传言里，他们要么开着形状各异的飞船，会做出地球上看来不可思议的动作，比如消失在视线里。在影视剧里要么就长着奇怪的身体，声音也很奇怪。在外星人眼里，地球人也是妥妥的外星人，所以把范围扩张到全宇宙的话，大家没什么不同。文贵先生说那个悬崖的故事，引出关于人类灵魂的话题。人的灵魂就像风，风可以到处刮，风是无影无踪的，但是人可以感受风，并且无处不在地感受风。人的肉体是在一些相对的物质条件下存在的有形状可触摸的实体，到了另外的环境，人的肉体就没有了，但是灵魂还是会存在的。来到地球的外星人也许有一些就是这样的，他们的肉体在地球这个环境里，是无法存在的，好比我们的人体在一万度的高温也是不能存在的，假如人的身体到了外太空，不穿宇航服的话，真空，人的身体会被迅速压瘪，然后碎成一块儿一块儿的，肉体就消亡了，但人的灵魂是存在的。所以外星人来到地球，他们可能就是一缕风，一些看不到的微小粒子，让人体会到的一种感受。为什么地球上的人大部分都相信有外星人的存在，因为人感受到了他们的存在，感知、感觉是不会骗人的。而那些我们看到的电影里，剧集里，拥有身体的外星人，奇奇怪怪的样子，也许就是地球外环境对他们身体的带来的某些形变，一旦回到他适宜的环境，又会变回原来的样子。
![](https://assets.gnews.org/wp-content/uploads/2021/12/1vZBIe9iovFKC010lnC_jUw.jpeg)天空之城 / 图源：网路
这么说来，灵魂是可以自由畅游宇宙的，但并不是每个灵魂都能到达想要去的地方。佛教里面的须弥山，据说是宇宙的中心，就不是随便哪个人都能去的，唐僧取经，一路上都是在修行，其实取经只是修行的一部分，人生中的修行是时时刻刻的。张高丽那样的，在地球上的那面悬崖上，他直接给跳下去了，从此以后就堕入魔道了，根本都没有接收他的地方，怎么可能再往上走，穿越太空飞向宇宙呢？
