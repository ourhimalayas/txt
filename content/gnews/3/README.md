![](https://assets.gnews.org/wp-content/uploads/2022/01/69F2D5D1-E3D1-40DD-BE81-E41092FC7278.jpeg)（图片来源：法新社）

1月15日，一份美国法院文件披露，谷歌（Google）与脸书（Facebook）联合签署的协议显示，两家公司合作操控线上广告市场的竞价。

据《自由时报》报导，一份美国多州联合指控谷歌反垄断诉讼案的法律文件显示，谷歌与脸书两大社交媒体公司的高管于2018年合作签署了代号为Jedi Blue的协议，该协议中谷歌公司为了让脸书放弃一种受出版者青睐的广告工具，同意在线上广告竞价上对脸书让利，从而达到共同双赢。然而，此举却损害了其他竞争者的权益。

2020年12月，由德州检察长派克斯顿（Ken Paxton）领军，美国多州联合控告谷歌垄断线上广告市场，在线上广告销售的数位生态系统中滥用职权，通过操纵广告竞价以逼退其他竞争者。
