[*图片来源*](https://m.zhihuimami.com/baike/jibing/6926.html)

近日被曝光的中共国家卫健委一份内部文件显示，中共病毒疫苗已导致众多接种者罹患白血病。该文件由中共卫健委发给河北、辽宁、四川、陕西等18个省份，名为《关于请关注部分自述接种新冠病毒疫苗后罹患白血病人员近期重点动向的函》在网络上热传。

文件称，有人员在微信群中自述本人或者家属接种新冠病毒疫苗后罹患白血病，并在微信中进行接龙，有下一步组织集体活动的可能。文件要求，鉴于中共“两会”和北京冬残奥会即将召开，各省卫生主管部门要与有关部门共同做好维稳工作。这表明接种中共病毒疫苗罹患白血病的情况，从地域跨度和人数规模都很大，并且中共官方针对性的维稳已经开始展开。

根据中共国家卫健委官网数据，截至2022年3月3日，31个省（自治区、直辖市）和新疆生产建设兵团累计报告接种新冠病毒疫苗314700.7万剂次。
