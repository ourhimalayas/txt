据《LifeSite》12月8日报道，特斯拉首席执行官兼 SpaceX 创始人埃隆·马斯克12月6日发出警告称，世界文明正面临因低出生率而导致崩溃的危险。

据报道，埃隆·马斯克在《华尔街日报》的年度CEO委员会上表示，他认为，世界文明的最大风险之一是低生育水平和快速少子化。

马斯克指出，现在很多人，包括聪明人，都认为世界上人口太多、人口增长失控。但根据全世界的出生率数据显示，事实完全相反。他警告说，如果人们没有更多的孩子，文明就会崩溃。

据公开统计数据显示，自1970年以来，世界人口增长速度在总体上急剧下降，从最初每年2.06%的出生率，快速下降至现在每年1.05%的水平。

《LifeSite》的报道还指出，根据人口研究所的研究结果，全世界至少有96个国家，即大约一半的国家，预计出生率在2020-2025年期间将低于生育更替水平。如果这一趋势继续下去，整个社会的人口数量将呈指数级下降，并最终面临灭绝的危险。

对此，有网友表示，现在全世界的人口出生率本已急剧下降，而中共又用病毒和毒疫苗对全人类进行种族灭绝。本就岌岌可危的人类文明如今又被中共这个恐怖组织裹挟，以火箭般的速度加速崩溃。为了人类能继续生存下去，为了世界文明不快速走向崩溃，全世界必须站起来反抗中共了。

正如爆料革命和新中国联邦创始人郭文贵先生所言，灭共是人类正义的需要，必须将中共铲除，否则人类会加速走向灭亡。
