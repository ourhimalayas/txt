## 黄鼠狼给鸡拜年，中共再给非洲提供10亿剂疫苗

29日晚中共与非洲合论坛第八第八届部长级会议开幕式中，中共高层决定为非洲提供10亿剂CCP病毒疫苗，其中6亿剂为免费赠送，4亿剂为中共企业与相关非洲国家联合生产的方式提供。中共还将为非洲国家援助实施10个医疗卫生项目，向非洲派遣1500名医疗队员和公共卫生专家。

当然这一切都是黄鼠狼给鸡拜年，中共终实在执行阴谋集团的“减人计划”，所谓为实现非盟确定的2022年60%非洲人口接种新冠疫苗的目标。此次中共的非洲疫苗提供计划，正是中共最近在非洲放的新变种**[****B.1.1.529****]**CCP病毒而准备的下一阶段计划，也就是先放毒，再所谓救毒的套路。

新的剧本显然是新变种由非洲出现，然后再传染到亚洲，欧洲，进而全球又一次推动加强针，俗话说免费的东西才是最贵的，这句话非常正确，此次提供给非洲的10亿剂绝非是免费的，只是所谓有6亿剂是免费，再说了以假大空中共的尿性来看，这6亿剂一定有很大的猫腻，很大可能是假疫苗，而另4亿剂是与非洲相关企业合作生产，这意味着这4亿剂会产生利益，况且这本来就是中共一边放毒，一边所谓救毒，所产生的一切影响对于中共来说都是巨大的利益，一方面非洲更加需求着中共提供疫苗，实际上腐败的非洲领导也能从中获得回扣，从而加大采购中共的疫苗，另一方面变种病毒从非洲流入欧美，正是有利于中共的灭白计划，此次放毒，中共又是多方面获得好处。

再从另一个角度来看，且不说这一切都是阴谋，中共就是一付吃里扒外的，崇洋媚外德行，也不管国内老百姓死活，中共国的百姓有6，7亿收入每月低于1000人民币，中共不管，却在之前就计划援助非洲600亿美元，非洲留学生来可以拿每月几万人民币的补助，而这些所谓的留学生连中文都不会，这简直就与清朝的所谓“宁赠友邦，不予家奴”如出一辙。不过中共比清朝邪恶多了，现在是黄鼠狼给非洲拜年，全是阴谋！
