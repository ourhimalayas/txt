## 有声｜美中情局局长：中共是最大挑战 美需积极迎战
` 秘密翻译组-即时新闻播报组` [轉載自GNews](https://gnews.org/zh-hans/2351442/)

**By:秘翻新闻**

**2022-04-15**
![](https://assets.gnews.org/wp-content/uploads/2022/04/IMAGE-2022-04-15-193634.jpg)
据[《福克斯新闻网》](https://www.foxnews.com/politics/chinas-xi-jinping-is-a-silent-partner-in-putins-aggression-in-ukraine-cia-director-burns-warns)4月14日报道，美国中央情报局，英文简称CIA局长威廉‧伯恩斯周四在乔治亚理工学院发表演讲时表示，中共是美国在21世纪唯一的一个最重要的地缘政治挑战，美国需要积极迎战。

伯恩斯指出，中共在许多方面给中情局带来了有史以来最重大的考验。中共国的GDP约为15万亿美元，是120个国家的头号贸易伙伴，并且是人工智能、5G、无人机、高超音速技术和网络应用领域的领先者。他还指习近平是普京侵略乌克兰的“沉默伙伴”。

伯恩斯还说：“在过去几年里，中共至少入侵了150家美国公司以窃取商业机密，并储存了多达1000枚核弹头，拘留了100万中(共)国公民，仅仅因为他们是穆斯林。中共还逮捕了数千名香港人。”

伯恩斯表示，对于一个情报机构的长期成功而言，最重要的莫过于其与中共的竞争结果，以及在未来几年的这种竞争中如何组织好自己。

（ 秘翻新闻 巴拉克 编译报道）
