2. 控制供应和分散力量：

尽管进行了虚假的慈善运动，但俄罗斯可能很快就能决定谁能获得重要的粮食供应，谁不能获得。由此给俄罗斯带来的战略收益也可能意味着故意在世界上一些最脆弱的国家和地区制造有针对性的粮食安全危机。由于也门、利比亚、黎巴嫩、斯里兰卡和苏丹等国严重依赖乌克兰粮食，即使是短期的供应延迟也可能产生严重后果。

一些地区的人道主义紧急情况、内乱甚至武装冲突是俄罗斯操纵全球粮食市场的可预见后果。许多分析人士仍然认为，2011 年的阿拉伯之春最直接的原因是谷物价格的变化。

当大量的人努力满足人类最基本的需求时，冲突和起义是自然的反响。这种不稳定将极大地分散美国、欧洲和其他致力于缓解此类冲突和人道主义危机的盟国的注意力。这种对世界其他地区的注意力和资源的流失将为俄罗斯不受干扰地经营创造更多空间。

3. 针对其他非北约国家

随着北约寻求加强其在黑海和波罗的海（Baltic Seas）附近北约成员国的军事存在，乌克兰给俄罗斯的教训可能是：“只要你不攻击联盟，我们就不会反击。”因此，俄罗斯可能会认为占领乌克兰是不够的。摩尔多瓦、格鲁吉亚等剩下的独立地区也是目标，甚至像哈萨克斯坦这样的中亚国家——最近欢迎俄罗斯的“维和人员”，是伊朗和阿富汗的主要粮食来源地——也可能是俄罗斯的直接目标。

如果俄罗斯能够利用这些国家的关系和经济驱动力，同时利用其在乌克兰的潜在成功所产生的恐吓因素，这种方法可能特别有吸引力。如果正如许多人所承认的那样，普京的真正目标是重建俄罗斯的领土，那么这种收复前苏联加盟共和国的做法一点也不牵强。

4. 利用中国条约
