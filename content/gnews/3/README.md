## 陕西西安因疫情升温中风险地区数量暴增

**撰稿：日本东京方舟农场 幽蘭自得**

据《大纪元》12月22日报道，陕西西安疫情升温，除确诊病例增加外，该市又新增27个中风险地区。目前当地共有41个中、高风险地区。

据报道，因西安市雁塔区多地出现确诊病例，自12月22日起，雁塔区长延堡街道明德门西区42号楼、雁塔区长延堡街道融尚第十区6号楼、雁塔区长延堡街道明德八英里7号楼、雁塔区长延堡街道鑫泰园西区22号、雁塔区电子城街道唐园小区12号楼、雁塔区电子城街道万象春天10号楼等27个区域新增为中风险地区。截至22日，西安市共有高风险地区1个，中风险地区40个。

![](https://assets.gnews.org/wp-content/uploads/2021/12/1-280.jpg)图为2021年10月20日西安民众排队做核酸检测（图片来自Getty Images）

中共国一位病毒学专家对《中（共）国新闻周刊》表示，西安在发现首例确诊的时候，病毒可能已经传播好多代了。

武汉大学公共卫生学院教授于传华称，随着西安全员核酸推进，未来不排除新病例的数字还会在高位运行。
