## 秘翻在线：乌克兰表示俄罗斯为隐瞒战死者规模拒绝接收本国军队尸体

![](https://assets.gnews.org/wp-content/uploads/2022/04/그림1-17.jpg)图片来源：本文链接
据韩国《亚洲经济》4月9日报道，有主张称，俄罗斯为了掩盖入侵乌克兰后战死者的规模，拒绝接收本国军队的尸体。

据美国《华盛顿邮报》(WP)8日(当地时间)报道，乌克兰政府试图将乌克兰境内的太平间或冷冻列车的7000多具俄罗斯军人尸体送回俄罗斯，但遭到俄罗斯的拒绝。

乌克兰总统办公室助理奥列克西·阿雷斯托维奇主张：“俄罗斯入侵第三天，我们本想把3000具俄军的尸体运往俄罗斯，但遭到了拒绝”，“虽然多次向俄罗斯提议接收尸体，但俄方连讨论都不想讨论”。

从以前开始，俄罗斯就对公布战死者规模表现出了敏感的反应。虽然战争已过去6周，但俄军尚未正式公布死亡人数。克里姆林宫（俄罗斯总统室）发言人佩斯科夫在接受英国《天空新闻》采访时表示，正式阵亡者为1351人，但这与西方国家及乌克兰的推算相差甚远。据北约估计，在过去6周内，俄罗斯军队中有7000至1.5万人死亡。

而俄罗斯内部也没有讨论战死者的规模。俄罗斯总统普京在2015年签署了可以保密所有军人死亡的法令。

乌克兰打开了俄军家属了解俄军死亡者或俘虏信息的窗口。乌克兰内务部建立了可以搜索俄军死亡者或战俘的照片或获取家属信息的在线申请网站和telegram（通讯软件）频道。

俄罗斯的这种做法，无疑会让战死或被俘的本国士兵及其家属寒心。
