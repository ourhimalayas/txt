12月19日郭文贵先生GTV、Gettr大直播中透露，自美国建国200多年以来国债规模已达30万亿美元（最新31.40万亿美元，而GDP为20.94万亿美元），而美国国会批准允许的国债上线总额（允许超发印的钱）是GDP的1倍-1.5倍。自2019年新冠病毒出世到现在，美国已印了接近10万亿美元的法币。
![](https://assets.gnews.org/wp-content/uploads/2021/12/935157c2ee3403471c91b83882fcb0ba.jpg)图片来源盖特


世界各国通过连续非常规的QE手段严重推高了房地产价值，其中美国房地产总价值为35万亿美元，英法德意四国总共为30多万亿美元，日本10万亿美元，中国一家已达100多万亿美元。然而，本该与法币发行对应的全球生产能力，却因为疫情的不断打击并没有增长，所以造成通货膨胀大幅推高了物价。

量化宽松营造的假象是大家手里的法币数额增多，但实际法币的真实购买力在下降。这也是全球经济的缩影，通货紧缩、通货膨胀并行，最终导致流动性减弱。新冠疫情导致各国生产力不平衡，供应链紧缺、物流环节不通畅等一系列的社会问题会迸发式涌现。

所以，全世界前所未有的至暗时刻即将到来，因而一定要做好5年到10年甚至更长时间各种准备。不仅仅是全球超发法币导致的通胀，还有实际上高达300万亿美元的债务，最终唯独经济崩塌一条路可走。

俗话说不破不立，各国滥权超发法币也给了代表未来出路的区块链数字货币机遇，人类可以第一次和平迅速而不用流血地改变货币的属性。从世界历史上看，当要选择国家货币作为衡量货物价值的时候，一定是伴随着血腥的改朝换代。

从去年到现在，所有人被前所未有地卷入到数字货币战、网络战、金融战、毒疫苗战、生化病毒战的战争之中。数字货币是第一战，各路群雄都在争夺未来数字货币的定价权、发行权和监管权。

当埋藏在金融房产界的核弹爆炸，世界范围的房地产倒塌和法币崩溃之际，数字货币迎来“春秋群雄”的竞争大时代，最后通过残酷淘汰进入“三国演义”。生存下来的赢者必定是跟随户用多，系统运行成本低，保密安全性强，有同宗、同信、同文化。最后，我们新中国联邦一定是“三国”之中最重要的国，一定要当“司马懿”。

人类已处在数字货币代替法币的“春秋时代”，世界也必然走向“三国”的新数字货币时代，而新中国联邦绝对是其中重要的一极。
