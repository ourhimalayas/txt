根据德国之声中文网3月17日报道，西藏歌手才旺罗布个人新浪微博被禁言，他于2月25日发布了最后一条微博，并配上了自己的四张黑白照片。

根据无法经过独立方核实的消息，才旺罗布已经于2月25日在西藏首都拉萨的布达拉宫前自焚身亡。

藏人行政中央3月12日公布了这一消息，有关才旺罗布自焚死亡的消息源自流亡藏人媒体以及藏人行政中央外交与新闻部人权事务处。报道指出，由于中共当局严格的信息控制，他的自焚抗议的细节、遗言、最后死因、个人诉求等，外界仍然不明。

该报道称，中共当局于3月2日联系了才旺罗布的家人，通知他们才旺罗布已经去世，而中共当局并没有将他的遗体移交给他的家人。

报道还称，在西藏发生反对中共政府的抗议活动，包括自焚抗议活动后，中共政府会立即加强对当地藏人的限制和镇压，对通讯渠道进行严格的审查，许多抗议活动的细节有时多年后才能获悉。
