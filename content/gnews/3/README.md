## 李嘉诚旗下药妆零售商屈臣氏宣布退出俄罗斯市场

**作者:  **青衣
![](https://assets.gnews.org/wp-content/uploads/2022/03/5-39.jpg)
（图片来自《南洋商报》）

香港富商李嘉诚旗下的国际药妆品牌“屈臣氏”日前宣布将关闭在俄罗斯的47家门店。不过该集团是以经营不佳为由，避谈此决定是否关乎对俄制裁。

据《南阳商报》3月17日报道，在俄罗斯经营近17年之后，香港富豪李嘉诚麾下的零售业务商“屈臣氏集团”将撤出这个市场，加入到众多在俄罗斯入侵乌克兰后退出该国的国际企业行列。

不过，据报道，该公司并没有将退出的原因归咎于战事，而是在声明中称其在1月底就已做出上述决定，原因是“企业在俄罗斯市场的业绩不佳”。

屈臣氏集团在2005年收购了俄罗斯当地零售商Spektr Group，借此打入俄罗斯市场。该集团隶属于李嘉诚的旗舰企业集团“长江和记实业”，而“长江和记”目前是由李嘉诚的长子李泽钜领导。

屈臣氏在周三（3月16日）的声明中表示，将在6月之前关闭在圣彼得堡的47家门店。但屈臣氏的发言人拒绝向媒体评论制裁与公司业务的关系。

在屈臣氏发布上述撤离计划之前，苹果、宜家、麦当劳等许多国际品牌，已响应西方制裁宣布撤离俄罗斯。俄政府则威胁要没收这些企业的在俄资产。
