## 法国国民议会批准了疫苗加强针接种要求以维持健康证的有效性

作者：纽约香草山农场 – 四季

![](https://assets.gnews.org/wp-content/uploads/2021/08/logo-5.jpg)





据[布赖特巴特新闻网](https://www.breitbart.com/europe/2022/01/17/560000-may-lose-covid-pass-after-booster-shot-requirement/)1月17日报道，在1月8日法国发生了反对疫苗护照的10万人抗议活动后，法国国民议会仍然于1月15日以185票对69票表决通过了对该国中共病毒护照立法的修改，要求民众注射疫苗加强针以维持其健康证的有效性。

![](https://assets.gnews.org/wp-content/uploads/2022/01/GettyImages-1237737263-e1642260976702-640x480-1.jpg)Photo by DAMIEN MEYER/AFT Via Getty Image

作为这项新政策的结果，预计有56万名已接种了双重疫苗的法国人的将停用其通行证。

除了第三针接种的要求之外，法国参议院周日还为中共病毒通行证转变为正式的疫苗护照扫清了道路，中共病毒阴性筛查不再是进入咖啡馆和餐馆等公共场所的有效证明。

12-15岁的人将能够保留以前的健康通行证，该通行证允许他们在检测结果为阴性时可进入餐馆或电影院等公共场所。
