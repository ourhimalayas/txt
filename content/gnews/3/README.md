## 俄罗斯试图阻止西方公司逃离该国

评论：***西方企业投资俄罗斯是为了获取稳定的回报，现在俄罗斯入侵乌克兰，这些企业站在西方世界道义的一边撤离投资，使俄罗斯金融系统除了swift的制裁外，雪上加霜。***

据《cnn.com》作者：Mark Thompson，2022年3月1日报道：
![](https://assets.gnews.org/wp-content/uploads/2022/03/image-190.png)
伦敦/莫斯科（CNN财经）俄罗斯总统弗拉基米尔-普京正试图阻止西方企业因其对乌克兰发动战争的决定而逃离该国的流动。

据塔斯社和俄新社周二报道，俄罗斯总理米修斯廷宣布了旨在阻止企业外流的资本管制措施。他说，西方公司做出决定是因为“政治压力”，在这种压力消失之前，他们将被禁止出售俄罗斯资产。

据引述，米修斯廷说，“为了使企业能够做出明智的决定，已经准备了一份总统令草案，对退出俄罗斯资产实行临时限制，我们期望那些在我国投资的人能够继续在这里工作。”

石油巨头英国石油公司(BP)是自上周俄罗斯军队入侵乌克兰以来放弃俄罗斯的最著名公司之一。它周日表示，它正计划退出其在俄罗斯最大石油公司Rosneft及其合资企业的19.75%的股份–相当于在俄罗斯最大的外国投资之一。

此后，其他公司也纷纷效仿，包括壳牌（RDSA）和挪威的Equinor。
法国道达尔能源公司(TotalEnergies)周二表示，它不会为俄罗斯的项目提供新的资金，并正在评估西方的巨大制裁对其在该国的现有投资的影响。

全球大型投资基金正与企业一起试图抛售俄罗斯资产。挪威首相周日表示，挪威1.3万亿美元的主权财富基金将剥离47家俄罗斯公司的股份以及俄罗斯政府债券。
自从美国、欧盟和其他西方盟友对该国大部分银行系统实施制裁以来，俄罗斯一直在争相防止金融崩溃，包括冻结莫斯科多年来为保护俄罗斯经济而储备的价值数千亿美元的外汇储备。分析人士说，这些措施可能会导致银行危机。

卢布周一暴跌约25%，目前价值约1美分。自2014年俄罗斯首次入侵乌克兰，吞并克里米亚并引发更为有限的制裁以来，它已经损失了大约一半的价值。俄罗斯的股市本周还没有开盘，但在海外上市的俄罗斯公司的股票已经崩溃。

俄罗斯官员已经采取了紧急措施，试图稳定金融系统。中央银行将利率提高了一倍多，达到20%，并暂时禁止俄罗斯经纪商出售外国人持有的证券。政府已下令出口商将其80%的外汇收入兑换成卢布，并禁止俄罗斯居民在俄罗斯境外进行银行转账。

米修斯廷说，“我相信，制裁的压力最终会消退，那些不会屈服于外国政客的口号而缩减在我国的项目的人将会获胜，”
