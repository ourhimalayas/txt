## 樱花FM┃防卫省研究攻击型无人机
` 樱花FM` [轉載自GNews](https://gnews.org/zh-hans/2281106/)

音频：东京樱花团/战友520

![](https://assets.gnews.org/wp-content/uploads/2022/04/040322.png)

3月30日获悉，防卫省将从明年开始正式研究使用攻击型无人机。俄罗斯入侵乌克兰时，乌克兰军队使用土耳其制造的“Byragtal TB2”无人机，展示了强大的能力。无人机战争逐渐成为现实，但目前日本自卫队只限于侦察型，因此，距离正式引进还很遥远。乌克兰军队从进攻开始的第3天开始，用无人机攻击了俄罗斯军队的地对空导弹。据悉，在前线物资补给路线上，破坏了运输车等，成功推迟了进军。另外，用募集的民间无人机拍摄的受害视频在网上公开，用于情报战，还得到了各国的支援。 让无人机失控的电子战机器也显示出了存在感。俄罗斯军队的无人机试图破坏乌克兰军队的无人机基地，但由于乌克兰的电子战机器发射电波，无人机的主要活动领域——地面1公里以内的低空制空权被阻止。

自卫队也在关注无人机。陆上自卫队首长、陆上幕僚长吉田圭秀17日表示：“使用大量廉价无人机的集体侦察和攻击确实是有效的军事手段。”但是引进情况仅限于灾难应对及掌握战况的1~2米左右的小型飞机，可以进行远程飞行的大型飞机从本月开始在空自三泽基地（青森）部署3台侦察型飞机。防卫省在明年的预算中，计入了3000万日元的小型攻击型无人机的有效性和外国机体等调查费，但目前还处于“从运用方面进行研究”（负责人）阶段。

分析无人机影像的庆应大学教授古谷知之（国土安全保障）表示：“与把活生生的人送上战场相比，替换能力更强的机器人将在现代战争中占据重量。尤其在人口减少的社会中有效。”针对敌方无人机的无人机攻击是否违背了日本的基本姿态“专守防卫”的讨论也在进行当中。政府在年内进行的国家安全保障战略(NSS)修改中，有必要明确无人机战争的地位。
