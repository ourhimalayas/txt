## 米勒强调，盖特捍卫言论自由反对删除文化

撰稿：首尔天池农场—GeorgeC（乔治）
![](https://assets.gnews.org/wp-content/uploads/2022/01/1-GTV-Gnews-HEADER-8.png)
1月21日，盖特总裁杰森·米勒先生在接受亚当·卡罗拉播客节目采访中，详细阐述了盖特的价值观，就是不以政治观点来区别对待盖特用户，反对广泛存在于其他社交媒体的歧视现象，反对取消文化。
![](https://assets.gnews.org/wp-content/uploads/2022/01/Capture1-60.jpg)
米勒先生表示，去年7月4日推出以后，盖特的定位就是一个真正被用户拥有的平台。而不是像大型科技公司和社交媒体公司所做的那样，对不认同的观点贴上标签甚至加以警告。他认为，中共病毒和治疗中共病毒药品（如青蒿素、伊维菌素、硫酸羟氯喹等）的信息，被大量主流社交媒体打压删除就是一个很好的例子。

杰森·米勒先生进一步以著名播客主持人乔·罗根等名人，因为遭受推特打压而加入盖特为例说明删除文化。

同时，米勒先生还明确表达了对中共的态度和观点。他说，盖特没有任何来自中共的投资资金，公司本身和中共也没有任何联系。并且，他认同美国正在建立的关于中共资金的审查制度。而更重要的是，米勒先生认为在美国有很多爱国的美籍华人，中共不等于中国人，中共在西藏、新疆等地犯下的反人类罪行必将受到审判。
