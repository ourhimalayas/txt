## 中共抗疫一瞥：权贵享特供，平民户外冻

![This image has an empty alt attribute; its file name is Screen-Shot-2021-08-30-at-7.37.59-PM.png](https://assets.gnews.org/wp-content/uploads/2021/08/Screen-Shot-2021-08-30-at-7.37.59-PM.png)
1月3日，据综合消息，西安市由于疫情扩散，在上月23日开始实施封城。 民众只被允许出门接种疫苗，除此之外全部不准离开住家，形同被监管。 近一周来，每日新增有症状确诊人数仍超过百人，无症状者并未被列入统计，其数应更甚于确诊者。

![](https://assets.gnews.org/wp-content/uploads/2022/01/Slide3-2.jpeg)

有一盖特视频显示，一曲江新区权贵之家，收到官方特地准备且亲自送来的猪肉、鸡肉、蛋、时蔬等，满满一桌。 另一个盖特视频则呈现出几位大人抱着小孩，站在尚有余雪的户外，抗议官员为了防疫将社区清空，把社区所有住民赶到户外凛冽的寒风下哆嗦。

为了达成中央要求在1月4日「社会面清零」的指令，西安市政府直接把有阳性病例出现的社区所有住户赶出住家，并集中送到城外的管控区，以此来塑造城内疫情已受到控制的假象。 因此受苦的只有平民百姓，在寒天中挨饿受冻，以及增加感染病毒或加重病情的风险。
