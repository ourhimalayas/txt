格兰瑟姆的这一观点也再次印证了郭文贵先生此前的爆料。郭先生曾指出，美股在此次世界性的经济危机中必然会出现暴跌，美元将可能失去其在世界中的霸主地位，而喜币和喜联储将会成为世界经济的未来。
