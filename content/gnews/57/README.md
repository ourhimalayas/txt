## 英国官方数据很讽刺：自然免疫与两剂疫苗的效果一致

**英国官方统计数据显示，未接种疫苗的新冠病毒康复患者与接种过两针疫苗的英国人对德尔塔变种病毒享有相同的免疫力。**

参照国家统计办公室今天发布的报告，未接种疫苗的英国人第二次感染德尔塔变种病毒的可能性要减少71%可能性，而对于接种两剂辉瑞或者阿斯利康疫苗的人群来说，感染这一变种病毒的可能性减少了67%。

国家统计办公室宣称，”没有证据表明”（接种）疫苗比感染新冠病毒（痊愈）本身能提供更多的免疫力，尽管其他一些研究显示情况正好相反。

这一研究成果是基于来自于英国各地8000多名阳性案例，而时间在5-8月之间，恰好是德尔塔病毒肆虐的时期。

科学家们仍然试图揭开自然免疫和疫苗免疫到底可以持续多久的谜底？

疫苗的保护大约可以持续5个月。这也是为什么50岁以上的英国人在这个秋天要接种加强针的原因。但是，自然免疫的持续时间一直都是一个谜，而新的变种病毒的出现也让这一情况更为复杂。

科学家们采集了来自于未接种疫苗、完全接种疫苗和两剂疫苗接种人群的志愿者们的样本。而其中一些志愿者们曾经感染过新冠病毒。

通过统计学的分析，**我们发现两剂接种者被感染的风险概率下降了64%到70%之间，这也是我们在文章开头说的67%的保护率。**

**对于已经感染过病毒但是未接种疫苗的志愿者来说，这一风险下降了65%到77%之间。**

**重叠的概率区间意味着两种人群在统计学上没有不同。**

在报告中，**国家统计办公室称****“****没有证据表明在降低感染率方面，接种两剂疫苗人群与获得自然免疫的人群有任何不同。****”**

同时，国家统计办公室也发现，两剂辉瑞接种者的保护要稍稍高于两剂阿斯利康接种者。

但是，办公室说，两种疫苗保护性的不同在统计学上并不显著，而这种稍稍的不同也可能是由于阿斯利康要比辉瑞接种时间早造成的。

**这份报告没有对感染病毒后的不良反应进行评估，这其中就包括了长期新冠后遗症，**这也是许多成年感染者挥之不去的阴影。

批评政府决定为12岁以上的学生接种疫苗的人认为，自然免疫对孩子会更好，因为这避免了接种疫苗的副作用。

这里有万分之一的概率会得心肌炎。而心肌炎是一种比较罕见的心脏感染，但孩子感染上了不一定很严重。

这种情况在男孩里有较高比例发生，医生说大多数的病例都是可以医治的。但是，疫苗对孩子长期的影响目前还是不明确。与之相比，年轻孩子因新冠感染进入ICU的比例大约在50万分之一。

数据也确认了疫苗在防止传播上的作用要低于当阿尔法病毒盛行的时候，这一时间段是在2020年12月到2021年5月之间。

那一时期，两剂疫苗接种后可以把感染的风险降低79%。比较而言，因感染阿尔法病毒而获得自然免疫后可以把再次感染的风险降低65%（自然免疫）。

新冠病毒感染调查分析部门的主任莎拉·克罗夫茨在其博客里分析，在两次记录期间疫苗免疫力可能的下降是对疫苗免疫力减弱的警告。

她说，“近期的数据，包括那些在接种了第二剂疫苗后经历了较长平均时期的参与者们。”

“如果疫苗的有效性随着时间的的变化而改变（效力减弱）那么这更可能在最新的分析里捕获到。”她也说较小的样本也可能会影响数据。

我们只用了99个人作为参照组，他们在5月到8月间感染了病毒。但是，有几乎4000个两剂疫苗接种者参加了调查。

要想算作自然免疫，那这个人一定不能接种哪怕一剂的疫苗。

而且，当我们比较未接种者和接种者病毒感染风险的时候，我们需要检测每一组的结果。

随着时间的推移，更多的人接种了疫苗，从而导致了我们未接种疫苗人群样本很低。

依据卫生部的数据，大约4.53千万人，或者10个人中的9个人已经接种了两剂疫苗。

并且大约一半的50岁以上的成年人已经接种了第三剂加强针的注射，这大约是在第三针对这一年纪组开放后的一个月。

在9月16日，大约3.1百万人已经接种了第三针。而据估计有6.5百万人是有资格接种的。

关于新冠免疫的争论目前还是未知的。而一系列的相互冲突的研究也让探寻真相更加困难。

**上个月在以色列发表的研究发现未接种人群的自然免疫能力要比没有被新冠感染过的两剂接种人群对于新冠的免疫力高****13****倍。**
