- ![](https://assets.gnews.org/wp-content/uploads/2022/01/1-280.png)图片来源：Medium


近日，一份关于中共在加拿大间谍活动的政府报告，指责中共政府对华人社区进行了“系统的情报收集、说服、影响和操纵活动”。

在加拿大《环球新闻》的报道中，加拿大官员称，中共侨务办公室(OCAO)的任务是“影响或控制”社区成员，并对持不同政见者和少数族裔使用“胁迫手段”，“在社会各个层面对海外华人（OA）进行恐吓，”报告说，“OACO对海外华人行为管理是通过激励或抑制，以及对华人社区情报收集、监视和颠覆来完成的”。

该报告非常直接概述了中共的情报活动，在最近披露的一起法庭案件中，涉及一名试图移民加拿大的前国务院侨务办公室(OCAO)成员。

中共声称侨务办公室对生活在国外的中国公民负责，但加拿大的报告提出了OCAO在“从事间谍活动”，并且已得知其在加拿大活动。

报告指出：“OCAO参与了针对海外中共异见人士的秘密行动和胁迫，并实施了对海外华人活动的情报收集”。

OCAO致力于打击被认定为威胁中共的个人，并组织和监控所有海外华人企业、学生、文化、媒体和政治网络。

由加拿大边境服务局（CBSA）的安全检查部门撰写的2020年3月的报告，是加拿大官方对中共在该国和世界各地的间谍和外国影响活动作出的罕见评估。

上周，加拿大联邦法院驳回了前OCAO成员张勇提出的上诉，该上诉试图推翻拒绝其永久居留权申请的决定。

法院裁定，处理他的案件的移民官“合理地确定”，并有“合理的理由确信他在加拿大正在进行针对海外华人社区的信息收集”。

渥太华的维吾尔族领导人穆罕默德·土赫提说，CBSA报告的细节以及法官的裁决是准确的，其中谈到了加拿大的间谍活动和外国干涉威胁。

他声称自己也是被攻击的对象之一。2020年7月，在他将于议会中国委员会作证的几个小时前，他收到一条消息，说他的母亲“已经死了”。

针对多伦多的维吾尔族集会，当地也组织了反抗议活动，他怀疑这些抗议是由海外事务办公室策划的。

“中共政府花费了大量资源来支持加拿大和其他地方的海外事务组织，”这位活动人士说。

“这些团体在北京有非常充足的资金，破坏了加拿大的民主和自由。这是一个非常有根据的决定，应该适用于所有联邦政府机构，以防止间谍和监视，并保护加拿大人的安全和福祉。”

CBSA报告称，**中共的情报部门是世界上最活跃的情报部门之一，并说他们在不断发展，“变得更具侵略性”。**

虽然对间谍人数的估计各不相同，但据报道，“**中共可以正确地声称拥有世界上最大的、最无形的、但也是最活跃的情报部门……因为任何中国人，特别是来自中国的人，从学生到CEO，都是潜在的活跃情报资产**，”加拿大边境管理局的报告说。

报告说，中国共产党一心想着“消除批评政权或挑战其权力地位的OC群体中的敌对言论或潜在威胁”。

为了减轻这些威胁，中共依靠OCAO，该组织在几乎每个国家的中共大使馆、领事馆和代表机构设立了附属机构，OCAO亲自与当地的华人社区进行联络。

它说：“OCAO专注于消除对政权的威胁，如促进台湾独立的人。同样，据了解，OCAO还在海外协调反法轮功的工作。”

同时，**OCAO“识别中华人民共和国的敌人，监控侨民、操纵组织，并参与秘密情报收集。OCAO专注于消除对中共政权的威胁，并试图在其开展业务的国家中破坏这些威胁。”**

前保守党议员肯尼·赵（锦荣）说，中共“认为海外华人是一种资产，可以通过操纵和控制来自私地利用，以促成他们的利益”。

赵在上一次联邦选举中失去了他在卑诗省列治文的席位，因为他声称受到中共特工的袭击，他说加拿大在打击外国干涉方面做得不够。
