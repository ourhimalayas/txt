#### 编译：小红帽

根据美国医学会（AMA）的数据，截至6月，高达96%的美国医生已经“接种”了中共病毒（Covid-19）疫苗。讽刺的是，而其中一些最直言不讳的人现在已经死了。

凯文·沃尔什（Kevin Walsh）医生，一位在弗吉尼亚州罗阿诺克市拥有一家私人诊所的妇产科医生，在脸书上推广中共病毒疫苗后仅六周就去世了。在他的帖子中，沃尔什将中共病毒与小儿麻痹症相提并论，试图吓唬他的追随者去接种疫苗。

来自澳洲昆士兰州贝诺瓦的医生贾斯汀·纳赛尔（Justin Nasser）也遭遇了同样的命运。11月14日，根据新闻简报称，纳赛尔“因心脏病发作意外死亡”。根据The Covid Blog报道，昆士兰州和澳大利亚所有其他州都需要为所有医护人员接种 mRNA 或病毒载体 DNA疫苗。

俄勒冈州瓦尔德波特的斯蒂芬妮·博什（Stephanie Bosch）医生也于10月13日因肺栓塞而 “突然且意外地 ”死亡。肺栓塞是与mRNA（辉瑞-生物技术公司，莫德娜）和病毒载体DNA（强生公司）疫苗相关的一个已知和常见的不良事件。

纽约州波基普西的克雷格·香农（Craig Shannon）博士在家中突然去世。两份不同的讣告声称，他死于非霍奇金淋巴瘤，然而如果这是真的，那么他应该死在医院或临终关怀房中，而不是在自己的住所。

10月14日，阿克苏瓦西拉的埃利奥特·加尼翁（Elliott Gagnon）博士也同样“在家中意外去世”。就在几个月前，这位整形外科医生的私人诊所还脸书上呼吁患者“请大家行动起来，如果你或你的孩子（目前12岁以上）没有接种疫苗，请与你的医生联系。”

马萨诸塞州西哈特菲尔德的丹尼尔·麦克布莱德（Daniel McBride）医生，他在10月20日骑完16英里的自行车后因心脏病发作而突然死亡。这位医生是一位 “狂热的自行车爱好者 ”和全能运动员，据悉，他在此之前总体上是健康的。

俄亥俄州玛丽埃塔的贾纳克·帕特尔（Janak Patel）博士于10月28日“突然死亡”。当他被 “发现倒地，无法抢救”时，他一直在做急诊室医生的正常工作。

The Covid Blog警告说“当权者知道，医护人员的疫苗强制令使合格人员受伤和死亡，从而导致人手大量短缺。许多医护人员为了避免接种疫苗而辞职，导致更多的人手短缺。”

“还有大量由疫苗引起的病人涌入医院。甚至美国国家公共电台（NPR）也承认，医院里挤满了没有患中共病毒的重病患者。这正在形成一种局面，有数百万病人却没有人来帮助他们。”
