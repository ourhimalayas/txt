## 疫情｜上海确诊病例居全国最高 当局出台多项支援平息民愤

**图/文｜CP**
**发布时间｜29/03/2022**
![](https://assets.gnews.org/wp-content/uploads/2022/03/上海確診病例居全國最高-當局出台多項支援平息民憤.jpg)

[点击关注官方盖特](https://gettr.com/user/mightygroup)


上海疫情持续严峻，昨日（28日）起以黄浦江为界实行两段式封城，进行全民核酸筛查。封城首日，上海确诊病例再创新高，录得4477宗本土个案，占当日全国总病例的八成。

上海市卫健委一级巡视员吴干渝表示，昨日是浦东、浦南及毗邻区域封锁筛查的第一天，当局累计组织1.7万名采样人员，设置6300个采样点。截至昨日深夜，共筛查超过826万人，发现3831宗阳性病例，大部分来自浦东新区。

另外，由于部分地区长时间封控，民众怨声载道，各种不利政权的言论不断在网上发酵，为了平息民愤，上海市发出《告通知书》要求各街道、居委会要保障核酸筛查期间居民的生活需求，并开通热线，为物资补给及就医配药提供服务，个别独居老人或特殊群体还要配备义工上门一对一提供帮助。

不仅如此，当局还提出21项扶持中小微企业的补贴措施，对包括零售、餐饮和交通等多个行业实施减税、免租、贴息等纾困支援。

有经济学家认为，上海是中国最富裕的一线城市，亦是中共的纳税大省，保守估计，上海封城8天至少令中共第一季与第二季经济成长较去年同期减少0.4%。
