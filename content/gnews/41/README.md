1977那个年代照片是相当奢侈的东西。那时候华国锋的两个凡是把对毛贼东的个人崇拜抬到了与神齐平的地步。正如现下习近平把自己称作太阳一样，那些专门为这个太阳创作的歌曲尽管在社交媒体上根本没什么人去听，其实对习近平这种懂音乐的人来说，创作水平相当差，但他不得不如此，毕竟真正的歌曲是不可能去歌颂独裁的，独裁也会抹杀掉人的创造力。华国锋彼时正要去做毛贼东的接班人，两个凡是是他自己所珍藏的毛贼东的遗言，然而毕竟毛贼东人死了，人死如灯灭，谁还会把他的屁话当神一样供着，并且两个凡是是华国锋提出的，也只有华国锋这样的蠢人，沾沾自喜拿着所谓的皇上的圣旨遗嘱，好比大清皇帝把遗嘱藏在正大光明的牌匾之下，其实里面的内容早让皇后改成了自己竖立的继承大统的阿哥。如果这个太后势力顽固，辅佐的新帝有手腕，那不愁坐不稳这江山，主要就是那些篡权夺位的阿哥们，太后不给力，前朝也不给力，也就两三年朝野又会被颠覆了。华国锋走的是后面这个剧本，所以不久以后，很快，邓小平采取了策略，马上就把他干掉了。当时邓小平还去中国的南海画了个圈，歌里不是唱着，神话般的崛起一座城，叫深圳。紧接着1978年就否定了两个凡是，改革开放了。华国锋这段就过去了。人虽然还活着，但是已经成为了历史的老人，属于满清遗老，活着跟死了没差别。和毛贼东一样，人死如灯灭，尽管华国锋头上一连串的官衔，但是真正掌权的那个人是邓小平了，没人再把他当回事儿了。是的，宇宙无限大，地球就是一粒沙，一段人生而已，华国锋在中共历史上做过皇帝，但几乎没人记得住他。

七哥说给习近平算过命，目前为止，所有的一切都在命数里走着，一步也不曾差错。当年爆料南普陀会议的时候，就连江志诚都不曾被告知，据说这个准确的发起人是曾家。曾是有信仰的，当年就是他去的南普陀，又是拜又是做法事，南普陀一夜之间成为大菩萨的道场，来往的信众络绎不绝，听说最近都要上市了。南普陀会议当年就是要把习近平当作傀儡，然后进行共产党统治全球的计划。这个计划里的那个沉船计划，意思就是江家这一伙儿人的船都停好了，着陆了，习近平自然就是傀儡了。可是不巧的是，七哥从当年给盘古奠基取五湖四海之水土的人那里听说了很多不可思议的事情，尤其是南普陀山上的那个土，风水先生说这个南普陀决定的事情大概率是办不成。其实从最自然的角度来说，肯定是办不成的，江家所谓的沉船里面的巨量财宝，那不是江家的，那都是从老百姓身上，几代人搜刮来的，凭什么这些钱就得白给共产党这伙子人？无奈中国这片大地是被诅咒的，总是有这些太邪门的事情发生，这个给中国人下蛊的做法和妖术每年都要进行，怪不得共产党敢十拿九稳操作这件事。

如今的习近平中共政府似乎也走到了两个凡是的时刻。不论习近平死不死，他指定的那个接班人一定是有危险的，好似华国锋一样，他就是所谓的毛贼东的接班人，但两个凡是是他自己说的。那么凭什么现在的习近平时代，其实任意一个人都可以跳出来说自己是习近平的接班人，并且发明一些和习时代类似所谓的语录来当作自己的政治筹码。这就是七哥所说的，20大是共产党真正的腥风血雨，党内九千万党员都要被迫卷入这些无端的斗争中。

七哥说起1977当年的照片，中共当年的内斗，和现在有很多共通的地方。也许这个照片出现的时间恰好就预示了真正的中共历史的类似时刻。太多的细节老百姓都不知道，今天有了互联网，记录了一切，共产党想要篡改也没办法。七哥说还有几张照片，他相信都会给到他手里的，也许又是某些历史的时刻，那些曾经类似的历史，一样一样浮现出来。时间让人记录，同样的时间，不同的人生，1977年七哥拿袖子擦鼻涕把胳膊擦到发亮，而中南坑当时在内斗杀人；今天七哥已经不擦鼻涕了，享受着全世界最好的物质生活，并且和战友们进行着最无私最纯真的精神沟通，中南坑还是在内斗杀人。啧啧啧，共产党不死谁死，连时间都唾弃这样的人。
