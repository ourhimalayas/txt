编译：东京樱花团/Hopkins


```
【编者按】中共海军正着眼于使用海外港口，包括大西洋港口。华盛顿智库战略与国际研究中心 (csis) 已经确定了中共参与的撒哈拉以南非洲近 50 个港口，有的正在建设中。 csis 表示，中共可能会利用它们“扩大其军事和政治影响力”，引起美国的国家安全担忧。《经济学家》周刊12月11日发表了这篇文章。
```

![](https://assets.gnews.org/wp-content/uploads/2021/12/1-155.jpg)

```
当中共于 2016 年在非洲之角的吉布提开始建造其第一个海外海军基地时，中共官员试图减轻西方对其国家扩大军事足迹的担忧。他们说，海外基地（如图）只是为了支持多国打击海盗的努力，帮助确保重要的航道安全，并使中共能够保护该地区的公民。五年后，吉布提的设施仍然是中共军队唯一的外国堡垒。
```



```
但中共官员正在悄悄地考虑建造更多基地的可能性。去年 11 月，五角大楼关于中共军力的年度报告中确定了 13 个国家，中共可能已经考虑在这些国家设立其他基地。它们的范围从中亚的塔吉克斯坦到非洲大西洋沿岸的安哥拉。美国官员认为，中共已与包括柬埔寨和阿拉伯联合酋长国在内的几个国家签订了秘密基地或港口准入协议。
```



```
海外有近100个民用港口作为投资者或运营商在其中有一定的影响力，这可能对中共有所帮助。华盛顿智库战略与国际研究中心 (csis) 已经确定了中共参与的撒哈拉以南非洲近 50 个港口，有的正在建设中。 csis 表示，中共可能会利用它们“扩大其军事和政治影响力”（见地图）。中共海军舰艇曾访问过一些基地。
```



```
12 月 5 日，《华尔街日报》援引美国情报报告称，中共希望在赤道几内亚部署海军。那里的基地将使中共海军在非洲大西洋一侧获得第一个永久立足点。虽然距离华盛顿将近 10,000 公里，但华府官员仍然非常担心。 10 月，美国政府派遣副国家安全顾问乔恩·费纳 (Jon Finer) 访问这个拥有 140 万人口的小产油国，并建议其领导人抵制中共的示好。据信，Finer 先生告诉赤道几内亚的官员，允许中共开设基地会引起美国的国家安全担忧。
```



```
中共正在成为一个更具冒险精神的海军强国。它在本世纪大大加强了其舰队，聚集了比美国海军更多的船只（虽然吨位没有更多）。但这些船只中的大多数都停靠在印度洋-太平洋水域。目前还不清楚中共是否有更远的军事野心。中共海军不能指望维持一支面向大西洋的舰队，能够与美国及其盟国抗衡，美国及其盟国的舰艇、基地和补给线在大西洋中部和北部占据主导地位。
```



```
或许，中共希望其海军更多地进入港口的原因与其在吉布提描述的动机非常相似：为中共贸易确保航道、打击海盗和保护海外中国公民。 2011 年，随着利比亚陷入内战，海军帮助从利比亚紧急疏散了 30,000 名中国公民，海军开始感到对远程基地的需求更加迫切。
```
