【新西兰RNZ 4月20日】 所罗门群岛无视澳大利亚的警告，与中共国签署了一项新协议，人们担心它可能会让中共国在南太平洋建立军事存在。澳大利亚总理莫里森驳斥了关于澳大利亚政府破坏关键安全关系的指控。
![](https://assets.gnews.org/wp-content/uploads/2022/04/1028714402_0_19_5472_3114_1920x0_80_0_0_2936718b8fce7da3eb32903015ac0090.jpg)
澳大利亚工党称处理这一问题是二战以来澳大利亚在太平洋地区最大的外交政策失误。它质疑为什么在协议草案泄露后政府派太平洋部长泽德塞塞利亚，而不是外交部长玛丽斯佩恩前往霍尼亚拉。

塞塞利亚参议员上周被派往所罗门群岛，力求最后努力说服霍尼亚拉政府放弃这笔交易，但现在看来，这次访问毫无结果。

影子外长Penny Wong表示，莫里森去年无视有关该协议的警告，本应亲自干预以确保该协议永远不会签署。外交部长佩恩驳斥了Penny Wong的评估，否认莫里森政府的不作为助长了局势。

莫里森在阿德莱德告诉记者，他是故意决定派遣参议员塞塞利亚，因为公开向所罗门群岛就该协议施加压力会适得其反。他还表示，整个地区一直面临来自中共国的“强烈”接触，并表示北京在太平洋地区做出了“各种承诺”和“各种投资”，这可能“非常有说服力”。他说：“这是我们现在正在处理的挑战，我们已经处理了很多年。这不是一个新问题，这些威胁仍然存在”。

澳大利亚、新西兰和美国都对这种情况可能为其他太平洋小国开创先例表示担忧。

美国国家安全委员会发言人艾德丽安·沃森表示，美国就此召集了来自澳大利亚、日本和新西兰的高级官员的会议。美国高级官员库尔特坎贝尔定于本周晚些时候访问所罗门群岛，因为美国警告说该安全协议开创了“令人担忧的先例”。
