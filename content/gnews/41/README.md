你要知道新西兰是什么地方？新西兰是海航藏自己情人的地方，是一辈子的情人，而且有大量的资产在那里。我给你们举个例子，平安保险的马明哲，平安保险的孙建一，有几千亩的豪宅在那。而且我告诉你，那几个情人的豪宅里埋的黄金不亚于一个国家的、（收藏的）珠宝不亚于一个国家，没有一样不是真东西。孙建一和马明哲都各有各的情人，都有几个私生子。王健最起码有六个私生子在新西兰，大概有五六个私生子在墨尔本，他一共有三十多个私生子，都是试管（婴儿）方式找人代孕的。王岐山最重要的（藏钱处）是澳大利亚和新西兰，王岐山已经“死”了，下一步核心的关键是孟建柱、孙力军、傅政华，他们是我们下一个要拿下的。孙力军的老窝就是在澳大利亚，还有新西兰。而且孙力军此人在澳大利亚藏的珠宝和字画，还有黄金白银，绝对不是你想象的数，是惊天的数！好多餐厅的老板都被他们买通了，都是他们的线人，澳大利亚有孟建柱和孙力军的线。傅政华的钱在新西兰，有些也在澳大利亚。我们把这三人下掉了，就是下了盗国贼的刀把子。四百万个公检法人员，绝对会把他们给生吃了。刀把子没了，就剩下枪把子了。

**郭文贵先生2018年9月12日直播：**

双汇未来也是中国一个巨大的炸弹。头一段时间，美国有两位检察官跟我聊天的时候说“我想跟你聊聊双汇，还有在美国的收购的这几家企业”，他们微笑着跟我说。我一听我就明白了，美国已经盯上双汇了，双汇在香港的总部，包括购买美国史密斯企业，和美国高盛的合作，还有上市，还有融资，美国已经高度关注了。

而且他们问了我几个企业的名字，把我吓了一大跳，因为那是温云松的，温家宝家族的。温家最发大财的，其中第一是平安，第二就是双汇，还有一个，是美国高盛下面的很多(让温家)发了大财。下一步美国一定在这儿着手。他问我的时候，因为我和万龙董事长认识，还有他下面一个小兄弟，姓杨，这小伙子特别不错。由于这种关系，当时我说对不起，今天我不想谈漯河双汇的事情。

说说平安当年的事情……平安最大创始人马明哲，马明哲唯一留下的合伙人叫孙建一，平安里面最重要的段伟红，董事，现在被抓了，还有我的好哥们车峰。当时共产党里边最牛叉的人，搞了30年调查的——合法调查——就是马建副部长，还有邱进。邱进的哥哥叫邱跃，北京市规委的主任，是我朋友。黄燕那点事，我啥不知道啊？！当时（总）书记是谁？北京规委主任是谁啊？胡锦涛的夫人刘永清。我认识刘永清的时候我还不知道她是胡锦涛的老婆，你说我啥不知道？这钱咱不要，郭文贵要是在国内敢要这人的钱，我活不到今天，早被弄死一千回，一万回都不止了，咱不要。

所以平安集团，玩多大？马云，马化腾，他俩算个屁都不是，他俩那叫什么？现在是称作“网络金融”，严格讲，今天真正的要叫他们金融骗子。真正中国金融老大就一个——平安，平安里就是马明哲，马明哲的女儿在香港，在中环有一个叫中环中心的，当年是香港最牛的富豪的，刘銮雄，后来卖给了何鸿燊的四太梁安琪。那个种着石头外表的楼，在那个楼里边你们上去，到第二层大堂曾经有一个红酒房，红酒销售处就是马明哲女儿开的。马明哲女儿跟我见过N次面，她都不记得我了，因为我叫了另外的名字，我姓吴，我不姓郭，叫吴楠，她都不记得了，太多人了。那时候卖的最好的酒叫什么romantic county。干嘛的？就洗钱么。马明哲，我跟他见过几次，马明哲装神弄鬼的，介绍我跟马明哲见面的人，在两周以前被抓了。
