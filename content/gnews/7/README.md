![](https://assets.gnews.org/wp-content/uploads/2022/01/F83891BB-7A1F-44C4-A8F0-82F0C2285816.jpeg)（图片来自美联社）

俄罗斯当局1月13日表示，俄国率领的2000多名军队已开始撤出哈萨克斯坦。

法新社报道，俄罗斯国防部表示，集体维和部队已开始准备将装备和物资装载到俄罗斯航空太空军军事运输机上，以返回常驻基地。

这是以俄罗斯为首的集体安全公约组织（Collective Security Treaty Organisation，CSTO）首度派出维和部队。据称，先前出动这些军队是因为哈萨克斯坦燃料价格飙涨引起的和平抗议演变成前所未见的暴乱，造成数十人死亡。
