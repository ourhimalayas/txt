# **救命，电商狗没单了**

连续几天失眠，感觉压力好大，想找个地方倾述下……我做淘宝五年了，从当初和老公两人在家做淘宝到后来搬出去拥有了一个六七人的小团队，再到现在两人，感觉真的像做梦一般，我想这一次可能真的撑不下去了……这几天淘宝单子越来越少了，每天的营业额都不够付房租，看着一仓库的货没有地方出去，心里真的很不是滋味。现在每天就发十几单，有时候甚至只有几单，有些地区发出去也是退回来，一年五六万的房租，银行还有两百多万房贷，再这样下去下个月的房贷也还不上，马上要断供了，没有退路，前方的路不知道怎么走，无助又无奈，想放弃又不甘心，不晓得该怎么办。
