## 大企业对黑客束手无策

图片来源： FAZ

德国领先的电子零售连锁店受到黑客攻击的严重打击。周一晚上，Media Markt 和 Saturn 拥有十多个国家组织和数百家商店的欧洲商店成为网络攻击的目标，攻击者显然设法渗透到零售连锁店的核心 IT 区域，以便通过故意冻结数据造成损害，从而使其暂时无法使用。

隶属于杜塞尔多夫 Ceconomy Holding 的公司立即通知了相关部门，目前正在全力确定受黑客攻击影响的系统，评估损坏情况，并在必要时进行补救。但这可能需要时间。因为至少有一些系统目前由于大量冻结数据而不能工作。

对市场的攻击是对全球公司、组织和政府机构的一系列攻击的一部分。例如，其中包括德国软件公司、微软、管道运营商和美国政府。自病毒危机爆发以来，他们的计算机和 IT 系统越来越多地受到黑客的攻击和劫持。攻击者实际上使用特殊软件将受影响计算机上的数据作为人质，并要求赎金以释放这些数据，通常要求以加密货币的形式索要赎金。金额可达数百万欧元。

追踪黑客并不容易。鉴于其技术系统的复杂性，这个问题似乎使许多公司不知所措。根据 IT 咨询公司埃森哲的一项研究，世界上每二家大公司在面对复杂的网络攻击时几乎都无能为力。 55% 的全球公司目前无法识别此类攻击，无法对其采取有效行动并迅速限制其潜在后果。

在攻击数量连续数月增长的背景下，全球 80% 的大公司在过去几个月都增加了网络安全预算。但是攻击浪潮几乎不受控制地滚滚而来。与此同时，保护他们的 IT 系统变得如此昂贵，以至于在埃森哲研究中调查的近 4800 家公司中的大多数表示，所需要的的资金无法在可持续的基础上提供。换句话说，网络安全要花费数十亿美元。

在病毒危机期间将工作转移到家庭办公为许多系统中的攻击者打开了薄弱的环节。 黑客充分利用了这一点。 8 月，柏林行业协会 Bitkom 估计，仅网络攻击对德国经济造成的损失每年就达到 2200 亿欧元。 这是冠状病毒大流行爆发前几年的两倍多。自去年以来，十分之九的公司受到了黑客攻击。这是调查公司在调查了一千多家公司后得出的结论。来自两家公司的 IT 专家将在未来几周内为Media Markt和Saturn忙得不可开交。
