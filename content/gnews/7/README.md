美国历史最悠久的科学杂志已跌至新低。《科学美国人》发表了一篇热门文章，抨击已故的威尔逊（E. O. Wilson）在遗传学工作中支持“种族主义”思想。

威尔逊写了许多关于社会生物学的书籍，该领域旨在解释基于进化和遗传学的社会行为。威尔逊将他的分析应用于动物和人类。

威尔逊最有影响力的著作之一《社会生物学：新综合》确实让《科学美国人》热门文章的作者感到不安，他声称威尔逊“促成了先天与后天的错误二分法，并催生了人类之间的差异可以通过基因、遗传和其他生物学机制来解释的概念。”

声称基因在人类行为中发挥作用的说法，如何以某种方式成为了“有争议的”，这仍然未知。**但自从乔治****·****弗洛伊德事件之后，突然间一切都变成了种族主义者，包括遗传学。**

“当然，你可以争论基因起作用的程度，但即使是威尔逊本人也写了许多论文，研究基因和文化在决定行为方面的错综复杂的相互作用，”Medium上的一份报告写道。

**《科学美国人》不再可信**

《科学美国人》文章的作者继续声称，整个遗传学研究都是“种族主义者”，但除了她对此事的感受外，没有提供任何证据。

她还抨击“白人经验主义”提出了遗传学的概念，“女权主义立场理论”认为，这种观点将“白人”置于“人类状况和我们世界的有价值观察者”的地位。

**女权主义者对白人的这种持续的仇恨并不是什么新鲜事**，但近年来它确实有所增加，**涵盖了从科学到自然，再到医学的方方面面。根据“醒来”的暴民的说法，因为“白人”，一切都是问题。**

有时很难判断《科学美国人》中的文章是讽刺的还是真实的。当今，界限如此模糊，以至于“新闻”读起来都像是某种病态的笑话。

您可能还记得，早在 2019 年夏天，《科学美国人》就发表了一篇文章，推动“无性别”的未来。

接受同性恋 LGBTQ 暴徒的教条，那篇文章否认科学，声称基因与生物性别无关。它认为，一个人是他、她或它，可以随心所欲的是任何性别。

《科学美国人》曾经是可信的，值得注意的是。但不知何故，它被颠覆者渗透，他们将叙述从科学转移到伪科学的怀抱。

现在，《科学美国人》是一个不再有任何科学依据的期刊的笑话。在这一点上，它也可能叫《疯子》杂志。

“像这样将威尔逊拖入泥潭，对任何边缘化社区都是不公平的。威尔逊不仅是生物学专家，而且在拯救地球的斗争中做出了巨大贡献，” Medium报告解释道。

“‘达尔文的继承人’并不是他唯一的标签，他还经常被称为‘生物多样性之父’，帮助普及了生物多样性这个词……越来越多地，我们开始意识到生物多样性的丧失对我们的星球是多么有害……威尔逊竭尽全力扭转这种损失。”

“这是威尔逊的真正遗产，这是我们应该为之奋斗的**。如果我们为了宣传某种意识形态而开始审查科学，我们永远也达不到目标。”**
