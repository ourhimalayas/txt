## 中概股人人网再度触发熔断
` 澳喜农场` [轉載自GNews](https://gnews.org/zh-hans/1742823/)

撰稿人：东南
![](https://assets.gnews.org/wp-content/uploads/2021/12/熔断.png)
（图片来源网络）

据TechWeb网报道12月10日中国概念股人人网暴跌超53%，盘中多次触发熔断，最新股价为12.15美元/股，市值为2.90亿美元。此前，人人网宣布，美国纽约州最高法院商业法庭今天就其股东衍生诉讼案举行了听证会，以审议批准和解协议的动议。听证会上法院拒绝批准该和解动议。法院驳回了协议书中关于确定有权从和解资金中获得分配的人人公司A类普通股和美国存托股份持有人的记录日期的程序。

此前人人网高管被指控将公司资金用于个人投资目的，违反了信托义务。

根据媒体报道，2019年3月，原告 ( Heng Ren Silk Road Investments LLC、Oasis Investments II Master Fund Ltd、Jodi Aram ) 等对陈一舟、赵克仁、DCM、OPI和Duff&Phelps提出合并投诉，认为公司内部人士导致人人在出售OPI交易中以被低估的价格转让其投资，使部分被告受益，损害了人人的利益。
2021年10月8日，人人网宣布与原告签订和解协议，将创建一个金额高于3亿美元（约19.1亿人民币）的和解基金，超过了所请求的赔偿金额。截至登记日的人人网 A 类普通股和美国存托股 ( 每股美国存托股代表 45 股 A 类普通股 ) 股东，将有资格从和解基金中获得补偿。当时人人网宣称，如该协议得到批准并生效，相关诉讼将被撤消，并将由法庭指定的管理人根据和议分配和解基金。

还记得当时和解消息传出后，人人的股价出现了大幅度的上涨。但这种假扮的辉煌似乎没有维持很久，昨晚的突如期来的熔断下跌53%还是被吓了一跳，说实话2015年之后的人人网，到底是做什么的网站还有谁知道吗？把这种过气的品牌拿出来，用假大空的业绩包装出来的公司犹如茅草搭建的房屋，当全球灭共的飓风刮过之后剩下的只是一地鸡毛，有些散户韭菜们还抱有希望这些中概股能像“滴滴”那样重新回港上市并能再次“东山再起”。但中概股退市后去港股就有救吗？中国股市早已经是满目疮痍了，不行央行干脆就再来几次降准吧。
