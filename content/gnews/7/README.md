![](https://assets.gnews.org/wp-content/uploads/2022/02/10-28.png)图片来源 2022年2月16日，加拿大隊在北京冬奧領獎時，集體「打掃領獎台」 (SEBASTIEN BOZON/AFP via Getty Images)
据《新唐人》2月19日消息。 2月16日，北京冬奥短道速滑男子5000米接力赛结束，加拿大队以6分41秒257的成绩率先冲线，获得金牌。

在当天的颁奖仪式上，加拿大队5名队员登上领奖台前，一起伸出右手做出打扫奖台的象征性动作，左右方向各一次，然后才上台欢呼。

这个动作是加拿大队的首创。 2018年的平昌冬奥会上，一名加拿大运动员领奖前曾做出打扫奖台的动作。其本人事后没有正面回应，但此举被媒体解读为“抗议韩国的赛场肮脏”。

这次，加拿大队在受访时称，他们只是“希望在迎接新奖牌的时候，领奖台是干净漂亮的”。
