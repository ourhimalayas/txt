#### **【雅典娜快讯】，紧贴时事脉搏**

- ![](https://assets.gnews.org/wp-content/uploads/2022/01/布拉莫斯反舰导弹系统.png)图片来源：搜狐


菲律宾购买了价值3.749亿美元（5.195亿美元）的超音速导弹系统，能够从陆地、海上和空中向敌方船只开火，从而提高了在南中国海争议水域的地位。

菲国防部长德尔芬·洛伦扎纳（Delfin Lorenzana）周五证实，菲律宾已经与印度签署了购买布拉莫斯反舰导弹系统的协议。这种由印度和俄罗斯联合开发的超音速巡航导弹是世界上最快的导弹之一。

安全分析人士表示，该系统标志着菲律宾战略的转变。近年来，菲律宾与中共国海军和海岸警卫队的船只发生了几次冲突。

中共国坚称，它对南海90%的海域拥有历史性权利，包括菲律宾声称拥有的领土。北京无视常设仲裁法院2016年驳回中共国主张的裁决。

新加坡国际战略研究所的安全与国防分析师尤安·格雷厄姆（Euan Graham）说：“菲律宾正在逐步转变能力，使该国有足够的资金来阻止侵略性的海军力量。”

他将与印度的交易描述为对菲律宾的“重大承诺”。他分析：“这表明他们正在重新调整国防方向，从国内安全转向常规防御。中共国是菲律宾唯一的严重威胁”。

这也提醒人们，那些受中共国海上扩张威胁最大的国家正在增强其防御能力。

越南和印度尼西亚已经拥有反舰导弹系统，前者也在与中共国进入其领海的行为作斗争，后者则经常受到中共国捕鱼船队边界推进的刺激。

澳大利亚正在为其F/A-18超级大黄蜂战斗机获取类似的能力。

格雷厄姆博士说，中共国还拥有广泛的反舰导弹系统，在某种程度上，那些受到这个崛起大国威胁的国家正在效仿中共国。

中共国已经利用远程精确导弹能力，使其他国家的海军更难在其海岸线附近行动。因此，越南、菲律宾和澳大利亚正像镜子一样紧跟中共国的军事能力的增长并采取反制措施。

如果中共国是一只大豪猪，那么其他这些国家就会把自己变成小刺猬。他们想确保侵略者会有风险。对菲律宾来说，鉴于它是一个难以防御的群岛国家，这一点至关重要。

菲律宾也是该地区第一个从印度购买这一系统的国家。

洛伦扎纳先生说，与印度政府谈判的合同内容包括为该国海军交付三个电池，培训操作员和维护人员以及必要的综合后勤支持（ILS）包。据了解，菲律宾军队也有可能获得两个电池。

去年，中共国对澳大利亚、美国和英国之间形成AUKUS协议做出了愤怒的反应，该协议被广泛认为是对中共国扩张举措的回应。

去年11月，习近平告诉该地区领导人，中共国“是促进该地区持久和平的共同努力”的一部分，并将“永远不会称霸，更不会欺负小国”。然而，就在一周前，菲律宾政府报告说，三艘中共国海岸警卫队的船只在有争议的南沙群岛对两艘菲律宾补给船进行了拦截并发射了高压水炮。
