- 开往圣彼得堡的416英尺货船在英吉利海峡被拦截
- 货船“波罗的海领袖”号被法国海警拦截并扣押进一步检查
- 一艘满载汽车的货船于周六在滨海布洛涅（Boulogne-sur-Mer）被扣押
- 法国当局称该船涉嫌违反欧盟贸易制裁

![](https://assets.gnews.org/wp-content/uploads/2022/02/image-2750.png)图片来源：每日邮报
最新的贸易制裁开始打击俄罗斯经济，周六早些时候，一艘开往圣彼得堡的货船在英吉利海峡被拦截。

据了解，这艘416英尺长名为“波罗的海领袖”的被扣留货船属于 Promsvyaz 银行，本周受欧盟制裁的俄罗斯五家主要银行之一。

Promsvyaz银行在（英国）下议院被提名为受制裁的五家俄罗斯国有金融机构之一。

与此同时，美国财政部的一份文件称：“波罗的海领袖”由 Promsvyaz 银行所有，该银行被描述为对俄罗斯国防工业“具有系统重要性”。

在乌克兰入侵后，海事官员有权拦截和扣押涉嫌违反欧盟制裁的俄罗斯相关船只。

据了解，这艘船在离开法国城市鲁昂后，在诺曼底翁弗勒尔（Honfleur）附近被海关官员截获，这是俄罗斯资产在运输途中被冻结的第一起事件。

法国当局表示，这艘满载汽车的货船所有者涉嫌违反与乌克兰战争有关的贸易制裁。“波罗的海领袖”号于2月19日抵达法国鲁昂市，并在那里停靠了将近六天，直到周五晚上离开。该船原本计划于3月3日星期四抵达圣彼得堡。

该海事区的区域通讯官马格宁（Véronique Magnin）船长表示，该船随后在凌晨3点至4点（格林威治标准时间凌晨2点/3点）之间转移到法国滨海布洛涅港，海关官员正在进行检查。

据说“波罗的海领袖”号上的所有19名船员都在配合调查。该过程可能需要长达48小时。

马格宁船长认为，这艘船“强烈怀疑与制裁所针对的俄罗斯利益有关”。

“波罗的海领袖” 号是一艘22年前建造的滚装货船，预计将于3月6日停靠在圣彼得堡。该船悬挂俄罗斯国旗。

英国广播公司援引一位官员的话说：“应法国政府的要求，它已被带到法国港口，因为它被怀疑属于欧盟制裁莫斯科的目标公司。”

据说俄罗斯驻法国大使馆正在就“扣押了一艘货船”向法国当局“寻求解释”。

巴黎的一位发言人告诉俄罗斯塔斯社通讯社，船长已致电大使馆，大使馆随后联系了法国官员。
