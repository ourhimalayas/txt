**区块链结构**

**区块链是由包含三个项目的“区块”组成的。**

**首先是数据本身**。在比特币的例子中，这包括所有与特定交易有关的信息，如日期和数量；其次是区块的**哈希值**，这是一个识别区块和其内容的唯一值。

【备注：哈希值(hash values)是使用哈希函数计算得到的值，哈希函数是是一种从任何一种数据中创建小的数字“指纹”的方法。】

**在任何特定的区块链上，哈希值将共享相同的格式。修改一个区块的数据也将导致一个完全不同的哈希值。**

第三个也是最后一个项目是**前一个区块的哈希值**，它有助于区块链的“链”部分。此功能使某人几乎不可能篡改区块链的数据，因为他们的链副本会与所有其他用户发生冲突。
