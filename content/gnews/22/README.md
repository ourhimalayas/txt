![](https://assets.gnews.org/wp-content/uploads/2021/12/np_file_130652.jpg)图片：来自网络


中共在全球社交媒体上放出大量用于推进威权主义计划的虚假帐号，让这些帐号看起来像真的，也有大量的关注量。他们还会找出在网上批评中共国的人——搞清楚他们是谁，住在哪里。

中共国政府发起了一场全球网络运动，用来提升自身形象并消除关于其侵犯人权的指控。大部分工作都在幕后进行，在自动发帖的机器人网络和难以追踪的网络角色的掩饰下进行。

目前，《纽约时报》（The New York Times）审阅的一组新文件详细揭示了中共官员如何命令私营企业按需生成内容、吸引关注者、追踪批评人士，并为信息战提供其他服务。这类操作越来越多地在脸书和推特等国际平台上出现，而这些平台在国内都被中共政府屏蔽了。

这些文件出自承包商的投标书，它们提供了一个难得的机会，让人们得以一窥中共庞大的官僚机构如何在全球社交媒体上散播宣传信息和塑造舆论。在《时报》（The Times）就这些文件联络中共国政府后，它们被下架了。
![](https://assets.gnews.org/wp-content/uploads/2021/12/image-1357.png)5月，发布在网上的一则通知，号召承包商进行投标。
5月21日，上海警方一处分局发布文件，向私人承包商招标，内容是中共国官场中所谓的舆论管理。官员们一直依靠技术承包商来帮助他们追踪国内社交媒体动态，并通过审查和在国内散播虚假内容来主动塑造舆论。直到最近，官员及舆论管理业才将注意力转向国外。
![](https://assets.gnews.org/wp-content/uploads/2021/12/image-1358.png)公安局文件的第14页详细说明了所需的服务。
上海警方希望在推特（Twitter）、脸书（Facebook）和其他主流社交媒体平台上创建数百个虚假帐号。该分局强调这项任务时效性较强，表明它希望这些帐号能做好迅速行动并引导言论的准备。

过去两年里，类似于上海警方想购买的这种僵尸帐户网络已经推动网上亲中共的流量激增。有时候，这些帐号网络在社交媒体上通过点赞或转发来支持政府官方帐户。其他时候，它们会攻击那些批评政府政策的社交媒体用户。

最近，脸书移除了500个帐号，因为它们被用来传播一位名叫威尔逊·爱德华兹（Wilson Edwards）的瑞士生物学家的言论，据称他在文章中写到美国正在扰乱世界卫生组织（World Health Organization）追溯新冠病毒起源的工作。瑞士驻华大使馆表示，威尔逊·爱德华兹这个人并不存在，但中公官媒已经引用了这位虚假科学家的指控。
![](https://assets.gnews.org/wp-content/uploads/2021/12/image-1359.png)公安局文件的第14页详细说明了所需的服务。
上海警方在社交媒体上的努力并不仅仅关乎数据，这部分文件强调了从动用大量僵尸帐号这种简单粗暴的策略转向破坏性更强的行动。

警方想要实现的是复杂性和影响力的升级：塑造一批拥有活跃关注者的帐号，在必要时能用来实现政府的目的。

这样的要求表明，警方官员明白通过这些花钱买来的帐号同公众进行密切互动的必要。随着社交媒体公司越来越经常地移除那些看起来不真实或是行为雷同的帐号，这种更深层次的互动会让虚假帐号更具可信度。

虚假信息专家表示，水军网络与中公政府的关联之所以引人注目，是因为它们缺乏与其他帐号的互动。尽管它们可被用来攻击他人，增加官方政府内容的点赞数量，但由于这些自动生成帐号的关注者很少，所以它们的个人影响力基本上也比较小。
![](https://assets.gnews.org/wp-content/uploads/2021/12/image-1360.png)公安局文件的第14页详细说明了所需的服务。
当局使用了中共国网警中常见的“落地”一词，指的是追踪社交媒体帐户背后的真实使用者。

中共国的网络警察越来越频繁地追捕和威胁发表意见的互联网用户。起初，其代理人专注于本地社交媒体平台。2018年，他们开展了一场新的运动，拘留中共国境内找到方法绕过政府封锁的推特用户，并迫使他们删除自己的帐户。

现在，该运动已扩展到居住在中共国境外的华人。该文件阐明了上海警方希望找出某些帐户背后人员的身份，并追踪其用户与大陆的联系。然后，警方就可以威胁用户在中共国的家人，或在帐户持有人返回中共国时将其拘捕，以迫使这些网络批评者删除帖子甚至删除整个帐户。
![](https://assets.gnews.org/wp-content/uploads/2021/12/image-1361.png)公安局文件的第14页详细说明了所需的服务。
在中共国先前的信息宣传活动中，疑似僵尸网络的帐户被用来给政府和官方媒体的帖子增加不切实际的点赞和转发数量。人为制造的流量通过推荐算法，可以使这些帖子更有可能被许多社交媒体网站和搜索引擎上显示出来。

最近几周，一个疑似僵尸网络的帐户出现了类似模式，大肆宣传官方媒体记者发布的证据，声称网球运动员彭帅是安全的，在北京自由地外出吃晚餐，并参加了青少年网球锦标赛。

上海警方非常清楚地解释了该部门想要的功能，可以看出他们十分熟悉社交媒体上的推荐算法。它的做法强调了宣传官员们熟知的事情：在短时期内，一堆垃圾帐户可以令官方帐户的帖子看似正被疯狂传播，从而为其增加曝光率和可信度。
![](https://assets.gnews.org/wp-content/uploads/2021/12/image-1362.png)公安局文件的第14页详细说明了所需的服务。
随着中共宣传活动在海外发展，他们越来越依赖视觉媒体。官员们正在寻找这样一家公司，不仅能维护和使用假帐户，还能生成原创内容。这对视频的需求很高。

《时报》查阅的另一份文件显示，上海市的同一公安分局于11月从另一家公司购买了视频制作服务。警方要求供应商每月至少提供20个视频，并在国内外社交媒体上发布。该文件将这项任务称为原创视频制作，用于打好“舆论战”。

今年早些时候，《纽约时报》和ProPublica的一项分析显示，数千个描绘维吾尔少数民族成员过着幸福自由生活的视频是一场宣传运动的关键部分，推特最终认为这些视频归属于中共。当推特关闭这些帖子背后的帐户网络时，它也关闭了据说帮助拍摄了宣传视频的承包商相关的帐户。推特发言人拒绝置评。

文件显示，在该上海市公安分局的招标公开三周后，一家名为上海云邻（Shanghai Cloud Link）的公司中标。在竞标提案中，该公司自称只有20名员工。根据其创始人魏国林的领英页面，该公司与跨国公司合作，在“数字政府”和“智慧城市”领域提供服务。

魏国林没有回应置评请求。上海浦东公安局没有回复通过传真发送的置评请求。

上海云邻提案中的工作可能只是冰山一角。中共国各地的地方政府和警方也提出了类似的服务要求，以影响海外社交媒体，但往往措辞含糊，偶尔会透露一些细节。例如，根据《时报》查阅的文件，2017年，内蒙古警方购买软件，允许政府水军直接在中共国境内外的多个社交媒体网站上发帖。

在另一个案例中，承包商下载了数百个脸书公共时间线的访问凭证，使其能够收集有关何人何时对哪些贴子发表评论的数据。脸书没有立即置评。

上海云邻的中标提供了一个窗口，可以从中了解某些类型的虚假信息服务的成本。
![](https://assets.gnews.org/wp-content/uploads/2021/12/image-1363.png)![](https://assets.gnews.org/wp-content/uploads/2021/12/image-1364.png)上海云邻的中标报价表，成本以人民币为单位列出。
在许多情况下，技术承包商希望直接向中共当局出售硬件和软件。在这种情况下，上海云邻的提案表明了一种基于服务的新模式：政府按月付费，就像一种用于社交媒体操纵的订阅。
