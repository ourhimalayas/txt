## 〖信仰之声】宁失一眼，不失全身
` 纽约香草山MOS03` [轉載自GNews](https://gnews.org/zh-hans/1604848/)

作者：纽约香草山福音部 山城小哥

![](https://assets.gnews.org/wp-content/uploads/2021/10/logo-5-768x103-6.jpg)



![](https://assets.gnews.org/wp-content/uploads/2021/10/image-367.png)

文贵先生最近两天直播说到许多如马云之流的中国“企业家”被中共绑架着到海外为中共拿回他们帮中共代持的隐藏在海外的从中国人民那里抢劫而来的财富。许多“企业家”竟然相信中共的承诺:只要为中共拿回这些钱，中共就给他们和他们的家人一条生路，不灭口他们。

可是，中共的话能信吗？人怎能去相信一个公然鼓吹无神论的杀人组织的承诺？如果历史让我们学到了任何东西，那就是:中共从来不信守它的任何承诺。对中共效忠的问题的关键在于:你效忠的对象随时准备背叛你，甚至杀了你。

我们能理解这些“企业家”们被中共用亲人的生命安全威胁的难处。可是，既然钱拿回去是死路一条，拿不回去也是死路一条。拿回去反而是帮助了谋杀自己和亲人的凶手。不如弃暗投明，抓住出国的机会，联系爆料革命，勇敢地站出来，把本属于中国人的钱交给新中国联邦，勇敢地出来爆料灭共。拿出勇气和国内的亲人断掉联系，就算中共威胁要迫害他们，拿出一份勇气勇敢地对中共说“你若煮了我父亲，请分我一杯羹”。

这样，有爆料革命做后盾，中共反而未必敢动“企业家”们的亲人，或者有必要动他们的亲人，亲人可能因此而安全了。即使亲人遇害，自己也有机会为亲人伸张公义。其实我们每个人的肉体从一出生都是被判了死缓，迟早会毁灭。与其以一个帮凶的身份毫无价值地死去，受神的严厉审判。不如勇敢地站出来为正义发声，反抗邪恶，不仅自己以往的过犯可以洗白，而且也不会虚度此生。即使亲人不幸遇害，相信神也必定给他们一个好的归属和安排。因为耶稣基督说:“为义受迫的人有福了，因为天国是他们的”。

何必要全家人一起被中共彻底毁灭呢？何必要一家人全下地狱呢？人，该切割的时候，就要学会切割。主耶稣基督说:如果你的手或脚使你犯罪，就砍掉它！因为肢体残缺着进入永生，总比四肢健全却被丢进永远不灭的火中好。如果你的一只眼睛使你犯罪，就剜出来丢掉它吧！独眼进入永生，总比双目健全却被丢进地狱的火中好。(马太福音 18:8-9)

更何况，可能因我们得救赎，神一喜悦，开恩，在我们的祈求下，我们原本逝去的亲人也得救赎了，得享在永恒里与神同在的福乐。因为神是仁慈的，祂听义人的祈求。

说到这里，其实这些企业家们能不能做出正确的选择，最核心的还是在于他们怎么看待生命，认不认识那位荣耀的神和主耶稣基督。

有人可能会说面对现在的中共，只有这些企业家才会有这样的难题，我因为身在其外，不用经历这样痛苦的切割和抉择，所以说得轻巧。其实，不管在哪个世代，但凡要让生命过得合神心意，追求永恒价值和福分的人，都始终都会面临着同样的问题。这些人想救自己亲人得肉体短暂的生命，但我是想救我亲人得永恒的生命。

主耶稣基督说:“爱父母过于爱我的人不配做我的门徒；爱儿女过于爱我的人不配做我的门徒；不肯背起他的十字架跟从我的人不配做我的门徒。试图保全自己生命的反而会失去生命，但为我舍弃生命的反而会得到生命。”(马太福音10:37-39 )

任何人在追求真正有价值，神永恒生命和荣耀的路上。都会有许多要切割的，甚至是自己合理的需要都要被切割，这需要耶稣基督式的勇气，牺牲和奉献，完全的无我，完全的只有神，完全的一心向善。对于那些我们目前无能为力去改变和救赎的亲人，我们所能做的只能是为他们祷告，特别是如果因为他们的羁绊，我们无法侍奉神的时候，该切割就要切割，即使那痛苦如剜掉一只眼。宁失一眼，不失全身，你怎么知道神某天会不会开恩让你又得回自己的眼睛呢？或许神让他暂时离开你，却是为了让你永远得到他。

在《旧约》时代，神拣选了利未人做祭司的族群来侍奉祂。那他们是怎样得到这份荣耀和恩典的呢？当时以色列人也是造了假神，造了“金牛犊”（那个时代的CCP)，并且跪拜。于是神借摩西的口吩咐说:“你们各人带着刀，从这个门到那个门，走遍整个营，不论遇见的是兄弟、伙伴还是邻居，只管杀他们。”利未人便照摩西的话去做。约杀了三千不肯悔改，跪拜假神的亲人同胞。正是因为此大义灭亲的举动，爱主胜过爱亲人的举动，才让神把祭司（与神最亲近的人)的职分给了利未人。而利未人和中共控制的“企业家”一样，在之前是被神诅咒的一群人。

现在神不是要我们杀掉亲人，而是要我们该放下的要放下，该切割先切割。更何况这其实不仅是我们，也是我们的亲人唯一得救的机会。真心希望神开这些“企业家”的心，保守他们的路，做出正确的选择。也祈求神开我们的心，保守我们的路。做一个“宁失一眼，不失全身”的明智勇敢之人。
