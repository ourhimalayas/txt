来自路透社REUTERS 2021年12月17日的报道，

**摘要翻译：**

香港，12月17日（路透社）——中共国官方正计划禁止富途控股有限公司（Futu Holdings Ltd）和向上金融（UP Fintech Holding Ltd）等互联网证券商向大陆客户提供离岸交易服务，这是过去一年中搅乱众多行业的广泛监管打击中的最新进展。

这两家在纳斯达克上市的中共国公司是该行业的两个最大的玩家，禁令将使中共国数百万股民无法在美国和香港等市场轻松进行证券交易。消息人士称：禁令的潜在动机是对数据安全和资本外流的担忧。

即将到来的限制措施紧随着前一波的取缔政策，该政策在过去一年里对从技术到教育再到房地产等一系列行业产生了广泛的影响。

有四位消息人士接受了路透社采访，其中一位说：受最新打击政策影响的公司可能会在“未来几个月”收到禁令的通知。所有消息人士都拒绝透露身份，因为他们无权向媒体发声。

富途控股和向上金融在香港证券及期货事务监察委员会都有注册许可，但该许可并不适用于中共国内地。消息人士称，中共国内地没有专门从事跨境交易的互联网证券商的运营牌照。

市值55亿美元的富途控股有限公司在给路透社的一份声明中说，它一直在与中共国当局沟通，但没有收到任何有关禁令的正式回复。它补充说，公司运营一切正常。该公司在4月份的后续股票发行的招股书中指出，其业务可能会受到中共国立场变化的影响，因为中共对法律法规的有最终解释权。

向上金融的估值为7.37亿美元，它说它一直遵循全球监管机构制定的规则，并将遵守和执行任何新规定。富途控股的股票最近上涨了1.5%，而向上金融的股票下跌了2%左右。在路透社报道之后，两家公司的股价在周五的盘前交易中都有所下跌。

中共国证券监督管理委员会(CSRC)、国家外汇管理局(SAFE)和中央银行没有及时应请求做出评论。中共当局在10月对“跨境”券商表示担忧，加剧了这两家公司的股价的下跌，自今年2月的峰值以来，这两家公司的股价已经暴跌超过80%。

在过去的一年里，中共当局一直在对许多产业进行打击，数据安全已经成为主要问题。10月，中共官媒《人民日报》警告说，如果美国证券交易委员会等机构要求证券商提供数据，大量的客户交易信息将面临风险。

其中三位消息人士说，中共当局也担心资本外流，并担心这些公司迅速增长的业务可能会触犯中共国的外汇管制议程。据其中两位消息人士称，富途控股的高管一直在游说包括中共国证监会、国家外汇管理局和央行在内的金融监管机构，但尚未收到任何积极反馈。

这两位消息人士说，禁令将影响到富途控股等公司的一大块业务。其中一位消息人士说，富途控股大约40%的客户是用中共国身份证开的交易账户。其他大多数账户都是由持有美国、新加坡和香港身份证的人开设的。

在游戏和社交媒体巨头腾讯控股（0700.HK）的支持下，截至9月底，富途控股有260万客户开设了一个或多个交易账户。7-9月，富途控股的交易额从去年同期的1.01万亿港元增至1.4万亿港元（1790亿美元），其中美股和港股交易占比超过90%。

据其中一位消息人士称，个人仍然可以用中共大陆身份证在富途控股上开设新账户，但该公司现强烈要求这些客户开设海外银行账户。

除了富途控股和向上金融等券商提供的服务外，中共大陆投资者只能通过连接香港和中共大陆股票市场，所谓的合格境内机构投资者（QDII）的方案来投资中共国境外的证券市场。这两种方案都受到严格的监管。
