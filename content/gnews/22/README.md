编辑：李易通

- ![](https://assets.gnews.org/wp-content/uploads/2022/04/图片2-16.png)图片来源：雅典娜设计组


据剑桥大学贾奇商学院战略学讲师哈姆扎·穆达西尔在《对话》杂志上的评论文章称，埃隆·马斯克将在Twitter的董事会上宣布社交媒体新时代的到来。在那里“言论自由”的理念将得到真正遵守。

这位观察家说，从该应用的狂热用户到其最大的股东，埃隆·马斯克已经准备好在Twitter掀起大浪，为社交媒体平台用户提供更多控制权。

坎布里奇：世界首富埃隆·马斯克以不到30亿美元的价格收购了Twitter的9.2%的股份，现在他是Twitter的最大股东。这一消息推动推特的股价上涨了27%，并且在盘后交易中继续增长。

马斯克成为Twitter的最大股东，确实解决了最近投资者的另外两个问题。

首先，马斯克一直威胁要创建一个竞争性的社交媒体平台，在那里“言论自由”的理念将得到遵守。这表明，马斯克是带着推动生态系统变革的愿景来的，这一点不可轻视。

其次，马斯克的出现有可能加速推特的创新和竞争力，而该公司过去在这方面做得不够好。

马斯克本人是著名的Twitter用户，他也一直呼吁让社交媒体的算法开放源代码。但这样一个大胆的举动可能会给Facebook和其它公司带来压力，逼其效仿，这样有可能使社交媒体对每个人都更好。
