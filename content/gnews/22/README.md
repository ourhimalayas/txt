## 美国会议员将结束中共对美国农田的购买狂潮

![](https://assets.gnews.org/wp-content/uploads/2021/12/das22-scaled-4.jpg)图片来源：大纪元
中共国多年来一直在美国购买农业用地，一位美国议员表示，为了保障美国食品供应链，这种趋势必须中止结束。
为此，华盛顿州共和党众议员丹·纽豪斯(Dan Newhouse)议员在6月针对众议院2022财年《农业拨款立法，(H.R.4356) 》提出了一项修正案。在最近一次接受《新唐人(NTD)电视台，NTD》采访时，该议员解释了他的修正案将做什么的作用。
“坦率地说，中共国是一个对敌手。我们要确保能够控制我们的食物供应。我认为这是一个自然的、，重要的，国家安全问题。，”纽豪斯表示说。
该修正案于6月30日由众议院拨款委员会(House Appropriations Committee)一致通过。 7月29日，众议院批准了将该农业拨款立法，作为2022年七项支出措施 (H.R.4502) 一揽子计划的一部分。
如果获得通过，该修正案将授权农业部长根据立法内容禁止中共国、伊朗、朝鲜和俄罗斯拥有的公司在美国购买农业用地。在中共国，私营企业和国有企业之间没有区别，因为中国共产党可以通过中共国法律或通过党内基层组织对私营企业实施控制。
纽豪斯说，他的修正案中的条款目前的语言已经被修改。他最初只提到了中共国 ( pdf )，而没有提到其他三个国家。
“在法案规则制定过程中，民主党对其进行了一些修改，将其他几个国家也包括在内。，”他说。“但事实仍然是，共产主义中共国才是威胁。他们是那些并非我们朋友的邪恶国家清单上正在购买美国大部分资产的国家邪恶国家，不是我们的朋友。而这才是重点所在。”
过去十年中，中共国公司一直在购买美国农业用地。根据美国农业部( USDA ) 的数据，截至2020年初，中共的国投资者在美国已控制了191,652英亩土地，价值约 186万美元，而在截至2010年底，中共国投资者控制了这一数字为13,720英亩，价值 81,425 美元。

**收购公司**
其中一项交易涉及中共国肉类加工企业《“万洲国际”，（(WH Group)）》，该集团于2013年以47亿美元收购了总部位于弗吉尼亚州的“《史密斯菲尔德食品公司”》(Smithfield Foods)食品公司。通过此次收购，这家中共国公司现在拥有美国最大的猪肉生产商，以及14.6,000万英亩的优质农田。
另一笔项交易涉及两名中共国企业家，他们于2011年在犹他州购买了一个面积为2.2万,000英亩的牧场，种植苜蓿，并将其出口到中共国。
中共国的农业投资并不局限于美国。根据美国农业部2018年的一份报告 ( pdf )，中共国对农业、林业和渔业方面的直接海外投资，从2009年的3亿美元跃升至2016年的33亿美元。报告发现，这些海外投资与共产党政权的政策密切相关，包括“一带一路” 倡议”(（BRI)）。
报告写道：“中共国官员制定了雄心勃勃的农业投资战略计划，以重塑农业贸易格局模式，并提高中共国在全球市场的影响力。”
中共国的海外投资还包括收购购买和投资外国农业企业。据报道称，“《万洲国际”》2017年收购了总部位于加利福尼亚的猪肉加工商Clougherty Packing和波兰的一家肉类和家禽加工公司。
该报告称，另一家中共国公司“《光明食品集团”》(Brights Food)在2010年至2016年间投资了七家外国公司。这些公司包括新西兰的一家乳制品公司、澳大利亚的一家酸奶公司、法国的一家葡萄酒公司、英国的一家谷物公司和意大利的一家橄榄油公司。
纽豪斯说，他对他的在修正案中采取了 “积极主动 “的行动，以应对中共投资带来的挑战，避免在“问题变得过于严重，大到以至于我们无法再对之纠正之前”，以应对中共国投资带来的挑战。
“我们看到了这种趋势。，”他说。“我们看到了被中国共产党政府收购购买的土地和公司的数量。我们现在就应该阻止它了。”
