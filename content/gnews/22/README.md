## 香港疫情失控 医疗系统即将全面崩溃
` NFSC News` [轉載自GNews](https://gnews.org/zh-hans/2024055/)

作者：纽约香草山教育部- Noemi
![This image has an empty alt attribute; its file name is Screen-Shot-2021-08-30-at-7.37.59-PM.png](https://assets.gnews.org/wp-content/uploads/2021/08/Screen-Shot-2021-08-30-at-7.37.59-PM.png)
当地时间2月16日，据相关媒体消息显示，香港当天通报了超过四千两百起中共病毒新增确诊病例，以及9起新增病毒死亡病例，死亡患者中还包括一名年仅三岁的女童。当地目前等候入院的患者已经高达一万两千人。
![](https://assets.gnews.org/wp-content/uploads/2022/02/截屏2022-02-18-上午2.26.17.png)
有报道提及，香港地区当下的疫情传播情况十分复杂，当地医疗系统中已有120多名医护人员确诊感染中共病毒，另有20名医院职员被列为病毒的密切接触者。一位香港医院管理局的经理人在接受采访时指出，这些医护人员的感染途径均为社区或家居感染，加之各大医院患者人数骤增，香港公立医院眼下已经不胜负荷。

消息人士透露，当地一家医院因急诊室人数爆棚，而不得不在急诊室旁的露天空地上搭起多排帐篷，医护人员只能在露天环境中照料等候就诊或入院的病人。

对此，香港医院管理局前行政人员表示，在未来一星期内，香港累积的中共病毒感染患者可能超过10万例，或将导致当地医疗系统面临全面崩溃的后果。
