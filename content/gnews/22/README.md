出生于乌克兰的寡头米哈伊尔-沃特福德周一被发现在英国最大的庄园之一被绞死。

据英国《卫报》报道，现年 66 岁的石油和天然气大亨沃特福德被发现在家中死亡,情况不明，但并不可疑。另据《太阳报》报道，沃特福德的尸体周一在他位于萨里郡,弗吉尼亚沃特的温特沃斯庄园被一名园丁发现后，随后当局被叫去他家。

虽然《太阳报》报道称沃特福德是俄罗斯人，但据《卫报》报道，他在苏联解体前于1955年出生于乌克兰。该媒体指出，他出生于米哈伊尔-托尔斯泰亚，在21世纪初移居英国时，他将自己的姓氏英文化。 据报道，他的三个孩子和他的爱沙尼亚籍妻子简幸存下来

据《太阳报》报道，这位寡头的母亲住在他位于萨里的别墅式住宅的邻近公寓里. 据称她在离开乌克兰后坚持保留她的苏联身份证。沃特福德在2015年接受《星期日泰晤士报》采访时说，他的母亲“坚信共产党人将回到俄罗斯和乌克兰……我们祈祷她的梦想永远不会实现”。
