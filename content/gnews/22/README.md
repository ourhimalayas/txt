美国可能不会在对俄战争期间向乌克兰派兵，但这并不意味着美国人不能加入乌克兰的战斗。美国官员表示，到目前为止，乌克兰军方在阻止俄罗斯军队方面做得非常出色，但这可能是一场漫长的战争，乌克兰正在寻求外国盟友的帮助。
“乌克兰国际领土防御军团”由乌克兰总统泽连斯基创建，旨在帮助加强对抗俄罗斯入侵的斗争。
周日，一名美国高级国防官员告诉美联社，俄罗斯人已经放慢了脚步并感到沮丧。
这位官员告诉美联社说：“他们受到了阻碍，遭到了乌克兰人的抵制，在某种程度上，他们在燃料、后勤和维护问题方面都已经感到了压力。” “但正如我之前所说，我们需要非常清醒地认识到这是战争，战争是丑陋的，很冷酷，很血腥，而且是不完全可以预测的。”
许多退伍军人联系了军事时报新闻社，询问他们如何加入战斗。在退伍军人和普通美国人可以加入乌克兰的战斗之前，需要做以下六件事。

1. 联系乌克兰驻华盛顿大使馆，让他们知道您打算加入乌克兰国际领土防御军团。如果您不能亲自访问大使馆，则可以致电或发送电子邮件，这是[乌克兰的国际大使馆列表](https://www.embassypages.com/ukraine)。
2. 需要的重要文件。这些包括身份证、国际旅行护照、任何兵役记录、与执法机构的合作或战斗历史。
3. 需要带文件去面试。这些对于所有面对面的面试都是必要的。
4. 写一份入伍申请。这是参加乌克兰对俄罗斯的运动，联系乌克兰驻华盛顿大使馆必須做的。
5. 获取有关如何前往乌克兰以及携带物品的说明。推荐的物品包括：军装、头盔、防弹衣、背心、腰带，以及战斗手套、战术眼镜等装备。
6. 组织程序。乌克兰官员和领事将帮助引导您进入乌克兰。
大使馆将提供他们的联系方式，无论是在华盛顿还是您申请的任何国家。这是[乌克兰大使馆和领事馆的完整列表](https://www.embassypages.com/ukraine)。
