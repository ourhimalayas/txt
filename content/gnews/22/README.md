据《新闻极限》3月19日引述路透社的报道，周六，乌克兰总统泽连斯基(Volodymyr Zelenskyy)呼吁与莫斯科进行全面和谈，以阻止莫斯科对其祖国的入侵，并警告说，否则俄罗斯将需要“几代人”才能从战争的损失中恢复过来。

报道说，自2月24日发起进攻以来，面临顽强抵抗，俄罗斯军队在推进过程中基本停滞不前，甚至在进攻首都基辅时也是如此。为了恢复攻势，他们围攻城市，将城市地区炸成废墟。

周六早些时候，基辅（Kyiv）、切尔尼戈夫（Chernihiv）和日托米尔（Zhytomyr）地区响起了空袭警报，但没有关于新袭击的即时报告。

文章认为，对阻止俄罗斯总统普京发动的军事入侵，孤立俄罗斯经济的前所未有的西方制裁几乎没有作用。

在俄罗斯说它正在“收紧”关键港口马里乌波尔的后，乌克兰国防部周五承认“暂时”失去了亚速海（Azov Sea）的通道，该海与黑海相连，对乌克兰来说是一个重大损失。

但西方官员表示，俄罗斯损失惨重，有迹象表明其军队士气低落，泽连斯基在视频信息中也暗示这一趋势，他敦促重新谈判以结束战争。

周六早些时候，他说：“我希望每个人现在都能听到我的话，特别是在莫斯科。会谈的时候到了，是时候谈谈了。现在是恢复乌克兰的领土完整和正义的时候了，否则，俄罗斯的损失将是你需要几代人才能恢复的”。

根据报道，3月2日，俄罗斯承认其近500名士兵死亡，但此后就再没有给出最新情况。乌克兰方面称现在俄罗斯死亡的士兵人数为几千人，而《路透社》无法独立核实相关死亡人数。

根据乌克兰和俄罗斯的相关报告，在本周的谈判中有了一些进展，以达成一项政治方案，保证乌克兰在北约之外的安全保护。

但乌克兰表示，立即停火和撤出俄罗斯军队的必要性仍然存在，并且双方都相互指责对方拖延了谈判。

俄罗斯方面，普京发誓要继续入侵乌克兰，直到成功。他周五在莫斯科的一个足球场向数万名挥舞俄罗斯国旗的人承诺，俄罗斯“绝对会完成我们所有的计划”。
