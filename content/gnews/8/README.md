#### 中俄关系

习近平死乞白赖花巨资买来的与俄罗斯的关系，反而处处让普京看不起。虽然习近平一上台就和俄罗斯先后签订了2700亿美元石油大单，4000亿美元天然气合同，拉拢俄罗斯迫切之心昭然若揭。并且自习2012年以來，中共军队以上合组织名义与俄罗斯举行了5次陆上军事演习，并且和俄罗斯单独举行了6次海上联合军事演习，6次特殊代号军事演习，包括今年8月9日至13日中俄一次最大的军事演习，习时代中共军队与俄罗斯联合军事演习超过胡锦涛时代5倍，看上去就是一副结成军事盟友的样子，好像在说，咱和俄罗斯在一块玩，你们谁敢打我？

其实，这只是习近平的一厢情愿。虽说俄罗斯确实是中共力抗外部压力的最佳伙伴，而且都受到美国的打压，而且习普两人似乎也有“强人统治”的惺惺相惜，但要普京真心力挺中共，只是习近平不切实际的幻想。

中共近年来透过“一带一路倡议”在欧亚地区圈地插旗立威，虽然引来非议，但确实也扩大了区域政经的影响力。不过，这也正式踩进了俄罗斯的后花园，难道作为前苏联重要继承者的俄罗斯会看不出来？面对中国崛起及当前国际形势，普京在美中俄三边关系一直保持着理性的选择，就是与中美都保持着不对等距离的交往策略，由此来看，中俄关系恐怕是“短多长空”罢了。

2017年4月5日，普京签署了俄联邦新宪法。这个新宪法明确将“对上帝的信仰”确定为俄的核心价值。其中规定：“俄罗斯联邦，团结于其千年历史，维护其祖先的记忆，祖先给予我们理念、对上帝的信仰，以及俄罗斯国家的持续发展，联邦承认这个由历史形成的个体。”这一规定不仅将今天的俄罗斯与历史的俄罗斯联系起来，而且将今天的俄罗斯与对“上帝的信仰”联系起来，这与信奉无神论的中共是有本质区别并且完全对立的。

2007年10月30日，俄政治迫害遇难者纪念日，普京前往莫斯科南郊的布托佛，悼念死于斯大林大清洗的遇难者。普京说：“我们很清楚，1937年是大清洗运动最为猖獗的一年。不过在那之前，残酷的迫害已经持续数年之久。”这样的悲剧“发生于那些表面上很吸引人，但内容空洞的理念凌驾于人的生命、自由和权利等人类最基本的价值观之上之际”，“数以十万、乃至百万计的人们被杀害，被送往监狱，被枪杀和折磨”，“这些人拥有自己的想法，并且不惧怕将其说出。他们是国家的精华。
普京这次演讲抨击的暴政，正是中共当政72年来一直在做的事。

俄总统普京与中共领导人习近平的关系似乎不错，但这不是建立在共同的价值观基础上的，而是建立在互相利用的利益基础上的。对普京来说，他不过是在利用中共为俄国家利益最大化服务而已，根本没把中共当成“自家人”。

习近平一上台就和俄罗斯签订了总价近7000亿美元的两个大单，解决了俄罗斯被美国制裁的燃眉之急，而每次与俄罗斯的联合军事演习，中共都承担全部费用，等于中共出军费帮俄罗斯练兵，有习近平这么好的冤大头，普京何乐而不为？但要俄罗斯为中共卖命，那却是不可能的。

据俄罗斯共产党中央国际部发布的通报，从2021年9月下旬开始，普京当局对俄罗斯共产党展开了持续抓捕行动，抓捕了俄共在莫斯科市、莫斯科州等地的数十个头目及支持者。俄共中央委员会大楼、俄共莫斯科市委大楼，被警方查封三天。俄共中央第一副主席梅列尼科夫的办公室也被查封。这一事件非同寻常。原因何在？有一种说法是，普京得知中共长期暗中支持俄共推翻他，近期更有加剧态势，因此痛下决心，启动了大抓捕。
