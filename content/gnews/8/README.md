3.**主流媒体曝 科学研究证明疫苗无用 **

[综合外媒12月18日消息，华盛顿大学和瑞士制药商Humabs Biomed SA研究人员针对市面上6种疫苗进行测试发现，市售所有疫苗对奥米炕几乎无效。所测试的六种疫苗包括强生、莫德纳、辉瑞、阿斯利康、中共国药、俄国Sputnik V，在对抗奥米炕基本无效。 在测试过程中发现，12位施打强生疫苗者，只有1人产生出抗体；接种中共国药集团疫苗的13人中，只有3人产生了对奥米炕变异株的抗体；接种俄罗斯Sputnik V疫苗的人，无一人产生抗体。 另外，在莫德纳、阿斯利康方面的数据显示，虽然他们有产生出抗体，但保护力仍下降33倍至44倍。研究数据发现，施打2剂辉瑞在预防奥米炕感染方面的效果微乎其微。](https://www.reuters.com/business/healthcare-pharmaceuticals/jj-sinopharm-sputnik-v-shots-weaker-against-omicron-study-shows-2021-12-17/)

4. **美国会通过国防授权法案 邀台参与环太军演 **

[据《中时》12月16日消息，美国参议院本月16日通过2022年国防授权法案，法案将送交美国总统拜登签署后生效。法案同时建议美国防部邀台参与2022年环太平洋军演，并呼吁制定计划协助台湾强化不对称防卫能力。 2022年度国防授权法案总额高达7680亿美元，其中包括为了威吓中共所提出的太平洋”吓阻倡议”的71亿美元，以及多项提升台湾自我防卫能力的具体措施。 国会指出，美国政策是维持美军阻止中共国武力犯台、造成既成事实的能力，并要求美国国防部长递交台湾防卫能力评估报告，协助台湾发展不对称防卫能力，并加强美国国民兵与台湾合作。](https://www.chinatimes.com/realtimenews/20211216000641-260408?chdtv)
