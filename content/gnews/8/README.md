【摘要】中共已经闯到我们家门口了！北京与所罗门群岛签署安全协议着眼于在南太平洋的第一个军事立足点——这将使澳大利亚处于危险之中：中共与南太平洋国家所罗门群岛将签署了范围广泛的协议；将允许中国武警和军事人员驻扎在那里；陆军警告说，如果中国船只在所罗门群岛作业，情况将会改变，澳大利亚的军事行动可能会受到新安全协议的严重影响。（图片说明）中国在该地区日益增长的存在使他们试图将自己立足于澳大利亚的家门口

澳大利亚联邦警察专员（AFP Commissioner）里斯·克肖（Reece Kershaw）知道中国将提出某种形式的培训建议，但他本人并没有看到拟议的安全协议谅解备忘录。

他希望澳大利亚仍将是所罗门群岛警方的首选合作伙伴。

“我们已经在那里呆了很长时间……（并且）它有点个人化，我们在那儿确实失去了我们的一名官员。”他在参议院估计听证会上说。

“所以我们不会让其他国家进来尝试改变整个框架，我们还在有效工作。

“那里的社区犯罪有所下降，警察比以往任何时候都更加专业，我们希望保持作为该地区首选合作伙伴的地位。

密克罗尼西亚总统大卫·帕努埃洛（David Panuelo）对协议表达了“严重的安全担忧”，并恳求索加瓦雷先生重新考虑。

“我担心我们——太平洋岛屿——将成为这些大国之间未来对抗的中心，”他周三写道。

他还担心中国的军事存在会引发冲突，太平洋岛屿将成为附带损害。

帕努埃洛总统还担心，这种存在可能不仅对整个太平洋地区而且对世界产生长期影响。

‘这不是不可能的恐惧，它以前发生过。密克罗尼西亚联邦和所罗门群岛都是二战期间的战场。”他写道。

“正如我们从俄罗斯对乌克兰的入侵中看到的那样，如果大国认为这符合他们的利益，就会选择暴力，而不必考虑我们的利益，比如不带来附带损害。”

此前，中国对澳大利亚不尊重的“对南太平洋岛国的殖民主义心态”进行了粗暴的抨击。

北京对一篇独立文章呼吁堪培拉介入所罗门群岛以取代令人担忧的与中国签订新安全协议感到愤怒。

在大卫·卢埃林-史密斯（David Llewellyn-Smith）呼吁斯科特·莫里森（Scott Morrison）应该在所罗门群岛上与共产主义超级大国“强制直接对抗”并使用“一切必要手段”将中国赶出该地区之后，中国官方媒体声称暴露了澳大利亚“歇斯底里和丑陋的脸”。

独立文章还将这种情况描述为“澳大利亚的古巴导弹危机”，并称“澳大利亚不可能允许这笔交易继续进行”。

澳大利亚财政部长西蒙·伯明翰（Simon Birmingham）否认澳大利亚在其太平洋外交政策上失败，尽管总理和内阁部的一名高级官员透露他不知道总理斯科特·莫里森和他的所罗门同行之间的任何电话交谈。

伯明翰部长说，政府非常有意识地确保我们尊重每个太平洋岛国的主权。

总理莫里森先生也表示，所罗门群岛政府没有就澳大利亚提供的支持提出任何异议。
