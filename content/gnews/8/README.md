## 有声｜俄罗斯表示已经对伊朗核协议作出书面保证

**By:秘翻新闻**

**2022-03-16**
![](https://assets.gnews.org/wp-content/uploads/2022/03/截屏2022-03-16-下午2.14.29.png)
《[路透社](https://www.reuters.com/world/middle-east/russia-says-it-has-written-guarantees-iran-nuclear-deal-2022-03-15/)》华盛顿3月15日报道，周二，俄罗斯外长拉夫罗夫，已经书面保证可以履行作为伊朗核协议缔约方的职责，暗示俄罗斯允许2015年伊朗核协议的重启。

报道指出，根据这项正式被称为《联合全面行动计划》的协议，伊朗同意限制其核项目，使其更难以开发核弹，以换取全球经济制裁的缓解。

此前，在莫斯科与伊朗外长侯赛因·阿米拉多拉希安举行的新闻发布会上，拉夫罗夫还否认俄罗斯是恢复2015年协议的障碍。

美国国务院发言人内德·普莱斯对记者说，恢复核协议不会成为俄罗斯逃避因乌克兰战争而实施制裁的“逃生口”。但他也表示，不会制裁俄罗斯参与核项目，但不能、也不会向俄罗斯提供其他保证。

伊朗表示，美国缺乏解决维也纳核谈判中几个悬而未决问题的“政治意愿”。这个伊斯兰共和国坚持要求华盛顿取消与人权和恐怖主义相关的制裁，包括2019年对其精锐的革命卫队实施的制裁。
