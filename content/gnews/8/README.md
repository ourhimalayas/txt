基辅和莫斯科之间关于可能的和平解决方案的会谈从早上开始。美国总统拜登将于周一与盟友交谈。在国际政府间气候变化会议上，一名俄罗斯外交官“代表所有无法阻止这场冲突的俄罗斯人”道歉。

在预定的俄罗斯和乌克兰之间的和平谈判的前一天，一名俄罗斯外交官出人意料地为俄罗斯的袭击道歉。正如《华盛顿邮报》报道的那样，这发生在周日政府间气候变化专门委员会的一次转换会议上。

据与会者称，俄罗斯代表团团长奥列格·阿尼西莫夫说：“让我代表所有无法阻止这场冲突的俄罗斯人道歉。”

这位气候科学家在 195 个国家的会议上补充说，所有知道正在发生的事情的俄罗斯人都找不到这次袭击的理由。阿尼西莫夫还表示，他钦佩乌克兰代表团尽管在他们的国家发生了战争，但仍参加了气候谈判。

在乌克兰-白俄罗斯边境会面

解决俄罗斯和乌克兰之间冲突的会谈定于周一上午开始。这是由俄罗斯国家机构塔斯社和《乌克兰真理报》报道的。塔斯在提到一位匿名消息人士时写道，原定于周日开始的会议推迟的原因是“乌克兰代表团的后勤工作”。

俄罗斯代表团已经离开明斯克，前往会场。乌克兰总统府此前曾宣布，基辅没有为计划在白俄罗斯戈梅利地区的乌白边境举行的会谈设定任何条件。
