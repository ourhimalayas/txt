**疫情|新冠疫苗副作用     打新冠疫苗后副作用综合征**

先声明一下：对于新冠疫苗，我其实是很支持去打的，但是没想到自己接种后的反应这么大，当然也是因人而异，我身边很多人打完之后是没有症状的，看到最近国内又有新增病例，加上阿三那边特别严重，所以还是建议大家尽快去接种，保护好自己。当然，可能也会有小伙伴会出现和我一样的症状，所以出这一篇也是为了解答小伙伴的疑虑。

5月9日接种第一针，接种的是北京科兴的疫苗，共两针，第二针间隔3一8周，在医院留观半小时后离开，当天没有任何不适，包括针眼位置无疼痛感。

5月10日下午四五点左右出现身体乏力，想睡觉，低烧 37.1度的症状。练完瑜伽八点左右开始出现流鼻涕、打喷嚏等感冒症状。

5月11日嗜睡，流鼻涕，乏力，低烧。

一度以为是自己感冒了，但又觉得这三十几度的大热天也没有吹空调怎么可能，于是在网上搜了下才知道属于接种疫苗后的不良反应，相当于现在是得了新冠之后的症状。

到了晚上感觉症状加重了，体温到37.5℃左右，伴有头痛，期间没有吃药只是不停的喝水。

5月12日早上测体温降下来了36.9度，但还是会有鼻塞和乏的症状，不过整体状态比昨天好很多，下午的时候针眼位置有些痒。

5月13日基本恢复正常，稍微有一些流鼻涕和鼻塞的症状。

我在此期间是没有吃任何药，只是不停的喝水，最严重的是发烧那天，我觉得休息还是蛮重要的，所以如果有类似症状的小伙伴可以请假先休息一天，但是一直没有好转的话一定要去医院检查，千万不能耽搁！！
