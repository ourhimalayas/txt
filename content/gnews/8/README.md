![](https://assets.gnews.org/wp-content/uploads/2021/10/tempsnip74.png)图片来源：华盛顿邮报

如果 Facebook 的算法认为帖子不好，Facebook 可以将其得分减半，从而在用户的供稿中将该帖子的大部分实例推倒。但根据文件，一些帖子的分数可能高达十亿。将天文分数减半以“降级”它仍然会留下足够高的分数以出现在用户提要的顶部。

“**可怕的想法：公民降级不起作用，**”一名 Facebook 员工指出。

实验文化在 Facebook 根深蒂固，工程师们拉动杠杆并衡量结果。2012 年发表，于 2014 年进行的一项实验，试图操纵用户提要中显示的帖子的情绪效价，使其变得更积极或更消极，然后观察他们自己的帖子是否改变以适应这些情绪，这引发了道德问题，《邮报》当时报道。另一个是豪根本月向国会报告的，**涉及关闭一部分用户的安全措施**，作为比较，看看这些措施是否有效。

根据文件，一组之前未报道的实验，涉及让一些人更频繁地加入他们随机选择的一些朋友的提要中——然后，一旦实验结束，检查这对朋友是否继续交流。一位研究人员假设，换句话说，**Facebook 可以使关系变得更加紧密**。

2017 年，Facebook 试图扭转令人担忧的下降趋势，**即在该网站上发帖和相互交谈的人数下降**，**表情符号的反应为其提供了五个新的杠杆。当时，每个情绪反应都值得五个赞。逻辑是，反应表情符号表明，该帖子比喜欢的人产生了更大的情感印象；对表情符号做出反应比单击或轻按“赞”按钮更进一步。**但 Facebook 对公众对这些反应的重视程度不以为然：该公司在 2017 年告诉 Mashable，它只是“比喜欢多一点”。

此举符合文件中强调的一种模式，其中 Facebook 将其试图鼓励用户采用的新功能的权重设置得非常高。通过训练算法来优化这些功能，Facebook 的工程师几乎确保它们会被广泛使用和看到。不仅如此，**任何在 Facebook 上发帖，并希望覆盖广泛受众（包括出版商和政治演员）的人，都会不可避免地发现某些类型的帖子比其他类型的帖子效果更好。**
![](https://assets.gnews.org/wp-content/uploads/2021/10/tempsnip73.png)马克·扎克伯格承认，反应可以用来表示不喜欢。
 
有一次，首席执行官马克·扎克伯格甚至在对用户评论的公开回复中，鼓励用户使用愤怒的反应来表示他们不喜欢某些东西，尽管这会使 Facebook 更频繁地显示类似的内容。

对一个帖子的回复，表明比点击反应按钮更大的努力，权重甚至更高，高达 30 倍。 **Facebook 发现，用户朋友在网站上的互动会创造一种良性循环，促使用户发布更多信息。** 《华尔街日报》上个月报道了 Facebook 更加重视评论、回复评论和回复转发——这是它所谓的“有意义的社交互动”指标的一部分——**进一步刺激了分裂的政治帖子。**（那篇文章还提到了愤怒表情符号的早期权重，但没有对其影响进行后续辩论。）

该指标的目标是“通过优先考虑激发互动，尤其是家人和朋友之间的对话的帖子，改善人们的体验，”莱佛说。

愤怒表情符号的权重第一次降级是在 2018 年，当时 Facebook 将其降为点赞值的四倍，所有情绪的权重保持不变。

但很明显，并非所有的情绪反应都是一样的。根据 2020 年的一份文件，愤怒是六种表情符号反应中使用最少的，每周点击 4.29 亿次，而反应是 630 亿次喜欢和 110 亿次“爱”。根据 2019 年的一份文件，Facebook 的数据科学家发现，对有问题的帖子的愤怒反应“更加频繁”：“公民低素质新闻、公民错误信息、公民毒性、健康错误信息和健康抗病毒内容”。**其当年的研究表明：愤怒的反应是将政治人物“武器化”。**
