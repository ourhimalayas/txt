兄弟姐妹们大家好，现在是7月10号。这看得我有点像装病的样子？抱歉今天早上也没有发盖特，今天是发不了盖特。明天星期天都需要直播一般的情况下，我实在直播不了了，一直嗓子还痛。昨天晚上回到家，今天看了医生是热感冒，病毒性的热感冒。嗓子里面就像有个火棍在这一样，然后头顶，一直两个头顶疼。而且在热伤风的时候又摸了红花油，导致病毒加剧。主要原因就是最近睡觉太少，免疫力低了。

头一段从六月一直到七月，大家都知道在忙活大事。到了新办公室那空调十八度开了两天，每个人都没有想到把空调给降下来，每个人都往身上加衣服，就没有一个人去想到把空调给降下来。我这几天都在想，这不就是中国人的效应吗？共产党把人类降到最低了，但从来没想到干掉共产党的 — 都往身上加衣服。七月一号，二号，三号，四号都在十八度，我们每个人，长岛哥，战友们都往身上加衣服，我也加衣服 — 就没想到把空调给降下来。到了七月五号，六号才想起来空调能调。这个是很大的伤害。这就是人呐，当领导有多重要，如果我们有一个人喊着说：“咱得把空调降下来”。都不至于这个样子。

这几天纽约特别热，冷冷热热的。很多人给我发信息关心我，战友们抱歉，我真的是没有时间一一而回，没办法，一直疼。我刚刚给嗓子倒了药 — 龙角散 — 我才能说话。从昨天倒现在基本没说话。还很多事情要做，给你们挣钱，还惦记着大家 — 太多事儿了。我在这里说一下：明天不直播了，因为7哥热感冒了。

事实上我过去每年都有一次热感冒，过年都有一次重感冒。到了美国以后就没有了，但这一次又来了，又抹了红花油。人家说疯了吧？这个时候你热重风再摸这么重的红花油。所以说都来了。

啥都不说了，兄弟姐妹们，姆该嗮了！大家还是一句话：别忘了健身，保重身体，该双修双修，但保重身体！不行了，我说不下去了，说了嗓子痛的不行了。
