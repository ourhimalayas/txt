## 俄罗斯对乌克兰马里乌波尔剧院的空袭造成 300 人死亡
` 加拿大温哥华英里农场` [轉載自GNews](https://gnews.org/zh-hans/2230645/)

编译：文宝
![](https://assets.gnews.org/wp-content/uploads/2022/03/4-108.png)
**马里乌波尔市政府3月25日援引目击者的话说，上周俄罗斯空袭了被围困的乌克兰城市马里乌波尔的一个用作防空洞的剧院，约有 300 人死亡。当剧院于 3 月 16 日被击中时，外面用俄语张贴了一个巨大的文字“儿童”，旨在从上方的天空中看到。目前尚不清楚紧急救援人员是否已经完成了现场挖掘。 空袭发生后不久，乌克兰议会人权专员 Ludmyla Denisova 表示有 1,300 多人当时在大楼内避难。**

**马里乌波尔市一直是战争中最严重破坏的现场。 自入侵初期以来，东部港口一直处于围困之中。 据信，仍有数万人被困在室内，无法获得食物、药品、电力或供热。“俄罗斯轰炸我们，因为他们无法击败我们”。
马里乌波尔市政府表示，克里姆林宫的主要政党已在被围困城市郊区的一个购物中心开设了一个政治办公室。 根据该市Telegram的消息，统一俄罗斯办事处正在为在附近俄罗斯支持的分离主义地区运作的运营商分发宣传材料和手机卡。**

**自 3 月初围攻开始以来，马里乌波尔的通讯联系几乎被切断。 手机、电视和无线电信号塔一直是俄罗斯空袭和炮击的目标。联合国人权办公室表示，自俄罗斯入侵开始以来，乌克兰已有 1,081 名平民丧生。**
