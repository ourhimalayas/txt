据OAN网站3月7日报道，周一，在与立陶宛外交部长举行的联合新闻发布会上，美国国务卿安东尼·布林肯重申，如果任何北约成员国受到攻击，美国准备为它们提供保护。

![](https://assets.gnews.org/wp-content/uploads/2022/03/AP22066528335372.jpg)2022年3月7日星期一，美国国务卿安东尼·布林肯在拉脱维亚里加的国家艺术博物馆与拉脱维亚外交部长埃德加斯·林克维奇举行的新闻发布会上讲话。美国国务卿安东尼·布林肯正在对波罗的海三国进行闪电式访问，这三个国家在目睹俄罗斯加紧入侵乌克兰时日益感到紧张。(Olivier Douliery/Pool via AP)

布林肯补充说，没有人应该怀疑美国的决心，并指出美国正在与北约盟国就防御态势进行接触，包括延长在波罗的海国家的军队部署。

此前，美国在最近几周向波兰和其他国家增派了数千军队。美国众议院准备批准向乌克兰提供100亿美元的援助。周二，美国总统拜登签署行政命令，宣布美国立即禁止从俄罗斯进口石油和其他能源，以对其进行惩罚。这项行政命令还禁止美国在俄罗斯能源领域开展新的投资，并禁止美国人为那些投资俄罗斯能源业的外国公司提供资金。
