**4、镍价大涨迫使LME暂停交易，也将清算所暴露在镁光灯下**
![img](https://media.gettr.com/group22/origin/2022/03/09/10/faebcb16-822a-30ca-4bbf-d22628f6bf9f/4db6c09f25ac1f3138400099bc3f4337_500x0.png)
路透伦敦3月8日 – 随着镍价翻番至每吨100,000美元上方，伦敦金属交易所(LME)周二采取紧急措施暂停镍交易。镍价之所以飙升，是因为在主要镍生产国俄罗斯遭到西方制裁之际，一家全球主要生产商进行空头回补。交易员称，一些镍持仓者也难以向其清算所支付保证金。清算所是金融市场的一个关键组成部分，站在买方和卖方之间，如果有一方违约，清算所将承担风险。全球最大的清算所包括伦敦证券交易所集团(LSE)旗下的伦敦清算所(LCH)、ICE Clear和CME Clear。这些清算所每年处理数以万亿美元计的股票、债券、衍生品和金属交易。
