俄罗斯国家电视台“俄罗斯24”播出的视频，乌克兰士兵袭击村庄，记者称他们为“民族主义者”和“新纳粹分子”。关于战争的消息都来自乌克兰东部的分离主义地区顿涅茨克和卢甘斯克自称的“人民共和国”。

俄罗斯于2月21日批准该地区以俄语为主成为独立的国家，俄方声称，它只能出面“保护”两国免受乌克兰迫在眉睫的袭击。用俄方新闻的话说，“实现去纳粹化的唯一途径就是军事行动”。

在俄罗斯媒体中，全世界人民目睹的乌克兰其他地区的战斗在很大程度上被忽视了。马里乌波尔被俄罗斯轰炸摧毁。哈尔科夫、切尔尼戈夫、赫尔松和日托米尔等城镇被俄罗斯空袭摧毁，留下烧焦的住宅和建筑碎片。俄罗斯电视台很少报道这样的景象。就算是报道出来，也是乌军干的。

普京在一次演讲中说，西方唯一的目标是“毁灭俄罗斯”，**普京认为，乌克兰并不是一个真正的国家，自古以来就是俄罗斯的一部分。**

正如乔治·奥威尔小说中的世界，乌克兰战争只能称为“特殊军事行动”。**3月4日俄罗斯通过法律，规定将其描述为“战争”、“攻击”或“入侵”是非法的。违反一个人最高可被判处15年监禁。**在俄罗斯大众媒体中没有发现任何反对战争的消息。俄罗斯在战争开始后的几周内爆发了反战抗议活动，并拘留或逮捕了15,000多人，但从未在国家电视台播出过。

当战争开始俄罗斯通过新的法律，仅有的两家独立媒体“TV Rain”和“Echo Moscow Radio”被关闭。**所有广播公司都直接或间接受到政府控制。除一些年轻人外，大多数俄罗斯人都从电视上获取新闻和信息。互联网上的来源，如 Facebook、Twitter、Instagram和其他外国社交媒体平台均被屏蔽。**
