路透社11月29日电，随着习近平主席领导下的中共国加强对台湾的军事恐吓，一系列外国潜艇技术供应商在其政府的批准下，正在协助一项在台湾建造潜艇的秘密计划。路透社的一项调查发现，台北已秘密从至少七个国家采购技术、部件和人才，以帮助其建立一支水下舰队，该舰队有可能对中共国的任何袭击造成沉重打击。

CNN 11月29日电，一位高级国防官员周一表示，五角大楼将专注于在关岛和澳大利亚建立基地，以更好地作好美国军队对抗中共国的准备。这位高级国防官员说，印度太平洋地区对国防部来说是一个主要的焦点，因为Austin部长强调“中共国是该地区的进逼挑战”。
