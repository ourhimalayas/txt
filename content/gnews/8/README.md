作为一个正常人，很难站在畜生的角度来看事情。一般我们看人，都是站在好的角度来看，但现在面对最邪恶的CCP，它们却比畜生还要邪恶万倍，即便把它们想像得再坏，都远不及他们实际坏的万亿分之一。只有亲身经历过CCP的坏，才能切身感受到它们的邪恶，这种邪恶与我们听说的和想像的是完全不同的。

我亲身经历的这件小事情，至今都让我感受到无比的愤恨与可怕。谁也不知道，这件事的背后是否会隐含着极为可怕的阴谋，但愿是我杞人忧天，但愿是我小题大作，也但愿是我以小人之心度“君子”之腹，如果没有亲身经历，我想我会这么认为的，但亲身经历之后，才知道到底有多可怕，绝不是大多数人所认为的稀松和平常。

中共制毒放毒，毒疫苗打了一轮又一轮，万幸遇先生，坚持不打苗，但现在又被逼着核酸检测，否则寸步难行。之前虽然也知道核酸检测绝非单纯的采样，一定有“醉翁之意”，迫不得已决定去做检测，去之前做了一定的准备，还特意驱车几十公里到一个偏僻医院，感觉会排队少，检测可能也没那么严格。到了之后只有四五个人，轮到我，扫码付款，录信息，医生拿一根看起来像棉棒的东西往我喉咙捅，慌忙后退，万幸没有捅到喉咙，只是在舌头表面和上颚划了两下。但就恰恰这两下，让我心情一下子冰凉至极，捶胸顿足，极为后悔。早知如此，打死都不会做这种狗屁玩意。因为仅这两下，舌头和上颚瞬间感觉到毛焦火辣，比单纯的开水烫伤还要恐怖，整个头都是痛的。单纯采样，绝不可能会有这种异样的感觉，两个小时过后，我的口腔还是麻的，头还是一样的痛，即使拿铁签刮过舌头上颚，都不可能出现这种情况，因此棉棒上一定是加了东西，通过粘膜迅速到达血液，才有可能会如此严重。至于具体加了什么，只有魔鬼才清楚，普通人是绝对想像不到的，实际只会更坏，不会更好。

三四个小时过去，我的头依然胀痛，没有遇到先生之前只知道痛，却到死都不知道为什么痛！谁又能想像CCP竟是如此畜生，普通人根本无法想象这帮畜生到底有多坏！十多个小时过去，头还是一直疼着，整个脑子木木的，像空了一样。写下我的可怕经历，希望引起大家的重视，这是我第一次做核酸检测，只是被棉棒划到舌头和上颚，但却引起持续和剧烈的头痛，尤其是额头两旁和后脖子上面的后脑，连头顶都感觉到一跳一跳的，整个口腔和面部都是麻木的，舌头上颚被接触到的部位像被开水烫伤了一样，尤其是后脑，我感觉像变了一个人，整个头木木的，精神状态和之前完全不一样了。本人从小几乎不打针不吃药，对有害物质很敏感，这次算是阴沟里翻船。

这次核酸检测后我电话问过一两个人，也说做过核酸之后喉咙一直不舒服。我有理由相信，核酸检测绝不是为了单纯采样，而一定是添加了东西在里面，而且绝不是什么好东西，否则不可能以防疫之名来干这种事。它们在决定干坏事的那一刻起，就一定会想着务必把坏事做得完美，既然干了就一定干彻底，以免引火烧身，偷鸡不成蚀把米。如果证明了一件事情是有问题的，那相关的其它事情也一定有问题，甚至更有问题，隐藏得越深，问题就越大，只是没有亲身经历过不会想到如此可怕。我不知道有多少人有类似经历，但我相信实际情况绝对比我感觉到的更可怕，也更残酷。

中共割韭菜刀刀见血，自然也不会放过核酸检测，每一次核酸检测几十元到几百不等，有多少人次做？这可是无数亿的黑心钱啊！更为可耻的是它们到底往人体粘膜里下了什么毒？只有亲身经历过，才知道CCP这帮畜生做的每件事都是先要钱再要命——就像核酸检测一样——CCP从没偏离过这两大主题。而可悲的是，全世界都被CCP玩弄于股掌之中，若没有先生的爆料革命，我们是永远不可能知道它到底有多邪恶，到底有多可怕！
