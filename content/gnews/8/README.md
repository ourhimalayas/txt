## 疫苗强制令生效，昆士兰警察和医护出现抵制

![](https://assets.gnews.org/wp-content/uploads/2021/12/d4280a0d18337897.png)图片来源：AP Images
翻译：月映万川
编辑：守望黎明

新数据显示，昆士兰州将近2%的卫生系统员工和大约1%的警务人员拒绝了疫苗强制令。

昆士兰州疫苗强制令的生效日期对医疗工作者和警察分别是12月15日和12月17日。

昆士兰警察系统共有17,200名员工，截止12月19日，有100名警察和71名后勤人员因疫苗的相关规定而被停职。

在12月21日，因在冲浪者天堂警察局出现的一例阳性病例，52名警察被隔离。另外，宾利警署的49名警察也在隔离中，因为他们有同事检出新冠阳性。

昆士兰警务系统发言人表示：“12月14日，警察局长卡塔琳娜·卡罗尔（Katarina Carroll）发布了一项新指示，要求所有雇员在2021年12月17日之前接种第二剂新冠疫苗，并根据澳大利亚疫苗免疫委员会的建议，在此后的一个月内接受加强针。”

这位发言人表示：“昆士兰警察接种疫苗的最大意义，就是要让民众与他们接触时有信任和安全感。拒绝听从警察局长指令的雇员将被停职。”

而昆士兰医疗系统的114,874名员工中，有2131人选择不遵从疫苗强制令。此外，另有3246名医疗系统的员工目前处于假期状态而未执行疫苗强制令。如果，把这一人数也考虑进去，大约有5%的医疗系统员工目前没有接种。

昆士兰卫生部的一位女发言人说，任何在提供护理的设施内工作或出入的人员，都必须完全接种疫苗。她说：”没有接种疫苗和没有申请豁免的工作人员将不能在提供护理的岗位工作。”

昆士兰医疗系统的工作人员已经提出了2640份豁免申请，其中385份被批准，831份被拒绝，1424份待批。

在布里斯班最高法院有几例针对疫苗强制令的诉讼正在进行中，涉及警务人员，卫生工作者和昆士兰提供救护车服务的雇员。

第一个案子涉及7名警务人员以及13名卫生和救护人员。此案已被列入明年2月14日开始的为期三天的民事审判。

第二个案子由单独一名护士和24名警察工作人员组成。警察案将于2月23日重新开庭庭，护士案将于2月11日开庭。两的案子都将进行指导性听证。

最高法院已经发出命令，禁止昆士兰警察局、昆士兰卫生局和昆士兰救护车服务局根据疫苗接种状况对任何申请人采取行动，以使他们挑战疫苗接种令的法律诉讼能够顺利进行。

昆士兰卫生部拒绝就此事发表评论，但警方发言人表示，已经为那些参与法律行动的警员和工作人员做出了置换安排。”包括换岗、易地安排、放假或要求他们戴口罩上岗。”
