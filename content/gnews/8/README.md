## 航空事故背后的人性光辉
` 纽约香草山MOS03` [轉載自GNews](https://gnews.org/zh-hans/2235050/)

作者：香草山健身部-新世界的一员
![This image has an empty alt attribute; its file name is Screen-Shot-2021-08-30-at-7.37.59-PM.png](https://assets.gnews.org/wp-content/uploads/2021/08/Screen-Shot-2021-08-30-at-7.37.59-PM.png)
东航事故牵动了世界人民的心，联合国秘书长、以及多国政要就东航空难表示哀悼。向遇难者家属以及中国政府和人民表示哀悼，只有为数不多的，如俄粉们的俄爹那样的邪恶独裁无动于衷。
全世界哀悼132个生命，可是在国内你感觉不出真心惋惜自己的同胞，只有无端的恨美国倒是发自内心的，更是借着毁机事故打广告，让我深切哀悼132个无辜的生命的同时也哀悼着那些人堕落的人性。
中共又把这个事故扣在了美国头上，我对它们的表现一点都不吃惊，因为中共总是不放过任何一个事故愚弄民意，恶毒的拉民族仇恨。太多的人深信不疑的说是美国搞的鬼，我问她们美国攻击民航的理由是什么？美国真要这样打，中共领导人还能活着？
只可惜网络上各种带风向，加上消息相对闭塞，使得太多的国家对我们做了好事却背着黑锅，特别是自由灯塔的美国更是中共营造给中国人民的假想敌，而俄罗斯对我们坏事做尽却宣传出那么多顶礼膜拜的俄粉脑残。
今天就讲一个同样是东航的一起事故，那是1993年4月6日，一架麦道MD-11型客机（编号：B-2171）从上海飞往洛杉矶。
当飞机飞行到美国阿拉斯加州阿留申群岛南约950海里的太平洋上空时，机组人员因不小心碰到了襟翼/缝翼控制手柄而导致飞机的前缘缝翼放下，在外侧缝翼放下之后，飞机因俯仰角过大而触发了失速警报。短短几秒钟内飞机急坠五千英尺，迅速下降到了两万八千英尺左右，飞机的俯仰角逐渐趋于稳定。但是在急坠过程中机上的乘客已经出现了伤亡，且飞机遭受到了破坏，机组人员希望能够紧急迫降对乘客进行救援。当时最好的选择就是离事故最近的俄罗斯，可俄罗斯拒绝迫降，刚有一点生的希望的乘客们陷入了恐慌，就这样在飞机损坏情况不明了又身处大洋上空还被拒绝迫降的情况下，乘客们忍受着生死煎熬。在这读秒如年的气氛下经受长时间的地狱般煎熬，广播中传来了机长的通知，“飞机准备迫降在阿拉斯加阿留申群岛的薛米亚美国空军基地（大家自发的爆发出掌声）。他们完全是出于人道考量，才接受了我们的迫降请求！”
就在大家重燃活下去的希望的时候，机长又说“但是这个岛太小，机场不具备降落大型民用客机的条件，跑道不够长，没有足够的照明设施。加上眼下气候恶劣，有大风暴，能见度很低。我们飞机自身的受损情况又不明，起落架不知道能不能打开。所以能否安全降落仍是未知数。请大家做好自救准备！”就这样，在生死转换中乘客们都接近崩溃，黑暗中，大家开始写起了遗书。飞机在漫天的黑暗中飞行，乘客绝望中静静等待死神降临，这种煎熬着实叫人生不如死（写到这里我倒是很希望俄粉们体验一下）！
阿留申群岛是当年二次大战时美日双方争夺的军事要地，薛米亚美国空军基地位于该群岛最北端。“此时此刻的我，对美军存着戒心，还有不少敌意。美国的军事基地让我们中国的飞机降落？也不怕泄露机密？真有些不可思议。”（机上乘客回忆当时自己的心情）
飞机迫降的薛米亚岛是一个长4.39公里，宽6.95公里的小岛。由于机场的跑道比较短，也没有能供大型客机降落的照明设备，美军身着防火服和防火面具，冒着生命危险站在跑道两侧，用手电筒将跑道照得通明雪亮，跑道旁边停满了救护车和消防车。迫降成功后美军救护人员冲进了机舱，他们首先把有骨折的重伤员抬了下去。 随后美军军医（蓝色衣服小姐姐）登机救治受伤人员。原来为应对最糟糕的结果，美军在接受迫降请求后，提前派出军用直升机前往2000公里以外的安克雷奇，把从睡梦中被急救电话唤醒的医生们接来支援重伤员被救护车送走后，其他伤员或未伤乘客被带上客车，送去基地大礼堂。礼堂里摆放着床铺凳子，以及准备好的各种食品饮料。许多美军士兵把自己的食品、用品、床垫、被褥等等也奉献了出来。按伤势程度的不同，伤员的病床旁绑着不同颜色的姓名标签。军医在急救处理着伤员，给流血伤员包扎止血。年轻士兵们则在照顾伤员，他们有的眼睛发红，有的挂着泪水。
因怕亲人担心，惊魂未定的乘客比划着问是否有电话可用，美军说这里没有公共电话，但接着安慰道：“不着急，你等一下！”请示过后，他跑回来，把乘客带去了办公室，让他放心地打免费国际长途，同时用广播告诉所有伤员乘客，可以过来排队打电话。
最终轻伤员和未受伤旅客被派来的阿拉斯加客机接走。重伤员分批乘坐C130运输机送往安克雷奇的数家医院，一位严重受伤的空姐呕吐不止，吐得满床满地。一位年轻士兵跑了过去，跪在她面前，看不出一丝一毫犹豫清理呕吐物并给空姐擦拭弄脏了的衣服和床铺，不顾呕吐物的气味，一位亲历者自问：“这就是我以往印象中的美国鬼子吗？

面对机毁人亡的人道主义求助，俄罗斯选择了拒绝，美国选择了伸出援助之手，而且还是一个空军基地，孰是孰非国人自然会懂得自行判断，可万恶的中共刻力隐瞒这样的真相，以至于这样的事实几乎无人知道，我幸运的从公众号里获得这样的真相，对伟大的美国只有感动和感恩，正因自由民主法制下才会出现如此美丽的人性光辉，我们的七哥才毅然决然的带领中国人歼灭邪恶，走西学东用，为中国的伟大打下了万年的文明基石！
