## 中共发动的经济世界大战
` 洛杉矶盘古农场 HIMALAYA LA PANGU` [轉載自GNews](https://gnews.org/zh-hans/1753634/)

【简评】***中共魔鬼政权加入世贸组织，从落后国家跃升为世界强国。中共鼓吹的“一带一路”项目更是用尽各种非法手段（知识产权盗窃、培养科技商业领域间谍、大量产业补贴和非法货币贬值和债务陷进外交等），重创世界经济格局，引起经济世界大战。经过四年爆料革命坚持不懈的传播中共恶魔的真相，美国及其盟国开始觉醒，要将中共踢出WTO。 “多行不义必自毙”，中共因加入世贸而演变成强国，也会因被踢出WTO而导致下行的经济崩溃。作为中共邪魔的受害者，世界各国也只有联合起来，集体围猎把亲手培植的中共恶狼猎杀，捍卫正常国际经济秩序而战！***

据《National Review》作者：参议员汤姆·科顿，2021年12月11日发布：
![](https://assets.gnews.org/wp-content/uploads/2021/12/3207.png)2021年1月19日，工人们在中国青岛的港口参加 “一带一路” RCEP航线线的开通庆祝仪式。图片来源：(Zhang Jingang/VCG via Getty Images)
中共国20年前加入世贸组织是一个严重的错误。国会可以通过立即终止北京的特殊贸易地位开始纠正这一错误。

二十年前的今天，中共国加入世界贸易组织，承诺为全人类带来更大的和平和繁荣。在为这一历史性讨论时，时任美国总统比尔-克林顿傲慢地坚称： “我认为，具有讽刺意味的是，如此多的美国人担心一个强大的中国在21世纪对世界的影响。”我们现在正在见证当时那些美国人是多么正确。

在过去20年里，中共政府将其近17万亿美元的经济作为武器，以发动了一场无情的经济世界大战，这场战争着实给每个大陆都带来痛苦，给全球无数社区带来毁灭。

自2001年以来，中共国经济增长了近1200%，从一个三流落后地区跃升为世界第二大经济体、第一大出口国和工业强国。当前，中共国生产世界上四分之一的汽车，三分之一的商船，40%的手机，70%的电视机和96%的运输集装箱。

更令人担忧的是，中共已经扼制了许多基本材料的生产，并在关键技术领域获取了主导地位。中共国不仅制造世界上一半以上的钢铁，生产我们普通药物中三分之二的有效成分，还加工85%的稀土元素。中共国在人工智能和量子计算领域也取得长足进展，拥有世界500台运行最快的超级计算机中的200台，并生产世界70%的无人机。

通过其消费和生产，中共控制着近四分之一的全球贸易。更令人震惊的是，有70%的国家与中国的贸易额超过与美国的贸易额。

在许多方面，比其生产能力相比，中共国的消费潜力更强大。几十年来，中共国一直被视为 “未来的市场”，西方公司一直渴望追寻其14亿消费者。尽管未来似乎遥不可及，进入中国市场的途径仍受到各种限制，但企业公司仍然对中共国无底限地纵容。中共国像一条巨大的琵琶鱼，用开放市场和廉价劳动力的承诺成功吸引成千上万的西方公司越来越接近它。

不可否认，中共国现在是一个经济超级大国。它通过无休止的欺诈和谎言、欺骗和偷窃的简单组合获得这种地位。中国共产党已部署了全套经济武器，包括知识产权盗窃、企业间谍活动、强制技术转让、广泛的产品倾销和货币操纵。

知识产权盗窃是中共国最明目张胆、最有利可图、最持久的经济战行为。中共是世界上最多产的知识产权盗窃贼。中共对美国知识产权盗窃已高达80%，其生产的假冒商品占所有运入美国的假货总量的70%以上。同时，近一半是联邦调查局的反间谍案件，涉及经济间谍活动。对知识产权盗窃而言，中共每年至少两次窃取相当于阿肯色州的经济体量。

中共政府甚至利用派往西方大学的学生作为技术间谍和盗窃的特工。每年新知识产权盗窃事件层出不穷，中共公民偷窃研究机构和实验室的“皇冠上的宝石”–从先进的火箭和军事隐形技术到尖端的商业技术。中共政府利用大学校长的天真、懦弱和腐败，只要中共为他们提供资金，资助其不断增长的捐赠基金，他们就愿意保持沉默。

即使那些没有在大学校园里积极从事间谍活动的学者，仍对美国的技术和创新战略有深入了解，这使中共具备了重要的竞争优势。我们的大学系统实际上是在培训中共下一代的军事工程师和科学家。

对于强制技术转让而言，中共以敲诈代替抢劫。为了让西方公司获得或保持进入中国市场的机会，中共经常强迫它们与中国公司合作，分享或授权给出最有价值的商业机密。这必然导致中国公司与政府分享这些机密，进而利用他们的知识来复制西方技术，向全球市场倾销廉价的仿制品。

中共还将其公司部署到美国和欧洲，以获取西方技术并将其带回大陆。例如，从2010年到2018年，中国航空工业集团公司（AVIC）斥资30多亿美元从美国和欧洲收购重要的航空技术，这些技术推动了中共国先进的无人驾驶飞行器、活塞发动机和飞机发动机技术的发展。中航公司恰好是中共人民解放军唯一的军用飞机供应商，得到中共政府的鼎力支持。与中共军方有关的其他国有企业也在汽车和半导体行业进行类似的采购。以上这些并不是中共自称的“自由市场 “采购：这些采购是中共政府对本应属于西方的技术进行的不公平反竞争性国家收购。这些收购本质上是盗窃。

中共领导层变得更加肆无忌惮，采取了前所未有的法律手段，每周向西方知识产权盗窃的受害者收取一百万美元，除非他们放弃对中共公司的知识产权案件，并提交给腐败的中共法院裁决他们的索赔。这不是一个正常的国家所为，而是一个日益具有侵略性的国家行动。中共为在技术军备竞赛中取得进展，已经掠夺了无数西方企业家、商业领袖和科学家的毕生成果，而且他们还打算掠夺更多。

中共还利用其补贴产业来摧毁外国竞争对手。中共国钢铁和铝业可以说明了这一点。2000年至2017年期间，中共国的粗钢产量从全球总量的15%上升到全球产量的50%。反观同期美国，64%的原钢生产商倒闭，50%的基本氧气炉被关闭或闲置，35%的钢铁工人失去工作。这场灾难并不是因为中共国具备某些先天竞争优势；而是由于中共的大量补贴和有目的的 “产能过剩”。2018年，美国商务部认定，继续享受进口补贴的中共钢铁 “在经济上不可持续”，并威胁到“损害国家安全”。

同样，在2001年至2016年期间，中共原生未锻轧铝产量猛增了10倍，从全球产量的13%增加到54%。从2014年到2015年，中共引发国际铝市场上供应过剩导致价格暴跌近30%。因此，造成世界各地的冶炼厂和工厂闲置或关闭。

中共一再重复同样的游戏规则：向全球市场倾销廉价的仿制品，消灭竞争对手，然后抬高价格。通过非法串通、操纵价格和倾销产品等手段破坏和转移世界上大部分制药业后，中共公司再大幅提高药品价格。在维生素C的案例中，中共公司接管这一行业后，将维生素的价格提高了600%。

中共通过非法贬值其货币来扩大其产品倾销。低汇率会给一个国家的出口商带来人为的优势–中共已最大限度地利用这一点。2018年，为抵消美国的关税，中共将其货币贬值至少8%。虽然美国财政部随后在2019年将中共国列为货币操纵国，但中共几十年来一直在攫取货币贬值的好处。

近年来，中共国的经济侵略更加公开化，从邓小平的韬光养晦过渡到习近平的重拳对抗。随着世界意识到中共构成的威胁，中共政府的回应是进一步将其市场武器化。

2019年，北京威胁要切断美国的稀土供应，并在大流行病期间威胁要限制我们的基本药物供应。中共的官方宣传报纸发表了一篇文章，断言如果中共行使其杠杆作用，切断药品供应，“美国将陷入新型冠状病毒流行的地狱”。2010年，中共兑现了类似的威胁，暂时切断日本的稀土供应，进而破坏日本电子工业。

中共采用其他侵略性策略打击我们的盟友。针对加拿大2018年逮捕华为首席财务官孟晚舟，中共不仅将加拿大外交官扣为人质，还大规模削减加拿大猪肉和油菜籽产品的进口，而这些产品对加拿大农民至关重要。由于这些限制，加拿大对中共国的出口暴跌16%，造成加拿大生产商损失数十亿美元。去年，当澳大利亚推动对Covid-19病毒来源进行独立调查时，中共也同样对澳大利亚生产商进行打击。中共禁止进口澳大利亚棉花和牛肉，对大麦征收80%的关税，限制赴澳旅游，并对澳大利亚葡萄酒展开官方贸易调查。

当欧盟对中共在新疆的种族灭绝行为实施象征性的制裁时，中共对此进行报复，制裁五名欧洲议会议员、二十几名欧盟大使和几个与欧盟结盟的智库。这种过度反应几乎破坏了欧盟与中共国的贸易协定谈判。

中共为支持其品牌企业，对代表欧盟国家采用无耻的敲诈策略，威胁要在2019年拒绝与丹麦达成贸易协议，并威胁要在2020年限制与荷兰的贸易，除非他们接纳华为和中芯国际。这些都是一个帝国主义共产主义国家的行动，而不是一个自由市场国家的举措。

然而，与习近平倡议“一带一路”项目的帝国野心相比，所有这些行动都显得微不足道。这一庞大的基础设施计划已包括130多个国家的项目，预计耗资1万亿美元—按今天的美元计算，相当于七个马歇尔计划。

中共已推行一系列财政和实践上均不规范的项目，不能期望为其东道国获得适当的投资回报。在某些情况下，这些项目似乎由中共的贪婪和东道国的轻率共同促成。在其他情况下，“一带一路”项目显而易见是中共“债务陷阱外交”举措。在斯里兰卡，中共为几乎未使用但具有战略意义的汉班托塔港提供超过10亿美元的高息贷款。此外，它还资助了一个价值2.1亿美元的机场，该机场最终平均每天只有7名乘客。不出所料，当斯里兰卡无法偿还债务时，中共要求将港口租借99年，并成功实现这一要求。

同样，乌干达在2015年从中共借款2亿美元，用于扩建其唯一的国际机场。贷款协议中附加了多项掠夺性条款，包括如果乌干达政府不偿还贷款，就可能没收机场。许多人担心中共可能很快就会获取这一宝贵资产。

中共还资助了运作不良的贝尔格莱德-布达佩斯铁路线。贝尔格莱德站被称为该市历史上 “最大的交通灾难”。据预计，该项目到2500年才能盈利，还不包括维护费。同样，在欧洲的第一个“一带一路”项目中，中共将黑山的债务从GDP的63%提高到80%，用于修建一条永远无法证明成本的道路。中共在东非最引人注目的铁路项目被贴上“通往无名之地的铁路”的标签。

这些项目对东道国来说更加雪上加霜，因为用于建设的材料通常在中共国制造。同样，大多数工人也是中国人。在穿越老挝的铁路项目中，中共为该项目带来2-5万名中国工人，而只有7000名老挝人被雇用。对非洲和南美的工人来说，他们极为熟悉这种原始交易，东道国工人一直被排除在自己国家的基础设施项目之外。这些自我交易的安排抵消了基础设施项目的许多益处，这些项目通常会教授工人相关技能并促进国内就业。

更为雪上加霜的是，中共的 “一带一路”项目一直被投诉质量不高且易损坏。一位中亚官员指出，中共国人 “非常擅长按照你的预算进行建设。但十年或十五年后，你必须重建一切。” 这究竟是由于中共国的无能，还是为维持长期客户而不择手段，这一点值得深思。

尽管“一带一路”项目经常会变成“烂尾工程”，但中共国的贷方仍然收取回报，数以万计的中国工人找到工作，无数产能过剩的的中共公司为其产品找到出路。在某些情况下，这些项目还具有关键战略目标，即建立陆上贸易路线、规避美国监管的海上通道、扩大华为等国家品牌企业的影响力、导致地处关键国家的依赖性，或获得对稀土和淡水等重要资源的控制。

中共从“一带一路”项目中攫取财富和影响力，而东道国只剩下沉重的债务和不合格的基础设施。这与伙伴关系和合作相去甚远——这是大规模的寄生和剥削，也是中共经济战的又一例证。

中共侵略的受害者遍布世界各地。在美国，可以在上千个空心化的工业城镇和工人阶级社区以及被鸦片类药物危机蹂躏的社区的停尸房中找到。板板封住的仓库和空荡荡的工厂成为中共国加入世界贸易组织的庄严纪念碑。2001年12月11日，作为西方世界犯下历史上最严重的经济错误之一的日子而臭名昭著。

国会可以通过立即终止中国的永久正常贸易关系地位，开始纠正其异常判断错误。在20年前，这种特殊的贸易地位为中共国加入世界贸易组织铺平道路，撤销其地位是恢复我们防御的第一步。

西方世界其他国家必须与美国一起，对中国政府施加巨大压力。我们所有的国家都赋予了这个恶魔政权权力，我们必须联合起来，结束它的经济世界战争。我们必须拒绝和终止中共“一带一路”项目，结束对中共国关键技术和材料的经济依赖，实施严厉的制裁和关税，直到中共国停止其非法经济行为和可怕的人权侵犯举措。美国和西方盟国还必须大幅限制向中共国出口关键技术和资本——我们决不能向中国共产主义者出售绞杀我们的绳索。这场运动不会是快速、容易或轻松的，但这是必要的。我们必须发动一场经济上的持久战，剥夺中共的经济特权和自负的权力，否则就有被掠夺和超越的危险。

中国共产党是一帮经济罪犯，必须将其排除在国际贸易秩序之外。正义、公平和良知要求我们这样做。

汤姆·科顿（Tom Cotton），阿肯色州参议员。
