![](https://assets.gnews.org/wp-content/uploads/2021/12/AF075E87-A2BD-43ED-A518-6CA4E7AC0766.jpeg)图为佳兆业2017年度全年业绩公布会（图片来自大纪元）

据《大纪元》12月8日报道，佳兆业集团控股有限公司再次发布公告宣布：短暂停止股份买卖。佳兆业的这次停牌距离其上次复牌还不到两周时间。

据报道，在香港股市开盘前，佳兆业集团发布公告表示，该公司自2021年12月8日上午9:00起短暂停止买卖，等待公司发布包含内部信息的公告。

《德国之声》的报道指出，佳兆业是中共国第27大房地产公司，同时也是负债最多的公司之一。该公司12月3日宣布，其2021年12月7日到期的6.5% 优先票据的交换要约及同意征求失效，意味着其寻求更多偿债时间的尝试失败。根据公告，这笔票据未偿还的本金总额为4亿美元。消息一出，佳兆业股票大跌。

佳兆业曾于11月5日停牌，11月25日方才复牌。时隔不到两周时间，佳兆业再次宣布停牌。

据《华尔街日报》披露，截至6月底，佳兆业集团未偿美元债约109亿美元。根据目前的状况显示，其财务危机已进一步加深。
