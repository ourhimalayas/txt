## 有声｜布林肯宣布取消与俄罗斯外长会晤

**2022-02-23**
![](https://assets.gnews.org/wp-content/uploads/2022/02/12-29.png) 图片来源(CAROLYN KASTER/POOL/AFP via Getty Images)
《大纪元》2月23日报道，周二，美国国务卿安东尼‧布林肯宣布，鉴于俄罗斯对乌克兰采取的最新行动，他已取消原计划与俄罗斯外长谢尔盖‧拉夫罗夫本周在日内瓦举行的会晤。

继俄罗斯总统普京承认乌克兰东部两个由分裂分子控制的地区独立，并宣布将派驻维和部队进入该地区后，美国总统拜登和美国高级官员周二表示，普京的举动标志着俄罗斯对乌克兰的入侵已经开始。作为回应，拜登宣布了对俄罗斯的第一批制裁措施。

布林肯在同一天表示，俄罗斯的入侵正在开始，而且俄罗斯已经明确表明它完全拒绝外交手段，在这个时候继续举行这次会议是没有意义的。他还补充说，美国与美国的盟友和伙伴们进行了磋商，他们都同意这一决定。

报道指出，布林肯的举措是一个最新迹象，表明拜登政府不再相信俄罗斯在认真寻求和平解决危机的方案，美国与俄罗斯在乌克兰问题上的外交途径正在迅速关闭。
