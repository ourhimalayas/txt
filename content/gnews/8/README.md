**循证医学的幻觉**

所谓循证医学的意思是，医学实践应当以证据为基础，这是几十年来临床实践发展的方向。上周的英国医学杂志（BMJ）发表了一篇评论，切中时弊地抨击了临床研究已经广泛而且深层次地被大药厂绑架，以至于所谓循证医学仍然只不过是一种幻觉。

循证医学的有效性取决于临床试验的可靠数据，但这些临床试验大部分是由制药行业进行，然后以资深学者的名义报告。醫藥领域主要由少數非常大的製藥公司主導，它們競爭市場份額，但有效地聯合起來努力擴大該市場。由於私有化而對生物醫學研究的短期刺激受到了自由市場擁護者的稱讚，但對醫學的意想不到的長期後果卻是嚴重的。

那些在學術界取得成功的人成为药厂眼中的“關鍵意見領袖”，由此他們不再扮演獨立、公正的科學家並批判性地評估藥物的性能，而是成為營銷主管所稱的“產品冠軍”。 而批評者则面臨被期刊拒絕发表、法律威脅以及職業生涯的潛在破壞。旨在促進科學客觀性和公正性的機構（即公共實驗室、獨立的科學期刊和学术會議）完全受政治和商業權力的支配；既得利益永遠凌駕於證據的合理性之上。

这篇文章完全不涉及中共病毒疫苗，伊维菌素和羟氯喹的研究，可是我们知道，它所谈到的临床研究被大药厂绑架的问题却在这三方面达到了登峰造极的地步，无论是辉瑞的临床研究造假，辉瑞报告的被迫披露，伊维菌素和羟氯喹研究中种种挑战人们智商底线的乌龙事件，这些事件的性质和恶果都是史无前例的。这里面所暴露的深层次问题，最终的解决方案首先是灭共，其次是回归正道主义，既要珍视信仰，也崇尚科学精神，唯真不破。
