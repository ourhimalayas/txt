## 有声｜美印会谈在即 美国警告印度不要进口俄罗斯能源

By:**秘翻新闻**

**2022-04-11**
![](https://assets.gnews.org/wp-content/uploads/2022/04/1-114.png)
《[路透社](https://www.reuters.com/world/biden-will-speak-indias-modi-us-warns-india-imports-russian-energy-2022-04-10/)》华盛顿4月10日报道，美国白宫表示，美国总统拜登周一将与印度总理莫迪进行虚拟会晤，并明确表示，不希望看到印度增加从俄罗斯进口能源。

报道称，西方对俄罗斯实体实施制裁后，印度受到石油大幅折扣的诱惑，已经从俄罗斯购买了至少1300万桶俄罗斯原油。路透社编纂的数据显示，去年全年的石油产量约为1600万桶。

俄罗斯长期以来一直是印度最大的国防设备供应商。分析认为，相较于美国的国防设备，俄罗斯设备更具成本优势。

最近，美国负责国际经济事务的副国家安全顾问辛格访问了印度，他说，美国不会为印度从俄罗斯进口能源设定任何“红线”，但不希望看到印度“快速加速”购买能源。

辛格还表示，美国准备帮助印度实现能源和国防供应多元化。印度是世界第三大石油进口国和消费国。

周日，白宫新闻秘书普萨基在一份声明中说，拜登总统将继续就俄乌战争的后果，以及对全球粮食供应和商品市场的不稳定影响进行密切磋商。
