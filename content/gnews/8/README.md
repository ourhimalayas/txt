美国白宫3月20日宣布，拜登总统将于3月25日和26日访问波兰。

拜登将在26日会见波兰总统杜达。由于俄罗斯军事入侵导致大量乌克兰难民涌入波兰的现状，他们将讨论应对乌克兰难民的问题。
