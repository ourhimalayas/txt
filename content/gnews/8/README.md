在奥米克戎病毒的攻击下，率先给全民接种新冠疫苗的以色列及医疗专家委员会于去年12月21日批准，为60岁以上人群和医务工作人员接种第四针新冠疫苗，接种第四针者需距离第三针疫苗四个月以上。

疫苗接种已从26号正式开始，近一个月来效果怎样呢？

以色列今年1月14日重症患者，创下近三个月新高。自疫情以来，人口仅970万的以色列已累计确诊170万病例，逾8000人病逝。

耶路撒冷赫尔佐格医院的院长说，95%的重病患者都接种了疫苗，完全接种疫苗的人占住院病人的85%到90%。疫情在医院里爆发，因为一个住院病人感染很多人，而且不是零星的。疫苗的有效性确实在减弱。

据悉，以色列财政部长李伯曼刚接种疫苗四针，仍感染新冠病毒。

以色列外交部长拉皮德1月10号宣布，确诊感染新冠病毒。

以色列卫生部强推疫苗 ，声称加强针注射后就能增强保护力，已成笑话。
