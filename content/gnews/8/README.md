中共从乌克兰获得了几乎所有它想要的军事技术，包括其第一艘航空母舰辽宁号。《环球时报》也承认，没有乌克兰，中共就没有今天的国防成就。

在2013年12月，当乌克兰弃核之时，中共承诺向乌克兰提供核安全保障。

去年6月，乌克兰副经济部长塔拉斯·卡奇卡表示，中国是乌克兰在亚洲最重要的战略伙伴。6[月30日](https://www.kmu.gov.ua/en/news/ukrayina-ta-kitaj-pidpisali-ugodu-pro-spivpracyu-u-galuzi-budivnictva-infrastrukturi)，乌克兰与中共国签署了深化基础设施项目合作协议。协议条款为中方提供优惠融资以及公路、桥梁和机场建设。

去年7月，乌克兰总统泽连斯基首次与中共领导人习近平通电话。泽连斯基表示，“乌方高度重视，致力于发展更紧密的乌中战略伙伴关系，乌克兰有望成为中国企业通往欧洲的桥梁”。
