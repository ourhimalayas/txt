港府专家顾问、港大微生物学系教授袁国勇为屠杀仓鼠辩护，将责任推予市民没有全部接种疫苗，尤其长者。他称该宠物店的仓鼠可能是在荷兰或香港有人将病毒传予仓库内的仓鼠，再由仓鼠传人。袁国勇说，科学与医学不能解决“情”这个议题，他仅对仓鼠爱好者失去宠物深表同情。他支持鱼护署的灭杀行动，称署方没有足够动物检疫设施、人手及检测能力，时间亦不容许取得所有基因组数据。

人类被赋予的责任是管理地球，但人类俨然认为自己是地球的主人却毫不爱惜自己的家园，净干着毁坏地球家园的蠢事。科学的发展不是为了令地球更美好，而是增加掠夺和伤害，中共利用生物科技将新冠病毒当成生物武器毒害全球，妄想着要当全世界的主人，面对全球追责，惯性地将播毒的责任推给动物们，并借助各种所谓的科学定动物的“罪”。港府的专家们也学会了用“人类利益”这个伟大借口，圈禁港人、逼迫港人打新冠毒疫苗，还要屠杀生灵转移病毒真相。

连日来Delta病毒被港府专家们嫁祸给仓鼠，导致数千弱小生命被鱼护署屠杀，宛如被中共暗杀的香港孩子们的悲惨景象。科学成为了杀生的工具，血的债却要堆在人民的头上，港府不愧是邪恶中共的傀儡。那些自以为聪明的专家们玩弄科技愚弄他人，但其所言所行都已被记录史册，他们的罪行无可推诿，最终会被反噬万劫不复。
