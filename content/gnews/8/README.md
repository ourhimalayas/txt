微博上有人发消息，说这个冬天只能烧柴火取暖，解封也买不上煤，因为政府统一采购、凭票买煤。

这就回到上世纪改革开放以前的计划经济年代了。因为现在的煤价很高，所以政府就管控、统一制定价格。但煤炭企业也不是做慈善的，政府指定价格会使煤炭企业亏损很多，那他们肯定会想办法不生产或者少生产，这就走上了几十年前的一切凭票购买商品的年代。现在是煤票，之后就会有粮票、布票、油票。六中全会习近平大权独揽之后，他一定会加速走向老毛时代的计划经济。**中国人吃不上饭的年代又要回来了。**

上世纪毛贼执政时期，中国人绝大多数都吃不饱饭，在所谓的三年自然灾害期间，还饿死了几千万人。估计用不了多久，有些地方的民众重又走上那条悲惨之路。

这不，第一步，企业大裁员已经开始。如同上世纪90年代末几千万国企工人下岗潮，这次是从钢铁行业开始的。这个消息没有官媒报道，网上对于这个消息也是封的严严的，只能从某些博主下面的评论里找到一点点的消息。

网传鞍钢、本钢开始裁员。说此次“改革”主要是精简干部，如年龄偏大的人员，不能遵纪遵章守纪上班的人员等，并给出了六条退出方式。因病不能上班的人按病休办法执行；已工作满15年不愿意上班的人，可以保留劳动关系自谋出路，企业仍然为其交保险；男50周岁以下、女40周岁以下，可以息工，（自我）创业最多三年，三年后必须回单位报到，息工创业期间，企业仍然为其交保。距离法定退休年龄大于五年的，可以与企业协商，申请解除劳动关系，买断剩余工龄。这与2000年企业下岗买断剩余工龄的说法如出一辙。现在过了20年，大规模下岗潮又回来了。

接着说裁员条例，距离法定退休年龄小于或等于三年的，可以自愿申请离岗，企业为其交保险，每月领取生活费、加工龄工资、加保留工资。不申请离岗的，将竞聘上岗，落聘的转岗进入人才赋能中心，参加转岗学习，每月领取生活费、加工龄工资、加保留工资。

这六条其实就是把年龄超过40岁，非青壮年的全部裁员，每月给你点生活费，不至于现在就让你饿死。大家一直说在中国过了35岁，尤其是四十岁，就基本找不到工作了，所以很多人拼命挤进国企。这下好了，鞍钢、本钢这种大国企也开始裁员了，而且目标计划裁员50%以上，也就是只留一半年轻力壮能干活的，老弱病残全回家。鞍钢在册职工4万人，本钢在册职工5.9万人，一下子就裁员了四至五万人。

目前中国钢企现在日子都不好过。武钢女45、男55内退；韶钢女45、男55内退。就是年龄大了，哪都不要你了，回家吧，国企也不是养老的地方。那抚顺阜新的煤矿企业呢？辽化、盘锦辽河油田呢？是不是也要大规模裁员？东北这些年越来越差，估计很多企业都会走上下岗裁员这条路。

以前很多私营企业，也就是所谓血汗工厂还能安置一些就业人员，现在全球供应链中断，世界经济低迷，消费起不来，沿海加工企业也活不下去都濒临破产。当2000年的下岗潮之后，进入WTO而使中国经济起死回生。这次，全球都在跟中共国脱钩，外企在撤离，中低端制造业在加速迁往东南亚，中国这么多四五十岁失业人员该怎么办？

前一次工人下岗大潮，虽然没有了收入，但也没有负债啊。而现在，有多少家庭背负着一屁股的房贷、车贷。银行也不是做慈善的，银行不可能说，你下岗，没工作了，把你的房贷、车贷免了。

2008年美国次贷危机之后，美国多少家庭被赶到了简陋的房车里、甚至大街上，中国是否会出现这一幕？

由于这两年的疫情，旅游、航空、饭店等吸纳大量人员岗位的行业基本没有正常营业过，行业收入每况愈下。后来中共国要让韭菜多生娃，打击教培行业，又又近1000万的从业人员也都失业。因为课外培训不只是老师，还有相关行政人员和网络人员。最近很多人说，以前送外卖只有男士，现在送外卖也出现了很多女士，可想而知这内卷有多厉害。还有很多私企35岁、40岁就让你回家的呢，还没有地方要你。

现在大型国企这波裁员潮，让40至60岁这些人可怎么过日子啊？很多家庭孩子还在上大学，外卖、滴滴能容纳那么多人吗？现在中共还能印钱发点，以后怎么办？这两年也已经物价飞涨了，再印下去，那岂不是成了“金圆券”吗?

以前只要不是乱世，在中国的历史上，一个王朝的周期一般就是200年到300年。而在现代社会里，政权崩溃速度肯定会快很多。因为以前古人从冬至就在家歇息，一直到下年开春儿都不用干活，还有各种祭祀，一年中大半年都在休假，根本没有什么效率；以前也没有工业化，也没有现代化，所以豪强兼并的速度也很慢；以前的地主是用土地产权来剥削，后来到了工业化，资本家用生产设备剥削，这样的速度就比以前快了很多。

就是生产力越发达，效率越高，财富集中的速度越快。封建农耕时代，土地从分散到集中，至少要两三百年甚至几百年才能完成；而工业化时代，从小业主到垄断托拉斯，一般不到100年也就够了。互联网时代如果不加限制的话，十年就能集中所有人的财富。你现在就能看出来，富人的利息都花不完，而穷人拼命工作还欠了一堆债务。有了现代金融制度之后，现在的全球富豪大佬，用现代金融债权制度来剥削老百姓，只需要十几年的时间，财富就全都弄到他们手里去了，老百姓已经没有活路。

所以豪强兼并的速度加快，那么王朝更替的速度也会加快，所以共产党也就只有70几年的寿命。
