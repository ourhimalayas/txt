2022年4月1日，由30辆巴士组成的车队载着从马里乌波尔和梅利托波尔撤离的人员抵达扎波罗热的登记中心，乘客们纷纷下车。照片。Emre Caylak/AFP via Getty Images

一支红十字会车队周六将试图从被围困的马里乌波尔市撤离乌克兰平民，此前周五的一次尝试失败了。

**滚动新闻：**根据一份声明，红十字国际委员会由三辆车和九名人员组成的团队周五试图帮助市民离开，但“安排和条件使其无法继续进行”。该组织说：“他们将在周六再次尝试，以促进平民安全离开马里乌波尔。”

**总体情况：**港口城市马里乌波尔已面临数周的俄罗斯军事轰炸。官员们说，成千上万的人仍然被困，食物、水和药品供应不足。路透社报道说，已经计划在周六建立一个人道主义走廊，帮助人们通过私人交通工具从马里乌波尔撤离，并为马里乌波尔居民提供巴士，使其离开别尔江斯克市。

红十字国际委员会说：“为使行动取得成功，各方必须遵守协议并提供必要的条件和安全保障。“

**他们在说什么：**据联合国报道，红十字国际委员会的一位发言人在日内瓦对记者说：“这里没有B计划，我们已经工作了数周，并向媒体介绍了我们为带着援助进入马里乌波尔并允许平民安全离开该市而做出的努力。““对于马里乌波尔的人民来说，时间已经不多了”。
