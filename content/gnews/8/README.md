4月1日FNN综合在线消息：法新社30日报道，乌克兰总统泽伦斯基宣布解除乌克兰驻摩洛哥和格鲁吉亚大使的职务，因为他们在鼓励所担当的国家向乌克兰提供武器和实施对俄罗斯的制裁方面努力不足。

泽伦斯基在视频演讲中说：有些人的工作增强了乌克兰保卫自己，为自己的未来而战的能力，还有些人则坚持自己的立场，浪费我们的时间。我决定解雇后者！

格鲁吉亚未能加入对俄罗斯的制裁，还禁止志愿兵前往乌克兰增援。之后，驻格鲁吉亚大使于3月被召回。

泽伦斯基先生说：很抱歉，如果既不能实现武器增援，也不能令所负责国家加入对俄罗斯贸易的制裁阵营，我希望你另谋高就。

他还呼吁说：我希望驻扎在拉丁美洲、中东和东南亚的外交官和军官在未来的日子里拿出业绩。因为外交战线是一个重要的岗位，每个人都必须高效工作。
