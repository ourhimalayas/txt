## GTV平台紧急关闭

作者：银河系农场 – 骄子
![This image has an empty alt attribute; its file name is Screen-Shot-2021-08-30-at-7.37.59-PM.png](https://assets.gnews.org/wp-content/uploads/2021/08/Screen-Shot-2021-08-30-at-7.37.59-PM.png)
3月3日，郭文贵先生在盖特平台上宣布，由于中共持续对GTV黑客攻击并篡改数据，为了保护战友们的信息安全和投资者的利益，GTV投资委员会和法律委员会决定，GTV平台已经于当天凌晨12:00彻底关闭。
![](https://assets.gnews.org/wp-content/uploads/2022/03/1-46.jpg)
郭先生补充，目前，新中国联邦的所有直播节目都可以在盖特平台上观看。至于投资者，请与喜马拉雅联盟委员会联系。

五年来，中共的黑客从未停止过对郭先生直播的攻击。同时，中共政府对郭先生在各个社交媒体平台上的打压与封禁也从未止歇。但是，这些都不能阻止G系列遇难呈祥且越来越强大。

郭先生此前反复强调，GTV作为传播真相、消灭中共的信息平台，会经历三次浴火重生。而这次的紧急关闭，新中国联邦则对此早有预案。

爆料革命的战友们也表示，期待GTV在不久的将来会以全新的面貌回归。
