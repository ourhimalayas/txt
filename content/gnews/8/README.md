## 拥有2500万人口的上海仍然处于封锁状态 压力越来越大
` 西班牙巴塞罗那喜悦农场` [轉載自GNews](https://gnews.org/zh-hans/2345362/)

素材编译: Mandel

![](https://assets.gnews.org/wp-content/uploads/2022/04/xin_png.001-1-1280x140-28.jpg)

![](https://assets.gnews.org/wp-content/uploads/2022/04/image-1730-edited.png)CNN是唯一经历过上海封锁的美国媒体。点击这里观看视频。图片来源: CNN

上海（CNN）- 大多数清晨，我在上海居住的狭窄的巷子里传来远处扩音器的回声，召唤我和我的邻居们从家里出来，去做强制性新冠测试。

我戴上口罩，拿着手机，在穿着防护服的志愿者们敲门之前就走了出去。如果你错过了这次呼叫，他们会持续敲门，直到有人回答。没有人可以例外。

这个拥有2,500万人口的庞大城市是中国有史以来最大的新冠疫情的中心，也是该国付出最大努力的地方。没有人被允许离开他们的住宅区，甚至是去购买食物，这意味着我们要依靠政府或私人送货司机，大量的需求让他们焦头烂额。这给系统带来了巨大的压力 —— 对许多人来说，这些限制比病毒的威胁更让人不安。

![](https://assets.gnews.org/wp-content/uploads/2022/04/image-1738-edited.png)来自CNN的记者David Culver在中国上海的封锁期间进行了一次常规的新冠测试

在我的公寓外，身穿防护服的社区工作人员带领我和我的邻居们排着社交安全距离的队伍经过我们锁着的前门，这是我唯一被允许走出公寓的时间。但他们从未带领我们走出大门——大门已经被弔锁和自行车锁封了三个多星期了。

当我们走到一张复盖着蓝色帐篷的桌子前时，医务人员正在那里等待进行测试，我感到一阵情绪波动——对被允许出去呼吸新鲜空气和享受春天的阳光感到欣慰，同时也感到焦虑—— 如果我的测试结果呈阳性怎么办？我担心自己会被送进上海简陋的隔离系统，呆上几天或几周。隔离设施的图片显示，我可能会面临狭窄、不卫生的条件，垃圾桶满溢，没有自来水和肮脏的公共厕所。

但我更担心会发生些什么在我的救援犬Chairman身上。（译者注：Chairman是作者的小狗的名字）

如果你的检测结果呈阳性，你的宠物会怎么样，这仍然是一个令人不安的灰色地带，没有明确的解决方案。网上流传着宠物被遗弃的恐怖故事，最近有一只宠物被一个穿着防护服的人用铲子打死了。

如果我被带去隔离，我希望当地的某个兽医或社区团体可以会被允许照顾我的狗。我已经装了一小袋Chairman的必需品，放在门边，以防我被送走后有人能收留它。

但这是不太可能的。除了必要的工人之外，整个城市就像我一样，被封锁了，被锁死了。

![](https://assets.gnews.org/wp-content/uploads/2022/04/image-1741-edited.png)David Culver担心，如果自己测试结果呈阳性并不得不进行隔离，那么他的狗Chairman可能会发生些什么。

争相购买额外的食物

3月下旬，在城市的居民被命令呆在家里之前，恐慌的买家让杂货店的货架上空空如也。

现在，绝望的情绪已经出现了。

视频显示，人们在向社区工作人员尖叫，恳求他们提供食物，说他们正在挨饿。其他视频显示，在一个隔离区食品分发点，人们为一小批蔬菜而争吵。

在我的社区，政府每隔几天就送一次食物。运送的食物包括一盒蔬菜和鸡蛋，或一块真空密封的猪肉或一些传统的中医药等。仅仅是发放的食物还不足以养活一个人，更不用说整个家庭，超过一天左右。

我的食物是定量供应的，我对运来的盒子里的食物和我的社区能够提供的任何额外食物进行了充分利用。最近，我的大部分饭菜都是鸡蛋和胡萝卜的组合——你必须要有创造力。

许多社区在中国社交媒体应用程序微信上建立了与邻居们的群聊。偶尔会有团购食物的服务，但选择有限。商店被关闭，送货司机被封锁，供应链被破坏。

![](https://assets.gnews.org/wp-content/uploads/2022/04/image-1742-edited.png)CNN的David Culver在大多数日子里试图订购额外的食物，他所在的大院里的居民正在交易食物以填补短缺。

我的一个邻居在聊天群里写道：“如果我没有食物，我应该怎么办？”社区联络员回信说：“目前没有团购——蔬菜现在供应短缺。”

我在封城的日子里，大部分时间都在试图下多个食品订单，希望有一个能到。上周，我在午夜后被一个电话吵醒——我的一个订单真的出现了。

我急切地联系我们的社区联络员，让他们帮忙取回订单，但经过一天的工作，他们都已经睡着了。因此，我不得不把订的杂货放在大院外街道上的一个箱子里，直到早上6点，希望在我拿到它的时候没有被拿走或变质。值得庆幸的是，到了早上，东西还在那里。

我们中的一些人已经想到了创建无接触的“投放点”的办法，在那里交换食物以改变我们的饮食。

例如，在一次从社区新冠测试中走回家后，我的一个邻居给我发了一条信息。她把一块奶酪留在了她自行车上方的阴凉处。当我后来出门去做新冠测试时，我拿走了她的奶酪，用两个橙子代替。然后，当她被允许出去做下一次新冠测试时，她就可以收起水果。

当局似乎听到了这些抱怨。上周末，上海副市长宗明在一次新闻发布会上哽咽了，为未能达到预期目标而向市民道歉。周一，当局承诺开始放宽对一些地区的限制。

![](https://assets.gnews.org/wp-content/uploads/2022/04/image-1743-edited.png)食品包裹被送到被封锁的住宅小区，但有些人说他们没有足够的食物

愤怒和不确定的未来

从武汉开始，我已经报道了这次中国疫情的各个方面。由于中央政府推行 “零感染” 政策，公众似乎已经忘记了早期的处理不当和对最初传播的被指控的掩盖行为。

两年来，中国在很大程度上成功地将病毒拒之门外，方法是关闭边境并引入一个看似复杂的接触者追踪系统，利用智能手机技术来追踪我们和我们可能接触到的病毒。

官员们已经完善了大规模测试，有能力快速处理拥有数千万人口的城市。他们主要依靠有针对性的快速封锁——关闭有确诊病例或密切接触者的社区、办公室甚至购物中心——从而试图避免封锁整个城市，以尽量减少社会和经济损失。

最近几个月，包括西安、天津和深圳在内的整个城市都进入了封锁状态，但没有像上海这样的规模，在那里，遏制病毒的肾上腺素和公共精神已经被疲劳、沮丧和绝望所取代。

在我600平方英尺（译者注：55.74平方米）的公寓里，我问自己，这些真的发生了吗？在上海，在所有的地方？

作为一个拥有高楼大厦和餐馆林立的现代城市，上海曾经可以与巴黎和纽约等国际大都市相媲美。现在，数以百万计的居民正在为基本的生活必需品而争先恐后地逃离他们的家。

这并不是说上海的生活不会恢复原状，但过去几周的”行动” 或 “不行动”，加上对过去两年来以预防新冠病毒的名义突然出现的严厉限制措施的持续不确定性，使许多人感到与这个城市和彼此之间的联系越来越疏远。

周一，美国国务院命令非必要的领事人员及其家属离开该市，理由是新冠病例激增，以及强制实行的遏制新冠病毒的限制措施所带来的影响。

我认识的大多数外籍人士要么已经离开，要么决心离开。原因是什么？“这是不可持续发展的”是一个常见的说法。

在精神上。情感上。身体上。这都不是（可持续发展的）。

由CNN 记者David Culver于2022年4月12日报道。
