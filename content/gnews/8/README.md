在俄乌战争爆发前后，各国政府纷纷尽全力免费撤侨。但中共却从一开始掩盖战争即将爆发的真相，到战争爆发后又以安全得不到保障为由，迟迟不撤侨。即便撤离了极少一部分人，那也是撤的中共官员及其亲戚和关系户，完全置数万滞留在乌克兰的中国同胞的生死于不顾。

在此情况之下，新中国联邦和法治基金迅速组织力量奔赴乌克兰前线展开紧急撤离救援，并且分文不取。中共见此被迫又演起了“撤侨”的假戏。然而，中共方面不仅以高价机票勒索急于撤离的同胞，并且还在登机前的深夜强制他们排练唱红歌“歌唱祖国”，以配合中共的宣传。

据《阿波罗新闻网》3月9日报道，3月7日，有从乌克兰逃到波兰的中国留学生称，第二天可以乘坐中共的撤侨飞机回国了，但每人需购买18000元（人民币，下同）的高价机票，并且半夜还被叫起来彩排合唱，为的是登机前在机场唱红歌感谢祖国。

3月8日网上流传的一段视频证实了上述说法。在波兰，准备登机回国的留学生确实手举“感谢祖国，感谢使馆”的血色横幅，在集体演唱红歌。现场还有多家中共国媒体在拍摄视频。

有逃到罗马尼亚的中国人在网上吐槽称：回国机票17999元/人，一家四口刚逃出乌克兰，根本无力支付近10万元的机票钱。有媒体向中共大使馆求证，使馆人员证实了此说法，但却声称这是航空公司的决定，与大使馆无关。

17999元/人的价格是平时机票价格的近3倍。有网友指责说，其它国家政府安排的撤侨包机一般都是免费的，从未见有收取高价机票的。另有未经证实的消息称，中共最先撤离的是留学生，17999元的机票只是学生价，而一般人的机票价格超过3万元。

除了高价机票之外，根据中共的防疫规定，中国公民乘机回国后还要面临强制隔离，并被收取高价隔离费、餐饮费、医学检测费用等，每人的费用在一万元左右。
