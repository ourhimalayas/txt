**背景信息**

我看到一个信息：2022年3月1日之后，中共要禁止网络上有关宗教信仰的讲道、资料。这意味着关于信仰的管控要达到一个新高度了。

近年来，中共对民众的信仰控制有目共睹的，但是宗教信仰人士还是可以通过软件进行一些联络和知识的沟通。3月1号以后，中共如此明目张胆的限网，已经撕掉了遮羞布，开始彻底封锁信仰信息了，宗教信仰人士的处境会越来越糟。

**比信仰控制更恐怖的是信仰的变异**

今天我们讲的内容：比信仰控制更恐怖的是信仰的变异。以下我们谈的是一个真实的基督教会案例，如有雷同实属巧合：

在北京的一个基督教教会，总共有150多圣徒，周日礼拜聚会一般在80人以上，W牧师言必引自圣经，宣称中国境内最懂解开圣经，他规定手下的20多位教师的讲道必须用他的稿子，不许随意发挥，每个月只给教师很少的工资，并实行精神控制。

W牧师还建立了教会学校，教会学校的家长都被要求委身在自己的教会，这样有利于教会稳定，圣徒不易流失。

这位牧师宣传的是孩子们长大后不用去世俗从事什么丰功伟业，在这里好好信神做义工就可以，从不在学术上让孩子们努力学习。这显然与圣经是相背离的，因为圣经·申命记28:13是要圣徒作首不作尾，居上不居下，这样才有神的荣耀。W牧师建立学校的目的，是为了留住圣徒，这样就会收到大量的奉献。W牧师教育老师们不用在教学上努力，只要不断的给家长们查经就行，其实就是洗脑。

这位牧师拥有绝对的权利，建立的执事会的商业就建立在教会里，比如卖酒的商店、食品部、豆腐坊、艺术培训机构。这位牧师还允许圣徒从事传销，只要传销得来的钱十分之一交给教会就可以。他还开发基督化APP、小程序，准备打造一个虚拟币的世界。

牧师拥有多套房产，自己的办公室所用的桌椅、设施设备都是世界顶级品牌，有一些事专门从海外运过来的，就像住在自己的宫殿里。平时钓钓鱼，生活的很享受。

这位牧师除了自己享受物质，还在情欲上玷污好多位女圣徒，甚至在行淫的时候，还能说出圣经章节来。

而这位牧师带领的圣徒又是怎样的呢？除了读圣经，除了会讲道，在生命上是一塌糊涂，没有行为，没有爱心，不传福音。

为了达到自己的目的，把圣经顺应自己的目的去讲，就成了教义上的异端。

把奉献的钱拿来买来物质享受，圣经说这叫“肉体与圣灵是为敌的”，肉体占了上风，那么圣灵就不与他同在了。

只会讲道，自己却行不出来，越来越多的已经信神人被绊倒，让更多的人不去信神。

**中国的信仰已经变异**

中共国的教会占比最多的是未注册而又隐蔽的家庭教会。这些教会这些年很隐蔽的进行着宗教活动，年复一年，基督教的信仰人士没有减少，反而每年增加。看上去很令人振奋。

中共国的基督教会，在中共的打击下，已经变得左右逢迎更懂生存之道。但是这种生存只是人数增加，他们变得更加聪明，更加不信任人，于是就出现了牧师名下多套房产，紧接着就是人性更多的考验和物质的挥霍，最后良心一点点泯灭。

在教会内部，众多的信徒明显的感觉到在自己身边，在平日里喊的弟兄姊妹中，平日里聚会聚餐嘻嘻哈哈的人中，很难找到一个愿意互相信任的生死之交。生活中的基督徒总是信仰挂在嘴边，而行为又是另外一回事，与不信的人没有多大区别。

**要有希望要有盼望**

在中共国这样的畸形的土壤上，生长出来的有中共国特色的信仰已经变异，不再是一块金字招牌。但是信仰人士不要放弃希望，尽管我们面临各种的不如人意，尽管生活在黑暗中，但是还是要有美好的盼望，从做好自己开始，从自己的家庭幸福开始，不要效法那些黑暗的人和黑暗的事。
