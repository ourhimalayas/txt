美国正在与全球的盟友和伙伴合作，向乌克兰人民提供安全援助和人道主义援助。

美国与盟友和伙伴合作，已经对俄罗斯实施了破坏性的制裁。

美国军队还向北约联盟的前线国家部署了人员和资源，以确保北约的每一寸领土都得到保卫。

周三 (3月16日)，美国总统乔-拜登在白宫说，全世界团结一致，支持乌克兰，并决心让普京付出非常沉重的代价。并签署了一项由美国国防部向乌克兰军方直接转让价值8亿美元设备的命令，以帮助乌克兰人民捍卫其自由。

白宫发布了一份援助清单。该计划包括800套 “毒刺 “防空系统，2000套 “标枪”，1000套轻型反装甲武器和6000套AT-4反装甲系统。

该计划包括100套战术无人驾驶航空系统。

国防部还将向乌克兰部队转让100台榴弹发射器、5000支步枪、1000支手枪、400挺机枪和400支霰弹枪，以及超过2000万发小武器弹药和榴弹发射器及迫击炮弹。

一揽子计划还包括25,000套防弹衣和25,000顶头盔。

此外，美国正在与盟国合作，以确定并向乌克兰转让更远距离的防空系统。

拜登说，该一揽子计划表明我们致力于将美国最先进的系统送至乌克兰用于其防御。

拜登还指出，美国实际上早在俄罗斯入侵乌克兰之前，就在帮助乌克兰。

美国从2021年3月开始发送了6.5亿美元的安全援助。这包括乌克兰军队使用的反空和反装甲武器，并取得了巨大效果。

因此，当入侵开始时，他们手中已经有了反击俄罗斯推进的各种武器。

普京宣布俄罗斯军队入侵乌克兰时，美国又送给乌克兰3.5亿美元。

上周末，拜登又批准了2亿美元的资金，以保持武器和弹药源源不断地运往乌克兰。

拜登今天签署的命令，使美国对乌克兰的新安全援助总额达到10亿美元。
