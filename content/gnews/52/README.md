据加拿大《CTV新闻》12月7日报道，安大略省政府当天宣布，为了更有效监测公共卫生事态，更多了解“奥密克戎”变体病毒状况，将继续无限期延迟重新开放计划。

报道称，最新数据表明，目前“奥密克戎”疫情对该省“新冠病毒”感染病例和住院患者人数的不断大幅上升影响不大。此外，由于重症监护室的病床占用率可能会成倍增长，因此各大医院将再次面临资源不足的压力。

据悉，安大略省现在的七天平均每日新增病例高达近千个，是自今年6月第三波疫情以来又一高峰。由于越来越多的人在室内聚集，预计冬季的感染人数还会持续增加。
