1、[**孟晚舟突获释 大陆媒体乱阵脚**](https://www.aboluowang.com/2021/0925/1651463.html)

9月24日，孟晚舟突然获释返回中国。这一突发事件被外媒争相报道，但与外媒相反的是大陆媒体却显得猝不及防，乱成了一锅粥。在孟晚舟已经登上返回中国的飞机大约5个小时后，新华社在其微信公众号终于出了一篇通稿。而在此之前的4、5个小时里，大陆许多网站登出相关的简讯，却都在短短的几分钟或半个小时就被“404”（删除）。

综合媒体报道，9月24日，华为首席财务员孟晚舟通过视频在纽约布鲁克林法庭出庭受审，在她承认数个错误做法，并与美国检察官达成暂缓起诉协议之后，美政府同意撤回向加拿大司法部提出的将孟晚舟引渡到美国的请求。而且孟晚舟当天已乘坐一架包机从温哥华机场起飞返回中国。

据大纪元报道，按照协议，检方将起诉的文件延迟到2022年12月1日，也就是孟晚舟被捕四年后撤销针对她的指控，前提是她满足某些条件。根据达成的DPA协议，孟晚舟对多项银行欺诈指控表示不认罪；但是作为协议的一部分，她承认曾向银行作假、帮助华为破坏了针对伊朗的经济制裁。

2、[**中共称与孟晚舟事件无关的两加拿大人质获释回家**](https://www.aboluowang.com/2021/0925/1651428.html)

加拿大总理特鲁多周五（25日）说，两名被北京拘留超过1000天的加拿大公民在美东时间晚上约7点30分已离开中国领空，并于周六返回加拿大。特鲁多说，“这两个人经历了难以置信的艰难磨难。在过去的1000天里，他们展示了力量、毅力、韧性和优雅。”孟晚舟今天获释，结束在加拿大近3年居家监禁生活，晚间随即飞回中国。孟晚舟被捕事件引发一场外交风暴，中共之后以涉嫌刺探情报为由，拘留加拿大前外交官康明凯（Michael Kovrig）和商人史佩弗（Michael Spavor）。北京当局曾公开否认此事与孟晚舟事件有关。然而孟晚舟今天获释后路透社快讯报道，加拿大总理杜鲁道表示，两名遭中方关押的加拿大人也将于25日上午返国。

3、[**配合美国 印度启动史上最大军改**](https://www.aboluowang.com/2021/0925/1651310.html)

彭博24日引述印度官员指出，印度总理莫迪高喊军事改革已行之有年，终于在近期付诸行动，逐步将原本各自为政且欠缺协调的陆、海、空三军，整合成统一的指挥机制，可望在2024年成形，以回应及配合美国联合印度－太平洋盟邦围堵中共的政策。印度在上个月成立“军队事务部”。经过整合的印军至少会有4个战区司令部，西、东部战区司令部分别负责监控巴基斯坦和中国，海上司令部负责印度洋防务，最后则是空中司令部，至于北部的查摩与克什米尔则暂时不会调整。
