## 废除“郭七条”，战友行动在即

作者：纽约香草山农场福音部-恩永存
![This image has an empty alt attribute; its file name is Screen-Shot-2021-08-30-at-7.37.59-PM.png](https://assets.gnews.org/wp-content/uploads/2021/08/Screen-Shot-2021-08-30-at-7.37.59-PM.png)
12月8日，郭文贵先生在直播中公开宣布，即日起，废除“郭七条”中“不反习”该条，并吹响了街头革命的号角。

![](https://assets.gnews.org/wp-content/uploads/2021/12/Slide1-12.jpeg)

郭先生表示，中共一贯泯灭人性，处事没有底线，采取了极端手段对待数千名的盘古员工和郭先生的老父亲——中共也会用相同的手段对付新中国联邦人和所有世界爱好和平之人。郭先生强调，对待中共这种恐怖分子，世界正义人士必须诛之。因此，新中国联邦人必须开始行动。

当天参与直播的战友也相继发表了参与此次行动的决心。他们谈到，既然已经公开露面直播，便早已把生死置之度外。战友们强调，能够成为灭共大军的一员是他们的荣幸。

其后，郭先生也在直播中收到了来自长岛伟哥的消息，当中提及，喜马拉雅联盟委员会将即刻组成全球三年连续抗议行动三人指挥小组，由老班长、长岛哥和大卫组成。同时，联盟将尽快公布抗议行动方案，欢迎全球爆料革命战友通过农场报名参与，也请各农场核心团队尽快组成行动小组并设立指挥负责人，配合联盟统一行动。

对此，战友们也反应迅速，纷纷在盖特平台上发帖宣传，并积极配合农场参与行动。
