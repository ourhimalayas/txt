![](https://assets.gnews.org/wp-content/uploads/2022/01/Water-pollution-1.jpg)水污染 / 图源：网路

中共国曾经依靠丰富的水资源、土地，和廉价劳动力，在WTO全球贸易合作里成为了世界工厂。其实本身中国大陆的水资源分布就是极度不合理的，西部常年干旱，沙漠广布，经过了这么多年的急功近利，不合理地开发和使用水资源，加上工业化污染，目前中共大陆内，数千条河流消失，剩下的水资源中，80%—90%的地下水和50%的河水无法饮用；50%以上的地下水和25%的河水无法用于工农业。水资源如此匮乏的状态，让中共国不得不开始侵占和威胁他国，中共曾在湄公河上建造水坝，从而导致了东南亚地区的常年干旱和洪涝灾难；中共也在新疆等地将河流分流，直接给中亚下游地区带来毁灭性影响。中共对全亚洲水资源的掌控野心，在喜马拉雅。七哥直播中曾提到喜马拉雅是全亚洲最重要的水源地，86-90percent的河流自此发源。中共控制西藏，进而控制全亚洲的水资源。

俗话说一方水土养一方人，中共大陆可用水资源常年缺乏，而水是万物之源，老百姓的生活无法不受其影响。还有郭先生在大直播中所说，中国人都被共产党泯灭了人性，男女之间的自然情爱都变成了羞耻之谈，一切都是假的，一切都是干的。天干物燥，火烛流萤，红色万里，血染江山。男女两情相悦要流水，真挚真诚的感情要流水，无论是流眼泪还是流水，这个人必须是真性情的流露。经常能够见到一些面孔，笑比哭难看，生气比开心自然，因为这些长期生活在谎言中的人，从来没有真正的笑容，次次都是表演出来的笑容，表演出来的开心，而生气是真实的生气，哭是真实的哭，真实的情绪表达是自然的，不违背人性的，所以才会让人看上去是，哭起来更好看一些。还有一个形容是关于男女相爱的，说你可以掩饰动作，掩饰笑容，但你无法掩饰你眼里的光芒，实实在在就是男女之间产生荷尔蒙的准确表达。这些全部都是需要个人人性的存在的。

共产党不允许个人化的任何东西存在，一切以集体为主。集体中的每个人都是机械的，虽然也在进行着人类生活的一切活动，比如从小长大，成年，恋爱，结婚，生子，养孩子，退休，养老，死去。似乎中共国内的一个人的一生就是这样过完的，尤其是体制内的公务员，从第一天开始坐在办公室，就已经看到了往后几十年的生活，明日复明日，今日一切如此，never changed，everything as usual。没有什么值得悲哀的，毕竟想要干公务员的人都着急地来排队呢，千万人挤一座独木桥，不来白不来。虽然这样枯燥的日子和一眼能够看到头的公务员工作明明就是对人的最大摧残，然而在共产党这里，这是一个铁饭碗，你一辈子饿不死，当然你也一辈子上不去。在中共国从事公务员工作的人有千千万，然而最后能够从基层一直往上升级，那是不可能的。基本上没有什么硬背景一辈子就已经确定了。
