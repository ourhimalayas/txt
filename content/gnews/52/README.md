## 中共超限战灭港行动（五十一）倒行逆施激民愤 616民阵发起大游行
` 喜馬拉雅-粵語組` [轉載自GNews](https://gnews.org/zh-hans/1405907/)

#### 搜集/撰文：岁月如歌 / 封面合成：文粤


![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/07/19025040/7191.jpg)
为了支援冲突中受伤或者被捕的示威人士，前立法会议员吴霭仪和何秀兰以及艺人何韵诗、学者许宝强于15日宣布成立反送中受伤，被捕之人道支援基金『简称：612人道基金』 ，为有需要的示威者提供紧急医疗、法律及相关援助。

梁凌杰先生坠亡让港人非常痛心，以及政府的态度也令香港市民愤怒至极。第二天，民阵发起第四次大规模游行，参予人数有200万+1人。民阵召集人岑子杰称，游行人数反映出市民非常愤怒，原因一林政的回应「完全面向北京」，没有正面回应市民诉求，二是不满警方「一次又一次的暴力镇压」。林郑月娥虽然宣布暂停修例二审程序，想愚弄市民，和假惺惺的道歉以此平缓民愤，但她对自己强行修例以及警察暴力镇压没有半点歉意，她只暂停修例并不肯撤回，及中共高调支持修例与林郑月娥激怒了香港人反对修例的决心。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/07/19025111/7192.jpg)图片来自BBC
民阵发表五大诉求：一、完全撤回《逃犯条例》修订草案；二、撤回暴动定性；三、释放被捕者且撤回控罪；四、成立独立调查委员会，追究警队涉嫌滥用武力的问题；五、行政长官林郑月娥辞职下台。林郑政府一次次藐视民意，为中共在香港犯罪提供法律保护，把香港人甚至世界各国途经香港之人亲手送与中共整治，这是十分凶险的，是人都害怕中共不透明的，共产党式法治。为了自己和后代市民没有退路只有反抗。市民身穿黑衣从维园出发浩浩荡荡一路呐喊着五大诉求的口号一路向政总推进。

在全世界的目光注视之下，香港在短短的八天内发生了三次大规模的游行事件，也目睹了香港警方暴力镇压的场面。这次属于第四次游行，市民由于在2014年发生过雨伞革命，他们都有了经验，上街游行全副武装保护好自己，戴口罩，护目镜，护肤品，头盔，雨伞等，在警方射出催泪弹未爆炸前赶紧用水泼息，用加密电报相互通信息。占中运动之后，很多香港人都明白，在民主社会中，没有一个声音，可以代表香港人，相同的政治诉求的人们聚在一起，成为反送中游行中不同的组成部分，他们不再希望自己再像雨伞运动时那样，被一些大型团体领导着，无法表达自己真正的愿望。这是反送中运动与雨伞革命的不同点。香港前立法会议员罗冠聪表示，没有任何组织可以包含所有人的诉求。这也使反送中游行成为「没有领袖的运动」。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/07/19025139/7193.jpg)图片来自法广
在当天示威游行到下午6:00多的时候，中联办发表声明：表示支持林郑，由于外国势力干预抹黑，使事件变得复杂。而港区代表陈永棋推诿说“「有人想香港乱」，呼吁反对修例的市民不要被他人利用。这是共产党一贯的口吻和措辞，有事都是美国佬及西方反华势力在背后操纵指使”。独裁政权认为，从来都是别人欺负他，把责任撇的一干二净。不但中国共产党如此，朝鲜，古巴等国的共产党也如此，将其责任过错推​​诿到美国等西方民主国家。中联办在早上的时候就约见了中共圈养的人大代表和政协委员，一起商量如何对付游行示威，一群香港吸血鬼经过密谋商议得出结论：中央对林郑月娥的决定表示“尊重、理解、支持”。说好听点是这帮人商议，说不好听其实就是来接圣旨，然后做复读机罢了。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/07/19025206/7194.jpg)图片来自光华日报
下午7点左右，有4男16女示威者不适，一人危殆，香港发生了一幕『人潮红海』宛如摩西分红海的壮观镜头，这样举措让香港人的素质和文明惊艳世界，茫茫人海自动让道救护车的画面令人动容，香港人每次大游行后都会主动清理街道垃圾，分类打好包装，充分体现香港极高的修养，香港人高度文明配拥有民主法治，可惜他们这些细微的动作感动世界文明国家却说服不了独裁中共。这是人类的悲哀，遇到邪恶，只有强与弱统治与被统治，没有文明。

直到晚上10:00多士威人士仍不肯散去，不少人在政府总部外面留守，市民认为林郑月娥没有撤回恶法，也没有收回暴动定性，大家不肯原谅林郑的『虚伪的道歉』。岑子杰表示，「三罢」行动将会有越来越多的市民响应。
