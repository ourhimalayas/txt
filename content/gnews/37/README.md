**2.欧洲多国宣布取消限制，疫情要宣告结束？**
![](https://assets.gnews.org/wp-content/uploads/2022/02/image003-1.jpg)
**【天维网1悦29日综合报道】据路透社1月26日报道，尽管欧洲的Omicron感染人数仍居高不下，但丹麦计划于下周取消国内剩余的所有新冠防控措施，而此前一周英国、爱尔兰和荷兰也发布了类似公告。**

**在长达一个月的封锁之后，这个北欧国家于两周前放松了限制，允许电影院和音乐场所重新开放，但仍有一些限制措施，包括限制餐厅的营业时间和强制佩戴口罩。

丹麦卫生部长Magnus Heunicke在给议会的一封信中表示，政府打算遵循专家小组周二（1月25日）发布的建议，在2月1日取消所有限制措施。如果这一决定得到议会的批准，这将是北欧国家中最广泛的放宽限制举措。

按照计划，丹麦将于2月1日放开限制。届时，夜总会可以在晚上10点之后重新营业，餐厅也能够提供酒水服务，顾客进店消费时无需出示疫苗证书，通勤者搭乘公交车时无需佩戴口罩，取消对商店顾客人数的限制。但仍保留了入境者的检测和隔离措施，这些规定将再保留4周。

无独有偶，英国、爱尔兰和荷兰上周也发布了类似公告，决定取消大多数新冠防控措施，即使Omicron变体的感染率仍然很高。

**5.口罩新规有变化，婚礼现场不需要戴口罩**
![](https://assets.gnews.org/wp-content/uploads/2022/02/image006.jpg)
**【天维网综合报道】新西兰政府昨天（1月29日）澄清了在Omicron疫情爆发期间，哪些场合需要佩戴口罩，哪些场合不需要佩戴。**

**在公众抗议和请愿之后，政府官员澄清了婚礼上是否每个人都需要戴口罩的规定，包括拍照时。

“在婚礼上，新郎和新娘戴着口罩亲吻对方，这是很滑稽的事情。”国家党议员Chris Bishop昨天在推特上写到。

卫生部昨天表示，从2月4日起，食品和饮料企业、有近距离接触的企业、活动和聚会现场都需要佩戴口罩。

但是， “在专用场地上举行的聚会，包括婚礼现场，可以不用佩戴口罩，你可以摘下口罩吃东西、喝水或运动”。

另外，公众必须佩戴“真正的口罩”，能够固定在耳朵或头部。

同时，四年级及以上的学生在搭乘教育部资助的学校交通和公共交通工具时必须佩戴口罩，而且所有受疫苗强制措施约束的工作人员，在从事面向公众的工作时必须佩戴医用口罩。**
