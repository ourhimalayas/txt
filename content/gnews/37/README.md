欧盟已冻结俄罗斯总统弗拉基米尔·普京和外交部长谢尔盖·拉夫罗夫的资产以应对乌克兰的入侵，并将这两个名字添加到欧盟制裁名单中。该措施意味着欧盟银行将被禁止向它们提供资金，尽管这将取决于金融实体的合作程度。

周五晚上，欧盟外交政策负责人何塞普·博雷尔证实了这一决定。博雷尔指出，普京是继白俄罗斯总统亚历山大·卢卡申科和叙利亚总统巴沙尔·阿萨德之后第三位受到欧盟个人制裁的世界领导人。

本周早些时候，欧盟制裁了 351 名杜马议员和普京核心圈子的成员。然而，为了保持外交渠道畅通，普京和拉夫罗夫不会受到旅行禁令的约束，这是一项经常伴随资产冻结的常规制裁。

尽管法国总统埃马纽埃尔·马克龙（Emmanuel Macron）最近在周四晚上与普京进行了交谈，但在边界危机期间，西方外交的局限性已经暴露出来，现在变成了一场全面的战争。

普京和拉夫罗夫的名字被排除在之前的一揽子制裁之外。但随着军事攻击以压倒性的速度展开，极大地增加了西方国家提出更严厉和更激进惩罚的可能性。

周四的初步措施随后被传递给欧盟的外交部长，以进行最后的润色。正是在这些部长级讨论中，冻结普京和拉夫罗夫的金融资产的选项作为一个可行的选项重新回到了谈判桌上。

在欧盟外交部长会议后，周五晚上公布了最终制裁措施。整个一揽子计划将在欧盟官方期刊上公布后生效。

爱尔兰外交和国防部长西蒙·科文尼在前往部长级会议之前对记者说：“我们今天将就此达成协议。我认为这两个名字都将被添加到一长串其他名字中。”他表示，将普京和拉夫罗夫排除在外将“破坏制裁的可信度”，但他补充说，旅行禁令暂时不应受到打击。他还说：“必须始终为对话和外交接触敞开大门，如果这涉及旅行，则需要提供便利。”

冻结普京的资产是一种象征性而非惩罚性的举动。围绕普京角色的默默无闻也延伸到了他的个人财富。据《财富》杂志报道，普京的净资产可能在 700 亿美元到 1500 亿美元之间。这些财富中有多少储存在欧盟领土上还有待观察。

当被问及如何追踪总统的资产以冻结它们时，博雷尔说. “相信我，我并不知道普京先生和拉夫罗夫先生的财富的秘密。这不是我的职责。但有人会处理这件事。”

这位高级外交官补充说，第三套制裁方案正在酝酿之中，但没有透露细节。但他说欧盟正在根据俄罗斯的行为探索所有可能性。

博雷尔证实，将俄罗斯从国际支付系统 SWIFT 中驱逐不在当前的一揽子计划中，但有可能保留在未来的谈判中。

拜登政府在周五表示，将对俄罗斯总统弗拉基米尔·普京和外交部长谢尔盖·拉夫罗夫实施制裁，以回应莫斯科对乌克兰的袭击。并对两人实施旅行禁令，并承诺稍后提供更多细节。
