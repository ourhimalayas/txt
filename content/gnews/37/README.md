## 中共不灭，人祸永存
` 纽约香草山MOS02` [轉載自GNews](https://gnews.org/zh-hans/1818276/)

作者：纽约香草山健身部 – 新世界的一员
![This image has an empty alt attribute; its file name is Screen-Shot-2021-08-30-at-7.37.59-PM.png](https://assets.gnews.org/wp-content/uploads/2021/08/Screen-Shot-2021-08-30-at-7.37.59-PM.png)
十年浩劫，中共所谓的“伟大时代”，过去了几十年，至今没有那一个人敢据实撰写这个罪恶滔天的年代，到底有多少良人被害，还得灭了中共以后，由新中国联邦给出真实的数据。

十年动荡首当其冲的是那些知识分子，迫害致死的数不胜数，这一场惨无人道对文化的摧残，对人性的残害是绝无仅有的，连当面活活打死无辜的人都不会觉得有犯罪感的恐怖年代，活在二十世纪的习太阳却还想复制一遍，曾少年时期受尽文革屈辱生死磨难的习神，如今为了帝位霸权又进行一场比文革还惨无人道的杀人运动，可见世界原本是没有魔的，是人心坏了成的魔，中共不灭，人祸永存，所以歼灭中共是正义的需要。

![](https://cdn.discordapp.com/attachments/902176588546850836/927560339892084776/IMG_9492.jpg)

笔者读过太多文革惨案，阅读时几度落泪，在那个荒唐的年代，全国上下受意识形态的蛊惑，竟然家家户户都觉得害人斗人折磨人是一种正确的政治选择，发生的每一场批斗，上演在中华大地的每一个角落，都是现场版的恐怖片！

一句真话，几句公道，立马就成反革命，人和人之间的相处，翻脸比翻书还要快，今天你斗我，明天我斗你，斗的天混地暗，祸及生命的家庭数不胜数。

今天在公众号翻阅到吕荧先生的事，看多了文革期间诸多悲剧的笔者对吕先生有种莫名的叹息，既为他看不清当时的时局背道而驰惋惜，也向他铮铮铁骨的知识分子傲气敬礼！

![](https://cdn.discordapp.com/attachments/902176588546850836/927559826576408596/IMG_9491.png)

吕荧先生在民国时代研究俄国文学和左派立场，他大部分时间都在四川、贵州等偏远省份执教，因为总是批评国民政府不为学校所容，他1947年从贵州大学辞职，辗转到了台湾。在国民党败退前夕，他反其道而行之，导致了他悲剧的一生，他的重返大陆想一展抱负的乌托邦美梦，最终碎在受尽屈辱的红色地狱。

1949年5月，躲在台湾师范大学教美学的吕荧，在国民党败退台湾时迫不及待的收拾包袱返回大陆，进入山东大学执教。开始了他认为的理想生活。

1951年，全国上下轰轰烈烈批判电影《武训传》，武训的家乡山东大学的师生们也紧随潮流大张旗鼓鞭打这个曾经的乡贤。吕荧转不过弯来，在课堂上说：“武训也是你们山东的一个圣人，他办义学错在哪里？没有钱，到处募捐甚至乞讨，正表现了他对办学的坚韧执着，怎么成了罪人呢？”

这番言论在当时这样的环境中视其大逆不道，当然逃不过被批判，。政治风向标《文艺报》甚至点名痛责，学校方表示只要他认错就可以出面息事宁人，可他拒不认错，在召开对他的批判大会时，悄悄找熟人，跑到人民文学出版社当了翻译，逃过一劫。

1955年，震惊全国的文艺界第一大案“胡风案”发端。有人举报胡适“在文艺创作上否认了革命实践和思想改造的意义”，被划为“反党集团”，整个文艺界火力全开，一致口诛笔伐。

![](https://cdn.discordapp.com/attachments/902176588546850836/927560382002913280/IMG_9493.jpg)

1955年5月25日，由郭沫若牵头，文联召集七百多位文艺界大咖召开批判大会，其中也包括在民国时代和胡风就有交集的吕荧。郭沫若带头：反革命分子必须加以镇压，其后20多名代表在发言中一致痛骂胡风，与之划清界限。不在发言名单内的吕荧在如此狂热的氛围中几次站起来想说都被坐在旁边的同事制止，他就干脆径直走上讲台抢过话筒说：“对于胡风我认为不应该说是政治问题，而是学术问题，是文艺观的一种争论，更不能说他是反革命……”

这段话在当时正是石破天惊，当场遭到郭沫若斥责，一大堆人就冲上台来把他直接赶下台。其实吕荧和胡风相识却没有交情。两个人早年因为在文艺创作上的观点不合拍还激烈争论过。但就是这么点交情，在万马齐喑的环境中，爱才惜才又正义不屈的吕荧就敢站出来说人话。没有意外罪恶年代，吕荧的人话带来了他灭顶之灾。他走出会场后就被宣布隔离审查长达一年之久。直到高层点名放生，但是1966年文革兴起，他的这段历史不可避免的又被捞出来，被定性为“漏网的胡风反革命集团分子”，被批斗恶打押送至北京良乡农场清河农场，长期劳改、受尽屈辱又缺衣少食，这个文弱又硬骨头的书生，面对看不到半点光明的年代，熬不过精神的困苦，患精神分裂症，瘦得只有50多斤……1969年3月5日，吕荧死于劳改农场，年仅55岁。

他的狱友用一张苇席包裹他的躯体，就地掩埋。墓碑只有半块砖头，上面是用粉笔的“吕荧”二字。当然，最后尸骨无存。如今留给世人的只是吕荧先生的美学著作！
再看如今的文艺界，多的是那唯唯诺诺跪活在红爪下的假文艺，假文明，当下的这样的一个乱世，于国于民都比当年文革期间的残酷和伤害有过之而无不及，虽没有批斗胜过批斗，没有打喵的人在别人眼里你就是不负责，你就是想害人，你就是妨碍国家执法的罪人，同样的万众一心，全国上下就是没有人站出来说真相讲真话，所幸的是文革期间没有希望只有死望，而现在有了新中国联邦，只有希望不用绝望，更有真相开心智，保命有良药，保财有喜币，希望受骗的人们早日清醒，更希望有更多的人知道新中国联邦，坚挺人性，守护良知，支持正义，揭露邪恶，曝光恶党恶性，当光明降临大地时，就可以无愧的说，在这个史无前例的乱世中，在此惊心动魄的善恶大决战中，你做了一个爱自己爱家人爱国家的正常人，让后代子孙撰写时有几笔好词好诗。
