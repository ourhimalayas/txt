![](https://assets.gnews.org/wp-content/uploads/2022/02/图片1-1.jpg)图片来源：C-SPAN
2月2日，《国家脉动》调查记者娜塔莉·温特斯（Natalie Winters）发布报道，披露NIH（美国国家卫生研究院）的一个教育委员会的联合主席兼任一个与中共有关系的游说团体顾问。

文章说，亚当·霍特（Adam Hott）在NIH临床测序证据生成研究小组工作，他也隶属于美国腹地中国协会（USHCA）。他在这个有争议的团体教育委员会任职，而该委员会的宗旨却是：“汇集K12和高等教育的资源，让研究、专业知识和新工作的人致力于美中合作”。

报道说，美国腹地中国协会与中共国国外游说团体的关系被揭露后，中共党员和中共公司购买美国农田的报道引起了议员们对国家安全的关注。

除了与中共政府的各个部门合作，美国腹地中国协会还“自豪地与”中美交流基金会（CUSEF）合作。根据美国政府的说法，该组织是中共“统战部”的一部分，它试图“收编和消除可能反对执政的中国共产党的政策和权威的人们”，并“影响外国政府采取支持北京所指定的政策的行动或立场”。

美国腹地中国协会还与中国人民对外友好协会（CPAFFC）共同举办活动，后者被认为是统战部的“公开亮相”，是“中共国的公开武装”。美国国务院将该协会定性为寻求“直接和恶意游说”美国国家和地方领导人的组织。

尽管各界对此有着这样的关注，霍特还是担任了NIH临床测序证据生成研究小组的教育和成果回报分部的联合主席。

根据他在领英（LinkedIn）个人资料中的披露，他自2019年以来一直担任这个职位，而“这个工作组专注于基因组测序结果和对患者、家庭、护理人员和医疗服务提供者的教育的设计和交付”。

霍特也发表了几篇期刊文章，概述学校的科学课程，也包含有直接由NIH财政支持的论文。他撰写的其他论文包括“生物技术101：遗传学和生物技术教育推广计划”和“理论与实践：非科学专业生物入门课中的遗传学内容”。
