2021年7月19日，美国与欧盟、英国、澳大利亚、加拿大、新西兰、日本和北约发表联合声明，共同指责中共国国安部门主导和支持的恶意网络黑客攻击行为不仅损害了美国的国家安全和利益，也威胁到世界多个国家的经济与国家安全。

声明指出，今年1月初到3月期间，与中共国主要情报机构有关联的黑客使用了超过50种策略、技术和方式对微软(Microsoft Corp., MSFT)的Microsoft Exchange 电子邮件软件系统发起不同寻常的无差别攻击，导致数十万家机构和企业受到影响。

美国总统拜登当天在白宫就美国经济发表讲话时被问及有关中共国利用黑客发动网络攻击的事情时表示，他会在周二（7月20日）收到一份关于中共国非军方情报机构在使用黑客软件攻击美国企业方面所扮演角色的详细报告，针对中共黑客网络攻击的调查还没有结束。

在当天举行的白宫新闻简报会上，Jen Psaki在回答记者提问时也强调，欧盟、英国、澳大利亚、加拿大、新西兰、日本和北约在内众多盟国和伙伴正在与美国一起揭露和批评中共国国家安全部门的恶意网络活动。这也是北约第一次谴责中共国的网络黑客活动。众多美国盟友实际上正在采取措施，不仅公开发表意见，而且肯定会采针对中共国的网络黑客行为采取行动。

今年3月2日，微软副总裁 Tom Burt 博客（Blog）称,一家位于中国共的黑客组织，名为Hafnium，自今年1月初以来，连续一个多月时间，利用微软用于企业和政府数据中心的 Exchange Server 邮件和日历软件（ Exchange Server mail and calendar software ）存在的漏洞，通过网络攻击，窃取了至少250,000家美国的国防承包商、学校和其他实体组织电子邮件帐户的访问权限，从那里盗取信息。

在美国政府发表对中共国国安部门的网络黑客行为谴责的同时，世界其它众多国家也发表了类似的谴责：

2021年7月19日，英国外交部门发表声明，称要中共国为包括今年稍早对微软电子邮件系统等一系列网络黑客攻击行为负责。英国外交大臣拉布称，若中共国不停止这种网络攻击，他们将追究责任。

欧盟外交与安全政策高级代表博雷利（Josep Borrell）在一份声明中称，中共国窃取知识产权和进行间谍活动的恶意网络活动已经严重影响欧盟的经济、安全、民主和整个社会。

北约呼吁中共国履行国际承诺和义务，要与受恶意网络攻击的国家站在一起，打击在网络威胁和共同挑战。

日本政府表示，日本公司遭到了一个很可能与中共国政府有关联的APT40网络黑客组织的攻击。日本内阁官方长官加藤胜信在例行记者会上表示，日本政府正密切注视这一情况，并对此“很担心”。

挪威外交部长索雷德(Ine Eriksen Soereide)在一份声明中说，挪威情报机构的调查显示，挪威及其盟友、欧盟和微软证实了3月份这次网络攻击来自中共国。中共国的网络攻击破坏了挪威最重要的民主机构的电子邮件系统，他们将始终对这类入侵做出反应。

对中共国安部门领导的网络黑客打击行动正在展开。美国司法部7月19日宣布，位于加州圣迭亚戈的一个联邦大陪审团在今年5月，已经对4位居住在中共国的网络黑客提出起诉。这4名黑客分别是，海南省国安厅的特工丁晓阳、程庆民、朱允敏和来自于中共国安全部门其它公司的吴淑荣。这四人作为海南仙盾职责的一部分，通过编写黑客软件，侵入美国、奥地利、柬埔寨、加拿大、德国、印尼、马来西亚、挪威、沙特、南非、瑞士和英国等国的航天、国防、教育、政府、卫生、生物制药及海事部门的电脑系统，盗取行业机密和商业机密信息，并管理海南仙盾的其他黑客。

事实上，加拿大国防部下属的通讯安全局（CSE）在2020年11月18日发布的一份网络威胁评估报告中，就点名指出中俄伊朝是网络犯罪的主力军，他们有可能瞄准加拿大的关键机构，也可能为未来行动布局，非常可能试图开发出干扰电力供应的新的网络黑客能力。
