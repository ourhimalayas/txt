## 港府称将推进立法量刑，针对CIA的&#8221;中共国任务中心&#8221;
` GTV新闻访谈` [轉載自GNews](https://gnews.org/zh-hans/1591379/)

作者：在远方

![This image has an empty alt attribute; its file name is Screen-Shot-2021-08-30-at-7.37.59-PM.png](https://assets.gnews.org/wp-content/uploads/2021/08/Screen-Shot-2021-08-30-at-7.37.59-PM.png)



10月7日美国中情局(CIA)宣布，将会成立两个全新的中心应对来自中共的战略竞争和挑战，其中一个名称就是”中国任务中心”。CIA现任局长 威廉·伯恩斯 称美国正面临着最严峻的地缘政治考验，而中共政府就是最主要的威胁。”中国任务中心”旨在更好地调配全球资源搜集和分析中国动向。
![](https://assets.gnews.org/wp-content/uploads/2021/10/1013-AM-（2、4）视频备用.001.jpeg)
针对中情局这一不加掩饰的举措，香港保安局长邓炳强称，”香港基本法”第23条中所涉及的间谍罪量刑，在本地立法时要“加强着墨”。目前的《港区国安法》已经涵盖包括叛国、煽动叛乱、窃取国家机密、禁止外国政治组织或团体在香港活动、禁止香港本地政治组织或团体与外国建立联系等。但港共政府发现有关法例不足以应付过去两年发生的所谓”危害国家安全”的情况，特别在应对境外”间谍”方面。也是因此，中共人大常委香港代表谭耀宗不遗余力的，在促进本届港府尽快完成并实施香港基本法第23条，以授予香港本地警察及国安机构更大的权限，应对来自海外的间谍及颠覆活动。而这个23条正是 引发“反送中”街头运动的恶政源头！
