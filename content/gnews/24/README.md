**1. 世卫专家提议G7集团成立调查组调查病毒来源**

美国乔治城大学世卫组织公共卫生法和人权合作中心主任[劳伦斯·戈斯汀](https://www.rfi.fr/cn/%E5%9B%BD%E9%99%85/20210703-%E6%9C%89%E4%B8%93%E5%AE%B6%E5%BB%BA%E8%AE%AE%E7%AA%81%E7%A0%B4%E4%B8%96%E5%8D%AB%E7%94%B1g7%E7%BB%84%E5%BB%BA%E6%9C%BA%E6%9E%84%E8%B0%83%E6%9F%A5%E7%97%85%E6%AF%92%E6%9D%A5%E6%BA%90?ref=tw_i)（Lawrence Gostin）说到，一年半来，世卫组织一直任由中共国摆布，如果依靠世卫组织，将永远不可能找到病毒来源。并表示，美国等国家可以通过自己的情报部门，把各种线索拼凑起来，修改国际卫生法，赋予世卫组织权力，或者另外组建一个新的机构专门负责调查病毒起源。

世卫组织顾问小组成员杰米·梅茨尔（Jamie Metzl）和同事提议，由G7集团成立一个新的调查组负责调查病毒来源。

2. **民调显示，近三分之一的美国人认为卫生官员在疫苗安全问题上撒谎**

[拉斯穆森民调](https://www.breitbart.com/politics/2021/07/03/poll-nearly-one-third-of-americans-think-health-officials-are-lying-about-vaccine-safety/)显示，32%的美国成年人认为公共卫生官员在新冠病毒疫苗的安全性上说谎；在不打算接种疫苗的人中，这一数字上升到近三分之二。

在已经接种疫苗的人中，21%的人认为公共卫生官员在疫苗的安全性上说谎；而在未接种疫苗的人中，这一数字上升到50%；在那些没有接种冠状病毒疫苗,并且将来也不打算接种疫苗的人中，不相信公共卫生官员的比例达到63%。

**3. 台湾欢迎立陶宛在台设立代表处**

[立陶宛外交部长](https://www.voachinese.com/a/taiwan-welcomes-lithuania-decision-to-set-up-representative-office-20200703/5952145.html?utm_source=twitter&amp;utm_medium=social&amp;utm_campaign=dlvr.it)兰斯伯格斯(​Gabrielius Landsbergis)表示，除了在澳大利亚建立大使馆外，立陶宛计划在韩国、新加坡和台湾设立代表处。台湾外交部发言人欧江安高度欢迎立陶宛政府拓展与台湾合作交流、加强双边友好关系的积极作为。

兰斯伯格斯表示，立陶宛并非反对中国，而是希望减少对中共国的经贸依赖，以免在政治上受制于中共国，影响国家安全利益。

4. **拜登总统下令调查大规模网络攻击**

[拜登总统](https://www.reuters.com/technology/biden-says-uncertain-who-is-behind-latest-ransomware-attack-2021-07-03/?taid=60e0c7a0bcdfda0001432ea0&amp;utm_campaign=trueAnthem:+Trending+Content&amp;utm_medium=trueAnthem&amp;utm_source=twitter)周六表示，已指示美国情报机构调查袭击数百家美国企业的幕后黑手，将在周日收到有关最新攻击的简报。

在6月16日的日内瓦峰会上，拜登敦促普京总统打击来自俄罗斯的网络黑客，并警告说，如果此类勒索软件攻击继续扩散，后果将不堪设想。拜登总统称，如果确定是俄罗斯所为，美国将做出回应。

美国网络安全和基础设施安全局在周五的一份声明中表示，正在采取行动，了解和解决这次网络攻击。

**5. 中共国前公安人员讲述目睹 “工业化 “摘取器官的经历**

一位来自中国郑州市、现居美国的前公安人员鲍勃（化名）向[大纪元](https://www.zerohedge.com/geopolitical/former-police-officer-recounts-witnessing-industrialized-organ-harvesting-china)记者描述了目睹“工业化”摘取器官的经历。这个“工业化”的供应链将活人转化为产品，用于器官交易。其中的参与者包括，司法系统、警察、监狱、医生和发布指令的中共（CCP）官员。鲍勃称，摘取死刑犯的器官是公开的秘密；不仅如此，他们还从活着的良心犯身上摘取器官，特别是法轮功学员。

鲍勃说，“在巨大利润的驱使下，所谓的人权和人道主义问题没有任何位置”，希望“中国人民能够从中共专制统治中解放出来，找到自由”。
