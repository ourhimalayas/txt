前言

我知道这一天终究会来，政府步步紧逼，我的父母为了进食肆吃饭，上周悄悄接种后才告知我。
和女儿争论时我无力的哭了。
这半年来，我们在预防上也加了维C维D，我总是告诉两个孩子，哪怕不上班不上学，我们也不打疫苗。
