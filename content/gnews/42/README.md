## 投资加密货币的简要提示

翻译/编辑：Rianna

- ![](https://assets.gnews.org/wp-content/uploads/2022/04/a1649502001496.jpg)图片来源：雅典娜设计组


作为一个不受监管的市场，关于谁可以购买加密货币以及他们可以购买多少的规则很少。对于澳大利亚投资者来说，如果下一届政府实施去年布拉格加密货币政策报告的建议，关于加密货币交易所及其使用的法规可能会出台。

但目前的情况是，散户投资者（即净资产低于250万美元或年收入低于25万美元的投资者）投资加密资产的主要方式是通过不受监管的加密货币交易平台或交易所直接购买单位。在提供一些基本身份证明后，他们可以在几分钟内做到这一点。大多数交易所不仅可以访问比特币和以太这样的大盘代币，还可以访问数百个“alt-coins”。

近年来出现了一些本地加密专业交易所，包括（但不限于）BTC Markets、SwyftX和Kraken。一些股票市场经纪人还宣布了允许加密货币交易的计划，包括英联邦银行的CommSec、SelfWealth、Stake和Superhero，但还没有人执行此功能。澳大利亚人还可以进入许多离岸交易所。

通过不受监管的交易所购买加密资产直接单位的投资者将面临棘手的存储问题。他们通常会将资产保存在“热钱包”中，这是一个容易遭受网络犯罪的基于互联网的帐户，或者一个“冷钱包”，一种没有连接到互联网但有丢失、被盗或损坏风险的物理设备，或者澳大利亚的一些小额投资应用程序通过托管投资计划提供加密货币的风险敞口。ASX上市的Raiz Invest有一个选项，包括最多5%的比特币分配，而专注于加密的微型投资者Bamboo提供由比特币、以太、黄金和白银组成的投资组合。这些是针对低平衡的入门级投资者，如果客户愿意，他们一次只需投入几美元。

但符合复杂批发投资者测试的投资者（净资产超过250万美元或年收入超过25万美元的投资者）可以访问一些投资于一系列加密货币的传统管理基金。澳大利亚有越来越多的加密专家专业投资公司。他们通常会收取相对较高的费用，但在一系列法律义务中，他们有义务为单位持有人的最佳利益行事。

公司监管机构澳大利亚证券和投资委员会向投资于比特币和以太的交易所交易基金（ETF）表示认可，这些将提供给散户投资者。一些基金经理已申请在ASX或竞争对手Cboe Australia上列出这些产品，但他们尚未开始交易。

投资者还可以通过投资资产负债表上持有加密货币或参与加密代币挖掘和分销的股票来无意中进入市场的风险敞口。一些当地ETF提供了这些股票的风险敞口。

无论您选择哪种方法，专家都建议您“从小处开始，边做边学”。另一些人说，鉴于市场的风险和波动程度，你根本不应该投资加密资产。

投资者应该考虑他们的目标、风险舒适度，并考虑获得专业的财务建议。尽管后者有其自身的复杂性，但许多顾问无法根据他们的许可和专业保险安排条款适当讨论或推荐加密资产。但无论如何都要问问他们，或者自己做研究。
