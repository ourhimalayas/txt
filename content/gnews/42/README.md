美东时间7月1日文贵先生及其团队出现在纽约的新办公室里，GETTR创始人——美国著名媒体人——杰森·米勒（Jason Miller）先生，接受了文贵先生的采访。这个新办公室以前的主人是CNN，CNN的黯然退场和GETTR的即将推出，形成了鲜明的对比，也预示着历史即将翻开崭新的一页。采访中他们谈到了习近平的讲话，也谈到了共产党和跨国科技巨头已经严重侵蚀了西方世界的言论自由。

中共在百年冥诞之时，习近平的讲话公开对全世界爱好和平、民主、法治、自由的人们发起了挑战，文贵先生认为习近平在天安门城楼上的那番讲话和台下那些整齐划一的群众表演，立刻让人联想到了北朝鲜。他认为共产党这么做完全是与世界脱轨，完全是与全世界民主、法治的文明标准背道而驰。

习近平在讲话中完全不提中共病毒给全世界人民带来的生命威胁，对杀害全人类毫无悔意。更可怕的是，共产党已经做好了绑架全中国人民的准备。共产党这种完全不负责任、挑战世界、无视生命的流氓黑社会的本质，已经毫无忌惮的展现在了世人面前。共产党已经摘下了最后的遮羞布，完全失控，变成了人类公敌。

文贵先生认为在西方世界并没有很多人注意到，习近平在讲话中多次提到了两个百年的计划。前一百年，共产党要完全统治中国人民、统治亚洲，而后一百年共产党要输出共产主义、统治世界。习近平已经明确告诉了全世界，并且还强调：共产党是中国人民唯一的代表，共产党和中国人民不能被分开，他将用共产主义的管理模式来管理整个世界。文贵先生认为虽然西方世界对于马克思主义的认识比较深刻，但是对于共产党将要向全世界输出共产主义管理模式这件事情，认识得并不深刻。

在谈到中共时，米勒先生提到了共产主义对整个世界的危害，也提到了中共跟大科技巨头的狼狈为奸。毫无疑问共产主义就是邪恶的化身，会吞噬世界上所有人的自由。表现在社交媒体上，共产党跟大政府、威权政府相互勾结，跟大财团、巨型高科技垄断企业在经济利益上紧紧捆绑，在经济领域表现为国家资本主义，就像过去二战时期的法西斯治下的经济形态是一样的。

一个权威的小圈子整天告诉你应该说什么、应该做什么、应该过什么样的生活、应该想什么、不应该想什么，甚至连应该如何向上帝祈祷都是他们说了算。这正是中国老百姓甚至是全世界老百姓现在所面临的。像共产党这样的法西斯政权，他们压根就不在乎老百姓，他们只在乎金钱和权力。那些跟中共做生意的大科技公司，他们才不在乎中共如何剥削和压迫中国老百姓呢，他们只在乎能不能把自己的产品卖出去，只要能做生意，怎么都可以。

文贵先生认为在美国，媒体作为第四权力平衡两党政治的作用，已经完全丧失，支持保守力量的人们基本上是无法发声的。社交媒体、包括很多主流媒体都是一边倒的，人们希望有一个新的平台可以发声。GETTR这个社交媒体平台，短期的目标，就是为了让人们能自由表达自己的政治观点，可以不怕被封号、被禁声、或者被社交平台强制踢出。现在不仅仅是在美国，在巴西，在印度，在奈及利亚，支持保守力量的人们都处在被科技大佬们不同程度的迫害中。奥威尔的小说《1984》描述的情形正在现实中上演。那些心怀恶念的人，为了金钱和权力，要把世界上每一个人都玩弄于他们的股掌之间。

GETTR这个社交媒体平台就是为打破共产党和科技大佬的魔咒应运而生的。在接受文贵先生采访后，米勒先生还接受了福克斯的采访，7月2日米勒先生又做客班农先生战斗室，继续为GETTR的推出做最后的宣传。GETTR将在美国独立日7月4日当天隆重推出，从上午10点开始，该平台将正式开始用户注册以及各项运营活动，在GETTR上也可以看到川普总统的身影！
