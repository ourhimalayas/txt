#### 撰稿：日本东京方舟农场  幽蘭自得

![](https://assets.gnews.org/wp-content/uploads/2022/01/9B2D6A4D-D88A-4656-AB19-C37DBE3068DC.jpeg)图为英国外交大臣特拉斯（图片来自路透社）

1月21日，英国与澳大利亚举行外交与国防部长“2+2”会谈，在会谈前夕，两国表示，加强英澳之间防卫联系，是反制中共国愈加侵略态度的关键一环，两国将在网络领域“反击”中共国、俄罗斯利用科技破坏自由民主。

据《自由时报》报道，英国外交大臣特拉斯（Liz Truss）、国防大臣华勒斯（Ben Wallace）与澳大利亚外交部长潘恩（Marise Payne）、国防部长杜登（Peter Dutton）在澳大利亚举行的《澳大利亚—英国部长级咨商会议》（AUKMIN），意在巩固澳、英、美成立三方安全联盟（AUKUS）后的安全和贸易关系。

杜登表示，网络安全将是这次会议的一大重点，“英国与澳洲都时常受到来自俄罗斯、中共国、伊朗和其他国家的（网络）攻击”，两国会“反击”；潘恩20日与特拉斯签署印太基础建设投资协议后表示，澳大利亚和英国将协调网络制裁机制，强化吓阻、升高敌意国家在网络活动的成本，澳大利亚正致力于与英国这样的伙伴，一起挑战利用科技破坏自由和民主的恶意行为者。

特拉斯则在《澳洲人报》专访中指出，此行是为了建构一个“自由网络”，而澳大利亚是促成此自由民主国家联盟的重要一部分，两国将讨论如何反制中共国在印太地区的影响力，以及AUKUS成立后的战略移转等议题。她指出，俄罗斯、中共国已愈加具有侵略性，中共国的经济胁迫手段也变得更为猖獗，尤其是针对立陶宛和澳大利亚，英国将会对中、俄的侵略性态度发出声音。

据悉，英国也正致力与日本、印度、印度尼西亚等印太国家加强经济、防卫合作。英国将脱离欧盟后重心转向印度—太平洋地区，列为其外交和安全政策的重点；英国也一如澳大利亚，展现出愿意在台湾和香港人权等问题上对抗中共国。
