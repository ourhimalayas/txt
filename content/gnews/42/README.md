- 人权高级专员入疆调查维吾尔族人权状况的希望几无可能


墨博士点评，当前西方国家和国际组织对中共进行的施压有两个重点。其一，中共对少数民族的反人类罪以及人权问题；其二，病毒溯源及超限战的真相。对于前者，中共自认为已经拿捏到位，驾轻就熟了。因为中共早就发现，如果利益输送到位、“蓝金黄”到位，还有经济指标到位，西方国家的人权控制是可以随意扭曲与化解的。当中的手段包括假意邀请西方媒体去新疆调研，其间在新疆运用维稳力量以及各级政府的严控，做好沿途的一切安排，设定好路线和人员去蔽瞒调查人员。这就是为什么之前西方媒体掌握的中共在新疆和西藏的反人类罪行的证据，其实并不多也不实锤，大部分都是边角信息的主要原因。但是现在的国际形势已然转变，病毒溯源这把达摩克利斯之剑正高悬在了中共的头上。因此，中共在病毒溯源问题上非常小心，同时也在人权调查上开始担心——中共非常害怕人权问题将成为西方社会和美国全面调查中共的一个突破口。一旦人权状况可以被调查，这意味着病毒溯源和超限战一样可以被调查。因此，中共处理此事必定非常小心，一定不会同意存在有关这两个关键问题的任何调查。

- 中共百年党庆在即，崔天凯此刻离职归国不情不愿


从中共国驻美国大使崔天凯的离职信件中反映出来，实际上他并不想这么早地回到中共墙内。他在信中的这番话实际是说给习总跟党内高层听的，是想为自己开脱，是意图表达他自己已经为习总和中共尽了全力，并帮助中共保留了与美国通话的通道之意。中美在十字口之上的论调其实就是自欺欺人的瞎话，实际上，中美早就在一条决裂之路上了。崔天凯之所以有此论调，只是为其回去减免办事不力之罪的托词。他想传达给习和中共一个错误的信息，就是中美的关系在他的努力下仍然有回头的余地，中共仍然有勾兑的可能，他还是给未来驻美大使留下了一点空间的。墨博士认为，崔大使的这番话其实透露出他预感到自己回到中共墙内，想穿军装都不一定有什么机会了，穿囚服的几率应该会更高些。

- 燕山大学教授称，推翻爱因斯坦相对论之现象见怪不怪


墨博士觉得，中共国某些教授的研究真的跟科学没什么关系，其实更多的是迎合目前的政治需求而已，跟几十年前的文革一模一样。这一怪象已然验证了“文革2.0”的到来。中共国强烈的文革种种迹象显现之际，即是其国之将亡，妖孽四起，天下无道，心忧作怪之时。也可以说，习总加速师的成神之路，其实就是中共国全面踏向的地狱之路。文革时的大运动、大斗争、大动乱和大浩劫其实已经悄悄地开始，而且墨博士觉得，这一次可能比上一次文革更加惨烈，因为现如今中共手里已然拥有了比以前更加恐怖的统治武器、病毒、科技和监控，这些都是对中国老百姓生命财产安全真正的最大威胁。

- 白宫对伊朗新总统的反应体现了其对外工作的重心就在中共


墨博士觉得，美国对付中共的国际政策其实比较鲜明了。除了拉拢盟友对中共进行群殴模式以外，另一方面就是积攒和回收自己之前的部署和力量。也就是说，在中东、阿富汗和朝鲜问题上，其实美国并不想过多参与，而是想把力量收回。这些问题将尽量地交给盟友去解决，美国则是以提供技术和信息为主。这样美国就有能力整合现在所有的资源、军力、财力和政治。之所以这么做，说明美国已经看到了中共开启的超限战是对美国所有的国家力量的一个严重挑战，认识到了它必须集合所有的力量才能应对这个头号威胁。
