![](https://assets.gnews.org/wp-content/uploads/2022/04/截屏2022-03-22-上午10.53.46-1536x113-1.png)
【大纪元4月14日讯】据田纳西州纳什维尔媒体报道，州参议院4月六日通过第 2188 号法案，允许人们在药店柜台购买或通过药店分发伊维菌素用于包括COVID-19患者的治疗。该法案还规定，善意行事的药剂师或处方医生将被免于纪律处分和民事责任。参议员拉斯提˙克劳（Rusty Crowe）和弗兰克·尼斯利（ Frank Niceley）是该法案的两位发起人。截至周三，关于伊维菌素的这项立法尚未在州众议院通过。
![](https://assets.gnews.org/wp-content/uploads/2022/04/image002.png)
克劳和尼斯利就该立法发表声明说：“伊维菌素是众多治疗选择之一，已被证明可有效治疗 COVID-19”。

但该州在其网站上也表示“ 联邦食品药物管理局（FDA） 尚未授权或批准伊维菌素用于预防或治疗人类或动物感染 COVID-19”。美国国立卫生研究院在其网站上说：“没有足够的证据表明 COVID-19 治疗指南专家小组推荐或反对使用伊维菌素治疗 COVID-19。”
