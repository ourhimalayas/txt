## 韩国暂不考虑全面禁止外国人入境
` 首尔天池农场` [轉載自GNews](https://gnews.org/zh-hans/1704821/)

韩国首尔天池农场—–追着曙光跑

据韩国釜山日报11月29日的报道，韩国中央防疫对策本部宣布，暂时不考虑为防范中共病毒新变异型“Omicron毒株”而限制所有外国人入境的方案。
![](https://assets.gnews.org/wp-content/uploads/2021/11/WhatsApp-Image-2021-11-29-at-19.58.55.jpeg)图片来源：原新闻
韩国中央防疫对策本部海外出入境管理组组长金柱心（音译）29日在记者会上表示，“目前没有考虑禁止所有外国人入境。”

为防止中共病毒新变异型“Omicron毒株”进入韩国，韩政府前一天28日0时起全面禁止来自最早发现“Omicron毒株”的南非，及其邻国博茨瓦纳、津巴布韦、纳米比亚、莱索托、埃斯瓦蒂尼、莫桑比克、马拉维共8个非洲国家的外籍人员入境。

金柱心（音译）补充说，如果 “Omicron毒株” 确诊病例发生在这八个国家以外的国家，他们不会考虑将这些国家纳入入境禁令名单。但是，会继续监测感染病例等情况，并观察扩散风险趋势，在必要时增加禁止入境的国家名单。
