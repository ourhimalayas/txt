**2.****英国银行寻求偿还1050亿美元的****中共病毒****贷款**

[随着逾750亿英镑(1,040亿美元)的国家担保紧急贷款回归时间临近，英国银行在应对疫情期间必须小心翼翼地对待企业。试图限制自身和纳税人的损失，但避免重蹈2008年金融危机的覆辙，当时银行被迫支付数百万英镑用于大规模贷款偿还策略。贷方承诺这次会有所不同。由于首次偿还中共病毒贷款，英国四大银行共聘用了750多名收债专家，并正在提供如何敏感处理客户的培训。巴克莱(Barclays)商业银行主管汉娜•巴纳德(Hannah Barnard)表示:“我们进行了训练营培训，以确保他们准备好了。”](https://businesshala.com/empathy-bootcamp-uk-banks-seek-payback-on-105-bln-covid-loans/)

[随着首批主要市场之一开始从疫情中获得政府支持的贷款，全球将关注英国银行的表现。政府的初步估计是，考虑到信贷问题和欺诈，最受欢迎的小企业能够借入高达 50,000 英镑的反弹贷款计划的损失可能高达 60%。虽然政府为贷款提供100%或80%的担保，这限制了银行潜在的财务痛苦，但这些贷款必须由国家规定在偿还前收取，一些银行家表示，这些成本意味着它们可能会造成总体损失。](https://businesshala.com/empathy-bootcamp-uk-banks-seek-payback-on-105-bln-covid-loans/)

**4.****全球多达1,500家企业受到日前勒索软件攻击的影响**

[美国信息技术公司Kaseya的执行长(CEO)周一表示，全球有800至1500家企业受到了日前以Kaseya为主要目标的勒索软件攻击的影响。总部位于佛罗里达州的这家IT公司的执行长Fred Voccola在接受采访时说，很难估计周五攻击事件的确切影响，因为被攻击的主要是Kaseya客户的客户。Kaseya为IT外包服务商提供软件工具：这些服务商通常为规模太小或资源有限而没有自己的技术部门的公司处理后台工作。](https://in.finance.yahoo.com/news/1-500-businesses-affected-ransomware-225028129.html)

[这些工具之一在周五被攻破，使黑客能够使五大洲的数百家企业陷入瘫痪。虽然大多数受影响的是如牙医诊所等小企业，但瑞典和新西兰受到的影响尤为严重，前者有数百家超市因收银机无法使用而不得不关闭，后者有学校和幼儿园被迫关闭计算机。声称对此次攻击行动负责的黑客索要7,000万美元，尽管他们在与一位网络安全专家及路透的私下谈话中表示愿意降低赎金要求。](https://in.finance.yahoo.com/news/1-500-businesses-affected-ransomware-225028129.html)

[黑客的一名代表周一早些时候告诉路透：“我们随时准备进行谈判。”Voccola拒绝说他是否准备接受黑客的提议。根据网络安全公司ESET发表的研究报告，大约有十几个不同的国家受到了这次攻击的影响。周日，白宫表示，美国政府正在“基于国家风险评估”，向勒索软件的受害者提供援助。](https://in.finance.yahoo.com/news/1-500-businesses-affected-ransomware-225028129.html)

**5.** **空中出租车初创公司Volocopter获得关键生产认证**

[德国飞行出租车初创公司Volocopter周二表示，将收购长期合作伙伴DG 飞机制造公司，以确保其符合欧盟航空安全监管机构的生产标准。此次收购的条款尚未披露，这意味着Volocopter除了现有的欧洲航空安全局设计批准外，现在还获得了符合欧盟航空安全局规定的生产组织批准。Volocopter在一份声明中表示:“Volocopter是目前第一家也是唯一一家电动垂直起降公司，该公司既获得了设计和生产机构的批准，可以将其飞机推进商业发射。”Volocopter希望在2024年巴黎奥运会前将其双座飞行出租车投入常规服务，这款出租车看起来像一个超大的无人机。该公司还在开发一种类似的货运无人机。](https://money.usnews.com/investing/news/articles/2021-07-06/air-taxi-startup-volocopter-gains-key-production-certification)

**6.****油价接近三年高点**

[在全球主要产油国未能就生产计划达成一致后，油价周二逼近3年高点，拖累石油货币和债券收益率。欧洲股市因通胀加速的前景而震荡。中国也曾以另一场高调的打击行动吓坏了科技行业。但是主要的行动都是围绕着黑色的东西。石油输出国组织(OPEC)及其盟友OPEC+周一被迫放弃谈判，此前阿联酋拒绝了将产量限制延长8个月的提议。一些OPEC+消息人士表示，8月不会增产，而其他人则表示，未来几天将召开一次新的会议，他们认为8月将增产。原油交易员们并不是为了寻找答案。他们将布伦特原油价格推高至77.66美元，为2018年10月以来的最高水平，美国原油价格达到2014年底以来的最高水平，为76.90美元/桶。今年油价上涨了约50%，自去年因新冠肺炎疫情而出现暴跌以来，油价上涨了385%以上。UniCredit分析师表示:“如果未来几周不增加一些原油供应，鉴于市场紧张，布伦特原油可能会突破80美元/桶的关口。”](https://money.usnews.com/investing/news/articles/2021-07-05/asia-opens-higher-but-china-tech-worries-weigh-on-hong-kong)

**7.****本田、日产6月份在****中共国****销量大幅下滑**

[日本汽车制造商本田汽车和日产汽车6月份在中国的销量大幅下滑，中国是全球最大的汽车市场。本田6月份在中共国销量为118168辆，较上年同期下降17%。日产在一份声明中说，上月在华销量为114,605辆，同比下降16.3%。中国汽车工业协会(CAAM)周一表示，预计6月份中国汽车销量将达到193万辆，较上年同期下降16.3%。另外，只公布中国季度销量的通用汽车公司说，4月至6月间在华销量超过75万辆，较去年同期增长5.2%。](https://money.usnews.com/investing/news/articles/2021-07-06/honda-nissan-saw-china-sales-tumble-in-june)

**8.****欧元区5月零售销售反弹超过预期**

[周二公布的数据显示，欧元区5月零售销售增幅高于预期，4月为下滑，主要受非食品和汽车燃料购买推动。欧盟统计局说，欧元区19个国家5月份零售额环比增长4.6%，较上年同期增长9.0%。欧盟统计局周二说，4月份零售额环比下降3.9%，但同比增长23.3%。欧盟统计局表示，随着欧盟解除对冠状病毒的限制，欧元区经济出现反弹，非食品类产品(不包括汽车燃料)的销售在4月份暴跌6.1%之后，5月份环比增长8.8%，同比增长14.8%。汽车燃油销售环比增长8.1%，同比增长28.4%。食品、饮料和烟草的购买量基本稳定，环比略微下降0.2%，同比上升0.1%。](https://money.usnews.com/investing/news/articles/2021-07-06/euro-zone-retail-sales-rebound-more-than-expected-in-may)
