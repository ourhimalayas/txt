## 澳大利亚政治家参加坎培拉抗议活动以声援加拿大卡车司机

![](https://assets.gnews.org/wp-content/uploads/2022/02/11129609.jpg)2022年2月5日，在澳大利亚坎培拉旧议会大厦外举行的反疫苗接种集会上，抗议者们举着旗帜。(图源：奥联社)
《大纪元时报》2022年2月4日报道，澳大利亚政治家乔治·克里斯坦森(George Christensen)议员呼吁结束中共病毒疫苗强制令，同时参加在该国首都举行的“前往坎培拉的车队”抗议活动，以声援加拿大和美国卡车司机的车队。

在他在脸书上直播的一段视频中，这位经常因为其对中共病毒治疗、大科技公司审查制度和疫苗强制令的看法而受到诽谤的保守派政治家出现在成千上万不同的人群中，在旧议会大厦外一起呼吁结束这场“闹剧”。

克里斯坦森在2月5日说：“这里确实有成千上万来自各行各业的人。你知道，他们声称我们是边缘居民，但看看这里的人。看看那些爸爸妈妈们。他们有一线工人，有警察、医生、护士、飞行员——他们都因为这些疫苗强制令而失去了他们的工作。”

“他们所希望的只是结束这场闹剧。因此，今天这将是一次非常和平的抗议活动。传达的信息是：我们希望恢复我们的自由。这一开始就不是任何人可以夺走的。”他说：“我们希望恢复我们的自由。我们希望这些疫苗强制令结束。我们希望结束医疗隔离；包括封锁、州边界关闭等等所有的限制。”

“适可而止吧。现在是时候了。澳大利亚的人们–他们变得非常、非常愤怒。”他补充道。社交媒体上充斥着来自抗议者和支持者的视频和图片，他们从全国各地驱车来到坎培拉，以声援“前往渥太华的车队”。

其他人穿着军装并举着牌子，其中一个牌子上写着：“从什么时候开始，两个星期的时间就可以让(疫情)曲线变平，并同意加入世界上最大的医学实验。” 人们高呼“澳大利亚人(Aussie)，澳大利亚人，澳大利亚人，嗨(oi)，嗨，嗨。”并唱着“我们不需要接种疫苗。” 抗议者和支持者在社交媒体上分享信息，鼓励非暴力行为，并就如何向警察表明他们没有参与任何“有偿”的捣乱行为提供建议。

“你可以选择永远生活在恐惧中，或者永远生活在自由中。”一位曾在澳大利亚为从越战归来的士兵担任厨师的退伍老兵在坎培拉告诉“澳大利亚转折点” (Turning Point Australia)组织。在加拿大，该运动见证了成千上万的卡车司机动员起来，以应对正在进行的中共病毒疫苗强制令和政府规定的严苛限制。

在澳大利亚，疫苗强制令已经在全国范围内被广泛实施，并在政治上得到了两党的大力支持以及商界和医疗界的支持。然而，这仍然是一个有争议的问题。
