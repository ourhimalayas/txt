——从美联储看喜联储

**撰稿：像风一样自由**
![](https://assets.gnews.org/wp-content/uploads/2021/12/Screenshot-2021-12-05-220243.png)
美联储是全球经济金融最重要的机构之一。说美联储咳嗽，美国和全球经济、金融体系就会感冒，这个比喻可能并不夸张。2021年11月1日喜币上市，喜联储（Himalaica Reserve）这个崭新的名词也正式出现在全球的视野中。那么喜联储未来能不能像美联储那样，在全球经济金融中发挥呼风唤雨、举足轻重的作用呢？对比美联储成长的历史进程，喜联储很可能以新的王者身份登上全球经济金融的舞台。

首先，从喜联储和美联储产生的历史时机来来看，两者有着惊人的相似性。美联储的英文：Federal Reserve System (Fed)，即美国联邦储备系统。它是美国的中央银行体系，依据美国国会通过的1913年《联邦储备法案》而创设。1907年全球第一次金融危机的爆发，直接促使后来的美国联邦政府建立了美联储体系，以避免再度发生类似的银行危机。美联储就是在这样的历史危机时刻而诞生的，进而发展、完善、成熟的。而喜联储的出现也极具历史相似性。目前，全球法定货币严重超发，全球的经济、金融体系的危机爆发在即，或者说正在爆发。在全球经济金融变革和洗牌的关键时刻，数字货币正以崭新的姿态登上历史舞台。喜联储作为数字中央银行体系登上世界金融的风口浪尖，正如同当年的美联储一样，成为如今的全球经济金融问题化危为机的一把利剑。那美联储这个时候为什么不能发挥当年力挽狂澜的作用呢？这大概也是历史的轮回。所谓时势造英雄，喜联储可能就是那个即将拯救全球经济金融的那个新的王者。

其次，从组织的独立性来看，喜联储不仅有着类似美联储的独立性，而且有着自身的独特性。美联储的组织结构设计是全世界最复杂的，这是因为其制度设计的初衷是为了使美联储能够保持一定独立性，不被个别集团控制，因为某个集团不可能在所有职位都安插自己的人马，所以能够相对公正地代表各个地区利益集团的利益博弈。喜联储注册在巴拿马，没有在美国注册，恰恰避开了复杂监管体系的制约，这也是喜联储独立性的体现。喜联储没有美联储组织结构那么复杂，或许正是它的优势。美联储如今在实际运行中存在的问题和毛病，可能正是年轻的喜联储可以避免，这也成了它先天的优势。

此外，从世界的认可度看，喜联储的发展潜能很可能得到类似美联储的全球认可度，从而确立自己的金融“江湖”地位。全球范围内，美元是被世界范围认可度最高的货币之一，它也是流通性最高的货币之一。几天前，文贵先生透露，金融实力排名欧洲前三的某国将为喜联储颁发数字银行牌照。在美国的数字银行收购也正在进行当中。这些意味着喜联储作为中央银行开始向数字商业银行领域开始进发。喜联储的体系建设已经迈出实质性的步伐。喜联储可以说真正拉开自身金融体系建设的历史大幕。未来可能会有更多这样的商业银行机构加入喜联储，当世界的认可程度达到一定的水准，世界一定会看到喜联储——这个名字不是徒有虚名，而是实至名归。

文贵先生不久前讲到危中化机的理念和“江湖”这个概念。如果把国际经济金融体系当作一个“江湖”，即将出现的世界经济金融危机可能是这个“江湖”重新洗牌的历史时刻。在这场人类历史上前所未有的危机中，喜联储可能就如同当年的美联储，从历史危机中诞生，进而发展壮大，成为化解世界经济金融危机的钥匙。王者才能笑傲江，喜联储可能就是那个崭新的时代王者。
