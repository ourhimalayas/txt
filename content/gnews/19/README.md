## 普京下令在乌克兰开展军事行动，要求投降

![](https://assets.gnews.org/wp-content/uploads/2022/02/23ukraine-briefing9-master1050-edited.jpg)POOL PHOTO BY SERGEY GUNEEV/EPA, VIA SHUTTERSTOCK
官方和媒体说，俄罗斯军队星期四向乌克兰的一些城市发射了导弹，并在乌克兰海岸登陆了部队，此前普京总统授权在东部开展他所谓的特别军事行动。

在普京在俄罗斯国家电台发表电视讲话之后，在乌克兰首都基辅可以听到黎明前的宁静中的爆炸声。

国际文传电讯社说，首都主要机场附近枪声大作，城市上空传来警笛声。

乌克兰外交部长德米特罗-库莱巴在推特上说，普京已经发动了对乌克兰的全面入侵，和平的乌克兰城市处于被攻击中。

美国总统拜登，对美国几周前就预测到的入侵作出反应，说这是侵略战争。乌克兰将会保卫它自己并且会赢，世界将能也必须阻击普京。必须现在行动。说他为乌克兰人民祈祷，“现在他们正在遭受一场无端和无理的攻击”，同时承诺以严厉的制裁进行回击。

拜登在一项声明中说他将会和G7的领导们和美国的盟友们和合作伙伴给俄罗斯施加严厉的制裁。

俄罗斯要求北约停止东扩，同时普京重申其立场，即乌克兰加入美国领导的大西洋军事联盟是不可接受的。

他说他已经授权了军事行动，在俄罗斯没有任何选择的情况下，只有通过保卫自己来抵抗他所说的来自现代乌克兰，即拥有4400万人民的民主国家的威胁。

俄罗斯军事行为的全部范围现在不是很清楚，但是普京说他们的计划没有包括对乌克兰领导的占领，他们不打算用武力强加任何东西。

在纽约举行的联合国安理会的紧急会议上，普京说他已经要求俄罗斯武力去保护人民并呼吁乌克兰军队去放下他们的武器。

乌克兰总统沃洛季米尔-泽伦斯基说，说俄罗斯已经在乌克兰基础设施和边防部队实施了导弹袭击，且在很多城市可以听到爆炸声。一位官员还报告了不间断的网络攻击。

泽伦斯基说，已经宣布戒严，且他已经向世界领导们呼吁施加所有可能的制裁于俄罗斯包括想要破坏整个乌克兰的普京。

在普京下达命令数小时后，俄罗斯媒体报道俄罗斯国防部说它们已经拔掉了乌克兰空军基地的基础设施并降低了其防空能力。

早前，乌克兰媒体报道在基辅的军事指挥中心和在东北部哈尔科夫市已经被导弹击中，同时俄罗斯则在南部港口城市敖德萨和马里乌波尔登陆。

路透社的一名目击者后来在马里乌波尔听到三声巨大的爆炸声。据俄新社报道，东部由俄罗斯支持的分离主义者后来说，他们已经占领了两个城镇。乌克兰当局没有立即发表评论。

在侵略开始后的数小时，分离主义者向莫斯科发出请求，要求帮助阻止所谓的乌克兰侵略–美国将这种说法视为俄罗斯的宣传。

全球市场和美国债券收益率跳水，美国同时美元和黄金火箭般走高在普京讲话后。布伦特石油窜至100美元/桶自从2014年以来第一次。

**‘决定性的方式’**

成群结队的人等着转账和购买食物和水必需品在基辅。同时交通在往西通往波兰边境城市发生了阻塞。

已排除在乌克兰实地部署美国军队的拜登说，普京选择了一场有预谋的战争，将带来灾难性的生命损失和人类痛苦。

拜登在与泽伦斯基的通话中表示俄罗斯应独自为此次袭击带来的死亡和破坏负责，同时美国和他的盟友和合作伙伴将以一个团结和决定性的的方式进行反击。

法国总统马克龙谴责俄罗斯的行为同时北约秘书长延斯-斯托尔滕贝格说，北约盟国将开会处理俄罗斯 “鲁莽和无端攻击 “的后果。

联合国秘书长安东尼奥-古特雷斯在安理会会议后发言，在最后一刻恳请普京 “以人类的名义 “停止战争。

**中国再次呼吁各方保持克制，并拒绝了一名外国记者将俄罗斯的行动描述为侵略的说法。**

乌克兰以安全风险高为由关闭了其空域，而欧洲的航空监管机构则警告说，在俄罗斯和白俄罗斯的接壤地区飞行有危险。

**周一，普京承认东部分离主义地区为独立地区，并下令部署他所谓的维和人员，此举被西方称为侵略的开始，此后，炮击愈演愈烈。**

为了反击普京周一的宣言，西方国家和日本对俄罗斯银行和个人施加了制裁，但在侵略开始之前暂不采取最严厉的措施。
