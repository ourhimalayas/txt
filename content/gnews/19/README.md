## 在俄罗斯入侵乌克兰期间，Meta如何摸索宣传节制

![](https://assets.gnews.org/wp-content/uploads/2022/04/xin_png.001-5.jpg)



![](https://assets.gnews.org/wp-content/uploads/2022/04/1-213.jpg)在这幅2022年2月26日拍摄的插图中，Meta标志被放置在一面俄罗斯国旗上。REUTERS/Dado Ruvic/插图

4月11日（路透社）–在3月9日乌克兰城市马里乌波尔的一家妇幼医院被炸的几天后，声称袭击事件从未发生的评论开始涌入代表应用程序所有者Meta

乌克兰总统沃洛基米尔-泽伦斯基（Volodymyr Zelenskiy）公开表示，这次轰炸至少杀死三个人，包括一个小孩。留着血的大肚子妇女在废墟中逃跑，她们的双手抱着肚子的画面立即引发了全世界的愤怒。

其中一个最受关注的妇女是乌克兰时尚和美容影响者玛丽安娜-维舍吉尔斯卡娅。她的穿着圆点睡衣在医院楼梯间穿梭的照片在袭击后广泛被流传，被美联社摄影师所拍摄。

根据两位在Facebook和Instagram直接管理冲突内容的承包商说，对这位准妈妈的在线支持表达迅速变成对她的Instagram帐号的攻击。他们以保密协议为由，禁止公开他们讨论自己的工作，所以以匿名方式接受了路透社的采访。

涉及到这位美丽影响者的案例只是Meta的内容政策以及执行机制在乌克兰入侵期间使亲俄宣传得以实现的一个例子。

俄罗斯官方抓住了这些图片，将他们和她的光鲜的INS照片放在一起试图说明观看者这次袭击是虚构的。在国有电视和社交媒体，以及在联合国安理会会议厅，莫斯科错误地指称维舍吉尔斯卡娅在乌克兰军队精心策划的骗局中化了妆以及穿了多套衣服。

主持人说，成群结队指责这位影响者的两面派以及演员身份的评论出现在她摆出化妆管的INS旧帖子下面。

该人士回忆说，在攻击的高峰期，包括对该女子的虚假指控的评论占了一位版主内容队列中的大部分内容，该队列通常会包含涉嫌违反Meta无数政策的帖子。

主持人告诉路透社，“这些帖子很卑鄙”，而且似乎是精心策划的，但是他们大多数没有违反公司的规则，这位人士称，因为他们没有直接提到攻击。“我不能再对他们做些什么”。

路透社无法联系到维舍吉尔斯卡娅。

Meta公司拒绝对它涉及到Vishegirskaya的活动发表评论，但在给路透社的一篇声明中，它说多个团队正在处理这一问题。

声明说“我们有单独和专业的队伍，和外部合作伙伴来审查错误信息和不真实的行为，我们一直在应用我们的政策来有力地打击这种活动，在整个战争期间。”

Meta政策主管尼克-克莱格周三单独告诉记者，该公司正在考虑新的措施以解决来自俄罗斯政府网页的错误信息和骗局，但没有详细说明。

俄罗斯数字发展、通信和大众传媒部以及克里姆林宫没有对评论请求作出回应。

乌克兰代表没有对评论请求做出回应

**‘政策的精神’**

在一个几百人组成的审核中心，这两名承包商是Meta公司在冲突中控制内容的战争中的步兵。他们是Meta公司与世界各地的外包公司签订合同以执行其规则的数万名低薪工人之一。

在入侵期间，这家科技巨头一直尝试将自己定位为负责任的网络言论管理者，俄罗斯将其称为用来解除和“剥夺”邻国武装的“特别行动”。

战争开始几天，Meta公司对俄罗斯国家媒体施加了制裁，同时取下了一个小型的协调假帐户网络，说它试图破坏对乌克兰政府的信任。

它后来说它已经撤下了另一个基于俄罗斯的网络，该网络错误地报告人们的仇恨言论或者欺凌等违法行为，同时击退了之前被禁用的网络重返平台的尝试。

同时，该公司尝试为该地区的用户开辟空间，用来表达他们对俄罗斯入侵的愤怒，并以Meta公司不允许的方式发出武装呼吁。

在乌克兰以及东欧及高加索地区的11其它国家，它对禁止仇恨言论，暴力威胁等的规则制定了一系列临时的“政策精神”豁免政策，根据路透社看到的对版主的Meta指示，这些改变旨在尊重这些政策的一般性原则而非字面意思。

例如，根据路透社看到的指示，它允许“针对俄罗斯士兵的非人性言论”和呼吁杀死俄罗斯总统弗拉基米尔－普京及其盟友白俄罗斯总统亚历山大－卢卡申科，除非这些呼吁被认为是可信的或包含其他目标。

3月10日路透社的一篇报道公开了Meta公司的分割情况，莫斯科方面对该公司提起了刑事诉讼，这些变化成为美达公司的一个热点。俄罗斯还在其境内禁止Facebook和Instagram，一家法院指责Meta从事“极端主义活动”。

路透社报道后，Meta公司收回了例外条款的内容。根据路透社审查的文件，Meta的公开声明以及对两位Meta工作人员，两位在欧洲的版主和另一地区处理英语内容的第三名版主的采访，它先是将这些例外情况限制在乌克兰，然后完全取消了这一项。

这些文件为了解Meta公司如何解释其政策（称之为社区标准）提供了一个罕见的视角。该公司表明它的系统是中立的以及基于规则的。

评论者说这往往是被动的，被商业考虑和新闻周期驱动，而不是原则驱动。在包括缅甸，叙利亚和埃塞尔比亚在内的其它全球冲突中，Meta公司也被这种抱怨所困扰。社交媒体研究员说这种做法允许该公司可以逃避对其政策如何其服务的36亿用户的责任。

正如纽约时报首次报道的那样，对乌克兰问题的指导意见的变化给版主带来了困惑和挫折感，他们说他们平均有90秒的时间来决定某个帖子是否违反了政策。路透社与三位版主独立证实了这种挫败感。

路透社3月10日报道了这些豁免后，Meta政策主管尼克-克莱格（Nick Clegg）第二天在一份声明中说，Meta将只允许在乌克兰发表此类言论。

两天后，克莱格告诉员工，根据路透社3月13日看到的一份公司内部文件，该公司将完全撤销允许用户呼吁杀死普京和卢卡申科的豁免。

文件显示，3月底，该公司将剩余的仅适用于乌克兰的豁免权延长至4月30日。路透社率先报道了这一延期，它允许乌克兰人继续从事某些类型的暴力和非人性的言论，而这些言论通常是被禁止的。

根据路透社看到的信息副本，在公司内部，一些Meta员工在内部社交媒体上写着，Facebook允许乌克兰人发表言论，而这些言论对于发布有关中东和世界其他地区以往冲突的用户来说，会被认为是越权的。

一位员工写道：“似乎这项政策是在说，如果仇恨言论和暴力针对的是‘正确’的人，那就可以。”

同时，这些人告诉路透社，Meta没有给这些版主任何的指引以提高他们禁用宣传有关俄罗斯入侵的虚拟叙述的帖子的能力，例如否认发生了平民死亡。

公司拒绝就其对版主的指导发表评论。

否认暴力悲剧的发生

理论上，Meta确实有一条规则，应该可以让版主处理那些对怀孕的美女影响者Vishegirskaya进行毫无根据的谩骂的暴徒们。据美联社报道，她在马里乌波尔医院爆炸案中幸存下来，并分娩了她的孩子。

Meta公司的骚扰政策禁止用户“发布有关暴力悲剧或暴力悲剧受害者的内容”，根据它自己网站上发布的社区规则。它在删除俄罗斯驻伦敦大使馆在3月9日的袭击发生后推崇有关马里乌波尔爆炸事件的虚假说法的帖子时引用了这一规则。

但是其中两位版主说，由于该规则的定义很窄，它只能被用于与随后发生的针对美女影响者的网上仇恨运动作斗争。

他们说，直接地指责爆炸是伪造的帖子有资格被删除，但如“你真是一个好演员”会被认为过于含糊，必须保留，虽然潜台词很明显。

他们补充说，假使指导意见能让Meta使得评论者能够考虑背景并且执行政策精神，将会有帮助。

Meta公司拒绝就该规则是否适用于Vishegirskaya账户上的评论发表评论。

同时，事实证明，即使是明确的帖子，Meta的执法系统也难以捉摸。

据以色列监督组织FakeReporter称，爆炸案发生一周后，俄罗斯大使馆的帖子版本仍在Facebook上至少8个俄罗斯官方账户上流传，包括其在丹麦、墨西哥和日本的大使馆。

其中一张照片显示，美联社在马里乌波尔的照片上贴了一个红色的“假”标签，文字上声称对维舍吉尔斯卡娅的袭击是一个骗局，同时指出读者在她的INS帐户上看到“500多条来自真实用户的评论”，谴责她参与了所谓的骗局。

Meta公司的一位发言人证实，Meta在3月16日移除这些帖子，在路透社向他们公司询问这些帖子的数小时后。Meta拒绝就为什么这些帖子能逃过他们自己的检测机制进行评论。

在接下来的一天，3月17日，Meta公司指定Vishegirskaya为“非自愿公众人物”，这意味着版本最终可以开始根据公司的欺凌和骚扰政策删除这些评论，他们告诉路透社。

但是改变，他们说，来得太迟了。与该女子相关的帖子的流量已经放缓到涓涓细流。
