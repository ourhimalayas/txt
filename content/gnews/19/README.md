## 达沃斯党就是世界的共产党
` GTV新闻访谈` [轉載自GNews](https://gnews.org/zh-hans/1890021/)

作者：日本方舟农场 – Andy5

![This image has an empty alt attribute; its file name is Screen-Shot-2021-08-30-at-7.37.59-PM.png](https://assets.gnews.org/wp-content/uploads/2021/08/Screen-Shot-2021-08-30-at-7.37.59-PM.png)

郭文贵先生在1月19日的大直播中指出，达沃斯党就是面向全世界的共产党，而中共就是中国境内的达沃斯集团。
![](https://assets.gnews.org/wp-content/uploads/2022/01/1-327.jpg)
郭先生解释道，近年来，世界上仅有为数不多的几个人可以参与达沃斯论坛。而对于近几年的西方世界而言，即使是美国总统或日本大财团都不在其中。这也导致了达沃斯的核心只有以中国共产党为首的极少数人，且这些核心无论身处何种党派，追根溯源都能找到与创立共产党的马克思、恩格斯等的联系。

郭先生也补充道，达沃斯党一直以来通过投资来影响文化娱乐界，将自己的价值观塑造为主流价值。其后，他们又通过影视剧等形式将此类价值观灌输给全球民众，进而能决定每个人的价值取向。郭先生认为，这不仅仅是洗脑，而是在一个人出生前就控制住了他的思想。

对此，郭先生列举了中共近年来的一系列操作：先是习近平在2017年声称要给全世界开方把脉的行为。随后，王毅代表中共，威胁世界“要打开潘多拉盒子”。最后就是持续至今的中共病毒疫情——这些举动绝非偶然，而是出于达沃斯党的计划。他们计划消除世界的部分人口，试图借由病毒继续奴役世界。

郭先生最后强调，人们对中共病毒疫情真相的探究一定会追究到达沃斯党的责任，而全世界对共产党的逐渐清楚的认识，将促成全球的大觉醒。
