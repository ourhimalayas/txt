希特勒当年为了让年轻孩子去入伍当兵，把军装做得尤其贴身好看。希特勒本人只有一米多，穿上军装感觉高大挺拔了许多。在对音乐的运用上，更是到了炉火纯青的地步，军队屠戮的性质配上音乐的节奏，再加上洗脑，让无辜老百姓或者家里的孩子就给变成了杀手，并且他们洋溢着崇拜，希特勒独裁是他们的信仰，犹太族人都应该被杀。杀害犹太人的行为，大部分是希特勒的党卫军或者是叫盖世太保干的，大部分是那些被漂亮军装蛊惑的孩子们，还有一些旧有的警察等，这就是名副其实的特务部门。习近平独裁拥有同样的部门，叫做国安部，直接听命于习，原来的所谓军队和公检法人员都不再拥有执法权，基本都是国家安全部直接执行习的抓捕命令。整个国家所有的权利都集中在习的一家人身上，那些常委全部都得往后排，当然公检法的地位比不上国安了。


![](https://assets.gnews.org/wp-content/uploads/2022/01/HHV2-hnknmqw2802272.jpg)希特勒军装 / 图源：网路
在古老中国的历史上，特务部门都是有专业的服饰或者暗号的，毕竟特务这项工作本来就是不能泄露机密的。希特勒的军装和音乐，迷惑了年轻人，让他们被深刻地洗脑，于他们自身来说，外表的好看和丰厚的报酬，足以让同龄的其他孩子羡慕。毛贼东时代也有类似的服饰，红卫兵的蓝绿军装，要宽大不合体，把人的线条全部都盖住，上衣不像上衣，裤子不像裤子，人也不像个人样，斗天斗地斗人，全部回到原始社会。这成为共产主义的传统，那些想要对共产党效忠的白手套们，如马云马化腾等，还有HBO的杨贝贝，只要穿上这个衣服拍照就行了，这形成了共产党的扭曲文化。包括很多现在读书的小孩们，很多人要穿这个衣服演节目，无论是幼儿园还是小学中学，几乎所有老师都要去排演这种抗日剧，让孩子接受共产党洗脑，服饰、道具是必须的。另外国内还有很多景点，都卖这种衣服，游人可以租用，穿着拍照上传互联网。还有很多照相馆里，比如婚纱摄影的，总少不了共产党红卫兵蓝绿军装主题。甚至有些地方的餐馆，也是采用的红卫兵主题。甚至国内互联网上兴起的一股红卫兵服饰cosplay风，年轻人根本不懂那是如何邪恶的东西，只是跟风穿着拍照而已。但是很明显，这种丑军装风格陡然变成了时尚，很可怕的就是，形成了一种文化，一种大家根本不知其邪恶真相却在处处传播的文化。
![](https://assets.gnews.org/wp-content/uploads/2022/01/9ca3-fypikws9058932-edited.jpg)红卫兵主题照 / 图源：网路
自从融入WTO以后，中共国的老百姓热切地和这个世界交流。西方舶来文化当然成为时尚的网红们争相试验的原始创意来源库。无论是音乐无论是电影更无论是服饰，直接抄袭是一种，乱七八糟地搭配也是一种，中国人在对文化的进化过程中，几乎是一下子从红卫兵蛮荒时代一步大跨越进了世界最前沿潮流时代，不用说对文化的理解了，就单纯是把一件衣服穿成合适合身的样子都非常艰难。好比一个小孩还没学会走路，一下子就要进行百米冲刺了。骨骼没有发育完全，肉体的爆发力更需要长时间的专业训练，什么都没有，但依然匍匐在跑道上，时不时还会有阵阵自我激励的喊声，“文化自信”。习近平时代大量地抄袭美国，几乎所有警察的制服都是照搬美国的，没有一点自己的服饰文化。谈到共产党，必然谈起的是毛时代红卫兵的衣服，习近平根本没有什么代表性的服饰。

共产党独裁靠的是对人的尊严的侮辱和吊打，尽管红卫兵的衣服邋遢滞后又不合身，还是有很多人去做红卫兵、红小兵。这是非常极端的做法，都能够让人瞬间就得到超出个人承受能力的荷尔蒙，换句话说，这种快乐会要人命。对他人的侮辱，在第一次得到满足的时候，再后来的给予，都必须是不断加码的，好比吸毒，第一次拿卡片吸到鼻子里，次数多了就没那么刺激了，需要用针管直接注射到体内。希特勒的很多军装都非常好看，音乐也非常隆重，那时候的人们对独裁的看法认知没有那么多，到了共产主义时期，尤其是中国共产党这里，用的都不再是那些东西，一件衣服只是表面的，不能再诱骗更多，必须进行的是人性的直接摧残。把尊严全都掠夺，衣服穿着再好看再精致还有什么用呢？

这成为共产党成功统治洗脑的利器，但同时也成为了杀死共产党的尖刀。能够让其强大的，必然会导致其灭亡。一个人对文化的理解，表现在方方面面，衣服是一方面，音乐也是一方面，并且这种自信是要有不断积淀的过程。共产党几乎把中国传统文化全阉割了，学俄罗斯学得不像，学英法更不像，学美国那就别说了，中国人站在世界上被共产党的恶意破坏规则全代表了。中国人，最具表现的是民国时代的服饰，自信就是底气，传统文化走过来的一步一步，沉淀在基因里，这种东西共产党没有，但他们可以在中国大地上横行统治那么多年，因为他们到处偷一点西方文化，嫁接到中国来，四不像总比一点没有要好。

习近平独裁不允许外来文化，口口声声要扶植传统文化。而传统文化的基因已经没有了，剩下的全部都是迷信，龙头改鸟头就是最好的见证。并且习所谓的传统文化，其实就是共产党的泯灭人性的暴行，并不是什么文化。红卫兵的衣服想要给全国人民每天都穿，对于已经接受西方舶来时尚文化的中国人来说是不可能的。习近平要效仿毛贼东，中国人的记忆里已经有一个太阳了，叫东方出了个毛贼东，习太阳不可能再成为太阳，他复制了毛，就相当于自己把自己杀死。和金家还有点不同，那是有血缘关系的一家人。

自己有多重要，做自己就有多难。文化都是表面，自信是持久平和。有一点点感悟关于七哥每次说的，你做你自己就行了。
