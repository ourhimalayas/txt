## 欧盟宣布对俄罗斯联邦实施第五轮制裁
` 日本東京方舟農場` [轉載自GNews](https://gnews.org/zh-hans/2297663/)

**撰稿：日本东京方舟农场 北极**

据焦点报4月5日报道，本周二欧盟对俄罗斯实施第五轮制裁，将影响俄罗斯联邦煤炭出口和其银行系统。

![](https://assets.gnews.org/wp-content/uploads/2022/04/图片-1-7.jpeg)欧洲联盟委员会主席乌尔苏拉·冯德莱恩（Ursula von der Leyen）。（图片来自焦点报）

欧洲联盟委员会主席乌尔苏拉·冯德莱恩（Ursula von der Leyen）指出，新的制裁措施是针对俄罗斯军方在布查地区犯下的罪行。制裁措施包括：

1.   禁止从俄罗斯进口煤炭；

2.   全面禁止四家俄罗斯主要银行的交易（这四家银行占俄罗斯银行业市场份额的23%），包括俄罗斯联邦第二大银行VTB；

3.   禁止俄罗斯船只进入欧盟国家的港口（人道主义援助例外），该禁令也适用于俄罗斯联邦和白俄罗斯的道路运输运营商；

4.   禁止向俄罗斯出口半导体、机械和运输设备;

5.   禁止进口俄罗斯商品，如海鲜，酒精饮料。

冯德莱恩称，欧盟不仅制定针对俄罗斯联邦个人的新制裁名单，还准备制定新的限制措施，特别是将影响俄罗斯石油的出口。
