来自《每日邮报》2022年3月15日的报道，据熟悉谈话内容的消息人士称，美国告诉盟友，俄罗斯在入侵乌克兰的几天后就曾向中共国请求提供武装作战无人机。最近几天，华盛顿加强了对中共的警告，称如果它提供武器或其他援助支持莫斯科，它将面临“严重后果”。

尽管在每天的简报中被反复问及,但白宫新闻秘书珍·普萨基(Jen Psaki)没有说明什么行动会触发乔·拜登总统的红线，也没有描述这些后果可能是什么。

周一上午，沙利文在罗马一家酒店会见了中共中央政治局委员杨洁篪。一位高级政府官员称这是一次“坦率的意见交流”，并说这次“紧张”的会晤计划已久。

上周，美国高级情报官员表示，他们认为，中共领导人可能担心，如果他们提供外交掩护，莫斯科的入侵以及弗拉基米尔·普京总统日益残暴的战术将对他们产生不利影响。

美国官员和盟友曾多次表示，如果中共站在俄罗斯一边，就有可能面临“二级制裁”。而在周日，他们声称普京已经向他的盟友请求为其踉踉跄跄的入侵提供军事援助。

俄罗斯周一否认它需要中共国的帮助。普京总统的发言人德米特里·佩斯科夫(Dmitry Peskov)说：“没有，俄罗斯自身有继续这项行动的潜力，正如我们所说的，这项行动正在按照计划展开，并将按时、全面完成。”

美国官员认为，普京期望在为期两天的闪电攻势中拿下乌克兰首都基辅。相反，几乎三周了，他的部队仍在与乌克兰军队作战，后者利用“打了就跑”的战术——以及包括反坦克榴弹在内的大量外国武器的涌入——达到了致命的效果。

中共国在联合国大会谴责俄罗斯入侵的投票中投了弃权票，而不是投反对票，官员们用 “战争”一词来描述莫斯科坚称的“特别军事行动”。

上周，美国中央情报局局长比尔·伯恩斯(Bill Burns)说，中共领导人习近平可能对战争的发展方式感到担忧，而且战争的丑陋性质有可能造成连带的罪责。

他说：“习主席可能有点不安，因为他看到普京总统促使美国人和欧洲人更紧密地联系在一起，强化了跨大西洋联盟，这在入侵开始之前是有点难以想象的。”
