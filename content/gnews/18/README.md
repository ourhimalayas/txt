12月23日，曾经市值接近300亿的银亿集团，以32亿的价格被梓禾瑾芯股权投资合伙企业接盘。ST银亿现在已进入《重整计划》执行阶段，如果期间不能执行或不执行，将被法院进行破产清算，终止上市。

银亿集团创立于1994年，熊续强是该集团创始人，该集团是一家以高端制造、能源、新材料、房地产开发和国内外贸易为主的综合性跨国企业，其旗下房地产开发年营业额超过百亿，2018年位列中国500强第215位，宁波市百强企业第3位。

在2018胡润百富榜上，银亿创始人熊续强还以295亿财富位列榜单第95位，为宁波首富，一年后熊续强身家暴降超过275亿，速跌出胡润榜，直到被梓禾瑾芯接盘之前更是面临破产。

根据天眼查资料显示：梓禾瑾芯成立于2020年，是一家股权投资合伙企业，赤骥控股集团此次接盘银亿将持有近60%股份，最终受益人为宁波籍商人叶骥。

叶骥曾于2007年在澳洲学习财务专业，后回国进入宁波城投集团股份有限公司，以及宁波市产城生态建设集团。后者原是国有企业，2007年改为股份合作制企业，叶骥的父亲叶礼诚任董事长兼总经理，直至2017年变更为叶骥。
