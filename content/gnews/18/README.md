据中共新华社11日消息，西安卫生健康委副主任张波在西安疫情防控工作新闻发布会上表示，此轮疫情防控，按照中共指示划分了封控区、管控区和防范区的三个等级。并宣布“全市疫情得到了有效控制。”这就是中共对疫情社会面迅速清零的纳粹化做法的结果。张波对此不忘记感谢“在广大市民的配合下”取得了这样一场人民内部战争的胜利。

- [**西安宣布人民内部战役胜利需满足的三个条件，划区封控者对该区老百姓有着绝对的生杀大权**](http://sn.news.cn/2022-01/11/c_1128251360.htm)**。**


该消息透露，按照划区调整责任要落实 “谁划定、谁调整”的原则，坚持梯次推进，由轻及重，由点到面，规范有序调整防控措施。”从“谁划定，谁调整”的原则看，划区者对该区域老百姓有着绝对的生杀大权，一般这样的好差事不会落入好人手里。因为，封控区、管控区的调整需要同时满足3个硬性条件：

- *确保近14天该区域内无新增病例或无症状感染者。*
- *确保该区域内最后1名密切接触者转运至集中营超过14天，且核酸检测为阴性；如集中营仍有新增的密切接触者，则解封时间重新计算。*
- *确保，该区域内所有人员解封前2天核酸筛查，均为阴性。*


该消息认为，管控区会根据疫情防控实际逐步调整等级，按照“解封不解防”的原则，实行“个人防护、少出不聚”，严格限制人员聚集，严格室内公共场所聚集。提倡线上办公，从严控制聚会、聚餐。对娱乐体育场所、培训机构实施暂停营业，市民非必要不离开本区县、开发区等封控措施意味着西安只是被强制解决了感染者的做法。

这些纳粹做法无论如何掩盖，也难以抹杀热心网友将1月6日西安发生的悲剧传播出来。正如喜马拉雅洛杉矶盘古农场（Himalaya\_Pangu1）[@Himalaya\_Pangu1](https://gettr.com/post/pmj6c2efaa)用户的视频消息可见，“中共，你把中国老百姓当人吗？”，“西安老百姓被隔离到停车场、行车隧道和看守所里。 天寒地冻，没热饭吃没热水喝，一群人抢一个厕所。 这里面有中共官员和家属吗？”

- [**按下西安，浮起了“河南”，河南省安阳最新数据显示，当地本轮疫情累计确诊已增至84例**](http://m.news.cn/2022-01/11/c_1128251776.htm)**。**


中共新华社11日援引“河南省安阳市委宣传部消息，自2022年1月8日首次报告确诊病例以来，安阳市连日出现新增病例。11日上午发布的最新数据显示，1月10日0时至11日8时，该市新增本土确诊病例58例（内黄县1例、汤阴县56例、文峰区1例）。1月8日以来累计报告确诊病例84例，均在定点医院隔离治疗。均无死亡报告。

河南在2021年经受特大水灾后，奥密克戎又一次入侵这个中共争夺的中原地带，据奥喜要闻[@ahnewsfeed](https://gettr.com/post/pndkob7dc7)盖特消息对河南疫情发出了疑问，称：10日河南新增确诊60例？安阳与天津属同一传播链？

另据推特网友“修行人”的说法，河南此轮疫情爆发前，河南地图就已经将即将爆发疫情区域标注成红色，很深的红色。遗憾的是地图不见了，可参考以下[该网友的图](https://twitter.com/ayanami789/status/1357891643911905280?s=20)。

xianghuang[@xianghuang13](https://twitter.com/xianghuang13/status/1480375215372734465)说：

“河南疫情的快速扩散，让我回想到几天前定中看到的河南地图，地图轮廓内是红色基调，中北部和中西部地区颜色偏深些。结合当前河南疫情中北部中西部日趋严重的情形，马上意识到，提前几天看到的原来是河南疫情地图呀。地图是整个河南省地图，或许不久河南疫情会扩散至全省范围”。

中共新华社11日援引天津疫情防控指挥部消息称，根据当前疫情形势，将天津市西青区王稳庄镇、大寺镇全域划定为管控区，实行封区管控措施。值得注意的是，[天津在清零时，小孩子同样被带去集中营（视频）](https://twitter.com/yuanzhtw/status/1480667275434856451)。西安模式已经被天津继承，乃至全国。

该消息透露，管控区内人员实行“人不出区、严禁聚集”，“按照统一安排，有序下楼到指定地点进行核酸检测。每户家庭每两天可派1人，在严格做好个人防护的前提下，在指定地点购买生活必需品。”这些对人民进行军事化管控即纳粹做法，全民实施家庭监狱计划。对所有出入人员落实“有序下楼”查证、验码、测温、登记。健康码赋橙码。”西青区疫情防控指挥部将立即对物资供给实施按需分配，“保障生活必需品供应。”
