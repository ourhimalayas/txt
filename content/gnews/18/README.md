## 2021年度中国结婚人口急剧下降
` NFSC News` [轉載自GNews](https://gnews.org/zh-hans/2037124/)

作者：台灣寶島農場—尋文客
![This image has an empty alt attribute; its file name is Screen-Shot-2021-08-30-at-7.37.59-PM.png](https://assets.gnews.org/wp-content/uploads/2021/08/Screen-Shot-2021-08-30-at-7.37.59-PM.png)
布赖特巴特新闻网2月18日援引《南华早报》的相关数据数据，揭示了中国近年来结婚人口呈现明显下降的趋势。
![](https://assets.gnews.org/wp-content/uploads/2022/02/截屏2022-02-20-上午9.22.40.png)
数据表明，2021年前三季度中共国婚姻登记数量较2019年同期下降17.5%。其中江苏省结婚人数连续五年下降，浙江省杭州市2021年结婚人数还不到2011年的80%。

与此同时，中国46.5%的结婚者年龄在30岁以上。结婚人口的下降、结婚者年龄偏大等现象将进一步降低中国的人口出生率。据悉，2021年中共国出生率已降至中共国家统计局1949年记录以来最低水平。

据社交媒体反映，民众普遍认为，近年来中共政权利令智昏，导致民间经济滞胀，高生活成本引发“结婚难”这一现象。去年以来，中共多年以来的盗国行为使得中国地方债、金融票据危机，房地产泡沫相继爆雷；2022年新春伊始，中共为延续其政权，不惜花费万亿民脂民膏承办一场冬奥会，妄图获取国际支持。2020中共病毒疫情爆发以来，中共有恃无恐的“人口红利”已不复存在，而今中共政权妄图通过三胎政策刺激民间生育，需面对中国人民“生育躺平”的现实。
