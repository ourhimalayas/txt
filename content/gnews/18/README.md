据[路透社](https://www.reuters.com/article/china-africa-xi-vaccine-donation-1130-idCNKBS2IF04W)11月30日的消息：习近平表示中（共）国将免除非洲最不发达国家2021年年底到期未还的政府间无息贷款债务；将向非洲金融机构提供100亿美元授信额度，并设立中非跨境人民币中心。

11月26日，国务院新闻办公室发布《新时代的中非合作》白皮书，并举行新闻发布会。外交部非洲司司长吴鹏介绍，中国支持减轻非洲国家的债务负担，积极落实20国集团暂缓最贫困国家债务偿付的倡议，在G20成员中，中国是缓债金额最大的，已经同19个非洲国家签署了缓债协议和共识。中国已宣布免除与中国有外交关系的非洲最不发达国家、重债穷国、内陆发展中国家、小岛屿国家，截止到2018年底到期未偿付的政府间的无息贷款。新冠肺炎疫情暴发以后，中国宣布免除15个非洲国家到2020年底的无息贷款。

根据[国家国际发展合作署官网](http://www.cidca.gov.cn/2021-11/26/c_1211463152.htm)，2013年至2018年中国对外援助金额为2702亿元人民币，其中对非洲国家的援助占比44.65%，包括无偿援助、无息贷款和优惠贷款【编者注：约为1206亿元人民币】。2000年至2020年，在非洲建成的公路铁路超过13000公里，建设了80多个大型电力设施，援建了130多个医疗设施、45个体育馆、170多所学校，为非洲培训各领域人才共计16万余名。

[据尼日利亚通讯社12月18日的报道](https://finance.sina.com.cn/jjxw/2021-12-20/doc-ikyakumx5202478.shtml)，尼日利亚债务管理办公室总干事欧尼亚表示，尼日利亚截至今年9月30日的存量债务总额为379亿美元，这一数字包括联邦政府、36个州政府和联邦首都直辖区的外债存量，从中国的贷款总额为35.9亿美元，占外债总额的9.47%。

[路透社](https://www.reuters.com/article/zambia-debt-idCNL6S2R402B)2021年10月8日的新闻显示，截止6月底赞比亚政府和国有企业欠中国的债务超过60亿美元。而中国非洲研究项目(CARI)上个月发布的一份研究报告称，赞比亚对中国贷款的债务为66亿美元，大约是该国前政府所披露数额的两倍。

另据[台湾媒体《科技新报》](https://technews.tw/2021/12/03/african-nations-mend-and-make-do-as-china-tightens-belt-and-road/)12月3日的报道，中共一带一路的投资2015年高峰期时曾投入1252.5亿美元。2017年中国投资非洲基础建设资金达110亿美元，至2020年仅剩33亿美元。2020年借给138个相关国家的资金为470亿美元，而在2019年则有将近1000亿美元。2021年一带一路上半年投资193亿美元，2020上半年则为275亿美元。

2020年5月19日，[《纽约时报》](https://cn.nytimes.com/business/20200519/china-loans-coronavirus-belt-road/)曾报道，巴基斯坦外长致电中共外长，提出“重组”数十亿美元贷款的请求。吉尔吉斯斯坦、斯里兰卡和许多非洲国家也向北京发出了类似的重组、推迟还款或免除今年到期的数百亿美元贷款的请求。在过去20年中，中共向各国注资数千亿美元。德国研究机构基尔研究所（Kiel Institute）估计中共向发展中国家发放的贷款达5200亿美元甚至更多。
