## 容易陷入低频牢笼的言行和念头
` 正道人` [轉載自GNews](https://gnews.org/zh-hans/2158897/)

作者：纽约香草山教育部 – 正道人

![This image has an empty alt attribute; its file name is Screen-Shot-2021-08-30-at-7.37.59-PM.png](https://assets.gnews.org/wp-content/uploads/2021/08/Screen-Shot-2021-08-30-at-7.37.59-PM.png)



正念、正思、正行



我们的念头和内心想法决定了当下人生的经历和遭遇，所以每个人都要为自己的言行和念力负责，毕竟今生的人间体验和来世的能量频率都和当下人生的所有念头有直接关系，无法逃避，因果守恒真实不虚。只有摒弃那些低频的想法和念头，才可以让正念当道，让自己不会陷入低频的陷阱，留下永远的遗憾。

究竟哪些念头会把我们带入低频的牢笼无法自拔？这里列出排在前7位的低频念头：

**（****1）恐惧：**所有恐惧的念头，不仅仅是对死亡和疾病的恐惧，也包括恐高，怕交际，不敢在众人面前演讲，害怕孤独，恐惧黑暗，担心失败等。恐惧本身就会让能量场扭曲，无法达到和谐自在，也会限制创造力，障碍人生的美好体验；

**（****2）愤怒：**愤怒是仅次于恐惧的低频能量，会让一个人无法从自我的心灵捆绑中走出来，人为地为自己设立一个“痛苦”牢笼，不断自残和伤害自己，却常常迁怒于人，把自己当成受害者；

**（****3）憎恨：**莫名的憎恨也是压倒很多迷失心灵的最后一根稻草，让自己无法原谅他人，其实根源是无法宽恕自己。憎恨的念头等于慢性自杀，长时间的憎恨念头会导致暴力和冲突，甚至产生同归于尽的想法，让自己的心灵忍受千刀万剐的折磨；

**（****4）委屈：**总感到委屈的心灵其实已经被低频能量充满，看到的都是外人的罪恶，执着于自我的判断模式中，陷入了“我委屈”、“不公平”的自我毁灭模式中，却无法自知；

**（****5）嫉妒：**妒忌的心是很多罪恶行为的起点，而且会一发而不可收，这种无可救药的低频想法会让一个人远离今生的快乐，也会和来世的喜乐彻底无缘；

**（****6）悲伤：**无法从悲伤中走出来的人，基本上无法有正能量的，因为悲伤的低频能量会彻底扭曲一个人的理性思考，也会把这种“受害者”、“悲惨者”的信念带到来世；

**（7）羞愧：**因为遗憾和悔恨导致的羞愧，可以让一个人永远失去自信心，也就彻底失去了希望，不仅对自己，也包括对人生与周围一切的人和事。
