据《纽约邮报》报道，近期欧洲各地爆发了针对中共病毒疫苗授权和封锁的抗议活动，目前欧洲出台了新的严格规定，措辞是“以遏制冬季的病毒浪潮”。对新措施感到愤怒的示威者聚集在奥地利、克罗地亚、意大利、北爱尔兰、法国领土瓜德罗普和荷兰。

周六在荷兰政府的部分封锁，发生了两晚的骚乱。一天前，警方在鹿特丹向抗议规定的人们开了枪。 据英国广播公司报道，在海牙，抗议者向警察投掷烟花，并点燃自行车，而警察则用马、警犬和警棍驱赶人群。警方说，一块石头从一辆载着一名病人的救护车的窗户里扔了出去。 官员们表示，至少五名警察在混战中受伤，其中一名膝盖受伤的警察被救护车带走。尽管该国70%以上的人已经完全接种了疫苗，但在病例出现创纪录激增后，该国上周末实施了为期三周的部分封锁，酒吧和餐馆等公用场所必须在晚上8点关门，禁止人群观看体育赛事。

在北爱尔兰，数百名反对疫苗护照的人聚集在贝尔法斯特市政厅外，那里的圣诞市场在周六开放，规定是必须有疫苗接种证明或者中共病毒检测呈阴性的证明才能进入。

在意大利，3000名抗议者出现在罗马古老的战车竞技场马克西姆斯广场，抗议当地工作场所、餐馆、电影院、剧院、体育场馆和健身房需要病毒的“绿色通行证”疫苗证书。 一条用意大利国旗的红、白、绿三种颜色写着的横幅上写“像我们这样的人从不放弃！”

周六，在奥地利政府宣布新的全国封锁并计划在2022年2月要求接种疫苗后，包括极右翼团体在内的数万人也加入了抗议活动。抗议者高呼：反抗! 并在封锁前在巨大的海尔登广场吹起了哨子。封锁将于周一开始，持续10天，也有可能会达到20天。人们只有在购买食品杂货、看医生或锻炼身体等情况下才能离开家。 周六晚上，示威者向警察投掷瓶子和啤酒罐，并燃放烟火，警察随后使用胡椒喷雾驱散人群。警方表示，有几名抗议者被拘留，但没有提供具体数字。

与此同时，在克罗地亚，数千人聚集在一起，手持旗帜和民族主义和宗教象征，抗议公共部门员工强制接种疫苗。

据瑞士公共广播电台SRF报道，在瑞士，2000人聚集在一起，反对即将举行的是否批准政府新冠肺炎限制法律的公投，称这歌举措充满了歧视性质。

法国海外领土之一的加勒比海瓜德罗普岛也爆发了暴力抗议活动。法国中共病毒健康通行证是进入餐馆、咖啡馆、文化场所和体育场馆的必备通行证，为了应对这一规定，抗议者实施了道路封锁，并点燃了汽车。 法国内政部长达尔马宁说，至少29人一夜之间被当局拘留。法国当局说，他们将向该岛增派约200名警官，并将于周二从下午6点至凌晨5点实施宵禁。

中共病毒疫苗导致人类社会彻底分裂，暴政有恃无恐，追求自由的人们绝不会妥协。
