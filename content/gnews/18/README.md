 法新社照片 / Ed Jones*
中共国最高检察院在11月5日发表声明宣布，现年52岁的原中共国公安部党委委员、副部长孙立军因涉嫌在其任职期间受贿而被捕。

孙立军于2018年3月成为公安部副部长，2020年5月被免职。在此一个月后，他正式受到中共中央纪律检查委员会的内部调查，并于9月被开除党籍。

最高检察院的声明补充说，对孙的逮捕，是反腐败监督机构国家监察委员会调查结束后进行的。但是，该声明并没有提供更多细节，只是最后说孙的案件“正在进一步处理”。

这份简短的声明留下了一个悬而未决的问题，即中共国当局是否也调查了孙在公安部任职之外的犯罪行为。孙曾是以侵犯法轮功信徒的人权而臭名昭著的610办公室的副主任，这个办公室是中共政权设立的一个类似盖世太保的机构。在他的政治生涯中，孙还曾是孟建柱的私人助手，孟建柱曾于2012年至2017年间担任中共国政法委书记。

孙的被捕正值中共国的政治敏感时期，因为中共中央委员会将于11月8日召开为期四天的六中全会。此次中央委员会全会的职能之一是确定由25人组成的中共国最高决策机构的人选，即政治局委员。
