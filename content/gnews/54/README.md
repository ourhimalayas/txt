## 新冠疫苗可能迫使病毒变异

翻译：Jenny

译文整理：

专家：开始提这个刺突蛋白时，我们发现了这些变体发生了被称为“病毒被迫变异选择”的现象。给观众举个例子，大多数人都听说过耐抗生素的细菌 。你可以给某人注射抗生素，但它不会有任何效果。因为细菌产生了耐药性。

如果你以美国大肠杆菌为例，这是一种存在于大多数人的大肠中的细菌。大肠杆菌的大部分对细菌没有抵抗力。但有些是，有些对细菌有抵抗力，同一类的抗生素会有不同的耐药菌群。所以当你处理问题时，给某人使用抗生素，它将杀死那些没有抗药性的细菌，是吧？但它会留下那些有耐药性的细菌，它们将成为族群，被称作原生菌群，处理这种情况的方法是，**停止使用抗生素，激发那些没有耐药性的细菌再次“茁壮成长”。**

那好，现在我们用这种方法来做疫苗，通过蛋白质来锁定HQ1（注：应该是初始病毒），我们知道，人体显然已经形成了一种免疫反应，尽管这是一种必要的直接免疫反应。如果你看一下感染数据，它干扰了T细胞，产生了问题并重启了适应性免疫系统，那么做什么了呢？它（刺突蛋白）绕开了HQ1部分。你能看得出刺突蛋白病毒与HQ1的刺突蛋白病毒相比差距很大，变异病毒才能更容易生存下来。

那么，我们第一次看到这种情况是什么时候？是去年在英国的Alpha变体病毒上首次看到了这一点。现在大家都忘了Delta病毒了。但开始是Alpha变体。然后Beta变体出现了，然后你知道，我们有多个变异病毒，越往后你越能够跟踪这一点。我就是这样知道的，所以你懂的。我一直在跟踪那些对这些变体和病毒进行基因型测序的人，以了解真正发生的情况。

**我已经在一些演讲中揭露并展示了所发生的情况，那就是随着我们连续接种疫苗，你可以看到，它不仅没有减少世界上任何国家的SARS Cov 2感染 的数量。反倒它已经从HQ1 变异到Alpha 到 Beta 再到Delta、Lambda 和 Mu，未来甚至到更多的变异。**

**我们人为的强制病毒选择了变异，我们给病毒施加压力，使它们选择朝着与HQ1更不相似的病毒去变异，事实证明了加强针并不能改变这种情况，因为那只是鼓励更多的“病毒压力选择”。**

我知道我不该笑，因为这是个令人担忧的问题。但我耻笑的不是那些令人担忧的打了疫苗的人，而是研发并推动加强针的人。他们看起来并不明白自己在做什么

**嗯，这很可悲，所以说我们应该用对待抗生素的方式。解决细菌耐药性的方法是要脱离抗生素。这就告诉我们，同样的方法撤回药物疫苗以停止“病毒被迫变异选择 ”。**

这样，HQ1和其它的就会恢复平衡，人们在人与人之间转移时产生的所有这些自然免疫力就会赶上，事实上，有一篇论文在三、四天前发表。这再次表明，我谈到了所有的部分，因此，冠状病毒病毒有刺突蛋白，它们有谷蛋白，它们有包膜，它们有膜。事实证明，人类的膜液体能做出了最好的抗体反应。因此，我们人类最善于保护自己从识别病毒到避免被攻击的部分根本就不在疫苗中。

这段视频最令人震惊的是结尾处，当主持人问原来的疫苗对这些变种病毒仍然有效，这种说法是假的吗？

答案：是假的。这就意味着旧疫苗对新的变异病毒有效性很低或者完全无效。

专家说“**如果你看一下数据，这些变体是如何与疫苗的耐药性有关联的，它再次表明，这些变体在刺突蛋白中出现得越多，它们对疫苗的反应就越小。辉瑞公司已经说过了，瞧，我们知道这将会发生。辉瑞首席执行官说，没关系，因为我们已经改变了一些东西。以便Pfizer可以在91天内，假如我们的疫苗对某些变异不起作用，我们可以制造一种新的疫苗。**

专家表示新的疫苗无需进行审批。而不是根据目前的规则和机制，持续的追加加强针，美国食品和药物管理局一开始就没有进行太多的审查。一切都在紧急使用授权（Emergency Use Authorization）保护下畅通无阻。

关于Moderna疫苗，专家爆料Moderna的疫苗主任，当他们在谈论加强针的小组讨论时居然说T细胞对免疫力并不重要。他表示作为modernna疫苗的主任。认为T细胞不重要，根本不配做那份工作。
