## 【中日财经爆】提高能源自给率 日本将启动国内油气田勘探项目
` 東京櫻花団-Tokyo Sakura Group` [轉載自GNews](https://gnews.org/zh-hans/1870854/)

撰稿：东京樱花团/麗子
![](https://assets.gnews.org/wp-content/uploads/2022/01/image-1347.png)[图片来源](https://www.inpex.co.jp/business/flow/development-production/)
17日，日本最大的油气开发公司INPEX宣布，将于3月开始在岛根县和山口县近海地区启动勘探项目。这将是日本近30年来首个海上油气田项目。

目前，在全球范围内原油、液化天然气（LNG）等能源价格飞涨，这个项目不仅有助于提高国内天然气自给率，从经济安全角度来看，本国内少量能源供给也很重要。

将要勘探的油气田位于距岛根县滨田市约130公里，距山口县萩市约150公里处，水深约240米。预计年产气量约为93万吨，相当于国内天然气消费量的1.2%，国內自给率将提高到3.4%，比目前水平提高1.1个百分点。

根据2011年三维物理勘测船的勘测结果和INPEX公司的调查结果，以及从2014年起进行了为期三年的钻探，成功地证实了该地点天然气的存在。

日本目前几乎所有的石油和天然气都依赖进口。然而，由于需要稳定的能源供应，内阁于去年10月批准的能源基本计划要求提高“独立开发比例”，2022年提高到60%以上。本次项目也将有助于实现这一目标。

虽然作为化石燃料，天然气的需求会随着脱碳而减少，但在燃烧过程中不排放二氧化碳且有广泛使用前景的氢气燃料来自天然气，所以本次项目也有望为国内制氢做出贡献。
