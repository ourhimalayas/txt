**1、擅自“割肉”导致巨亏，交易员向上市公司赔超5500万**
![擅自“割肉”导致巨亏交易员向上市公司赔超5500万_ 证券时报网](http://epaper.stcn.com/paper/zqsb/res/1/1/2022-01/28/A004/res01_attpic_brief.jpg)
期货交易员因擅自“割肉”向上市公司赔付超过5500万元，而公司在后续的账务处理时却摆了乌龙，引发了市场热议。昨日，上市公司金字火腿发布一则关于商品期货套期保值业务的进展公告。按照公告，去年8月底，该公司在生猪期货套期保值过程，在4个月份合约上买入，1个月后，由于市场不断下跌，该公司期货交易员擅自将持有的合约进行了平仓操作，导致公司旗下账户总计亏损5510.53万元。根据公司相关规章制度，金字火腿要求交易员承担全部责任。截至2021年9月30日，金字火腿收到期货交易员支付的5510.53万元的赔偿。但是，公司收到赔付资金后，对于投资损失和赔偿收入均未进行账务处理，而错将这笔赔偿款计入了收回期货本金。为此，金字火腿向投资者公开致歉。

**2、乐视案余波未了,中介机构遭调查，连带60多家IPO公司中招**
![乐视案余波未了？中介机构遭调查连带60多家IPO公司中招！_凤凰网](https://x0.ifengimg.com/ucms/2022_05/8785E09F29562F6CD440EF0C2398E6CFF9A94318_size26_w589_h345.jpg)
因中介机构被证监会立案调查，1月26日至今多家IPO企业中止审查，涉及12家科创板IPO企业、49家创业板IPO企业。终止原因看到，上述公司分别踩雷北京市金杜律师事务所（简称“金杜律师事务所”）、信永中和会计师事务所（简称“信永中和”）、中德证券有限责任公司（简称“中德证券”）等三家中介机构。据了解，中德证券被证监会立案事涉乐视网，今年1月18日，山西证券曾披露称，控股子公司中德证券因在乐视网2016年非公开发行股票项目中，保荐业务涉嫌违法违规，证监会决定对中德证券立案。对于金杜律师事务所、信永中和被立案的原因，网传也跟乐视网有关。Wind数据显示，三家涉事中介机构负责的IPO项目不止于此。这也是继2021年8月之后，A股市场再度出现IPO项目大面积中止的情形。

**3、成交价109亿，华融证券72%股权受让确定**
![中国华融以109亿元挂牌出让华融证券72%股份月底前或有结果_中国经济网——国家经济门户](https://img.cls.cn/images/20211203/19s61sUDZp.png)
1月27日晚间，中国华融在港交所公告称，本公司于北金所挂牌披露期结束后，征集到一名意向受让方国新资本有限公司（下称“国新资本”），于2022年1月27日，本公司与国新资本签署了《国有产权交易合同》。本次转让的对价为109亿元，转让完成后，中国华融不再持有华融证券股权。去年12月3日，中国华融称，拟转让华融证券71.99%股权，作价109亿元。

**5、白宫正在准备就加密货币发布行政命令**
![白宫将通过行政命令发布新的加密货币政策– 寻币吧](https://www.blockvalue.com/uploads/autoimage7/2022012501caqlfmp1vtw.jpg)
乔-拜登的白宫预计将在未来几周内发布一项关于美国政府将对数字资产采取行动的行政命令。一位“熟悉白宫计划”的消息人士告诉《巴伦周刊》(Barron’s)，该行政命令将以国家安全备忘录的形式发布。拜登的备忘录将指派一些政府实体研究加密货币、稳定币和非同质化代币(NFT)，目的是制定一个可行的监管框架。这位消息人士称：“这是为了全面审视数字资产，并制定一套政策，使政府在这一领域的努力保持一致性。”
