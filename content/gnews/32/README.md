在后面偷偷发笑的共产党，感觉自己又赢了，赢大发了。在对待台湾问题上，能让此届美国政府如此下跪，相信也只有千年一帝习大神了。也许习神真得认为“人定胜天”，认为搞定了一切不利于台湾大一统的人事条件，就能左右天意。亏习神还是这么迷信的人。天生气的时候，水汽会增多，人也一样，情绪高涨就会莫名出很多汗。中国古代常说“天人合一”，人的情绪会在天气里变化，天气的情绪到底是什么样的？（这里的天气，指得是极端天气，就是很多年没见到的，尤其造成了自然灾害的天气。像降水量达到十几年来的最高）。参考了一些历史，发现每次共产党要对台湾进行“大一统”的时候，老天总是会带来很多水汽。

- 1954年8月，共产党计划炮击金门，由于长江水患的原因无法调动兵力，暂时帮助台湾延缓了一些时间。但是雨季一过，9.22日共产党就开始炮击金门了


- 2021年7月，河南大暴雨，加上共产党内斗恶意泄洪，地铁和京广隧道泄洪淹死了众多无辜的老百姓，同时也淹掉了共产党的地下对台作战指挥中心


- 2021年9月，也是河南多地又是连日暴雨，洪涝。而在9.24日，共产党对台湾进行了实弹核弹演习，等同于开战


- 2021年10月，山西遭遇连日大暴雨，和河南比有过之而无不及，大雨冲垮了大桥，共产党泄洪又把老百姓泡在水里。也就是这几天，我们从文贵先生的直播中得知，台湾彻底被卖了，时时刻刻处在危险之中，也许某天一醒来，台湾突然遍地小粉红了，电视里都在宣传共产党解放了台湾，到处都是共产党的爪牙
