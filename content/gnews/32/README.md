![](https://assets.gnews.org/wp-content/uploads/2022/03/20220311.jpg)
连日来的战争阴云一直笼罩在乌克兰的上空，也蔓延至波兰的梅迪卡口岸。战争带来的摧残、伤亡、痛苦，也让上天飘下了雪的泪花。

梅迪卡口岸，一个连接生死和希望的地方，每天都有大量的乌克兰难民逃到此地，他们渴望平安。他们拖着疲惫不堪的身躯到达口岸后，多么希望能有个取暖的地方休息一会，再来一点吃的，一点喝的，多么希望有个天使来帮助他们。

可是每天的难民实在太多了，只有一点民间自发的力量，没有一个国际组织在这里帮助，实在是僧多粥少！

有道是战争无情人有情，在郭文贵先生带领下，新中国联邦和法治基金及时伸出温暖的双手，大卫、小Sara等战友们组成的第一支救援力量迅速赶到了梅迪卡，接下来第二批、第三批新中国联邦的战友到达前线，建立了最好的帐篷等硬件设施，还有最充足的毛毯、水果、食物，对难民进行帮助。五月花农场的大锅、文桑、chunhuajin、默默无闻等战友也到达了救援现场，我多么希望能像他们一样，也能代表五月花农场，去前线帮助那些难民和同胞们。

在救助过程中发生了许许多多感人的事情，有战友水土不服但仍然战斗，有的战斗到呕吐胃酸，有的战友累的瘦了好几斤，许多战友的脸被冷风吹的又红又黑，就像刀割一样，让人心疼。

最令我感动的是，一位外国记者采访文戈战友，问道：“你们在这里面临最大的难题是什么？”文戈回答道：“我们面临的最大问题是，我们提供了这么好的帮助，但我们很难将这些信息高效地发给所有需要帮助的人，如果可以的话，请把这些信息传播给所有需要帮助的人”。多么令人感动的回答，与中共相比，他们是能少管一个是一个，我们是能多救一个是一个。这一刻我感觉我们新中国联邦战友们就是那道温暖的光，用真诚、用善良、用无私照耀着每一个需要帮助的人。

感恩万佛万神，感恩文贵先生和所有战友们艰苦卓绝的付出！
