## 【聚焦澳洲】澳洲外交官看透战狼大国的玻璃心——中共国的“围城”心态

#### 校对/导读：守望黎明

图片来源：The Sydney Morning Herald

【导读】弗朗西斯-亚当森女士是澳洲资深外交官，她熟悉汉语并曾任澳洲驻华大使，是名副其实的中国通。在卸下外交部副部长的职务改任南澳州长的卸任演讲中，她畅所欲言，深刻剖析了中共国一方面强权在握盛气凌人；另一方面又毫无安全感，一副弱者“坐困愁城”的极为矛盾的双面心态。活脱脱地勾勒出了中共极为扭曲的外强中干的两面人特征。对于澳洲因为率先倡议对新冠病毒源头进行深入调查而遭致的中共疯狂报复，亚当森女士表示无怨无悔。她还直截了当地把中共国直接向澳洲媒体递交不满清单的做法，称为“大臭棋”。

澳大利亚即将离任的高级外交官弗朗西斯-亚当森（Frances Adamson）警告说，中共国满满的不安全感和手中的巨大权能是一对不稳定的组合。她还说中共国的 “围城心态” 和拒绝监督的态度对任何一方都不是好事。

这位周五刚刚结束五年任期的外交部副部长还建议，在当前不确定性不断增加的情况下，澳大利亚需要在外交和对外援助方面加大投入，以发挥更大的国际影响力。

去年以来澳大利亚与中共国的关系不断恶化。亚当森女士对此进行了说明，她并不后悔在没有获得国际支持的情况下，澳大利亚率先独立推动对冠状病毒的来源调查。

因为澳大利亚对COVID-19来源调查的推动，还有禁用华为5G网络等决定，招致了北京当局的贸易报复，涉及金额超过了200亿美元。

亚当森女士周三在堪培拉国家新闻俱乐部发言时说，中共国在习近平的领导下，通过强化意识形态和压制不同的声音， 已经让“时钟倒转”。

她说:“人们很难理解，这样一个强权大国居然会既被野心驱使又同时被不安全感所困扰。它有一种深深的防卫心态——哪怕在它把自己的利益凌驾于别人之上时，还是会感到外部威胁。”

“它动不动就怀疑自己被别人遏制了，总不能根据自己和对方的特点判断问题。”

“这种“围城心态”— 一种不愿意接受监查和无法客观讨论分歧的心态—对各方都非常不利。”

她说她总是这样提醒自己：“向外部其他国家施加压力，在内部的每个人也都会各自感受到反弹的力道。”

“不安全感和权力这两者可以构成不稳定的组合，如果处理不当会让情况更加恶化。我们需要了解我们的对手这种状况”。

亚当森女士曾在2011年至2015年期间担任澳大利亚驻华大使，熟悉中共国情况。她认为中共国政权存在更迭的可能性，但应该不会很快发生。但她补充说：“专制政权在本质上是很脆弱的。如果发生改变，也很有可能来得相当相当快。”

亚当森女士说，中国大使馆去年向堪培拉的《新闻九台》、《悉尼先驱晨报》和《时代报》递交了一份列有14项不满的清单，这真是中共国的一招 “大臭棋”。“我实在无法理解他们为什么要这么干。”

她说，澳中关系的修复可能需要时间，这种关系只有在“中共国能够摆正自己和别人的相处之道 ”之后才会向好。

她说：“中国还是非常务实的，如果它选择与我们和其他国家坐下来建立建设性和互利的关系，我们将会积极应对，其他国家也会这么做。”

当澳大利亚成为第一个呼吁对新冠大瘟疫的源头进行独立调查时，引起了北京的愤怒。虽然澳大利亚最终与欧盟合作，通过世生WHO进行了调查，但这次调查并没有得到其他国家的认可。澳洲外交部长玛丽斯-佩恩（Marise Payne）在澳大利亚广播公司的《内幕》节目中也宣布不接受这个调查结果。

亚当森女士说，某些人对政府要求再次调查的批评“让人感到莫名其妙”，因为 “很显然进行这样的调查是必需的”。但她也承认，“无论哪个国家对深入调查最先发声，提出启动这样一个绝对必要的行动，都会遭遇中共国同样的反应。”
