亡腚肛携闫丽梦（以下称之为蛇妖闫）及所谓的“博士军团”轮番上阵，使尽浑身解数，拼命诬陷、诋毁、栽赃、污名、谩骂文贵先生和爆料革命。其言语之恶毒，用心之险恶，态度之猖狂尽皆显露。透过屏幕都能感受到几个屑小之徒的无耻和歹毒。

先分析下亡腚肛使用的套路。第一，把郭文贵先生定义为中共特务，定义为中共的海外总代理。第二，揽以毒灭共之功，把揭露病毒真相的功劳归功于亡腚肛、归功于闫丽梦、归功于119。

那么，为何亡腚肛选择这个时间点出来砸郭？今年5月，文贵先生说以毒灭共已经部署完成。事实也是如此，中欧协议的终止，美国即将出台的病毒报告，表明爆料革命推动的以毒灭共已经进入了快车道，任何力量任何人都不可阻挡。所以，亡腚肛、蛇妖闫迫不及待地窜了出来，抢夺以毒灭共的果实。从亡腚肛在节目中反复提及119可知，亡腚肛早有预谋，想独霸病毒的话语权了，独揽病毒真相之功。好像只有亡腚肛、蛇妖闫才是病毒真相的揭露者，以毒灭共都是他们的功劳。

事实上，在119之前的115，文贵先生已经爆料了病毒真相。而且，早在2019年，文贵先生公开示警全世界，黑暗即将来临。告诉世人，中共即将打开潘多拉盒子。还记得文贵先生多次提到的已经到了欧洲的中共军方病毒专家吗？那可是掌握中共病毒核心机密的人，手握病毒的直接证据。可见，没有亡腚肛、没有蛇妖闫，文贵先生领导的爆料根革命照样让世界了解病毒真相，照样以毒灭共！亡腚肛、蛇妖闫，省省吧，别犯大头证了！

据文贵先生透露，今年5月份左右，亡腚肛已经拿了中共海外特务的钱。拿人手短，吃人嘴软，这也是亡腚肛急于跳出来砸郭的原因。至于亡腚肛为何从“爆料革命战友”变成中共的走狗，原因很简单，亡腚肛从开始就没有相信爆料革命，没有相信文贵先生。亡腚肛从来就不是爆料革命战友，纯粹就是一个投机份子。从亡腚肛的历史可知，他是一个典型的渣男。青年时期游手好闲、一事无成，把老板的女儿泡到手后靠吃软饭度日。结婚后继续风流成性，导致3次婚姻，留下6个孩子。离婚时与前妻对簿公堂，争夺不属于他的财产。孩子管生不管养，就是现在，还有个孩子在日本，在战友的家里生活。亡腚肛来到美国后生活都是问题，于是成为伪民运的一个小跟班。在文贵先生的提携和帮助下能够养家糊口了，并逐渐成为“爆料革命的大咖”。如果没有文贵先生和爆料革命，亡腚肛恐怕还在黑暗潮湿的地下室里与蚊蝇为伍。现在，亡腚肛居然住在文贵先生赠与的豪宅中砸郭，此等屑小不知感恩、恩将仇报。此等行为为世人所不齿，为天理所不容。
