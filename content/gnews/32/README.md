**收****229****张限聚令告票后　铜锣湾篮球市集如常举行　负责人：别急着交罚款**

铜锣湾陈东里8号昨日（17 日）傍晚有人聚集，警方指，该处举行篮球比赛及音乐派对，向 229 人发限聚令告票。主办方表示今（18日）市集将会如常举行，惟现场人流疏落，外来档主均退出活动。负责人对昨日事件感无奈，建议被发告票者别急着缴款，应向法庭解释在场原因。

**萧泽颐：反恐演习要「全民参与」　批准游行需考虑国家安全**

警方将7.1 剌警案定性为「孤狼式恐袭」，警务处处长萧泽颐今日在无线《讲清讲楚》上表示，涉及国家安全的问题，人人都有责任，指警方会加强反恐演习，而且「全民都要参与」，举例说会在港铁、商场、酒店内进行演习。他透露，近来已在油塘一商场内举行演习，员工亦有参加。他承认，不能强迫市民参加，但尽量希望多做演习，也提高公众警觉性及灌输正确意识，往后可以做到「全民反暴、全民反恐」。

**萧泽颐：循宣扬恐怖主义方向查港大学生会　不排除搜献花致谢者寓所**

警方国安处日前因港大学生会评议会曾通过「感激梁健辉」议案，搜查港大学生会办事处、学苑及校园电视会室。警务处处长萧泽颐在无线节目《讲清讲楚》上表示，正循国安法 27 条宣扬恐怖主义调查，搜证是为了调查涉案学生的意图。对于连日有人到铜锣湾现场献花，他不排除会调查献花及表达谢意人士，包括搜查其住所。

萧泽颐在节目上表示，港大学生会评议会的议案，可能等同认同、美化凶徒的行为，「绝对有机会」触犯国安法，如令不同群体产生敌视，亦有可能违反煽动罪。

主持追问，对于曾到事发现场献花及表达谢意人士，是否亦要调查其意图及目的，包括到其家中搜证，又或调查网上言论。萧泽颐回应指不排除这样可能性。他又解释，宣扬的定义在于该行为有没有美化恐怖主义，宣扬其他人都去效尤。

另外，《香港 01》刊出萧泽颐的专访，对于港大学生已经就议案道歉，萧泽颐强调，「犯法就是犯法」，「…别以为做完之后出来道歉就可能会无罪。」

对于过往不时发生袭警案，「71 刺警案」首被定性为「孤狼式袭撃」，萧泽颐称，很难个别案件直接比较，指联合国反恐怖主义措施条例有清楚列明恐怖主义行为，今次事件疑犯明显是受失实信息影响，令警方决定有此定性。

**剧团「剧道场」宣布道别　两部作品将成最后演出　创办人欧锦棠︰外在环境迅速转变**

艺人欧锦棠近年活跃于舞台剧制作，2010 年和太太万斯敏创立剧团「剧道场」，刚过十周年，欧锦棠今（17 日）在FB 宣布，即将公演的作品《十诫》和《诀别》，将会是剧团的道别演出。

欧锦棠表示，去年是「剧道场」十周年，慨叹「一个十岁的剧团，应该是满有活力且不断茁壮成长，没想到一切的理所当然，随着外在环境瞬（迅）速的转变而产生不一样的结果。」他指，原本音乐剧《十诫》是「剧道场」十周年的重头作品，惟因疫情由去年延至今年八月，紧接《诀别》，剧团将用「这两个制作向大家道别，用戏剧诉说我们暂别的原由，相信是最好的表达。」

他又表示，票房所余座位已不多，而近日康文署放宽剧院入座人数限制，剧团却无法受惠，经与署方调解，《十诫》每场只能上限加多十数个座位，《诀别》则加无可加。欧锦棠诚邀各支持者出席观赏，「与我们好好道别，祝愿珍重！」

欧锦棠去年接受《明报》访问时曾表示，原本计划2020 年上演4 部剧，逾20 场演出，但最终因疫情，近十个月没有演出机会，剧团没有收入，但疫情期间他照付租金与员工酬劳，即使有艺发局、防疫抗疫基金支援仍负担不少。

卫生防护中心宣布，今日（18日）新增2宗中共病毒确诊个案，均为输入个案。另外，最新一批复必泰疫苗已经付运抵港，约86万剂。
