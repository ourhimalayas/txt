2022年1月15日，米兰，法国病毒学家，2008年诺贝尔生理学或医学奖获得者卢克·蒙塔尼耶呼吁：“**让我们立即停止大规模接种疫苗**”。

“对我来说，即使在肠道中也有一些小细菌的图像，这些图像充满了病毒。这是一场细菌和病毒之间的战斗，即使有适当的营养和卫生，也必须打败它。治愈的不是疫苗，而是消除这种疾病的药物组合。

有一个巨大的战略错误，一些东西被综合和孤立，一个巨大的战略错误。与一开始所说的相反，这些疫苗没有提供保护，而且这种情况正在缓慢出现。这是当今科学界公认的。不仅是科学实验证明了这一点，而且对所有患者的影响都进行了分析以证明这一点。

它不能像承诺的那样保护，反而可以促进其他感染。用于这种病毒疫苗的蛋白质实际上是有毒的，它是一种毒药。疫苗不应该是杀戮，而是保护，由于这种疫苗，已经有很多人死亡，甚至是有严重问题的年轻运动员。**今天给儿童接种这些疫苗是绝对的犯罪行为**。

它还可能导致大脑中非常严重的神经疾病。由于这种疫苗的长期影响，有很多人正在死亡。 神经系统疾病也可能在第一次给药时出现，而不仅仅是从第二次给药。即使在今天，也没有人能够预测这些现在已经接种过疫苗的人中有多少在未来会出现严重的神经系统问题。

我要求我所有的同事绝对停止接种这种疫苗。今天的医生完全了解我在说什么，因此他们应该立即进行干预，因为人类的未来岌岌可危。许多国家已经忘记了治疗，不仅有疫苗，还有一些尚未使用但效果很好的药物，例如抗生素。

哪一天能够拯救人类，这取决于你，尤其是那些未接种疫苗的人。只有未接种疫苗的人才能拯救接种疫苗的人。接种疫苗的人无论如何都会联系医疗中心以获救。我们必须避免倾听那些无权这样做的人并让他们发声，让科学说话。我再说一遍：**只有未接种疫苗的人才能拯救人类**。

起初，大型制药跨国公司出于经济问题对疫苗非常感兴趣。现在这已经超越了他们，我们必须回归对科学真理的尊重。有必要仔细跟踪临床情况的演变，尤其是接种 1、2 或 3 剂疫苗的情况，因为有科学研究涉及严重的脑部疾病。我们需要消除影响科学新闻的迷雾。重要的是医学和科学真理在媒体和主流中显得准确。
