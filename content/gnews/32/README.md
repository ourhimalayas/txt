首先，为什么说信仰是有性繁殖呢？我大概理解为信仰就是人们对一种思想或概念的坚定不移的追求和向往。在不同的时代信仰的外在表现是不同的，比如在欧洲封建时代人们对骑士精神和贵族精神的追求和向往，在现代西方社会人们对自由，法治，民主等普世价值观的追求和向往，以及未来我相信对我们新中国联邦的正道主义的追求和向往，都是信仰在不同时代的不同表现形式。我认为，无论是骑士精神还是普世价值亦或是未来的正道主义，其内在本质都是对正义和善良的维护，对上天的敬畏和对万物自然和谐共处法则的遵循。
