《朝日新闻》网站12月18日报道，台湾举行了近三年来的第一次全国性公投。对是否停止从美国进口使用生长促进剂的猪肉等四个问题，均被否决。

蔡英文的政策得到了公众的认可，政府希望这些能为加强与美国的关系和加入跨太平洋伙伴关系（TPP）提供动力。

除了猪肉进口问题外，三大主题分别是：1，已冻结建设的核电站的运行，2，天然气发电相关设施计划建设地点的变更，3,全民公投和大规模选举恢复到同一天。选民投票率约为41%。执政的民主党呼吁反对一切，而最大的反对派国民党则敦促所有人同意。

今年1月，与中国保持距离的蔡英文政府取消了对从美国进口经过促进生长药物处理的猪肉的禁令，以改善与美国的关系。然而，虽然许多台湾人支持蔡英文政府的外交政策，但他们认为食品安全是另一个问题。在2018年的最后一次公投中，尽管日本和台湾之间正处于蜜月期，但因福岛第一核电站事故而继续解除对部分日本食品的禁运也未获批准。
