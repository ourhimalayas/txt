《环球时报》记者25日深夜从消息人士处独家获悉，**美国国务院拟以“中国疫情状况”为由，批准其驻华使领馆“授权撤离”。**分析人士对《环球时报》记者表示，这是上月白宫宣布“不派官方代表团”出席冬奥会后，华盛顿图谋对北京冬奥会搞所谓“外交抵制”的又一行动。在未来一个月中，不排除美国将采取更多针对中国和北京冬奥会的恶劣举措。

**据了解，“授权撤离”系美国驻外机构针对特定的紧急事件，决定将驻外人员撤离驻在国的行为，需获得美国国务院高级官员批准后方可实施。**

相关消息人士告诉《环球时报》记者，2020年1月新冠肺炎疫情暴发之初，美国曾单方面宣布临时关闭驻武汉总领馆并撤离领馆人员。“联想到美国近来一系列挑拨煽动破坏北京冬奥会的行径，此次美国选择在虎年春节和北京冬奥会前夕再次‘授权撤离’，来者不善，目的显然并非单纯防疫考量”，**该消息人士称，美国此举应为制造恐慌情绪，抹黑中国防疫成效，干扰北京冬奥会成功举行。**

公开资料显示，截至美国东部时间1月24日下午6时20分，全美共报告新冠肺炎确诊病例超过7000万例、死亡病例超过86万例，在此前24小时内，新增确诊近79万例、新增死亡1583例。而根据中国国家卫健委最新数据，1月24日0-24时，中国本土新增病例只有18例。

“中国无疑是当前世界上最安全的地方之一”，上述知情人士评论称，中国疫情防控工作的严谨科学和防疫策略的行之有效同美国疫情持续混乱和失败形成鲜明反差。**“所以，美国企图通过打疫情牌、将体育政治化向中国施压，不得人心，也不会得逞，只会将美国置于国际社会的对立面，进一步沦为国际笑柄。”**

值得注意的是，美国国务院拟作出上述决定的时间节点正处在北京冬奥会开幕前不到十天。

中国外交学院教授李海东25日在接受《环球时报》记者采访时表示，**这一时间节点的选择清晰地传递出美国的意图，它是继去年12月初华盛顿宣布“不派出官员代表团”出席冬奥会之后，又一次明确的、针对北京冬奥会的杯葛行为。**

他表示，上述两个举动，再加上美国最近大面积暂停该国赴华航班等行动，本质上都是美国对北京冬奥会图谋搞所谓“外交抵制”计划的一部分。“美国想借此达成的目的，通俗点说就是：你家办喜事，我家故意来吊丧。”他这样形容称，美国把冬奥会这一大型体育赛事和中国抗疫工作等国家正常治理行为统统视为“北京扩大其国际影响力”的举措，并因此认为需要不惜一切手段予以打压。
