2月8日Q电视新闻消息：出席了北京冬奥会开幕式的巴布亚新几内亚总理马拉佩被确诊感染中共病毒(新冠病毒Covid-19)。

根据中共国外交部的消息，马拉佩于本月3日抵达北京，随后的PCR测试结果呈阳性，接受了检查和治疗。

马拉佩出席了4日的开幕式，但随后与中共国总理李克强的会晤是通过网络在线进行的。

中共国外交部强调，来自巴布亚新几内亚的访问团在北京逗留期间遵守了疫情防控管理规定。因此，不会带来影响。
