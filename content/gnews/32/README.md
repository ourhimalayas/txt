伦敦帝国理工学院YouGov Covid-19行为追踪数据中心收集了有关全球人们应对中共病毒的行为和态度。该调查显示了美国、英国、法国、德国、意大利、挪威、丹麦等多个国家的人们接种新冠疫苗意愿的月度数据。每次都有超过500名18岁及以上的受访者参与。调查将将受访者分为四类：至少接种了一剂疫苗、未接种疫苗但愿意接种疫苗、未接种疫苗且不确定是否愿意接种疫苗、以及未接种疫苗且不愿接种疫苗。第一个图表显示了2021年2月28日至6月30日期间这四个类别在美国的百分比，而第二个图表显示了多个国家/地区在2021年6月30日那一时间点的数据。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/07/09140420/Picture1111.png)

![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/07/09140630/Picture1222.png)
图 2. 对来自多个国家的四类人的态度进行比较。图表显示，美国和法国拒绝疫苗的人比例最高。

从第一张图可以看出，在美国，大多数愿意接种疫苗的人已经接种了疫苗，而无论主流媒体如何，总有大约30%的人不愿意接种疫苗。尽管媒体整天宣传疫苗的有效性、政府不遗余力地推动接种疫苗甚至推出抽奖活动，美国那30%的人群一直坚决拒绝接种疫苗。毫无疑问，他们拥有独立的思考能力，并且他们知道疫苗对身体的真正风险。估计他们总数在8000万左右，这一数字跟川普获得的选票数一致。笔者认为他们很可能同时也是支持川普的保守派人士。
