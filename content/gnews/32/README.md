#### 翻译                   意大利达芬奇农场              小白鸽
![](https://assets.gnews.org/wp-content/uploads/2022/02/Schermata-2022-02-16-alle-16.14.39.png)图片来源：eventiavversinews.it
这是英格兰足球赛季中出现的第二个心脏不适事件。在球迷中发生了 “医疗事故”后，纽卡斯尔联队与阿斯顿维拉的比赛在周日暂时推迟。在纽卡斯尔1-0战胜杰拉德的维拉比赛中，有医生在看台上救治一名球迷。俱乐部在其网站上发布的一份比赛报告中证实比赛是出于医疗原因而终止。

令人惊讶的是， 这已经是本赛季第二次因医疗紧急情况而终止在纽卡斯尔主场比赛。

在那场比赛中一名球迷在上半场比赛快结束时心脏骤停，被送往医院。

幸运的是， 当天就在后面30排坐着汤姆•普里查德医生， 他马上冲到病人身边。

普里查德医生后来告诉BBC:“这是一般在工作中经常能看到的情况，但在这之前我从未在医院外遇到过类似情况。”

英格兰足协实施了英国政府的方案： 在10月1日以前其俱乐部球员和球迷必须接种新冠疫苗。如果不打两针新冠疫苗， 俱乐部球员及其成员将无法入场， 球迷也无法进入体育场。
