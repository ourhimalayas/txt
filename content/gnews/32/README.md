外交官杂志的作者梅西·郭（Mercy Kuo）定期与全球各地的主题专家、政策实践者和战略思想家接触，听取他们对美国亚洲政策的不同见解。这次是与特雷弗·泰勒博士关于“跨太平洋观点系列”的对话。他是伦敦皇家联合服务学院（RUSI）国防管理教授级研究员。

鉴于澳大利亚缺乏核基础设施，请分析一下堪培拉选择核动力潜艇而不是柴电潜艇的决策。

由于所涉及的费用和风险，这对（AUKUS）协议的所有三方来说肯定是一个艰难的选择。

**核潜艇具有一系列的作战优势，但主要是它们可以在不浮出水面的情况下比柴电潜艇部署更长的时间。**对核潜艇水下作业的限制只基于它们能携带多少食物以及海军能要求其船员在艇上服务多长时间。另一方面，核潜艇的开发和建造成本要高得多，而且它们需要专业的劳动力来建造、操作和支援。

促使澳大利亚转向核潜艇的各种确切的平衡因素，除其政府最高层，外界并不知晓。对法国潜艇进度迟缓的不满以及确认美国和英国是否愿意合作，这些肯定都起到了一些作用。

**澳大利亚海军认为在中共国附近部署一艘或甚至两艘核动力潜艇具有的优势，**同时意识到即使是长程柴电潜艇也无法实现这一目标。

是英国的“机敏号”或美国的“弗吉尼亚”级核潜艇？哪种选择最有利于澳大利亚的军事实力？

如果可以从一个正在运行的生产线上购买潜艇，那么“机敏”号的成本会更低，而且生产能力更充足。然而，“机敏”号生产即将下线，其供应链可能已经无法满足需求。美国和英国的潜艇部门都没有出口的能力，所以必须找到一些有能力来帮助澳大利亚。

下一步各国政府进入实施安排的细节，将遇到英国和美国确定转让哪些技术和知识产权的棘手问题。美国有可能希望澳大利亚拥有洛杉矶级潜艇的一个更新型号，而不是其最终的替代者——“弗吉尼亚”级。考虑到澳大利亚以前在组建潜艇船员方面存在的困难，“机敏”号所需的船员人数较少（大约是弗吉尼亚级的四分之三），这可能会增加对英国本土的潜艇的吸引力。

请解释一下澳大利亚将如何从美国和英国的核潜艇技术中受益。

像许多其他国家一样，**澳大利亚希望国防项目可以增加就业**，但其就业规模将很大程度的受到其新潜艇投入使用进度影响。作为一个粗略的规划，澳大利亚的潜艇创造工作岗位越多，交付周期就越长。从长远来看，如果**澳大利亚选择了民用核能发电**，它将要求更多的专业知识背景来实施完成。

随着澳大利亚决定购买战斧导弹，多大的有效射程将扩张澳大利亚海军的打击力量？

这可能是一个重大的改进，因为澳大利亚的霍巴特驱逐舰和正在建造的猎人级舰艇都没有装备射程可以攻击中国大陆目标的导弹。**最新的“战斧”导弹的射程超过1000英里。**使用这些导弹也需要覆盖范围广的目标情报系统，而且很难设想脱离AUKUS联盟背景发射这些导弹。

在未来18个月的AUKUS计划中，堪培拉在与华盛顿和伦敦的谈判中面临那些挑战和选择？

整个议程将是复杂而漫长的。澳大利亚将不得不在它最想要的东西和时间之间作出决定。无论在澳大利亚建造的船只型号比例如何，**都必须为它们的到来建立相应的基础设施，以及它们将在其中运行的核监管制度**。堪培拉将需要与它的两个合作伙伴就教育和培训事宜达成一致。如果澳大利亚人员能在几年内接受培训并在英国或美国的(核)潜艇上担任船员，这显然是有益的。澳大利亚的潜艇将需要一名国家级别指挥官，担任这种职位所需的培训和之前的多种工作经历需要多年培养才能达到要求。

总之，关键的挑战将是政治上的——调和澳大利亚的期望和其合作伙伴愿意提供的东西，以及财政上的——为所有需要做的事情找到持续的资金。资金任务清单包括与法国海军集团达成和解方案的费用。
