1992年南斯拉夫解体成6个主权国家，分别是斯洛文尼亚、克罗地亚、塞尔维亚、黑山、马其顿与波黑，历史上塞尔维亚是个很牛叉的大公国，出了个著名的科姆（克雷姆纳村庄的塔拉比奇叔侄）这样的先知与特斯拉这样牛叉的人物，但后来跟苏联混就逐渐衰退没落了。塞尔维亚的两个核心人物迪萨科维奇、科萨维奇，一个是国防部长，一个是军委主席，俩人都被中共彻底蓝金黄收买了，他们全家都能说中国话、吃中国饭，在海南置地、上海买房，几乎完全中国化了，没事儿就飞到香港等中共国地盘住上一段时间。

塞尔维亚总共才700万人和4万多军人，人均月工资不到600美金，每年GDP不到500亿美元，整个国家总共才19架飞机，其中8架还是不能飞的废铁，交通运输大量依靠火车系统。塞尔维亚人对中国人很友好、对俄罗斯很崇拜。当下在塞尔维亚最牛的企业就是中共国的河钢、阿里巴巴、微信等47家企业。塞尔维亚旅游业和农业都很发达，产的粮食约定需卖给中共国。

塞尔维亚所处的巴尔干地区有着特殊的战略地位，因此中共解放军6架运-20战略运输机4月8日将首批100枚红旗导弹FK-3及其反导系统运往塞尔维亚，据说中共4月9日公布此事后西方军事专家都吓懵了。塞尔维亚将其部署在西部的250旅，基本上覆盖了北约大部分空域。这些FK-3红旗导弹系统并不是卖给塞尔维亚的，而是白送给它的。中共不但白送给塞尔维亚，还答应给它重建通信与火车系统。

中共马上还要给俄罗斯最起码送上100倍的FK-3红旗导弹系统，首批1600枚能打掉北约一两百架战机。中共的FK-3已生产了50万枚，习近平已命令全部发出去备战，其空军和海军联合作战部麾下的几十个地下作战单元就已部署了大概30万枚。最近中共正在加紧生产另外150万枚，其中近20万枚是专门瞄准台湾的。

中共在塞尔维亚部署FK-3导弹这一票的主要目的，就是利用原来培养南斯拉夫这个独裁国家的关系打进这个战略楔子，同时给土耳其、沙特、伊朗、叙利亚、突尼斯、也门、保加利亚等中共的大小跟班们一个巨大的武器供应之诱惑。习共此举就是直截了当：“你不是调查我支持俄罗斯吗？老子就干了怎么的！”实际上就是直接对美欧竖起大中指这个意思。

这个FK-3红旗地对空导弹系统最初完全抄苏联的技术，经过共军的反复改进从雷达制导到智能制导、电脑定位后，打击准确率从过去完全蒙打的12%到达现在所谓的80%，且一个弹头能带10个分弹头，有效打击距离大概在2万5到3万米之间。FK-3导弹系统非常便宜，且可与最简单的雷达系统匹配，平角发射出去后就直跟目标去了，非常危险！

FK-3红旗导弹系统必须得用战略运输机运-20运送，悲哀的是运-20最核心的技术却是乌克兰提供的。更加悲哀的是，郭先生将中共运送FK-3红旗导弹系统给塞尔维亚的情报给了英国和美国，但他们却都不相信，说共产党不敢这么做。同时，中共某艘民用船只现在就停在美国家门口，船上载有大量导弹，一百个古巴导弹危机跟这比都不值一提！

中共的战术导弹系统除了FK-3，还有一个更厉害的FK-9，也叫红-900或红-9。中共还有两个非常厉害的大杀器即战略洲际导弹系统，一个是东风-5系列洲际导弹系统，东风-5甲一般般，东风-5丙很厉害，射程在13000-15000公里，携带12个百万级TNT当量的战略核武器分弹头，且应用的是末端自动制导的分导式系统。

另一个是东风-15系列洲际导弹系统，东风-15B马马虎虎，但东风-15E和东风-15EN很牛叉，射程近10000公里。中共已将东风-15布置到了伊拉克、伊朗和沙特，并把FK-3送到了伊拉克、沙特、叙利亚、伊朗、突尼斯等邪恶国家，对西方乃至全世界的威胁远超当年的古巴导弹危机！中共解放军二炮部队现在称火箭军，其番号为8010、8011、8013、8014、8017的部队均布置了大杀器东风-15EN、东风-41和东风-5等战略洲际导弹系统。

中共马上又要给塞尔维亚和俄罗斯分别送去60架和800架长虹无人机，给中东的叙利亚、也门、沙特、伊拉克、伊朗等邪恶国家准备的1600架无人机也即将生产完毕等待交付。这次交付的无人机进行了显著改进，增加了小型导弹的装载量，尤其是增强的自动跟踪功能，达到移动式美制“毒刺”导弹的攻击效果专打北约战机，有效射程从过去的27公里提升到50公里甚至120公里。

中共这次的红旗战术导弹系统与东风战略导弹系统的布局，涵盖塞尔维亚、保加利亚、阿塞拜疆、哈萨克斯坦、白俄罗斯、中东、印度南部、巴基斯坦、伊朗、沙特、伊拉克、吉布提、阿根廷，甚至澳大利亚的后院所罗门，一场正义与邪恶之间的大战、血战即将来临！
