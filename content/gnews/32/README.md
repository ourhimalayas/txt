**译文：**

一年前比特币尚未进入大多数人的视野，自从去年11月起，它的价格令人瞠目结舌地屡创新高。

对此，澳大利亚参议院的金融和科技特别委员会在极短时间内为政府制定了一项全面的计划，将加密货币纳入主流。自其职权范围扩大到对数字资产的机遇和风险进行深入研究以来，仅仅过了七个月时间。

由参议员安德鲁·布拉格担任主席的该委员会各个行业的参与度非常高。今年已收到了100多份建议书，表明市场希望政策制定者围绕具有挑战性的话题展开讨论。

其中包括治理的权限下放；（市场）对包括商业银行和中央银行在内的传统中介机构的不信任；**以及包括分布式账本和云计算在内新技术的强大组合，这使得提供新金融服务的全球互联网协议，能够在传统金融体系监管者的范围之外发展**。

在过去的一年里，数字资产领域迅速增长，（大到）政府无法忽视。美国银行在本月报告中说，该行业的价值为2.8万亿美元。该报告发现，**截至****6月，全球有2.21亿用户交易过加密货币或使用过基于区块链的应用程序，比一年前高出四倍。委员会取得的证据显示，17%的澳大利亚人拥有一种加密货币；**澳大利亚税务局告知，**近年来有超过****60万名纳税人投资于数字资产，**

但所有这些投资都是在监管黑洞中进行的。

持有价值数十亿美元的数字资产的交易所在澳大利亚几乎不受监管。银行已经避开了这个行业。澳洲证券和投资委员会（ASIC）给与了豁免权，并进行有限的监督。澳大利亚证券交易所拒绝让加密货币企业上市。相对而言，ASIC的做法比较温和。委员会希望政府结束这种逃避监管和希望市场自生自灭做法，因为它现在已经太大了。

这份澳大利亚技术和金融特别委员会(ATFC)报告，比人们对比特币及其每日价格走势的迷恋，或看到一些加密资产被推向令人眼花缭乱的高度的代币（NFT）热潮，要深入得多。**委员会认为，去中心化技术会引发一波新的服务，包括在以太坊上运作的更透明的商业模式，并建议澳大利亚最好是拥抱而不是回避即将到来的东西。**

这不仅仅是关于1350亿美元的去中心化金融（DeFi）市场，而是全新数字经济的基础，从互联网搜索和游戏到社交媒体和计算机存储的新模式，其中更多的利益被分配给用户而不是集中的技术巨头。如果第三代基于去中心化区块链协议的互联网服务（Web3.0）真的出现了，报告认为澳洲必须谨慎行事，在国内培养新兴技能，（政策）支持这些新生代企业的创立，而不是眼睁睁地看着加密货币“蛋糕”被他国瓜分。

当然，许多人将对上述场景中的监管和控制的想法不以为然。“死硬派”不希望（加密货币市场）与传统金融系统的“轨道”运行发生任何关系。加密货币（天生）有反叛的、反建制的特质。此外，“去中心化的自治组织”存在在互联网上；（各国）地方规则在实践中难以实施。

但这次调查清楚地表明，尽管存在挑战，澳大利亚（目前）的公司法并不适合数字化的未来发展。

银行可能会公开欢迎将不受监管的参与者纳入监管网络，（因为）银行喜欢公平的竞争环境。但私下里，银行家们担心自动化计算机协议所提供的金融服务会造成（实际）脱离监管。

虽然比特币表明可以在银行之外创建货币系统，但Synthetix、TracerDAO等十几种去中心化金融应用程序——它们都与澳大利亚有联系，如此看来银行业务在未来依然大有可为。

本委员会希望监管机构和财政部为此做好准备。它希望联邦政府确定政策方向，鼓励企业家和新的参与者，同时确保提高对投资者的最低保护水平。

当然，加密货币也创造了许多新的风险。缺乏（类似破产重组）“安全港”的保护将使一些开发者感到紧张，让政府官僚机构跟上步伐也将是一个巨大的挑战。整个领域的发展如此之快，任何新的法律都需要确保在生效前不会过时。目前，世界上没有任何一个司法管辖区的加密货币监管是正确的。

此报告核心指出，政府最大的风险是无所作为。
