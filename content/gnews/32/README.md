- ![](https://assets.gnews.org/wp-content/uploads/2022/01/KTBCGBXLMM5CJ8VPX.png)图片来源：REUTERS


英国政府声称，自2020年3月以来，英国有超过150,000人死于新冠。但来自其他机构如国家卫生署以及国家统计局的数据，则清楚地实情并非如此。

英国国家统计局最近在回应一项信息问询时说，截至2021年第三季度末，在英格兰和威尔士地区登记的新冠死亡人数有17371人。但就在一个月后，在回应另一项信息问询时，他们表示在2020年2月1日至2021年12月31日期间，新冠的实际死亡人数只有6183人。

而英国国家卫生署在官方数据中指出，截至22年1月12日，在英格兰的医院中没有其他严重基础疾病的人只有4670名死于新冠。那么，为什么我们被告知的英国新冠死亡人数有这么多不同的版本呐？哪个数字又是正确的呢？

英国政府的数字–新冠死亡人数是15万

英国政府通过数据公告网页声称，截至2022年1月13日，自2020年3月以来，全英国已有151,855人死于新冠。

在1月13日发表的一份报告中，药监局声称“根据这份报告，全英国有超过149,541人在冠状病毒新冠检测呈阳性的28天内死亡，接种疫苗是减少死亡最有效的唯一方法……”。

但是英国政府、政府机构和主流媒体使用的这个数字可能是有史以来最具误导性的统计资料。这是因为他们宣传的并不是新冠的真实死亡人数，而是检测呈阳性后28天内死亡的人数。这意味着，那些每天都虔诚地进行新冠测试，然后在卫生署的统计程序上确认其阳性测试结果，然后在三周后碰巧在事故中丧生的人，也都被统计到了新冠死亡当中。

很显然，这15万例新冠死亡当中的绝大多数，实际上是不是死于新冠是很值得商榷的。这就是为什么不同的英国政府机构向我们提供了一堆数据，关键是各部门的数据无声地揭示出，这些数字相互之间是矛盾的。这反到证实了所宣称的15万例新冠死亡中的绝大多数实际上不是死于新冠。真是太棒了！

国家统计局信息问询–17,371例新冠死亡

国家统计局在2021年12月16日对信息查询的答复中确认，截至2021年第三季度末，只有17,371人死于新冠。

那个查询系统显示：英格兰和威尔士地区新冠死亡数字是 2020年9400人，2021年一季度6483人、二季度346人、三季度1142人。由此可以清楚地看到，这些数字加起来远远没有达到英国政府和主流媒体声称的15万新冠死亡人数，如果我们把2021年第三季度末的死亡总数加起来，我们还可以看出，新冠疫苗在减少死亡人数方面并没有真正起到很大的作用。

根据国家统计局的数据，在2020年3月1日至12月31日期间，有9,400例新冠死亡。英国第一批新冠疫苗是在2020年12月8日接种的，老年人和最易感染新冠的人优先。但奇怪的是，在2021年的前三个季度，又发生了7,971例新冠死亡。而原先本来宣称新冠疫苗是能够把新冠死亡的风险降低95%的。

这里我们又遇到一个新问题。一个月后，国家统计局对另一个信息问询作出回应，称截至2021年底，死亡数字降到了6,183。

国家统计局信息问询–6,183例新冠的死亡

信息问询请求国家统计局提供单纯由新冠引起的死亡人数，即新冠是死亡证书上列出的唯一死因。

国家统计局于22年1月17日对问询作出的回应显示，在2020年2月1日至2021年12月31日期间，死亡人数只有6183人，绝大多数发生在老年人口中，56%的死亡发生在80岁以上的人，74%的死亡发生在70岁以上的人。这与英国政府及其机构宣称的150,000多起新冠死亡，以及一个月前国家统计局宣称的17,371起死亡事件相差甚远。但为什么数字会如此不同？

国家统计局还显示，在20年2月1日至21年12月31日期间，20岁以下死于新冠的人数仅仅是3个人，这正能说明，英国政府决定推翻疫苗接种和免疫联合委员会的决定，让儿童注射实验性的新冠疫苗，是多么可耻的事情！

国家卫生署–4,670例新冠死亡

一份题为“新冠死亡总人数——每周摘要”的文件显示，截至1月12日下午4点，总共有101,763人死亡，并不是英国政府告诉我们的150,000多人。然而，该数据同时显示，在过去两年中，只有4670名没有其他严重疾病的人死于英格兰医院的新冠。

我们到底该相信谁？

可以肯定的是，英国政府、政府机构和主流媒体使用的官方的超150,000的新冠死亡数字是不可靠的。这个数字被用来向民众传播恐惧，并为两年来的严厉新冠限制措施背书。两年的时间已经匆匆逝去，人们将永远无法找回。

国家统计局的死亡数字是6,183人，而卫生署的数字是4,670人，在这两个数字中，一定是国家统计局的数字更准一些。事实上，卫生署所说的仅在英格兰的某些年龄组的死亡人数就比统计局在英格兰和威尔士两地的死亡人数还多，这意味着卫生署的数字根本就不可信。

因此，在2020年2月1日至2021年12月31日这一年零十一个月期间，死于新冠的真实人数是6,183人，而不是英国广播公司（BBC）作为黑暗里程碑报告给你的150,000人。
