3月10日，文貴大直播中，郭文貴先生向新中國聯邦（New Federal State of China, 縮寫NFSC）戰友們發出預警，在俄烏戰爭之後，中共也必將會走向滅亡。

郭先生表示，俄羅斯的入侵戰爭一打響，即已經宣告了俄軍失敗的結局。世界人民對於民主法治的社會制度已經達成了高度認可，不會坐視獨裁者向無辜的國家發動侵略戰爭。

![](https://assets.gnews.org/wp-content/uploads/2022/03/图片-1-9.jpeg)2022年3月9日文貴大直播（圖片來源於網絡）

目前俄羅斯內部反戰情緒高漲，國際社會對俄羅斯的制裁一輪接一輪，俄國已面臨經濟崩潰，普京面臨衆叛親離的悲慘境地。

郭先生透露，習近平本來以爲俄烏戰爭會導致美歐之間出現嫌隙，藉此俄羅斯橫掃烏克蘭，中共則乘機拿下臺灣。但是根據目前俄軍遇到的頑強抵抗來看，背後支持烏克蘭的美國最終會成爲戰爭的最大受益者，一方面證明了美國具有無比強大的實力來穩定混亂的世界局面，另一方面也會讓歐洲更加清醒，認識到對美國的依賴只會越來越強。

俄軍戰敗後，俄羅斯將不會再做爲一個主權國家存於世上，而烏克蘭將會作爲勇敢抗擊侵略者的英雄國家贏得世人敬仰。

對於俄烏戰爭發展到現在的局勢，中共和俄羅斯之間的狼狽爲奸已被這個世界上的民主社會共同唾棄，做爲與俄有着“堅如磐石”關係的中共，面對世界民主國家的一致譴責，中共總書記習近平會害怕嗎？

對此郭文貴先生向NFSC戰友發出預警，以習近平的性格必定會冒險一搏。中共目前威脅着世界的石油能源供應，在霍爾木茲海峽、伊朗、沙特、阿聯酋和卡塔爾到非洲安哥拉都已滲透多年。中共的策略，就是要利用中東皇室的利益和對石油的依賴挑起一場能源戰爭打垮世界的經濟，繼而侵佔臺灣。

中共有14億人口生產力提供全球的供應鏈，還有無限的“鎖鏈女”供享用。通過上述誘惑，中共會逼迫美國與之達成“新型大國關係”，共同統治地球。但是，獨裁的體制必被消滅，美國也看清了中共的邪惡本質，面對中共的拼命一搏，會給予迅猛打擊，中共必將灰飛煙滅。
