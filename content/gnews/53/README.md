## 欧洲议会将通过动议抵制北京冬奥会制裁中港官员

据《美国之音》7月8日报道，欧洲议会预计将在周四（7月8日）通过一项动议，制裁香港官员，抵制北京冬季奥运会。

根据香港《南华早报》的报道，欧洲议会的主要政党已经在周二和周三完成了这个动议的起草工作，对中共损害香港民众的自由权利，以及封杀香港民主派媒体《苹果日报》的做法，提出严厉批评。动议还要求对香港有关官员进行制裁，并对北京冬季奥运会进行抵制。

报道称，这项动议将在周四付诸辩论和表决。预计，该动议将会以绝对多数票获得通过。虽然这个动议不具法律约束力，但它将能够为欧盟委员会和欧盟理事会了解欧洲议会的态度提供参考。

负责欧盟日常事务的欧盟委员会目前还没有就是否支持抵制北京冬奥会和制裁香港官员的问题表明态度。

《南华早报》称，欧盟的这份动议草案敦促议会对香港行政长官林郑月娥、律政司司长郑若骅、保安局局长邓炳强、前警务处处长卢伟聪、政务司司长李家超等官员采取“定向制裁”。

被列入制裁名单的不仅有香港官员，还有中共国大陆官员，如中共国国务院港澳办主任夏宝龙和他的前任张晓明、香港中联办主任骆惠宁和中央驻港国安公署署长郑雁雄，以及“在通过相关措施和参与旨在破坏香港高度自治和自由权利行动的中华人民共和国的一些机构”。

与此同时，英国主要反对党工党周二也敦促英国政府和王室对2022年北京冬季奥运会进行政治抵制，除非中共国同意联合国派人进入中共国，就新疆发生的系统性侵害人权行为进行调查。

此前，英国政府对中共国侵犯人权的行为多次提出谴责，并呼吁联合国派人去中共国新疆，调查有关当地维吾尔穆斯林遭受酷刑、强迫绝育等迫害的指控。

不过，英国政府至今一直没有公开表示抵制北京冬奥会。而美国国务卿布林肯上个月表示，华盛顿正在就是否抵制这次冬季奥运会的问题谋求盟国的共识。

《法新社》的报道援引了英国工党外交事务发言人莉萨·南迪的话。南迪说：“我们一直在要求政府对这种（新疆的）令人恐怖的做法采取更为有力的行动。包括对新疆所发生的事情负有责任的高级官员实施更为广泛的制裁，以及对强迫劳动现象采取更为有力的应对措施。”

南迪呼吁利用这次运动会的机会向中（共）国施压，要求中共国允许联合国人员无限制进入新疆，展开充分、透明和独立的调查。而在此前，英国等国家的政府已经就此提出过多次要求，但一直没有得以落实。”

工党议员还表示，如果到今年9月14日联合国大会第76次会议召开之际，联合国仍然没有得到中共国的许可，“英国政府就不应该派部长级官员、王室成员或者高级代表，以官方身份前往北京参加冬奥会开幕式。”

工党指出，如果英国等国家采取政治抵制，将会发出一个强烈的信号，表明全世界对维吾尔人的遭遇给予了高度的关注。同时，防止中共当局把这次运动会变成它的一次公关手段。
