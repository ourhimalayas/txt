**译文**

维多利亚州警方叫停了媒体机构拍摄和播放墨尔本抗议活动的空中直播，这一令人震惊的对电视实施的前所未有的限制令，在实行几个小时后被撤回。

民航安全局9月22日批准了维多利亚州警方的要求，即在抗议活动的第三天，禁止除警方直升机之外的所有飞机飞越墨尔本中央商务区。

但是，在维多利亚州警方的禁飞令实行四个小时后，这项史无前例的限制令被收回，改为允许媒体在获得警方批准后飞越中央商务区。

媒体仍将被禁止进行视像直播，而必须将其空中镜头或者延后60分钟或者直到警方行动结束后方可播出。这些限制措施至少在9月26日上午9点之前仍然有效。

警方在一份发表的声明中说。”这是因为抗议者在随时关注空中直播，直播影响了警方的行动，使警员的安全受到威胁。”

如果电视频道敢实况直播任何影像，空域将再被关闭。

在9月22日下午这项规定发布后，各电视台都表示出愤怒，纷纷说这将阻碍他们报道重大新闻事件。

第九频道的新闻和时事主管达伦·威克激烈抨击这一规定，他认为此举 “太过分了”。他说，在做出这一决定之前，并没有征求媒体的意见。

威克先生对墨尔本3AW电台说：”媒体必须能够向公众展示正在发生的事情，限制报导这样的事不该在我们这里发生，澳大利亚不是一个警察国家。”

“现在的这样一项规定是史无前例的，也让人感觉极不舒服。”

“在接下来的几天里，媒体的直升机报道都要受此规定限制，这是很过分的，实际上也开出了一个非常危险的先例。”

威克先生提到了2014年在悉尼的Lindt咖啡馆发生的劫持事件，他说新南威尔士州警方在那么危险的事件发生时，还在征求媒体的配合。

他说：”几年前的Lindt咖啡馆劫持事件时我在悉尼，警方当时与媒体沟通，要求我们在半小时内停止播放现场图片，以便他们能够将警察调动到出击位置，我们配合了这一要求。”

“但现在这不是那种危急的事件，这不过是一群在街上溜达抗议的怪汉……在接下来的几天里禁止媒体直升机对此进行实时报道是很过分的。”

“我们正在进入新闻审查制度，正在变成警察国家”。

州长丹尼尔·安德鲁斯的办公室说，阻止电视台空中直播抗议现场的决定，与他们无关。

据透露，一架第七频道、第九频道、天空新闻和澳大利亚广播公司共享的媒体直升机，在9月22日下午2点左右停飞。

第七频道的新闻总监克雷格·麦克弗森也批评了这一决定，并说这种做法是在 “剪掉新闻的翅膀”。

他说：”我从来没有见过这样处理事情的，依据的是什么？一下子滑落了很多档次，主要是有违常识。我不认为新闻直升机在空中的实拍，会给抗议者提供任何程度的情报。 “

他说，在自己从事媒体行业的40年中，从未见过这样的事情。”并没有什么实质性的理由，只是把媒体禁足了。墨尔本这座澳大利亚第二大城市的中心正在发生重大的新闻事件，人们需要航空直播见证这一切。但这项决定阻碍了电视台的报导。”

直播者们是在空中被告知了限播的决定，当时他们的直升机刚刚加油，正返回中央商务区准备拍摄9月22日的抗议活动的更多画面。

维多利亚州州长丹尼尔·安德鲁斯的一位女发言人说。”这些是维州警察独立作出的决定，政府没有参与。”

抗议活动的画面一直在世界各地广泛流传。

在已经播出的媒体直升机拍摄的一些最混乱的场景中，包括数以千计的抗议者星期二在墨尔本的主要干道–西门高速公路上造成混乱，使交通瘫痪，交通堵塞了几个小时。

维多利亚州警方在遭到媒体抨击后，于9月22日下午6点发表的声明中说：”虽然这一决定是出于与抗议活动有关的安全原因，但我们理解媒体的关切。”

“因此，维多利亚州警方将会制定一项规定，让媒体机构得以在中央商务区上空飞行。”
