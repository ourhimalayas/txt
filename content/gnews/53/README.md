**中共国粗暴改变盘古酒店现状后又粗暴查封酒店，并严重侵犯酒店所有员工和郭文贵先生年迈老父亲的人权，我们新中国联邦人对此发动报复行动，一千天的行动绝不手软。**

盘古酒店的外貌和盘古经营权被中共非法改变后，预示着共产党政治根基已经动摇。人们已经看到美国针对北京冬奥会抵制的公文发布，这件事非同小可，根据郭文贵先生（七哥）援引一位国际友人的观点看，“取消北京冬奥会基本上给共产党宣判死刑了”。只不过是哪一天完的事，（[40:25 时段](https://gettr.com/post/pj3xefa5bc)）七哥认为，“严格讲，取消冬奥会比取消世贸还严重，他说，这意味着今后再没有任何国家的国际组织来中国。所以，奥运会打到他的痛处，奥运会被取消的同一天，盘古被查封。”

这件事的发生不是偶然，爆料革命战友都会想到盘古出现一件灵异的事件，这件事也令中共不寒而栗。七哥曾说，当时盘古酒店工地出现两条蛇，一条黑蛇、一条白蛇。笔者查考了一下，白蛇的神话故事家喻户晓，白蛇象征爱情，而黑蛇，根据[闽北民间神话](https://kknews.cc/zh-my/culture/a3zzamx.amp)，是救死扶伤的民间正道代表，也有人们说，黑蛇是“龙神下凡”。说到这里，笔者恍然大悟，这难道不是七哥对同胞爱的凝聚、对战友们的集结而掀起的“正道主义”灭共运动吗？ 这场灭共运动自2017年正式打响以来，四年多时间，我们已经动摇了中共政权的根本。

这场运动自始至终充满着神圣的使命感，这要从盘古“黑白蛇”的灵异事件说起，[七哥说](https://xiaomu2018.blogspot.com/2018/04/29google-earth-20012009-22-1999-2017.html)，“2001年底盘古大观动工时挖出来的两条蛇，一黑一白，其中一个让当时施工队的人打死了，结果这个人被清土车弄死了。另外一个蛇放生到香山卧佛寺。2009年时，这个蛇竟然真的回到了我们盘古大观，爬到了我住的一个院的门上面，都有录像。他们跟我说以后，我说千万别打死它，从楼上跑下去，然后他就弄个小筐把它给弄下来，大家都给它磕头，有录像。用我们酒店的小篮子，开车又送回到香山给放生。”蛇在中共国民间一直视为龙一样的存在，有些蛇的出现超乎人的想像。也因此，盘古大观的主楼建筑为什么是龙头造型，就源自这对黑白蛇。

更加神奇的是，习近平将自己中南海的椅子改成龙椅，真以为自己是皇帝。但奥运会被抵制后，你甚至可以听到习近平的龙椅开始咯吱咯吱地摇动声了。

然而欺民贼们永远都不相信灭共是神的旨意，仍然在那里讨论什么《[民主峰会](https://youtu.be/5Q6IuSjMqnM)》，似乎这是他们的饭碗。但新中国联邦对此有独家消息披露，他完全超出人的猜测。可以说，美国领头官方抵制北京冬奥会，完全是考虑美国国家利益的问题，你永远都不要相信一个以偷窃手段夺取民主选票当选总统的拜登有真民主，内部原因非常现实而骨感。

七哥说，这是“拜登政府经过仔细评估的结果”，根据白宫的考量，拜登有三件事他解决不了，“第一，美国政府要去了，运动员将会有一千人的规模，染了病毒谁负责？总统和国会议员都不负责。第二，钱谁出？出钱者要出付钱的责任，财政部说没有这个预算。第三，一旦这期间（中共国）打台湾了，运动员被扣成人质怎么办？或者运动员之间发生种族事件了，或者是有病毒感染者当场死亡的事件发生怎么办？（笔者认为，以中共的邪恶，通过对运动员的基因分析，定点的让谁死是完全能做到），有应急方案没有？由谁负责？这种情况下，如果美国要去，五眼联盟都会跟着去，几十个联盟国要去，美国要不要对这些盟友负责任？这些一考量，都无法决定去参加。”（[1:48:57](https://gettr.com/post/pj3xefa5bc) 时段）

新中国联邦人永远都是“唯真不破”、客观分析，脚踏实地的灭共，既不是路大脑袋夸大美军贝雷帽、神话美式民主，也不像欺民贼整天见风就是雨，虚张民主声势，障人眼目、更不像九指妖欺哄圣灵，许诺假上帝欺骗基督徒战友们。七哥一直以来的灭共战略的主导思想就是“正道主义”，因为只有我们坚持真、求正，我们就一定立于不败之地。最后那条盘古被打死的蛇不会是蛇妖闫吧？但蛇妖闫毕竟还没修炼成精，这条被打死的蛇也不知道是白蛇还是黑蛇，应该就是习近平这条恶魔吧，而打死蛇的也死了，这会不会是当今袁世凯、戈尔巴乔夫？这不就是以共灭共吗？

笔者认为，这些灵异事件联系起来都是上承天意。因为中共“以人血建城、以罪孽立邑的有祸了！”（[哈巴谷书 2:12](https://cnbible.com/habakkuk/2-12.htm)）中共政权必被“盘古的火”焚烧灭绝。
