- 首先，美国人永远不会接受高于4美金一桶的油价，否则，美国政治将会大变；
- 其次，在食物方面，当面包的单价超过4至5美元时，美国社会也将进入动荡；
- 再者，如果能源，比如天然气和电力的价格涨至历史价格的120%，执政党必将下台。
