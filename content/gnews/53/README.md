## “厂里停电”90年代的记忆又回来了
` 银河勇气之星` [轉載自GNews](https://gnews.org/zh-hans/1552715/)

![](https://assets.gnews.org/wp-content/uploads/2021/09/电荒假-1.jpg)图片来源于网络![](https://assets.gnews.org/wp-content/uploads/2021/09/1-28.png)图片来源于网络
中共国内近日大规模限电，有的企业甚至被逼停产到十一长假。壮哉中共泱泱大国，怎么就连供电这种最基本的事情都搞不定呢？文贵先生在直播中说，造成这个限电的原因，是煤价越来越高，发出来的电赔钱，每收一块钱电费就赔掉7分钱，他就不愿意发电。还有一个更夸张的事情就是污染特别严重。再加上和原材料（煤炭）供应国澳大利亚又打又闹的。而国内物价奇高，工资不涨，现在就是缺电缺得特别严重，需要1度电只能满足0.4度。

自工业革命以来，用电量成为衡量一个地区发达的指标之一。那么最起码的，不能停电。发电方式有很多种，目前较常见的有以下几种，火力发电、水力发电、核能发电、风力发电、太阳能发电、地热能发电等。随着工业结构的不断调整，以及几百年来人类文明的进步，和对“雾都伦敦”（即污染）的恐惧，人类越来越重视对环境的保护。像污染最严重的火力发电，就是靠燃烧煤/天然气/石油等来进行发电（其中煤是最主要的能源），其燃烧后产生的严重污染是大气污染的主要污染源之一。在先进发达的地区，清洁能源的使用率越来越高。

在这种世界范围的大背景下，中共国也紧跟世界的步伐。各种水电站大坝的修建，核电厂的修建，在表面上看来风光无限啊。但内部到底是什么情况？

以三峡大坝为例：中共国的三峡大坝是全世界最大的水力发电站，发电量如何？当年修建三峡大坝的时候，时任国务院总理李鹏说过，“三峡大坝建成了，大半个中国8分钱一度电”。想问问有哪个中国的普通老百姓家庭用电是8分钱一度？有没有一个？8分钱一度电的意思，就是电很多很多，用不完，很便宜。那最近怎么就给企业停电限电了？不仅仅是最近，从去年疫情以来就是。甚至，连居民用电也开始限制。秋天的广州还有30多度，老百姓家里停电了；去年年底冬天，南方有些很冷的地方也限电。老百姓夏天热死，冬天冻死。而中共给出的解释是一贯的推脱和不负责任，“这话是李鹏说的，你找他去啊”，意思就是李鹏人去世了，你还找不着人说理了。

以中共核电站为例：截止到2020年，中国核电机组数量将达到90余台，从装机容量上讲，将超过法国成为世界第二的核电大国，仅次于美国。但是发电量如何？根据国际原子能机构数据，2019年，中共国核能发电量占本国总发电量比重仅仅为4.9%。在这个图里排列是倒数。中共国引以为傲的数量最多，那些核电站到底是用来干嘛的？
![](https://assets.gnews.org/wp-content/uploads/2021/09/2-16.png)图片来源于网络
还有一个数据。2018年中国发电量，燃煤火电在中国2018年总发电量中的占比达64%。（数据来源：IHS Markit 2019年5月）燃煤火电占据这么大比例的最主要原因就是其发电成本相对最低，能赚钱。结果是现在中国发电装机总体过剩，而且还挤压了其他新能源发电的发展空间。那么一旦煤炭涨价了，就会造成用电短缺，像文贵先生讲的一样，都是假的。中共表面上好大喜功弄了很多面子工程，实际上屁用没有，大部分靠进口的能源，例如煤炭，一旦发生波动，毫无疑问一定会出问题的。

我们都知道中共假骗偷，而以上这些数据只是冰山一角。
