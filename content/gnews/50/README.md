郭文贵先前直播曾明确点出，中共欲拿下台湾必须靠“台湾卖台贼”，将有计划的引起台湾社会动乱（水电、运输、网路等问题）后，卖台贼可能擅自代表台湾人民对中共喊话，希望中共接管台湾等，为中共攻台制造理由并尽力排除国际介入可能。（中华民国与中共目前仍在国际争夺“中国合法代表政府地位”）
