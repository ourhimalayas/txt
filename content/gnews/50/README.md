**据《stripes》 作者：Daniel Flatley, Janet Lorin，2021年7月19日报道：**

美国国会旨在削弱中共国在美国招募科学家和学者的能力，作为华盛顿对抗这个亚洲国家日益增长的影响力的更广泛行动的一部分。

最近通过的一项加强美国研究和发展的众议院法案将禁止得到北京支持的科学家和学者参与美国资助的研究项目。

“多年来，国会、联邦研究机构、国家安全机构和大学一直在努力根除恶性的外国人才招聘。”提出该措施的爱荷华州共和党代表兰迪-费恩斯特拉在委员会的立法听证会上说，“现在是简单地禁止他们接受美国纳税人的钱的时候了。”

中共国外交部发言人赵立坚周一在北京举行的例行新闻发布会上回避了有关该法案的问题，说他不知道此事。

众议院通过这项限制是世界上两个最大的经济体之间关系紧张的又一个迹象，即使是在吸引了数十万中共国学生和学者来美国的学术研究层面也是如此。拜登政府正计划采取一系列行动来保护美国在该地区的利益，包括周五向投资者发出在香港做生意的风险警告。同时，国会正在采取行动，在贸易、人权和知识产权方面惩罚北京政府。

事实证明，如何应对北京通过传统间谍活动以外的手段获取敏感或专有信息的企图，对立法者和执法官员来说都是令人困扰的问题。根据司法部的数据，在所有商业秘密盗窃案中，约有60%的案件与中共国有关，在所有经济间谍案中，约有80%的案件涉及有利于中共国政府的行为。

参议院在这方面做了一个尝试，提议对来自海外的对美国高等教育机构的捐赠进行更严格的联邦审查。然而，在学院和大学抱怨其要求将是苛刻的，并最终对美国的创新有害之后，当它被纳入更广泛的立法以提高美国的竞争力时，它被搁置了。

费恩斯特拉的条款将让国家科学基金会制定一项要求，即个人必须证明他们不是“恶意的外国人才招聘计划”的积极参与者。虽然它包括朝鲜、俄罗斯和伊朗，但它显然是针对中共国的“千人计划”，以吸引美国顶级机构的科学家和企业家。中国政府支持的这一举措与几起引人注目的逮捕事件有关，其中包括哈佛大学教授查尔斯-利伯，他被指控就其参与该计划向联邦当局作虚假陈述。利伯不认罪，正在等待审判。

美国大学的贸易团体美国教育委员会说，自从中共国的人才招聘计划成为2019年参议院报告的主题后，它一直在关注千人计划，政府关系主任Sarah Spreitzer说。该报告认为，中共国一直在为自己的经济和军事利益寻求美国的研究和专业知识。

Spreitzer说，立法者们已经明确表示，他们不想停止所有的学术交流，他们理解这些研究伙伴关系和学术交流的重要性。

“我们将不得不密切关注人才招聘计划是如何定义的。”她说，“如果狭义地专注于特定的国家，那会更好。任何其他的东西，更狭隘地关注它以解决这些具体的问题，都比禁止这些国家之间的任何学术或学术交流项目要好。”

安全和新兴技术中心的研究分析员艾米丽-温斯坦说，众议院的做法消除了过于狭隘地关注中共国或千人计划所固有的一些危险，这是司法部针对经济间谍和影响行动的中共国倡议的批评之一。

“我们并不想建立我们的国家安全机构来针对一个国家，因为那样会自取灭亡。”温斯坦在一次采访中说，“这不是专门针对谁的东西。例如，这不是适用于中共国研究人员的事情。这是适用于美国的任何一个人，他们正在做违背美国国家安全利益的事情，甚至违背美国学术自由的事情。”

但是，倡导组织“正义全球”的负责人托比塔-周说，这项措施可能对研究产生有害的影响，并扼杀两国间的合作。

周说：“所有这些打击措施都是基于对来自中共国的研究人员所构成的威胁的一种夸大的感觉。我还担心，这其中有很多是来自对中共国政府运作方式的深刻误解，其中包含了对中国人和中国国家的一些坦率的种族主义假设，这些假设在很多情报机构中长期存在。”

费恩斯特拉在听证会上说，他的提案不会“禁止科学思想的合法交流和合作”，并补充说，它仍将允许研究人员参加国际会议和其他公开的研究交流；他们只是不能在接受任何有关国家的支持时这样做。

众议院正在对参议院2500亿美元的《美国创新和竞争法》进行商讨，以尽快作出回应。众议院科学委员会的最高共和党人弗兰克-卢卡斯代表说，他认为众议院和参议院的版本可以在今年9月或10月之前达成一致。除了增加国家科学基金会的授权支出和建立一个新的技术局以帮助美国与中共国竞争之外，立法者还希望附加高达520亿美元的紧急拨款，以吸引半导体制造商到美国。
