**各国央行对数字货币绝对控制的愿景**

控制狂有完全监督和控制您所做的每一笔交易的能力：如果一笔交易超出了数字货币监控算法设定的参数就可以当场取消；如果一次买卖与购买者的社会信用评分不一致则不会通过。一个避开这些好管闲事的人的简单办法就是用现金支付，以保留一定程度的经济活动隐私，因为当你用现金支付时，当局是无法监控和跟踪你的交易的，当然当局不喜欢你这样做。

通过数字货币，他们有无数种方式来控制你自己的花钱方式：通过每月数字货币定额，限制你每月只能花多少钱，也就是说，即使你账户里有大笔存款也不能随便花；你的数字货币还会被指定一个截止日期，必须在该日期前全部花掉，否则过期就作废了。

使用数字货币的时间和限额将取决于控制狂编造的经济模型，以确保什么样的支出节奏才能最好地调节经济状态：经济放缓了就把到期日提前；经济正在升温就降低人们的月消费限额。至于如何在其它县、市、州使用你的数字货币，取决于数字货币警察允不允许，甚至会受到某些处罚。

数字货币的这一系列骚操作看起来很不可思议，但只要有一点抽象的认识，这个命题就变得非常可能，尤其是当控制狂和科技精英一起公开倡导数字货币的时候。极权政府的核心是央行数字货币（CBDCs）取代现金后投放市场的推广使用，并打着消除不法交易、打击黑市、防止疫情蔓延的幌子，还称这都是为了你好。

使用CBDC和现金之间存在巨大差异。使用CBDC的一个关键区别是，央行将在有助于当局利益的法律法规下拥有绝对控制权，并在履行其责任的过程中肆意使用，且独家拥有执行这一系列监管和控制的技术。毫无疑问，控制狂想要绝对控制你的花钱方式，进而控制你的财产，因为金钱本身就是财产的一部分。你的钱代表你已经花费的时间、付出的牺牲、学到的技能，以及为赚钱和存钱所冒的风险。

控制狂想把他们的意志强加于你，替你决定如何使用和享受你的财产，但想要干扰你私生活的不只是控制狂，富有的精英阶层及其裙带资本家也很享受获得绝对控制权的大好前景。市场就像极权主义政府一样，不喜欢不确定性，想知道你到底有多少钱并想买什么，但这对我们老百姓来说却是灾难。在从CBDC的开始实施到残存的经济隐私丧失的过程中，在极权主义的控制一切的愿景完全实现之前，控制狂们下的央行行长和科技精英们是不会罢手的！

**韩国加密货币运营商须向****FSC****提交满足新合规要求的证据以申请上市许可证临近最后期限**

2021年9月24日是所有的韩国加密货币运营商向韩国金融服务委员会（FSC）提交满足新合规要求的证据，以申请正式上市许可证的最后期限。FSC要求所有交易所都必须出示使用韩国银行实名制账户的证据，因为客户对他们在小型加密货币平台上持有的资产要求更多、更高的保护。FSC称这些新措施旨在抑制韩国人对加密货币交易的热情，因为人们担心，在工资增长受到抑制、就业市场冰封而房地产价格却不断上涨的情况下，尤其是年轻一代的散户投资者，将会为了促进交易而过度借贷。

然而，除了韩国四大交易平台之外，韩国的银行大多拒绝参与任何其它交易所的风险评估程序。同时FSC的这一要求也受到不少加密货币运营商的反对。在韩国银行和加密货币运营商双方大都反对的情况下，估计韩国现有的60家加密货币运营商中，有近40家将因不能满足这些要求而被迫关闭。

最近中共当局以担心电力供应将进入即将到来的冬季用电旺季为由，冠以“加密货币活动将严重影响经济和社会发展，直接威胁国家安全，扰乱金融秩序”的借口，采取激进措施再次加大了对加密货币矿工的追捕力度。一些矿工最终离开了中共国，一些矿工则留下来并试图避开中共政府的监管。一些矿工伪装成数据存储设施里面的研究人员，试图维持挖矿业务；有些矿工却说业务并没受影响，理由是经常更换新设备，确保这些设备在一个地点不超过100台。在几个省份，针对在大学、研究机构和数据中心等场所进行非法挖矿的检查已经加强，河北省已要求挖矿企业进行自我合规检查，以确保在9月30日前停止挖矿。
![](https://assets.gnews.org/wp-content/uploads/2021/09/16-8.png)图片来源：网络
从10月份开始，中共政府将实施监控和跟踪加密挖矿活动的工具，确保不再有人继续进行加密货币的挖矿。

**比特币将如何修复货币、拯救世界**

不可否认，我们生活在一个巨大变化的时代，比特币就是其中的一部分。斯特劳斯（William Strauss）和郝伊（Nil Howe）合著的《第四次转折（The Fourth Turning）》一书涵盖了几个世纪的历史，书中阐明历史长河中每一代人的变化都很明显，60年代的人与50年代或80年代的人都明显不同，人口总数与其他因素导致社会每100年左右就会发生一次巨变。我们当下就正处在“第四个转折点”，将有更大程度的变化，很快会看到地理、社会、宗教、信仰体系的变化，可能还会看到经济和货币本身的本质变化。

早些时候，由于互联网还不够发达，比特币没有条件被广泛使用。如今，全球经济正面临着运行了半个世纪并将我们的世界推向深渊的老派法币体系的巨大变革，老派法币体系已经不可持续，因为它不仅让人们的货币贬值，而且导致了一些更严重的罪恶。在一个健全的货币体系里，人们会更加重视自己辛苦挣来的货币，可以自主保管和控制自己的钱。如果政客们不强行让人们为战争、营利性监狱和严格监管买单，人们将希望能够更加仔细地审查他们纳税的钱到底花在了哪里！ 央行的问题始于人民的货币贬值，而真正的问题是它集中了权力，创造了大量政府承包商们激烈争夺的金钱蜜罐，而他们获得不计其数的货币补贴最有效的方式是制造恐慌，这样纳税人就会把自己的钱送回这些公司，形成你不给政客钱就意味着不给暴君钱的恐怖氛围，这是及其邪恶的。而去中心化的比特币因正好可以解决这个问题而应运而生，并对各国央行失败的货币政策形成终极对冲。不管你相不相信比特币是更好的货币，我们现正处于第四个转折点都是很难否认的。20多年前开始的政府大规模支出，在2008年左右随着政府救助的无底洞开支面前，开始到达不可持续的水平。在过去的一年半时间里，人们见证了在过去50年或更长时间里所经历的最剧烈变化。尤其是去年，我们看到央行的印钞量比历史上任何时候都多，这种大规模印钞也因此催化了比特币的诞生。

![](https://assets.gnews.org/wp-content/uploads/2021/09/17-6.png)图片来源：网络

受到世界上几乎每个人关注，并有数千万人持有的比特币，从最贫穷的到最富有的，从权力殿堂到街边小店，已成了一种全球流通的货币现象。比特币不但创造了财富，而且围绕它建立了一个完整的产业。比特币在全球金融体系中的地位比大多数人意识到的还要重要，虽然其价值并没有反映在目前的市值中。尽管比特币的规模仍然小于苹果币，但它的影响、重要性和传递的信息却大得多。萨尔瓦多已经采用比特币作为该国的法定货币，一些主流银行和中间商也提供了比特币敞口及其相关产品，一个具备不同质素的比特币元素之完整生态环境已经形成。

面临着货币体系巨变的时代，已存在了12年的比特币将扮演什么角色呢？考虑到比特币在当前世界所起的作用，以及中央银行系统弱点的背景时，说比特币可能拯救人类并不为过。当前的央行体系有太多很难克服的漏洞，和平使用另一种形式的货币体系替代之，也许才是最好的解决办法。如此人民的钱将更加和平，也不再受胁迫，而且是完全自由的。这将是世界上最具意义的和平革命之一，无需一枪一炮却可以让战争和侵略这头野兽饿死，而这些战争和侵略都是由伪善的不受节制印刷的法币供血的。我们终于看到了隧道尽头的光明，终于有希望建立一个以自愿贸易交换和坚实而健全的货币为基础的社会，人们用这种货币来衡量财富，就像几个世纪前黄金在社会流通中的作用一样。稳健的货币能激励储蓄，激励明智的投资，激励人们规避风险并促进创新。

比特币是我们现在所处的技术革命中的核心部分，将会使我们的全球经济体系和货币体系发生根本性的变化。俗话说“金钱是万恶之源”，因为金钱和经济一直是世界斗争的漩涡，奴隶制、侵略战争等罪恶，以及贸易路线、艺术、语言和文化等美好事物都受到了金钱的绑架。但这句话现在未必正确，因为金钱一直是人类发展最重要的工具之一，是人们根据所提供的商品和服务来权衡、调合和分享价值的一种方式，正如几个世纪前我们看到的政教分离，我们也将很快看到金钱货币与国家政权之间的公正分离，即钱政分离。健全的货币不仅可以减弱暴君和恶官的权力，还可以增强人民的权力，可见金钱的力量是可以与政府分庭抗礼的，可聚可离，足以跟政府讲数。

自愿使用的去中心化货币将使人们变得更加自由，集中化的权力却降低了人们在社会活动中的作用和意义。解决了资金的本质问题，就能解决世界存在的大问题！

不过，去中心化的比特币看似可以解决当下绝对中心化的传统货币体系带来的极权统治和货币贬值等巨大政治经济问题，但却也带来了洗钱、资助犯罪集团和恐怖主义等同样罪恶的问题，是从一个极端走向另一个极端的本质问题。而且，这种去中心化的虚拟货币体系，理论上任何懂货币加密技术的个人和公司都可以研制出独立的一套，未来的世界将出现无数个去中心化虚拟货币体系，混乱而多样的货币系统在纷繁复杂的社会活动中，尤其是在人性的固有弱点和缺点中将是另一种灾难。同时没有现金形式的虚拟币体系，在电网维护以及其他自然灾害导致的断电状态面前也将束手就擒。只有以实物黄金和美元为双锚的、绝对中心化的新中国联邦HDO和HCN双币系统，既规避了集权和货币贬值，也通过确保进入喜联储的每一分钱都是绝对干净的途径，杜绝了洗钱等犯罪活动的出现，HDO的现金模式更是填补了断电情况下的金融交易不间断！
