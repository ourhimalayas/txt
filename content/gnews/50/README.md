据此，中共国新华社（7月20日）一篇题为《[郑州遭遇历史极值暴雨](http://www.xinhuanet.com/photo/2021-07/20/c_1127676001.htm)》消息证实郑州市遭遇一次离奇罕见（极值暴雨）强降雨。单日降雨量突破历史建站以来极值。“降雨造成郑州市区严重内涝，市内交通中断，多处小区停水停电。”

据[郑州气象局统计数据显示](https://www.chinatimes.com/cn/realtimenews/:~:text=%E9%83%91%E5%B7%9E%E6%B0%94%E8%B1%A1%E5%B1%80%E7%BB%9F%E8%AE%A1%E6%95%B0%E6%8D%AE%E6%98%BE%E7%A4%BA%EF%BC%8C%E9%83%91%E5%B7%9E20%E6%97%A516%E6%97%B6%E8%87%B317%E6%97%B6%EF%BC%8C%E4%B8%80%E5%B0%8F%E6%97%B6%E7%9A%84%E9%99%8D%E9%9B%A8%E9%87%8F%E8%BE%BE%E5%88%B0201.9%E6%AF%AB%E7%B1%B3%E3%80%82%2019%E6%97%A520%E6%97%B6%E8%87%B320%E6%97%A520%E6%97%B6%EF%BC%8C%E5%8D%95%E6%97%A5%E9%99%8D%E9%9B%A8%E9%87%8F%E8%BE%BE552.5%E6%AF%AB%E7%B1%B3%E3%80%82,17%E6%97%A520%E6%97%B6%E8%87%B320%E6%97%A520%E6%97%B6%EF%BC%8C%E4%B8%89%E5%A4%A9%E9%99%8D%E9%9B%A8%E9%87%8F%E6%80%BB%E9%87%8F%E9%AB%98%E8%BE%BE617.1%E6%AF%AB%E7%B1%B3%E3%80%82%20%E5%85%B6%E4%B8%AD%E5%B0%8F%E6%97%B6%E9%99%8D%E6%B0%B4%E9%87%8F%E5%92%8C%E5%8D%95%E6%97%A5%E9%99%8D%E6%B0%B4%E9%87%8F%E5%9D%87%E5%B7%B2%E7%AA%81%E7%A0%B4%E8%87%AA1951%E5%B9%B4%E9%83%91%E5%B7%9E%E5%BB%BA%E7%AB%99%E4%BB%A5%E6%9D%A560%E5%B9%B4%E7%9A%84%E6%AD%B7%E5%8F%B2%E8%AE%B0%E5%BD%95%E3%80%82)：

郑州20日16时至17时，一小时的降雨量达到201.9毫米。

19日20时至20日20时，单日降雨量达552.5毫米。

17日20时至20日20时，三天降雨量总量高达617.1毫米。

其中小时降水量和单日降水量均已突破自1951年郑州建站以来60年的历史记录。

据称：郑州常年平均全年降雨量为640.8毫米，此次三天的降雨量相当于以往一年的降雨总量。
