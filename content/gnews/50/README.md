![](https://assets.gnews.org/wp-content/uploads/2022/04/图片1-10.jpg)图片来源：阿纳多卢通讯社
据《国会山报》4月5日报道，周二，美国、英国和澳大利亚承诺共同开发高超音速导弹，这是去年三国宣布的安全条约的一部分。

在三国领导人的一份联合声明中，他们承诺“开始就高超音速导弹和反高超音速以及电子战能力开展新的三方合作”。

而白宫发布的一份纪要称，该三国集团将“共同努力，加速发展先进的高超音速和反超音速能力”。

报道说，为了深化在印太地区的合作，美国、英国和澳大利亚在去年9月份启动了AUKUS三边安全伙伴关系，该条约被广泛认为是为了对抗中共国的军事进步和在印太地区日益增长的影响力。

去年，参谋长联席会议主席马克·米利（Mark Milley）将军当时在接受电视采访时证实，中共国去年测试了一个高超音速武器系统。

美国和俄罗斯也曾试图开发高超音速导弹。周二，CNN报道说，美国上个月成功测试了一枚高超音速导弹，但为了避免在乌克兰战争中与俄罗斯的紧张局势升级，美国一直将其保密。

报道指，作为伙伴关系的一部分，美国和英国正在帮助澳大利亚获得核动力潜艇。去年宣布该协议的这一内容时激怒了法国，因为它导致巴黎失去了向澳大利亚提供潜艇的有利可图的国防合同。

此外，通过安全伙伴关系，这三个国家还在网络、人工智能和量子技术等新兴技术方面进行合作。

根据周二的联合声明，拜登总统、英国首相鲍里斯·约翰逊（Boris Johnson）和澳大利亚总理斯科特·莫里森（Scott Morrison）评估了该协议的进展，并申明了他们对“自由和开放的印太地区”的承诺。

声明说：“鉴于俄罗斯对乌克兰的无端、无理和非法入侵，我们重申我们对一个尊重人权、法治和不受胁迫地和平解决争端的国际体系的坚定承诺”。
