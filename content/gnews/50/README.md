据《网关专家》7月13日报道，根据联邦“疫苗不良事件报告系统”（VAERS）的数据显示，这周全美共有9125人因接种CCP病毒疫苗而死亡。而上周，他们报告了6985人死亡，一周内增加死亡人数2410人次。 此外，上周美国因CCP病毒死亡的人数有1505人。

今年与疫苗有关的死亡人数绝对激增。根据美国疾病控制与预防中心自己的数据，2021年前3个月，美国因接种疫苗而死亡的人数超过1750人，而现在这个数字激增到9195人。

“疫苗不良事件报告系统”（VAERS）数据库包含使用美国许可的疫苗进行免疫接种后未经核实的不良事件报告（疾病、健康问题和/或症状）的信息。此接受任何人的报告，并可通过电子方式提交。目前已经有411,931份关于疫苗的不良反应报告。

这些数据报告表明，美国上周死于疫苗的人数超过了美国上周死于病毒的人数。
