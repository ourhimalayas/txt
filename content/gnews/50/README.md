## 【每日聚焦】俄乌战事4月17日简要

1、基辅和乌克兰东北部：在过去24小时内，该地区没有重大变化。

2、哈尔科夫和伊久姆：哈尔科夫市周围的俄军还守着阵地，尽管乌克兰联合行动总部的指挥官在4月16日声称，乌军在该市周围重新夺回了一些领土。俄军继续在伊久姆及其周围集结，包含了来自此前在基辅周围作战的部队，在哈尔科夫及周围地区活动了一段时间的第6、第20集团军以及第1近卫坦克军得到了第35、第36集团军以及第106空降师的增援，这些增援部队此前都在基辅周围作战并且战损严重。此外还有俄东部军区第68军团所部，可能是来自驻扎在萨哈林岛的第39独立摩步旅，也在乌克兰东部活动，据乌克兰总参谋部报告，这支部队此前可能没有参加在乌克兰的战斗，因为他们原定的进入叙利亚的任务被取消。乌克兰总参谋部声称，俄军在伊久姆周围集结了多达22个营级战术群，准备用于向斯拉夫扬斯克和巴文科伏进攻。乌克兰总参谋部报告说，昨天俄军向伊久姆以南大约20公里处的多夫根斯克村发动了一次略有规模的进攻，但是被击退。

3、顿涅茨克州和卢甘斯克州：俄军继续炮击波帕斯纳亚、鲁别日诺耶和北顿涅斯克，并且对波帕斯纳亚周围进行了小规模的战术攻击，然而，过去24小时内没有在这些地区取得明显的进展。乌克兰总参谋部报告说，俄军4月16日在阿瓦迪夫卡及其周边地区进行了炮击，可能是在为该地区的进攻行动做准备。阿瓦迪夫卡就在顿涅茨克市以北，在通往斯拉夫扬斯克的公路上，俄军在这个方向进攻，可以呼应从伊久姆朝斯拉夫扬斯克方向的进攻。

4、马里乌波尔：战斗在马里乌波尔中部继续进行，俄军进入到乌克兰国民警卫队第12旅的基地，夺取了马里乌波尔中部相对靠近海滩的一个警察站。乌军则继续进行小块防御，特别是在亚速工厂，俄军继续猛烈攻击。俄罗斯及其代理人似乎正准备宣布马里乌波尔战役的胜利，因为顿涅茨克人民共和国领导人丹尼斯-普希林在该市开设了统一俄罗斯党办公室。

5、南部战线：俄军炮击了赫尔松和第聂伯罗彼得罗夫斯克地区边境的检查站。

6、乌克兰击沉”莫斯科”号是一个重大事件，会引发俄罗斯加大空袭和导弹攻击进行报复。尽管如此，因为俄罗斯黑海舰队在几周前就把陆战队投入到马里乌波尔周围的战斗中，这意味着俄罗斯海军在接下来的战争中不太可能有更多的力量来发挥更大的作用。因此即便俄军加大空袭和导弹攻击也不太可能对战争的结果产生决定性的影响，因为没有理由相信俄罗斯能够一直拥有足够的空中力量和导弹储备，能够足以改变平衡。所以战争到了这一阶段，其决定性战斗仍将在乌克兰东部的地面上进行。
