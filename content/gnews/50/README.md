## 美国将反击中共钢铁产业的不公平竞争
` GTV新闻访谈` [轉載自GNews](https://gnews.org/zh-hans/1637764/)

作者：纽约香草山农场 – 007
![This image has an empty alt attribute; its file name is Screen-Shot-2021-08-30-at-7.37.59-PM.png](https://assets.gnews.org/wp-content/uploads/2021/08/Screen-Shot-2021-08-30-at-7.37.59-PM.png)
11月2日，美国贸易代表戴琪出席了由美国钢铁协会和美国钢铁制造商协会共同在华盛顿举办的大会。会上，戴琪指出，美国政府将全力组织中共过剩的钢铁进口产品对美国的钢铁产业和市场造成损害，同时将努力提振美国钢铁产业的竞争力。

![](https://assets.gnews.org/wp-content/uploads/2021/11/Slide6-2.jpeg)

戴琪表示，钢铁产业是制造业的基础，过去20年间，中共政府通过巨额补贴、牺牲环境、劳工剥削和低价倾销等不公平的竞争手段，使得中共的钢铁产量猛增，但这是不可持续的。

对此，戴琪补充道，美国将动用并开发各式新工具，同时通过与其他经济体和国家合作，保护美国免受多年来因中共不公平竞争而造成的损害。
