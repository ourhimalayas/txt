据《大纪元》2021年7月4日报道，在7月1日举行的中共成立100周年的庆典活动中，中共总书记习近平提到要让“外来势力”“头破血流”。这番言论引发国际热议。7月4日，前美国国务院发言人摩根‧奥塔格斯（Morgan Ortagus）在接受《福克斯》采访时，评析了习近平的讲话。

习近平在讲话中称，“绝不允许任何外来势力欺负、压迫、奴役我们。谁妄想这样干，必将在十四亿多中国人民用血肉筑成的钢铁长城面前碰得头破血流！”

《福克斯》的主持人表示，习近平的话听起来很咄咄逼人。中共似乎想成为统治世界的超级力量，显示出中共不仅要在地区称霸，还有全球野心。

奥塔格斯对此评论说：“我们从战略角度看，习近平通过百年（庆典上）的讲话想传达什么信息？在80、90年代甚至21世纪初，中共的策略是‘韬光养晦’，安静地‘崛起’，不在国际舞台上兴风作浪。现在他（习近平）说：‘我们不能被欺负’，要在国际上发挥影响力。”

奥塔格斯还抨击了奥巴马和拜登政府的对华政策，因为奥巴马政府认为中共崛起和美国衰落是“不可避免的”。

奥塔格斯回忆说，川普政府上任时，川普总统和蓬佩奥国务卿都拒绝了这种理论，他们说：“我们不接受这种说法，我们不相信中共会接管我们，而我们只能‘不可避免’地衰落。”

6月中旬，奥塔格斯在《福克斯》的另一次采访中提到，在川普总统真正开始打击中共威胁之前，世界上大部分地区，包括美国两党都没有意识到这种威胁。她说：“这种（中共崛起、美国衰落的）想法，我认为拜登政府也大部分继承了这种想法。”不过，她在7月4日的采访中表示，拜登政府现在也对中共“做出很多强硬姿态”了。

“一定要记住这一点，这（应对中共）将是我们这个时代的挑战，”奥塔格斯表示：“华盛顿的所有人要清醒过来，面对新的现实。”

习近平在百年讲话中谈到一系列话题，其中包括香港、“统一台湾”等。奥塔格斯对此表示：“我认为习近平不仅仅是在测试美国，也在测试全世界。他们夺走了香港的民主自由，而川普政府因此对（中共）进行了制裁。”

奥塔格斯表示，目前一些美国企业还在迎合中共，而美国需要团结一心对抗中共。她说：“所有的公共部门和私营部门都要站起来对抗它们。”

奥塔格斯还指出，在中共统治中国的100年中，仅“大跃进”就导致了四千五百多万中国人被迫害致死。100年中还有强制堕胎；100年来的酷刑、迫害；以及对人权的完全无视。

奥塔格斯最后说：“中共在百年之际威胁台湾，威胁香港，威胁我们。我们需要记住，中共代表着什么？他们代表着100年来对本国人民的暴政。”

习近平的“七一”讲话杀气腾腾，完全是一副向全世界“亮剑”的姿态。真不知他究竟有何底气可以如此“自信”。不过据悉，习近平讲话的英文版，里面的“战狼”字眼都被替换弱化了。这种掩耳盗铃的做法究竟想要达到什么目的？难道全世界的政府和媒体都没有能熟读中文含义的人吗？笔者认为，这不过是中共在末路狂奔中对国内百姓的又一次愚弄，是中共煽动民族主义，绑架十多亿国民垂死挣扎的最后的疯狂。

由此也可以看出，中共虽然嘴上强硬，但背后所透出的其实是无比的虚弱和胆怯，而这也是中共快速走向灭亡的前兆。
