澳大利亚[联邦贸易部长丹·特汉（Dan Tehan](https://cn.theaustralian.com.au/2021/07/08/55058/)）上周与台湾贸易部长邓振中举行会谈，会议内容的主题是双边能源贸易，其中包括澳洲氢气出口的潜力。这是继4月份以来，澳洲与台湾贸易部长的第二次会谈。

早在2017年，澳洲政府就曾考虑与台湾签署自由贸易协定(FTA)，但当时澳洲受到中共外交部发言人的警告。中共[外交部发言人陆慷](https://www.sohu.com/a/205746737_114731)表示：“坚决反对中共国的建交国与台湾当局进行任何形式的官方接触和往来，坚决反对中共国的建交国与台湾当局商签任何具有主权意涵和官方性质的协定。我们希望有关国家充分认识到台湾问题的敏感性和复杂性，切实恪守一个中国原则，慎重、妥善处理好有关问题。”

鉴于中共的威胁，澳洲自2015年与中共国签署双边自贸协定后，就一直与台湾保持一定的距离，以免触怒中共当局。然而近两年来，澳洲因为提出要对中共病毒的源头进行独立调查而不断受到中共的威胁和报复，自此中共与澳洲的关系跌入冰点。在中澳之间无话可谈的情况下，澳洲不再理会中共，决定再次推动与台湾的自由贸易协定谈判。

台湾是澳洲的十大贸易伙伴中，唯一没有与之签订自由贸易协定的。澳洲和台湾官员一直在讨论达成自由贸易协定的可能性，同时，台湾也表示希望加入11国的《跨太平洋伙伴关系全面进步协定》。

澳洲政府中许多人都赞成与台湾达成自贸协定，原因有二：一是台湾的市场很重要；二是澳洲已经没有什么可以失去的了。

[尽管丹·特汉拒绝评论这次会谈](https://cn.theaustralian.com.au/2021/07/08/55058/)，但特汉的发言人表示：“特汉部长与台湾的会谈主要集中在能源方面。” 澳洲驻台湾办事处代表珍妮·布卢姆菲尔德（Jenny Bloomfield）确认，双方的讨论富有成效。

“重启和台湾的自由贸易谈判合乎澳洲一直提倡的贸易开放原则”，悉尼大学商学院[李薇](https://www.dw.com/zh/%E6%BE%B3%E5%A4%A7%E5%88%A9%E4%BA%9A%E5%92%8C%E5%8F%B0%E6%B9%BE%E7%AD%BE%E7%BD%B2%E8%87%AA%E8%B4%B8%E5%8D%8F%E5%AE%9A%E6%98%AF%E5%90%A6%E6%9C%89%E5%A5%BD%E5%A4%84/a-56532039)博士在接受《德国之声》采访时表示，“澳洲与台湾之间的贸易谈判符合澳洲的经济利益。澳洲与台湾的经贸关系互补性强。双方的贸易协定会减少中澳贸易摩擦对澳洲出口商的压力，比如目前受中共国制裁的煤炭、牛肉、小麦等，未来可以转销到台湾市场。台湾对澳洲的出口主要集中在电子和交通产品，例如电信设备、计算机以及摩托车和自行车。澳洲向台湾主要出口产品是资源类产品，例如煤炭、铁矿石、天然气和铜。”

从澳大利亚与台湾的自由贸易谈判会议可以看到，澳洲政府已经无惧来自中共的压力，坚决地与中共政府抗衡，因为与一个有民主自由法治的台湾政府打交道，远远胜过中共这样的流氓政府。而中共已经在经济上对澳大利亚用尽了所有可用的阴招，另一方面又不得不高价从澳洲大量采购铁矿石等必需产品，在这场没有硝烟的经济战，最后失败的只会是中共！
