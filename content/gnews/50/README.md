中国郑州发生号称“千年一遇”暴雨，整个城市泡在水中，地铁浸水，车厢内有乘客淹死以及乘客求救的视频传出后，牵动了海内外民众的心。

目前官方声称，因极值暴雨致25人死亡7人失联，但外界认为，光是在地铁内的死亡人数就不只这个数。许多网友在微博中讽刺，当天打开电视收看中国官媒《央视》，本想关心郑州灾情，“我以为是郑州，仔细一看是欧洲”，竟然在报导欧洲洪灾； “最不要脸的喉舌，没有之一”；“这政府对人民哪有半点的责任心？”。

而中共官媒央视、人民日报、环球时报等不仅轻描淡写地报导河南洪灾，而且更是大篇幅地调侃欧洲抗洪不力，满屏皆是“忙着关心意大利的水灾”。同时，中共又开始抓“不当言论者”和辟谣，“享道出行”微信公从号首当其冲。有网友指出：惨痛的损失，不追究救援不力的责任，反而却盯所谓的不当“言论者”？让人不适的是政府的无为……

关于此次水灾，共产党处理方法一如既往，首先。选择性断电断网络，防止信息外泄，威胁知情者和受害人家属，绝不允许将真实情况外露，严厉封锁消息，如不配合下场很严重。郑州“享道出行”刚刚说出一点真相就被迫道歉，道歉的内幕虽不得而知，但是按照邪恶中共一贯做法，逼迫威胁，否则就会运用国家力量，是公司就让公司不仅破产还要负债累累，名誉扫地，可能还有牢狱之灾甚至生命消失。是个人的话，那更是不堪一击，如武汉疫情初期，2019年12月30日李文亮医生他在一个微信群中发布“华南海鲜市场确诊7例SARS”的消息。四天后，他受到当局的警示和训诫。2月6日在医院去世，死因一直受质疑，生前被封口，死后遭封网。

同样在武汉疫情期间，1月23日武汉被封城，城内的民众受困，城内、城外的民众无法了解城内疫情的信息，那时武汉女作家方方用日记的形式记录了她的所见所闻，为城内城外的民众提供了难得的信息，方方日记曾在海内外广受喜爱，得到网民的追捧和传播。但是方方日记在海外出版的消息却引发轩然大波，短短几天便从受追捧的偶像成为中国的“叛徒”、“卖国贼”、“给西方递刀子的人”。究其原因，国际上五眼联盟要求追查病毒起源是否起源于武汉实验室，并且要求赔偿，中共为转移群众的恐惧、愤怒和质疑的情绪，急需树立一个“假想敌”、“内奸”转移视线。于是中共动用宣传工具，利用五毛们和左派们的文革思维，引导普通民众在网络上对方方进行了铺天盖地的谩骂和攻击，致使方方身心疲惫，生不如死。
