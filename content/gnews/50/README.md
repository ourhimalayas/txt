## 战争大救援历史再现-危难中展现人性光辉

作者：纽约香草山农场教育部 – 马克文/朱丽叶
![](https://assets.gnews.org/wp-content/uploads/2022/01/1-GTV-Gnews-HEADER-8.png)
二次大战期间，纳粹德国对欧洲诸种族及政治群体展开迫害，犹太人遭到广泛系统性屠杀。当时欧洲共有近900万犹太人，其中近三分之二被害，包括近150万儿童。在犹太民族将被灭绝的危难时刻，发生了许多不顾个人生死，救援犹太人的故事，在危难中展现了人性的光辉。根据史料记载：
![](https://assets.gnews.org/wp-content/uploads/2022/03/Capture00-34.jpg)
记录1

德国纳粹对波兰人严格规定，凡是对犹太人提供任何形式帮助的波兰人及其家人，均会被判处死刑。二战期间，有300万非犹太波兰人被杀害，其中多达5万人因救援犹太人而遭处决，波兰成为二战拯救犹太人最多的国家。战后以色列把保护犹太人的波兰人认定为国际义人。

记录2

中华民国奥地利外交官何凤山不忍看着犹太人在维也纳等死，勇敢地打开了向犹太人发放签证之门，给犹太人签发数千个赴中国的“生命签证”，当时的上海犹太难民顶峰时多达3万人。战后，何凤山被以色列授予最高的荣誉——国际义人。以色列在耶路撒冷为何凤山建立纪念碑，碑上刻着“永远不能忘记的中国人”，联合国正式誉何凤山为“中国的辛德勒”。

记录3

奥斯卡·辛德勒，德国商人。他在二战中拯救超过1100名他工厂里的犹太人。

2022年2月24日，俄罗斯入侵乌克兰，总统泽连斯基率领乌克兰人民拼死抵抗。战争造成几百万难民背井离乡，其中大多数是儿童、老人和妇女，大量百姓死于战火之中。

正当民众被困乌克兰，孤立无援之际。新中国联邦、法治基金、法治社会立即开展组织国际大救援行动。他们租用大巴车深入乌克兰战区，几天就救出几百个儿童、妇女、学生等难民，转运到波兰、西班牙等国。目前大救援还在继续中，一批批救援队赶赴救援前线，救援行动计划租用上百辆大巴车、数十架飞机做百次飞行，将把有需要的民众送往美国和加拿大，以及亚洲安全的国家。一场人类史上大规模的战区国际大救援正在展开，新中国联邦人展现出的人性光辉，再次照亮世界。
