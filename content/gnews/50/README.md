**1.日本对科技前沿领域的留学生采取更严格的入境审查程序，主要针对中共国**

[据读卖新闻7月20日的报道，日本政府已经于今年春季开始更严格地对在国内大学学习的外国人留学生进行入境审查。在日本大学研究可转用于军事目的的先进技术时，大学不仅要对其学术背景和工作经历进行汇报，还要对资助其留学和研究的相关组织和企业进行报告。这一政策主要是针对中共国，防止可转用于军事目的的先进技术外流。过去，大学只需向该机构提交接收留学生的同意书和描述其最终学术背景的简历。在新的政策下，从事火箭材料和人工智能等领域的留学生将被要求提交过去的教育背景、工作经历、出身组织和雇佣关系等详细背景信息。](https://news.yahoo.co.jp/articles/06eee08322abddf4539afb766ad1e0e27e86cba8)

**2****.****又有9名东京奥运会相关人员确认感染中共病毒**

[据每日新闻7月20日的报道，东京奥运・残奥会组委会20日宣布，又有9名奥运会相关人员中共病毒检测呈阳性。除了一名停留在奥运村的海外运动员外，首次发现一名志愿者也被感染病毒。至此，东京奥运会相关人员共有67人确认感染中共病毒。组委会表示将不会公布感染者的性别和国籍。经国际体育组织同意，在比赛开始前6小时内进行PCR检测后呈阴性才可以参加比赛，而呈阳性的运动员在隔离期间不能参加比赛。](https://news.yahoo.co.jp/articles/e8bca9c550254990ac83f2b3235d54e00c1accf9?tokyo2020)

**3.疑似是失踪的乌干达运动员的男子在三重县境内受到警方监护**

[据关西电视台7月20日的报道，一名似乎是失踪的乌干达运动员的男子于20日在三重县受到警方监护，失踪前他在大阪府泉佐野市参加东京奥运会前的集训。失踪的男子是20岁的举重运动员Julius Ssekitoleko (20岁)。7月16日，Ssekitoleko从大阪府泉佐野市的酒店失踪，房间里留下写着“我在乌干达生活很艰难，想在日本工作”的纸条。据搜索人员称，20日下午3点左右，一名被认为是Ssekitoleko的男性在三重县四日市南警察局受到监护。](https://news.yahoo.co.jp/articles/709d2dfb0fe2f369ec82bdbb755e81923054f721?tokyo2020)

**4.大阪市处分1109名政府职员，因在疫情期间违反聚餐规定**

[据产经新闻7月20日的报道，因在疫情期间违反聚餐规定，大阪市20日宣布对1109名市政府职员进行处分。当时为防止中共病毒疫情蔓延，大阪市曾呼吁市民不要多人数聚餐。市政府对今年3月1日至4月4日的期间内“5人以上”或“晚9点后”的违反聚餐规定的行为进行走访调查。市政府表示，“对职员多人数聚餐的行为感到遗憾，并深表歉意。我们将更加努力确保职员遵守规定。”](https://news.yahoo.co.jp/articles/9c6ed1c2782702ec9f5ddd997d1859c1ca5ef198)

**5.****日本三大经济团体领袖均将缺席奥运开幕式**

[据共同社7月20日的报道，日本经济团体联合会会长十仓雅和在20日的例行记者会上表明了不出席东京奥运会开幕式的意向。日本商工会议所和经济同友会也已分别透露主席三村明夫和代表干事樱田谦悟将缺席。至此，奥运开幕式上三大经济团体领袖将一同缺席。东京奥运会和残奥会的最高级别赞助商松下也于同日表示，社长楠见雄规将放弃出席。此外，丰田汽车于19日宣布社长丰田章男等相关人员将缺席。经济界与东京奥运保持距离的动向愈加明显。](https://nordot.app/790131032027217920)

**6.****疫苗接种证明可在30多个国家使用**

[据共同社7月20日的报道，一位政府官员20日透露，政府即将签发用于证明接种中共病毒疫苗历史的“疫苗护照”，预计可在30多个国家使用。部分已达成协议的国家将于21日在外交部网站上公布。而与日本有频繁商业人员往来的美国和中共国的协调目前并不顺利。疫苗护照旨在帮助赴海外人员入境审查和旅馆入住。拥有者具有可免除入境隔离或有所放宽的优势。本月26日起可在市政府申请发行，费用将由政府承担。](https://nordot.app/790194947331538944)
