新浪网7月19日转载中国新闻网消息，中共商务部公布将在上海、北京、天津、重庆、广州五市率先开展国际消费中心城市培育建设。

2019年10月，中共商务部等14部门联合发布《关于培育建设国际消费中心城市的指导意见》，提出在5年内建成若干消费潜力大、国际化水平高、面向世界的综合性国际消费城市。《指导意见》对中美贸易战印发的国内经济衰退有着明显的“前瞻性”指导，但此后随之而来的2020年CCP病毒大流行导致全球经济停滞；受病毒溯源和种族灭绝罪行影响，越来越多的国家选择疏远和减少与中共的贸易来往，中共的外向型经济依旧收到严重冲击。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/07/20155903/Screen-Shot-2021-07-20-at-3.48.04-PM.png)
2020年下半年，中共为了促进国内资本流通，在疫情阶段性区域爆发的前提下大搞倡旅游促消费，一定程度上有助于“内循环”。然而，在种族灭绝罪行的“法定”下，美国、欧洲这两个最大的中共贸易伙伴关系先后“告吹”，而日本也在虚与委蛇下逐步与中共经济贸易“脱钩”，大量日企撤离。其次，各国政府对CCP病毒真相洞若观火，而民意和与之对应的制裁法律手段逐渐健全，中共与各国的关系将不可避免陷入对立，在此大形势下，外资的削减以及对中共一带一路等渗透手段的剪除行动也层出不穷，中共市场对外资的吸引力也越来越低。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/07/20155852/Screen-Shot-2021-07-20-at-3.48.08-PM.png)


再者，虽然中共有14亿韭菜和庞大的市场规模，但一方面因教育、社保、医疗等开支加大造成的社会矛盾增多，维稳经费的逐年增大依赖于外资的持续供血，而外资的撤离势必导致中共政府高压态势难以运营。另一方面中共需要庞大的外资用以支付能源、物资的采购，而疫情因素以及中共的公私合营都在盘剥外贸型企业的利润，中小型外贸企业纷纷破产倒闭，在此形势下，外汇趋紧程度更加严峻。最后，由于国际市场大宗商品价格上涨，由此引发的生产原料价格上涨，对国内的初加工和制造企业而言，生产成本的加大造成终端消费商品价格攀高，民众的日常消费金额上涨、开支加大，而收入减少，都预示着报复性消费的“短暂性”，难以长久。在缺乏外资供给，以及中共横征暴敛下，底层民众的财富被更大力度的剥削，用以消费的结余越来越少，最终导致无钱消费，主打国际消费中心的奢求也只能“胎死腹中”。
