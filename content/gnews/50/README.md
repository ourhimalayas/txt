## 美军增派25架F-22进驻关岛，百架五代匿踪战机云集印太

撰文：台湾宝岛农场 快慢机
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/07/19100837/%E8%9E%A2%E5%B9%95%E6%93%B7%E5%8F%96%E7%95%AB%E9%9D%A2-2021-07-19-220532.jpg)
美军夏威夷太平洋空军司令部日前对外界透露，将于近期增派约25架F-22“猛禽”战机，部署至印太防区的关岛基地。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/07/19100846/%E8%9E%A2%E5%B9%95%E6%93%B7%E5%8F%96%E7%95%AB%E9%9D%A2-2021-07-19-220546.jpg)
据悉，美军就本轮F-22“猛禽”战机，大规模增派至印太防区的调动部署，所给出的官方解释为——执行“太平洋钢铁-2021”联合演习的任务需要。该批F-22将于近期陆续抵达，美军位于西太平洋地区的关岛基地展开部署。

参与上述移防的这25架F-22战机，此前分别部署于夏威夷西卡姆空军基地，以及阿拉斯加的埃尔门多夫.理查森联合基地。

夏威夷的卡西姆空军基地，距中共大陆的距离约8000公里左右。驻守着美空军第19与199空中打击中队，装备有近20架F-22。

阿拉斯加的埃尔门多夫.理查森联合基地内，距中共大陆的最短距离约5000公里左右。驻守有美空军第90、302及525联合空中打击中队，配备至少40余架F-22。

美军目前共装备187架，F-22“猛禽”型五代匿踪隐身战机。其中绝大部份均部署于美国本土进行防御。这次美军极为罕见的，从美国本土空军基地，一次性抽调近25架F-22战机，前突进驻印太防区的美军关岛基地展开部署。

如果仅仅是为了执行“太平洋钢铁-2021”演习行动的话，大可不必如此兴师动众。因为在夏威夷及日本冲绳嘉手纳空军基地内，美军就常备保持着约30架左右的F-22战机用于日常战备警戒。所以，这次美军向印太防区内，一次性增派如此规模的F-22战机行动，更大的背后主因极有可能是，为了保持在该地区对于中共的绝对空中优势，所展开的战略部署调整。

除了先进五代机翘楚的F-22外，另外一款目前正在加速大量装备，美军及其北约盟军的， F-35系列战机。也已陆续在印太防区投入实战化部署。
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/07/19100855/%E8%9E%A2%E5%B9%95%E6%93%B7%E5%8F%96%E7%95%AB%E9%9D%A2-2021-07-19-220600.jpg)
F-22由于其所装备搭载的超前科技过于先进，因此被美国军方严令出口。不过另外一款颇具代表性的，美制先进五代主打型战机F-35，则被美军及其北约盟军所大批量争先采购及装备。

就匿踪性隐身而言，F-22属于几乎360度全向型隐身。而F-35则属于相对单一的正向型隐身。简单来说，F-22可以做到近乎360度全方位，规避敌军雷达扫描搜索；而F-35的机身正面是反雷达匿踪效果最强的扇区，其它方向上的隐身效果相对于正面，则有些许差距。

对比F-22高昂的造价与后期维护成本、以及单一的陆基起飞模式而言。F-35相对低廉的制造与后期保养成本，以及其所衍生出的后续三个不同版本机型，极大的扩充了该机遂行多种任务的能力。

F-35A与F-22同属于传统陆基起降型，主要装备于空军；F-35B属于垂直短距起降型，主要装备于海军陆战队；F-35C属于舰载型，主要装备于海军航母战斗群。

截止目前，印太防区内的美、日、澳、韩四国，至少实际完成列装部署F-35系列战机逾百架。其中近半数将部署于海上移动两栖攻击平台之上。对比中共眼下屈指可数的所谓五代机歼-20而言，空中碾压态势已基本毫无悬念。
