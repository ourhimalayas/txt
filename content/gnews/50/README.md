## 中共军方公然违反国际海庭对南海主权裁决，裁决周年之际 &#8220;驱赶&#8221;美国军舰

#### 责编：胡天
![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/07/12105557/1.png.jpg)图片来源：路透社
7月12日据路透社北京报道，中共军方表示，在国际法庭裁定对南海主权裁决五年后的今天， “驱赶 “了一艘所谓非法进入中国西沙群岛附近水域的美国军舰。

中国人民解放军南部战区司令部在一份声明中说，美国本福号军舰未经中国批准进入该水域，严重侵犯了中国的主权，破坏了南海的稳定。我们敦促美国立即停止这种挑衅行为。

2016年的7月12日，海牙常设仲裁法院裁定中国对南海没有历史所有权，但中共表示将不会理会该裁决。

美国海军在一份声明中说，本福尔德号（Benfold）在西沙群岛附近的主张符合国际法的航行权利和自由。根据《海洋法公约》所反映的国际法，所有国家的船只，包括其军舰，都享有无害通过领海的权利。

美国国务卿安东尼布林肯在周日声明中说，海洋自由是所有国家的 “持久”利益，中华人民共和国继续胁迫和恐吓东南亚沿海国家，威胁着这一关键的全球通道的航行自由。

而中国外交部发言人赵立坚在周一的例行发布会上说，美国正在破坏该地区的和平与稳定。

中共已经发出信号决心占领台湾，这已经使台湾海峡成为世界的主要焦点，台湾海峡形成了美国和中共国之间的主要对抗点。而且中共国显然并不仅于此，今日又在南海主权裁决周年之际标志性地 “驱赶”美国军舰，显然它想把南海也变成一个战争热点，把美国和亚洲及周边的国际军事力量卷进这个漩涡。这场战争可能比我们想象的要早来到。
