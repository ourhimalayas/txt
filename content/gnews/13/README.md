4月13日，英国皇家属地的泽西岛法院下令，冻结俄罗斯富豪阿布拉莫维奇拥有的超过70亿美元的资产，这是俄罗斯入侵乌克兰后，英国政府对俄罗斯实施的制裁。

阿布拉莫维奇是英超切尔西俱乐部的老板，据说他与普京政府关系密切。

泽西岛是一个“避税天堂”，据英国《金融时报》报道，此次冻结的资产数额堪比瑞士当局迄今扣押的 900 名俄罗斯人的总资产，未来，受英国政府影响的避税天堂可能会加强对隐藏资产的俄罗斯人的制裁。
