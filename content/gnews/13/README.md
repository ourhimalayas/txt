## 数千名拒绝接种疫苗的情报人员可能被开除

翻译：Lulumi

![](https://assets.gnews.org/wp-content/uploads/2021/11/Representative-Chris-Stewart-700x420-1.jpg)2019年11月19日，众议员克里斯·斯图尔特（犹他州共和党）向前乌克兰特使库尔特·沃尔克大使和前国家安全委员会官员蒂姆·莫里森提问，他们在华盛顿特区国会山的众议院情报委员会作证。(Jacquelyn Martin/POOL/AFP via Getty Images)

一位美国国会议员对未接种疫苗的情报人员可能被解雇表示担忧，如果他们不能遵守拜登政府本月晚些时候为文职人员接种COVID-19疫苗的最后期限。

犹他州共和党众议员克里斯·斯图尔特（Chris Stewart）周五说，美国所有18个机构的情报部门的COVID-19疫苗接种率是不同的。

“你有可能在同一天解雇成千上万的人。而且它不像你在Craigslist上发布广告，让人们在周四之前申请。”斯图尔特说。

“我的问题是，如果我们这样做，对国家安全有什么影响？”他对可能解雇数以千计的情报人员提出质疑，由于他们从事的是高度专业化的工作，以及完成安全审查的困难，他们特别难以被取代。

斯图尔特引用了他说已经提供给众议院情报委员会但尚未公开发布的信息。

据美联社报道，这位共和党立法者说，截至10月底，一些情报机构至少有20%的工作人员没有接种疫苗。在其他机构，多达40%的情报人员仍未接种疫苗。

在给《大纪元时报》的一份电子邮件声明中，斯图尔特办公室的一位发言人证实，美联社发表的报道中所列数字是准确的。

该发言人写道：“不幸的是，任何比这更具体的东西都是机密。”总统乔·拜登已下令所有文职雇员，以及那些受雇于联邦承包商和接受联邦支持的实体，在11月22日之前全面接种疫苗，这与政府的许多病毒恢复计划有关。

![](https://assets.gnews.org/wp-content/uploads/2021/11/president-biden-2-1200x800-1.jpg)2021年11月6日，美国总统乔·拜登在华盛顿白宫国宴厅讲话时回答了有关美国边境的问题。(Alex Brandon/AP Photo)

没有接种疫苗或在11月最后期限前没有获得豁免的联邦雇员可能面临14天或更少的停职，随后可能被解雇。

国家情报局局长拒绝了宣布在18个机构的情报界工作的大约10万人的总体疫苗接种率的请求。

中情局上周公开披露，其97%的工作人员接种了中共病毒疫苗。负责运营美国间谍卫星的国家侦察局报告说，疫苗接种率为90%。

拜登政府的国家情报办公室主任艾弗里·海因斯（Avril Haines）上周拒绝说明该机构的工作人员中有多大比例已经接种了中共病毒疫苗。

![](https://assets.gnews.org/wp-content/uploads/2021/11/Avril-Haines-1200x800-1.jpeg)2021年10月27日，国家情报局局长艾薇儿·海因斯在中情局局长威廉·伯恩斯旁边，在华盛顿国会山的众议院情报委员会听证会上作证。(Jacquelyn Martin/AP)

拜登的疫苗接种强制令在全国范围内面临巨大阻力，在全国各地引发了抗议。包括芝加哥、丹佛、西雅图和纽约在内的多个城市也已下令员工接种疫苗。10月下旬，纽约数以千计的警察、消防员和其他城市雇员走上街头抗议。

自由律师事务所等公民自由团体报告说，有数以千计的联邦军事和文职人员以及企业雇员与他们联系，寻求法律援助，以医疗或宗教理由豁免这些疫苗强制令。

众议院情报委员会的民主党人说，他们相信绝大多数情报工作者会遵守拜登的疫苗强制令。

众议员杰森·克罗（Jason Crow）在接受采访时说：“如果有人不愿意做必要的事情来保护他们自己的健康和他们单位的健康，这实际上是对他们有效开展工作的能力提出了质疑。”

前美国空军成员斯图尔特（Stewart）说，他自己已经接种了CCP病毒疫苗，但他反对政府强制接种疫苗。

他说：“如果你说，‘你必须这样做，我们不会考虑任何例外情况’，这就是你让人们钻牛角尖的地方。”
