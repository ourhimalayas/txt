据《六度网》11月19日消息，《法新社》当日报道称，联合国周五呼吁中共提供证据，证明中共国网球选手彭帅安好。

联合国人权事务高级专员办事处发言人莉兹·特罗塞尔（Throssell）11月19日在日内瓦举行的新闻发布会上表示，掌握有关彭帅的下落以及她是否安全的证据，十分重要。

特罗塞尔同时代表联合国有关组织，敦促中共对彭帅的性侵犯指控进行全面和透明的调查。

《法新社》的报道指出，35岁的彭帅不久前在社交媒体上指控中共前副总理张高丽强迫她发生性关系，并让她成为他的情妇。张高丽在2013年至2018年期间，是中共国最有权势的中共中央政治局七名常委之一。

在彭帅发出对张高丽的指控后，中共的网络审查人员很快从社交网络上删除了任何有关彭帅指控的痕迹。她最初在微博上发布的帖子也被删除了。从那时起，彭帅就再没露过面。

最近几天，世界网球界诸多名人纷纷在Twitter上以“Where Is Peng Shuai”为标签，表达对彭帅的关注，齐声询问“彭帅在哪里”。

负责世界女子职业网球赛事的WTA也同样发出呼吁 ，要求中共对彭帅的性侵犯指控进行彻底、公平和透明的调查。
