#### 编译：小红帽

以色列政府表示，不会寻求延长有争议的监视计划。在该计划中，其安全机构 Shin Bet通过手机跟踪感染了micron新冠病毒（Covid-19）变种的人。

“目前决定不再继续使用它监视，”总理纳夫塔利·贝内特（Naftali Bennett）和卫生部长尼赞·霍洛维茨（Nitzan Horowitz）周四在一份联合声明中说。

声明称，此举符合政府政策，该政策要求“谨慎使用并每天检查”跟踪。

去年在以色列大流行高峰期间首次实施的监视计划在上周日被重新提起，内阁成员对该举措表示不安。该措施是犹太国家对Omicron菌株反应的一部分，其中还包括关闭以色列对外国人的空中边界，并指定非洲大部分地区（而不仅仅是发现新变种的大陆南部）作为“红色区域”。

如将该计划延长至周四午夜之后，需要政府寻求以色列议会的批准，但国会议员几乎都不支持延长该计划。

公众也对该计划表示愤慨，四个人权组织向最高法院提交请愿书，要求取消该措施。就在首相和卫生部长发表声明前几个小时，该申请被驳回。

霍洛维茨后来在推特上解释说，政府决定不再继续进行电话追踪，因为“除了保护健康之外，我们还需要保护隐私和人权，即使在紧急情况下也是如此。”

Shin Bet使用的监控技术将感染者的位置与附近的其他手机进行匹配，以确定谁可能与该人有过接触。
