## 有声|独裁者之死—卡扎菲

配音：香草山佛教部 – 李傻
![This image has an empty alt attribute; its file name is Screen-Shot-2021-08-30-at-7.37.59-PM.png](https://assets.gnews.org/wp-content/uploads/2021/08/Screen-Shot-2021-08-30-at-7.37.59-PM.png)
卡扎菲是终结利比亚内战的重要人物，他也向世界证明了专制统治者的最后结局必定是灭亡。他出生于1942年的利比亚南部，1969年在一次没有流血的政变中，推翻了国王伊德里斯一世， 成立利比亚共和国，时年仅27岁。
![](https://assets.gnews.org/wp-content/uploads/2022/03/7-25.jpg)
卡扎菲在上台后，大搞文化革命，个人崇拜，强迫民众学习其思想，牢固控制媒体，称自己是利比亚的精神“向导”，民众在进食前也要背诵其著作绿皮书。卡扎菲建立个人绝对的权威统治，其个人意志主导了整个国家的意志，民众长期以来敢怒而不敢言。实行独裁统治，残酷镇压异议人士。利比亚的法律禁止一切反对卡扎菲的政治团体活动。违反该法律的人士被判刑，甚至被判死刑。

对外，为西非的几个反叛团体提供训练营地，支持一些激进组织，包括爱尔兰共和军和巴勒斯坦解放组织等。

1982年 ，在谴责利比亚支持国际恐怖组织之后，美国禁止从利比亚进口石油。美国前总统里根送卡一个绰号叫“中东疯狗”。

1986年，美国和利比亚的船只在锡德拉湾发生冲撞。利比亚还涉嫌参与了一起发生在柏林的夜店爆炸事件。两起事件让美国政府下令轰炸利比亚首都黎波里和第二大城市班加西。

1988年 ，一架泛美航空公司的客机在苏格兰洛克比上空爆炸，利比亚成为首要嫌疑国。

1992年，联合国对利比亚进行全面制裁。

2008年初，利比亚发生抗议和骚乱，北约开始对卡扎菲政权实施军事打击，利比亚反叛武装把卡扎菲从的黎波里驱逐。

2011年10月20日，过渡政府军取得大部份地区的胜利，最后向亲卡扎菲部队的据点发动攻击。卡扎菲在逃亡时遇到北约战机空袭，最后狼狈不堪，躲在两个水泥排水管中，身体多处被射中受伤，很快被发现拖了出来。被抓后，受到殴打躏虐折磨。最后，据报，车队在运送过程中，遭遇卡扎菲支持者袭击，在双方交火中，卡扎菲头部中弹而亡，死前，乞求士兵“不要开枪”。卡扎菲死后，民众纷纷涌出家门走上大街，不断欢呼，军人放枪，司机鸣笛，分享喜悦激动之情。

卡扎菲之死是象徵利比亚内战终结的重要标志，也是阿拉伯之春的重大事件。卡扎菲带给世界最大的贡献，就是揭示出：一切专制统治者的最后结局必定是灭亡。
