今天，澳大利亚移民部长亚历克斯·霍克亲自取消了他的签证，理由是认为他在澳大利亚的存在可能会煽动“内乱”并鼓励其他人避免接种中共病毒疫苗。明天的法庭，将是德约科维奇最后一次法律反抗的机会，如果失败，他将被禁止进入澳洲3年，这意味着他在年满37或38岁之后才有机会重返澳网，挑战他自己创造的九次澳网冠军纪录。

塞尔维亚总统亚历山大·武契奇（Aleksandar Vučić）在社交媒体上谴责“骚扰”和针对“世界上最好的网球运动员”的“政治迫害”，而外交部则表示他“被引诱到澳大利亚接受羞辱” 。
