## 赵长鹏继续跟着普京大帝玩儿，没有兑现其承诺捐赠乌克兰政府1000万美元。
` mayflowertranslate` [轉載自GNews](https://gnews.org/zh-hans/2191786/)

编译：波士顿五月花—-思澈

乌克兰加密货币交易所KUNA创始人称，币安加密货币交易所仍在继续与普京合作。

![](https://assets.gnews.org/wp-content/uploads/2022/03/908C362B-2DD6-4310-A122-AEAE8BFC6F8C.jpeg)

乌克兰加密货币交易所KUNA创始人迈克尔·乔巴尼安（Michael Chobanian）说，币安继续与俄罗斯政府积极合作，俄罗斯市场长期以来一直是币安的重要市场。

2019年，赵长鹏称俄罗斯是”我们的主要市场”，“我们一直在寻找任何社区的合作伙伴，特别是在俄罗斯。”在最近接受彭博社采访时，赵长鹏说，金融制裁不是一个”加密特定问题”，阻止所有俄罗斯人进入加密交易所将是”不道德的”。

Chobanian也是乌克兰区块链协会的主席，周四（2022/03/17）出席了美国参议院银行、住房和城市事务委员会关于 “数字资产在非法金融中的作用 “的听证会。在那次听证会上，他指责币安加密货币交易所没有对俄罗斯政府的制裁采取足够积极的立场。

Chobanian还表示，币安还没有做出它在2月份的一份声明中承诺的1000万美元的捐款。币安当时说，它将把钱送给乌克兰政府间组织和非政府组织，用于人道主义救援。
乔巴尼安说”他们说，他们向乌克兰政府捐赠了1000万美元。好吧，我没有看到那1000万美元。没有人知道去了哪里，”

点评：赵长鹏与俄罗斯和中共鬼混就是快步走进火葬场，如同郭文贵先生所言，赵长鹏的下场一定会比王健还惨。
