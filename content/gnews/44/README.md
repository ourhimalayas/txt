美国情报机构一直坚持认为Sars-Cov2是通过武汉的一个实验室事故进入世界的。与此同时，中共国正试图想出关于病毒进口的晦涩解释。世卫组织终于想对此事进行说明。

在寻找CCP病毒的来源方面，世界卫生组织（WHO）现在正在施加压力：在给成员国的一封信中，它列出了研究重点，并宣布成立一个新的常设工作组来寻找新病原体的来源。世卫组织负责人谭德赛强调，除了对2019年底首次出现病毒的中共国城市武汉的野生动物市场进行检查外，还必须对那里的实验室进行检查。

中共国已将世卫组织第一个研究小组的行程推迟了数月，尽管世卫组织希望继续进行调查，但中共国政府没有采取任何行动，允许这项工作继续进行。

这个问题很敏感。美国指责中共国阻止透明的分析，中共怀疑美国想把病毒的传播归咎于中共自己。当局正在散布牵强附会的理论，即病毒也可能通过来自国外的冷冻货物进入中共国。

经过几个月的争论，世卫组织的团队在1月份才被允许前往中共国，在3月底报告说：“很可能到非常可能”病毒是通过中间宿主从动物身上跳到人类身上的。人们认为该病毒“极不可能”意外地从病毒实验室中逃出并传播。

然而，美国却坚持实验室事故的论点。美国总统拜登在5月底表示，至少在美国情报机构的部分地区，认为这是可能的，并下令进行进一步检查。情报部门将在8月底进行报告。另一方面，中共国始终强调，寻找发源地必须是“全球”的。美国正以其“政治操纵”使搜索工作复杂化。

**评论：**距离美国总统拜登给出的90天病毒调查时间的最后期限越来越近，而近期WHO总干事谭德赛也提出要重新对武汉病毒实验室进行调查。德国媒体则仍在慢慢的透露着病毒的实验室事故理论，本报道中也写出了中共在向外，特别是美国推责。
多国及世卫组织要求前往武汉病毒实验室进行进一步调查，这明显是要从事实和科学角度来调查病毒来源，中共却一再称是将此事“政治化”，也就是说除了他给出的病毒自然理论外，其他的调查都是在将病毒溯源“政治化”。那你对李文亮所作的一切都是什么呢，是客观科学的对待么？
全球各国目前公开解决病毒的方法就是为公民接种疫苗，而疫苗安全性的讨论则愈演愈烈，特别是新型Delta变种病毒出现后，接种率最高的以色列由于感染率上升重新封国，这一定会导致夏天过后全球新一轮的大流行的加强。在疫苗拖延战术逐步失效以后，经济也会更加趋向低谷，各国还有什么办法安抚民众呢？如果没有，则最好的办法就是剑指中共，联合要求前往武汉病毒研究所进行CCP病毒溯源调查。
