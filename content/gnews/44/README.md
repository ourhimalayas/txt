（早报讯）英国首相约翰逊周六（4月9日）访问乌克兰首都基辅时表示，在乌克兰城镇发现平民的尸体，已“永久玷污”俄罗斯总统普京的名声。

综合外电报道，约翰逊说：“普京在布查和伊尔平等地方的行为是战争罪行，永久玷污了他和其政府的名声。”

约翰逊也称赞乌克兰人是“不畏困难”的英雄，抵御俄罗斯对基辅的进攻，“展现了狮子般的勇气”。他说：“俄罗斯人以为能在几天内就吞并乌克兰，基辅可在数小时内攻下……他们错了。”

约翰逊和乌克兰总统泽连斯基会面后也承诺，英国将为乌克兰提供援助，让该国“不会再被侵略”。他透露，英国将为乌克兰提供120辆装甲车及新反舰导弹系统，以应对接下来在乌克兰东部的关键战役。

此前，约翰逊已在周五（8日）宣布，英国将向乌克兰提供总值1亿英镑（约1亿7800新元）的“优质军备”，包括更多防空导弹与800枚反坦克导弹。

泽连斯基呼吁西方国家效仿英国，为乌克兰提供更多军事援助。
