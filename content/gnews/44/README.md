来自《每日邮报》2021年10月17日澳大利亚报道

**摘要翻译：**

他们会拿走你的肝脏，你甚至不会知道。中共国现实生活中的“鱿鱼游戏”；每周有数千人的器官被摘取，用于“按需杀人”的市场——而为什么世界却无力阻止。

自上个月在网飞公司（Netflix）首播以来，韩国恐怖剧《鱿鱼游戏》已经吸引了90多个国家和地区的观众，并迅速成为该流媒体平台历史上收视率最高的国际节目。

人权组织称，中国共产党每年从10万名名持不同政见者和政治犯身上摘除心脏、肾脏、肝脏和眼角膜，一个由政府运营的“按需杀人”器官贩卖网络在大规模运作。但国际社会仍然无力阻止屠杀，因为世界卫生组织被迫不加质疑地接受这个极权国家“不充分且具误导性”的医院数据。

就在Netflix热播剧上映前一周，在联合国人权事务高级专员办事处称，中共国“针对被拘留的特定种族、语言或宗教少数群体”并每年赚取10亿美元后，中共政府愤怒地否认存在国家支持的器官摘取计划。

来自人权理事会的九名联合国特别报告员，花了一年多的时间挖掘证人证词并调查中共国高度可疑的器官捐献率，以揭示可怕的“按需杀人”市场的新情况。

声明说：“联合国人权专家今天表示，他们对在中共国被拘留的少数民族，包括法轮功修炼者、维吾尔人、藏人、穆斯林和基督徒被‘摘取器官’的报道感到极为震惊。”

“根据收到的指控，据说从囚犯身上摘取的最常见器官是心脏、肾脏、肝脏、角膜，以及较少见的肝脏部分。”

这份严峻的声明中还指出，这种形式的所谓贩运，在很大程度上依赖于发誓保护患者的熟练的医护人员，包括“外科医生、麻醉师和其他医学专家”，以及各种公共部门专业人员的参与。

声明说：“一些囚犯如果不放弃自己的信仰或拒绝与警方合作，就会收到警方的死亡威胁和摘取器官的威胁。”

中共国器官移植系统的“危险信号”之一是，接受器官移植者可以在特定时间和地点预约手术。

在其他国家医疗系统中，这种情况不会发生，因为外科医生无法预测选择成为器官捐献者的人何时会死亡。

但是，作为2019年独立的中共国法庭听证会的一部分，向中共国医院拨打的卧底电话显示，在“按需杀人”的系统下，患者可以多快地接受手术。

尽管几十年来，中共政府支持的令人不安的器官贩运一直有据可查，但国际社会在阻止这一可怕的交易方面几乎无能为力。

她告诉澳大利亚《每日邮报》：“最近对中共国目前器官移植系统的一项统计分析表明，中共国一直以来提供的数据都是伪造的。”

“当你从中共官方数据中查看医院收入、床位利用率和手术团队数量时……这个数字更有可能是每年6万到10万例移植手术之间。”

中共国法庭发现，维吾尔穆斯林少数民族以及被禁止的法轮功修炼者，基本上被用作国内外买家的“器官库”。

尽管有这些发现，中共政府却一再否认在中共国存在强摘人体器官，称联合国的声明是“捏造”和“诽谤”。
