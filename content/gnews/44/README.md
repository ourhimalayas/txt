# 重要事件

- [**06.26 Zero Hedge：西方是否在为对抗共产主义中国而进行战略重组？**](https://www.zerohedge.com/geopolitical/west-strategically-reorganizing-fight-communist-china)
最近，西方世界的三个主要峰会分别为七国集团（G7）、北约和美国-欧盟的峰会相继举行，凸显了一个重大的战略转变，以对抗中共（CCP）带来的系统性威胁。如果说2020年是以大流行病肆虐全球和美中之间出现新冷战为标志的一年，那么2021年将是西方为对抗共产主义中国而进行战略重组的一年。


自2021年开始，中国共产党在推进其全球野心方面变得更加直截了当–在联合国与美国争夺霸权，宣称 “时间和势头 “站在它这边，”东方正在崛起，西方正在衰落”，并通过其所谓的 “狼性战士 “外交表现出越来越好战的姿态。中国共产党的强硬言论和行为大大加速了西方国家对其主宰世界的野心形成共识，促使西方开始对北京进行全面的战略重组。

6月10日，美国总统乔-拜登和英国首相鲍里斯-约翰逊签署了《新大西洋宪章》，承诺共同捍卫民主和基于规则的国际秩序。

紧接着，6月13日的七国集团峰会、6月14日的北约峰会和6月15日的美欧峰会都发表了公报，概述了战略重组的主要框架，以应对中共带来的重大威胁。

中共对台湾的挑衅不断升级

自2021年初以来，中共在台湾海峡的军事挑衅行为明显增加，该地区的战争风险也随之增加。在美国的带领下，西方国家多次联合发声回应，并加强了在台湾海峡、东海、南海和印度太平洋的军事存在，以遏制中共的军事冒险。

一周前刚刚举行的七国集团、北约、美欧峰会，加上4月16日的美日峰会、5月21日的美韩峰会、5月27日的日欧峰会、6月9日的日澳2+2会谈，都提出了台湾海峡问题，重申了地区和平与稳定的重要性。中共对民主台湾发动战争的战略压力空前加大。这表明，台湾问题现在已经上升到影响全球战略格局演变的风向标的高度。

对西方的 “系统性挑战

北约在其公报中首次将中共列为系统性威胁，称其公开的 “野心和自信行为对基于规则的国际秩序和与联盟安全相关的领域构成系统性挑战”。

北约秘书长延斯-斯托尔滕贝格称这次峰会是该组织的一个 “关键时刻”，呼吁各成员国对抗中国的崛起。

在敦促北约修改其战略方针以对抗中共威胁的同时，美国也在推动印太战略，即四国战略，并加强美日和美韩军事联盟。这表明，民主国家之间的军事联盟正在深化。

6月14日，北约各国元首和政府首脑在布鲁塞尔北约总部举行的北约峰会期间合影留念。

经济和技术挑战

西方不仅要应对中共的军事挑衅，还要应对其经济和技术挑战。为了确保中共不利用各种非法手段在世界经历数字转型的同时超越西方，有必要在共同价值观的基础上加强协调和整合，正如美国-欧盟峰会的联合声明（pdf）中所述。”我们决心推动数字转型，刺激贸易和投资，加强我们的技术和工业领导力，促进创新，保护和促进关键和新兴技术和基础设施。我们计划在我们共同的民主价值观（包括尊重人权）的基础上合作开发和部署新技术，并鼓励兼容的标准和法规。”

此外，声明还说，”为了启动这一积极的议程，并提供一个有效的合作平台，我们建立一个高级别的欧盟-美国贸易和技术委员会（TTC）”。

在这方面，在最近的峰会期间已经出现了一些积极的进展。例如，美国和欧盟达成协议，停止在空客和波音争端中长期存在的惩罚性贸易关税，并同意对抗 “非市场经济国家”。同时，加拿大和欧盟还启动了一个新的伙伴关系，以确保关键矿产的供应链，减少对中国的依赖。

B3W对抗中共的BRI

自2013年中共通过 “一带一路 “倡议（BRI，又称 “一带一路”）推出全球基础设施发展战略以来，许多国家陷入了债务和腐败的困境。七国集团峰会同意发起一个新的全球基础设施倡议，名为 “重建更美好的世界”（B3W），这是一个由主要民主国家主导的、高标准的、价值驱动的、透明的基础设施伙伴关系，以帮助缩小发展中国家超过40万亿美元的基础设施需求的资金缺口。这是西方对中国共产党的BRI做出的第一个战略回应。

结论

目前，西方国家对中共的立场仍然是三条腿走路–“合作、竞争和系统性对抗”。这表明，西方对共产党政权仍有一定的幻想和顾忌，还没有完全认识到中共的真实面目。任何一厢情愿的 “合作 “想法都会被中共利用，并被其蔑视。如果你还有疑问，只要看看中国外交部发言人赵立坚在6月16日的例行记者会上是怎么说的：”美欧对中国提出了所谓的’合作、竞争、系统性竞争’的三分法，其实质就是要在损害中国利益的同时，占中国的便宜。据中国新闻门户网站新浪网报道，”相信并鼓吹这种论调的人要么是自己的智力有问题，要么是低估了中国人的智力。

显然，西方正在进行的战略重组有其积极意义，但它需要更好地了解中共的真实性质，否则，西方将不可避免地继续被共产党政权所利用。
