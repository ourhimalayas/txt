而早在1月14日的大直播中，郭先生爆料，制作”零容忍”的团队已经多人被抓，并提到江泽民曾说过，“你们要懂得中共的力量，当中共说零容忍的时候，零的大小是由我们中共决定的”。郭先生评论道，所谓的“零”就是一把手和二把手加起来，零的大小完全由几个人决定，零的范围并不包括他们自己，中共一定会对一些所谓的腐败、违法行为和人员进行打击，但打击的对象、打击的方式、打击的力度没有任何客观标准，完全由中共根据自己的意愿决定。

分析人士指出，仅在网上能查到的2021年公安系统、法律系统落马的官员就有几十位，这就印证了郭先生所说，中共会用底下各级官员的滚滚人头平民愤、搞党内斗争，维持自己高层继续在中华大地上作恶的状态，这也是为习近平个人的二十大连任在党内定调。
