今年初，杭州的首套和二套房贷利率分别只有5.2%和5.4%，短短半年后分别上调了70和60个BP。7月2日，《中国证券报》记者致电多家银行予以证实。杭州市房贷再次上调，上调的银行包括工行、建行等，利率由此前的首套房5.6%、二套5.7%，提高到首套5.9%、二套6%。假设贷款300万元，等额本息30年还款，如果购房者买的是首套房，还款总额增加48万元，每个月多还1321元。

值得注意的是，不仅多家银行上调房贷利率，一些银行已经“暂停”受理房贷业务，尤其是二手房房贷正在收紧，甚至停贷。包括广州、深圳、合肥、武汉、南京等地，均有媒体曝出当地房贷告急，甚至有银行直接停办了房贷业务。虽然银行出面辟谣，否认暂停房贷业务，但仍难以抚平购房人的担忧，一些地方还出现了房贷放款周期拉长，需要漫长排队的现象。

进入7月份，中共国的经济衰退迹象已经非常明显，房地产是中共国经济的风向标，失去房地产对经济数据的支撑，将会对中共国经济产生“多米诺骨牌”效应。

如今推出的收紧二手房房贷政策，部分地区甚至取消了住房公积金贷款买房，消费者买房时不得不走商业贷款从而偿还更多的利率。这一切迹象都表明，中共国银行是真的没钱了，多年来赖以支撑的“房地产经济”岌岌可危。房价会出现大幅度下跌吗？有地区已经出现。二手房真停贷后，更可能的结果是大部分普通人无法卖出自己的房产，直白了说就是自己几十年来积累的最大财富将失去流动性，甚至财富归于零。这中共国经济崩塌的迹象越来越明显，假擀面仗子还能撑多久？
