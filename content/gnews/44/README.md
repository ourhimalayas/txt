据雅虎新闻网消息，英国路透社25日报道说，加拿大总理特鲁多呼吁，包括加拿大在内的西方国家需要团结起来，建立一个共同战线，他谴责中共试图让西方国家相互对抗。特鲁多谴责说：“自由市场中的中（共）国有时非常狡猾，让西方国家相互对立。我们需要比以往任何时候都要更加团结，更加果断。这样中（共）国就不可能用它的伎俩来让我们互相对立。”

此前，加拿大也加入了与美国，英国和澳大利亚等国采取的行动，不派政府代表参加北京的冬奥会。

2018年，应美国的要求，加拿大逮捕了中国电信巨头华为的首席财务官孟晚舟。她因与违反伊朗制裁有关的欺诈指控而被美国通缉。孟晚舟被逮捕后，中共国拘留了两名加拿大人，在一次闭门审判中这两人被指控收集国家机密和其它信息，这导致了两国关系的恶化。今年9月，在孟晚舟同意与美国达成认罪协议后不久，两名加拿大人也被释放，但中共国否认此事为“人质外交”。
