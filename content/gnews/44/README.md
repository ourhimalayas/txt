**1.****因中共病毒疫情长期化，政府考虑继续对航空企业减税**

[据时事通信社7月4日的报道，为了援助受中共病毒感染扩大长期化影响的航空企业，日本政府考虑2022年度以后继续减免航空燃料税。虽然有人预计随着国内中共病毒疫苗接种的逐步推进，旅客乘坐航空工具出行的需求会迅速回升，但以今年5月中旬的时点来看，不仅国际线旅客数比疫情发生前下降了5%，国内线亦下降了约20%。国土交通省的官员称航空业巨头的经营状况也不佳。](https://news.yahoo.co.jp/articles/aa4b5ae78f4491ff5203234da9f23f3b02e97438)

**2.****塞尔维亚奥运代表团1人中共病毒检测呈阳性**

[据共同社报道，日本厚生劳动省7月4日宣布，3日抵达东京羽田机场的塞尔维亚奥运会代表团的5名成员中，1名30多岁的赛艇运动员在机场的中共病毒抗原检测中呈阳性。据厚生劳动省称，该运动员已被送往疗养设施，同队的其余4人因为有密切接触史已送往机场附近的隔离设施。为了进行赛前集训，他们原计划前往富山县南砺市。据南砺市透露，集训将被取消，而去机场迎接的市政府职员与该代表队没有接触。](https://news.yahoo.co.jp/articles/ab1895764724e8a1a5aa4108409e0b4c7255d9d9)

**3.****东京奥运会4成场次将采取零观众措施，观众人数上限也作出调整**

[据共同社7月4日的报道，东京奥运会观众人数上限调整为会场容量的50%且最多5000人，而购票人数超过5000人的大规模场次（包括开闭幕式）以及晚上9点之后举行的赛事（约占全部场次的4成）将采取零观众入场的措施。由于东京最近中共病毒疫情再次扩大，6月决定的最大1万人的观众上限难以应对目前的局面。](https://news.yahoo.co.jp/articles/544219b0f0e139e490a6cd0f0373656f38e952fd?tokyo2020)

**4.****热海市再次展开失联人员搜救行动**

[据共同社7月4日的报道，针对静冈县热海市伊豆山发生的大规模泥石流灾害，县警与消防、自卫队4日再次展开失联人员的搜救行动。现场由于大雨的影响导致地面松软，搜救人员冒着二次灾害的风险展开工作。海上保安总部从3日开始连夜对被泥石流侵袭的伊豆山港沿岸地区展开搜救。泥石流灾害现场从上午6时开始陆续迎来警察局与自卫队的车辆与重型设备，泥沙的清除工作也已经开始。当地目前仍然持续阵雨的天气。海上保安总部称，泥石流造成现场附近的海水浑浊。](https://news.yahoo.co.jp/articles/5b2221b6c3e2f22a0d8a6f92b3541eace5f8674c)

**5.东京圈1都3县疫情蔓延重点措施将延长**

[根据NHK7月5日的消息，考虑到东京圈内1都3县中共病毒疫情恐再次扩大的现状，政府内部普遍表示必须延长疫情蔓延重点措施，本周四将作出决定。另外，如果延长该措施，东京奥运会夜间比赛和大型会场将采取零观众入场的举措，预计本周晚些时候将进行讨论。冲绳县的疫情紧急事态宣言和东京等10个都道府县的疫情蔓延重点措施原定于本月11日结束。而东京圈内1都3县的周平均新增感染人数增加趋势明显，令人担忧疫情再次扩大。](https://www3.nhk.or.jp/news/html/20210705/k10013119841000.html)

**6.****民众不满疫情与奥运会的应对措施，东京都议会选举使自民党感受到了秋季众议院选举的危机**

[据时事通信社7月5日的报道，自民党试图在7月4日的东京都议会选举中与公明党联合，夺回多数席位，但该目标似乎难以达成。民众不满菅义伟政府对中共病毒疫情以及东京奥运会的应对措施，自民党感受到了秋季即将举行的众议院选举的危机。“如果照此下去，众议院选举将不妙。”一名自民党高官毫不掩饰对4日晚东京都议会选举的焦虑。上次选举是在2017年，东京都知事小池百合子一马当先，自民党遭遇历史性失利，席位跌至有史以来最低的23席。](https://news.yahoo.co.jp/articles/d45b6391fcc154ab1ff578d8228719f08a73dee5)
