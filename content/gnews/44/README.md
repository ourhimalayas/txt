![]()![](https://gnews-media-offload.s3.amazonaws.com/wp-content/uploads/2021/06/19100909/%E9%97%AE%E9%97%AE%E9%97%AE.jpg)
据《每日曝光》（THE DAILY EXPOSE）报道，6月25日星期五，马特·汉考克报道，英国公共卫生局发布了一份报告，该报告显示，大多数CCP病毒的死亡人数属于接种过疫苗的人。统计表格显示，从2021年2月1日到2021年6月21日，在50岁以上的人群中已经有9571例疑似确诊的德尔塔变体病例，明显高于未接种者。其中仅在过去28天内就有8025人得到证实。
![]()![](https://i2.wp.com/dailyexpose.co.uk/wp-content/uploads/2021/06/image-133.png?resize=639%2C135&amp;ssl=1)
当CCP病毒疫苗获得紧急使用授权时，有关当局并不知道它们是否会起作用。进行的有限试验的接种疫苗者是否会患上严重疾病，没有衡量接种疫苗者是否仍会感染和传播病毒。
