**3. DeFi资产更值得拥有吗？**

compound和aave属于非生产性的代币。而uniswap和sushiswap这两种属于生产性的代币，它们是未来加密资产的圣杯，同时代表有治理的权益，但同时也有链上现金流的权利。而且它们允许持有者以协议的费用在某种情况下，拥有以通胀奖励的形式获取被动收益回报。

unitswap和compound都属于以太坊的区块链上去中心化的交易平台的协议，都是以那个Ethereum的代币提供流通性的交易作为服务模式，但都是用DeFi的一种应用。compound和aave属于借贷协议的交易合约。

aave和compound这些资金池可以做贷款的协议，可以做这么多银行的交易业务，也就是意味着以后我们各种各样的金融产品就直接会嫁接到像以太坊设计的区块链上，以各种交易合约的形式来进行的DeFi金融的应用。

DeFi就是干掉华尔街未来区块链的技术。 未来我们可能不需要华尔街了，所有这些大的金融投资集团或者是机构，都会被Blockchain上面的这些新兴的科技公司取代，以证券型代币的模式发股票。所以GTV可能也会以证券型代币的模式发股票。

中心化的金融系统是怎样运作的：比如说，我们去超市买东西，我们付款的时候刷卡，刷卡机会收到我们想要消费的指令，刷卡机会跟终端进行数据交互，终端会跟我们的银行或者是中央机构确认，我们卡里面是不是有钱等信息，终端收到中央机构确认的同时，把信息传递给我们刷卡的pos机，这样子我们交易就完成了。

现在所有的信息存储形式中心化了，被科技公司等社交媒体垄断，就形成了所谓的霸权。这已经背离了原来信息共享、信息透明、让全世界的人都可以公平的得到所有信息的初衷，现在已经很大程度上为利益服务。

中心化平台的所有信息是要与中央机构进行交互的，用中央的机构进行验证。比特币的验证用到共识机制，Concensus mechanism，而这也是挖矿的核心。比特币挖矿就是解数学题，第一个解完的有机会建立下一个区块链，并得到比特币奖励。在建区块链的过程中，为确保要交易的人的加密货币没有被花掉，就涉及到验证做数学题。这给了验证的时间窗口。矿工可以在区块链上的分布式账本去验证，货币没有被重复的使用过，这样子的验证之后，可以在整个区块链上broadcast已经确认的交易，可以进入到下个区块链建设当中。所以Prof of word，工作量的证明，就是刚刚说的验证像你交易是不是合法的。Concensus mechanism，共识化机制，就是去中心化地验证这些信息的核心。这工作量证明，这从字面上就能看出来谁干得多，谁得到奖励的机会就大；谁对这条链贡献的算力大、谁对这条链记账记得多，那么谁，就有可能更多的得到奖励。

分布式账本是去中心化的，以消除对中央机构或中介机构处理验证或认证交易的需要。他们在处理我们这些信息的时候，都有中央的处理器。如果把分布式的记账技术应用到这些科技项目上来时，是可以通过smart contract，把系统设置到不需要人为干预的状况，这样就可以保证项目的独立性。分布式账本中的所有时间都有时间戳，并被赋予唯一的密码学签名，分布式账本上的所有参与者都可以查看所有相关的记录，这就是现在去中心化最重要的一点：所有的参与者都有自己的账本。如果有谁想要去篡改账本，他必须把平台上所有的参与者51%的系统黑掉，才能更改记录这套系统。这是非常难的，所以这就是为什么区块链技术是非常安全的系统的原因之一。

**4. GTV的股票将来会是以STO的形式发行吗？**

STO，就是Security Token Offering，他代表的是一种资产证券权益。

Token其实就是代币，它可以分很多支。像现在比较火的，是加密货币Crypto currency。服务型公司或者卖产品的可以发行Utility Token。而在公司中的股份可以用Security Token越过天使轮A轮、B轮、C轮、D轮，直接把股份做成Token，在未来加密货币交易平台上进行交易，这就是项目股权的Token。

STO打通了一种新的投资渠道。

以前人们靠投资医疗健康行业的指数基金公司的股票，公司再去大学对接新技术。所以科研机构其实非常依赖这些医药公司，这导致医药公司变得中心化。而现在，比如做医药药物开发的 Molecure公司，通过Blockchain技术在平台上对接有治病救人技术的科研人员和金融市场的资金，打开了一种新的投资渠道。其意义就是去中心化，能够使supply和demand更好地、更多元化地结合在一起。而且在结合的同时，效率高、成本低，不需要所有的中介，比如上市公司、风投等环节。

现在的知识产权其实是很难进行交易的，我们没有非常成熟的市场，又没有办法给它拆分。但是通过区块链技术可以做出一套代币。过去知识产权可能就是大的科技公司拥有了，以后我们每个人都可以投。项目的知识产权可以做STO，把非流动性的资产变成可分割的流动性资产。各种项目也可以打包成证券型的Token，就像现在买股票基金一样。

理论上只要是有价值的东西，以Token的形式表现出来，那么在区块链上就可以买卖，可以抵押给银行，这样一定程度上就绕开了现实世界上市IPO这套繁琐的流程。喜马拉雅Exchange也可以是交易和融资平台。

未来期权期货等金融衍生品也都可以搬到区块链来。

Token可以承载信息、也可以承载价值、还可以承载合约。所以Token可以用一种稳定币的形式表现出来，也可以是一种社区币。

传统金融资产已经逐步进入数据经济，这就是变革的时代。不久的将来，会配合加密货币 ，AI，5G，之后才会形成整个新的货币金融体系辉煌的蓝图。
