![](https://assets.gnews.org/wp-content/uploads/2022/03/4-74.jpg)[图片来源](https://m.lbkrs.com/social/1605845)
3月21日，祥生控股集团在港交所发公告称，未能按期支付2亿美元债的1200万美元利息。

祥生控股集团在公告中表示，因受宏观经济、房地产市场环境及金融环境、多轮疫情等多种不利因素影响，本集团流动性出现阶段性问题。

据悉，上述美元债将于2023年到期，年利率为12%，有笔1200万美元利息应在2022年2月18日到期支付。按优先票据的条款，祥生控股集团有30天宽限期支付利息，但直至宽限期满，祥生控股集团并未支付利息。

此外，1月18日，祥生控股集团称，就一笔将于2022年1月23日到期的美元债完成了交换要约，根据交换要约有效提交作交换及获接纳的2022年票据已被注销。不过，因上述交换为非公开要约，目前尚未得知更多信息。

2月15日晚间，穆迪发布公告称，出于自身商业原因，撤销对祥生控股集团的“Caa2”企业家族评级和“Caa3”高级无抵押债券评级，评级撤销前展望为“负面”。
