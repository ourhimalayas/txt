【德国之声】 澳大利亚周三（4月13日）要求所罗门群岛总理苏加瓦雷不要与中共签署一份有争议的安全协议。上个月，一份泄露出来的安全协议草案在太平洋地区引起了震荡，特别是因为草案内容允许中共在所罗门群岛进行安全和海军部署的措施。
![](https://assets.gnews.org/wp-content/uploads/2022/04/image005-5.jpg)为了防止中共在太平洋地区扩张其势力，澳大利亚高官13日向所罗门群岛总理表示，希望该国不要签署与中共的安全协议。与此同时，一名美国高级将领也表示，所罗门群岛签署协议后可能会后悔。
这项消息加剧了美国及其盟国对中共可能在南太平洋建立海军基地的担忧，因为它们认为这将使北京的海军力量远远超出其国界。澳大利亚政府派遣太平洋事务部长赛索嘉前往所罗门群岛首府荷尼亚拉，亲自与该岛国总理会面。

赛索嘉在事后的一份声明中说，他要求苏加瓦雷“考虑不签署该协议”，并本着区域开放和透明的精神，与该地区的安全框架进行协商。

正在澳大利亚出访的美国海军陆战队司令伯杰13日警告，中共提出深化与所罗门群岛的安全关系将附带条件。伯杰暗示，与北京达成协议的国家可能会在未来后悔。
