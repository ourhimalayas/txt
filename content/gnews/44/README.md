**1.日本政府要求纺织企业加强应对人权侵害问题**

[据共同社7月12日的报道，经济产业省决定要求相关行业制定指导方针，促使国内纺织企业确认其业务合作伙伴是否存在强迫劳动和不公平待遇等侵犯人权的行为。因怀疑侵害人权，全世界正在逐步抵制来自新疆的棉花。日本政府将加强应对整个供应链下劳动者的权利保障。同日，经济产业省发布了一份报告，总结了纺织业的人权应对措施。](https://nordot.app/787216952244224000)

**2.****读卖新闻民意调查显示菅义伟内阁不支持率创新高**

[读卖新闻7月9日至11日实施的全国民意调查显示，菅义伟内阁的支持率为37%，与上一期调查（6月4日至6日）持平，同为去年9月内阁组建以来最低水平。不支持率则为53%，超过上一期的50%，创内阁组建以来的新高。今年不支持率已经连续3期超过支持率。支持率不高的原因据称是民众对中共病毒疫情以及奥运会的政府应对的不满。关于政党支持率，自民党为36%（上一期33%），立宪民主党5%（5%），无党派43%（48%）。](https://news.yahoo.co.jp/articles/384dbd1ef7a2e4b132887728c19d69103ba5a27e)

**3****.****内阁官房长官加藤胜信称将暂时免除中共病毒疫苗接种证明的发行费，且可当日领取**

[据时事通信社的报道，内阁官房长官加藤胜信7月12日在记者会上宣布，对证明中共病毒疫苗接种历史的“疫苗护照”，暂时免除发行费用。自本月26日起，可持接种证明发行申请书和护照去居民卡所登记的市区町村申请发行该接种证明。据加藤胜信称，接种证明可以在申请的当日领取。](https://news.yahoo.co.jp/articles/dfdc8bbb07042cf988277ca8afb7ba9ed81c9a0f)

**4.****日本四家啤酒巨头上半年销量均下滑**

[据共同社7月12日的报道，日本四家啤酒巨头2021年上半年（1～6月）啤酒类销量12日出炉。受中共病毒疫情下餐厅停业或停止提供酒类等影响，商用啤酒陷入苦战，各公司均出现同比下滑。关于包括发泡酒、第三类啤酒在内的啤酒类销量，麒麟啤酒销售额减少2%，三得利啤酒减少11%，札幌啤酒减少5%，朝日啤酒减少8%。2021年上半年因大城市圈等成为紧急事态宣言或防止蔓延等重点措施的对象，商用桶装和瓶装啤酒销售情况不佳。去年10月《酒税法》修改后税金增加的第三类啤酒市场也明显萎缩。](https://china.kyodonews.net/news/2021/07/c1a40dcf31e7.html)

**5.****酒业协会抗议经济再生担当大臣西村康稔的“粗暴发言”**

[据每日新闻7月12日的消息，经济再生担当大臣西村康稔要求酒类经销商停止与继续提供酒类的餐厅进行交易。针对其“粗暴发言”，酒业协会12日在自民党本部向政务调查会长下村博文提出抗议，要求撤回该命令。酒业协会会长吉田精孝对下村博文说，“酒类经销商被视为坏人，业内对此有很大的不满。希望向政府传达。” 下村博文回应称“我们深知当前的严峻形势，会做出努力。”本月9日，西村康稔撤回了他的言论，但仍然要求酒类经销商停止与餐厅交易。](https://news.yahoo.co.jp/articles/e10facb134e9dc73638b74a27795e9aee720f0c0)

**6.****大阪市等地削减8月上半月的中共病毒疫苗供应**

[根据NHK7月13日的报道，厚生劳动省公布了中共病毒疫苗8月上半月的供应量，对大阪市、名古屋市等有一定疫苗库存的地区，把按人口分配的“基本计划”中制定的供应数减少10%。削减的疫苗供应给疫苗不足的其他地方政府。厚生劳动省12日通报，1万盒辉瑞疫苗（含1170万剂）将于8月2日至9日向全国各地的市区町村配送。](https://www3.nhk.or.jp/news/html/20210713/k10013136051000.html)
