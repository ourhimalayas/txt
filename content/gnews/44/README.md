前言：中共国对文革从未允许过真正反省，更谈不上对文革的彻底否定和定罪，它事实上至今也未曾画上句号。按华国锋的意思，文革，这只是第一次。因为文革的本质就是愚者反对智者，这是中共统治工具的核心。这或许是邓小平看到的危机——被劫持的“国家利益”，一个不合法的政权。要想改变这一现状，文化大革命是一个迈不过去的坎儿，它触及到所有人利益。因此，邓小平（以下简称“邓”）选择了不反毛泽东，就意味着不反对共产主义体制，当然也就不反对文化大革命，因为**对文革彻底清算将会影响中共执政的合法性**。邓小平不是一个绝对的毛主义者，这一点可以确信，因为他也是文革受害者，但在邓看来，毛的极端做法，只会毁掉共产党。如何很好地对文革有一个交代，才是下一步开放国门的关键，即法理依据——拨乱反正——改革开放。可以说，特色的社会主义是改革开放的法理依据。在笔者看来，这是升级版完美犯罪心理使然。

**一、邓小平使镰刀斧头的功能增强**

拨乱反正运动是邓小平幕后主导，*在1978年12月的中共十一届三中全会上，邓小平取代中共中央主席、国务院总理华国锋成为中国实际上的最高领导人，否定了“以阶级斗争为纲”的路线，提出了“以经济建设为中心”的路线*。（见《[拨乱反正](https://zh.wikipedia.org/wiki/%E6%8B%A8%E4%B9%B1%E5%8F%8D%E6%AD%A3)》）这是一个爆炸性的消息，因此，中共国人记住了[十一届三中全会](https://zh.wikipedia.org/wiki/%E4%B8%AD%E5%9B%BD%E5%85%B1%E4%BA%A7%E5%85%9A%E7%AC%AC%E5%8D%81%E4%B8%80%E5%B1%8A%E4%B8%AD%E5%A4%AE%E5%A7%94%E5%91%98%E4%BC%9A%E7%AC%AC%E4%B8%89%E6%AC%A1%E5%85%A8%E4%BD%93%E4%BC%9A%E8%AE%AE)，被视为改变中国人命运的大会。首先邓的两名亲信进政治局常委，胡耀邦推选为中共中央政治局常委、中央书记处总书记，赵紫阳也进入中央政治局常委会。*华国锋迫于形势，辞去国务院总理一职，由赵紫阳接任*。（见《文革》）因为，这次大会否定了毛的继任者华国锋的“[两个凡是](https://zh.wikipedia.org/wiki/%E4%B8%A4%E4%B8%AA%E5%87%A1%E6%98%AF)”，实际上间接地否认了毛的极端路线。提出了《实践是检验真理的唯一标准》。但这另一些人受到鼓舞，认为共产党敢于直面改革。

在提出《实践是检验真理的唯一标准》实施前，*中共十一届六中全会一致通过了《关于建国以来党的若干历史问题的决议》，全面否定了文化大革命*。这一《决议》可谓实实在在检验了邓小平对文革的态度是否是遵循真理的原则下“坚持实践”。首先，人们必须要了解“实践”一词的中西方语用差距，中共对于“实践”的解释完全是上升到一种类似于马克思主义哲学的迷惑性：即[实践就是人们能动地改造和探索现实世界一切客观物质的社会性活动](https://baike.baidu.com/item/%E5%AE%9E%E8%B7%B5/5800441)。这就很好地解释了“能动地改造和探索现实世界”的活动。他为其改革一切的成败做好了完美的铺垫，即失败了也是实践中的能动性不足，如希特勒发动超限生物战的能动性、各样运动式杀人的能动性等等；而西方语境中对“实践”的定义：[一种通过反复学习的方法](https://en.wikipedia.org/wiki/Practice)，如艺术创作过程和艺术幻像实验和练习，从而使人的能力不断提高的现象等等。总而言之，比较具体化，而不是一种只能意会的虚无幻觉或任意解释。按照西方人理解的逻辑看，中国共产党领导人邓小平是要决心通过不断学习西方民主的过程中提高自身执政能力和执政合法性，而不是相反。可见，西方社会一直这样判断的。实际上，习特勒强硬做法和无视生命权的极端行为使国际社会完全看清楚了共产党的虚伪与邪恶。

正如德国之声2017年的一次专访中所提到的那样，“邓小平克服了文革和毛泽东主义，但[中国没有过像前苏联‘去斯大林化’那样的‘去毛泽东化](https://www.dw.com/zh/%E4%B8%93%E8%AE%BF%E9%82%93%E5%B0%8F%E5%B9%B3%E8%AE%A9%E4%B8%AD%E5%85%B1%E8%8E%B7%E5%BE%97%E4%BA%86%E6%96%B0%E7%9A%84%E5%90%88%E6%B3%95%E6%80%A7/a-46767811)’。”这是中共国的根本性问题，即中国掌握国家机器的人在深受其害后没有反省，反而继续高举“镰刀与斧头”戕害自己的居民（中国人非公民身份）的同时，在政治与经济实践中[增强了镰刀与斧头](https://youtu.be/L5CeyaosqrY%22%EF%BF%BDHYPERLINK%20%22https:/youtu.be/L5CeyaosqrY)的功能，并发动了第三次世界大战——超限生物基因战！但这一切正处于全面溃败中。

**二、邓小平修宪成仍为一纸空文**

邓小平掌权力后重新修订了《中华人民共和国宪法》，即全国人大于1982年通过了具有历史性意义的[《八二宪法》](https://zh.wikipedia.org/wiki/%E4%B8%AD%E5%8D%8E%E4%BA%BA%E6%B0%91%E5%85%B1%E5%92%8C%E5%9B%BD%E5%AE%AA%E6%B3%95_%281982%E5%B9%B4%29)，即《中华人民共和国宪法》1982年修订版。被其解读为“突出了宪政主义的精神和思想。”这是否和所描述的那样呢？

《八二宪法》里删除和修改了7条非法条款删除了诸如“在无产阶级专政下继续革命” 的文革语言。其中有几条是这样的：

- 取消了“国家由中国共产党领导”的条款。
- 规定了包括中国共产党在内的任何组织或者个人都不得有超越宪法和法律的特权。
- 对于关键的军队归属问题，废除七五宪法和七八宪法由中国共产党、中央委员会主席统帅军队的规定。相对于五四宪法，该宪法采用了折中方案，即设立一个中华人民共和国中央军事委员会领导武装力量。
- 中华人民共和国宪法 (1975年）中有关罢工自由的内容一同遭到删除，其修改理由此后一直难以服众。


增加条款中有哪些重点呢？这更是一个“实践就是检验真理”的能动性问题。事实上，取消国家由中国共产党领导的条款形同虚设捍卫国家利益、保障每一个家庭的安全是国家军队的神圣职责，而在中共国，人民解放军被勒令[永远听党指挥](https://epaper.gmw.cn/gmrb/html/2011-06/20/nw.D110000gmrb_20110620_1-03.htm)。也就是说，这意味着这个国家的权力机构——政府永远都不允许他的人民起来推翻它。这种典型的极权制度最大的弊端是，它的国家军队反而失去了军人的灵魂，他不可能是强大的军队，它也意味着这个国家政府也不允许它的枪杆子（军队）强大到不忠诚的时候起来推翻它。这就是一个恶性循环、逆淘汰的机制之一，笔者曾在《毛为集权统治设计矛盾体系》一文中说过，[毛泽东决不容许“枪指挥党”一言既出以后，中共国注定会弱化军队力量](https://gnews.org/zh-hans/1102074/)。而更可耻的是中共政党在自己规定的宪法“任何组织或者个人都不得有超越宪法和法律的特权”的前提下将其变为废话，因为此类 “任何组织或者个人”超越特权的例子举不胜举，妇幼皆知。由此可见，中共国《宪法》就是中共掩耳盗铃的可笑做法；而“中央军事委员会领导武装力量”直接被毛泽东“决不容许枪指挥党”的言论驳回。尽管邓小平时代修宪，但毛的许多重大政策都不能变。至于针对“罢工自由”被删除引争议一事更不值一提，在天安门64事件中可得到佐证，任何个人或团体，包括工人罢工、学生罢课都不被允许。

**三、邓小平的经济体制与集中力量办大事**

自《八二宪章》出炉后，1988年对该宪法进行补充，即“允许私营经济出现，并准许土地使用权转让”。当时绝大多数人认为这是“顺应当时的改革形势”作出的最大利好。即所谓“正式确立了“[社会主义市场经济体制](https://www.investopedia.com/articles/investing/081514/socialist-economies-how-china-cuba-and-north-korea-work.asp)”，并将中国共产党领导的“多党合作和政治协商制度”写入宪法中的序言。即后来人们所熟知的“中华人民共和国政治协商委员会”和“全国人民代表大会”的“两会体制。”这些看似有一定民主味道的东西，细分析起来，是一种彻头彻尾的自我暴露——既没有民主、也没有市场经济体，因为他的前提是社会主义。这是典型的“当婊子树牌坊”的经济体。

首先，社会主义经济与资本主义经济明显不同的特点是“[商品和劳务的生产是以使用价值为基础的”](https://www.investopedia.com/articles/investing/081514/socialist-economies-how-china-cuba-and-north-korea-work.asp)。这种使用价值受制于政府的需要，表面上看好像是“防止生产不足和生产过剩”，实际上是完全相反。比如，社会主义制度一定是专制政府，连可能的半民主制度都没有，那么，这样的政体下的商品和劳动力服从于政府需求时，就会在许多重大商品价值取向上偏离人民群众实际需要，比如CCP举国之力，制造生物武器争夺世界地缘政治的需求就不是市场需求所致。再比如，大型基建和不动产项目泛滥的时候，规划偏离实际需要就会劳民伤财，反而制造过剩产能，[中共国房地产即将崩溃](https://www.secretchina.com/news/gb/2021/06/25/976175.html)就是这个结局。而西方发达国家的自由市场经济“在资本主义经济体系中，商品和服务的生产是为了产生利润和资本积累，而不是基于它们的用途和价值。”显然，“产生利润和资本积累”才是人正常的追逐利润的心态，换言之，不赚钱的事不会去做，但赚钱的事一定要看商品和服务的“用途和价值”，而不是眼前利益或政治需要。一个显著的例子，生物学家在研究[COVID-19病毒毒性增强实验](https://youtu.be/wsKidRcF68U)时就背离了正常用途和其价值。而这恰恰证明了，为什么中共国一直热衷于社会主义市场经济，目的就在于此。中共在玩弄市场经济供需关系的文字游戏时可以利用互联网信息不对等现状为社会灌输错误信息，使其可以继续玩弄完美犯罪心里，认为社会主义市场经济的好处就是中央集中统筹规划办大事，而这个“集中力量办大事”就是一句完美犯罪式的欺骗，与之相反的是——[COVID19](https://www.worldometers.info/coronavirus/?utm_campaign=homeAdvegas1?)大规模杀人。

现行国际经济体主要就是资本主义体制，问题是，一个极大的国际玩笑是，社会主义经济体制目前只有中共国、朝鲜和古巴。而中共国又是世界第二大经济体。这好比在说，社会主义经济体可以和资本主义市场经济分庭抗礼。事实上显然不是这样，但中共国经济地位跻身世界第二又是一个事实。如果说，社会主义市场经济是非人性化的市场，那么这种不尊重市场规律的经济体做大做强的问题出在哪里？前美国副总统彭斯在「传统基金会」年度「李秉喆讲座」上的讲话时说：“[事实真相时：中国问题是美国一手豢养和造成的”](https://www.gettr.com/post/p4dgab)，中共的强大一直受到美国企业科技技术和某些公众人物以牺牲美国利益的基础上推动的结果。这是我们美国一手造成的。

道理很简单，中共国若失去美国的背书和各界技术支持，社会主义经济体的好处真的就不是集中力量办大事了，更不是高效，而是极端低能的朝鲜模式。可以说，它必将走向衰退而退出历史舞台。这是一个不争的事实。因为“集中力量办大事”的理论本身就是完美欺骗，在它给国家财富带来巨大效应的前提是你要“[伪装成自由市场经济体](https://www.reuters.com/article/usa-china-wto-eusuit-0617-mon-idCNKCS1TJ05N)”，这就是中共一直欺骗西方世界的原因所在。问题也很快来了，这成就了社会主义的国家资本主义。即按需分配的商品和服务产生的巨大利益都落到国家财政里去了，国家财政又被少数控制国家财富之人劫持着，而且，越是极权，控制控制国家财富的盗国者人数越来越少（盗国贼绑架国家的家族经济）。而劳动者的处境则越来越沦为一种被榨取利益后的牺牲品。

**四、结论：邓小平借“清污运动”向毛魂效忠**

中共国内人士一直认为邓小平执政后的“思想解放”是一个朝着正确方向迈进的举动。事与愿违，邓小平在1983年发起的“[清除精神污染](https://zh.wikipedia.org/wiki/%E6%B8%85%E9%99%A4%E7%B2%BE%E7%A5%9E%E6%B1%A1%E6%9F%93)”运动旨在对政治、文艺、思想界领域的一种自由意志出现的打压。认为在社会主义社会中出现“[异化](https://zh.wikipedia.org/wiki/%E9%A6%AC%E5%85%8B%E6%80%9D%E7%9A%84%E7%95%B0%E5%8C%96%E8%AB%96)”现象。中共认为，“这种异化现象倡导人道主义、接受非马克思经济理论、提倡艺术美学的自由主义倾向。”换言之，在邓的温和派专制政府，人有了“自由主义倾向”都是错误的，因此，清除这种“精神污染”是中共80年代一种新的社会主义言论审查的开始。在处理“什么该清、什么不该清”的政治争执中，影响了共产党内部各派别的争论，和对胡耀邦处理手法的评价。

那么，“异化”究竟是怎样一种现象？这是马克思用来批判资本主义市场经济主体中劳动者被资本家撕裂的现象。他的这种矛盾体理论把这种现象称为“外化”了，认为：“工人在劳动中耗费的力量越多，他亲手创造出来反对自身的，异己的对象世界的力量就越大，他本身，他的内部世界就越贫乏，归他所有的东西就越少。宗教方面的情况也是如此。人奉献给上帝的越多，他留给自己的就越少。”

事实上，马克思这种荒谬的理论正好可以利用它来打击中共国“社会主义经济体”本身被“异化”了的政治体制——特色的社会主义。在这种本身被异化了的政治体制下，工人在劳动中耗费的力量越多，他亲手创造出来的价值更多地被剥夺，通过高税收和离谱的商品价格使其劳动成果无形中被异化了。而劳动者若资本积累越大，就被允许进入高级劳动者阶级——[白手套](https://chinadigitaltimes.net/space/%E7%99%BD%E6%89%8B%E5%A5%97)（被中共允许强大的民营企业主勒令其为洗钱工具）。反过来，白手套的存在感越不安全，被异化了的阶级利益就越有问题。政府剥削的力量就越大，白手套本身，以及白手套内部世界就越贫乏，归他所有的东西就越少。这一点，人们可以参考阿里巴巴的马云以及死去的海航集团主席王健和陆续消失的白手套们。在这种被异化了的马云现在还有存在感吗？由此可见一斑，社会主义所谓市场经济体的存在就是家族经济国家化的剥削体制。在这个体制中，工人的劳动所得不是被所谓异化了，是被“家国化”了，相反，资本主义市场经济体每一个劳动者所得是根据你的劳动价值大小获得回报，他既不是“外化”了，也不是“异化”了，更不是“家国化”了，而是一种正常化了的而社会模式。举一个简单的例子，合理的劳动时间、富人多纳税的高税收模式下，穷人在期间获得相应的福利保护。商品价格更是市场供需所致。一种合理化的社会模式恰恰在资本主义经济体下得到展示。比如，中共国一句谚语所说，“人不为己，天诛地灭”。

一个正常的社会是以个体和家庭为单位的发展模式。只有每个人的利益获得保护的前提下，才有家庭的保障，才有个体和和团队间的合作产生的高效率和高创造力。因此，邓小平政府实际上是延续毛泽东的高压政策，即人的思想不能被释放，才有“思想解放”运动、人的言论不能各抒己见，才有本着为“国家安全”负责的言论审查制度。换言之，人的自由言论如果威胁到所谓国家安全，一个基本的思维逻辑是，这个国家已经威胁到所有人的基本生存权——威权主义。而邓小平在[八九64](https://youtu.be/AAJ8RsVqcm8)那天以实际行动证明了这一点。

*有一种说法认为：64的爆发标志着改革开放以来邓小平等人在中国大陆推动的后期政治体制改革失败*。这一说法是有争议的，从宏观来看，64天安门屠杀是中共今日准备要血债血偿的历史轮回。从狭义角度看，恰恰相反，64后，中共国经济真正开始好转，直到胡锦涛时期达到顶峰。事实证明，中共屠夫不因此受到国际社会摒弃，反而拿到国际社会“通行绿卡”。据BBC中文网报道称：“[当抗议学生来到美国大使馆门口要求避难时，使馆决定不予收容](https://www.bbc.com/zhongwen/simp/world-48100530)。”学运被镇压后，美国老布什政府玩弄虚伪外交，一边强烈谴责暴力、断绝与中国的军事合作；另一边先后派出两位特使访华，维系中美关系。并在一年后，美国无条件延续了中国享有的贸易最惠国待遇。老布什政府对华政策也仅仅是被批评“过于软弱”。今天看来，64事件恰恰是中共国向以美国为首的全世界发动生物战的一个伏笔。美国也因此上演了一场美国版“农夫与蛇”的故事。更加令人困惑的是，国际社会对暴政的惰性容忍到了一种令人发指的地步，中共发动生物战一年多来，西方世界还在讨论如何定性这一场战争并予以反击。这不是一场国际讽刺，而这恰是让国际社会看到了各国政客们集体堕落的表现。也正因为这一点，中共看到了金钱外交的优势在于举国之财力向国际社会个个击破地收买，使其“坏事变成好事”。将完美犯罪心理伎俩玩弄到一个无与伦比的水准，美前副总统称其为“[全知之眼](https://baike.baidu.com/item/%E5%85%A8%E7%9F%A5%E4%B9%8B%E7%9C%BC/8219624)”。讽喻中共自高地要取代上帝。而这进一步隐喻为消灭中共时刻临近了。
