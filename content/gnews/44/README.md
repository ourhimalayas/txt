浙江海事局本次发布的东海军演航行警告，明显是针对前日（7月15号），美军特种作战司令部专用军机，降落台湾事件发生后，所作出的仓促应对。

中共国防部对美军机抵台常态化的反应也显得十分苍白无力。毕竟，事实胜于雄辩。任凭你中共自顾自的在那边上演暴跳如雷的独角戏，奈何美国的军机已然轻松从容的起降台湾。眼下，美台双方的军事交流互动方面，已处于事实上的明面化、常态化。

本次划定的军演区域基本处于浙江近海水域范围，以浙江台州头门港码头附近海域为基准，相距台湾本岛的最近区间不足300公里。这也是中共军方近年期间内，所举行的历次军演区域，距离台湾本岛最近的一次。

恼羞成怒的中共这次恐怕是要“动真格”的了。竟首次表明将在本次为期6天的大规模军演当中 ——“实际使用武器训练”！

浙江海事局所发布的这则东海军演航行警告中，最为抢眼的便是将“实际使用武器训练”一句。其本质与实弹射击雷同。但此时使用这类一般不常用的技术性措辞，难免会给人产生某种遐想。难不成中共此前的绝大部分规模不等的各类军演，都是过家家似得假把式？

还别说，日常使用军事模型类教具训练，在中共军方一线基层部队的常规作训当中是极为普遍的。战士手持仿真枪在模拟训练的情况更加比比皆是。原因有三，首先是安全；其次是减少枪械磨损；再有就是降低弹药损耗。

试想如此严重脱离实战的作训模式，所训练出的部队，又有几分所谓强悍的战力呢？中共军方如若真的有其吹嘘的那般实力，就不应仓促的组织此类哗众取宠的所谓“实际使用武器训练”的军演。派兵登陆钓鱼岛，或者将入台的美国军机直接击落！这样方能显示出你中共军方强大威武的战力。
