**据《福克斯新闻》作者： Brian Flood，2021年7月19日发布:**

白宫因支持大科技公司”勾结”禁止COVID”错误信息”传播者而受到抨击，一名记者对古巴问题上的进步人士进行了坦诚的评估，拜登总统在MSNBC上受到抨击，这些都是今天的媒体头条。

罗利说：”CRT（批判性种族理论）活动家不会停止，醒目的左派不会停止，直到他们完全改变美利坚合众国。”

作家和独立记者海伦·罗利是来自中国的移民，她认为批判性种族理论伤害了亚裔美国人社区的成员，他们还没有取得美国功利主义所带来的成功。

“批判性种族理论是一种马克思主义意识形态，”罗利告诉福克斯新闻，”它基本上用种族划分取代了阶级划分。我们的国家正朝着错误的方向发展。”

作家肯尼·徐上周告诉福克斯新闻，亚裔美国人社区的成功驳斥了批判性种族理论。

亚裔美国人的成功驳斥了批判性种族理论，《不便的少数族裔》作者说。

“批判性种族理论的整个理念是通过说’嘿，我们的法律、我们的社会、我们的结构本质上是针对有色人种的堆积’来发展的，”徐说，”亚裔美国人证明，即使面对你所处的任何种族叙述，你仍然可以成功，这个国家是充满机会的土地。”

罗利同意徐的论点，但更进一步，宣称批判性种族理论实际上是在伤害亚裔美国人社区。她说，”亚裔美国人”是一个广泛的术语，包括多个族裔群体，继续谈论该社区的成功成员，加上批判性种族理论活动家的行动，会阻碍其他人的道路。

“这是一个非常多样化的群体，对吗？所以很难代表这些人，你知道，来自或接近20个不同国家的原籍和种族，”罗利说，”按收入计算，这个群体中有些人做得很好，有些人做得不是很好。”

她觉得批判性种族理论是有害的，因为还没有获得成功的亚裔美国人应该有机会享受到平等的择优录取。罗利担心人们不了解批判性种族理论的真正含义，她觉得它基本上是在告诉人们，由于他们的肤色，他们要么注定受益，要么注定失败。

海伦-罗利：作为一个移民，我在这个七月四日庆祝美国和美国人民的善良。

罗利说：”他们过度简化，把社会分为压迫者和被压迫者，所有白人就自然是压迫者，所有有色人种就自然被压迫。”

“因为亚裔美国人的总体经济成就、教育程度，批判种族理论者又不知道该把亚裔放在哪里，”她说，”所以他们就把亚裔美国人归类为与白人相邻；他们不属于被压迫的群体，所以我发现这很危险。”

罗利不希望亚裔美国人所经历的歧视被忽视，他指出，从1882年的排华法案到第二次世界大战中的日裔美国人被拘禁。

罗利说：”如此多的亚裔美国人在这些逆境中获得了成功，这有力地证明了美国梦对任何人来说都是可能的，肤色在这个国家不是命运。因此，批判种族理论是错误的。”

**什么是批判性种族理论？**

曾写过《反击：中国的侵略如何适得其反》、《孔子从未说过》和《破碎的欢迎垫》的罗利认为，那些歧视合格的亚裔美国人以维持其学生群体的特定种族构成的学校对贫穷的亚裔美国儿童造成了重大伤害。

她说：”活动家们所做的是关闭他们经济流动的唯一途径。他们来自贫穷的移民家庭。他们和我一样，没有其他资源，没有关系，没有遗产，没有代际财富，什么都没有……靠自己的努力上好学校，这是唯一的办法。批判种族理论活动家正在从他们身上夺走这个机会。”

罗利一直是批判性种族理论的直言不讳的反对者，她撰写的专栏标题为《亚裔美国人成为反对批判性种族理论的强大声音》和《清醒的亚裔美国人精英并不代表所有亚裔美国人》。她认为，许多其他可能成为批判性种族理论批评者的人因为取消文化而不敢说话，但在为时已晚之前我也需要做些什么。
