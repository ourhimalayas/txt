## 美议员批评白宫使用中共来源的检测试剂
` NFSC News` [轉載自GNews](https://gnews.org/zh-hans/1951246/)

作者：台湾农场 – zhong
![This image has an empty alt attribute; its file name is Screen-Shot-2021-08-30-at-7.37.59-PM.png](https://assets.gnews.org/wp-content/uploads/2021/08/Screen-Shot-2021-08-30-at-7.37.59-PM.png)
2月1日，据当地媒体揭露，美国政府邮寄给该国公民使用的中共病毒家用检测试剂盒中，有很大一部分是由中共国医疗设备制造商在加州的子公司生产的，其包装盒上明确贴有“中共国制造”的标签。
![](https://assets.gnews.org/wp-content/uploads/2022/02/截屏2022-02-03-上午8.44.52.png)Sohu
资料显示，该医疗设备制造商曾经获得过一家中共公司数千万美元的投资，但这一中共公司曾因与中共军方有联系而被美国政府列入黑名单中。据悉，联邦政府已经与该制造商签署了价值超过21亿美元的合约。

对此，美众议院医生核心小组主席温斯特鲁普在接受采访时批评道，联邦政府不应将美国的国家安全与健康依赖于中共国。相反的，政府应尽快从中共撤资，制造属于美国自身的、安全的医疗防护用品。同时，北卡罗来纳州的众议员也抨击道，美国正在与来自中共的病毒做斗争，为此，白宫不应该支持与中共政府有关联的制造商，尤其是那些与大流行病毒相关的中共医疗用品供应商。
