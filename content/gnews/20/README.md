我去俄罗斯，我特别不喜欢莫斯科，我喜欢圣彼得堡。圣彼得堡有它的神秘和它的文化、它的建筑和它的地域，而且我从那儿每次都飞土耳其，我飞到土耳其以后就感觉这是完全不同的世界。那么近的地方，靠近死海的地方，但是绝对又不是一个地方。俄罗斯有它绝对的文化和优势，绝对不是一般的优势。而且我说过这世界上，在欧洲最豪华的私人飞机、最大的飞机，甚至说私人飞机的升级版是由俄罗斯人代理的。这么看很腐败，百分之六七十豪华飞机是他们的，全世界最豪华的游艇几乎（占了）百分之七八十，甚至百分之八十以上，第一个创造者都是俄罗斯人。当你看到人家的私人飞机、私人游艇的时候，你就觉得中国人土到已经不能再土了，土得不行了。然后在莫斯科我最爱去的餐厅，不是中国餐厅，是在海德公园一个俄罗斯餐厅。你说我们中国人，到俄罗斯吃中国餐？我吃俄罗斯餐去。
