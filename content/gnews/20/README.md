## 谷爱玲的赞助商展示了“腐败和软弱的美国公司”

![](https://assets.gnews.org/wp-content/uploads/2022/02/image-1633.png)图片来源：foxnews.com

福克斯新闻主持人威尔·凯恩（Will Cain）在“塔克·卡尔森今夜秀”（Tucker Carlson Tonight）中解释了美国出生的谷爱凌（Eileen Gu）在北京奥运会上代表中国参赛的内容。

谷在北京奥运会上代表中国参赛。 她由“路易威登”和“维多利亚的秘密”等公司赞助。

“…… 很少有美国公司没有做过像谷爱凌一样的事情，” 凯恩说。 “他们已经背弃了美国以换取中国的财富，这就是为什么她是一个象征。你想看一些真正令人不安的东西——这不只是美国的忘恩负义的孩子，这是腐败的和软弱的美国公司的象征。”

凯恩指出“这是资本主义固有的不道德，缺乏道德、原则的本质。”

他称谷的背叛是“忘恩负义，背叛了这个不仅养育了她，还把她变成了世界级滑雪运动员的国家，因为只有美国才拥有才能提供她训练和设施。她却为了钱背叛了自己的国家，简直是可耻。”

凯恩将她比作一长大了搬出“温暖之家”的孩子，警告她说：她“很快就会……后悔”。

凯恩指出，中国网球运动员彭帅在去年 11 月指控中共前副总理张高丽性侵后“神秘失踪了”。

“我认为，谷爱玲将不得不牺牲掉她的美国护照，”他说。 “所以，欢迎去到中国。我希望你背叛美国所获得的明星和财富都值得，因为你实际上已经出卖光了。”
