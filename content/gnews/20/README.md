## 被挤出加拿大:中国移动CMLink业务今日停止运营
` MG FARM` [轉載自GNews](https://gnews.org/zh-hans/1802320/)

作者：Sloth
![](https://assets.gnews.org/wp-content/uploads/2021/12/1-366-e1640801108526.jpg)
今天一则突发信息震动加拿大华人圈，中国移动加拿大子公司12月28日突然在其官网宣布停止运营其CMLink业务。中国移动提供的中国内地和香港号码将被收回，会给许多用户带来影响，特别是那些使用这些号码办理微信支付和银行业务的用户。

据介绍，对于加拿大华人来说，国内手机号绑定了太多账号，如果收不到国内短信就会有诸多不便。想要回国探亲又不想忍受高昂漫游费就要换卡换号，而中国移动CMLink业务就给出了完美的解决方案——一卡双号，加中落地畅行无阻。

加拿大联邦政府以“出于国家安全考虑”为由，勒令中国移动国际公司（China Mobile International，简称CMI）剥离其在加拿大子公司的股份。该公司已向联邦法院提出申请，要求撤销这项决定。

据加通社报道：CMI是一家中国国有的电信企业，于2015年成立加拿大分公司，提供移动通讯服务，包括预付费电话计划等。该公司本身并不在加拿大境内拥有或经营任何电讯网络设施，只是租用研科电信（Telus Communications Inc）的网络提供服务。

现在还没等法庭做出决定中国移动加拿大子公司突然宣布停止运营其CMLink业务。

在中国移动加拿大官网“终止服务用户操作指南”中，中国移动加拿大表示，应加拿大联邦政府要求，中国移动加拿大子公司CMLink业务将于2022年1月5日起停止运营。

根据该指南，中国移动加拿大自2021年12月28日起（加东时间），停止新用户购卡、携号转入、CMLink一卡多号新申请、充值服务、套餐变更、补换卡。所有2022年1月5日前未启用的SIM卡无法再启用。

原有用户的主套餐自2021年12月28日起，将不再接受主套餐变更申请。在2021年12月28日当天仍有效的套餐，套餐到期后将免费续订，免费续订后的套餐可继续免费使用至2022年3月31日。2022年4月1日开始，所有套餐将无法继续使用。

有关叠加包，一卡多号业务，营销赠送等，CMLink也做出相应调整。此外，自动扣费将从2021年12月28日起，停止自动扣费，已绑定银行卡自动充值缴费服务会被取消。终止服务用户操作指南还就退款安排，携号转出做出安排。
