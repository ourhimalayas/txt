#### 编译：小红帽

随着澳大利亚新南威尔士州已接种两剂中共病毒（COVID-19）疫苗的居民的自由扩大，该州又出现了187例中共病毒病例和7例死亡病例。

根据该州修订中共病毒的路线图，从周一开始，完全接种疫苗的人可以在室内和室外进行更多的聚会，而学校的学生则可以进行集会和远足。

上周，议会委员会被告知，大约4900名教师尚未告知教育部他们的疫苗接种情况。

新南威尔士州即将为16岁及以上的人提供90%的全面疫苗接种覆盖率，其中89.8%的人已经接种两剂疫苗。

所有未接种疫苗的新州居民将不得不等到12月15日或该州达到95%的两剂疫苗接种率时才能享受更大的自由。

周日，在帕拉马塔、卧龙岗、库兰加塔和中央海岸，数百名身着白色衣服的抗议者参加了反对疫苗强制接种的集会，作为 “夺回防线（Reclaim The Line）”集会的一部分。

从周一开始，位于悉尼库多斯银行竞技场的大规模中共病毒疫苗接种中心关闭。该中心自8月9日开放以来已经提供了超过36万剂疫苗。该场馆将恢复举办体育和娱乐活动。
