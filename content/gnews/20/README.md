## 自由车队2022组织者在GoFundMe的最新声明
` GTV News Talk` [轉載自GNews](https://gnews.org/zh-hans/1926857/)

作者：加拿大温哥华扬帆农场 – Spirit
![This image has an empty alt attribute; its file name is Screen-Shot-2021-08-30-at-7.37.59-PM.png](https://assets.gnews.org/wp-content/uploads/2021/08/Screen-Shot-2021-08-30-at-7.37.59-PM.png)
自由车队2022在GoFundMe1月28日筹款已经过7百万美元，约有9万人参加了捐款。该组织者今日早些时候更新了声明，全文内容如下：

![](https://assets.gnews.org/wp-content/uploads/2022/01/unknown-33.png)

对加拿大同胞来说，政府嚣张时代应该结束。 现任政府正在实施的强制规定在破坏我们的企业、行业和生计的基础。

 我们是一个和平的国家，曾帮助保护全球各地的国家免受政府暴政对其人民的压迫，现在同样的事情却在加拿大这里发生。我们要把战斗带到联邦政府的门口，要求他们停止所有反人民的强制规定。

现在小企业被摧毁，房屋被破坏，人们被虐待，剥夺了人们最基本的生活需要。 作为加拿大人，我们有责任结束这强制措施。 这是当务之急，因为如果我们不这样做，我们的国家将不再是我们所热爱的国家。

 我们这样做是为了我们的子孙后代，也是为了重新获得我们的生活。 我们要求捐款用来帮助解决燃料费用，并希望得到食物和住宿，以帮助缓解这项艰巨任务的压力。 这是为我们的自由付出的一个小代价。 我们感谢你们的捐款，知道你们正在帮助这个曾经美丽的国家重塑她的面貌。

 为了让你们的慷慨捐款顺利进行，Go Fund Me的好心人将直接把捐款发送给我们的大宗燃料供应商，现在正在解决细节问题，这意味着你们辛苦赚来的钱将直接流向预定的对象，不需要通过其他任何人。 任何剩余的捐款将被捐赠给一个可靠的退伍军人组织，该组织将由捐赠者选择。 

筹集到的资金将被分配给我们的卡车司机，以帮助他们解决旅途中的费用问题。 资金将首先用于帮助支付我们卡车司机的燃料费用，如果需要，将用于协助提供食物，如果需要，将用于提供住所\*\*。

参考链接：[https://www.gofundme.com/f/taking-back-our-freedom-convoy-2022](https://www.gofundme.com/f/taking-back-our-freedom-convoy-2022)
