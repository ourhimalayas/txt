## [小雨专栏]-在路上 051——水篇

北方普遍缺水，不像南方雨水充足，到处山清水秀，小河星罗密布。但非常难得的是在外婆家门口，却有一条大河，一条小溪，大河不好玩，平时水深浪大，不敢近步；潺潺小溪却是我的挚爱。小溪旱季的时候小得几乎感觉不到，很像平时浇地的水渠，只有看到裸露的河床和岸边的杨柳才知道这是条小溪；小溪犬牙差（ci）互、明灭可见，时而聚拢成一汪池水，时而隐藏在草丛中，以为在此断流了，又突然从前面的乱石堆里冒出一汪清水。让人捉摸不定，又信心笃定，弯弯曲曲，又好像无处不在，柔软地平淌包容着，倔犟地蜿蜒伸张着，一直通向大河，通向大海……

那时候，外婆经常带着我来这里洗衣服，一般会选择溪水汇集的地方，坐在大石头上。外婆把衣服泡到清水里面，马上就会吸引一大群机敏的小鱼群围过来，溪水顶多到膝盖，我就会挽起裤腿装模作样地淌过去，每次鱼没逮着一条，衣服弄地倒是湿透。后来就索性不管不顾了，外婆在那边洗衣服，我就泡在水里扑腾，本来安静惬意的小溪被我折腾得不成样子，刚才的小鱼群瞬间了无踪迹，剩下我一个人在无趣地发呆。在水里时间长了，腿上就会引来蚂蝗，它吸血的时候身体缩成小小的一团，逃跑的时候身体又可以拉得长长的，很是神奇。没了小鱼来了蚂蝗，长大后才知道蚂蟥学名水蛭，外婆说蚂蝗附在腿上的时候，不要用力扯它，容易断，反而更麻烦，只要在蚂蝗身上稍微撒一点洗衣粉或盐它就自然脱落了。如法炮制，更好玩，我就故意把腿放到水中吸引蚂蝗上腿，然后把它们放在石板上，用洗衣粉在蚂蝗周围简单的撒一圈，然后看它们在圈里一伸一缩表演逃跑，乐此不疲。后来看《西游记》，悟空为保护师傅用金箍棒也画了个圈，难道悟空小时候也干过类似勾当？想想悟空也如此，自己就越发得意。

外婆家是依山而建，后面是一个不高的小山，虽然不高但每次拾阶而上还是气喘吁吁的，山上多是依山种植靠天收的粮食和蔬菜。外婆却唯独喜欢向日葵，春天的时候，她会把路两边到处撒种子，一到夏天，小路两旁排满笔挺的向日葵，个个昂首挺胸，迎着太阳，咧着笑脸，黄艳艳，亮闪闪，绿茵茵，风一吹，一起向您点头，像是在欢迎打招呼，又好像是在说向阳而生未来可期。

自助天眷顾，好运越逍遥；

可怜颠倒路，梦中一小草。

记得好像是八三年，有一场大暴雨，连着倾盆大雨几天几夜，引起山洪爆发，山上的洪水裹挟着泥浆填满了整个院子，屋门都不敢打开。大雨间隙，大人们忙着清淤修葺（qi），我也忙，我第一忙着我的向日葵，山路被冲得沟沟坎坎，路边靠天吃饭的向日葵也被冲得七零八落，有些还被冲走了，留下的经过雨水洗礼，更是精神抖擞，绿得可爱，还是那样地咧着大嘴笑，我和外婆小心翼翼地把它们一棵棵扶正、培土、压紧压实；我第二忙的是我的小溪，远远就听到涛声阵阵，原来的潺潺小溪，现在却像一条黄色的蛟龙，水漫树顶，摧枯拉朽，裹挟一切，金刚怒目，舍我其谁。

站在山坡上，身边葵花朵朵，绿叶拂面，花香袭人，远望小溪，黄蛟荡荡，横扫一切，抬望眼，黑云压城，山雨欲来，看来大人们又要忙了，我也要忙了……
