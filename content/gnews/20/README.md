## 我同住的老爸被证实新冠阳性后 我的处置纪录
` 香草山醫療部` [轉載自GNews](https://gnews.org/zh-hans/1902040/)

![](https://assets.gnews.org/wp-content/uploads/2022/01/logo-5-768x103-37.jpg)

作者：纽约香草山 医疗部 麦可医师



大概十天前，我爸留了小纸条在客厅的桌上：“我被检测出新冠阳性了，要隔离”。我戴口罩进入他的房间，他望着我，很久说不出话，手有点抖，慢慢说道：“老人活动中心检测出7人阳性，需要在家隔离“，后来我还知道，老人公寓有21户华人刚检出新冠阳性，其中有人每天都到老人活动中心活动，因为能够拿到政府补贴的免费中餐，价值每天$30，是比较吸引人的。
![](https://assets.gnews.org/wp-content/uploads/2022/01/老人新冠.png)
我爸是83岁的健康老人，是活动中心唯一一位每天自己开车来回的老人，他有血糖偏高，甘油三酯稍高。他每天都有吃我提供给他的药，包括深海鱼油，尼克酸Niacin，锌，维生素D，复合维生素，降脂药Atorvastatin，降糖药Metformin，还有日本的Umeken Beta Glucan 药丸 （增强免疫力）每天五美元，还有青蒿素吃5天停10天，每周硫酸羟氯喹200毫克。

老父亲的症状很轻微，自觉痰多，轻咳嗽，鼻子有分泌物。他没有乏力，发热，肌肉酸痛，胸痛，胸闷，没有气促，头疼，没有味觉嗅觉缺失，没有神经系统，心血管系统症状，他的胃口有下降。

我马上给了他如下的药物：

1.伊维菌素每天30毫克两天(合0.5毫克每公斤体重)；然后降到18毫克两天；然后15毫克两天，随后停药。

2.阿司匹林325毫克每天，吃三周（防止血栓），然后减到81毫克每天。

3. NAC 2400毫克每天，吃一周（抗病毒，痰液容易咳出），然后减到600毫克每天。

4.锌（Zinc Gluconate）增加到每天100毫克，一周后降为25毫克每天基础量。

5.维生素D3增加到每天10000IU，一周后降为基础量每天5000IU。

6.维生素C每天 2000毫克，一周后降为基础量每天 1000毫克。

7.Mucinex 一天四次，吃五天，使痰液容易咳出。

8.当伊维菌素停药后，开始吃黑孜然籽油（Black Seed Oil）5000毫克每天(合80毫克每公斤体重)，重新恢复每周的硫酸羟氯喹200毫克。黑孜然籽油一个月后减半量。

其他可以选择的药物包括褪黑色素（抗病毒），佛伏沙明（抗抑郁药的抗病毒作用），槲皮素（强抗氧化剂）。不建议使用抗菌素和激素。如有发热，喉咙痛，可用 Motrin 600毫克, 或Tylenol 625毫克每次，一天三次。

对老人的看护，需要密切注意他的呼吸，咳嗽频率，每两小时测量血氧含量，测量时病人不用带口罩。如果血氧常在93%左右波动，就要注意清理鼻孔，需要灌洗鼻腔。

对病人情绪方面的梳理包括：1）理解。2）不责备， 不要谈“怎么得上的？做错什么了？”。3）需要淡化，普通化，“很多人都感染上了，症状都不重”，“我们有药物，你身边有我们懂医学的”，“你身体内有永久性的抵抗力了！不需要打疫苗，抵抗力还比其他人好，您这次症状这么轻，都是平时保养和锻练的结果”。

在饮食方面，以清淡为主，多喝蜜糖水，喝粥，面食，水果，酸奶。

在活动方面，老人家喜欢上网看电视剧，我建议他经常做深呼吸，只要困了就休息。老爸在隔离期间一天能够睡12小时以上。房间内需要有加湿器，毛巾，纸巾，餐具要分开。

作为密集接触者的笔者，我刚好在老父阳性3天前受了风寒，在冰雨中搬运了四大袋化冰盐回家，第二天就有感冒，流涕等上呼吸道症状。在得知父亲新冠阳性后，我自己的用药也包括了伊维菌素，锌，维D，维C， NAC，剂量一样。我没有用阿司匹林，我发现我的鼻涕颜色有暗黄色，显示有鼻窦炎。喉咙痒，痰多，讲话不能说完整句子，说话就咳嗽，是支气管炎的症状之一。我果断给自己阿奇霉素五天(Azithromycin 处方药)，在咳嗽很厉害时，我用了Promethazine + Codeine 糖浆止咳，很管用。这个是处方药，也是瘾君子的入道之爱，不过可以放在家里以备严重干咳之用。

清洗鼻窦，是我每天的一个常规做法。先配置5%生理盐水 （1000毫升纯净水放入50毫克食盐），取90毫升配置好的生理盐水放入干净容器，加入药房买的10% Providone-Iodine 10毫升，稀释后的深棕色的液体为 1% Providone-Iodine灌洗液。在微波炉加热45秒成温热，用塑料注射器（网上买）10毫升分别注入鼻孔，头要往上扬，眼睛看天花板，全过程屏住呼吸。我的经验是30毫升灌洗液就可以灌满鼻窦，且有少量液体滴入口中，这时把头伸到洗手盆，让灌洗液流出，重复三次，直到灌洗液用完。之后用5%生理盐水灌洗两遍，整个过程10分钟，是FLCCC推荐的步骤。我自己在灌洗液中加入了几滴婴儿洗发水，摇匀用于灌洗，因为Penn State 的文章显示，只用 1%婴儿洗发水灌洗鼻腔都能灭活99.99%的新冠病毒，这和肥皂洗手是一个道理，只可惜一个简单的灌洗概念，都会被埋没一年多。

以上是我个人在最近10天内，处置关于新冠阳性老人的症状、治疗，以及家人防护措施的纪录，谨供参考。笔者没有进行新冠测试，因为我都做了我可以做的了。大概四周后，我们会抽血检查抗体。希望大家做好防护，备好药品，以积极的心态来应对一切。
