## 面对中共国日益增长的实力，美国国会正在战斗中
` 索尼克` [轉載自GNews](https://gnews.org/zh-hans/1355345/)

翻译：文复

在唐纳德特朗普的领导下，环城公路对中共国的看法从一种相对自满转变为一种近乎危言耸听的看法。

两党就需要对中共国实力的崛起做出重大回应达成了共识，现在推动了重大的参议院和众议院新的中共国立法。虽然这两项法案都是对中共国日益增强的实力的有力回应，但它们对美国对华政策的极端转向提出了关键选择。

参议院关于中共国的法案（战略竞争法案或 SCA，最近作为 S.1260 大中共国立法的一部分通过）充满了对中国动机和意图的煽动性断言。它包含的政策变化将大大增加与北京发生冲突的可能性。

其中突出的措施和言辞似乎将导致美国对台政策发生根本性变化。

尽管没有违反美国的“一个中国”政策，但该法案声称将台湾与中共国分开符合美国的重大利益。因此，该立法将破坏自 1979 年以来允许美国在没有对抗或冲突的情况下与台湾和中共国建立关系的长期政策。

台湾海峡 北京将台湾视为一个分离的省份。

该法案将禁止美国务院和其他机构限制美国政府官员与台湾政府“直接和经常性的”互动，并迫使国务院改变对台湾的其他外交协议。

该法案明确暗示，独立的台湾是美国至关重要的国家安全利益，甚至对夏威夷的防御至关重要。它还呼吁美国与台湾军方进行联合军事训练和演习，这是史无前例的一步，将直接加剧台海紧张局势。

美国政策的这种根本性变化不仅有可能引发军事冲突，而且还会使与中共国在共同关心的各个领域取得进展变得更加困难，包括贸易、气候变化和管理未来的流行病。

SCA 远远超出了拜登政府愿意做的任何事情，拜登政府已明确表示希望维持长期的台湾政策。这些措施大概是为了通过展示美国对台湾的重视程度来阻止北京对台湾采取激进措施。

但放弃“一个中国”的政策会适得其反，将中共国推向绝境，除了使用武力之外别无选择。此外，也没有确凿的证据表明中共国决定以武力夺取台湾。

在习近平的领导下，中共国并没有从根本上改变通过长期政治施压和恐吓而非直接军事入侵的方式以达到统一台湾的强烈偏好。

事实上，参谋长联席会议主席马克米利将军刚刚在公开证词中告诉国会，中共国目前没有意图或能力以武力征服台湾。

在中共国没有明确向军事力量转变的情况下，SCA 中的措施有可能升级并引发中共国更具侵略性的行动，甚至包括因台湾独立而与美国开战。这种紧张局势的加剧显然会增加对台湾人民和美国在该地区的盟友的威胁。

幸运的是，众议院外交事务主席格雷戈里·米克斯（Gregory Meeks）提出的战略竞争法的众议院对应法案（EAGLE 法案（H.R.3524）强调了同样的问题，但大大减低了加剧亚洲紧张局势的可能性。

与 SCA 一样，EAGLE 法案明确承认台湾是“美国印太战略的重要组成部分”，并以多种方式表达对台湾的支持。但它并没有包含迫使美国政府改变与台湾有关的数十年外交协议，并朝着军事上反对统一的承诺迈进的条款。

与 SCA 一样，EAGLE 法案包含广泛的政策声明和调查结果，表明美国承诺防止中共国在印度太平洋地区占据主导地位，支持台湾及其在那里的盟友，以及在世界其他地区与中国竞争。

然而，EAGLE法案也指出，美国和中共国应该“努力降低冲突风险”，改善两军沟通以“协助危机管理，利益一致的地方，通过与中共国合作来“加强稳定和减少猜疑”。

虽然 EAGLE 法案在印度-太平洋地区确立了一种侵略姿态，但它赋予行政部门灵活性，不会以升级军事紧张局势、疏远该地区的潜在盟友或破坏在共同关心的问题上取得外交进展的可能性来管理同中共国的关系。

可以预见的是，在 EAGLE 法案中达成的常识平衡已经导致一些人攻击它是“软”的。

据报道，米克斯主席承受着来自众议院对中共国鹰派的巨大压力，要求其纳入参议院法案中的措施，尤其是那些扭转美国数十年对台政策的措施。国会应该遵循米克斯主席的领导，拒绝这种做法。

美国正处于与中共国关系的转折点。人们普遍认为，需要进行重大变革。但这些变化可能以极端方式发生，通过引发与中共国的升级循环，危及我们的太平洋盟友和地区和平与发展，或者它们可能以一种方式发生，即果中共国愿意的话，美国也会乐于采取温和的路线.

虽然它是闭门进行的，很少受到公众的关注，但目前参众两院之间关于我们的台湾政策要走多远的冲突是做出这一选择的关键点之一。
