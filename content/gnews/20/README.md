2021年11月21日

在11月16日的一封公开信中，维加诺大主教呼吁建立一个 “反全球主义联盟”，联合起来对抗迫在眉睫的全球政变——新世界秩序的威胁。” 他说：”如果攻击是全球性的，防御也必须是全球性的。

去年6月，维加诺给美国总统唐纳德-川普致信并警告他，目前关于冠状病毒大流行和乔治·弗洛伊德暴乱的危机，是善恶力量之间永恒的属灵争战的一部分。

“正在进行的调查必将揭示那些不仅在医疗保健领域，还在政治、经济和媒体领域里操控这次疫情危机的人员的真正责任。我们将会发现，在这巨大的社会工程的运行中，有一些能决定人类命运的人，居然僭越职权，违背公民及其政府代表的意愿。”——卡洛-马里亚-维加诺大主教，2020年6月7日

在2020年10月25日给川普总统的第二封信中，维加诺主教在开头说到：“此刻，反上帝和反人类的全球阴谋，正在威胁着全世界的命运。”

在几天前发布的对全世界的呼吁中，维加诺主教提议“反全球主义联盟”组织起来，反对“一场全球政变。在这场政变中，金融和意识形态精英成功地控制了部分国家政府、公共和私人机构、媒体、司法机构、政治家和宗教领袖。”

维加诺说：“我呼吁执政者、政治和宗教领袖、知识分子和所有善良的人，邀请他们联合起来，组成一个联盟，发表反全球主义宣言。”

维加诺将他的视频和呼吁书发给了《网关专家》（The Gateway Pundit）网站，下面是这两份文件的副本。

以下是大主教呼吁的全文，其原文发布在《网关专家》（The Gateway Pundit）网站。

建立反全球主义联盟的呼吁

作者：卡洛-马里亚-维加诺大主教

两年来，我们目睹了一场全球政变，在这场政变中，一个金融和意识形态精英成功地控制了部分国家政府、公共和私人机构、媒体、司法机构、政治家和宗教领袖。所有这些人都不约而同地成为这些新主人的奴隶，因为新主人对他们许以权力、金钱和社会认可。直到昨天还被认为是不可侵犯的基本权利，以紧急状态的名义被肆意践踏：今天是健康紧急状态，明天是生态紧急状态，后天就会是互联网紧急状态。

这场全球政变剥夺了公民任何防卫的可能性，因为立法、行政和司法权互相勾结，完全背离了法律、正义及其存在的价值所在。这是一场全球性的政变，因为这种对公民的犯罪攻击遍布全世界，只有极个别的例外。这是一场世界性的战争，敌人是我们所有人，甚至是那些对正在发生的事情尚一无所知的人们。这场战争不是用武器，而是用不合法的规则、邪恶的经济政策和对天赋人权不可容忍的限制来进行的。

在很大程度上由这场政变的阴谋家们资助的超国家组织，正在干预各个国家的政府和数十亿人的生活、社交和健康。他们这样做当然是为了钱，但更多的是为了集中权力，以建立一个全球性的独裁政权。这就是世界经济论坛推动的大重置，联合国的2030议程。这是新世界秩序的计划，建立一个全球共和国来奴役所有人，以一种全人类的宗教来取消对基督的信仰。

面对这场全球政变，有必要组建一个国际反全球化联盟，汇集所有想要反对独裁统治、不愿成为一个隐形权力的奴隶、不愿取消自己的身份、个性和宗教信仰的人们。如果攻击是全球性的，防御也必须是全球性的。

我呼吁执政者、政治和宗教领袖、知识分子和所有善良的人们，邀请他们联合起来，组成一个联盟，发表反全球主义宣言，逐点驳斥新世界秩序的错误和偏差，并提出具体的替代方案，以共同利益、基督教的道德原则、传统价值，保护生命和自然家庭、保护企业和工作、促进教育和研究以及尊重创造为灵感，制定政治方案。

这个反全球主义联盟必须把那些有意愿摆脱暴政的地狱枷锁、确认自己主权的国家聚集在一起，与那些有共同原则、同样渴望自由、正义和善良的国家和人民达成相互合作的协议。它必须谴责精英阶层的罪行，查明责任人，向国际法庭起诉他们，并限制他们的过度权力和恶劣影响。它必须制止游说集团的行动，首先要打击腐败的国家官员和IT行业人员，并冻结用于破坏社会秩序的资本。

在那些政府屈从于精英阶层的国家，应当建立公民抵抗运动和民族解放委员会，包括社会各阶层代表，要求对政治进行彻底的改革，着眼于共同利益，坚决反对全球主义议程的新马尔萨斯项目。

我邀请所有想要捍卫传统基督教社会的人尽快召开一个国际论坛。各国代表聚在一起，要在论坛上提出严肃、具体、明确的建议。我向政治领导人和关心其公民福祉的执政者发出呼吁，抛弃旧的政党制度，以及权力和金钱奴役下的制度所强加的逻辑。我呼吁从东到西的基督教国家一起，邀请国家元首和机构、经济、劳工、大学、医疗和信息行业的正义力量加入一个共同的项目，颠覆旧有制度，击破这些人类的敌人企图分而治之的敌对行动。我们绝不接受对手制定的规则，因为这些规则正是为了阻止我们做出反应，并组织快速有效的反击。

我呼吁各国及其公民在我们的主耶稣基督的十字架下结盟，他是唯一的国王和救主，是和平之君。In hoc signo vinces.（凭此徽号汝必得胜）

让我们建立这个反全球主义联盟，让我们制定一个简单而明确的计划，让我们把人类从极权统治中解放出来，因为它集合了有史以来最残暴的独裁政权的罪恶。如果我们继续拖延，如果我们不了解笼罩在我们所有人头上的威胁，如果我们不组织起来进行坚定而勇敢的抵抗来予以反击，那么这个正在到处肆虐的地狱政权将无法被阻止。

愿全能的上帝帮助我们，保护我们。

+ 卡洛·玛利亚·维加诺大主教，前教廷驻美国大使，2021年11月16日
