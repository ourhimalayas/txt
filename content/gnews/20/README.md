![](https://assets.gnews.org/wp-content/uploads/2022/02/7-30.png)图为2022年2月19日，乌克兰军队正在乌克兰东部的一个镇上巡逻。(ARIS MESSINIS/AFP via Getty Images)
《大纪元》2月21日报道，周日，美国哥伦比亚广播公司的国家安全记者大卫·马丁在《面向全国》电视节目中说，俄罗斯当地指挥官已经接到了入侵乌克兰的命令，并正在制定具体的行动计划。

马丁告诉节目主持人，情报显示，俄罗斯军队实际上已经接到了入侵的命令。因此，他们不仅越来越接近边境，进入这些攻击地点，而且当地指挥官正在制定具体计划，以确定他们如何在各自的战区调动部队。

周五，美国总统拜登告诉记者，他确信俄罗斯总统普京已经做出入侵乌克兰的决定，并表示美国认为俄罗斯军队打算在未来几天发动进攻。

周日，美国国务卿安东尼·布林肯在接受《面向全国》的采访时，在节目中重申了拜登的说法。他说:“我们所看到的一切都告诉我们，我们认为普京总统做出的入侵决定正在向前推进。”

布林肯还告诉主持人，俄罗斯和亲俄乌克兰分裂势力最近制造了挑衅事件，原定于周日结束的俄罗斯和白俄罗斯的联合军演也将无限期延长。
