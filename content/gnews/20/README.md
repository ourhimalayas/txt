据盖特官方账户3月14日报道，跨媒体交叉发布功能于今日上线盖特。该功能将实现用户的原创盖文同步发布在所关联的推特账户中。这意味着盖特用户能够在无需重复登陆其他社交媒体平台的情况下，与跨平台的关注者第一时间分享信息，实现交叉互动。据悉，本功能面向所有现阶段的500万盖特注册用户。
