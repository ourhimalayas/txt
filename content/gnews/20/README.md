## 国际足联提议对俄罗斯进行处罚
` NFSC News` [轉載自GNews](https://gnews.org/zh-hans/2098840/)

作者：纽约香草山教育部 – Naughty(文行)
![This image has an empty alt attribute; its file name is Screen-Shot-2021-08-30-at-7.37.59-PM.png](https://assets.gnews.org/wp-content/uploads/2021/08/Screen-Shot-2021-08-30-at-7.37.59-PM.png)
2月27日，国际足联通过了一系列对俄罗斯的制裁措施，这些措施针对在3月举行，对俄罗斯至关重要的世界杯预选赛：禁止出现俄罗斯的名字、国旗和国歌，并为其比赛设立中立场地。

![](https://assets.gnews.org/wp-content/uploads/2022/03/10-3.jpg)

参与世界杯争夺战的另外三个国家波兰、瑞典和捷克共和国则表示，仅仅这样的制裁还不足够，他们拒绝与俄罗斯进行比赛，以抗议俄罗斯入侵邻国乌克兰。国际足联年度世界最佳球员罗伯特·莱万多夫斯基，以及其他几位顶级球员，都支持抵制任何与俄罗斯比赛的决定。

波兰足协主席库莱萨表示，波兰国家队拒绝与俄罗斯一起比赛，无论球队的名称是什么。瑞典足协主席尼尔森也表示拒绝与俄罗斯比赛，并敦促国际足联取消 3 月份涉及该国的季后赛比赛。

其他足球机构也对俄罗斯采取了抵制行动：欧洲足球管理机构上周剥夺了圣彼得堡今年欧洲冠军联赛决赛的资格；周日英格兰足协表示，在可预见的未来，它不会在任何国际比赛中与俄罗斯比赛。
