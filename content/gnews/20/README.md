10月26日，在联大2758号决议通过满50年的第二天，美国国务卿布林肯发表声明支持台湾参与联合国，并说美国鼓励所有联合国会员国支持台湾“有意义”的参与。中华民国总统蔡英文立即感谢布林肯支持台湾参与联合国体系，并说“我们将继续与美国和其他志同道合的伙伴一起努力，克服国际社会面对的挑战”。

10月23日，美国国务院发表声明说，美国与台湾官员就如何让台湾可能“有意义”参与联合国事务进行了讨论。显示美国对台湾的支持还在升级。在过去，美国只是利用台湾的防疫成绩而单一提到台湾加入世卫组织的议题，之后开始支持台湾进入气候领域的国际合作。

美国国务卿布林肯26日的声明则更加明确地指出：将台湾排除在外，有损联合国与相关组织工作，美国鼓励所有联合国会员国，支持台湾有意义参与。美国是众多视台湾为价值伙伴、可信赖友人的联合国会员国之一。在国际社会面对前所未有数量的复杂全球议题之际，所有利益相关者都能协助解决问题十分关键，“这包含2400万名住在台湾的人们”。

布林肯强调：“台湾有意义参与联合国体系不是政治议题，而是务实议题”。布林肯回顾：过去50年绝大部分时间，台湾强健参与特定联合国专门机构，证明国际社会认为台湾贡献有其价值，但台湾近来却不被允许对联合国工作做出贡献。布林肯举例，每年有数千万名乘客行经台湾机场，台湾在国际民航组织（ICAO）大会却没有代表；外界虽有许多能向台湾“世界级”防疫学习之处，台湾却未参与世界卫生大会,众议员麦考尔(Michael MacCaul)在众议院外委会共和党的声明中说，“联合国2758号决议没有说台湾是中国的一部分，没有说中国可以在联合国代表台湾，没有涉及台湾主权问题，甚至根本没有包含‘台湾’这个词。”

国台办发言人马晓光称，美国在中美建交前曾经有过企图在联合国搞“两个中国”的劣迹，最终以失败告终。2758号决议通过之日，也是美国等国提出的所谓“双重代表权决议草案”遭否定之时。美方一些人的错误在于，无视历史和法理事实，否定联大2758号决议的权威性，企图为联合国裁决翻案，与国际社会大多数对抗。

马晓光强调，台湾是中国一部分。中国台湾地区参与国际组织及其活动，必须按照一个中国原则处理。美方一些人的错误在于，无视一个中国原则和国际关系基本准则，甚至与美国政府一再重申的“不支持台湾加入任何必须由主权国家才能参加的国际组织”立场相违背。所谓将“台湾排除在联合国、国际民航组织、世界卫生组织之外”，完全是无稽之谈。

马晓光称，民进党当局长期以来叫嚣“台湾代表权”、“台湾的国际空间”等谬论，就是想把台湾从中国分裂出去，本质就是搞“台独”。美方一些人的错误在于，助长民进党当局“台独”行径，违背美国政府长期以来不支持“台独”的立场。。
