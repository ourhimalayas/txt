据国家公共广播电台(National Public Radio）12月1日报道，卫生官员周三说，美国已经发现了第一例中共病毒的omicron变体。据美国疾病控制和预防中心CDC称，该病例是在加利福尼亚州的一个人身上发现的。

CDC在一份新闻稿中说：“该人是11月22日从南非回来的旅行者，这个人已经完全接种了疫苗，症状轻微，情况正在改善，自从检测结果呈阳性后一直进行自我隔离，所有密切接触者都已被联系上，并且检测结果为阴性”。

旧金山卫生部门和加州的州卫生部门证实了这一病例。在一份联合声明中，这两个机构说，该州的测试和基因组测序监测，发现了该变体。他们说：“我们必须对这种变异体保持警惕，但这并不是恐慌的原因。”

加州州长加文·纽森（Gavin Newsom）在关于此案的简报中说，此病例涉及一名最近曾在南非的旧金山居民。他说，该居民于11月21日开始返回美国，并于11月22日在旧金山登陆。

据纽森说，这个人在11月25日出现了中共病毒的症状，三天后得到了检测。他还补充说，在 11月29日阳性结果出来后，样本就被送往加州大学旧金山分校，在那里进行基因测序，证实中共病毒是omicron变体。

纽森还说，该居民没有住院，而且有望完全康复。
