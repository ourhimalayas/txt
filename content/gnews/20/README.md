## 以色列强行推动第四针疫苗接种工作

作者：东京樱花团—在远方
![This image has an empty alt attribute; its file name is Screen-Shot-2021-08-30-at-7.37.59-PM.png](https://assets.gnews.org/wp-content/uploads/2021/08/Screen-Shot-2021-08-30-at-7.37.59-PM.png)
当地时间12月21日，据有关媒体报道，以色列政府宣布，将于近期向该国60岁以上的民众提供第四剂中共病毒疫苗。除此之外，以色列本周也将继续扩大其旅行禁令覆盖的范围，其中，美国、德国、意大利、土耳其、加拿大等地也将被包含在内。另外，政府也大肆鼓励公务员通过网络远程工作，避免外出通勤。
![](https://assets.gnews.org/wp-content/uploads/2021/12/截屏2021-12-23-下午11.15.46.png)
但资料显示，截至目前，以色列全国已经有超过584万人完整接种了中共病毒疫苗，大约占总人口的63.4%。然而，在新一轮奥密克戎变种病毒的疫情中，以色列至少出现了340例新变种病毒确诊感染病例，同时，该国也出现了首例奥密克戎死亡病例。

对此，有评论分析称，在中共病毒疫苗接种比例如此高的情况下，以色列民众并没有真正得到对病毒的预防和保护，但以色列卫生部却罔顾疫苗无用的事实，强行推动灾难性的第四剂中共病毒疫苗接种工作。
