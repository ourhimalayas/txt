根据观察者网报道：乌克兰首都基辅市长克里琴科（Vitali Klitschko）指责德国在俄乌问题上不坚定并拒绝向乌克兰提供武器。

22日，乌克兰政府谴责德国拒绝向乌克兰提供武器，并召见德国驻乌大使以示抗议。但德国对此不为所动，德防长坚称，德方当前不会向乌克兰提供武器，并将阻止爱沙尼亚向乌克兰出口德国制的武器，但许诺会以其他形式支持乌方。

乌克兰对此十分不满，加之德国仍坚持俄德合作的“北溪2号”项目，两国之间的外交上摩擦愈加严重。对此，德外长曾表示，如果俄罗斯“入侵”乌克兰，德国将履行与美国达成的协议，废弃“北溪2号”。

但克里琴科仍对德国的立场表示怀疑。

《华尔街日报》23日文章称，德国在俄乌问题如此谨慎，是因为德国过于依赖俄罗斯的天然气供应。

见此情况，美国国务卿布林肯出来稳定军心并在23日表示，他毫不怀疑德国在俄乌问题上对抗俄罗斯的决心。有美国官员透露，拜登政府最近忙于与欧洲、中东、北非等国政府和公司磋商以制定天然气全球战略，以防俄乌紧张局势令欧洲天然气断供。
