## COVID疫苗种族灭绝 儿童医院满是婴儿患者
` 西班牙巴塞罗那喜悦农场` [轉載自GNews](https://gnews.org/zh-hans/1658068/)

**编译：JennyBall**

![](https://assets.gnews.org/wp-content/uploads/2021/11/tempsnip15.png)图片来源：naturalnews.com

《自然新闻》([Natural News](https://www.naturalnews.com/)) 调查记者史蒂夫·基尔希 (Steve Kirsch) 说，全国各地的儿童医院都挤满了患有严重心脏问题的新生儿，这一切都是因为他们的母亲接种了武汉冠状病毒 (Covid-19) 的“疫苗”。

他收到的一封电子邮件称，一家医院的三个新生儿急救病房(NICU)都挤满了婴儿患者。一名护士从他们的记录中证实，所有婴儿都来自完全接种疫苗的母亲，但不允许医院工作人员直接询问母亲的疫苗接种情况。

“因为这是一个三手或四手的谣言，我通常会在与任何人分享之前尝试确认它，”该电子邮件声明作为警告。

“但我最初的互联网搜索结果是空的，我担心这种事情可能会被我们最大医院的控制者、有效地压制一段时间——尤其是那些目前正忙于解雇拒绝接种疫苗的医生和护士的医院 。”

“我想知道是否值得与 Unity 联合项目的一些科学/医学顾问医生分享，这些医生可能与 NICU 的大医院“内部”有联系，只要他们可以信任地进行谨慎调查，而不会因为在确认之前公布故事而危及他们或我们的可信度，”电子邮件继续说。

“如果这个谣言能够得到证实和揭露，它本身就有可能扭转这场战争的潮流。”

**医院重症监护室被“完全接种疫苗”的患者挤得水泄不通**

基尔希要求任何阅读过他著作的医生对这封信的说法发表评论，要么挑战要么证实。 数百人做出了回应，这里有一些他们的故事。（相关内容：基尔希还写过关于 2800 万儿童必须接种疫苗以被“拯救”，据说只有 14 条来自“covid”。）

一位人士说，最近几天发表了几篇文章，证实重症监护室里挤满了生病和垂死的病人，其中许多人不得不坐在走廊里，因为没有更多的床位。

这位人士补充说：“房间里挤满了非 COVID 症状的患者，也就是说，因为新冠病毒，他们不在重症室，”此人补充道。 “怀疑的原因是疫苗的继发效应。”

“这里的问题是，这些患者有多少很快就会成为婴儿？ **媒体和医学界不会站出来告诉你这些， 这些信息只能来自我们这些受够了的基层，媒体和医学界更害怕这个政府，而不是 COVID**。”

另一位写道，**自从开始对孕妇注射福奇流感疫苗以来，自然流产率已经飙升，这一点已经“明确确实”。**

“我的儿媳没有接种疫苗，刚分娩，她的医生告诉她在怀孕期间不要接种疫苗，因为她前面的**三个病人在接种疫苗后都流产了**，”该人士解释说。

“我认为基尔希先生为信息提供了值得注意的警告，直到得到更好的确认，但任何关注 COVID 疫苗研究的人都不会感到惊讶， 会得到证实的。”

一个人分享了一个据称是殡仪馆的人的[视频](https://www.bitchute.com/video/ADFUsv5lFd0p/)，他说他看到注射了中共病毒疫苗的新生儿死亡人数，是正常人数的许多倍。

“他不确定有多少是足月前的，但对于与怀孕/分娩相关的异常死亡而言，这并没有真正影响，”该人解释说。

其他人质疑基尔希收到的电子邮件是否合法，暗示它可能是“受控制的反对派”的“毒饵”。虽然这当然是可能的，但由于 Covid 疫苗被广泛使用，婴儿住院人数似乎确实有所增加。

“这不仅仅是关于有心脏问题的新生儿，”另一位指出，年龄较大的儿童也受到疫苗伤害。“在人们开始变得有点好奇之前，我们还需要多少次看到年轻人同时发生神秘的心脏病发作？”

另一个推测，**如果疫苗刺突蛋白能够穿过血脑屏障，那么它们对胎盘的渗透性可能也相同。**

这位人士写道：“如果其中一些婴儿出现令人讨厌的反应，我不会感到惊讶。”“什么样的恶魔会告诉怀孕的人不要喝酒，而是一定要扎针，并给她注射一些实验性生物材料？”
