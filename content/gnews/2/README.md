根据美国国家情报局局长办公室的年度威胁评估，中共国正在发展历史上最强大的核武器力量，而俄罗斯将利用一切机会破坏美国及其盟国。

众议院情报委员会于周一晚间发布的这份报告，在一份长达31页的解密文件中称，“来年，美国及其盟国将面临日益复杂和相互关联的全球安全环境，其特点是大国竞争和冲突的幽灵日益增加，对所有国家和行为者的集体跨国威胁，将在消耗我们的注意力和有限的资源。”

周二，美国最高情报部门负责人将在众议院委员会作证时提出并扩展评估，这项年度评估代表了全国17个情报机构对美国面临的主要威胁达成共识，并被立法者和政策制定者用作做出关键决策，推进立法和制定预算的基准。
但是，该项评估是在俄罗斯上个月入侵乌克兰之前编写的，并且基于截至1月21日的可用信息。

评估警告称，俄罗斯决心在短期内“主宰乌克兰和其他国家”，同时不希望与美军发生直接冲突。“莫斯科将继续使用一系列工具来促进其自身利益或损害美国及其盟国的利益，我们预计，当俄罗斯的利益受到威胁，且预期的行动成本很低，或者它看到了利用出现权力真空的机会时，莫斯科会将自己置身于危机之中。”

另一方面，中国共产党**“将努力推动台湾‘统一’，削弱美国的影响力，在华盛顿与其合作伙伴之间挑拨离间，并培养一些有利于其威权体制的规范”**。

美国情报机构称，中国“将继续其历史上最大规模的核力量扩张和核武库多样化”，因为北京“对限制其计划的协议不感兴趣，也不会同意参与锁定美国或俄罗斯优势的谈判”。如中共国控制台湾，可能导致全球半导体芯片供应链出现更多中断。
文件称，“中国仍将是美国技术竞争力的最大威胁，因为北京的目标是关键领域以及美国及其盟国公司和机构的专有商业和军事技术”。中共国“几乎肯定有能力发动网络攻击，破坏美国境内的关键基础设施服务，包括针对石油和天然气管道和铁路系统的攻击。”
