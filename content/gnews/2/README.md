西方国家将于周四警告俄罗斯总统弗拉基米尔·普京，他的国家将为入侵乌克兰付出“毁灭性”的代价，正值为期三天的北约、七国集团和欧盟峰会召开，美国总统乔·拜登将出席这一史无前例的会议。

这次为维护西方统一而举行的高峰会议将在布鲁塞尔的北约总部拉开帷幕，跨大西洋防御联盟的领导人将同意在欧洲东翼增加军事力量。

北约30个国家对俄罗斯在长达一个月的入侵僵持不下，拟将战争升级的前景感到震惊，将同意向基辅调遣装备以防御生物、化学和核攻击。

在七国集团发达经济体的紧急会议中，将重申以大规模制裁来惩罚莫斯科的决心。接着，随着27个欧盟国家的峰会，占世界GDP一半以上的国家将在一天内举行会议。

加拿大总理贾斯汀·特鲁多在欧盟议会说：“我们必须确保入侵一个主权独立国家的决定最终成为失败的战略，这会给普京和俄罗斯带来毁灭性的代价。”

根据联合国的数据，俄罗斯对乌克兰的袭击已造成数千人死亡，乌克兰4400万人中的近四分之一被迫逃离家园，其中包括超过360万逃离至邻国。

普京错误地将这次入侵描述为旨在使乌克兰非军事化和“去纳粹化”的“特殊军事行动”。乌克兰和西方称普京发动了一场无端的侵略战争。

“普京总统犯了一个大错误，”北约秘书长斯托尔滕贝格在抵达会议现场时说，普京低估了乌克兰人民的抵抗和西方社会的团结。

在这场“一代人以来最严重的安全危机”中，斯托尔滕贝格说，“只要我们团结在一起，我们就是安全的。”

乌克兰总统泽连斯基将通过视频会议在北约和欧盟峰会上发表讲话。

他曾恳求北约在乌克兰上空设立禁飞区，但没有成功，但西方盟国对俄罗斯实施了全面制裁，并为乌克兰的国防提供了价值数十亿美元的武器和援助。

北约大幅增加了其在东部边境的存在，约有40,000名士兵从波罗的海扩展到黑海。北约秘书长斯托尔滕贝格说，领导人将讨论在保加利亚、罗马尼亚、匈牙利和斯洛伐克部署四个新的作战部队。
