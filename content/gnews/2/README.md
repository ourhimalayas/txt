世界人权组织“人权观察”1月13日发布了连续32年《世界人权报告》。有关中国的部分涉及了香港、新疆、西藏、人权活动者状况、言论自由、宗教自由、大规模监控和中共病毒等14项内容。

中共在2021年加倍对国内外的压迫，在香港以《国安法》摧毁了香港的人权保障和公民自由，许多民主派人士遭到任意逮捕拘押；半民主制度已成为“橡皮图章机构”，“选制改革”规定只有效忠中共人士才可当选香港立法会议员； 香港数十家公民社会组织宣布解散；当局打压新闻自由，数家媒体被迫关闭；同时也体现在对网络言论、学术自由 和艺术审查上。使自由香港沦为中国的一个新的控制区域，成功地实施了一国一治的目标。

“人权观察”是一个国际非政府组织（NGO），以调查、促进人权问题为宗旨，建立于1978年，原名为“赫尔辛基观察”。它根据世界人权宣言等广为接受得人权标准，撰写国际人权违反状况的研究报告。2021年7月，中共宣布了对包括人权观察中国部主任索菲理查森在内的6人实施制裁。
