打针前

前一个月每天晚上都有喝vc，前一周每天一个鸡蛋，前三天每天吃两个鸡蛋(我也是看攻略说多补充蛋白质)。

打针当天早上吃了两个鸡蛋，喝了牛奶和1.5L水

打针第天

快10点打的，打针时没有什么感受，针推进去也不疼，过程很轻松结束

打针后

1. 看了攻略，打完针疯狂甩胳膊，甩了20分钟，并且按摩胳膊周围🙃

2. 喝了个vc泡腾片➕1L水

打完2h没有反应，快13点去吃饭(听说打完针最好吃鸡肉)，朋友说我充满了energy

3. 然后就去逛街，试衣服有一直在抬胳膊，然后16点开始(6h后)有点酸疼了，19点(9h后)胳膊用力会疼，感觉和平常好久没运动后突然运动后的状态一样

4. 晚上想早睡都没睡着，半夜觉得睡得不舒服迷迷糊糊醒来过，但是没有发烧也没有看表

打完第天早上

1. 6点(20h后)难受醒了， 感觉手部脸部很热，额头是冰的，不过应该是低烧，还没有之前生病烧的严重。胳膊就是打针部位抬手或者用力的话会有点疼

2. 吃了退烧药，期间可能吹了点风感觉自己手脚冰凉额头发热，吃完药1 h后开始出汗

3. 9点多(23h后)准备出门上课了，期间感觉没有烧，10点多药劲过去又开始烧，中午午睡16点醒来全身很热，感觉有点高烧了，等了会开始手脚出汗变冰凉

4. 吃了颗退烧药，过了会儿出了汗，整个人还是很虚弱

5. 18点(32h后)整个人应该处于低烧状态

6. 00.30睡前又吃了个退烧药，整个人上半身很难受，说不上来是吃退烧药胃难受还是胸闷，有点发冷

7. 第二天早上9点多醒来(48h)感觉烧已经退了，身体也轻松很多，抬胳膊还有点疼

8. 下午16点开始又有些低烧，身子很乏，一直晕晕沉沉到睡前都是低烧状态

打完第三天

1. 早上9点(72h)醒来，感觉已经好很多，但是还是会有胸闷的感觉，心跳也很快

2. 下午14点没有发烧，没有其他反应。16点多感觉有些低烧，没吃药

打完第四天

晚上睡前感觉头有点烧烧的

第五天就完全活力四射啦
