1. 自小被洗脑

没有常识，没有逻辑，没有思考能力，没有是非标准，没有自我提高和发展的意图。

只有被每时每刻的洗脑，训练成了赵家“爱国贼”，赵家的“爱国狗”——只要一摇“爱国”这个铃铛，就“汪汪汪”拼命咬。

主人叫咬啥就咬啥，叫咬几口就咬几口。

2. 一直被蒙蔽

不知道真实的历史，看不见真相，被选择性的事件和知识，浇灌出扭曲的价值观和思维方式。

不知道抗日战争是蒋介石民国政府打的。

不知道三十年共产实验差一点亡国灭种。

不知道“红色联盟”土崩瓦解，弃暗投明的穷哥们都过上了好日子。

不知道这三十年改革的成果被中共霸占攫取了，改革的恶果自己承担和面对了。

3. 自始至终被恐吓

以暴力获得的政权，必定以暴力维护。

刀把子，枪杆子，始终是统治的得力工具。

凡是洗脑无效，用杀戮，干掉反对的，吓死吓傻质疑觉醒的。

那滴血的刀，冒烟的枪口，从没有从百姓脖子上离开。

恐惧，深深的植根在每个人的心里。

几千万无辜迫害致死的尸体，堆积出大赵帝国的江山基石。

跪着的膝盖，弯着的腰，耷拉的脑袋，是赵家奴隶的政治状态。

4. 穷极一生被压榨

小时候，用作业分数，把孩子从三四岁，一直压榨到成年。

长大了，用钱和账单，终其一生，都在忙碌奔波。

不敢稍有停顿。

任何差错，停顿，意外，都会被赵家的车轮，碾压成碎片粉末，都会变成被社会抛弃的残次品，失败者。

中共战车上，捆绑了13亿奴隶，日夜劳作。

来换取，活下去的资本和机会。

奴隶在一分钱一分钱的节省，来应对天价的账单。

和珅在一亿佰亿万亿的贪腐，转移，糟蹋，撒币。

5. 至今被拖延

除了纯度太高的锑（Sb），大家心里都明白：民主，利国利民。但是，不利中共。

一百年前，前辈，在抗争民主自由。

一百年后的今天，我们，依旧在抗争民主自由。

民主自由，有两个标志性的基本条件：言论自由，一人一票。

审视这一百年，白白被耽误浪费引入歧途。

中共，拖延一天是一天。

百姓，苟活一天是一天。

中国的改变：

需要精英部分的责任担当。

需要民众部分的勇气跟随。

每一个中国人，把自己当人，学会思考，开始行动。

从谎言中剥离，从噩梦中醒来，从奴隶中挣脱出来。

成为一个人，对的起自己，对得起孩子，对得起这一生老天爷赐予的生命。

人活着，要像花一样美丽绽放。

不是如猪狗一样，苟且偷生，奔波劳碌。
