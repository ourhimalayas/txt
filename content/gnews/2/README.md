俄罗斯总统普京承认乌克兰东部两大亲俄区域独立，并下令派军进驻后，美国总统拜登最先宣布制裁乌东两地。

拜登21日深夜签署行政令，禁止美国人在自称为顿涅茨克人民共和国及卢甘斯克人民共和国的乌东地区进行新投资、贸易及融资。白宫发言人普萨基强调，针对俄罗斯的制裁会陆续有来，期望俄罗斯当局拭目以待。

日本首相岸田文雄亦于今日（22日）较早时间表示，日本不会接受任何违反国际法的行为，已准备好联同美国及其余七大工业国成员一同制裁俄罗斯。

另外，欧盟委员会主席冯德莱恩及欧洲理事会主席米歇尔在联合声明中也斥责俄罗斯入侵乌克兰的行为，违反了国际法，并承诺欧盟将会以制裁的方式对付是次违法行为的参与者。

**不过，最值得关注的还是英国，自前苏联1991年解体后，英国成为俄罗斯超级富豪转移资产及移民热门地点，伦敦蓬勃金融市场亦为俄企吸纳外来资金首选，相信英国的制裁将对俄罗斯的特权阶级造成最大影响。**

**英国首相约翰逊早前表示，当局就俄乌局势进行了紧急会议，并就针对俄罗斯的“一揽子制裁”已达成共识，预计将在伦敦时间22日中午宣布对俄的经济制裁，同时，约翰逊强调，此次严厉措施只属首轮制裁，随着俄方行动升级制裁力度亦将随之加码，不排除将覆盖俄罗斯及各地的寡头巨富。**
