西安事变结束以后，中国共产党承认服从三民主义及中国国民党之领导地位，国民政府分批释放狱中的中共党员，取消苏维埃政府及苏维埃制度，红军驻地改为陕甘宁行政区，执行国民政府统一法令及民选制度，经费由行政院与陕西省政府拨款；红军改编为国民革命军，服从国民政府军事委员会及委员长蒋介石统一指挥，其编制人员、给养及补充与中央军同等待遇，红军中之最精壮者改编为3个国防师，另有直属之工兵、炮兵、通信、辎重等部队，红军地方部队改编为地方民团或保安队，马步芳部停止在河西走廊对红军西路军的进攻。从此岌岌可危的中共得以苟延残喘。
蒋介石在《西安半月记》称：此次事变，为我国民革命过程中一大顿挫。八年剿匪之功，预计将于两星期至一个月可竟全力者，几全毁于一旦。而西北国防交通、经济建设，竭国家社会数年之心力，经营敷设，粗有规模，经此叛乱，损失难计。。。建国程度，至少要后退三年。蒋坚决相信，造成中国共产党坐大、神州沉沦之导火线就是西安事变。
张学良在60年代初曾表示：当时我对共产党还没有正确的认识，我以为在抗日过程中，也可以和共产党相处，抗日胜利后，大家也可以合作共同建国。谁知，后来共产党是伪装抗日，而藉抗日扩大自己的力量。。。 共产党欺骗了全中国人民，我就是被他们欺骗中的一人。。。实在痛苦万分。为了给祸国殃民罪大恶极的罪魁和骗子打击，如果有机会的话，我愿贡献一切，甚至我的生命。
