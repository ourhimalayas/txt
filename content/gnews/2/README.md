## 港府逼关六四纪念馆 封杀线上网站
` GTV新闻访谈` [轉載自GNews](https://gnews.org/zh-hans/1566943/)

栏目记者/责任编辑：修文

![This image has an empty alt attribute; its file name is Screen-Shot-2021-08-30-at-7.37.59-PM.png](https://assets.gnews.org/wp-content/uploads/2021/08/Screen-Shot-2021-08-30-at-7.37.59-PM.png)



香港「六四記憶 · 人權博物館」网站的工作人员9月29日证实，该博物馆网站在香港遭到封锁，港人必须通过VPN翻墙才能链接该网站。馆方从未收到港府的任何通知，是技术人员检测后证实，该网站已在香港境内遭封锁。
![](https://assets.gnews.org/wp-content/uploads/2021/10/1001AM（1、2、3、6、7）視頻備用.005.jpeg)
六四纪念馆最初由长期举办悼念六四烛光晚会的「香港市民支持爱国民主运动联合会」筹办，实体馆在今年六月遭港府关闭，八月初改以线上方式重新开幕，线上馆同时也脱离支联会，改由海外团队独立运作。支联会近期饱受港府打压，已于日前自行解散。六四线上馆虽已与支联会切割，但仍被中共视为眼中钉，在香港遭到封锁。馆方称其网站上的内容均为公开资料与可查证的史料，痛批此举是删除历史记忆的可耻行为。

中共对于六四事件的封禁，在拿下香港以后完成了闭环，中国国土上再无可以公开纪念六四的地方。
