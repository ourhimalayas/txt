## 【每日聚焦】俄乌战事3月26日简要
` 蓝莲花农场` [轉載自GNews](https://gnews.org/zh-hans/2235516/)

![](https://assets.gnews.org/wp-content/uploads/2022/03/GNEWSLOGO1.jpg)
**撰稿：蓝莲花农场 文斌**

1、基辅西线：3月25日，基辅西北部的乌克兰反击战在伊尔平取得了小的进展，乌克兰军队在该镇以前被俄军占领的地区行动的视频证实了这一点。当地民政当局报告说，3月25日在伊尔平、霍斯托梅尔和马卡里夫发生过激烈交战。乌克兰总参谋部报告说，基辅西北部的俄军在3月25日没有取得任何进展，只限于炮击乌克兰阵地；并称有数目不详的俄罗斯东部军区部队在切尔诺贝利地区建立了一个野战营地。

2、切尔尼耶夫和苏梅：乌克兰部队于3月24日在基辅以东进行了成功的反击，夺回了Luk’yanivka（Brovary以东40公里）和其他几个小城镇；并且声称摧毁了9辆俄罗斯装甲运兵车和2辆坦克，击毙了40名俄军。乌克兰总参谋部报道说，在当地时间3月25日中午，在Brovary附近的俄罗斯军队为应对乌克兰的反击而转入防御。乌克兰部队在当地时间3月25日中午击退了俄军对切尔尼戈夫市的攻击，致使俄军正在进行的包围切尔尼戈夫的努力仍然没有成功。在过去24小时内，俄军没有在苏梅周围进行任何进攻行动 。

3、哈尔科夫：过去24小时内，乌克兰东北部的军事局势没有改变。俄罗斯军队继续炮击哈尔科夫，但在过去24小时内没有进行任何地面攻击，并且不太可能在未来几周内恢复直接攻击该城市的努力。哈尔科夫的民政当局表示，俄罗斯军队在3月24日至25日继续炮击人道主义援助分发点和民用基础设施。乌克兰总参谋部报告说，截至当地时间3月25日中午，乌克兰部队击退了俄罗斯对卡米扬卡（伊久姆南部）的攻击。俄军在未能通过直接攻击攻占伊久姆后，可能打算包围该城，但无法成功突破乌克兰的防御。

4、卢甘斯克州和顿涅茨克州：俄罗斯部队在过去24小时内暂停了在顿涅茨克州和卢甘斯克州的地面行动，但重新集结了部队 。

5、马里乌波尔：逐街逐巷争夺继续，一部俄军进攻到市中心，但车臣军队声称在3月24日占领了马里乌波尔的整个左岸（东岸）的信息尚无法证实。马里乌波尔市议会还报告说，俄罗斯执政党统一俄罗斯党在被占领的马里乌波尔开设了一个办公室，被称为”人道主义总部”，以发布亲俄新闻。乌克兰总参谋部还报告说，乌克兰军队于3月24日在别尔江斯克击沉了鳄鱼级登陆舰萨拉托夫号，并损坏了凯撒-库尼科号和新切尔卡斯克号登陆舰。

6、赫尔松：俄罗斯部队在过去24小时内没有在南部方向进行任何进攻行动。乌克兰总参谋部报告说，俄罗斯部队暂停了恢复作战能力的行动，多达五个俄军营级战术小组在扎波罗热方向转入防御，但没有说明这些俄军是在第聂伯罗河的哪一岸。乌军声称击毙了在赫尔松附近行动的俄罗斯第49集团军司令官。

7、俄军继续空袭乌克兰中西部城市，昨天向文尼察的乌克兰武装部队司令部发射了6枚巡航导弹，其中一些导弹被乌克兰防空部队拦截。日托米尔也受到一枚俄罗斯导弹袭击。

8、俄罗斯总参谋部3月25日发布了一份关于俄罗斯入侵乌克兰的第一个月的虚构报告，声称俄罗斯的主要目标是占领整个顿涅茨克州和卢甘斯克州，慌称俄军已经完成了”第一阶段行动的主要任务”，慌称俄军已经沉重打击了乌克兰军队战斗力，所以使俄罗斯能够专注于占领顿涅茨克州和卢甘斯克州的”主要目标”。
