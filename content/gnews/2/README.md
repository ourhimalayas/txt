一场有关习近平新书的研讨会被临时取消，引发德国社会有关孔子学院的讨论：孔子学院是推介中国语言文化的使者，还是中共渗透海外的宣传喉舌呢？《世界报》发表评论称，即将离任的默克尔总理一向是孔子学院在德国最具影响力的倡导者。2016年，北德城市施特拉尔松德（Stralsund）开设孔子学院时，默克尔就曾亲自前往祝贺。即使后来孔子学院引发种种争议，但默克尔总理仍不为所动。

这篇题为《北京伸向德国的长手——作为宣传工具的孔子学院》的长篇评论写道：目前德国大约有20所孔子学院。按照孔子学院自己的说法，他们的使命是推动中国语言文化的传播。然而，细究之下，这种比较却是站不住脚的：该学院同德国大学建立合作伙伴关系，由双方共同分担运作资金。但德国方面所能做的，却只局限于提供场地，教职人员和教材则全部由中方提供。 而最大的问题则在于，孔子学院直接受中共宣传机器管控。

2012年，习近平就任中共国领导人以来，中共的宣传机器也变得更具有专制威权色彩。批评者认为，孔子学院是中共的特洛伊木马，其目的是美化中共国的专制统治。而最近发生的事件恰恰证实了这种批评。

本周早些时候，《世界报》发行人奥斯特（ Stefan Aust）和具有多年驻华经历的记者盖格斯（Adrian Geiges）在杜伊斯堡埃森大学和汉诺威大学举办新书（美化中共领导人习近平）讨论会的计划被临时取消，活动主办方则是上述两所德国大学的孔子学院。孔子学院取消新书讨论活动的决定，在德国引起了轩然大波。事实上，由于担心学术自由受到来自中国的政治干涉，杜塞尔多夫大学和汉堡大学已经宣布取消了同孔子学院的合作。

评论接着写道：两名德国记者的新书推介活动被取消，也许能加快德国大学同孔子学院分道扬镳的进程。在这个问题上，北京的大外宣们很有可能搬起石头砸了自己的脚。长期以来，批评者们一直在呼吁取缔德国境内的孔子学院，其中包括自民党，绿党和一些学者。杜塞尔多夫大学前不久已经终止了同孔子学院的合作。批评者们长期以来对孔子学院的怀疑，也终于得到了证实：即孔子学院就是中共扩展影响力的工具。
